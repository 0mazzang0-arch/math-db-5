## 문제
$x$ 에 대한 부등식 $| 3x-1 | < x+a$ 의 해가 $-1 < x < 3$ 일 때, 양수 $a$ 의 값은?

## 보기
① $4$   ② $\frac{17}{4}$   ③ $\frac{9}{2}$   ④ $\frac{19}{4}$   ⑤ $5$

## 풀이
[정답] ⑤ 

부등식 $| 3x-1 | < x+a$ 에서 

(ⅰ) $x \ge \frac{1}{3}$ 일 때 
$3x-1 \ge 0$ 이므로 
$3x-1 < x+a$ 
$\therefore x < \frac{a+1}{2}$ 
$x \ge \frac{1}{3}$ 이고, 양수 $a$ 에 대하여 $\frac{1}{3} < \frac{a+1}{2}$ 이므로 
$\frac{1}{3} \le x < \frac{a+1}{2}$ 

(ⅱ) $x < \frac{1}{3}$ 일 때 
$3x-1 < 0$ 이므로 
$-3x+1 < x+a$ 
$\therefore x > \frac{1-a}{4}$ 
$x < \frac{1}{3}$ 이고, 양수 $a$ 에 대하여 $\frac{1-a}{4} < \frac{1}{3}$ 이므로 
$\frac{1-a}{4} < x < \frac{1}{3}$ 

(ⅰ), (ⅱ)에서 부등식의 해는 
$\frac{1-a}{4} < x < \frac{a+1}{2}$ 

그런데 주어진 부등식의 해가 $-1 < x < 3$ 이므로 
$\frac{1-a}{4} = -1$ 
$\therefore a = 5$
