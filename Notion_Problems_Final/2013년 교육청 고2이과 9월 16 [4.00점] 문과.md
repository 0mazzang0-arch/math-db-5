## 문제
다음은 $2$ 이상의 모든 자연수 $n$에 대하여
$$\sum_{k=1}^{n} \frac{1}{k^3} < \frac{1}{2} \left( 3 - \frac{1}{n^2} \right) \cdots (\star)$$
이 성립함을 수학적 귀납법으로 증명한 것이다.

(ⅰ) $n=2$ 일 때,
(좌변) $= (가)$ 이고 (우변) $= \frac{11}{8}$ 이므로 $(\star)$이 성립한다.
(ⅱ) $n=m$ ($m \ge 2$) 일 때, $(\star)$이 성립한다고 가정하면
$$\sum_{k=1}^{m} \frac{1}{k^3} < \frac{1}{2} \left( 3 - \frac{1}{m^2} \right)$$ 이다.
$n=m+1$ 일 때,
$$\sum_{k=1}^{m+1} \frac{1}{k^3} = \sum_{k=1}^{m} \frac{1}{k^3} + (나)$$
$$< \frac{1}{2} \left( 3 - \frac{1}{m^2} \right) + (나)$$
한편,
$$\frac{1}{2} \left( 3 - \frac{1}{(m+1)^2} \right) - \left\{ \frac{1}{2} \left( 3 - \frac{1}{m^2} \right) + (나) \right\}$$
$$= \frac{(다)}{2m^2(m+1)^3} > 0$$
이므로
$$\sum_{k=1}^{m+1} \frac{1}{k^3} < \frac{1}{2} \left( 3 - \frac{1}{(m+1)^2} \right)$$ 가 성립한다.
따라서 $n=m+1$ 일 때도 $(\star)$이 성립한다.
그러므로 $2$ 이상의 모든 자연수 $n$에 대하여 $(\star)$이 성립한다.

위의 (가)에 알맞은 수를 $a$, (나), (다)에 알맞은 식을 각각 $f(m)$, $g(m)$이라 할 때, $\frac{g(a)}{f(1)}$의 값은?

## 보기
① $35$   ② $36$   ③ $37$   ④ $38$   ⑤ $39$

## 풀이
[정답] ①

(ⅰ) $n=2$ 일 때,
(좌변) $= \frac{1}{1^3} + \frac{1}{2^3} = 1 + \frac{1}{8} = \frac{9}{8}$
따라서 $a = \frac{9}{8}$이다.

(ⅱ) $n=m+1$ 일 때,
$\sum_{k=1}^{m+1} \frac{1}{k^3} = \sum_{k=1}^{m} \frac{1}{k^3} + \frac{1}{(m+1)^3}$ 이므로
$f(m) = \frac{1}{(m+1)^3}$이다.

또한,
$$\frac{1}{2} \left( 3 - \frac{1}{(m+1)^2} \right) - \left\{ \frac{1}{2} \left( 3 - \frac{1}{m^2} \right) + \frac{1}{(m+1)^3} \right\}$$
$$= -\frac{1}{2(m+1)^2} + \frac{1}{2m^2} - \frac{1}{(m+1)^3}$$
$$= \frac{-m^2(m+1) + (m+1)^3 - 2m^2}{2m^2(m+1)^3}$$
$$= \frac{-m^3 - m^2 + m^3 + 3m^2 + 3m + 1 - 2m^2}{2m^2(m+1)^3}$$
$$= \frac{3m+1}{2m^2(m+1)^3}$$
이므로 $g(m) = 3m+1$이다.

따라서
$a = \frac{9}{8}$
$f(1) = \frac{1}{(1+1)^3} = \frac{1}{8}$
$g(a) = g\left( \frac{9}{8} \right) = 3 \left( \frac{9}{8} \right) + 1 = \frac{27}{8} + \frac{8}{8} = \frac{35}{8}$
$$\therefore \frac{g(a)}{f(1)} = \frac{\frac{35}{8}}{\frac{1}{8}} = 35$$
