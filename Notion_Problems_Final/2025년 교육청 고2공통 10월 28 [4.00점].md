## 문제
닫힌구간 $[0, 2]$에서 정의된 함수 $f(x)$가 다음 조건을 만족시킬 때, $\frac{f(0)}{f(2)}$의 값을 구하시오.

(가) 함수 $f(x)$는 $x=1$에서만 불연속이고, $\lim_{x \to 1} f(x) = 3f(2)$이다.
(나) $0 \le x \le 2$인 모든 실수 $x$에 대하여 $f(x) \ne 2$이고, $f(0) + f(2) = 4$이다.

## 풀이
함수 $g(x)$를 
$$ \begin{array}{l} g(x) = \left\{ \begin{array}{ll} f(x) & (0 \le x < 1, 1 < x \le 2) \\ 3f(2) & (x = 1) \end{array} \right. \end{array} $$
이라 하자. 
조건 (가)에 의하여 함수 $g(x)$는 닫힌구간 $[0, 2]$에서 연속이다. 
또한 조건 (나)에 의하여 $g(0) \ne 2$, $g(2) \ne 2$이고 $\frac{g(0) + g(2)}{2} = 2$이므로 
$g(0) < 2 < g(2)$ 또는 $g(2) < 2 < g(0)$이다. 
그러므로 사잇값 정리에 의하여 $0 < c < 2$이고 $g(c) = 2$인 $c$가 적어도 하나 존재한다. 
조건 (나)에 의하여 $0 \le x < 1$, $1 < x \le 2$에서 $g(x) = f(x) \ne 2$이므로 $c = 1$이다. 
$g(1) = 3f(2) = 2$에서 $f(2) = \frac{2}{3}$이다. 
$f(0) + f(2) = 4$에서 $f(0) = \frac{10}{3}$이다. 
따라서 $\frac{f(0)}{f(2)} = 5$이다.
