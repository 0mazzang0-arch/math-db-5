## 문제
파장이 $\lambda$ 이고 강도가 $I_0$ 인 광선이 어떤 용액을 통과하면 이 용액에 광선이 흡수되어 입사광의 강도가 감소된다. 광선이 농도가 $c$ 인 용액을 통과한 거리가 $d$ 일 때의 광선의 강도를 $I$ 라 하면 $I$ 와 입사광의 강도 $I_0$ 사이에는 
$$I = I_0 \times 10^{-acd} , \ a = \frac{4 \pi k}{\lambda}$$
(단, $k$ 는 소멸계수) 인 관계가 성립하고, 이 용액의 흡광도 $A$ 를 
$$A = \log I_0 - \log I$$
로 정의한다. 파장이 $\lambda_1$ 인 광선이 농도가 $0$ 이 아닌 어떤 용액을 통과한 거리가 $d_1$ 일 때의 흡광도를 $A_1$, 파장이 $2 \lambda_1$ 인 광선이 동일한 농도의 이 용액을 통과한 거리가 $4 d_1$ 일 때의 흡광도를 $A_2$ 라 하자. 이때, 소멸계수 $k$ 가 일정하다고 할 때 $\frac{A_2}{A_1}$ 의 값은? (단, $\lambda_1 d_1 \neq 0$ 이다.)

## 보기
① $8$   ② $2$   ③ $1$   ④ $\frac{1}{2}$   ⑤ $\frac{1}{8}$

## 풀이
$I = I_0 \times 10^{-acd}$ 에서 $\frac{I_0}{I} = 10^{acd}$ 이므로 흡광도 $A$ 는 다음과 같다.
$$A = \log \frac{I_0}{I} = \log 10^{acd} = acd = \frac{4 \pi k}{\lambda} cd$$
파장이 $\lambda_1$, 거리가 $d_1$ 일 때의 흡광도 $A_1$ 은
$$A_1 = \frac{4 \pi k}{\lambda_1} c d_1$$
파장이 $2 \lambda_1$, 거리가 $4 d_1$ 일 때의 흡광도 $A_2$ 는
$$A_2 = \frac{4 \pi k}{2 \lambda_1} c (4 d_1) = \frac{16 \pi k c d_1}{2 \lambda_1} = \frac{8 \pi k c d_1}{\lambda_1}$$
따라서 $\frac{A_2}{A_1}$ 의 값은 다음과 같다.
$$\therefore \frac{A_2}{A_1} = \frac{\frac{8 \pi k c d_1}{\lambda_1}}{\frac{4 \pi k c d_1}{\lambda_1}} = \frac{8}{4} = 2$$
정답은 ② 이다.
