## 문제
    
$0 \le x < 2\pi$ 일 때, 방정식 $2\sin^2 x + 3\cos x = 3$ 의 모든 해의 합은? 

## 보기
① $\frac{\pi}{2}$   ② $\pi$   ③ $\frac{3\pi}{2}$   ④ $2\pi$   ⑤ $\frac{5\pi}{2}$

## 풀이
[정답] ④ 

$2\sin^2 x + 3\cos x = 3$ 
$2(1 - \cos^2 x) + 3\cos x = 3$ 
$2\cos^2 x - 3\cos x + 1 = 0$ 
$(\cos x - 1)(2\cos x - 1) = 0$ 
$\cos x = 1$ 또는 $\cos x = \frac{1}{2}$ 
이때, $0 \le x < 2\pi$ 이므로 
$x = 0$ 또는 $x = \frac{\pi}{3}$ 또는 $x = \frac{5\pi}{3}$ 
따라서, 모든 해의 합은 $0 + \frac{\pi}{3} + \frac{5\pi}{3} = 2\pi$ 이다.
