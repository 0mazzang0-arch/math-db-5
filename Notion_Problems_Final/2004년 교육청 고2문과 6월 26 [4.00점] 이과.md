## 문제
자연수 $n$ 에 대하여 집합 $A_{n}$ 을 
$A_{n} = \left\{ x \mid \left| \frac{x}{n} - 2 \right| \leq 1, x \text{ 는 자연수} \right\}$
라 하고, $S_{n}$ 을 집합 $A_{n}$ 의 모든 원소들의 합이라 한다. 
이때, $S_{1} + S_{2} + S_{3}$ 의 값을 구하시오.

## 보기
(제공된 보기가 없습니다.)

## 풀이
$\left| \frac{x}{n} - 2 \right| \leq 1$ 에서 
$-1 \leq \frac{x}{n} - 2 \leq 1$ 
$1 \leq \frac{x}{n} \leq 3$ 
$\therefore n \leq x \leq 3n$ 

따라서 
$A_{1} = \{ 1, 2, 3 \}$ 
$A_{2} = \{ 2, 3, 4, 5, 6 \}$ 
$A_{3} = \{ 3, 4, 5, 6, 7, 8, 9 \}$ 
이므로 

$S_{1} = 6$ 
$S_{2} = 20$ 
$S_{3} = 42$ 

$\therefore S_{1} + S_{2} + S_{3} = 6 + 20 + 42 = 68$
