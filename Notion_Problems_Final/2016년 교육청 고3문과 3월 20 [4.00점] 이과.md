## 문제
자연수 $n$ 에 대하여 
$\left| \left( n + \frac{1}{2} \right)^2 - m \right| < \frac{1}{2}$
을 만족시키는 정수 $m$ 을 $a_{n}$ 이라 하자. $\sum_{k=1}^{5} a_{k}$ 의 값은?

## 보기
① $65$   ② $70$   ③ $75$   ④ $80$   ⑤ $85$

## 풀이
$\left| \left( n + \frac{1}{2} \right)^2 - m \right| < \frac{1}{2}$ 에서
$-\frac{1}{2} < \left( n + \frac{1}{2} \right)^2 - m < \frac{1}{2}$ 이다.
$\left( n + \frac{1}{2} \right)^2 = n^2 + n + \frac{1}{4}$ 이므로 
$-\frac{1}{2} < n^2 + n + \frac{1}{4} - m < \frac{1}{2}$
각 변에서 $\frac{1}{4}$ 을 빼면
$-\frac{3}{4} < n^2 + n - m < \frac{1}{4}$
이때 $m, n$ 은 정수이므로 $n^2 + n - m = 0$ 이어야 한다.
따라서 $m = n^2 + n$ 이고, $a_{n} = n^2 + n$ 이다.
$\sum_{k=1}^{5} a_{k} = \sum_{k=1}^{5} (k^2 + k)$
$= \frac{5 \times 6 \times 11}{6} + \frac{5 \times 6}{2}$
$= 55 + 15$
$= 70$

정답은 ②이다.
