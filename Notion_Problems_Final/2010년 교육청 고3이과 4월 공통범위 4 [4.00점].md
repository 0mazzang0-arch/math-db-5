## 문제
두 사건 $A$, $B$가 서로 독립이고 $P(A \cap B) = \frac{1}{4}$일 때, $P(A \cup B) = k - \frac{1}{4}$이 되도록 하는 실수 $k$의 최솟값은?



> [!example] 📸 (그림/그래프)

## 보기
① $\frac{1}{2}$   ② $\frac{5}{8}$   ③ $\frac{3}{4}$   ④ $\frac{7}{8}$   ⑤ $1$

## 풀이
$P(A) = a$, $P(B) = b$라 하면 $A$와 $B$가 서로 독립이므로 $P(A \cap B) = P(A) \times P(B) = ab$이다.
따라서, $ab = \frac{1}{4}$ ($0 < a \le 1$, $0 < b \le 1$)이다.
또한, $P(A \cup B) = P(A) + P(B) - P(A \cap B)$이므로 $a + b - \frac{1}{4} = k - \frac{1}{4}$에서 $a + b = k$이다.
산술·기하평균의 관계에 의하여
$k = a + b \ge 2\sqrt{ab} = 2\sqrt{\frac{1}{4}} = 1$ (단, $a=b=\frac{1}{2}$일 때 등호 성립)
이때 $0 < a \le 1$, $0 < b \le 1$이므로 $a = \frac{1}{4b}$에서 $b \le 1$이면 $a \ge \frac{1}{4}$이다. 
$a = 1, b = \frac{1}{4}$ 또는 $a = \frac{1}{4}, b = 1$일 때 $k$는 최댓값 $\frac{5}{4}$를 갖는다.
$\therefore 1 \le k \le \frac{5}{4}$
따라서 실수 $k$의 최솟값은 $1$이다.

정답: ⑤
