## 문제
다항함수 $f(x)$가 $\lim_{x \to \infty} \frac{f(x)}{x^{2}} = 2$, $\lim_{x \to 2} \frac{f(x)}{x - 2} = 7$을 만족시킬 때, $f(4)$의 값을 구하시오.

## 풀이
$\lim_{x \to \infty} \frac{f(x)}{x^{2}} = 2$이므로 $f(x) = 2x^{2} + ax + b$ ($a$, $b$는 상수)라 하자.
$\lim_{x \to 2} \frac{f(x)}{x - 2} = 7$에서 $\lim_{x \to 2} (x - 2) = 0$이므로 $\lim_{x \to 2} f(x) = f(2) = 0$이다.
$8 + 2a + b = 0$, $b = -2(a + 4)$
$f(x) = (x - 2)(2x + a + 4)$이므로
$\lim_{x \to 2} \frac{f(x)}{x - 2} = \lim_{x \to 2} \frac{(x - 2)(2x + a + 4)}{x - 2} = a + 8 = 7$에서 $a = -1$, $b = -6$이다.
따라서 $f(x) = 2x^{2} - x - 6$이므로 $f(4) = 32 - 4 - 6 = 22$이다.

[정답] $22$
