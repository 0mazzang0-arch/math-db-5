## 문제
함수 $f(x) = (x - 1)(x - 2)$와 최고차항의 계수가 $1$인 사차함수 $g(x)$가 다음 조건을 만족시킨다.

$$ \begin{array}{|c|} \hline \text{모든 실수 } a \text{에 대하여 } \lim_{x \to a} \frac{g(x) \times |f(x)|}{f(x)} \text{의 값과 } \lim_{x \to a} \frac{|g(x) - f(x)|}{g(x)} \text{의 값이 모두 존재한다.} \\ \hline \end{array} $$

$g(-1)$의 값을 구하시오.

## 보기


## 풀이
$f(x) = 0$에서 $x = 1$ 또는 $x = 2$이고, 모든 실수 $a$에 대하여 $\lim_{x \to a} \frac{g(x) \times |f(x)|}{f(x)}$의 값이 존재한다.

(ⅰ) $a = 1$일 때
$\lim_{x \to 1^-} \frac{g(x) \times |f(x)|}{f(x)} = \lim_{x \to 1^-} \frac{g(x) f(x)}{f(x)} = \lim_{x \to 1^-} g(x)$
$\lim_{x \to 1^+} \frac{g(x) \times |f(x)|}{f(x)} = \lim_{x \to 1^+} \frac{g(x) \{ -f(x) \}}{f(x)} = -\lim_{x \to 1^+} g(x)$
에서 $\lim_{x \to 1^-} g(x) = -\lim_{x \to 1^+} g(x) = \lim_{x \to 1} g(x)$
$\therefore \lim_{x \to 1} g(x) = 0$

(ⅱ) $a = 2$일 때
$\lim_{x \to 2^-} \frac{g(x) \times |f(x)|}{f(x)} = \lim_{x \to 2^-} \frac{g(x) \{ -f(x) \}}{f(x)} = -\lim_{x \to 2^-} g(x)$
$\lim_{x \to 2^+} \frac{g(x) \times |f(x)|}{f(x)} = \lim_{x \to 2^+} \frac{g(x) f(x)}{f(x)} = \lim_{x \to 2^+} g(x)$
에서 $-\lim_{x \to 2^-} g(x) = \lim_{x \to 2^+} g(x) = \lim_{x \to 2} g(x)$
$\therefore \lim_{x \to 2} g(x) = 0$

$g(x)$는 최고차항의 계수가 $1$인 사차함수이므로 (ⅰ), (ⅱ)에 의해 $g(1) = g(2) = 0$
따라서 $g(x) = (x - 1)(x - 2)h(x) = f(x)h(x)$라 놓을 수 있다.
모든 실수 $a$에 대하여 $\lim_{x \to a} \frac{|g(x) - f(x)|}{g(x)}$의 값이 존재해야 하고
$\lim_{x \to a} \frac{|g(x) - f(x)|}{g(x)} = \lim_{x \to a} \frac{|f(x)h(x) - f(x)|}{f(x)h(x)} = \lim_{x \to a} \frac{|f(x)| |h(x) - 1|}{f(x)h(x)}$ 이므로

(ⅰ) $a = 1$일 때
$\lim_{x \to 1^-} \frac{|g(x) - f(x)|}{g(x)} = \lim_{x \to 1^-} \frac{f(x) |h(x) - 1|}{f(x)h(x)} = \lim_{x \to 1^-} \frac{|h(x) - 1|}{h(x)}$
$\lim_{x \to 1^+} \frac{|g(x) - f(x)|}{g(x)} = \lim_{x \to 1^+} \frac{-f(x) |h(x) - 1|}{f(x)h(x)} = -\lim_{x \to 1^+} \frac{|h(x) - 1|}{h(x)}$
에서 $\lim_{x \to 1^-} \frac{|h(x) - 1|}{h(x)} = -\lim_{x \to 1^+} \frac{|h(x) - 1|}{h(x)} \cdots \cdots$ ㉠
$h(1) = 0$이면 ㉠의 값이 존재하지 않으므로 $h(1) \neq 0$
$\therefore h(1) = 1$

(ⅱ) $a = 2$일 때
$\lim_{x \to 2^-} \frac{|g(x) - f(x)|}{g(x)} = \lim_{x \to 2^-} \frac{-f(x) |h(x) - 1|}{f(x)h(x)} = -\lim_{x \to 2^-} \frac{|h(x) - 1|}{h(x)}$
$\lim_{x \to 2^+} \frac{|g(x) - f(x)|}{g(x)} = \lim_{x \to 2^+} \frac{f(x) |h(x) - 1|}{f(x)h(x)} = \lim_{x \to 2^+} \frac{|h(x) - 1|}{h(x)}$
에서 $-\lim_{x \to 2^-} \frac{|h(x) - 1|}{h(x)} = \lim_{x \to 2^+} \frac{|h(x) - 1|}{h(x)} \cdots \cdots$ ㉡
$h(2) = 0$이면 ㉡의 값이 존재하지 않으므로 $h(2) \neq 0$
$\therefore h(2) = 1$

(ⅰ), (ⅱ)에서 $h(1) = 1$, $h(2) = 1$이고 $h(x)$는 최고차항의 계수가 $1$인 이차식이므로
$h(x) = (x - 1)(x - 2) + 1$
$g(x) = f(x)h(x)$에 대입하면
$g(x) = (x - 1)(x - 2) \{ (x - 1)(x - 2) + 1 \} = (x - 1)^2(x - 2)^2 + (x - 1)(x - 2)$
$\therefore g(-1) = 36 + 6 = 42$
