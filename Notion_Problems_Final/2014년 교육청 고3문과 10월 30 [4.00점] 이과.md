## 문제
수열 $\{ a_{n} \}$ 이 다음 조건을 만족시킨다.

(가) $| a_{n} | + a_{n+1} = n+6$ ($n \ge 1$)
(나) $\sum_{n=1}^{40} a_{n} = 520$

$\sum_{n=1}^{30} a_{n}$ 의 값을 구하시오.

## 보기

## 풀이
(가)에서 
$a_{n} + a_{n+1} \le | a_{n} | + a_{n+1} = n+6$ 

$n=2k-1$ 일 때 
$a_{2k-1} + a_{2k} \le (2k-1)+6 = 2k+5$ 

그런데 
$\sum_{n=1}^{40} a_{n} = \sum_{k=1}^{20} ( a_{2k-1} + a_{2k} ) \le \sum_{k=1}^{20} (2k+5) = 520$ 

(나)에서 
$\sum_{n=1}^{40} a_{n} = 520$ 이므로 
$a_{2k-1} = | a_{2k-1} |$ ($k=1, 2, \dots, 20$) 

$a_{2k-1} \ge 0$ 이므로 
$a_{2k-1} + a_{2k} = 2k+5$ 

따라서 
$\sum_{n=1}^{30} a_{n} = \sum_{k=1}^{15} ( a_{2k-1} + a_{2k} ) = \sum_{k=1}^{15} (2k+5) = 315$

정답: $315$
