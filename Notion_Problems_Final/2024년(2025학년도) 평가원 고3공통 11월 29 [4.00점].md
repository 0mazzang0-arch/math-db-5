## 문제
정규분포 $N(m_{1}, \sigma_{1}^{2})$ 을 따르는 확률변수 $X$ 와 정규분포 $N(m_{2}, \sigma_{2}^{2})$ 을 따르는 확률변수 $Y$ 가 다음 조건을 만족시킨다.

모든 실수 $x$ 에 대하여
$P(X \leq x) = P(X \geq 40 - x)$ 이고,
$P(Y \leq x) = P(X \leq x + 10)$ 이다.

$$ \begin{array}{|c|c|} \hline z & P(0 \leq Z \leq z) \\ \hline 0.5 & 0.1915 \\ \hline 1.0 & 0.3413 \\ \hline 1.5 & 0.4332 \\ \hline 2.0 & 0.4772 \\ \hline \end{array} $$

$P(15 \leq X \leq 20) + P(15 \leq Y \leq 20)$ 의 값을 다음 표준정규분포표를 이용하여 구한 것이 $0.4772$ 일 때, $m_{1} + \sigma_{2}$ 의 값을 구하시오. (단, $\sigma_{1}$ 과 $\sigma_{2}$ 는 양수이다.)

## 풀이
확률변수 $X$ 는 정규분포 $N(m_{1}, \sigma_{1}^{2})$ 을 따르고 확률변수 $Y$ 는 정규분포 $N(m_{2}, \sigma_{2}^{2})$ 을 따를 때, 모든 실수 $x$ 에 대하여
$P(X \leq x) = P(X \geq 40 - x)$
가 성립하므로
$m_{1} = \frac{x + 40 - x}{2} = 20$

또한 모든 실수 $x$ 에 대하여
$P(Y \leq x) = P(X \leq x + 10)$
$Z_{1} = \frac{X - 20}{\sigma_{1}}$, $Z_{2} = \frac{Y - m_{2}}{\sigma_{2}}$ 라 하면 확률변수 $Z_{1}$, $Z_{2}$ 는 표준정규분포 $N(0, 1)$ 을 따르므로
$P\left( Z_{2} \leq \frac{x - m_{2}}{\sigma_{2}} \right) = P\left( Z_{1} \leq \frac{x + 10 - 20}{\sigma_{1}} \right)$
이때, 모든 실수 $x$ 에 대하여
$\frac{1}{\sigma_{2}} x - \frac{m_{2}}{\sigma_{2}} = \frac{1}{\sigma_{1}} x - \frac{10}{\sigma_{1}}$
이 성립하므로 $\sigma_{1} = \sigma_{2}$, $m_{2} = 10$

따라서
$P(15 \leq X \leq 20) = P(5 \leq Y \leq 10) = P(10 \leq Y \leq 15)$ 이므로
$P(15 \leq X \leq 20) + P(15 \leq Y \leq 20)$
$= P(10 \leq Y \leq 15) + P(15 \leq Y \leq 20)$
$= P(10 \leq Y \leq 20)$
$= P\left( \frac{10 - 10}{\sigma_{2}} \leq Z_{2} \leq \frac{20 - 10}{\sigma_{2}} \right)$
즉 $P\left( 0 \leq Z_{2} \leq \frac{10}{\sigma_{2}} \right) = 0.4772$ 이므로 표에서 $z = 2.0$ 일 때 $0.4772$ 이다.
$\frac{10}{\sigma_{2}} = 2$, $\sigma_{2} = 5$

$\therefore m_{1} + \sigma_{2} = 20 + 5 = 25$
