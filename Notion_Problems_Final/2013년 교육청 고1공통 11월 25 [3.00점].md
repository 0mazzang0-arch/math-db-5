## 문제
좌표평면에서 점 $(1, 4)$를 점 $(-2, a)$로 옮기는 평행이동에 의하여, 원 $x^{2} + y^{2} + 8x - 6y + 21 = 0$은 원 $x^{2} + y^{2} + bx - 18y + c = 0$으로 옮겨진다. 세 실수 $a$, $b$, $c$의 합 $a+b+c$의 값을 구하시오.

## 보기

## 풀이
점 $(1, 4)$를 점 $(-2, a)$로 옮기는 평행이동에 의하여 좌표평면 위의 점은 $x$축의 방향으로 $-3$만큼, $y$축의 방향으로 $a-4$만큼 옮겨진다. 

$x^{2} + y^{2} + 8x - 6y + 21 = 0$에서 
$(x+4)^{2} + (y-3)^{2} = 4 \cdots \cdots$ $㉠$ 

$x^{2} + y^{2} + bx - 18y + c = 0$에서 
$(x + \frac{b}{2})^{2} + (y-9)^{2} = 81 - c + \frac{b^{2}}{4} \cdots \cdots$ $㉡$ 

$㉠$의 원의 중심 $(-4, 3)$이 평행이동에 의하여 $㉡$의 원의 중심 $(-\frac{b}{2}, 9)$로 옮겨지므로 
$-4 + (-3) = -\frac{b}{2}$ 에서 $b = 14$
$3 + (a - 4) = 9$ 에서 $a = 10$

평행이동을 하여도 원의 반지름의 길이는 변하지 않으므로 
$4 = 81 - c + \frac{196}{4}$ 
$4 = 81 - c + 49$
$4 = 130 - c$
따라서 $c = 126$ 

$\therefore a+b+c = 10 + 14 + 126 = 150$
