## 문제
$A = \{ x | x^{2} + ax + b \leq 0 \}$, $B = \{ x | x^{2} + cx + d \leq 0 \}$에 대하여
$A \cup B = \{ x | -5 \leq x \leq 4 \}$,
$A \cap B = \{ x | -3 \leq x \leq 1 \}$,
$A - B = \{ x | -5 \leq x < -3 \}$
일 때, $a + b + c + d$의 값은?

## 보기
① $8$   ② $-8$   ③ $0$   ④ $14$   ⑤ $-14$

## 풀이
두 이차함수 $f(x)$와 $g(x)$를 
$f(x) = x^{2} + ax + b$, $g(x) = x^{2} + cx + d$라 하면
$A \cup B = \{ x | -5 \leq x \leq 4 \}$, $A \cap B = \{ x | -3 \leq x \leq 1 \}$로부터 두 함수 $f(x)$, $g(x)$의 $x$절편은 $-5$, $-3$, $4$, $1$이다. 
또한, 조건 $A - B = \{ x | -5 \leq x < -3 \}$로부터 $-5$와 $1$은 $f(x)$의 $x$절편이 되고, $-3$과 $4$는 $g(x)$의 $x$절편이 됨을 알 수 있다. 
따라서
$A = \{ x | (x + 5)(x - 1) \leq 0 \}$, $B = \{ x | (x + 3)(x - 4) \leq 0 \}$이고
$f(x) = x^{2} + 4x - 5$에서 $a = 4$, $b = -5$
$g(x) = x^{2} - x - 12$에서 $c = -1$, $d = -12$이다.
$\therefore a + b + c + d = 4 + (-5) + (-1) + (-12) = -14$

정답: ⑤
