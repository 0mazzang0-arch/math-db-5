## 문제
다음은 $x$에 대한 다항식 $ax^{9} + bx^{8} + 1$이 다항식 $x^{2} - x - 1$로 나누어떨어지기 위한 정수 $a$, $b$의 값을 구하는 과정의 일부이다.

$$
\begin{array}{|l|}
\hline
\text{방정식 } x^{2} - x - 1 = 0 \text{의 두 근을 } p, q \text{라 하면} \\
p + q = 1, pq = -1 \text{이다. 따라서} \\
p^{2} + q^{2} = (\text{가}), p^{4} + q^{4} = (\text{나}) \text{이다.} \\
x \text{에 대한 다항식 } ax^{9} + bx^{8} + 1 \text{이 } x^{2} - x - 1 \text{로 나누어떨어지면} \\
ap^{9} + bp^{8} = -1 \cdots\cdots ① \\
aq^{9} + bq^{8} = -1 \cdots\cdots ② \\
\text{이다. } ①, ② \text{의 양변에 각각 } q^{8}, p^{8} \text{을 곱하여 정리하면} \\
ap + b = -q^{8} \cdots\cdots ③ \\
aq + b = -p^{8} \cdots\cdots ④ \\
\text{이다. } ③ \text{에서 } ④ \text{를 뺀 식으로부터 } a(p - q) = p^{8} - q^{8} \text{이고,} \\
p \neq q \text{이므로 } a = \frac{p^{8} - q^{8}}{p - q} \text{이다. 따라서 } a = (\text{다}) \text{이다.} \\
\hline
\end{array}
$$

위의 $(\text{가})$, $(\text{나})$, $(\text{다})$에 알맞은 수를 각각 $r$, $s$, $t$라 할 때, $r + s + t$의 값은?

## 보기
① $27$   ② $29$   ③ $31$   ④ $33$   ⑤ $35$

## 풀이
$p + q = 1$, $pq = -1$이므로
$r = p^{2} + q^{2} = (p + q)^{2} - 2pq = 1^{2} - 2 \times (-1) = 3$
$s = p^{4} + q^{4} = (p^{2} + q^{2})^{2} - 2p^{2}q^{2} = 3^{2} - 2 \times (-1)^{2} = 7$

$x$에 대한 다항식 $ax^{9} + bx^{8} + 1$이 $x^{2} - x - 1$로 나누어떨어질 때의 몫을 $Q(x)$라 하면
$ax^{9} + bx^{8} + 1 = (x^{2} - x - 1)Q(x)$
위 식의 양변에 $x = p, x = q$를 각각 대입하면
$ap^{9} + bp^{8} = -1 \cdots\cdots ①$
$aq^{9} + bq^{8} = -1 \cdots\cdots ②$
$①, ②$의 양변에 각각 $q^{8}, p^{8}$을 곱하면
$ap(pq)^{8} + b(pq)^{8} = -q^{8}$
$aq(pq)^{8} + b(pq)^{8} = -p^{8}$
$pq = -1$을 대입하여 정리하면
$ap + b = -q^{8} \cdots\cdots ③$
$aq + b = -p^{8} \cdots\cdots ④$
$③$에서 $④$를 빼면 $a(p - q) = p^{8} - q^{8}$이므로
$t = a = \frac{p^{8} - q^{8}}{p - q} = \frac{(p^{4} + q^{4})(p^{4} - q^{4})}{p - q} = \frac{(p^{4} + q^{4})(p^{2} + q^{2})(p^{2} - q^{2})}{p - q}$
$t = \frac{(p^{4} + q^{4})(p^{2} + q^{2})(p + q)(p - q)}{p - q} = (p^{4} + q^{4})(p^{2} + q^{2})(p + q)$
$t = 7 \times 3 \times 1 = 21$

따라서 $r = 3, s = 7, t = 21$이므로
$r + s + t = 3 + 7 + 21 = 31$

정답: ③
