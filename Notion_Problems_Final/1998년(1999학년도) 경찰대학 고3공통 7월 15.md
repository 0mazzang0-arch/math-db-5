## 문제
$\lim _{n \to \infty} \frac{\cos^{n} x-\sin^{n} x}{\cos^{n} x+\sin^{n} x}$를 치역으로 하는 집합의 원소의 개수는?
($0 < x < \frac{\pi}{2}$)

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
주어진 식을 변형하면 $\lim_{n \to \infty} \frac{1 - \tan^n x}{1 + \tan^n x}$ 가 된다.
$\lim_{n \to \infty} r^n$ 값은 $r$ 의 크기에 따라 나누어 고려하면 다음과 같다.

$1$) $|\tan x| < 1$ : $0 < x < \frac{\pi}{4}$ 또는 $\frac{3}{4} \pi < x < \pi$ 인 경우
$\lim_{n \to \infty} \tan^n x = 0$ 이므로 $\lim_{n \to \infty} \frac{1 - \tan^n x}{1 + \tan^n x} = 1$

$2$) $|\tan x| > 1$ : $\frac{\pi}{4} < x < \frac{\pi}{2}$ 또는 $\frac{\pi}{2} < x < \frac{3}{4} \pi$ 인 경우
$\lim_{n \to \infty} \tan^n x = \infty$ 이므로 $\lim_{n \to \infty} \frac{1 - \tan^n x}{1 + \tan^n x} = -1$

$3$) $\tan x = 1$ : $x = \frac{\pi}{4}$ 인 경우
$\lim_{n \to \infty} \tan^n x = 1$ 이므로 준 식의 값은 $0$

따라서, $\lim_{n \to \infty} \frac{\cos^n x - \sin^n x}{\cos^n x + \sin^n x}$ 의 값은 $3$ 가지이다.
