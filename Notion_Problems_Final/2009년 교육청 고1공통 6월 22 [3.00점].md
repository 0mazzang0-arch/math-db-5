## 문제
등식 $( a + b - 3 ) x + ab + 1 = 0$ 이 $x$ 값에 관계없이 항상 성립할 때, 상수 $a$, $b$ 에 대하여 $a^2 + b^2$ 의 값을 구하시오.

## 보기

## 풀이
등식 $( a + b - 3 ) x + ab + 1 = 0$ 은 $x$ 에 대한 항등식이므로
$a + b - 3 = 0$, $ab + 1 = 0$
$\therefore a + b = 3$, $ab = -1$
따라서
$a^2 + b^2 = ( a + b )^2 - 2ab = 3^2 - 2 \times ( -1 ) = 11$

정답: $11$
