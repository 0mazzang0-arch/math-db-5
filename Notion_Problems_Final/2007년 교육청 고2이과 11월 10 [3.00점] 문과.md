## 문제
$2$ 이상의 자연수 $m$, $n$ 과 양의 실수 $a$, $b$ 에 대하여 <보기>에서 항상 옳은 것을 모두 고르면?

$$ \begin{array}{|l|} \hline \text{ㄱ. } \sqrt[n]{a} \sqrt[n]{b} = \sqrt[n]{ab} \\ \text{ㄴ. } \sqrt[n]{a} < \sqrt[n]{b} \text{ 이면 } a < b \text{ 이다.} \\ \text{ㄷ. } m < n \text{ 일 때, } \sqrt[m]{a} = \sqrt[n]{b} \text{ 이면 } a < b \text{ 이다.} \\ \hline \end{array} $$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
ㄱ. $\sqrt[n]{a} \sqrt[n]{b} = \sqrt[n]{ab}$ (참)

ㄴ. $\sqrt[n]{a} > 0$, $\sqrt[n]{b} > 0$ 이므로 $\sqrt[n]{a} < \sqrt[n]{b}$ 의 양변을 $n$ 제곱하여도 부등호 방향이 변하지 않는다.
$\therefore a < b$ (참)

ㄷ. (반례) $m = 2$, $n = 3$, $a = \frac{1}{4}$, $b = \frac{1}{8}$ 이면
$\sqrt[2]{\frac{1}{4}} = \frac{1}{2}$, $\sqrt[3]{\frac{1}{8}} = \frac{1}{2}$ 이므로
$\sqrt[m]{a} = \sqrt[n]{b}$ 이지만 $a = \frac{1}{4} > b = \frac{1}{8}$ 이다. (거짓)

따라서 옳은 것은 ㄱ, ㄴ이다.
