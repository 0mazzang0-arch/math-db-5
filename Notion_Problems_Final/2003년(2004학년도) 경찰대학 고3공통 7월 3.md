## 문제
사차다항식 $a x^{4} + b x^{3} + c x^{2} - 3 x$ 가 $x^{2} + 4$ 로 나누어떨어질 때, 방정식 $a x^{4} + b x^{3} + c x^{2} - 3 x = 0$ 의 모든 근의 합은? (단, $a$, $b$, $c$ 는 실수)

## 보기
① $-\frac{c}{3}$   ② $-\frac{3}{c}$   ③ $\frac{3}{c}$   ④ $\frac{c}{3}$   ⑤ $-b$

## 풀이
주어진 $4$ 차 다항식이 $x^{2} + 4$ 로 나누어떨어지므로
$a x^{4} + b x^{3} + c x^{2} - 3 x = ( x^{2} + 4 ) \cdot Q ( x ) \cdots (1)$
식 $(1)$은 $x$ 에 관한 항등식이므로 $x = 2 i$ 를 대입하면
$16 a - 8 b i - 4 c - 6 i = 0$
$4 ( 4 a - c ) - 2 ( 4 b + 3 ) i = 0$
$a$, $b$, $c$ 는 실수이므로 실수부와 허수부가 각각 $0$ 일 때 등식이 성립한다. 따라서,
$a = \frac{1}{4} c$, $b = - \frac{3}{4}$
방정식 $a x^{4} + b x^{3} + c x^{2} - 3 x = 0$ 의 모든 근의 합은
$- \frac{b}{a} = \frac{- ( - \frac{3}{4} )}{\frac{1}{4} c} = \frac{3}{c}$

정답: ③
