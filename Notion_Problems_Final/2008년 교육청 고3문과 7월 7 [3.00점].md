## 문제
$1$이 아닌 양의 실수 $x$, $y$에 대하여 연산 $◎$을 $x ◎ y = \log_{x} y + \log_{y} x$로 정의할 때, <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $a$, $b$, $k$는 $1$이 아닌 양수)

## 보기
ㄱ. $4 ◎ 16 = \frac{5}{2}$
ㄴ. $a^{k} ◎ b^{k} = a ◎ b$
ㄷ. $a^{b} ◎ b^{a} = a ◎ b^{\frac{a}{b}}$

① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
ㄱ. $4 ◎ 16 = \log_{4} 16 + \log_{16} 4 = 2 + \frac{1}{2} = \frac{5}{2}$ (참)

ㄴ. $a^{k} ◎ b^{k} = \log_{a^{k}} b^{k} + \log_{b^{k}} a^{k} = \frac{k}{k} \log_{a} b + \frac{k}{k} \log_{b} a = \log_{a} b + \log_{b} a = a ◎ b$ (참)

ㄷ. $a^{b} ◎ b^{a} = \log_{a^{b}} b^{a} + \log_{b^{a}} a^{b} = \frac{a}{b} \log_{a} b + \frac{b}{a} \log_{b} a$
또한, $a ◎ b^{\frac{a}{b}} = \log_{a} b^{\frac{a}{b}} + \log_{b^{\frac{a}{b}}} a = \frac{a}{b} \log_{a} b + \frac{1}{\frac{a}{b}} \log_{b} a = \frac{a}{b} \log_{a} b + \frac{b}{a} \log_{b} a$
따라서 $a^{b} ◎ b^{a} = a ◎ b^{\frac{a}{b}}$ (참)

그러므로 <보기>에서 옳은 것은 ㄱ, ㄴ, ㄷ이다.
