## 문제
실수 전체의 집합 $R$의 두 부분집합
$$ A = \{ x | x^{2} - x - 12 \le 0 \}, B = \{ x | x < a \text{ 또는 } x > b \} $$
가 다음 조건을 만족시킨다.

(가) $A \cup B = R$
(나) $A - B = \{ x | -3 \le x \le 1 \}$

두 상수 $a$, $b$에 대하여 $b - a$의 값을 구하시오.

## 풀이
$x^{2} - x - 12 \le 0$ 에서 
$(x + 3)(x - 4) \le 0$ 이므로 
$-3 \le x \le 4$ 이다.
그러므로 
$A = \{ x | -3 \le x \le 4 \}$
조건 (가)에서 
$A \cup B = R$ 이므로 
$a \ge -3$ 이고 $b \le 4$ 이다. 
또, 
$A - B = A \cap B^{c}$ 이고 
$B^{c} = \{ x | a \le x \le b \}$ 이다. 

$i)$ $a > -3$ 인 경우 집합 $A \cap B^{c}$ 의 원소 중 가장 작은 수는 $a$ 이므로 $A - B = \{ x | -3 \le x \le 1 \}$ 을 만족시키지 않는다. 그러므로 $a = -3$ 이다. 

$ii)$ $b \le 4$ 인 경우 집합 $A \cap B^{c}$ 의 원소 중 가장 큰 수는 $b$ 이므로 $A - B = \{ x | -3 \le x \le 1 \}$ 을 만족시키기 위한 $b$ 의 값은 $1$ 이다. 

두 조건을 만족시키는 두 집합의 관계를 수직선 위에 나타내면 그림과 같다. 

> [!example] 📸 (그림/그래프)

따라서 $a = -3$, $b = 1$ 이므로 $b - a = 1 - (-3) = 4$ 이다.

정답: $4$
