## 문제
$2$ 이상의 세 실수 $a$, $b$, $c$가 다음 조건을 만족시킨다.

(가) $\sqrt[3]{a}$는 $ab$의 네제곱근이다.
(나) $\log_{a} bc + \log_{b} ac = 4$

$a = \left( \frac{b}{c} \right)^{k}$이 되도록 하는 실수 $k$의 값은?

## 보기
① $6$   ② $\frac{13}{2}$   ③ $7$   ④ $\frac{15}{2}$   ⑤ $8$

## 풀이
(가)에서 $\sqrt[3]{a}$는 $ab$의 네제곱근이므로 
$(\sqrt[3]{a})^{4} = ab$
$a^{\frac{4}{3}} = ab$, $b = a^{\frac{1}{3}}$

(나)에서 
$\log_{a} bc + \log_{b} ac = 4$
$\log_{a} a^{\frac{1}{3}}c + \log_{a^{\frac{1}{3}}} ac = 4$
$\frac{1}{3} \log_{a} a + \log_{a} c + 3(\log_{a} a + \log_{a} c) = 4$
$\frac{1}{3} + \log_{a} c + 3 + 3\log_{a} c = 4$
$\frac{10}{3} + 4\log_{a} c = 4$
$4\log_{a} c = \frac{2}{3}$
$\log_{a} c = \frac{1}{6}$, $c = a^{\frac{1}{6}}$

따라서 $a = \left( \frac{b}{c} \right)^{k} = \left( \frac{a^{\frac{1}{3}}}{a^{\frac{1}{6}}} \right)^{k} = \left( a^{\frac{1}{3} - \frac{1}{6}} \right)^{k} = a^{\frac{k}{6}}$이므로 
$1 = \frac{k}{6}$
$k = 6$

정답: ①
