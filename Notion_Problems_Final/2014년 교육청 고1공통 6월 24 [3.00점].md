## 문제
연립부등식 $\left\{ \begin{array}{l} |x - 1| \le 6 \\ (x - 2)(x - 8) \le 0 \end{array} \right.$ 의 해가 $\alpha \le x \le \beta$ 일 때, $\alpha + \beta$ 의 값을 구하시오.

## 풀이
부등식 $|x - 1| \le 6$ 에서
$-6 \le x - 1 \le 6$
$\therefore -5 \le x \le 7$

부등식 $(x - 2)(x - 8) \le 0$ 에서
$2 \le x \le 8$

따라서 주어진 연립부등식의 해는 $2 \le x \le 7$ 이므로
$\alpha = 2$, $\beta = 7$
$\therefore \alpha + \beta = 9$

정답: $9$
