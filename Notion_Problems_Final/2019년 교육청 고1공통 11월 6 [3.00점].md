## 문제
다항식 $( x ^{2} +x  ) ^{2} +2  ( x ^{2} +x  ) -3$ 이 $( x ^{2} +ax-1  )  ( x ^{2} +x+b  )$ 로 인수분해될 때, 두 상수 $a$, $b$ 에 대하여 $a+b$ 의 값은?

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
$x ^{2} +x=t$ 라 하면 
$$( x ^{2} +x  ) ^{2} +2  ( x ^{2} +x  ) -3$$ 
$$=t ^{2} +2t-3$$ 
$$=  ( t-1  )  ( t+3  )$$ 
$$=  ( x ^{2} +x-1  )  ( x ^{2} +x+3  )$$ 
에서 
$a=1$ , $b=3$ 
따라서 
$a+b=1+3=4$

정답: ④
