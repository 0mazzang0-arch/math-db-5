## 문제
양수 $t$에 대하여 닫힌구간 $[0, \frac{2}{t}]$에서 정의된 두 함수 
$f(x) = \sqrt{3} \sin (t \pi x)$, $g(x) = -3 \cos (t \pi x)$가 있다. $0 < k < \frac{2}{t}$인 상수 $k$에 대하여 $f(k) = g(k) = 3k$일 때, $60(t + k)$의 값을 구하시오.

## 풀이
$0 < k < \frac{2}{t}$에서 $0 < tk \pi < 2 \pi$이다.

$f(k) = 3k$에서 $\sin (tk \pi) = \sqrt{3}k > 0$이고, 
$g(k) = 3k$에서 $-3 \cos (tk \pi) = 3k$, 즉 $\cos (tk \pi) = -k < 0$이므로
$\frac{\pi}{2} < tk \pi < \pi$이다.

$f(k) = g(k)$에서 $\sqrt{3} \sin (tk \pi) = -3 \cos (tk \pi)$이므로
$\tan (tk \pi) = -\sqrt{3}$이다.

따라서 $tk \pi = \frac{2}{3} \pi$에서 $tk = \frac{2}{3}$이다.

$f(k) = 3k$에서 $\sqrt{3} \sin (\frac{2}{3} \pi) = \frac{3}{2} = 3k$이므로
$k = \frac{1}{2}$, $t = \frac{4}{3}$이다.

따라서 
$60(t + k) = 60(\frac{4}{3} + \frac{1}{2}) = 110$이다.

정답: $110$
