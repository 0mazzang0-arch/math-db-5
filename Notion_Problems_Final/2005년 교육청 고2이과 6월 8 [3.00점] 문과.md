## 문제
세 수 $A = 2^{\frac{1}{2}}$, $B = 3^{\frac{1}{3}}$, $C = 9^{\frac{1}{9}}$ 의 대소 관계는?

## 보기
① $A < B < C$   ② $B < A < C$   ③ $B < C < A$   ④ $C < B < A$   ⑤ $C < A < B$

## 풀이
$A = 2^{\frac{1}{2}}$ 이면
$A^{18} = ( 2^{\frac{1}{2}} )^{18} = 2^{9} = 512$

$B = 3^{\frac{1}{3}}$ 이면
$B^{18} = ( 3^{\frac{1}{3}} )^{18} = 3^{6} = 729$

$C = 9^{\frac{1}{9}}$ 이면
$C^{18} = ( 9^{\frac{1}{9}} )^{18} = 9^{2} = 81$

$\therefore 81 < 512 < 729$ 이므로 $C < A < B$

정답: ⑤
