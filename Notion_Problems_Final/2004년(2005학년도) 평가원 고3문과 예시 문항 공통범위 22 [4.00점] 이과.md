## 문제
  
`$3^{30}$`은 십진법으로 `$n$`자리 자연수이고 가장 큰 자리의 숫자가 `$a$`이다. 이때, `$n+a$`의 값을 구하시오. (단, `$\log_{10} 2 = 0.3010$`, `$\log_{10} 3 = 0.4771$`로 계산한다.)

## 풀이

먼저 `$\log_{10} 3^{30}$`을 계산해 보면
`$\log_{10} 3^{30} = 30 \log_{10} 3 = 30 \times 0.4771 = 14.313$`이므로
`$3^{30}$`은 `$15$`자리의 수이다.
`$\therefore n = 15$`
한편, `$14.3010 < 14.313 < 14.4771$`이므로
`$14 + 0.3010 < 14.313 < 14 + 0.4771$`
`$\log_{10} 10^{14} + \log_{10} 2 < \log_{10} 3^{30} < \log_{10} 10^{14} + \log_{10} 3$`
`$\leftrightarrow \log_{10} (2 \times 10^{14}) < \log_{10} 3^{30} < \log_{10} (3 \times 10^{14})$`
`$\therefore 2 \times 10^{14} < 3^{30} < 3 \times 10^{14}$`
따라서, `$3^{30}$`의 가장 큰 자리의 숫자는 `$2$`이므로
`$a = 2$`
`$\therefore n + a = 15 + 2 = 17$`

정답: `$17$`
