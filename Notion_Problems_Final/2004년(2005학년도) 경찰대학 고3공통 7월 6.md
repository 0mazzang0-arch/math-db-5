## 문제
$a=2^{\sqrt{2}}$ 일 때, 다음 보기 중 옳은 것만을 있는 대로 고른 것은?

$$ \begin{array}{|l|} \hline \text{가. } a^{\sqrt{2}}=4 \\ \text{나. } a < 3 \\ \text{다. } \frac{5}{2} < a \\ \hline \end{array} $$

## 보기
① 가   ② 나   ③ 가, 나   ④ 나, 다   ⑤ 가, 나, 다

## 풀이
(가) $a^{\sqrt{2}} = (2^{\sqrt{2}})^{\sqrt{2}} = 2^{2} = 4$
(나) 양변에 로그를 취하면 
$\log a = \log 2^{\sqrt{2}} = \sqrt{2} \log 2 < \log 3$
$\sqrt{2} < \frac{\log 3}{\log 2}$
그런데, $\sqrt{2} < \frac{3}{2}$ 이고 $\frac{3}{2} < \frac{\log 3}{\log 2}$ 이므로
$\sqrt{2} < \frac{\log 3}{\log 2}$
(다) $\frac{5}{2} < a$ 에서 $5 < 2^{\sqrt{2}+1}$
양변에 $2$ 를 곱하여 로그를 취하면
$1 < (\sqrt{2} + 2) \log 2$
그런데 $\sqrt{2} + 2 > 3.4$ 이고 $\log 2 > 0.3$ 이므로
$(\sqrt{2} + 2) \log 2 > 3.4 \times 0.3 = 1.02 > 1$
이상에서 가, 나, 다 모두 옳다.
