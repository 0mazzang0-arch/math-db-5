## 문제
수열 $\{ a_{n} \}$은 $0 < a_{1} < 1$이고, 모든 자연수 $n$에 대하여 다음 조건을 만족시킨다.

(가) $a_{2n} = a_{2} \times a_{n} + 1$
(나) $a_{2n+1} = a_{2} \times a_{n} - 2$

$a_{8} - a_{15} = 63$일 때, $\frac{a_{8}}{a_{1}}$의 값은?

## 보기
① $91$   ② $92$   ③ $93$   ④ $94$   ⑤ $95$

## 풀이
조건 (가) $a_{2n} = a_{2} \times a_{n} + 1$에서
$n = 1$을 대입하면 $a_{2} = a_{2} \times a_{1} + 1$이므로
$a_{2} \times a_{1} = a_{2} - 1$ $\cdots$ ㉠

$n = 2$를 대입하면 $a_{4} = ( a_{2} )^{2} + 1$ $\cdots$ ㉡

$n = 4$를 대입하면
$a_{8} = a_{2} \times a_{4} + 1$
$= a_{2} \times \{ ( a_{2} )^{2} + 1 \} + 1$ (∵ ㉡)
$= ( a_{2} )^{3} + a_{2} + 1$ $\cdots$ ㉢

조건 (나) $a_{2n+1} = a_{2} \times a_{n} - 2$에서
$n = 1$을 대입하면
$a_{3} = a_{2} \times a_{1} - 2$
$= a_{2} - 3$ (∵ ㉠) $\cdots$ ㉣

$n = 3$을 대입하면
$a_{7} = a_{2} \times a_{3} - 2$
$= a_{2} \times ( a_{2} - 3 ) - 2$ (∵ ㉣)
$= ( a_{2} )^{2} - 3a_{2} - 2$ $\cdots$ ㉤

$n = 7$을 대입하면
$a_{15} = a_{2} \times a_{7} - 2$
$= a_{2} \times \{ ( a_{2} )^{2} - 3a_{2} - 2 \} - 2$ (∵ ㉤)
$= ( a_{2} )^{3} - 3 ( a_{2} )^{2} - 2a_{2} - 2$ $\cdots$ ㉥

이때, $a_{8} - a_{15} = 63$이므로 ㉢, ㉥에 의해
$\{ ( a_{2} )^{3} + a_{2} + 1 \} - \{ ( a_{2} )^{3} - 3 ( a_{2} )^{2} - 2a_{2} - 2 \} = 63$
$3 ( a_{2} )^{2} + 3a_{2} + 3 = 63$
$( a_{2} )^{2} + a_{2} - 20 = 0$
$( a_{2} - 4 ) ( a_{2} + 5 ) = 0$

$\therefore a_{2} = 4$ 또는 $a_{2} = -5$

(ⅰ) $a_{2} = -5$일 때, ㉠에서
$-5a_{1} = -6$
$\therefore a_{1} = \frac{6}{5}$
$0 < a_{1} < 1$이므로 주어진 조건을 만족시키지 않는다.

(ⅱ) $a_{2} = 4$일 때, ㉠에서
$4a_{1} = 3$
$\therefore a_{1} = \frac{3}{4}$
$0 < a_{1} < 1$이므로 주어진 조건을 만족시킨다.

(ⅰ), (ⅱ)에서 $a_{1} = \frac{3}{4}$, $a_{2} = 4$
따라서 ㉢에 대입하면
$a_{8} = 4^{3} + 4 + 1 = 69$

$\therefore \frac{a_{8}}{a_{1}} = \frac{69}{\frac{3}{4}} = 92$

정답: ②
