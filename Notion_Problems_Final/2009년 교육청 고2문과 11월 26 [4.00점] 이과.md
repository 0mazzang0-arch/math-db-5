## 문제
$100$ 보다 큰 두 실수 $a$, $b$ 가 $2 [\log a] + 3 [\log b] = 13$ 을 만족할 때, $\log ab$ 의 정수부분은 $\alpha$ 또는 $\beta$ 이다. 이때, $\alpha + \beta$ 의 값을 구하시오. (단, $[x]$ 는 $x$ 보다 크지 않은 최대의 정수이다.)

## 보기

## 풀이
$a$, $b$ 는 $100$ 보다 큰 실수이므로 $[\log a]$, $[\log b]$ 가 $2$ 이상의 정수이다.
$2 [\log a] + 3 [\log b] = 13$ 에서 이를 만족하는 정수는 $[\log a] = 2$, $[\log b] = 3$ 뿐이다.
$[\log a] = 2$, $[\log b] = 3$ 이므로
$2 < \log a < 3$, $3 \le \log b < 4$
따라서 $5 < \log a + \log b = \log ab < 7$ 이다.
그러므로 $\log ab$ 의 정수부분은 $5$ 또는 $6$ 이다.
$\alpha = 5$, $\beta = 6$ 이라 하면
$\alpha + \beta = 11$
