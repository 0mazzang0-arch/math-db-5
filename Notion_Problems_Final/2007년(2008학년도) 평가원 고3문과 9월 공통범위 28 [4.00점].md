## 문제
연속확률변수 $X$ 가 갖는 값의 범위는 $0 \le X \le 2$ 이고 확률밀도함수의 그래프는 다음과 같다.

> [!example] 📸 (그림/그래프)

두 양수 $a$, $b$ 에 대하여 $p_1 = P(0 \le X \le a)$, $p_2 = P(a < X \le b)$, $p_3 = P(b < X \le 2)$ 이다. 세 확률 $p_1$, $p_2$, $p_3$ 이 순서대로 등차수열을 이루고 $a+b = \frac{4}{3}$ 일 때, $b$ 의 값은? (단, $a < b$ 이다.)

## 보기
① $\frac{11}{12}$   ② $1$   ③ $\frac{13}{12}$   ④ $\frac{7}{6}$   ⑤ $\frac{5}{4}$

## 풀이
$p_1$, $p_2$, $p_3$ 이 등차수열이므로, 등차중항의 관계에서 $2 p_2 = p_1 + p_3$ 이고, $p_1 + p_2 + p_3 = 1$ 에서 $3 p_2 = 1$ 이다. 

$\therefore p_2 = \frac{1}{3}$ 

한편 $p_2$ 는 오른쪽 그림에서 어두운 부분의 넓이와 같으므로

> [!example] 📸 (그림/그래프)

$S = \frac{1}{2} ( \frac{a}{2} + \frac{b}{2} ) \cdot (b-a) = \frac{1}{4} (a+b) (b-a) = \frac{1}{3}$

위 식에서 $a+b = \frac{4}{3}$ 이므로 $\frac{1}{4} \cdot \frac{4}{3} \cdot (b-a) = \frac{1}{3}$ 에서 $b-a = 1$ 이다.

$a+b = \frac{4}{3}$ 와 $b-a = 1$ 을 연립하여 풀면 $2b = \frac{7}{3}$ 이므로 $b = \frac{7}{6}$ 이다.
