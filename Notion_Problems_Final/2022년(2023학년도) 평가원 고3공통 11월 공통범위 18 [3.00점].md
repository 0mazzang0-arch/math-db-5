## 문제
두 수열 $\{ a_{n} \}$, $\{ b_{n} \}$ 에 대하여 
$\sum_{k=1}^{5} ( 3 a_{k} + 5 ) = 55$, $\sum_{k=1}^{5} ( a_{k} + b_{k} ) = 32$ 
일 때, $\sum_{k=1}^{5} b_{k}$ 의 값을 구하시오.

## 풀이
$\sum_{k=1}^{5} ( 3 a_{k} + 5 ) = 55$ 에서 
$3 \sum_{k=1}^{5} a_{k} + 25 = 55$, 
$\sum_{k=1}^{5} a_{k} = 10$ 

$\sum_{k=1}^{5} ( a_{k} + b_{k} ) = 32$ 에서 
$\sum_{k=1}^{5} a_{k} + \sum_{k=1}^{5} b_{k} = 32$ 

$\therefore \sum_{k=1}^{5} b_{k}$ 
$= - \sum_{k=1}^{5} a_{k} + 32$ 
$= -10 + 32$ 
$= 22$
