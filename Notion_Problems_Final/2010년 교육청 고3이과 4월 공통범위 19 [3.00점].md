## 문제
(가) $f \left( \frac{5}{2} \right) = 2\sqrt{2}$
(나) 임의의 실수 $x$, $y$에 대하여 $f(x+y) = 2f(x)f(y)$이다.

실수 전체의 집합에서 정의된 함수 $f(x) = 2^{ax+b}$가 위 두 조건을 모두 만족시킬 때, $a^2 + b^2$의 값을 구하시오. (단, $a, b$는 상수이다.)

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
$(가)$에서 $f \left( \frac{5}{2} \right) = 2^{\frac{5}{2}a+b}$이므로
$2^{\frac{5}{2}a+b} = 2\sqrt{2} = 2^{\frac{3}{2}}$
$\therefore \frac{5}{2}a+b = \frac{3}{2} \cdots$ ㉠

$(나)$에서 $f(x+y) = 2f(x)f(y)$에 $x = 0$, $y = 0$을 대입하면
$f(0) = 2f(0)f(0)$이고, $f(x) = 2^{ax+b} > 0$이므로 $f(0) > 0$이다.
$f(0) = 2^b = \frac{1}{2} = 2^{-1}$
$\therefore b = -1$

$b = -1$을 ㉠에 대입하면
$\frac{5}{2}a - 1 = \frac{3}{2}$
$\frac{5}{2}a = \frac{5}{2}$
$\therefore a = 1$

따라서 $a = 1, b = -1$이므로
$a^2 + b^2 = 1^2 + (-1)^2 = 2$이다.

정답: ②
