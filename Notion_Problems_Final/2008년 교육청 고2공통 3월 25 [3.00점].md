## 문제
이차함수 $y = x^{2} + a x + b$의 그래프가 두 직선 $y = -x + 4$와 $y = 5x + 7$에 동시에 접할 때, 두 상수 $a$, $b$의 곱 $a b$의 값을 구하시오.

## 보기

## 풀이
(ⅰ) $y = x^{2} + a x + b$의 그래프와 직선 $y = -x + 4$가 접할 때 
$x^{2} + a x + b = -x + 4$, 즉 $x^{2} + ( a + 1 ) x + b - 4 = 0$의 판별식을 $D_{1}$이라 하면 
$D_{1} = ( a + 1 )^{2} - 4 ( b - 4 ) = 0$ 
$\therefore a^{2} + 2 a - 4 b + 17 = 0$ $\cdots$ ㉠ 

(ⅱ) $y = x^{2} + a x + b$의 그래프와 직선 $y = 5x + 7$이 접할 때 
$x^{2} + a x + b = 5x + 7$, 즉 $x^{2} + ( a - 5 ) x + b - 7 = 0$의 판별식을 $D_{2}$라 하면 
$D_{2} = ( a - 5 )^{2} - 4 ( b - 7 ) = 0$ 
$\therefore a^{2} - 10 a - 4 b + 53 = 0$ $\cdots$ ㉡ 

(ⅰ), (ⅱ)에서 ㉠, ㉡을 연립하면 
$a = 3$, $b = 8$ 

$\therefore a b = 3 \times 8 = 24$

정답: $24$
