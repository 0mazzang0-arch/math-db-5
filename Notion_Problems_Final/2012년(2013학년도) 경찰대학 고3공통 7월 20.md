## 문제
방정식 $x[x]+187= [x^{2} ]+[x]$ 의 근의 개수는? 
(단, $[x]$ 는 $x$ 보다 크지 않은 최대 정수이다.)

## 보기
① $58$   ② $67$   ③ $76$   ④ $85$   ⑤ $94$

## 풀이
$x[x]+187= [x^{2} ]+[x]$ 에서 $[x^{2} ]+[x]$ 이 정수이므로 $x[x]+187$ 도 정수이다.
즉, $x[x]$ 가 정수이다.
$x=n+ \alpha$ ($n$ 은 정수, $0 \leq \alpha < 1$) 이라 할 때, $[x]=n$ 이므로
$x[x]=n(n + \alpha) = n^{2}+ \alpha n$
$\alpha n$ 이 정수이므로 $\alpha$ 는 분모가 $n$ 인 유리수로 나타낼 수 있다.
즉, $\alpha = \frac{k}{n}$ ($0 \leq k < n$, $k$ 는 정수) 을 만족시키는 $k$ 가 존재한다.
따라서 $x=n+\frac{k}{n}$ ($0 \leq k < n$) 이라 하면 $x[x]+187= [x^{2} ]+[x]$ 에서
$(n+\frac{k}{n} ) n+187= [ (n+\frac{k}{n})^{2} ]+n$
$n^{2}+k+187= [n^{2}+2 k+\frac{k^{2}}{n^{2}} ]+n$
이때 $0 \leq k < n$ 이므로 $0 \leq \frac{k^{2}}{n^{2}} < 1$ 이다.
$n^{2}+k+187= n^{2}+2 k+n$
$n+k=187 \cdots ㉠$
그런데 $0 \leq k < n$ 이므로 $k = 187 - n$ 을 대입하면
$0 \leq 187 - n < n$
$n \leq 187$ 이고 $187 < 2n$ 에서 $\frac{187}{2} < n$ 이다.
즉, $\frac{187}{2} < n \leq 187 \cdots ㉡$
$n$ 은 정수이므로 $㉠, ㉡$ 에서 가능한 $(n, k)$ 의 순서쌍의 개수는
$187 - 93 = 94$ 이다.
따라서 주어진 방정식의 근의 개수는 $94$ 개다.

정답: ⑤
