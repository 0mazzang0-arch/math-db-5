## 문제
두 함수 $f(x) = [x^{2}]$ 과 $g(x) = [x]^{2}$ 에 대하여 <보기>에서 옳은 것만을 있는 대로 고른 것은?

$$
\begin{array}{|l|}
\hline
\text{ㄱ. } f(\sqrt{2}) > g(\sqrt{2}) \\
\text{ㄴ. } x \text{ 가 정수이면 } f(x) = g(x) \text{ 이다.} \\
\text{ㄷ. } f(x) = g(x) \text{ 이면 } x \text{ 는 정수이다.} \\
\hline
\end{array}
$$

(단, $[x]$ 는 $x$ 보다 크지 않은 최대의 정수이다.)

## 보기
① $ㄱ$   ② $ㄱ, ㄴ$   ③ $ㄱ, ㄷ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
ㄱ. $f(\sqrt{2}) = [(\sqrt{2})^{2}] = [2] = 2$, $g(\sqrt{2}) = [\sqrt{2}]^{2} = 1^{2} = 1$
$\therefore f(\sqrt{2}) > g(\sqrt{2})$ (참)

ㄴ. $x = n$ ($n$ 은 정수)이면 
$f(x) = [n^{2}] = n^{2}$, $g(x) = [n]^{2} = n^{2}$
$\therefore f(x) = g(x)$ (참)

ㄷ. (반례) $x = 0.5$ 일 때
$f(x) = [0.5^{2}] = [0.25] = 0$
$g(x) = [0.5]^{2} = 0^{2} = 0$
$f(x) = g(x)$ 이지만, $x$ 는 정수가 아니다. (거짓)

이상에서 옳은 것은 $ㄱ, ㄴ$ 이다.

정답: ②
