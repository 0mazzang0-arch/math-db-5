## 문제
수열 $\{ a_{n} \}$, $\{ b_{n} \}$ 에 대하여 <보기>에서 항상 옳은 것만을 있는 대로 고른 것은?

$$
\begin{array}{|l|}
\hline
ㄱ. \text{ 수열 } \{ a_{n}^{2} \} \text{ 이 발산하면 수열 } \{ a_{n} \} \text{ 은 발산한다.} \\
ㄴ. \text{ 수열 } \{ a_{n} + b_{n} \}, \{ a_{n} - b_{n} \} \text{ 이 모두 수렴하면 수열 } \{ a_{n}^{2} \}, \{ b_{n}^{2} \} \text{ 도 모두 수렴한다.} \\
ㄷ. \text{ 수열 } \{ a_{n} + b_{n} \}, \{ a_{n} b_{n} \} \text{ 이 모두 수렴하면 수열 } \{ a_{n} \}, \{ b_{n} \} \text{ 중 적어도 하나는 수렴한다.} \\
\hline
\end{array}
$$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
[정답] ③

$ㄱ.$ 대우명제: 수열 $\{ a_{n} \}$ 이 수렴하면 수열 $\{ a_{n}^{2} \}$ 이 수렴한다. 가 참이므로 주어진 명제는 참이다. (참)

$ㄴ.$ $\lim_{n \to \infty} ( a_{n} + b_{n} ) = \alpha$ , $\lim_{n \to \infty} ( a_{n} - b_{n} ) = \beta$ 라 두자. 

$\lim_{n \to \infty} a_{n} = \lim_{n \to \infty} \left\{ \frac{( a_{n} + b_{n} ) + ( a_{n} - b_{n} )}{2} \right\}$ 
$= \lim_{n \to \infty} \left( \frac{a_{n} + b_{n}}{2} \right) + \lim_{n \to \infty} \left( \frac{a_{n} - b_{n}}{2} \right)$ 
$= \frac{\alpha + \beta}{2}$ 

$\lim_{n \to \infty} b_{n} = \lim_{n \to \infty} \left\{ \frac{( a_{n} + b_{n} ) - ( a_{n} - b_{n} )}{2} \right\}$ 
$= \lim_{n \to \infty} \left( \frac{a_{n} + b_{n}}{2} \right) - \lim_{n \to \infty} \left( \frac{a_{n} - b_{n}}{2} \right)$ 
$= \frac{\alpha - \beta}{2}$ 

$\lim_{n \to \infty} a_{n}^{2} = \left( \frac{\alpha + \beta}{2} \right)^{2}$ , $\lim_{n \to \infty} b_{n}^{2} = \left( \frac{\alpha - \beta}{2} \right)^{2}$ 
따라서 수열 $\{ a_{n}^{2} \}$, $\{ b_{n}^{2} \}$ 은 수렴한다. (참)

$ㄷ.$ $a_{n} = ( -1 )^{n}$ , $b_{n} = ( -1 )^{n+1}$ 이면
$a_{n} + b_{n} = 0$ , $a_{n} b_{n} = -1$ 로 모두 수렴하지만 수열 $\{ a_{n} \}$, $\{ b_{n} \}$ 은 모두 발산한다. (거짓)

따라서 항상 옳은 것은 $ㄱ, ㄴ$ 이다.
