## 문제
다음 조건을 만족시키는 실수 $a$의 값의 범위는?

(가) $0 < a < 1$
(나) $a + \sqrt{7}$과 $a + 2\sqrt{2}$ 사이에 있는 정수의 개수는 $1$이다.

## 보기
① $8 - 3\sqrt{7} < a < 3 - 2\sqrt{2}$   ② $8 - 3\sqrt{7} < a < 3 - \sqrt{7}$   ③ $3 - 2\sqrt{2} < a < 3 - \sqrt{7}$   ④ $3 - 2\sqrt{2} < a < 2 - \sqrt{2}$   ⑤ $3 - \sqrt{7} < a < 2 - \sqrt{2}$

## 풀이
$\sqrt{4} < \sqrt{7} < \sqrt{8} < \sqrt{9}$ 이므로 
$2 < \sqrt{7} < 2\sqrt{2} < 3$ 이다. 
모든 변에 $a$를 더하면 
$a + 2 < a + \sqrt{7} < a + 2\sqrt{2} < a + 3$ 이 성립한다.

조건 (가)에서 $a > 0$ 이므로 
$a + 2 > 2$ 이고, 
$a < 1$ 이므로 
$a + 3 < 4$ 이다.

따라서 
$2 < a + \sqrt{7} < a + 2\sqrt{2} < 4$ 이다.

조건 (나)에서 $a + \sqrt{7}$과 $a + 2\sqrt{2}$ 사이에 한 개의 정수가 있으려면 그 정수는 $3$이어야 한다.
즉, $a + \sqrt{7} < 3 < a + 2\sqrt{2}$ 가 성립해야 한다.

$a + \sqrt{7} < 3$ 에서 
$a < 3 - \sqrt{7} \cdots \cdots$ ㉠ 

$a + 2\sqrt{2} > 3$ 에서 
$a > 3 - 2\sqrt{2} \cdots \cdots$ ㉡ 

㉠, ㉡에서 구하는 $a$의 값의 범위는 
$3 - 2\sqrt{2} < a < 3 - \sqrt{7}$ 이다.

정답: ③
