## 문제
실수 전체의 집합 $R$의 두 부분집합
$A = \{ x \mid x^2 - x - 6 > 0 \}$, $B = \{ x \mid x^2 + ax + b \le 0 \}$
이 다음 조건을 모두 만족시킬 때, 두 상수 $a$, $b$에 대하여 $a - b$의 값을 구하시오.

(가) $A \cup B = R$
(나) $A \cap B = \{ x \mid -5 \le x < -2 \}$

## 풀이
$A = \{ x \mid (x - 3)(x + 2) > 0 \} = \{ x \mid x < -2$ 또는 $x > 3 \}$이므로 조건 (가)와 (나)를 만족하는 범위를 수직선 위에 나타내면 그림과 같다.

> [!example] 📸 (그림/그래프)

그러므로
$B = \{ x \mid x^2 + ax + b \le 0 \} = \{ x \mid -5 \le x \le 3 \}$이다.
$x^2 + ax + b = (x + 5)(x - 3) = x^2 + 2x - 15 \le 0$에서
$a = 2$, $b = -15$
따라서 $a - b = 2 - (-15) = 17$

정답: $17$
