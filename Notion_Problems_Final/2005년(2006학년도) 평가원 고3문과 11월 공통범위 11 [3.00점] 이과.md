## 문제
양수 $a$에 대하여 $\log a$의 정수부분과 소수부분을 각각 $f(a)$, $g(a)$라 할 때, <보기>에서 옳은 것을 모두 고른 것은?
ㄱ. $f(2006) = 3$
ㄴ. $g(2) + g(6) = g(12) + 1$
ㄷ. $f(ab) = f(a) + f(b)$ 이면 $g(ab) = g(a) + g(b)$ 이다.

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
$\log a = f(a) + g(a)$ ($f(a)$는 정수, $0 \le g(a) < 1$)

ㄱ. $\log 2006 = \log (10^3 \times 2.006) = 3 + \log 2.006$
$\therefore f(2006) = 3$ (참)

ㄴ. $g(2) + g(6) = \log 2 + \log 6 = \log 12$
$g(12) + 1 = \{ \log 12 - f(12) \} + 1$
$f(12) = 1$이므로
$g(12) + 1 = \log 12 - 1 + 1 = \log 12$ (참)

ㄷ. $\log ab = f(ab) + g(ab)$ 이고 $f(ab) = f(a) + f(b)$ 이므로
$g(ab) = \log ab - f(ab)$
$= (\log a + \log b) - \{ f(a) + f(b) \}$
$= \{ \log a - f(a) \} + \{ \log b - f(b) \}$
$= g(a) + g(b)$ (참)

따라서 ㄱ, ㄴ, ㄷ 모두 옳다.

정답: ⑤
