## 문제
방정식 $2 \log_{4} ( 5 x + 1 ) = 1$의 실근을 $\alpha$라 할 때, $\log_{5} \frac{1}{\alpha}$의 값을 구하시오.

## 보기

## 풀이
$2 \log_{4} ( 5 x + 1 ) = 1$의 실근을 $\alpha$라 하면 
$2 \log_{4} ( 5 \alpha + 1 ) = 1$
$\log_{2} ( 5 \alpha + 1 ) = 1$
$5 \alpha + 1 = 2$
$\therefore \alpha = \frac{1}{5}$
$\therefore \log_{5} \frac{1}{\alpha} = \log_{5} 5 = 1$

정답: $1$
