## 문제
$x$에 대한 이차방정식 $x^{2} + ( m + n ) x - mn = 0$의 한 근이 $4 + \sqrt{2} i$일 때, $m^{2} + n^{2}$의 값을 구하시오. (단, $i = \sqrt{-1}$이고, $m$, $n$은 실수이다.)

## 보기

## 풀이
계수가 실수인 이차방정식 $x^{2} + ( m + n ) x - mn = 0$의 한 근이 $4 + \sqrt{2} i$이면 다른 한 근은 $4 - \sqrt{2} i$이다.
따라서 근과 계수의 관계에 의하여
$-( m + n ) = ( 4 + \sqrt{2} i ) + ( 4 - \sqrt{2} i )$
$\therefore m + n = -8$

$-mn = ( 4 + \sqrt{2} i ) ( 4 - \sqrt{2} i )$
$\therefore mn = -18$

따라서
$m^{2} + n^{2} = ( m + n )^{2} - 2 mn = ( -8 )^{2} - 2 \times ( -18 ) = 100$
