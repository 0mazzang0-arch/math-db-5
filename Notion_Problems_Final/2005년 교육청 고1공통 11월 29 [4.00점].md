## 문제
$n \ge 2$인 정수 $n$에 대하여 부등식 
$\sqrt{n+1} - \sqrt{n} < \frac{1}{2\sqrt{n}} < \sqrt{n} - \sqrt{n-1}$이 성립함을 이용하여 
$\frac{1}{2} \left( \frac{1}{\sqrt{2}} + \frac{1}{\sqrt{3}} + \frac{1}{\sqrt{4}} + \dots + \frac{1}{\sqrt{81}} \right)$의 값을 계산하였을 때, 정수 부분의 값을 구하시오.

## 보기

## 풀이
$n=2, 3, 4, \dots, 81$을 $\sqrt{n+1} - \sqrt{n} < \frac{1}{2\sqrt{n}} < \sqrt{n} - \sqrt{n-1}$에 대입하면 

$\sqrt{3} - \sqrt{2} < \frac{1}{2\sqrt{2}} < \sqrt{2} - \sqrt{1}$ 
$\sqrt{4} - \sqrt{3} < \frac{1}{2\sqrt{3}} < \sqrt{3} - \sqrt{2}$ 
$\sqrt{5} - \sqrt{4} < \frac{1}{2\sqrt{4}} < \sqrt{4} - \sqrt{3}$ 
$\vdots$ 
$\sqrt{82} - \sqrt{81} < \frac{1}{2\sqrt{81}} < \sqrt{81} - \sqrt{80}$ 

위의 식을 변변 더하면 
$\sqrt{82} - \sqrt{2} < \frac{1}{2} \left( \frac{1}{\sqrt{2}} + \frac{1}{\sqrt{3}} + \dots + \frac{1}{\sqrt{81}} \right) < \sqrt{81} - \sqrt{1}$ 

$7 < 9 - \sqrt{2} < \sqrt{82} - \sqrt{2} < \frac{1}{2} \left( \frac{1}{\sqrt{2}} + \frac{1}{\sqrt{3}} + \dots + \frac{1}{\sqrt{81}} \right) < 8$이므로 
정수 부분의 값은 $7$이다.

정답: $7$
