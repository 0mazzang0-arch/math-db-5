## 문제
자연수 $n$ ($n \ge 2$)에 대하여 직선 $y = -x + n$과 곡선 $y = | \log_2 x |$가 만나는 서로 다른 두 점의 $x$좌표를 각각 $a_n$, $b_n$ ($a_n < b_n$)이라 할 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?

## 보기
$$ \begin{array}{|l|} \hline \text{ㄱ. } a_2 < \frac{1}{4} \\ \text{ㄴ. } 0 < \frac{a_{n+1}}{a_n} < 1 \\ \text{ㄷ. } 1 - \frac{\log_2 n}{n} < \frac{b_n}{n} < 1 \\ \hline \end{array} $$

① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
정답 ④

그래프를 그려 $a_n$, $b_n$을 나타낸 후에 ㄱ, ㄴ을 따져보자.
먼저 $y = -x + n$과 $y = | \log_2 x |$의 그래프를 그리면 다음과 같아.

> [!example] 📸 (그림/그래프)

> [!example] 📸 (그림/그래프)

ㄱ. $a_2$는 [그림 $2$]와 같이 $y = -x + 2$와 $y = | \log_2 x |$의 그래프의 교점의 $x$좌표라구.
또, $2 = | \log_2 x |$에서 $\log_2 x = \pm 2$
$\therefore$ $x = \frac{1}{4}, 4$
$\therefore$ $a_2 > \frac{1}{4}$ (거짓)

ㄴ. [그림 $1$]에서와 같이 $a_{n+1}$은 $y = -x + n + 1$과 $y = | \log_2 x |$의 교점의 $x$좌표이므로
$0 < a_{n+1} < a_n$
$\therefore$ $0 < \frac{a_{n+1}}{a_n} < 1$ (참)

$b_n$의 범위와 $b_n$이 $y = -x + n$과 $y = \log_2 x$의 교점의 $x$좌표임을 활용해.
ㄷ. [그림 $1$]에서 $1 < b_n < n$이므로 $\frac{b_n}{n} < 1$이지?
또, $b_n$은 $y = -x + n$과 $y = \log_2 x$의 교점의 좌표니까
$-b_n + n = \log_2 b_n$
$\therefore$ $b_n = n - \log_2 b_n$
이때, $\log_2 b_n < \log_2 n$이므로
$\frac{b_n}{n} = 1 - \frac{\log_2 b_n}{n} > 1 - \frac{\log_2 n}{n}$
$\therefore$ $1 - \frac{\log_2 n}{n} < \frac{b_n}{n} < 1$ (참)

따라서, 옳은 것은 $ㄴ, ㄷ$이야..
