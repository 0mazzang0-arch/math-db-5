## 문제
$a^{2} + b^{2} = 4$ 인 복소수 $z = a + bi$ 에 대하여 $\frac{i}{z - 1}$ 가 양의 실수일 때, $z^{2}$ 의 값은? (단, $a$, $b$ 는 실수이다.)

## 보기
① $-2 + 2\sqrt{3}i$   ② $2 + 2\sqrt{3}i$   ③ $2 - 2\sqrt{3}i$   ④ $2\sqrt{3} + 2i$   ⑤ $2\sqrt{3} - 2i$

## 풀이
$\frac{i}{z - 1} = \frac{i}{a - 1 + bi} = \frac{i(a - 1 - bi)}{(a - 1)^{2} + b^{2}} = \frac{b}{(a - 1)^{2} + b^{2}} + \frac{a - 1}{(a - 1)^{2} + b^{2}}i$
이므로, $\frac{i}{z - 1}$ 가 양의 실수이면 $\frac{a - 1}{(a - 1)^{2} + b^{2}} = 0$ 이고 $\frac{b}{(a - 1)^{2} + b^{2}} > 0$ 이어야 한다.
즉, $a = 1$, $b > 0$ 이다.
$a^{2} + b^{2} = 4$ 에 $a = 1$ 을 대입하면 $1^{2} + b^{2} = 4$ 에서 $b^{2} = 3$ 이다.
$b > 0$ 이므로 $b = \sqrt{3}$ 이다.
따라서 $z = 1 + \sqrt{3}i$ 이다.
$z^{2} = (1 + \sqrt{3}i)^{2} = 1 + 2\sqrt{3}i - 3 = -2 + 2\sqrt{3}i$

정답은 ① 이다.
