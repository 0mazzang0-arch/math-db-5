## 문제
방정식 $4x^{3} + 1003x + 1004 = 0$ 의 세 근을 $\alpha$, $\beta$, $\gamma$ 라 할 때, $(\alpha + \beta)^{3} + (\beta + \gamma)^{3} + (\gamma + \alpha)^{3}$ 의 값을 구하시오.

## 풀이
$4x^{3} + 1003x + 1004 = 0$ 의 세 근이 $\alpha$, $\beta$, $\gamma$ 이므로 삼차방정식의 근과 계수와의 관계에 의하여

$\alpha + \beta + \gamma = 0$ $\cdots$ ㉠
$\alpha\beta + \beta\gamma + \gamma\alpha = \frac{1003}{4}$ $\cdots$ ㉡
$\alpha\beta\gamma = -251$ $\cdots$ ㉢

따라서
$(\alpha + \beta)^{3} + (\beta + \gamma)^{3} + (\gamma + \alpha)^{3}$
$= (-\gamma)^{3} + (-\alpha)^{3} + (-\beta)^{3}$ (∵ ㉠)
$= -(\alpha^{3} + \beta^{3} + \gamma^{3})$
$= -\{(\alpha + \beta + \gamma)(\alpha^{2} + \beta^{2} + \gamma^{2} - \alpha\beta - \beta\gamma - \gamma\alpha) + 3\alpha\beta\gamma\}$
$= -3\alpha\beta\gamma$ (∵ ㉠, ㉡)
$= 753$ (∵ ㉢)

정답: $753$
