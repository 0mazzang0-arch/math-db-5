## 문제
복소수 $z_{1} = 1 + 2i$에 대하여 $z_{2} = \overline{z_{1}} + (1 + i)$, $z_{3} = \overline{z_{2}} + (1 + i)$, $z_{4} = \overline{z_{3}} + (1 + i)$라 하자. 같은 방법으로 $z_{5}$, $z_{6}$, $\cdots$을 차례로 정할 때, $z_{100}$은? (단, $i = \sqrt{-1}$이고 $\overline{z}$는 $z$의 켤레복소수이다.)

## 보기
① $100 + i$   ② $100 - i$   ③ $100 + 2i$   ④ $101 - i$   ⑤ $101 + 2i$

## 풀이
$z_{2}$, $z_{3}$, $z_{4}$, $z_{5}$, $\cdots$를 차례로 구해보면 

$z_{2} = \overline{z_{1}} + (1 + i) = 1 - 2i + (1 + i) = 2 - i$ 

$z_{3} = \overline{z_{2}} + (1 + i) = 2 + i + (1 + i) = 3 + 2i$ 

$z_{4} = \overline{z_{3}} + (1 + i) = 3 - 2i + (1 + i) = 4 - i$ 

$z_{5} = \overline{z_{4}} + (1 + i) = 4 + i + (1 + i) = 5 + 2i$ 

$\cdots \cdots \cdots \cdots \cdots$ 

실수부분은 $2$, $3$, $4$, $5$, $\cdots$로 나타나고, 허수부분은 $-1$, $2$가 반복되어 나타난다. 

따라서 $z_{100} = 100 - i$이다.

정답: ②
