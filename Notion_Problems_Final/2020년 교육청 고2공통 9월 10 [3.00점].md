## 문제
삼각형 $ABC$ 에서 $\frac{2}{\sin A} = \frac{3}{\sin B} = \frac{4}{\sin C}$ 일 때, $\cos C$ 의 값은?

## 보기
① $-\frac{1}{2}$   ② $-\frac{1}{4}$   ③ $0$   ④ $\frac{1}{4}$   ⑤ $\frac{1}{2}$

## 풀이
삼각형 $ABC$ 에서 $\overline{BC} = a, \overline{CA} = b, \overline{AB} = c$ 라 하면 사인법칙에 의하여 
$\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C}$ 이므로 
$a : b : c = 2 : 3 : 4$ 이다.
$a = 2k, b = 3k, c = 4k$ ($k > 0$) 이라 하면 코사인법칙에 의하여 
$\cos C = \frac{a^{2} + b^{2} - c^{2}}{2ab}$
$\cos C = \frac{(2k)^{2} + (3k)^{2} - (4k)^{2}}{2 \cdot 2k \cdot 3k}$
$\cos C = \frac{4k^{2} + 9k^{2} - 16k^{2}}{12k^{2}}$
$\cos C = -\frac{3k^{2}}{12k^{2}} = -\frac{1}{4}$

정답: ②
