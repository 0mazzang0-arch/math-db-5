## 문제
$\alpha = 2 - 7i$, $\beta = -1 + 4i$ 일 때, $\alpha \bar{\alpha} + \bar{\alpha} \beta + \alpha \bar{\beta} + \beta \bar{\beta}$ 의 값은? (단, $i = \sqrt{-1}$ 이고, $\bar{\alpha}$, $\bar{\beta}$ 는 각각 $\alpha$, $\beta$ 의 켤레복소수이다.)

## 보기
① $8$   ② $9$   ③ $10$   ④ $11$   ⑤ $12$

## 풀이
$\alpha = 2 - 7i$, $\beta = -1 + 4i$ 이므로 
$\bar{\alpha} = 2 + 7i$, $\bar{\beta} = -1 - 4i$ 이다.

따라서
$\alpha \bar{\alpha} + \bar{\alpha} \beta + \alpha \bar{\beta} + \beta \bar{\beta}$
$= \bar{\alpha} ( \alpha + \beta ) + \bar{\beta} ( \alpha + \beta )$
$= ( \alpha + \beta ) ( \bar{\alpha} + \bar{\beta} )$

이때
$\alpha + \beta = ( 2 - 7i ) + ( -1 + 4i ) = 1 - 3i$
$\bar{\alpha} + \bar{\beta} = ( 2 + 7i ) + ( -1 - 4i ) = 1 + 3i$

이므로 식의 값은
$( 1 - 3i ) ( 1 + 3i ) = 1^2 - ( 3i )^2 = 1 + 9 = 10$

정답은 ③이다.
