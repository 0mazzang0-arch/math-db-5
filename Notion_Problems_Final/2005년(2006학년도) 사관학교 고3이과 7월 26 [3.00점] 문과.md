## 문제
$xy=10$ 이고 $1 \le x \le 10^4$ 일 때, 
$(\log_{10} y)^3 + 3(\log_{10} x)^2 - 6\log_{10} x + 15$
의 최댓값을 구하시오.

## 보기

## 풀이
$\log_{10} x=X$, $\log_{10} y=Y$ 로 치환하면 주어진 조건에 의하여 $X+Y=1$, $0 \le X \le 4$ 이고, 주어진 식은 다음과 같이 정리할 수 있다. 

$(\log_{10} y)^3 + 3(\log_{10} x)^2 - 6\log_{10} x + 15 = Y^3 + 3X^2 - 6X + 15$ 
$= (1-X)^3 + 3X^2 - 6X + 15 = -X^3 + 6X^2 - 9X + 16$ 

$f(X) = -X^3 + 6X^2 - 9X + 16$ 라 하면 제한변역 $0 \le X \le 4$ 에서 최댓값을 구하면 된다. 

$f'(X) = -3X^2 + 12X - 9 = -3(X^2 - 4X + 3) = -3(X-1)(X-3)$ 
이므로 $X=1, 3$ 에서 각각 극소와 극대를 가진다. 경계값과 극댓값, 극솟값을 구하면 

$f(0) = 16$, $f(1) = 12$, $f(3) = 16$, $f(4) = 12$ 이므로 최댓값은 $16$ 이다. 

정답: $16$
