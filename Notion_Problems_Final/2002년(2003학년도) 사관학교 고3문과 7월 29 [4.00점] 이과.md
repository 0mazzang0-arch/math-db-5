## 문제
$(\because \sin^{2} A + \cos^{2} A = 1 \iff \sin^{2} A + \sin^{2} (90^{\circ} - A) = 1)$ 수열 $\{a_{n}\}$을 $a_{n} = (2n)^{\circ}$라 하고, 한 변의 길이가 $\sin a_{n}$인 정사각형의 면적을 $A_{n}$으로 정의한다. 이때, $\sum_{n=1}^{45} A_{n}$의 값을 구하시오.

## 풀이
$A_{n} = \sin^{2} a_{n}$ 이므로 

$\sum_{n=1}^{45} A_{n}$ 
$= \sin^{2} a_{1} + \sin^{2} a_{2} + \cdots + \sin^{2} a_{45}$ 
$= \sin^{2} 2^{\circ} + \sin^{2} 4^{\circ} + \cdots + \sin^{2} 88^{\circ} + \sin^{2} 90^{\circ}$ 

$88^{\circ} = 90^{\circ} - 2^{\circ}$ 이므로 $\sin^{2} 88^{\circ} = \cos^{2} 2^{\circ}$ 이다. 
따라서 $\sin^{2} 2^{\circ} + \sin^{2} 88^{\circ} = 1$ 과 같은 쌍이 $2^{\circ}$부터 $44^{\circ}$까지 총 $22$개 존재한다.
$= 22 + \sin^{2} 90^{\circ}$
$= 22 + 1 = 23$

정답: $23$
