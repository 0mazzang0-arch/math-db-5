## 문제
집합 $A = \{ 2, 3, 4, 5 \}$에서 연산 $\ast$를 다음과 같이 정의한다.
$$ a \ast b = \begin{cases} a - (a - b) [ \log_{a} b ], & b = 2 \text{ 또는 } b = 3 \\ b - (b - a) [ \log_{b} a ], & b = 4 \text{ 또는 } b = 5 \end{cases} $$
($[x]$는 $x$를 넘지 않는 최대 정수이다.)
이때, 방정식 $4 \ast (4 \ast x) = x$를 만족하는 $x$의 개수는?

## 보기
① $0$   ② $1$   ③ $2$   ④ $3$   ⑤ $4$

## 풀이
연산 $4 \ast x$를 각각 계산하면
$x = 2$일 때, $4 \ast 2 = 4 - (4 - 2) [ \log_{4} 2 ] = 4 - 2 \cdot [ \frac{1}{2} ] = 4 - 0 = 4$
$\therefore 4 \ast 2 = 4$

$x = 3$일 때, $4 \ast 3 = 4 - (4 - 3) [ \log_{4} 3 ] = 4 - [ \log_{4} 3 ] = 4 - 0 = 4$
$\therefore 4 \ast 3 = 4$

$x = 4$일 때, $4 \ast 4 = 4 - (4 - 4) [ \log_{4} 4 ] = 4 - 0 = 4$
$\therefore 4 \ast 4 = 4$

$x = 5$일 때, $4 \ast 5 = 5 - (5 - 4) [ \log_{5} 4 ] = 5 - [ \log_{5} 4 ] = 5 - 0 = 5$
$\therefore 4 \ast 5 = 5$

그러므로 방정식 $4 \ast (4 \ast x) = x$에서
(ⅰ) $x = 2$ 이면 $4 \ast (4 \ast 2) = 4 \ast 4 = 4$ (X)
(ⅱ) $x = 3$ 이면 $4 \ast (4 \ast 3) = 4 \ast 4 = 4$ (X)
(ⅲ) $x = 4$ 이면 $4 \ast (4 \ast 4) = 4 \ast 4 = 4$ (O)
(ⅳ) $x = 5$ 이면 $4 \ast (4 \ast 5) = 4 \ast 5 = 5$ (O)

$\therefore x = 4$ 또는 $x = 5$
집합 $A$에서 이를 만족하는 $x$의 개수는 $2$이다.
