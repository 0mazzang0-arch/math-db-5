## 문제
두 자리 자연수의 집합 $N$에서 실수 전체의 집합 $R$로 대응되는 함수 $f(x)$를 $f(x)= \log x-[ \log x]$라 정의할 때, <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $[x]$는 $x$를 넘지 않는 최대의 정수이다.)

ㄱ. 함수 $f(x)$의 최솟값은 $0$이다. 
ㄴ. $f(x)+f(y)=1$을 만족하는 $x$, $y$가 존재한다. 
ㄷ. $2$ 이상의 자연수 $n$에 대하여 $f (x^{n} )=n f(x)$이면 $10 \leq x<10 \sqrt[n]{10}$이다. 

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
ㄱ. $f(x)$가 소수부분이므로 최솟값은 $f(10)=0$ (참) 

ㄴ. $f(20)+f(50)=1$ (참) 

ㄷ. $\log x=1+ \alpha$ (단, $0 \leq \alpha < 1$)이라고 하면 
$\log x^{n}=n+n \alpha$, $0 \leq n \alpha < n$이고, 
i) $0 \leq n \alpha < 1$ 일 때, 
$f (x^{n} )=n \alpha$, $n f(x)=n \alpha$이므로 $f (x^{n} )=n f(x)$ 
$\therefore 0 \leq \alpha < \frac{1}{n}$ 이고 $1 \leq \log x < 1 + \frac{1}{n}$ 
$\therefore 10 \leq x < 10^{1+ \frac{1}{n}}$ 

ii) $1 \leq n \alpha < n$ 일 때, 
$f (x^{n} )=n \alpha - [n \alpha]$이고 $[n \alpha]$의 값은 $1, 2, \dots, n-1$이므로 $f (x^{n} ) \neq n f(x)$ 

따라서 $10 \leq x < 10^{1+ \frac{1}{n}} = 10 \sqrt[n]{10}$ (참) 

정답: ⑤
