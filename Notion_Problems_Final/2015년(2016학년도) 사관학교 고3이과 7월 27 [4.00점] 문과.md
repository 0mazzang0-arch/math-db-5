## 문제
좌표평면에서 곡선 $y= \cos 2x$ 가 두 직선 $x=t$, $x=-t$ ($0 < t < \frac{\pi}{4}$)와 만나는 점을 각각 $P$, $Q$라 하고, 곡선 $y= \cos 2x$ 가 $y$축과 만나는 점을 $R$라 하자. 세 점 $P$, $Q$, $R$를 지나는 원의 중심을 $C(0, f(t))$라 할 때, $\lim_{t \to 0+} f(t) = \alpha$ 이다. $100 \alpha$ 의 값을 구하시오.

## 보기

## 풀이
정답: $75$

선분 $PQ$의 수직이등분선이 $y$축이므로 $f(t)$는 선분 $PR$을 수직이등분하는 직선의 $y$절편이다.
$P(t, \cos 2t)$, $R(0, 1)$이므로 선분 $PR$의 중점은 $(\frac{t}{2}, \frac{\cos 2t + 1}{2})$ 이고 선분 $PR$의 기울기는 $\frac{\cos 2t - 1}{t}$ 이다.
따라서 선분 $PR$을 수직이등분하는 직선의 방정식은
$$y = -\frac{t}{\cos 2t - 1} \left( x - \frac{t}{2} \right) + \frac{\cos 2t + 1}{2}$$
$\therefore f(t) = -\frac{t}{\cos 2t - 1} \left( 0 - \frac{t}{2} \right) + \frac{\cos 2t + 1}{2}$
$= \frac{t^2}{2(\cos 2t - 1)} + \frac{\cos 2t + 1}{2}$
$= \frac{t^2(\cos 2t + 1)}{2(\cos^2 2t - 1)} + \frac{\cos 2t + 1}{2}$
$= -\frac{t^2(\cos 2t + 1)}{2\sin^2 2t} + \frac{\cos 2t + 1}{2}$

$\therefore \alpha = \lim_{t \to 0+} f(t)$
$= \lim_{t \to 0+} \left( -\frac{t^2(\cos 2t + 1)}{2\sin^2 2t} + \frac{\cos 2t + 1}{2} \right)$
$= \lim_{t \to 0+} \left( - \left( \frac{2t}{\sin 2t} \right)^2 \cdot \frac{\cos 2t + 1}{8} + \frac{\cos 2t + 1}{2} \right)$
$= -\frac{1}{4} + 1 = \frac{3}{4}$

따라서 $100 \alpha = 75$ 이다.
