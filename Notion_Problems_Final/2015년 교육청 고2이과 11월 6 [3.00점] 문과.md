## 문제
함수 $f(x) = (x + \pi)\sin x$ 에 대하여 $f'(0)$ 의 값은?

## 보기
① $-\pi$   ② $-\frac{\pi}{2}$   ③ $0$   ④ $\frac{\pi}{2}$   ⑤ $\pi$

## 풀이
$f(x) = (x + \pi)\sin x$ 를 $x$ 에 대하여 미분하면 곱의 미분법에 의해
$f'(x) = \sin x + (x + \pi)\cos x$
$x = 0$ 을 대입하면
$f'(0) = \sin 0 + (0 + \pi)\cos 0 = 0 + \pi \times 1 = \pi$
따라서 정답은 ⑤이다.
