## 문제
등차수열 $\{ a_{n} \}$ 이 다음 조건을 만족시킨다.

(가) $a_{1} + a_{2} + a_{3} = 159$
(나) $a_{m-2} + a_{m-1} + a_{m} = 96$

인 자연수 $m$ 에 대하여 $\sum_{k=1}^{m} a_{k} = 425$ 일 때, $a_{11}$ 의 값을 구하시오. (단, $m > 3$)

## 풀이
등차수열 $\{ a_{n} \}$ 의 공차를 $d$ 라 하면
(가)에서 $a_{1} + ( a_{1} + d ) + ( a_{1} + 2d ) = 159$ 이므로
$a_{1} + d = 53 \cdots$ ㉠

(나)에서 $( a_{m} - 2d ) + ( a_{m} - d ) + a_{m} = 96$ 이므로
$a_{m} - d = 32 \cdots$ ㉡

㉠, ㉡에서 $a_{1} + a_{m} = 85$ 이므로
$\sum_{k=1}^{m} a_{k} = \frac{m}{2} ( a_{1} + a_{m} ) = \frac{m}{2} \times 85 = 425$ 이고
$m = 10$ 이다.

또한 ㉠, ㉡에서 $a_{1} = 56$, $d = -3$ 이므로
$a_{n} = -3n + 59$

따라서 $a_{11} = -3 \times 11 + 59 = 26$

[정답] $26$
