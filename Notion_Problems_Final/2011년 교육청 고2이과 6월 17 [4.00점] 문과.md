## 문제
양의 실수 $x$ 에 대하여 $\log x$ 의 정수부분을 $f(x)$, 소수부분을 $g(x)$ 라 하자. $m$, $n$ 이 자연수일 때, 옳은 것만을 <보기>에서 있는 대로 고른 것은?

$$ \begin{array}{|l|} \hline \text{<보 기>} \\ \text{ㄱ. } f(n)+g(n) \text{ 의 최솟값은 } 0 \text{ 이다.} \\ \text{ㄴ. } f(m)=f(n)=2 \text{ 이면 } |m-n| \text{ 의 최댓값은 } 900 \text{ 이다.} \\ \text{ㄷ. } m \neq n \text{ 이고 } g(m)=g(n)= \log 7 \text{ 이면 } |m-n| \text{ 의 최솟값은 } 70 \text{ 이다.} \\ \hline \end{array} $$

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄱ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
[정답] ① 

ㄱ. $n$ 이 증가할수록 $\log n$ 의 값은 증가하므로 $\log n=f(n)+g(n)$ 은 $n=1$ 일 때 최솟값을 갖는다. $f(n)+g(n)$ 의 최솟값은 $\log 1=0$ (참) 

ㄴ. $f(m)=f(n)=2$ 이면 $m$, $n$ 은 세 자리의 자연수이므로 $|m-n|$ 의 최댓값은 $|999-100|=899$ (거짓) 

ㄷ. $g(n)= \log 7$ 을 만족시키는 자연수 $n$ 의 값은 $7, 70, 700, 7000, \cdots$ 이다. $m \neq n$ 이고 $g(m)=g(n)= \log 7$ 일 때, $|m-n|$ 의 최솟값은 $|70-7|=63$ (거짓) 

따라서 옳은 것은 ㄱ이다.
