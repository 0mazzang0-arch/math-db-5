## 문제
서로 다른 두 자연수 `$a$`, `$b$`에 대하여 두 확률변수 `$X$`, `$Y$`가 각각 정규분포 `$N(a, \sigma^2)$`, `$N(2b-a, \sigma^2)$`을 따른다. 확률변수 `$X$`의 확률밀도함수 `$f(x)$`와 확률변수 `$Y$`의 확률밀도함수 `$g(x)$`가 다음 조건을 만족시킬 때, `$a+b$`의 값을 구하시오.

(가) `$P(X \le 11) = P(Y \ge 11)$`
(나) `$f(17) < g(10) < f(15)$`

## 보기

## 풀이
조건 (가)에서 
`$$ P(X \le 11) = P\left(Z \le \frac{11-a}{\sigma}\right) $$`
`$$ P(Y \ge 11) = P\left(Z \ge \frac{11-(2b-a)}{\sigma}\right) = P\left(Z \ge \frac{11-2b+a}{\sigma}\right) $$`
`$P(X \le 11) = P(Y \ge 11)$`이므로
`$$ \frac{11-a}{\sigma} + \frac{11-2b+a}{\sigma} = 0 $$`
`$$ \therefore b = 11 $$`
확률변수 `$X$`의 평균이 `$a$`, 확률변수 `$Y$`의 평균이 `$22-a$`이고, 두 확률변수 `$X$`, `$Y$`의 표준편차가 같으므로 두 확률밀도함수 `$f(x)$`, `$g(x)$`는 `$x=11$`에 대하여 대칭인 함수이다.

> [!example] 📸 (그림/그래프)

`$g(10) = f(12)$`이므로 조건 (나)에서 
`$$ f(17) < g(10) = f(12) < f(15) $$`
함수 `$f(x)$`는 `$x=a$`에 대하여 대칭이므로 함숫값의 대소 관계에 의해 평균 `$a$`의 범위를 구하면 다음과 같다.
`$$ |15-a| < |12-a| < |17-a| $$`
`$12 < a < 17$`라 하면
`$$ |15-a| < a-12 < 17-a $$`
`$$ \frac{27}{2} < a < \frac{29}{2} $$`
`$$ \therefore a = 14 $$`
따라서 `$a+b = 14+11 = 25$`이다.

정답: `$25$`
