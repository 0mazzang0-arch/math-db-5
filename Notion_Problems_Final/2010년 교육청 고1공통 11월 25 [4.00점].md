## 문제
집합 $X = \{ 1, 2, 3 \}$ 에 대하여 함수 $f : X \to X$ 가 그림과 같이 주어져 있다.

> [!example] 📸 (그림/그래프)
> ($1 \to 2$, $2 \to 3$, $3 \to 1$ 로 대응되는 일대일대응 그래프)

$f^{1}(x) = f(x)$, $f^{n+1}(x) = f(f^{n}(x)) (n = 1, 2, 3, \cdots)$ 이라 할 때, $f^{2010}(2) + f^{2011}(3)$ 의 값을 구하시오.

## 풀이
$f(1) = 2$, $f(2) = 3$, $f(3) = 1$ 이므로
$f^{3}(1) = f(f(f(1))) = f(f(2)) = f(3) = 1$ 이다.
같은 방법으로 
$f^{3}(2) = 2$, $f^{3}(3) = 3$ 이므로 
$f^{3}(x) = x$ 이다. 
그러므로 $f^{3} = I$ (항등함수) 이므로 
$f^{2010}(2) = (f^{3})^{670}(2) = I(2) = 2$ 
$f^{2011}(3) = f((f^{3})^{670}(3)) = f(I(3)) = f(3) = 1$ 
이다. 
따라서 
$f^{2010}(2) + f^{2011}(3) = 2 + 1 = 3$
