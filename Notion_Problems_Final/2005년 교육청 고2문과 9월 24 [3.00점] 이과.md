## 문제
$\log \frac{1}{2} + \log \frac{2}{3} + \log \frac{3}{4} + \cdots + \log \frac{n}{n+1} = -2$ 일 때, 자연수 $n$ 의 값을 구하시오.

## 풀이
$\log \frac{1}{2} + \log \frac{2}{3} + \log \frac{3}{4} + \cdots + \log \frac{n}{n+1} = -2$
$\log \left( \frac{1}{2} \times \frac{2}{3} \times \frac{3}{4} \times \cdots \times \frac{n}{n+1} \right) = -2$
$\log \frac{1}{n+1} = -2$
$\frac{1}{n+1} = 10^{-2} = \frac{1}{100}$
$n + 1 = 100$
$\therefore n = 99$

정답: $99$
