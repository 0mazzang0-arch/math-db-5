## 문제
원 $x^{2} + (y + 4)^{2} = 10$을 $x$축의 방향으로 $-4$만큼, $y$축의 방향으로 $2$만큼 평행이동하였더니 원 $x^{2} + y^{2} + ax + by + c = 0$과 일치하였다. $a + b + c$의 값은? (단, $a$, $b$, $c$는 상수이다.)

## 보기
① $14$   ② $16$   ③ $18$   ④ $20$   ⑤ $22$

## 풀이
원 $x^{2} + (y + 4)^{2} = 10$을 $x$축의 방향으로 $-4$만큼, $y$축의 방향으로 $2$만큼 평행이동하면
$$(x + 4)^{2} + \{ (y - 2) + 4 \}^{2} = 10$$
$$(x + 4)^{2} + (y + 2)^{2} = 10$$
$$x^{2} + 8x + 16 + y^{2} + 4y + 4 = 10$$
$$x^{2} + y^{2} + 8x + 4y + 10 = 0$$
이 원이 $x^{2} + y^{2} + ax + by + c = 0$과 일치하므로
$a = 8$, $b = 4$, $c = 10$
따라서
$a + b + c = 8 + 4 + 10 = 22$

정답: ⑤
