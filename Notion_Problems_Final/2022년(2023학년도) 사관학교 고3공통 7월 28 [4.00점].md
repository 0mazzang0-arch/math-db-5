## 문제
두 집합 $X = \{ 1, 2, 3, 4 \}$, $Y = \{ 0, 1, 2, 3, 4, 5, 6 \}$ 에 대하여 $X$ 에서 $Y$ 로의 함수 $f$ 중에서 $f( 1 ) + f( 2 ) + f( 3 ) + f( 4 ) = 8$ 을 만족시키는 함수 $f$ 의 개수는?

## 보기
① $137$   ② $141$   ③ $145$   ④ $149$   ⑤ $153$

## 풀이
정답: ④

$f( 1 ) + f( 2 ) + f( 3 ) + f( 4 ) = 8$ 을 만족시키는 경우의 수는 $f( n ) ( 1 \le n \le 4 )$ 에서 중복을 허락하여 $8$ 번 뽑는 경우의 수에서 $f( n )$ 의 값이 $7$ 또는 $8$ 이 되는 경우의 수를 제외한 수와 같다.

$\therefore {}_4 H_8 - ( {}_4 C_1 \times {}_3 C_1 + {}_4 C_1 )$
$= {}_{11} C_8 - 16$
$= {}_{11} C_3 - 16$
$= \frac{11 \times 10 \times 9}{3 \times 2 \times 1} - 16$
$= 165 - 16 = 149$
