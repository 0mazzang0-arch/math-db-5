## 문제
임의의 각 $\theta$ 에 대하여 다음 보기 중 항상 옳은 것만을 있는 대로 고른 것은?

ㄱ. $\sin \left( \frac{\pi}{2} + \theta \right) = \cos (\pi + \theta)$
ㄴ. $\cos \left( \frac{\pi}{2} + \theta \right) = \sin (\pi + \theta)$
ㄷ. $\tan \left( \frac{\pi}{2} + \theta \right) = \frac{1}{\tan (\pi + \theta)}$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄷ$   ④ $ㄱ, ㄴ$   ⑤ $ㄴ, ㄷ$

## 풀이
[정답] ②

ㄱ. $\sin \left( \frac{\pi}{2} + \theta \right) = \cos \theta$ 이고 $\cos (\pi + \theta) = -\cos \theta$ 이다.
$\therefore$ 거짓

ㄴ. $\cos \left( \frac{\pi}{2} + \theta \right) = -\sin \theta$ 이고 $\sin (\pi + \theta) = -\sin \theta$ 이다.
$\therefore$ 참

ㄷ. $\tan \left( \frac{\pi}{2} + \theta \right) = -\cot \theta$ 이고 $\frac{1}{\tan (\pi + \theta)} = \frac{1}{\tan \theta} = \cot \theta$ 이다.
$\therefore$ 거짓

따라서 임의의 각 $\theta$ 에 대하여 항상 옳은 것은 $ㄴ$이다.
