## 문제
$1 < a < b$인 두 실수 $a$, $b$에 대하여 옳은 것을 <보기>에서 모두 고른 것은?

$$\begin{array}{|l|} \hline \text{<보 기>} \\ ㄱ. \log_{b} a < \log_{a} b \\ ㄴ. \frac{1}{a} \log a < \frac{1}{b} \log b \\ ㄷ. 2 \log(a + b) < \log 2(a^{2} + b^{2}) \\ \hline \end{array}$$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄷ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
[정답] ③ 

ㄱ. $1 < a < b$에서 $0 < \log a < \log b$이다. 
$\log_{b} a = \frac{\log a}{\log b} < \frac{\log b}{\log a} = \log_{a} b$ (참) 

ㄴ. (반례) $a = 10$, $b = 100$일 때 
$10 < 100$이지만 
$\frac{1}{10} \log 10 > \frac{1}{100} \log 100$ (거짓) 

ㄷ. $2 \log(a + b) < \log 2(a^{2} + b^{2}) \iff \log(a + b)^{2} < \log 2(a^{2} + b^{2})$ 
$2(a^{2} + b^{2}) - (a + b)^{2} = a^{2} - 2ab + b^{2} = (a - b)^{2} > 0$ ($\because a < b$) 
$(a + b)^{2} < 2(a^{2} + b^{2})$ 
$\therefore 2 \log(a + b) < \log 2(a^{2} + b^{2})$ (참)
