## 문제

다음은 삼차다항식 $P(x)$가 $P(k) = \frac{k+1}{k}$ ($k=1, 2, 3, 4$)를 만족할 때, $P(-1)$의 값을 구하는 과정이다.

$$ \begin{array}{|l|} \hline Q(x) = x P(x) - (x+1) \cdots ㉠ \\ \text{이라 하면 } Q(x) \text{는 사차다항식이다.} \\ \text{자연수 } k (1 \le k \le 4) \text{에 대하여 } Q(k) = (가) \text{이므로} \\ Q(x) = a(x-1)(x-2)(x-3)(x-4) (a \text{는 상수}) \cdots ㉡ \\ \text{그런데 ㉠에서 } Q(0) = -1 \text{이므로 ㉡에서 } a = (나) \\ \text{따라서 ㉠, ㉡에서 } P(-1) = -Q(-1) = (다) \\ \hline \end{array} $$

위의 과정에서 $(가), (나), (다)$에 알맞은 값을 구하시오.

## 보기

① $(가): 0, (나): -\frac{1}{24}, (다): 5$   ② $(가): -1, (나): \frac{1}{60}, (다): 5$   ③ $(가): 0, (나): -\frac{1}{24}, (다): 2$   ④ $(가): -1, (나): \frac{1}{60}, (다): 2$   ⑤ $(가): 0, (나): -\frac{1}{60}, (다): 5$

## 풀이

자연수 $k$ ($1 \le k \le 4$)에 대하여
$Q(k) = k P(k) - (k+1) = k \cdot \frac{k+1}{k} - (k+1) = 0$
$\therefore (가) = 0 \cdots (가)$

$Q(0) = -1$이므로 ㉡에서
$Q(0) = a(0-1)(0-2)(0-3)(0-4) = 24a = -1$
$\therefore a = -\frac{1}{24} \cdots (나)$

따라서 ㉠에서 $x = -1$을 대입하면
$Q(-1) = -1 \cdot P(-1) - (-1+1) = -P(-1)$
$\therefore P(-1) = -Q(-1)$
㉡에서
$P(-1) = -a(-1-1)(-1-2)(-1-3)(-1-4)$
$P(-1) = -(-\frac{1}{24}) \cdot (-2) \cdot (-3) \cdot (-4) \cdot (-5)$
$P(-1) = \frac{1}{24} \cdot 120 = 5 \cdots (다)$

따라서 정답은 ①이다.
