## 문제
$\log x$의 정수부분이 $3$이고, $\log x$의 소수부분과 $\log \sqrt{x}$의 소수부분의 합이 $\frac{3}{4}$이다. 이때, $\log \sqrt{x}$의 소수부분은?

## 보기
① $\frac{1}{3}$   ② $\frac{5}{12}$   ③ $\frac{1}{2}$   ④ $\frac{7}{12}$   ⑤ $\frac{2}{3}$

## 풀이
$\log x$의 소수부분을 $\alpha$라 하면
$\log x = 3 + \alpha$ ($0 \le \alpha < 1$)
$\log \sqrt{x} = \frac{1}{2} \log x = \frac{3 + \alpha}{2} = 1 + \frac{1 + \alpha}{2}$
$0 \le \frac{1 + \alpha}{2} < 1$이므로 문제의 조건에서
$\alpha + \frac{1 + \alpha}{2} = \frac{3}{4}$
$\therefore \alpha = \frac{1}{6}$
따라서 $\log \sqrt{x}$의 소수부분은
$\frac{1 + \alpha}{2} = \frac{1 + \frac{1}{6}}{2} = \frac{7}{12}$

정답: ④
