## 문제
전체집합 $U = \{ x | x \text{ 는 } 7 \text{ 이하의 자연수 } \}$ 의 두 부분집합 $A = \{ 1, 3, 5, 7 \}$, $B = \{ 4, 5, 6, 7 \}$ 에 대하여 집합 $(A^{C} \cup B )^{C} \cup B$ 의 모든 원소의 합은?

## 보기
① $24$   ② $25$   ③ $26$   ④ $27$   ⑤ $28$

## 풀이
$(A^{C} \cup B )^{C} = A \cap B^{C} = A - B = \{ 1, 3 \}$ 이므로
$(A^{C} \cup B )^{C} \cup B = \{ 1, 3, 4, 5, 6, 7 \}$
따라서 구하는 모든 원소의 합은 $1 + 3 + 4 + 5 + 6 + 7 = 26$ 이다.

정답: ③
