## 문제
이차방정식 $x^{2}-x+2=0$의 두 근이 $\alpha$, $\beta$이고, 삼차방정식 $x^{3}-3x+5=0$의 세 근이 $p$, $q$, $r$일 때, $(p\alpha+q\beta)^{2}+(p\beta-q\alpha)^{2}+r^{2}\alpha^{2}+r^{2}\beta^{2}$의 값을 구하시오.

## 보기


## 풀이
근과 계수의 관계에 의하여
$\alpha + \beta = 1$, $\alpha\beta = 2$
$p + q + r = 0$, $pq + qr + rp = -3$, $pqr = -5$

주어진 식을 전개하면
$(p\alpha + q\beta)^{2} + (p\beta - q\alpha)^{2} + r^{2}\alpha^{2} + r^{2}\beta^{2}$
$= p^{2}\alpha^{2} + 2pq\alpha\beta + q^{2}\beta^{2} + p^{2}\beta^{2} - 2pq\alpha\beta + q^{2}\alpha^{2} + r^{2}\alpha^{2} + r^{2}\beta^{2}$
$= (p^{2} + q^{2} + r^{2})\alpha^{2} + (p^{2} + q^{2} + r^{2})\beta^{2}$
$= (p^{2} + q^{2} + r^{2})(\alpha^{2} + \beta^{2})$

이때
$p^{2} + q^{2} + r^{2} = (p + q + r)^{2} - 2(pq + qr + rp) = 0^{2} - 2(-3) = 6$
$\alpha^{2} + \beta^{2} = (\alpha + \beta)^{2} - 2\alpha\beta = 1^{2} - 2(2) = -3$

따라서 구하는 값은
$6 \times (-3) = -18$
