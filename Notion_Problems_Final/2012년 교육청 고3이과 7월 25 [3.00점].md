## 문제
$f(x) = 3x^2 + x + \int_{0}^{2} f(t) dt$일 때, $f(2)$의 값을 구하시오.

## 보기

## 풀이
$\int_{0}^{2} f(t) dt = a$라 하면,
$f(x) = 3x^2 + x + a$이다.
$\int_{0}^{2} (3t^2 + t + a) dt = a$
$[t^3 + \frac{1}{2}t^2 + at]_{0}^{2} = a$
$8 + 2 + 2a = a$
$\therefore a = -10$
$f(x) = 3x^2 + x - 10$
$\therefore f(2) = 3 \times 2^2 + 2 - 10 = 4$

정답: $4$
