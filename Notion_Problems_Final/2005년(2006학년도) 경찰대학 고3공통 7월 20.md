## 문제
각 자리의 숫자가 $1$, $2$, $3$ 만으로 이루어지고 $3$ 의 배수인 $4$ 자리 자연수의 개수는?

## 보기
① $23$   ② $25$   ③ $27$   ④ $29$   ⑤ $31$

## 풀이
$1$, $2$, $3$ 으로 이루어진 $4$ 자리 자연수가 $3$ 의 배수가 되려면 각 자리수의 합이 $3$ 의 배수가 되어야 한다. 즉, 
$(1, 1, 1, 3)$, $(1, 1, 2, 2)$, $(1, 2, 3, 3)$, $(2, 2, 2, 3)$, $(3, 3, 3, 3)$ 이 가능하다. 각각의 경우에 대해 나열하는 방법을 고려하면

$1)$ $1, 1, 1, 3$ 일 때 : $\frac{4!}{3!} = 4$ (가지)
$2)$ $1, 1, 2, 2$ 일 때 : $\frac{4!}{2!2!} = 6$ (가지)
$3)$ $1, 2, 3, 3$ 일 때 : $\frac{4!}{2!} = 12$ (가지)
$4)$ $2, 2, 2, 3$ 일 때 : $\frac{4!}{3!} = 4$ (가지)
$5)$ $3, 3, 3, 3$ 일 때 : $\frac{4!}{4!} = 1$ (가지)

따라서, 가능한 모든 경우의 수는 $4+6+12+4+1=27$ (가지)이다.

정답: ③
