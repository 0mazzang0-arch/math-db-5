## 문제
함수 
$$ f ( x ) = \begin{cases} \frac{a \sqrt{x + 2} + b}{x - 2} & ( x \neq 2 ) \\ 2 & ( x = 2 ) \end{cases} $$
가 $x = 2$에서 연속일 때, 두 상수 $a$, $b$에 대하여 $2a - b$의 값을 구하시오.

## 보기

## 풀이
$\lim_{x \to 2} ( a \sqrt{x + 2} + b ) = 0$ 이므로 
$2a + b = 0$ 

$\lim_{x \to 2} \frac{a \sqrt{x + 2} - 2a}{x - 2} = \lim_{x \to 2} \frac{a}{\sqrt{x + 2} + 2} = 2$ 

따라서 $a = 8$, $b = -16$ 이므로 
$2a - b = 32$

정답: $32$
