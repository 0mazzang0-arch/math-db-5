## 문제
모든 실수 $x$ 에 대하여 이차부등식 $x^{2} - 2 ( k - 2 ) x - k^{2} + 5k - 3 \ge 0$ 이 성립하도록 하는 모든 정수 $k$ 의 값의 합은?

## 보기
① $2$   ② $4$   ③ $6$   ④ $8$   ⑤ $10$

## 풀이
[정답] ③

모든 실수 $x$ 에 대하여 이차부등식 $x^{2} - 2 ( k - 2 ) x - k^{2} + 5k - 3 \ge 0$ 이 성립하므로 방정식 $x^{2} - 2 ( k - 2 ) x - k^{2} + 5k - 3 = 0$ 의 판별식을 $D$ 라 하면
$\frac{D}{4} = ( k-2 )^{2} - ( -k^{2} + 5k - 3 ) \le 0$
$2k^{2} - 9k + 7 \le 0$
$( k-1 ) ( 2k-7 ) \le 0$
$\therefore 1 \le k \le \frac{7}{2}$
따라서 정수 $k$ 는 $1$, $2$, $3$ 이므로 구하는 합은 $6$ 이다.

[다른 풀이]
부등식 $x^{2} - 2 ( k - 2 ) x - k^{2} + 5k - 3 \ge 0$ 에서
$x^{2} - 2 ( k - 2 ) x + ( k - 2 )^{2} - ( k - 2 )^{2} - k^{2} + 5k - 3 \ge 0$
$\therefore \{ x - ( k - 2 ) \}^{2} - 2k^{2} + 9k - 7 \ge 0$
이 부등식이 모든 실수 $x$ 에 대하여 성립하므로
$-2k^{2} + 9k - 7 \ge 0$
$2k^{2} - 9k + 7 \le 0$
$( k-1 ) ( 2k-7 ) \le 0$
$\therefore 1 \le k \le \frac{7}{2}$
따라서 정수 $k$ 는 $1$, $2$, $3$ 이므로 구하는 합은 $6$ 이다.
