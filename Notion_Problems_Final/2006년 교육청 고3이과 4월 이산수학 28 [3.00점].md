## 문제
부등식 $\cos 2 x - 3 \sin x + 1 \leq 0$ 을 만족시키는 $x$ 의 범위는? (단, $0 \leq x < 2 \pi$)

## 보기
① $\frac{\pi}{6} \leq x \leq \frac{5}{6} \pi$   ② $\frac{\pi}{3} \leq x \leq \frac{7}{6} \pi$   ③ $\frac{5}{6} \pi \leq x \leq \frac{4}{3} \pi$   ④ $\frac{4}{3} \pi \leq x \leq \frac{5}{3} \pi$   ⑤ $\frac{7}{6} \pi \leq x \leq \frac{11}{6} \pi$

## 풀이
부등식 $\cos 2 x - 3 \sin x + 1 \leq 0$ 에서 $\cos 2 x = 1 - 2 \sin^{2} x$ 를 대입하면
$( 1 - 2 \sin^{2} x ) - 3 \sin x + 1 \leq 0$
$- 2 \sin^{2} x - 3 \sin x + 2 \leq 0$
$2 \sin^{2} x + 3 \sin x - 2 \geq 0$
$( 2 \sin x - 1 ) ( \sin x + 2 ) \geq 0$

모든 $x$ 에 대하여 $- 1 \leq \sin x \leq 1$ 이므로
$\sin x + 2 > 0$
따라서 $2 \sin x - 1 \geq 0$ 에서
$\sin x \geq \frac{1}{2}$

$0 \leq x < 2 \pi$ 일 때, $\sin x \geq \frac{1}{2}$ 을 만족하는 $x$ 의 범위는
$\therefore \frac{\pi}{6} \leq x \leq \frac{5}{6} \pi$

정답은 ①이다.
