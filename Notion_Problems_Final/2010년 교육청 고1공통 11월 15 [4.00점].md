## 문제
방정식 $x^{3}-1=0$의 한 허근을 $\omega$라 할 때, <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $\bar{\omega}$는 $\omega$의 켤레복소수이다.)

## 보기
ㄱ. $\omega^{10} = \omega$
ㄴ. $\frac{\omega^{2}}{1+\omega} + \frac{\bar{\omega}}{1+\bar{\omega}^{2}} = -2$
ㄷ. $\omega^{4n} + (\omega+1)^{4n} + 1 = 0$을 만족시키는 $30$ 이하의 양의 정수 $n$의 개수는 $20$이다.

① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
방정식 $x^{3}-1=0$에서 $(x-1)(x^{2}+x+1)=0$이므로 $\omega$는 $x^{2}+x+1=0$의 한 허근이고, 방정식의 계수가 실수이므로 $\bar{\omega}$도 $x^{2}+x+1=0$의 근이다.
따라서 $\omega^{2}+\omega+1=0$, $(\bar{\omega})^{2}+\bar{\omega}+1=0$, $\omega^{3}=1$, $(\bar{\omega})^{3}=1$이다.

ㄱ. $\omega^{10} = (\omega^{3})^{3} \cdot \omega = 1^{3} \cdot \omega = \omega$ (참)

ㄴ. $\omega^{2}+\omega+1=0$, $(\bar{\omega})^{2}+\bar{\omega}+1=0$에서 $1+\omega = -\omega^{2}$, $1+(\bar{\omega})^{2} = -\bar{\omega}$이다.
$\frac{\omega^{2}}{1+\omega} + \frac{\bar{\omega}}{1+(\bar{\omega})^{2}} = \frac{\omega^{2}}{-\omega^{2}} + \frac{\bar{\omega}}{-\bar{\omega}} = -1 + (-1) = -2$ (참)

ㄷ. $\omega^{4n} + (\omega+1)^{4n} + 1$에서 $\omega+1 = -\omega^{2}$이므로
$\omega^{4n} + (-\omega^{2})^{4n} + 1 = \omega^{4n} + \omega^{8n} + 1 = (\omega^{3} \cdot \omega)^{n} + (\omega^{3})^{2n} \cdot \omega^{2n} + 1 = \omega^{n} + \omega^{2n} + 1$

$n=3k$ ($k=1, 2, 3, \dots$)일 때,
$\omega^{2n} + \omega^{n} + 1 = \omega^{6k} + \omega^{3k} + 1 = 1+1+1 = 3$

$n=3k+1$ ($k=0, 1, 2, \dots$)일 때,
$\omega^{6k+2} + \omega^{3k+1} + 1 = \omega^{2} + \omega + 1 = 0$

$n=3k+2$ ($k=0, 1, 2, \dots$)일 때,
$\omega^{6k+4} + \omega^{3k+2} + 1 = \omega^{4} + \omega^{2} + 1 = \omega + \omega^{2} + 1 = 0$

따라서 $\omega^{4n} + (\omega+1)^{4n} + 1 = 0$을 만족시키는 $n$은 $3$의 배수가 아닌 양의 정수이다.
$30$ 이하의 양의 정수 중 $3$의 배수는 $10$개이므로, 구하는 $n$의 개수는 $30-10=20$이다. (참)

이상에서 ㄱ, ㄴ, ㄷ 모두 옳다.
