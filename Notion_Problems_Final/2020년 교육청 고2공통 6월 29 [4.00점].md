## 문제
자연수 $k$ ($k \le 39$)에 대하여 함수 
$f(x) = 2 \log_{1/2}(x - 7 + k) + 2$
의 그래프와 원 $x^2 + y^2 = 64$가 만나는 서로 다른 두 점의 $x$좌표를 $a, b$라 하자. 다음 조건을 만족시키는 $k$의 최댓값과 최솟값을 각각 $M, m$이라 할 때, $M + m$의 값을 구하시오.

(가) $ab < 0$
(나) $f(a)f(b) < 0$

## 풀이
$x$의 값이 증가할 때 $f(x)$의 값은 감소하므로 조건 (가)와 (나)를 만족하기 위해서는 두 교점이 제 $2$사분면과 제 $4$사분면에 각각 한 개씩 존재해야 한다.

> [!example] 📸 (그림/그래프)

따라서 $-8 < f(0) < 8, f(-8) > 0, f(8) < 0$이다.

(ⅰ) $-8 < f(0) < 8$일 때,
$f(0) = 2 \log_{1/2}(-7 + k) + 2$이므로
$-8 < 2 \log_{1/2}(-7 + k) + 2 < 8$에서 
$\frac{57}{8} < k < 39$

(ⅱ) $f(-8) > 0$일 때,
$f(-8) = 2 \log_{1/2}(-15 + k) + 2$이므로 
$2 \log_{1/2}(-15 + k) + 2 > 0$에서 
$k < 17$

(ⅲ) $f(8) < 0$일 때,
$f(8) = 2 \log_{1/2}(1 + k) + 2$이므로 
$2 \log_{1/2}(1 + k) + 2 < 0$에서 
$k > 1$

(ⅰ), (ⅱ), (ⅲ)에 의해서 $\frac{57}{8} < k < 17$이다. 
따라서 $k$의 최댓값 $M = 16$, 최솟값 $m = 8$이므로 
$M + m = 24$
