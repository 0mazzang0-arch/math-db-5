## 문제
부등식 $n < \int_{0}^{2} \sqrt{27 + 2 \sin x} dx < n + 1$을 만족하는 양의 정수 $n$을 구하시오.

## 보기

## 풀이
구간 $0 \le x \le 2$에서 $0 \le \sin x \le 1$이므로 
$$\int_{0}^{2} \sqrt{27} dx < \int_{0}^{2} \sqrt{27 + 2 \sin x} dx < \int_{0}^{2} \sqrt{29} dx$$ 
$$\left[ \sqrt{27} x \right]_0^{2} < \int_{0}^{2} \sqrt{27 + 2 \sin x} dx < \left[ \sqrt{29} x \right]_0^{2}$$ 
$$\sqrt{108} < \int_{0}^{2} \sqrt{27 + 2 \sin x} dx < \sqrt{116}$$ 
$$10 < \sqrt{108} < \int_{0}^{2} \sqrt{27 + 2 \sin x} dx < \sqrt{116} < 11$$ 
따라서 $n = 10$이다.
