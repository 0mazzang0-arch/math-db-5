## 문제
함수 $f (x) = x^{3} + a x^{2} + b x + 4$가 다음 조건을 만족시키도록 하는 두 정수 $a$, $b$에 대하여 $f (1)$의 최댓값을 구하시오.
모든 실수 $\alpha$에 대하여 $\lim_{x \to \alpha} \frac{f (2 x + 1)}{f (x)}$의 값이 존재한다.

## 풀이
모든 실수 $\alpha$에 대하여 $\lim_{x \to \alpha} \frac{f (2 x + 1)}{f (x)}$의 값이 존재하려면 $\lim_{x \to \alpha} f (x) = 0$일 때 $\lim_{x \to \alpha} f (2 x + 1) = 0$이어야 한다.
$\cdots \cdots$ ㉠
즉, $f (\alpha) = 0$이면 $f (2 \alpha + 1) = 0$
이때 $\alpha \neq 2 \alpha + 1$이면 ㉠에 의하여
$f (\alpha) = 0$, $f (2 \alpha + 1) = 0$, $f (4 \alpha + 3) = 0$, $f (8 \alpha + 7) = 0, \cdots$
이 되어 방정식 $f (x) = 0$이 무수히 많은 실근을 가지게 되므로 모순이 된다.
따라서 $\alpha = 2 \alpha + 1$에서 $\alpha = -1$이므로 방정식 $f (x) = 0$의 실근은 $-1$뿐이다.

$f (-1) = 0$에서 $-1 + a - b + 4 = 0$, $b = a + 3$
$f (x) = (x + 1) \{ x^{2} + (a - 1) x + 4 \}$
이고, 함수 $f (x)$의 상수항이 $4$이므로 방정식 $f (x) = 0$은 $x = -1$을 삼중근으로 가질 수 없다.
따라서 방정식 $f (x) = 0$, 즉 $(x + 1) \{ x^{2} + (a - 1) x + 4 \} = 0$은 $x = -1$의 실근 하나와 두 허근을 가져야 한다.
이차방정식 $x^{2} + (a - 1) x + 4 = 0$의 판별식을 $D$라 하면
$D = (a - 1)^{2} - 4^{2} < 0$
$(a - 5) (a + 3) < 0$
$\therefore -3 < a < 5$

$f (1) = 2 a + 8$에서 $a$가 정수이므로 $f (1)$의 최댓값은 $a = 4$일 때 $16$이다.

정답: $16$
