## 문제
좌표축 위에 있지 않은 두 점 $A ( p, q ), B ( r, s )$ 가 원 $O : x^{2} + y^{2} = 1$ 위에 있고 $\overline{AB} = \sqrt{2}$ 이면, 두 점 $C ( p, r ), D ( q, s )$ 도 원 $O$ 위에 있고 $\overline{CD} = ( \text{다} )$ 임을 증명하는 과정이다.

---
**[증명]**
두 점 $A ( p, q ), B ( r, s )$ 가 원 $x^{2} + y^{2} = 1$ 위의 점이므로
$p^{2} + q^{2} = 1 \cdots \cdots ㉠$
$r^{2} + s^{2} = 1 \cdots \cdots ㉡$

$\overline{AB}^{2} = ( r - p )^{2} + ( s - q )^{2} = 2 \times ( \text{가} ) = 2$ 이므로
$( \text{가} ) = 1 \cdots \cdots ㉢$

$㉢$ 을 $q$ 에 대하여 정리하여 $㉠$ 에 대입하면
$p^{2} + q^{2} = p^{2} \left( 1 + \frac{r^{2}}{s^{2}} \right) = 1$ 이므로
$( \text{나} )$ 이 성립한다.

따라서 $p^{2} + r^{2} = q^{2} + s^{2} = 1$ 이므로 두 점 $C ( p, r ), D ( q, s )$ 는 원 $x^{2} + y^{2} = 1$ 위에 있다.
또한 $\overline{CD}^{2} = ( q - p )^{2} + ( s - r )^{2} = 2 - 2 ( pq + rs )$ 이다.

$㉢$ 과 $( \text{나} )$ 에 의하여 $pq + rs$ 의 값을 구할 수 있다.
$\therefore \overline{CD} = ( \text{다} )$
---

위의 증명 과정에서 $( \text{가} ), ( \text{나} ), ( \text{다} )$ 에 알맞은 것은?

## 보기
$$ \begin{array}{|c|c|c|c|} \hline & (\text{가}) & (\text{나}) & (\text{다}) \\ \hline ① & 1-pr+qs & p^{2} = r^{2} & 1 \\ \hline ② & 1-pr-qs & p^{2} = s^{2} & 1 \\ \hline ③ & 1-pr+qs & p^{2} = s^{2} & \sqrt{2} \\ \hline ④ & 1-pr-qs & p^{2} = s^{2} & \sqrt{2} \\ \hline ⑤ & 1-pr+qs & p^{2} = r^{2} & \sqrt{2} \\ \hline \end{array} $$

① $ ( \text{가} ): 1-pr+qs, ( \text{나} ): p^{2} = r^{2}, ( \text{다} ): 1 $
② $ ( \text{가} ): 1-pr-qs, ( \text{나} ): p^{2} = s^{2}, ( \text{다} ): 1 $
③ $ ( \text{가} ): 1-pr+qs, ( \text{나} ): p^{2} = s^{2}, ( \text{다} ): \sqrt{2} $
④ $ ( \text{가} ): 1-pr-qs, ( \text{나} ): p^{2} = s^{2}, ( \text{다} ): \sqrt{2} $
⑤ $ ( \text{가} ): 1-pr+qs, ( \text{나} ): p^{2} = r^{2}, ( \text{다} ): \sqrt{2} $

## 풀이
정답: ④

두 점 $A ( p, q ), B ( r, s )$ 가 원 $x^{2} + y^{2} = 1$ 위의 점이므로
$p^{2} + q^{2} = 1 \cdots \cdots ㉠$, $r^{2} + s^{2} = 1 \cdots \cdots ㉡$

$\overline{AB}^{2} = ( r - p )^{2} + ( s - q )^{2} = r^{2} - 2pr + p^{2} + s^{2} - 2sq + q^{2}$
$= ( p^{2} + q^{2} ) + ( r^{2} + s^{2} ) - 2 ( pr + qs ) = 1 + 1 - 2 ( pr + qs ) = 2 ( 1 - pr - qs )$
$\overline{AB} = \sqrt{2}$ 에서 $\overline{AB}^{2} = 2$ 이므로
$2 ( 1 - pr - qs ) = 2 \Rightarrow 1 - pr - qs = 1 \cdots \cdots ㉢$
따라서 $( \text{가} ) = 1 - pr - qs$ 이고, $pr + qs = 0$ 이다.

$㉢$ 에서 $qs = -pr$ 이므로 $q = - \frac{pr}{s}$ 이다. 이를 $㉠$ 에 대입하면
$p^{2} + \left( - \frac{pr}{s} \right)^{2} = p^{2} + \frac{p^{2} r^{2}}{s^{2}} = p^{2} \left( 1 + \frac{r^{2}}{s^{2}} \right) = p^{2} \left( \frac{s^{2} + r^{2}}{s^{2}} \right) = \frac{p^{2}}{s^{2}} = 1$
따라서 $p^{2} = s^{2}$ 이므로 $( \text{나} ) : p^{2} = s^{2}$ 이다.

$p^{2} = s^{2}$ 이면 $q^{2} = 1 - p^{2} = 1 - s^{2} = r^{2}$ 이므로 두 점 $C ( p, r ), D ( q, s )$ 는 원 $x^{2} + y^{2} = 1$ 위에 있다.

또한 $p^{2} = s^{2}$ 에서 $p = s$ 또는 $p = -s$ 이다.
$pr + qs = 0$ 에 대입하면
1) $p = s$ 일 때, $sr + qs = s ( r + q ) = 0$ 이므로 $q = -r$ 이고, $pq + rs = s ( -r ) + rs = 0$ 이다.
2) $p = -s$ 일 때, $-sr + qs = s ( q - r ) = 0$ 이므로 $q = r$ 이고, $pq + rs = ( -s ) r + rs = 0$ 이다.
항상 $pq + rs = 0$ 이므로
$\overline{CD}^{2} = ( q - p )^{2} + ( s - r )^{2} = q^{2} - 2pq + p^{2} + s^{2} - 2rs + r^{2}$
$= ( p^{2} + q^{2} ) + ( r^{2} + s^{2} ) - 2 ( pq + rs ) = 1 + 1 - 2 \times 0 = 2$
$\therefore \overline{CD} = \sqrt{2}$
따라서 $( \text{다} ) = \sqrt{2}$ 이다.
