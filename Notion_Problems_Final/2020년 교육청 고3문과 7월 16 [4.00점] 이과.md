## 문제
한 개의 주사위를 세 번 던질 때 나오는 눈의 수를 차례로 $a$, $b$, $c$라 하자. $a+b+c$의 값을 확률변수 $X$라 할 때, 다음은 확률변수 $X$의 평균 $E(X)$를 구하는 과정이다.

$$ \begin{array}{|l|} \hline 3 \le a+b+c \le 18 \text{ 이므로 확률변수 } X \text{ 가 가질 수 있는 값은 } 3, 4, 5, \dots, 18 \text{ 이다.} \\ a, b, c \text{ 가 각각 } 6 \text{ 이하의 자연수이므로 } 7-a, 7-b, 7-c \text{ 는 각각 } 6 \text{ 이하의 자연수이다.} \\ \\ 3 \le k \le 18 \text{ 인 자연수 } k \text{ 에 대하여 } a+b+c=k \text{ 일 확률 } P(X=k) \text{ 와 } \\ (7-a)+(7-b)+(7-c)=k \text{ 일 확률 } P(X=3 \times (가) - k) \text{ 는 서로 같다.} \\ \\ \text{그러므로 확률변수 } X \text{ 의 평균 } E(X) \text{ 는 } \\ E(X) = \sum_{k=3}^{18} k \times P(X=k) \\ = 3 \times P(X=3) + 4 \times P(X=4) + 5 \times P(X=5) \\ \quad + \dots + 17 \times P(X=17) + 18 \times P(X=18) \\ = (나) \times \sum_{k=3}^{10} P(X=k) \\ \\ \text{이때, 확률질량함수의 성질에 의하여 } \sum_{k=3}^{18} P(X=k) = 1 \text{ 이므로 } \\ \sum_{k=3}^{10} P(X=k) = (다) \\ \\ \text{따라서 } E(X) = (나) \times (다) \\ \hline \end{array} $$

위의 $(가)$, $(나)$, $(다)$에 알맞은 수를 각각 $p$, $q$, $r$라 할 때, $\frac{p+q}{r}$의 값은?

## 보기
① $49$   ② $\frac{105}{2}$   ③ $56$   ④ $\frac{119}{2}$   ⑤ $63$

## 풀이
$1 \le a \le 6$ 이면 $1 \le 7-a \le 6$ 이 성립한다.
$a, b, c$ 가 각각 $6$ 이하의 자연수이므로 $7-a, 7-b, 7-c$ 도 각각 $6$ 이하의 자연수이다.
$3 \le k \le 18$ 인 자연수 $k$ 에 대하여 $a+b+c=k$ 를 만족시키는 $6$ 이하의 자연수 순서쌍 $(a, b, c)$ 의 개수는 $(7-a)+(7-b)+(7-c)=k$ 즉, $a+b+c = 3 \times 7 - k$ 를 만족시키는 $6$ 이하의 자연수 순서쌍 $(a, b, c)$ 의 개수와 같다.

따라서 $P(X=k) = P(X=21-k)$ 이 성립하며, 이를 $P(X=3 \times (가) - k)$ 와 비교하면 $(가) = 7$ 임을 알 수 있다. 즉, $p = 7$ 이다.

확률변수 $X$ 의 평균 $E(X)$ 는 다음과 같이 계산된다.
$E(X) = \sum_{k=3}^{18} k \times P(X=k)$
$= 3 \times P(X=3) + 4 \times P(X=4) + \dots + 18 \times P(X=18)$
위에서 구한 대칭성 $P(X=k) = P(X=21-k)$ 에 의해
$E(X) = (3+18) \times P(X=3) + (4+17) \times P(X=4) + \dots + (10+11) \times P(X=10)$
$= 21 \times \sum_{k=3}^{10} P(X=k)$
따라서 $(나) = 21$ 이며, $q = 21$ 이다.

확률질량함수의 성질에 의해 $\sum_{k=3}^{18} P(X=k) = 1$ 이고, 대칭성에 의해 $\sum_{k=3}^{10} P(X=k) = \sum_{k=11}^{18} P(X=k)$ 이므로
$\sum_{k=3}^{10} P(X=k) = \frac{1}{2}$ 이다.
따라서 $(다) = \frac{1}{2}$ 이며, $r = \frac{1}{2}$ 이다.

최종적으로 구하는 값은
$\frac{p+q}{r} = \frac{7+21}{\frac{1}{2}} = 28 \times 2 = 56$ 이다.

정답은 ③이다.
