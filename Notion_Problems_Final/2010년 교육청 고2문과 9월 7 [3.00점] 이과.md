## 문제
세 수 $A = \sqrt[3]{\frac{1}{4}}$, $B = \sqrt[4]{\frac{1}{6}}$, $C = \sqrt[3]{\sqrt{\frac{1}{15}}}$의 대소 관계를 바르게 나타낸 것은?

## 보기
① $A < B < C$   ② $A < C < B$   ③ $B < A < C$   ④ $B < C < A$   ⑤ $C < A < B$

## 풀이
$A = \sqrt[3]{\frac{1}{4}} = \left( \frac{1}{4} \right)^{\frac{1}{3}} = \left( \frac{1}{4} \right)^{\frac{4}{12}} = \left( \frac{1}{256} \right)^{\frac{1}{12}}$
$B = \sqrt[4]{\frac{1}{6}} = \left( \frac{1}{6} \right)^{\frac{1}{4}} = \left( \frac{1}{6} \right)^{\frac{3}{12}} = \left( \frac{1}{216} \right)^{\frac{1}{12}}$
$C = \sqrt[3]{\sqrt{\frac{1}{15}}} = \left( \frac{1}{15} \right)^{\frac{1}{6}} = \left( \frac{1}{15} \right)^{\frac{2}{12}} = \left( \frac{1}{225} \right)^{\frac{1}{12}}$

$\frac{1}{256} < \frac{1}{225} < \frac{1}{216}$ 이므로
$\therefore A < C < B$

정답 ②
