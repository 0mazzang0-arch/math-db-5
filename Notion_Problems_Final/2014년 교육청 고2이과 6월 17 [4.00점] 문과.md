## 문제
두 집합
$A = \{ x \mid ( \frac{1}{2} )^{2x^{2}} - ( \frac{1}{2} )^{13x-6} \ge 0, x \text{ 는 정수} \}$,
$B = \{ x \mid \log_{3} ( 5x+6 ) - \frac{2}{\log_{x} 3} > 0, x \text{ 는 정수} \}$
에 대하여 집합 $A-B$의 모든 원소의 합은?

## 보기
① $3$   ② $5$   ③ $7$   ④ $9$   ⑤ $11$

## 풀이
$( \frac{1}{2} )^{2x^{2}} - ( \frac{1}{2} )^{13x-6} \ge 0$에서
$( \frac{1}{2} )^{2x^{2}} \ge ( \frac{1}{2} )^{13x-6}$
$2x^{2} - 13x + 6 \le 0$이고
$( 2x - 1 )( x - 6 ) \le 0$
$\frac{1}{2} \le x \le 6$이므로
$A = \{ 1, 2, 3, 4, 5, 6 \}$이다.

$\log_{3} ( 5x+6 ) - \frac{2}{\log_{x} 3} > 0$에서
$\log_{3} ( 5x+6 ) - \log_{3} x^{2} > 0$
$\iff \log_{3} ( 5x+6 ) > \log_{3} x^{2}$
$x^{2} - 5x - 6 < 0$
$( x - 6 )( x + 1 ) < 0$이므로
$-1 < x < 6$이다.

로그의 정의에 의해 밑은 $1$이 아닌 양수이고 진수는 양수여야 하므로
$x > 0, x \neq 1$
따라서 $0 < x < 6, x \neq 1$이므로
$B = \{ 2, 3, 4, 5 \}$이다.

그러므로
$A - B = \{ 1, 6 \}$이다.
따라서 $A - B$의 모든 원소의 합은 $7$이다.
