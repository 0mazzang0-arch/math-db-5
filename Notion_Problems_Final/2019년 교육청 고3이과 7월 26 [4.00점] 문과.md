## 문제
실수 전체의 집합에서 미분가능한 함수 $f(x)$가 다음 조건을 만족시킨다.

(가) $f(1) = 0$
(나) $0$이 아닌 모든 실수 $x$에 대하여 $\frac{xf'(x) - f(x)}{x^2} = xe^x$이다.

$f(3) \times f(-3)$의 값을 구하시오.

## 보기


## 풀이
제시된 조건 (나)에서
$\frac{xf'(x) - f(x)}{x^2} = \left( \frac{f(x)}{x} \right)' = xe^x$
양변을 $x$에 대하여 적분하면
$\frac{f(x)}{x} = \int xe^x dx = (x - 1)e^x + C$ ($C$는 적분상수)
조건 (가)에서 $f(1) = 0$이므로
$\frac{f(1)}{1} = (1 - 1)e^1 + C = 0$
따라서 $C = 0$이다.
$\frac{f(x)}{x} = (x - 1)e^x$ 에서
$f(x) = x(x - 1)e^x$
$f(3) = 3(3 - 1)e^3 = 6e^3$
$f(-3) = -3(-3 - 1)e^{-3} = 12e^{-3}$
따라서
$f(3) \times f(-3) = 6e^3 \times 12e^{-3} = 72$

정답: $72$
