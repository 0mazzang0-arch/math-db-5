## 문제
수열 $\{ a_{n} \}$은 $a_{1} = 1$이고, 모든 자연수 $n$에 대하여 
$$a_{n+1} + a_{n} = n + 3$$
을 만족시킨다. $a_{4}$의 값은?

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
$a_{n+1} + a_{n} = n + 3$에서 
$$a_{n+1} = n + 3 - a_{n}$$ 

$n = 1$일 때, $a_{1} = 1$이므로 
$$a_{2} = 1 + 3 - a_{1} = 1 + 3 - 1 = 3$$ 

$n = 2$일 때, 
$$a_{3} = 2 + 3 - a_{2} = 2 + 3 - 3 = 2$$ 

$n = 3$일 때, 
$$a_{4} = 3 + 3 - a_{3} = 3 + 3 - 2 = 4$$ 

따라서 정답은 ④이다.
