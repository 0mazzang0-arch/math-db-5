## 문제
서로 다른 두 점에서 만나는 두 곡선 $C_{1} : y = x^{2} - 2x + 2$, $C_{2} : y = -x^{2} + ax + b$ 의 한 교점을 $P$ 라 하고, 점 $P$ 에서 두 곡선 $C_{1}$, $C_{2}$ 에 접하는 직선을 각각 $l$, $m$ 이라 하자. 두 접선 $l$, $m$ 이 서로 수직일 때, 곡선 $C_{2}$ 는 두 실수 $a$, $b$ 의 값에 관계없이 일정한 점 $Q$ 를 지난다. 다음은 점 $Q$ 의 좌표를 구하는 과정이다.

$f(x) = x^{2} - 2x + 2$, $g(x) = -x^{2} + ax + b$ 라 하고, 두 곡선 $C_{1}$, $C_{2}$ 의 한 교점 $P$ 의 $x$ 좌표를 $t$ 라 하자.
두 접선 $l$, $m$ 이 서로 수직이므로 $f'(t)g'(t) = -1$ 에서
$4t^{2} - 2(a + 2)t + (\text{가}) = 0 \cdots ㉠$
$f(t) = g(t)$ 에서
$2t^{2} - (a + 2)t + 2 - b = 0 \cdots ㉡$
$㉠$, $㉡$ 에서 $b = (\text{나}) - a$ 를 $y = -x^{2} + ax + b$ 에 대입하고 $a$ 에 관하여 정리하면,
$a(x - 1) - x^{2} - y + (\text{나}) = 0 \cdots ㉢$
$㉢$ 에서 $x - 1 = 0$, $-x^{2} - y + (\text{나}) = 0$ 을 만족시키는 $x$ 와 $y$ 의 값을 구하면 점 $Q$ 의 좌표는 $(1, (\text{다}))$ 이다.
위의 $(\text{가})$ 에 알맞은 식을 $h(a)$ 라 하고, $(\text{나})$ 와 $(\text{다})$ 에 알맞은 수를 각각 $\alpha$, $\beta$ 라 할 때, $h(\alpha) \times h(\beta)$ 의 값은?

## 보기
① $4$   ② $8$   ③ $12$   ④ $16$   ⑤ $20$

## 풀이
$f(x) = x^{2} - 2x + 2$ 에서 $f'(x) = 2x - 2$
$g(x) = -x^{2} + ax + b$ 에서 $g'(x) = -2x + a$
두 접선 $l$, $m$ 이 서로 수직이므로 $f'(t)g'(t) = -1$ 에서
$(2t - 2)(-2t + a) = -1$
$-4t^{2} + 2at + 4t - 2a = -1$
$4t^{2} - 2(a + 2)t + 2a - 1 = 0 \cdots ㉠$
따라서 $h(a) = 2a - 1$ 이다.

또한 교점 $P$ 에서 $f(t) = g(t)$ 이므로
$t^{2} - 2t + 2 = -t^{2} + at + b$
$2t^{2} - (a + 2)t + 2 - b = 0 \cdots ㉡$
$㉠$ 에서 $2t^{2} - (a + 2)t + \frac{2a - 1}{2} = 0$ 이므로 $㉡$ 과 비교하면
$2 - b = \frac{2a - 1}{2}$
$4 - 2b = 2a - 1$
$2b = 5 - 2a$
$b = \frac{5}{2} - a$
따라서 $(\text{나})$ 에 알맞은 수 $\alpha = \frac{5}{2}$ 이다.

이를 $y = -x^{2} + ax + b$ 에 대입하면
$y = -x^{2} + ax + \frac{5}{2} - a$
$a(x - 1) - x^{2} - y + \frac{5}{2} = 0 \cdots ㉢$
이 식이 $a$ 의 값에 관계없이 성립하려면
$x - 1 = 0 \implies x = 1$
$-x^{2} - y + \frac{5}{2} = 0 \implies -1 - y + \frac{5}{2} = 0 \implies y = \frac{3}{2}$
따라서 점 $Q$ 의 좌표는 $(1, \frac{3}{2})$ 이므로 $\beta = \frac{3}{2}$ 이다.

$h(\alpha) \times h(\beta) = h(\frac{5}{2}) \times h(\frac{3}{2})$
$= (2 \times \frac{5}{2} - 1) \times (2 \times \frac{3}{2} - 1)$
$= 4 \times 2 = 8$

정답: ②
