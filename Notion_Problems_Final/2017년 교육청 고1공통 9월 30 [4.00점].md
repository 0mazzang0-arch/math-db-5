## 문제
좌표평면 위에 두 원 
$C_{1} : x^{2} + ( y-4 )^{2} = 4$, 
$C_{2} : ( x-6 )^{2} + ( y-4+6 \sqrt{3} )^{2} = 16$
이 있다. 원 $C_{1}$ 위를 움직이는 점 $P ( x_{1}, y_{1} )$과 원 $C_{2}$ 위를 움직이는 점 $Q ( x_{2}, y_{2} )$가 다음 조건을 만족시킨다.

(가) $0 \le x_{1} \le 1$, $\frac{2x_{1} + x_{2}}{3} = 2$
(나) $y_{1} \le 4$, $y_{2} \ge 4-6 \sqrt{3}$

선분 $PQ$가 그리는 도형의 넓이가 $a-b \pi$일 때, $a+9b$의 값을 구하시오. (단, $a$, $b$는 유리수이다.)

## 풀이
조건 (가)에서 $x_{2} = -2x_{1} + 6$이므로 점 $Q ( -2x_{1} + 6, y_{2} )$라 하면 점 $Q$는 원 $C_{2}$ 위의 점이므로 
$( -2x_{1} )^{2} + ( y_{2}-4+6 \sqrt{3} )^{2} = 16$ $\cdots \cdots$ ㉠ 
점 $P ( x_{1}, y_{1} )$는 원 $C_{1}$ 위의 점이므로 
$x_{1}^{2} + ( y_{1}-4 )^{2} = 4$ $\cdots \cdots$ ㉡ 
㉠, ㉡을 연립하여 정리하면 
$( y_{1}-4 )^{2} = ( \frac{y_{2}}{2}-2+3 \sqrt{3} )^{2}$ 
조건 (나)에서 $y_{1}-4 \le 0$, $\frac{y_{2}}{2}-2+3 \sqrt{3} \ge 0$이므로 
$-y_{1}+4 = \frac{y_{2}}{2}-2+3 \sqrt{3}$ 
$2y_{1}+y_{2} = 12-6 \sqrt{3}$ 
$\therefore \frac{2y_{1}+y_{2}}{3} = 4-2 \sqrt{3}$ 
따라서 점 $( 2, 4-2 \sqrt{3} )$은 선분 $PQ$를 $1:2$로 내분하는 점이다. 

$x_{1}=0$일 때, $P ( 0, 2 )$, $Q ( 6, 8-6 \sqrt{3} )$ 
$x_{1}=1$일 때, $P ( 1, 4- \sqrt{3} )$, $Q ( 4, 4-4 \sqrt{3} )$ 
이때 직선 $PQ$의 방정식은 $y = - \sqrt{3} x+4$이므로 두 원 $C_{1}$, $C_{2}$의 중심을 지난다. 
$0 \le x_{1} \le 1$이므로 선분 $PQ$가 지나간 부분은 그림의 어두운 부분과 같다. 

> [!example] 📸 (그림/그래프)

원 $C_{1}$의 중심을 $O_{1} ( 0, 4 )$, 원 $C_{2}$의 중심을 $O_{2} ( 6, 4-6 \sqrt{3} )$, $A ( 0, 2 )$, $B ( 1, 4- \sqrt{3} )$, $C ( 2, 4-2 \sqrt{3} )$, $D ( 6, 8-6 \sqrt{3} )$이라 하자. 점 $B$에서 $y$축에 내린 수선의 발을 $H$라 하면 삼각형 $O_{1}HB$는 직각삼각형이고 $\overline{BH}=1$, $\overline{O_{1}H}= \sqrt{3}$ 
따라서 $\angle HO_{1}B = 30^{\circ}$ 

$S_{1}$의 넓이는 삼각형 $O_{1}AC$의 넓이에서 부채꼴 $O_{1}AB$의 넓이를 뺀 값과 같다. 삼각형 $O_{1}AC$의 넓이는 $\frac{1}{2} \times 2 \times 2 = 2$, 부채꼴 $O_{1}AB$의 넓이는 $\pi \times 2^{2} \times \frac{30^{\circ}}{360^{\circ}} = \frac{\pi}{3}$이므로 
$S_{1}$의 넓이는 $2 - \frac{\pi}{3}$이다. 
삼각형 $O_{1}AC$와 삼각형 $O_{2}DC$는 닮음이다. 
닮음비가 $1:2$이므로 $S_{1}$과 $S_{2}$의 넓이의 비는 $1:4$이다. 
따라서 $S_{2}$의 넓이는 $8 - \frac{4}{3} \pi$이다. 
따라서 선분 $PQ$가 그리는 도형의 넓이는 $10 - \frac{5}{3} \pi$이므로 
$a = 10$, $b = \frac{5}{3}$ 

$\therefore a+9b = 25$
