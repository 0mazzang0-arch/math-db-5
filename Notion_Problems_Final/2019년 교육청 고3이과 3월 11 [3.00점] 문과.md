## 문제
함수 $f(x) = \tan(\pi x^{2} + ax)$ 가 $x = \frac{1}{2}$ 에서 극솟값 $k$ 를 가질 때, $k$ 의 값은? (단, $a$ 는 상수이다.)

## 보기
① $-\sqrt{3}$   ② $-1$   ③ $-\frac{\sqrt{3}}{3}$   ④ $0$   ⑤ $\frac{\sqrt{3}}{3}$

## 풀이
$f(x) = \tan(\pi x^{2} + ax)$ 에서 
$$f'(x) = (2\pi x + a)\sec^{2}(\pi x^{2} + ax)$$
$x = \frac{1}{2}$ 에서 극솟값을 가지므로 
$$f'\left(\frac{1}{2}\right) = (\pi + a)\sec^{2}\left(\frac{\pi}{4} + \frac{a}{2}\right) = 0$$
$\sec^{2}\left(\frac{\pi}{4} + \frac{a}{2}\right) \neq 0$ 이므로 
$$a = -\pi$$
따라서 $f(x) = \tan(\pi x^{2} - \pi x)$ 에서 극솟값 $k$ 는 
$$k = f\left(\frac{1}{2}\right) = \tan\left(\frac{\pi}{4} - \frac{\pi}{2}\right) = \tan\left(-\frac{\pi}{4}\right) = -1$$

정답: ②
