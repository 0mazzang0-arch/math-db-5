## 문제
수열 $\{ a_{n} \}$ 이 모든 자연수 $n$ 에 대하여 
$$ a_{n} + a_{n+4} = 12 $$
를 만족시킬 때, $\sum_{n=1}^{16} a_{n}$ 의 값을 구하시오.

## 보기

## 풀이
수열 $\{ a_{n} \}$ 이 모든 자연수 $n$ 에 대하여 $a_{n} + a_{n+4} = 12$ 이므로 

$$ \sum_{n=1}^{16} a_{n} = ( a_{1} + a_{5} ) + ( a_{2} + a_{6} ) + ( a_{3} + a_{7} ) + ( a_{4} + a_{8} ) + ( a_{9} + a_{13} ) + ( a_{10} + a_{14} ) + ( a_{11} + a_{15} ) + ( a_{12} + a_{16} ) $$

$$ = 12 \times 8 $$

$$ = 96 $$
