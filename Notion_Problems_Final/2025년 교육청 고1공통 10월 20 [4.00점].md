## 문제
$0$이 아닌 실수 $a$에 대하여 좌표평면 위의 서로 다른 세 점 $A(2a, 0)$, $B$, $C$가 다음 조건을 만족시킨다.

$$ \begin{array}{|l|} \hline \circ \text{ 삼각형 } ABC \text{ 의 무게중심의 좌표는 } (0, 2) \text{ 이다.} \\ \circ \overline{AB} = \overline{AC} \\ \hline \end{array} $$

다음은 $\overline{BC} = 2\sqrt{a^2 + 1}$일 때, 점 $B$의 $x$좌표와 $y$좌표의 합을 구하는 과정이다. (단, 점 $B$의 $x$좌표는 점 $C$의 $x$좌표보다 크다.)

선분 $BC$의 중점을 $M(b, c)$, 삼각형 $ABC$의 무게중심을 $G$라 하면, 점 $G(0, 2)$는 선분 $AM$을 $2:1$로 내분하는 점이므로 $b = -a$, $c = (가)$이다.
$\overline{AB} = \overline{AC}$이고 직선 $AM$의 기울기가 $-\frac{1}{a}$이므로 직선 $BC$의 방정식은
$$y = (나) \times \{x - (-a)\} + (가)$$
이다. 점 $B$의 $x$좌표를 $k$라 하면 점 $B$의 $y$좌표는 $(나) \times (k + a) + (가)$이다.
$\overline{BM} = \frac{1}{2}\overline{BC}$이고 점 $B$의 $x$좌표가 점 $C$의 $x$좌표보다 크므로 $k = (다)$이다.
따라서 점 $B$의 $x$좌표와 $y$좌표의 합은 $(라)$이다.

위의 (가), (라)에 알맞은 수를 각각 $p$, $q$라 하고, (나), (다)에 알맞은 식을 각각 $f(a)$, $g(a)$라 할 때, $f(p) \times g(q)$의 값은?

## 보기
① $-10$   ② $-\frac{19}{2}$   ③ $-9$   ④ $-\frac{17}{2}$   ⑤ $-8$

## 풀이
선분 $BC$의 중점을 $M(b, c)$, 삼각형 $ABC$의 무게중심을 $G$라 하면, 점 $G(0, 2)$는 선분 $AM$을 $2:1$로 내분하는 점이므로
$$\left( \frac{2 \times b + 1 \times 2a}{2 + 1}, \frac{2 \times c + 1 \times 0}{2 + 1} \right) = (0, 2)$$
에서 $2a + 2b = 0$, $2c = 6$이다.
따라서 $b = -a$, $c = 3$이다. $\therefore p = 3$

$\overline{AB} = \overline{AC}$이므로 직선 $AM$과 직선 $BC$는 서로 수직이다.
직선 $AM$의 기울기가 $\frac{3 - 0}{-a - 2a} = -\frac{1}{a}$이므로 직선 $BC$의 기울기는 $a$이다. $\therefore f(a) = a$
직선 $BC$는 기울기가 $a$이고 점 $M(-a, 3)$을 지나므로 직선 $BC$의 방정식은
$$y = a \times \{x - (-a)\} + 3$$
이다. 점 $B$의 $x$좌표를 $k$라 하면 점 $B$의 $y$좌표는 $a(k + a) + 3$이다.

$B(k, a(k + a) + 3)$, $M(-a, 3)$이므로
$$\begin{aligned} \overline{BM} &= \sqrt{(-a - k)^2 + \{3 - (ak + a^2 + 3)\}^2} \\ &= \sqrt{(a + k)^2 + a^2(a + k)^2} \\ &= \sqrt{(a + k)^2(a^2 + 1)} \end{aligned}$$
이고, $\overline{BM} = \frac{1}{2}\overline{BC} = \sqrt{a^2 + 1}$이므로
$$\sqrt{(a + k)^2(a^2 + 1)} = \sqrt{a^2 + 1}$$
$$\sqrt{(a + k)^2} = 1 \Rightarrow |a + k| = 1$$
$k = -a + 1$ 또는 $k = -a - 1$이다.
점 $B$의 $x$좌표가 점 $C$의 $x$좌표보다 크므로 점 $B$의 $x$좌표 $k$는 점 $M$의 $x$좌표 $-a$보다 커야 한다.
그러므로 $k = -a + 1$이다. $\therefore g(a) = -a + 1$

점 $B$의 $y$좌표는 $a(-a + 1 + a) + 3 = a + 3$이므로 점 $B$의 $x$좌표와 $y$좌표의 합은
$$(-a + 1) + (a + 3) = 4$$
이다. $\therefore q = 4$

따라서 $p = 3$, $q = 4$, $f(a) = a$, $g(a) = -a + 1$이므로
$$f(p) \times g(q) = f(3) \times g(4) = 3 \times (-4 + 1) = -9$$
정답은 ③이다.
