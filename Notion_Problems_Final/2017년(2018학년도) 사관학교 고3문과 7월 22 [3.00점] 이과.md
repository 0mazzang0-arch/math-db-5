## 문제
확률변수 $X$ 가 이항분포 $B ( 300 , \frac{2}{5} )$ 를 따를 때, $V ( X )$ 의 값을 구하시오.

## 풀이
확률변수 $X$ 가 이항분포 $B ( n , p )$ 를 따를 때, 분산 $V ( X ) = n p q$ (단, $q = 1 - p$) 이다.
$n = 300$, $p = \frac{2}{5}$, $q = 1 - \frac{2}{5} = \frac{3}{5}$ 이므로
$V ( X ) = 300 \times \frac{2}{5} \times \frac{3}{5} = 72$
따라서 정답은 $72$ 이다.
