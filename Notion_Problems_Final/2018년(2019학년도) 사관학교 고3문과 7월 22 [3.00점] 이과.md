## 문제
$\sqrt{3 \sqrt[4]{27}} = 3^{\frac{q}{p}}$ 일 때, $p+q$ 의 값을 구하시오. (단, $p$ 와 $q$ 는 서로소인 자연수이다.)

## 풀이
$\sqrt{3 \sqrt[4]{27}} = ( 3 \times 27^{\frac{1}{4}} )^{\frac{1}{2}} = ( 3 \times (3^{3})^{\frac{1}{4}} )^{\frac{1}{2}} = ( 3 \times 3^{\frac{3}{4}} )^{\frac{1}{2}} = ( 3^{\frac{7}{4}} )^{\frac{1}{2}} = 3^{\frac{7}{8}}$
따라서 $\frac{q}{p} = \frac{7}{8}$ 이고, $p$, $q$ 는 서로소인 자연수이므로
$p = 8$, $q = 7$
$\therefore p+q = 8+7 = 15$
