## 문제
두 함수 $f(x) = x^{2} - 4x$와 $y = g(x)$가 임의의 실수 $h$에 대하여 
$$g(x+h) - g(x) = \int_{x}^{x+h} f(t) dt$$
일 때, 방정식 $g(x) = 0$의 모든 근의 합은?

## 보기
① $6$   ② $5$   ③ $4$   ④ $3$   ⑤ $2$

## 풀이
$$\frac{g(x+h)-g(x)}{h} = \frac{1}{h} \int_{x}^{x+h} f(t) dt$$ 
위 식의 양변에 극한을 취하면 
$$\lim_{h \to 0} \frac{g(x+h)-g(x)}{h} = \lim_{h \to 0} \frac{1}{h} \int_{x}^{x+h} f(t) dt$$ 
$g'(x) = f(x)$이므로 
$$g(x) = \int f(x) dx = \int (x^{2} - 4x) dx$$ 
$$= \frac{1}{3}x^{3} - 2x^{2} + C$$ 
$g(x) = 0$에서 
$$\frac{1}{3}x^{3} - 2x^{2} + C = 0$$
이다. 삼차방정식의 근과 계수의 관계에 의하여 모든 근의 합은 
$$-\frac{-2}{\frac{1}{3}} = 6$$
정답: ①
