## 문제
세 함수 $f(x) = (1+r_1)^x$, $g(x) = \left( 1 + \frac{r_2}{2} \right)^{2x}$, $h(x) = \left( 1 + \frac{r_3}{4} \right)^{4x}$ 에 대하여 $f(10) = g(10) = h(10)$ 일 때, $r_1$, $r_2$, $r_3$ 의 대소 관계를 옳게 나타낸 것은? (단, $r_1$, $r_2$, $r_3$ 는 양의 실수이다.)

## 보기
① $r_1 < r_2 < r_3$   ② $r_1 < r_3 < r_2$   ③ $r_2 < r_1 < r_3$   ④ $r_2 < r_3 < r_1$   ⑤ $r_3 < r_2 < r_1$

## 풀이
$f(10) = g(10)$ 에서 
$(1+r_1)^{10} = \left( 1 + \frac{r_2}{2} \right)^{20}$ 
$1+r_1 = \left( 1 + \frac{r_2}{2} \right)^2 = 1 + r_2 + \left( \frac{r_2}{2} \right)^2$ 
$r_1 = r_2 + \left( \frac{r_2}{2} \right)^2$ 
$\therefore r_1 > r_2$ 

$g(10) = h(10)$ 에서 
$\left( 1 + \frac{r_2}{2} \right)^{20} = \left( 1 + \frac{r_3}{4} \right)^{40}$ 
$1 + \frac{r_2}{2} = \left( 1 + \frac{r_3}{4} \right)^2 = 1 + \frac{r_3}{2} + \left( \frac{r_3}{4} \right)^2$ 
$\frac{r_2}{2} = \frac{r_3}{2} + \left( \frac{r_3}{4} \right)^2$ 
$r_2 = r_3 + 2 \cdot \left( \frac{r_3}{4} \right)^2$ 
$\therefore r_2 > r_3$ 

위 식들에 의하여 대소 관계는 다음과 같다.
$r_3 < r_2 < r_1$ 

정답: ⑤
