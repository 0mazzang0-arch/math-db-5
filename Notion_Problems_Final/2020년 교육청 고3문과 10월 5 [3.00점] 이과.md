## 문제
등차수열 $\{ a_n \}$ 에 대하여
$a_1 + a_2 + a_3 = 15$,
$a_3 + a_4 + a_5 = 39$
일 때, 수열 $\{ a_n \}$ 의 공차는?

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
등차수열 $\{ a_n \}$ 의 공차를 $d$ 라 하면
$a_3 = a_1 + 2d$, $a_4 = a_2 + 2d$, $a_5 = a_3 + 2d$ 이므로
$a_3 + a_4 + a_5 = a_1 + a_2 + a_3 + 6d$
즉, $6d = ( a_3 + a_4 + a_5 ) - ( a_1 + a_2 + a_3 )$
$6d = 39 - 15$
$6d = 24$
따라서 $d = 4$
