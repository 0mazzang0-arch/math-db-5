## 문제
다항식 $f(x)$, $g(x)$에 대하여 세 집합 $A$, $B$, $C$를
$$A = \{ x \mid f(x)=0 \}$$
$$B = \{ x \mid g(x)=0 \}$$
$$C = \{ x \mid (f(x)+g(x))^{3}=(f(x))^{3}+(g(x))^{3} \}$$
라고 하자. $n(A)=4$이고 $n(B)=3$일 때, $n(C)$의 최솟값은? (단, $n(X)$는 집합 $X$의 원소의 개수)

## 보기
① $3$   ② $4$   ③ $5$   ④ $6$   ⑤ $7$

## 풀이
집합 $C$의 조건 제시법에 주어진 방정식을 정리하면
$$(f(x)+g(x))^{3}=(f(x))^{3}+(g(x))^{3}$$
$$(f(x))^{3} + 3(f(x))^{2}g(x) + 3f(x)(g(x))^{2} + (g(x))^{3} = (f(x))^{3} + (g(x))^{3}$$
$$3(f(x))^{2}g(x) + 3f(x)(g(x))^{2} = 0$$
$$3f(x)g(x)\{ f(x)+g(x) \} = 0$$
이므로 $f(x)=0$ 또는 $g(x)=0$ 또는 $f(x)+g(x)=0$이다.
즉, $C = A \cup B \cup \{ x \mid f(x)+g(x)=0 \}$이다.
$n(A)=4$, $n(B)=3$일 때, 집합 $C$의 원소의 개수가 최소가 되기 위해서는 $B \subset A$이고 $\{ x \mid f(x)+g(x)=0 \} \subset A$인 경우를 생각할 수 있다.
따라서 $n(C)$의 최솟값은 $n(A)=4$일 때이다.

정답: ②
