## 문제
함수 $f(x) = x^2 - 8x + a$에 대하여 함수 $g(x)$를
$$ g(x) = \left\{ \begin{array}{ll} 2x + 5a & (x \ge a) \\ f(x+4) & (x < a) \end{array} \right. $$
라 할 때, 다음 조건을 만족시키는 모든 실수 $a$의 값의 곱을 구하시오.

$$ \begin{array}{|l|} \hline \text{(가) 방정식 } f(x) = 0 \text{ 은 열린구간 } (0, 2) \text{ 에서 적어도 하나의 실근을 갖는다.} \\ \text{(나) 함수 } f(x)g(x) \text{ 는 } x = a \text{ 에서 연속이다.} \\ \hline \end{array} $$

## 보기

## 풀이
주어진 이차함수 $f(x)$는 축의 방정식이 $x = 4$이고, 방정식 $f(x) = 0$은 열린 구간 $(0, 2)$에서 적어도 하나의 실근을 가지므로
$f(0) = a > 0$, $f(2) = a - 12 < 0$
$\therefore 0 < a < 12$

조건 (나)에서 함수 $f(x)g(x)$가 $x = a$에서 연속이어야 하므로
$f(a)g(a) = (a^2 - 8a + a)(2a + 5a) = 7a^2(a - 7)$

$\lim_{x \to a^+} f(x)g(x) = \lim_{x \to a^+} (x^2 - 8x + a)(2x + 5a) = 7a^2(a - 7)$

$\lim_{x \to a^-} f(x)g(x) = \lim_{x \to a^-} (x^2 - 8x + a)f(x + 4)$
$= (a^2 - 8a + a)\{(a + 4)^2 - 8(a + 4) + a\}$
$= a(a - 7)(a^2 + a - 16)$

함수 $f(x)g(x)$가 $x = a$에서 연속이므로 좌극한과 우극한(함숫값)이 같아야 한다.
$7a^2(a - 7) = a(a - 7)(a^2 + a - 16)$
$a(a - 7)\{7a - (a^2 + a - 16)\} = 0$
$a(a - 7)(-a^2 + 6a + 16) = 0$
$a(a - 7)(a^2 - 6a - 16) = 0$
$a(a - 7)(a - 8)(a + 2) = 0$

$\therefore a = 7$ 또는 $a = 8$ ($\because 0 < a < 12$)

따라서 모든 실수 $a$의 값의 곱은 $7 \times 8 = 56$이다.

정답: $56$
