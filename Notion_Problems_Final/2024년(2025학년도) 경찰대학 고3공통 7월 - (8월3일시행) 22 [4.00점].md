## 문제

자연수 $n$ 에 대하여 집합 $\{ x \mid x \le \log_{2} (x+n), x \text{ 는 자연수} \}$ 의 원소의 개수를 $f(n)$ 이라 할 때, $\sum_{n=1}^{20} f(n)$ 의 값을 구하시오.

## 풀이

> [!example] 📸 (그림/그래프)

위의 그래프에서 $f(n) = k$ ( 단, $k$ 는 자연수)라 하면 
$x = k$ 일 때 $k \le \log_{2} (k+n)$ 
$x = k+1$ 일 때 $\log_{2} (k+1+n) < k+1$ 
이 성립하므로 두 식을 연립하면 
$2^k - k \le n < 2^{k+1} - k - 1 \quad \cdots$ ㉠ 
이 성립한다. ㉠의 양변의 $k$ 값에 자연수를 대입하여 조건을 만족하는 자연수 $n$ 을 구하면 아래와 같다. 

(ⅰ) $k = 1$ 일 때 
$2 - 1 \le n < 2^2 - 2$, $n = 1$ 
$f(n) = 1$ 인 자연수 $n$ 은 $1$ 뿐이다. 

(ⅱ) $k = 2$ 일 때 
$2^2 - 2 \le n < 2^3 - 3$, $n = 2, 3, 4$ 
$f(n) = 2$ 인 자연수 $n$ 은 $n = 2, 3, 4$ 이다. 

(ⅲ) $k = 3$ 일 때 
$2^3 - 3 \le n < 2^4 - 4$, $5 \le n < 12$ 
$f(n) = 3$ 인 자연수 $n$ 은 $n = 5, 6, 7, \dots, 11$ 이다. 

(ⅳ) $k = 4$ 일 때 
$2^4 - 4 \le n < 2^5 - 5$, $12 \le n < 27$ 
$f(n) = 4$ 인 자연수 $n$ 은 $n = 12, 13, 14, \dots, 26$ 이다. 

이상에서 
$\sum_{n=1}^{20} f(n) = 1 \times 1 + 2 \times 3 + 3 \times 7 + 4 \times 9 = 64$

정답: $64$
