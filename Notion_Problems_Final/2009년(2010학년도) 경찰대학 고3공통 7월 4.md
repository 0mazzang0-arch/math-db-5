## 문제
삼차다항식 $f(x) = x^3 - 6x^2 + 3x + 7$에 대하여 서로 다른 세 실수 $\alpha$, $\beta$, $\gamma$가 $f(\alpha) = f(\beta) = f(\gamma) = -3$을 만족시킬 때, $\alpha^2 + \beta^2 + \gamma^2$의 값은?

## 보기
① $14$   ② $21$   ③ $26$   ④ $30$   ⑤ $35$

## 풀이
$f(\alpha) = f(\beta) = f(\gamma) = -3$이므로 
$f(\alpha) + 3 = f(\beta) + 3 = f(\gamma) + 3 = 0$이다. 

$g(x) = f(x) + 3$이라 하면 최고차항의 계수가 $1$인 삼차식 $g(x)$는 $x-\alpha$, $x-\beta$, $x-\gamma$를 인수로 갖는다. 
따라서 
$g(x) = (x-\alpha)(x-\beta)(x-\gamma)$이다.

방정식 $g(x) = 0$은 $x^3 - 6x^2 + 3x + 10 = 0$이며, 이 방정식의 세 근이 $\alpha$, $\beta$, $\gamma$이므로 근과 계수의 관계에 의하여 
$\alpha + \beta + \gamma = 6$, 
$\alpha\beta + \beta\gamma + \gamma\alpha = 3$, 
$\alpha\beta\gamma = -10$이다.

따라서 
$\alpha^2 + \beta^2 + \gamma^2$
$= (\alpha + \beta + \gamma)^2 - 2(\alpha\beta + \beta\gamma + \gamma\alpha)$
$= 6^2 - 2 \cdot 3 = 36 - 6 = 30$

정답은 ④이다.
