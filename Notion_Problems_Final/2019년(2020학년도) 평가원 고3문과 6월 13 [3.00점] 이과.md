## 문제
자연수 $n$에 대하여 $x$에 대한 이차방정식 $x^2 - nx + 4(n - 4) = 0$이 서로 다른 두 실근 $\alpha, \beta$ ($\alpha < \beta$)를 갖고, 세 수 $1, \alpha, \beta$가 이 순서대로 등차수열을 이룰 때, $n$의 값은?

## 보기
① $5$   ② $8$   ③ $11$   ④ $14$   ⑤ $17$

## 풀이
세 수 $1, \alpha, \beta$가 이 순서대로 등차수열을 이루므로 
$2\alpha = 1 + \beta$ $\cdots \cdots$ ㉠ 
이차방정식 $x^2 - nx + 4(n - 4) = 0$을 풀면 
$(x - 4)(x - n + 4) = 0$ 
$\therefore x = 4, n - 4$

(ⅰ) $n - 4 > 4$ 일 때, 즉, $n > 8$ 이고 $\alpha < \beta$ 이므로 
$\alpha = 4, \beta = n - 4$ 
$8 = 1 + n - 4$ 
$\therefore n = 11$

(ⅱ) $n - 4 < 4$ 일 때, 즉 $0 < n < 8$ 에서 
$\alpha = n - 4, \beta = 4$ 
$2(n - 4) = 1 + 4$
$2n - 8 = 5$
$\therefore n = \frac{13}{2}$ 
$n$은 자연수이므로 성립하지 않는다. 

(ⅰ), (ⅱ)에서 $n = 11$

정답: ③
