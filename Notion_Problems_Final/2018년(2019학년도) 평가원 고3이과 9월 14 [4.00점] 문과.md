## 문제
실수 $k$ 에 대하여 함수 $f(x) = \cos^{2} \left( x - \frac{3}{4}\pi \right) - \cos \left( x - \frac{\pi}{4} \right) + k$의 최댓값은 $3$, 최솟값은 $m$이다. $k+m$의 값은?

## 보기
① $2$   ② $\frac{9}{4}$   ③ $\frac{5}{2}$   ④ $\frac{11}{4}$   ⑤ $3$

## 풀이
$x - \frac{3}{4}\pi = \theta$라 하면 $x - \frac{\pi}{4} = \theta + \frac{\pi}{2}$이므로
$f(x) = \cos^{2} \theta - \cos \left( \theta + \frac{\pi}{2} \right) + k$
$= \cos^{2} \theta + \sin \theta + k$
$= 1 - \sin^{2} \theta + \sin \theta + k$
$= - \left( \sin \theta - \frac{1}{2} \right)^{2} + k + \frac{5}{4}$

최댓값은 $k + \frac{5}{4} = 3$이므로 $k = \frac{7}{4}$
최솟값은 $\sin \theta = -1$일 때이므로
$m = - \left( -1 - \frac{1}{2} \right)^{2} + \frac{7}{4} + \frac{5}{4} = - \frac{9}{4} + 3 = \frac{3}{4}$

따라서 $k + m = \frac{7}{4} + \frac{3}{4} = \frac{5}{2}$

정답: ③
