## 문제
그림과 같이 직선 $x+y=2$ 위의 점 $P(a, b)$ ($ab \neq 0$)에서 $x$축, $y$축에 내린 수선의 발을 각각 $Q$, $R$라 하고, 점 $P$를 지나고 직선 $QR$에 수직인 직선을 $l$이라 하자.

> [!example] 📸 (그림/그래프)

다음은 직선 $l$이 점 $P$의 위치에 관계없이 항상 일정한 점을 지남을 보이는 과정이다.

$$ \begin{array}{|l|} \hline \text{점 } P(a, b)\text{는 직선 } x+y=2 \text{ 위의 점이므로 } b = 2-a \\ \text{이때 직선 } l\text{의 기울기는 (가)이므로} \\ l\text{의 방정식은} \\ y-(2-a) = \text{(가)} \times (x-a) \cdots \text{㉠} \\ \text{한편, ㉠이 } a\text{의 값에 관계없이 항상 성립하려면} \\ x = \text{(나)} \\ y = \text{(다)} \\ \text{따라서 직선 } l\text{은 점 } P\text{의 위치에 관계없이 항상 일정한 점을 지난다.} \\ \hline \end{array} $$

위의 과정에서 (가)에 알맞은 식을 $f(a)$라 하고, (나), (다)에 알맞은 수를 각각 $\alpha$, $\beta$라 할 때, $f( \frac{4}{3} ) + \alpha + \beta$의 값은? (단, $O$는 원점이다.)

## 보기
① $4$   ② $5$   ③ $6$   ④ $7$   ⑤ $8$

## 풀이
점 $P(a, b)$는 직선 $x+y=2$ 위의 점이므로 $a+b=2$에서 $b=2-a$이다.
점 $P$에서 $x$축, $y$축에 내린 수선의 발은 각각 $Q(a, 0)$, $R(0, b)$이다.
직선 $l$은 직선 $QR$와 수직이고, 직선 $QR$의 기울기는 $-\frac{b}{a}$이므로 직선 $l$의 기울기는 $\frac{a}{b} = \frac{a}{2-a}$이다.
따라서 (가)에 알맞은 식 $f(a) = \frac{a}{2-a}$이다.

직선 $l$의 방정식은 $y-(2-a) = \frac{a}{2-a}(x-a)$이다.
양변에 $2-a$를 곱하여 정리하면
$(2-a)y - (2-a)^2 = a(x-a)$
$2y - ay - (4 - 4a + a^2) = ax - a^2$
$2y - ay - 4 + 4a - a^2 = ax - a^2$
이 식을 $a$에 대하여 정리하면
$(x+y-4)a + (4-2y) = 0$
위 등식이 $a$의 값에 관계없이 항상 성립하려면
$x+y-4=0$, $4-2y=0$
따라서 $x=2$, $y=2$이다.
즉, $\alpha=2$, $\beta=2$이다.

그러므로
$f( \frac{4}{3} ) = \frac{\frac{4}{3}}{2-\frac{4}{3}} = \frac{\frac{4}{3}}{\frac{2}{3}} = 2$
이때 $f( \frac{4}{3} ) + \alpha + \beta = 2 + 2 + 2 = 6$이다.
