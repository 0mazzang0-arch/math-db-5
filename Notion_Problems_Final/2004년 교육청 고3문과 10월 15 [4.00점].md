## 문제
어느 상품의 수요량 $D$와 판매가격 $P$ 사이에는 
$$\log _{a} D=\log _{a} c- \frac{1}{3} \log _{a} P$$
($a$, $c$는 양의 상수, $a \neq 1$)인 관계가 성립한다고 한다. 이 상품의 판매가격이 $P _{1}$, $4P _{1}$일 때의 수요량을 각각 $D _{1}$, $D _{2}$라 할 때, $\frac{D _{2}}{D _{1}}$의 값은?

## 보기
① $2^{-\frac{2}{3}}$   ② $2^{-\frac{1}{3}}$   ③ $2^{-\frac{1}{2}}$   ④ $2^{\frac{1}{3}}$   ⑤ $2^{\frac{2}{3}}$

## 풀이
주어진 관계식 $\log_{a} D = \log_{a} c - \frac{1}{3} \log_{a} P$에 각각의 값을 대입하면 다음과 같다.

판매가격이 $P_1$일 때,
$\log_{a} D_{1} = \log_{a} c - \frac{1}{3} \log_{a} P_{1} \cdots$ ㉠

판매가격이 $4P_1$일 때,
$\log_{a} D_{2} = \log_{a} c - \frac{1}{3} \log_{a} (4P_{1}) \cdots$ ㉡

㉡ $-$ ㉠을 하면
$$\log_{a} D_{2} - \log_{a} D_{1} = \left(\log_{a} c - \frac{1}{3} \log_{a} 4P_{1}\right) - \left(\log_{a} c - \frac{1}{3} \log_{a} P_{1}\right)$$
$$\log_{a} \frac{D_{2}}{D_{1}} = \frac{1}{3} (\log_{a} P_{1} - \log_{a} 4P_{1})$$
$$\log_{a} \frac{D_{2}}{D_{1}} = \frac{1}{3} \log_{a} \frac{P_{1}}{4P_{1}}$$
$$\log_{a} \frac{D_{2}}{D_{1}} = \log_{a} \left(\frac{1}{4}\right)^{\frac{1}{3}}$$

따라서
$$\frac{D_{2}}{D_{1}} = \left(\frac{1}{4}\right)^{\frac{1}{3}} = (2^{-2})^{\frac{1}{3}} = 2^{-\frac{2}{3}}$$

정답: ①
