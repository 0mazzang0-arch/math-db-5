## 문제
첫째항이 `$2$`이고 공비가 정수인 등비수열 `$\{ a_{n} \}$`과 자연수 `$m$`이 다음 조건을 만족시킬 때, `$a_{m}$`의 값을 구하시오.

(가) `$4 < a_{2} + a_{3} \leq 12$`
(나) `$\sum_{k=1}^{m} a_{k} = 122$`

## 풀이
첫째항이 `$2$`이고 공비를 `$r$` (단, `$r$`은 정수)이라 하면
`$a_{n} = 2r^{n-1} \cdots$ ㉠`
㉠에서 `$a_{2} = 2r, a_{3} = 2r^{2}$`이므로 조건 (가)에 대입하면
`$4 < 2r + 2r^{2} \leq 12$`, `$2 < r + r^{2} \leq 6$`
따라서 `$0 < r^{2} + r - 2$`, `$r^{2} + r - 6 \leq 0$`를 풀면
`$0 < (r+2)(r-1)$`에서 `$r < -2$` 또는 `$r > 1 \cdots$ ㉡`
`$(r+3)(r-2) \leq 0$`에서 `$-3 \leq r \leq 2 \cdots$ ㉢`
㉡, ㉢에 의해 `$-3 \leq r < -2$` 또는 `$1 < r \leq 2$`
그런데 공비 `$r$`이 정수이므로 `$r = -3$` 또는 `$r = 2$`

(나) 조건에서 `$\sum_{k=1}^{m} a_{k} = 122$`이므로
(ⅰ) `$r = -3$`인 경우
`$\frac{2\{(-3)^{m} - 1\}}{-3 - 1} = 122$`, `$\frac{(-3)^{m} - 1}{-2} = 122$`
`$(-3)^{m} - 1 = -244$`, `$(-3)^{m} = -243$`
`$\therefore m = 5$`
`$a_{m} = 2(-3)^{m-1}$`이므로 `$a_{5} = 2(-3)^{4} = 162$`

(ⅱ) `$r = 2$`인 경우
`$\frac{2(2^{m} - 1)}{2 - 1} = 122$`, `$2^{m} - 1 = 61$`, `$2^{m} = 62$`
`$m$`은 자연수이므로 조건에 모순이 된다.

(ⅰ), (ⅱ)에서 만족하는 `$m = 5$`이므로
`$a_{m} = a_{5} = 162$`

정답: `$162$`
