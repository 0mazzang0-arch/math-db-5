## 문제
두 집합 $A = \{ x \mid \frac{(x-2)^{2}}{x-4} \le 0 \}$, $B = \{ x \mid x^{2}-8x+a \le 0 \}$에 대하여 $A \cup B = \{ x \mid x \le 5 \}$일 때, 상수 $a$의 값은?



> [!example] 📸 (그림/그래프)

## 보기
① $7$   ② $10$   ③ $12$   ④ $15$   ⑤ $16$

## 풀이
[정답] ④ 

집합 $A$의 분수부등식 $\frac{(x-2)^2}{x-4} \le 0$에서 $x-4 \ne 0$이므로 양변에 $(x-4)^2$을 곱하면 
$(x-2)^2(x-4) \le 0$, $x \ne 4$
부등식의 해는 $x < 4$

따라서 집합 $A \cup B = \{ x \mid x \le 5 \}$가 되려면 집합 $B$의 이차부등식 $x^2-8x+a \le 0$은 
$(x-\alpha)(x-5) \le 0$ ($\alpha \le 4$)
이 되어야 한다. 

그런데 $\alpha + 5 = 8$이므로 $\alpha = 3$
$\therefore a = 3 \times 5 = 15$
