## 문제
두 실수 $x$, $y$ 에 대하여 $\sin x + \sin y = 1$, $\cos x + \cos y = \frac{1}{2}$ 일 때, $\cos (x-y)$ 의 값은?

## 보기
① $\frac{5}{8}$   ② $\frac{3}{8}$   ③ $\frac{1}{8}$   ④ $-\frac{3}{8}$   ⑤ $-\frac{5}{8}$

## 풀이
$(\sin x + \sin y)^2 = \sin^2 x + \sin^2 y + 2 \sin x \sin y = 1$  
$(\cos x + \cos y)^2 = \cos^2 x + \cos^2 y + 2 \cos x \cos y = \frac{1}{4}$  
위 두 등식의 변끼리 더하면  
$(\sin^2 x + \cos^2 x) + (\sin^2 y + \cos^2 y) + 2(\sin x \sin y + \cos x \cos y) = 1 + \frac{1}{4}$  
$1 + 1 + 2 (\sin x \sin y + \cos x \cos y) = \frac{5}{4}$  
$2 + 2(\cos x \cos y + \sin x \sin y) = \frac{5}{4}$  
$2 \cos (x-y) = \frac{5}{4} - 2 = -\frac{3}{4}$  
$\therefore \cos (x-y) = -\frac{3}{8}$  

정답: ④
