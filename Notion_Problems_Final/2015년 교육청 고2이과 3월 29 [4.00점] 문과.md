## 문제
수열 $\{ a_{n} \}$을 다음과 같이 정의하자.

집합 $A_{n} = \{ x | (x - n)(x - 2n + 1) \le 0 \}$에 대하여 $25 \in A_{n}$이면 $a_{n} = 1$이고, $25 \notin A_{n}$이면 $a_{n} = -1$이다. 

$\sum_{k=1}^{m} a_{k} = -20$을 만족시키는 자연수 $m$의 값을 구하시오.

## 보기


## 풀이
집합 $A_{n} = \{ x | (x - n)(x - 2n + 1) \le 0 \}$에서 
$(x - n)(x - 2n + 1) \le 0$ 
$n \le x \le 2n - 1$ 
$\therefore A_{n} = \{ x | n \le x \le 2n - 1 \}$ 

$25 \in A_{n}$인 $n$의 값의 범위를 구하면 
$n \le 25 \le 2n - 1$ 
$n \le 25$이고 $n \ge 13$에서 $13 \le n \le 25$ 

$\therefore a_{n} = \begin{cases} -1 & (1 \le n \le 12 \text{ 또는 } n \ge 26) \\ 1 & (13 \le n \le 25) \end{cases}$ 

$\sum_{k=1}^{m} a_{k} = -20$에서 $m \ge 26$이므로 
$\sum_{k=1}^{m} a_{k} = \sum_{k=1}^{12} a_{k} + \sum_{k=13}^{25} a_{k} + \sum_{k=26}^{m} a_{k}$ 
$= (-1) \times 12 + 1 \times 13 + (-1) \times (m - 25)$ 
$= -m + 26 = -20$ 

$\therefore m = 46$
