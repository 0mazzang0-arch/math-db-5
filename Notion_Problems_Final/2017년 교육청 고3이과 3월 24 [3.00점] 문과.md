## 문제
구간 $($ $-1$ $,$ $\infty$ $)$에서 정의된 함수 $f(x) = xe^{x} + e$의 역함수를 $g(x)$라 할 때, $60g'(e)$의 값을 구하시오.

## 풀이
$g(e) = a$라 하면, $f(a) = e$이다.
$f(a) = ae^{a} + e = e$에서
$ae^{a} = 0$이므로 $a = 0$이다.
$f'(x) = (x + 1)e^{x}$이므로
$f'(0) = (0 + 1)e^{0} = 1$이다.
역함수의 미분법에 의해
$g'(e) = \frac{1}{f'(g(e))} = \frac{1}{f'(0)} = \frac{1}{1} = 1$
따라서
$60g'(e) = 60 \times 1 = 60$
