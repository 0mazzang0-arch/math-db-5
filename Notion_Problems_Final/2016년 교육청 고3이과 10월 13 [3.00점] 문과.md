## 문제
함수 $f(x) = e^{x+1}(x^{2} + 3x + 1)$ 이 구간 $(a, b)$ 에서 감소할 때, $b - a$ 의 최댓값은?

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
함수 $f(x)$ 의 도함수 $f'(x)$ 는
$f'(x) = e^{x+1}(x^{2} + 3x + 1) + e^{x+1}(2x + 3)$
$= e^{x+1}(x^{2} + 5x + 4)$
함수 $f(x)$ 가 감소하려면 $f'(x) < 0$ 이어야 하므로
$e^{x+1}(x^{2} + 5x + 4) < 0$
이때 모든 실수 $x$ 에 대하여 $e^{x+1} > 0$ 이므로
$x^{2} + 5x + 4 < 0$
$(x + 4)(x + 1) < 0$
$-4 < x < -1$
따라서 함수 $f(x)$ 는 구간 $(-4, -1)$ 에서 감소한다.
구간 $(a, b)$ 가 $(-4, -1)$ 에 포함되어야 하므로 $b - a$ 의 최댓값은
$(-1) - (-4) = 3$
정답: ③
