## 문제
좌표평면에서 두 점 $A(4, a)$, $B(2, 1)$을 직선 $y=x$에 대하여 대칭이동한 점을 각각 $A'$, $B'$이라 하고, 두 직선 $AB$, $A'B'$의 교점을 $P$라 하자. 두 삼각형 $APA'$, $BPB'$의 넓이의 비가 $9:4$일 때, $a$의 값은? (단, $a>4$)

## 보기
① $5$   ② $\frac{11}{2}$   ③ $6$   ④ $\frac{13}{2}$   ⑤ $7$

## 풀이
[정답] ② 

두 점 $A(4, a)$, $B(2, 1)$을 직선 $y=x$에 대하여 대칭이동한 점은 각각 $A'(a, 4)$, $B'(1, 2)$이다. 
두 직선 $AA'$, $BB'$은 각각 직선 $y=x$와 서로 수직이므로 두 직선 $AA'$, $BB'$은 서로 평행하다. 
따라서 두 삼각형 $APA'$, $BPB'$은 서로 닮은 삼각형이다. 
두 삼각형 $APA'$, $BPB'$의 넓이의 비가 $9:4$이므로 두 삼각형 $APA'$, $BPB'$의 닮음비는 $3:2$이다. 
$\therefore \overline{AA'} : \overline{BB'} = 3:2$ 

$\overline{AA'} = \sqrt{(a-4)^2 + (4-a)^2} = \sqrt{2}(a-4)$ (단, $a>4$) 

$\overline{BB'} = \sqrt{(2-1)^2 + (1-2)^2} = \sqrt{2}$ 

$\overline{AA'} : \overline{BB'} = 3:2$에서 

$\sqrt{2}(a-4) : \sqrt{2} = 3:2$ 

$2(a-4) = 3$에서 $a = \frac{11}{2}$ 

> [!example] 📸 (그림/그래프)

[다른 풀이1] 
두 점 $A(4, a)$, $B(2, 1)$을 직선 $y=x$에 대하여 대칭이동한 점은 각각 $A'(a, 4)$, $B'(1, 2)$이다. 
두 직선 $AA'$, $BB'$은 각각 직선 $y=x$와 서로 수직이므로 두 직선 $AA'$, $BB'$은 서로 평행하다. 
따라서 두 삼각형 $APA'$, $BPB'$은 서로 닮은 삼각형이다. 
두 삼각형 $APA'$, $BPB'$의 넓이의 비가 $9:4$이므로 두 삼각형의 닮음비는 $3:2$이다. 

$\overline{AP} : \overline{BP} = 3:2$ 
점 $P$는 선분 $AB$를 $3:2$로 내분하는 점이므로 점 $P$의 좌표는 
$\left( \frac{3 \times 2 + 2 \times 4}{3+2}, \frac{3 \times 1 + 2 \times a}{3+2} \right)$, 즉 $\left( \frac{14}{5}, \frac{2a+3}{5} \right)$ 
두 직선 $AB$, $A'B'$은 직선 $y=x$에 대하여 서로 대칭이므로 두 직선 $AB$, $A'B'$의 교점 $P$는 직선 $y=x$ 위의 점이다. 

$\frac{14}{5} = \frac{2a+3}{5}$ 

$a = \frac{11}{2}$ 

[다른 풀이2] 
두 점 $A(4, a)$, $B(2, 1)$을 직선 $y=x$에 대하여 대칭이동한 점은 각각 $A'(a, 4)$, $B'(1, 2)$이다. 
두 직선 $AA'$, $BB'$은 각각 직선 $y=x$와 서로 수직이므로 두 직선 $AA'$, $BB'$은 서로 평행하다. 
따라서 두 삼각형 $APA'$, $BPB'$은 서로 닮은 삼각형이다. 
점 $A$와 직선 $y=x$ 사이의 거리와 점 $B$와 직선 $y=x$ 사이의 거리의 비가 닮음비 $3:2$와 같다. 

$\frac{|4-a|}{\sqrt{1^2 + (-1)^2}} : \frac{|2-1|}{\sqrt{1^2 + (-1)^2}} = 3:2$ 

$2(a-4) = 3$ (단, $a>4$) 

$a = \frac{11}{2}$ 

[참고] 
$a$의 값에 따라 두 삼각형 $APA'$, $BPB'$의 존재 여부와 두 삼각형의 닮음비가 달라진다. 
(ⅰ) $a < -1$ 또는 $-1 < a < 3$ 또는 $a > 5$일 때 
$APA'$, $BPB'$이 존재하고, 점 $A$와 직선 $y=x$ 사이의 거리가 점 $B$와 직선 $y=x$ 사이의 거리보다 크므로 삼각형 $APA'$의 넓이가 삼각형 $BPB'$의 넓이보다 크다. 
(ⅱ) $a = 3$일 때 
$AB$, $A'B'$이 서로 평행하므로 두 삼각형 $APA'$, $BPB'$이 존재하지 않는다. 
(ⅲ) $a = 5$일 때 
$APA'$, $BPB'$이 존재하고, 점 $A$와 직선 $y=x$ 사이의 거리와 점 $B$와 직선 $y=x$ 사이의 거리가 같으므로 삼각형 $APA'$의 넓이와 삼각형 $BPB'$의 넓이가 같다. 
(ⅳ) $3 < a < 4$ 또는 $4 < a < 5$일 때 
$APA'$, $BPB'$이 존재하고, 점 $A$와 직선 $y=x$ 사이의 거리가 점 $B$와 직선 $y=x$ 사이의 거리보다 작으므로 삼각형 $APA'$의 넓이가 삼각형 $BPB'$의 넓이보다 작다. 
(ⅴ) $a = 4$일 때 
$A$가 직선 $y=x$ 위의 점이므로 삼각형 $APA'$이 존재하지 않는다. 
(ⅵ) $a = -1$일 때 
$A$, $A'$, $B$, $B'$이 모두 직선 $y = -x+3$ 위에 있으므로 두 삼각형 $APA'$, $BPB'$이 존재하지 않는다.
