## 문제
정수 전체의 집합의 두 부분집합 
$A = \{ x \mid \log_{2} (x+1) \le k \}$, 
$B = \{ x \mid \log_{2} (x-2) - \log_{\frac{1}{2}} (x+1) \ge 2 \}$
에 대하여 $n(A \cap B) = 5$를 만족시키는 자연수 $k$의 값은?

## 보기
① $3$   ② $4$   ③ $5$   ④ $6$   ⑤ $7$

## 풀이
$\log_{2} (x+1) \le k$에서 진수의 조건에 의하여 $x > -1$이고, 
$x+1 \le 2^{k}$이므로 
$A = \{ x \mid -1 < x \le 2^{k}-1, x \text{는 정수} \}$ 

$\log_{2} (x-2) - \log_{\frac{1}{2}} (x+1) \ge 2$에서 
$\log_{2} (x-2) + \log_{2} (x+1) \ge 2$ 
$\log_{2} (x-2)(x+1) \ge 2$ 
진수의 조건에 의하여 $x > 2$이고, 
$(x-2)(x+1) \ge 4$이므로 
$x^{2}-x-6 \ge 0$
$(x-3)(x+2) \ge 0$
$x \ge 3$ 또는 $x \le -2$
$x > 2$와의 공통 범위는 $x \ge 3$
따라서 $B = \{ x \mid x \ge 3, x \text{는 정수} \}$ 

$n(A \cap B) = 5$이므로 $A \cap B = \{ x \mid 3 \le x \le 2^{k}-1, x \text{는 정수} \}$의 원소의 개수가 $5$개여야 한다.
원소는 $3, 4, 5, 6, 7$이므로
$2^{k}-1 = 7$ 
$2^{k} = 8$ 
따라서 자연수 $k$의 값은 $3$이다.

정답: ①
