## 문제
다항식 $(x^{2} - x)^{2} + 2x^{2} - 2x - 15$ 가 $(x^{2} + ax + b)(x^{2} + ax + c)$ 로 인수분해될 때, 세 상수 $a$, $b$, $c$ 에 대하여 $a + b + c$ 의 값은?

## 보기
① $-2$   ② $-1$   ③ $0$   ④ $1$   ⑤ $2$

## 풀이
$x^{2} - x = t$ 로 놓으면 

$$(x^{2} - x)^{2} + 2x^{2} - 2x - 15$$ 
$$= t^{2} + 2t - 15$$ 
$$= (t + 5)(t - 3)$$ 
$$= (x^{2} - x + 5)(x^{2} - x - 3)$$ 

이므로 
$a = -1$, $b = 5$, $c = -3$ 또는 
$a = -1$, $b = -3$, $c = 5$ 

$\therefore a + b + c = 1$

[정답] ④
