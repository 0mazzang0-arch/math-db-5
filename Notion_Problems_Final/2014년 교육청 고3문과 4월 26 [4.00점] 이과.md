## 문제
$x$ 에 대한 부등식 $( 3^{x + 2} - 1 ) ( 3^{x - p} - 1 ) \le 0$ 을 만족시키는 정수 $x$ 의 개수가 $20$ 일 때, 자연수 $p$ 의 값을 구하시오.

## 보기

## 풀이
주어진 부등식의 양변에 $3^{-2} \times 3^{p}$ 을 곱하면 
$( 3^{x} - 3^{-2} ) ( 3^{x} - 3^{p} ) \le 0$ 

$p$ 가 자연수이므로 
$3^{-2} \le 3^{x} \le 3^{p}$ 
$\therefore -2 \le x \le p$ 

$-2 \le x \le p$ 를 만족시키는 정수 $x$ 는 $-2$, $-1$, $0$, $1$, $\dots$, $p$ 
이때, 정수 $x$ 의 개수는 $p + 3 = 20$ 
따라서 $p = 17$

정답: $17$
