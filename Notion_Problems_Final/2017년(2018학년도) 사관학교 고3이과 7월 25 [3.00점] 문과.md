## 문제
도함수가 실수 전체의 집합에서 연속인 함수 $f(x)$가 다음 조건을 만족시킨다.

(가) 모든 실수 $x$에 대하여 $f(-x)=-f(x)$이다.
(나) $f(\pi)=0$
(다) $\int_{0}^{\pi} x^{2} f'(x) dx = -8\pi$

$\int_{-\pi}^{\pi} (x + \cos x) f(x) dx = k\pi$일 때, $k$의 값을 구하시오.

## 보기

## 풀이
$y=f(x)$의 그래프는 원점에 대하여 대칭이므로 $f(x) \cos x$는 원점에 대하여 대칭이고 $x f(x)$는 $y$축에 대하여 대칭이다. 
따라서 $\int_{-\pi}^{\pi} (x + \cos x) f(x) dx = \int_{-\pi}^{\pi} x f(x) dx + \int_{-\pi}^{\pi} f(x) \cos x dx$에서 기함수의 정적분 성질에 의해
$\int_{-\pi}^{\pi} (x + \cos x) f(x) dx = 2 \int_{0}^{\pi} x f(x) dx$이다.

부분적분법에 의해
$\int_{0}^{\pi} x^{2} f'(x) dx = [x^{2} f(x)]_{0}^{\pi} - 2 \int_{0}^{\pi} x f(x) dx$
$= \pi^{2} f(\pi) - 0^{2} f(0) - 2 \int_{0}^{\pi} x f(x) dx$
조건 (가)에 의해 $f(0)=0$이고, 조건 (나)에 의해 $f(\pi)=0$이므로
$= 0 - 2 \int_{0}^{\pi} x f(x) dx = -8\pi$
따라서 $2 \int_{0}^{\pi} x f(x) dx = 8\pi$이다.

그러므로 $\int_{-\pi}^{\pi} (x + \cos x) f(x) dx = 8\pi = k\pi$에서
$k=8$이다.
