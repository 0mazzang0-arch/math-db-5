## 문제
$2$ 이상의 자연수 $n$에 대하여 $1 \leq a < b \leq n$, $1 \leq c < d \leq n$을 만족하고, 좌표평면 위의 네 직선 $x=a$, $x=b$, $y=c$, $y=d$로 둘러싸인 직사각형의 둘레의 길이가 $2n$이 되도록 자연수 $a$, $b$, $c$, $d$를 택한다. 다음은 $b-a$의 값을 확률변수 $X$라 할 때 $E(X) = \frac{n}{2}$임을 보이는 과정이다.

$$
\begin{array}{|l|}
\hline
\text{확률변수 } X\text{가 가질 수 있는 가장 작은 값은 } 1\text{, 가장 큰 값은 } (가) \text{이다.} \\
X=k \text{ 일 때, } b-a=k \text{ 이므로 } 1 \leq a \leq n-k \text{ 이고,} \\
d-c=n-k \text{ 이므로 } 1 \leq c \leq (나) \text{ 이다.} \\
\text{그러므로 } P(X=k) = \frac{(n-k) \times (나)}{\sum_{i=1}^{(가)} (n-i) \times i} \text{ 이다.} \\
\text{따라서 } E(X) = \sum_{k=1}^{(가)} k \times P(X=k) \\
= \frac{6}{(다)} \sum_{k=1}^{(가)} k \times (n-k) \times (나) = \frac{n}{2} \\
\hline
\end{array}
$$

위의 과정에서 $(가)$, $(나)$, $(다)$에 알맞은 식을 각각 $f(n)$, $g(k)$, $h(n)$이라 할 때, $\frac{h(7)}{f(8) \times g(6)}$의 값은?

## 보기
① $4$   ② $5$   ③ $6$   ④ $7$   ⑤ $8$

## 풀이
$a$와 $b$는 $1 \leq a < b \leq n$을 만족하는 자연수이고, $b-a$의 값이 확률변수 $X$이므로 $X$가 가질 수 있는 가장 작은 값은 $1$, 가장 큰 값은 $n-1$이다.
따라서 $(가)$에 알맞은 식은 $f(n) = n-1$이다.

$X=k$일 때 $b-a=k$에서 $a=b-k$이므로 $1 \leq a \leq n-k$이다.
직사각형의 둘레의 길이가 $2n$이므로 $2(b-a) + 2(d-c) = 2n$에서 $(b-a) + (d-c) = n$이다.
$k + (d-c) = n$이므로 $d-c = n-k$이다.
이때 $c=d-(n-k)$이고 $1 \leq c < d \leq n$에서 $d \leq n$이므로 $c \leq n-(n-k)=k$이다.
자연수 $c$의 범위는 $1 \leq c \leq k$이므로 $(나)$에 알맞은 식은 $g(k) = k$이다.

$1 \leq a \leq n-k$이고 $1 \leq c \leq k$이므로 $X=k$가 되는 경우의 수는 $(n-k) \times k$이다.
전체 경우의 수는 $\sum_{i=1}^{n-1} (n-i) \times i$이므로
$P(X=k) = \frac{(n-k) \times k}{\sum_{i=1}^{n-1} (n-i) \times i}$이다.
분모를 계산하면
$$ \sum_{i=1}^{n-1} (n-i) \times i = \sum_{i=1}^{n-1} (ni - i^2) = \frac{n(n-1)(n+1)}{6} $$
$E(X)$를 구하는 과정에서
$$ E(X) = \sum_{k=1}^{n-1} k \times P(X=k) = \frac{6}{n(n-1)(n+1)} \sum_{k=1}^{n-1} k \times (n-k) \times k $$
이므로 $(다)$에 알맞은 식은 $h(n) = n(n-1)(n+1)$이다.

$f(n)=n-1$, $g(k)=k$, $h(n)=n(n-1)(n+1)$에서
$$ \frac{h(7)}{f(8) \times g(6)} = \frac{7 \times 6 \times 8}{7 \times 6} = 8 $$

정답은 ⑤이다.
