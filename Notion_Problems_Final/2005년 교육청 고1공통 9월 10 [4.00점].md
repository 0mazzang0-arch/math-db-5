## 문제
실수 $x$에 대하여 $n \leq x < n+1$을 만족하는 정수 $n$을 $x$의 정수부분이라고 한다. 다음은 $\sqrt{2} + \sqrt{6}$의 정수부분을 구하는 과정이다. (단, $a$, $b$, $c$, $d$는 정수이다.)

$x = \sqrt{2} + \sqrt{6}$으로 놓으면
$x^{2} = 8 + \sqrt{a}$
그런데, $\sqrt{a}$의 정수부분은 $b$이므로
$c \leq x^{2} < c+1$
이다. 따라서
$d \leq x < d+1$
이므로 $x = \sqrt{2} + \sqrt{6}$의 정수부분은 $d$이다.
위의 과정에서 네 정수 $a$, $b$, $c$, $d$의 합 $a+b+c+d$의 값은?

## 보기
① $65$   ② $68$   ③ $71$   ④ $74$   ⑤ $77$

## 풀이
[정답] ③ 

$x = \sqrt{2} + \sqrt{6}$으로 놓으면 
$x^{2} = 2+6+2 \sqrt{12} = 8 + \sqrt{48}$
이때 $6^{2} < 48 < 7^{2}$이므로 $6 < \sqrt{48} < 7$이다. 
따라서 $\sqrt{48}$의 정수부분은 $6$이다.
그러므로 $a = 48$, $b = 6$이다.
$x^{2} = 8 + \sqrt{48}$이고 $6 < \sqrt{48} < 7$이므로
$14 < x^{2} < 15$이다.
따라서 $c = 14$이다.
$\sqrt{14} < x < \sqrt{15}$에서 $3 < \sqrt{14} < \sqrt{15} < 4$이므로 
$3 < x < 4$이다.
따라서 $\sqrt{2} + \sqrt{6}$의 정수부분은 $3$이므로 $d = 3$이다.
$\therefore a+b+c+d = 48+6+14+3 = 71$
