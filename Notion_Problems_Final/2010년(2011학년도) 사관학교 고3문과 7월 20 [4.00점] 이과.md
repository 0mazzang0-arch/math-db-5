## 문제
세 수열 $\{ a_n \}$, $\{ b_n \}$, $\{ c_n \}$ 에 대하여 옳은 것만을 <보기>에서 있는 대로 고른 것은?

$$
\begin{array}{|l|}
\hline
ㄱ. \text{ } 0 < a_n < b_n \text{ (} n = 1, 2, 3, \dots \text{) 이고 } \lim_{n \to \infty} b_n = \infty \text{ 이면 } \lim_{n \to \infty} \frac{a_n}{b_n^2} = 0 \text{ 이다.} \\
ㄴ. \text{ 수열 } \{ a_n \} \text{ 이 발산하고 수열 } \{ a_n b_n \} \text{ 이 수렴하면 } \lim_{n \to \infty} b_n = 0 \text{ 이다.} \\
ㄷ. \text{ } a_n < b_n < c_n \text{ (} n = 1, 2, 3, \dots \text{) 이고 } \lim_{n \to \infty} (n+1) a_n = \lim_{n \to \infty} (n-1) c_n = 1 \text{ 이면 } \lim_{n \to \infty} n b_n = 1 \text{ 이다.} \\
\hline
\end{array}
$$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄱ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
정답: ④

ㄱ. $0 < a_n < b_n$ 이고, $\lim_{n \to \infty} b_n = \infty$ 이므로 
$\lim_{n \to \infty} \frac{a_n}{b_n^2} = \lim_{n \to \infty} \frac{1}{b_n} \times \frac{a_n}{b_n} = 0 \times 0 = 0$ (참) 

ㄴ. $\{ a_n \} : 1, 0, 1, 0, 1, 0, \dots$ 이고 $\{ b_n \} : 0, 1, 0, 1, 0, 1, \dots$ 이면 $\{ a_n \}$ 은 발산하고 수열 $\{ a_n b_n \}$ 은 수렴하지만 $\lim_{n \to \infty} b_n$ 은 발산한다. (거짓) 

ㄷ. $a_n < b_n < c_n$ ($n = 1, 2, 3, \dots$) 이고 $\lim_{n \to \infty} (n+1) a_n = \lim_{n \to \infty} (n-1) c_n = 1$ 이므로 
$a_n = \frac{1}{n+a}$, $c_n = \frac{1}{n+b}$ ($a > b$) 꼴이다. 
$\frac{1}{n+a} < b_n < \frac{1}{n+b}$ ($a > b$) 에서 
$\frac{n}{n+a} < n b_n < \frac{n}{n+b}$ ($a > b$) 이므로 
$\lim_{n \to \infty} n b_n = 1$ 이다. (참) 
따라서 옳은 것은 ㄱ, ㄷ이다.
