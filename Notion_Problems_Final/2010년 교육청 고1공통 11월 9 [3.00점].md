## 문제
실수 $a$, $b$, $c$, $d$에 대하여 $a > b$, $c > d$일 때, 대소 관계가 항상 성립하는 것만을 <보기>에서 있는 대로 고른 것은?

ㄱ. $ac + bd > bc + ad$
ㄴ. $ac > bd$
ㄷ. $\sqrt{a - b} + \sqrt{c - d} > \sqrt{a + c - b - d}$

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄷ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
$a > b$, $c > d$이므로 $a - b > 0$, $c - d > 0$이다.

ㄱ. $(ac + bd) - (bc + ad) = (a - b)(c - d) > 0$이므로
$ac + bd > bc + ad$이다. (참)

ㄴ. [반례] $a = 1$, $b = -2$, $c = 1$, $d = -1$이면
$a > b$, $c > d$를 만족하지만
$ac = 1$, $bd = 2$이므로 $ac < bd$이다. (거짓)

ㄷ. $(\sqrt{a - b} + \sqrt{c - d})^2 - (\sqrt{a + c - b - d})^2$
$= (a - b) + 2\sqrt{a - b}\sqrt{c - d} + (c - d) - (a + c - b - d)$
$= 2\sqrt{a - b}\sqrt{c - d} = 2\sqrt{(a - b)(c - d)} > 0$
$\sqrt{a - b} + \sqrt{c - d} > 0$, $\sqrt{a + c - b - d} > 0$이므로
$\sqrt{a - b} + \sqrt{c - d} > \sqrt{a + c - b - d}$이다. (참)

이상에서 옳은 것은 ㄱ, ㄷ이다.
