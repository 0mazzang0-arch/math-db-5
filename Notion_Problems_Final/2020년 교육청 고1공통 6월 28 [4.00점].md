## 문제
두 양수 $p$, $q$에 대하여 이차함수 $f(x) = -x^2 + px - q$가 다음 조건을 만족시킬 때, $p^2 + q^2$의 값을 구하시오.

$$ \begin{array}{|l|} \hline \text{(가) } y = f(x) \text{의 그래프는 } x\text{축에 접한다.} \\ \text{(나) } -p \le x \le p \text{에서 } f(x) \text{의 최솟값은 } -54 \text{이다.} \\ \hline \end{array} $$

## 보기

## 풀이
$f(x) = -x^2 + px - q = -(x - \frac{p}{2})^2 + \frac{p^2}{4} - q$
(가)에 의하여 함수 $f(x)$의 그래프가 $x$축에 접하므로 $\frac{p^2}{4} - q = 0$이다.
따라서 $q = \frac{p^2}{4}$, $f(x) = -(x - \frac{p}{2})^2$

$f(x)$의 꼭짓점의 $x$좌표가 $\frac{p}{2}$이므로 (나)에 의하여 $-p \le x \le p$에서 $f(x)$의 최솟값은 $f(-p)$이다.
$f(-p) = -\frac{9p^2}{4} = -54$이고 $p^2 = 24$

$q = \frac{p^2}{4}$이므로 $q = 6$
$\therefore p^2 + q^2 = 60$
