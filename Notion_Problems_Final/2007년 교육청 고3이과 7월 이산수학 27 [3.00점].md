## 문제
$0 \le x \le 4\pi$ 일 때, 삼각방정식 $\sin x + \sqrt{3} \cos x = -1$ 의 실근의 개수는?

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
$\sin x + \sqrt{3} \cos x = 2 \sin (x + \frac{\pi}{3}) = -1$ 이므로 

$\sin (x + \frac{\pi}{3}) = -\frac{1}{2}$ 의 근은 

$\frac{5}{6}\pi$, $\frac{3}{2}\pi$, $\frac{17}{6}\pi$, $\frac{7}{2}\pi$ 이고 근의 개수는 $4$ 이다.
