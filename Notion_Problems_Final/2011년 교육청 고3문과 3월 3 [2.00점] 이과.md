## 문제
$\lim_{n \to \infty} \frac{n(n + \cos n \pi)}{n^{2} + 1}$ 의 값은?

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
모든 자연수 $n$ 에 대하여 $\cos n \pi = (-1)^{n}$ 이므로
$\lim_{n \to \infty} \frac{n^{2} + n \cos n \pi}{n^{2} + 1} = \lim_{n \to \infty} \frac{1 + \frac{1}{n} \cos n \pi}{1 + \frac{1}{n^{2}}} = \lim_{n \to \infty} \frac{1 + \frac{(-1)^{n}}{n}}{1 + \frac{1}{n^{2}}} = 1$

[다른 풀이]
모든 자연수 $n$ 에 대하여 $-1 \le \cos n \pi \le 1$ 이므로
$-\frac{n}{n^{2} + 1} \le \frac{n}{n^{2} + 1} \cos n \pi \le \frac{n}{n^{2} + 1}$
그런데 $\lim_{n \to \infty} \frac{n}{n^{2} + 1} = \lim_{n \to \infty} \frac{\frac{1}{n}}{1 + \frac{1}{n^{2}}} = 0$ 이므로
$\lim_{n \to \infty} \frac{n}{n^{2} + 1} \cos n \pi = 0$
또, $\lim_{n \to \infty} \frac{n^{2}}{n^{2} + 1} = \lim_{n \to \infty} \frac{1}{1 + \frac{1}{n^{2}}} = 1$
$\therefore \lim_{n \to \infty} \frac{n(n + \cos n \pi)}{n^{2} + 1} = \lim_{n \to \infty} \frac{n^{2}}{n^{2} + 1} + \lim_{n \to \infty} \frac{n}{n^{2} + 1} \cos n \pi = 1$
