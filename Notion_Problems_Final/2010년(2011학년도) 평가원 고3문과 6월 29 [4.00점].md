## 문제
이차정사각행렬 $A$, $B$, $P$가 
$$AP=P \left( \begin{array}{cc} a & 0 \\ 0 & b \end{array} \right), \quad BP=P \left( \begin{array}{cc} c & 0 \\ 0 & d \end{array} \right)$$
를 만족시킨다. $P$가 역행렬을 가질 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?

$$\begin{array}{|l|} \hline \text{ㄱ. } a=c\text{이고 } b=d\text{이면 } A=B\text{이다.} \\ \text{ㄴ. } AB=BA \\ \text{ㄷ. } A-B\text{가 역행렬을 가지면 } a \neq c\text{이고 } b \neq d\text{이다.} \\ \hline \end{array}$$

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄱ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
[정답] ⑤ 

ㄱ. $A=P \left( \begin{array}{cc} a & 0 \\ 0 & b \end{array} \right) P^{-1}$, $B=P \left( \begin{array}{cc} c & 0 \\ 0 & d \end{array} \right) P^{-1}$이므로 $a=c$, $b=d$이면 $A=B$이다. (참)

ㄴ. $AB = P \left( \begin{array}{cc} a & 0 \\ 0 & b \end{array} \right) P^{-1} P \left( \begin{array}{cc} c & 0 \\ 0 & d \end{array} \right) P^{-1}$
$= P \left( \begin{array}{cc} a & 0 \\ 0 & b \end{array} \right) \left( \begin{array}{cc} c & 0 \\ 0 & d \end{array} \right) P^{-1}$
$= P \left( \begin{array}{cc} ac & 0 \\ 0 & bd \end{array} \right) P^{-1}$

$BA = P \left( \begin{array}{cc} c & 0 \\ 0 & d \end{array} \right) P^{-1} P \left( \begin{array}{cc} a & 0 \\ 0 & b \end{array} \right) P^{-1}$
$= P \left( \begin{array}{cc} c & 0 \\ 0 & d \end{array} \right) \left( \begin{array}{cc} a & 0 \\ 0 & b \end{array} \right) P^{-1}$
$= P \left( \begin{array}{cc} ac & 0 \\ 0 & bd \end{array} \right) P^{-1}$
$\therefore AB = BA$ (참)

ㄷ. $A-B = P \left( \begin{array}{cc} a & 0 \\ 0 & b \end{array} \right) P^{-1} - P \left( \begin{array}{cc} c & 0 \\ 0 & d \end{array} \right) P^{-1}$
$= P \left[ \left( \begin{array}{cc} a & 0 \\ 0 & b \end{array} \right) - \left( \begin{array}{cc} c & 0 \\ 0 & d \end{array} \right) \right] P^{-1}$
$= P \left( \begin{array}{cc} a-c & 0 \\ 0 & b-d \end{array} \right) P^{-1}$
에서 $A-B$의 역행렬이 존재하므로 $(a-c)(b-d) \neq 0$이다.
따라서 $a \neq c$이고 $b \neq d$이다. (참)

따라서 옳은 것은 ㄱ, ㄴ, ㄷ이다.
