## 문제
집합 $X = \{ 1, 2, 3, 4, 5, 6 \}$에 대하여 다음 조건을 만족시키는 함수 $f : X \to X$의 개수는?

(가) $2 f ( 1 ) + 2 f ( 2 ) + f ( 6 ) = f ( 3 ) + 16$
(나) $f ( 3 ) \le f ( 4 ) \le f ( 5 ) \le f ( 6 )$

## 보기
① $288$   ② $300$   ③ $312$   ④ $324$   ⑤ $336$

## 풀이
조건 (가)에서 
$f ( 6 ) - f ( 3 ) = 2 [ 8 - \{ f ( 1 ) + f ( 2 ) \} ] \cdots \cdots$ ㉠ 

조건 (나)에서 
$f ( 6 ) - f ( 3 ) \ge 0$ 이고 
$f ( 3 )$, $f ( 6 )$ 은 각각 $6$ 이하의 자연수이므로 
$f ( 6 ) - f ( 3 ) = 0$ 또는 $f ( 6 ) - f ( 3 ) = 2$ 또는 $f ( 6 ) - f ( 3 ) = 4$ 

(ⅰ) $f ( 6 ) - f ( 3 ) = 0$ 인 경우 
$f ( 3 )$, $f ( 6 )$ 의 값을 정하는 경우의 수는 $6$ 
$f ( 1 ) + f ( 2 ) = 8$ 이고, 
$f ( 1 )$, $f ( 2 )$ 의 값을 정하는 경우의 수는 $5$ 
$f ( 3 ) \le f ( 4 ) \le f ( 5 ) \le f ( 3 )$ 이므로
$f ( 4 )$, $f ( 5 )$ 의 값을 정하는 경우의 수는 $1$ 
$6 \times 5 \times 1 = 30$ 

(ⅱ) $f ( 6 ) - f ( 3 ) = 2$ 인 경우 
$f ( 3 )$, $f ( 6 )$ 의 값을 정하는 경우의 수는 $4$ 
$f ( 1 ) + f ( 2 ) = 7$ 이고, 
$f ( 1 )$, $f ( 2 )$ 의 값을 정하는 경우의 수는 $6$ 
$f ( 3 ) \le f ( 4 ) \le f ( 5 ) \le f ( 3 ) + 2$ 
$f ( 4 )$, $f ( 5 )$ 의 값을 정하는 경우의 수는 서로 다른 세 수 $f ( 3 )$, $f ( 3 ) + 1$, $f ( 3 ) + 2$ 중에서 $2$ 개를 택하는 중복조합의 수와 같으므로 
${}_3H_2 = {}_4C_2 = 6$ 
$4 \times 6 \times 6 = 144$ 

(ⅲ) $f ( 6 ) - f ( 3 ) = 4$ 인 경우 
$f ( 3 )$, $f ( 6 )$ 의 값을 정하는 경우의 수는 $2$ 
$f ( 1 ) + f ( 2 ) = 6$ 이고, 
$f ( 1 )$, $f ( 2 )$ 의 값을 정하는 경우의 수는 $5$ 
$f ( 3 ) \le f ( 4 ) \le f ( 5 ) \le f ( 3 ) + 4$ 
$f ( 4 )$, $f ( 5 )$ 의 값을 정하는 경우의 수는 서로 다른 다섯 수 $f ( 3 )$, $f ( 3 ) + 1$, $f ( 3 ) + 2$, $f ( 3 ) + 3$, $f ( 3 ) + 4$ 중에서 $2$ 개를 택하는 중복조합의 수와 같으므로 
${}_5H_2 = {}_6C_2 = 15$ 
$2 \times 5 \times 15 = 150$ 

(ⅰ), (ⅱ), (ⅲ)에 의하여 구하는 경우의 수는 
$30 + 144 + 150 = 324$

정답: ④
