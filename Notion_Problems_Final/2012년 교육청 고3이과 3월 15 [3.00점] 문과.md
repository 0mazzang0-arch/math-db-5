## 문제
열린구간 $( 0, \frac{\pi}{2} )$에서 정의된 미분가능한 함수 $f(x)$는 다음 조건을 만족시킨다.

(가) $f'(x) = 1 + \{ f(x) \}^{2}$
(나) $f( \frac{\pi}{4} ) = 1$

함수 $g(x) = \ln f'(x)$에 대하여 $g'( \frac{\pi}{4} )$의 값은?

## 보기
① $1$   ② $\frac{3}{2}$   ③ $2$   ④ $\frac{5}{2}$   ⑤ $3$

## 풀이
[정답] ③

$g(x) = \ln f'(x)$
$= \ln ( 1 + \{ f(x) \}^{2} )$에서
$g'(x) = \frac{2f(x)f'(x)}{1 + \{ f(x) \}^{2}}$
$= \frac{2f(x) [ 1 + \{ f(x) \}^{2} ]}{1 + \{ f(x) \}^{2}}$
$= 2f(x)$

따라서
$g'( \frac{\pi}{4} ) = 2f( \frac{\pi}{4} ) = 2 \cdot 1 = 2$

[다른풀이]
$g(x) = \ln f'(x)$이므로
$g'(x) = \frac{f''(x)}{f'(x)} = \frac{2f(x)f'(x)}{f'(x)} = 2f(x)$

따라서
$g'( \frac{\pi}{4} ) = 2f( \frac{\pi}{4} ) = 2 \cdot 1 = 2$
