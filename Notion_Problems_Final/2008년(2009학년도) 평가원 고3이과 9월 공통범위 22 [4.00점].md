## 문제
수열 $\{ a_{n} \}$ 의 제 $n$ 항 $a_{n}$ 을 자연수 $k$ 의 양의 제곱근 $\sqrt{k}$ 를 소수점 아래 첫째 자리에서 반올림하여 $n$ 이 되는 $k$ 의 개수라 하자. $\sum_{i=1}^{10} a_{i}$ 의 값을 구하시오.

## 보기


## 풀이
$\sqrt{k}$ 를 소수점 아래 첫째 자리에서 반올림하여 $n$ 이므로 
$n - \frac{1}{2} \leq \sqrt{k} < n + \frac{1}{2}$
$n^{2} - n + \frac{1}{4} \leq k < n^{2} + n + \frac{1}{4}$

$n$ 과 $k$ 가 자연수이므로
$n^{2} - n + 1 \leq k \leq n^{2} + n \cdots$ ㉠

㉠을 만족하는 자연수 $k$ 의 개수는
$a_{n} = (n^{2} + n) - (n^{2} - n + 1) + 1 = 2n$

$\therefore \sum_{i=1}^{10} a_{i} = \sum_{i=1}^{10} 2i = 2 \times \frac{10 \times 11}{2} = 110$

정답: $110$
