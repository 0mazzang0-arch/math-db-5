## 문제

두 양수 $a$, $b$ ($b \neq 1$)가 다음 조건을 만족시킬 때, $a^{2} + b^{2}$의 값은?

(가) $(\log_{2} a)(\log_{b} 3) = 0$
(나) $\log_{2} a + \log_{b} 3 = 2$

## 보기
① $3$   ② $4$   ③ $5$   ④ $6$   ⑤ $7$

## 풀이
[정답] ② 

조건 (가)에서 $(\log_{2} a)(\log_{b} 3) = 0$이므로 $\log_{2} a = 0$ 또는 $\log_{b} 3 = 0$이다. 
$\log_{b} 3 = 0$을 만족시키는 $b$는 존재하지 않는다. 
따라서 $\log_{2} a = 0$이므로 $a = 1$이다. 

조건 (나)에서 $\log_{b} 3 = 2$이므로 $b^{2} = 3$이다. 
$a^{2} + b^{2} = 1 + 3 = 4$이다. 

[다른 풀이] 

$\log_{2} a$와 $\log_{b} 3$을 두 근으로 하고 최고차항의 계수가 $1$인 이차방정식은 
$x^{2} - (\log_{2} a + \log_{b} 3)x + (\log_{2} a)(\log_{b} 3) = 0$이다. 
$x^{2} - 2x = 0$ 
$\therefore x = 0, 2$ 

따라서 $\log_{2} a = 0, \log_{b} 3 = 2$ 또는 $\log_{2} a = 2, \log_{b} 3 = 0$이다. 
$\log_{b} 3 = 0$을 만족시키는 $b$는 존재하지 않으므로 
$\log_{2} a = 0, \log_{b} 3 = 2$이다. 

그러므로 $a = 1, b^{2} = 3$이고 $a^{2} + b^{2} = 4$이다.
