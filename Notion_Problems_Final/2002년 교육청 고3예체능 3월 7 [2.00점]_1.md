## 문제
$-1 \le x \le 3$ 에서 정의된 함수 $f(x)=-2 x^{2}+8 x+a$ 의 최댓값이 $5$ 가 되도록 상수 $a$ 의 값을 정할 때, 함수 $f(x)$ 의 최솟값은?

## 보기
① $-11$   ② $-12$   ③ $-13$   ④ $-14$   ⑤ $-15$

## 풀이
$f(x) = -2 x^{2}+8 x+a = -2(x-2)^{2}+a+8$ 이므로 최댓값은 
$f(2) = a+8=5$
$\therefore a = -3$
이때, $-1 \le x \le 3$ 에서 최솟값은 꼭짓점 $x=2$ 에서 가장 멀리 떨어진 $x=-1$ 일 때 갖는다.
$f(-1) = -2-8+a = -10-3 = -13$

정답: ③
