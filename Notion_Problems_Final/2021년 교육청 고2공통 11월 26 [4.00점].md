## 문제
$0 \le x < 2\pi$ 에서 $x$ 에 대한 부등식 
$( 2a+6 ) \cos x - a \sin^2 x + a + 12 < 0$ 
의 해가 존재하도록 하는 자연수 $a$ 의 최솟값을 구하시오.

## 보기


## 풀이
$(2a+6) \cos x - a \sin^2 x + a + 12 < 0$ 
$(2a+6) \cos x - a(1 - \cos^2 x) + a + 12 < 0$ 
$a \cos^2 x + (2a+6) \cos x + 12 < 0$ 
$( a \cos x + 6 ) ( \cos x + 2 ) < 0$ 에서 
$\cos x + 2 > 0$ 이므로 
$a \cos x + 6 < 0$ 
$a > 0$ 이므로 
$\cos x < -\frac{6}{a}$ 
$0 \le x < 2\pi$ 에서 부등식 $\cos x < -\frac{6}{a}$ 의 해가 존재하기 위해서는 
$-\frac{6}{a} > -1$ 이어야 한다. 
따라서 $a > 6$ 이며 자연수 $a$ 의 최솟값은 $7$ 이다.

정답: $7$
