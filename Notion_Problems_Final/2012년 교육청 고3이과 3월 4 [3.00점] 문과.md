## 문제
$0 < x < \frac{\pi}{2}$ 에서 함수 $f(x) = \frac{(\cos x + \sin x)^2}{\sin 2x}$ 의 최솟값은?

## 보기
① $1$   ② $\frac{3}{2}$   ③ $2$   ④ $\frac{5}{2}$   ⑤ $3$

## 풀이
$f(x) = \frac{(\cos x + \sin x)^2}{\sin 2x}$
$= \frac{\sin^2 x + \cos^2 x + 2 \sin x \cos x}{\sin 2x}$
$= \frac{1 + \sin 2x}{\sin 2x}$
$= 1 + \frac{1}{\sin 2x}$

$0 < x < \frac{\pi}{2}$ 에서 $0 < 2x < \pi$ 이므로 $0 < \sin 2x \leq 1$ 이다.
따라서 $f(x)$ 는 $\sin 2x = 1$ 일 때 최솟값 $1 + \frac{1}{1} = 2$ 를 갖는다.

정답: ③
