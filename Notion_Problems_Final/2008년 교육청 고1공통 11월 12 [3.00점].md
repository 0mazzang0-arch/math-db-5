## 문제
두 집합 $A = \{ x | x^{2} - ax + b \le 0 \}$, $B = \{ x | x^{2} + x - a > 0 \}$ 에 대하여 $A \cap B = \{ x | 1 < x \le 3 \}$ 일 때, $A \cup B$ 는?

## 보기
① $\{ x | -1 < x \le 3 \}$   ② $\{ x | -2 \le x \le 3 \}$   ③ $\{ x | x \le -1 \text{ 또는 } x > 1 \}$   ④ $\{ x | x < -2 \text{ 또는 } x \ge 1 \}$   ⑤ $\{ x | x < -2 \text{ 또는 } x \ge -1 \}$

## 풀이
$A \cap B = \{ x | 1 < x \le 3 \}$ 이므로 $x^{2} + x - a = 0$ 의 한 근이 $1$ 이고, $x^{2} - ax + b = 0$ 의 한 근이 $3$ 이다.
따라서 $a = 2$, $b = -3$ 이다.

$A = \{ x | x^{2} - 2x - 3 \le 0 \} = \{ x | -1 \le x \le 3 \}$
$B = \{ x | x^{2} + x - 2 > 0 \} = \{ x | x < -2 \text{ 또는 } x > 1 \}$

$\therefore A \cup B = \{ x | x < -2 \text{ 또는 } x \ge -1 \}$

정답: ⑤
