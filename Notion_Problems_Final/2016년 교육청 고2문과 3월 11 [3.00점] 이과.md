## 문제
실수 $x$에 대하여 두 조건 $p$, $q$가 다음과 같다.
$p: x^{2}-2x-3 \le 0$
$q: |x-a| \le b$
$p$는 $q$이기 위한 필요충분조건일 때, $ab$의 값은? (단, $a$, $b$는 상수이다.)

## 보기
① $-2$   ② $-1$   ③ $0$   ④ $1$   ⑤ $2$

## 풀이
두 조건 $p$, $q$의 진리집합을 각각 $P$, $Q$라 하자.

$p: x^{2}-2x-3 \le 0$에서
$(x-3)(x+1) \le 0$
$-1 \le x \le 3$
$\therefore P = \{ x | -1 \le x \le 3 \}$

$q: |x-a| \le b$에서
$-b \le x-a \le b$
$a-b \le x \le a+b$
$\therefore Q = \{ x | a-b \le x \le a+b \}$

$p$는 $q$이기 위한 필요충분조건이므로 $P=Q$이다.
따라서 $a-b = -1$, $a+b = 3$이므로 두 식을 연립하면
$a = 1$, $b = 2$
$\therefore ab = 2$

정답: ⑤
