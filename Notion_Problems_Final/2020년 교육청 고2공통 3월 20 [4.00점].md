## 문제
$x$ 에 대한 사차방정식 
$$ x^{4} + (3-2a)x^{2} + a^{2}-3a-10 = 0 $$
이 실근과 허근을 모두 가질 때, 이 사차방정식에 대하여 <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $a$ 는 실수이다.)

## 보기
ㄱ. $a=1$ 이면 모든 실근의 곱은 $-3$ 이다.  
ㄴ. 모든 실근의 곱이 $-4$ 이면 모든 허근의 곱은 $3$ 이다.  
ㄷ. 정수인 근을 갖도록 하는 모든 실수 $a$ 의 값의 합은 $-1$ 이다.  

① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
**정답: ⑤**

ㄱ. $x^{4} + (3-2a)x^{2} + a^{2}-3a-10=0$ 에서 $a=1$ 이면 
$x^{4} + x^{2} - 12 = 0$
$(x^{2} - 3)(x^{2} + 4) = 0$
$(x + \sqrt{3})(x - \sqrt{3})(x + 2i)(x - 2i) = 0$
$x = -\sqrt{3}$ 또는 $x = \sqrt{3}$ 또는 $x = -2i$ 또는 $x = 2i$
따라서 실근은 $x = -\sqrt{3}$ 또는 $x = \sqrt{3}$ 이므로 모든 실근의 곱은 $(-\sqrt{3}) \times \sqrt{3} = -3$ 이다. (참)

ㄴ. $x^{4} + (3-2a)x^{2} + a^{2}-3a-10=0$ 에서
$x^{4} + (3-2a)x^{2} + (a-5)(a+2) = 0$
$(x^{2} - (a-5))(x^{2} - (a+2)) = 0$
$x^{2} = a-5$ 또는 $x^{2} = a+2$
$x^{2} = a-5$ 에서 $a-5 \ge 0$ 이면 실근을 갖고 $a-5 < 0$ 이면 허근을 갖는다.
$x^{2} = a+2$ 에서 $a+2 \ge 0$ 이면 실근을 갖고 $a+2 < 0$ 이면 허근을 갖는다.
사차방정식이 실근과 허근을 모두 가지려면 $a+2 \ge 0$ 이고 $a-5 < 0$ 이어야 하므로 
$-2 \le a < 5$
이때 방정식의 실근은 $x^{2} = a+2$ 에서 $x = \pm \sqrt{a+2}$ 이고, 
허근은 $x^{2} = a-5$ 에서 $x = \pm \sqrt{5-a}i$ 이다.
모든 실근의 곱이 $-4$ 이려면
$(-\sqrt{a+2}) \times \sqrt{a+2} = -4$
$a+2 = 4$, 즉 $a = 2$
$a = 2$ 일 때 방정식의 허근은 $x^{2} = 2-5 = -3$ 에서 $x = \pm \sqrt{3}i$ 이다.
따라서 모든 허근의 곱은 $(-\sqrt{3}i) \times (\sqrt{3}i) = 3$ 이다. (참)

ㄷ. ㄴ에서 $-2 \le a < 5$ 이고 $0 \le \sqrt{a+2} < \sqrt{7}$ 이다.
사차방정식이 정수인 근을 가지려면 $x^{2} = a+2$ 의 근이 정수여야 한다. (허근은 정수가 될 수 없다.)
따라서 $\sqrt{a+2}$ 의 값이 $0, 1, 2$ 가 되어야 한다.
$\sqrt{a+2} = 0 \implies a = -2$
$\sqrt{a+2} = 1 \implies a = -1$
$\sqrt{a+2} = 2 \implies a = 2$
만족하는 모든 실수 $a$ 의 값의 합은 $(-2) + (-1) + 2 = -1$ 이다. (참)

이상에서 옳은 것은 ㄱ, ㄴ, ㄷ 이다.
