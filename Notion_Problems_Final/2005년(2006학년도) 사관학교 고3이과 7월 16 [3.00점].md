## 문제
자연수 $n$ 에 대하여 $f(n)= \left[ \frac{n}{4} \right]$ 이라 하고, 수열 $\{ a_{n} \}$ 을 $a_{1}=a_{2}=a_{3}=0$, $a_{n}= \sum_{k=1}^{f(n)} k$ ($n=4, 5, 6, \cdots$) 으로 정의할 때, $\sum_{n=1}^{28} a_{n}$ 의 값은? (단, 실수 $x$ 에 대하여 $[x]$ 는 $x$ 보다 크지 않은 최대의 정수이다.)

## 보기
① $204$   ② $212$   ③ $220$   ④ $224$   ⑤ $252$

## 풀이
$a_{n}= \sum_{k=1}^{f(n)} k = \frac{f(n) \{ f(n)+1 \}}{2} = \frac{\left[ \frac{n}{4} \right] \left( \left[ \frac{n}{4} \right]+1 \right)}{2}$ 이다.

(i) $1 \leq n < 4$ 일 때, $\left[ \frac{n}{4} \right]=0$ 이므로 $a_{n}=0$
(ii) $4 \leq n < 8$ 일 때, $\left[ \frac{n}{4} \right]=1$ 이므로 $a_{n}=1$
(iii) $8 \leq n < 12$ 일 때, $\left[ \frac{n}{4} \right]=2$ 이므로 $a_{n}=3$
(iv) $12 \leq n < 16$ 일 때, $\left[ \frac{n}{4} \right]=3$ 이므로 $a_{n}=6$
(v) $16 \leq n < 20$ 일 때, $\left[ \frac{n}{4} \right]=4$ 이므로 $a_{n}=10$
(vi) $20 \leq n < 24$ 일 때, $\left[ \frac{n}{4} \right]=5$ 이므로 $a_{n}=15$
(vii) $24 \leq n < 28$ 일 때, $\left[ \frac{n}{4} \right]=6$ 이므로 $a_{n}=21$
(viii) $n=28$ 일 때, $\left[ \frac{n}{4} \right]=7$ 이므로 $a_{n}=28$

따라서 구하는 값은 다음과 같다.
$$
\begin{array}{l}
\sum_{n=1}^{28} a_{n} = (a_{1}+a_{2}+a_{3}) + (a_{4}+a_{5}+a_{6}+a_{7}) + \cdots + (a_{24}+a_{25}+a_{26}+a_{27}) + a_{28} \\
= 3 \times 0 + 4 \times (1+3+6+10+15+21) + 28 \\
= 4 \times 56 + 28 \\
= 224 + 28 \\
= 252
\end{array}
$$

정답: ⑤
