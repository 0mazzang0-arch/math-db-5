## 문제

다음 조건을 만족시키는 자연수 $n$의 개수는? (단, $[x]$는 $x$보다 크지 않은 최대의 정수이다.)

(가) $[\log_{3} n] = 3$
(나) $[\log n^{2}] = [\log 2n] + 2$

## 보기

① $12$   ② $14$   ③ $16$   ④ $18$   ⑤ $20$

## 풀이

$[\log_{3} n] = 3$에서 $3^{3} \leq n < 3^{4}$이므로 $27 \leq n < 81$이다.
이 범위에서 $[\log 2n] = 1$ 또는 $[\log 2n] = 2$이다.

(i) $[\log 2n] = 1$일 때
즉, $27 \leq n < 50$일 때, (나)에서 $[\log n^{2}] = 3$이므로
$[2 \log n] = 3$
$3 \leq 2 \log n < 4$
$\frac{3}{2} \leq \log n < 2$
$\therefore \log 10\sqrt{10} \leq \log n < \log 100$
$10\sqrt{10} \leq n < 100$
$10\sqrt{10} \approx 31.62\dots$ 이고 주어진 범위가 $27 \leq n < 50$이므로
$32 \leq n < 50$
따라서 자연수 $n$의 개수는 $50 - 32 = 18$이다.

(ii) $[\log 2n] = 2$일 때
즉, $50 \leq n < 81$일 때, (나)에서 $[\log n^{2}] = 4$이므로
$[2 \log n] = 4$
$4 \leq 2 \log n < 5$
$2 \leq \log n < \frac{5}{2}$
$\therefore \log 100 \leq \log n < \log 100\sqrt{10}$
$100 \leq n < 100\sqrt{10}$
주어진 범위가 $50 \leq n < 81$이므로 이를 만족시키는 자연수 $n$의 값은 존재하지 않는다.

(i), (ii)에서 자연수 $n$의 개수는 $18$이다.

**[다른풀이]**

$[\log_{3} n] = 3$에서 $27 \leq n < 81 \dots$ ㉠
㉠에서 $27 \leq n < 32$일 때, $n^{2}$은 세 자리의 정수이므로 $[\log n^{2}] = 2$이다.
$32 \leq n < 81$일 때, $n^{2}$은 네 자리의 정수이므로 $[\log n^{2}] = 3$이다.
또, $27 \leq n < 50$일 때, $2n$은 두 자리의 정수이므로 $[\log 2n] = 1$이다.
$50 \leq n < 81$일 때, $2n$은 세 자리의 정수이므로 $[\log 2n] = 2$이다.

(나) $[\log n^{2}] = [\log 2n] + 2$를 만족시키려면
$[\log n^{2}] = 3$이고 $[\log 2n] = 1$이어야 하므로
$32 \leq n < 50$이다.
따라서 구하는 자연수 $n$의 개수는 $50 - 32 = 18$이다.
