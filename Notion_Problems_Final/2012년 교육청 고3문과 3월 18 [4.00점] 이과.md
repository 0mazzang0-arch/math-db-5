## 문제
두 이차정사각행렬 $A$, $B$는 다음 조건을 만족시킨다.

(가) $B A + B = E$
(나) $A^{2} B = A + E$

<보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $E$는 단위 행렬이다.)

---
**<보기>**
ㄱ. 행렬 $B$의 역행렬이 존재한다.
ㄴ. $A B = B A$
ㄷ. 행렬 $A B$의 모든 성분의 합은 $-2$이다.

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
ㄱ. 조건 (가)에서 $B(A + E) = E$ 이므로 행렬 $B$의 역행렬은 $A + E$이다. 따라서 행렬 $B$의 역행렬이 존재한다. (참)

ㄴ. ㄱ에서 $B^{-1} = A + E$ 이므로 $A = B^{-1} - E$이다.
$A B = (B^{-1} - E) B = E - B$
$B A = B (B^{-1} - E) = E - B$
따라서 $A B = B A$이다. (참)

ㄷ. $A B = B A$ 이므로 조건 (나)에서
$A^{2} B = A(A B) = A(B A) = A(E - B) = A - A B = A + E$
$A - A B = A + E$ 이므로 $-A B = E$
즉, $A B = -E$이다.
$A, B$는 이차정사각행렬이므로 $E = \begin{pmatrix} 1 & 0 \\ 0 & 1 \end{pmatrix}$이다.
$A B = -E = \begin{pmatrix} -1 & 0 \\ 0 & -1 \end{pmatrix}$
따라서 행렬 $A B$의 모든 성분의 합은 $(-1) + 0 + 0 + (-1) = -2$이다. (참)

이상에서 옳은 것은 ㄱ, ㄴ, ㄷ이다.
