## 문제
다항함수 $y=f(x)$의 도함수 $y=f'(x)$의 그래프가 그림과 같을 때, 옳은 것만을 <보기>에서 있는 대로 고른 것은?

> [!example] 📸 (그림/그래프)

$$
\begin{array}{|l|}
\hline
\text{ㄱ. 구간 } [a, f] \text{에서 } f(x) \text{의 변곡점은 } 4 \text{개이다.} \\
\text{ㄴ. 구간 } [a, e] \text{에서 } f(x) \text{가 극대가 되는 } x \text{의 개수는 } 1 \text{개이다.} \\
\text{ㄷ. 구간 } [a, e] \text{에서 } f(x) \text{의 최댓값은 } f(c) \text{이다.} \\
\hline
\end{array}
$$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
[정답] ③

ㄱ. 함수 $f(x)$는 원점 $O$와 $x=b$, $c$, $e$에서 변곡점을 가진다. ($참$)
ㄴ. $f(x)$는 $x=d$에서 극대가 된다. ($참$)
ㄷ. 구간 $[a, e]$에서 $f(x)$는 $x=d$에서 최댓값 $f(d)$를 가진다. 즉, 최댓값은 $f(d)$이다. ($거짓$)

따라서 옳은 것은 ㄱ, ㄴ이다.
