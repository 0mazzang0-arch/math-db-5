## 문제
유리함수 $f(x) = \frac{8x}{2x-15}$ 와 수열 $\{a_{n}\}$ 에 대하여 $a_{n} = f(n)$ 이다. $\sum_{n=1}^{m} a_{n} \le 73$ 을 만족시키는 자연수 $m$ 의 최댓값을 구하시오.

## 풀이
곡선 $f(x) = 4 + \frac{60}{2x-15}$ 의 그래프는 점 $(\frac{15}{2}, 4)$ 에 대하여 대칭이다.

곡선 $y = f(x)$ 의 그래프는 점 $(\frac{15}{2}, 4)$ 에 대하여 대칭이므로 다음이 성립한다.
$f(7) + f(8) = 8$
$f(6) + f(9) = 8$
$f(5) + f(10) = 8$
$f(4) + f(11) = 8$
$f(3) + f(12) = 8$
$f(2) + f(13) = 8$
$f(1) + f(14) = 8$

위의 관계식으로부터
$\sum_{n=1}^{14} a_{n} = f(1) + f(2) + f(3) + \cdots + f(14) = 56$
또한,
$a_{15} = f(15) = 8$
$a_{16} = f(16) = 4 + \frac{60}{2 \times 16 - 15} = 7 + \frac{9}{17} < 8$
$a_{17} = f(17) = 4 + \frac{60}{2 \times 17 - 15} = 4 + \frac{60}{19} > 4$ 이므로

$\sum_{n=1}^{16} a_{n} = 56 + 8 + (7 + \frac{9}{17}) = 71 + \frac{9}{17} < 73$
$\sum_{n=1}^{17} a_{n} = 71 + \frac{9}{17} + (4 + \frac{60}{19}) > 73$ 이다.

따라서 이를 만족시키는 자연수 $m$ 의 최댓값은 $16$ 이다.
