## 문제
두 다항식 $P$, $Q$의 최대공약수를 $P \triangle Q$, 최소공배수를 $P \nabla Q$로 나타내기로 하자. 세 다항식 
$A = x^{2} - 4x + 3$
$B = x^{2} - x - 6$
$C = x^{3} + 4x^{2} + x - a$
에 대하여 $(A \triangle B) \nabla C$가 $x$에 대한 삼차식일 때, 자연수 $a$의 값을 구하시오.

## 보기

## 풀이
두 다항식 $A$와 $B$를 인수분해하면 
$A = (x - 1)(x - 3)$
$B = (x + 2)(x - 3)$
이므로 $A$와 $B$의 최대공약수인 $A \triangle B$는 
$A \triangle B = x - 3$이고 
$(A \triangle B) \nabla C = (x - 3) \nabla (x^{3} + 4x^{2} + x - a)$
가 된다. 

$(A \triangle B) \nabla C$는 $x - 3$과 $x^{3} + 4x^{2} + x - a$의 최소공배수이고, 이 최소공배수가 $x$에 대한 삼차식이므로 $x - 3$은 $x^{3} + 4x^{2} + x - a$의 인수이다. 

$f(x) = x^{3} + 4x^{2} + x - a$라 할 때, 인수정리에 의해 $f(3) = 0$이므로 
$f(3) = 3^{3} + 4 \times 3^{2} + 3 - a = 0$
따라서 $a = 27 + 36 + 3 = 66$

[정답] $66$
