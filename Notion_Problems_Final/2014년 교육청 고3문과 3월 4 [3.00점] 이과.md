## 문제
등차수열 $\{ a_{n} \}$ 에 대하여 $a_{3} = 10$, $a_{4} - a_{2} = 4$ 일 때, $a_{8}$ 의 값은?

## 보기
① $18$   ② $20$   ③ $22$   ④ $24$   ⑤ $26$

## 풀이
수열 $\{ a_{n} \}$ 의 첫째항을 $a$, 공차를 $d$ 라 하면 
$a_{3} = 10$ 에서 $a + 2 d = 10$ $\cdots$ ㉠ 

$a_{4} - a_{2} = 4$ 에서 
$(a + 3 d) - (a + d) = 4$
$2 d = 4$
$\therefore d = 2$ 

㉠에 대입하면 $a = 6$ 이다. 

$\therefore a_{8} = a + 7 d = 6 + 7 \times 2 = 20$

정답: ②
