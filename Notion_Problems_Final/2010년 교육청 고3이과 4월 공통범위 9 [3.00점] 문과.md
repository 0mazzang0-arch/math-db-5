## 문제
다음 두 조건을 모두 만족시키는 모든 양의 실수 $x$의 곱은? (단, $[x]$는 $x$보다 크지 않은 최대의 정수이다.)

(가) $[\log x] = [\log 365]$
(나) $\log x^3 - [\log x^3] = \log \frac{1}{x} - [\log \frac{1}{x}]$

## 보기
① $10^{9}$   ② $10^{\frac{19}{2}}$   ③ $10^{10}$   ④ $10^{\frac{21}{2}}$   ⑤ $10^{11}$

## 풀이
(가) $[\log x] = [\log 365] = 2$에서 $2 \le \log x < 3$이다.
(나) $\log x^3$과 $\log \frac{1}{x}$의 소수부분이 같으므로 $\log x^3 - \log \frac{1}{x} = 4 \log x$는 정수이다.

$2 \le \log x < 3$이므로 $8 \le 4 \log x < 12$이다. 
$4 \log x$는 정수이므로 $4 \log x = 8, 9, 10, 11$이다.
따라서, $x = 10^{2}, 10^{\frac{9}{4}}, 10^{\frac{5}{2}}, 10^{\frac{11}{4}}$이므로 모든 양의 실수 $x$의 곱은
$10^{2 + \frac{9}{4} + \frac{5}{2} + \frac{11}{4}} = 10^{\frac{19}{2}}$이다.

정답: ②
