## 문제
다음은 모든 자연수 $n$에 대하여
$$\sum_{k=1}^{2n} (-1)^{k-1} \frac{1}{k} = \sum_{k=1}^{n} \frac{1}{n+k} \cdots\cdots (\ast)$$
이 성립함을 수학적 귀납법으로 증명한 것이다.

$(\ast)$에서 $S_{n} = \sum_{k=1}^{2n} (-1)^{k-1} \frac{1}{k}$, $T_{n} = \sum_{k=1}^{n} \frac{1}{n+k}$이라 하자.
(ⅰ) $n=1$ 일 때,
$S_{1} = (가) = T_{1}$ 이므로 $(\ast)$이 성립한다.
(ⅱ) $n=m$ 일 때, $(\ast)$이 성립한다고 가정하면 $S_{m} = T_{m}$이다.
$n=m+1$ 일 때, $(\ast)$이 성립함을 보이자.
$S_{m+1} = S_{m} + \frac{1}{2m+1} + (나)$,
$T_{m+1} = T_{m} + (다) + \frac{1}{2m+1} + \frac{1}{2m+2}$

$S_{m+1} - T_{m+1} = S_{m} - T_{m}$ 이고, $S_{m} = T_{m}$ 이므로 $S_{m+1} = T_{m+1}$ 이다.
$n=m+1$ 일 때도 $(\ast)$이 성립한다.
(ⅰ), (ⅱ)에 의하여 모든 자연수 $n$에 대하여 $(\ast)$이 성립한다.

위의 (가)에 알맞은 수를 $a$라 하고, (나), (다)에 알맞은 식을 각각 $f(m)$, $g(m)$이라 할 때, $a+ \frac{g(5)}{f(14)}$의 값은?

## 보기
① $\frac{7}{2}$   ② $\frac{9}{2}$   ③ $\frac{11}{2}$   ④ $\frac{13}{2}$   ⑤ $\frac{15}{2}$

## 풀이
(ⅰ) $n=1$ 일 때,
$S_{1} = \sum_{k=1}^{2} (-1)^{k-1} \frac{1}{k} = 1 - \frac{1}{2} = \frac{1}{2}$
$T_{1} = \sum_{k=1}^{1} \frac{1}{1+k} = \frac{1}{2}$
이므로 $a = \frac{1}{2}$이다.

(ⅱ) $n=m+1$ 일 때,
$S_{m+1} = \sum_{k=1}^{2m+2} (-1)^{k-1} \frac{1}{k} = S_{m} + (-1)^{2m} \frac{1}{2m+1} + (-1)^{2m+1} \frac{1}{2m+2}$
$= S_{m} + \frac{1}{2m+1} - \frac{1}{2m+2}$
이므로 $f(m) = -\frac{1}{2m+2}$이다.

$T_{m+1} = \sum_{k=1}^{m+1} \frac{1}{m+1+k} = \frac{1}{m+2} + \frac{1}{m+3} + \cdots + \frac{1}{2m} + \frac{1}{2m+1} + \frac{1}{2m+2}$
$T_{m} = \frac{1}{m+1} + \frac{1}{m+2} + \cdots + \frac{1}{2m}$ 이므로
$T_{m+1} = T_{m} - \frac{1}{m+1} + \frac{1}{2m+1} + \frac{1}{2m+2}$
이므로 $g(m) = -\frac{1}{m+1}$이다.

따라서 $a = \frac{1}{2}$, $f(m) = -\frac{1}{2m+2}$, $g(m) = -\frac{1}{m+1}$일 때,
$f(14) = -\frac{1}{2 \cdot 14 + 2} = -\frac{1}{30}$
$g(5) = -\frac{1}{5+1} = -\frac{1}{6}$
$a + \frac{g(5)}{f(14)} = \frac{1}{2} + \frac{- \frac{1}{6}}{- \frac{1}{30}} = \frac{1}{2} + 5 = \frac{11}{2}$

정답은 ③이다.
