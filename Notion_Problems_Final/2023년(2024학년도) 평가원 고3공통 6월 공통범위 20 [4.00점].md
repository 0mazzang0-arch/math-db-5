## 문제
최고차항의 계수가 `$1$` 인 이차함수 `$f(x)$` 에 대하여 함수 
`$$ g(x) = \int_{0}^{x} f(t) dt $$`
가 다음 조건을 만족시킬 때, `$f(9)$` 의 값을 구하시오.

`$x \ge 1$` 인 모든 실수 `$x$` 에 대하여 
`$g(x) \ge g(4)$` 이고 `$|g(x)| \ge |g(3)|$` 이다.

## 보기

## 풀이
[정답] `$39$`

`$x \ge 1$` 인 모든 실수 `$x$` 에 대하여 
`$g(x) \ge g(4) \cdots \cdots$ ㉠`
`$|g(x)| \ge |g(3)| \cdots \cdots$ ㉡`
가 성립한다.

(ⅰ) `$g(4) \ge 0$` 일 때,
`$x=4$` 일 때 `$g(4) \ge |g(3)|$`
`$x=3$` 일 때 `$g(3) \ge g(4)$`
`$\therefore |g(3)| \le g(4) \le g(3)$`
`$g(4) \ge 0$`, `$g(3) \ge 0$` 이므로 `$g(3) = g(4)$`
`$x \ge 1$` 인 모든 실수 `$x$` 에 대하여 `$g(x) \ge g(3) = g(4)$` 이므로 함수 `$g(x)$` 는 `$x=3$` 과 `$x=4$` 모두에서 극솟값을 가져야 한다. 그러나 `$g(x)$` 는 삼차함수이고 극솟값은 많아야 하나이므로 모순이다.

(ⅱ) `$g(4) < 0$` 일 때,
`$x \to \infty$` 에 따라 `$g(x) \to \infty$` 이므로 사잇값 정리에 의해 `$x \ge 1$` 인 범위에서 `$g(\alpha) = 0$` 인 `$\alpha$` 가 존재한다.
`$|g(3)| \le |g(\alpha)| = 0$` 이므로 `$g(3) = 0$` 이다.
`$g(0) = 0$` 이고, `$g(x)$` 는 최고차항의 계수가 `$\frac{1}{3}$` 인 삼차함수이므로
`$g(x) = \frac{1}{3} x (x-3) (x-a)$` (`$a$` 는 상수)
`$= \frac{1}{3} x^{3} - \frac{3+a}{3} x^{2} + ax$`

㉠에서 함수 `$g(x)$` 는 `$x=4$` 에서 극소이므로 `$g'(4) = 0$`
`$g'(x) = x^{2} - \frac{2}{3} (3+a) x + a$` 에서
`$g'(4) = 16 - \frac{8}{3} (3+a) + a = 8 - \frac{5}{3} a = 0$`
`$\therefore a = \frac{24}{5}$`, `$g'(x) = x^{2} - \frac{26}{5} x + \frac{24}{5}$`
`$g'(x) = \frac{1}{5} (5x-6) (x-4)$` 이므로
`$f(9) = g'(9) = \frac{1}{5} \times 39 \times 5 = 39$`
