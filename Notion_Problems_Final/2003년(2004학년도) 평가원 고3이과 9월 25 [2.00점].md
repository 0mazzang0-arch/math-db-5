## 문제
$\log _{2} x + \log _{4} \frac{1}{x} = 5$ 일 때, $\log _{4} x - \log _{2} \frac{1}{x}$ 의 값을 구하시오.

## 풀이
$\log _{2} x + \log _{4} \frac{1}{x} = \log _{2} x - \log _{2} x^{\frac{1}{2}}$
$= \log _{2} x^{\frac{1}{2}}$
$= \frac{1}{2} \log _{2} x$
$= 5$
$\therefore x = 2^{10} \cdots$ ㉠

$\therefore \log _{4} x - \log _{2} \frac{1}{x} = \log _{2} x^{\frac{1}{2}} + \log _{2} x$
$= \log _{2} x^{\frac{3}{2}}$
$= \log _{2} (2^{10})^{\frac{3}{2}}$
$= \log _{2} 2^{15}$ (∵ ㉠)
$= 15$

정답: $15$
