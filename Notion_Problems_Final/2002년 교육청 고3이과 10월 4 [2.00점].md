## 문제
$A = \{ x \mid \frac{1}{x+2} + \frac{1}{x-3} \geq 0 \}$, $B = \{ x \mid x^{2} - 2x - 8 \leq 0 \}$ 에 대하여 집합 $A \cap B$ 에 속하는 모든 정수의 합은?

## 보기
① $3$   ② $4$   ③ $5$   ④ $6$   ⑤ $7$

## 풀이
$A: \frac{1}{x+2} + \frac{1}{x-3} \geq 0$ 에서
$\frac{2x-1}{(x+2)(x-3)} \geq 0$
$(x+2)(2x-1)(x-3) \geq 0$ (단, $x \neq -2$, $x \neq 3$)
$\therefore -2 < x \leq \frac{1}{2}$ 또는 $x > 3$

$B: x^{2} - 2x - 8 \leq 0$ 에서
$(x+2)(x-4) \leq 0$
$\therefore -2 \leq x \leq 4$

따라서 $A \cap B = \{ x \mid -2 < x \leq \frac{1}{2}$ 또는 $3 < x \leq 4 \}$ 이다.
집합 $A \cap B$ 에 속하는 정수는 $-1$, $0$, $4$ 이므로 모든 정수의 합은 $3$ 이다.
