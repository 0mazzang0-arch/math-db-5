## 문제
다항식 $f(x) = x^2 + px + q$ ($p$, $q$는 실수)가 다음 두 조건을 만족시킨다.

(가) 다항식 $f(x)$를 $x-1$로 나눈 나머지는 $1$이다.
(나) 실수 $a$에 대하여 이차방정식 $f(x) = 0$의 한 근은 $a+i$이다.

$p+2q$의 값은? (단, $i = \sqrt{-1}$)

## 보기
① $2$   ② $4$   ③ $6$   ④ $8$   ⑤ $10$

## 풀이
조건 (가)에서 
$f(1) = 1 + p + q = 1$
$\therefore p + q = 0 \cdots$ ㉠

조건 (나)에서 계수가 실수인 이차방정식 $x^2 + px + q = 0$의 한 근이 $a + i$이므로 다른 한 근은 $a - i$이다. 따라서 근과 계수의 관계에 의하여
$-p = (a + i) + (a - i) = 2a$
$\therefore p = -2a \cdots$ ㉡
$q = (a + i)(a - i) = a^2 + 1 \cdots$ ㉢

㉡, ㉢에서 $p + q = a^2 - 2a + 1$이므로 ㉠에 의하여
$p + q = a^2 - 2a + 1 = 0$, $(a - 1)^2 = 0$
$\therefore a = 1$, $p = -2$, $q = 2$ ($\because$ ㉡, ㉢)

따라서 $p + 2q = -2 + 2(2) = 2$

정답: ①
