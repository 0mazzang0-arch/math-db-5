## 문제
어느 지역의 학생들의 $1$일 인터넷 사용시간 $X$는 평균이 $m$분, 표준편차가 $30$분인 정규분포를 따른다. 이 지역 학생들을 대상으로 $9$명을 임의로 추출하여 조사한 $1$일 인터넷 사용시간의 표본평균을 $\bar{X}$라 하자. 함수 $G(k), H(k)$를
$$G(k) = P(X \leq m + 30k), H(k) = P(\bar{X} \geq m - 30k)$$
라 할 때, 옳은 것만을 <보기>에서 있는 대로 고른 것은?

$$
\begin{array}{|l|}
\hline
\text{<보 기>} \\
ㄱ. \ G(0) = H(0) \\
ㄴ. \ G(3) = H(1) \\
ㄷ. \ G(1) + H(-1) = 1 \\
\hline
\end{array}
$$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄱ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
$X$는 정규분포 $N(m, 30^2)$을 따르므로 크기가 $9$인 표본평균 $\bar{X}$에 대하여 
$E(\bar{X}) = m$, $\sigma(\bar{X}) = \frac{30}{\sqrt{9}} = 10$이다.
따라서, $\bar{X}$는 정규분포 $N(m, 10^2)$을 따른다.

$G(k) = P(X \leq m + 30k) = P\left(Z \leq \frac{m + 30k - m}{30}\right) = P(Z \leq k)$
$H(k) = P(\bar{X} \geq m - 30k) = P\left(Z \geq \frac{m - 30k - m}{10}\right) = P(Z \geq -3k)$

ㄱ. $G(0) = P(Z \leq 0) = 0.5$, $H(0) = P(Z \geq 0) = 0.5$
$\therefore G(0) = H(0)$ (참)

ㄴ. $G(3) = P(Z \leq 3) = 0.5 + P(0 \leq Z \leq 3)$
$H(1) = P(Z \geq -3) = P(-3 \leq Z \leq 0) + 0.5 = P(0 \leq Z \leq 3) + 0.5$
$\therefore G(3) = H(1)$ (참)

ㄷ. $G(1) = P(Z \leq 1)$, $H(-1) = P(Z \geq 3)$이므로
$G(1) + H(-1) = P(Z \leq 1) + P(Z \geq 3) = 1 - P(1 \leq Z \leq 3)$
$P(1 \leq Z \leq 3) > 0$이므로 $G(1) + H(-1) < 1$ (거짓)

따라서 보기 중 옳은 것은 ㄱ, ㄴ이다.
