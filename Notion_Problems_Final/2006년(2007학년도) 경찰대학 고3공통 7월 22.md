## 문제
$1$보다 큰 서로 다른 세 자연수 $x$, $y$, $z$가 $\frac{1}{x} + \frac{1}{y} + \frac{1}{z} > 1$을 만족할 때, $\frac{1}{x} + \frac{1}{y} + \frac{1}{z}$의 최솟값은?

## 보기
① $\frac{6}{5}$   ② $\frac{7}{6}$   ③ $\frac{13}{12}$   ④ $\frac{31}{30}$   ⑤ $\frac{37}{36}$

## 풀이
$x < y < z$라 하면 
$1 < \frac{1}{x} + \frac{1}{y} + \frac{1}{z} < \frac{3}{x} \implies 1 < \frac{3}{x} \implies x < 3$

$x$, $y$, $z$가 $1$보다 큰 자연수이므로 
$\therefore x = 2$

또한, 
$\frac{1}{2} + \frac{1}{y} + \frac{1}{z} > 1$에서
$\frac{1}{2} < \frac{1}{y} + \frac{1}{z} < \frac{2}{y} \implies \frac{2}{y} > \frac{1}{2} \implies y < 4$

$x < y$이므로 
$\therefore y = 3$

또한, 
$\frac{1}{2} + \frac{1}{3} + \frac{1}{z} > 1 \implies \frac{1}{z} > \frac{1}{6} \implies z < 6$

$y < z$이므로 
$\therefore z = 4$ 또는 $5$

따라서, $\frac{1}{x} + \frac{1}{y} + \frac{1}{z}$의 최솟값은 $z = 5$일 때인
$\frac{1}{2} + \frac{1}{3} + \frac{1}{5} = \frac{31}{30}$

정답은 ④이다.
