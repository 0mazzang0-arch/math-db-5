## 문제
세 양수 $a$, $b$, $c$가 다음 조건을 만족시킨다. 

$$ \begin{array}{l} 
\text{(가) } \frac{1}{a} - \frac{1}{c} < 0 \\ 
\text{(나) } \frac{1}{a} + \frac{1}{c} < \frac{1}{b} 
\end{array} $$

$a$, $b$, $c$의 대소 관계를 바르게 나타낸 것은? 

## 보기
① $a < b < c$   ② $b < a < c$   ③ $b < c < a$   ④ $c < a < b$   ⑤ $c < b < a$

## 풀이
조건 (가)에서 $a$, $c$는 양수이므로 
$\frac{1}{a} < \frac{1}{c}$ 
$\therefore c < a$ 

조건 (나)에서 $\frac{1}{a} > 0$이고 $\frac{1}{a} + \frac{1}{c} < \frac{1}{b}$이므로 
$\frac{1}{c} < \frac{1}{a} + \frac{1}{c} < \frac{1}{b}$ 
$\therefore \frac{1}{c} < \frac{1}{b}$ 

따라서 $b$, $c$가 양수이므로 $b < c$에서 $b < c < a$ 

[다른 풀이] 
조건 (가)에서 $ac > 0$이고, $\frac{1}{a} - \frac{1}{c} < 0$이므로 
$c - a = ac \left( \frac{1}{a} - \frac{1}{c} \right) < 0$ 
$\therefore c < a \cdots \cdots$ ㉠ 

조건 (나)에서 $bc > 0$이고 $0 < \frac{1}{a} < \frac{1}{b} - \frac{1}{c}$이므로 
$c - b = bc \left( \frac{1}{b} - \frac{1}{c} \right) > 0$ 
$\therefore c > b \cdots \cdots$ ㉡ 

㉠, ㉡에서 $b < c < a$ 

정답: ③
