## 문제
$1 < a < 4$ 인 실수 $a$ 에 대하여 함수 $y = \log_{a} x$ 의 그래프와 함수 $y = \frac{1}{x}$ 의 그래프가 만나는 점을 $A(p, q)$ 라 할 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?



> [!example] 📸 (그림/그래프)

## 보기
$$ \begin{array}{|l|} \hline \text{ㄱ. } pq = 1 \\ \text{ㄴ. } a = 2 \text{ 일 때, } p > \sqrt{2} \text{ 이다.} \\ \text{ㄷ. 원점 } O \text{ 와 점 } B(p + q, 0) \text{ 에 대하여 삼각형 } AOB \text{ 의 넓이를 } S(p) \text{ 라 할 때, } \\ \quad S(p) < \frac{a + 1}{2a} \text{ 이다.} \\ \hline \end{array} $$

① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
$f(x) = \frac{1}{x}$, $g(x) = \log_{a} x$ 라 하자.

ㄱ. 점 $A(p, q)$ 는 함수 $y = f(x)$ 의 그래프 위의 점이므로 $q = \frac{1}{p}$, 즉 $pq = 1$ (참)

ㄴ. $f(x) > g(x)$ 이면 $0 < x < p$ 이고 $f(x) < g(x)$ 이면 $x > p$ 이다.
$f(\sqrt{2}) = \frac{1}{\sqrt{2}}$ 이고 $a = 2$ 이므로 $g(\sqrt{2}) = \frac{1}{2}$ 이다.
$f(\sqrt{2}) > g(\sqrt{2})$ 이다.
따라서 $p > \sqrt{2}$ (참)

ㄷ. 점 $B(p + q, 0)$ 에 대하여 삼각형 $AOB$ 의 넓이 $S(p)$ 는
$S(p) = \frac{1}{2} \times (p + q) \times q = \frac{pq}{2} + \frac{q^{2}}{2} = \frac{1}{2} + \frac{1}{2p^{2}}$ 이다.
한편, $f(\sqrt{a}) = \frac{1}{\sqrt{a}}$ 이고 $g(\sqrt{a}) = \frac{1}{2}$ 이다.
$1 < \sqrt{a} < 2$ 이므로 $f(\sqrt{a}) > g(\sqrt{a})$ 이다.
따라서 $p > \sqrt{a}$ 이고, $\frac{1}{p} < \frac{1}{\sqrt{a}}$ 이다.
그러므로 $S(p) = \frac{1}{2} + \frac{1}{2p^{2}} < \frac{1}{2} + \frac{1}{2a} = \frac{a + 1}{2a}$ (참)
