## 문제
$x > 0$에서 미분가능한 함수 $f(x)$가 다음 조건을 만족시킨다.

(가) $f ( \frac{\pi}{2} ) = 1$
(나) $f(x) + x f'(x) = x \cos x$

$f(\pi)$의 값은?

## 보기
① $-\frac{2}{\pi}$   ② $-\frac{1}{\pi}$   ③ $0$   ④ $\frac{1}{\pi}$   ⑤ $\frac{2}{\pi}$

## 풀이
$(x f(x))' = f(x) + x f'(x) = x \cos x$에서
$\int (x f(x))' dx = \int x \cos x dx$
$x f(x) = x \sin x + \cos x + C$ (단, $C$는 적분상수)

조건 (가)에서 $f( \frac{\pi}{2} ) = 1$이므로
$\frac{\pi}{2} f( \frac{\pi}{2} ) = \frac{\pi}{2} \sin \frac{\pi}{2} + \cos \frac{\pi}{2} + C$
$\frac{\pi}{2} = \frac{\pi}{2} + C$이므로 $C = 0$

따라서 $x f(x) = x \sin x + \cos x$이다.
$x = \pi$를 대입하면
$\pi f(\pi) = \pi \sin \pi + \cos \pi$
$\pi f(\pi) = -1$
$f(\pi) = -\frac{1}{\pi}$

정답: ②
