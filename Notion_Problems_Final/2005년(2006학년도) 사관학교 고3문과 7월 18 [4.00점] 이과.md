## 문제
$1$보다 큰 세 실수 $a$, $b$, $c$에 대하여 두 등식 
$$\begin{cases} a^{2} b^{3} = 64 \\ 3 ( \log _{a} c ) ^{2} -2 ( \log _{b} c ) ^{2} =-( \log _{a} c ) ( \log _{b} c ) \end{cases}$$ 
이 성립하도록 하는 두 수 $a$와 $b$에 대하여 $\log _{2} ab$의 값은?

## 보기
① $1$   ② $\frac{3}{2}$   ③ $2$   ④ $\frac{5}{2}$   ⑤ $3$

## 풀이
$a^2 b^3 = 64$의 양변에 밑이 $2$인 로그를 취하면 
$2 \log_2 a + 3 \log_2 b = 6 \dots\dots (1)$ 
또한, 주어진 식 $3 ( \log _{a} c ) ^{2} + ( \log _{a} c ) ( \log _{b} c ) - 2 ( \log _{b} c ) ^{2} = 0$을 정리하면 
$(3\log_a c - 2 \log_b c )(\log_a c + \log_b c ) = 0$ 
여기서 $a, b, c$는 모두 $1$보다 큰 실수이므로 
$\log_a c + \log_b c > 0$ 
따라서, $3\log_a c = 2 \log_b c$ 
$3 \frac{\log_2 c}{\log_2 a} = 2 \frac{\log_2 c}{\log_2 b}$ 
$2\log_2 a = 3\log_2 b \dots\dots (2)$ 
식 $(1)$과 $(2)$로부터 $\log_2 b = 1$, $\log_2 a = \frac{3}{2}$이다. 따라서, 
$\log_2 ab = \log_2 a + \log_2 b = \frac{3}{2} + 1 = \frac{5}{2}$
