## 문제
수열 $\{ a_{n} \}$ 이 
$$\sum_{k=1}^{n} k a_{k} = n ( n+1 ) ( n+2 )$$
를 만족시킬 때, $\sum_{k=1}^{10} a_{k}$ 의 값은?

## 보기
① $185$   ② $195$   ③ $205$   ④ $215$   ⑤ $225$

## 풀이
$n a_{n} = \sum_{k=1}^{n} k a_{k} - \sum_{k=1}^{n-1} k a_{k}$ ($n \ge 2$)
$= n ( n+1 ) ( n+2 ) - ( n-1 ) n ( n+1 )$
$= 3 n ( n+1 )$

$\therefore a_{n} = 3 ( n+1 )$ ($n \ge 2$)

$\sum_{k=1}^{1} k a_{k} = a_{1} = 1 \times 2 \times 3 = 6 = 3 ( 1+1 )$
이므로 모든 자연수 $n$ 에 대하여 $a_{n} = 3 ( n+1 )$ 이다.

$\therefore \sum_{k=1}^{10} 3 ( k+1 ) = 3 ( \frac{10 \times 11}{2} + 10 ) = 3 \times 65 = 195$

정답은 ② 이다.
