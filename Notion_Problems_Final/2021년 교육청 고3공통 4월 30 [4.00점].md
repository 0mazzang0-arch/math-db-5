## 문제

다음 조건을 만족시키는 $14$ 이하의 네 자연수 $x_1, x_2, x_3, x_4$ 의 모든 순서쌍 $(x_1, x_2, x_3, x_4)$ 의 개수를 구하시오.

(가) $x_1 + x_2 + x_3 + x_4 = 34$
(나) $x_1$ 과 $x_3$ 은 홀수이고 $x_2$ 와 $x_4$ 는 짝수이다.

## 풀이

$6$ 이하의 음이 아닌 네 정수 $y_1, y_2, y_3, y_4$ 에 대하여
$x_1 = 2y_1 + 1$, $x_2 = 2y_2 + 2$, $x_3 = 2y_3 + 1$, $x_4 = 2y_4 + 2$ 라 하면
$x_1 + x_2 + x_3 + x_4 = 34$ 에서
$(2y_1 + 1) + (2y_2 + 2) + (2y_3 + 1) + (2y_4 + 2) = 34$
$y_1 + y_2 + y_3 + y_4 = 14 \dots$ ㉠

구하는 모든 순서쌍 $(x_1, x_2, x_3, x_4)$ 의 개수는 방정식 $y_1 + y_2 + y_3 + y_4 = 14$ 를 만족시키는 음이 아닌 네 정수 $y_1, y_2, y_3, y_4$ 의 모든 순서쌍 $(y_1, y_2, y_3, y_4)$ 에서 $y_k \ge 7$ 인 $4$ 이하의 자연수 $k$ 가 존재하는 순서쌍 $(y_1, y_2, y_3, y_4)$ 를 제외한 개수와 같다.

(ⅰ) 방정식 $y_1 + y_2 + y_3 + y_4 = 14$ 를 만족시키는 $(y_1, y_2, y_3, y_4)$ 의 개수는 $4$ 개의 변수에서 중복을 허락하여 $14$ 개를 택하는 중복조합의 수와 같으므로
${}_4H_{14} = {}_{17}C_{14} = {}_{17}C_3 = \frac{17 \times 16 \times 15}{3 \times 2 \times 1} = 680$

(ⅱ) $y_k \ge 7$ 인 $k$ 의 값이 $1$ 개인 경우
$y_1 \ge 7$ 이라 하자. $z_1 = y_1 - 7$ 이라 하면 방정식 ㉠은
$z_1 + y_2 + y_3 + y_4 = 7$
이를 만족시키는 $z_1, y_2, y_3, y_4$ 의 모든 $(z_1, y_2, y_3, y_4)$ 의 개수는 $4$ 개의 변수에서 중복을 허락하여 $7$ 개를 택하는 중복조합의 수와 같으므로
${}_4H_7 = {}_{10}C_3 = 120$
이 중 $y_k \ge 7$ 인 $k \in \{2, 3, 4\}$ 가 존재하는 경우는 $(z_1, y_2, y_3, y_4)$ 가 $(0, 7, 0, 0), (0, 0, 7, 0), (0, 0, 0, 7)$ 인 $3$ 가지이므로
$120 - 3 = 117$
따라서 $y_k \ge 7$ 인 $k$ 의 값이 $1$ 개인 $(y_1, y_2, y_3, y_4)$ 의 개수는
$4 \times 117 = 468$

(ⅲ) $y_k \ge 7$ 인 $k$ 의 값이 $2$ 개인 경우
$(y_1, y_2, y_3, y_4)$ 는
$(7, 7, 0, 0), (7, 0, 7, 0), (7, 0, 0, 7), (0, 7, 7, 0), (0, 7, 0, 7), (0, 0, 7, 7)$
의 $6$ 가지이다.

(ⅰ), (ⅱ), (ⅲ)에 의해 구하는 모든 순서쌍 $(x_1, x_2, x_3, x_4)$ 의 개수는
$680 - (468 + 6) = 206$
