## 문제
등비수열 $\{ a_{n} \}$ 의 첫째항부터 제 $n$ 항까지의 합 $S_{n}$ 에 대하여 $S_{3} = 21$ , $S_{6} = 189$ 일 때, $a_{5}$ 의 값은?

## 보기
① $45$   ② $48$   ③ $51$   ④ $54$   ⑤ $57$

## 풀이
[정답] ②

[해설]
첫째항이 $a$ 이고 공비가 $r$ 인 등비수열에서 $S_{3} = \frac{a(1-r^{3})}{1-r} = 21$ 이므로 
$S_{6} = \frac{a(1-r^{6})}{1-r} = \frac{a(1-r^{3})(1+r^{3})}{1-r} = 21(1+r^{3}) = 189$ 이다. 
$1+r^{3} = 9$ 이므로 $r^{3} = 8$ 이고 $r = 2$ 이다. 
$S_{3} = \frac{a(1-2^{3})}{1-2} = 7a = 21$ 이므로 $a = 3$ 이고 
$a_{5} = ar^{4} = 3 \times 2^{4} = 48$ 이다.

[다른 풀이]
$S_{3} = a+ar+ar^{2} = 21$ 
$S_{6} = a+ar+ar^{2} + r^{3}(a+ar+ar^{2}) = (1+r^{3})(a+ar+ar^{2}) = 21(1+r^{3})$ 
$S_{6} = 189$ 이므로 $21(1+r^{3}) = 189$ 에서 $r^{3} = 8$ 이고 $r = 2$ 이다. 
$S_{3} = a+2a+2^{2}a = 7a = 21$ 이므로 $a = 3$ 이다. 
$a_{5} = ar^{4} = 3 \times 2^{4} = 48$ 이다.
