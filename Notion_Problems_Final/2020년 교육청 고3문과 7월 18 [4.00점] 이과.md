## 문제

확률변수 $X$는 정규분포 $N(m_{1}, \sigma_{1}^{2})$, 확률변수 $Y$는 정규분포 $N(m_{2}, \sigma_{2}^{2})$를 따르고, 확률변수 $X$, $Y$의 확률밀도함수는 각각 $f(x)$, $g(x)$이다. $\sigma_{1} = \sigma_{2}$이고 $f(24) = g(28)$일 때, 확률변수 $X$, $Y$는 다음 조건을 만족시킨다.

(가) $P(m_{1} \le X \le 24) + P(28 \le Y \le m_{2}) = 0.9544$
(나) $P(Y \ge 36) = 1 - P(X \le 24)$

$P(18 \le X \le 21)$의 값을 다음 표준정규분포표를 이용하여 구한 것은?

$$
\begin{array}{c|c}
z & P(0 \le Z \le z) \\
\hline
0.5 & 0.1915 \\
1.0 & 0.3413 \\
1.5 & 0.4332 \\
2.0 & 0.4772
\end{array}
$$



> [!example] 📸 (그림/그래프)

## 보기

① $0.3830$   ② $0.5328$   ③ $0.6247$   ④ $0.6826$   ⑤ $0.7745$

## 풀이

표준편차가 같은 정규분포곡선의 모양은 항상 일정하다. 확률변수 $X$, $Y$는 표준편차가 같은 정규분포를 따르고, 조건 (가)에 의하여 $m_{1} < 24 < 28 < m_{2}$, $f(24) = g(28)$인 확률밀도함수 $f(x)$, $g(x)$의 그래프의 개형은 다음과 같다.

![그래프](https://latex.codecogs.com/png.latex?%5Ctext%7B%EA%B7%B8%EB%9E%98%ED%94%84%20%EC%83%9D%EB%9E%B5%7D)

$\sigma_{1} = \sigma_{2} = \sigma$라 하자.
$P(m_{1} \le X \le 24) = P(28 \le Y \le m_{2}) = \frac{0.9544}{2} = 0.4772$

$P(0 \le Z \le \frac{24 - m_{1}}{\sigma}) = P(\frac{28 - m_{2}}{\sigma} \le Z \le 0) = 0.4772$

$P(0 \le Z \le 2) = P(-2 \le Z \le 0) = 0.4772$ 이므로
$\frac{24 - m_{1}}{\sigma} = 2$, $\frac{28 - m_{2}}{\sigma} = -2$
$24 - m_{1} = 2\sigma$, $m_{2} - 28 = 2\sigma$ $\cdots$ ㉠

조건 (나)에 의하여
$P(Y \ge 36) = 1 - P(X \le 24)$
$P(X \le 24) = P(Z \le 2)$이므로
$P(Y \ge 36) = 1 - P(Z \le 2) = P(Z \ge 2)$

![그래프2](https://latex.codecogs.com/png.latex?%5Ctext%7B%EA%B7%B8%EB%9E%98%ED%94%842%20%EC%83%9D%EB%9E%B5%7D)

따라서 $\frac{36 - m_{2}}{\sigma} = 2$, 즉 $36 - m_{2} = 2\sigma$ $\cdots$ ㉡

㉠, ㉡에 의하여
$m_{2} - 28 = 36 - m_{2}$
$2m_{2} = 64 \implies m_{2} = 32$

이를 ㉡에 대입하면 $36 - 32 = 2\sigma \implies \sigma = 2$
이를 ㉠에 대입하면 $24 - m_{1} = 2(2) \implies m_{1} = 20$

$\therefore P(18 \le X \le 21)$
$= P(\frac{18 - 20}{2} \le Z \le \frac{21 - 20}{2})$
$= P(-1 \le Z \le 0.5)$
$= P(0 \le Z \le 1) + P(0 \le Z \le 0.5)$
$= 0.3413 + 0.1915$
$= 0.5328$
