## 문제
이차부등식 $x^{2} - 2x - 8 < 0$의 해와 부등식 $|x - a| < b$의 해가 서로 같을 때, 두 상수 $a$, $b$의 곱 $ab$의 값은?

## 보기
① $-5$   ② $-3$   ③ $-1$   ④ $1$   ⑤ $3$

## 풀이
$x^{2} - 2x - 8 < 0 \iff (x + 2)(x - 4) < 0 \iff -2 < x < 4$

$|x - a| < b \iff -b < x - a < b \iff a - b < x < a + b$

이때, 
$a - b = -2$, 
$a + b = 4$
$\therefore a = 1$, $b = 3$

따라서 
$ab = 3$

정답: ⑤
