## 문제
두 집합 $A = \{ x | x^{2}+3x-4 < 0 \}$, $B = \{ x | x^{2}+ax+b < 0 \}$에 대하여 $A \cup B = \{ x | x^{2}+x-12 < 0 \}$, $A \cap B = \{ x | x^{2}-1 < 0 \}$일 때, 상수 $a$, $b$의 합 $a+b$의 값은?

## 보기
① $-5$   ② $-3$   ③ $0$   ④ $3$   ⑤ $5$

## 풀이
두 집합 $A$, $B$를 정리하면 다음과 같다.
$A = \{ x | x^{2}+3x-4 < 0 \} = \{ x | (x+4)(x-1) < 0 \} = \{ x | -4 < x < 1 \}$
$A \cup B = \{ x | x^{2}+x-12 < 0 \} = \{ x | (x+4)(x-3) < 0 \} = \{ x | -4 < x < 3 \}$
$A \cap B = \{ x | x^{2}-1 < 0 \} = \{ x | (x+1)(x-1) < 0 \} = \{ x | -1 < x < 1 \}$

수직선을 이용하여 두 집합 $A$와 $B$의 관계를 나타내면 다음과 같다.

> [!example] 📸 (그림/그래프)

위 조건들을 만족하는 집합 $B$의 범위는 다음과 같다.
$B = \{ x | x^{2}+ax+b < 0 \} = \{ x | -1 < x < 3 \}$

이차부등식의 해가 $-1 < x < 3$이므로 다음과 같이 식을 세울 수 있다.
$x^{2}+ax+b = (x+1)(x-3) = x^{2}-2x-3$

따라서 $a = -2$, $b = -3$이므로
$a+b = -5$이다.
