## 문제

다음 조건을 만족시키는 두 자연수 $a$, $b$의 모든 순서쌍 ($a$, $b$)의 개수를 구하시오.

(가) $1 \le a \le 10$, $1 \le b \le 100$
(나) 곡선 $y = 2^{x}$이 원 $(x - a)^{2} + (y - b)^{2} = 1$과 만나지 않는다.
(다) 곡선 $y = 2^{x}$이 원 $(x - a)^{2} + (y - b)^{2} = 4$와 적어도 한 점에서 만난다.

## 보기

## 풀이

정답: $196$

원 ㉠: $(x - a)^{2} + (y - b)^{2} = 1 = 2^{0}$
원 ㉡: $(x - a)^{2} + (y - b)^{2} = 4 = 2^{2}$

곡선 $y = 2^{x}$이 원 ㉠과는 만나지 않고 원 ㉡과 적어도 한 점에서 만나기 위한 $b$의 범위는 다음과 같다.

$$
\begin{array}{l}
a=1 \text{ 일 때, } 2^{2} < b \le 2^{3} \\
a=2 \text{ 일 때, } 2^{0} < b \le 2^{1} \text{ 또는 } 2^{3} < b \le 2^{4} \\
a=3 \text{ 일 때, } 2^{1} < b \le 2^{2} \text{ 또는 } 2^{4} < b \le 2^{5} \\
a=4 \text{ 일 때, } 2^{2} < b \le 2^{3} \text{ 또는 } 2^{5} < b \le 2^{6} \\
a=5 \text{ 일 때, } 2^{3} < b \le 2^{4} \text{ 또는 } 2^{6} < b \le 100 \\
a=6 \text{ 일 때, } 2^{4} \le b < 2^{5} \\
a=7 \text{ 일 때, } 2^{5} \le b < 2^{6} \\
a=8 \text{ 일 때, } 2^{6} \le b < 100
\end{array}
$$

따라서 구하는 순서쌍 ($a$, $b$)의 개수는
$2^{2} + (2^{0} + 2^{3}) + (2^{1} + 2^{4}) + (2^{2} + 2^{5}) + (2^{3} + 36) + 2^{4} + 2^{5} + 37$
$= (2^{0} + 2^{1} + 2^{2} + \cdots + 2^{5}) + 37 + (2^{2} + 2^{3} + 2^{4} + 2^{5}) + 36$
$= \frac{1(2^{6} - 1)}{2 - 1} + \frac{2^{2}(2^{4} - 1)}{2 - 1} + 73$
$= 63 + 60 + 73 = 196$
