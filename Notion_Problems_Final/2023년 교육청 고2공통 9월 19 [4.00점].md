## 문제
수열 $\{ a_{n} \}$이 다음 조건을 만족시킨다.

(가) 네 수 $a_{1}$, $a_{3}$, $a_{5}$, $a_{7}$은 이 순서대로 공비가 양수인 등비수열을 이룬다.
(나) $8$ 이하의 모든 자연수 $n$에 대하여 $a_{n} \times a_{9-n} = 75$이다.

$a_{1} + a_{2} = \frac{10}{3}$, $\sum_{k=1}^{8} a_{k} = \frac{400}{3}$일 때, $a_{3} + a_{8}$의 값은?

## 보기
① $\frac{110}{3}$   ② $40$   ③ $\frac{130}{3}$   ④ $\frac{140}{3}$   ⑤ $50$

## 풀이
조건 (가), (나)에 의해 상수 $a$ ($a \neq 0$), $r$ ($r > 0$)에 대하여 
$a_{1} = a$, $a_{3} = ar$, $a_{5} = ar^{2}$, $a_{7} = ar^{3}$라 하자.

조건 (나)에 의해 $n$의 값을 대입하면 다음과 같다.
$a_{2} = \frac{75}{a_{7}} = \frac{75}{ar^{3}}$
$a_{4} = \frac{75}{a_{5}} = \frac{75}{ar^{2}}$
$a_{6} = \frac{75}{a_{3}} = \frac{75}{ar}$
$a_{8} = \frac{75}{a_{1}} = \frac{75}{a}$

$\sum_{k=1}^{8} a_{k}$를 묶어서 정리하면
$\sum_{k=1}^{8} a_{k} = (a_{1} + a_{3} + a_{5} + a_{7}) + (a_{2} + a_{4} + a_{6} + a_{8})$
$= (a + ar + ar^{2} + ar^{3}) + \left( \frac{75}{ar^{3}} + \frac{75}{ar^{2}} + \frac{75}{ar} + \frac{75}{a} \right)$
$= a(1 + r + r^{2} + r^{3}) + \frac{75}{ar^{3}}(1 + r + r^{2} + r^{3})$
$= \left( a + \frac{75}{ar^{3}} \right)(1 + r + r^{2} + r^{3})$
$= (a_{1} + a_{2})(1 + r + r^{2} + r^{3})$

$a_{1} + a_{2} = \frac{10}{3}$, $\sum_{k=1}^{8} a_{k} = \frac{400}{3}$이므로 
$\frac{10}{3}(1 + r + r^{2} + r^{3}) = \frac{400}{3}$
$1 + r + r^{2} + r^{3} = 40$
$r^{3} + r^{2} + r - 39 = 0$
$(r - 3)(r^{2} + 4r + 13) = 0$

$r$는 양의 실수이므로 $r = 3$이다.

$a_{1} + a_{2} = a + \frac{75}{ar^{3}} = a + \frac{75}{27a} = \frac{10}{3}$
$\frac{27a^{2} + 75}{27a} = \frac{10}{3}$
$27a^{2} + 75 = 90a$
$9a^{2} - 30a + 25 = 0$
$(3a - 5)^{2} = 0$
$a = \frac{5}{3}$

따라서 $a_{3} + a_{8}$의 값은
$a_{3} + a_{8} = ar + \frac{75}{a} = \left( \frac{5}{3} \times 3 \right) + \left( 75 \times \frac{3}{5} \right) = 5 + 45 = 50$
