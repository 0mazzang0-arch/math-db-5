## 문제
연속확률변수 $X$가 갖는 값의 범위는 $0 \le X \le 8$이고, $X$의 확률밀도함수 $f(x)$의 그래프는 직선 $x=4$에 대하여 대칭이다. 
$3P(2 \le X \le 4) = 4P(6 \le X \le 8)$일 때, $P(2 \le X \le 6)$의 값은?

## 보기
① $\frac{3}{7}$   ② $\frac{1}{2}$   ③ $\frac{4}{7}$   ④ $\frac{9}{14}$   ⑤ $\frac{5}{7}$

## 풀이
확률밀도함수 $f(x)$의 그래프는 직선 $x=4$에 대하여 대칭이므로 
$P(2 \le X \le 4) = P(4 \le X \le 6)$
$P(2 \le X \le 4) = P(4 \le X \le 6) = k$ $\cdots$ ㉠
라 하면 
$P(6 \le X \le 8) = \frac{1}{2} - k$
조건에서 $3P(2 \le X \le 4) = 4P(6 \le X \le 8)$이므로 
$3k = 4 \left( \frac{1}{2} - k \right)$
$3k = 2 - 4k$
$\therefore k = \frac{2}{7}$
㉠에서 
$P(2 \le X \le 6) = P(2 \le X \le 4) + P(4 \le X \le 6)$
$= 2k$
$= \frac{4}{7}$

정답: ③
