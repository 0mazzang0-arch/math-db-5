## 문제

> [!example] 📸 (그림/그래프)

그림과 같은 모양의 종이에 서로 다른 $3$가지 색을 사용하여 색칠하려고 한다. 이웃한 사다리꼴에는 서로 다른 색을 칠하고, 맨 위의 사다리꼴과 맨 아래의 사다리꼴에 서로 다른 색을 칠한다. $5$개의 사다리꼴에 색을 칠하는 방법의 수를 구하시오.

## 보기



## 풀이

서로 다른 $3$가지 색을 $A, B, C$라고 하면 맨 위와 맨 아래의 사다리꼴에 서로 다른 색을 칠하는 방법의 수는 
$${}_3C_1 \times {}_2C_1 = 6$$
이다. 이때 맨 위와 맨 아래의 사다리꼴에 $A, B$ 두 색을 칠한 경우 중간의 사다리꼴에 색을 칠하는 경우의 수는

$A - B - A - C - B$
$A - B - C - A - B$
$A - C - A - C - B$
$A - C - B - A - B$
$A - C - B - C - B$

로 $5$가지 이므로 구하는 경우의 수는 
$$6 \times 5 = 30$$

[정답] 
$30$
