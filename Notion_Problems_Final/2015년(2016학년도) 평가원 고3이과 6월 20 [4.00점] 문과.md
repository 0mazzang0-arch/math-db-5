## 문제
양수 $t$에 대하여 $\log t$의 소수부분을 $f(t)$라 하자. 자연수 $n$에 대하여 다음 조건을 만족시키는 모든 양수 $t$의 개수를 $a_{n}$이라 할 때, $a_{4}+a_{5}$의 값은?

(가) $1 \le t < 100$
(나) $f(t^{n}) + 2f(t) = 1$

## 보기
① $8$   ② $10$   ③ $12$   ④ $14$   ⑤ $16$

## 풀이
(1) $n=4$ 이면
(i) $1 \le t < 10$ 일 때
$0 \le \log t < 1$ 이므로 $0 \le 4 \log t < 4$
따라서, $m=1, 2, 3, 4$ 에 대하여
$f(t^{4}) = 4 \log t - (m-1)$

(단, $m-1 \le 4 \log t < m$)
$f(t^{4}) + 2f(t) = 4 \log t - (m-1) + 2 \log t = 6 \log t - m + 1 = 1$
$\therefore \log t = \frac{m}{6}$
그런데, $m=4$ 일 때 즉 $\log t = \frac{2}{3}$ 인 경우 $3 \le 4 \log t < 4$ 를 만족시키지 못한다.

(ii) $10 \le t < 100$ 일 때
$1 \le \log t < 2$ 이므로 $4 \le 4 \log t < 8$
따라서, $m=1, 2, 3, 4$ 에 대하여
$f(t^{4}) = 4 \log t - (m+3)$

(단, $m+3 \le 4 \log t < m+4$)
$f(t^{4}) + 2f(t) = 4 \log t - (m+3) + 2(\log t - 1) = 6 \log t - m - 5 = 1$
$\therefore \log t = \frac{m+6}{6}$
그런데, $m=4$ 일 때 즉 $\log t = \frac{5}{3}$ 인 경우 $7 \le 4 \log t < 8$ 을 만족시키지 못한다.

(i), (ii)에 의하여 $a_{4}=6$

(2) $n=5$ 일 때
(i) $1 \le t < 10$ 일 때
$0 \le \log t < 1$ 이므로 $0 \le 5 \log t < 5$
따라서, $m=1, 2, 3, 4, 5$ 에 대하여
$f(t^{5}) = 5 \log t - (m-1)$
$f(t^{5}) + 2f(t) = 5 \log t - (m-1) + 2 \log t = 7 \log t - m + 1 = 1$
$\therefore \log t = \frac{m}{7}$
그런데, $m=4$ 일 때 즉 $\log t = \frac{4}{7}$, $m=5$ 일 때 즉 $\log t = \frac{5}{7}$ 인 경우는 각각 $3 \le 5 \log t < 4$, $4 \le 5 \log t < 5$ 를 만족시키지 못한다.

(ii) $10 \le t < 100$ 일 때
$1 \le \log t < 2$ 이므로 $5 \le 5 \log t < 10$
따라서, $m=1, 2, 3, 4, 5$ 에 대하여
$f(t^{5}) = 5 \log t - (m+4)$

(단, $m+4 \le 5 \log t < m+5$)
$f(t^{5}) + 2f(t) = 5 \log t - (m+4) + 2(\log t - 1) = 7 \log t - m - 6 = 1$
$\therefore \log t = \frac{m+7}{7}$
그런데, $m=4$ 일 때 즉 $\log t = \frac{11}{7}$, $m=5$ 일 때 즉 $\log t = \frac{12}{7}$ 인 경우는 각각 $8 \le 5 \log t < 9$, $9 \le 5 \log t < 10$ 을 만족시키지 못한다.

(i), (ii)에 의하여 $a_{5}=6$
$\therefore a_{4} + a_{5} = 12$
