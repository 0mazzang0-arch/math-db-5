## 문제
방정식 $4 ( \sin ^ { 3 } x - \cos ^ { 3 } x ) = 3 ( \sin x - \cos x )$ 의 모든 실근의 합은? (단, $0 \le x \le \pi$)

## 보기
① $\pi$   ② $\frac { 5 } { 4 } \pi$   ③ $\frac { 3 } { 2 } \pi$   ④ $\frac { 7 } { 4 } \pi$   ⑤ $2 \pi$

## 풀이
$$ ( \sin x - \cos x ) ( 4 \sin ^ { 2 } x + 4 \sin x \cos x + 4 \cos ^ { 2 } x - 3 ) = 0 $$ 
$$ \sqrt { 2 } \sin ( x - \frac { \pi } { 4 } ) ( 2 \sin 2 x + 1 ) = 0 $$ 
$$ \sin ( x - \frac { \pi } { 4 } ) = 0 \text{ 또는 } \sin 2 x = - \frac { 1 } { 2 } $$ 
$$ x = n \pi + \frac { \pi } { 4 } \text{ 또는 } x = \frac { n \pi } { 2 } + ( - 1 ) ^ { n } \frac { 7 } { 12 } \pi $$ 
따라서 $0 \le x \le \pi$ 이므로, 해는 
$$ \frac { \pi } { 4 } , \frac { 7 } { 12 } \pi , \frac { 11 } { 12 } \pi $$ 
방정식 $4 ( \sin ^ { 3 } x - \cos ^ { 3 } x ) = 3 ( \sin x - \cos x )$ 의 모든 실근의 합은
$$ \frac { \pi } { 4 } + \frac { 7 } { 12 } \pi + \frac { 11 } { 12 } \pi = \frac { 21 } { 12 } \pi = \frac { 7 } { 4 } \pi $$

정답: ④
