## 문제
집합 $X=\{1, 2, 3, 4, 5\}$에 대하여 다음 조건을 만족시키는 함수 $f: X \to X$의 개수를 구하시오.

(가) $x=1, 2, 3, 4$일 때 $f(x+1)+3 \ge f(x)+x$이다.
(나) $f(2)$의 값은 홀수이다.

## 보기

## 풀이
조건 (가)에 의하여 

$f(2)+3 \ge f(1)+1$, 
$f(1) \le f(2)+2$ 

$f(3)+3 \ge f(2)+2$, 
$f(2) \le f(3)+1$ 

$f(4)+3 \ge f(3)+3$, 
$f(3) \le f(4)$ 

$f(5)+3 \ge f(4)+4$, 
$f(4) \le f(5)-1$ 

따라서 
$f(1) \le f(2)+2 \le f(3)+3 \le f(4)+3 \le f(5)+2 \quad \cdots \quad ㉠$ 

조건 (나)에서 $f(2)$의 값은 홀수이므로 

(ⅰ) $f(2)=1$일 때 
㉠에서 $f(1) \le 3$이므로 $f(1)$의 값을 정하는 경우의 수는 $3$이고 

$3 \le f(3)+3 \le f(4)+3 \le f(5)+2$ 

$0 \le f(3) \le f(4) \le f(5)-1$이므로 
$f(3)$, $f(4)$, $f(5)$의 값을 정하는 경우의 수는 $1, 2, 3, 4, 5$ 중 $3$개를 정하는 중복조합의 수에서 $f(5)-1$의 값이 $5$인 경우의 수를 뺀 값과 같다. 

${}_5H_3 - {}_5H_2 = {}_7C_3 - {}_6C_2 = 35 - 15 = 20$ 

$3 \times 20 = 60$ 

(ⅱ) $f(2)=3$일 때 
㉠에서 $f(1) \le 5$이므로 $f(1)$의 값을 정하는 경우의 수는 $5$이고 

$5 \le f(3)+3 \le f(4)+3 \le f(5)+2$ 

$2 \le f(3) \le f(4) \le f(5)-1$이므로 
$f(3)$, $f(4)$, $f(5)$의 값을 정하는 경우의 수는 $2, 3, 4, 5$ 중 $3$개를 정하는 중복조합의 수에서 $f(5)-1$의 값이 $5$인 경우의 수를 뺀 값과 같다. 

${}_4H_3 - {}_4H_2 = {}_6C_3 - {}_5C_2 = 20 - 10 = 10$ 

$5 \times 10 = 50$ 

(ⅲ) $f(2)=5$일 때 
㉠에서 $f(1) \le 7$이므로 $f(1)$의 값을 정하는 경우의 수는 $5$이고 

$7 \le f(3)+3 \le f(4)+3 \le f(5)+2$ 

$4 \le f(3) \le f(4) \le f(5)-1$이므로 
$f(3)$, $f(4)$, $f(5)$의 값을 정하는 경우의 수는 $4, 5$ 중 $3$개를 정하는 중복조합의 수에서 $f(5)-1$의 값이 $5$인 경우의 수를 뺀 값과 같다. 

${}_2H_3 - {}_2H_2 = {}_4C_3 - {}_3C_2 = 4 - 3 = 1$ 

$5 \times 1 = 5$ 

이상에서 구하는 함수의 개수는 
$60+50+5=115$ 

정답: $115$
