## 문제
다음은 $\sqrt[3]{2+\sqrt{5}}$를 $p+q\sqrt{5}$ ($p$, $q$는 유리수) 꼴로 나타내는 과정이다.

$$ \begin{array}{|l|} \hline \alpha=\sqrt[3]{2+\sqrt{5}}, \beta=\sqrt[3]{2-\sqrt{5}}\text{라 두면 } \alpha\beta = (가) \text{ 이다.} \\ \alpha+\beta=x\text{라 하면 } (\alpha+\beta)^{3}=\alpha^{3}+3\alpha^{2}\beta+3\alpha\beta^{2}+\beta^{3}\text{을 이용하여} \\ \text{삼차방정식 } x^{3}+3x-4=0\text{을 얻을 수 있다.} \\ \text{이 방정식의 실근은 } x=1\text{ 뿐이므로 } \alpha+\beta=1\text{이다.} \\ \text{따라서 } \alpha+\beta=1\text{과 } \alpha\beta = (가) \text{ 인 실수 } \alpha, \beta\text{를 두 근으로 하는} \\ \text{이차방정식은 } \underline{~~~~~~~~~} \text{ 이다.} \\ \text{이 이차방정식의 두 근 } \alpha, \beta\text{가 } \beta<0<\alpha\text{이므로 } \sqrt[3]{2+\sqrt{5}} = (나) \text{ 이다.} \\ \hline \end{array} $$

위의 과정에서 $(가)$, $(나)$에 들어갈 수를 각각 $a$, $b$라 할 때, $a+10b$의 값은?

## 보기
① $1+2\sqrt{5}$   ② $2+3\sqrt{5}$   ③ $3+4\sqrt{5}$   ④ $4+5\sqrt{5}$   ⑤ $5+6\sqrt{5}$

## 풀이
$\alpha = \sqrt[3]{2+\sqrt{5}}$, $\beta = \sqrt[3]{2-\sqrt{5}}$라 두면
$\alpha\beta = \sqrt[3]{2+\sqrt{5}} \cdot \sqrt[3]{2-\sqrt{5}} = \sqrt[3]{(2+\sqrt{5})(2-\sqrt{5})} = \sqrt[3]{4-5} = \sqrt[3]{-1} = -1$
이다. 따라서 $a = -1$이다.

$\alpha+\beta=x$라 하면
$(\alpha+\beta)^{3} = \alpha^{3}+\beta^{3}+3\alpha\beta(\alpha+\beta) = (2+\sqrt{5})+(2-\sqrt{5})+3(-1)x = 4-3x$
이므로 삼차방정식 $x^{3}+3x-4=0$을 얻을 수 있다.
$(x-1)(x^{2}+x+4)=0$에서 이 방정식의 실근은 $x=1$ 뿐이므로 $\alpha+\beta=1$이다.

따라서 $\alpha+\beta=1$과 $\alpha\beta=-1$인 실수 $\alpha, \beta$를 두 근으로 하는 최고차항의 계수가 $1$인 이차방정식은 $x^{2}-x-1=0$이다.
이 이차방정식의 해는 $x = \frac{1 \pm \sqrt{5}}{2}$이며, 조건에서 $\beta < 0 < \alpha$이므로
$\alpha = \sqrt[3]{2+\sqrt{5}} = \frac{1+\sqrt{5}}{2}$이다. 따라서 $b = \frac{1+\sqrt{5}}{2}$이다.

그러므로 $a+10b$의 값은
$a+10b = -1 + 10 \left( \frac{1+\sqrt{5}}{2} \right) = -1 + 5(1+\sqrt{5}) = 4+5\sqrt{5}$
이다.
