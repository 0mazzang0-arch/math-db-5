## 문제
$0 < \theta < \frac{\pi}{4}$ 인 $\theta$ 에 대하여 $(1 + \tan \theta) \tan 2\theta = 3$ 일 때, $\tan \theta$ 의 값은?

## 보기
① $\frac{1}{5}$   ② $\frac{3}{10}$   ③ $\frac{2}{5}$   ④ $\frac{1}{2}$   ⑤ $\frac{3}{5}$

## 풀이
배각의 공식에 의해 $\tan 2\theta = \frac{2 \tan \theta}{1 - \tan^2 \theta}$ 이므로
$$(1 + \tan \theta) \tan 2\theta = (1 + \tan \theta) \times \frac{2 \tan \theta}{1 - \tan^2 \theta}$$
$$(1 + \tan \theta) \times \frac{2 \tan \theta}{(1 - \tan \theta)(1 + \tan \theta)} = \frac{2 \tan \theta}{1 - \tan \theta}$$
조건에 의해 $\frac{2 \tan \theta}{1 - \tan \theta} = 3$ 이다.
$$2 \tan \theta = 3(1 - \tan \theta)$$
$$2 \tan \theta = 3 - 3 \tan \theta$$
$$5 \tan \theta = 3$$
따라서 $\tan \theta = \frac{3}{5}$ 이다.
