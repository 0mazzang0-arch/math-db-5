## 문제
$1 \le a \le b \le c \le d \le 10$을 만족시키는 자연수 $a$, $b$, $c$, $d$의 모든 순서쌍 $(a, b, c, d)$ 중에서 임의로 한 개를 선택한다. 선택한 순서쌍 $(a, b, c, d)$에서 $a \times b \times (c+1) \times (d+1)$이 홀수일 때, $b \times c$가 $3$의 배수일 확률은 $\frac{q}{p}$이다. $p+q$의 값을 구하시오. (단, $p$와 $q$는 서로소인 자연수이다.)

## 풀이
네 수의 곱이 홀수가 되려면 네 수 모두 홀수이어야 한다. 따라서 $a \times b \times (c+1) \times (d+1)$이 홀수이므로 $a$, $b$는 홀수, $c$, $d$는 짝수이다. 
$\cdots \cdots$ ㉠ 
이때 자연수 $a$, $b$, $c$, $d$는 $1 \le a \le b \le c \le d \le 10$을 만족하므로 

(ⅰ) $c=2$ 일 때 
$1 \le a \le b \le 2 \le d \le 10$이므로 
$a=b=1$ 
$d$를 택하는 경우의 수는 $5$이므로 순서쌍 $(a, b, c, d)$의 개수는 $1 \times 5 = 5$ 

(ⅱ) $c=4$ 일 때 
$1 \le a \le b \le 4 \le d \le 10$이므로 순서쌍 $(a, b, c, d)$의 개수는 ${}_{2} H_{2} \times 4 = {}_{3} C_{2} \times 4 = 12$ 

(ⅲ) $c=6$ 일 때 
$1 \le a \le b \le 6 \le d \le 10$이므로 순서쌍 $(a, b, c, d)$의 개수는 ${}_{3} H_{2} \times 3 = {}_{4} C_{2} \times 3 = 18$ 

(ⅳ) $c=8$ 일 때 
$1 \le a \le b \le 8 \le d \le 10$이므로 순서쌍 $(a, b, c, d)$의 개수는 ${}_{4} H_{2} \times 2 = {}_{5} C_{2} \times 2 = 20$ 

(ⅴ) $c=10$ 일 때 
$1 \le a \le b \le 10 \le d \le 10$이므로 순서쌍 $(a, b, c, d)$의 개수는 ${}_{5} H_{2} \times 1 = {}_{6} C_{2} \times 1 = 15$ 

이상에서 ㉠을 만족하는 순서쌍 $(a, b, c, d)$의 개수는 
$5 + 12 + 18 + 20 + 15 = 70$ 
$\cdots \cdots$ ㉡ 

이때 $b \times c$가 $3$의 배수가 되려면 $b$ 또는 $c$가 $3$의 배수가 되어야 한다. 

(ⅰ) $c=6$ 일 때 
$1 \le a \le b \le 6 \le d \le 10$이므로 순서쌍 $(a, b, c, d)$의 개수는 ${}_{3} H_{2} \times 3 = {}_{4} C_{2} \times 3 = 18$ 

(ⅱ) $b=3$ 일 때 
$1 \le a \le 3 \le c \le d \le 10$이므로 순서쌍 $(a, b, c, d)$의 개수는 $2 \times {}_{4} H_{2} = 2 \times {}_{5} C_{2} = 20$ 

(ⅲ) $b=9$ 일 때 
$1 \le a \le 9 \le c \le d \le 10$이므로 순서쌍 $(a, b, c, d)$의 개수는 $5 \times {}_{1} H_{2} = 5 \times {}_{2} C_{2} = 5$ 

(ⅳ) $b=3$, $c=6$ 일 때 
$1 \le a \le 3 \le 6 \le d \le 10$이므로 순서쌍 $(a, b, c, d)$의 개수는 $2 \times 3 = 6$ 

이상에서 $b \times c$가 $3$의 배수인 순서쌍 $(a, b, c, d)$의 개수는 
$18 + 20 + 5 - 6 = 37$ 
$\cdots \cdots$ ㉢ 

㉡, ㉢에서 구하는 확률은 $\frac{37}{70}$ 
따라서 $p=70$, $q=37$이므로 $p+q = 70 + 37 = 107$

정답: $107$
