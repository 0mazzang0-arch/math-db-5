## 문제
이차방정식 $x^2 + ax + b = 0$의 두 근이 $\alpha$, $\beta$이고, 이차방정식 $x^2 + bx + a = 0$의 두 근이 $\frac{1}{\alpha}$, $\frac{1}{\beta}$이다. 이때 $a+b$의 값은? (단, $a$, $b$는 실수이다.)

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
$x^2 + ax + b = 0$의 두 근이 $\alpha$, $\beta$이므로 근과 계수의 관계에 의하여
$\alpha + \beta = -a$, $\alpha\beta = b$ $\cdots\cdots$ ㉠

$x^2 + bx + a = 0$의 두 근이 $\frac{1}{\alpha}$, $\frac{1}{\beta}$이므로 근과 계수의 관계에 의하여
$\frac{1}{\alpha} + \frac{1}{\beta} = -b$, $\frac{1}{\alpha\beta} = a$ $\cdots\cdots$ ㉡

㉠, ㉡에서
$\frac{1}{\alpha} + \frac{1}{\beta} = \frac{\alpha+\beta}{\alpha\beta} = -b$이므로
$\frac{-a}{b} = -b$이다.
$\therefore a = b^2$

$\frac{1}{\alpha\beta} = a$이므로
$\frac{1}{b} = a$이다.
$\therefore ab = 1$

따라서 $b^2 \cdot b = 1$에서 $b^3 = 1$, $a^3 = 1$이고 $a$, $b$가 실수이므로
$a = 1$, $b = 1$이다.

$\therefore a + b = 2$
