## 문제
전체집합 $U = \{ x \mid x \text{ 는 } 20 \text{ 이하의 자연수} \}$ 의 세 부분집합
$A = \{ x \mid x \text{ 는 } 18 \text{ 의 약수} \}$,
$B = \{ x \mid x \text{ 는 } 12 \text{ 의 약수} \}$,
$C = \{ x \mid x \text{ 는 } 6 \text{ 의 약수} \}$
에 대하여 다음 중 옳은 것은?

## 보기
① $A \cup C = U$   ② $A \cap B = C$   ③ $A \cup B^{C} = C$   ④ $A - C = B$   ⑤ $B - C = \emptyset$

## 풀이
세 집합 $A$, $B$, $C$를 원소나열법으로 나타내면 다음과 같다.
$A = \{ 1, 2, 3, 6, 9, 18 \}$
$B = \{ 1, 2, 3, 4, 6, 12 \}$
$C = \{ 1, 2, 3, 6 \}$

① $A \cup C = \{ 1, 2, 3, 6, 9, 18 \} = A$
② $A \cap B = \{ 1, 2, 3, 6 \} = C$ (옳음)
③ $A \cup B^{C} = A \cup (U - B) = U - \{ 4, 12 \}$
④ $A - C = \{ 9, 18 \}$
⑤ $B - C = \{ 4, 12 \}$

따라서 옳은 것은 ②이다.

정답: ②
