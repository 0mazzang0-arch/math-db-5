## 문제
$0$ 이 아닌 서로 다른 세 실수 $p$, $q$, $r$ 에 대하여 삼차함수 $f(x) = (x-p)(x-q)(x-r)$ 라 할 때, $\frac{p^2}{f'(p)} + \frac{q^2}{f'(q)} + \frac{r^2}{f'(r)}$ 의 값은?

## 보기
① $-1$   ② $-\frac{1}{2}$   ③ $0$   ④ $\frac{1}{2}$   ⑤ $1$

## 풀이
$f'(x) = (x-q)(x-r) + (x-r)(x-p) + (x-p)(x-q)$ 이므로
$\frac{p^2}{f'(p)} + \frac{q^2}{f'(q)} + \frac{r^2}{f'(r)}$
$= \frac{p^2}{(p-q)(p-r)} + \frac{q^2}{(q-r)(q-p)} + \frac{r^2}{(r-p)(r-q)}$
$= \frac{p^2(q-r) - q^2(p-r) + r^2(p-q)}{(p-q)(p-r)(q-r)}$

이때, 분자를 인수분해하면
$p^2(q-r) - q^2(p-r) + r^2(p-q)$
$= p^2(q-r) - (q^2 - r^2)p + q^2r - qr^2$
$= (q-r) \{ p^2 - (q+r)p + qr \}$
$= (q-r)(p-q)(p-r)$

따라서
$\frac{p^2}{f'(p)} + \frac{q^2}{f'(q)} + \frac{r^2}{f'(r)} = \frac{(p-q)(p-r)(q-r)}{(p-q)(p-r)(q-r)} = 1$

정답: ⑤
