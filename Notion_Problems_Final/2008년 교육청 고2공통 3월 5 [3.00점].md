## 문제
두 집합 $X$, $Y$에 대하여 연산 $\triangle$를 $X \triangle Y = (X - Y) \cup (Y - X)$라고 정의할 때, 집합 $\{A \cap (B \cup C)\} \triangle (B \triangle C)$를 벤다이어그램으로 나타낸 것은?

## 풀이
두 집합 $X$, $Y$에 대하여 $X \triangle Y = (X - Y) \cup (Y - X)$는 대칭차집합을 의미한다.

주어진 식 $\{A \cap (B \cup C)\} \triangle (B \triangle C)$를 구하기 위해 각 부분을 벤다이어그램으로 나타내면 다음과 같다.

$1$. $A \cap (B \cup C)$은 집합 $A$와 ($B$와 $C$의 합집합)의 공통 부분이다.
$2$. $B \triangle C = (B - C) \cup (C - B)$는 $B$와 $C$의 대칭차집합이다.
$3$. $\{A \cap (B \cup C)\} \triangle (B \triangle C)$는 위 두 결과의 대칭차집합이다.

이를 만족하는 벤다이어그램을 찾으면 정답은 ④이다.

정답: ④
