## 문제
다음 중 집합 $A = \{3, 4\}$와 서로소인 집합은?

## 보기
① $\{4\}$ ② $\{1, 3, 5\}$ ③ $\{x \mid x \text{는 } 10 \text{ 이하의 홀수}\}$ ④ $\{x \mid x \text{는 } 10 \text{ 이하의 } 3 \text{의 배수}\}$ ⑤ $\{x \mid x \text{는 } 7 \text{의 양의 약수}\}$

## 풀이
$\{x \mid x \text{는 } 7 \text{의 양의 약수}\} = \{1, 7\}$ 이고
$\{1, 7\} \cap A = \emptyset$ 이므로 
$\{1, 7\}$ 와 $A$ 는 서로소이다.

정답: ⑤
