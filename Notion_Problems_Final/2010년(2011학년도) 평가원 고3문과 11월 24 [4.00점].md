## 문제
자연수 $A$에 대하여 $\log A$의 정수부분을 $n$, 소수부분을 $\alpha$라 할 때, $n \le 2\alpha$가 성립하도록 하는 $A$의 개수를 구하시오. (단, $3.1 < \sqrt{10} < 3.2$)

## 풀이
$0 \le \alpha < 1$이므로 $0 \le 2\alpha < 2$이다.
$n \le 2\alpha$에서 $n$은 정수이므로 $n = 0$ 또는 $n = 1$이다.

(ⅰ) $n = 0$일 때
$0 \le 2\alpha$에서 $\alpha \ge 0$이다.
$\log A = 0 + \alpha$ ($0 \le \alpha < 1$)이므로 $0 \le \log A < 1$에서 $1 \le A < 10$이다.
만족하는 자연수 $A$는 $1, 2, 3, \dots, 9$로 그 개수는 $9$이다.

(ⅱ) $n = 1$일 때
$1 \le 2\alpha$에서 $\alpha \ge \frac{1}{2}$이다.
$\log A = 1 + \alpha$ ($\frac{1}{2} \le \alpha < 1$)이므로 $1.5 \le \log A < 2$이다.
즉, $10^{1.5} \le A < 10^2$에서 $10\sqrt{10} \le A < 100$이다.
조건에서 $3.1 < \sqrt{10} < 3.2$이므로 $31 < 10\sqrt{10} < 32$이다.
따라서 자연수 $A$는 $32, 33, \dots, 99$이며, 그 개수는 $99 - 32 + 1 = 68$이다.

따라서 $A$의 개수는 $9 + 68 = 77$이다.

정답: $77$
