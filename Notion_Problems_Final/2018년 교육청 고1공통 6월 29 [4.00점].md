## 문제
최고차항의 계수가 음수인 이차다항식 $P(x)$ 가 모든 실수 $x$ 에 대하여 
$\{ P(x)+x \}^{2} = (x-a)(x+a)(x^{2}+5)+9$
를 만족시킨다. $\{ P(a) \}^{2}$ 의 값을 구하시오. (단, $a>0$)

## 보기

## 풀이
주어진 등식에서 $P(x)+x$ 가 이차다항식이므로 
$(x-a)(x+a)(x^{2}+5)+9$
도 이차다항식의 완전제곱식이어야 한다. 

$(x-a)(x+a)(x^{2}+5)+9$ 
$= (x^{2}-a^{2})(x^{2}+5)+9$ 
$= x^{4} + (5-a^{2})x^{2} - 5a^{2} + 9$ 
$= x^{4} + (5-a^{2})x^{2} + \frac{(5-a^{2})^{2}}{4} - \frac{(5-a^{2})^{2}}{4} - 5a^{2} + 9$ 
$= \{ x^{4} + (5-a^{2})x^{2} + \frac{(5-a^{2})^{2}}{4} \} - \frac{(5-a^{2})^{2} - 4(-5a^{2}+9)}{4}$ 
$= (x^{2} + \frac{5-a^{2}}{2})^{2} - \frac{(5-a^{2})^{2} - 4(-5a^{2}+9)}{4}$ 

에서 
$- \frac{(5-a^{2})^{2} - 4(-5a^{2}+9)}{4} = 0$ 
$(5-a^{2})^{2} - 4(-5a^{2}+9) = 0$ 
$a^{4} + 10a^{2} - 11 = 0$, 
$(a^{2}+11)(a^{2}-1) = 0$ 
$(a^{2}+11)(a+1)(a-1) = 0$ 

$\therefore a = 1$ ($\because a > 0$)

따라서 주어진 등식에서 
$\{ P(x)+x \}^{2} = (x^{2}+2)^{2}$ 

$\therefore P(x)+x = \pm(x^{2}+2)$
$P(x) = x^{2}-x+2$ 또는 $P(x) = -x^{2}-x-2$ 

그런데 $P(x)$ 의 이차항의 계수가 음수이므로 
$P(x) = -x^{2}-x-2$ 

$\therefore \{ P(a) \}^{2} = \{ P(1) \}^{2} = (-1-1-2)^{2} = 16$ 

---
**[다른 풀이]** 

주어진 등식에서 
$\{ P(x)+x \}^{2} = (x^{2}-a^{2})(x^{2}+5)+9$ 

$\therefore \{ P(x)+x \}^{2} = x^{4} + (5-a^{2})x^{2} - 5a^{2} + 9 \cdots \cdots$ ㉠ 

$P(x)$ 의 최고차항의 계수가 음수이므로 
$P(x)+x = -x^{2}+px+q$ ($p$, $q$ 는 실수) 
라 하면 ㉠에 대입하면 
$(-x^{2}+px+q)^{2} = x^{4} + (5-a^{2})x^{2} - 5a^{2} + 9$ 

$\therefore x^{4} - 2px^{3} + (p^{2}-2q)x^{2} + 2pqx + q^{2} = x^{4} + (5-a^{2})x^{2} - 5a^{2} + 9$ 

위 등식이 $x$ 에 대한 항등식이므로 
$-2p = 0 \therefore p = 0$ 

$p^{2}-2q = 5-a^{2}$ 
$\therefore a^{2} = 2q+5$ ($\because p = 0$) $\cdots \cdots$ ㉡ 

$q^{2} = -5a^{2} + 9$ 
$q^{2} + 10q + 16 = 0$ ($\because$ ㉡) 
$(q+8)(q+2) = 0$ 
$\therefore q = -8$ 또는 $q = -2$ 

그런데 $q = -8$ 이면 $a^{2} = -11 < 0$ 이므로 주어진 조건을 만족시키지 않으므로 $q = -2$ 이다. 
이것을 ㉡에 대입하면 
$a^{2} = 1 \therefore a = 1$ ($\because a > 0$) 

이것을 ㉠에 대입하면 $P(x)+x = -x^{2}-2$ 
$\therefore P(x) = -x^{2}-x-2$ 

$\therefore \{ P(a) \}^{2} = \{ P(1) \}^{2} = 16$
