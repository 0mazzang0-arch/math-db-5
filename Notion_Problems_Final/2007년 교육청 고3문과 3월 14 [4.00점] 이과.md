## 문제
$a$, $b$, $c$ 가 서로 다른 세 실수일 때, 이차함수 $f(x)=ax^{2}+2bx+c$ 에 대한 설명 중 옳은 것만을 <보기>에서 있는 대로 고른 것은?

$$
\begin{array}{|l|}
\hline
\text{<보 기>} \\
\text{ㄱ. } a, b, c \text{ 가 이 순서로 등차수열을 이루면 } f(1)=4b \text{ 이다.} \\
\text{ㄴ. } a, b, c \text{ 가 이 순서로 등차수열을 이루면 } y=f(x) \text{ 의 그래프는 } x \text{ 축과 서로 다른 두 점에서 만난다.} \\
\text{ㄷ. } a, b, c \text{ 가 이 순서로 등비수열을 이루면 } y=f(x) \text{ 의 그래프는 } x \text{ 축과 만나지 않는다.} \\
\hline
\end{array}
$$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
ㄱ. $a$, $b$, $c$ 가 이 순서로 등차수열을 이루므로 $2b=a+c$ 이다.
$\therefore f(1)=a+2b+c=2b+2b=4b$ (참)

ㄴ. 이차방정식 $ax^{2}+2bx+c=0$ 의 판별식을 $D$ 라 하면
$D=(2b)^{2}-4ac=(a+c)^{2}-4ac=(a-c)^{2}>0$ ($\because a \neq c$)
따라서 $y=f(x)$ 의 그래프는 $x$ 축과 서로 다른 두 점에서 만난다. (참)

ㄷ. $a$, $b$, $c$ 가 이 순서로 등비수열을 이루므로 $b^{2}=ac$ 이다.
$\frac{D}{4}=b^{2}-ac=0$
따라서 함수 $y=f(x)$ 의 그래프는 $x$ 축에 접한다. (거짓)
