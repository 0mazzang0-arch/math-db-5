## 문제
세 이차정사각행렬 $A$, $B$, $C$와 단위행렬 $E$에 대하여 $A B C = E$가 성립할 때, <보기>에서 항상 옳은 것만을 있는 대로 고른 것은?

ㄱ. $A^{-1} = B C$
ㄴ. $B C A = C A B$
ㄷ. 행렬 $B$의 역행렬이 존재한다.

## 보기
① ㄱ   ② ㄷ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
[정답] ⑤ 

ㄱ. $A(B C) = E$ 에서 $A^{-1} = B C$ (참) 
ㄴ. $A^{-1}(A B C) A = A^{-1} E A = E$ 에서 $B C A = E$ 
$C(A B C) C^{-1} = C E C^{-1} = E$ 에서 $C A B = E$ 
따라서 $B C A = C A B$ (참) 
ㄷ. $B(C A) = E$ 에서 $B^{-1} = C A$ 이므로 행렬 $B$의 역행렬이 존재한다. (참) 

따라서 ㄱ, ㄴ, ㄷ 모두 옳다.
