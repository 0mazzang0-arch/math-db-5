## 문제
$\log x$의 소수부분 $\alpha$가 $0 < \alpha < \frac{1}{4}$일 때, $\log x^2$의 소수부분과 $\log \frac{\sqrt{10}}{x^2}$의 소수부분의 합은?

## 보기
① $1$   ② $\frac{1}{2}$   ③ $\frac{1}{3}$   ④ $\frac{1}{4}$   ⑤ $\frac{1}{5}$

## 풀이
$\log x$의 정수부분을 $n$이라 하면
$\log x = n + \alpha$ ($0 < \alpha < \frac{1}{4}$)

$\log x^2 = 2 \log x = 2(n + \alpha) = 2n + 2\alpha$
$0 < 2\alpha < \frac{1}{2}$이므로 $\log x^2$의 소수부분은 $2\alpha$이다.

$\log \frac{\sqrt{10}}{x^2} = \frac{1}{2} \log 10 - \log x^2$
$= \frac{1}{2} - (2n + 2\alpha)$
$= -2n + \frac{1}{2} - 2\alpha$

$0 < 2\alpha < \frac{1}{2}$이므로 $\log \frac{\sqrt{10}}{x^2}$의 소수부분은 $\frac{1}{2} - 2\alpha$이다.

따라서 두 소수부분의 합은
$2\alpha + \left( \frac{1}{2} - 2\alpha \right) = \frac{1}{2}$

정답: ②
