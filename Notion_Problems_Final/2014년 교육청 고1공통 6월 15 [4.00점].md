## 문제
이차함수 $f(x) = (x-a)(x-b)$ 에 대하여 <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $a, b$ 는 실수이다.)



> [!example] 📸 (그림/그래프)

## 보기
ㄱ. $a = b$ 이면 모든 실수 $x$ 에 대하여 $f(x) \ge 0$ 이다.
ㄴ. 이차함수 $f(x)$ 의 최솟값은 $f(\frac{a+b}{2})$ 이다.
ㄷ. $0 < a < b$ 이면 $f(\frac{b-a}{2}) < f(\frac{a-b}{2})$ 이다.

① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
ㄱ. $a=b$ 이면 $f(x) = (x-a)^{2}$ 이므로 모든 실수 $x$ 에 대하여 $f(x) \ge 0$ 이다.

ㄴ. $f(x) = x^{2} - (a+b)x + ab$ 이므로
$$ \begin{array}{l} f(x) = x^{2} - (a+b)x + \left( \frac{a+b}{2} \right)^{2} + ab - \left( \frac{a+b}{2} \right)^{2} \\ f(x) = \left( x - \frac{a+b}{2} \right)^{2} + ab - \frac{a^{2} + 2ab + b^{2}}{4} \\ f(x) = \left( x - \frac{a+b}{2} \right)^{2} - \frac{a^{2} - 2ab + b^{2}}{4} \\ f(x) = \left( x - \frac{a+b}{2} \right)^{2} - \left( \frac{a-b}{2} \right)^{2} \end{array} $$
따라서 $f(x)$ 는 $x = \frac{a+b}{2}$ 일 때 최솟값 $f(\frac{a+b}{2}) = -\left( \frac{a-b}{2} \right)^{2}$ 를 가진다.

ㄷ. 이차함수 $y=f(x)$ 의 그래프는 아래로 볼록하고 $x = \frac{a+b}{2}$ 에서 최솟값을 가진다.
$0 < a < b$ 일 때, 대칭축으로부터 $x$ 값의 거리를 비교하면 다음과 같다.
$$ \begin{array}{l} \left| \frac{b-a}{2} - \frac{a+b}{2} \right| = \left| -a \right| = a \\ \left| \frac{a-b}{2} - \frac{a+b}{2} \right| = \left| -b \right| = b \end{array} $$
$0 < a < b$ 이므로 $x = \frac{b-a}{2}$ 가 $x = \frac{a-b}{2}$ 보다 대칭축에 더 가깝다.
따라서 $f(\frac{b-a}{2}) < f(\frac{a-b}{2})$ 이다.

**[다른 풀이]**
ㄷ. $f(\frac{b-a}{2}) - f(\frac{a-b}{2})$ 의 값을 직접 계산하면 다음과 같다.
$$ \begin{array}{l} f\left( \frac{b-a}{2} \right) = \left( \frac{b-a}{2} - a \right) \left( \frac{b-a}{2} - b \right) = \left( \frac{b-3a}{2} \right) \left( \frac{-a-b}{2} \right) = \frac{(3a-b)(a+b)}{4} \\ f\left( \frac{a-b}{2} \right) = \left( \frac{a-b}{2} - a \right) \left( \frac{a-b}{2} - b \right) = \left( \frac{-a-b}{2} \right) \left( \frac{a-3b}{2} \right) = \frac{(3b-a)(a+b)}{4} \\ f\left( \frac{b-a}{2} \right) - f\left( \frac{a-b}{2} \right) = \frac{a+b}{4} \{ (3a-b) - (3b-a) \} = \frac{a+b}{4} (4a-4b) = (a+b)(a-b) \end{array} $$
$0 < a < b$ 에서 $a+b > 0$, $a-b < 0$ 이므로
$(a+b)(a-b) < 0$
$\therefore f(\frac{b-a}{2}) < f(\frac{a-b}{2})$

이상에서 ㄱ, ㄴ, ㄷ 모두 옳다.
