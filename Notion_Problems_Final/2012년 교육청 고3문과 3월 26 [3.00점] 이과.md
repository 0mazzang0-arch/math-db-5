## 문제
수열 $\{ a_{n} \}$은 $a_{1} = 1$, $a_{2} = 1$이고 모든 자연수 $n$에 대하여 다음 조건을 만족시킨다.

(가) $a_{2n+2} - a_{2n} = 1$
(나) $a_{2n+1} - a_{2n-1} = 0$

$a_{100} + a_{101}$의 값을 구하시오.

## 풀이
$a_{3} = a_{1} = 1$, $a_{5} = a_{3} = 1$, $a_{7} = a_{5} = 1$, $\dots$ 이므로 $a_{2n-1} = 1$이다.
$a_{4} = a_{2} + 1 = 1 + 1 = 2$, 
$a_{6} = a_{4} + 1 = 2 + 1 = 3$, 
$a_{8} = a_{6} + 1 = 3 + 1 = 4$, 
$\dots$ 이므로 $a_{2n} = n$이다.
$\therefore a_{100} + a_{101} = 50 + 1 = 51$
