## 문제
$0 < a < b$인 $a$, $b$에 대하여 $N(a, b)$를 $a < x < b$에서 $\log x$의 소수부분과 $\log x^3$의 소수부분이 같은 실수 $x$의 개수라 하자. <보기>에서 옳은 것만을 있는 대로 고른 것은?

$$ \begin{array}{|l|} \hline \text{ㄱ. } N(\sqrt{10}, 1000) = 4 \\ \text{ㄴ. } p \text{ 가 정수이면 } N(10^p, 10^{p+10}) = 19 \text{ 이다.} \\ \text{ㄷ. } N(2^{10}, 2^{50}) = 25 \\ \hline \end{array} $$

(단, $\log 2 = 0.3010$으로 계산한다.)

## 보기
① $ㄱ$   ② $ㄱ, ㄴ$   ③ $ㄱ, ㄷ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
$\log x^3 - \log x = 3 \log x - \log x = 2 \log x = (\text{정수})$이므로

ㄱ. $\sqrt{10} < x < 1000$ 에서
$\frac{1}{2} < \log x < 3$ 이므로
$1 < 2 \log x < 6$
따라서 $2 \log x$의 값은 $2, 3, 4, 5$이다.
$\therefore N(\sqrt{10}, 1000) = 4$ (참)

ㄴ. $10^p < x < 10^{p+10}$ 에서
$p < \log x < p+10$ 이므로
$2p < 2 \log x < 2p + 20$
따라서 $2 \log x$의 값은 $2p+1, 2p+2, \cdots, 2p+19$로 그 개수는 $19$이다.
$\therefore N(10^p, 10^{p+10}) = 19$ (참)

ㄷ. $2^{10} < x < 2^{50}$ 에서
$10 \log 2 < \log x < 50 \log 2$
$3.010 < \log x < 15.050$ 이므로
$6.02 < 2 \log x < 30.10$
따라서 $2 \log x$의 값은 $7, 8, \cdots, 30$으로 그 개수는 $30 - 7 + 1 = 24$이다.
$\therefore N(2^{10}, 2^{50}) = 24$ (거짓)

따라서 옳은 것은 ㄱ, ㄴ이다.
