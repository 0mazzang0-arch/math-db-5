## 문제
두 수열 $\{ a_{n} \}$, $\{ b_{n} \}$의 일반항이 $a_{n} = \frac{(-1)^{n} + 3}{2}$, $b_{n} = p \times (-1)^{n+1} + q$일 때, <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $p$, $q$는 실수이다.)

## 보기
ㄱ. 수열 $\{ a_{n} \}$은 발산한다.
ㄴ. 수열 $\{ b_{n} \}$이 수렴하도록 하는 실수 $p$가 존재한다.
ㄷ. 두 수열 $\{ a_{n} + b_{n} \}$, $\{ a_{n} b_{n} \}$이 모두 수렴하면 $\lim_{n \to \infty} (a_{n}^{2} + b_{n}^{2}) = 6$이다.

① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄱ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
[정답] ③ 

ㄱ. 수열 $\{ a_{n} \}$은 $1$, $2$, $1$, $2$, $\dots$ 이므로 발산(진동)한다. (참) 

ㄴ. 수열 $\{ b_{n} \}$은 $p+q$, $-p+q$, $p+q$, $-p+q$, $\dots$ 이다. $p=0$인 경우 수열 $\{ b_{n} \}$은 $q$, $q$, $q$, $\dots$ 로 수렴하므로 실수 $p$가 존재한다. (참) 

ㄷ. 수열 $\{ a_{n} + b_{n} \}$은 $1+p+q$, $2-p+q$, $1+p+q$, $2-p+q$, $\dots$ 이다. $\{ a_{n} + b_{n} \}$이 수렴하기 위해서는 $1+p+q = 2-p+q$가 성립해야 하므로 $2p = 1$, 즉 $p = \frac{1}{2}$이다.
수열 $\{ a_{n} b_{n} \}$은 $1 \times (p+q)$, $2 \times (-p+q)$, $1 \times (p+q)$, $2 \times (-p+q)$, $\dots$ 이므로 수열 $\{ a_{n} b_{n} \}$이 수렴하기 위해서는 $1 \times (p+q) = 2 \times (-p+q)$가 성립해야 하므로 $p+q = -2p+2q$, 즉 $q = 3p$이다.
$p = \frac{1}{2}$이므로 $q = 3 \times \frac{1}{2} = \frac{3}{2}$이다.
이때 $1+p+q = 2-p+q = 3$이므로 $\lim_{n \to \infty} (a_{n} + b_{n}) = 3$이다.
또한 $1 \times (p+q) = 2 \times (-p+q) = 2$이므로 $\lim_{n \to \infty} a_{n} b_{n} = 2$이다.
$\lim_{n \to \infty} \{ (a_{n})^{2} + (b_{n})^{2} \} = \lim_{n \to \infty} \{ (a_{n} + b_{n})^{2} - 2a_{n} b_{n} \}$
$= \lim_{n \to \infty} (a_{n} + b_{n}) \times \lim_{n \to \infty} (a_{n} + b_{n}) - 2 \lim_{n \to \infty} a_{n} b_{n}$
$= 3^{2} - 2 \times 2 = 5$ (거짓)
따라서 옳은 것은 ㄱ, ㄴ이다.

[다른 풀이]
ㄷ. $a_{n} + b_{n} = \{ \frac{1}{2} \times (-1)^{n} + \frac{3}{2} \} + \{ p \times (-1)^{n+1} + q \}$
$= (\frac{1}{2} - p) (-1)^{n} + \frac{3}{2} + q$
수열 $\{ a_{n} + b_{n} \}$이 수렴하려면 $\frac{1}{2} - p = 0$, 즉 $p = \frac{1}{2}$이어야 한다.
$p = \frac{1}{2}$일 때,
$a_{n} b_{n} = \{ \frac{1}{2} \times (-1)^{n} + \frac{3}{2} \} \{ \frac{1}{2} \times (-1)^{n+1} + q \}$
$= \{ \frac{1}{2} \times (-1)^{n} + \frac{3}{2} \} \{ -\frac{1}{2} \times (-1)^{n} + q \}$
$= -\frac{1}{4} \times (-1)^{2n} + (\frac{q}{2} - \frac{3}{4}) (-1)^{n} + \frac{3}{2} q$
$= (\frac{q}{2} - \frac{3}{4}) (-1)^{n} - \frac{1}{4} + \frac{3}{2} q$
수열 $\{ a_{n} b_{n} \}$이 수렴하려면 $\frac{q}{2} - \frac{3}{4} = 0$, 즉 $q = \frac{3}{2}$이어야 한다.
$\lim_{n \to \infty} (a_{n} + b_{n}) = \frac{3}{2} + q = \frac{3}{2} + \frac{3}{2} = 3$
$\lim_{n \to \infty} a_{n} b_{n} = -\frac{1}{4} + \frac{3}{2} q = -\frac{1}{4} + \frac{9}{4} = 2$
따라서 $\lim_{n \to \infty} \{ (a_{n})^{2} + (b_{n})^{2} \} = \lim_{n \to \infty} \{ (a_{n} + b_{n})^{2} - 2a_{n} b_{n} \}$
$= \lim_{n \to \infty} (a_{n} + b_{n})^{2} - \lim_{n \to \infty} 2a_{n} b_{n}$
$= 3^{2} - 2 \times 2 = 5$
