## 문제
$3 ( \cos 60^{\circ} + \sin 30^{\circ} )^{2} + \tan 60^{\circ} \times \tan 30^{\circ}$의 값을 구하시오.

## 풀이
$3 ( \cos 60^{\circ} + \sin 30^{\circ} )^{2} + \tan 60^{\circ} \times \tan 30^{\circ}$
$= 3 ( \frac{1}{2} + \frac{1}{2} )^{2} + \sqrt{3} \times \frac{1}{\sqrt{3}}$
$= 3 \times 1^{2} + 1$
$= 3 + 1$
$= 4$

정답: $4$
