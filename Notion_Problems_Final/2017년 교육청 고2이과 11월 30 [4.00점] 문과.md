## 문제
최고차항의 계수가 양수인 사차함수 $f(x)$에서 $x$의 값이 $1$에서 $t$ ($t>1$)까지 변할 때의 평균변화율을 $g(t)$라 정의할 때, 함수 $g(t)$는 $t=2$에서 극댓값 $0$을 갖는다. $g(t)$의 최솟값이 존재할 때, 방정식 $f(x)=f(1)$의 서로 다른 실근의 합의 최솟값을 구하시오.

## 보기

## 풀이
$f(x)$에서 $x$의 값이 $1$에서 $t$ ($t>1$)까지 변할 때의 평균변화율 $g(t)$는 
$$g(t) = \frac{f(t)-f(1)}{t-1}$$ 
함수 $g(t)$가 $t=2$에서 극댓값 $0$을 가지므로 $f(1)=f(2)$이고, 함수 $f(x)$는 $x=2$에서 극댓값을 갖는다. 
그러므로 함수 $f(x)$는 
$f(x) = k(x-1)(x-2)^{2}(x-\alpha) + f(1)$ (단, $k>0$, $\alpha>2$) 
이고, 함수 $g(t)$는 $g(t) = k(t-2)^{2}(t-\alpha)$이다. 
또한, 함수 $g(t)$의 최솟값이 존재하기 위해서는 $\lim_{t \to 1+} g(t) \ge g(p)$를 만족시키는 $p$ ($p>2$)가 존재해야 한다. 
따라서 $\lim_{t \to 1+} g(t) = g(p)$를 만족시키는 $p$를 $a$라 하면 $p=a$일 때, 방정식 $f(x)=f(1)$의 실근의 합은 최소가 된다. $\cdots\cdots$ ㉠ 
두 함수 $f(x)$와 $g(t)$의 그래프의 개형은 다음과 같다.

함수 $g(t) = k(t-2)^{2}(t-\alpha)$에 대하여 
$g'(t) = 2k(t-2)(t-\alpha) + k(t-2)^{2}$ 
$= k(t-2)\{3t - (2\alpha+2)\}$ 
$g'(t)=0$에서 $t=2$, $\frac{2\alpha+2}{3}$ 
따라서 함수 $g(t)$는 $t = \frac{2\alpha+2}{3}$에서 극소이며 $a = \frac{2\alpha+2}{3}$ 
㉠에 의하여 
$\lim_{t \to 1+} g(t) = g\left(\frac{2\alpha+2}{3}\right)$ 
$k(1-\alpha) = k\left(\frac{2\alpha+2}{3}-2\right)^{2}\left(\frac{2\alpha+2}{3}-\alpha\right)$ 
$27(1-\alpha) = -4(\alpha-2)^{3}$ 
$4\alpha^{3}-24\alpha^{2}+21\alpha-5=0$ 
$(\alpha-5)(4\alpha^{2}-4\alpha+1)=0$ 
$\therefore \alpha=5$ ($\because \alpha>2$) 
방정식 $f(x)=f(1)$의 서로 다른 실근의 합이 최소일 때, 
$f(x) = k(x-1)(x-2)^{2}(x-5) + f(1)$ 
따라서 서로 다른 실근의 합의 최솟값은 
$1+2+5=8$
