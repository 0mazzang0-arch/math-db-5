## 문제
$\tan ( \alpha - \beta ) = \frac{7}{8}$, $\tan \beta = 1$ 일 때, $\tan \alpha$ 의 값을 구하시오. (단, $0 < \alpha < \frac{\pi}{2}$, $0 < \beta < \frac{\pi}{2}$)

## 보기

## 풀이
삼각함수의 덧셈정리에 의하여
$\tan ( \alpha - \beta ) = \frac{\tan \alpha - \tan \beta}{1 + \tan \alpha \tan \beta} = \frac{7}{8}$
이 식에 $\tan \beta = 1$ 을 대입하면
$\frac{\tan \alpha - 1}{1 + \tan \alpha \cdot 1} = \frac{7}{8}$
양변에 $8 ( 1 + \tan \alpha )$ 를 곱하여 정리하면
$8 ( \tan \alpha - 1 ) = 7 ( 1 + \tan \alpha )$
$8 \tan \alpha - 8 = 7 + 7 \tan \alpha$
$\tan \alpha = 15$

정답: $15$
