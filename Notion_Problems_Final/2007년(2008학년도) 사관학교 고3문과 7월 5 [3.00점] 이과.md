## 문제
다음과 같이 귀납적으로 정의된 수열 $\{ a_{n} \}$ 이 있다.
$a_{1} = 2$, $a_{n+1} a_{n} = ( \frac{1}{4} )^{n}$ ($n = 1, 2, 3, \dots$)
이때, $\sum_{n=1}^{\infty} a_{2n-1} + \sum_{n=1}^{\infty} a_{2n}$ 의 값은?

## 보기
① $\frac{17}{6}$   ② $\frac{19}{6}$   ③ $\frac{7}{2}$   ④ $\frac{23}{6}$   ⑤ $\frac{25}{6}$

## 풀이
주어진 점화식에 $n$ 대신 $2n-1$ 을 대입하면
$a_{2n} a_{2n-1} = ( \frac{1}{4} )^{2n-1}$ $\cdots$ (1)

또, 주어진 점화식에 $n$ 대신 $2n$ 을 대입하면
$a_{2n+1} a_{2n} = ( \frac{1}{4} )^{2n}$ $\cdots$ (2)

(1)식을 정리해서 (2)식에 대입해서 점화식을 유도하면
$a_{2n+1} = \frac{1}{4} a_{2n-1}$

주어진 수열 $\{ a_{2n-1} \}$ 은 초항 $a_{1} = 2$ 이고 공비가 $\frac{1}{4}$ 인 등비수열이다.
$\sum_{n=1}^{\infty} a_{2n-1} = \frac{2}{1 - \frac{1}{4}} = \frac{2}{\frac{3}{4}} = \frac{8}{3}$

마찬가지로, 짝수항들에 대해서도 점화식을 유도하면
$a_{2} a_{1} = ( \frac{1}{4} )^{1}$ 에서 $a_{2} \cdot 2 = \frac{1}{4}$이므로 $a_{2} = \frac{1}{8}$ 이고,
공비가 $\frac{1}{4}$ 인 등비수열이 된다.
$\sum_{n=1}^{\infty} a_{2n} = \frac{\frac{1}{8}}{1 - \frac{1}{4}} = \frac{\frac{1}{8}}{\frac{3}{4}} = \frac{1}{6}$

$\therefore$ 구하라는 수열의 합 $= \frac{8}{3} + \frac{1}{6} = \frac{17}{6}$

정답: ①
