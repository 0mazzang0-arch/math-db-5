## 문제
최고차항의 계수가 $1$인 삼차함수 $f(x)$에 대하여 곡선 $y=f(x)$ 위의 점 $(2, 4)$에서의 접선이 점 $(-1, 1)$에서 이 곡선과 만날 때, $f'(3)$의 값은?



> [!example] 📸 (그림/그래프)

## 보기
① $10$   ② $11$   ③ $12$   ④ $13$   ⑤ $14$

## 풀이
$f(x) = x^3 + ax^2 + bx + c$라 하면
$f'(x) = 3x^2 + 2ax + b$
이고, 두 점 $(2, 4)$, $(-1, 1)$을 지나는 직선의 방정식이 $y = x + 2$이므로

$f(2) = 4$에서 $4a + 2b + c = -4$ $\cdots\cdots$ ㉠
$f(-1) = 1$에서 $a - b + c = 2$ $\cdots\cdots$ ㉡
$f'(2) = 1$에서 $4a + b = -11$ $\cdots\cdots$ ㉢

㉠, ㉡, ㉢을 연립하여 풀면
$a = -3$, $b = 1$, $c = 6$

$f'(x) = 3x^2 - 6x + 1$
따라서 $f'(3) = 10$

**[다른 풀이]**
두 점 $(2, 4)$, $(-1, 1)$을 지나는 직선의 방정식이 $y = x + 2$이므로 곡선 $y=f(x)$와 접선 $y=x+2$의 관계에 의해
$f(x) - (x + 2) = (x - 2)^2 (x + 1)$
$f(x) = (x - 2)^2 (x + 1) + x + 2$
$f(x) = x^3 - 3x^2 + x + 6$

$f'(x) = 3x^2 - 6x + 1$
따라서 $f'(3) = 10$
