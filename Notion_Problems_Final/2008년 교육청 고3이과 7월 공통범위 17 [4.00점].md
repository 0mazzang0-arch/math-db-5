## 문제
$0 \le x \le 1$ 에서 정의된 다항함수 $f(x)$ 가 다음 두 조건을 만족한다.

Ⅰ. $1 < f(x) < 2$
Ⅱ. $x_{1} < x_{2}$ 이면 $f(x_{1}) < f(x_{2})$

이때, <보기>에서 옳은 것을 모두 고른 것은?

<보기>
ㄱ. $0 < x < 1$ 인 임의의 실수 $x$ 에 대하여 $f'(x) \ge 0$ 이다.
ㄴ. 방정식 $f(x) - 2x = 0$ 의 해가 열린구간 $(0, 1)$ 에 적어도 한 개 존재한다.
ㄷ. $\sum_{k=0}^{n-1} f \left( \frac{k}{n} \right) \frac{1}{n} < \int_{0}^{1} f(x) dx < \sum_{k=1}^{n} f \left( \frac{k}{n} \right) \frac{1}{n}$

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
ㄱ. 조건 Ⅱ에 의해 $f(x)$ 는 증가함수이다. $f(x)$ 가 다항함수이므로 모든 구간에서 미분가능하며, 증가함수의 정의에 따라 $0 < x < 1$ 인 임의의 실수 $x$ 에 대하여 $f'(x) \ge 0$ 이다. (참)

ㄴ. $g(x) = f(x) - 2x$ 라 하면 $f(x)$ 가 다항함수이므로 $g(x)$ 는 $[0, 1]$ 에서 연속이다.
조건 Ⅰ에 의해 $1 < f(0) < 2$ 이므로 $g(0) = f(0) - 2(0) = f(0) > 1 > 0$ 이다.
또한 $1 < f(1) < 2$ 이므로 $g(1) = f(1) - 2(1) = f(1) - 2 < 0$ 이다.
$g(0) > 0$ 이고 $g(1) < 0$ 이므로 사잇값 정리에 의해 $g(x) = 0$ 인 $x$ 가 열린구간 $(0, 1)$ 에 적어도 한 개 존재한다. (참)

ㄷ. $f(x)$ 가 증가함수이므로 구간 $\left[ \frac{k}{n}, \frac{k+1}{n} \right]$ 에서 $f \left( \frac{k}{n} \right) \le f(x) \le f \left( \frac{k+1}{n} \right)$ 이 성립한다.
각 변을 구간 $\left[ \frac{k}{n}, \frac{k+1}{n} \right]$ 에 대해 적분하면
$\frac{1}{n} f \left( \frac{k}{n} \right) < \int_{\frac{k}{n}}^{\frac{k+1}{n}} f(x) dx < \frac{1}{n} f \left( \frac{k+1}{n} \right)$
이를 $k=0$ 부터 $k=n-1$ 까지 합하면
$\sum_{k=0}^{n-1} f \left( \frac{k}{n} \right) \frac{1}{n} < \int_{0}^{1} f(x) dx < \sum_{k=1}^{n} f \left( \frac{k}{n} \right) \frac{1}{n}$ 이 성립한다. (참)

따라서 ㄱ, ㄴ, ㄷ 모두 옳다.
