## 문제
두 함수 $f(x) = x^2 - ax + b$ ($a > 0$), $g(x) = x^2 e^{-\frac{x}{2}}$에 대하여 상수 $k$와 함수 $h(x) = (f \circ g)(x)$가 다음 조건을 만족시킨다.

(가) $h(0) < h(4)$
(나) 방정식 $|h(x)| = k$의 서로 다른 실근의 개수는 $7$이고, 그 중 가장 큰 실근을 $\alpha$라 할 때 함수 $h(x)$는 $x = \alpha$에서 극소이다.

$f(1) = -\frac{7}{32}$일 때, 두 상수 $a, b$에 대하여 $a + 16b$의 값을 구하시오. (단, $\frac{5}{2} < e < 3$이고, $\lim_{x \to \infty} g(x) = 0$이다.)

## 풀이
$g'(x) = \frac{1}{2}x(4 - x)e^{-\frac{x}{2}}$ 이므로 $g(x)$는 $x = 0$에서 극솟값 $g(0) = 0$, $x = 4$에서 극댓값 $g(4) = \frac{16}{e^2}$을 갖는다.
또한, $\lim_{x \to \infty} g(x) = 0$, $\lim_{x \to -\infty} g(x) = \infty$이다.

> [!example] 📸 (그림/그래프)

> [!example] 📸 (그림/그래프)

(가)에서 $h(0) = f(g(0)) = f(0) = b$, $h(4) = f(g(4)) = f(\frac{16}{e^2}) > b$이므로, 이차함수 $f(x)$의 대칭축 $x = \frac{a}{2}$에 대하여 $0 < \frac{a}{2} < \frac{8}{e^2}$ 즉, $0 < a < \frac{16}{e^2}$이다.
(단, 조건에서 $\frac{5}{2} < e < 3$ 이므로 $\frac{16}{9} < \frac{16}{e^2} < \frac{16}{6.25} = 2.56$)

(나)를 만족시키려면 $h(x) = f(g(x))$의 그래프는 다음과 같아야 한다.

> [!example] 📸 (그림/그래프)

$f'(x) = 2x - a$
$g'(x) = \frac{1}{2}x(4 - x)e^{-\frac{x}{2}}$
$h'(x) = g'(x)f'(g(x)) = \frac{1}{2}x(4 - x)e^{-\frac{x}{2}} \{2g(x) - a\}$

$g(x) = \frac{a}{2}$ ($0 < \frac{a}{2} < \frac{8}{e^2}$) 인 지점에서 $h(x)$는 극솟값을 가지며, 조건 (나)에 의해 가장 큰 실근 $\alpha$에서 $h(x)$가 극소이면서 $|h(x)| = k$를 만족해야 하므로 $h(\alpha) = f(\frac{a}{2}) = -k$여야 한다.
또한 $x \to \infty$일 때 $g(x) \to 0$이므로 $h(x) \to f(0) = b$이고, 그래프 개형에 따라 $b = k$임을 알 수 있다.
따라서 $f(\frac{a}{2}) = -b$이다.

$f(x) = (x - \frac{a}{2})^2 + b - \frac{a^2}{4}$ 이므로 $f(\frac{a}{2}) = b - \frac{a^2}{4} = -b$
$\frac{a^2}{4} = 2b$, 즉 $b = \frac{a^2}{8}$

한편, $f(1) = 1 - a + b = 1 - a + \frac{a^2}{8} = -\frac{7}{32}$ 이므로
$32 - 32a + 4a^2 = -7$
$4a^2 - 32a + 39 = 0$
$(2a - 13)(2a - 3) = 0$

$0 < a < \frac{16}{e^2}$ 이므로 $a = \frac{3}{2}$ 이고, $b = \frac{9}{32}$ 이다.

$\therefore a + 16b = \frac{3}{2} + 16 \times \frac{9}{32} = \frac{3}{2} + \frac{9}{2} = 6$

정답: $6$
