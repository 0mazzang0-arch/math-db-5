## 문제
집합 $X = \{1, 2, 3, 4, 5, 6\}$에 대하여 다음 조건을 만족시키는 함수 $f : X \to X$의 개수는?

(가) $f(1) \times f(6)$의 값이 $6$의 약수이다.
(나) $2f(1) \le f(2) \le f(3) \le f(4) \le f(5) \le 2f(6)$

## 보기
① $166$   ② $171$   ③ $176$   ④ $181$   ⑤ $186$

## 풀이
조건 (가), (나)에 의하여
(ⅰ) $f(1) = 1$, $f(6) = 1$일 때
$2 \le f(2) = f(3) = f(4) = f(5) \le 2$
$f$의 개수는 $1$

(ⅱ) $f(1) = 1$, $f(6) = 2$일 때
$2 \le f(2) \le f(3) \le f(4) \le f(5) \le 4$
$f$의 개수는 ${}_{3}H_{4} = {}_{6}C_{4} = 15$

(ⅲ) $f(1) = 1$, $f(6) = 3$일 때
$2 \le f(2) \le f(3) \le f(4) \le f(5) \le 6$
$f$의 개수는 ${}_{5}H_{4} = {}_{8}C_{4} = 70$

(ⅳ) $f(1) = 1$, $f(6) = 6$일 때
공역의 원소가 $6$ 이하이므로 $2 \le f(2) \le f(3) \le f(4) \le f(5) \le 6$
$f$의 개수는 ${}_{5}H_{4} = {}_{8}C_{4} = 70$

(ⅴ) $f(1) = 2$, $f(6) = 3$일 때
$4 \le f(2) \le f(3) \le f(4) \le f(5) \le 6$
$f$의 개수는 ${}_{3}H_{4} = {}_{6}C_{4} = 15$

이상에서 구하는 함수 $f$의 개수는
$1 + 15 + 70 + 70 + 15 = 171$
