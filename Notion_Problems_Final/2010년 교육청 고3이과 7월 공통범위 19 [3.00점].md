## 문제
연속확률변수 $X$의 확률밀도함수 $f(x)$는 다음과 같다.

$f(x) = 1 - ax$ ($1 \le x \le 3$)

확률 $P(1 \le X \le 2) = \frac{q}{p}$일 때, $p + q$의 값을 구하시오. (단, $a$는 상수이며, $p$와 $q$는 서로소인 자연수이다.)



> [!example] 📸 (그림/그래프)

## 풀이
> [!예시] 📸 (그림/그래프)

$f(x) = 1 - ax$가 확률밀도함수이므로 전체 구간에서의 넓이는 $1$이어야 한다.
$\frac{1}{2} \times (f(1) + f(3)) \times (3 - 1) = 1$
$\frac{1}{2} \times (1 - a + 1 - 3a) \times 2 = 1$
$2 - 4a = 1$
$a = \frac{1}{4}$이다.

$P(1 \le X \le 2)$는 그래프 아래 어두운 부분의 넓이와 같으므로
$P(1 \le X \le 2) = \frac{1}{2} \times (f(1) + f(2)) \times (2 - 1)$
$P(1 \le X \le 2) = \frac{1}{2} \times \left( \frac{3}{4} + \frac{2}{4} \right) \times 1 = \frac{5}{8}$
따라서 $p = 8$, $q = 5$이므로
$\therefore p + q = 13$

[정답] $13$
