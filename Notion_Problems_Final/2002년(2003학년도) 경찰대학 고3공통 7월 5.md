## 문제
일차함수 $f(x)$ 에 대하여 $-1 \leq f(0) \leq 1$, $0 \leq f(1) \leq 2$ 일 때, $f(2)$ 의 범위는?

## 보기
① $1 \leq f(2) \leq 3$   ② $0 \leq f(2) \leq 3$   ③ $-1 \leq f(2) \leq 5$   ④ $0 \leq f(2) \leq 4$   ⑤ $-3 \leq f(2) \leq -1$

## 풀이
[정답] ③

일차함수 $f(x) = ax + b$ 라 하면,
주어진 조건에 의하여
$-1 \leq b \leq 1 \cdots$ ①
$0 \leq a + b \leq 2 \cdots$ ②

$f(2) = 2a + b$ 의 범위는
$2 \times$ ② $-$ ① 에 의하여
$-1 \leq 2a + b \leq 5$
