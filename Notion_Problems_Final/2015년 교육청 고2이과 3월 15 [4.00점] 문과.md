## 문제
다항식 $P(x)$를 $x-2$로 나누었을 때의 몫이 $Q(x)$, 나머지는 $3$이고, 다항식 $Q(x)$를 $x-1$로 나누었을 때의 나머지는 $2$이다. $P(x)$를 $(x-1)(x-2)$로 나누었을 때의 나머지를 $R(x)$라 하자. $R(3)$의 값은?

## 보기
① $5$   ② $7$   ③ $9$   ④ $11$   ⑤ $13$

## 풀이
다항식 $P(x)$를 $x-2$로 나누었을 때의 몫이 $Q(x)$이고 나머지가 $3$이므로 
$P(x) = (x-2)Q(x) + 3 \cdots \cdots$ ㉠ 

다항식 $Q(x)$를 $x-1$로 나누었을 때의 몫을 $Q_{1}(x)$라 하면 나머지가 $2$이므로 
$Q(x) = (x-1)Q_{1}(x) + 2 \cdots \cdots$ ㉡ 

㉡을 ㉠에 대입하여 정리하면 
$P(x) = (x-2)\{(x-1)Q_{1}(x) + 2\} + 3$ 
$= (x-2)(x-1)Q_{1}(x) + 2(x-2) + 3$ 
$= (x-1)(x-2)Q_{1}(x) + 2x-1$ 

따라서 $P(x)$를 $(x-1)(x-2)$로 나누었을 때의 나머지는 $R(x) = 2x-1$이므로 
$R(3) = 2 \times 3 - 1 = 5$

**[다른 풀이]**
다항식 $P(x)$를 $x-2$로 나누었을 때의 몫이 $Q(x)$이고 나머지가 $3$이므로 
$P(x) = (x-2)Q(x) + 3 \cdots \cdots$ ㉠ 

위 식의 양변에 $x=2$를 대입하면 $P(2) = 3 \cdots \cdots$ ㉡ 

다항식 $Q(x)$를 $x-1$로 나누었을 때의 몫을 $Q_{1}(x)$라 하면 나머지가 $2$이므로 
$Q(x) = (x-1)Q_{1}(x) + 2$ 

위 식의 양변에 $x = 1$을 대입하면 $Q(1) = 2$ 
따라서 ㉠의 양변에 $x = 1$을 대입하면 
$P(1) = (1-2)Q(1) + 3 = -1 \times 2 + 3 = 1$ 
$\therefore P(1) = 1 \cdots \cdots$ ㉢ 

$P(x)$를 $(x-1)(x-2)$로 나누었을 때의 몫을 $Q_{2}(x)$, $R(x) = ax+b$ ($a$, $b$는 상수)라 하면 
$P(x) = (x-1)(x-2)Q_{2}(x) + ax+b$ 

위 식의 양변에 $x = 1$, $x = 2$를 각각 대입하면 
$P(1) = a+b = 1$, $P(2) = 2a+b = 3$ ($\because$ ㉡, ㉢) 

위 두 식을 연립하면 $a=2$, $b=-1$ 
따라서 $R(x) = 2x-1$이므로 $R(3) = 5$
