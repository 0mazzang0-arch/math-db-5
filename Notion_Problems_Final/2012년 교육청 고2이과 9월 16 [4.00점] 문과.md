## 문제
자연수 $n$과 양수 $A$에 대하여 이차방정식 $x^{2} - \left( 3n + \frac{1}{3n} \right) x + 1 = 0$의 한 근은 $\log A^{3}$의 정수부분이고, 다른 한 근은 $\log A^{2}$의 소수부분이다. <보기>에서 옳은 것만을 있는 대로 고른 것은?

$$ \begin{array}{|c|} \hline \text{<보기>} \\ \text{ㄱ. } n=1 \text{ 이면 } \log A^{3} \text{의 정수부분은 } 3 \text{ 이다.} \\ \text{ㄴ. } \log A \text{의 소수부분은 } \frac{1}{6n} \text{ 이다.} \\ \text{ㄷ. } A^{12} \text{ 이 자연수가 되도록 하는 } n \text{의 개수는 } 2 \text{ 이다.} \\ \hline \end{array} $$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
이차방정식 $x^{2} - \left( 3n + \frac{1}{3n} \right) x + 1 = 0$의 두 근은 $(x - 3n) \left( x - \frac{1}{3n} \right) = 0$에서 $3n$과 $\frac{1}{3n}$이다.
$n$이 자연수이므로 $3n$은 정수이고, $0 < \frac{1}{3n} \leq \frac{1}{3}$이다.
따라서 $\log A^{3}$의 정수부분은 $3n$이고, $\log A^{2}$의 소수부분은 $\frac{1}{3n}$이다.

$\log A^{3} = 3n + \alpha$ ($0 \leq \alpha < 1$)라 하면,
$\log A = n + \frac{\alpha}{3}$
$\log A^{2} = 2n + \frac{2}{3} \alpha$
이때 $0 \leq \frac{2}{3} \alpha < \frac{2}{3}$이므로 $\log A^{2}$의 소수부분은 $\frac{2}{3} \alpha$이다.
조건에 의해 $\frac{2}{3} \alpha = \frac{1}{3n}$이므로 $\alpha = \frac{1}{2n}$이다.
즉, $\log A = n + \frac{1}{6n}$이다.

ㄱ. $n=1$일 때 $\log A = 1 + \frac{1}{6}$이므로 $\log A^{3} = 3 \left( 1 + \frac{1}{6} \right) = 3 + \frac{1}{2}$이다.
따라서 $\log A^{3}$의 정수부분은 $3$이다. (참)

ㄴ. $\log A = n + \frac{1}{6n}$에서 $n$은 정수이고 $n \geq 1$일 때 $0 < \frac{1}{6n} \leq \frac{1}{6} < 1$이므로 $\log A$의 소수부분은 $\frac{1}{6n}$이다. (참)

ㄷ. $\log A^{12} = 12 \log A = 12 \left( n + \frac{1}{6n} \right) = 12n + \frac{2}{n}$이다.
$A^{12} = 10^{12n + \frac{2}{n}}$이 자연수가 되려면 지수인 $12n + \frac{2}{n}$가 $0$ 이상의 정수여야 한다.
$n$은 자연수이므로 $n$은 $2$의 약수여야 한다.
따라서 $n=1$ 또는 $n=2$로 그 개수는 $2$이다. (참)

옳은 것은 ㄱ, ㄴ, ㄷ이다.
