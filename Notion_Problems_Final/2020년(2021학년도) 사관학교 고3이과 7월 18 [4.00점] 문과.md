## 문제
수열 $\{ a_n \}$이 모든 자연수 $n$에 대하여 다음 조건을 만족시킨다.

(가) $a_{2n+1} = -a_n + 3a_{n+1}$
(나) $a_{2n+2} = a_n - a_{n+1}$

$a_1 = 1$, $a_2 = 2$일 때, $\sum_{n=1}^{16} a_n$의 값은?

## 보기
① $31$   ② $33$   ③ $35$   ④ $37$   ⑤ $39$

## 풀이
(가), (나)에서
$a_{2n+1} = -a_n + 3a_{n+1} \cdots$ ㉠
$a_{2n+2} = a_n - a_{n+1} \cdots$ ㉡

㉠ $+$ ㉡을 하면
$a_{2n+1} + a_{2n+2} = 2a_{n+1}$

$n=1$ 일 때, $a_3 + a_4 = 2a_2$
$n=2$ 일 때, $a_5 + a_6 = 2a_3$
$\vdots$
$n=n-1$ 일 때, $a_{2n-1} + a_{2n} = 2a_n$

변변히 모두 더하면
$S_{2n} - a_1 - a_2 = 2(S_n - a_1)$
즉, $S_{2n} = 2S_n - a_1 + a_2$

그런데 조건에서 $a_1 = 1$, $a_2 = 2$이므로
$S_{2n} = 2S_n - 1 + 2 = 2S_n + 1$

$S_2 = a_1 + a_2 = 1 + 2 = 3$ 이므로
$S_4 = 2S_2 + 1 = 2(3) + 1 = 7$
$S_8 = 2S_4 + 1 = 2(7) + 1 = 15$
$S_{16} = 2S_8 + 1 = 2(15) + 1 = 31$

따라서 정답은 ①이다.
