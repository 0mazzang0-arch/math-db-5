## 문제
두 집합 $X = \{1, 2, 3, 4, 5\}$, $Y = \{1, 2, 3, 4\}$에 대하여 다음 조건을 만족시키는 함수 $f: X \rightarrow Y$의 개수는?

(가) $f(1) + f(2) = 4$
(나) $1$은 함수 $f$의 치역의 원소이다.

## 보기
① $145$   ② $150$   ③ $155$   ④ $160$   ⑤ $165$

## 풀이
$f(1) + f(2) = 4$를 만족시키는 경우는 $f(1) = 1$, $f(2) = 3$ 또는 $f(1) = f(2) = 2$ 또는 $f(1) = 3$, $f(2) = 1$의 $3$가지이다.

(ⅰ) $f(1) = 1$, $f(2) = 3$ 또는 $f(1) = 3$, $f(2) = 1$인 경우
조건 (나)에서 $1$은 이미 치역의 원소이므로, $f(3)$, $f(4)$, $f(5)$의 값을 정하는 경우의 수는 $1$, $2$, $3$, $4$ 중에서 $3$개를 택하는 중복순열의 수와 같다.
${}_{4} \Pi_{3} = 64$
따라서 함수 $f$의 개수는 $2 \times 64 = 128$

(ⅱ) $f(1) = f(2) = 2$인 경우
$f(1) = 2, f(2) = 2$이므로 조건 (나)를 만족시키기 위해 $f(3)$, $f(4)$, $f(5)$의 값 중 적어도 하나는 $1$이어야 한다.
그러므로 $f(3)$, $f(4)$, $f(5)$의 값을 정하는 경우의 수는 $1$, $2$, $3$, $4$ 중에서 $3$개를 택하는 중복순열의 수에서, $1$을 제외한 $2$, $3$, $4$ 중에서 $3$개를 택하는 중복순열의 수를 뺀 것과 같다.
함수 $f$의 개수는 ${}_{4} \Pi_{3} - {}_{3} \Pi_{3} = 64 - 27 = 37$

(ⅰ), (ⅱ)에 의하여 구하는 모든 함수 $f$의 개수는
$128 + 37 = 165$

정답: ⑤
