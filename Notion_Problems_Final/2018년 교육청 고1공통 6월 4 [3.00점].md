## 문제
등식 $x^{3} - x^{2} + x + 3 = (x - 1)(x^{2} + 1) + a$ 가 $x$ 에 대한 항등식일 때, 상수 $a$ 의 값은?

## 보기
① $2$   ② $4$   ③ $6$   ④ $8$   ⑤ $10$

## 풀이
[정답] ②

[해설]
등식 $x^{3} - x^{2} + x + 3 = (x - 1)(x^{2} + 1) + a$ 가 $x$ 에 대한 항등식이므로 $x$ 에 어떤 값을 대입하여도 항상 참이 되어야 한다.
$x = 1$ 을 대입하면 $1 - 1 + 1 + 3 = a$ 이다. 따라서 $a = 4$ 이다.

[다른 풀이]
등식의 우변을 정리하면 $x^{3} - x^{2} + x + 3 = x^{3} - x^{2} + x - 1 + a$ 이다. 항등식의 성질을 이용하여 양변의 동류항을 비교하면 $3 = -1 + a$ 이다. 따라서 $a = 4$ 이다.
