## 문제
$0 < a < \frac{4}{7}$인 실수 $a$와 유리수 $b$에 대하여 닫힌구간 $[ - \frac{\pi}{a} , \frac{2 \pi}{a} ]$에서 정의된 함수 $f(x) = 2 \sin(ax) + b$가 있다. 함수 $y = f(x)$의 그래프가 두 점 $A( - \frac{\pi}{2} , 0 )$, $B( \frac{7}{2} \pi , 0 )$을 지날 때, $30( a+b )$의 값을 구하시오.

## 보기

## 풀이
닫힌구간 $[ - \frac{\pi}{a} , \frac{2 \pi}{a} ]$에서 $0 < a < \frac{4}{7}$이므로 $- \frac{\pi}{a} < - \frac{7}{4} \pi$, $\frac{7 \pi}{2} < \frac{2 \pi}{a}$이다.
함수 $f(x) = 2 \sin(ax) + b$의 그래프가 두 점 $A( - \frac{\pi}{2} , 0 )$, $B( \frac{7}{2} \pi , 0 )$을 지나므로 

$f( - \frac{\pi}{2} ) = 2 \sin( - \frac{a}{2} \pi ) + b = -2 \sin( \frac{a}{2} \pi ) + b = 0$
$f( \frac{7 \pi}{2} ) = 2 \sin( \frac{7a}{2} \pi ) + b = 0$

따라서 $\sin( \frac{7a \pi}{2} ) = -\sin( \frac{a}{2} \pi )$
$0 < a < \frac{4}{7}$에서 $0 < \frac{a}{2} \pi < \frac{2}{7} \pi$, $0 < \frac{7a}{2} \pi < 2 \pi$이므로 
$\frac{7a \pi}{2} = 2 \pi - \frac{a}{2} \pi$ 또는 $\frac{7a \pi}{2} = \pi + \frac{a}{2} \pi$
따라서 $a = \frac{1}{2}$ 또는 $a = \frac{1}{3}$

(ⅰ) $a = \frac{1}{2}$ 일 때 
$f(x) = 2 \sin( \frac{1}{2} x ) + b$에서 
$f( - \frac{\pi}{2} ) = 2 \sin( - \frac{\pi}{4} ) + b = 2 \times ( - \frac{\sqrt{2}}{2} ) + b = - \sqrt{2} + b = 0$
이므로 $b = \sqrt{2}$
이는 $b$는 유리수라는 조건을 만족시키지 않는다. 

(ⅱ) $a = \frac{1}{3}$ 일 때 
$f(x) = 2 \sin( \frac{1}{3} x ) + b$에서 
$f( - \frac{\pi}{2} ) = 2 \sin( - \frac{\pi}{6} ) + b = 2 \times ( - \frac{1}{2} ) + b = - 1 + b = 0$
이므로 $b = 1$
이때 $f( \frac{7}{2} \pi ) = 0$이다. 

(ⅰ), (ⅱ)에서 $a = \frac{1}{3}$, $b = 1$이고
$30( a+b ) = 30 \times ( \frac{1}{3} + 1 ) = 40$

정답: $40$
