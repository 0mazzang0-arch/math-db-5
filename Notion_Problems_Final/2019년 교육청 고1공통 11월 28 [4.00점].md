## 문제
두 함수 $f(x) = x + a$, 
$$ g(x) = \left\{ \begin{array}{ll} 2x - 6 & (x < a) \\ x^{2} & (x \geq a) \end{array} \right. $$
에 대하여 
$(g \circ f)(1) + (f \circ g)(4) = 57$을 만족시키는 모든 실수 $a$의 값의 합을 $S$라 할 때, $10S^{2}$의 값을 구하시오.

## 보기

## 풀이
$(g \circ f)(1) = g(a + 1)$
이때 $a + 1 \geq a$이므로 $g(a + 1) = (a + 1)^{2}$이다.

$1)$ $a \leq 4$일 때
$(f \circ g)(4) = f(16) = a + 16$
$(g \circ f)(1) + (f \circ g)(4) = (a + 1)^{2} + a + 16 = 57$
$a^{2} + 3a + 17 = 57$
$a^{2} + 3a - 40 = (a - 5)(a + 8) = 0$
$\therefore a = -8$ (단, $a \leq 4$)

$2)$ $a > 4$일 때
$(f \circ g)(4) = f(2) = a + 2$
$(g \circ f)(1) + (f \circ g)(4) = (a + 1)^{2} + a + 2 = 57$
$a^{2} + 3a + 3 = 57$
$a^{2} + 3a - 54 = (a - 6)(a + 9) = 0$
$\therefore a = 6$ (단, $a > 4$)

모든 실수 $a$의 값의 합 $S = -8 + 6 = -2$
따라서 $10S^{2} = 10 \times (-2)^{2} = 40$
