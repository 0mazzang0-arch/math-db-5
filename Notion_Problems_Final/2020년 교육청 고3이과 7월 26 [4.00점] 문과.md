## 문제
삼각형 $ABC$에 대하여 $\angle A = \alpha$, $\angle B = \beta$, $\angle C = \gamma$라 할 때, $\alpha$, $\beta$, $\gamma$가 이 순서대로 등차수열을 이루고 $\cos \alpha$, $2 \cos \beta$, $8 \cos \gamma$가 이 순서대로 등비수열을 이룰 때, $\tan \alpha \tan \gamma$의 값을 구하시오. (단, $\alpha < \beta < \gamma$)

## 보기


## 풀이
$\alpha$, $\beta$, $\gamma$가 삼각형 $ABC$의 세 내각의 크기이므로
$\alpha + \beta + \gamma = \pi \cdots \cdots$ ㉠

$\alpha$, $\beta$, $\gamma$가 이 순서대로 등차수열을 이루므로
$\beta = \frac{\alpha + \gamma}{2}$, $\alpha + \gamma = 2\beta \cdots \cdots$ ㉡

㉠, ㉡에서
$3\beta = \pi$, $\beta = \frac{\pi}{3}$

$\beta = \frac{\pi}{3}$이므로 $\alpha + \gamma = \frac{2\pi}{3}$에서
$\cos (\alpha + \gamma) = \cos \frac{2\pi}{3} = -\frac{1}{2}$
$\cos \alpha \cos \gamma - \sin \alpha \sin \gamma = -\frac{1}{2} \cdots \cdots$ ㉢

$\cos \alpha$, $2 \cos \beta$, $8 \cos \gamma$가 이 순서대로 등비수열을 이루므로
$(2 \cos \beta)^{2} = 8 \cos \alpha \cos \gamma$
$\cos \alpha \cos \gamma = \frac{1}{8} \cdots \cdots$ ㉣

㉢, ㉣에서
$\sin \alpha \sin \gamma = \frac{5}{8} \cdots \cdots$ ㉤

따라서
$\tan \alpha \tan \gamma = \frac{\sin \alpha \sin \gamma}{\cos \alpha \cos \gamma} = 5$

정답: $5$
