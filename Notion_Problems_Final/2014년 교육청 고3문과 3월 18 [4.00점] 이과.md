## 문제
수열 $\{ a_{n} \}$ 에 대하여 $\sum_{n=1}^{20} a_{n} = p$ 라 할 때, 등식 $2 a_{n} + n = p$ ($n \geq 1$) 가 성립한다. $a_{10}$ 의 값은? (단, $p$ 는 상수이다.)

## 보기
① $\frac{2}{3}$   ② $\frac{3}{4}$   ③ $\frac{5}{6}$   ④ $\frac{11}{12}$   ⑤ $1$

## 풀이
$2 a_{n} + n = p$ 에 $n = 1, 2, 3, \dots, 20$ 을 대입하면
$2 a_{1} + 1 = p$
$2 a_{2} + 2 = p$
$\vdots$
$2 a_{20} + 20 = p$

위의 식들을 변끼리 모두 더하면
$2 (a_{1} + a_{2} + \dots + a_{20}) + (1 + 2 + \dots + 20) = 20 p$
$\sum_{n=1}^{20} a_{n} = p$ 이므로
$2 p + \frac{20 \times 21}{2} = 20 p$
$2 p + 210 = 20 p$
$18 p = 210$
$\therefore p = \frac{35}{3}$

$2 a_{10} + 10 = \frac{35}{3}$ 에서
$2 a_{10} = \frac{35}{3} - 10$
$2 a_{10} = \frac{5}{3}$
$a_{10} = \frac{5}{6}$

정답은 ③입니다.
