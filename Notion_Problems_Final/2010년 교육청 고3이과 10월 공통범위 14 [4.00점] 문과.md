## 문제
$0 < a < b$ 인 $a$, $b$ 에 대하여 $N(a, b)$ 를 $a < x < b$ 에서 $\log x$ 의 소수부분과 $\log x^{3}$ 의 소수부분이 같은 실수 $x$ 의 개수라 하자. <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $\log 2 = 0.3010$ 으로 계산한다.)

ㄱ. $N(\sqrt{10}, 1000) = 4$
ㄴ. $p$ 가 정수이면 $N(10^{p}, 10^{p+10}) = 19$ 이다.
ㄷ. $N(2^{10}, 2^{50}) = 25$

## 보기
① $ㄱ$   ② $ㄱ, ㄴ$   ③ $ㄱ, ㄷ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
$\log x^{3} - \log x = 3\log x - \log x = 2\log x$ 가 정수이어야 하므로
ㄱ. $\sqrt{10} < x < 1000$ 에서
$\frac{1}{2} < \log x < 3$
$1 < 2\log x < 6$
$2\log x$ 는 $2, 3, 4, 5$ 이므로
$\therefore N(\sqrt{10}, 1000) = 4$ (참)

ㄴ. $10^{p} < x < 10^{p+10}$ 에서
$p < \log x < p+10$
$2p < 2\log x < 2p+20$
$2\log x$ 는 $2p+1, 2p+2, \cdots, 2p+19$ 이므로
$\therefore N(10^{p}, 10^{p+10}) = 19$ (참)

ㄷ. $2^{10} < x < 2^{50}$ 에서
$10\log 2 < \log x < 50\log 2$
$20\log 2 < 2\log x < 100\log 2$
$20 \times 0.3010 < 2\log x < 100 \times 0.3010$
$6.02 < 2\log x < 30.10$
$2\log x$ 는 $7, 8, \cdots, 30$ 이므로 개수는 $30 - 7 + 1 = 24$
$\therefore N(2^{10}, 2^{50}) = 24$ (거짓)

따라서 옳은 것은 ㄱ, ㄴ이다.

정답: ②
