## 문제
$5$보다 큰 소수로는 나누어 떨어지지 않는 자연수 $a$를 소인수분해하면 $a = 2^{a_{1}} 3^{a_{2}} 5^{a_{3}}$과 같은 형태가 된다. 이때, $《 a 》 = ( a_{1}, a_{2}, a_{3} )$이라 하자. (소인수가 없는 경우 지수 $a_{i}$는 $0$으로 놓는다.) 예를 들어, $《 12 》 = ( 2, 1, 0 )$, $《 15 》 = ( 0, 1, 1 )$이다. 두 자연수 $a$와 $b$가 $《 a 》 = ( a_{1}, a_{2}, a_{3} )$, $《 b 》 = ( b_{1}, b_{2}, b_{3} )$일 때, <보기>에서 항상 옳은 것을 모두 고르면?

$$ \begin{array}{|l|} \hline \text{<보기>} \\ \text{ㄱ. } 《 ab 》 = ( a_{1} + b_{1}, a_{2} + b_{2}, a_{3} + b_{3} ) \\ \text{ㄴ. } b \text{가 } a \text{의 배수이면, } b_{i} \geq a_{i} ( i = 1, 2, 3 ) \\ \text{ㄷ. } a \text{와 } b \text{의 최대공약수 } c \text{는 } 《 c 》 = ( c_{1}, c_{2}, c_{3} ) \text{일 때, } \\ \quad c_{i} \text{는 } a_{i} \text{와 } b_{i} \text{ 중에서 크거나 같은 수 } ( i = 1, 2, 3 ) \\ \hline \end{array} $$

## 보기
① $ㄱ$   ② $ㄱ, ㄴ$   ③ $ㄱ, ㄷ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
$《 a 》 = ( a_{1}, a_{2}, a_{3} )$, $《 b 》 = ( b_{1}, b_{2}, b_{3} )$이므로 
$a = 2^{a_{1}} 3^{a_{2}} 5^{a_{3}}$, $b = 2^{b_{1}} 3^{b_{2}} 5^{b_{3}}$이 된다.

ㄱ. $ab = ( 2^{a_{1}} 3^{a_{2}} 5^{a_{3}} ) ( 2^{b_{1}} 3^{b_{2}} 5^{b_{3}} ) = 2^{a_{1} + b_{1}} 3^{a_{2} + b_{2}} 5^{a_{3} + b_{3}}$
$\therefore 《 ab 》 = ( a_{1} + b_{1}, a_{2} + b_{2}, a_{3} + b_{3} )$ (참)

ㄴ. $b$가 $a$의 배수이면 $b = ka$ ($k$는 자연수)라 하면, $a, b$는 $5$ 이하의 소수로만 나누어지므로 $k$의 소인수 또한 $2, 3, 5$ 중의 어느 하나로 반드시 구성된다.
따라서 $b_{i} \geq a_{i} ( i = 1, 2, 3 )$이다. (참)

ㄷ. [반례] $a = 12$, $b = 36$일 때,
$《 12 》 = ( 2, 1, 0 )$, $《 36 》 = ( 2, 2, 0 )$이고 $12$와 $36$의 최대공약수는 $12$이다.
$《 12 》$에서 $c_{i}$는 $a_{i}$와 $b_{i} ( i = 1, 2, 3 )$ 중에서 크거나 같은 수가 아니라 작거나 같은 수($\min$)이다. (거짓)

그러므로 옳은 것은 ㄱ, ㄴ이다.
정답: ②
