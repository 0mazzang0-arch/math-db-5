## 문제
다음은 $2$ 이상의 모든 자연수 $n$ 에 대하여 $1+\frac{1}{2^{2}}+\frac{1}{3^{2}}+ \cdots+\frac{1}{n^{2}}<2-\frac{1}{n}$ 임을 수학적 귀납법으로 증명한 것이다.

$$
\begin{array}{|l|}
\hline
( \text{i} ) \text{ } n = (가) \text{ 일 때,} \\
\quad (\text{좌변}) = 1 + \frac{1}{2^{2}} = \frac{5}{4} < 2 - \frac{1}{2} = \frac{3}{2} = (\text{우변}) \\
\quad n = (가) \text{ 일 때, 주어진 식은 성립한다.} \\
\\
( \text{ii} ) \text{ } n = k \text{ (} k \ge 2 \text{) 일 때, 주어진 식이 성립한다고 가정하면} \\
\quad 1 + \frac{1}{2^{2}} + \frac{1}{3^{2}} + \cdots + \frac{1}{k^{2}} < 2 - \frac{1}{k} \text{ 이다.} \\
\quad \text{양변에 } \frac{1}{(k+1)^{2}} \text{ 을 더하면} \\
\quad 1 + \frac{1}{2^{2}} + \frac{1}{3^{2}} + \cdots + \frac{1}{k^{2}} + \frac{1}{(k+1)^{2}} < 2 - \frac{1}{k} + \frac{1}{(k+1)^{2}} \\
\quad \text{이때, } \\
\quad \left\{ - \frac{1}{k} + \frac{1}{(k+1)^{2}} \right\} - (나) = - \frac{1}{k(k+1)^{2}} < 0 \text{ 이므로} \\
\quad 2 - \frac{1}{k} + \frac{1}{(k+1)^{2}} < 2 - \frac{1}{k+1} \text{ 이다.} \\
\quad n = k + 1 \text{ 일 때에도 성립한다.} \\
\\
n \ge 2 \text{ 인 모든 자연수 } n \text{ 에 대하여} \\
1 + \frac{1}{2^{2}} + \frac{1}{3^{2}} + \cdots + \frac{1}{n^{2}} < 2 - \frac{1}{n} \text{ 이 성립한다.} \\
\hline
\end{array}
$$

이 증명 과정에서 $(가)$, $(나)$에 알맞은 내용을 바르게 짝지은 것은?

## 보기
① $(가): 1, (나): \frac{1}{k+1}$   ② $(가): 1, (나): -\frac{1}{k+1}$   ③ $(가): 2, (나): -\frac{1}{k+1}$   ④ $(가): 2, (나): \frac{1}{k(k+1)}$   ⑤ $(가): 2, (나): \frac{1}{(k+1)^{2}}$

## 풀이
(i) $2$ 이상의 모든 자연수 $n$ 에 대하여 증명하는 것이므로 최소의 자연수 $n = 2$ 이다.
$\therefore (가) = 2$

(ii) $n = k$ 일 때의 우변에서 $n = k + 1$ 일 때의 우변을 유도하는 과정에서 다음 식이 성립해야 한다.
$\left\{ - \frac{1}{k} + \frac{1}{(k+1)^{2}} \right\} - (나) = - \frac{1}{k(k+1)^{2}}$

이때, 좌변의 식을 정리하면 다음과 같다.
$- \frac{1}{k} + \frac{1}{(k+1)^{2}} = - \frac{(k+1)^{2} - k}{k(k+1)^{2}} = - \frac{k^{2} + 2k + 1 - k}{k(k+1)^{2}}$
$= - \frac{k^{2} + k + 1}{k(k+1)^{2}} = - \frac{k(k+1) + 1}{k(k+1)^{2}}$
$= - \frac{1}{k+1} - \frac{1}{k(k+1)^{2}}$

따라서 식에 대입하면:
$\left\{ - \frac{1}{k+1} - \frac{1}{k(k+1)^{2}} \right\} - (나) = - \frac{1}{k(k+1)^{2}}$
$- \frac{1}{k+1} - (나) = 0$
$\therefore (나) = - \frac{1}{k+1}$

(가), (나)에 알맞은 값을 짝지으면 ③이다.

정답: ③
