## 문제
$-1 < a < b < 0$ 인 두 실수 $a$, $b$ 에 대하여 함수 $y=10^{x}-1$ 의 그래프 위의 두 점 $P(\log(a+1), a)$, $Q(\log(b+1), b)$ 가 있다. 세 직선 $PO$, $QO$, $PQ$ 를 이용하여 세 실수 
$A=(a+1)^{\frac{1}{a}}$, $B=(b+1)^{\frac{1}{b}}$, $C=\left(\frac{b+1}{a+1}\right)^{\frac{1}{b-a}}$ 
의 대소를 비교할 때, $A$, $B$, $C$ 의 대소 관계를 옳게 나타낸 것은? (단, $O$ 는 원점이다.)

> [!example] 📸 (그림/그래프)

## 보기
① $A < B < C$   ② $A < C < B$   ③ $B < A < C$   ④ $B < C < A$   ⑤ $C < A < B$

## 풀이
(직선 $QO$ 의 기울기) $= \frac{b}{\log(b+1)} = \frac{1}{\log(b+1)^{\frac{1}{b}}} \cdots ㉠$

(직선 $PO$ 의 기울기) $= \frac{a}{\log(a+1)} = \frac{1}{\log(a+1)^{\frac{1}{a}}} \cdots ㉡$

(직선 $PQ$ 의 기울기) $= \frac{b-a}{\log(b+1)-\log(a+1)} = \frac{1}{\log\left(\frac{b+1}{a+1}\right)^{\frac{1}{b-a}}} \cdots ㉢$

$㉠ > ㉡ > ㉢$ 이므로 

$\log(b+1)^{\frac{1}{b}} < \log(a+1)^{\frac{1}{a}} < \log\left(\frac{b+1}{a+1}\right)^{\frac{1}{b-a}}$

$(b+1)^{\frac{1}{b}} < (a+1)^{\frac{1}{a}} < \left(\frac{b+1}{a+1}\right)^{\frac{1}{b-a}}$

따라서 $B < A < C$ 이다.
