## 문제
> [!example] 📸 (그림/그래프)

`$1$`보다 큰 두 수 `$A$`, `$B$`에 대하여 `$\log A$`의 정수부분과 소수부분은 각각 `$m$`, `$\alpha$`이고 `$\log B$`의 정수부분과 소수부분은 각각 `$n$`, `$\beta$`일 때, 점 `$P$`, `$Q$`의 좌표를 `$P(m, n)$`, `$Q(\alpha, \beta)$`라 하자. 점 `$P$`는 곡선 `$y = \frac{16}{x} (x > 0)$` 위에 있고 점 `$Q$`는 직선 `$y = -x + 1$` 위에 있을 때, `$AB$`의 최댓값과 최솟값의 곱이 `$10^{k}$`이다. 실수 `$k$`의 값을 구하시오. 

> [!example] 📸 (그림/그래프)

## 풀이
`[정답] $27$`

`$\log A = m + \alpha$`, `$\log B = n + \beta$`
(`$m$`, `$n$`은 음이 아닌 정수, `$0 \le \alpha < 1$`, `$0 \le \beta < 1$`)

점 `$Q$`가 `$y = -x + 1$` 위에 있으므로
`$\alpha + \beta = 1$`

점 `$P$`가 `$y = \frac{16}{x}$` 위에 있으므로
`$mn = 16$`

`$\log AB = \log A + \log B$`
`$= (m + \alpha) + (n + \beta)$`
`$= m + n + (\alpha + \beta)$`
`$= m + n + 1$`

`$\therefore AB = 10^{m + n + 1}$`

양의 정수 `$m$`, `$n$`에 대하여 `$mn = 16$`일 때, `$m + n$`의 최댓값은 `$17$`, 최솟값은 `$8$`이므로
`$AB$`의 최댓값은 `$10^{17+1} = 10^{18}$`
`$AB$`의 최솟값은 `$10^{8+1} = 10^{9}$`

최댓값과 최솟값의 곱은
`$10^{18} \times 10^{9} = 10^{27}$`

`$\therefore k = 27$`
