## 문제
함수 $f(x) = \cos^{2} x - 4\cos \left( x + \frac{\pi}{2} \right) + 3$의 최댓값은?

## 보기
① $1$   ② $3$   ③ $5$   ④ $7$   ⑤ $9$

## 풀이
$f(x) = \cos^{2} x - 4\cos \left( x + \frac{\pi}{2} \right) + 3$
$= \cos^{2} x + 4\sin x + 3$
$= (1 - \sin^{2} x) + 4\sin x + 3$
$= -\sin^{2} x + 4\sin x + 4$

$\sin x = t$ ($-1 \le t \le 1$)로 치환하면
$y = -t^{2} + 4t + 4$

이차함수의 축의 방정식이 $t = 2$이므로
$-1 \le t \le 1$의 범위에서 $t = 1$일 때 최댓값을 갖는다.

최댓값은 $-1 + 4 + 4 = 7$이다.

정답은 ④이다.
