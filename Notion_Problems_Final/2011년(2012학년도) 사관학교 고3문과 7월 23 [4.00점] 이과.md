## 문제
$0 < a < b < 1$ 일 때, 직선 $y = 1$ 이 $y = \log_{a} x$ 의 그래프와 $y = \log_{b} x$ 의 그래프와 만나는 점을 각각 $P$, $Q$ 라 하고 직선 $y = -1$ 이 $y = \log_{a} x$ 의 그래프와 $y = \log_{b} x$ 의 그래프와 만나는 점을 각각 $R$, $S$ 라 하자. $PS$, $PR$, $QS$, $QR$ 의 기울기를 각각 $\alpha$, $\beta$, $\gamma$, $\delta$ 라 할 때, 다음 중 옳은 것은?

## 보기
① $\delta < \alpha < \beta < \gamma$
② $\gamma < \alpha < \delta < \beta$
③ $\gamma < \alpha < \beta < \delta$
④ $\gamma < \alpha = \delta < \beta$
⑤ $\alpha = \delta < \beta < \gamma$

## 풀이
[정답] ②

함수 $y = \log_{a} x$, $y = \log_{b} x$ 의 그래프와 직선 $y = 1$, $y = -1$ 를 그려 점 $P$, $Q$, $R$, $S$ 를 표시하면 다음과 같다.

> [!example] 📸 (그림/그래프)

$y = \log_{a} x$ 에서 $y = 1$ 이면 $x = a$ 이므로 $P(a, 1)$
$y = \log_{b} x$ 에서 $y = 1$ 이면 $x = b$ 이므로 $Q(b, 1)$
$y = \log_{a} x$ 에서 $y = -1$ 이면 $x = \frac{1}{a}$ 이므로 $R(\frac{1}{a}, -1)$
$y = \log_{b} x$ 에서 $y = -1$ 이면 $x = \frac{1}{b}$ 이므로 $S(\frac{1}{b}, -1)$

각 선분의 기울기는 다음과 같다.
$\alpha (\text{직선 } PS \text{의 기울기}) = \frac{-1 - 1}{\frac{1}{b} - a} = \frac{-2}{\frac{1}{b} - a}$
$\beta (\text{직선 } PR \text{의 기울기}) = \frac{-1 - 1}{\frac{1}{a} - a} = \frac{-2}{\frac{1}{a} - a}$
$\gamma (\text{직선 } QS \text{의 기울기}) = \frac{-1 - 1}{\frac{1}{b} - b} = \frac{-2}{\frac{1}{b} - b}$
$\delta (\text{직선 } QR \text{의 기울기}) = \frac{-1 - 1}{\frac{1}{a} - b} = \frac{-2}{\frac{1}{a} - b}$

$0 < a < b < 1$ 이므로 $1 < \frac{1}{b} < \frac{1}{a}$ 이다.
분모의 크기를 비교하면
$0 < \frac{1}{b} - b < \frac{1}{b} - a < \frac{1}{a} - b < \frac{1}{a} - a$ 이다.
역수를 취하고 $-2$ 를 곱하면 부동호의 방향이 바뀌므로
$\frac{-2}{\frac{1}{b} - b} < \frac{-2}{\frac{1}{b} - a} < \frac{-2}{\frac{1}{a} - b} < \frac{-2}{\frac{1}{a} - a}$
따라서 $\gamma < \alpha < \delta < \beta$ 이다.
