## 문제
$C = \{ a + bi \mid a, b \text{ 는 실수} \}$의 부분집합 $S$는 다음을 만족한다.

$$
\begin{array}{l}
\text{Ⅰ. } 0! \in S \\
\text{Ⅱ. } S\text{의 원소는 } 3\text{개} \\
\text{Ⅲ. } S\text{는 곱셈에 대하여 닫혀 있다.}
\end{array}
$$

이때, $S$의 세 원소를 $Z_{1}$, $Z_{2}$, $Z_{3}$라 할 때, $Z_{1}^{2} + Z_{2}^{2} + Z_{3}^{2}$의 값을 구하면?

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
$0! = 1$이므로 집합 $S$는 $1$을 원소로 가진다.
원소의 개수가 $3$개이므로 나머지 두 원소를 $1$이 아닌 서로 다른 두 수 $x$, $y$라 하자. 곱셈에 관하여 닫혀있기 위해 다음 중 하나를 만족해야 한다.

(ⅰ) $x^{2} = 1$인 경우
$x = -1$ ($\because x \neq 1$)이므로 $y^{2} = -1$을 만족해야 한다. 즉, $y = i$일 때 집합 $S$는 곱셈에 관하여 닫혀있지 않다. (예: $i \times (-1) = -i \notin S$)

(ⅱ) $x^{2} = x$인 경우
$x = 0$이므로 $y^{2} = 1$을 만족해야 한다. 따라서 $y = -1$이면 집합 $S = \{ -1, 0, 1 \}$이고 곱셈에 관하여 닫혀있다.

(ⅲ) $x^{2} = y$인 경우
$y^{2} = 1$ 또는 $y^{2} = x$를 만족해야 한다. $y^{2} = 1$인 경우에는 앞의 (ⅰ)과 (ⅱ)의 경우에 해당하며, $y^{2} = x$인 경우 $x = y = 1$이므로 조건에 부합하지 않는다.

따라서 조건에 부합하는 집합 $S = \{ 1, 0, -1 \}$이다.
$Z_{1}^{2} + Z_{2}^{2} + Z_{3}^{2} = 1^{2} + 0^{2} + (-1)^{2} = 2$

정답: ②
