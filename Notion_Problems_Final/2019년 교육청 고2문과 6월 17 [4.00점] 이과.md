## 문제
다음은 $0 < \theta < 2\pi$ 에서 $3 + 2\sin^{2} \theta + \frac{1}{3 - 2\cos^{2} \theta}$ 의 최솟값을 구하는 과정이다.

$3 + 2\sin^{2} \theta = t$ 로 놓으면
$3 + 2\sin^{2} \theta + \frac{1}{3 - 2\cos^{2} \theta} = t + \frac{1}{\text{(가)}}$
이다. $0 < \theta < 2\pi$ 에서 $t \geq 3$ 이므로 $\text{(가)} > 0$ 이다.
$t + \frac{1}{\text{(가)}} = t - 2 + \frac{1}{\text{(가)}} + 2 \geq 4$
이다. (단, 등호는 $t = \text{(나)}$ 일 때 성립한다.)
따라서 $3 + 2\sin^{2} \theta + \frac{1}{3 - 2\cos^{2} \theta}$ 은 $\theta = \text{(다)}$ 에서 최솟값 $4$ 를 갖는다.

위의 (가)에 알맞은 식을 $f(t)$, (나)와 (다)에 알맞은 수를 각각 $p$, $q$ 라 할 때, $f(p) + \tan^{2} \left( q + \frac{\pi}{3} \right)$ 의 값은?

## 보기
① $4$   ② $5$   ③ $6$   ④ $7$   ⑤ $8$

## 풀이
$3 + 2\sin^{2} \theta = t$ 로 놓으면
$3 - 2\cos^{2} \theta = 3 - 2(1 - \sin^{2} \theta) = 1 + 2\sin^{2} \theta = (3 + 2\sin^{2} \theta) - 2 = t - 2$
이므로
$3 + 2\sin^{2} \theta + \frac{1}{3 - 2\cos^{2} \theta} = t + \frac{1}{t - 2}$
이다.
$0 < \theta < 2\pi$ 에서 $0 \leq \sin^{2} \theta \leq 1$ 이므로 $3 \leq t \leq 5$ 이고, $t - 2 > 0$ 이다.
$t + \frac{1}{t - 2} = t - 2 + \frac{1}{t - 2} + 2 \geq 2\sqrt{(t - 2) \cdot \frac{1}{t - 2}} + 2 = 4$
이다. 
단, 등호는 $t - 2 = \frac{1}{t - 2}$ 에서 $(t - 2)^{2} = 1$
$t \geq 3$ 이므로 $t - 2 = 1$, 즉 $t = 3$ 일 때 성립한다.
따라서 $3 + 2\sin^{2} \theta = 3$ 일 때 $\sin \theta = 0$ 이므로 $0 < \theta < 2\pi$ 에서 $\theta = \pi$ 이고, 이때 최솟값 $4$ 를 갖는다.

그러므로
$f(t) = t - 2$, $p = 3$, $q = \pi$ 이다.
$f(p) + \tan^{2} \left( q + \frac{\pi}{3} \right) = f(3) + \tan^{2} \left( \pi + \frac{\pi}{3} \right)$
$= (3 - 2) + \tan^{2} \left( \frac{\pi}{3} \right)$
$= 1 + (\sqrt{3})^{2}$
$= 1 + 3 = 4$

정답: ①
