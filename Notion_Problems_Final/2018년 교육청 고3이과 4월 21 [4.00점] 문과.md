## 문제
$\frac{3}{5} < x < 4$에서 정의된 미분가능한 함수 $f(x)$가 $f(1) = 2$이고
$$f'(x) = \frac{1 - x^{2} \{f(x)\}^{3}}{x^{3} \{f(x)\}^{2}}$$
을 만족시킨다. 함수 $f(x)$의 역함수 $g(x)$가 존재하고 미분가능할 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
ㄱ. 함수 $f(x)$의 역함수가 $g(x)$이므로 $f(g(x)) = x$
$f(1) = 2$이므로 $g(2) = 1$
주어진 식에서 
$$f'(g(x)) = \frac{1 - \{g(x)\}^{2} \{f(g(x))\}^{3}}{\{g(x)\}^{3} \{f(g(x))\}^{2}} = \frac{1 - x^{3} \{g(x)\}^{2}}{x^{2} \{g(x)\}^{3}}$$
$f'(g(x)) = \frac{1}{g'(x)}$이므로
$$g'(x) = \frac{x^{2} \{g(x)\}^{3}}{1 - x^{3} \{g(x)\}^{2}} \cdots (*)$$
$x = 2$를 대입하면
$$g'(2) = \frac{2^{2} \times \{g(2)\}^{3}}{1 - 2^{3} \times \{g(2)\}^{2}} = \frac{2^{2}}{1 - 2^{3}} = -\frac{4}{7}$$
따라서 (참)

ㄴ. 식 $(*)$을 정리하면
$$g'(x) = x^{3} \{g(x)\}^{2} g'(x) + x^{2} \{g(x)\}^{3}$$
$$g'(x) = x^{2} \{g(x)\}^{2} \{x g'(x) + g(x)\}$$
$x$에 대하여 적분하면 $x g'(x) + g(x) = \{x g(x)\}'$이므로
$$\int g'(x) dx = \int \{x g(x)\}^{2} \{x g(x)\}' dx$$
$$g(x) = \frac{1}{3} x^{3} \{g(x)\}^{3} + C$$
(단, $C$는 적분상수)
$x = 2$를 대입하면 $1 = \frac{8}{3} + C$이므로 $C = -\frac{5}{3}$
$$g(x) = \frac{1}{3} x^{3} \{g(x)\}^{3} - \frac{5}{3} \cdots (**)$$
따라서 (참)

ㄷ. 식 $(**)$에 $x = 1$을 대입하여 정리하면
$$\{g(1)\}^{3} - 3g(1) - 5 = 0$$
$h(t) = t^{3} - 3t - 5$, $g(1) = \alpha$라 하면
$$h'(t) = 3(t + 1)(t - 1)$$
$t = -1$에서 극댓값 $-3$, $t = 1$에서 극솟값 $-7$을 가지므로 방정식 $h(t) = 0$은 하나의 실근 $\alpha$를 갖는다.
$h(2) = 8 - 6 - 5 = -3 < 0$,
$$h\left(\frac{5}{2}\right) = \frac{125}{8} - \frac{15}{2} - 5 = \frac{25}{8} > 0$$
이므로 $2 < \alpha < \frac{5}{2}$
$$2 < g(1) < \frac{5}{2}$$
따라서 (참)

따라서 옳은 것은 ㄱ, ㄴ, ㄷ이다.
