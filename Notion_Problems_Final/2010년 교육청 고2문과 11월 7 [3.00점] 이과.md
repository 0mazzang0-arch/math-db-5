## 문제
그림과 같이 두 곡선 $y = \log_{4} x$, $y = \log_{\frac{1}{4}} x$와 두 직선 $x = a$, $x = a + 2$가 만나는 점을 각각 $A$, $B$, $C$, $D$라 하자. 사각형 $ABCD$의 넓이가 $3$일 때, 상수 $a$의 값은? (단, $a > 1$)



> [!example] 📸 (그림/그래프)

## 보기
① $\frac{3}{2}$   ② $\frac{7}{4}$   ③ $2$   ④ $\frac{5}{2}$   ⑤ $3$

## 풀이
점 $A$, $B$, $C$, $D$의 좌표는 각각 다음과 같다.
$A(a, \log_{4} a)$, $B(a, \log_{\frac{1}{4}} a)$, $C(a + 2, \log_{\frac{1}{4}} (a + 2))$, $D(a + 2, \log_{4} (a + 2))$

이때 선분 $AB$와 $CD$의 길이를 구하면 다음과 같다.
$\overline{AB} = \log_{4} a - \log_{\frac{1}{4}} a = \log_{4} a - (-\log_{4} a) = 2 \log_{4} a = \log_{2} a$
$\overline{CD} = \log_{4} (a + 2) - \log_{\frac{1}{4}} (a + 2) = \log_{4} (a + 2) - (-\log_{4} (a + 2)) = 2 \log_{4} (a + 2) = \log_{2} (a + 2)$

사각형 $ABCD$는 사다리꼴이므로 그 넓이는 다음과 같다.
$\frac{1}{2} \times \{ \overline{AB} + \overline{CD} \} \times ( (a + 2) - a ) = 3$
$\frac{1}{2} \times \{ \log_{2} a + \log_{2} (a + 2) \} \times 2 = 3$
$\log_{2} \{ a(a + 2) \} = 3$
$a(a + 2) = 2^3 = 8$
$a^2 + 2a - 8 = 0$
$(a + 4)(a - 2) = 0$

$a > 1$이므로 $a = 2$이다.
