## 문제
부등식 $(x^{2}+2 x-y+1) (x^{2}+y^{2}-9) \leq 0$ 의 영역을 좌표평면 위에 어두운 부분으로 바르게 나타낸 것은? (단, 경계선은 포함한다.)

## 풀이
[정답] ①

$(x^{2}+2 x-y+1 ) (x^{2}+y^{2}-9 ) \leq 0$ 에서 
$\begin{cases} x^{2}+2 x-y+1 \geq 0 \\ x^{2}+y^{2}-9 \leq 0 \end{cases}$ 또는 $\begin{cases} x^{2}+2 x-y+1 \leq 0 \\ x^{2}+y^{2}-9 \geq 0 \end{cases}$

($i$) $\begin{cases} x^{2}+2 x-y+1 \geq 0 \\ x^{2}+y^{2}-9 \leq 0 \end{cases}$ 의 영역은 
$y=x^{2}+2 x+1$ 의 그래프의 아랫부분(경계선 포함)과 원 $x^{2}+y^{2}=9$ 의 내부(경계선 포함)의 공통부분으로 [그림 $1$]의 어두운 부분이다. 

($ii$) $\begin{cases} x^{2}+2 x-y+1 \leq 0 \\ x^{2}+y^{2}-9 \geq 0 \end{cases}$ 의 영역은 
$y=x^{2}+2 x+1$ 의 그래프의 윗부분 (경계선 포함)과 원 $x^{2}+y^{2}=9$ 의 외부 (경계선 포함)의 공통부분으로 [그림 $2$]의 어두운 부분이다. 

($i$), ($ii$)에 의하여 구하는 영역은 [그림 $3$]의 어두운 부분이다. (단, 경계선은 포함한다.)
