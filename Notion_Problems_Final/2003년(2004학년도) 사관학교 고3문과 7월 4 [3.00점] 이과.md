## 문제
서로 다른 두 실수 $a$, $b$ 에 대하여 두 집합 $\langle a, b \rangle$, $[a, b]$ 를
$$\langle a, b \rangle = \{ \{ a \}, \{ a, b \} \}$$
$$[a, b] = \{ a \} \cup \{ b \}$$
와 같이 정의할 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?

ㄱ. $\langle a, b \rangle = \langle b, a \rangle$
ㄴ. 집합 $\langle a, b \rangle$ 의 부분집합의 개수는 $4$ 개이다.
ㄷ. $[a, b] = [c, d]$ 이면 $a = c$, $b = d$ 이다.

## 보기
① ㄱ   ② ㄴ   ③ ㄷ   ④ ㄱ, ㄴ   ⑤ ㄴ, ㄷ

## 풀이
ㄱ. $\langle a, b \rangle = \{ \{ a \}, \{ a, b \} \}$, $\langle b, a \rangle = \{ \{ b \}, \{ b, a \} \}$ 이다. $a \neq b$ 이므로 $\{ a \} \neq \{ b \}$ 가 되어 $\langle a, b \rangle \neq \langle b, a \rangle$ 이다. (거짓)

ㄴ. 집합 $\langle a, b \rangle = \{ \{ a \}, \{ a, b \} \}$ 의 원소는 $\{ a \}$ 와 $\{ a, b \}$ 로 $2$ 개이다. 따라서 부분집합의 개수는 $2^{2} = 4$ (개)이다. (참)

ㄷ. $[a, b] = \{ a, b \}$, $[c, d] = \{ c, d \}$ 이므로 $[a, b] = [c, d]$ 이면 $\{ a, b \} = \{ c, d \}$ 이다. 이는 $a = c, b = d$ 일 수도 있지만 $a = d, b = c$ 일 수도 있으므로 $a = c, b = d$ 라고 확정할 수 없다. (거짓)

따라서 옳은 것은 ㄴ 이다.
