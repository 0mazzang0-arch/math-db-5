## 문제
다음 보기에서 옳은 것만을 있는 대로 고른 것은?

$$ \begin{array}{|l|} \hline \text{ㄱ. 함수 } y = \tan \frac{3\pi}{2}x - \sin 2\pi x \text{의 주기는 } 2 \text{이다.} \\ \text{ㄴ. 함수 } y = 2\pi + \cos 2\pi x \sin \frac{4\pi}{3}x \text{의 주기는 } 3 \text{이다.} \\ \text{ㄷ. 함수 } y = \sin \pi x - \left| \cos \frac{3\pi}{2} x \right| \text{의 주기는 } 2 \text{이다.} \\ \hline \end{array} $$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
[정답] ⑤

ㄱ. $f(x) = \tan \frac{3\pi}{2}x - \sin 2\pi x$라 하면
$y = \tan \frac{3\pi}{2}x$의 주기는 $\frac{\pi}{\frac{3\pi}{2}} = \frac{2}{3}$
함수 $y = \sin 2\pi x$의 주기는 $\frac{2\pi}{2\pi} = 1$
$f(x) = f(x + p)$를 만족시키는 최소의 양수는 $\frac{2}{3} \times 3 = 1 \times 2$에서 $2$이다.

ㄴ. $g(x) = 2\pi + \cos 2\pi x \sin \frac{4\pi}{3}x$라 하면
함수 $y = \cos 2\pi x$의 주기는 $\frac{2\pi}{2\pi} = 1$
$y = \sin \frac{4\pi}{3}x$의 주기는 $\frac{2\pi}{\frac{4\pi}{3}} = \frac{3}{2}$
$g(x) = g(x + p)$를 만족시키는 최소의 양수는 $1 \times 3 = \frac{3}{2} \times 2$에서 $3$이다.

ㄷ. $h(x) = \sin \pi x - \left| \cos \frac{3\pi}{2} x \right|$라 하면
$y = \sin \pi x$의 주기는 $\frac{2\pi}{\pi} = 2$
함수 $y = \left| \cos \frac{3\pi}{2} x \right|$의 주기는 $\frac{1}{2} \times \frac{2\pi}{\frac{3\pi}{2}} = \frac{2}{3}$
$h(x) = h(x + p)$를 만족시키는 최소의 양수는 $2 \times 1 = \frac{2}{3} \times 3$에서 $2$이다.

[참고]
주기가 각각 $a$, $b$인 두 주기함수 $f(x)$, $g(x)$에 대하여 $am = bn$을 만족하는 가장 작은 두 자연수 $m_1$, $n_1$에 대하여 함수 $f(x) + g(x)$의 주기는 $am_1$ (또는 $bn_1$)이다.
