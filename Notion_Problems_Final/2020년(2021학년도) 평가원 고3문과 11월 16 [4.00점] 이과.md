## 문제
$0 \le x \le 4\pi$ 일 때, 방정식 
$$4 \sin^2 x - 4 \cos \left( \frac{\pi}{2} + x \right) - 3 = 0$$
의 모든 해의 합은?

## 보기
① $5\pi$   ② $6\pi$   ③ $7\pi$   ④ $8\pi$   ⑤ $9\pi$

## 풀이
$\cos \left( \frac{\pi}{2} + x \right) = -\sin x$ 이므로 
$$4 \sin^2 x - 4 \cos \left( \frac{\pi}{2} + x \right) - 3$$
$$= 4 \sin^2 x + 4 \sin x - 3$$
$\sin x = t$ 라 하면 $t$ 의 범위는 $0 \le x \le 4\pi$ 에 의하여 
$$-1 \le t \le 1$$
준 식은 $4t^2 + 4t - 3$ 이므로 
$$(2t - 1)(2t + 3) = 0$$
$\therefore t = \frac{1}{2}$ 또는 $t = -\frac{3}{2}$
$\therefore t = \frac{1}{2}$ ($\because -1 \le t \le 1$)
즉, $0 \le x \le 4\pi$ 에서 $\sin x = \frac{1}{2}$ 을 만족하는 $x$ 의 값은 
$$x = \frac{\pi}{6}, \pi - \frac{\pi}{6}, 2\pi + \frac{\pi}{6}, 3\pi - \frac{\pi}{6}$$
따라서 모든 해의 합은 
$$\frac{\pi}{6} + \frac{5\pi}{6} + \frac{13\pi}{6} + \frac{17\pi}{6} = 6\pi$$
정답: ②
