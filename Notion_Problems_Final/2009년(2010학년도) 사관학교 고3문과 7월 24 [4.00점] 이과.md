## 문제
다음 등식을 만족시키는 세 실수 $a$, $b$, $c$가 있다.
$$ \left( \frac{1}{3} \right)^{a} = 2a, \quad \left( \frac{1}{3} \right)^{2b} = b, \quad \left( \frac{1}{2} \right)^{2c} = c $$
이때, 세 실수 $a$, $b$, $c$의 대소 관계를 옳게 나타낸 것은?

## 보기
① $a < b < c$   ② $a < c < b$   ③ $b < a < c$   ④ $b < c < a$   ⑤ $c < a < b$

## 풀이
주어진 식을 정리하면 다음과 같다.
$1)$ $\left( \frac{1}{3} \right)^{a} = 2a$
$2)$ $\left( \frac{1}{3} \right)^{2b} = b \Rightarrow \left( \left( \frac{1}{3} \right)^{2} \right)^{b} = b \Rightarrow \left( \frac{1}{9} \right)^{b} = b$
$3)$ $\left( \frac{1}{2} \right)^{2c} = c \Rightarrow \left( \left( \frac{1}{2} \right)^{2} \right)^{c} = c \Rightarrow \left( \frac{1}{4} \right)^{c} = c$

또한 $\left( \frac{1}{3} \right)^{2b} = b$를 $\left( \frac{1}{3} \right)^{b} = \sqrt{b}$의 형태로 보거나, 함수 $y = \left( \frac{1}{3} \right)^{x}$와 $y = 2x$의 교점의 $x$좌표를 $a$로 분석할 수 있다.

지수함수 $y = \left( \frac{1}{3} \right)^{x}$, $y = \left( \frac{1}{9} \right)^{x}$, $y = \left( \frac{1}{4} \right)^{x}$의 그래프와 직선의 교점을 이용하여 대소 관계를 비교하면 다음과 같다.

> [!example] 📸 (그림/그래프)
> (지수함수 $y = \left( \frac{1}{3} \right)^{x}$와 직선 $y = 2x$의 교점 $a$, $y = \left( \frac{1}{9} \right)^{x}$와 $y = x$의 교점 $b$ 등을 시각화)

따라서 세 실수의 대소 관계는 다음과 같다.
$\therefore a < b < c$

정답: ①
