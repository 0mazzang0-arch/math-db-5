## 문제
집합 $X = \{ 1, 2, 3, 4 \}$에 대하여 함수 $f(x)$는 $X$에서 $X$로의 일대일대응이다. 함수 $f(x)$가 $(f \circ f)(1) = 4$, $(f \circ f)(3) = 3$을 만족할 때, $2 f(1) + 4 f(3)$의 값은? (단, $(f \circ f)(x) = f(f(x))$)

## 보기
① $12$   ② $14$   ③ $16$   ④ $18$   ⑤ $20$

## 풀이
$(f \circ f)(1) = 4$에서
$i)$ $f(1) = 1$이면 $f(f(1)) = f(1) = 4$
$\therefore$ 모순
$ii)$ $f(1) = 2$이면 $f(f(1)) = f(2) = 4$
$\therefore f(1) = 2$, $f(2) = 4$
$iii)$ $f(1) = 3$이면 $\cdots$ ㉠
$f(f(1)) = f(3) = 4$
$(f \circ f)(3) = 3$에서 $f(f(3)) = f(4) = 3$ $\cdots$ ㉡
㉠, ㉡에서 $f(3) = 4$, $f(4) = 3$이면 일대일대응 조건을 만족하나, $f(1) = 3$이므로 $f(1) = f(4) = 3$이 되어 모순이 발생한다.
$iv)$ $f(1) = 4$이면 $f(f(1)) = f(4) = 4$
$\therefore$ 모순

$ii)$에서 $f(1) = 2$, $f(2) = 4$이므로 $f(3)$의 값은 일대일대응에 의해 $1$ 또는 $3$이다.
$f(3) = 1$이면 $f(f(3)) = f(1) = 3$이나, $f(1) = 2$이므로 $\therefore$ 모순
$f(3) = 3$이면 $f(f(3)) = f(3) = 3$을 만족하고, 남은 원소에 의해 $f(4) = 1$이다.

$\therefore f(1) = 2$, $f(2) = 4$, $f(3) = 3$, $f(4) = 1$
$\therefore 2 f(1) + 4 f(3) = 2 \cdot 2 + 4 \cdot 3 = 4 + 12 = 16$

정답: ③
