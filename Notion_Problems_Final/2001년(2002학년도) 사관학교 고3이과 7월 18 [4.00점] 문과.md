## 문제
다음은 임의의 실수 $a$, $b$에 대하여
$$ (가) \leq |a+b| \leq (나) $$
임을 증명한 것이다.

$$ \begin{array}{|l|} \hline \\ -|a| \leq a \leq |a| \text{ 이고 } -|b| \leq b \leq |b| \text{ 이므로 } \\ |a+b| \leq (나) \quad \cdots \cdots ㉠ \\ \text{㉠을 이용하면 } |a| \leq |a+b| + (다) \text{ 이므로 } \\ |a| - (다) \leq |a+b| \text{ 이다.} \\ \text{같은 방법으로 하면 } (다) - |a| \leq |a+b| \\ \therefore (가) \leq |a+b| \quad \cdots \cdots ㉡ \\ \text{㉠과 ㉡에 의해 } (가) \leq |a+b| \leq (나) \text{ 가 성립한다.} \\ \\ \hline \end{array} $$

위의 증명에서 $(가)$, $(나)$, $(다)$에 알맞은 것을 순서대로 적은 것은?

## 보기
① $(가): ||a| - |b||, (나): |a| - |b|, (다): |b|$   ② $(가): ||a| - |b||, (나): |a| + |b|, (다): -|b|$   ③ $(가): |a+b|, (나): |a-b|, (다): -|b|$   ④ $(가): ||a| - |b||, (나): |a| + |b|, (다): |b|$   ⑤ $(가): |a-b|, (나): |a+b|, (다): |b|$

## 풀이
$-|a| \leq a \leq |a|$, $-|b| \leq b \leq |b|$ 에서 각 변을 더하면
$$-(|a| + |b|) \leq a+b \leq |a| + |b|$$
이므로
$$|a+b| \leq |a| + |b| \quad \cdots \cdots ㉠$$
이다. 따라서 $(나) = |a| + |b|$ 이다.
$㉠$을 이용하면 임의의 실수 $a$, $b$에 대하여 $|a+b| \leq |a| + |b|$ 가 성립하므로
$$|a| = |(a+b) + (-b)| \leq |a+b| + |-b| = |a+b| + |b|$$
$$|a| \leq |a+b| + |b|$$
즉, $|a| - |b| \leq |a+b|$ 이다. 따라서 $(다) = |b|$ 이다.
같은 방법으로 하면
$$|b| = |(b+a) + (-a)| \leq |b+a| + |-a| = |a+b| + |a|$$
$$|b| - |a| \leq |a+b|$$
따라서 $-(|a+b|) \leq |a| - |b| \leq |a+b|$ 이므로 절댓값의 정의에 의해
$$||a| - |b|| \leq |a+b| \quad \cdots \cdots ㉡$$
이다. 따라서 $(가) = ||a| - |b||$ 이다.
$㉠$과 $㉡$에 의해
$$||a| - |b|| \leq |a+b| \leq |a| + |b|$$
가 성립한다.
그러므로 $(가): ||a| - |b||, (나): |a| + |b|, (다): |b|$ 이다.

정답은 ④이다.
