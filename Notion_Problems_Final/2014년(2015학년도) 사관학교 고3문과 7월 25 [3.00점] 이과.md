## 문제
수열 $\{ a_{n} \}$ 이 모든 자연수 $n$ 에 대하여
$$\sum_{k=1}^{n} ( a_{2k-1} + a_{2k} ) = 2n^{2} - n$$
을 만족시킨다. $a_{10} + a_{11} = 20$ 일 때, $a_{9} + a_{12}$ 의 값을 구하시오.

## 보기

## 풀이
$\sum_{k=1}^{n} ( a_{2k-1} + a_{2k} ) = 2n^{2} - n$ 이므로 $n \geq 2$ 일 때, 
$$a_{2n-1} + a_{2n} = \sum_{k=1}^{n} ( a_{2k-1} + a_{2k} ) - \sum_{k=1}^{n-1} ( a_{2k-1} + a_{2k} )$$
$$= ( 2n^{2} - n ) - \{ 2(n-1)^{2} - (n-1) \}$$
$$= 4n - 3$$
위 식에 $n=5$ 를 대입하면
$$a_{9} + a_{10} = 4 \times 5 - 3 = 17$$
$n=6$ 을 대입하면
$$a_{11} + a_{12} = 4 \times 6 - 3 = 21$$
$\therefore ( a_{9} + a_{10} ) + ( a_{11} + a_{12} ) = 17 + 21 = 38$
이때 $a_{10} + a_{11} = 20$ 이므로
$\therefore a_{9} + a_{12} = 38 - ( a_{10} + a_{11} ) = 38 - 20 = 18$

정답: $18$
