## 문제
수열 $\{ a_{n} \}$의 첫째항부터 제 $n$항까지의 합 $S_{n}$이 $S_{n} = n^{2} + 2^{n}$일 때, $a_{1} + a_{5}$의 값은?

## 보기
① $26$   ② $28$   ③ $30$   ④ $32$   ⑤ $34$

## 풀이
$a_{1} + a_{5}$
$= S_{1} + ( S_{5} - S_{4} )$
$= ( 1^{2} + 2^{1} ) + \{ ( 5^{2} + 2^{5} ) - ( 4^{2} + 2^{4} ) \}$
$= 3 + ( 57 - 32 ) = 28$

따라서 정답은 ②이다.
