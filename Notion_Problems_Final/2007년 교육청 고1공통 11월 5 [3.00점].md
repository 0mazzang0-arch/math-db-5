## 문제
세 집합 $A$, $B$, $C$에 대하여 
$A = \{ x \mid x^{2}-4 x-12<0 \}$ 
$B = \{ x \mid x^{2}-8 x+16>0 \}$ 
$C = \{ x \mid x^{2}-2 x+10 \le 0 \}$ 
일 때, $(A \cap B) \cup C$의 원소 중 정수의 개수는?

## 보기
① $3$   ② $4$   ③ $5$   ④ $6$   ⑤ $7$

## 풀이
$x^{2}-4 x-12<0$
$(x+2)(x-6)<0$
$-2<x<6 \cdots$ ㉠

$x^{2}-8 x+16>0 \rightarrow (x-4)^{2}>0 \rightarrow x \neq 4$ 인 모든 실수 $\cdots$ ㉡

$x^{2}-2 x+10 \le 0 \rightarrow (x-1)^{2}+9 \le 0 \rightarrow$ 해는 없다. $\cdots$ ㉢

㉠, ㉡, ㉢에 의하여 $(A \cap B) \cup C$의 원소 중 정수는 
$-1$, $0$, $1$, $2$, $3$, $5$이다.

따라서 만족하는 정수의 개수는 $6$개이다.

정답: ④
