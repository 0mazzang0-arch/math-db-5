## 문제
자연수 $a$, $b$, $c$ 가 $abc+ab+bc+ca+a+b+c=29$ 를 만족할 때, $a+b+c$ 를 구하면?

## 보기
① $3$   ② $5$   ③ $7$   ④ $9$   ⑤ $11$

## 풀이
[정답] ③

$abc+ab+bc+ca+a+b+c=29$ 에서
$( a+1  )  ( b+1  )  ( c+1  )  -1 =29$
$\therefore ( a+1  )  ( b+1  )  ( c+1  ) = 30 = 2 \cdot 3 \cdot 5$

$a$, $b$, $c$ 가 자연수이므로 $a+1, b+1, c+1$ 은 $2$ 이상의 정수이다.
따라서
$a+b+c = ( a+1 ) + ( b+1 ) + ( c+1 ) - 3 = (2+3+5) - 3 = 7$
