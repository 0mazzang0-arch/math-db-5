## 문제
$a_{1} a_{2} < 0$ 인 등비수열 $\{ a_{n} \}$ 에 대하여 
$a_{6} = 16$, $2a_{8} - 3a_{7} = 32$
일 때, $a_{9} + a_{11}$ 의 값은?

## 보기
① $-\frac{5}{2}$   ② $-\frac{3}{2}$   ③ $-\frac{1}{2}$   ④ $\frac{1}{2}$   ⑤ $\frac{3}{2}$

## 풀이
등비수열 $\{ a_{n} \}$ 의 공비를 $r$ 라 하면 $a_{1} a_{2} < 0$ 에서 
$a_{1} \times ( a_{1} r ) = a_{1}^{2} r < 0$ 
$\therefore r < 0$ 

$a_{6} = 16$ 에서 
$a_{1} r^{5} = 16$ $\cdots$ ㉠ 

$2a_{8} - 3a_{7} = 32$ 에서 
$2a_{1} r^{7} - 3a_{1} r^{6} = 32$
$a_{1} r^{5} ( 2r^{2} - 3r ) = 32$ $\cdots$ ㉡ 

㉠, ㉡을 연립하면 
$16 ( 2r^{2} - 3r ) = 32$
$2r^{2} - 3r = 2$
$2r^{2} - 3r - 2 = 0$
$( 2r + 1 ) ( r - 2 ) = 0$ 

$\therefore r = -\frac{1}{2}$ ($\because r < 0$) 

$\therefore a_{9} + a_{11}$ 
$= a_{1} r^{8} + a_{1} r^{10}$ 
$= a_{1} r^{5} ( r^{3} + r^{5} )$ 
$= 16 \times \left( -\frac{1}{8} - \frac{1}{32} \right)$ 
$= 16 \times \left( -\frac{5}{32} \right)$
$= -\frac{5}{2}$

정답은 ①입니다.
