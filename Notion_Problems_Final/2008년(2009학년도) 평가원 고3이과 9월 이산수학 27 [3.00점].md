## 문제
좌표평면에서 곡선 $y = \cos^{n} x$ ($0 < x < \frac{\pi}{2}$, $n = 2, 3, 4, \dots$)의 변곡점의 $y$ 좌표를 $a_n$이라 할 때, $\lim_{n \to \infty} a_n$의 값은?

## 보기
① $\frac{1}{e^2}$   ② $\frac{1}{e}$   ③ $\frac{1}{\sqrt{e}}$   ④ $\frac{1}{2e}$   ⑤ $\frac{1}{\sqrt{2e}}$

## 풀이
$y = \cos^n x$ 

$y' = -n \cos^{n-1} x \sin x$ 

$y'' = n(n-1) \cos^{n-2} x \sin^2 x - n \cos^{n-1} x \cos x$ 

$= n(n-1) \cos^{n-2} x (1 - \cos^2 x) - n \cos^{n-1} x \cos x$ 

$= n(n-1) \cos^{n-2} x - n(n-1) \cos^n x - n \cos^n x$ 

$= n(n-1) \cos^{n-2} x - n^2 \cos^n x$ 

$= \cos^{n-2} x (n^2 - n - n^2 \cos^2 x)$ 

$= \cos^{n-2} x (n^2 \sin^2 x - n)$ 

$0 < x < \frac{\pi}{2}$ 에서 $\cos x \neq 0$ 이므로 

$y'' = 0$ 에서 
$n^2 \sin^2 x - n = 0$, $\sin^2 x = \frac{1}{n}$ 

꼭지점의 좌표를 $(b_n, a_n)$ 이라 하면 

$\sin^2 b_n = \frac{1}{n}$ 이므로 
$a_n = \cos^n b_n = (\cos^2 b_n)^{\frac{n}{2}} = (1 - \frac{1}{n})^{\frac{n}{2}}$ 

$\therefore \lim_{n \to \infty} a_n = \lim_{n \to \infty} (1 - \frac{1}{n})^{\frac{n}{2}}$ 
$= \lim_{n \to \infty} \{(1 - \frac{1}{n})^{-n}\}^{-\frac{1}{2}}$ 
$= e^{-\frac{1}{2}} = \frac{1}{\sqrt{e}}$

[정답] ③
