## 문제
전체집합 $U = \{ 1, 2, 3, 4 \}$ 의 공집합이 아닌 두 부분집합 $A$, $B$ 에 대하여 두 명제
- $A$ 의 모든 원소 $x$ 에 대하여 $x^{2} - 3x < 0$ 이다.
- $B$ 의 어떤 원소 $x$ 에 대하여 $x \in A$ 이다.
가 있다. 두 명제가 모두 참이 되도록 하는 두 집합 $A$, $B$ 의 모든 순서쌍 $( A, B )$ 의 개수를 구하시오.

## 보기

## 풀이
조건 $x^{2} - 3x < 0$ 의 진리집합을 $P$ 라 하면
$x ( x-3 ) < 0$ 에서 $0 < x < 3$ 이므로
$P = \{ 1, 2 \}$
명제 '집합 $A$ 의 모든 원소 $x$ 에 대하여 $x^{2} - 3x < 0$ 이다.' 가 참이 되기 위해서는 집합 $A$ 가 집합 $P$ 의 공집합이 아닌 부분집합이어야 한다.
그러므로 $A = \{ 1 \}$ 또는 $A = \{ 2 \}$ 또는 $A = \{ 1, 2 \}$ 이다.
명제 '집합 $B$ 의 어떤 원소 $x$ 에 대하여 $x \in A$ 이다.' 가 참이 되기 위해서는 $A \cap B \neq \emptyset$ 이어야 한다.

(ⅰ) $A = \{ 1 \}$ 인 경우
집합 $B$ 는 $1$ 을 원소로 갖는 집합 $U$ 의 부분집합이므로 집합 $B$ 의 개수는 $2^{3} = 8$

(ⅱ) $A = \{ 2 \}$ 인 경우
집합 $B$ 는 $2$ 를 원소로 갖는 집합 $U$ 의 부분집합이므로 집합 $B$ 의 개수는 $2^{3} = 8$

(ⅲ) $A = \{ 1, 2 \}$ 인 경우
집합 $B$ 는 $1$ 또는 $2$ 를 원소로 갖는 집합 $U$ 의 부분집합이다.
ⅲ-가) $1$ 을 원소로 갖고, $2$ 를 원소로 갖지 않는 집합 $B$ 의 개수는 $2^{2} = 4$
ⅲ-나) $2$ 를 원소로 갖고, $1$ 를 원소로 갖지 않는 집합 $B$ 의 개수는 $2^{2} = 4$
ⅲ-다) $1$, $2$ 를 모두 원소로 갖는 집합 $B$ 의 개수는 $2^{2} = 4$

따라서 (ⅰ), (ⅱ), (ⅲ)에 의하여 구하는 순서쌍 $( A, B )$ 의 개수는
$8 + 8 + ( 4 + 4 + 4 ) = 28$

정답: $28$
