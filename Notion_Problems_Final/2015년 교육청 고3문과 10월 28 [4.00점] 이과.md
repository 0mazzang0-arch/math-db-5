## 문제
좌표평면 위의 점 $P$가 다음 규칙에 따라 이동한다. 

$$ \begin{array}{|l|} \hline \text{(가) } \\ \text{(나) 동전 } 1\text{개를 던져서 앞면이 나오면 } x\text{축의 방향으로 } 1\text{만큼 평행이동한다.} \\ \text{(다) 동전 } 1\text{개를 던져서 뒷면이 나오면 } x\text{축의 방향으로 } 1\text{만큼, } y\text{축의 방향으로 } 1\text{만큼 평행이동한다.} \\ \hline \end{array} $$

$1$개의 동전을 $6$번 던져서 점 $P$가 $(a, b)$로 이동하였다. $a+b$가 $3$의 배수가 될 확률이 $\frac{q}{p}$일 때, $p+q$의 값을 구하시오. (단, $p, q$는 서로소인 자연수이다.)

## 풀이
동전을 $1$번 던질 때마다 $x$좌표는 항상 $1$만큼 증가하므로, $6$번 던진 후의 $x$좌표 $a = 6$이다.
또한, 동전을 던져 뒷면이 나오는 횟수를 $b$라 하면 $0 \le b \le 6$이고, 이때의 $y$좌표는 $b$가 된다.
$a+b = 6+b$가 $3$의 배수가 되는 경우는 $b$가 $3$의 배수인 경우이므로
$b=0$ 또는 $b=3$ 또는 $b=6$이다.

$1)$ $b=0$일 확률: ${}_{6}C_{0} \left( \frac{1}{2} \right)^{0} \left( \frac{1}{2} \right)^{6} = \frac{1}{64}$
$2)$ $b=3$일 확률: ${}_{6}C_{3} \left( \frac{1}{2} \right)^{3} \left( \frac{1}{2} \right)^{3} = \frac{20}{64}$
$3)$ $b=6$일 확률: ${}_{6}C_{6} \left( \frac{1}{2} \right)^{6} \left( \frac{1}{2} \right)^{0} = \frac{1}{64}$

따라서 구하는 확률은
$\frac{1}{64} + \frac{20}{64} + \frac{1}{64} = \frac{22}{64} = \frac{11}{32}$
즉, $p=32, q=11$이므로
$p+q = 32+11 = 43$
