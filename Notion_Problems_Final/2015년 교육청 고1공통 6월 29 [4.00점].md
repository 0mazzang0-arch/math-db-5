## 문제
삼차다항식 $f(x)$가 다음 조건을 만족시킨다.

(가) $f(1) = 2$
(나) $f(x)$를 $(x-1)^2$으로 나눈 몫과 나머지가 같다.

$f(x)$를 $(x-1)^3$으로 나눈 나머지를 $R(x)$라 하자. $R(0) = R(3)$일 때, $R(5)$의 값을 구하시오.

## 풀이
조건 (나)에서 $f(x)$를 $(x-1)^2$으로 나누었을 때의 몫과 나머지가 같으므로 그때의 몫을 $ax+b$ ($a$, $b$는 상수)라 하면
$f(x) = (x-1)^2 (ax+b) + (ax+b) \cdots$ ㉠
조건 (가)에서 $f(1) = 2$이므로 ㉠의 양변에 $x=1$을 대입하면
$f(1) = a+b = 2$
$\therefore b = -a+2$
따라서 $ax+b = a(x-1)+2$이므로 이것을 ㉠에 대입하면
$f(x) = (x-1)^2 \{ a(x-1)+2 \} + a(x-1)+2$
$= a(x-1)^3 + 2(x-1)^2 + a(x-1) + 2$
따라서 $f(x)$를 $(x-1)^3$으로 나누었을 때의 나머지는
$R(x) = 2(x-1)^2 + a(x-1) + 2$
$R(0) = R(3)$이므로
$2-a+2 = 8+2a+2$
$\therefore a = -2$
따라서 $R(x) = 2(x-1)^2 - 2(x-1) + 2$이므로
$R(5) = 2(5-1)^2 - 2(5-1) + 2 = 2(4)^2 - 2(4) + 2 = 32 - 8 + 2 = 26$

정답: $26$
