## 문제
좌표평면 위의 두 점 $A(5, 12)$, $B(a, b)$에 대하여 선분 $AB$의 길이가 $3$일 때, $a^{2} + b^{2}$의 최댓값을 구하시오.

## 보기

## 풀이
두 점 $A(5, 12)$, $B(a, b)$에 대하여 선분 $AB$의 길이가 $3$이므로 
$$\sqrt{(a-5)^{2} + (b-12)^{2}} = 3$$ 

$$\therefore (a-5)^{2} + (b-12)^{2} = 9$$ 
따라서 점 $B$는 원 $(x-5)^{2} + (y-12)^{2} = 9$ 위의 점이다. 

> [!example] 📸 (그림/그래프)

원점 $O$에 대하여 $a^{2} + b^{2} = \overline{OB}^{2}$이므로 선분 $OB$의 길이가 최대일 때 $a^{2} + b^{2}$이 최댓값을 갖는다. 원의 반지름의 길이가 $3$이므로 선분 $OB$의 최댓값을 $M$이라 하면 

$$M = \overline{OA} + 3$$ 
$$= \sqrt{5^{2} + 12^{2}} + 3$$ 
$$= \sqrt{169} + 3$$ 
$$= 13 + 3$$ 
$$= 16$$ 

따라서 $a^{2} + b^{2}$의 최댓값은 
$$M^{2} = 16^{2} = 256$$

정답: $256$
