## 문제
다음 조건을 만족시키는 자연수 $a$, $b$, $c$, $d$, $e$의 모든 순서쌍 $(a, b, c, d, e)$의 개수는?

(가) $a+b+c+d+e=12$
(나) $| a^{2} - b^{2} | = 5$

## 보기
① $30$   ② $32$   ③ $34$   ④ $36$   ⑤ $38$

## 풀이
조건 (나)에서 
$a^{2} - b^{2} = -5$ 또는 $a^{2} - b^{2} = 5$
즉, 
$(b - a)(b + a) = 5$ 또는 $(a - b)(a + b) = 5$
이고 $a, b$는 자연수이므로 
$b - a = 1, b + a = 5$ 또는 $a - b = 1, a + b = 5$
따라서, $a = 2, b = 3$ 또는 $a = 3, b = 2$이다. 

또한, 조건 (가)에서 
$a + b + c + d + e = 12$
이므로 $c + d + e = 7$이고 $c, d, e$는 자연수이므로 
$c = c' + 1, d = d' + 1, e = e' + 1$ ($c', d', e'$은 음이 아닌 정수)
로 놓으면 
$(c' + 1) + (d' + 1) + (e' + 1) = 7$
$c' + d' + e' = 4$
이를 만족시키는 모든 순서쌍 $(c', d', e')$의 개수는 
${}_{3}H_{4} = {}_{3+4-1}C_{4} = {}_{6}C_{4} = {}_{6}C_{2} = \frac{6 \times 5}{2 \times 1} = 15$
따라서, 구하는 모든 순서쌍 $(a, b, c, d, e)$의 개수는 
$2 \times 15 = 30$
