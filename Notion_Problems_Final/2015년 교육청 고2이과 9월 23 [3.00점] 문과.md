## 문제
전체집합 $U = \{ x \mid x$는 자연수 $\}$의 두 부분집합 $A$, $B$에 대하여 $A = \{ x \mid x$는 $4$의 약수 $\}$, $B = \{ x \mid x$는 $12$의 약수 $\}$일 때, $A \subset X \subset B$를 만족시키는 집합 $X$의 개수를 구하시오.

## 보기

## 풀이
$A = \{ 1, 2, 4 \}$, $B = \{ 1, 2, 3, 4, 6, 12 \}$이므로 
$\{ 1, 2, 4 \} \subset X \subset \{ 1, 2, 3, 4, 6, 12 \}$이다. 
조건을 만족시키는 집합 $X$의 개수는 집합 $B$의 부분집합 중 $\{ 1, 2, 4 \}$를 반드시 원소로 갖는 집합의 개수와 같으므로
$2^{6-3} = 2^{3} = 8$
정답: $8$
