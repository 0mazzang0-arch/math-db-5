## 문제
집합 $S = \{ 0, 1, 2, \dots \}$에서 정의된 함수 $f : S \to S$가 $S$의 모든 원소 $m$, $n$에 대하여 다음을 만족시킨다.
$$ f(m+f(n)) = f(f(m)) + f(n) $$
다음 중 참인 것을 모두 고르면?

$$ \begin{array}{|l|} \hline \text{가. } f(0) = 0 \\ \text{나. } f(m+f(n)) = f(m) + f(n) \\ \text{다. } f(f(n)) = f(n) \\ \hline \end{array} $$

## 보기
① 가   ② 나   ③ 다   ④ 나, 다   ⑤ 가, 나, 다

## 풀이
주어진 식: $f(m+f(n)) = f(f(m)) + f(n)$

가. $m = 0$, $n = 0$인 경우
$f(0+f(0)) = f(f(0)) + f(0)$
$f(0) = \alpha$ ($\alpha \neq 0$)라고 가정하면
$f(\alpha) = f(\alpha) + \alpha$ 이므로 $\alpha = 0$이 되어 가정에 모순이다.
$\therefore f(0) = 0$ (참)

다. $m = 0$인 경우
$f(0+f(n)) = f(f(0)) + f(n)$
$f(f(n)) = f(0) + f(n)$
가에서 $f(0) = 0$이므로 $f(f(n)) = 0 + f(n) = f(n)$ (참)

나. 다의 결과인 $f(f(m)) = f(m)$을 원래의 식 $f(m+f(n)) = f(f(m)) + f(n)$에 대입하면
$f(m+f(n)) = f(m) + f(n)$ (참)

따라서 가, 나, 다 모두 참이다.

정답: ⑤
