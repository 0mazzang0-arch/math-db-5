## 문제
세 집합 $A$, $B$, $C$가 각각
$$ \begin{array}{l} A = \{ (a, a + 3b) \mid a, b \text{ 는 정수} \} \\ B = \{ (a + 3b, a) \mid a, b \text{ 는 정수} \} \\ C = \{ (a + 2b, 2a + b) \mid a, b \text{ 는 정수} \} \end{array} $$
일 때, 다음 중 $A$, $B$, $C$의 관계식으로 옳은 것은?

## 보기
① $A = B = C$   ② $A \neq B \neq C$   ③ $A = B \neq C$

## 풀이
$A = \{ (a, a + 3b) \mid a, b \text{ 는 정수} \}$는 $(x, y) = (a, a + 3b)$이므로 $y = x + 3b$ ($b$는 정수) 위의 점을 원소로 가진다. 

$B = \{ (a + 3b, a) \mid a, b \text{ 는 정수} \}$의 경우 같은 방법으로 자취를 구하면 $y = x - 3b$ ($b$는 정수) 위의 점을 원소로 가진다. 

$C = \{ (a + 2b, 2a + b) \mid a, b \text{ 는 정수} \}$는 $y = 2x - 3b$ ($b$는 정수) 위의 점을 원소로 가진다. 

그러므로 $A = B \neq C$

[정답] ③
