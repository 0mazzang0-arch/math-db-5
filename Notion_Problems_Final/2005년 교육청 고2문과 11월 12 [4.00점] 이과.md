## 문제
거듭제곱근에 대한 설명으로 <보기>에서 옳은 것을 모두 고르면?

$$
\begin{array}{|l|}
\hline
\text{ㄱ. } \sqrt[4]{\sqrt[3]{5}} = \sqrt[7]{5} \\
\text{ㄴ. } [\sqrt[3]{1}] + [\sqrt[3]{2}] + [\sqrt[3]{3}] + \cdots + [\sqrt[3]{36}] = 75 \\
\text{ㄷ. } [\sqrt{a}] + [\sqrt{10-a}] = [\sqrt{10}] \text{ 을 만족하는 자연수 } a \text{ 는 } 5 \text{ 개이다.} \\
\hline
\end{array}
$$

(단, $[x]$는 $x$를 넘지 않는 최대 정수이다.)

## 보기
① $ㄴ$   ② $ㄷ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
정답: ④

ㄱ. $\sqrt[4]{\sqrt[3]{5}} = \sqrt[12]{5} \neq \sqrt[7]{5}$ 이므로 옳지 않다.

ㄴ. 
$1 \le n < 8$ 이면 $[\sqrt[3]{n}] = 1$
$8 \le n < 27$ 이면 $[\sqrt[3]{n}] = 2$
$27 \le n \le 36$ 이면 $[\sqrt[3]{n}] = 3$
따라서 $[\sqrt[3]{1}] + [\sqrt[3]{2}] + \cdots + [\sqrt[3]{36}]$
$= 1 \times 7 + 2 \times 19 + 3 \times 10 = 75$ 이므로 옳다.

ㄷ. 
$1 \le a \le 10$ 이면 $1 \le [\sqrt{a}] \le 3$ 이다.
$[\sqrt{10}] = 3$ 이므로
(i) $[\sqrt{a}] = 1$, $[\sqrt{10-a}] = 2$ 인 경우
$[\sqrt{a}] = 1$ 이면 $1 \le \sqrt{a} < 2$ 이므로 $1 \le a < 4$
$[\sqrt{10-a}] = 2$ 이면 $2 \le \sqrt{10-a} < 3$ 이므로 $1 < a \le 6$
$\therefore 1 < a < 4$ 에서 $a = 2, 3$

(ii) $[\sqrt{a}] = 2$, $[\sqrt{10-a}] = 1$ 인 경우
같은 방법으로 $6 < a < 9$ $\therefore a = 7, 8$

(iii) $[\sqrt{a}] = 3$, $[\sqrt{10-a}] = 0$ 인 경우
같은 방법으로 $9 < a \le 10$ $\therefore a = 10$

따라서 (i), (ii), (iii)에서 $a = 2, 3, 7, 8, 10$ 이다. 
이를 만족하는 자연수 $a$ 의 개수는 $5$ 개이므로 옳다.

그러므로 옳은 것은 ㄴ, ㄷ이다.
