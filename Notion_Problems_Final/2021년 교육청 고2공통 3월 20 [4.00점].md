## 문제
세 집합 $X = \{ 1, 2, 3, 4 \}$, $Y = \{ 2, 3, 4, 5 \}$, $Z = \{ 3, 4, 5 \}$에 대하여 두 함수 $f : X \to Y$, $g : Y \to Z$가 다음 조건을 만족시킨다.

$$ \begin{array}{|l|} \hline \text{(가) 함수 } f \text{는 일대일대응이다.} \\ \text{(나) } x \in (X \cap Y) \text{이면 } g(x) - f(x) = 1 \text{이다.} \\ \hline \end{array} $$

<보기>에서 옳은 것만을 있는 대로 고른 것은?

$$ \begin{array}{|l|} \hline \text{ㄱ. 함수 } g \circ f \text{의 치역은 } Z \text{이다.} \\ \text{ㄴ. } f^{-1}(5) \geq 2 \\ \text{ㄷ. } f(3) < g(2) < f(1) \text{이면 } f(4) + g(2) = 6 \text{이다.} \\ \hline \end{array} $$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
ㄱ. 함수 $f$ 는 일대일대응이고 집합 $X \cap Y = \{ 2, 3, 4 \}$ 의 모든 원소 $x$ 에 대하여 $g(x) - f(x) = 1$ 이므로 $f(x) = 5$ 인 $x$ 가 존재하면 $g(x) = 6$ 이 되어 모순이다.
$X \cap Y = \{ 2, 3, 4 \}$ 의 모든 원소 $x$ 에 대하여 $f(x) \leq 4$ 이고 함수 $f$ 는 일대일대응이므로 
$\{ f(2), f(3), f(4) \} = \{ 2, 3, 4 \}$
$g(x) = f(x) + 1$ 에서 
$\{ g(2), g(3), g(4) \} = \{ 3, 4, 5 \}$
$g \circ f$ 의 치역은 $Z$ 이다. (참)

ㄴ. ㄱ에서 $\{ f(2), f(3), f(4) \} = \{ 2, 3, 4 \}$ 이고 함수 $f$ 는 일대일대응이므로 
$f(1) = 5$
$f^{-1}(5) = 1$ (거짓)

ㄷ. ㄴ에서 $f(1) = 5$ 이므로 
$f(3) < g(2) < f(1)$ 에서 
$f(3) < g(2) < 5$ $\cdots$ ㉠

$g(2) = 3$ 인 경우 
$f(2) = g(2) - 1 = 2$
$f$ 는 일대일대응이므로 $f(3) = 3$ 또는 $f(3) = 4$ 이나, 이는 ㉠의 $f(3) < 3$ 에 모순이다.

$g(2) = 4$ 인 경우 
$f(2) = g(2) - 1 = 3$ $\cdots$ ㉡
$f(3) < 4$ 이고 $f(3) \in \{ 2, 4 \}$ 이므로 $f(3) = 2$
$f$ 는 일대일대응이므로 $f(4) = 4$
$f(4) + g(2) = 4 + 4 = 8$ (거짓)

이상에서 옳은 것은 ㄱ이다.

정답: ①
