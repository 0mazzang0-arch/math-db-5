## 문제
집합 $C = \{ (x, y) \mid x + 2y \le 30, 3x + 2y \le 50, x \ge 0, y \ge 0 \}$ 와 집합 $M_{k} = \{ (x, y) \mid x + y = k \}$ (단, $k = 0, 1, 2, 3, \dots$) 에 대하여 $C \cap M_{k} \neq \emptyset$ 인 $k$ 중에서 가장 큰 값과 가장 작은 값의 합은?

## 보기
① $10$   ② $15$   ③ $\frac{50}{3}$   ④ $20$   ⑤ $25$

## 풀이
집합 $C$ 의 영역을 좌표평면에 표시하면 아래 그림과 같다. 

> [!example] 📸 (그림/그래프)

따라서, 조건 $C \cap M_{k} \neq \emptyset$ 을 만족하기 위해서 직선 $x + y = k$ 는 빗금 친 영역을 지나야 한다. 직선 $x + y = k$ 가 점 $(10, 10)$ 와 원점을 지날 때 $k$ 의 값은 각각 최댓값과 최솟값이다. 

$1$) 점 $(10, 10)$ 을 지날 때 : 
$x + y = 10 + 10 = k$ 
$\therefore k = 20$ 

$2$) 원점을 지날 때 : 
$x + y = 0 + 0 = k$ 
$\therefore k = 0$ 

따라서, 최대, 최소일 때의 $k$ 값의 합은 
$\therefore 20 + 0 = 20$
