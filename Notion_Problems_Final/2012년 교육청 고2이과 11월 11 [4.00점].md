## 문제
이차방정식 $25x^{2} - 25x + 4 = 0$의 두 근이 $\sin(a+b)$, $\sin(a-b)$일 때, $\frac{\tan a}{\tan b}$의 값은? (단, $0 < b < a < \frac{\pi}{4}$)

## 보기
① $\frac{3}{5}$   ② $\frac{3}{4}$   ③ $\frac{4}{5}$   ④ $\frac{4}{3}$   ⑤ $\frac{5}{3}$

## 풀이
$25x^{2} - 25x + 4 = 0$의 두 근이 $\frac{4}{5}$, $\frac{1}{5}$이고 주어진 조건 $0 < b < a < \frac{\pi}{4}$에 의하여 $\sin(a-b) < \sin(a+b)$이므로 
$\sin(a+b) = \frac{4}{5}$, $\sin(a-b) = \frac{1}{5}$이다. 

삼각함수의 덧셈정리에 의하여 
$\sin(a+b) = \sin a \cos b + \cos a \sin b = \frac{4}{5} \cdots$ ㉠ 
$\sin(a-b) = \sin a \cos b - \cos a \sin b = \frac{1}{5} \cdots$ ㉡ 

㉠, ㉡을 연립하여 $\sin a \cos b$와 $\cos a \sin b$의 값을 구하면 
$\sin a \cos b = \frac{1}{2}$ 
$\cos a \sin b = \frac{3}{10}$ 

따라서 구하는 값은 
$\frac{\tan a}{\tan b} = \frac{\frac{\sin a}{\cos a}}{\frac{\sin b}{\cos b}} = \frac{\sin a \cos b}{\cos a \sin b} = \frac{\frac{1}{2}}{\frac{3}{10}} = \frac{5}{3}$

정답은 ⑤이다.
