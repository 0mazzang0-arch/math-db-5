## 문제
$-2 \leq x \leq 5$ 에서 정의된 이차함수 $f(x)$ 가 $f(0) = f(4)$, $f(-1) + |f(4)| = 0$ 을 만족시킨다. 함수 $f(x)$ 의 최솟값이 $-19$ 일 때, $f(3)$ 의 값을 구하시오.



> [!example] 📸 (그림/그래프)

## 보기


## 풀이
$f(0) = f(4)$ 이므로 이차함수 $f(x)$ 의 대칭축은 $x = 2$ 이다.
$f(x) = a(x - 2)^{2} + b$ ($a$, $b$ 는 상수, $a \neq 0$)이라 하자.
이차함수 $f(x)$ 의 대칭축이 $x = 2$ 이므로 $f(-1) \neq f(4)$ 이다.
따라서 $f(-1) + |f(4)| = 0$ 에서 $f(-1) = f(4) = 0$ 은 성립하지 않으므로 $f(-1) = -|f(4)| < 0$ 이고 $|f(-1)| = |f(4)|$ $\cdots$ ㉠

(ⅰ) $a > 0$ 인 경우
$f(4) < f(-1) < 0$ 이 되어 ㉠을 만족시키지 않는다.

(ⅱ) $a < 0$ 인 경우
$f(-1) < 0$ 이므로 $f(4) > 0$ 이다.
$f(-1) + |f(4)| = 0$ 에서 $f(-1) + f(4) = 13a + 2b = 0 \cdots$ ㉡
$a < 0$ 이므로 $-2 \leq x \leq 5$ 에서 함수 $f(x)$ 의 최솟값은 $f(-2) = 16a + b = -19 \cdots$ ㉢
㉡, ㉢에서 $a = -2$, $b = 13$ 이다.
따라서 $f(x) = -2(x - 2)^{2} + 13$ 이므로 $f(3) = 11$ 이다.

정답: $11$
