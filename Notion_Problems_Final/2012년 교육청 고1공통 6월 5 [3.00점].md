## 문제
등식 `$ ( k + 3 ) x - ( 3k + 4 ) y + 5k = 0 $` 이 `$ k $` 의 값에 관계없이 항상 성립할 때, `$ x + y $` 의 값은?

## 보기
① `$ 6 $`   ② `$ 7 $`   ③ `$ 8 $`   ④ `$ 9 $`   ⑤ `$ 10 $`

## 풀이
[정답] ② 

등식 `$ ( k + 3 ) x - ( 3k + 4 ) y + 5k = 0 $` 을 `$ k $` 에 대하여 정리하면 
`$ ( x - 3y + 5 ) k + ( 3x - 4y ) = 0 $` 
위 등식은 `$ k $` 에 대한 항등식이므로 
`$ x - 3y + 5 = 0 $`, `$ 3x - 4y = 0 $` 
위 두 식을 연립하면 `$ x = 4 $`, `$ y = 3 $` 

`$ \therefore x + y = 7 $` 

[다른 풀이] 
등식 `$ ( k + 3 ) x - ( 3k + 4 ) y + 5k = 0 $` 은 `$ k $` 에 대한 항등식이므로 위 등식의 양변에 `$ k = 0 $` 을 대입하면 
`$ 3x - 4y = 0 \cdots \cdots ㉠ $` 
위 등식의 양변에 `$ k = -1 $` 을 대입하면 
`$ 2x - y - 5 = 0 \cdots \cdots ㉡ $` 
`$ ㉠ $`, `$ ㉡ $` 을 연립하면 `$ x = 4 $`, `$ y = 3 $` 

`$ \therefore x + y = 7 $`
