## 문제
전체집합 $U = \{ 1, 2, 3, 4, 5, 6, 7, 8 \}$의 두 부분집합 $A$, $B$가 다음 조건을 만족시킨다.

(가) $A \cap B = \{ 3, 5 \}$
(나) $A^{C} \cap B^{C} = \{ 1, 7 \}$

집합 $X$의 모든 원소의 합을 $S(X)$라 할 때, $S(A) = 2 S(B)$가 되도록 하는 두 집합 $A$, $B$에 대하여 $S(A)$의 값을 구하시오.

## 보기

## 풀이
$S(A \cap B) = 8$
$A^{C} \cap B^{C} = (A \cup B)^{C}$이므로 
$A \cup B = \{ 2, 3, 4, 5, 6, 8 \}$이고 
$S(A \cup B) = 28$
$S(A) + S(B) = S(A \cup B) + S(A \cap B) = 28 + 8 = 36$
이때 $S(A) = 2 S(B)$에서 $S(B) = \frac{1}{2} S(A)$이므로
$S(A) + S(B) = S(A) + \frac{1}{2} S(A) = \frac{3}{2} S(A) = 36$
따라서 $S(A) = 24$이다.
