## 문제
수열 $\{ a_{n} \}$ 에 대하여 $\sum_{k=1}^{5} a_{k} = 10$ 일 때, $\sum_{k=1}^{5} c a_{k} = 65 + \sum_{k=1}^{5} c$ 를 만족시키는 상수 $c$ 의 값을 구하시오.

## 풀이
$\sum_{k=1}^{5} c a_{k} = c \sum_{k=1}^{5} a_{k} = 10c$ 이고,
$\sum_{k=1}^{5} c = 5c$ 이다.
그러므로 주어진 식에 대입하면
$\sum_{k=1}^{5} c a_{k} = 65 + \sum_{k=1}^{5} c$
$10c = 65 + 5c$
$5c = 65$
$\therefore c = 13$

정답: $13$
