## 문제
`$1 \le x < 10$` 인 실수 `$x$` 에 대하여 `$\frac{x^{3}}{[x]}$` 의 값이 자연수가 되는 `$x$` 의 개수를 구하시오. (단, `$[x]$` 는 `$x$` 보다 크지 않은 최대의 정수이다.)

## 풀이
`$[x]=n$` 이라 두면 
`$n \le x < n+1$` 
`$\Rightarrow n^{3} \le x^{3} < (n+1)^{3} = n^{3} + 3n^{2} + 3n + 1$` 
`$\Rightarrow n^{2} \le \frac{x^{3}}{n} < \frac{n^{3} + 3n^{2} + 3n + 1}{n} = n^{2} + 3n + 3 + \frac{1}{n}$` 

따라서 `$[x]=n$` 일 때, `$\frac{x^{3}}{[x]}$` 가 자연수가 되는 `$x$` 는 총 
`$n^{2} + 3n + 3 - (n^{2}) + 1 = 3n + 4$` 개가 존재한다. 
그런데 `$1 \le x < 10$` 이므로 `$[x]$` 값은 `$1 \le [x] < 10$` 이며, 
`$1$`, `$2$`, `$\cdots$`, `$9$` 까지 총 `$9$` 개가 존재한다. 

따라서 `$x$` 의 개수 
`$= \sum_{n=1}^{9}(3n + 4) = 3 \cdot 45 + 4 \cdot 9 = 171$`

정답: `$171$`
