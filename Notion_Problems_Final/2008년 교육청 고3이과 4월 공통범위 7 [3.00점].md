## 문제
서로 다른 세 양의 실수 $A$, $B$, $C$가 다음 조건을 모두 만족할 때, $A$, $B$, $C$의 대소 관계를 바르게 나타낸 것은?

$$ \begin{array}{l} \text{(가) } \log_{10} \frac{A}{B} > 0 \\ \text{(나) } \log_{10} A - 2 \log_{10} B + \log_{10} C = 0 \end{array} $$

## 보기
① $A < B < C$   ② $A < C < B$   ③ $B < A < C$   ④ $B < C < A$   ⑤ $C < B < A$

## 풀이
(가)에서 $\log_{10} \frac{A}{B} > 0$ 이므로 $\log_{10} A > \log_{10} B$ 이고, 따라서 $A > B$ 이다.

(나)에서 $\log_{10} A - 2 \log_{10} B + \log_{10} C = 0$ 이므로
$\log_{10} A - \log_{10} B = \log_{10} B - \log_{10} C$ 이다.

(가)에 의해 $\log_{10} A - \log_{10} B > 0$ 이므로 
$\log_{10} B - \log_{10} C > 0$ 이 되어 $B > C$ 이다.

$\therefore C < B < A$

정답: ⑤
