## 문제
확률변수 $X$가 정규분포 $N(4, 3^{2})$을 따를 때, $\sum_{n=1}^{7} P(X \leq n) = a$이다. $10a$의 값을 구하시오.

## 보기

## 풀이
확률변수 $X$가 정규분포 $N(4, 3^{2})$을 따르므로 확률밀도함수의 그래프는 직선 $x=4$에 대하여 대칭이다.
따라서 다음의 성질이 성립한다.
$P(X \leq 1) + P(X \leq 7) = 1$
$P(X \leq 2) + P(X \leq 6) = 1$
$P(X \leq 3) + P(X \leq 5) = 1$
$P(X \leq 4) = \frac{1}{2}$

이 성질을 이용하여 $a$의 값을 구하면 다음과 같다.
$\sum_{n=1}^{7} P(X \leq n) = \{P(X \leq 1) + P(X \leq 7)\} + \{P(X \leq 2) + P(X \leq 6)\} + \{P(X \leq 3) + P(X \leq 5)\} + P(X \leq 4)$
$a = 1 + 1 + 1 + \frac{1}{2} = \frac{7}{2}$

따라서 $10a$의 값은 다음과 같다.
$10a = 10 \times \frac{7}{2} = 35$

정답 $35$
