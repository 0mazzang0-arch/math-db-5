## 문제
최고차항의 계수가 $1$ 인 삼차함수 $f(x)$ 의 역함수를 $g(x)$ 라 할 때, $g(x)$ 가 다음 조건을 만족시킨다.

$$ \begin{array}{|l|} \hline \text{(가) } g(x) \text{ 는 실수 전체의 집합에서 미분가능하고 } g'(x) \leq \frac{1}{3} \text{ 이다.} \\ \text{(나) } \lim_{x \to 3} \frac{f(x)-g(x)}{(x-3)g(x)} = \frac{8}{9} \\ \hline \end{array} $$

$f(1)$ 의 값은?

## 보기
① $-11$   ② $-9$   ③ $-7$   ④ $-5$   ⑤ $-3$

## 풀이
조건 (나)에서 $x \to 3$ 일 때, (분모) $\to 0$ 이므로 (분자) $\to 0$ 이어야 한다.

$$\lim_{x \to 3} \{ f(x)-g(x) \} = 0$$

$$\therefore f(3)=g(3)$$

또한, $f(x)$ 와 $g(x)$ 는 서로 역함수의 관계이므로

$$f(3)=g(3)=3 \cdots \cdots ㉠$$

$$\lim_{x \to 3} \frac{f(x)-g(x)}{(x-3) g(x)}$$

$$= \lim_{x \to 3} \frac{f(x)-f(3)-g(x)+g(3)}{(x-3) g(x)}$$

$$= \lim_{x \to 3} \left\{ \frac{f(x)-f(3)}{x-3} \times \frac{1}{g(x)} - \frac{g(x)-g(3)}{x-3} \times \frac{1}{g(x)} \right\}$$

$$= f'(3) \times \frac{1}{g(3)} - g'(3) \times \frac{1}{g(3)}$$

$$= \frac{1}{3} \{ f'(3)-g'(3) \} = \frac{8}{9}$$

이 때, $g'(3) = \frac{1}{f'(g(3))} = \frac{1}{f'(3)}$ 이므로

$$\frac{1}{3} \left\{ f'(3) - \frac{1}{f'(3)} \right\} = \frac{8}{9}$$

$$3 \{ f'(3) \}^2 - 8 f'(3) - 3 = 0$$

$$\{ 3 f'(3) + 1 \} \{ f'(3) - 3 \} = 0$$

$$\therefore f'(3) = -\frac{1}{3} \text{ 또는 } f'(3) = 3$$

그런데, $f(x)$ 의 삼차항의 계수는 양수이고 역함수가 존재해야 하므로 $f'(x) \geq 0$ 이다.

$$\therefore f'(3) = 3 \cdots \cdots ㉡$$

㉠에 의하여

$$f(x)-3 = (x-3) (x^2 + ax + b)$$

라 하고 양변을 $x$ 에 대하여 미분하면

$$f'(x) = x^2 + ax + b + (x-3)(2x+a) \cdots \cdots ㉢$$

㉡에 의하여

$$f'(3) = 9 + 3a + b = 3$$

$$\therefore 3a + b = -6 \cdots \cdots ㉣$$

㉢의 양변을 $x$ 에 대하여 미분하면

$$f''(x) = 2x + a + (2x + a) + 2(x-3) = 6x + 2a - 6$$

이때, 변곡점의 $x$ 좌표는 $6x + 2a - 6 = 0$ 에서 $x = \frac{3-a}{3}$

그런데, $f'(3) = 3$ 에서 $g'(3) = \frac{1}{f'(3)} = \frac{1}{3}$ 이므로 조건 (가)에 의하여 $f(x)$ 의 변곡점의 $x$ 좌표는 $3$ 이 되어야 한다. 즉,

$$\frac{3-a}{3} = 3$$

$$\therefore a = -6$$

㉣에 대입하면 $b = 12$ 이므로

$$f(x)-3 = (x-3) (x^2 + ax + b) = (x-3) (x^2 - 6x + 12)$$

$$\therefore f(1) = -2 \times 7 + 3 = -11$$
