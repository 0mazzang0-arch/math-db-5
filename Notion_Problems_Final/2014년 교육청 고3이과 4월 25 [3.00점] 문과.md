## 문제
$x$에 대한 부등식 $2^{2x+1} - (2n+1)2^x + n \le 0$을 만족시키는 모든 정수 $x$의 개수가 $7$일 때, 자연수 $n$의 최댓값을 구하시오.

## 보기


## 풀이
주어진 부등식은 다음과 같이 정리된다.
$$2(2^x)^2 - (2n+1)2^x + n \le 0$$
$$(2 \cdot 2^x - 1)(2^x - n) \le 0$$
$$2^{-1} \le 2^x \le n$$
부등식을 만족시키는 모든 정수 $x$의 개수가 $7$이므로, $x$가 가질 수 있는 값은 다음과 같다.
$x = -1, 0, 1, 2, 3, 4, 5$
따라서 $2^x$의 범위는 $2^{-1} \le 2^x \le 2^5$ 가 포함되어야 하며, $x = 6$은 포함되지 않아야 하므로 $n$의 범위는 다음과 같다.
$$2^5 \le n < 2^6$$
따라서 자연수 $n$의 최댓값은 $63$이다.

정답: $63$
