## 문제
두 집합 $A = \{ x \mid \log_{4} (\log_{2} x) \le 1 \}$, $B = \{ x \mid x^{2} - 5ax + 4a^{2} < 0 \}$ 에 대하여 $A \cap B = B$를 만족시키는 자연수 $a$의 개수는?

## 보기
① $4$   ② $5$   ③ $6$   ④ $7$   ⑤ $8$

## 풀이
진수의 조건에서 
$x > 0$ 이고 
$\log_{2} x > 0$ 
$\therefore x > 1$ 

$\log_{4} (\log_{2} x) \le \log_{4} 4$ 
$\log_{2} x \le 4$ 
$\therefore x \le 16$ 

$\therefore A = \{ x \mid 1 < x \le 16 \}$ 

$x^{2} - 5ax + 4a^{2} = (x - a)(x - 4a) < 0$ 
$\therefore a < x < 4a$ 

$\therefore B = \{ x \mid a < x < 4a \}$ 

$A \cap B = B$이면 $B \subset A$이므로 
$a \ge 1$ 이고 
$4a \le 16$ 
$\therefore 1 \le a \le 4$ 

자연수 $a$는 $1$, $2$, $3$, $4$이므로 개수는 $4$개이다.

정답: ①
