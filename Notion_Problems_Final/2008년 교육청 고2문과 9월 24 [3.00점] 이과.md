## 문제
집합 $A = \{ x \mid x = \left( \frac{1}{64} \right)^{\frac{1}{k}}, k \text{ 는 } 0 \text{ 이 아닌 정수} \}$ 의 원소 중 자연수인 것들의 합을 구하시오.

## 풀이
$\left( \frac{1}{64} \right)^{\frac{1}{k}} = (2^{-6})^{\frac{1}{k}} = 2^{-\frac{6}{k}}$ 이 자연수이려면 $-\frac{6}{k}$ 이 자연수여야 한다.
즉, $k$ 는 $-6$ 의 약수 중 음의 정수여야 하므로 $k$ 가 될 수 있는 값은 
$k = -1, -2, -3, -6$ 이다.
집합 $A$ 의 원소 중 자연수인 것은
$k = -1$ 일 때 $2^{-\frac{6}{-1}} = 2^{6} = 64$
$k = -2$ 일 때 $2^{-\frac{6}{-2}} = 2^{3} = 8$
$k = -3$ 일 때 $2^{-\frac{6}{-3}} = 2^{2} = 4$
$k = -6$ 일 때 $2^{-\frac{6}{-6}} = 2^{1} = 2$
따라서 구하는 합은
$64 + 8 + 4 + 2 = 78$ 이다.
