## 문제
최고차항의 계수가 $1$인 사차함수 $f(x)$가 모든 실수 $x$에 대하여 
$$f'(-x) = -f'(x)$$
를 만족시킨다. $f'(1) = 0$, $f(1) = 2$일 때, 다음 <보기>에서 옳은 것만을 있는 대로 고른 것은?

$$
\begin{array}{|l|}
\hline
\text{<보 기>} \\
\text{ㄱ. } f'(-1) = 0 \\
\text{ㄴ. 모든 실수 } k \text{에 대하여 } \int_{-k}^{0} f(x) dx = \int_{0}^{k} f(x) dx \\
\text{ㄷ. } 0 < t < 1 \text{인 모든 실수 } t \text{에 대하여 } \int_{-t}^{t} f(x) dx < 6t \\
\hline
\end{array}
$$



> [!example] 📸 (그림/그래프)

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
ㄱ. $f'(-x) = -f'(x)$이고 $f'(1) = 0$이므로 
$f'(-1) = -f'(1) = 0$ (참)

ㄴ. $f'(-1) = f'(1) = 0$이고, $f'(-x) = -f'(x)$에 $x=0$을 대입하면 $f'(0) = 0$이다.
$f'(x)$는 최고차항의 계수가 $4$인 삼차함수이므로
$f'(x) = 4x(x-1)(x+1) = 4x^3 - 4x$
이를 적분하면
$f(x) = x^4 - 2x^2 + C$
$f(1) = 2$이므로 $1 - 2 + C = 2$, 즉 $C = 3$이다.
$f(x) = x^4 - 2x^2 + 3$
$f(-x) = (-x)^4 - 2(-x)^2 + 3 = x^4 - 2x^2 + 3 = f(x)$이므로 $f(x)$는 $y$축 대칭인 우함수이다.
따라서 모든 실수 $k$에 대하여 $\int_{-k}^{0} f(x) dx = \int_{0}^{k} f(x) dx$이다. (참)

ㄷ. 함수 $f(x) = x^4 - 2x^2 + 3$의 그래프에서 $y$절편은 $(0, 3)$이다.
$0 < t < 1$인 범위에서 $f(x) = x^2(x^2 - 2) + 3$일 때, $x^2 - 2 < 0$이므로 $f(x) < 3$이 성립한다.
$\int_{-t}^{t} f(x) dx$는 곡선 $y = f(x)$와 $x$축, 그리고 $x = -t$, $x = t$로 둘러싸인 부분의 넓이이다.
반면 $y = 3$, $x$축, $x = -t$, $x = t$로 둘러싸인 직사각형의 넓이는 $3 \times 2t = 6t$이다.
해당 구간에서 $f(x) < 3$이므로
$\int_{-t}^{t} f(x) dx < \int_{-t}^{t} 3 dx = 6t$가 성립한다. (참)

따라서 옳은 것은 ㄱ, ㄴ, ㄷ이다.
