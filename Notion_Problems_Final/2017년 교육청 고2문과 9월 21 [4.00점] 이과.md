## 문제
곡선 $y = x^{2}$ 위의 점을 $P_{n}(x_{n}, x_{n}^{2})$이라 하자. 점 $P_{1}(0, 0)$이고, 직선 $P_{n}P_{n+1}$의 기울기를 $a_{n}$이라 할 때, 수열 $\{a_{n}\}$이 다음 조건을 만족시킨다. (단, $n$은 자연수이다.)

$$
\begin{array}{|l|}
\hline
\text{(가) } a_{1} = 3 \\
\text{(나) } d > 3 \text{인 상수 } d \text{에 대하여 } a_{n+1} = a_{n} + d \text{ } (n = 1, 2, 3, \dots) \text{ 이다.} \\
\hline
\end{array}
$$

<보기>에서 옳은 것만을 있는 대로 고른 것은?

## 보기
ㄱ. $x_{2} = 3$
ㄴ. $x_{20} = x_{19} + d$
ㄷ. $\sum_{k=1}^{10} (x_{2k+1} - x_{2k}) \le 100$을 만족시키는 $d$의 최댓값은 $13$이다.

① ㄱ   ② ㄷ   ③ ㄱ, ㄷ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
ㄱ. $P_{1}(0, 0)$, $P_{2}(x_{2}, x_{2}^{2})$이고 직선 $P_{1}P_{2}$의 기울기가 $a_{1}$이므로
$$ a_{1} = \frac{x_{2}^{2} - 0}{x_{2} - 0} = x_{2} = 3 $$
따라서 ㄱ은 옳다.

ㄴ. 직선 $P_{n}P_{n+1}$의 기울기 $a_{n}$은
$$ a_{n} = \frac{x_{n+1}^{2} - x_{n}^{2}}{x_{n+1} - x_{n}} = x_{n+1} + x_{n} $$
$\{a_{n}\}$은 첫째항이 $3$이고, 공차가 $d$ ($d > 3$)인 등차수열이므로
$$ a_{n} = x_{n+1} + x_{n} = 3 + (n - 1)d $$ ······ ㉠
㉠에 $n = 1, 2, 3, \dots$을 차례로 대입하면
$x_{2} = 3$
$x_{3} + x_{2} = 3 + d \implies x_{3} + 3 = 3 + d \implies x_{3} = d$
$x_{4} + x_{3} = 3 + 2d \implies x_{4} + d = 3 + 2d \implies x_{4} = 3 + d$
$x_{5} + x_{4} = 3 + 3d \implies x_{5} + 3 + d = 3 + 3d \implies x_{5} = 2d$
이와 같은 과정을 반복하면 일반항은 다음과 같다.
$x_{2n} = 3 + (n - 1)d$
$x_{2n-1} = (n - 1)d$
따라서 $x_{20} = 3 + 9d$, $x_{19} = 9d$이므로
$x_{20} = x_{19} + 3$이다.
그러므로 ㄴ은 옳지 않다.

ㄷ. $\sum_{k=1}^{10} (x_{2k+1} - x_{2k})$
$= (x_{3} - x_{2}) + (x_{5} - x_{4}) + \dots + (x_{21} - x_{20})$
위의 ㄴ에서 구한 규칙에 의해 $x_{2k+1} - x_{2k} = kd - \{3 + (k - 1)d\} = d - 3$이므로
$= \sum_{k=1}^{10} (d - 3) = 10(d - 3)$
$10(d - 3) \le 100$에서 $d - 3 \le 10$
즉, $d \le 13$이므로 $d$의 최댓값은 $13$이다.
따라서 ㄷ은 옳다.

따라서 옳은 것은 ㄱ, ㄷ이다.
