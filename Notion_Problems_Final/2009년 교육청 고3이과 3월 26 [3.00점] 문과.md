## 문제
$0 < a < b < 1$ 을 만족시키는 실수 $a$, $b$ 에 대하여 
$A = \log_{a} b$, $B = \log_{b} (a+1)$, $C = \log_{a+1} (b+1)$
이라 할 때, 다음 중 옳은 것은?

## 보기
① $A < B < C$   ② $A < C < B$   ③ $B < A < C$   ④ $B < C < A$   ⑤ $C < B < A$

## 풀이
$a < b < 1$ 에서 
$\log_{a} a > \log_{a} b > \log_{a} 1$ 
$\therefore 0 < \log_{a} b < 1$ … ㉠ 

$a+1 > 1$ 에서 
$\log_{b} (a+1) < \log_{b} 1 = 0$ … ㉡ 

$1 < a+1 < b+1$ 에서 
$\log_{a+1} (a+1) < \log_{a+1} (b+1)$ 
$\therefore \log_{a+1} (b+1) > 1$ … ㉢ 

㉠, ㉡, ㉢에서 
$\log_{b} (a+1) < \log_{a} b < \log_{a+1} (b+1)$ 
$\therefore B < A < C$

정답 ③
