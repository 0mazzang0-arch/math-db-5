## 문제
열린구간 $(0, 2\pi)$에서 정의된 함수 $f(x) = \frac{\sin x}{e^{2x}}$가 $x = a$에서 극솟값을 가질 때, $\cos a$의 값은?

## 보기
① $-\frac{2\sqrt{5}}{5}$   ② $-\frac{\sqrt{5}}{5}$   ③ $0$   ④ $\frac{\sqrt{5}}{5}$   ⑤ $\frac{2\sqrt{5}}{5}$

## 풀이
$f(x) = \frac{\sin x}{e^{2x}} = e^{-2x} \sin x$이므로
$f'(x) = -2e^{-2x} \sin x + e^{-2x} \cos x = e^{-2x}(-2 \sin x + \cos x)$이고
$f''(x) = -2e^{-2x}(-2 \sin x + \cos x) + e^{-2x}(-2 \cos x - \sin x) = e^{-2x}(3 \sin x - 4 \cos x)$이다.
이때 함수 $f(x)$는 $x = a$에서 극솟값을 가지므로 $f'(a) = 0, f''(a) > 0$이어야 한다. 이때 $e^{-2a} > 0$이므로
$-2 \sin a + \cos a = 0$ $\cdots$ ㉠,
$3 \sin a - 4 \cos a > 0$ $\cdots$ ㉡이 성립해야 한다. 
㉠에서 $\cos a = 2 \sin a$이므로 $\tan a = \frac{1}{2}$이다.
㉠을 ㉡에 대입하면 $-5 \sin a > 0$이다.
따라서 $\tan a > 0$이고, $\sin a < 0$이므로 $\pi < a < \frac{3}{2}\pi$이다.
$\sec^2 a = 1 + \tan^2 a = 1 + (\frac{1}{2})^2 = \frac{5}{4}$
$\cos a = \frac{1}{\sec a} = \pm \frac{2}{\sqrt{5}}$
$\therefore \cos a = -\frac{2\sqrt{5}}{5}$ ($\because \pi < a < \frac{3}{2}\pi$)

**[다른 풀이 1]**
미분가능한 함수 $f(x)$가 $x = a$에서 극값을 가지므로 $f'(a) = 0$이어야 한다.
$f(x) = \frac{\sin x}{e^{2x}} = e^{-2x} \sin x$이므로
$f'(x) = -2e^{-2x} \sin x + e^{-2x} \cos x = -e^{-2x}(2 \sin x - \cos x)$
이때 $-e^{-2a}(2 \sin a - \cos a) = 0$에서 $\tan a = \frac{1}{2}$이다.
ⅰ) $0 < a < \frac{\pi}{2}$일 때,
$0 < x < a$이면 $\sin x < \sin a$이고 $\cos x > \cos a$이므로
$2 \sin x - \cos x < 2 \sin a - \cos a = 0$ $\therefore f'(x) > 0$
$a < x < \frac{\pi}{2}$이면 $\sin x > \sin a$이고 $\cos x < \cos a$이므로
$2 \sin x - \cos x > 2 \sin a - \cos a = 0$ $\therefore f'(x) < 0$
따라서 함수 $f(x)$는 $x = a$에서 극댓값을 갖는다.

ⅱ) $\pi < a < \frac{3}{2}\pi$일 때,
$\pi < x < a$이면 $\sin x > \sin a$이고 $\cos x < \cos a$이므로
$2 \sin x - \cos x > 2 \sin a - \cos a = 0$ $\therefore f'(x) < 0$
$a < x < \frac{3\pi}{2}$이면 $\sin x < \sin a$이고 $\cos x > \cos a$이므로
$2 \sin x - \cos x < 2 \sin a - \cos a = 0$ $\therefore f'(x) > 0$
따라서 $x = a$에서 극솟값을 갖는다.
$\sec^2 a = 1 + \tan^2 a = \frac{5}{4}$에서 $\sec a = -\frac{\sqrt{5}}{2}$ ($\because \pi < a < \frac{3}{2}\pi$)
$\therefore \cos a = -\frac{2\sqrt{5}}{5}$

**[다른 풀이 2]**
$f(x) = \frac{\sin x}{e^{2x}}$는 미분가능한 함수이므로 몫의 미분법을 사용하면
$f'(x) = \frac{e^{2x} \cos x - 2e^{2x} \sin x}{(e^{2x})^2} = -\frac{2 \sin x - \cos x}{e^{2x}}$
이때 삼각함수의 합성에 의해서 $2 \sin x - \cos x = \sqrt{5} \sin(x - \alpha)$이므로
(단, $\cos \alpha = \frac{2}{\sqrt{5}}, \sin \alpha = \frac{1}{\sqrt{5}}$)
$f'(x) = -\frac{\sqrt{5} \sin(x - \alpha)}{e^{2x}}$
이때 $f'(x) = 0$에서 $\sin(x - \alpha) = 0$이므로
$x - \alpha = 0$ 또는 $x - \alpha = \pi$ $\therefore x = \alpha$ 또는 $x = \pi + \alpha$
이때 증가와 감소를 나타내는 표는 다음과 같다.

$$ \begin{array}{c|c|c|c|c|c|c|c} 
x & (0) & \cdots & \alpha & \cdots & \pi + \alpha & \cdots & (2\pi) \\ \hline 
f'(x) & & + & 0 & - & 0 & + & \\ \hline 
f(x) & & \nearrow & f(\alpha) & \searrow & f(\pi + \alpha) & \nearrow & 
\end{array} $$

그러므로 함수 $f(x)$는 $x = \pi + \alpha$에서 극솟값을 갖는다.
$\therefore a = \pi + \alpha$
$\therefore \cos a = \cos(\pi + \alpha) = -\cos \alpha = -\frac{2}{\sqrt{5}} = -\frac{2\sqrt{5}}{5}$

정답: ①
