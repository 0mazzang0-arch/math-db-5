## 문제
등차수열 $\{ a_{n} \}$ 에서 $a_{2} + a_{4} = 16$, $a_{8} + a_{12} = 58$ 일 때, $a_{17}$ 의 값을 구하시오.

## 보기

## 풀이
등차수열의 공차를 $d$ 라 하면
$( a_{8} + a_{12} ) - ( a_{2} + a_{4} ) = ( a_{8} - a_{2} ) + ( a_{12} - a_{4} ) = 6d + 8d = 14d$
$14d = 58 - 16$ 에서 $d = 3$
$a_{2} + a_{4} = ( a_{1} + 3 ) + ( a_{1} + 9 ) = 16$
$\therefore a_{1} = 2$
$\therefore a_{17} = a_{1} + 16d = 2 + 48 = 50$

정답: $50$
