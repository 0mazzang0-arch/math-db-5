## 문제
$0 < x < 2\pi$ 일 때, 방정식 $\cos^{2} x - \sin x = 1$ 의 모든 실근의 합은 $\frac{q}{p}\pi$ 이다. $p + q$ 의 값을 구하시오. (단, $p$, $q$ 는 서로소인 자연수이다.)

## 풀이
$\cos^{2} x - \sin x = 1$
$(1 - \sin^{2} x) - \sin x = 1$
$\sin^{2} x + \sin x = 0$
$\sin x (\sin x + 1) = 0$
$\sin x = 0$ 또는 $\sin x = -1$
$0 < x < 2\pi$ 이므로
$x = \pi$ 또는 $x = \frac{3}{2}\pi$
따라서 모든 실근의 합은
$\pi + \frac{3}{2}\pi = \frac{5}{2}\pi$
$\frac{q}{p}\pi = \frac{5}{2}\pi$ 이므로
$p = 2$, $q = 5$
$p + q = 2 + 5 = 7$

정답: $7$
