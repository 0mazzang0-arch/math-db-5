## 문제
집합 $A = \{ z \mid z = a + bi, a, b \text{는 정수} \}$이고, $yA = \{ \omega \mid \omega = yz, z \in A \}$일 때, $yA = A$를 만족하는 $y$의 값들의 총합은? (단, $y \in A$)

## 보기
① $0$   ② $1$   ③ $2$   ④ $i$   ⑤ $-i$

## 풀이
$yA = A$이면 $yA \subset A$이고 $A \subset yA$이다.

$y = p + qi$ ($p, q$는 정수)라 하면
(ⅰ) $x \in yA$일 때 $x \in A$여야 하므로,
$x = yz = (p + qi)(a + bi) = (pa - qb) + (pb + qa)i$
이때 $a, b, p, q$는 모두 정수이므로 $pa - qb$와 $pb + qa$도 정수이다. 따라서 임의의 $z \in A$에 대하여 $yz \in A$가 성립한다.

(ⅱ) $A \subset yA$이므로, 임의의 원소 $\omega \in A$에 대하여 $\omega = yx$를 만족하는 $x \in A$가 존재해야 한다.
즉, 모든 $\omega \in A$에 대하여 $x = \frac{\omega}{y} \in A$가 성립해야 한다.
$\omega = 1$이라 하면, $x = \frac{1}{y} \in A$여야 한다.
$y = p + qi$이므로
$\frac{1}{y} = \frac{1}{p + qi} = \frac{p - qi}{p^2 + q^2} = \frac{p}{p^2 + q^2} - \frac{q}{p^2 + q^2}i$
$\frac{1}{y} \in A$이려면 $\frac{p}{p^2 + q^2}$와 $\frac{q}{p^2 + q^2}$가 모두 정수여야 한다.
$\frac{p}{p^2 + q^2} = \alpha$, $\frac{q}{p^2 + q^2} = \beta$ ($\alpha, \beta$는 정수)라 하면
$p = \alpha(p^2 + q^2)$, $q = \beta(p^2 + q^2)$
각 식을 제곱하여 더하면
$p^2 + q^2 = (\alpha^2 + \beta^2)(p^2 + q^2)^2$
$\therefore (\alpha^2 + \beta^2)(p^2 + q^2) = 1$
이때 $\alpha^2 + \beta^2$과 $p^2 + q^2$는 모두 $0$ 이상의 정수이므로
$p^2 + q^2 = 1$
$p, q$는 정수이므로 가능한 순서쌍 $(p, q)$는
$(1, 0), (-1, 0), (0, 1), (0, -1)$
이에 따른 $y$의 값은
$1, -1, i, -i$
따라서 모든 $y$의 값의 총합은
$1 + (-1) + i + (-i) = 0$

정답: ①
