## 문제
곡선 $y=x^{2}$ 위의 임의의 점 $A(t, t^{2})$ ($0 < t < 1$)을 직선 $y=x$ 에 대하여 대칭이동한 점을 $B$ 라 하고 두 점 $A$, $B$ 에서 $y$ 축에 내린 수선의 발을 각각 $C$, $D$ 라 하자. 사각형 $ABDC$ 의 넓이가 $\frac{1}{8}$ 이 되는 상수 $t$ 의 값을 구하는 과정이다.

$$
\begin{array}{|l|}
\hline
\text{점 } A \text{ 에서 } y \text{ 축에 내린 수선의 발이 } C \text{ 이므로 } \overline{AC} = t \\
\text{점 } B \text{ 에서 } y \text{ 축에 내린 수선의 발이 } D \text{ 이므로 } \overline{BD} = t^{2} \\
\overline{DC} = (가) \text{ 이므로 사각형 } ABDC \text{ 의 넓이는} \\
\frac{1}{2} t^{2} \times (나) \\
\text{사각형 } ABDC \text{ 의 넓이가 } \frac{1}{8} \text{ 이므로} \\
\frac{1}{2} t^{2} \times (나) = \frac{1}{8} \\
\text{따라서 } t = (다) \\
\hline
\end{array}
$$

위의 (가), (나)에 알맞은 식을 각각 $f(t)$, $g(t)$ 라 하고, (다)에 알맞은 수를 $k$ 라 할 때, $f(k) \times g(k)$ 의 값은?



> [!example] 📸 (그림/그래프)

## 보기
① $\frac{\sqrt{2}-1}{4}$   ② $\frac{\sqrt{3}-1}{2}$   ③ $\frac{\sqrt{3}+1}{4}$   ④ $\frac{2\sqrt{2}-1}{2}$   ⑤ $\frac{2\sqrt{2}+1}{4}$

## 풀이
점 $B$ 는 점 $A(t, t^{2})$ 을 직선 $y=x$ 에 대하여 대칭이동한 점이므로 점 $B$ 의 좌표는 $(t^{2}, t)$ 이다.

점 $A$ 에서 $y$ 축에 내린 수선의 발이 $C$ 이므로 $\overline{AC} = t$ 이고, 점 $B$ 에서 $y$ 축에 내린 수선의 발이 $D$ 이므로 $\overline{BD} = t^{2}$ 이다.

$0 < t < 1$ 일 때 $t^{2} < t$ 이므로
$\overline{DC} = t - t^{2}$
사각형 $ABDC$ 는 한 쌍의 대변 $\overline{AC}$ 와 $\overline{BD}$ 가 $y$ 축에 수직으로 평행한 사다리꼴이므로 그 넓이는
$\frac{1}{2} \times (\overline{AC} + \overline{BD}) \times \overline{DC} = \frac{1}{2} (t + t^{2})(t - t^{2}) = \frac{1}{2} t(1 + t) \cdot t(1 - t) = \frac{1}{2} t^{2}(1 - t^{2})$
따라서 (가) $f(t) = t - t^{2}$, (나) $g(t) = 1 - t^{2}$ 이다.

사각형 $ABDC$ 의 넓이가 $\frac{1}{8}$ 이므로
$\frac{1}{2} t^{2}(1 - t^{2}) = \frac{1}{8}$
$t^{2}(1 - t^{2}) = \frac{1}{4}$
$4t^{4} - 4t^{2} + 1 = 0$
$(2t^{2} - 1)^{2} = 0$
$t^{2} = \frac{1}{2}$
$0 < t < 1$ 이므로 $t = \frac{\sqrt{2}}{2}$
따라서 (다) $k = \frac{\sqrt{2}}{2}$ 이다.

구하고자 하는 값 $f(k) \times g(k)$ 는
$f\left(\frac{\sqrt{2}}{2}\right) \times g\left(\frac{\sqrt{2}}{2}\right) = \left(\frac{\sqrt{2}}{2} - \left(\frac{\sqrt{2}}{2}\right)^{2}\right) \times \left(1 - \left(\frac{\sqrt{2}}{2}\right)^{2}\right)$
$= \left(\frac{\sqrt{2}}{2} - \frac{1}{2}\right) \times \left(1 - \frac{1}{2}\right)$
$= \frac{\sqrt{2}-1}{2} \times \frac{1}{2} = \frac{\sqrt{2}-1}{4}$

정답은 ① 이다.
