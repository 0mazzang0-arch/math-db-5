## 문제
좌표공간의 점 $A$($3$, $3$, $3$)과 중심이 원점 $O$인 구 $x^{2} + y^{2} + z^{2} = 9$ 위를 움직이는 점 $P$에 대하여 $\left| \frac{2}{3} \vec{OA} + \frac{1}{3} \vec{OP} \right|$의 최댓값은 $a + b\sqrt{3}$이다. $10$($a$ $+$ $b$)의 값을 구하시오. (단, $a$, $b$는 유리수이다.)

## 보기

## 풀이
점 $P$($x$, $y$, $z$)라 하면 $x^{2} + y^{2} + z^{2} = 9$이다.

$\frac{2}{3} \vec{OA} + \frac{1}{3} \vec{OP} = (2, 2, 2) + \left( \frac{x}{3}, \frac{y}{3}, \frac{z}{3} \right) = \left( \frac{x}{3} + 2, \frac{y}{3} + 2, \frac{z}{3} + 2 \right)$이므로

$\left| \frac{2}{3} \vec{OA} + \frac{1}{3} \vec{OP} \right|^{2} = \left( \frac{x}{3} + 2 \right)^{2} + \left( \frac{y}{3} + 2 \right)^{2} + \left( \frac{z}{3} + 2 \right)^{2}$

$= \frac{1}{9} \{ (x + 6)^{2} + (y + 6)^{2} + (z + 6)^{2} \}$ 이고,

$(x + 6)^{2} + (y + 6)^{2} + (z + 6)^{2}$은 점 $P$와 점 $B$($-6$, $-6$, $-6$)까지의 거리의 제곱과 같다.

점 $P$가 구 위의 점이므로 

$\overline{BP}^{2} \le (\overline{OP} + \overline{OB})^{2} = (3 + 6\sqrt{3})^{2}$

$\therefore \left| \frac{2}{3} \vec{OA} + \frac{1}{3} \vec{OP} \right|^{2} = \frac{1}{9} \overline{BP}^{2} \le \frac{1}{9} (3 + 6\sqrt{3})^{2} = (1 + 2\sqrt{3})^{2}$

따라서 $\left| \frac{2}{3} \vec{OA} + \frac{1}{3} \vec{OP} \right| \le 1 + 2\sqrt{3}$ 이므로

$a = 1$, $b = 2$이다.

$\therefore 10(a + b) = 10(1 + 2) = 30$

[정답] $30$
