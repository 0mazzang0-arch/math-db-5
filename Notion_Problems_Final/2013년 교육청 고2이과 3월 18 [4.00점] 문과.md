## 문제

다음은 $n$ 차 다항식 $P ( x )$ 가 $P ( k ) = \frac{k}{k+1}$ ($k=0, 1, 2, \cdots , n$)를 만족시킬 때, $P ( n+1 )$ 의 값을 구하는 과정이다.

$n$ 차 다항식 $P ( x )$ 는 $( k+1 ) P ( k ) -k=0$ ($k=0, 1, 2, \cdots , n$)을 만족시킨다. $Q ( x ) = ( x+1 ) P ( x ) -x$ 라 두면 $Q ( x )$ 는 $n+1$ 차 다항식이고 $Q ( 0 ) = Q ( 1 ) = Q ( 2 ) = \cdots = Q ( n ) = 0$ 이므로 $0$ 이 아닌 상수 $a$ 에 대하여 $Q ( x ) = a x ( x-1 ) ( x-2 ) \cdots ( x-n )$ 으로 나타낼 수 있다. 따라서 $( x+1 ) P ( x ) -x = a x ( x-1 ) ( x-2 ) \cdots ( x-n )$ $\cdots \cdots$ ㉠ 이다. ㉠의 양변에 $x=-1$ 을 대입하면 $a = \frac{( -1 )^{n+1}}{( 가 )}$ 이고, ㉠의 양변에 $x=n+1$ 을 대입하여 정리하면 $( n+2 ) P ( n+1 ) - ( n+1 ) = ( 나 )$ 이므로 $P ( n+1 ) = \frac{( 나 ) +n+1}{n+2}$ 이다. 따라서 $n$ 이 홀수이면 $P ( n+1 ) = 1$ 이고, $n$ 이 짝수이면 $P ( n+1 ) = ( 다 )$ 이다.

위의 과정에서 (가), (나), (다)에 알맞은 식을 각각 $f ( n )$, $g ( n )$, $h ( n )$ 이라 할 때, $\frac{f ( 4 ) \times g ( 5 )}{h ( 6 )}$ 의 값은?

## 보기

① $150$   ② $155$   ③ $160$   ④ $165$   ⑤ $170$

## 풀이

$n$ 차 다항식 $P ( x )$ 는 $( k+1 ) P ( k ) -k=0$ ($k=0, 1, 2, \cdots , n$)을 만족시킨다.
$Q ( x ) = ( x+1 ) P ( x ) -x$ 라 두면 $Q ( x )$ 는 $n+1$ 차 다항식이고 $Q ( 0 ) = Q ( 1 ) = Q ( 2 ) = \cdots = Q ( n ) = 0$ 이므로 $0$ 이 아닌 상수 $a$ 에 대하여 $Q ( x ) = a x ( x-1 ) ( x-2 ) \cdots ( x-n )$ 으로 나타낼 수 있다.

따라서 $( x+1 ) P ( x ) -x = a x ( x-1 ) ( x-2 ) \cdots ( x-n )$ $\cdots \cdots$ ㉠

㉠ 의 양변에 $x=-1$ 을 대입하면
$Q(-1) = (-1+1)P(-1) - (-1) = 1$
$a(-1)(-2)(-3) \cdots (-1-n) = a(-1)^{n+1}(n+1)!$
$1 = a(-1)^{n+1}(n+1)!$
$\therefore a = \frac{1}{(-1)^{n+1}(n+1)!} = \frac{(-1)^{n+1}}{(n+1)!}$
그러므로 (가)에 알맞은 식은 $f(n) = (n+1)!$ 이다.

㉠ 의 양변에 $x=n+1$ 을 대입하여 정리하면
$(n+1+1)P(n+1) - (n+1) = a(n+1)(n)(n-1) \cdots (1)$
$(n+2)P(n+1) - (n+1) = a(n+1)!$
$a = \frac{(-1)^{n+1}}{(n+1)!}$ 이므로
$(n+2)P(n+1) - (n+1) = \frac{(-1)^{n+1}}{(n+1)!} \times (n+1)! = (-1)^{n+1}$
그러므로 (나)에 알맞은 식은 $g(n) = (-1)^{n+1}$ 이다.

$P ( n + 1 ) = \frac{(-1)^{n+1} + n + 1}{n+2}$ 에서
$n$ 이 홀수이면 $P(n+1) = \frac{(-1)^{홀+1} + n + 1}{n+2} = \frac{1+n+1}{n+2} = 1$
$n$ 이 짝수이면 $P(n+1) = \frac{(-1)^{짝+1} + n + 1}{n+2} = \frac{-1+n+1}{n+2} = \frac{n}{n+2}$
그러므로 (다)에 알맞은 식은 $h(n) = \frac{n}{n+2}$ 이다.

구하는 값은 $\frac{f(4) \times g(5)}{h(6)}$ 이므로
$f(4) = (4+1)! = 5! = 120$
$g(5) = (-1)^{5+1} = (-1)^6 = 1$
$h(6) = \frac{6}{6+2} = \frac{6}{8} = \frac{3}{4}$

$\therefore \frac{120 \times 1}{\frac{3}{4}} = 120 \times \frac{4}{3} = 160$

정답은 ③ 이다.
