## 문제
등차수열 $\{ a_{n} \}$ 에 대하여 $a_{2} + a_{3} = 2 ( a_{1} + 12 )$ 일 때, 수열 $\{ a_{n} \}$ 의 공차는?

## 보기
① $2$   ② $4$   ③ $6$   ④ $8$   ⑤ $10$

## 풀이
등차수열 $\{ a_{n} \}$ 의 첫째항을 $a$, 공차를 $d$ 라 하면 
$a_{2} = a + d$, $a_{3} = a + 2d$
이를 주어진 등식에 대입하면 
$( a + d ) + ( a + 2d ) = 2 ( a + 12 )$
$2a + 3d = 2a + 24$
$3d = 24$
따라서 $d = 8$

정답: ④
