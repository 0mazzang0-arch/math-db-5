## 문제
다음은 $n \ge 2$인 모든 자연수 $n$에 대하여 부등식 
$$( 1 + \frac{1}{2} + \frac{1}{3} + \dots + \frac{1}{n} ) ( 1 + 2 + 3 + \dots + n ) > n^{2} \dots (*)$$
이 성립함을 수학적 귀납법을 이용하여 증명하는 과정이다.

주어진 식 $(*)$의 양변을 $\frac{n(n+1)}{2}$로 나누면 
$$1 + \frac{1}{2} + \frac{1}{3} + \dots + \frac{1}{n} > \frac{2n}{n+1} \dots ㉠$$
이다. $n \ge 2$인 자연수 $n$에 대하여
(ⅰ) $n=2$ 일 때,
(좌변) $= ( 가 )$, (우변) $= \frac{4}{3}$이므로 ㉠이 성립한다.
(ⅱ) $n=k$ ($k \ge 2$)일 때, ㉠이 성립한다고 가정하면
$$1 + \frac{1}{2} + \frac{1}{3} + \dots + \frac{1}{k} > \frac{2k}{k+1} \dots ㉡$$
이다. ㉡의 양변에 $\frac{1}{k+1}$을 더하면
$$1 + \frac{1}{2} + \frac{1}{3} + \dots + \frac{1}{k} + \frac{1}{k+1} > \frac{2k+1}{k+1}$$
$$\frac{2k+1}{k+1} - ( 나 ) = \frac{k}{(k+1)(k+2)} > 0$$
$$1 + \frac{1}{2} + \frac{1}{3} + \dots + \frac{1}{k} + \frac{1}{k+1} > ( 나 )$$
$n=k+1$ 일 때도 ㉠이 성립한다.
(ⅰ), (ⅱ)에 의하여 $n \ge 2$인 모든 자연수 $n$에 대하여 ㉠이 성립하므로 $(*)$도 성립한다.

위의 (가)에 알맞은 수를 $p$, (나)에 알맞은 식을 $f(k)$라 할 때, $8p \times f(10)$의 값은?

## 보기
① $14$   ② $16$   ③ $18$   ④ $20$   ⑤ $22$

## 풀이
[정답] ⑤ 

[해설] 
주어진 식 $(*)$의 양변을 $\frac{n(n+1)}{2}$로 나누면 
$$1 + \frac{1}{2} + \frac{1}{3} + \dots + \frac{1}{n} > \frac{2n}{n+1} \dots ㉠$$
이므로 $n \ge 2$인 자연수 $n$에 대하여
(ⅰ) $n=2$ 일 때,
(좌변) $= \frac{3}{2}$, (우변) $= \frac{4}{3}$이므로 ㉠이 성립한다.
(ⅱ) $n=k$ ($k \ge 2$)일 때, ㉠이 성립한다고 가정하면
$$1 + \frac{1}{2} + \frac{1}{3} + \dots + \frac{1}{k} > \frac{2k}{k+1} \dots ㉡$$
양변에 $\frac{1}{k+1}$을 더하면 
$$1 + \frac{1}{2} + \frac{1}{3} + \dots + \frac{1}{k} + \frac{1}{k+1} > \frac{2k+1}{k+1}$$
이때,
$$\frac{2k+1}{k+1} - \frac{2(k+1)}{k+2} = \frac{(2k+1)(k+2) - 2(k+1)^2}{(k+1)(k+2)} = \frac{2k^2+5k+2 - (2k^2+4k+2)}{(k+1)(k+2)} = \frac{k}{(k+1)(k+2)} > 0$$
이므로 
$$1 + \frac{1}{2} + \frac{1}{3} + \dots + \frac{1}{k} + \frac{1}{k+1} > \frac{2(k+1)}{k+2}$$
즉, $n=k+1$ 일 때도 ㉠이 성립한다.
(ⅰ), (ⅱ)에 의하여 $n \ge 2$인 모든 자연수 $n$에 대하여 ㉠이 성립하므로 $(*)$도 성립한다.
위 과정에서 $p = \frac{3}{2}$, $f(k) = \frac{2(k+1)}{k+2}$이므로
$$8p \times f(10) = 8 \times \frac{3}{2} \times \frac{2 \times 11}{10+2} = 8 \times \frac{3}{2} \times \frac{22}{12} = 12 \times \frac{22}{12} = 22$$
이다.
