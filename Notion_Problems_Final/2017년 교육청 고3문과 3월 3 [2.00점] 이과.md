## 문제
$\lim_{n \to \infty} \frac{3n^{2} - 2n + 1}{2n^{2} + 7}$ 의 값은?

## 보기
① $1$   ② $\frac{3}{2}$   ③ $2$   ④ $\frac{5}{2}$   ⑤ $3$

## 풀이
$$\lim_{n \to \infty} \frac{3n^{2} - 2n + 1}{2n^{2} + 7}$$
$$= \lim_{n \to \infty} \frac{3 - \frac{2}{n} + \frac{1}{n^{2}}}{2 + \frac{7}{n^{2}}}$$
$$= \frac{3 - 0 + 0}{2 + 0}$$
$$= \frac{3}{2}$$

정답: ②
