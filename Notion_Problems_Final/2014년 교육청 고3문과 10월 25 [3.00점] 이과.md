## 문제
양수 $x$에 대하여 $\log x$의 소수부분을 $f(x)$라 할 때, 집합 $A = \{ n \mid f(n) < \log 2, n \text{은 } 99 \text{ 이하의 자연수} \}$의 원소의 개수를 구하시오.

## 보기

## 풀이
$n$의 값을 구하면 다음과 같다.

(ⅰ) $1 \le n \le 9$
$\log n$의 정수부분은 $0$이므로 $f(n)$의 값은 $\log 1, \log 2, \cdots, \log 9$이다.
$f(n) < \log 2$인 $n$은 $1$이다.

(ⅱ) $10 \le n \le 99$
$\log n$의 정수부분은 $1$이므로 $f(n)$의 값은 $\log \frac{10}{10}, \log \frac{11}{10}, \cdots, \log \frac{99}{10}$이다.
$f(n) < \log 2$인 $n$은 $\frac{n}{10} < 2$를 만족해야 하므로 $n < 20$이다.
즉, $n$은 $10, 11, \cdots, 19$이다.

따라서 집합 $A$의 원소의 개수는 $1 + 10 = 11$이다.

정답: $11$
