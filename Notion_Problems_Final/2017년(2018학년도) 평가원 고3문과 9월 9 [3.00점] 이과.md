## 문제
실수 $x$에 대하여 두 조건 $p$, $q$가 다음과 같다. 

$$ p : (x + 2)(x - 4) \neq 0 $$
$$ q : -2 \le x \le 4 $$

다음 중 참인 명제는?

## 보기
① $p \to q$   ② $\sim p \to \sim q$   ③ $q \to \sim p$   ④ $q \to p$   ⑤ $\sim p \to q$

## 풀이
두 조건 $p$, $q$의 진리집합을 각각 $P$, $Q$라 하면 
$P = \{ x \mid x \neq -2, x \neq 4, x \text{는 실수} \}$ 
$Q = \{ x \mid -2 \le x \le 4 \}$ 

① $P \not\subset Q$이므로 명제 $p \to q$는 거짓이다. 
② 두 조건 $\sim p$, $\sim q$의 진리집합은 각각 $P^{c}$, $Q^{c}$이다. 이때, 
$P^{c} = \{ x \mid x = -2 \text{ 또는 } x = 4 \}$ 
$Q^{c} = \{ x \mid x < -2 \text{ 또는 } x > 4 \}$ 
$P^{c} \not\subset Q^{c}$이므로 $\sim p \to \sim q$는 거짓이다. 
③ $Q = \{ x \mid -2 \le x \le 4 \}$, $P^{c} = \{ x \mid x = -2 \text{ 또는 } x = 4 \}$이므로 $Q \not\subset P^{c}$이다. 
따라서 명제 $q \to \sim p$는 거짓이다. 
④ $Q = \{ x \mid -2 \le x \le 4 \}$, $P = \{ x \mid x \neq -2, x \neq 4, x \text{는 실수} \}$이므로 $Q \not\subset P$이다. 
(단, $x = -2, x = 4$일 때 $x \in Q$이지만 $x \notin P$이다.)
따라서 명제 $q \to p$는 거짓이다. 
⑤ $P^{c} = \{ x \mid x = -2 \text{ 또는 } x = 4 \}$, $Q = \{ x \mid -2 \le x \le 4 \}$이므로 $P^{c} \subset Q$이다. 
따라서 명제 $\sim p \to q$는 참이다. 

정답: ⑤
