## 문제
양수 $n$에 대하여 $\log n$의 정수부분과 소수부분을 각각 $f(n)$, $g(n)$이라 할 때, 다음 두 조건을 모두 만족시키는 자연수 $x$, $y$의 순서쌍 $(x, y)$의 개수를 구하시오.

(가) $f(2x) = f(y) = 0$
(나) $g(2x + y) < g(2x) + g(y)$

## 보기

## 풀이
(가)에서 $f(2x) = f(y) = 0$이므로
$0 \le \log 2x < 1$, $0 \le \log y < 1$이다.
이를 만족하는 자연수 $x$의 범위는 $1 \le x \le 4$이고, 자연수 $y$의 범위는 $1 \le y \le 9$이다.
따라서 조건을 만족하는 자연수 $x$, $y$의 전체 순서쌍 $(x, y)$의 개수는 $4 \times 9 = 36$이다.
또한 $3 \le 2x + y \le 17$이므로 조건 (나)에 따라 다음의 두 경우로 나눌 수 있다.

ⅰ) $3 \le 2x + y \le 9$일 때,
$f(2x + y) = 0$이므로 $g(2x + y) = \log(2x + y)$이다.
(나)에서 $\log(2x + y) < \log 2x + \log y$
$2x + y < 2xy$
$2xy - 2x - y > 0$
$(2x - 1)(y - 1) > 1$
이 식을 만족하지 않는 순서쌍 $(x, y)$는 $(1, 1)$, $(2, 1)$, $(3, 1)$, $(4, 1)$, $(1, 2)$의 $5$개이다. 이 $5$개의 경우를 제외하면 모두 성립한다.

ⅱ) $10 \le 2x + y \le 17$일 때,
$f(2x + y) = 1$이므로 $g(2x + y) = \log(2x + y) - 1$이다.
(나)에서 $\log(2x + y) - 1 < \log 2x + \log y$
$\log \frac{2x + y}{10} < \log 2xy$
$2x + y < 20xy$
$x \ge 1$, $y \ge 1$인 모든 자연수에 대하여 $20xy$가 $2x + y$보다 항상 크므로 이 식은 항상 성립한다.

그러므로 ⅰ)과 ⅱ)에 의해 조건을 만족하는 순서쌍의 개수는
$36 - 5 = 31$

정답: $31$
