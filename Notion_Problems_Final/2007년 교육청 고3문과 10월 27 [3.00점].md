## 문제
두 집합 $A = \{ x | 2^{x(x-3a)} < 2^{a(x-3a)} \}$, $B = \{ x | \log_{3}(x^{2} - 2x + 6) < 2 \}$ 에 대하여 $A \cap B = A$ 가 성립하도록 하는 실수 $a$ 의 값의 범위는?

## 보기
① $-1 \le a \le 0$   ② $-1 \le a \le \frac{1}{3}$   ③ $-\frac{1}{3} \le a \le 1$   ④ $\frac{1}{3} \le a \le 3$   ⑤ $1 \le a \le 3$

## 풀이
$A = \{ x | 2^{x(x-3a)} < 2^{a(x-3a)} \}$
$= \{ x | (x-a)(x-3a) < 0 \}$

$B = \{ x | \log_{3}(x^{2} - 2x + 6) < 2 \}$
$= \{ x | x^{2} - 2x + 6 < 3^{2} \}$
$= \{ x | x^{2} - 2x - 3 < 0 \}$
$= \{ x | -1 < x < 3 \}$

$A \cap B = A$ 즉, $A \subset B$ 가 성립해야 하므로

(ⅰ) $a > 0$ 일 때
$A = \{ x | a < x < 3a \} \subset \{ x | -1 < x < 3 \} = B$ 에서
$3a \le 3$ 이므로 $0 < a \le 1$

(ⅱ) $a = 0$ 일 때
$A = \{ x | x^{2} < 0 \} = \emptyset \subset B$ 이므로
$a = 0$

(ⅲ) $a < 0$ 일 때
$A = \{ x | 3a < x < a \} \subset \{ x | -1 < x < 3 \} = B$ 에서
$3a \ge -1$ 이므로 $-\frac{1}{3} \le a < 0$

(ⅰ), (ⅱ), (ⅲ)에서 $-\frac{1}{3} \le a \le 1$ 이다.

정답: ③
