## 문제
자연수 $n$ 에 대하여 
$$ A_{n} = \{ x \mid 2n-1 \leq \log_{2}x < 2n+1, x \text{ 는 자연수} \} $$
일 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?

## 보기
ㄱ. $A_{1} = \{ 2, 3, 4, 5, 6, 7 \}$
ㄴ. $A_{n} \cap A_{n+1} = \emptyset$
ㄷ. 집합 $A_{n}$ 의 원소의 개수는 $3 \cdot 2^{2n-1}$ 개이다.

① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
ㄱ. $n=1$ 일 때, $1 \leq \log_{2} x < 3$ 이므로 $2 \leq x < 8$ 이다. 
따라서 $A_{1} = \{ 2, 3, 4, 5, 6, 7 \}$ (참)

ㄴ. $A_{n} = \{ x \mid 2^{2n-1} \leq x < 2^{2n+1}, x \text{ 는 자연수} \}$ 이고,
$A_{n+1} = \{ x \mid 2^{2(n+1)-1} \leq x < 2^{2(n+1)+1}, x \text{ 는 자연수} \}$ 이므로
$A_{n+1} = \{ x \mid 2^{2n+1} \leq x < 2^{2n+3}, x \text{ 는 자연수} \}$ 이다.
두 집합의 범위를 비교하면 $A_{n} \cap A_{n+1} = \emptyset$ 임을 알 수 있다. (참)

ㄷ. $A_{n}$ 의 원소의 개수는 
$2^{2n+1} - 2^{2n-1} = 2^{2n-1}(2^{2} - 1) = 3 \cdot 2^{2n-1}$ 이다. (참)

따라서 옳은 것은 ㄱ, ㄴ, ㄷ이다.

정답: ⑤
