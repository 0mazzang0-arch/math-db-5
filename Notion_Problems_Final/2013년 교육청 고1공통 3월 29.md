## 문제
두 자연수 $a$, $b$ 에 대하여 $\frac{a}{b}$ 를 소수점 아래 첫째 자리에서 반올림하면 $5$ 이다. $2 a - 5 b = 23$ 일 때, $a b$ 의 값을 구하시오.

## 보기


## 풀이
$\frac{a}{b}$ 를 소수점 아래 첫째 자리에서 반올림하면 $5$ 이므로
$\frac{9}{2} \leq \frac{a}{b} < \frac{11}{2}$
$9 b \leq 2 a < 11 b$ 이다.
이때 $2 a = 23 + 5 b$ 이므로 이를 대입하면
$9 b \leq 23 + 5 b < 11 b$ 에서
$3.83 \cdots = \frac{23}{6} < b \leq \frac{23}{4} = 5.75$
따라서 자연수 $b$ 의 값은 $4$ 또는 $5$ 이다.

$1)$ $b = 4$ 이면 $2 a = 23 + 20 = 43$ 에서 $a = \frac{43}{2}$ 이므로 자연수 조건에 부적합하다.
$2)$ $b = 5$ 이면 $2 a = 23 + 25 = 48$ 에서 $a = 24$ 이다.

따라서 $a = 24$, $b = 5$ 이므로
$a b = 24 \times 5 = 120$

정답: $120$
