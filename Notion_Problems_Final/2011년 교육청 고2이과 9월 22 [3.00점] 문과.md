## 문제
수열 $\{ a_{n} \}$, $\{ b_{n} \}$ 에 대하여 $\sum_{n=1}^{\infty} ( 2a_{n} - 3 ) = 300$, $\sum_{n=1}^{\infty} ( 2b_{n} + 3 ) = 180$ 일 때, $\sum_{n=1}^{\infty} ( a_{n} + b_{n} )$ 의 값을 구하시오.

## 보기


## 풀이
$\sum_{n=1}^{\infty} ( 2a_{n} - 3 ) = 300$ …… ①
$\sum_{n=1}^{\infty} ( 2b_{n} + 3 ) = 180$ …… ② 
두 무한급수가 수렴하므로 ① $+$ ② 하면 
$\sum_{n=1}^{\infty} ( 2a_{n} - 3 ) + \sum_{n=1}^{\infty} ( 2b_{n} + 3 ) = \sum_{n=1}^{\infty} ( 2a_{n} + 2b_{n} ) = 480$ 
$2 \sum_{n=1}^{\infty} ( a_{n} + b_{n} ) = 480$ 
$\therefore \sum_{n=1}^{\infty} ( a_{n} + b_{n} ) = 240$

정답: $240$
