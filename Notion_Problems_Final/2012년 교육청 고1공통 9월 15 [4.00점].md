## 문제
계수가 실수인 다항식 $P(x)$에 대하여 방정식 $x^{3} + 2x - 1 = 0$의 서로 다른 세 근이 모두 방정식 $(x^{2} + x + 1)P(x) = 1$의 근이 되도록 하는 차수가 최소인 다항식 $P(x)$를 구하는 과정이다.

$x^{3} + 2x - 1 = 0$의 서로 다른 세 근이 모두 방정식 $(x^{2} + x + 1)P(x) = 1$의 근이므로 $(x^{2} + x + 1)P(x) - 1 = (x^{3} + 2x - 1)Q(x)$인 다항식 $Q(x)$가 존재한다. 
즉, $(x^{2} + x + 1)P(x) = (x^{3} + 2x - 1)Q(x) + 1$이다. 
그런데, $x^{3} + 2x - 1$을 $x^{2} + x + 1$로 나눈 몫과 나머지는 각각 $x - 1$, (가)이므로 
$(x^{2} + x + 1)P(x) = (x - 1)(x^{2} + x + 1)Q(x) +$ (가) $Q(x) + 1 \cdots \cdots$ ㉠ 
이다. 등식 ㉠을 만족하는 다항식 $P(x)$의 차수가 최소가 되기 위해서는 $Q(x)$가 다항식이므로 
(가) $Q(x) + 1 = x^{2} + x + 1$이어야 한다. 
따라서 $Q(x) =$ (나)이다. 
그러므로 구하고자 하는 다항식 $P(x) =$ (다)이다. 

위의 과정에서 (가), (나), (다)에 알맞은 식을 각각 $f(x)$, $g(x)$, $h(x)$라 할 때, $f(1) + g(3) + h(5)$의 값은?

## 보기
① $16$   ② $17$   ③ $18$   ④ $19$   ⑤ $20$

## 풀이
$x^{3} + 2x - 1$을 $x^{2} + x + 1$로 나누면
$$ \begin{array}{l} x^{3} + 2x - 1 = (x^{2} + x + 1)(x - 1) + 2x \end{array} $$
이므로 (가)에 알맞은 식은 $f(x) = 2x$이다.

등식 ㉠에서
$(x^{2} + x + 1)P(x) = (x - 1)(x^{2} + x + 1)Q(x) + 2xQ(x) + 1$
이 식을 만족하는 $P(x)$의 차수가 최소가 되려면 $2xQ(x) + 1$이 $x^{2} + x + 1$의 배수여야 한다.
$2xQ(x) + 1 = x^{2} + x + 1$
$2xQ(x) = x^{2} + x = x(x + 1)$
따라서 $Q(x) = \frac{1}{2}x + \frac{1}{2}$이므로 (나)에 알맞은 식은 $g(x) = \frac{1}{2}x + \frac{1}{2}$이다.

$(x^{2} + x + 1)P(x) = (x - 1)(x^{2} + x + 1)Q(x) + (x^{2} + x + 1)$
양변을 $x^{2} + x + 1$로 나누면
$P(x) = (x - 1)Q(x) + 1 = (x - 1)\left(\frac{1}{2}x + \frac{1}{2}\right) + 1 = \frac{1}{2}(x^{2} - 1) + 1 = \frac{1}{2}x^{2} + \frac{1}{2}$
따라서 (다)에 알맞은 식은 $h(x) = \frac{1}{2}x^{2} + \frac{1}{2}$이다.

각 식에 값을 대입하면
$f(1) = 2 \times 1 = 2$
$g(3) = \frac{1}{2} \times 3 + \frac{1}{2} = 2$
$h(5) = \frac{1}{2} \times 5^{2} + \frac{1}{2} = \frac{25}{2} + \frac{1}{2} = 13$

따라서 $f(1) + g(3) + h(5) = 2 + 2 + 13 = 17$이다.

정답: ②
