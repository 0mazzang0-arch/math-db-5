## 문제
실수 $a$, $b$, $c$가 $a b c > 0$를 만족할 때, 이차방정식 $a x^{2} + b x + c = 0$의 두 근이 서로 다른 부호를 갖기 위한 필요충분조건은?

## 보기
① $b^{2} - 4 a c > 0$ ② $b < 0$ ③ $b c > 0$ ④ $\frac{c}{a} - \frac{b}{a} > 0$ ⑤ $a + b + c > 0$

## 풀이
이차방정식 $a x^{2} + b x + c = 0$의 두 근이 서로 다른 부호의 실근을 가지려면 두 근의 곱이 음수이면 된다. 
$$\alpha \beta = \frac{c}{a} < 0 \quad \therefore a c < 0$$ 
이때 조건에서 $a b c > 0$ 이므로 $b < 0$ 이어야 한다.

정답: ②
