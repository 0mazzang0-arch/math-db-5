## 문제
함수 $f(x) = \sqrt{5} \sin x + 2 \cos x + a$의 최댓값이 $7$일 때, 상수 $a$의 값은?

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
$f(x) = \sqrt{5} \sin x + 2 \cos x + a$
$= \sqrt{(\sqrt{5})^{2} + 2^{2}} \sin(x + \alpha) + a$
$= 3 \sin(x + \alpha) + a$
(단, $\sin \alpha = \frac{2}{3}, \cos \alpha = \frac{\sqrt{5}}{3}$)
에서 $-1 \le \sin(x + \alpha) \le 1$이므로 함수 $f(x)$의 최댓값은 $3 + a$이다.
$3 + a = 7$에서 $a = 4$이다.

정답: ④
