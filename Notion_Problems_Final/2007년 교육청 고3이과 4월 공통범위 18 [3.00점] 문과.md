## 문제
수열 $\{ a_{n} \}$이 모든 자연수 $n$에 대하여 
$$ 3n^{2} - 5n - 1 < a_{n} < 3n^{2} + n + 2 $$
를 만족시킬 때, $\lim_{n \to \infty} \frac{a_{n}}{n^{2} + 2n + 2}$의 값을 구하시오.

## 풀이
모든 자연수 $n$에 대하여 $n^{2} + 2n + 2 > 0$이므로 각 변을 $n^{2} + 2n + 2$로 나누면
$$ \frac{3n^{2} - 5n - 1}{n^{2} + 2n + 2} < \frac{a_{n}}{n^{2} + 2n + 2} < \frac{3n^{2} + n + 2}{n^{2} + 2n + 2} $$
이때
$$ \lim_{n \to \infty} \frac{3n^{2} - 5n - 1}{n^{2} + 2n + 2} = 3, \lim_{n \to \infty} \frac{3n^{2} + n + 2}{n^{2} + 2n + 2} = 3 $$
이므로 수열의 극한의 대소 관계(샌드위치 정리)에 의하여
$$ \lim_{n \to \infty} \frac{a_{n}}{n^{2} + 2n + 2} = 3 $$
이다.

정답: $3$
