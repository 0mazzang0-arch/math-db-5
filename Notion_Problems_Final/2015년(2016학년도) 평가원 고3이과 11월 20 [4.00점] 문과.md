## 문제
양수 $x$ 에 대하여 $\log x$ 의 정수부분을 $f(x)$ 라 하자. 
$f(n+10)=f(n)+1$ 
을 만족시키는 $100$ 이하의 자연수 $n$ 의 개수는?

## 보기
① $11$   ② $13$   ③ $15$   ④ $17$   ⑤ $19$

## 풀이
$1 \le n \le 100$ 이므로 $0 \le f(n) \le 2$ 이다.

(ⅰ) $f(n)=0$ 즉 $1 \le n \le 9$ 일 때
$f(n+10)=1$ 이어야 하므로
$10 \le n+10 < 100$
$\therefore 1 \le n \le 9$

(ⅱ) $f(n)=1$ 즉 $10 \le n \le 99$ 일 때
$f(n+10)=2$ 이어야 하므로
$100 \le n+10 \le 1000$
$\therefore 90 \le n \le 99$

(ⅲ) $f(n)=2$ 즉 $n=100$ 일 때
$f(n+10)=f(110)=2$ 이므로
$f(n+10)=3$ 을 만족하지 않는다.

(ⅰ), (ⅱ), (ⅲ)에서 구하는 자연수 $n$ 의 개수는 $9+10=19$ 이다.

정답: ⑤
