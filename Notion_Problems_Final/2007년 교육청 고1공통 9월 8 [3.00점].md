## 문제
서로 다른 세 개의 실수로 이루어진 집합 $A = \{ a, b, c \}$에 대하여 $\{ x + y | x \in A, y \in A, x \neq y \} = \{ 11, 13, 16 \}$일 때, 집합 $A$의 원소 중 가장 큰 수는?

## 보기
① $8$   ② $9$   ③ $10$   ④ $11$   ⑤ $12$

## 풀이
집합 $A = \{ a, b, c \}$에 대하여 $a < b < c$라 하자. 이때, 주어진 집합을 원소나열법으로 나타내면 $\{ a + b, b + c, c + a \}$이다. 

$a + b < a + c < b + c$이므로 

$a + b = 11$ $\cdots$ ㉠ 
$a + c = 13$ $\cdots$ ㉡ 
$b + c = 16$ $\cdots$ ㉢ 

㉠ $+$ ㉡ $+$ ㉢을 하면 
$2(a + b + c) = 40$ 

$\therefore a + b + c = 20$ $\cdots$ ㉣ 

㉣ $-$ ㉠을 하면 
$c = 9$ 

따라서 집합 $A$의 원소 중 가장 큰 수는 $9$이다. 

정답: ②
