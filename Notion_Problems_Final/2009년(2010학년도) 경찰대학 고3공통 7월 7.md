## 문제
임의의 두 집합 $X$, $Y$에 대하여 $X \triangle Y = (X - Y) \cup (Y - X)$로 정의하자. 전체집합 $U = \{ x \mid x \text{는 } 10 \text{ 이하의 자연수} \}$의 두 부분집합 $A$, $B$가 $A = \{ x \mid x \text{는 } 6 \text{의 약수} \}$, $A \triangle B = \{ 2, 5, 8, 10 \}$를 만족시킬 때, 집합 $B$의 부분집합의 개수는?

## 보기
① $16$   ② $32$   ③ $64$   ④ $128$   ⑤ $256$

## 풀이
집합 $A \triangle B = (A - B) \cup (B - A)$를 벤다이어그램에 나타내면 다음 그림의 어두운 부분과 같다. 

> [!example] 📸 (그림/그래프)

$A = \{ x \mid x \text{는 } 6 \text{의 약수} \} = \{ 1, 2, 3, 6 \}$이고, 
$A \triangle B = \{ 2, 5, 8, 10 \}$이므로 
$A - B = \{ 2 \}$, 
$A \cap B = \{ 1, 3, 6 \}$, 
$B - A = \{ 5, 8, 10 \}$이다.

또, $U = \{ x \mid x \text{는 } 10 \text{ 이하의 자연수} \} = \{ 1, 2, \dots, 10 \}$이므로 이를 벤다이어그램에 나타내면 다음과 같다.

> [!example] 📸 (그림/그래프)

집합 $B = (A \cap B) \cup (B - A) = \{ 1, 3, 6, 5, 8, 10 \}$이 되어 $n(B) = 6$이다.
따라서 집합 $B$의 부분집합의 개수는 $2^{6} = 64$이다.
