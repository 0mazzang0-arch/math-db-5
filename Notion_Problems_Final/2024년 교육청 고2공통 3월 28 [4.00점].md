## 문제
$1$보다 큰 자연수 $k$에 대하여 전체집합 $U = \{ x \mid x \text{는 } k \text{ 이하의 자연수} \}$의 두 부분집합 $A = \{ x \mid x \text{는 } k \text{ 이하의 짝수} \}$, $B = \{ x \mid x \text{는 } k \text{의 약수} \}$가 $n(A) \times n((A \cup B)^{C}) = 15$를 만족시킨다. 집합 $(A \cup B)^{C}$의 모든 원소의 곱을 구하시오.

## 보기

## 풀이
$n(A) \times n((A \cup B)^{C}) = 15$에서 $n(A)$는 $15$의 양의 약수이다.

(ⅰ) $n(A) = 1$일 때
$A = \{ 2 \}$이므로 $k = 2$ 또는 $k = 3$이다.
$k = 2$이면 $U = \{ 1, 2 \}$, $B = \{ 1, 2 \}$에서 $(A \cup B)^{C} = \emptyset$, $n((A \cup B)^{C}) = 0$이므로 조건을 만족시키지 않는다.
$k = 3$이면 $U = \{ 1, 2, 3 \}$, $B = \{ 1, 3 \}$에서 $(A \cup B)^{C} = \emptyset$, $n((A \cup B)^{C}) = 0$이므로 조건을 만족시키지 않는다.

(ⅱ) $n(A) = 3$일 때
$A = \{ 2, 4, 6 \}$이므로 $k = 6$ 또는 $k = 7$이다.
$k = 6$이면 $U = \{ 1, 2, 3, \cdots, 6 \}$, $B = \{ 1, 2, 3, 6 \}$에서 $(A \cup B)^{C} = \{ 5 \}$, $n((A \cup B)^{C}) = 1$이므로 조건을 만족시키지 않는다.
$k = 7$이면 $U = \{ 1, 2, 3, \cdots, 7 \}$, $B = \{ 1, 7 \}$에서 $(A \cup B)^{C} = \{ 3, 5 \}$, $n((A \cup B)^{C}) = 2$이므로 조건을 만족시키지 않는다.

(ⅲ) $n(A) = 5$일 때
$A = \{ 2, 4, 6, 8, 10 \}$이므로 $k = 10$ 또는 $k = 11$이다.
$k = 10$이면 $U = \{ 1, 2, 3, \cdots, 10 \}$, $B = \{ 1, 2, 5, 10 \}$에서 $(A \cup B)^{C} = \{ 3, 7, 9 \}$, $n((A \cup B)^{C}) = 3$이므로 $n(A) \times n((A \cup B)^{C}) = 5 \times 3 = 15$가 되어 조건을 만족시킨다.
$k = 11$이면 $U = \{ 1, 2, 3, \cdots, 11 \}$, $B = \{ 1, 11 \}$에서 $(A \cup B)^{C} = \{ 3, 5, 7, 9 \}$, $n((A \cup B)^{C}) = 4$이므로 조건을 만족시키지 않는다.

(ⅳ) $n(A) = 15$일 때
$A = \{ 2, 4, 6, \cdots, 30 \}$이므로 $k = 30$ 또는 $k = 31$이다.
$k = 30$이면 $U = \{ 1, 2, 3, \cdots, 30 \}$, $B = \{ 1, 2, 3, 5, 6, 10, 15, 30 \}$에서 $(A \cup B)^{C} = \{ 7, 9, 11, 13, 17, 19, 21, 23, 25, 27, 29 \}$, $n((A \cup B)^{C}) = 11$이므로 조건을 만족시키지 않는다.
$k = 31$이면 $U = \{ 1, 2, 3, \cdots, 31 \}$, $B = \{ 1, 31 \}$에서 $(A \cup B)^{C} = \{ 3, 5, 7, \cdots, 29 \}$, $n((A \cup B)^{C}) = 14$이므로 조건을 만족시키지 않는다.

(ⅰ) ~ (ⅳ)에서 두 집합 $A$, $B$가 조건을 만족시키도록 하는 $k$는 $k = 10$이다.
이때 $(A \cup B)^{C} = \{ 3, 7, 9 \}$이므로 모든 원소의 곱은 $3 \times 7 \times 9 = 189$이다.

정답: $189$
