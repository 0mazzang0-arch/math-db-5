## 문제
직사각형 $A B C D$ 의 내부의 점 $P$ 가 
$$\vec{PA} + \vec{PB} + \vec{PC} + \vec{PD} = \vec{CA}$$
를 만족시킨다. <보기>에서 옳은 것만을 있는 대로 고른 것은?

$$
\begin{array}{|l|}
\hline
\text{<보기>} \\
ㄱ. \vec{PB} + \vec{PD} = 2 \vec{CP} \\
ㄴ. \vec{AP} = \frac{3}{4} \vec{AC} \\
ㄷ. \text{삼각형 } ADP \text{의 넓이가 } 3 \text{이면 직사각형 } ABCD \text{의 넓이는 } 8 \text{이다.} \\
\hline
\end{array}
$$



> [!example] 📸 (그림/그래프)

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
$\vec{CA} = \vec{PA} - \vec{PC}$ 이므로 
$\vec{PA} + \vec{PB} + \vec{PC} + \vec{PD} = \vec{PA} - \vec{PC}$ 에서 
$\vec{PB} + \vec{PD} = - 2\vec{PC}$ 

ㄱ. $\vec{PB} + \vec{PD} = -2 \vec{PC} = 2 \vec{CP}$ (참) 

ㄴ. $\frac{\vec{PB} + \vec{PD}}{2} = - \vec{PC}$ 
$BD$ 의 중점을 $E$ 라 하면 
$\vec{PE} = - \vec{PC}$ 

$P$ 는 선분 $EC$ 의 중점이므로 
$\vec{AP} = \frac{3}{4} \vec{AC}$ (참)

ㄷ. 삼각형 $ADC$ 의 넓이는 삼각형 $ADP$ 의 넓이의 $\frac{4}{3}$ 이므로 삼각형 $ADC$ 의 넓이는 
$3 \times \frac{4}{3} = 4$ 
$ABCD$ 의 넓이는 $4 \times 2 = 8$ 이다. (참)

이상에서 옳은 것은 $ㄱ$, $ㄴ$, $ㄷ$이다.
