## 문제
$1 \le x \le 3$ 에서 함수 $f(x) = 1 + \left( \frac{1}{3} \right)^{x-1}$ 의 최댓값은?

## 보기
① $\frac{5}{3}$   ② $2$   ③ $\frac{7}{3}$   ④ $\frac{8}{3}$   ⑤ $3$

## 풀이
[정답] ② 

함수 $f(x) = 1 + \left( \frac{1}{3} \right)^{x-1}$ 은 $1 \le x \le 3$ 에서 $x$ 의 값이 증가하면 $f(x)$ 의 값은 감소하므로 함수 $f(x)$ 는 $x = 1$ 에서 최댓값을 갖는다. 
따라서 $1 \le x \le 3$ 에서 함수 $f(x)$ 의 최댓값은 
$f(1) = 1 + \left( \frac{1}{3} \right)^{0} = 2$ 이다.
