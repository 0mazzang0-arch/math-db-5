## 문제
삼차함수 $f(x)$ 의 도함수 $y=f'(x)$ 의 그래프가 그림과 같을 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?

> [!example] 📸 (그림/그래프)

$$ 
\begin{array}{l} 
\text{ㄱ. } f(0) < 0 \text{ 이면 } |f(0)| < |f(2)| \text{ 이다.} \\
\text{ㄴ. } f(0)f(2) \ge 0 \text{ 이면 함수 } |f(x)| \text{ 가 } x=a \text{ 에서 극소인 } a \text{ 의 값의 개수는 } 2 \text{ 이다.} \\
\text{ㄷ. } f(0)+f(2)=0 \text{ 이면 방정식 } |f(x)| = f(0) \text{ 의 서로 다른 실근의 개수는 } 4 \text{ 이다.}
\end{array} 
$$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
ㄱ. $f(0) < 0$ 이면 함수 $f(x)$ 의 그래프의 개형은 다음과 같다.

> [!example] 📸 (그림/그래프)

이 때, $f(2) < f(0) < 0$ 이므로 $|f(0)| < |f(2)|$ 이다.

ㄴ. $f(0)f(2) \ge 0$ 일 때 $f(0) > f(2)$ 이므로 함수 $y=|f(x)|$ 의 그래프의 개형을 각 경우에 따라 그리면 다음과 같다.
(ⅰ) $f(0) > f(2) > 0$ 일 때,
함수 $y=f(x)$ 와 $y=|f(x)|$ 의 그래프의 개형은 다음과 같다.

> [!example] 📸 (그림/그래프)

> [!example] 📸 (그림/그래프)

그러므로 $|f(\alpha)|=0 (\alpha \neq 2)$ 라 하면 $a$ 의 값은 $\alpha$ 와 $2$ 로 $2$ 개이다.

(ⅱ) $f(0) > f(2)$ 이고, $f(2) = 0$ 일 때,
함수 $y=f(x)$ 와 $y=|f(x)|$ 의 그래프의 개형은 다음과 같다.

> [!example] 📸 (그림/그래프)

> [!example] 📸 (그림/그래프)

그러므로 $|f(\alpha)|=0 (\alpha \neq 2)$ 라 하면 $a$ 의 값은 $\alpha$ 와 $2$ 로 $2$ 개이다.

(ⅲ) $f(0) = 0$ 이고 $f(2) < 0$ 일 때,
함수 $y=f(x)$ 와 $y=|f(x)|$ 의 그래프의 개형은 다음과 같다.

> [!example] 📸 (그림/그래프)

> [!example] 📸 (그림/그래프)

그러므로 $|f(\alpha)|=0 (\alpha \neq 2)$ 라 하면 $a$ 의 값은 $0$ 과 $\alpha$ 로 $2$ 개이다.

(ⅳ) $f(2) < f(0) < 0$ 일 때,
함수 $y=f(x)$ 와 $y=|f(x)|$ 의 그래프의 개형은 다음과 같다.

> [!example] 📸 (그림/그래프)

> [!example] 📸 (그림/그래프)

그러므로 $|f(\alpha)|=0$ 라 하면 $a$ 의 값은 $0$ 과 $\alpha$ 로 $2$ 개이다.

따라서 (ⅰ), (ⅱ), (ⅲ), (ⅳ)에 의하여 극소인 $a$ 의 값의 개수는 $2$ 이다.

ㄷ. $f(0)+f(2)=0$ 이므로 $f(2) = -f(0)$ 이다.
함수 $y=f(x)$ 와 $y=|f(x)|$ 의 그래프의 개형은 다음과 같다.

> [!example] 📸 (그림/그래프)

> [!example] 📸 (그림/그래프)

이때, 방정식 $|f(x)| = f(0)$ 의 서로 다른 실근의 개수는 함수 $y=|f(x)|$ 의 그래프와 직선 $y=f(0)$ 과의 서로 다른 교점의 개수와 같다.
위의 그림에서 함수 $y=|f(x)|$ 의 그래프와 직선 $y=f(0)$ 의 서로 다른 교점의 개수는 $4$ 개이므로 서로 다른 실근의 개수도 $4$ 개이다.

이상에서 옳은 것은 $ㄱ, ㄴ, ㄷ$ 이다.
