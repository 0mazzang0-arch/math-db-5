## 문제
등차수열 $\{a_n\}$ 에 대하여 $a_2 = 16$, $a_5 = 10$ 일 때, $a_k = 0$ 을 만족시키는 $k$ 의 값을 구하시오.

## 보기

## 풀이
등차수열 $\{ a_n \}$ 의 공차를 $d$ 라 하면
$a_5 = a_2 + 3d = 16 + 3d = 10$
$\therefore d = -2$
따라서
$a_1 = a_2 - d = 16 - (-2) = 18$
이므로
$a_n = 18 + (n-1) \times (-2) = -2n + 20$
즉, $a_k = -2k + 20 = 0$ 이므로 $k = 10$

[정답] $10$
