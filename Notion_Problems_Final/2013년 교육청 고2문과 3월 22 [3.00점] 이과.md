## 문제
$P = \{ x \mid 10 \leq x \leq 26 \}$, $Q = \{ x \mid 1 \leq x \leq a \}$이다. 두 집합 $P$, $Q$가 각각 두 조건 $p$, $q$의 진리집합일 때, 명제 $p \rightarrow q$가 참이 되게 하는 실수 $a$의 최솟값을 구하시오.

## 보기

## 풀이
명제 $p \rightarrow q$가 참이므로 조건 $p$의 진리집합 $P$가 조건 $q$의 진리집합 $Q$의 부분집합이다. ($P \subseteq Q$)
즉, $\{ x \mid 10 \leq x \leq 26 \} \subseteq \{ x \mid 1 \leq x \leq a \}$ 가 성립해야 하므로 $26 \leq a$ 이다.
따라서 실수 $a$의 최솟값은 $26$이다.
