## 문제
함수 $f(x)$에 대하여 $f'(x) = 3x^{2} + 4x + 5$이고 $f(0) = 4$일 때, $f(1)$의 값을 구하시오.

## 풀이
주어진 식 $f'(x) = 3x^{2} + 4x + 5$을 부정적분하면
$f(x) = \int (3x^{2} + 4x + 5) dx$
$= x^{3} + 2x^{2} + 5x + C$
그런데, $f(0) = 4$이므로
$C = 4$

$\therefore f(x) = x^{3} + 2x^{2} + 5x + 4$

$x = 1$을 대입하면
$f(1) = 1 + 2 + 5 + 4 = 12$

정답: $12$
