## 문제
$0 < x < y < z < 1$ 일 때, 세 수 $A = \frac{\frac{x}{y}}{z}$, $B = \frac{x}{\frac{y}{z}}$, $C = \frac{z}{\frac{y}{x}}$ 의 대소 관계로 옳은 것은?

## 보기
① $A < B < C$   ② $B < C < A$   ③ $A < B = C$   ④ $B = C < A$   ⑤ $A = B = C$

## 풀이
$A = \frac{x}{y} \times \frac{1}{z} = \frac{x}{yz}$, 
$B = x \times \frac{z}{y} = \frac{xz}{y}$, 
$C = z \times \frac{x}{y} = \frac{xz}{y}$ 이므로 
$B = C \cdots \cdots$ ㉠ 

또, $0 < z < 1$ 에서 $\frac{1}{z} > 1$ 이므로 $z < \frac{1}{z}$ 
따라서 $\frac{x}{y} > 0$, $\frac{1}{z} - z > 0$ 이므로 
$A - B = \frac{x}{yz} - \frac{xz}{y} = \frac{x}{y} \left( \frac{1}{z} - z \right) > 0$ 
($\because \frac{x}{y} > 0, \frac{1}{z} - z > 0$) 
$\therefore A > B \cdots \cdots$ ㉡ 

㉠, ㉡에서 $B = C < A$

정답: ④
