## 문제
수열 $\{ a_{n} \}$이 $3 n - 1 < n a_{n} < 3 n + 2$를 만족시킬 때, $\lim_{n \to \infty} a_{n}$의 값은?

## 보기
① $\frac{1}{3}$   ② $\frac{1}{2}$   ③ $2$   ④ $3$   ⑤ $5$

## 풀이
$3 n - 1 < n a_{n} < 3 n + 2$의 각 변을 $n$으로 나누면 
$\frac{3 n - 1}{n} < a_{n} < \frac{3 n + 2}{n}$이다. 
이때, 
$\lim_{n \to \infty} \frac{3 n - 1}{n} = 3$, 
$\lim_{n \to \infty} \frac{3 n + 2}{n} = 3$이므로 
수열의 극한의 대소 관계에 의하여 
$\lim_{n \to \infty} a_{n} = 3$이다. 

정답은 ④이다.
