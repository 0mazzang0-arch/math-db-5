## 문제
이차정사각행렬 $A$, $B$, $C$ 에 대하여 <보기>에서 옳은 것만을 있는 대로 고른 것은?

ㄱ. $A^{2} B = E$ 이면 $A B A = E$ 이다.
ㄴ. $A + B + C = E$ 이면 $A B - B A = A C - C A$ 이다.
ㄷ. $A^{3} + A + E = O$ 일 때, $A$ 의 역행렬은 존재한다.

(단, $O$ 는 영행렬이고 $E$ 는 단위행렬이다.)

## 보기
① ㄴ   ② [정보 없음]   ③ [정보 없음]   ④ ㄱ, ㄷ   ⑤ [정보 없음]

## 풀이
ㄱ. $A^{2} B = A ( AB ) = E$ 이므로 $A^{-1} = AB$ 이다.
따라서 $ABA = A^{-1} A = E$ 이다. (참)

ㄴ. $A + B + C = E$ 에서 $B = E - A - C$ 이므로
$AB - BA = A ( E - A - C ) - ( E - A - C ) A$
$= A - A^{2} - AC - ( A - A^{2} - CA )$
$= A - A^{2} - AC - A + A^{2} + CA$
$= CA - AC$
즉, $AB - BA = CA - AC$ 이다. (거짓)

ㄷ. $A^{3} + A + E = O \iff -A ( A^{2} + E ) = E$
따라서 $A^{-1} = -A^{2} - E$ 가 존재한다. (참)
