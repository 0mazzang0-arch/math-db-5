## 문제
이차방정식 $x^{2} + x - 1 = 0$의 두 근을 $\alpha$, $\beta$라 할 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?

$$
\begin{array}{|l|}
\hline
\text{ㄱ. } \alpha + \beta = -1 \\
\text{ㄴ. } \frac{\beta}{\alpha} + \frac{\alpha}{\beta} = -3 \\
\text{ㄷ. } \alpha^{5} + \beta^{5} + \alpha^{4} + \beta^{4} = \alpha^{3} + \beta^{3} \\
\hline
\end{array}
$$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
$\alpha$, $\beta$는 이차방정식 $x^{2} + x - 1 = 0$의 두 근이므로 근과 계수의 관계에 의하여
$\alpha + \beta = -1$, $\alpha \beta = -1$이다.

ㄱ. $\alpha + \beta = -1$이므로 참이다.

ㄴ. $\frac{\beta}{\alpha} + \frac{\alpha}{\beta} = \frac{\alpha^{2} + \beta^{2}}{\alpha \beta} = \frac{(\alpha + \beta)^{2} - 2 \alpha \beta}{\alpha \beta} = \frac{(-1)^{2} - 2(-1)}{-1} = \frac{1 + 2}{-1} = -3$
따라서 참이다.

ㄷ. $\alpha^{5} + \beta^{5} + \alpha^{4} + \beta^{4} - \alpha^{3} - \beta^{3}$
$= \alpha^{3} (\alpha^{2} + \alpha - 1) + \beta^{3} (\beta^{2} + \beta - 1)$
이때 $\alpha$, $\beta$는 이차방정식 $x^{2} + x - 1 = 0$의 두 근이므로
$\alpha^{2} + \alpha - 1 = 0$, $\beta^{2} + \beta - 1 = 0$이다.
따라서 $\alpha^{5} + \beta^{5} + \alpha^{4} + \beta^{4} - \alpha^{3} - \beta^{3} = \alpha^{3} \cdot 0 + \beta^{3} \cdot 0 = 0$
즉, $\alpha^{5} + \beta^{5} + \alpha^{4} + \beta^{4} = \alpha^{3} + \beta^{3}$이므로 참이다.

따라서 옳은 것은 ㄱ, ㄴ, ㄷ이다.
