## 문제
$-2 \le x \le 2$ 에서 정의된 이차함수 $y = x^{2} + 6x + a$ 의 최솟값이 $-2$ 일 때, 최댓값을 구하시오. (단, $a$ 는 상수이다.)

## 보기

## 풀이
$y = x^{2} + 6x + a = (x + 3)^{2} + a - 9$ 이므로 
주어진 범위 $-2 \le x \le 2$ 는 축의 방정식 $x = -3$ 의 오른쪽에 위치한다.
따라서 함수 $y = x^{2} + 6x + a$ 는 $-2 \le x \le 2$ 에서 증가한다.

$x = -2$ 에서 최솟값 $a - 8$ 을 가지고,
$x = 2$ 에서 최댓값 $a + 16$ 을 가진다.

최솟값이 $-2$ 이므로
$a - 8 = -2$
$\therefore a = 6$

따라서 구하는 최댓값은
$a + 16 = 6 + 16 = 22$ 이다.

정답: $22$
