## 문제
부등식 $9^{x} - 3^{x+2} + 18 < 0$의 해가 $\alpha < x < \beta$일 때, $3^{\alpha} \times 3^{\beta}$의 값을 구하시오.

## 보기

## 풀이
$9^{x} - 3^{x+2} + 18 < 0$에서 
$3^{x} = t$라 하면 $t > 0$이고 주어진 부등식은 
$t^{2} - 9t + 18 < 0$
$(t - 3)(t - 6) < 0$
$3 < t < 6$
$\therefore 3 < 3^{x} < 6$
이 때 주어진 부등식의 해가 $\alpha < x < \beta$이므로 
$3^{\alpha} < 3^{x} < 3^{\beta}$
따라서 $3^{\alpha} = 3, 3^{\beta} = 6$
$\therefore 3^{\alpha} \cdot 3^{\beta} = 3 \times 6 = 18$

정답: $18$
