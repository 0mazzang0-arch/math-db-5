## 문제
두 지수함수 $f(x) = a^{bx-1}$, $g(x) = a^{1-bx}$이 다음 조건을 만족시킨다.

(가) 함수 $y=f(x)$의 그래프와 함수 $y=g(x)$의 그래프는 직선 $x=2$에 대하여 대칭이다.
(나) $f(4) + g(4) = \frac{5}{2}$

두 상수 $a$, $b$의 합 $a+b$의 값은? (단, $0 < a < 1$)

## 보기
① $1$   ② $\frac{9}{8}$   ③ $\frac{5}{4}$   ④ $\frac{11}{8}$   ⑤ $\frac{3}{2}$

## 풀이
$f(x) = a^{bx-1}$의 그래프와 $g(x) = a^{1-bx}$의 그래프는 직선 $x=2$에 대하여 대칭이므로 $f(2) = g(2)$가 성립한다.
따라서 $a^{2b-1} = a^{1-2b}$에서
$2b - 1 = 1 - 2b$
$4b = 2$
$\therefore b = \frac{1}{2}$

$f(4) = g(0)$, $g(4) = f(0)$이므로
$f(4) + g(4) = g(0) + f(0) = \frac{5}{2}$
$a^{4b-1} + a^{1-4b} = a^{4(\frac{1}{2})-1} + a^{1-4(\frac{1}{2})} = \frac{5}{2}$
$a^1 + a^{-1} = \frac{5}{2}$
$a + \frac{1}{a} = \frac{5}{2}$
양변에 $2a$를 곱하여 정리하면
$2a^2 - 5a + 2 = 0$
$(a-2)(2a-1) = 0$
$0 < a < 1$이므로 $a = \frac{1}{2}$이다.
$\therefore a+b = \frac{1}{2} + \frac{1}{2} = 1$

정답: ①
