## 문제
$3 \theta$ 는 제 $1$ 사분면의 각이고 $4 \theta$ 는 제 $2$ 사분면의 각일 때, $\theta$ 는 제 $m$ 사분면 또는 제 $n$ 사분면의 각이다. $m+n$ 의 값은? (단, $m \neq n$)

## 보기
① $3$   ② $4$   ③ $5$   ④ $6$   ⑤ $7$

## 풀이
[정답] ①

$2k \pi < 3 \theta < 2k \pi + \frac{\pi}{2}$ ($k$ 는 정수)에서
$\frac{2k}{3} \pi < \theta < \frac{2k}{3} \pi + \frac{\pi}{6}$ 이므로
$\frac{8k}{3} \pi < 4 \theta < \frac{8k}{3} \pi + \frac{2}{3} \pi$

이때 $4 \theta$ 가 제 $2$ 사분면의 각이 되려면
(ⅰ) $k = 0$ 일 때
$0 < 4 \theta < \frac{2}{3} \pi$ 에서 $\frac{1}{2} \pi < 4 \theta < \frac{2}{3} \pi$
$\frac{\pi}{8} < \theta < \frac{\pi}{6}$ 이므로 $\theta$ 는 제 $1$ 사분면의 각이다.

(ⅱ) $k = 1$ 일 때
$\frac{8 \pi}{3} < 4 \theta < \frac{10}{3} \pi$ 에서 $\frac{8}{3} \pi < 4 \theta < 3 \pi$
$\frac{2}{3} \pi < \theta < \frac{3}{4} \pi$ 이므로 $\theta$ 는 제 $2$ 사분면의 각이다.

(ⅲ) $k = 2$ 일 때
$\frac{16}{3} \pi < 4 \theta < 6 \pi$ 에서 제 $2$ 사분면에 속하는 $4 \theta$ 는 존재하지 않는다.

(ⅳ) $k = 3$ 일 때
$8 \pi < 4 \theta < \frac{26}{3} \pi$ 에서 $\frac{17}{2} \pi < 4 \theta < \frac{26}{3} \pi$
$\frac{17}{8} \pi < \theta < \frac{13}{6} \pi$ 이므로 $\theta$ 는 제 $1$ 사분면의 각이다.

(ⅴ) $k = 4$ 일 때
$\frac{32}{3} \pi < 4 \theta < \frac{34}{3} \pi$ 에서 $\frac{32}{3} \pi < 4 \theta < 11 \pi$
$\frac{32}{12} \pi < \theta < \frac{11}{4} \pi$ 이므로 $\theta$ 는 제 $2$ 사분면의 각이다.

(ⅵ) $k = 5$ 일 때
$\frac{40}{3} \pi < 4 \theta < 14 \pi$ 에서 제 $2$ 사분면에 속하는 $4 \theta$ 는 존재하지 않는다.

이상에서 $k$ 를 $3$ 으로 나눌 때 나머지가 $0$ 이면 $\theta$ 는 제 $1$ 사분면에 속하고 $3$ 으로 나눌 때 나머지가 $1$ 이면 $\theta$ 는 제 $2$ 사분면에 속하며 $3$ 으로 나눌 때 나머지가 $2$ 이면 조건을 만족시키는 $\theta$ 는 존재하지 않는다.
따라서 $m = 1$, $n = 2$ 라 하면
$m + n = 1 + 2 = 3$

[다른 풀이]
$3 \theta$ 가 제 $1$ 사분면의 각이므로
$2k \pi < 3 \theta < 2k \pi + \frac{\pi}{2}$ ($k$ 는 정수) $\cdots\cdots$ ㉠

$4 \theta$ 가 제 $2$ 사분면의 각이므로
$2l \pi + \frac{\pi}{2} < 4 \theta < 2l \pi + \pi$ ($l$ 은 정수) $\cdots\cdots$ ㉡

㉠, ㉡에서
$2(l - k) \pi < 4 \theta - 3 \theta < 2(l - k) \pi + \pi$
$\therefore 2(l - k) \pi < \theta < 2(l - k) \pi + \pi$

$l - k$ 는 정수이므로 $\theta$ 는 제 $1$ 사분면 또는 제 $2$ 사분면의 각이다.
따라서 $m, n$ 은 $1, 2$ 이므로
$m + n = 3$
