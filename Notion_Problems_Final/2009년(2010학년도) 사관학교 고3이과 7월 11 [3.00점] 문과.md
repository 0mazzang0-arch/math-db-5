## 문제
$10 < a < b < 50$ 인 두 자연수 $a$, $b$ 에 대하여 $\log_{2} a$ 의 소수부분과 $\log_{2} b$ 의 소수부분이 같을 때 순서쌍 $(a, b)$ 의 개수는?

## 보기
① $15$   ② $16$   ③ $17$   ④ $18$   ⑤ $19$

## 풀이
임의의 실수 $x$ 에 대하여 $x = n + \alpha$ ($n$ 은 정수, $0 \le \alpha < 1$) 일 때, $n$ 을 $x$ 의 정수부분, $\alpha$ 를 $x$ 의 소수부분이라 하자.

$\log_{2} a$ 의 소수부분과 $\log_{2} b$ 의 소수부분이 같으므로 $\log_{2} b - \log_{2} a$ 는 정수이어야 한다.
$10 < a < b < 50$ 이므로
$\log_{2} 10 < \log_{2} a < \log_{2} 50$
$\log_{2} 10 < \log_{2} b < \log_{2} 50$
$0 < \log_{2} b - \log_{2} a < \log_{2} 50 - \log_{2} 10 = \log_{2} 5 = 2. \dots$

따라서 $\log_{2} \frac{b}{a} = 1$ 또는 $\log_{2} \frac{b}{a} = 2$ 이다.

$1$) $\log_{2} \frac{b}{a} = 1$ 일 때, $b = 2a$
$10 < a < 2a < 50$ 이므로 $10 < a < 25$
따라서 $a = 11, 12, \dots, 24$ 까지 가능 (개수: $14$개)

$2$) $\log_{2} \frac{b}{a} = 2$ 일 때, $b = 4a$
$10 < a < 4a < 50$ 이므로 $10 < a < 12.5$
따라서 $a = 11, 12$ 만 가능 (개수: $2$개)

따라서 순서쌍 $(a, b)$ 의 개수는 $14 + 2 = 16$ 이다.

정답: ②
