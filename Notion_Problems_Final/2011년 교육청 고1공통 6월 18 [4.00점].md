## 문제
복소수 $\alpha$, $\beta$가 $\alpha^{2} = 2i$, $\beta^{2} = -2i$를 만족시킬 때, <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $i = \sqrt{-1}$)

$$ \begin{array}{|l|} \hline \text{ㄱ. } \alpha\beta = 2 \\ \text{ㄴ. } (\alpha + \beta)^{4} = 16 \\ \text{ㄷ. } \frac{\alpha - \beta}{\alpha + \beta} \text{ 는 실수이다.} \\ \hline \end{array} $$

## 보기
① ㄴ   ② ㄷ   ③ ㄱ, ㄴ   ④ ㄱ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
ㄱ. (거짓) 
$(\alpha\beta)^{2} = \alpha^{2}\beta^{2} = (2i)(-2i) = 4$이므로 
$\alpha\beta = 2$ 또는 $\alpha\beta = -2$이다. 

ㄴ. (참) 
$(\alpha\beta)^{2} = 4$에서 $\alpha\beta = 2$ 또는 $\alpha\beta = -2$이다. 
$\alpha^{2} + \beta^{2} = 0$이므로 
$(\alpha + \beta)^{2} = \alpha^{2} + \beta^{2} + 2\alpha\beta = 2\alpha\beta$ 
$(\alpha + \beta)^{4} = (2\alpha\beta)^{2} = 4\alpha^{2}\beta^{2} = 16$ 

ㄷ. (거짓) 
$(\alpha + \beta)^{2} = \alpha^{2} + 2\alpha\beta + \beta^{2}$이고 
$(\alpha - \beta)^{2} = \alpha^{2} - 2\alpha\beta + \beta^{2}$이므로 
$\left( \frac{\alpha + \beta}{\alpha - \beta} \right)^{2} = \frac{(\alpha + \beta)^{2}}{(\alpha - \beta)^{2}} = \frac{\alpha^{2} + \beta^{2} + 2\alpha\beta}{\alpha^{2} + \beta^{2} - 2\alpha\beta} = -1$ 
$\frac{\alpha + \beta}{\alpha - \beta}$는 실수가 아니다. 
이상에서 옳은 것은 ㄴ이다. 

**[다른 풀이]** 
$\alpha = a + bi$ ($a$, $b$는 실수)라 하고 $\alpha^{2} = 2i$에 대입하여 정리하면 
$\alpha^{2} = (a + bi)^{2} = (a^{2} - b^{2}) + 2abi = 2i$ 
그러므로 두 수의 실수부분을 비교하면 $a^{2} - b^{2} = 0$에서 $(a - b)(a + b) = 0$이 되어 $a + b = 0$ 또는 $a - b = 0$이다. 즉, $a = -b$ 또는 $a = b$ 
또 두 수의 허수부분을 비교하면 $ab = 1$이다. 
그러므로 $a = -b$에서 $a$ 대신 $-b$를 대입하면 $-b^{2} = 1$ 
그런데 $b$는 실수이므로 $b^{2} = -1$이 될 수 없다. 
이번에는 $a = b$에서 $a$ 대신 $b$를 대입하면 $b^{2} = 1$이다. 
즉, $b = 1$ 또는 $b = -1$ 
그러므로 $a = 1$, $b = 1$일 때 $\alpha = 1 + i$이고 
$a = -1$, $b = -1$일 때 $\alpha = -1 - i$이다. 

마찬가지로 $\beta = c + di$ ($c$, $d$는 실수)라 하고 $\beta^{2} = -2i$에 대입하여 $\beta$를 구하면 
$\beta = 1 - i$ 또는 $\beta = -1 + i$ 

ㄱ. (거짓) $\alpha = 1 + i$, $\beta = -1 + i$이면 $\alpha\beta = -2$이다. 
ㄴ. (참) $\alpha + \beta$의 값은 $2$, $-2$, $2i$, $-2i$가 될 수 있으므로 $(\alpha + \beta)^{4} = 16$이다.

정답: ①
