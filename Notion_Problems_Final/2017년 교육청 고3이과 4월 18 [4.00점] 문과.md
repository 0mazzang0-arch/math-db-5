## 문제

다음은 자연수 $n$에 대하여 부등식
$$\sum_{k=1}^{n} \left( \frac{k}{k+1} \cdot {}_{n}C_{k} \right) < 100$$
을 만족시키는 $n$의 최댓값을 구하는 과정이다.

이항정리를 이용하여 $(1+x)^{n}$을 전개하면
$$(1+x)^{n} = \sum_{k=0}^{n} ( (\text{가}) \cdot x^{k} )$$
$\cdots \cdots$ ㉠
위 식의 양변에 $x=1$을 대입하면
$$2^{n} = {}_{n}C_{0} + {}_{n}C_{1} + {}_{n}C_{2} + \cdots + {}_{n}C_{n}$$
$\cdots \cdots$ ㉡
㉠의 양변을 $0$에서 $1$까지 적분하여
$$\frac{2^{n+1}}{n+1} - \frac{1}{n+1} = {}_{n}C_{0} + \frac{1}{2} {}_{n}C_{1} + \frac{1}{3} {}_{n}C_{2} + \cdots + \frac{1}{n+1} {}_{n}C_{n}$$
$\cdots \cdots$ ㉢
을 얻는다. ㉡과 ㉢에서
$(\text{나}) + \frac{1}{n+1}$
$$= \frac{1}{2} {}_{n}C_{1} + \frac{2}{3} {}_{n}C_{2} + \frac{3}{4} {}_{n}C_{3} + \cdots + \frac{n}{n+1} {}_{n}C_{n}$$
$$= \sum_{k=1}^{n} \left( \frac{k}{k+1} \cdot {}_{n}C_{k} \right)$$
이므로 부등식 $\sum_{k=1}^{n} \left( \frac{k}{k+1} \cdot {}_{n}C_{k} \right) < 100$을 만족시키는 $n$의 최댓값은 $(\text{다})$이다.

위의 과정에서 $(\text{가})$에 알맞은 식에 대하여 $k=1$일 때의 식을 $f(n)$, $(\text{나})$에 알맞은 식을 $g(n)$, $(\text{다})$에 알맞은 수를 $p$라 할 때, $f(6) \cdot g(5) + p$의 값은?

## 보기

① $115$   ② $120$   ③ $125$   ④ $130$   ⑤ $135$

## 풀이

이항정리를 이용하여 $(1+x)^{n}$을 전개하면
$$(1+x)^{n} = \sum_{k=0}^{n} ( {}_{n}C_{k} \cdot x^{k} )$$
$\cdots \cdots$ ㉠
위 식의 양변에 $x=1$을 대입하면
$$2^{n} = {}_{n}C_{0} + {}_{n}C_{1} + {}_{n}C_{2} + \cdots + {}_{n}C_{n}$$
$\cdots \cdots$ ㉡
㉠의 양변을 $0$에서 $1$까지 적분하면
$$\int_{0}^{1} (1+x)^{n} dx = \left[ \frac{(1+x)^{n+1}}{n+1} \right]_{0}^{1} = \frac{2^{n+1}}{n+1} - \frac{1}{n+1}$$
$$\int_{0}^{1} ( {}_{n}C_{0} + {}_{n}C_{1} x + {}_{n}C_{2} x^{2} + \cdots + {}_{n}C_{n} x^{n} ) dx$$
$$= \left[ {}_{n}C_{0} x + \frac{1}{2} {}_{n}C_{1} x^{2} + \frac{1}{3} {}_{n}C_{2} x^{3} + \cdots + \frac{1}{n+1} {}_{n}C_{n} x^{n+1} \right]_{0}^{1}$$
$$= {}_{n}C_{0} + \frac{1}{2} {}_{n}C_{1} + \frac{1}{3} {}_{n}C_{2} + \cdots + \frac{1}{n+1} {}_{n}C_{n}$$
이므로
$$\frac{2^{n+1}}{n+1} - \frac{1}{n+1} = {}_{n}C_{0} + \frac{1}{2} {}_{n}C_{1} + \cdots + \frac{1}{n+1} {}_{n}C_{n}$$
$\cdots \cdots$ ㉢
㉡에서 ㉢을 빼면
$$\frac{n-1}{n+1} \cdot 2^{n} + \frac{1}{n+1} = \frac{1}{2} {}_{n}C_{1} + \frac{2}{3} {}_{n}C_{2} + \frac{3}{4} {}_{n}C_{3} + \cdots + \frac{n}{n+1} {}_{n}C_{n} = \sum_{k=1}^{n} \left( \frac{k}{k+1} \cdot {}_{n}C_{k} \right)$$
이므로
$$\sum_{k=1}^{n} \left( \frac{k}{k+1} \cdot {}_{n}C_{k} \right) = \frac{n-1}{n+1} \cdot 2^{n} + \frac{1}{n+1} < 100$$
$h(n) = \frac{n-1}{n+1} \cdot 2^{n} + \frac{1}{n+1}$이라 하면
$h(7) = \frac{769}{8} < 100$
$h(8) = \frac{1793}{9} > 100$이므로
$n$의 최댓값은 $7$이다.
따라서 $f(n) = {}_{n}C_{1}$, $g(n) = \frac{n-1}{n+1} \cdot 2^{n}$, $p=7$이고
$f(6) \cdot g(5) + p = 6 \cdot \frac{64}{3} + 7 = 135$
정답은 ⑤이다.
