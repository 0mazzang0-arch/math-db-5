## 문제
자연수 $m$, $n$에 대하여 $x$에 대한 이차방정식 $mx^{2} - 10x + n = 0$의 두 근이 서로 다른 소수일 때, 모든 $n$값의 합은?

## 보기
① $30$   ② $31$   ③ $32$   ④ $33$   ⑤ $34$

## 풀이
이차방정식 $mx^{2} - 10x + n = 0$의 소수인 두 근을 $p$, $q$라 하면 근과 계수의 관계에 의하여 
$p+q = \frac{10}{m}$, $pq = \frac{n}{m}$이다. 

$p+q = \frac{10}{m}$은 자연수이므로 가능한 $m$은 $1$, $2$, $5$, $10$이다. 
$pq = \frac{n}{m}$에서 $n = mpq$이므로

(ⅰ) $m = 1$일 때
$p$, $q$에 대하여 $p+q = 10$이므로 $p = 3$, $q = 7$ (또는 $p = 7$, $q = 3$)
$\therefore n = pq = 21$

(ⅱ) $m = 2$일 때
$p$, $q$에 대하여 $p+q = 5$이므로 $p = 2$, $q = 3$ (또는 $p = 3$, $q = 2$)
$\therefore n = 2pq = 12$

(ⅲ) $m = 5$일 때
$p+q = 2$를 만족시키는 서로 다른 소수 $p$, $q$는 존재하지 않는다.

(ⅳ) $m = 10$일 때
$p+q = 1$을 만족시키는 소수 $p$, $q$는 존재하지 않는다.

이상에서 가능한 자연수 $n$은 $21$, $12$이므로 구하는 합은 $21 + 12 = 33$이다.
