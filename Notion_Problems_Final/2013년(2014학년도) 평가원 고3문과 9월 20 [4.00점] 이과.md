## 문제
자연수 $n$ 에 대하여 실수 $a$ 가 $10^{n} < a < 10^{n+1}$ 을 만족시킨다. $\log a$ 의 소수 부분과 $\log \sqrt[n]{a}$ 의 소수 부분의 합이 정수이고 $(n+1) \log a = n^{2} + 8$ 일 때, $\frac{\log a}{n}$ 의 값은?

## 보기
① $\frac{57}{56}$   ② $\frac{22}{21}$   ③ $\frac{11}{10}$   ④ $\frac{6}{5}$   ⑤ $\frac{17}{12}$

## 풀이
$10^{n} < a < 10^{n+1}$ 에서 
$n < \log a < n + 1$ 

$\log a = n + \alpha$ ($0 < \alpha < 1$) 라고 놓으면 
$\log \sqrt[n]{a} = \frac{1}{n} \log a = \frac{1}{n}(n + \alpha) = 1 + \frac{\alpha}{n}$ 

$\log a$ 와 $\log \sqrt[n]{a}$ 의 소수 부분의 합이 정수이므로 
$\alpha + \frac{\alpha}{n} = 1$ 
$\therefore \alpha = \frac{n}{n + 1}$ 

또한 $(n + 1) \log a = n^{2} + 8$ 에서 
$(n + 1)(n + \alpha) = n^{2} + 8$ 
$(n + 1) \left( n + \frac{n}{n + 1} \right) = n^{2} + 8$ ($\because \alpha = \frac{n}{n + 1}$) 
$n(n + 1) + n = n^{2} + 8$
$n^{2} + 2n = n^{2} + 8$
$2n = 8$ 

$\therefore n = 4$, $\alpha = \frac{4}{5}$ 
$\therefore \frac{\log a}{n} = \frac{4 + \frac{4}{5}}{4} = \frac{6}{5}$

정답: ④
