## 문제
두 복소수 
$\alpha = \frac{1+i}{2i}$, $\beta = \frac{1-i}{2i}$
에 대하여 $(2\alpha^2 + 3)(2\beta^2 + 3)$ 의 값은? (단, $i = \sqrt{-1}$이다.)

## 보기
① $6$   ② $10$   ③ $14$   ④ $18$   ⑤ $22$

## 풀이
$\alpha = \frac{1+i}{2i}$ 에서 
$\alpha^2 = \frac{2i}{-4} = -\frac{i}{2}$ 이고, 
$\beta = \frac{1-i}{2i}$ 에서 
$\beta^2 = \frac{-2i}{-4} = \frac{i}{2}$ 이므로 
$2\alpha^2 = -i$, $2\beta^2 = i$ 이다. 

$\therefore (2\alpha^2 + 3)(2\beta^2 + 3) = (3-i)(3+i) = 10$

[다른 풀이] 
$\alpha + \beta = \frac{2}{2i} = \frac{1}{i} = -i$, 
$\alpha\beta = \frac{1-i^2}{-4} = \frac{2}{-4} = -\frac{1}{2}$ 이고 
$\alpha^2 + \beta^2 = (\alpha + \beta)^2 - 2\alpha\beta = (-i)^2 - 2 \times \left( -\frac{1}{2} \right) = -1 + 1 = 0$ 이므로 
$(2\alpha^2 + 3)(2\beta^2 + 3) = 4(\alpha\beta)^2 + 6(\alpha^2 + \beta^2) + 9$
$= 4 \times \frac{1}{4} + 6 \times 0 + 9 = 10$

정답: ②
