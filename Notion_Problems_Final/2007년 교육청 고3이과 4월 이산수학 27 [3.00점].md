## 문제
$\cos 2x + 3 \sin x = -1$ 의 모든 근의 합은? (단, $0 \le x < 2\pi$ 이다.)

## 보기
① $\frac{\pi}{2}$   ② $\pi$   ③ $2\pi$   ④ $3\pi$   ⑤ $4\pi$

## 풀이
$\cos 2x = 1 - 2 \sin^2 x$ 이므로 주어진 식에 대입하면
$1 - 2 \sin^2 x + 3 \sin x = -1$
$1 - 2 \sin^2 x + 3 \sin x + 1 = 0$
$2 \sin^2 x - 3 \sin x - 2 = 0$
$(2 \sin x + 1)(\sin x - 2) = 0$
$\sin x = -\frac{1}{2}$ ($\because -1 \le \sin x \le 1$ 이므로 $\sin x - 2 \neq 0$)
$0 \le x < 2\pi$ 이므로
$x = \frac{7}{6}\pi$ 또는 $x = \frac{11}{6}\pi$
따라서 모든 근의 합은
$\frac{7}{6}\pi + \frac{11}{6}\pi = \frac{18}{6}\pi = 3\pi$
