## 문제
$-1 \le x \le 2$ 에서 함수 $f(x) = \left( \frac{3}{a} \right)^x$ 의 최댓값이 $4$ 가 되도록 하는 모든 양수 $a$ 의 값의 곱은?

## 보기
① $16$   ② $18$   ③ $20$   ④ $22$   ⑤ $24$

## 풀이
[정답] ②

$f(x) = \left( \frac{3}{a} \right)^x$ 에서

(ⅰ) $\frac{3}{a} > 1$, 즉 $0 < a < 3$ 일 때,
$f(x)$ 는 증가함수이므로 $x = 2$ 에서 최댓값을 갖는다.
$f(2) = \left( \frac{3}{a} \right)^2 = 4$ 에서 $a^2 = \frac{9}{4}$
$a = \pm \frac{3}{2}$
$0 < a < 3$ 이므로 $a = \frac{3}{2}$

(ⅱ) $\frac{3}{a} = 1$, 즉 $a = 3$ 일 때,
$f(x) = 1$ 이므로 함수 $f(x)$ 의 최댓값이 $4$ 가 아니다.

(ⅲ) $0 < \frac{3}{a} < 1$, 즉 $a > 3$ 일 때,
$f(x)$ 는 감소함수이므로 $x = -1$ 에서 최댓값을 갖는다.
$f(-1) = \left( \frac{3}{a} \right)^{-1} = \frac{a}{3} = 4$ 에서
$a = 12$

(ⅰ), (ⅱ), (ⅲ)에서 모든 양수 $a$ 의 값의 곱은
$\frac{3}{2} \times 12 = 18$
