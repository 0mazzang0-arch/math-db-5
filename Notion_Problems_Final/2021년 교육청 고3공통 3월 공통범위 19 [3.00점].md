## 문제
수열 $\{ a_n \}$ 의 첫째항부터 제 $n$ 항까지의 합을 $S_n$ 이라 하자. 
$a_1 = 2$, $a_2 = 4$ 이고 $2$ 이상의 모든 자연수 $n$ 에 대하여 
$a_{n+1} S_n = a_n S_{n+1}$
이 성립할 때, $S_5$ 의 값을 구하시오.

## 풀이
$S_{n+1} = a_{n+1} + S_n$ 이므로 
$a_{n+1} S_n = a_n (a_{n+1} + S_n)$, 
$(S_n - a_n) a_{n+1} = a_n S_n$ 
즉, $S_{n-1} a_{n+1} = a_n S_n$ ($n \ge 2$) $\cdots$ ㉠ 

$a_1 = S_1 = 2$, $a_2 = 4$ 이므로 
$S_2 = a_1 + a_2 = 6$ 

㉠에서 $a_{n+1} = \frac{a_n S_n}{S_{n-1}}$ $\cdots$ ㉡ 

㉡에 $n = 2$, $3$, $4$ 를 차례로 대입하면 
$a_3 = \frac{a_2 S_2}{S_1} = \frac{4 \times 6}{2} = 12$ 에서 
$S_3 = S_2 + a_3 = 6 + 12 = 18$ 

$a_4 = \frac{a_3 S_3}{S_2} = \frac{12 \times 18}{6} = 36$ 에서 
$S_4 = S_3 + a_4 = 18 + 36 = 54$ 

$a_5 = \frac{a_4 S_4}{S_3} = \frac{36 \times 54}{18} = 108$ 

따라서 $S_5 = S_4 + a_5 = 54 + 108 = 162$

정답: $162$
