## 문제
확률변수 $X$ 와 $Y$ 는 각각 정규분포 $N(0, 2^{2})$ 과 $N(3, 1^{2})$ 을 따른다. 실수 $a$, $b$ 에 대하여 두 확률 $P(X < a)$ 와 $P(Y < b + 3)$ 이 같을 때, $a$ 와 $b$ 의 관계식은?

## 보기
① $a = b - 3$  ② $a = b + 3$  ③ $a = b$  ④ $a = 2b$  ⑤ $a = 4b$

## 풀이
확률변수 $X$ 의 평균이 $0$, 표준편차가 $2$ 이므로 확률변수 $Z = \frac{X - 0}{2}$ 은 표준정규분포 $N(0, 1)$ 을 따른다.
$\therefore P(X < a) = P\left( Z < \frac{a - 0}{2} \right)$ $\cdots$ ㉠ 

확률변수 $Y$ 의 평균이 $3$, 표준편차가 $1$ 이므로 확률변수 $Z = \frac{Y - 3}{1}$ 은 표준정규분포 $N(0, 1)$ 을 따른다.
$\therefore P(Y < b + 3) = P\left( Z < \frac{b + 3 - 3}{1} \right) = P(Z < b)$ $\cdots$ ㉡ 

이때, $P(X < a) = P(Y < b + 3)$ 이므로 
$P\left( Z < \frac{a}{2} \right) = P(Z < b)$ ($\because$ ㉠, ㉡) 

$\therefore \frac{a}{2} = b$
즉, $a = 2b$
