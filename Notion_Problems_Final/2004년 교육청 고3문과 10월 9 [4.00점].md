## 문제
연속확률변수 $X$ 가 취하는 값의 범위가 $0 \le X \le 2$ 이고, 확률밀도함수 $f(x)$ 가 $f(x) = kx$ ($0 \le x \le 2$) 일 때, 확률 $P(0 \le X \le k)$ 의 값은? (단, $k$ 는 상수이다.)

## 보기
① $\frac{1}{32}$   ② $\frac{1}{16}$   ③ $\frac{1}{8}$   ④ $\frac{1}{4}$   ⑤ $\frac{1}{2}$

## 풀이
$0 \le x \le 2$ 에서 $f(x) = kx$ 의 그래프와 $x$ 축 사이의 넓이가 $1$ 이어야 하므로
$2k = 1$ $\therefore k = \frac{1}{2}$
$\therefore P(0 \le X \le k) = P(0 \le X \le \frac{1}{2}) = \frac{1}{16}$

정답은 ② 입니다.
