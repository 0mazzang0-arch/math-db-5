## 문제
점 $A(1, 6)$을 한 꼭짓점으로 하는 삼각형 $ABC$의 두 변 $AB$, $AC$의 중점을 각각 $M(x_1, y_1)$, $N(x_2, y_2)$라 하자. $x_1 + x_2 = 2$, $y_1 + y_2 = 4$일 때, 삼각형 $ABC$의 무게중심의 좌표는?

## 보기
① $(\frac{1}{2}, \frac{2}{3})$ ② $(\frac{1}{2}, 1)$ ③ $(1, \frac{2}{3})$ ④ $(1, 2)$ ⑤ $(2, 1)$

## 풀이
꼭짓점 $B$, $C$의 좌표를 각각 $(a_1, b_1)$, $(a_2, b_2)$라 하면 두 점 $M$, $N$은 두 변 $AB$, $AC$의 중점이므로 
$1 + a_1 = 2x_1$, $6 + b_1 = 2y_1$
$1 + a_2 = 2x_2$, $6 + b_2 = 2y_2$
그런데 $x_1 + x_2 = 2$, $y_1 + y_2 = 4$이므로 
$(1 + a_1) + (1 + a_2) = 2(x_1 + x_2) = 2 \times 2 = 4$ 에서 $a_1 + a_2 = 2$
$(6 + b_1) + (6 + b_2) = 2(y_1 + y_2) = 2 \times 4 = 8$ 에서 $b_1 + b_2 = -4$
따라서 삼각형 $ABC$의 무게중심의 좌표는 
$(\frac{1 + a_1 + a_2}{3}, \frac{6 + b_1 + b_2}{3}) = (\frac{1 + 2}{3}, \frac{6 - 4}{3}) = (1, \frac{2}{3})$이다.
