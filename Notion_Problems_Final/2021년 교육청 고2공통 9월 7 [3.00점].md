## 문제
등차수열 $\{ a_{n} \}$ 에 대하여 $a_{3} + a_{6} = 25$, $a_{8} = 23$ 일 때, $a_{4}$ 의 값은?

## 보기
① $11$   ② $12$   ③ $13$   ④ $14$   ⑤ $15$

## 풀이
등차수열 $\{ a_{n} \}$ 의 첫째항을 $a$, 공차를 $d$ 라 하면 

$a_{3} + a_{6} = ( a + 2d ) + ( a + 5d ) = 2a + 7d = 25$
$a_{8} = a + 7d = 23$ 이므로 
위의 두 식을 연립하면 $a = 2$, $d = 3$ 이다.

따라서 
$a_{4} = a + 3d = 2 + 3 \times 3 = 11$

정답: ①
