## 문제
자연수 $n$ 에 대하여 $\log_{10} n$ 의 값을 소수점 아래 첫째 자리에서 반올림한 수를 $a_{n}$ 이라 할 때, $\sum_{n=1}^{20} a_{n}$ 의 값은?

## 보기
① $15$   ② $17$   ③ $19$   ④ $21$   ⑤ $23$

## 풀이
$n$ 이 자연수이므로 $\log_{10} n \ge 0$ 이고, $\log_{10} n$ 의 값을 소수점 아래 첫째 자리에서 반올림한 수의 범위를 생각하면

(ⅰ) $0 \le \log_{10} n < 0.5$ 일 때, $a_n = 0$
이때, $1 \le n < 10^{0.5}$ 이므로 $1 \le n < \sqrt{10}$
따라서, 이를 만족하는 $n$ 의 값은 $1, 2, 3$ 이다.

(ⅱ) $0.5 \le \log_{10} n < 1.5$ 일 때, $a_n = 1$
이때, $10^{0.5} \le n < 10^{1.5}$ 이므로 $\sqrt{10} \le n < 10\sqrt{10} \cdots$ ㉠
여기서 $20 < 10\sqrt{10}$ 이므로 ㉠을 만족하는 $n$ 의 값은 $4, 5, 6, \dots, 20$ 이다.

$\therefore \sum_{n=1}^{20} a_{n} = (a_{1} + a_{2} + a_{3}) + (a_{4} + \cdots + a_{20})$
$= 0 + 0 + 0 + \underbrace{1 + 1 + \cdots + 1}_{17\text{개}}$
$= 17$

[다른 풀이]
$\log_{10} 1 = 0, \log_{10} 2 = 0.3010, \log_{10} 3 = 0.4771 < 0.5$,
$\log_{10} 4 = 2\log_{10} 2 > 0.5, \dots, \log_{10} 10 = 1, \dots, \log_{10} 20 = \log_{10} 10 + \log_{10} 2 < 1.5$

따라서 $a_1 = a_2 = a_3 = 0, a_4 = a_5 = \cdots = a_{20} = 1$

$\therefore \sum_{n=1}^{20} a_{n} = (a_{1} + a_{2} + a_{3}) + (a_{4} + \cdots + a_{20})$
$= 0 + 0 + 0 + \underbrace{1 + 1 + \cdots + 1}_{17\text{개}}$
$= 17$
