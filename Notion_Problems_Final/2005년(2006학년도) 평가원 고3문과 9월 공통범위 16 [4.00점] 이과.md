## 문제
세 확률변수 $X$, $Y$, $W$ 는 각각 다음과 같다.

$$ \begin{array}{|l|} \hline \text{(가) } X \text{ 는 이항분포 } B \left( 100, \frac{1}{5} \right) \text{ 을 따른다.} \\ \text{(나) } Y \text{ 는 이항분포 } B \left( 225, \frac{1}{5} \right) \text{ 을 따른다.} \\ \text{(다) } W \text{ 는 이항분포 } B \left( 400, \frac{1}{5} \right) \text{ 을 따른다.} \\ \hline \end{array} $$

<보기>에서 옳은 것을 모두 고른 것은?

$$ \begin{array}{|l|} \hline \text{ㄱ. } P \left( \left| \frac{X}{100} - \frac{1}{5} \right| < \frac{1}{10} \right) < P \left( \left| \frac{W}{400} - \frac{1}{5} \right| < \frac{1}{10} \right) \\ \text{ㄴ. } P \left( \left| \frac{X}{100} - \frac{1}{5} \right| < \frac{1}{10} \right) < P \left( \left| \frac{Y}{225} - \frac{1}{5} \right| < \frac{1}{25} \right) \\ \text{ㄷ. } P \left( \left| \frac{Y}{225} - \frac{1}{5} \right| < \frac{1}{25} \right) < P \left( \left| \frac{W}{400} - \frac{1}{5} \right| < \frac{1}{25} \right) \\ \hline \end{array} $$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄷ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
$X$ 는 이항분포 $B \left( 100, \frac{1}{5} \right)$ 를 따르므로 
$E(X) = 100 \times \frac{1}{5} = 20$, 
$V(X) = 100 \times \frac{1}{5} \times \frac{4}{5} = 4^{2}$ 이므로 
$N (20, 4^{2} )$ 의 정규분포를 따른다. 
그러므로 
$P \left( \left| \frac{X}{100} - \frac{1}{5} \right| < \frac{1}{10} \right)$ 
$= P (|X - 20| < 10)$ 
$= P \left( \left| \frac{X - 20}{4} \right| < 2.5 \right)$ 
$= P (|Z| < 2.5)$ 이다. 

위와 같은 방법으로 
$P \left( \left| \frac{Y}{225} - \frac{1}{5} \right| < \frac{1}{25} \right) = P (|Z| < 1.5)$ 
$P \left( \left| \frac{W}{400} - \frac{1}{5} \right| < \frac{1}{25} \right) = P (|Z| < 2)$ 
$P \left( \left| \frac{W}{400} - \frac{1}{5} \right| < \frac{1}{10} \right) = P (|Z| < 5)$ 

따라서 각 보기를 확인하면 다음과 같다.
ㄱ. $P (|Z| < 2.5) < P (|Z| < 5)$ (참)
ㄴ. $P (|Z| < 2.5) < P (|Z| < 1.5)$ (거짓)
ㄷ. $P (|Z| < 1.5) < P (|Z| < 2)$ (참)

$\therefore$ 옳은 것은 $ㄱ, ㄷ$이다.
