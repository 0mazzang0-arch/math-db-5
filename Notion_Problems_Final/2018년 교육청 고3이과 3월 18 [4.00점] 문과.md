## 문제
다음은 부등식 $\sum_{k=1}^{n} \{ 2k \times ( _{n} C_{k} )^{2} \} \geq 10 \times _{2n} C_{n+1}$을 만족시키는 자연수 $n$의 최솟값을 구하는 과정이다.

$(1+x)^{2n}$의 전개식에서 $x^{n}$의 계수는 (가) 이다.
$(1+x)^{n} (1+x)^{n}$의 전개식에서 $x^{n}$의 계수는 
$\sum_{k=0}^{n} ( _{n} C_{k} \times _{n} C_{n-k} ) = \sum_{k=0}^{n} ( _{n} C_{k} )^{2}$
이다. 그러므로 
$\sum_{k=1}^{n} \{ 2k \times ( _{n} C_{k} )^{2} \}$
$= \sum_{k=1}^{n} \{ k \times ( _{n} C_{k} )^{2} \} + \sum_{k=1}^{n} \{ k \times ( _{n} C_{n-k} )^{2} \}$
$= \{ ( _{n} C_{1} )^{2} + 2 \times ( _{n} C_{2} )^{2} + \cdots + n \times ( _{n} C_{n} )^{2} \} + \{ ( _{n} C_{n-1} )^{2} + 2 \times ( _{n} C_{n-2} )^{2} + \cdots + n \times ( _{n} C_{0} )^{2} \}$
$= (나) \times \{ ( _{n} C_{0} )^{2} + ( _{n} C_{1} )^{2} + \cdots + ( _{n} C_{n} )^{2} \}$
$= (나) \times (가)$
이다. 
따라서 부등식 $\sum_{k=1}^{n} \{ 2k \times ( _{n} C_{k} )^{2} \} \geq 10 \times _{2n} C_{n+1}$을 만족시키는 자연수 $n$의 최솟값은 (다) 이다.

위의 (가), (나)에 알맞은 식을 각각 $f(n)$, $g(n)$이라 하고, (다)에 알맞은 수를 $p$라 할 때, $f(3)+g(3)+p$의 값은?

## 보기
① $32$   ② $34$   ③ $36$   ④ $38$   ⑤ $40$

## 풀이
[정답] ① 

$(1+x)^{2n}$의 전개식에서 $x^{n}$의 계수는 $_{2n} C_{n}$이다. 
$(1+x)^{n} (1+x)^{n}$의 전개식에서 $x^{n}$의 계수는 
$\sum_{k=0}^{n} ( _{n} C_{k} \times _{n} C_{n-k} ) = \sum_{k=0}^{n} ( _{n} C_{k} )^{2}$이므로 
$\sum_{k=0}^{n} ( _{n} C_{k} )^{2} = _{2n} C_{n}$이 성립한다. 

그러므로 
$\sum_{k=1}^{n} \{ 2k \times ( _{n} C_{k} )^{2} \}$
$= \sum_{k=1}^{n} \{ k \times ( _{n} C_{k} )^{2} \} + \sum_{k=1}^{n} \{ k \times ( _{n} C_{n-k} )^{2} \}$
$= \{ ( _{n} C_{1} )^{2} + 2 \times ( _{n} C_{2} )^{2} + \cdots + n \times ( _{n} C_{n} )^{2} \} + \{ ( _{n} C_{n-1} )^{2} + 2 \times ( _{n} C_{n-2} )^{2} + \cdots + n \times ( _{n} C_{0} )^{2} \}$
$= \{ ( _{n} C_{1} )^{2} + 2 \times ( _{n} C_{2} )^{2} + \cdots + n \times ( _{n} C_{n} )^{2} \} + \{ n \times ( _{n} C_{0} )^{2} + (n-1) \times ( _{n} C_{1} )^{2} + \cdots + ( _{n} C_{n-1} )^{2} \}$
$= n \times ( _{n} C_{0} )^{2} + n \times ( _{n} C_{1} )^{2} + \cdots + n \times ( _{n} C_{n} )^{2}$
$= n \times \{ ( _{n} C_{0} )^{2} + ( _{n} C_{1} )^{2} + \cdots + ( _{n} C_{n} )^{2} \}$
$= n \times _{2n} C_{n}$이다. 

한편, 부등식 $\sum_{k=1}^{n} \{ 2k \times ( _{n} C_{k} )^{2} \} \geq 10 \times _{2n} C_{n+1}$에 대입하면,
$n \times _{2n} C_{n} \geq 10 \times _{2n} C_{n+1}$
$n \times \frac{(2n)!}{n! \times n!} \geq 10 \times \frac{(2n)!}{(n+1)! \times (n-1)!}$
$n \times \frac{1}{n} \geq 10 \times \frac{1}{n+1}$
$1 \geq \frac{10}{n+1}$
$n+1 \geq 10$
$n \geq 9$
따라서 부등식을 만족시키는 자연수 $n$의 최솟값은 $9$이다. 

(가)에 알맞은 식 $f(n) = _{2n} C_{n}$, 
(나)에 알맞은 식 $g(n) = n$, 
(다)에 알맞은 수 $p = 9$이므로 
$f(3) + g(3) + p = _{6} C_{3} + 3 + 9 = 20 + 3 + 9 = 32$이다.
