## 문제
두 이차정사각행렬 $A$, $B$가
$$ A^{2} + B = 3E, A^{4} + B^{2} = 7E $$
를 만족시킬 때, <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $E$는 단위행렬이다.)

$$
\begin{array}{|l|}
\hline
\text{ㄱ. } AB = BA \\
\text{ㄴ. } B^{-1} = A^{2} \\
\text{ㄷ. } A^{6} + B^{3} = 18E \\
\hline
\end{array}
$$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄱ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
[정답] ⑤

ㄱ. $B = 3E - A^{2}$이므로
$AB = A(3E - A^{2}) = 3A - A^{3}$
$BA = (3E - A^{2})A = 3A - A^{3}$
$\therefore AB = BA$ (참)

ㄴ. $A^{2} = 3E - B$이므로
$A^{4} = (3E - B)^{2} = B^{2} - 6B + 9E$
$A^{4} + B^{2} = 7E$에서
$(B^{2} - 6B + 9E) + B^{2} = 7E$
$2B^{2} - 6B + 9E = 7E$
$2B^{2} - 6B = -2E$
$B^{2} - 3B = -E$
$B(B - 3E) = -BA^{2} = -E$
$\therefore B^{-1} = A^{2}$ (참)

ㄷ. ㄴ에서 $A^{2}B = BA^{2} = E$이므로
$(A^{2} + B)(A^{4} + B^{2}) = A^{6} + A^{2}B^{2} + BA^{4} + B^{3}$
$= A^{6} + (A^{2}B)B + (BA^{2})A^{2} + B^{3}$
$= A^{6} + EB + EA^{2} + B^{3}$
$= A^{6} + B^{3} + (A^{2} + B)$
$3E \times 7E = A^{6} + B^{3} + 3E$
$21E = A^{6} + B^{3} + 3E$
$\therefore A^{6} + B^{3} = 18E$ (참)

따라서 옳은 것은 $ㄱ, ㄴ, ㄷ$이다.
