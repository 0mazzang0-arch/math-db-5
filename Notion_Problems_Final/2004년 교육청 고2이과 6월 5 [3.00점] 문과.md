## 문제
세 수 $A = \sqrt[4]{2}$, $B = \sqrt[12]{9}$, $C = \sqrt[6]{\sqrt{7}}$의 대소 관계를 바르게 나타낸 것은?

## 보기
① $A < B < C$   ② $A < C < B$   ③ $B < A < C$   ④ $C < A < B$   ⑤ $C < B < A$

## 풀이
$A = 2^{\frac{1}{4}}$, $B = 9^{\frac{1}{12}}$, $C = \sqrt[12]{7} = 7^{\frac{1}{12}}$이므로 각 수를 $12$제곱하면
$A^{12} = (2^{\frac{1}{4}})^{12} = 2^{3} = 8$
$B^{12} = (9^{\frac{1}{12}})^{12} = 9$
$C^{12} = (7^{\frac{1}{12}})^{12} = 7$
따라서 $7 < 8 < 9$이므로 $C^{12} < A^{12} < B^{12}$이다.
$A > 0$, $B > 0$, $C > 0$이므로 대소 관계는 $C < A < B$이다.

정답: ④
