## 문제
수열 $\{ a_{n} \}$ 에서 
$$ a_{n} = 1 + \frac{1}{2} + \frac{1}{3} + \dots + \frac{1}{n} \quad ( n = 1, 2, 3, \dots ) $$
일 때, $30 a_{30} - ( a_{1} + a_{2} + a_{3} + \dots + a_{29} )$ 의 값을 구하시오.

## 풀이
$$ 30 a_{30} - ( a_{1} + a_{2} + a_{3} + \dots + a_{29} ) $$
$$ = a_{30} + ( a_{30} - a_{1} ) + ( a_{30} - a_{2} ) + \dots + ( a_{30} - a_{29} ) $$
$$ = 1 + 2 \times \frac{1}{2} + 3 \times \frac{1}{3} + \dots + 30 \times \frac{1}{30} = 30 $$
따라서 구하는 값은 $30$ 이다.
