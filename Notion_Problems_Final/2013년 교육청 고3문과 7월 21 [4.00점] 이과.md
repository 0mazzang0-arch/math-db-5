## 문제
최고차항의 계수가 $1$인 삼차함수 $f(x)$가 $f(0)=0$, $f(\alpha)=0$, $f'(\alpha)=0$이고 함수 $g(x)$가 다음 두 조건을 만족시킬 때, $g(\frac{\alpha}{3})$의 값은? (단, $\alpha$는 양수이다.)

**(가)** $g'(x) = f(x) + xf'(x)$
**(나)** $g(x)$의 극댓값이 $81$이고 극솟값이 $0$이다.



> [!example] 📸 (그림/그래프)

## 보기
① $56$   ② $58$   ③ $60$   ④ $62$   ⑤ $64$

## 풀이
삼차함수 $f(x)$의 최고차항의 계수가 $1$이고 $f(0)=0, f(\alpha)=0, f'(\alpha)=0$이므로
$f(x) = x(x-\alpha)^2$
이다.

조건 (가)에서 $g'(x) = f(x) + xf'(x) = (xf(x))'$이므로
$g(x) = xf(x) + C = x^2(x-\alpha)^2 + C$
이다.

$g(x)$의 도함수 $g'(x)$를 구하면
$g'(x) = 2x(x-\alpha)^2 + 2x^2(x-\alpha) = 2x(x-\alpha)(x-\alpha+x) = 2x(x-\alpha)(2x-\alpha)$
이므로, $y=g(x)$는 $x=0$, $x=\alpha$에서 극솟값을 갖고, $x=\frac{\alpha}{2}$에서 극댓값을 갖는다.

조건 (나)에서 극솟값이 $0$이므로
$g(0) = C = 0$
$g(\alpha) = C = 0$
따라서 $C=0$이고 $g(x) = x^2(x-\alpha)^2$이다.

또한 극댓값이 $81$이므로
$g(\frac{\alpha}{2}) = (\frac{\alpha}{2})^2(\frac{\alpha}{2}-\alpha)^2 = \frac{\alpha^2}{4} \cdot \frac{\alpha^2}{4} = \frac{\alpha^4}{16} = 81$
$\alpha^4 = 81 \times 16 = 3^4 \times 2^4 = 6^4$
$\alpha > 0$이므로 $\alpha = 6$이다.

따라서 $g(x) = x^2(x-6)^2$이며, 구하고자 하는 값은
$g(\frac{\alpha}{3}) = g(\frac{6}{3}) = g(2)$
$g(2) = 2^2(2-6)^2 = 4 \times (-4)^2 = 4 \times 16 = 64$
이다.
