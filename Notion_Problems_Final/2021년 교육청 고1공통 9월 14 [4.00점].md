## 문제
이차방정식 $x ^{2} + 2x + 3 = 0$의 서로 다른 두 근을 $\alpha$, $\beta$라 할 때, $\frac{1}{\alpha ^{2} + 3\alpha + 3} + \frac{1}{\beta ^{2} + 3\beta + 3}$의 값은?

## 보기
① $-\frac{1}{3}$   ② $-\frac{1}{2}$   ③ $-\frac{2}{3}$   ④ $-\frac{5}{6}$   ⑤ $-1$

## 풀이
이차방정식 $x ^{2} + 2x + 3 = 0$의 서로 다른 두 근이 $\alpha$, $\beta$이므로 이차방정식의 근과 계수의 관계에 의해 
$\alpha + \beta = -2$, $\alpha\beta = 3$
또한 $\alpha$, $\beta$가 이차방정식의 근이므로 
$\alpha ^{2} + 2\alpha + 3 = 0$에서 $\alpha ^{2} + 3\alpha + 3 = \alpha$
$\beta ^{2} + 2\beta + 3 = 0$에서 $\beta ^{2} + 3\beta + 3 = \beta$
따라서 
$\frac{1}{\alpha ^{2} + 3\alpha + 3} + \frac{1}{\beta ^{2} + 3\beta + 3} = \frac{1}{\alpha} + \frac{1}{\beta} = \frac{\alpha + \beta}{\alpha\beta} = -\frac{2}{3}$

정답: ③
