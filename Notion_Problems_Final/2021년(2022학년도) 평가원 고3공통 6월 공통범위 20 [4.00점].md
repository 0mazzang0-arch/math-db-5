## 문제
실수 $a$ 와 함수 $f(x) = x^3 - 12x^2 + 45x + 3$ 에 대하여 함수 
$$g(x) = \int_{a}^{x} \{ f(x) - f(t) \} \{ f(t) \}^4 dt$$
가 오직 하나의 극값을 갖도록 하는 모든 $a$ 의 값의 합을 구하시오.

## 보기

## 풀이
$$g(x) = \int_{a}^{x} \{ f(x) - f(t) \} \times \{ f(t) \}^4 dt$$
$$= f(x) \int_{a}^{x} \{ f(t) \}^4 dt - \int_{a}^{x} \{ f(t) \}^5 dt$$
$g'(x)$ 를 구하면 다음과 같다.
$$g'(x) = f'(x) \int_{a}^{x} \{ f(t) \}^4 dt + f(x) \times \{ f(x) \}^4 - \{ f(x) \}^5$$
$$= f'(x) \int_{a}^{x} \{ f(t) \}^4 dt$$
$$= (3x^2 - 24x + 45) \int_{a}^{x} \{ f(t) \}^4 dt$$
$$= 3(x - 3)(x - 5) \int_{a}^{x} \{ f(t) \}^4 dt$$
$g'(x) = 0$ 에서 $x = 3$ 또는 $x = 5$ 또는 $\int_{a}^{x} \{ f(t) \}^4 dt = 0$ 이다.
극값이 오직 하나이므로 $g'(x) = 0$ 의 실근 중 좌우에서 부호 변화가 일어나는 실근이 오직 하나이어야 한다.
이때 $h(x) = \int_{a}^{x} \{ f(t) \}^4 dt$ 라 하면 $h(a) = 0$ 이고 $h'(x) = \{ f(x) \}^4 \ge 0$ 이므로 $h(x)$ 는 항상 증가하는 함수이다.
따라서 $\int_{a}^{x} \{ f(t) \}^4 dt = 0$ 의 실근은 $x = a$ 뿐이다.
$g'(x) = 3(x - 3)(x - 5)(x - a) \times (\text{양수})$ 꼴에서 부호 변화가 한 번만 일어나려면 $x = a$ 가 $x = 3$ 또는 $x = 5$ 를 반드시 근으로 가져야 한다. (중근이 되어 부호 변화가 사라짐)
즉, $a = 3$ 또는 $a = 5$ 이다.
따라서 모든 $a$ 값의 합은 $3 + 5 = 8$ 이다.

[정답] $8$
