## 문제
$\lim_{n \to \infty} \frac{\left( \frac{m}{5} \right)^{n+1} + 2}{\left( \frac{m}{5} \right)^{n} + 1} = 2$ 가 되도록 하는 자연수 $m$ 의 개수는?

## 보기
① $5$   ② $6$   ③ $7$   ④ $8$   ⑤ $9$

## 풀이
(ⅰ) $0 < \frac{m}{5} < 1$, 즉 $0 < m < 5$ 이면
$\lim_{n \to \infty} \left( \frac{m}{5} \right)^{n} = 0$ 이므로
$\lim_{n \to \infty} \frac{\left( \frac{m}{5} \right)^{n+1} + 2}{\left( \frac{m}{5} \right)^{n} + 1} = \frac{0 + 2}{0 + 1} = 2$
$m$ 의 값은 $1, 2, 3, 4$ 이다.

(ⅱ) $\frac{m}{5} = 1$, 즉 $m = 5$ 이면
$\lim_{n \to \infty} \left( \frac{m}{5} \right)^{n} = 1$ 이므로
$\lim_{n \to \infty} \frac{\left( \frac{m}{5} \right)^{n+1} + 2}{\left( \frac{m}{5} \right)^{n} + 1} = \frac{1 + 2}{1 + 1} = \frac{3}{2}$
따라서 $m \neq 5$ 이다.

(ⅲ) $\frac{m}{5} > 1$, 즉 $m > 5$ 이면
$\lim_{n \to \infty} \left( \frac{m}{5} \right)^{n} = \infty$ 이므로
$\lim_{n \to \infty} \frac{\left( \frac{m}{5} \right)^{n+1} + 2}{\left( \frac{m}{5} \right)^{n} + 1} = \lim_{n \to \infty} \frac{\frac{m}{5} + 2 \times \frac{1}{\left( \frac{m}{5} \right)^{n}}}{1 + \frac{1}{\left( \frac{m}{5} \right)^{n}}} = \frac{\frac{m}{5} + 0}{1 + 0} = \frac{m}{5}$
즉, $\frac{m}{5} = 2$ 에서 $m = 10$ 이다.

따라서 $\lim_{n \to \infty} \frac{\left( \frac{m}{5} \right)^{n+1} + 2}{\left( \frac{m}{5} \right)^{n} + 1} = 2$ 가 되도록 하는 자연수 $m$ 의 개수는 $1, 2, 3, 4, 10$ 으로 총 $5$ 이다.
