## 문제
최고차항의 계수가 $1$인 삼차함수 $f(x)$와 실수 전체의 집합에서 연속인 함수 $g(x)$가 다음 조건을 만족시킬 때, $f(4)$의 값을 구하시오.

$$ \begin{array}{l} \text{(가) 모든 실수 } x \text{에 대하여 } f(x) = f(1) + (x - 1)f'(g(x)) \text{이다.} \\ \text{(나) 함수 } g(x) \text{의 최솟값은 } \frac{5}{2} \text{이다.} \\ \text{(다) } f(0) = -3, f(g(1)) = 6 \end{array} $$

## 보기

## 풀이
조건 (가)에서
$f(x) = f(1) + (x - 1)f'(g(x))$
이므로 $x \neq 1$일 때,
$f'(g(x)) = \frac{f(x) - f(1)}{x - 1} \cdots \cdots$ ㉠
이때, 두 점 $(1, f(1))$, $(x, f(x))$를 지나는 직선의 기울기가 $f'(g(x))$이고, 조건 (나)에서 $g(x) \ge \frac{5}{2}$이므로 두 점 $(1, f(1))$, $(\frac{5}{2}, f(\frac{5}{2}))$를 지나는 직선은 점 $(\frac{5}{2}, f(\frac{5}{2}))$에서 접하는 직선이다.
두 점 $(1, f(1))$, $(\frac{5}{2}, f(\frac{5}{2}))$를 지나는 직선의 방정식을 $y = mx + n$이라 하면
$f(x) = (x - 1)\left(x - \frac{5}{2}\right)^2 + mx + n \cdots \cdots$ ㉡
조건 (다)에서 $f(0) = -3$이므로
$f(0) = -\frac{25}{4} + n = -3$, $n = \frac{13}{4}$
㉠에서
$\lim_{x \rightarrow 1} f'(g(x)) = \lim_{x \rightarrow 1} \frac{f(x) - f(1)}{x - 1}$
$f'(g(1)) = f'(1) \cdots \cdots$ ㉢
㉡의 양변을 $x$에 대하여 미분하면
$f'(x) = \left(x - \frac{5}{2}\right)^2 + 2(x - 1)\left(x - \frac{5}{2}\right) + m$
$= 3x^2 - 12x + m + \frac{45}{4}$
㉢에서 $g(1) \ge \frac{5}{2}$이므로 $g(1) \neq 1$
따라서 함수 $y = f'(x)$의 그래프의 대칭축이 $x = 2$이므로
$\frac{g(1) + 1}{2} = 2$에서 $g(1) = 3$
조건 (다)에서 $f(3) = 6$이므로 ㉡에 대입하면
$f(3) = \frac{1}{2} + 3m + \frac{13}{4} = 6$, $m = \frac{3}{4}$
따라서 $f(x) = (x - 1)\left(x - \frac{5}{2}\right)^2 + \frac{3}{4}x + \frac{13}{4}$이므로
$f(4) = 13$

**[다른 풀이]**
최고차항의 계수가 $1$이고 $f(0) = -3$이므로
$f(x) = x^3 + ax^2 + bx - 3$
이라 하면 $f'(x) = 3x^2 + 2ax + b$
조건 (가)에서
$f(x) = f(1) + (x - 1)f'(g(x))$
이므로 $x \neq 1$일 때,
$f'(g(x)) = \frac{f(x) - f(1)}{x - 1} \cdots \cdots$ ㉠
이때, 두 점 $(1, f(1))$, $(x, f(x))$를 지나는 직선의 기울기가 $f'(g(x))$이고, 조건 (나)에서 $g(x) \ge \frac{5}{2}$이므로 두 점 $(1, f(1))$, $(\frac{5}{2}, f(\frac{5}{2}))$를 지나는 직선은 점 $(\frac{5}{2}, f(\frac{5}{2}))$에서 접하는 직선이다.
그러므로 직선 $y - f(\frac{5}{2}) = f'(\frac{5}{2})(x - \frac{5}{2})$는 점 $(1, f(1))$을 지난다. 즉,
$1 + a + b - 3 - \{ (\frac{5}{2})^3 + a(\frac{5}{2})^2 + b(\frac{5}{2}) - 3 \}$
$= \{ 3 \times (\frac{5}{2})^2 + 5a + b \} (1 - \frac{5}{2})$
이 식을 정리하면
$-\frac{117}{8} - \frac{21}{4}a - \frac{3}{2}b = (\frac{75}{4} + 5a + b) (-\frac{3}{2})$
$\frac{9}{4}a = -\frac{108}{8}$
$a = -6$
따라서 $f(x) = x^3 - 6x^2 + bx - 3$이고, $f'(x) = 3x^2 - 12x + b$이다.
한편, ㉠에서
$\lim_{x \rightarrow 1} f'(g(x)) = \lim_{x \rightarrow 1} \frac{f(x) - f(1)}{x - 1}$
이때, $g(x)$는 연속함수이므로 $g(1) = k$라 하면 좌변은
$\lim_{x \rightarrow 1} f'(g(x)) = \lim_{x \rightarrow 1} \{ 3\{g(x)\}^2 - 12g(x) + b \}$
$= 3k^2 - 12k + b \cdots \cdots$ ㉡
또, 우변은
$\lim_{x \rightarrow 1} \frac{f(x) - f(1)}{x - 1} = f'(1)$
$= 3 - 12 + b = b - 9 \cdots \cdots$ ㉢
㉡, ㉢에서
$3k^2 - 12k + b = b - 9$
$k^2 - 4k + 3 = 0$, $(k - 1)(k - 3) = 0$
$k = 1$ 또는 $k = 3$
$g(1) = 1$ 또는 $g(1) = 3$
이때, 조건 (나)에서 $g(x)$의 최솟값이 $\frac{5}{2}$이므로
$g(1) = 3$
따라서 조건 (다)에서 $f(g(1)) = f(3) = 6$이므로
$27 - 54 + 3b - 3 = 6$, $b = 12$
따라서 $f(x) = x^3 - 6x^2 + 12x - 3$이므로
$f(4) = 4^3 - 6 \times 4^2 + 12 \times 4 - 3 = 13$
