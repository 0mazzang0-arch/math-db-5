## 문제
$1 < x < 3$ 이고 $3 < y < 5$ 인 두 실수 $x$, $y$ 에 대하여 집합 $A = \{ 1, x, 3, y, 5 \}$ 의 부분집합 중 원소의 개수가 $2$ 인 부분집합을 각각 $A_{1}$, $A_{2}$, $A_{3}$, $\dots$, $A_{10}$ 이라 하고, 집합 $A_{k}$ ($k = 1, 2, 3, \dots, 10$) 의 두 원소 중 작은 수를 $m_{k}$, 큰 수를 $M_{k}$ 라 하자. $m_{k}$ 와 $M_{k}$ 가 다음 조건을 만족시킬 때, $12x + 6y$ 의 최댓값을 구하시오.

(가) $m_{1} + m_{2} + m_{3} + \dots + m_{10} \le 19$
(나) $M_{1} + M_{2} + M_{3} + \dots + M_{10} \le 41$

## 보기

## 풀이
집합 $A$ 의 원소를 작은 것부터 나열하면 $1 < x < 3 < y < 5$ 이다.
원소의 개수가 $2$ 인 부분집합 $10$ 개에서 각 원소가 작은 수 $m_{k}$ 가 되는 횟수는 다음과 같다.
- $1$ 이 가장 작은 수인 경우: $\{1, x\}, \{1, 3\}, \{1, y\}, \{1, 5\}$ ($4$ 개)
- $x$ 가 가장 작은 수인 경우: $\{x, 3\}, \{x, y\}, \{x, 5\}$ ($3$ 개)
- $3$ 이 가장 작은 수인 경우: $\{3, y\}, \{3, 5\}$ ($2$ 개)
- $y$ 가 가장 작은 수인 경우: $\{y, 5\}$ ($1$ 개)

따라서 $\sum_{k=1}^{10} m_{k} = 4(1) + 3x + 2(3) + 1y = 3x + y + 10$ 이다.
조건 (가)에 의해 $3x + y + 10 \le 19$ 이므로 $3x + y \le 9$ 이다.

또한, 각 원소가 큰 수 $M_{k}$ 가 되는 횟수는 다음과 같다.
- $5$ 가 가장 큰 수인 경우: $\{1, 5\}, \{x, 5\}, \{3, 5\}, \{y, 5\}$ ($4$ 개)
- $y$ 가 가장 큰 수인 경우: $\{1, y\}, \{x, y\}, \{3, y\}$ ($3$ 개)
- $3$ 가 가장 큰 수인 경우: $\{1, 3\}, \{x, 3\}$ ($2$ 개)
- $x$ 가 가장 큰 수인 경우: $\{1, x\}$ ($1$ 개)

따라서 $\sum_{k=1}^{10} M_{k} = 4(5) + 3y + 2(3) + 1x = x + 3y + 26$ 이다.
조건 (나)에 의해 $x + 3y + 26 \le 41$ 이므로 $x + 3y \le 15$ 이다.

$12x + 6y = k$ 라 할 때, $k$ 가 최대가 되는 지점은 두 직선 $3x + y = 9$ 와 $x + 3y = 15$ 의 교점이다.
두 식을 연립하면:
$y = 9 - 3x$ 를 $x + 3y = 15$ 에 대입
$x + 3(9 - 3x) = 15 \Rightarrow x + 27 - 9x = 15 \Rightarrow -8x = -12 \Rightarrow x = 1.5$
$y = 9 - 3(1.5) = 4.5$

$x = 1.5$, $y = 4.5$ 는 주어진 범위 $1 < x < 3$ 과 $3 < y < 5$ 를 만족한다.
따라서 $12x + 6y$ 의 최댓값은:
$12(1.5) + 6(4.5) = 18 + 27 = 45$

정답: $45$
