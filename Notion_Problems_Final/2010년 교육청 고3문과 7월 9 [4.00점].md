## 문제
영행렬이 아닌 두 이차정사각행렬 $A$, $B$에 대하여 옳은 것만을 <보기>에서 있는 대로 고른 것은?
(단, $O$는 영행렬이고, $E$는 단위행렬이다.)

ㄱ. $A B = O$ 이면 $A^{2} B^{2} = O$
ㄴ. $A + B = E$ 이면 $A B = B A$
ㄷ. $A^{2} = O$ 이면 행렬 $A + E$의 역행렬이 존재하지 않는다.

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
ㄱ. $A B = O$ 이면 $A^{2} B^{2} = A (A B) B = A O B = O$ 이다. (참)

ㄴ. $A + B = E$ 에서 $A (A + B) = A E$ 이므로 
$A^{2} + A B = A$ $\cdots$ ① 
$A + B = E$ 에서 $(A + B) A = E A$ 이므로 
$A^{2} + B A = A$ $\cdots$ ② 
①, ②에 의해 $A B = B A = A - A^{2}$ 이 성립한다. (참)

ㄷ. $A^{2} = O$ 이면 $A^{2} - E = -E$ 이므로 
$(A - E) (A + E) = -E$ 
$(A + E) (E - A) = E$ 
따라서 $A + E$ 의 역행렬 $E - A$ 가 존재한다. (거짓)

따라서 옳은 것은 $ㄱ, ㄴ$이다.
