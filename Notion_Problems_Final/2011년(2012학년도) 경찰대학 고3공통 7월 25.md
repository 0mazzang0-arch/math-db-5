## 문제
자연수 $n$에 대하여 수직선 위의 점 $A_n (x_n)$이 다음 조건을 만족시킬 때, 모든 $a$의 값의 합은?

(가) $x_1 = 1$, $x_2 = a$ (단, $a$는 자연수)
(나) $A_{n+2}$는 선분 $A_n A_{n+1}$을 $1-t : t$로 내분하는 점이다. (단, $0 < t < 1$)
(다) $\lim_{n \to \infty} x_n$의 값이 정수가 되게 하는 실수 $t$의 개수는 $11$이다.



> [!example] 📸 (그림/그래프)

## 보기
① $45$   ② $47$   ③ $49$   ④ $51$   ⑤ $53$

## 풀이
조건 (나)에 의해서 다음 점화식이 성립한다.
$$x_{n+2} = t x_n + (1 - t) x_{n+1}$$
위 식을 변형하면
$$x_{n+2} - x_{n+1} = -t (x_{n+1} - x_n)$$
따라서 $\{x_{n+1} - x_n\}$은 첫째항이 $a - 1$이고 공비가 $-t$인 등비수열을 이룬다.
$$\therefore x_n = x_1 + \sum_{k=1}^{n-1} (x_{k+1} - x_k) = 1 + \sum_{k=1}^{n-1} (a - 1)(-t)^{k-1}$$
$$\therefore \lim_{n \to \infty} x_n = 1 + \sum_{n=1}^{\infty} (a - 1)(-t)^{n-1} = 1 + \frac{a - 1}{1 + t}$$
분수함수 $y = 1 + \frac{a - 1}{1 + t}$는 $t > -1$인 범위에서 감소하므로 $0 < t < 1$에서
$$\frac{a + 1}{2} < 1 + \frac{a - 1}{1 + t} < a$$
(ⅰ) $a$가 홀수일 때, 조건 (다)에서
$$a - \frac{a + 1}{2} - 1 = 11$$
$$\therefore a = 25$$
(ⅱ) $a$가 짝수일 때, 조건 (다)에서
$$a - \frac{a + 2}{2} = 11$$
$$\therefore a = 24$$
(ⅰ), (ⅱ)에서 구하는 모든 $a$의 값의 합은
$$25 + 24 = 49$$

정답: ③
