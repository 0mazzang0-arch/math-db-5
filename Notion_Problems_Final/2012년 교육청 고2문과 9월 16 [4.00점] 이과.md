## 문제
자연수 $n$과 양수 $A$에 대하여 이차방정식 $x^{2} - \left( 3n + \frac{1}{3n} \right) x + 1 = 0$의 한 근은 $\log A^{3}$의 정수부분이고, 다른 한 근은 $\log A^{2}$의 소수부분이다. <보기>에서 옳은 것만을 있는 대로 고른 것은?

$$ \begin{array}{|l|} \hline \text{ㄱ. } n=1 \text{ 이면 } \log A^{3} \text{ 의 정수부분은 } 3 \text{ 이다.} \\ \text{ㄴ. } \log A \text{ 의 소수부분은 } \frac{1}{6n} \text{ 이다.} \\ \text{ㄷ. } A^{12} \text{ 이 자연수가 되도록 하는 } n \text{ 의 개수는 } 2 \text{ 이다.} \\ \hline \end{array} $$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
[정답] ⑤

$x^{2} - \left( 3n + \frac{1}{3n} \right) x + 1 = 0$의 두 근은 $3n$과 $\frac{1}{3n}$이므로, $\log A^{3}$의 정수부분은 $3n$이고 $\log A^{2}$의 소수부분은 $\frac{1}{3n}$이다.

$\log A^{3} = 3n + \alpha$ ($0 \le \alpha < 1$)라 하자.
$\log A^{2} = 2n + \frac{2}{3} \alpha$이고 $0 \le \frac{2}{3} \alpha < \frac{2}{3}$이므로 $\log A^{2}$의 소수부분은 $\frac{2}{3} \alpha$이다.
$\frac{2}{3} \alpha = \frac{1}{3n}$이므로 $\alpha = \frac{1}{2n}$이다.
$\log A = n + \frac{1}{6n}$

ㄱ. $n=1$일 때 $\log A^{3}$의 정수부분은 $3$이다. (참)
ㄴ. $\log A$의 소수부분은 $\frac{1}{6n}$이다. (참)
ㄷ. $\log A^{12} = 12 \log A = 12n + \frac{2}{n}$이므로 $A^{12} = 10^{12n + \frac{2}{n}}$이다.
$A^{12}$이 자연수가 되기 위해서는 $12n + \frac{2}{n}$가 정수여야 하므로 $n=1$ 또는 $n=2$일 때 $A^{12}$은 자연수이다.
따라서 $n$의 개수는 $2$이다. (참)
그러므로 ㄱ, ㄴ, ㄷ 모두 옳다.
