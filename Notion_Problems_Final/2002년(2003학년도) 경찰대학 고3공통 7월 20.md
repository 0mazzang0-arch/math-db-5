## 문제
닫힌구간 $[0, \pi]$ 에서 $f(x) = \frac{\sin x - 1}{\cos x - 3}$ 의 최댓값은?

## 풀이
**[정답]** 
$\frac{1}{2}$

**[해설]** 
$\cos x = X$, $\sin x = Y$ 라 하면 
$-1 \le X \le 1$, $0 \le Y \le 1$ 이고, 
$\cos^{2} x + \sin^{2} x = 1$ 이므로 
$X^{2} + Y^{2} = 1$ ($-1 \le X \le 1, 0 \le Y \le 1$) $\cdots$ $㉡$ 

주어진 함수는 $y = \frac{Y-1}{X-3}$ 이므로 
$y$ 는 원 위의 점 $(X, Y)$ 와 $(3, 1)$ 을 잇는 직선의 기울기이다. 

> [!example] 📸 (그림/그래프)

그래프에서 $y$ 의 최댓값은 $\frac{1}{2}$ 이다. 

**[다른 풀이]** 
주어진 식을 미분하면 
$f'(x) = \frac{\cos x \cdot (\cos x - 3) - (\sin x - 1) \cdot (-\sin x)}{(\cos x - 3)^{2}}$ 
$= \frac{1 - (\sin x + 3 \cos x)}{(\cos x - 3)^{2}}$ 
$= \frac{1 - \sqrt{10} \sin(x + \alpha)}{(\cos x - 3)^{2}}$ 
($\sin \alpha = \frac{3 \sqrt{10}}{10}$, $\cos \alpha = \frac{\sqrt{10}}{10}$) 

$x = \frac{\pi}{2}$ 일 때, $\sin (\frac{\pi}{2} + \alpha) = \cos \alpha = \frac{\sqrt{10}}{10}$ 이므로 $f' (\frac{\pi}{2}) = 0$ 이고 극소가 된다. 

$x = 0$ 일 때, $f(0) = \frac{1}{2}$ 
$x = \frac{\pi}{2}$ 일 때, $f(\frac{\pi}{2}) = 0$ 
$x = \pi$ 일 때, $f(\pi) = \frac{1}{4}$ 

따라서, 닫힌구간 $[0, \pi]$ 에서 함수 $f(x)$ 는 $x = 0$ 에서 최댓값 $\frac{1}{2}$ 을 갖는다.
