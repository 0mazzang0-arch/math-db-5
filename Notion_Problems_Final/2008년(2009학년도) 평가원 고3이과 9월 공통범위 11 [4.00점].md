## 문제
다항함수 $f(x)$가 다음 두 조건을 만족한다.

(가) $f(0)=0$
(나) $0 < x < y < 1$인 모든 $x$, $y$에 대하여 $0 < xf(y) < yf(x)$

세 수 $A=f'(0)$, $B=f(1)$, $C=2 \int_{0}^{1} f(x)dx$의 대소 관계를 옳게 나타낸 것은?



> [!example] 📸 (그림/그래프)

## 보기
① $A < B < C$   ② $A < C < B$   ③ $B < A < C$   ④ $B < C < A$   ⑤ $C < A < B$

## 풀이
[정답] ④

$f(x)$는 (가)에 의해 $(0,0)$을 지나고 (나)에 의해 $0 < x < y < 1$일 때 $f(x) > 0$, $\frac{f(y)}{y} < \frac{f(x)}{x}$임을 알 수 있다.

$A$는 $\square OPQS$의 넓이이고 $B$는 $\square OPRR'$의 넓이, $C$는 어두운 부분의 넓이 $T$의 넓이의 $2$배이다.

$\square OPQS > \square OPRR'$이므로 $A > B$

$2 \triangle ORP < 2 \times (\text{어두운 부분 } T\text{의 넓이})$ 이므로 $B < C$

$2 \triangle OPQ < 2 \times (\text{어두운 부분 } T\text{의 넓이})$ 이므로 $A > C$

따라서, $A > C > B$ 이므로 $B < C < A$이다.
