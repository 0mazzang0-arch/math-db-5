## 문제
실수 $x$에 대한 두 조건
$p$ : $|x-a| \le 1$,
$q$ : $x^2 - 2x - 8 > 0$
에 대하여 $p \to \sim q$가 참이 되도록 하는 실수 $a$의 최댓값은?

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
조건 $p$의 진리집합을 $P$라 하면
$|x-a| \le 1$
$a-1 \le x \le a+1$
$P = \{ x | a-1 \le x \le a+1 \}$

조건 $q$에 대하여 조건 $\sim q$는
$x^2 - 2x - 8 \le 0$
$(x-4)(x+2) \le 0$
$-2 \le x \le 4$
따라서 $\sim q$의 진리집합은 $Q^c = \{ x | -2 \le x \le 4 \}$이다.

명제 $p \to \sim q$가 참이 되려면 $P \subseteq Q^c$이어야 하므로
$-2 \le a-1$이고 $a+1 \le 4$이어야 한다.
$-1 \le a$ 이고 $a \le 3$
즉, $-1 \le a \le 3$

따라서 실수 $a$의 최댓값은 $3$이다.

정답: ③
