## 문제
$x > 0$ 에서 정의된 함수 
$$ f(x) = \left\{ \begin{array}{ll} 0 & ( 0 < x \le 1 ) \\ \log_{3} x & ( x > 1 ) \end{array} \right. $$
에 대하여 $f(t) + f\left(\frac{1}{t}\right) = 2$ 를 만족시키는 모든 양수 $t$ 의 값의 합은? 

## 보기
① $\frac{76}{9}$   ② $\frac{79}{9}$   ③ $\frac{82}{9}$   ④ $\frac{85}{9}$   ⑤ $\frac{88}{9}$

## 풀이
(ⅰ) $0 < t < 1$ 일 때, 
$\frac{1}{t} > 1$ 이므로 
$f(t) + f\left(\frac{1}{t}\right) = 0 + \log_{3} \frac{1}{t} = 2$ 에서 
$t = \frac{1}{9}$ 이다. 

(ⅱ) $t = 1$ 일 때, 
$f(t) + f\left(\frac{1}{t}\right) = 0 + 0 = 0 \neq 2$ 

(ⅲ) $t > 1$ 일 때, 
$0 < \frac{1}{t} < 1$ 이므로 
$f(t) + f\left(\frac{1}{t}\right) = \log_{3} t + 0 = 2$ 에서 
$t = 9$ 이다. 

따라서 $f(t) + f\left(\frac{1}{t}\right) = 2$ 를 만족시키는 모든 양수 $t$ 의 값의 합은 
$\frac{1}{9} + 9 = \frac{82}{9}$

정답: ③
