## 문제
모든 항이 양수인 수열 $\{ a_{n} \}$이 모든 자연수 $n$에 대하여
$$\sum_{k=1}^{n} a_{k} = \frac{a_{n}^{2}+1}{2 a_{n}}$$
을 만족시킨다. 다음은 일반항 $a_{n}$이
$$a_{n} = \sqrt{n} - \sqrt{n-1} \quad \cdots \cdots (*)$$
임을 수학적 귀납법을 이용하여 증명한 것이다.

$$ \begin{array}{|l|} \hline \\ \text{(i) } n=1 \text{ 일 때, } \\ a_{1} = \frac{a_{1}^{2}+1}{2 a_{1}} \text{ 에서 } a_{1} > 0 \text{ 이므로 } \\ (\text{좌변}) = a_{1} = 1, \text{ (우변)} = 1 - 0 = 1 \text{ 이다. } \\ n=1 \text{ 일 때 } (*) \text{ 이 성립한다. } \\ \\ \text{(ii) } n=m \text{ 일 때 } (*) \text{ 이 성립한다고 가정하면 } \\ a_{m} = \sqrt{m} - \sqrt{m-1} \text{ 이므로 } \\ \sum_{k=1}^{m+1} a_{k} = \sum_{k=1}^{m} a_{k} + a_{m+1} \\ = \sum_{k=1}^{m} (\sqrt{k} - \sqrt{k-1}) + a_{m+1} \\ = (가) + a_{m+1} \\ \\ \frac{a_{m+1}^{2}+1}{2 a_{m+1}} = (가) + a_{m+1} \\ a_{m+1}^{2} + 2 \sqrt{m} \times a_{m+1} - 1 = 0 \\ a_{m+1} > 0 \text{ 이므로 } \\ a_{m+1} = (나) - \sqrt{m} \\ n=m+1 \text{ 일 때도 } (*) \text{ 이 성립한다. } \\ \\ \hline \end{array} $$

(i), (ii)에 의하여 모든 자연수 $n$에 대하여 $a_{n} = \sqrt{n} - \sqrt{n-1}$ 이다.
위의 (가)와 (나)에 알맞은 식을 각각 $f(m)$, $g(m)$이라 할 때, $f(25) + g(35)$의 값은?

## 보기
① $8$   ② $9$   ③ $10$   ④ $11$   ⑤ $12$

## 풀이
(i) $n=1$ 일 때,
$a_{1} = \frac{a_{1}^{2}+1}{2 a_{1}}$ 에서 $a_{1}^{2} = 1$
$a_{1} > 0$ 이므로 $a_{1} = 1$.
(우변) $= 1 - 0 = 1$.
$n=1$ 일 때 $(*)$이 성립한다.

(ii) $n=m$ 일 때 $(*)$이 성립한다고 가정하면 $a_{m} = \sqrt{m} - \sqrt{m-1}$ 이므로
$$ \begin{array}{l} \sum_{k=1}^{m+1} a_{k} = \sum_{k=1}^{m} a_{k} + a_{m+1} \\ = \sum_{k=1}^{m} (\sqrt{k} - \sqrt{k-1}) + a_{m+1} \\ = (\sqrt{1} - \sqrt{0}) + (\sqrt{2} - \sqrt{1}) + \cdots + (\sqrt{m} - \sqrt{m-1}) + a_{m+1} \\ = \sqrt{m} + a_{m+1} \end{array} $$
따라서 $(가)$에 알맞은 식은 $f(m) = \sqrt{m}$이다.

$\sum_{k=1}^{m+1} a_{k} = \frac{a_{m+1}^{2}+1}{2 a_{m+1}}$ 이므로
$$ \begin{array}{l} \frac{a_{m+1}^{2}+1}{2 a_{m+1}} = \sqrt{m} + a_{m+1} \\ a_{m+1}^{2} + 1 = 2 \sqrt{m} \times a_{m+1} + 2 a_{m+1}^{2} \\ a_{m+1}^{2} + 2 \sqrt{m} \times a_{m+1} - 1 = 0 \end{array} $$
$a_{m+1} > 0$ 이므로 근의 공식에 의해
$a_{m+1} = \frac{-2 \sqrt{m} + \sqrt{4m + 4}}{2} = -\sqrt{m} + \sqrt{m+1}$
따라서 $(나)$에 알맞은 식은 $g(m) = \sqrt{m+1}$이다.

$n=m+1$ 일 때도 $(*)$이 성립한다.
(i), (ii)에 의하여 모든 자연수 $n$에 대하여 $a_{n} = \sqrt{n} - \sqrt{n-1}$ 이다.

$f(m) = \sqrt{m}$, $g(m) = \sqrt{m+1}$ 이므로
$f(25) + g(35) = \sqrt{25} + \sqrt{35+1} = 5 + 6 = 11$
