## 문제
$x \ge 0$에서 정의된 함수 $f(x) = a \cos bx + c$의 최댓값이 $3$, 최솟값이 $-1$이다. 그림과 같이 함수 $y = f(x)$의 그래프와 직선 $y = 3$이 만나는 점 중에서 $x$좌표가 가장 작은 점과 두 번째로 작은 점을 각각 $A$, $B$라 하고, 함수 $y = f(x)$의 그래프와 $x$축이 만나는 점 중에서 $x$좌표가 가장 작은 점과 두 번째로 작은 점을 각각 $C$, $D$라 하자. 사각형 $ACDB$의 넓이가 $6\pi$일 때, $0 \le x \le 4\pi$에서 방정식 $f(x) = 2$의 모든 해의 합은? (단, $a$, $b$, $c$는 양수이다.)

> [!example] 📸 (그림/그래프)

## 보기
① $6\pi$   ② $\frac{13}{2}\pi$   ③ $7\pi$   ④ $\frac{15}{2}\pi$   ⑤ $8\pi$

## 풀이
함수 $f(x) = a \cos bx + c$의 최댓값이 $3$, 최솟값이 $-1$이고 $a$가 양수이므로
$a + c = 3$, $-a + c = -1$
두 식을 연립하면 $a = 2$, $c = 1$이다.
함수 $f(x) = 2 \cos bx + 1$의 주기가 $\frac{2\pi}{b}$이므로
$\overline{AB} = \frac{2\pi}{b}$

$0 \le x \le \frac{2\pi}{b}$에서 방정식 $2 \cos bx + 1 = 0$의 해는
$\cos bx = -\frac{1}{2}$
$bx = \frac{2\pi}{3}$ 또는 $bx = \frac{4\pi}{3}$
$x = \frac{2\pi}{3b}$ 또는 $x = \frac{4\pi}{3b}$이므로
$\overline{CD} = \frac{4\pi}{3b} - \frac{2\pi}{3b} = \frac{2\pi}{3b}$

사각형 $ACDB$의 높이는 $y = 3$과 $y = 0$($x$축) 사이의 거리이므로 $3$이다.
사각형 $ACDB$의 넓이는
$\frac{1}{2} \times \left( \frac{2\pi}{3b} + \frac{2\pi}{b} \right) \times 3 = 6\pi$
$\frac{1}{2} \times \frac{8\pi}{3b} \times 3 = 6\pi$
$\frac{4\pi}{b} = 6\pi$이므로 $b = \frac{2}{3}$
따라서 $f(x) = 2 \cos \frac{2}{3}x + 1$이다.

$0 \le x \le 4\pi$에서 방정식 $f(x) = 2$는
$2 \cos \frac{2}{3}x + 1 = 2$
$\cos \frac{2}{3}x = \frac{1}{2}$
$0 \le x \le 4\pi$일 때 $0 \le \frac{2}{3}x \le \frac{8}{3}\pi$이므로
$\frac{2}{3}x = \frac{\pi}{3}, \frac{5\pi}{3}, \frac{7\pi}{3}$
$x = \frac{\pi}{2}, \frac{5\pi}{2}, \frac{7\pi}{2}$
따라서 모든 해의 합은
$\frac{\pi}{2} + \frac{5\pi}{2} + \frac{7\pi}{2} = \frac{13}{2}\pi$

정답: ②
