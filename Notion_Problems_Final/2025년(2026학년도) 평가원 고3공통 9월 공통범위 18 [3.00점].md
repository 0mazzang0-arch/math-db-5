## 문제
등차수열 $\{ a_{n} \}$ 에 대하여 $a_{3} = 6$, $2a_{5} - a_{4} = 15$ 일 때, $a_{11}$ 의 값을 구하시오.

## 풀이
등차수열 $\{ a_{n} \}$ 의 공차를 $d$ 라 하면
$2a_{5} - a_{4} = 2 ( a_{1} + 4d ) - ( a_{1} + 3d ) = a_{1} + 5d$
$2a_{5} - a_{4} = 15$ 이므로
$a_{1} + 5d = 15 \cdots$ ㉠
$a_{3} = a_{1} + 2d = 6 \cdots$ ㉡
㉠, ㉡을 연립하여 풀면
$d = 3$, $a_{1} = 0$
$\therefore a_{11} = a_{1} + 10d = 0 + 10 \times 3 = 30$

정답: $30$
