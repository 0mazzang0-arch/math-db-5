## 문제
영행렬이 아닌 이차정사각행렬 $A$ 가 $A^2 = 3A$ 를 만족시킨다. 다음은 모든 자연수 $n$ 에 대하여 행렬 $(A-E)^n$ 을 $(A-E)^n = a_n A + (-1)^n E$ 와 같이 나타낼 때, 수열 $\{ a_n \}$ 의 일반항을 구하는 과정이다. (단, $E$ 는 단위행렬이다.)

$$ \begin{array}{|l|} \hline \text{자연수 } n \text{ 에 대하여} \\ (A-E)^{n+1} = \{ a_n A + (-1)^n E \} (A-E) \\ = a_n A^2 - a_n A + (-1)^n A + (-1)^{n+1} E \\ \text{이고, } A^2 = 3A \text{ 이므로} \\ (A-E)^{n+1} = (2 a_n + (가)) A + (-1)^{n+1} E \\ \text{이다. 그러므로} \\ a_{n+1} = 2 a_n + (가) \cdots \cdots ㉠ \\ \text{이다. 따라서 } 2 \text{ 이상인 자연수 } n \text{ 에 대하여} \\ a_n + a_{n+1} = 2 (a_{n-1} + a_n) \\ \text{이다. 또한 } a_1 + a_2 = 1 + 1 = 2 \text{ 이므로 모든 자연수 } n \text{ 에 대하여} \\ a_n + a_{n+1} = (나) \cdots \cdots ㉡ \\ \text{이다. ㉠과 ㉡에 의해 } \\ 3 a_n + (-1)^n = (나) \\ \text{이다. 따라서 } a_n = \frac{(나) + (-1)^{n+1}}{3} \text{ 이다.} \\ \hline \end{array} $$

위의 과정에서 $(가)$, $(나)$에 알맞은 식을 각각 $f(n)$, $g(n)$ 이라 할 때, $f(9) \times g(5)$ 의 값은?

## 보기
① $-32$   ② $-16$   ③ $8$   ④ $16$   ⑤ $32$

## 풀이
$(A-E)^{n+1} = a_n A^2 - a_n A + (-1)^n A + (-1)^{n+1} E$ 에서 $A^2 = 3A$ 이므로
$(A-E)^{n+1} = a_n(3A) - a_n A + (-1)^n A + (-1)^{n+1} E$
$= (2 a_n + (-1)^n) A + (-1)^{n+1} E$
따라서 $f(n) = (-1)^n$ 이다.

또한, $a_{n+1} = 2 a_n + (-1)^n$ 에서
$a_n + a_{n+1} = 3 a_n + (-1)^n$ 이고, 
$a_n + a_{n+1} = 2(a_{n-1} + a_n)$ 이며 $a_1 + a_2 = 2$ 이므로
수열 $\{ a_n + a_{n+1} \}$ 은 첫째항이 $2$ 이고 공비가 $2$ 인 등비수열이다.
따라서 $g(n) = a_n + a_{n+1} = 2^n$ 이다.

$f(9) = (-1)^9 = -1$
$g(5) = 2^5 = 32$
$f(9) \times g(5) = -1 \times 32 = -32$

정답은 ① 이다.
