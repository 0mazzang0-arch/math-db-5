## 문제
명제 $a \leq x \leq a+2$ 이면 $-3 \leq x \leq 5$ 이다가 참이 되게 하는 정수 $a$ 의 개수를 구하시오.

## 보기


## 풀이
명제 $a \leq x \leq a+2$ 이면 $-3 \leq x \leq 5$ 이다가 참이 되기 위해서는
$\{x \mid a \leq x \leq a+2 \} \subset \{x \mid -3 \leq x \leq 5 \}$ 이어야 한다.

따라서 $a \geq -3$ 이고 $a+2 \leq 5$ 를 만족해야 한다.
$a+2 \leq 5$ 에서 $a \leq 3$ 이므로,
$-3 \leq a \leq 3$ 이다.

> [!example] 📸 (그림/그래프)

이를 만족하는 정수 $a$ 는 $-3, -2, -1, 0, 1, 2, 3$ 으로 $7$ 개이다.

정답: $7$
