## 문제
양의 실수 $a$, $b$는 다음 조건을 만족시킨다.

Ⅰ. $a$는 정수부분이 세 자리인 수이다.
Ⅱ. $\log_{10} b$의 정수부분은 $-1$이다.

이때, $ab$의 값의 범위는?

## 보기
① $1 \leq ab < 10^{2}$   ② $10 \leq ab < 10^{3}$   ③ $10^{2} \leq ab < 10^{4}$   ④ $10^{3} \leq ab < 10^{5}$   ⑤ $10^{4} \leq ab < 10^{6}$

## 풀이
조건 Ⅰ에서 $a$는 정수부분이 세 자리인 수이므로
$2 \leq \log_{10} a < 3$ 이고

조건 Ⅱ에서 $\log_{10} b$의 정수부분이 $-1$이므로
$-1 \leq \log_{10} b < 0$ 이다.

두 부등식을 각 변끼리 더하면
$1 \leq \log_{10} a + \log_{10} b < 3$
$1 \leq \log_{10} ab < 3$ 이다.

따라서 로그의 정의에 의해
$10^{1} \leq ab < 10^{3}$
$10 \leq ab < 10^{3}$ 이다.

정답: ②
