## 문제
$A$ 지점과 $B$ 지점을 자전거로 왕복하는 데 갈 때는 시속 $20$ $km$, 올 때는 시속 $15$ $km$로 달렸더니, 올 때 걸린 시간은 갈 때 걸린 시간보다 $1$ 시간 $30$ 분이 더 걸렸다. 
다음은 $A$ 지점과 $B$ 지점 사이의 거리를 구하는 과정이다. 

$$ \begin{array}{|l|} \hline \text{갈 때에는 } 1 \text{ } km \text{ 가는데 } 3 \text{ 분, 올 때에는 } 1 \text{ } km \text{ 가는데 } (가) \text{ 분 걸리므로} \\ 1 \text{ } km \text{ 가는데 } (나) \text{ 분씩 차이가 난다. 따라서 } (다) \text{ } km \text{ 가는데 } 1 \text{ 시간 } 30 \text{ 분 차이가 난다.} \\ \text{그러므로 } A \text{ 지점과 } B \text{ 지점 사이의 거리는 } (다) \text{ } km \text{ 이다.} \\ \hline \end{array} $$

이때, $(다)$에 알맞은 수를 구하시오.

## 보기

## 풀이
$A$ 지점과 $B$ 지점을 왕복하는 데 갈 때는 $1$ $km$ 에 $\frac{60}{20} = 3$ (분), 올 때는 $1$ $km$ 에 $\frac{60}{15} = 4$ (분) 걸리므로 $1$ $km$ 가는데 $4 - 3 = 1$ 분씩 차이가 난다. 따라서, $a$ $km$ 가는데 $a$ 분 차이가 난다. $1$ 시간 $30$ 분은 $90$ 분이므로 $(다)$에 알맞은 수는 $90$ 이다. 그러므로 $A$ 지점과 $B$ 지점 사이의 거리는 $90$ $km$ 이다.

정답: $90$
