## 문제
두 집합 $A = \{1, 2, 3, 4\}$, $B = \{3, 4, 5, 6\}$ 에 대하여 $A \cap X = A$, $X \cup (A \cup B) = A \cup B$ 를 만족시키는 집합 $X$ 의 개수는?

## 보기
① $1$   ② $2$   ③ $4$   ④ $8$   ⑤ $16$

## 풀이
$A \cap X = A$ 에서 $A \subset X$ 이고, $X \cup (A \cup B) = A \cup B$ 에서 $X \subset (A \cup B)$ 이다.
그런데 $A = \{1, 2, 3, 4\}$ 이므로 집합 $X$ 에 집합 $A$ 의 원소인 $1, 2, 3, 4$ 가 모두 속해야 한다.
또한 $A \cup B = \{1, 2, 3, 4, 5, 6\}$ 이므로 집합 $X$ 가 될 수 있는 경우는 다음과 같다.

(ⅰ) 원소 $5, 6$ 이 모두 속하지 않는 경우
$X = \{1, 2, 3, 4\}$
(ⅱ) 원소 $5$ 는 속하고 $6$ 은 속하지 않는 경우
$X = \{1, 2, 3, 4, 5\}$
(ⅲ) 원소 $5$ 는 속하지 않고 $6$ 은 속하는 경우
$X = \{1, 2, 3, 4, 6\}$
(ⅳ) 원소 $5, 6$ 이 모두 속하는 경우
$X = \{1, 2, 3, 4, 5, 6\}$

따라서 집합 $X$ 의 개수는 $4$ 이다.

**[다른 풀이]**
$A = \{1, 2, 3, 4\}$ 이고 $A \cup B = \{1, 2, 3, 4, 5, 6\}$ 이므로 두 원소 $5, 6$ 이 집합 $X$ 에 각각 속하는 경우, 속하지 않는 경우로 나누어 집합 $X$ 의 개수를 구하면 $2^{2} = 4$ 이다.

정답: ③
