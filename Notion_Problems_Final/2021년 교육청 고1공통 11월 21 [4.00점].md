## 문제
$1 \le a < b$인 두 상수 $a$, $b$에 대하여 세 집합
$A = \{ (x, y) \mid y = \frac{4}{3}x \text{ 이고 } (x+2)^2 + (y+1)^2 = 1 \}$
$B = \{ (x, y) \mid y = \frac{4}{3}x \text{ 이고 } (x-a-1)^2 + (y-a)^2 = a^2 \}$
$C = \{ (x, y) \mid y = \frac{4}{3}x \text{ 이고 } (x-b-1)^2 + (y-b)^2 = b^2 \}$
이 있다. $n(A \cup B \cup C) = 3$일 때, $a+b$의 값은?

## 보기
① $\frac{14}{5}$   ② $3$   ③ $\frac{16}{5}$   ④ $\frac{17}{5}$   ⑤ $\frac{18}{5}$

## 풀이
세 원
$$(x+2)^2 + (y+1)^2 = 1$$
$$(x-a-1)^2 + (y-a)^2 = a^2$$
$$(x-b-1)^2 + (y-b)^2 = b^2$$
을 차례로 $O_1$, $O_2$, $O_3$이라 하자. 집합 $A$, $B$, $C$는 좌표평면에서 직선 $y = \frac{4}{3}x$가 세 원 $O_1$, $O_2$, $O_3$과 각각 만나는 점의 집합이다.

원 $O_1$의 중심 $(-2, -1)$과 직선 $y = \frac{4}{3}x$, 즉 $4x - 3y = 0$ 사이의 거리가
$$\frac{|-8+3|}{\sqrt{4^2 + 3^2}} = \frac{5}{5} = 1$$
이고 원 $O_1$의 반지름의 길이가 $1$이므로 원 $O_1$과 직선 $y = \frac{4}{3}x$는 한 점에서 만난다.
그러므로 $n(A) = 1$이다.

세 원 $O_1$, $O_2$, $O_3$은 모두 $x$축에 접하고 원 $O_1$의 중심은 제 $3$사분면, 두 원 $O_2$, $O_3$의 중심은 제 $1$사분면 위에 있으므로 원 $O_1$은 두 원 $O_2$, $O_3$과 만나지 않는다.
그러므로 $A \cap (B \cup C) = \emptyset$이다.

$n(A) = 1$, $A \cap (B \cup C) = \emptyset$이므로 $n(A \cup B \cup C) = 3$이려면
$n(B \cup C) = 2$ $\cdots$ ㉠

두 원 $O_2$, $O_3$의 중심 $(a+1, a)$, $(b+1, b)$는 모두 직선 $y = x-1$ 위의 점이다.
직선 $y = x-1$ 위의 점 $(k+1, k)$ ($k \ge 1$)을 중심으로 하고 반지름의 길이가 $k$인 원에 대하여 원의 중심 $(k+1, k)$와 직선 $y = \frac{4}{3}x$ ($4x - 3y = 0$) 사이의 거리는
$$\frac{|4(k+1)-3k|}{\sqrt{4^2 + 3^2}} = \frac{k+4}{5}$$
이므로 점 $(k+1, k)$ ($k \ge 1$)을 중심으로 하고 반지름의 길이가 $k$인 원과 직선 $y = \frac{4}{3}x$는
$k = 1$이면 $k = \frac{k+4}{5}$이므로 서로 접하고,
$k > 1$이면 $k > \frac{k+4}{5}$이므로 서로 다른 두 점에서 만난다.

$1 \le a < b$에서
$a \ge 1$이므로 $n(B) \ge 1$
$b > 1$이므로 $n(C) = 2$ $\cdots$ ㉡

㉠, ㉡에서 $B \subset C$이고 $a \ne b$이면 $B \ne C$이므로 $n(B) < n(C) = 2$이다.
$1 \le n(B) < 2$에서 $n(B) = 1$이므로 원 $O_2$와 직선 $y = \frac{4}{3}x$는 서로 접하고 $a = 1$이다.

$(x-2)^2 + (y-1)^2 = 1$에 $y = \frac{4}{3}x$를 대입하면
$$(x-2)^2 + \left( \frac{4}{3}x-1 \right)^2 = 1$$
$$\frac{25}{9}x^2 - \frac{20}{3}x + 4 = \left( \frac{5}{3}x-2 \right)^2 = 0$$
$x = \frac{6}{5}$, $y = \frac{8}{5}$이므로 $B = \{ (\frac{6}{5}, \frac{8}{5}) \}$이다.

$B \subset C$이므로 $(\frac{6}{5}, \frac{8}{5}) \in C$이다.
점 $(\frac{6}{5}, \frac{8}{5})$이 원 $O_3$ 위의 점이어야 하므로 세 원 $O_1, O_2, O_3$과 직선 $y = \frac{4}{3}x$의 관계는 다음과 같다.

> [!example] 📸 (그림/그래프)

$(x-b-1)^2 + (y-b)^2 = b^2$에 $x = \frac{6}{5}$, $y = \frac{8}{5}$을 대입하면
$$\left( \frac{6}{5}-b-1 \right)^2 + \left( \frac{8}{5}-b \right)^2 = b^2$$
$$\left( \frac{1}{5}-b \right)^2 + \left( \frac{8}{5}-b \right)^2 = b^2$$
$$b^2 - \frac{18}{5}b + \frac{13}{5} = (b-1)\left( b- \frac{13}{5} \right) = 0$$
$b > a = 1$이므로 $b = \frac{13}{5}$이다.
따라서 $a+b = 1 + \frac{13}{5} = \frac{18}{5}$이다.
