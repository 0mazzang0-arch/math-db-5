## 문제
등차수열 $\{a_{n}\}$ 에 대하여 $a_{2} = 14$, $a_{4} + a_{5} = 23$ 일 때, $a_{7} + a_{8} + a_{9}$ 의 값을 구하시오.

## 보기

## 풀이
공차를 $d$ 라고 하면
$a_{4} + a_{5} = (a_{2} + 2d) + (a_{2} + 3d)$
$= 14 + 2d + 14 + 3d$
$= 28 + 5d = 23$
$5d = -5$, $d = -1$
따라서 $a_{8} = a_{2} + 6d = 14 - 6 = 8$
그러므로 $a_{7} + a_{8} + a_{9} = 3a_{8} = 3 \times 8 = 24$
