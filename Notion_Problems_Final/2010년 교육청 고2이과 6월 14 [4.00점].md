## 문제
이차정사각행렬 $A$, $B$에 대하여 $C(A, B) = AB - BA$라 하자. 정수 $k$에 대하여 이차정사각행렬의 집합 $M_k$를
$$ M_k = \{ (a_{ij}) \mid i, j \in \{1, 2\}, i-j \neq k \text{ 이면 } a_{ij} = 0 \} $$
으로 정의할 때, <보기>에서 옳은 내용만을 있는 대로 고른 것은? (단, $O$는 영행렬이고 $E$는 단위행렬이다.)

**<보 기>**
ㄱ. $A$, $B \in M_0$이면 $C(A, B) = O$이다.
ㄴ. $|k| \ge 2$이면 $M_k = \{O, E\}$이다.
ㄷ. $|k| < 2$일 때, $A$, $B \in M_k$이면 $C(A, B) = O$이다.

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄷ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
ㄱ. $A, B \in M_0$이면 $i-j \neq 0$일 때 $a_{ij}=0$이므로
$A = \left( \begin{array}{cc} a_{11} & 0 \\ 0 & a_{22} \end{array} \right)$, $B = \left( \begin{array}{cc} b_{11} & 0 \\ 0 & b_{22} \end{array} \right)$이다.
대각행렬끼리의 곱은 교환법칙이 성립하므로 $AB = BA$이다.
$\therefore C(A, B) = AB - BA = O$ (참)

ㄴ. $|k| \ge 2$이면 임의의 $i, j \in \{1, 2\}$에 대하여 $i-j$의 값은 $-1, 0, 1$ 중 하나이므로 항상 $i-j \neq k$가 성립한다.
따라서 모든 성분 $a_{ij} = 0$이 되어야 한다.
즉, $A \in M_k$이면 $A = \left( \begin{array}{cc} 0 & 0 \\ 0 & 0 \end{array} \right) = O$ 뿐이다.
그러므로 $M_k = \{O\}$이다. (거짓)

ㄷ. $|k| < 2$이면 정수 $k$는 $0, 1, -1$이다.
(ⅰ) $k=0$이면 ㄱ에 의해서 참이다.
(ⅱ) $k=1$이면 $i-j \neq 1$인 모든 $a_{ij}=0$이므로 $a_{11}=a_{12}=a_{22}=0$이다.
$A = \left( \begin{array}{cc} 0 & 0 \\ a_{21} & 0 \end{array} \right)$, $B = \left( \begin{array}{cc} 0 & 0 \\ b_{21} & 0 \end{array} \right)$이므로 $AB = BA = O$이다.
(ⅲ) $k=-1$이면 $i-j \neq -1$인 모든 $a_{ij}=0$이므로 $a_{11}=a_{21}=a_{22}=0$이다.
$A = \left( \begin{array}{cc} 0 & a_{12} \\ 0 & 0 \end{array} \right)$, $B = \left( \begin{array}{cc} 0 & b_{12} \\ 0 & 0 \end{array} \right)$이므로 $AB = BA = O$이다.
$\therefore C(A, B) = O$ (참)

따라서 옳은 것은 ㄱ, ㄷ이다.
