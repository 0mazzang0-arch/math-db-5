## 문제
부등식 $log_{2} (x^{2}-x ) < 1- log_{\frac{1}{2}}x$를 만족시키는 모든 $x$의 값의 범위가 $\alpha < x < \beta$일 때, $\alpha + \beta$의 값은?

## 보기
① $2$   ② $4$   ③ $6$   ④ $8$   ⑤ $10$

## 풀이
$x^{2}-x$와 $x$는 로그의 진수이므로 
$x^{2}-x > 0$, $x > 0$에서
$x > 1$ $\cdots$ ㉠

부등식 $log_{2} (x^{2}-x ) < 1- log_{\frac{1}{2}}x$에서
$- log_{\frac{1}{2}}x = - log_{2^{-1}}x = log_{2}x$이므로
$log_{2} (x^{2}-x ) < log_{2}2 + log_{2}x$
$log_{2} (x^{2}-x ) < log_{2}2x$

밑 $2$가 $1$보다 크므로
$x^{2}-x < 2x$
$x^{2}-3x < 0$
$x (x-3 ) < 0$에서
$0 < x < 3$ $\cdots$ ㉡

㉠, ㉡에 의하여 구하는 모든 $x$의 값의 범위는
$1 < x < 3$
따라서 $\alpha = 1$, $\beta = 3$이므로
$\alpha + \beta = 1 + 3 = 4$
