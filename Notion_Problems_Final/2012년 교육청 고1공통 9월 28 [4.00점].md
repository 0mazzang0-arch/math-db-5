## 문제

> [!example] 📸 (그림/그래프)

그림과 같이 반지름의 길이가 $8$ 인 원 $O$ 의 내부에 반지름의 길이가 각각 $r_{1}$, $r_{2}$, $r_{3}$ 인 세 원 $O_{1}$, $O_{2}$, $O_{3}$ 이 있다. 네 원 $O$, $O_{1}$, $O_{2}$, $O_{3}$ 의 중심이 한 직선 위에 있고 원 $O_{1}$, $O_{3}$ 은 각각 원 $O$ 와 내접하며 원 $O_{2}$ 는 원 $O_{1}$, $O_{3}$ 과 동시에 외접한다. 원 $O_{1}$, $O_{2}$, $O_{3}$ 의 넓이의 합이 어두운 부분의 넓이와 같을 때, $r_{1} r_{2} + r_{2} r_{3} + r_{3} r_{1}$ 의 값을 구하시오. (단, $O_{1}$, $O_{2}$, $O_{3}$ 의 중심의 위치는 서로 다르다.)

## 풀이

$r_{1} + r_{2} + r_{3} = 8$ 이고, 주어진 그림의 어두운 부분과 $O_{1}$, $O_{2}$, $O_{3}$ 의 넓이의 합이 같으므로 $O_{1}$, $O_{2}$, $O_{3}$ 의 넓이의 합은 원 $O$ 의 넓이의 $\frac{1}{2}$ 이다. 따라서 

$\pi r_{1}^{2} + \pi r_{2}^{2} + \pi r_{3}^{2} = \frac{1}{2} \times 64 \pi$ 

$\therefore r_{1}^{2} + r_{2}^{2} + r_{3}^{2} = 32$ 

$\therefore r_{1} r_{2} + r_{2} r_{3} + r_{3} r_{1} = \frac{1}{2} \{ ( r_{1} + r_{2} + r_{3} )^{2} - ( r_{1}^{2} + r_{2}^{2} + r_{3}^{2} ) \}$ 

$= \frac{1}{2} ( 64 - 32 ) = 16$

정답: $16$
