## 문제
세 실수 $a$, $b$, $c$에 대하여 다항식 $P(x) = x^3 - ax^2 + bx - c$는 다음 조건을 만족시킨다.

(가) $2+i$는 삼차방정식 $P(x) = 0$의 근이다.
(나) $P(x)$를 일차식 $x-1$로 나눈 나머지는 $1$이다.

이때, $a+b+c$의 값을 구하시오. (단, $i = \sqrt{-1}$)

## 보기

## 풀이
조건 (가)에서 계수가 실수인 삼차방정식 $P(x) = 0$의 한 근이 $2+i$이므로 다른 한 근은 켤레복소수인 $2-i$이다.
$P(x) = 0$의 다른 한 근을 $\alpha$라 하면
$P(x) = x^3 - ax^2 + bx - c$
$= (x - (2+i))(x - (2-i))(x - \alpha)$
$= (x^2 - 4x + 5)(x - \alpha)$

조건 (나)에서 나머지 정리에 의해 $P(1) = 1$이므로
$P(1) = (1^2 - 4 \cdot 1 + 5)(1 - \alpha) = 2(1 - \alpha) = 1$
$1 - \alpha = \frac{1}{2}$
$\therefore \alpha = \frac{1}{2}$

따라서 $P(x) = (x^2 - 4x + 5)(x - \frac{1}{2})$이므로 $P(-1)$의 값을 구하면
$P(-1) = ((-1)^2 - 4(-1) + 5)(-1 - \frac{1}{2})$
$= (1 + 4 + 5)(-\frac{3}{2})$
$= 10 \cdot (-\frac{3}{2}) = -15$

한편, $P(x) = x^3 - ax^2 + bx - c$에 $x = -1$을 대입하면
$P(-1) = (-1)^3 - a(-1)^2 + b(-1) - c = -1 - a - b - c$
$-1 - a - b - c = -15$
$- (a + b + c) = -14$
$\therefore a + b + c = 14$

정답: $14$
