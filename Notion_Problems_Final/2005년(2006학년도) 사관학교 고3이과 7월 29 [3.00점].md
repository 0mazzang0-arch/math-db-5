## 문제
다항함수 $f(x)$에 대하여 
$$\lim_{x \to \infty} \frac{f(x)}{x^3 - 2x^2 + 3x - 4} = 1$$, 
$$\lim_{x \to 1} \frac{f(x)}{x^2 - 3x + 2} = 4$$
가 성립하고, 극한 $\lim_{x \to 2} \frac{13 f(x)}{x^2 - 3x + 2}$가 $\alpha$로 수렴할 때, 상수 $\alpha$의 값을 구하시오.

## 풀이
$\lim_{x \to \infty} \frac{f(x)}{x^3 - 2x^2 + 3x - 4} = 1$이므로 $f(x)$는 $x^3$ 항의 계수가 $1$인 삼차함수이다. 
또한 $\lim_{x \to 1} \frac{f(x)}{x^2 - 3x + 2} = 4$로부터 $f(x) = (x - 1)g(x) = (x - 1)(x^2 + ax + b)$로 나타낼 수 있다. 

$$\lim_{x \to 1} \frac{(x - 1)(x^2 + ax + b)}{x^2 - 3x + 2} = \lim_{x \to 1} \frac{x^2 + ax + b}{x - 2} = 4$$
이므로 $a + b = -5 \cdots\cdots (1)$ 

$\lim_{x \to 2} \frac{13f(x)}{x^2 - 3x + 2} = \lim_{x \to 2} \frac{13(x - 1)(x^2 + ax + b)}{(x - 1)(x - 2)} = \alpha$로 수렴하기 위해서 $x \to 2$일 때 분모 $\to 0$이므로 분자 $\to 0$이어야 한다. 따라서 
$4 + 2a + b = 0 \cdots\cdots (2)$ 

식 $(1)$과 $(2)$로부터 $a = 1$, $b = -6$이다. 
따라서 
$$\lim_{x \to 2} \frac{13(x - 1)(x^2 + x - 6)}{(x - 1)(x - 2)} = \lim_{x \to 2} \frac{13(x - 1)(x - 2)(x + 3)}{(x - 1)(x - 2)}$$ 
$$= \lim_{x \to 2} 13(x + 3) = 65$$

정답: $65$
