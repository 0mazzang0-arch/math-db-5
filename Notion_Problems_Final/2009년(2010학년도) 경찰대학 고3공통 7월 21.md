## 문제
수열 $\{ a_{n} \}$ 이 다음 세 조건을 만족시킨다.

(가) $a_{1} = \frac{1}{10}$
(나) 모든 자연수 $n$ 에 대하여 $a_{n} > 0$, $a_{n} \neq 1$
(다) 어떤 양수 $x$ ($x \neq 1$) 에 대하여 $\log_{a_{n}} x + \log_{a_{n+1}} x = \log x$

자연수 $n$ 에 대하여 $b_{n} = a_{1} a_{2} a_{3} a_{4} \cdots a_{2n-1} a_{2n}$ 으로 정의할 때, 무한급수 $\sum_{n=1}^{\infty} b_{n}$ 의 값은?

## 보기
① $\frac{1}{9}$   ② $\frac{\sqrt{10}-1}{9}$   ③ $\frac{\sqrt{10}}{9}$   ④ $\frac{4}{9}$   ⑤ $\frac{\sqrt{10}+1}{9}$

## 풀이
(다)에서 $\log_{a_{n}} x + \log_{a_{n+1}} x = \log x$ 이므로 좌변을 상용로그로 바꾸면 
$$\frac{\log x}{\log a_{n}} + \frac{\log x}{\log a_{n+1}} = \log x$$ 
$x \neq 1$ 이므로 양변을 $\log x$ 로 나누면 
$$\frac{1}{\log a_{n}} + \frac{1}{\log a_{n+1}} = 1$$ 
$$\frac{1}{\log a_{n+1}} = 1 - \frac{1}{\log a_{n}} \therefore \log a_{n+1} = \frac{\log a_{n}}{\log a_{n} - 1}$$ 
$a_{1} = \frac{1}{10}$ 이므로 
$$\log a_{2} = \frac{\log \frac{1}{10}}{\log \frac{1}{10} - 1} = \frac{-1}{-1-1} = \frac{-1}{-2} = \frac{1}{2} \therefore a_{2} = 10^{\frac{1}{2}}$$ 
$\log a_{2} = \frac{1}{2}$ 이므로 
$$\log a_{3} = \frac{\frac{1}{2}}{\frac{1}{2} - 1} = -1 \therefore a_{3} = 10^{-1} = a_{1}$$ 
따라서 $a_{1} = 10^{-1}$, $a_{2} = 10^{\frac{1}{2}}$, $a_{3} = 10^{-1}$, $a_{4} = 10^{\frac{1}{2}}, \cdots$ 이므로 
$$a_{2n-1} a_{2n} = 10^{-1} \cdot 10^{\frac{1}{2}} = 10^{-\frac{1}{2}} (n = 1, 2, 3, \cdots)$$ 
$$\therefore b_{n} = (a_{1} a_{2}) (a_{3} a_{4}) \cdots (a_{2n-1} a_{2n})$$ 
$$= (10^{-\frac{1}{2}}) (10^{-\frac{1}{2}}) \cdots (10^{-\frac{1}{2}}) = (10^{-\frac{1}{2}})^{n}$$ 
따라서 수열 $\{ b_{n} \}$ 은 첫째항이 $10^{-\frac{1}{2}}$, 공비가 $10^{-\frac{1}{2}}$ 인 등비수열이므로 
$$\sum_{n=1}^{\infty} b_{n} = \frac{10^{-\frac{1}{2}}}{1 - 10^{-\frac{1}{2}}} = \frac{\frac{1}{\sqrt{10}}}{1 - \frac{1}{\sqrt{10}}} = \frac{1}{\sqrt{10}-1} = \frac{\sqrt{10}+1}{9}$$ 
정답은 ⑤이다.
