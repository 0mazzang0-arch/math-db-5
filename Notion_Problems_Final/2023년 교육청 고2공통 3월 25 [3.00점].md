## 문제
$-5 \le x \le -1$ 에서 함수 $f(x) = \sqrt{-ax+1}$ ($a > 0$) 의 최댓값이 $4$ 가 되도록 하는 상수 $a$ 의 값을 구하시오.

## 보기

## 풀이
$a$ 가 양수이므로 $-5 \le x \le -1$ 에서 함수 $y = f(x)$ 의 그래프는 그림과 같다.

> [!example] 📸 (그림/그래프)

따라서 $f(x) = \sqrt{-ax+1}$ 은 $x = -5$ 일 때 최대이고 최댓값이 $4$ 이므로
$f(-5) = 4$
$\sqrt{5a+1} = 4$
$5a+1 = 16$
따라서 $a = 3$

**정답: $3$**
