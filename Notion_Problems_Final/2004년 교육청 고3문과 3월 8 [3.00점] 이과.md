## 문제
좌표평면 위의 점 $(x, y)$를 행렬 
$$ \left( \begin{array}{c} x \\ y \end{array} \right) $$
로 나타내기로 하자. 영역 $D = \{ (x, y) \mid 0 \leq x \leq 1, 0 \leq y \leq 1 \}$에 속하는 임의의 두 점 $(a, b)$, $(c, d)$에 대응하는 행렬 $A = \left( \begin{array}{c} a \\ b \end{array} \right)$, $B = \left( \begin{array}{c} c \\ d \end{array} \right)$에 대하여 $A+B = \left( \begin{array}{c} p \\ q \end{array} \right)$라 할 때, 점 $(p, q)$가 나타내는 영역의 넓이는?

## 보기
① $2$   ② $\frac{5}{2}$   ③ $3$   ④ $\frac{7}{2}$   ⑤ $4$

## 풀이
[정답] ⑤ 

$$ A+B = \left( \begin{array}{c} a+c \\ b+d \end{array} \right) = \left( \begin{array}{c} p \\ q \end{array} \right) $$ 

$0 \leq a \leq 1$, $0 \leq b \leq 1$, $0 \leq c \leq 1$, $0 \leq d \leq 1$이므로 
$0 \leq a+c \leq 2$, $0 \leq b+d \leq 2$이다.

따라서 $0 \leq p \leq 2$, $0 \leq q \leq 2$가 성립한다.
점 $(p, q)$가 나타내는 영역은 가로의 길이가 $2$이고 세로의 길이가 $2$인 정사각형이 되므로 구하는 영역의 넓이는 $4$이다.
