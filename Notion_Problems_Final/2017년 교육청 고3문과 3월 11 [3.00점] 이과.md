## 문제
첫째항이 양수인 등비수열 $\{a_{n}\}$이
$a_{1} = 4 a_{3}$, $a_{2} + a_{3} = -12$
를 만족시킬 때, $a_{5}$의 값은?

## 보기
① $3$   ② $4$   ③ $5$   ④ $6$   ⑤ $7$

## 풀이
등비수열 $\{a_{n}\}$의 첫째항을 $a$, 공비를 $r$라 하면
$a_{2} + a_{3} = ar + ar^{2} = a(r + r^{2}) = -12$ …… ㉠
이고, 첫째항이 양수이므로 $a > 0$이다.
$a(r + r^{2}) = -12$에서 $r + r^{2} = r(1 + r) < 0$이므로
$-1 < r < 0$이다.
한편, $a_{1} = 4 a_{3}$에서
$a = 4 a r^{2}$
$r^{2} = \frac{1}{4}$
$-1 < r < 0$이므로 $r = -\frac{1}{2}$이다.
㉠에 $r = -\frac{1}{2}$을 대입하면
$a \left( -\frac{1}{2} + \frac{1}{4} \right) = -12$
$a \left( -\frac{1}{4} \right) = -12$
$a = 48$
따라서 $a_{5}$의 값은
$a_{5} = a r^{4} = 48 \times \left( -\frac{1}{2} \right)^{4} = 48 \times \frac{1}{16} = 3$
정답은 ①이다.
