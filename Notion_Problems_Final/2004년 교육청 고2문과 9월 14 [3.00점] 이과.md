## 문제
기약분수 $\frac{m}{n}$이 소수점 아래 셋째 자리까지의 유한소수라 할 때, $\frac{m}{n}$을 (가)배 하면 정수가 된다. 이 정수를 $k$라 하면 $\frac{m}{n} \times$ (가) $= k$이다. 이때, $m$과 $n$은 서로소이므로, $n$은 (가)의 약수이다. 따라서 $n$을 소인수분해하면 $n = $ (나)와 같이 된다. $\cdots$ (이하 생략) $\cdots$
위의 과정에서 (가), (나)에 알맞은 것은?

## 보기
① (가) $m$, (나) $5^{\alpha} (\alpha = 0, 1, 2, 3)$   ② (가) $m$, (나) $2^{\alpha} \cdot 5^{\beta} (\alpha = 0, 1, 2, 3, \beta = 0, 1, 2, 3)$   ③ (가) $10^{3}$, (나) $2^{\alpha} (\alpha = 0, 1, 2, 3)$   ④ (가) $10^{3}$, (나) $5^{\alpha} (\alpha = 0, 1, 2, 3)$   ⑤ (가) $10^{3}$, (나) $2^{\alpha} \cdot 5^{\beta} (\alpha = 0, 1, 2, 3, \beta = 0, 1, 2, 3)$

## 풀이
$\frac{m}{n}$이 소수점 아래 셋째 자리까지의 유한소수이므로, $10^{3}$배를 하면 정수가 된다. $n$은 $10^{3} = 2^{3} \cdot 5^{3}$의 약수이므로, 소인수분해 하면 $2^{\alpha} \cdot 5^{\beta} (\alpha = 0, 1, 2, 3, \beta = 0, 1, 2, 3)$과 같다. 다음은 $0 < m < n$인 자연수 $m$, $n$에 대하여 기약분수 $\frac{m}{n}$이 소수점 아래 셋째 자리까지의 유한소수가 되기 위한 조건을 구하는 과정의 일부이다.
