## 문제
확률변수 $X$가 정규분포 $N(5, 3^{2})$을 따를 때, $P(|X-5| \le 3) = 0.6826$이다. 확률변수 $Y$를 $Y=2X+1$이라 할 때, $P(Y \ge 17)$의 값은?

## 보기
① $0.1037$   ② $0.1587$   ③ $0.3174$   ④ $0.3413$   ⑤ $0.6826$

## 풀이
확률변수 $X$가 정규분포 $N(5, 3^{2})$을 따르므로 $m=5$, $\sigma=3$이다.
$P(|X-5| \le 3) = P(-3 \le X-5 \le 3)$
$= P(2 \le X \le 8)$
$= P(m-\sigma \le X \le m+\sigma)$
$= 2P(m \le X \le m+\sigma) = 0.6826$
$\therefore P(m \le X \le m+\sigma) = 0.3413$

따라서, 확률변수 $Y = 2X+1$에 대하여
$P(Y \ge 17) = P(2X+1 \ge 17)$
$= P(2X \ge 16)$
$= P(X \ge 8)$
$= P(X \ge m+\sigma)$
$= \frac{1}{2} - P(m \le X \le m+\sigma)$
$= 0.5 - 0.3413$
$= 0.1587$

정답: ②
