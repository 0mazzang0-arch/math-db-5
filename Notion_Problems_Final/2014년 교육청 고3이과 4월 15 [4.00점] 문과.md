## 문제
$$\int_{e^{2}}^{e^{3}} \frac{a + \ln x}{x} dx = \int_{0}^{\frac{\pi}{2}} (1 + \sin x) \cos x dx$$가 성립할 때, 상수 $a$의 값은?

## 보기
① $-2$   ② $-1$   ③ $0$   ④ $1$   ⑤ $2$

## 풀이
주어진 등식의 좌변에서 $\ln x = s$라 하면 
$$\int_{e^{2}}^{e^{3}} \frac{a + \ln x}{x} dx = \int_{2}^{3} (a + s) ds$$ 
$$= \left[ as + \frac{1}{2} s^{2} \right]_{2}^{3} = a + \frac{5}{2}$$ 
주어진 등식의 우변에서 $\sin x = t$라 하면 
$$\int_{0}^{\frac{\pi}{2}} (1 + \sin x) \cos x dx = \int_{0}^{1} (1 + t) dt$$ 
$$= \left[ t + \frac{1}{2} t^{2} \right]_{0}^{1} = \frac{3}{2}$$ 
$$a + \frac{5}{2} = \frac{3}{2} \text{ 이므로 } a = -1$$ 
따라서 $a$의 값은 $-1$이다.

정답: ②
