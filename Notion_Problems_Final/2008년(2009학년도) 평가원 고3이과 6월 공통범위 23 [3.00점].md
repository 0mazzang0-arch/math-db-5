## 문제
모든 계수가 정수인 삼차함수 $y=f(x)$는 다음 조건을 만족시킨다.

(가) 모든 실수 $x$에 대하여 $f(-x)=-f(x)$이다.
(나) $f(1)=5$
(다) $1 < f'(1) < 7$

함수 $y=f(x)$의 극댓값은 $m$이다. $m^{2}$의 값을 구하시오.

## 보기


## 풀이
$f(x) = ax^{3} + bx^{2} + cx + d$ ($a$, $b$, $c$, $d$는 정수)에서 기함수 조건(∵ (가)) 때문에
$b = d = 0$
따라서 $f(x) = ax^{3} + cx$이다.
(나)에서 $f(1) = a + c = 5 \cdots \cdots$ ㉠
또한 (다)에서 $1 < f'(1) < 7$이므로
$1 < 3a + c < 7$
㉠에서 $c = 5 - a$로 두면
$1 < 3a + 5 - a < 7$
$-4 < 2a < 2$
$-2 < a < 1$
따라서 정수 $a$는 $-1$과 $0$인데, $f(x)$가 삼차함수이므로 $a = -1$이다.
또한 $c = 6$
따라서 $f(x) = -x^{3} + 6x$이다.
미분하면 $f'(x) = -3x^{2} + 6 = 0$
$x^{2} = 2$이므로 $x = \sqrt{2}$에서 극댓값 $m$을 가진다.
$m^{2} = \{ f(\sqrt{2}) \}^{2}$
$= (4\sqrt{2})^{2}$
$= 32$

정답: $32$
