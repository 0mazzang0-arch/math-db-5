## 문제
전체집합 $U = \{ x | x \text{는 } 20 \text{ 이하의 자연수} \}$의 두 부분집합 $A = \{ 1, 3, a - 1 \}$, $B = \{ a^{2} - 4 a - 7, a + 2 \}$에 대하여 $( A \cap B^{C} ) \cup ( A^{C} \cap B ) = \{ 1, 3, 8 \}$일 때, 상수 $a$의 값은?

## 보기
① $5$   ② $6$   ③ $7$   ④ $8$   ⑤ $9$

## 풀이
$( A \cap B^{C} ) \cup ( A^{C} \cap B ) = ( A - B ) \cup ( B - A )$이고 벤 다이어그램으로 나타내면 그림과 같다. 

> [!example] 📸 (그림/그래프)

$( A - B ) \cup ( B - A )$의 원소가 $1$, $3$, $8$이고 $n( A ) = 3$, $n( B ) = 2$이므로 $n( A \cap B ) = 1$이어야 한다. 
$\therefore a - 1 \in A \cap B$ 

$a - 1$은 $B$의 원소 중 어느 하나와 같아야 한다. 
(i) $a - 1 \neq a + 2$ 
(ii) $a - 1 = a^{2} - 4 a - 7$이고 $a + 2 = 8$일 때, $a = 6$ 
(i), (ii)에 의하여 $a = 6$
