## 문제
전체집합 $U$가 실수 전체의 집합일 때, 실수 $x$에 대한 두 조건 $p$, $q$가
$p: a(x-1)(x-2)<0$, $q: x>b$
이다. 두 조건 $p$, $q$의 진리집합을 각각 $P$, $Q$라 할 때, <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $a$, $b$는 실수이다.)

$$ \begin{array}{|l|} \hline \text{ㄱ. } a=0 \text{ 일 때, } P = \emptyset \text{ 이다.} \\ \text{ㄴ. } a>0, b=0 \text{ 일 때, } P \subset Q \text{ 이다.} \\ \text{ㄷ. } a<0, b=3 \text{ 일 때, 명제 } \sim p \text{ 이면 } q \text{ 이다. 는 참이다.} \\ \hline \end{array} $$

## 보기
① ㄱ   ② ㄱ, ㄴ   ③ ㄱ, ㄷ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
ㄱ. $a=0$ 이면 $0 \times (x-1)(x-2)<0$ 이 되어 이 부등식을 만족시키는 실수 $x$는 존재하지 않으므로 $P = \emptyset$ 이다.

ㄴ. $a>0$, $b=0$ 이면 조건 $p$의 진리집합은 $P = \{ x | 1 < x < 2 \}$, 조건 $q$의 진리집합은 $Q = \{ x | x > 0 \}$ 이다.
$\therefore P \subset Q$

ㄷ. $a<0$, $b=3$ 이면 조건 $p$의 진리집합은 $P = \{ x | x < 1 \text{ 또는 } x > 2 \}$ 이다.
조건 $\sim p$의 진리집합은 $P^c = \{ x | 1 \le x \le 2 \}$ 이다.
조건 $q$의 진리집합은 $Q = \{ x | x > 3 \}$ 이다.
$\therefore P^c \not\subset Q$
즉 명제 $\sim p$ 이면 $q$ 이다. 는 거짓이다.

이상에서 옳은 것은 ㄱ, ㄴ이다.
