## 문제
세 실수 $a$, $b$, $c$ ($a < b < c$)에 대하여 $x$에 대한 연립부등식 
$$ \begin{cases} (x-a)(x-b)>0 \\ (x-b)(x-c)>0 \end{cases} $$
의 해가 $x < -4$, $x > 3$일 때, 이차부등식 $x^2 + ax + c < 0$의 해는?



> [!example] 📸 (그림/그래프)

## 보기
① $-3 < x < 4$   ② $3 < x < 4$   ③ $-3 < x < 1$   ④ $-1 < x < 3$   ⑤ $1 < x < 3$

## 풀이
$a < b < c$이므로 
$(x - a)(x - b) > 0$ 의 해는 $x < a$ 또는 $x > b$
$(x - b)(x - c) > 0$ 의 해는 $x < b$ 또는 $x > c$

따라서, 연립부등식 
$$ \begin{cases} (x-a)(x-b)>0 \\ (x-b)(x-c)>0 \end{cases} $$
의 해는 공통 범위를 구하면 $x < a$ 또는 $x > c$이다.

문제에서 해가 $x < -4$, $x > 3$이라고 하였으므로
$\therefore a = -4$, $c = 3$

이 값을 이차부등식 $x^2 + ax + c < 0$에 대입하면
$x^2 - 4x + 3 < 0$
$(x - 1)(x - 3) < 0$

$\therefore 1 < x < 3$

정답: ⑤
