## 문제
급수 $\sum_{n=1}^{\infty} (x-1)(\log_{2} x)^{n}$ 이 수렴할 때, <보기>에서 옳은 것을 모두 고른 것은?

## 보기
ㄱ. 수렴하기 위한 $x$ 값의 범위는 $\frac{1}{2} < x < 2$ 이다.
ㄴ. 급수의 합이 $1$ 이 되도록 하는 $x$ 의 값은 한 개 존재한다.
ㄷ. 급수 $\sum_{n=1}^{\infty} \left( \frac{\log_{2} x - 1}{2} \right)^{n}$ 은 수렴한다.

① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
ㄱ. 급수 $\sum_{n=1}^{\infty} (x-1)(\log_{2} x)^{n}$ 이 수렴하려면 첫째항이 $0$ 이거나 공비 $r$ 이 $-1 < r < 1$ 이어야 한다.
첫째항: $(x-1)\log_{2} x = 0 \Rightarrow x=1$
공비: $-1 < \log_{2} x < 1 \Rightarrow \frac{1}{2} < x < 2$
따라서 수렴하기 위한 $x$ 의 범위는 $\frac{1}{2} < x < 2$ 이다. (참)

ㄴ. 급수의 합이 $1$ 이므로
$\frac{(x-1)\log_{2} x}{1-\log_{2} x} = 1$
$(x-1)\log_{2} x = 1 - \log_{2} x$
$x \log_{2} x - \log_{2} x = 1 - \log_{2} x$
$x \log_{2} x = 1 \Rightarrow \log_{2} x = \frac{1}{x}$
$y = \log_{2} x$ 와 $y = \frac{1}{x}$ 의 그래프는 $\frac{1}{2} < x < 2$ 인 범위 내에서 반드시 한 개의 교점이 생긴다. (참)

ㄷ. ㄱ에서 구한 범위 $\frac{1}{2} < x < 2$ 에 의해
$-1 < \log_{2} x < 1$
$-2 < \log_{2} x - 1 < 0$
$-1 < \frac{\log_{2} x - 1}{2} < 0$
따라서 급수 $\sum_{n=1}^{\infty} \left( \frac{\log_{2} x - 1}{2} \right)^{n}$ 의 공비 $r'$ 이 $-1 < r' < 0$ 이므로 이 급수는 수렴한다. (참)

정답: ⑤
