## 문제
$a > 0$, $b > 0$, $c > 0$일 때, 보기에서 항상 성립하는 부등식을 모두 고른 것은?

ㄱ. $\frac{1}{a} + \frac{1}{b} \geq \frac{4}{a+b}$
ㄴ. $\sqrt{a} + \sqrt{b} > \sqrt{a+b}$
ㄷ. $a + b + c \geq \sqrt{ab} + \sqrt{bc} + \sqrt{ca}$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄷ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
[정답] ⑤

ㄱ. $\frac{1}{a} + \frac{1}{b} - \frac{4}{a+b} = \frac{a+b}{ab} - \frac{4}{a+b} = \frac{(a+b)^{2} - 4ab}{ab(a+b)} = \frac{(a-b)^{2}}{ab(a+b)} \geq 0$ (등호는 $a=b$ 일 때 성립) (참)

ㄴ. $(\sqrt{a} + \sqrt{b})^{2} - (\sqrt{a+b})^{2} = a + b + 2\sqrt{ab} - (a+b) = 2\sqrt{ab} > 0$
$\therefore \sqrt{a} + \sqrt{b} > \sqrt{a+b}$ (참)

ㄷ. $a + b \geq 2\sqrt{ab}$, $b + c \geq 2\sqrt{bc}$, $c + a \geq 2\sqrt{ca}$
$2(a + b + c) \geq 2(\sqrt{ab} + \sqrt{bc} + \sqrt{ca})$
$\therefore a + b + c \geq \sqrt{ab} + \sqrt{bc} + \sqrt{ca}$ (등호는 $a = b = c$ 일 때 성립) (참)

따라서 옳은 것은 ㄱ, ㄴ, ㄷ 이다.
