## 문제
등차수열 { $a_{n}$ } 에 대하여 ($a_{1} + a_{2}$) : ($a_{3} + a_{4}$) = $1 : 2$ 가 성립할 때, $a_{1} : a_{4}$ 는? (단, $a_{1} \neq 0$ 이다.)

## 보기
① $1 : 2$   ② $1 : 3$   ③ $2 : 3$   ④ $2 : 5$   ⑤ $3 : 5$

## 풀이
[정답] ②

등차수열 { $a_{n}$ } 의 공차를 $d$ 라 하면
($a_{1} + a_{2}$) : ($a_{3} + a_{4}$) = ($a_{1} + a_{1} + d$) : ($a_{1} + 2d + a_{1} + 3d$) = $1 : 2$ 에서
$2a_{1} + 5d = 4a_{1} + 2d$ $\therefore 2a_{1} = 3d$
$\therefore a_{1} : a_{4} = a_{1} : (a_{1} + 3d) = a_{1} : 3a_{1} = 1 : 3$
