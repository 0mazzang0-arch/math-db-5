## 문제
부등식 $|x + 1| + |x - 2| < 5$를 만족시키는 정수 $x$의 개수를 구하시오.

## 보기

## 풀이
(i) $x < -1$ 일 때,
$-2x + 1 < 5$ 에서
$-2 < x < -1$

(ii) $-1 \le x < 2$ 일 때,
$x + 1 - x + 2 < 5$ 가 항상 성립하므로
$-1 \le x < 2$

(iii) $x \ge 2$ 일 때,
$2x - 1 < 5$ 에서
$2 \le x < 3$

(i), (ii), (iii)에 의하여 부등식 $|x + 1| + |x - 2| < 5$ 의 해는 $-2 < x < 3$ 이다.
따라서 정수 $x$ 는 $-1, 0, 1, 2$ 이므로 개수는 $4$ 이다.

정답: $4$
