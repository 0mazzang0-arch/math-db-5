## 문제
두 상수 $a$, $b$에 대하여 실수 전체의 집합에서 미분가능한 함수 $f(x)$가 다음 조건을 만족시킨다.

(가) $0 \le x < 4$일 때, $f(x) = x^3 + ax^2 + bx$이다.  
(나) 모든 실수 $x$에 대하여 $f(x+4) = f(x) + 16$이다.  

$\int_{4}^{7} f(x) dx$의 값은?



> [!example] 📸 (그림/그래프)

## 보기
① $\frac{255}{4}$   ② $\frac{261}{4}$   ③ $\frac{267}{4}$   ④ $\frac{273}{4}$   ⑤ $\frac{279}{4}$

## 풀이
함수 $f(x)$가 실수 전체의 집합에서 미분가능하므로 실수 전체의 집합에서 연속이다.
함수 $f(x)$가 $x=4$에서 연속이므로
$\lim_{x \to 4^+} f(x) = \lim_{x \to 4^-} f(x) = f(4)$

$4 \le x < 8$에서의 함수 $y=f(x)$의 그래프는 $0 \le x < 4$에서의 함수 $y=f(x)$의 그래프를 $x$축의 방향으로 $4$만큼, $y$축의 방향으로 $16$만큼 평행이동한 그래프와 일치한다.
모든 실수 $x$에 대하여 $f(x+4) = f(x) + 16$이므로
$\lim_{x \to 4^+} f(x) = \lim_{x \to 0^+} f(x+4) = \lim_{x \to 0^+} \{f(x) + 16\} = 0 + 16 = 16$
$\lim_{x \to 4^-} f(x) = \lim_{x \to 4^-} (x^3 + ax^2 + bx) = 64 + 16a + 4b$
$f(4) = f(0) + 16 = 16$
$16 = 64 + 16a + 4b$ 에서 $b = -4a - 12$

함수 $f(x)$가 $x=4$에서 미분가능하므로
$\lim_{x \to 4^+} \frac{f(x) - f(4)}{x - 4} = \lim_{x \to 4^-} \frac{f(x) - f(4)}{x - 4}$
$\lim_{x \to 4^+} \frac{f(x) - f(4)}{x - 4} = \lim_{x \to 0^+} \frac{f(x+4) - f(4)}{x} = \lim_{x \to 0^+} \frac{\{f(x) + 16\} - 16}{x} = \lim_{x \to 0^+} \frac{x^3 + ax^2 + bx}{x} = \lim_{x \to 0^+} (x^2 + ax + b) = b$
$\lim_{x \to 4^-} \frac{f(x) - f(4)}{x - 4} = \lim_{x \to 4^-} \frac{(x^3 + ax^2 + bx) - 16}{x - 4} = \lim_{x \to 4^-} \frac{ax(x - 4) + (x - 4)(x^2 + 4x + 4)}{x - 4} = \lim_{x \to 4^-} \{x^2 + (a + 4)x + 4\} = 16 + 4(a + 4) + 4 = 4a + 36$
따라서 $b = 4a + 36$
위에서 구한 $b = -4a - 12$를 대입하면
$-4a - 12 = 4a + 36$
$8a = -48 \Rightarrow a = -6$, $b = 12$

그러므로 $0 \le x < 4$에서 $f(x) = x^3 - 6x^2 + 12x$이다.
조건 (나)에 의해 $\int_{4}^{7} f(x) dx$를 구하면, $x = t + 4$로 치환할 때 $dx = dt$이고 $x$가 $4$에서 $7$까지 변할 때 $t$는 $0$에서 $3$까지 변하므로
$\int_{4}^{7} f(x) dx = \int_{0}^{3} f(t+4) dt = \int_{0}^{3} \{f(t) + 16\} dt$
$= \int_{0}^{3} (t^3 - 6t^2 + 12t + 16) dt$
$= \left[ \frac{1}{4}t^4 - 2t^3 + 6t^2 + 16t \right]_{0}^{3}$
$= \frac{81}{4} - 2(27) + 6(9) + 16(3)$
$= \frac{81}{4} - 54 + 54 + 48 = \frac{81}{4} + 48 = \frac{273}{4}$
