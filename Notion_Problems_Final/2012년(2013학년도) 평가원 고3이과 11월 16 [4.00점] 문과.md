## 문제
두 이차정사각행렬 $A$, $B$가
$$ 2A^2 + AB = E, AB + BA = 2A + E $$
를 만족시킬 때, <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $E$는 단위행렬이다.)

$$ \begin{array}{|l|} \hline \text{<보 기>} \\ \text{ㄱ. } A^{-1} = 2A + B \\ \text{ㄴ. } B = 2A + 2E \\ \text{ㄷ. } (B - E)^2 = O \text{ (단, } O \text{는 영행렬이다.)} \\ \hline \end{array} $$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄱ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
[정답] ③ 

ㄱ. $2A^2 + AB = E$ 에서 
$A(2A + B) = E$ 
$\therefore A^{-1} = 2A + B$ (참) 

ㄴ. $2A^2 + AB = E$ 에서 
$AB = E - 2A^2$ 

$AB + BA = 2A + E$ 에서 
$AB = 2A + E - BA$ 이므로 

$E - 2A^2 = 2A + E - BA$ 

$-2A^2 = 2A - BA$ 

$BA = 2A^2 + 2A$ 

$\therefore B = (2A^2 + 2A)A^{-1} = 2A + 2E$ (참) 

ㄷ. $2A^2 + AB = E$ 에서 
$2A^2 + A(2A + 2E) = E$ 

$\therefore 4A^2 = -2A + E$ 

따라서 
$(B - E)^2$ 
$= (2A + E)^2$ 
$= 4A^2 + 4A + E$ 
$= (-2A + E) + 4A + E$
$= 2A + 2E = B$ 

그런데, $B = O$ 이면 주어진 조건에서 
$2A^2 = E$ 이고, $2A = -E$ ($A = -\frac{1}{2}E$) 이므로 
$2(-\frac{1}{2}E)^2 = \frac{1}{2}E = E$ 가 되어 모순이다. 
즉, $B \neq O$ 이므로 
$(B - E)^2 \neq O$ (거짓) 

따라서 옳은 것은 $ㄱ, ㄴ$이다.
