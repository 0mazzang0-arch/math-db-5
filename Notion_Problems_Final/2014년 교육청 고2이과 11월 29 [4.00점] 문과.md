## 문제
양수 $x$에 대하여 $\log x$의 정수부분과 소수부분을 각각 $f(x)$, $g(x)$라 하자. 다음 조건을 만족시키는 두 양수 $a$, $b$에 대하여 $ab=10^{\frac{n}{m}}$일 때, $m+n$의 값을 구하시오. (단, $m$, $n$은 서로소인 자연수이다.)

$$ \begin{array}{l}
(\text{가}) \log \frac{a}{b} = 1 \\
(\text{나}) g(a) = 4g \left( \frac{1}{a} \right) \\
(\text{다}) f(a) + g(b^2) = \frac{18}{5}
\end{array} $$

## 보기

## 풀이
$\log a = f(a) + g(a)$ ($f(a)$는 정수, $0 \le g(a) < 1$)
$\log b = f(b) + g(b)$ ($f(b)$는 정수, $0 \le g(b) < 1$)

(가)에 의하여 $\log a - \log b = 1$이므로 $g(a) = g(b)$이다.
$\therefore f(a) - f(b) = 1$, $g(a) = g(b) \cdots$ ㉠

(나)에서
(i) $g(a) = 0$일 때, $g \left( \frac{1}{a} \right) = 0$이므로 (나)를 만족한다.
(ii) $0 < g(a) < 1$일 때, $g \left( \frac{1}{a} \right) = 1 - g(a)$이므로
$g(a) = 4 \{ 1 - g(a) \}$, $g(a) = \frac{4}{5}$

$\therefore g(a) = 0$ 또는 $g(a) = \frac{4}{5} \cdots$ ㉡

(다)에 의하여 $f(a) = 3$이고 $g(b^2) = \frac{3}{5}$
(i) $0 \le g(b) < \frac{1}{2}$일 때,
$g(b^2) = 2g(b)$이므로 $g(b) = \frac{3}{10}$
(ii) $\frac{1}{2} \le g(b) < 1$일 때,
$g(b^2) = 2g(b) - 1$이므로 $g(b) = \frac{4}{5}$

$\therefore g(b) = \frac{3}{10}$ 또는 $g(b) = \frac{4}{5} \cdots$ ㉢

㉠, ㉡, ㉢에서
$f(a) = 3$, $f(b) = 2$, $g(a) = g(b) = \frac{4}{5}$

$\therefore a = 10^{\frac{19}{5}}$, $b = 10^{\frac{14}{5}}$
따라서 $ab = 10^{\frac{33}{5}}$이므로 $m + n = 38$

정답: $38$
