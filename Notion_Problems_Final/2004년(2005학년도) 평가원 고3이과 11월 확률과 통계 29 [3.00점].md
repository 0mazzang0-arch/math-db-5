## 문제
자연수 $n$ 에 대하여 등식 
$${}_{n} C _{n} + {}_{n+1} C _{n} + {}_{n+2} C _{n} = {}_{n+3} C _{n+1}$$ 
이 성립함을 다음과 같이 증명하였다.

$$
\begin{array}{|l|}
\hline \\
{}_{n+3} C _{n+1} \text{ 은 집합 } A = \{ 1, 2, \dots, n+3 \} \text{ 의 부분집합 중에서 원소의 개수가 } n+1 \text{ 인 부분집합의 개수이다.} \\
\text{이것을 다른 방법으로 세어보자.} \\
\\
(i) \text{ 집합 } A \text{ 의 부분집합 중에서, 가장 큰 원소가 (가)이고 원소의 개수가 } n+1 \text{ 인 부분집합의 개수는 } {}_{n} C _{n} \text{ 이다.} \\
(ii) \text{ 집합 } A \text{ 의 부분집합 중에서, 가장 큰 원소가 (나)이고 원소의 개수가 } n+1 \text{ 인 부분집합의 개수는 } {}_{n+1} C _{n} \text{ 이다.} \\
(iii) \text{ 집합 } A \text{ 의 부분집합 중에서, 가장 큰 원소가 (다)이고 원소의 개수가 } n+1 \text{ 인 부분집합의 개수는 } {}_{n+2} C _{n} \text{ 이다.} \\
\\
(i), (ii), (iii) \text{ 중에서 한 가지 경우만 일어날 수 있으므로 합의 법칙에 의하여} \\
{}_{n} C _{n} + {}_{n+1} C _{n} + {}_{n+2} C _{n} = {}_{n+3} C _{n+1} \text{ 이 성립한다.} \\ \\
\hline
\end{array}
$$

위의 과정에서 (가), (나), (다)에 알맞은 것은?

## 보기
① $(가): n-1, (나): n, (다): n+1$ ② $(가): n, (나): n+1, (다): n+2$ ③ $(가): n+1, (나): n+2, (다): n+3$ ④ $(가): n+2, (나): n+1, (다): n$ ⑤ $(가): n+3, (나): n+2, (다): n+1$

## 풀이
$(i)$ 집합 $A$ 의 부분집합 중에서, 가장 큰 원소가 $n+1$ 이고 원소의 개수가 $n+1$ 인 부분집합의 개수를 구하면, 가장 큰 원소가 $n+1$ 이 되어야 하므로, $n+1$ 은 반드시 부분집합의 원소로 선택하고, $n+2, n+3$ 은 선택해서는 안 된다. 그리고, $n+1$ 보다 작은 $n$ 개의 원소 중에서 $n$ 개를 고르면 된다.
따라서, ${}_{n} C _{n}$

$(ii)$ 집합 $A$ 의 부분집합 중에서, 가장 큰 원소가 $n+2$ 이고 원소의 개수가 $n+1$ 인 부분집합의 개수를 구하면, 가장 큰 원소가 $n+2$ 이 되어야 하므로, $n+2$ 는 반드시 부분집합의 원소로 선택하고, $n+3$ 은 선택해서는 안 된다. 그리고, $n+2$ 보다 작은 $n+1$ 개의 원소 중에서 $n$ 개를 고르면 된다.
따라서, ${}_{n+1} C _{n}$

$(iii)$ 집합 $A$ 의 부분집합 중에서, 가장 큰 원소가 $n+3$ 이고 원소의 개수가 $n+1$ 인 부분집합의 개수를 구하면, 가장 큰 원소가 $n+3$ 이 되어야 하므로, $n+3$ 은 반드시 부분집합의 원소로 선택되어야 한다. 그리고, $n+3$ 보다 작은 $n+2$ 개의 원소 중에서 $n$ 개를 고르면 된다.
따라서, ${}_{n+2} C _{n}$

위의 과정에 의하여
$(가) = n+1$, $(나) = n+2$, $(다) = n+3$
정답은 ③이다.
