## 문제
양의 실수 전체의 집합에서 정의된 두 함수 $f(x)$, $g(x)$가 다음 세 조건을 만족한다.

(가) $f(x)$의 값은 정수이다.
(나) $0 \leq g(x) < 1$
(다) $10^{f(x)+g(x)}=x$

$a=f(1)+f(11)+f(111)$일 때, $g(a)$의 값은?

## 보기
① $0$   ② $\log 2$   ③ $\log \sqrt{5}$   ④ $\log 3$   ⑤ $\frac{1}{2}$

## 풀이
$f(x)$는 정수, $0 \leq g(x) < 1$이고, $\log x = f(x) + g(x)$이므로 $f(x)$, $g(x)$는 각각 $\log x$의 정수부분과 소수부분이다.

$1 = 10^{0}$이므로 $0 \leq \log 1 < 1$이다.
즉, $f(1) = 0$이다.

$10^{1} < 11 < 10^{2}$이므로 $1 < \log 11 < 2$이다.
즉, $f(11) = 1$이다.

$10^{2} < 111 < 10^{3}$이므로 $2 < \log 111 < 3$이다.
즉, $f(111) = 2$이다.

$a = f(1) + f(11) + f(111) = 0 + 1 + 2 = 3$

$1 \leq a < 10$이므로 $0 \leq \log a < 1$이다. 
즉, $f(a) = 0$이다.

$\log 3 = 0 + g(a)$
따라서 $g(a) = \log 3$

**[다른 풀이]**
(다)의 식에 $x=1$을 대입하면 $10^{f(1)+g(1)} = 1 = 10^{0}$에서 $f(1)+g(1)=0$이다.
따라서 $f(1)=0$이다.

(다)의 식에 $x=11$을 대입하면 $10^{f(11)+g(11)} = 11 = 10^{1.\dots}$에서 $f(11)+g(11)=1.\dots$이다.
따라서 $f(11)=1$이다.

(다)의 식에 $x=111$을 대입하면 $10^{f(111)+g(111)} = 111 = 10^{2.\dots}$에서 $f(111)+g(111)=2.\dots$이다.
따라서 $f(111)=2$이다.

$a = f(1)+f(11)+f(111) = 0+1+2 = 3$

(다)의 식에 $x=3$을 대입하면 $10^{f(3)+g(3)} = 3 = 10^{0.\dots}$에서 $f(3)+g(3)=0.\dots$이다.
$f(3)=0$이므로 $f(a)=0$이다.

(다)의 식에 $x=a$를 대입하면 $10^{f(a)+g(a)} = a$이다.
$10^{g(a)} = a = 3$
따라서 구하는 값은 $g(a) = \log 3$이다.

정답: ④
