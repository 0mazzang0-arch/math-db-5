## 문제
두 수 $2^{n}$ 과 $5^{n}$ 의 최고 자릿수가 $a$ 로 같아지도록 하는 자연수 $n$ 과 $a$ 에 대하여 옳은 것만을 <보기>에서 있는 대로 고른 것은?

$$ \begin{array}{|l|} \hline \text{<보기>} \\ \text{ㄱ. } a \cdot 10^{p} < 2^{n} < (a + 1) \cdot 10^{p} \text{ 인 자연수 } p \text{ 가 있다.} \\ \text{ㄴ. } a^{2} < 10^{r} < (a + 1)^{2} \text{ 인 자연수 } r \text{ 가 있다.} \\ \text{ㄷ. } a \text{ 의 값이 } 7 \text{ 이 되도록 하는 } n \text{ 이 있다.} \\ \hline \end{array} $$

## 보기
① $ㄱ$   ② $ㄱ, ㄴ$   ③ $ㄱ, ㄷ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
$2^{1} = 2$, $2^{2} = 4$, $2^{3} = 8$, $2^{4} = 16$, $\cdots$
$5^{1} = 5$, $5^{2} = 25$, $5^{3} = 125$, $5^{4} = 625, \cdots$
이므로 자연수 $n$ 은 $4$ 보다 큰 자연수임을 알 수 있다.

ㄱ. $2^{n}$ 의 최고 자리의 숫자가 $a$ 이고 $2^{n}$ 이 두 자리 이상의 자연수이므로
$a \cdot 10^{p} < 2^{n} < (a + 1) \cdot 10^{p} \cdots$ ㉠
을 만족하는 자연수 $p$ 가 존재한다. (참)

ㄴ. $5^{n}$ 의 최고자리의 숫자가 $a$ 이고 $5^{n}$ 도 두 자리 이상의 자연수이므로
$a \cdot 10^{q} < 5^{n} < (a + 1) \cdot 10^{q} \cdots$ ㉡
을 만족하는 자연수 $q$ 가 존재한다.
㉠과 ㉡을 변끼리 곱하면
$a^{2} \cdot 10^{p+q} < 2^{n} \cdot 5^{n} < (a + 1)^{2} \cdot 10^{p+q}$
$a^{2} \cdot 10^{p+q} < 10^{n} < (a + 1)^{2} \cdot 10^{p+q}$
$a^{2} < 10^{n-p-q} < (a + 1)^{2}$
$n - p - q$ 는 자연수이므로 $r = n - p - q$ 라 하면
$a^{2} < 10^{r} < (a + 1)^{2}$ 을 만족하는 자연수 $r$ 이 존재한다. (참)

ㄷ. $a$ 는 한 자리의 자연수이고 조건 ㄴ을 만족해야 하므로 $a^{2} < 10^{r} < (a + 1)^{2}$ 에서
$r = 1$ 일 때, $a^{2} < 10 < (a + 1)^{2}$ 이므로 $a = 3$ 이어야 한다. ($\because 3^{2} < 10 < 4^{2}$)
$r = 2$ 일 때, $a^{2} < 100 < (a + 1)^{2}$ 을 만족하는 한 자리 자연수 $a$ 는 존재하지 않는다.
따라서 $a$ 의 값은 $3$ 뿐이므로 $a = 7$ 이 되도록 하는 $n$ 은 존재하지 않는다. (거짓)

이상에서 옳은 것은 ㄱ, ㄴ이다.

정답: ②
