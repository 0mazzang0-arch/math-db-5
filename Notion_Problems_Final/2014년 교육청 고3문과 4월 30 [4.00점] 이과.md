## 문제
양수 $x$ 에 대하여 $\log x$ 의 정수부분과 소수부분을 각각 $f(x)$, $g(x)$ 라 할 때, 두 양수 $a$, $b$ 가 다음 조건을 만족시킨다.

$$
\begin{array}{|l|}
\hline
\text{(가) } f(a) + g(b) = \frac{9}{4} \\
\text{(나) } g(a) = g\left(\frac{1}{a}\right) + g(b) \\
\text{(다) } f(b) = f\left(\frac{1}{b}\right) + f(a^{5}) \\
\hline
\end{array}
$$

$ab = 10^{\frac{n}{m}}$ 일 때, $m + n$ 의 값을 구하시오. (단, $m$, $n$ 은 서로소인 자연수이다.)

## 보기

## 풀이
(가)에서 $f(a) + g(b) = 2 + \frac{1}{4}$ 이므로 
$f(a) = 2$, $g(b) = \frac{1}{4}$ 이다.

(나)에서 $g(b) = \frac{1}{4}$ 이므로 $g(a) \neq 0$ 이다.
이때 $g\left(\frac{1}{a}\right) = 1 - g(a)$ 이므로 
$g(a) = 1 - g(a) + \frac{1}{4}$, 즉 $g(a) = \frac{5}{8}$ 이다.

$\log a = f(a) + g(a) = 2 + \frac{5}{8} = \frac{21}{8}$ 이므로 
$\therefore a = 10^{\frac{21}{8}}$ 이다.

(다)에서 $g(b) \neq 0$ 이므로 $f\left(\frac{1}{b}\right) = -1 - f(b)$ 이다.
$a^{5} = 10^{\frac{105}{8}} = 10^{13 + \frac{1}{8}}$ 에서 $f(a^{5}) = 13$ 이므로 
$f(b) = -1 - f(b) + 13$, 즉 $f(b) = 6$ 이다.

$\log b = f(b) + g(b) = 6 + \frac{1}{4} = \frac{25}{4}$ 이므로 
$\therefore b = 10^{\frac{25}{4}}$ 이다.

$\therefore ab = 10^{\frac{21}{8} + \frac{25}{4}} = 10^{\frac{71}{8}}$
$ab = 10^{\frac{n}{m}}$ 에서 $n = 71$, $m = 8$ 이다. 
따라서 $m + n = 8 + 71 = 79$ 이다.
