## 문제
자연수 $n$ 에 대하여 다항식 $2x^2 - 3x + 1$ 을 $x - n$ 으로 나누었을 때의 나머지를 $a_n$ 이라 할 때, $\sum_{n=1}^{7} ( a_n - n^2 + n )$ 의 값을 구하시오.

## 보기

## 풀이
나머지정리에 의하여 $a_n = 2n^2 - 3n + 1$ 이므로
$\sum_{n=1}^{7} ( a_n - n^2 + n )$
$= \sum_{n=1}^{7} ( 2n^2 - 3n + 1 - n^2 + n )$
$= \sum_{n=1}^{7} ( n^2 - 2n + 1 )$
$= \sum_{n=1}^{7} ( n - 1 )^2$
$= \sum_{k=1}^{6} k^2$
$= \frac{6 \times 7 \times 13}{6} = 91$
따라서 정답은 $91$ 이다.
