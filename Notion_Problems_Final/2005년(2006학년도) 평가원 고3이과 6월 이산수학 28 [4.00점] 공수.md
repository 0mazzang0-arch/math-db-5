## 문제
자연수 $n$을 두 자연수 $n_{1}$, $n_{2}$의 합 $n = n_{1} + n_{2}$ ($n_{1} \geq n_{2}$)으로 나타내는 방법의 수를 $p(n)$이라 하자. 예를 들어, $p(3) = 1$이다. <보기>에서 항상 옳은 것을 모두 고른 것은?

**<보기>**
ㄱ. $p(n) = \left[ \frac{n}{2} \right]$ (단, $[x]$는 $x$보다 크지 않은 최대의 정수이다.)
ㄴ. $p(m + n) = p(m) + p(n)$ (단, $m, n$은 자연수이다.)
ㄷ. $p(n^{2}) = \{ p(n) \}^{2}$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄷ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
[정답] ① 

ㄱ. (i) $n$ 이 짝수일 때 
$n = 2k$ ($k = 1, 2, 3, \cdots$) 라 하면 
$n = (2k - 1) + 1 = (2k - 2) + 2 = \cdots = (2k - k) + k$ 
$\therefore P(n) = k = \left[ \frac{n}{2} \right]$ 

(ii) $n$ 이 홀수일 때 
$n = 2k - 1$ ($k = 1, 2, 3, \cdots$) 이라 하면 
$n = (2k - 2) + 1 = (2k - 3) + 2 = \cdots = (2k - k) + (k - 1)$ 
$\therefore P(n) = k - 1 = \left[ \frac{n}{2} \right]$ 

(i), (ii)에서 $P(n) = \left[ \frac{n}{2} \right]$ ($\therefore$ 참) 

ㄴ. $m = 3, n = 5$ 라 하면 
$m + n = 8$ 이고 
$P(3) = \left[ \frac{3}{2} \right] = 1$, $P(5) = \left[ \frac{5}{2} \right] = 2$, $P(8) = \left[ \frac{8}{2} \right] = 4$ 이므로 
$P(3 + 5) \neq P(3) + P(5)$ 
$P(m + n) \neq P(m) + P(n)$ (거짓) 

ㄷ. $n = 3$ 일 때 
$P(3^{2}) = P(9) = \left[ \frac{9}{2} \right] = 4$ 
$\{ P(3) \}^{2} = 1^{2} = 1$ 
$P(3^{2}) \neq \{ P(3) \}^{2}$ 
$P(n^{2}) \neq \{ P(n) \}^{2}$ (거짓) 

이상에서 옳은 것은 ㄱ뿐이다.
