## 문제
등식 $[ \log x ] = 3 ( \log x - [ \log x ] )$ 를 만족시키는 모든 양수 $x$ 의 값의 곱을 $A$ 라 할 때, $10 \log A$ 의 값을 구하시오. (단, $[ x ]$ 는 $x$ 보다 크지 않은 최대의 정수이다.)

## 풀이
$\log x$ 의 지표를 $n$, 가수를 $\alpha$ ($0 \le \alpha < 1$) 라 하면 $[ \log x ] = n$, $\log x - [ \log x ] = \alpha$ 이다.

주어진 등식 $[ \log x ] = 3 ( \log x - [ \log x ] )$ 에서
$n = 3 \alpha$
$0 \le \alpha < 1$ 이므로 $0 \le 3 \alpha < 3$ 이고, $n$ 은 정수이므로
$n = 0$ 또는 $n = 1$ 또는 $n = 2$ 이다.

즉, $3 \alpha = 0$ 또는 $3 \alpha = 1$ 또는 $3 \alpha = 2$ 이므로
$\alpha = 0$ 또는 $\alpha = \frac{1}{3}$ 또는 $\alpha = \frac{2}{3}$ 이다.

(i) $n = 0, \alpha = 0$ 일 때,
$\log x = 0$ 이므로 $x = 10^{0} = 1$

(ii) $n = 1, \alpha = \frac{1}{3}$ 일 때,
$\log x = 1 + \frac{1}{3} = \frac{4}{3}$ 이므로 $x = 10^{\frac{4}{3}}$

(iii) $n = 2, \alpha = \frac{2}{3}$ 일 때,
$\log x = 2 + \frac{2}{3} = \frac{8}{3}$ 이므로 $x = 10^{\frac{8}{3}}$

따라서 주어진 등식을 만족시키는 모든 양수 $x$ 의 값의 곱 $A$ 는
$A = 1 \times 10^{\frac{4}{3}} \times 10^{\frac{8}{3}} = 10^{\frac{12}{3}} = 10^{4}$ 이다.

$\therefore 10 \log A = 10 \log 10^{4} = 10 \times 4 = 40$

정답: $40$
