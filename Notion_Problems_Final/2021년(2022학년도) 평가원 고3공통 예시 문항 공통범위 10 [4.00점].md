## 문제
$\frac{1}{2} < \log a < \frac{11}{2}$ 인 양수 $a$ 에 대하여 $\frac{1}{3} + \log \sqrt{a}$ 의 값이 자연수가 되도록 하는 모든 $a$ 의 값의 곱은?

## 보기
① $10^{10}$   ② $10^{11}$   ③ $10^{12}$   ④ $10^{13}$   ⑤ $10^{14}$

## 풀이
$\frac{1}{2} < \log a < \frac{11}{2}$ 이므로 각 변에 $\frac{1}{2}$ 을 곱하면
$\frac{1}{4} < \log \sqrt{a} < \frac{11}{4}$
각 변에 $\frac{1}{3}$ 을 더하면
$\frac{1}{3} + \frac{1}{4} < \frac{1}{3} + \log \sqrt{a} < \frac{1}{3} + \frac{11}{4}$
$\frac{7}{12} < \frac{1}{3} + \log \sqrt{a} < \frac{37}{12}$
이때 $\frac{1}{3} + \log \sqrt{a}$ 의 값이 자연수이므로
$\frac{1}{3} + \log \sqrt{a} = 1, 2, 3$
$\log \sqrt{a} = \frac{2}{3}, \frac{5}{3}, \frac{8}{3}$ 에서 $\frac{1}{2} \log a = \frac{2}{3}, \frac{5}{3}, \frac{8}{3}$
$\log a = \frac{4}{3}, \frac{10}{3}, \frac{16}{3}$
모든 $a$ 의 값을 $a_{1}, a_{2}, a_{3}$ 이라 하면
$\log ( a_{1} \times a_{2} \times a_{3} ) = \frac{4}{3} + \frac{10}{3} + \frac{16}{3} = \frac{30}{3} = 10$
$\therefore a_{1} \times a_{2} \times a_{3} = 10^{10}$

정답은 ①이다.
