## 문제
$x$에 대한 삼차방정식 $ax^{3} + 2bx^{2} + 4bx + 8a = 0$이 서로 다른 세 정수를 근으로 갖는다. 두 정수 $a$, $b$가 $|a| \le 50$, $|b| \le 50$일 때, 순서쌍 $(a, b)$의 개수를 구하시오.

## 풀이
[정답] $46$

삼차방정식 $ax^{3} + 2bx^{2} + 4bx + 8a = 0$에서
$a(x^{3} + 8) + 2bx(x + 2) = 0$
$a(x + 2)(x^{2} - 2x + 4) + 2bx(x + 2) = 0$
$\therefore (x + 2)\{ax^{2} - 2(a - b)x + 4a\} = 0$ ($a \neq 0$)

위 방정식이 서로 다른 세 정수를 근으로 가지므로 방정식 $ax^{2} - 2(a - b)x + 4a = 0$이 $-2$가 아닌 서로 다른 두 정수를 근으로 가져야 한다. 이 이차방정식의 두 정수인 근을 $\alpha$, $\beta$ ($\alpha < \beta$)라 하면 근과 계수의 관계에 의하여
$\alpha + \beta = \frac{2(a - b)}{a} \cdots$ ㉠
$\alpha \beta = \frac{4a}{a} = 4 \cdots$ ㉡

㉡에서 $\alpha$, $\beta$는 정수이므로
$\alpha = 1$, $\beta = 4$ 또는 $\alpha = -4$, $\beta = -1$
따라서 ㉠에서
$\frac{2(a - b)}{a} = 5$ 또는 $\frac{2(a - b)}{a} = -5$
$\therefore b = -\frac{3}{2}a$ 또는 $b = \frac{7}{2}a$

(ⅰ) $b = -\frac{3}{2}a$ 일 때
$a = 32$이면 $b = -\frac{3}{2} \times 32 = -48$이므로 $a$, $b$의 순서쌍 $(a, b)$는
$(2, -3)$, $(4, -6)$, $\dots$, $(32, -48)$,
$(-2, 3)$, $(-4, 6)$, $\dots$, $(-32, 48)$
로 $32$개다.

(ⅱ) $b = \frac{7}{2}a$ 일 때
$a = 14$이면 $b = \frac{7}{2} \times 14 = 49$이므로 $a$, $b$의 순서쌍 $(a, b)$는
$(2, 7)$, $(4, 14)$, $\dots$, $(14, 49)$,
$(-2, -7)$, $(-4, -14)$, $\dots$, $(-14, -49)$
로 $14$개다.

(ⅰ), (ⅱ)에서 주어진 조건을 만족시키는 순서쌍 $(a, b)$의 개수는 $32 + 14 = 46$이다.
