## 문제
$1$보다 큰 실수 $x$에 대하여 $\log x$의 소수부분을 $\alpha$라 하면 $(\log x)^{2} + \alpha^{2} = 8$이 성립한다. 이때, $\log x$의 값은?

## 보기
① $1 + \sqrt{2}$   ② $2 \sqrt{2}$   ③ $1 + \sqrt{3}$   ④ $\frac{5}{2}$   ⑤ $\frac{8}{3}$

## 풀이
정답: ③

$\log x = n + \alpha$라 하자. 


단, $n$은 정수, $0 \le \alpha < 1$

준식에 대입해서 정리하면
$(n + \alpha)^{2} + \alpha^{2} = 8$
$n^{2} + 2 \alpha n + 2 \alpha^{2} = 8$

$x > 1$이면서 $n$값은 정수이므로 $n = 0, 1, 2, \dots$ 등이 가능하다.

(ⅰ) $n = 0$인 경우
$2 \alpha^{2} = 8 \Rightarrow \alpha^{2} = 4$를 만족시키는 실수 $\alpha$ ($0 \le \alpha < 1$)는 존재하지 않는다.

(ⅱ) $n = 1$인 경우
$1^{2} + 2 \alpha (1) + 2 \alpha^{2} = 8 \Rightarrow 2 \alpha^{2} + 2 \alpha - 7 = 0 \Rightarrow \alpha = \frac{-1 \pm \sqrt{15}}{2}$
$\alpha$의 범위를 만족시키지 않는다.

(ⅲ) $n = 2$인 경우
$2^{2} + 2 \alpha (2) + 2 \alpha^{2} = 8 \Rightarrow 2 \alpha^{2} + 4 \alpha - 4 = 0 \Rightarrow \alpha^{2} + 2 \alpha - 2 = 0 \Rightarrow \alpha = -1 + \sqrt{3}$ (단, $0 \le \alpha < 1$)

$n \ge 3$인 경우 소수부분 조건을 만족시키는 $\alpha$는 존재하지 않는다.

따라서 $\log x = n + \alpha = 2 + (-1 + \sqrt{3}) = 1 + \sqrt{3}$이다.
