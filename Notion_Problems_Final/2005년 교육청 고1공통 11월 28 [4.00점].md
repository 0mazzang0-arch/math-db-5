## 문제
자연수 $a$, $b$, $c$에 대하여 $\sqrt{a + \sqrt{24}} = \sqrt{b} + \sqrt{c}$ 가 성립할 때, $a + b + c$ 의 최솟값을 구하시오.

## 보기


## 풀이
$\sqrt{a + \sqrt{24}} = \sqrt{a + 2 \sqrt{6}} = \sqrt{b} + \sqrt{c}$ 이므로 
$a = b + c$, $bc = 6$ 이다.
$6 = 1 \times 6 = 2 \times 3$ 이므로
$\begin{cases} b = 1 \\ c = 6 \end{cases}$, $\begin{cases} b = 6 \\ c = 1 \end{cases}$, $\begin{cases} b = 2 \\ c = 3 \end{cases}$, $\begin{cases} b = 3 \\ c = 2 \end{cases}$ 이다.

이때 $a + b + c$ 의 값은 다음과 같다.
i) $\{b, c\} = \{1, 6\}$ 일 때, 
$a = 1 + 6 = 7$ 이므로
$a + b + c = 7 + 1 + 6 = 14$

ii) $\{b, c\} = \{2, 3\}$ 일 때, 
$a = 2 + 3 = 5$ 이므로
$a + b + c = 5 + 2 + 3 = 10$

따라서 $a + b + c$ 의 최솟값은 $10$ 이다.
