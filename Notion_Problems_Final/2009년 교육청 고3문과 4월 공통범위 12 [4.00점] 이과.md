## 문제
자연수 `$m$`에 대하여 수열 `$\{ a_{n} \}$`의 일반항이 `$a_{n}= \left[ \frac{n}{m} \right]$`일 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?
(단, `$\left[ x \right]$`는 `$x$`보다 크지 않은 최대의 정수이다.)

$$ \begin{array}{|l|} \hline \text{ㄱ. } m=2 \text{ 일 때, } a_{5}=2 \text{ 이다.} \\ \text{ㄴ. } m=3 \text{ 일 때, } \sum_{k=1}^{100} a_{k}=1683 \text{ 이다.} \\ \text{ㄷ. } \sum_{k=1}^{mn} a_{k}=\frac{n(mn-m+2)}{2} \\ \hline \end{array} $$

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄷ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
정답: ③

ㄱ. `$m=2$` 일 때, `$a_{5}= \left[ \frac{5}{2} \right] = 2$` 이다. (참)

ㄴ. `$m=3$` 일 때,
`$\sum_{k=1}^{100} \left[ \frac{k}{3} \right] = \left( \left[ \frac{1}{3} \right] + \left[ \frac{2}{3} \right] + \left[ \frac{3}{3} \right] \right) + \left( \left[ \frac{4}{3} \right] + \left[ \frac{5}{3} \right] + \left[ \frac{6}{3} \right] \right) + \dots + \left( \left[ \frac{97}{3} \right] + \left[ \frac{98}{3} \right] + \left[ \frac{99}{3} \right] \right) + \left[ \frac{100}{3} \right]$`
`$= (0+0+1) + (1+1+2) + (2+2+3) + \dots + (32+32+33) + 33$`
`$= \sum_{j=0}^{32} (3j+1) + 33$`
`$= 3 \sum_{k=1}^{32} k + 33 + 33$`
`$= 3 \times \frac{32 \times 33}{2} + 66 = 1584 + 66 = 1650$` 이다. (거짓)

ㄷ. `$\sum_{k=1}^{mn} \left[ \frac{k}{m} \right]$` 에서
각 `$j=1, 2, \dots, n-1$` 에 대하여 `$\left[ \frac{k}{m} \right] = j$` 인 `$k$` 의 값은 `$jm, jm+1, \dots, (j+1)m-1$` 의 `$m$` 개가 존재한다.
단, `$k=mn$` 일 때 `$\left[ \frac{mn}{m} \right] = n$` 이다.
따라서, `$\sum_{k=1}^{mn} \left[ \frac{k}{m} \right] = \sum_{j=1}^{n-1} (m \times j) + n$`
`$= m \sum_{k=1}^{n-1} k + n$`
`$= m \times \frac{(n-1)n}{2} + n$`
`$= \frac{mn^2 - mn + 2n}{2}$`
`$= \frac{n(mn-m+2)}{2}$` (참)
