## 문제
거듭제곱근에 대한 설명으로 옳은 것만을 <보기>에서 있는 대로 고른 것은? (단, $[x]$ 는 $x$ 를 넘지 않는 최대 정수이다.)

$$ \begin{array}{|l|} \hline \text{<보 기>} \\ \text{ㄱ. } \sqrt[4]{\sqrt[3]{5}} = \sqrt[7]{5} \\ \text{ㄴ. } [\sqrt[3]{1}] + [\sqrt[3]{2}] + [\sqrt[3]{3}] + \cdots + [\sqrt[3]{36}] = 75 \\ \text{ㄷ. } [\sqrt{a}] + [\sqrt{10-a}] = [\sqrt{10}] \text{ 을 만족하는 자연수 } a \text{ 는 } 5 \text{ 개이다.} \\ \hline \end{array} $$

## 보기
① $ㄴ$   ② $ㄷ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
ㄱ. 
$\sqrt[4]{\sqrt[3]{5}} = \sqrt[12]{5}$ 이므로 
$\sqrt[12]{5} \neq \sqrt[7]{5}$
따라서 거짓이다.

ㄴ. 
$1 \le n < 8$ 이면 $[\sqrt[3]{n}] = 1$ ($7$ 개)
$8 \le n < 27$ 이면 $[\sqrt[3]{n}] = 2$ ($19$ 개)
$27 \le n \le 36$ 이면 $[\sqrt[3]{n}] = 3$ ($10$ 개)
$$ \begin{array}{l} [\sqrt[3]{1}] + [\sqrt[3]{2}] + \cdots + [\sqrt[3]{36}] \\ = (1 \times 7) + (2 \times 19) + (3 \times 10) \\ = 7 + 38 + 30 \\ = 75 \end{array} $$
따라서 참이다.

ㄷ. 
$1 \le a \le 10$ 이면 $1 \le [\sqrt{a}] \le 3$ 이다.
$[\sqrt{10}] = 3$ 이므로 $[\sqrt{a}] + [\sqrt{10-a}] = 3$ 을 만족해야 한다.

(i) $[\sqrt{a}] = 1, [\sqrt{10-a}] = 2$ 인 경우
$[\sqrt{a}] = 1$ 이면 $1 \le \sqrt{a} < 2$ 이므로 $1 \le a < 4$
$[\sqrt{10-a}] = 2$ 이면 $2 \le \sqrt{10-a} < 3$ 이므로 $4 \le 10-a < 9$
즉, $1 < a \le 6$
따라서 공통 범위 $1 < a < 4$ 에서 자연수 $a = 2, 3$

(ii) $[\sqrt{a}] = 2, [\sqrt{10-a}] = 1$ 인 경우
같은 방법으로 $4 \le a < 9$ 이고 $1 < a \le 6$ 이므로 $6 < a < 9$
따라서 자연수 $a = 7, 8$

(iii) $[\sqrt{a}] = 3, [\sqrt{10-a}] = 0$ 인 경우
같은 방법으로 $9 \le a < 16$ 이고 $9 < a \le 10$ 이므로 $9 < a \le 10$
따라서 자연수 $a = 10$

따라서 (i), (ii), (iii)에서 만족하는 자연수 $a$ 는 $2, 3, 7, 8, 10$ 으로 총 $5$ 개이다.
따라서 참이다.
