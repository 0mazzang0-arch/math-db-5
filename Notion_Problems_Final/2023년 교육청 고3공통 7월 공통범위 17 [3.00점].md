## 문제
함수 $f(x)$ 에 대하여 $f'(x) = 9x^{2} - 8x + 1$ 이고 $f(1) = 10$ 일 때, $f(2)$ 의 값을 구하시오.

## 풀이
$f(x) = \int (9x^{2} - 8x + 1) dx$
$= 3x^{3} - 4x^{2} + x + C$ (단, $C$ 는 적분상수)

$f(1) = 3 - 4 + 1 + C = 10$ 이므로
$C = 10$

즉, $f(x) = 3x^{3} - 4x^{2} + x + 10$
따라서 $f(2) = 3 \times 2^{3} - 4 \times 2^{2} + 2 + 10$
$= 24 - 16 + 2 + 10 = 20$

정답: $20$
