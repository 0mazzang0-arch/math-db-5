## 문제
양수 $x$에 대하여 $\log x$의 정수부분과 소수부분을 각각 $f(x)$, $g(x)$라 할 때, 두 양수 $a$, $b$가 다음 조건을 만족시킨다.

$$
\begin{array}{l}
(가) \text{ } 10 \le a < 100 \\
(나) \text{ } g(a) = f(b) + \frac{1}{2} \\
(다) \text{ } g(b) = g \left( \frac{1}{b} \right) + g(a)
\end{array}
$$

$ab = 10^{\frac{n}{m}}$일 때, $m+n$의 값을 구하시오. (단, $m$, $n$은 서로소인 자연수이다.)

## 풀이
(가)에서 $10 \le a < 100$이므로 $\log a$의 정수부분은 $f(a) = 1$이다.

(나)에서 $g(a) = f(b) + \frac{1}{2}$이고 $0 \le g(a) < 1$이므로
$0 \le f(b) + \frac{1}{2} < 1$
$-\frac{1}{2} \le f(b) < \frac{1}{2}$
$f(b)$는 정수이므로 $f(b) = 0$이다.
따라서 $g(a) = 0 + \frac{1}{2} = \frac{1}{2}$이다.

$f(b) = 0$이므로 $0 \le \log b < 1$이다.
만약 $g(b) = 0$이면 $\log b = 0$이 되어 $g \left( \frac{1}{b} \right) = g(0) = 0$이다. 이 경우 (다)에서 $0 = 0 + \frac{1}{2}$이 되어 모순이다.
따라서 $g(b) \ne 0$이고, $g \left( \frac{1}{b} \right) = 1 - g(b)$이다.

(다)에 대입하면
$g(b) = 1 - g(b) + g(a)$
$g(b) = 1 - g(b) + \frac{1}{2}$
$2g(b) = \frac{3}{2}$
$\therefore g(b) = \frac{3}{4}$

$\log a = f(a) + g(a) = 1 + \frac{1}{2} = \frac{3}{2}$
$\log b = f(b) + g(b) = 0 + \frac{3}{4} = \frac{3}{4}$

$\log ab = \log a + \log b = \frac{3}{2} + \frac{3}{4} = \frac{9}{4}$
$ab = 10^{\frac{9}{4}}$

따라서 $n = 9$, $m = 4$이며, $m, n$은 서로소인 자연수이다.
$m + n = 4 + 9 = 13$

정답: $13$
