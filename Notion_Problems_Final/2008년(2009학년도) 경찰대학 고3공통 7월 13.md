## 문제
$-2 \leq X \leq 4$ 의 모든 값을 취하는 확률변수 $X$ 의 확률밀도함수 $f(x)$ 는 다음을 만족시킨다.
$$f(1-x)=f(1+x)$$
$P(1 \leq X \leq 3)=2 P(3 \leq X \leq 4)$ 이고 $P(0 \leq X \leq 1)=\frac{1}{4}$ 일 때, $P(0 \leq X \leq 3)$ 의 값은?

## 보기
① $\frac{5}{12}$   ② $\frac{1}{2}$   ③ $\frac{7}{12}$   ④ $\frac{2}{3}$   ⑤ $\frac{3}{4}$

## 풀이
[정답] ③ 

조건에서 $f(1-x)=f(1+x)$ 이므로 확률변수 $X$ 의 확률밀도함수 $f(x)$ 는 $x=1$ 에 대하여 대칭이고, $-2 \leq X \leq 4$ 이므로 확률밀도함수의 성질에 의해 
$$P(-2 \leq X \leq 1)=P(1 \leq X \leq 4)=\frac{1}{2}$$ 
그러므로 조건 $P(1 \leq X \leq 3)=2 P(3 \leq X \leq 4)$ 에 의해 
$$P(1 \leq X \leq 3)=\frac{1}{3}$$ 
구하고자 하는 확률은 다음과 같다.
$$P(0 \leq X \leq 3) = P(0 \leq X \leq 1)+P(1 \leq X \leq 3)$$ 
$$=\frac{1}{4}+\frac{1}{3}=\frac{7}{12}$$
