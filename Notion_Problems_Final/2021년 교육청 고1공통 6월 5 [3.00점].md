## 문제
$x$의 값에 관계없이 등식 $3x^{2} + ax + 4 = bx(x - 1) + c(x - 1)(x - 2)$가 항상 성립할 때, $a + b + c$의 값은? (단, $a$, $b$, $c$는 상수이다.)

## 보기
① $-6$   ② $-5$   ③ $-4$   ④ $-3$   ⑤ $-2$

## 풀이
주어진 등식이 $x$에 대한 항등식이므로 $3x^{2} + ax + 4 = bx(x - 1) + c(x - 1)(x - 2)$에 적절한 값을 대입한다.

$x = 1$을 대입하면:
$3(1)^{2} + a(1) + 4 = b(1)(1 - 1) + c(1 - 1)(1 - 2)$
$3 + a + 4 = 0$
$a + 7 = 0$ 이므로 $a = -7$이다.

$x = 0$을 대입하면:
$3(0)^{2} + a(0) + 4 = b(0)(0 - 1) + c(0 - 1)(0 - 2)$
$4 = c(-1)(-2)$
$4 = 2c$ 이므로 $c = 2$이다.

$x = 2$를 대입하면:
$3(2)^{2} + a(2) + 4 = b(2)(2 - 1) + c(2 - 1)(2 - 2)$
$12 + 2a + 4 = 2b$
$2a + 16 = 2b$
위에서 구한 $a = -7$을 대입하면:
$2(-7) + 16 = 2b$
$-14 + 16 = 2b$
$2 = 2b$ 이므로 $b = 1$이다.

따라서 $a + b + c = -7 + 1 + 2 = -4$이다.

정답: ③
