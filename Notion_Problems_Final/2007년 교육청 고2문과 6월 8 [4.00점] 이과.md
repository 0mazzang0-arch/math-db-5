## 문제
세 조건 $p$, $q$, $r$은 다음과 같다.
$p : |x| \leq 2$
$q : x(x - 3) < 0$
$r : [x]^{2} - [x] - 2 \leq 0$ (단, $[x]$는 $x$를 넘지 않는 최대 정수)
세 명제 $p \rightarrow q$, $q \rightarrow r$, $r \rightarrow p$의 참, 거짓을 순서대로 나열하면?

## 보기
① 참, 참, 참   ② 참, 참, 거짓   ③ 참, 거짓, 거짓   ④ 거짓, 참, 참   ⑤ 거짓, 참, 거짓

## 풀이
정답: ⑤

세 조건 $p, q, r$을 만족하는 $x$의 범위는 다음과 같다.
$p : |x| \leq 2 \rightarrow -2 \leq x \leq 2$
$q : x(x - 3) < 0 \rightarrow 0 < x < 3$
$r : [x]^{2} - [x] - 2 \leq 0$
$([x] - 2)([x] + 1) \leq 0$
$-1 \leq [x] \leq 2$
$[x]$는 정수이므로 $[x] = -1, 0, 1, 2$
$\therefore -1 \leq x < 3$

각 명제의 참, 거짓을 판별하면 다음과 같다.
1. $p \rightarrow q$: $[-2, 2] \subset (0, 3)$은 거짓이다. (예: $x = -1$)
2. $q \rightarrow r$: $(0, 3) \subset [-1, 3)$은 참이다.
3. $r \rightarrow p$: $[-1, 3) \subset [-2, 2]$는 거짓이다. (예: $x = 2.5$)

따라서 참, 거짓을 순서대로 나열하면 거짓, 참, 거짓이다.
