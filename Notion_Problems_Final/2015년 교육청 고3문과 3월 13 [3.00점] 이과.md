## 문제
양의 실수 $x$ 에 대하여 $f(x) = \log x$ 이다. 세 실수 $f(3)$, $f(3^{t} + 3)$, $f(12)$ 가 이 순서대로 등차수열을 이룰 때, 실수 $t$ 의 값은?

## 보기
① $\frac{1}{4}$   ② $\frac{1}{2}$   ③ $\frac{3}{4}$   ④ $1$   ⑤ $\frac{5}{4}$

## 풀이
$f(3) = \log 3$, $f(3^{t} + 3) = \log(3^{t} + 3)$, $f(12) = \log 12$ 가 이 순서대로 등차수열을 이루므로 등차중항의 성질에 의하여 다음이 성립한다.

$\log(3^{t} + 3) = \frac{\log 12 + \log 3}{2}$
$\log(3^{t} + 3) = \frac{\log 36}{2}$
$\log(3^{t} + 3) = \log \sqrt{36}$
$\log(3^{t} + 3) = \log 6$

따라서 $3^{t} + 3 = 6$ 이다.
$3^{t} = 3$
$t = 1$

정답은 ④ 이다.
