## 문제
연립부등식 
$$ \begin{cases} 2x+1 < x-3 \\ x^2+6x-7 < 0 \end{cases} $$
의 해가 $ \alpha < x < \beta $ 일 때, $ \beta - \alpha $ 의 값을 구하시오.

## 풀이
부등식 $ 2x+1 < x-3 $ 의 해는 $ x < -4 $ 이고 
$ x^2+6x-7 = (x-1)(x+7) < 0 $ 의 해는 $ -7 < x < 1 $ 이므로 연립부등식의 해는 
$ -7 < x < -4 $ 
따라서 $ \alpha = -7 $, $ \beta = -4 $ 이므로 
$ \beta - \alpha = -4 - (-7) = 3 $
