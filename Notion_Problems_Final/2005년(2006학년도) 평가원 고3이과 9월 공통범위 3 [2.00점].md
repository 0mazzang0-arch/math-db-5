## 문제
두 실수 $a$, $b$에 대하여 $\lim_{x \to 1} \frac{\sqrt{x^2 + a} - b}{x - 1} = \frac{1}{2}$ 일 때, $ab$의 값은?

## 보기
① $6$   ② $7$   ③ $8$   ④ $9$   ⑤ $10$

## 풀이
$x \to 1$일 때 (분모) $\to 0$이므로
(분자) $= \sqrt{1 + a} - b = 0$
$\therefore b = \sqrt{1 + a}$

주어진 식의 분자를 유리화하면
$\lim_{x \to 1} \frac{\sqrt{x^2 + a} - \sqrt{1 + a}}{x - 1}$
$= \lim_{x \to 1} \frac{(x^2 + a) - (1 + a)}{(x - 1)(\sqrt{x^2 + a} + \sqrt{1 + a})}$
$= \lim_{x \to 1} \frac{x^2 - 1}{(x - 1)(\sqrt{x^2 + a} + \sqrt{1 + a})}$
$= \lim_{x \to 1} \frac{(x - 1)(x + 1)}{(x - 1)(\sqrt{x^2 + a} + \sqrt{1 + a})}$
$= \lim_{x \to 1} \frac{x + 1}{\sqrt{x^2 + a} + \sqrt{1 + a}}$
$= \frac{2}{2\sqrt{1 + a}} = \frac{1}{\sqrt{1 + a}}$

이 값이 $\frac{1}{2}$이므로
$\frac{1}{\sqrt{1 + a}} = \frac{1}{2}$
$\sqrt{1 + a} = 2$
$1 + a = 4$
$\therefore a = 3$

$b = \sqrt{1 + a} = \sqrt{1 + 3} = 2$

$\therefore ab = 3 \times 2 = 6$

정답: ①
