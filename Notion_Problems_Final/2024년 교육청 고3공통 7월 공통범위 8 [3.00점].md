## 문제
삼차함수 $f(x)$가 모든 실수 $x$에 대하여 
$xf'(x) = 6x^{3} - x + f(0) + 1$
을 만족시킬 때, $f(-1)$의 값은?

## 보기
① $-2$   ② $-1$   ③ $0$   ④ $1$   ⑤ $2$

## 풀이
모든 실수 $x$에 대하여 
$xf'(x) = 6x^{3} - x + f(0) + 1$ 

$x = 0$일 때, 
$0 = f(0) + 1$ 

$f(0) = -1$ 

$xf'(x) = 6x^{3} - x + (-1) + 1 = x(6x^{2} - 1)$ 

$f'(x) = 6x^{2} - 1$ 

$f(x) = 2x^{3} - x + C$ (단, $C$는 적분상수) 

$f(0) = C = -1$ 

$f(x) = 2x^{3} - x - 1$ 

따라서 $f(-1) = 2(-1)^{3} - (-1) - 1 = -2 + 1 - 1 = -2$

정답: ①
