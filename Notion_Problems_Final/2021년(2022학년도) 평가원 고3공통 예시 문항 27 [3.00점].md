## 문제
집합 $X = \{1, 2, 3, 4\}$에 대하여 다음 조건을 만족시키는 모든 함수 $f : X \to X$의 개수는?

(가) $f(1) + f(2) + f(3) \ge 3f(4)$
(나) $k = 1$, $2$, $3$일 때 $f(k) \neq f(4)$이다.

## 보기
① $41$   ② $45$   ③ $49$   ④ $53$   ⑤ $57$

## 풀이
(i) $f(4) = 1$이면 $f(1) + f(2) + f(3) \ge 3$
$f(1)$, $f(2)$, $f(3) \in \{2, 3, 4\}$에서 $3^3 = 27$가지

(ii) $f(4) = 2$이면 $f(1) + f(2) + f(3) \ge 6$
$f(1)$, $f(2)$, $f(3) \in \{1, 3, 4\}$에서
$1 + 1 + 1 = 3$, $1 + 1 + 3 = 5$이므로 이것을 제외하면
$3^3 - (1 + 3) = 23$가지

(iii) $f(4) = 3$이면 $f(1) + f(2) + f(3) \ge 9$
$f(1)$, $f(2)$, $f(3) \in \{1, 2, 4\}$에서
$4 + 4 + 4 = 12$, $4 + 4 + 2 = 10$, $4 + 4 + 1 = 9$
$1 + 3 + 3 = 7$가지

(iv) $f(4) = 4$이면 $f(1) + f(2) + f(3) \ge 12$
$f(1)$, $f(2)$, $f(3) \in \{1, 2, 3\}$에서 $3 + 3 + 3 = 9$이므로 만족하는 경우는 없다.

$\therefore 27 + 23 + 7 = 57$

정답: ⑤
