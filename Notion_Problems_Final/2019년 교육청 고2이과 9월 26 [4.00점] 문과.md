## 문제
첫째항과 공비가 모두 자연수인 등비수열 $\{ a_{n} \}$ 에 대하여 $5 \le a_{2} \le 6$, $42 \le a_{4} \le 96$ 일 때, $\sum_{n=1}^{5} a_{n}$ 의 값을 구하시오.

## 보기

## 풀이
등비수열 $\{ a_{n} \}$ 의 첫째항을 $a$, 공비를 $r$ 라 하자. 

$5 \le a_{2} \le 6$ 에 의하여 $5 \le ar \le 6$ 이고 $a$ 와 $r$ 는 자연수이므로 $ar = 5$ 또는 $ar = 6$ 이다.

$42 \le a_{4} \le 96$ 에 의하여 $42 \le ar^{3} \le 96$ 이므로
$\frac{42}{ar} \le r^{2} \le \frac{96}{ar} \cdots$ ㉠ 

i) $ar = 5$ 일 때, 
㉠에 의하여 $\frac{42}{5} \le r^{2} \le \frac{96}{5}$ 이므로 $8.4 \le r^{2} \le 19.2$ 이다.
이를 만족시키는 자연수 $r$ 는 $3$ 또는 $4$ 이나, $ar = 5$ 를 만족시키는 자연수 $a$ 는 존재하지 않는다. 

ii) $ar = 6$ 일 때, 
㉠에 의하여 $\frac{42}{6} \le r^{2} \le \frac{96}{6}$ 즉, $7 \le r^{2} \le 16$ 이므로 $r = 3$ 또는 $r = 4$ 이다.
$ar = 6$ 을 만족시키는 자연수 $a, r$ 는 $r = 3$, $a = 2$ 이다. (이때 $r = 4$ 이면 $a$ 는 자연수가 아니다.)

따라서 $\sum_{n=1}^{5} a_{n} = \frac{2 \times ( 3^{5} - 1 )}{3 - 1} = 242$

정답: $242$
