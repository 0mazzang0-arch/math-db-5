## 문제
함수 $f(x)$ 가 다음과 같다.
$$f(x) = \begin{cases} -3x+a & (x < -1) \\ x^3+bx^2+cx & (-1 \le x < 1) \\ -3x+d & (x \ge 1) \end{cases}$$
함수 $f(x)$ 가 모든 실수 $x$ 에 대하여 미분가능하도록 네 실수 $a$, $b$, $c$, $d$ 의 값을 정할 때, $a+b+c+d$ 의 값은?

## 보기
① $-10$   ② $-8$   ③ $-6$   ④ $-4$   ⑤ $-2$

## 풀이
함수 $f(x)$ 가 $x = -1$ 과 $x = 1$ 에서 미분가능해야 한다.

$1$ ) $x = -1$ 에서 미분가능할 조건:
연속성: $f(-1) = 3+a = -1+b-c$
미분계수: $f'(-1) = -3 = 3(-1)^2 + 2b(-1) + c = 3-2b+c$

$2$ ) $x = 1$ 에서 미분가능할 조건:
연속성: $f(1) = -3+d = 1+b+c$
미분계수: $f'(1) = -3 = 3(1)^2 + 2b(1) + c = 3+2b+c$

미분계수 식을 정리하면:
$-2b+c = -6$
$2b+c = -6$
두 식을 연립하면 $4b = 0$ 이므로 $b = 0$, $c = -6$ 이다.

이를 연속성 식에 대입하면:
$3+a = -1+0-(-6) = 5 \implies a = 2$
$-3+d = 1+0+(-6) = -5 \implies d = -2$

따라서 
$\therefore a+b+c+d = 2+0-6-2 = -6$

정답은 ③이다.
