## 문제
$1$이 아닌 두 양수 $a$, $b$에 대하여 $n \le \log_{a} b < n+1$ ($n$은 정수)이 성립할 때, $f(a, b) = n$으로 정의한다. <보기> 중 옳은 것만을 있는 대로 고른 것은?

$$ \begin{array}{l} | \text{ < 보 기 > } | \\ \text{ㄱ. } f(2, 9) = 4 \text{ 이다.} \\ \text{ㄴ. } f(a, b) = 2 \text{ 이면 } f(b, a) = 0 \text{ 이다.} \\ \text{ㄷ. } f(a, b) = -2 \text{ 이면 } f(b, a) = -1 \text{ 이다.} \end{array} $$

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
ㄱ. $3 < \log_{2} 9 < 4$이므로 $f(2, 9) = 3$이다. (거짓)

ㄴ. $f(a, b) = 2$이면 $2 \le \log_{a} b < 3$이다.
각 변에 역수를 취하면 $\frac{1}{3} < \log_{b} a \le \frac{1}{2}$이다.
$0 \le \log_{b} a < 1$이므로 $f(b, a) = 0$이다. (참)

ㄷ. $f(a, b) = -2$이면 $-2 \le \log_{a} b < -1$이다.
각 변에 역수를 취하면 $-1 < \log_{b} a \le -\frac{1}{2}$이다.
$-1 \le \log_{b} a < 0$이므로 $f(b, a) = -1$이다. (참)

따라서 옳은 것은 ㄴ, ㄷ이다.
