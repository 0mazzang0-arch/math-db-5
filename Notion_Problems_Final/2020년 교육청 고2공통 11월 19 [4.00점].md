## 문제
다음은 공차가 $1$ 보다 크고 $a_{3} + a_{5} = 2$ 인 등차수열 $\{ a_{n} \}$ 에 대하여 $\sum_{k=1}^{5} ( a_{k}^{2} - 5 | a_{k} | )$ 의 값이 최소가 되도록 하는 수열 $\{ a_{n} \}$ 의 공차를 구하는 과정이다.

$$ \begin{array}{|l|} \hline a_{3} + a_{5} = 2 \text{ 에서 } a_{4} = (가) \\ \text{등차수열 } \{ a_{n} \} \text{ 의 공차를 } d \text{ 라 하고, } \sum_{k=1}^{5} a_{k}^{2} \text{ 과 } \sum_{k=1}^{5} | a_{k} | \text{ 를 각각 } d \text{ 에 대한 식으로 나타내면} \\ \sum_{k=1}^{5} a_{k}^{2} = 15d^{2} - 10d + 5 \\ \sum_{k=1}^{5} | a_{k} | = (나) \\ \text{따라서 } \sum_{k=1}^{5} ( a_{k}^{2} - 5 | a_{k} | ) \text{ 의 값이 최소가 되도록 하는 수열 } \{ a_{n} \} \text{ 의 공차는 } (다) \text{ 이다.} \\ \hline \end{array} $$

위의 $(가)$, $(다)$ 에 알맞은 수를 각각 $p$, $q$ 라 하고, $(나)$ 에 알맞은 식을 $f(d)$ 라 할 때, $f(p + 2q)$ 의 값은?

## 보기
① $21$   ② $23$   ③ $25$   ④ $27$   ⑤ $29$

## 풀이
$a_{3} + a_{5} = 2a_{4}$ 이므로 $a_{3} + a_{5} = 2$ 에서 $a_{4} = 1$ 이다.
등차수열 $\{ a_{n} \}$ 의 공차를 $d$ 라 하면
$a_{1} = 1 - 3d$, $a_{2} = 1 - 2d$, $a_{3} = 1 - d$, $a_{4} = 1$, $a_{5} = 1 + d$ 이다.
$d > 1$ 이므로 $a_{1} < 0$, $a_{2} < 0$, $a_{3} < 0$, $a_{5} > 0$ 이다.
$\sum_{k=1}^{5} a_{k}^{2}$ 과 $\sum_{k=1}^{5} | a_{k} |$ 를 각각 $d$ 에 대한 식으로 나타내면

$\sum_{k=1}^{5} a_{k}^{2} = (1 - 3d)^{2} + (1 - 2d)^{2} + (1 - d)^{2} + 1^{2} + (1 + d)^{2} = 15d^{2} - 10d + 5$

$\sum_{k=1}^{5} | a_{k} | = | 1 - 3d | + | 1 - 2d | + | 1 - d | + | 1 | + | 1 + d |$
$= (3d - 1) + (2d - 1) + (d - 1) + 1 + (1 + d) = 7d - 1$

그러므로
$\sum_{k=1}^{5} ( a_{k}^{2} - 5 | a_{k} | ) = (15d^{2} - 10d + 5) - 5(7d - 1)$
$= 15d^{2} - 45d + 10$
$= 15( d - \frac{3}{2} )^{2} - \frac{95}{4}$

$d > 1$ 이므로 $\sum_{k=1}^{5} ( a_{k}^{2} - 5 | a_{k} | )$ 의 값이 최소가 되도록 하는 수열 $\{ a_{n} \}$ 의 공차는 $\frac{3}{2}$ 이다.
따라서 $p = 1$, $q = \frac{3}{2}$, $f(d) = 7d - 1$ 이므로
$f(p + 2q) = f(1 + 3) = f(4) = 7 \times 4 - 1 = 27$ 이다.
