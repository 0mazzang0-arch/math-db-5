## 문제
두 함수 $f(x) = kx^2 - 2x$, $g(x) = e^{3x} + 1$이 있다. 함수 $h(x) = (f \circ g)(x)$에 대하여 $h'(0) = 42$일 때, 상수 $k$의 값을 구하시오.

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
두 함수 $f(x) = kx^2 - 2x$, $g(x) = e^{3x} + 1$에서 
$f'(x) = 2kx - 2$, $g'(x) = 3e^{3x}$이다.
$g(0) = e^0 + 1 = 2$, $g'(0) = 3e^0 = 3$이고
$f'(2) = 4k - 2$이다.
함수 $h(x) = (f \circ g)(x)$에서 합성함수의 미분법에 의해
$h'(x) = f'(g(x))g'(x)$이므로
$h'(0) = f'(g(0))g'(0) = f'(2) \times g'(0)$이다.
$h'(0) = (4k - 2) \times 3 = 42$
$4k - 2 = 14$
$4k = 16$
따라서 $k = 4$이다.

정답: ④
