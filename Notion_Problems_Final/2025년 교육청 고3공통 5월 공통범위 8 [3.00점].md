## 문제
부등식 $2^{|x|} + \frac{64}{2^{|x|}} \le 20$을 만족시키는 정수 $x$의 개수는?

## 보기
① $6$   ② $7$   ③ $8$   ④ $9$   ⑤ $10$

## 풀이
$2^{|x|} = t$ ($t \ge 1$)이라 하면 
$t + \frac{64}{t} \le 20$에서 
$t^2 - 20t + 64 \le 0$, 
$(t - 4)(t - 16) \le 0$ 
$4 \le t \le 16$ 
$2^{2} \le 2^{|x|} \le 2^{4}$에서 밑 $2$가 $1$보다 크므로 
$2 \le |x| \le 4$, 
$-4 \le x \le -2$ 또는 $2 \le x \le 4$ 
따라서 부등식을 만족시키는 정수 $x$의 개수는 $6$이다.

정답: ①
