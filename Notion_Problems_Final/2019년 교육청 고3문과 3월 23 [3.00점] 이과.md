## 문제
함수 $y = \frac{2x - 7}{x - 3}$ 의 그래프의 점근선은 두 직선 $x = a$, $y = b$ 이다. 두 상수 $a$, $b$ 의 곱 $ab$ 의 값을 구하시오.

## 보기

## 풀이
$y = \frac{2x - 7}{x - 3}$
$= \frac{2(x - 3) - 1}{x - 3}$
$= \frac{-1}{x - 3} + 2$
에서 함수 $y = \frac{2x - 7}{x - 3}$ 의 그래프의 두 점근선의 방정식은 $x = 3$, $y = 2$ 이므로 
$a = 3$, $b = 2$
따라서 $ab = 6$
