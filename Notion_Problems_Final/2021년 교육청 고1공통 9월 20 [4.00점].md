## 문제
복소수 $z = \frac{-1+\sqrt{3}i}{2}$ 에 대하여 <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $i = \sqrt{-1}$)

$$
\begin{array}{|l|}
\hline
ㄱ. z^3 = 1 \\
ㄴ. z^4 + z^5 = -1 \\
ㄷ. z^n + z^{2n} + z^{3n} + z^{4n} + z^{5n} = -1 \text{ 을 만족시키는 } 100 \text{ 이하의 모든 자연수 } n \text{ 의 개수는 } 66 \text{ 이다.} \\
\hline
\end{array}
$$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄱ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
ㄱ. 
$z^{2} = \left( \frac{-1+\sqrt{3}i}{2} \right)^{2} = \frac{-1-\sqrt{3}i}{2}$
$z^{3} = z^{2} \times z = \left( \frac{-1-\sqrt{3}i}{2} \right) \left( \frac{-1+\sqrt{3}i}{2} \right) = 1$ (참)

ㄴ. ㄱ에 의해 
$z^{4} + z^{5} = z + z^{2} = \frac{-1+\sqrt{3}i}{2} + \frac{-1-\sqrt{3}i}{2} = -1$ (참)

ㄷ. ㄱ에 의해 자연수 $k$ 에 대하여 
$z = z^{4} = z^{7} = \dots = z^{3k-2} = \frac{-1+\sqrt{3}i}{2}$
$z^{2} = z^{5} = z^{8} = \dots = z^{3k-1} = \frac{-1-\sqrt{3}i}{2}$
$z^{3} = z^{6} = z^{9} = \dots = z^{3k} = 1$

(ⅰ) $n = 3k-2$ 일 때, 
$z^{n} = z^{3k-2} = \frac{-1+\sqrt{3}i}{2}$
$z^{2n} = z^{3(2k-1)-1} = \frac{-1-\sqrt{3}i}{2}$
$z^{3n} = z^{3(3k-2)} = 1$
$z^{4n} = z^{3(4k-2)-2} = \frac{-1+\sqrt{3}i}{2}$
$z^{5n} = z^{3(5k-3)-1} = \frac{-1-\sqrt{3}i}{2}$
$z^{n} + z^{2n} + z^{3n} + z^{4n} + z^{5n} = -1$ 을 만족시키는 $100$ 이하의 모든 자연수 $n$ 은 $1, 4, 7, \dots, 100$ 이고 그 개수는 $34$

(ⅱ) $n = 3k-1$ 일 때, 
$z^{n} = z^{3k-1} = \frac{-1-\sqrt{3}i}{2}$
$z^{2n} = z^{3(2k)-2} = \frac{-1+\sqrt{3}i}{2}$
$z^{3n} = z^{3(3k-1)} = 1$
$z^{4n} = z^{3(4k-1)-1} = \frac{-1-\sqrt{3}i}{2}$
$z^{5n} = z^{3(5k-1)-2} = \frac{-1+\sqrt{3}i}{2}$
$z^{n} + z^{2n} + z^{3n} + z^{4n} + z^{5n} = -1$ 을 만족시키는 $100$ 이하의 모든 자연수 $n$ 은 $2, 5, 8, \dots, 98$ 이고 그 개수는 $33$

(ⅲ) $n = 3k$ 일 때, 
$z^{n} = z^{2n} = z^{3n} = z^{4n} = z^{5n} = 1$
$z^{n} + z^{2n} + z^{3n} + z^{4n} + z^{5n} = 5$
$z^{n} + z^{2n} + z^{3n} + z^{4n} + z^{5n} = -1$ 을 만족시키는 $100$ 이하의 자연수는 없다.

(ⅰ), (ⅱ), (ⅲ)에 의해 모든 자연수 $n$ 의 개수는 $34 + 33 = 67$ (거짓)

따라서 옳은 것은 ㄱ, ㄴ이다.
