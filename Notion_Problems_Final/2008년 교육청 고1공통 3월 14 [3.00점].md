## 문제
그림은 점 $(1, 0)$ 을 지나는 이차함수 $y = ax^{2} + bx + c$ 의 그래프를 나타낸 것이다. 이때 <보기>에서 옳은 것을 모두 고른 것은?

$$ \begin{array}{|l|} \hline \text{<보기>} \\ \text{ㄱ. } b > 0 \\ \text{ㄴ. } ab + c > 0 \\ \text{ㄷ. } a - b + c > 0 \\ \hline \end{array} $$



> [!example] 📸 (그림/그래프)

## 보기
① $ㄴ$   ② $ $   ③ $ $   ④ $ㄴ, ㄷ$   ⑤ $ $

## 풀이
ㄱ. 그래프가 위로 볼록하므로 $a < 0$ 이고, 축이 $y$ 축의 왼쪽에 있으므로 $-\frac{b}{2a} < 0$ 이다. 따라서 $b < 0$ 이다. (거짓)

ㄴ. $y$ 절편이 양수이므로 $c > 0$ 이다. $a < 0, b < 0$ 이므로 $ab > 0$ 이다. 따라서 $ab + c > 0$ 이다. (참)

ㄷ. 그래프에서 $x = -1$ 일 때 $y > 0$ 이므로 $a(-1)^{2} + b(-1) + c = a - b + c > 0$ 이다. (참)

이상에서 옳은 것은 $ㄴ, ㄷ$ 이다.
