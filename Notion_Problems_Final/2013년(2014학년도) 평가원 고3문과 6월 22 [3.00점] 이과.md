## 문제
등차수열 $\{ a_n \}$ 에 대하여 $a_3 = 8$, $a_6 - a_4 = 12$ 일 때, $a_6$ 의 값을 구하시오.

## 보기

## 풀이
정답: $26$

등차수열 $\{ a_n \}$ 의 공차를 $d$ 라 하면
$a_6 - a_4 = 12$ 에서 $2d = 12$ 이므로 $d = 6$
따라서 $a_6 = a_3 + 3d = 8 + 3 \times 6 = 26$

[다른 풀이]
등차수열 $\{ a_n \}$ 의 첫째항을 $a$, 공차를 $d$ 라 하면
$a_3 = 8$ 에서 $a + 2d = 8$ $\cdots \cdots$ $㉠$
$a_6 - a_4 = 12$ 에서 $(a + 5d) - (a + 3d) = 12$
$2d = 12$ 이므로 $d = 6$
$d = 6$ 을 $㉠$ 에 대입하면 $a + 12 = 8$ 이므로 $a = -4$
따라서 $a_6 = -4 + (6 - 1) \times 6 = 26$
