## 문제
자연수 $n$에 대하여 $f(n) = \left( \frac{1-i}{1+i} \right)^{n} + \left( \frac{1+i}{1-i} \right)^{n}$이라 할 때, $f(1) + f(2) + f(3) + \dots + f(19) + f(20)$의 값은? (단, $i = \sqrt{-1}$이다.)

## 보기
① $-20$   ② $-2$   ③ $0$   ④ $2$   ⑤ $20$

## 풀이
$\frac{1-i}{1+i} = \frac{(1-i)^{2}}{(1+i)(1-i)} = \frac{-2i}{2} = -i$이고,
$\frac{1+i}{1-i} = \frac{(1+i)^{2}}{(1-i)(1+i)} = \frac{2i}{2} = i$이므로
$f(n) = (-i)^{n} + i^{n}$이다.

$f(1) = (-i) + i = 0$
$f(2) = (-i)^{2} + i^{2} = -1 - 1 = -2$
$f(3) = (-i)^{3} + i^{3} = i - i = 0$
$f(4) = (-i)^{4} + i^{4} = 1 + 1 = 2$

이때 $f(n)$은 $4$를 주기로 반복되며, $f(1) + f(2) + f(3) + f(4) = 0 + (-2) + 0 + 2 = 0$이다.
따라서 $f(1) + f(2) + f(3) + \dots + f(19) + f(20)$의 값은
$5(f(1) + f(2) + f(3) + f(4)) = 5 \times 0 = 0$이다.
