## 문제
다항식 $x^{3} - 4x^{2} + 7x + 4$를 이차항의 계수가 $1$인 서로 다른 두 이차식 $A(x)$, $B(x)$로 나눈 나머지가 모두 $2x + 6$이다. 다항식 $A(x) + B(x)$를 $x - 4$로 나눈 나머지는?

## 보기
① $11$   ② $12$   ③ $13$   ④ $14$   ⑤ $15$

## 풀이
다항식 $x^{3} - 4x^{2} + 7x + 4$를 두 이차식 $A(x)$, $B(x)$로 나눈 나머지가 $2x + 6$으로 같으므로 $A(x)$, $B(x)$는 $(x^{3} - 4x^{2} + 7x + 4) - (2x + 6)$의 인수이다.

$(x^{3} - 4x^{2} + 7x + 4) - (2x + 6)$
$= x^{3} - 4x^{2} + 5x - 2$
$= (x - 1)^{2} (x - 2)$

두 이차식 $A(x)$, $B(x)$는 위 식의 서로 다른 인수이고 이차항의 계수가 $1$이므로
$A(x) = (x - 1)^{2}$, $B(x) = (x - 1)(x - 2)$ 또는 $A(x) = (x - 1)(x - 2)$, $B(x) = (x - 1)^{2}$ 이다.

$\therefore A(x) + B(x) = (x - 1)^{2} + (x - 1)(x - 2)$

다항식 $A(x) + B(x)$를 $x - 4$로 나눈 나머지는 다항식 $A(x) + B(x)$의 $x$에 $4$를 대입한 값과 같다.

$\therefore A(4) + B(4) = (4 - 1)^{2} + (4 - 1)(4 - 2)$
$= 3^{2} + 3 \times 2$
$= 9 + 6 = 15$

정답: ⑤
