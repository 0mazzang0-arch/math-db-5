## 문제
좌표평면 위의 두 점 $A(0, 0)$, $B(5, 0)$과 $1$보다 큰 자연수 $n$에 대하여 $\overline{AP} : \overline{PB} = 1:n$을 만족하는 점 $P(x, y)$들의 집합을 $T_n$이라 하자. 집합 $T_n$의 임의의 두 원소 $P$, $Q$에 대하여 $\overline{PQ}$의 최댓값을 $M(n)$이라고 할 때, $\sum_{n=2}^{\infty} \frac{10M(n)}{n}$의 값을 구하시오.

## 풀이
점 $P(x, y)$에 대하여 $\overline{AP} : \overline{PB} = 1:n$이므로
$\sqrt{x^2 + y^2} : \sqrt{(x-5)^2 + y^2} = 1:n$이다.
이를 정리하면
$n^2(x^2 + y^2) = (x-5)^2 + y^2$
$(n^2 - 1)x^2 + 10x + (n^2 - 1)y^2 = 25$
$x^2 + \frac{10}{n^2 - 1}x + y^2 = \frac{25}{n^2 - 1}$
$(x + \frac{5}{n^2 - 1})^2 + y^2 = \frac{25}{n^2 - 1} + \frac{25}{(n^2 - 1)^2}$
$(x + \frac{5}{n^2 - 1})^2 + y^2 = (\frac{5n}{n^2 - 1})^2$ 이다.

집합 $T_n$의 임의의 두 원소 $P$, $Q$에 대하여 $\overline{PQ}$의 최댓값 $M(n)$은 이 원의 지름과 같으므로
$M(n) = \frac{10n}{n^2 - 1}$이다.

따라서 구하는 값은
$\sum_{n=2}^{\infty} \frac{10M(n)}{n} = \sum_{n=2}^{\infty} \frac{100}{n^2 - 1}$
$= 50 \sum_{n=2}^{\infty} \left( \frac{1}{n-1} - \frac{1}{n+1} \right)$
$= 50 \left\{ \left( 1 - \frac{1}{3} \right) + \left( \frac{1}{2} - \frac{1}{4} \right) + \left( \frac{1}{3} - \frac{1}{5} \right) + \cdots \right\}$
$= 50 \left( 1 + \frac{1}{2} \right) = 75$이다.

[정답] $75$
