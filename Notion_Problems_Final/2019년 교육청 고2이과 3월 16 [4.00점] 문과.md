## 문제
함수 $f(x)=x^2-(k+1)x+2k$ ($k$는 $2$가 아닌 실수)에 대하여 함수 $g(x)$를 $g(x)=(f \circ f)(x)$라 하자. 다음은 다항식 $g(x)-x$는 다항식 $f(x)-x$로 나누어떨어짐을 보이는 과정이다.

$$
\begin{array}{|l|}
\hline
\text{모든 실수 } x\text{에 대하여} \\
f(x)-x = (x-k)((가)) \\
\text{이다. 함수 } g(x) = (f \circ f)(x) = f(f(x))\text{에 대하여} \\
g(k) = f(f(k)) = (나) \\
g(2) = f(f(2)) = (다) \\
\text{다항식 } g(x)-x\text{는 } x-k\text{와 }(가)\text{를 인수로 가지므로 다항식 } g(x)-x\text{는} \\
\text{다항식 } f(x)-x\text{로 나누어떨어진다.} \\
\hline
\end{array}
$$

위의 $(가)$, $(나)$에 알맞은 식을 각각 $p(x)$, $q(k)$라 하고, $(다)$에 알맞은 수를 $a$라 할 때, $p(5)+q(4)+a$의 값은?

## 보기
① $9$   ② $10$   ③ $11$   ④ $12$   ⑤ $13$

## 풀이
[정답] ①

함수 $f(x)=x^2-(k+1)x+2k$ ($k$는 $2$가 아닌 실수)에서 모든 실수 $x$에 대하여
$f(x)-x = x^2-(k+2)x+2k = (x-k)(x-2)$
이다. 이때 $f(k)-k=0$, $f(2)-2=0$에서
$f(k)=k$, $f(2)=2$
함수 $g(x)=(f \circ f)(x)=f(f(x))$에 대하여
$g(k)=f(f(k))=f(k)=k$
$g(2)=f(f(2))=f(2)=2$
$g(k)-k=0$, $g(2)-2=0$에서 다항식 $g(x)-x$는 $x-k$와 $x-2$를 인수로 가지므로 다항식 $g(x)-x$는 다항식 $(x-k)(x-2)$, 즉 $f(x)-x$로 나누어떨어진다.

$(가)$ $p(x)=x-2$, $(나)$ $q(k)=k$, $(다)$ $a=2$이므로
$p(5)+q(4)+a = (5-2)+4+2 = 9$
