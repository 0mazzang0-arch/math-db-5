## 문제
서로 다른 두 실수 $a$, $b$에 대하여 두 집합 $\langle a, b \rangle$, $[a, b]$를 
$$\langle a, b \rangle = \{ \{ a \}, \{ a, b \} \}$$
$$[a, b] = \{ a \} \cup \{ b \}$$
와 같이 정의할 때, <보기>에서 옳은 것을 모두 고르면?

ㄱ. $\langle a, b \rangle = \langle b, a \rangle$
ㄴ. 집합 $\langle a, b \rangle$의 부분집합의 개수는 $4$개이다.
ㄷ. $[a, b] = [c, d]$이면 $a = c$, $b = d$이다.

## 보기
① ㄱ   ② ㄴ   ③ ㄷ   ④ ㄱ, ㄴ   ⑤ ㄴ, ㄷ

## 풀이
ㄱ. $\langle a, b \rangle = \{ \{ a \}, \{ a, b \} \}$, $\langle b, a \rangle = \{ \{ b \}, \{ b, a \} \}$이므로 $\langle a, b \rangle \neq \langle b, a \rangle$ (거짓)

ㄴ. 집합 $\langle a, b \rangle = \{ \{ a \}, \{ a, b \} \}$에서 $a \neq b$이므로 원소의 개수는 $\{ a \}$와 $\{ a, b \}$로 $2$개이다. 따라서 부분집합의 개수는 $2^{2} = 4$ (개)이다. (참)

ㄷ. $[a, b] = \{ a \} \cup \{ b \} = \{ a, b \}$, $[c, d] = \{ c \} \cup \{ d \} = \{ c, d \}$이다.
$[a, b] = [c, d]$이면 $\{ a, b \} = \{ c, d \}$이지만, 집합에서는 원소의 순서가 상관없으므로 $a = d$, $b = c$인 경우도 존재한다. 따라서 반드시 $a = c$, $b = d$인 것은 아니다. (거짓)

정답: ②
