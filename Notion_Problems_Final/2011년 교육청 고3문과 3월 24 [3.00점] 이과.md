## 문제
수열 $\{ a_{n} \}$의 계차수열 $\{ b_{n} \}$에 대하여 $\lim_{n \to \infty} a_{n} = 100$, $b_{n} = \frac{4}{n^{2} + 2n}$ 일 때, $a_{1}$의 값을 구하시오.

## 보기


## 풀이
$b_{n} = \frac{4}{n(n+2)} = 2 \left( \frac{1}{n} - \frac{1}{n+2} \right)$

$a_{n} = a_{1} + \sum_{k=1}^{n-1} b_{k}$
$= a_{1} + \sum_{k=1}^{n-1} 2 \left( \frac{1}{k} - \frac{1}{k+2} \right)$
$= a_{1} + 2 \{ (1 - \frac{1}{3}) + (\frac{1}{2} - \frac{1}{4}) + \cdots + (\frac{1}{n-1} - \frac{1}{n+1}) \}$
$= a_{1} + 2 (1 + \frac{1}{2} - \frac{1}{n} - \frac{1}{n+1})$

$\lim_{n \to \infty} a_{n} = \lim_{n \to \infty} \{ a_{1} + 2 (1 + \frac{1}{2} - \frac{1}{n} - \frac{1}{n+1} ) \}$
$= a_{1} + 3 = 100$
이므로 $a_{1} = 97$ 이다.

정답: $97$
