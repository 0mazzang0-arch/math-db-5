## 문제
공차가 $d$ ($d \neq 0$)인 등차수열 $\{ a_{n} \}$에 대하여 수열 $\{ T_{n} \}$을
$$T_{n} = a_{1} - a_{2} + a_{3} - a_{4} + \cdots + (-1)^{n-1} a_{n} \quad (n = 1, 2, 3, \dots )$$
으로 정의할 때, <보기>에서 옳은 것을 모두 고른 것은?

$$
\begin{array}{|l|}
\hline
\text{ㄱ. } T_{4} = 2d \\
\text{ㄴ. } T_{5} = a_{3} \\
\text{ㄷ. 수열 } \{ T_{2n} \} \text{은 등차수열이다.} \\
\hline
\end{array}
$$

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄱ, ㄷ   ⑤ ㄴ, ㄷ

## 풀이
ㄱ. $T_{4} = ( a_{1} - a_{2} ) + ( a_{3} - a_{4} ) = -d + ( -d ) = -2d$ (거짓)

ㄴ. $T_{5} = T_{4} + a_{5} = -2d + a_{1} + 4d = a_{1} + 2d = a_{3}$ (참)

ㄷ. $T_{2n} = ( a_{1} - a_{2} ) + ( a_{3} - a_{4} ) + \cdots + ( a_{2n-1} - a_{2n} )$
$T_{2} = -d$
$T_{4} = -2d$
$\cdots, T_{2n} = -nd$
따라서 수열 $\{ T_{2n} \}$은 첫째항이 $-d$이고 공차가 $-d$인 등차수열이다. (참)

그러므로 옳은 것은 ㄴ, ㄷ이다.
