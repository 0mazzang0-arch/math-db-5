## 문제
두 자연수 $a$, $b$에 대하여 $0 \le x \le 8$에서 정의된 함수 $f(x) = a \sin \frac{\pi}{4}x + b$이다. 집합 $\{x \mid f(x) = n, n \text{은 자연수}\}$의 모든 원소의 합이 $22$일 때, $a^2 + b^2$의 값을 구하시오.

## 풀이
함수 $y = a \sin \frac{\pi}{4}x + b$의 주기는 $8$이고, 치역은 $\{y \mid b - a \le y \le b + a\}$이다.
정수 $k$에 대하여 $A_k = \{x \mid f(x) = k\}$라 할 때, $i \ne j$이면 $A_i \cap A_j = \emptyset$이다.
임의의 집합 $X (\ne \emptyset)$에 대하여 집합 $X$의 모든 원소의 합을 $S(X)$라 하자.

$A_{b+a} = \{2\}$, $S(A_{b+a}) = 2$
$A_b = \{0, 4, 8\}$, $S(A_b) = 12$
$A_{b-a} = \{6\}$, $S(A_{b-a}) = 6$

집합 $A = \{x \mid f(x) = n, n \text{은 자연수}\}$라 할 때, $(A_b \cup A_{b+a}) \subset A$이고 $S(A_b \cup A_{b+a}) = 14$이다.

(ⅰ) $b = 1$인 경우
(ⅰ-①) $a = 1$일 때
$f(x) = \sin \frac{\pi}{4}x + 1$이고 $A_b \cup A_{b+a} = A$이다.
$S(A) = 14$이므로 조건을 만족시키지 않는다.

(ⅰ-②) $a \ge 2$일 때
$b < m < b + a$인 정수 $m$에 대하여 $S(A_m) = 4$이다.
원소의 합이 $22$가 되어야 하므로 $14 + 4 \times (\text{정수 } m \text{의 개수}) = 22$에서 정수 $m$의 개수는 $2$이다.
$(b + a) - b - 1 = 2$에서 $a = 3$이다.
$A_b \cup A_{b+1} \cup A_{b+2} \cup A_{b+3} = A$, $S(A) = 22$이므로 조건을 만족시킨다.

(ⅱ) $b \ge 2$인 경우
(ⅱ-①) $a = 1$일 때
$f(x) = \sin \frac{\pi}{4}x + b$이고 $A_{b-a} \cup A_b \cup A_{b+a} = A$이다.
$S(A) = 6 + 12 + 2 = 20$이므로 조건을 만족시키지 않는다.

(ⅱ-②) $a \ge 2$일 때
$b - a < b - 1 < b$이므로 정수 $b - 1$에 대하여 $S(A_{b-1}) = 12$이다.
$(A_{b-1} \cup A_b \cup A_{b+a}) \subset A$이고 $S(A_{b-1} \cup A_b \cup A_{b+a}) = 12 + 12 + 2 = 26$이다.
$S(A) > 22$이므로 조건을 만족시키지 않는다.

(ⅰ), (ⅱ)에 의하여 $a = 3$, $b = 1$이다.
$a^2 + b^2 = 3^2 + 1^2 = 10$
