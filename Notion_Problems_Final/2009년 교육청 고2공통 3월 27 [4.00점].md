## 문제
> [!example] 📸 (그림/그래프)

그림과 같이 두 점 ( $4$ , $0$ ), ( $0$ , $2$ )를 지나는 직선 $l$이 있다. 직선 $l$ 위의 임의의 점 ( $x$ , $y$ )에 대하여 등식 $x^{2} + ay^{2} + bx + c = 0$이 성립하도록 실수 $a$, $b$, $c$를 정할 때, $| a | + | b | + | c |$의 값을 구하시오.

## 풀이
직선 $l$의 $x$절편, $y$절편이 각각 $4$, $2$이므로 직선 $l$의 방정식은
$$\frac{x}{4} + \frac{y}{2} = 1$$
$$\therefore y = -\frac{1}{2}x + 2 \cdots \cdots ㉠$$
$㉠$을 주어진 등식에 대입하면
$$x^{2} + a \left( -\frac{1}{2}x + 2 \right)^{2} + bx + c = 0$$
$$\left( 1 + \frac{a}{4} \right) x^{2} - ( 2a - b ) x + ( 4a + c ) = 0$$
위 식은 $x$에 대한 항등식이므로
$$\begin{array}{l} 1 + \frac{a}{4} = 0, \\ 2a - b = 0, \\ 4a + c = 0 \end{array}$$
위 세 식을 연립하면
$$a = -4, b = -8, c = 16$$
$$\therefore | a | + | b | + | c | = 28$$

**[다른 풀이]**
직선 위의 어떤 점을 대입하여도 등식이 성립해야 하므로 직선 위의 점 ( $0$ , $2$ ), ( $2$ , $1$ ), ( $4$ , $0$ )을 대입하면
$$\begin{array}{l} 4a + c = 0 & \cdots \cdots ㉠ \\ a + 2b + c = -4 & \cdots \cdots ㉡ \\ 4b + c = -16 & \cdots \cdots ㉢ \end{array}$$
$㉠, ㉡, ㉢$을 연립하면
$$a = -4, b = -8, c = 16$$
$$\therefore | a | + | b | + | c | = 28$$

**정답: $28$**
