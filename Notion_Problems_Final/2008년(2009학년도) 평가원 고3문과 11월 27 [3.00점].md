## 문제
자연수 $n$ 에 대하여 $\log n$ 의 소수부분을 $f(n)$ 이라 할 때, 집합 $A = \{ f(n) \mid 1 \le n \le 150, n \text{ 은 자연수} \}$ 의 원소의 개수는?

## 보기
① $131$   ② $133$   ③ $135$   ④ $137$   ⑤ $139$

## 풀이
[정답] ③

$1 \le n \le 9$ 일 때, $\log n$ 의 정수부분은 $0$ 이므로 소수부분 $f(n)$ 의 값은 
$\log 1, \log 2, \log 3, \dots, \log 9 \dots ㉠$ 
의 $9$ 개이고, 이들은 서로 다른 값이다. 

$10 \le n \le 99$ 일 때, $\log n$ 의 정수부분이 $1$ 이므로 소수부분 $f(n)$ 의 값은 
$\log \frac{10}{10}, \log \frac{11}{10}, \log \frac{12}{10}, \dots, \log \frac{99}{10} \dots ㉡$ 
의 $90$ 개이고, 이들은 서로 다른 값이며, 이들 중 
$\log \frac{10}{10}, \log \frac{20}{10}, \log \frac{30}{10}, \dots, \log \frac{90}{10}$ 
의 $9$ 개는 $㉠$ 의 값과 중복된다. 

$100 \le n \le 150$ 일 때, $\log n$ 의 정수부분은 $2$ 이므로 소수부분 $f(n)$ 의 값은 
$\log \frac{100}{100}, \log \frac{101}{100}, \log \frac{102}{100}, \dots, \log \frac{150}{100}$ 
의 $51$ 개이고, 이들은 서로 다른 값이며, 이들 중 
$\log \frac{100}{100}, \log \frac{110}{100}, \log \frac{120}{100}, \dots, \log \frac{150}{100}$ 
의 $6$ 개는 $㉠, ㉡$ 의 값과 중복된다. 

따라서 집합 $A$ 의 원소의 개수는 
$9 + (90 - 9) + (51 - 6) = 135$ (개) 
이다.
