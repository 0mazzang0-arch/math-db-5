## 문제
곡선 $y=x^{2}$과 직선 $y=nx-2$의 두 교점을 각각 $A$, $B$라 하자. 원점 $O$와 두 점 $A$, $B$를 잇는 두 직선 $OA$, $OB$의 기울기를 각각 $a_{n}$, $b_{n}$이라 할 때, $\sum_{n=3}^{20} (a_{n}+b_{n})$의 값은?

## 보기
① $195$   ② $199$   ③ $203$   ④ $207$   ⑤ $211$

## 풀이
곡선 $y=x^{2}$과 직선 $y=nx-2$의 두 교점을 각각 $A(\alpha, \alpha^{2})$, $B(\beta, \beta^{2})$이라 하면 
$a_{n} = \frac{\alpha^{2}}{\alpha} = \alpha$
$b_{n} = \frac{\beta^{2}}{\beta} = \beta$
$a_{n} + b_{n} = \alpha + \beta$

한편 $y=x^{2}$과 $y=nx-2$를 연립하면
$x^{2} = nx-2$
$x^{2} - nx + 2 = 0$
이차방정식의 두 근이 $\alpha$, $\beta$이므로 근과 계수의 관계에 의하여
$\alpha + \beta = n$

$\therefore a_{n} + b_{n} = n$

$\therefore \sum_{n=3}^{20} (a_{n} + b_{n}) = \sum_{n=3}^{20} n$
$= \sum_{n=1}^{20} n - (1+2)$
$= \frac{20 \times 21}{2} - (1+2)$
$= 207$

**[다른 풀이]**
곡선 $y=x^{2}$과 직선 $y=nx-2$의 두 교점을 각각 $A(\alpha, n\alpha-2)$, $B(\beta, n\beta-2)$라 하면 
$a_{n} = \frac{n\alpha-2}{\alpha} = n - \frac{2}{\alpha}$
$b_{n} = \frac{n\beta-2}{\beta} = n - \frac{2}{\beta}$

$a_{n} + b_{n} = 2n - \left( \frac{2}{\alpha} + \frac{2}{\beta} \right) = 2n - \frac{2(\alpha+\beta)}{\alpha\beta}$

한편 $y=x^{2}$과 $y=nx-2$를 연립하면
$x^{2} = nx-2$
$x^{2} - nx + 2 = 0$
이차방정식의 두 근이 $\alpha$, $\beta$이므로 근과 계수의 관계에 의하여
$\alpha + \beta = n$, $\alpha\beta = 2$

$\therefore a_{n} + b_{n} = 2n - \frac{2n}{2} = n$

$\therefore \sum_{n=3}^{20} (a_{n} + b_{n}) = \sum_{n=3}^{20} n$
$= \sum_{n=1}^{20} n - (1+2)$
$= \frac{20 \times 21}{2} - (1+2)$
$= 207$

정답: ④
