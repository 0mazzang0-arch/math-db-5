## 문제
다음 등식을 만족하는 소수 $p$는 $2$개 존재한다.
$$\frac{1}{p} = \sum_{n=1}^{\infty} \left( \frac{a}{6^{2n-1}} + \frac{b}{6^{2n}} \right)$$
($0 \le a < 6$, $0 \le b < 6$, $a$와 $b$는 정수이다.)
위 등식을 만족하는 두 소수의 합을 구하시오.

## 보기

## 풀이
$$\frac{1}{p} = \sum_{n=1}^{\infty} \left( \frac{a}{6^{2n-1}} + \frac{b}{6^{2n}} \right)$$
$$= \frac{a}{6} + \frac{b}{6^{2}} + \frac{a}{6^{3}} + \frac{b}{6^{4}} + \cdots$$
$$= \left( \frac{a}{6} + \frac{a}{6^{3}} + \frac{a}{6^{5}} + \cdots \right) + \left( \frac{b}{6^{2}} + \frac{b}{6^{4}} + \frac{b}{6^{6}} + \cdots \right)$$
$$= \frac{\frac{a}{6}}{1 - \frac{1}{36}} + \frac{\frac{b}{36}}{1 - \frac{1}{36}}$$
$$= \frac{6a + b}{35}$$
따라서 $\frac{1}{p} = \frac{6a + b}{35}$에서 $p = \frac{35}{6a + b}$이다.
($0 \le a < 6$, $0 \le b < 6$, $a$, $b$는 정수이므로)
그런데 $p$가 소수이므로 $6a + b$는 $35$의 약수 중 $p$를 소수로 만드는 값이어야 한다.
$1)$ $6a + b = 5$일 때, $p = 7$ (소수)
$2)$ $6a + b = 7$일 때, $p = 5$ (소수)
따라서 만족하는 소수 $p$는 $5$와 $7$로 $2$개 존재한다.
그러므로 두 소수의 합은 $5 + 7 = 12$이다.

정답: $12$
