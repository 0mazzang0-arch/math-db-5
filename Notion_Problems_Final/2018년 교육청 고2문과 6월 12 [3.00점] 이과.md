## 문제
두 함수 
$$ f ( x ) = \begin{cases} ( x - 1 )^{2} & ( x \neq 1 ) \\ 1 & ( x = 1 ) \end{cases} $$, 
$g ( x ) = 2x + k$ 에 대하여 함수 $f ( x ) g ( x )$ 가 실수 전체의 집합에서 연속이 되도록 하는 상수 $k$ 의 값은?

## 보기
① $-2$   ② $-1$   ③ $0$   ④ $1$   ⑤ $2$

## 풀이
$f ( x )$ 가 $x \neq 1$ 인 모든 실수에서 연속이고, $g ( x )$ 는 실수 전체의 집합에서 연속이므로 $f ( x ) g ( x )$ 가 실수 전체에서 연속이 되기 위해서는 $x = 1$ 에서 연속이면 된다. 

$\lim_{x \to 1} f ( x ) g ( x ) = \lim_{x \to 1} ( x - 1 )^{2} ( 2x + k ) = 0$ 
$f ( 1 ) g ( 1 ) = 1 \times ( 2 + k )$ 

$\lim_{x \to 1} f ( x ) g ( x ) = f ( 1 ) g ( 1 )$ 이므로 
$2 + k = 0$ 이다. 
따라서 $k = -2$ 이다.

정답: ①
