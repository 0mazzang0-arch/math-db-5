## 문제
등차수열 $\{ a_{n} \}$ 에 대하여 $S_{n} = \sum_{k=1}^{n} a_{k}$, $T_{n} = \sum_{k=1}^{n} | a_{k} |$ 라 할 때, $S_{n}$, $T_{n}$ 은 다음 조건을 만족시킨다.

$$
\begin{array}{|l|}
\hline
(가) \ S_{7} = T_{7} \\
(나) \ 6 \text{ 이상의 모든 자연수 } n \text{ 에 대하여 } S_{n} + T_{n} = 84 \text{ 이다.} \\
\hline
\end{array}
$$

$T_{15}$ 의 값은?

## 보기
① $96$   ② $102$   ③ $108$   ④ $114$   ⑤ $120$

## 풀이
[정답] ④

조건 (가), (나)에 의하여 $S_{7} = T_{7}$ 이고 $S_{7} + T_{7} = 84$ 이므로 $S_{7} = 42$ 이다.
$S_{7} = T_{7}$ 이므로 $7$ 이하의 모든 자연수 $n$ 에 대하여 $a_{n} \ge 0 \cdots \cdots$ ㉠
조건 (나)에 의하여 $6$ 이상의 모든 자연수 $n$ 에 대하여 
$(S_{n+1} + T_{n+1}) - (S_{n} + T_{n}) = 0$
$a_{n+1} + | a_{n+1} | = 0$
$a_{n+1} \le 0 \cdots \cdots$ ㉡
㉠, ㉡에 의하여 $0 \le a_{7} \le 0$ 이므로 $a_{7} = 0$ 이다.
수열 $\{ a_{n} \}$ 의 첫째항을 $a$, 공차를 $d$ 라 하자.
$S_{7} = \frac{7(2a + 6d)}{2} = 42$, $a_{7} = a + 6d = 0$ 에서 $a = 12$, $d = -2$ 이다.
$\therefore a_{n} = 14 - 2n$
$S_{15} = \frac{15 \times (24 - 28)}{2} = -30$
$\therefore S_{15} + T_{15} = 84$
따라서 $T_{15} = 84 - S_{15} = 114$ 이다.
