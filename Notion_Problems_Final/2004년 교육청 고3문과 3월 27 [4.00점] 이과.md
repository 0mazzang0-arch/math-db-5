## 문제
다음 두 조건을 만족시키는 실수 $x$를 모두 곱한 값을 $M$이라 할 때, $\log_{10} M$의 값을 구하시오. (단, $[x]$는 $x$보다 크지 않은 최대의 정수이다.)

Ⅰ. $[\log_{10} x] = 6$
Ⅱ. $\log_{10} x^{2} - [\log_{10} x^{2}] = \log_{10} \frac{1}{x} - [\log_{10} \frac{1}{x}]$

## 보기

## 풀이
Ⅰ에서 
$6 \le \log_{10} x < 7$ … ㉠ 
Ⅱ에서 
$\log_{10} x^{2}$과 $\log_{10} \frac{1}{x}$의 소수부분이 같으므로 
$\log_{10} x^{2} - \log_{10} \frac{1}{x} = 3 \log_{10} x = (\text{정수})$ 
㉠에서 
$18 \le 3 \log_{10} x < 21$ 이므로 
$3 \log_{10} x = 18, 19, 20$ 
$\log_{10} x = 6, \frac{19}{3}, \frac{20}{3}$ 
$x = 10^{6}, 10^{\frac{19}{3}}, 10^{\frac{20}{3}}$ 
$\log_{10} M = \log_{10} (10^{6} \times 10^{\frac{19}{3}} \times 10^{\frac{20}{3}})$ 
∴ $\log_{10} M = 19$

정답: $19$
