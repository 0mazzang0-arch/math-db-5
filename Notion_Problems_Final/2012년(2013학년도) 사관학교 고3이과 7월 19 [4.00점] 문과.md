## 문제
닫힌구간 $[0, \frac{\pi}{2}]$에서 정의된 함수 $f(x) = \frac{\sin 2x}{1 + \sin x}$에 대하여 <보기>에서 옳은 것만을 있는 대로 고른 것은?

$$
\begin{array}{|l|}
\hline
\text{ㄱ. } f(x) \ge 0 \\
\text{ㄴ. } f'(c) = 0 \text{인 } c\text{가 열린구간 } (0, \frac{\pi}{2}) \text{에 존재한다.} \\
\text{ㄷ. 함수 } f(x)\text{의 그래프와 } x\text{축으로 둘러싸인 부분의 넓이는 } 2 - 2 \ln 2\text{이다.} \\
\hline
\end{array}
$$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄱ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
ㄱ. 구간 $[0, \frac{\pi}{2}]$에서 $0 \le \sin 2x \le 1$이고 $1 \le 1 + \sin x \le 2$이므로 
$f(x) = \frac{\sin 2x}{1 + \sin x} \ge 0$이다. (참) 

ㄴ. $f(0) = 0$, $f(\frac{\pi}{2}) = 0$이므로 롤의 정리에 의해서 
$f'(c) = 0$인 $c$가 열린 구간 $(0, \frac{\pi}{2})$에 존재한다. (참) 

ㄷ. 열린 구간 $(0, \frac{\pi}{2})$에서 $\sin 2x \neq 0$이므로 열린 구간 $(0, \frac{\pi}{2})$에서 $f(x) > 0$이다. 
ㄴ에서 $f(0) = 0$, $f(\frac{\pi}{2}) = 0$이므로 함수 $f(x)$의 그래프와 $x$축으로 둘러싸인 부분의 넓이는 $\int_0^{\frac{\pi}{2}} f(x) dx$이다. 

$1 + \sin x = t$라 하면 $\cos x dx = dt$이고 
$x = 0$일 때 $t = 1$이고 $x = \frac{\pi}{2}$일 때 $t = 2$이므로 
$$
\begin{array}{l}
\int_0^{\frac{\pi}{2}} f(x) dx \\
= \int_0^{\frac{\pi}{2}} \frac{\sin 2x}{1 + \sin x} dx \\
= \int_0^{\frac{\pi}{2}} \frac{2 \sin x \cos x}{1 + \sin x} dx \\
= \int_1^2 \frac{2(t-1)}{t} dt \\
= 2 \int_1^2 (1 - \frac{1}{t}) dt \\
= 2 [t - \ln t]_1^2 \\
= 2(1 - \ln 2) \\
= 2 - 2 \ln 2
\end{array}
$$
(참) 
따라서 ㄱ, ㄴ, ㄷ 모두 옳다. 

**[다른 풀이]** 
ㄴ. $f'(x) = \frac{2 \cos 2x (1 + \sin x) - \sin 2x \cos x}{(1 + \sin x)^2}$ 
$f'(0) = 2$, $f'(\frac{\pi}{2}) = -1$이므로 중간값의 정리에 의해서 
$f'(c) = 0$인 $c$가 열린 구간 $(0, \frac{\pi}{2})$에 존재한다. (참) 

**[별해]** 
ㄱ. 주어진 구간에서 $1 + \sin x > 0$이고 $\sin^2 x \ge 0$이므로 $f(x) \ge 0$이다. (참) 
ㄴ. $f(\frac{\pi}{2}) = f(0) = 0$이므로 평균값의 정리에 의하여 $c$가 주어진 구간에 존재한다. (참) 
ㄷ. $\int_0^{\frac{\pi}{2}} \frac{\sin 2x}{1 + \sin x} dx$에서 $\sin x = t$로 치환하면 
$\int_0^{\frac{\pi}{2}} \frac{\sin 2x}{1 + \sin x} dx = \int_0^1 \frac{2t}{1 + t} dt = \int_0^1 (2 - \frac{2}{1 + t}) dt$ 
$= 2 - 2 \int_1^2 \frac{1}{t} dt = 2 - 2 \ln 2$이므로 (참)
