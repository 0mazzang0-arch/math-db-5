## 문제
$x$에 대한 연립부등식
$$ \begin{cases} ax^{2} + (a + b)x + a + b + 1 < 0 \\ (a + b)x^{2} + (a + b + 1)x + a < 0 \end{cases} $$
을 만족시키는 모든 $x$의 값의 범위가 $x < p$일 때, <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $a$, $b$, $p$는 실수이다.)

| <보기> |
| :--- |
| ㄱ. $a = -1$일 때, $p = -1$이다. |
| ㄴ. $b > 0$ |
| ㄷ. $a^{3} \le -1$ |

## 보기
① $ㄱ$   ② $ㄱ, ㄴ$   ③ $ㄱ, ㄷ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
ㄱ. $a = -1$일 때, 연립부등식은 다음과 같다.
$$ \begin{cases} -x^{2} + (b - 1)x + b < 0 \\ (b - 1)x^{2} + bx - 1 < 0 \end{cases} $$
(ⅰ) $-x^{2} + (b - 1)x + b < 0$의 해는
(ⅰ-1) $b > -1$일 때, $x < -1$ 또는 $x > b$
(ⅰ-2) $b < -1$일 때, $x < b$ 또는 $x > -1$
(ⅰ-3) $b = -1$일 때, $x \neq -1$인 모든 실수

(ⅱ) $(b - 1)x^{2} + bx - 1 < 0$의 해는
(ⅱ-1) $b - 1 > 0$이면 $(b - 1)x^{2} + bx - 1 = 0$의 판별식 $D = b^{2} + 4(b - 1) > 0$이므로 해가 $x < \alpha$ 또는 $x > \beta$ 형태가 되어 연립부등식의 해가 $x < p$가 될 수 없다.
(ⅱ-2) $b - 1 < 0$이면 이차부등식의 해는 $x < \alpha$ 또는 $x > \beta (\alpha < \beta)$, $x \neq \alpha$인 모든 실수, 또는 모든 실수 중 하나이므로 연립부등식의 해가 $x < p$가 될 수 없다.
(ⅱ-3) $b - 1 = 0$, 즉 $b = 1$이면 부등식은 $x - 1 < 0$이 되어 $x < 1$이다.
이때 (ⅰ)의 부등식은 $-x^{2} + 1 < 0$이므로 $x < -1$ 또는 $x > 1$이다.
따라서 연립부등식의 해는 $x < -1$이 되어 $p = -1$이다. (참)

ㄴ. 연립부등식의 해가 $x < p$이려면 두 부등식 중 적어도 하나는 일차부등식이어야 한다.
(ⅰ) $a = 0$인 경우
$$ \begin{cases} bx + b + 1 < 0 \\ bx^{2} + (b + 1)x < 0 \end{cases} $$
(ⅰ-1) $b > 0$일 때, $x < -\frac{b+1}{b}$와 $-\frac{b+1}{b} < x < 0$의 공통부분이 없으므로 해가 없다.
(ⅰ-2) $b = 0$일 때, $1 < 0$이 되므로 해가 없다.
(ⅰ-3) $b < 0$일 때, 연립부등식의 해는 $x > -\frac{b+1}{b}$ 또는 $x > 0$ 중 하나가 포함된 형태가 되어 $x < p$가 될 수 없다.
(ⅱ) $a + b = 0$인 경우
$$ \begin{cases} ax^{2} + 1 < 0 \\ x + a < 0 \end{cases} $$
(ⅱ-1) $a > 0$일 때, $ax^{2} + 1 > 0$이므로 해가 없다.
(ⅱ-2) $a < 0$일 때, $ax^{2} + 1 < 0$에서 $x < -\sqrt{-\frac{1}{a}}$ 또는 $x > \sqrt{-\frac{1}{a}}$이고, $x + a < 0$에서 $x < -a$이다.
이때 $-\sqrt{-\frac{1}{a}} < -a \le \sqrt{-\frac{1}{a}}$이면 연립부등식의 해가 $x < -\sqrt{-\frac{1}{a}}$가 된다.
$a + b = 0$이고 $a < 0$이므로 $b = -a > 0$이다. (참)

ㄷ. ㄴ에 의해 $a < 0$이고 $-a \le \sqrt{-\frac{1}{a}}$이므로 양변을 제곱하면
$a^{2} \le -\frac{1}{a}$
$a$가 음수이므로 양변에 $a$를 곱하면 부동호 방향이 바뀐다.
$a^{3} \ge -1$ (거짓)

따라서 옳은 것은 $ㄱ, ㄴ$이다.
