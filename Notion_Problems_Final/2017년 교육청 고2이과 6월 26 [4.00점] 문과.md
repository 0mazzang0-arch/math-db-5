## 문제
$1$이 아닌 두 양수 $a$, $b$에 대하여 $x = \log_{2} a$, $y = \log_{2} b$라 하면 
$x^{2} - 4xy + y^{2} = 0$이 성립한다. 
$\log_{8} a^{\frac{1}{y}} + \log_{8} b^{\frac{1}{x}}$의 값을 $k$라 할 때, $27k$의 값을 구하시오.

## 풀이
로그의 정의로부터 
$a = 2^{x}$, $b = 2^{y}$이다. 

$\log_{8} a^{\frac{1}{y}} + \log_{8} b^{\frac{1}{x}}$ 
$= \log_{8} 2^{\frac{x}{y}} + \log_{8} 2^{\frac{y}{x}}$ 
$= \log_{8} 2^{\frac{x}{y} + \frac{y}{x}}$ 
$= \log_{8} 2^{\frac{x^{2} + y^{2}}{xy}}$ 
$= \frac{x^{2} + y^{2}}{3xy}$ 
$= \frac{4xy}{3xy}$ 
$= \frac{4}{3}$ 
이므로 $k = \frac{4}{3}$이다. 따라서 
$27k = 36$ 

**[다른 풀이]** 

$\log_{8} a^{\frac{1}{y}} + \log_{8} b^{\frac{1}{x}}$ 
$= \log_{2^{3}} a^{\frac{1}{y}} + \log_{2^{3}} b^{\frac{1}{x}}$ 
$= \frac{1}{3y} \log_{2} a + \frac{1}{3x} \log_{2} b$ 
$= \frac{x}{3y} + \frac{y}{3x}$ 
$= \frac{x^{2} + y^{2}}{3xy}$ 
$= \frac{4}{3}$ 
이므로 $k = \frac{4}{3}$이다. 
따라서 $27k = 36$
