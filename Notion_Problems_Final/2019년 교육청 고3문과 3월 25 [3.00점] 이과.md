## 문제
첫째항이 $4$인 수열 $\{a_{n}\}$이 모든 자연수 $n$에 대하여 $a_{n+2} = a_{n+1} + a_{n}$을 만족시킨다. $a_{4} = 34$일 때, $a_{2}$의 값을 구하시오.

## 풀이
$a_{2} = p$라 하면
$a_{1} = 4$, $a_{2} = p$,
$a_{3} = a_{2} + a_{1} = p + 4$,
$a_{4} = a_{3} + a_{2} = (p + 4) + p = 2p + 4$
이므로 $2p + 4 = 34$에서 $p = 15$
따라서 $a_{2} = 15$

정답: $15$
