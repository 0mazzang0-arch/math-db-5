## 문제
실수 전체의 집합에서 미분가능한 함수 $f(x)$ 가 모든 실수 $x$ 에 대하여 $f(1+x)=f(1-x)$, $f(2+x)=f(2-x)$ 를 만족시킨다. 실수 전체의 집합에서 $f'(x)$ 가 연속이고, $\int_{2}^{5} f'(x) dx=4$ 일 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?

ㄱ. 모든 실수 $x$ 에 대하여 $f(x+2)=f(x)$ 이다.
ㄴ. $f(1)-f(0)=4$
ㄷ. $\int_{0}^{1} f(f(x)) f'(x) dx=6$ 일 때, $\int_{1}^{10} f(x) dx = \frac{27}{2}$ 이다.

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
ㄱ. $f(2+x) = f(2-x)$ 이고, $2-x = 1+(1-x)$ 이므로 $f(1+x)=f(1-x)$ 성질을 이용하면
$f(2+x) = f(2-x) = f(1+(1-x)) = f(1-(1-x)) = f(x)$
따라서 $f(x+2) = f(x)$ 이므로 옳은 것이다.

ㄴ. $\int_{2}^{5} f'(x) dx = [f(x)]_{2}^{5} = f(5)-f(2)=4$ 이다.
ㄱ의 주기성에 의하여 $f(5)=f(1)$ 이고, $f(2)=f(0)$ 이므로
$f(1)-f(0)=4$
따라서 옳은 것이다.

ㄷ. $f(0)=a$ 라 하면 ㄴ에 의해 $f(1)=a+4$ 이다.
$\int_{0}^{1} f(f(x)) f'(x) dx=6$ 에서 $f(x)=t$ 로 치환하면 $\frac{dt}{dx}=f'(x)$ 이므로
$\int_{f(0)}^{f(1)} f(t) dt = \int_{a}^{a+4} f(t) dt = 6$
$f(x)$ 의 주기가 $2$ 이므로 $\int_{a}^{a+4} f(t) dt = 2 \int_{a}^{a+2} f(t) dt = 2 \int_{0}^{2} f(t) dt = 6$
즉, $\int_{0}^{2} f(t) dt = 3$ 이다.
또한 $f(1+x)=f(1-x)$ 이므로 $x=1$ 에 대하여 대칭이다.
$\int_{0}^{1} f(x) dx = \int_{1}^{2} f(x) dx = \frac{1}{2} \int_{0}^{2} f(x) dx = \frac{3}{2}$
$\int_{0}^{10} f(x) dx = 5 \int_{0}^{2} f(x) dx = 5 \times 3 = 15$ 이므로
$\int_{1}^{10} f(x) dx = \int_{0}^{10} f(x) dx - \int_{0}^{1} f(x) dx = 15 - \frac{3}{2} = \frac{27}{2}$
따라서 옳은 것이다.

따라서 옳은 것은 ㄱ, ㄴ, ㄷ이다.
