## 문제
함수 $f(x) = x + 1$ 에 대하여 
$$\int_{-1}^{1} \{ f(x) \}^{2} dx = k \left( \int_{-1}^{1} f(x) dx \right)^{2}$$
일 때, 상수 $k$ 의 값은?

## 보기
① $\frac{1}{6}$   ② $\frac{1}{3}$   ③ $\frac{1}{2}$   ④ $\frac{2}{3}$   ⑤ $\frac{5}{6}$

## 풀이
먼저 $\int_{-1}^{1} \{ f(x) \}^{2} dx$ 의 값을 구하면 다음과 같다.
$$\int_{-1}^{1} \{ f(x) \}^{2} dx = \int_{-1}^{1} (x + 1)^{2} dx$$ 
$$= \int_{-1}^{1} (x^{2} + 2x + 1) dx$$ 
$$= 2 \int_{0}^{1} (x^{2} + 1) dx$$ 
$$= 2 \left[ \frac{1}{3} x^{3} + x \right]_{0}^{1}$$ 
$$= 2 \left( \frac{1}{3} + 1 \right) = \frac{8}{3}$$ 

다음으로 $\int_{-1}^{1} f(x) dx$ 의 값을 구하면 다음과 같다.
$$\int_{-1}^{1} f(x) dx = \int_{-1}^{1} (x + 1) dx$$ 
$$= 2 \int_{0}^{1} 1 dx$$ 
$$= 2 [x]_{0}^{1} = 2$$ 

주어진 식 $\int_{-1}^{1} \{ f(x) \}^{2} dx = k \left( \int_{-1}^{1} f(x) dx \right)^{2}$ 에 대입하면
$$\frac{8}{3} = k \times 2^{2}$$ 
$$\frac{8}{3} = 4k$$ 
$$\therefore k = \frac{2}{3}$$
