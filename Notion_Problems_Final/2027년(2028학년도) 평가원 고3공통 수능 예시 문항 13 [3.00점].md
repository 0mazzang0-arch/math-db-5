## 문제
확률변수 $X$가 정규분포를 따르고 다음 조건을 만족시킬 때, $\sigma (X)$의 값은? (단, $Z$는 표준정규분포를 따르는 확률변수이다.)

(가) $P (X \ge 1) + P (X \ge 7) = 1$
(나) $P ( | X - 5 | \le 3 ) + P ( X \le 0 ) = P ( Z \le 1 )$

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
조건 (가)에서 
$P (X \ge 1) + P (X \ge 7) = 1$
$P ( X \ge 1 ) + 1 - P ( X \le 7 ) = 1$
$P ( X \ge 1 ) = P ( X \le 7 )$
$\therefore E (X) = m = \frac{1 + 7}{2} = 4$

확률변수 $X$의 표준편차를 $\sigma$라 하면 정규분포 $N ( 4, \sigma^{2} )$을 따르고, $Z = \frac{X - 4}{\sigma}$라 하면 확률변수 $Z$는 표준정규분포 $N ( 0, 1 )$을 따른다.

조건 (나)에서 
$P ( | X - 5 | \le 3 ) + P ( X \le 0 ) = P ( Z \le 1 )$
$P ( 2 \le X \le 8 ) + P ( X \le 0 ) = P ( Z \le 1 )$
$P ( \frac{2 - 4}{\sigma} \le Z \le \frac{8 - 4}{\sigma} ) + P ( Z \le \frac{0 - 4}{\sigma} ) = P ( Z \le 1 )$
$P ( - \frac{2}{\sigma} \le Z \le \frac{4}{\sigma} ) + P ( Z \le - \frac{4}{\sigma} ) = P ( Z \le 1 )$

정규분포의 대칭성에 의해 $P ( - \frac{2}{\sigma} \le Z \le \frac{4}{\sigma} ) = P ( - \frac{4}{\sigma} \le Z \le \frac{2}{\sigma} )$ 이므로
$P ( - \frac{4}{\sigma} \le Z \le \frac{2}{\sigma} ) + P ( Z \le - \frac{4}{\sigma} ) = P ( Z \le 1 )$
$P ( Z \le \frac{2}{\sigma} ) = P ( Z \le 1 )$
$\frac{2}{\sigma} = 1$
$\therefore \sigma = 2$

정답: ②
