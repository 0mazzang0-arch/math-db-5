## 문제
다음은 $0 < \theta < \frac{\pi}{2}$ 일 때, $\sin \theta \cos \theta$, $\sin \theta + \cos \theta$, $1$ 을 세 변의 길이로 하는 삼각형이 존재함을 증명한 것이다.

$0 < \theta < \frac{\pi}{2}$ 이므로
$0 < \sin \theta < 1$, $0 < \cos \theta < 1$
$\therefore 0 < \sin \theta \cos \theta < 1 \cdots \cdots$ ㉠
또, $\sin \theta$ (가) $\sin^{2} \theta$, $\cos \theta$ (가) $\cos^{2} \theta$ 이므로
$\sin \theta + \cos \theta$ (가) $\sin^{2} \theta + \cos^{2} \theta = 1 \cdots \cdots$ ㉡
㉠, ㉡에서 세 수 $\sin \theta \cos \theta$, $\sin \theta + \cos \theta$, $1$ 중 (나) 가(이) 가장 크다.
이때, (다) $+ 1 - ($ (나) $)$
$= (\sin \theta - 1)(\cos \theta - 1) > 0$
$\therefore$ (다) $+ 1 >$ (나)
따라서 주어진 수를 세 변의 길이로 하는 삼각형이 존재한다.

위의 증명 과정에서 (가), (나), (다)에 알맞은 것을 순서대로 나열하면?

## 보기
① (가) $: >, (나) : \sin \theta + \cos \theta, (다) : \sin \theta + \cos \theta$
② (가) $: <, (나) : \sin \theta \cos \theta, (다) : \sin \theta + \cos \theta$
③ (가) $: >, (나) : \sin \theta + \cos \theta, (다) : \sin \theta \cos \theta$
④ (가) $: <, (나) : 1, (다) : \sin \theta \cos \theta$
⑤ (가) $: =, (나) : \sin \theta \cos \theta, (다) : 1$

## 풀이
[정답] ③

(가) $0 < \sin \theta < 1$, $0 < \cos \theta < 1$ 이므로
$\sin \theta > \sin^{2} \theta$, $\cos \theta > \cos^{2} \theta$
$\therefore$ (가) 에는 $>$

(나) (가)로부터 $\sin \theta + \cos \theta$ 가 $1$보다 크고, ㉠에서 $\sin \theta \cos \theta$ 는 $1$보다 작으므로 $\sin \theta + \cos \theta$ 가 가장 크다.
$\therefore$ (나) 에는 $\sin \theta + \cos \theta$

(다) 짧은 두 변의 길이의 합은 가장 긴 변의 길이보다 큼을 보이면 된다.
(나)에서 가장 긴 변이 $\sin \theta + \cos \theta$ 임을 알았으므로, 나머지 두 변 $\sin \theta \cos \theta$ 와 $1$의 합이 이보다 큰지 확인한다.
$\sin \theta \cos \theta + 1 - (\sin \theta + \cos \theta) = (\sin \theta - 1)(\cos \theta - 1)$
이때 $\sin \theta < 1$, $\cos \theta < 1$ 이므로 $(\sin \theta - 1) < 0$, $(\cos \theta - 1) < 0$ 이다.
따라서 $(\sin \theta - 1)(\cos \theta - 1) > 0$ 이 성립한다.
$\therefore$ (다) 에는 $\sin \theta \cos \theta$
