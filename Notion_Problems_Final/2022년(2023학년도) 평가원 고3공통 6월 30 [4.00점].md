## 문제
주머니에 $1$부터 $12$까지의 자연수가 각각 하나씩 적혀 있는 $12$개의 공이 들어 있다. 이 주머니에서 임의로 $3$개의 공을 동시에 꺼내어 공에 적혀 있는 수를 작은 수부터 크기 순서대로 $a$, $b$, $c$라 하자. $b-a \geq 5$일 때, $c-a \geq 10$일 확률은 $\frac{q}{p}$이다. $p+q$의 값을 구하시오. (단, $p$와 $q$는 서로소인 자연수이다.)

## 풀이
$b-a \geq 5$인 사건을 $E$, $c-a \geq 10$인 사건을 $F$라 하면 구하는 확률은 $P(F | E)=\frac{P(E \cap F)}{P(E)}$이다.
모든 순서쌍 $(a, b, c)$의 개수는 ${}_{12}C_{3}=220$이다.
이때 $b-a \geq 5$를 만족시키는 순서쌍 $(a, b)$는
$$ \begin{array}{l} (1, 6), (1, 7), (1, 8), \dots, (1, 11) \\ (2, 7), (2, 8), \dots, (2, 11) \\ \vdots \\ (6, 11) \end{array} $$
각 $a$의 값에 따른 $c$의 개수는 다음과 같다.
$$ \begin{array}{l} a=1 \text{ 일 때 } c \text{ 의 개수는 } 6+5+4+3+2+1=21 \\ a=2 \text{ 일 때 } c \text{ 의 개수는 } 5+4+3+2+1=15 \\ a=3 \text{ 일 때 } c \text{ 의 개수는 } 4+3+2+1=10 \\ a=4 \text{ 일 때 } c \text{ 의 개수는 } 3+2+1=6 \\ a=5 \text{ 일 때 } c \text{ 의 개수는 } 2+1=3 \\ a=6 \text{ 일 때 } c \text{ 의 개수는 } 1 \end{array} $$
이므로 $b-a \geq 5$를 만족시키는 모든 순서쌍 $(a, b, c)$의 개수는
$21+15+10+6+3+1=56$
즉, $P(E)=\frac{56}{220}=\frac{14}{55}$이다.
한편, $b-a \geq 5$이고 $c-a \geq 10$인 경우는
$$ \begin{array}{l} a=1, c=11 \text{ 일 때 } b=6, 7, 8, 9, 10 \\ a=1, c=12 \text{ 일 때 } b=6, 7, 8, 9, 10, 11 \\ a=2, c=12 \text{ 일 때 } b=7, 8, 9, 10, 11 \end{array} $$
이므로 $b-a \geq 5$이고 $c-a \geq 10$인 모든 순서쌍 $(a, b, c)$의 개수는
$5+6+5=16$
즉, $P(E \cap F)=\frac{16}{220}=\frac{4}{55}$이다.
따라서 $P(F | E)=\frac{P(E \cap F)}{P(E)} = \frac{\frac{4}{55}}{\frac{14}{55}} = \frac{2}{7}$이다.
$p=7, q=2$이므로 $p+q=7+2=9$이다.
**정답: $9$**
