## 문제
$\log 2$ 가 무리수임을 이용하여 등식 $a \log 20 + \frac{b}{\log_{8} 100} + 3 = 0$ 을 만족시키는 유리수 $a$, $b$ 의 값을 정할 때, $a^{2} + b^{2}$ 의 값을 구하시오.

## 보기

## 풀이
$\log 20 = 1 + \log 2$
$\frac{1}{\log_{8} 100} = \frac{\log 8}{\log 100} = \frac{3}{2} \frac{\log 2}{\log 10} = \frac{3}{2} \log 2$ 이므로
$a ( 1 + \log 2 ) + \frac{3b}{2} \log 2 + 3 = 0$
$( a + 3 ) + \left( a + \frac{3b}{2} \right) \log 2 = 0$
이때 $a + 3$, $a + \frac{3b}{2}$ 는 유리수이고 $\log 2$ 가 무리수이므로
$a + 3 = 0$, $a + \frac{3b}{2} = 0$
$\therefore a = -3$, $b = 2$
$\therefore a^{2} + b^{2} = 13$

[정답] $13$
