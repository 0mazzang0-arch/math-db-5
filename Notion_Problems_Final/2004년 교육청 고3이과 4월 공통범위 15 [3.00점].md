## 문제
부등식 $x^{4} - x^{3} < x^{2} + 2x$를 만족시키는 모든 실수 $x$에 대하여 이차함수 $f(x) = -x^{2} + x + k$가 항상 양의 값을 갖도록 하는 실수 $k$의 값의 범위는?

## 보기
① $k > 2$   ② $k \geq 2$   ③ $0 < k < 2$   ④ $0 \leq k < 2$   ⑤ $0 < k \leq 2$

## 풀이
부등식을 정리하면
$x^{4} - x^{3} - x^{2} - 2x < 0$
$x ( x - 2 ) ( x^{2} + x + 1 ) < 0$
이때 $x^{2} + x + 1 > 0$이므로
$0 < x < 2$
$0 < x < 2$인 모든 실수 $x$가 $f(x) = -x^{2} + x + k > 0$을 만족시키려면
$f(0) \geq 0$, $f(2) \geq 0$이어야 한다.
$f(0) = k \geq 0$
$f(2) = -2^{2} + 2 + k = -4 + 2 + k = k - 2 \geq 0$
그러므로 $k \geq 2$이다.

정답: ②
