## 문제
$0 \le x \le \pi$ 일 때, 방정식 $(\sin x + \cos x)^2 = \sqrt{3} \sin x + 1$ 의 모든 실근의 합은?

## 보기
① $\frac{7}{6}\pi$   ② $\frac{4}{3}\pi$   ③ $\frac{3}{2}\pi$   ④ $\frac{5}{3}\pi$   ⑤ $\frac{11}{6}\pi$

## 풀이
$(\sin x + \cos x)^2 = \sqrt{3} \sin x + 1$ 에서
$1 + 2 \sin x \cos x = \sqrt{3} \sin x + 1$
$\sin x (2 \cos x - \sqrt{3}) = 0$
$0 \le x \le \pi$ 이므로
$\sin x = 0$ 일 때, $x = 0$ 또는 $x = \pi$
$\cos x = \frac{\sqrt{3}}{2}$ 일 때, $x = \frac{\pi}{6}$
따라서 모든 실근의 합은 $0 + \pi + \frac{\pi}{6} = \frac{7}{6}\pi$

정답: ①
