## 문제
두 수열 $\{ a_{n} \}$, $\{ b_{n} \}$은 $a_{1} = a_{2} = 1$, $b_{1} = k$이고, 모든 자연수 $n$에 대하여
$$ a_{n+2} = (a_{n+1})^{2} - (a_{n})^{2} , b_{n+1} = a_{n} - b_{n} + n $$
을 만족시킨다. $b_{20} = 14$일 때, $k$의 값은?

## 보기
① $-3$   ② $-1$   ③ $1$   ④ $3$   ⑤ $5$

## 풀이
$a_{1} = a_{2} = 1$이고 $a_{n+2} = (a_{n+1})^{2} - (a_{n})^{2}$이므로
$a_{3} = (a_{2})^{2} - (a_{1})^{2} = 1^{2} - 1^{2} = 0$
$a_{4} = (a_{3})^{2} - (a_{2})^{2} = 0^{2} - 1^{2} = -1$
$a_{5} = (a_{4})^{2} - (a_{3})^{2} = (-1)^{2} - 0^{2} = 1$
$a_{6} = (a_{5})^{2} - (a_{4})^{2} = 1^{2} - (-1)^{2} = 0$
$a_{7} = (a_{6})^{2} - (a_{5})^{2} = 0^{2} - 1^{2} = -1$
$\vdots$
따라서 수열 $\{ a_{n} \}$은 $n \ge 2$일 때, $1, 0, -1$이 차례대로 반복된다.

$b_{n+1} = a_{n} - b_{n} + n$에서 $b_{n} + b_{n+1} = a_{n} + n$이므로
$$ \begin{array}{l} b_{1} + b_{2} = a_{1} + 1 \\ -(b_{2} + b_{3}) = -(a_{2} + 2) \\ b_{3} + b_{4} = a_{3} + 3 \\ -(b_{4} + b_{5}) = -(a_{4} + 4) \\ \vdots \\ -(b_{18} + b_{19}) = -(a_{18} + 18) \\ b_{19} + b_{20} = a_{19} + 19 \end{array} $$
위의 식들을 변끼리 더하면
$$ \begin{array}{l} b_{1} + b_{20} = (a_{1} - a_{2} + a_{3} - a_{4} + \dots - a_{18} + a_{19}) + (1 - 2 + 3 - 4 + \dots - 18 + 19) \end{array} $$
이때 $a_{1} = 1$이고, $n \ge 2$에 대하여 $(a_{n} - a_{n+1} + a_{n+2})$의 꼴을 살펴보면
$(a_{2} - a_{3} + a_{4}) = 1 - 0 + (-1) = 0$
$(a_{5} - a_{6} + a_{7}) = 1 - 0 + (-1) = 0$
이와 같이 $3$개씩 묶었을 때 합이 $0$이 되므로
$a_{1} - (a_{2} - a_{3} + a_{4}) + (a_{5} - a_{6} + a_{7}) - \dots + (a_{17} - a_{18} + a_{19}) = 1 - 0 + 0 - 0 + 0 - 0 + 0 = 1$
또한, $1 - 2 + 3 - 4 + \dots - 18 + 19 = (1 - 2) + (3 - 4) + \dots + (17 - 18) + 19 = (-1) \times 9 + 19 = 10$
따라서 $b_{1} + b_{20} = 1 + 10$이다.
$b_{1} = k$, $b_{20} = 14$이므로
$k + 14 = 11$
$\therefore k = -3$

정답: ①
