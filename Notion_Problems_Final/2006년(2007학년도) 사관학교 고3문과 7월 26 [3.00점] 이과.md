## 문제
수열 $\{ a_{n} \}$ 의 일반항이
$a_{n} = n - 4 \left[ \frac{n}{4} \right]$
일 때, $\sum_{n=1}^{25} a_{n}$ 의 값을 구하시오.
(단, $[x]$ 는 $x$ 보다 크지 않은 최대의 정수이다.)

## 보기


## 풀이
(ⅰ) $1 \leq n < 4$ 일 때 
$\left[ \frac{n}{4} \right] = 0$ 이므로 
$\sum_{n=1}^{3} n = 6$ 

(ⅱ) $4 \leq n < 8$ 일 때 
$\left[ \frac{n}{4} \right] = 1$ 이므로 
$\sum_{n=4}^{7}(n - 4) = \sum_{n=1}^{4}(n - 1) = 6$ 

(ⅲ) $8 \leq n < 12$ 일 때 
$\left[ \frac{n}{4} \right] = 2$ 이므로 
$\sum_{n=8}^{11}(n - 8) = \sum_{n=1}^{4}(n - 1) = 6$ 

이런 식으로 계속하게 되면 
$12 \leq n < 16$ 일 때, $16 \leq n < 20$ 일 때, $20 \leq n < 24$ 일 때도 모두 $6$ 이고, 
$24 \leq n \leq 25$ 일 때 
$0 + 1 = 1$ 이므로 

$\sum_{n=1}^{25} a_{n} = 6 \times 6 + 1 = 37$

**정답: $37$**
