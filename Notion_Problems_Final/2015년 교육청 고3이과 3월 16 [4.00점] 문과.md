## 문제
그림과 같이 두 곡선 $y = a^{x}$, $y = b^{x}$ ($1 < a < b$)가 직선 $y = t$ ($t > 1$)과 만나는 점의 $x$ 좌표를 각각 $f(t)$, $g(t)$라 할 때, $2f(a) = 3g(a)$가 성립한다. $f(c) = g(27)$을 만족시키는 실수 $c$의 값은?

> [!example] 📸 (그림/그래프)

## 보기
① $6$   ② $9$   ③ $12$   ④ $15$   ⑤ $18$

## 풀이
$a^{f(t)} = t$ 이므로
$f(t) = \log_{a} t$

$b^{g(t)} = t$ 이므로
$g(t) = \log_{b} t$

$2f(a) = 3g(a)$ 이므로
$2 \log_{a} a = 3 \log_{b} a$ 에서
$2 = 3 \log_{b} a$
$\log_{b} a = \frac{2}{3}$
즉, $\log_{a} b = \frac{3}{2}$

$f(c) = g(27) = \log_{b} 27$
$= \frac{\log_{a} 27}{\log_{a} b}$
$= \frac{\log_{a} 27}{\frac{3}{2}}$
$= \frac{2}{3} \log_{a} 27$
$= \log_{a} 27^{\frac{2}{3}}$
$= \log_{a} 9$

따라서 $c = 9$ 이다.
