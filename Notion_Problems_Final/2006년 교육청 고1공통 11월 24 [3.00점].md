## 문제
상수 $a$, $b$에 대하여 다항식 $ax^{3} + bx^{2} + 1$이 $x^{2} - x - 1$로 나누어떨어질 때, $a^{2} + b^{2}$의 값을 구하시오.

## 보기

## 풀이
다항식 $ax^{3} + bx^{2} + 1$이 $x^{2} - x - 1$로 나누어떨어지므로 
$ax^{3} + bx^{2} + 1 = (x^{2} - x - 1)(ax - 1)$ 
$\Rightarrow ax^{3} + bx^{2} + 1 = ax^{3} - (a + 1)x^{2} - (a - 1)x + 1$ 

위 식은 항등식이므로 
$b = -a - 1$, $-a + 1 = 0$ 

$\therefore a = 1$, $b = -2$ 

따라서, $a^{2} + b^{2} = 1^{2} + (-2)^{2} = 5$
