## 문제
$\log_{10} x = [ \log_{10} x ]$를 만족하는 $0 < x < 1$인 모든 $x$ 값들의 합을 $S$라 할 때, $99S$의 값을 구하시오. (단, $[ x ]$는 $x$보다 크지 않은 최대의 정수)

## 보기
(해당 문제에는 객관식 보기가 없습니다.)

## 풀이
$\log_{10} x = [ \log_{10} x ]$이므로 $\log_{10} x$의 가수는 $0$이다. 
따라서 $\log_{10} x = n$ ($n$은 정수)이다.
이때 $0 < x < 1$이므로 $\log_{10} x < 0$이다.
따라서 $n$은 음의 정수이다.
그러므로 $x$는 다음과 같다.
$x = \frac{1}{10}, \frac{1}{10^{2}}, \frac{1}{10^{3}}, \dots, \frac{1}{10^{n}}, \dots$

$S$는 이 모든 $x$의 합이므로 무한등비급수의 합 공식에 의해
$$ S = \frac{1}{10} + \frac{1}{10^{2}} + \frac{1}{10^{3}} + \dots + \frac{1}{10^{n}} + \dots = \frac{\frac{1}{10}}{1 - \frac{1}{10}} = \frac{1}{9} $$
$\therefore 99S = 99 \times \frac{1}{9} = 11$

**정답: $11$**
