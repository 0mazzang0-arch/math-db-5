## 문제
정수 $n$에 대하여 두 집합 $A(n)$, $B(n)$이
$$A(n) = \{ x | \log_{2} x \le n \}$$
$$B(n) = \{ x | \log_{4} x \le n \}$$
일 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?

## 보기
ㄱ. $A(1) = \{ x | 0 < x \le 1 \}$
ㄴ. $A(4) = B(2)$
ㄷ. $A(n) \subset B(n)$일 때, $B(-n) \subset A(-n)$이다.

① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄱ, ㄷ$   ⑤ $ㄴ, ㄷ$

## 풀이
집합의 조건 제시법을 풀면 $A(n) = \{ x | 0 < x \le 2^n \}$, $B(n) = \{ x | 0 < x \le 4^n \}$이다.

ㄱ. $2^1 = 2$이므로 $A(1) = \{ x | 0 < x \le 2 \}$이다. 따라서 주어진 문장은 거짓이다.

ㄴ. $2^4 = 16$이고 $4^2 = 16$이므로 $2^4 = 4^2$이다.
따라서 $A(4) = \{ x | 0 < x \le 16 \}$, $B(2) = \{ x | 0 < x \le 16 \}$이 되어 $A(4) = B(2)$이다. 따라서 참이다.

ㄷ. $A(n) \subset B(n)$이면 $0 < 2^n \le 4^n$이다.
이 식의 양변에 지수 $-1$을 취하면 부등호의 방향이 바뀌므로 $(2^n)^{-1} \ge (4^n)^{-1}$이 되어 $2^{-n} \ge 4^{-n} > 0$이다.
이때 $A(-n) = \{ x | 0 < x \le 2^{-n} \}$, $B(-n) = \{ x | 0 < x \le 4^{-n} \}$이므로 $B(-n) \subset A(-n)$이 성립한다. 따라서 참이다.

이상에서 옳은 것은 ㄴ, ㄷ이다.
