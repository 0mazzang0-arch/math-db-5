## 문제
연속확률변수 $X$가 정규분포 $N(n, \frac{n^{2}}{4})$를 따를 때, $P(n \leq X \leq 120) = P(0 \leq Z \leq 1)$을 만족시키는 자연수 $n$의 값은? (단, $Z$는 표준정규분포를 따른다.)

## 보기
① $50$   ② $60$   ③ $70$   ④ $80$   ⑤ $90$

## 풀이
연속확률변수 $X$가 정규분포 $N(n, (\frac{n}{2})^{2})$을 따르므로 표준편차 $\sigma = \frac{n}{2}$이다.
확률변수 $X$를 표준화하면 $Z = \frac{X-n}{\frac{n}{2}}$이므로,
$P(n \leq X \leq 120) = P(\frac{n-n}{\frac{n}{2}} \leq Z \leq \frac{120-n}{\frac{n}{2}}) = P(0 \leq Z \leq \frac{2(120-n)}{n})$이다.
주어진 조건에서 $P(n \leq X \leq 120) = P(0 \leq Z \leq 1)$이므로
$\frac{2(120-n)}{n} = 1$
$240 - 2n = n$
$3n = 240$
$n = 80$
따라서 정답은 ④이다.
