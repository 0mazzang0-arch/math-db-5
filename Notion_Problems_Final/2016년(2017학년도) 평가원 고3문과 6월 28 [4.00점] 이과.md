## 문제
양수 $a$ 에 대하여 함수 $f(x) = x^3 + ax^2 - a^2 x + 2$ 가 닫힌 구간 $[-a, a]$ 에서 최댓값 $M$, 최솟값 $\frac{14}{27}$ 를 갖는다. $a + M$ 의 값을 구하시오.

## 풀이
$f'(x) = 3x^2 + 2ax - a^2 = (x + a)(3x - a)$
$f'(x) = 0$ 에서 $x = -a$ 또는 $x = \frac{a}{3}$
즉, 함수 $f(x)$ 는 $x = -a$ 에서 극대이고 $x = \frac{a}{3}$ 에서 극소이므로 함수 $f(x)$ 는 $x = \frac{a}{3}$ 에서 극솟값 $f\left(\frac{a}{3}\right)$ 을 갖는다.

$f\left(\frac{a}{3}\right) = \left(\frac{a}{3}\right)^3 + a\left(\frac{a}{3}\right)^2 - a^2\left(\frac{a}{3}\right) + 2$
$= -\frac{5}{27} a^3 + 2$

$-\frac{5}{27} a^3 + 2 = \frac{14}{27}$ 에서
$a^3 = 8$, $a = 2$

$f(x) = x^3 + 2x^2 - 4x + 2$ 이므로 닫힌 구간 $[-2, 2]$ 에서 최댓값 $M$ 은
$f(2) = 2^3 + 2 \times 2^2 - 4 \times 2 + 2 = 10$
$f(-2) = (-2)^3 + 2 \times (-2)^2 - 4 \times (-2) + 2 = 10$
이므로 $M = 10$

따라서 $a + M = 2 + 10 = 12$

**정답: $12$**
