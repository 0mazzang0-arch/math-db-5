## 문제
함수 $y = \frac{16}{x}$ 의 그래프와 함수 $y = -x^{2} + a$ 의 그래프가 서로 다른 두 점에서 만날 때, 상수 $a$ 의 값은?

## 보기
① $9$   ② $10$   ③ $11$   ④ $12$   ⑤ $13$

## 풀이
두 그래프가 서로 다른 두 점에서 만나는 것은 오른쪽 그림과 같을 때이다.

> [!example] 📸 (그림/그래프)

$x = t$ 에서 접한다면
$\frac{16}{t} = -t^{2} + a$ $\cdots$ ㉠

$y = \frac{16}{x}$ 을 미분하면
$y' = -\frac{16}{x^{2}}$ 이고,

$y = -x^{2} + a$ 을 미분하면
$y' = -2x$

이므로
$-\frac{16}{t^{2}} = -2t$ 에서
$t^{3} = 8$
$t = 2$

㉠에 대입하면
$\frac{16}{2} = -2^{2} + a$
$8 = -4 + a$ 에서
$a = 12$

정답: ④
