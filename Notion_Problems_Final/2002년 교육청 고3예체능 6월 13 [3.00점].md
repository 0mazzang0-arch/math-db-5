## 문제
실수 전체의 집합을 정의역으로 하는 함수 $f(x) = [2x] - 2[x]$ 의 치역은? (단, $[x]$ 는 $x$ 보다 크지 않은 최대의 정수이다.)

## 보기
① $\{0\}$   ② $\{-1, 0\}$   ③ $\{-1, 1\}$   ④ $\{0, 1\}$   ⑤ $\{-1, 0, 1\}$

## 풀이
$[x] = n$ 이면 $n \le x < n + 1$ 이고 $2n \le 2x < 2n + 2$ 이므로 $[2x] = 2n$ 또는 $[2x] = 2n + 1$ 이다.

i) $[2x] = 2n$ 일 때
$[2x] - 2[x] = 2n - 2n = 0$

ii) $[2x] = 2n + 1$ 일 때
$[2x] - 2[x] = 2n + 1 - 2n = 1$

따라서 함수 $f(x)$ 의 치역은 $\{0, 1\}$ 이다.
