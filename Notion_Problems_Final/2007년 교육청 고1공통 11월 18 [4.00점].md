## 문제
원 $x^{2} + y^{2} - 4x - 2y = a - 3$이 $x$축과 만나고, $y$축과 만나지 않도록 하는 실수 $a$의 값의 범위는?

## 보기
① $a > -2$   ② $a \geq -1$   ③ $-1 \leq a < 2$   ④ $-2 < a \leq 2$   ⑤ $-2 \leq a < 3$

## 풀이
$x^{2} + y^{2} - 4x - 2y = a - 3$에서
$(x - 2)^{2} + (y - 1)^{2} = a + 2$
즉, 중심이 $(2, 1)$이고, 반지름의 길이가 $\sqrt{a + 2}$인 원이므로

$x$축과 만나려면
$\sqrt{a + 2} \geq 1$, $a \geq -1$ $\cdots$ ㉠

$y$축과 만나지 않으려면
$0 < \sqrt{a + 2} < 2$, $-2 < a < 2$ $\cdots$ ㉡

㉠, ㉡를 동시에 만족시키는 $a$의 값의 범위는
$-1 \leq a < 2$

정답: ③
