## 문제
함수 $f(x) = a x^{2} - 8 x + c$ 에 대하여 $f(f(1)) = f(f(2)) = f(f(3))$ 일 때, $a + 2 c$ 의 값을 구하면? (단, $a$, $c$ 는 정수)

## 보기
① $16$   ② $18$   ③ $20$   ④ $22$   ⑤ $24$

## 풀이
[정답] $20$

$f(f(1)) = f(f(2)) = f(f(3)) = k$ 라 하면 함수 $f(x)$ 는 이차함수이므로 $f(x) = k$ 인 $x$ 값은 최대 $2$ 개다. 따라서 $f(1)$, $f(2)$, $f(3)$ 중 $2$ 개는 같은 값이어야 한다.

$$ \begin{array}{l} \text{① } f(1) = f(3) \text{ 인 경우: } a + c - 8 = 9 a + c - 24 \implies a = 2 \\ \text{② } f(1) = f(2) \text{ 인 경우: } a + c - 8 = 4 a + c - 16 \implies a = \frac{8}{3} \\ \text{③ } f(2) = f(3) \text{ 인 경우: } 4 a + c - 16 = 9 a + c - 24 \implies a = \frac{8}{5} \end{array} $$

$a$ 가 정수이므로 $f(1) = f(3)$ 이고 대칭축은 $x = 2$ 이다. 
또한, $f(f(1)) = f(f(2))$ 이므로 $f(1)$ 과 $f(2)$ 는 $x = 2$ 에 대하여 대칭이다. 즉,

$$ \frac{f(1) + f(2)}{2} = 2 $$

$a = 2$ 를 대입하면 $f(1) = 2 - 8 + c = c - 6$, $f(2) = 8 - 16 + c = c - 8$ 이므로
$$ \frac{(c - 6) + (c - 8)}{2} = 2 $$
$$ \frac{2 c - 14}{2} = 2 \implies c - 7 = 2 \implies c = 9 $$

따라서, $a + 2 c = 2 + 2 \times 9 = 20$ 이다.
