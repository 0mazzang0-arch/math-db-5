## 문제
최고차항의 계수가 $1$이고 $f(-3)=f(0)$인 삼차함수 $f(x)$에 대하여 함수 $g(x)$를
$$ g(x) = \begin{cases} f(x) & (x < -3 \text{ 또는 } x \ge 0) \\ -f(x) & (-3 \le x < 0) \end{cases} $$
이라 하자. 함수 $g(x)g(x-3)$이 $x=k$에서 불연속인 실수 $k$의 값이 한 개일 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?

ㄱ. 함수 $g(x)g(x-3)$은 $x=0$에서 연속이다.
ㄴ. $f(-6) \times f(3) = 0$
ㄷ. 함수 $g(x)g(x-3)$이 $x=k$에서 불연속인 실수 $k$가 음수일 때 집합 $\{ x \mid f(x)=0, x \text{는 실수} \}$의 모든 원소의 합이 $-1$이면 $g(-1)=-48$이다.

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
ㄱ. 
$\lim_{x \to 0-} g(x)g(x-3) = -f(0) \times f(-3)$
$\lim_{x \to 0+} g(x)g(x-3) = f(0) \times \{ -f(-3) \}$
$g(0)g(-3) = f(0) \times \{ -f(-3) \}$
이므로 함수 $g(x)g(x-3)$은 $x=0$에서 연속이다.

ㄴ. 함수 $g(x)g(x-3)$이 $x=k$에서 불연속인 실수 $k$의 값이 한 개이므로 $k=-3$ 또는 $k=3$이다.
(ⅰ) 함수 $g(x)g(x-3)$이 $x=-3$에서 연속이고, $x=3$에서 불연속인 경우
$\lim_{x \to -3-} g(x)g(x-3) = f(-3) \times f(-6)$
$\lim_{x \to -3+} g(x)g(x-3) = -f(-3) \times f(-6)$
$g(-3)g(-6) = -f(-3) \times f(-6)$이므로
$f(-3) \times f(-6) = 0 \cdots \cdots$ ㉠
$\lim_{x \to 3-} g(x)g(x-3) = f(3) \times \{ -f(0) \}$
$\lim_{x \to 3+} g(x)g(x-3) = f(3) \times f(0)$
$g(3)g(0) = f(3) \times f(0)$이므로
$f(3) \times f(0) \neq 0 \cdots \cdots$ ㉡
$f(-3)=f(0)$이므로 ㉠, ㉡에 의해 $f(-6)=0$이다.

(ⅱ) 함수 $g(x)g(x-3)$이 $x=3$에서 연속이고, $x=-3$에서 불연속인 경우
위와 같은 방법으로 $f(3)=0$이다.
(ⅰ), (ⅱ)에 의하여 $f(-6)=0$ 또는 $f(3)=0$이므로
$f(-6) \times f(3) = 0$이다.

ㄷ. $k < 0$이므로 $k=-3$이다. 따라서 $x=3$에서 연속이어야 하므로 $f(3)=0$이다.
$f(x) = (x-3)(x^2 + ax + b)$라 하자. (단, $a, b$는 상수)
$f(-3)=f(0)$이므로
$-6(9-3a+b) = -3b$, $b=6a-18$
$f(x) = (x-3)(x^2 + ax + 6a - 18)$
(ⅰ) 방정식 $x^2+ax+6a-18=0$이 $3$이 아닌 서로 다른 두 실근을 갖는 경우
$f(x)=0$의 세 실근의 합은 $3 + (-a) = -1$에서 $a=4$이다.
이때 $x^2+4x+6=0$은 실근을 갖지 않으므로 모순이다.
(ⅱ) 방정식 $x^2+ax+6a-18=0$이 중근을 갖는 경우
$f(x)=0$의 서로 다른 두 실근의 합은 $3 + \left( -\frac{a}{2} \right) = -1$에서 $a=8$이다.
이때 $x^2+8x+30=0$은 중근을 갖지 않으므로 모순이다.
(ⅲ) 방정식 $x^2+ax+6a-18=0$이 $3$과 $-4$를 실근으로 갖는 경우
$3 + (-4) = -a$, $3 \times (-4) = 6a-18$에서 $a=1$이다.
$f(x) = (x-3)(x^2 + x - 12) = (x-3)^2(x+4)$
$g(-1) = -f(-1) = -(-4)^2 \times 3 = -48$
따라서 옳은 것은 ㄱ, ㄴ, ㄷ이다.
