## 문제
$\tan x = \sqrt{3}$ 일 때, $\sin^4 x + \cos^4 x$ 의 값은?

## 보기
① $\frac{1}{8}$   ② $\frac{1}{4}$   ③ $\frac{3}{8}$   ④ $\frac{1}{2}$   ⑤ $\frac{5}{8}$

## 풀이
주어진 식을 변형하면 
$$\sin^4 x + \cos^4 x = (\sin^2 x + \cos^2 x)^2 - 2 \sin^2 x \cos^2 x$$ 
$\tan x = \sqrt{3}$ 인 경우 
$\sin x = \pm \frac{\sqrt{3}}{2}$, $\cos x = \pm \frac{1}{2}$ (복호동순) 
이므로 
$\sin^4 x + \cos^4 x = 1 - 2 \sin^2 x \cos^2 x$ 
$= \frac{5}{8}$

정답: $\frac{5}{8}$
