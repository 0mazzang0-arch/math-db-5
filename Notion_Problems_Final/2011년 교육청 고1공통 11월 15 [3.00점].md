## 문제
다음은 다항식 $f(x)$를 $(2x-3)(x+1)$로 나눈 몫이 $Q(x)$, 나머지가 $x+7$일 때, $f(3x+1)$을 $3x+2$로 나눈 나머지를 구하는 과정이다.

$$ \begin{array}{l} \text{다항식 } f(x)\text{를 } (2x-3)(x+1)\text{로 나눈 몫이 } Q(x)\text{이고 나머지가 } x+7\text{이므로} \\ f(x) = (2x-3)(x+1)Q(x) + x+7 \\ \text{이다. 한편,} \\ f(3x+1) = (6x-1)(3x+2)Q(3x+1) + (\text{가}) \\ = (3x+2) \{ (6x-1)Q(3x+1) + 1 \} + (\text{나}) \\ \text{이므로 } f(3x+1)\text{을 } 3x+2\text{로 나눈 나머지는 } (\text{나})\text{이다.} \end{array} $$

위의 과정에서 $(\text{가})$에 알맞은 식을 $P(x)$, $(\text{나})$에 알맞은 값을 $r$라 할 때, $r \times P(2)$의 값은?

## 보기
① $66$   ② $72$   ③ $78$   ④ $84$   ⑤ $90$

## 풀이
다항식 $f(x)$를 $(2x-3)(x+1)$로 나누었을 때의 몫이 $Q(x)$이고 나머지가 $x+7$이므로
$f(x) = (2x-3)(x+1)Q(x) + x+7$
이다. 

$f(3x+1)$을 구하기 위해 $x$ 대신 $3x+1$을 대입하면
$f(3x+1) = \{2(3x+1)-3\}\{(3x+1)+1\}Q(3x+1) + (3x+1)+7$
$= (6x+2-3)(3x+2)Q(3x+1) + 3x+8$
$= (6x-1)(3x+2)Q(3x+1) + 3x+8$
따라서 $(\text{가})$에 알맞은 식 $P(x) = 3x+8$이다.

또한, $3x+8$을 $3x+2$로 나누면
$3x+8 = (3x+2) \times 1 + 6$
이므로 식을 다시 정리하면
$f(3x+1) = (3x+2) \{ (6x-1)Q(3x+1) + 1 \} + 6$
이다. 따라서 $(\text{나})$에 알맞은 값 $r = 6$이다.

$P(x) = 3x+8$, $r=6$이므로
$r \times P(2) = 6 \times (3 \times 2+8) = 6 \times 14 = 84$
정답은 ④이다.
