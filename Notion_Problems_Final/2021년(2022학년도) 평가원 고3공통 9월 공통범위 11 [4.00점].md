## 문제
다항함수 $f(x)$가 모든 실수 $x$에 대하여
$$xf(x) = 2x^3 + ax^2 + 3a + \int_{1}^{x} f(t) dt$$
를 만족시킨다. $f(1) = \int_{0}^{1} f(t) dt$일 때, $a + f(3)$의 값은? (단, $a$는 상수이다.)

## 보기
① $5$   ② $6$   ③ $7$   ④ $8$   ⑤ $9$

## 풀이
[정답] ④

$$xf(x) = 2x^3 + ax^2 + 3a + \int_{1}^{x} f(t) dt \cdots \text{㉠}$$

㉠의 양변에 $x=1$을 대입하면
$$f(1) = 2 + a + 3a + 0$$
이므로
$$f(1) = 2 + 4a \cdots \text{㉡}$$

㉠의 양변에 $x=0$을 대입하면
$$0 = 3a + \int_{1}^{0} f(t) dt$$
즉, $0 = 3a - \int_{0}^{1} f(t) dt$이므로
$$\int_{0}^{1} f(t) dt = 3a \cdots \text{㉢}$$

$f(1) = \int_{0}^{1} f(t) dt$이므로 ㉡, ㉢에서
$$2 + 4a = 3a$$
즉, $a = -2$, $f(1) = -6$

㉠의 양변을 미분하면
$$f(x) + xf'(x) = 6x^2 + 2ax + f(x)$$
이므로
$$xf'(x) = 6x^2 + 2ax$$
$$f'(x) = 6x + 2a = 6x - 4$$

따라서
$$f(x) = \int f'(x) dx = 3x^2 - 4x + C \text{ ($C$는 적분상수)}$$
$f(1) = 3 - 4 + C = -6$에서 $C = -5$

따라서 $f(3) = 27 - 12 - 5 = 10$이므로
$$a + f(3) = -2 + 10 = 8$$
