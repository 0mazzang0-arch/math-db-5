## 문제
자연수 $n$에 대하여 원점 $O$와 점 $(n, 0)$을 이은 선분을 밑변으로 하고, 높이가 $h_n$인 삼각형의 넓이를 $a_n$이라 하자. 수열 $\{ a_n \}$은 첫째항이 $\frac{1}{2}$인 등비수열일 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?

$$
\begin{array}{|l|}
\hline
\text{ㄱ. 모든 자연수 } n \text{에 대하여 } a_n = \frac{1}{2} \text{이면 } h_n = \frac{1}{n} \text{이다.} \\
\text{ㄴ. } h_2 = \frac{1}{4} \text{이면 } a_n = \left( \frac{1}{2} \right)^n \text{이다.} \\
\text{ㄷ. } h_2 < \frac{1}{2} \text{이면 } \lim_{n \to \infty} n h_n = 0 \text{이다.} \\
\hline
\end{array}
$$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄷ$   ④ $ㄱ, ㄴ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
[정답] ⑤

$\triangle OAB$의 넓이가 $a_n$이므로 
> [!example] 📸 (그림/그래프)

$a_n = \frac{1}{2} n h_n$
이 때 $a_n$은 첫째항이 $\frac{1}{2}$인 등비수열이므로 공비를 $r$라 하면 
$a_n = \frac{1}{2} r^{n-1}$이다.

ㄱ. $a_n = \frac{1}{2}$이면 $\frac{1}{2} n h_n = \frac{1}{2}$에서 $h_n = \frac{1}{n}$이다. (참)

ㄴ. $\frac{1}{2} n h_n = \frac{1}{2} r^{n-1}$에서 $n = 2$를 대입하면 
$h_2 = \frac{1}{2} r = \frac{1}{4}$이므로 $r = \frac{1}{2}$이다. 
$\therefore a_n = \frac{1}{2} \left( \frac{1}{2} \right)^{n-1} = \left( \frac{1}{2} \right)^n$ (참)

ㄷ. $h_2 = \frac{1}{2} r$이므로 $0 < h_2 < \frac{1}{2}$에서 
$0 < \frac{1}{2} r < \frac{1}{2}$
$0 < r < 1$이므로 $\lim_{n \to \infty} a_n = \lim_{n \to \infty} a_1 r^{n-1} = 0$이다.
$n h_n = 2 a_n$이므로 
$\lim_{n \to \infty} n h_n = \lim_{n \to \infty} 2 a_n = 0$ (참)

따라서 ㄱ, ㄴ, ㄷ 모두 옳다.
