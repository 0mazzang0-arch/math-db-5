## 문제
지름의 길이가 $1$ 인 원에 내접하는 사각형 $ABCD$ 가 다음 조건을 만족시킨다.

(가) 선분 $AB$ 는 원의 지름이다.  
(나) $\overline{AD} = \overline{CD} < \frac{\sqrt{2}}{2}$

사각형 $ABCD$ 의 둘레의 길이가 $\frac{19}{8}$ 일 때, 선분 $AD$ 의 길이는 $\frac{q}{p}$ 이다. $p+q$ 의 값을 구하시오. (단, $p$ 와 $q$ 는 서로소인 자연수이다.)

## 풀이
$\angle DBA = \theta$ 라 하자. 조건 (나)에서 $\overline{AD} = \overline{CD}$ 이므로 호 $AD$ 와 호 $CD$ 의 길이가 같다. 따라서 원주각의 성질에 의해 $\angle CBD = \angle ABD = \theta$ 이다. 
이때 $\angle CBA = 2\theta$ 이고, $0 < \theta < \frac{\pi}{4}$ 이다.

조건 (가)에서 $\angle ADB = \angle ACB = \frac{\pi}{2}$ 이므로
직각삼각형 $ADB$ 에서 $\overline{AD} = \sin \theta$
직각삼각형 $ACB$ 에서 $\overline{BC} = \cos 2\theta$

사각형 $ABCD$ 의 둘레의 길이가 $\frac{19}{8}$ 이므로
$$ \begin{array}{l} \overline{AD} + \overline{DC} + \overline{CB} + \overline{BA} = \frac{19}{8} \\ \sin \theta + \sin \theta + \cos 2\theta + 1 = \frac{19}{8} \\ 2\sin \theta + 1 - 2\sin^{2} \theta + 1 = \frac{19}{8} \\ -2\sin^{2} \theta + 2\sin \theta + 2 = \frac{19}{8} \\ \sin^{2} \theta - \sin \theta + \frac{3}{16} = 0 \\ 16\sin^{2} \theta - 16\sin \theta + 3 = 0 \\ (4\sin \theta - 3)(4\sin \theta - 1) = 0 \end{array} $$

$\therefore \sin \theta = \frac{1}{4}$ ($\because 0 < \theta < \frac{\pi}{4}$ 이므로 $\sin \theta < \frac{\sqrt{2}}{2}$)

따라서 $\overline{AD} = \sin \theta = \frac{1}{4}$ 이므로
$p = 4, q = 1$
$p+q = 4+1 = 5$

정답: $5$
