## 문제
함수 $y = \ln ( x - a ) + b$의 그래프는 점 $( 2 , 5 )$를 지나고, 직선 $x = 1$을 점근선으로 갖는다. $a + b$의 값은? (단, $a, b$는 상수이다.)

## 보기
① $3$   ② $4$   ③ $5$   ④ $6$   ⑤ $7$

## 풀이
함수 $y = \ln ( x - a ) + b$의 그래프가 점 $( 2 , 5 )$를 지나므로
$\ln ( 2 - a ) + b = 5$ $\cdots$ ㉠

함수 $y = \ln ( x - a ) + b$의 그래프의 점근선의 방정식이 $x = a$이므로
$a = 1$ $\cdots$ ㉡

㉠, ㉡에 의하여 $a = 1$, $b = 5$
따라서 $a + b = 1 + 5 = 6$
