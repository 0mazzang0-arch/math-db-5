## 문제
두 수열 $\{ a_{n} \}$, $\{ b_{n} \}$은 $a_{1} = 1$, $b_{1} = -1$이고, 모든 자연수 $n$에 대하여
$a_{n+1} = a_{n} + b_{n}$, $b_{n+1} = 2\cos \frac{a_{n}}{3} \pi$
를 만족시킨다. $a_{2021} - b_{2021}$의 값은?

## 보기
① $-2$   ② $0$   ③ $2$   ④ $4$   ⑤ $6$

## 풀이
$a_{1} = 1$, $b_{1} = -1$이므로
$a_{2} = 1 + ( -1 ) = 0$, $b_{2} = 2\cos \frac{\pi}{3} = 1$
$a_{3} = 0 + 1 = 1$, $b_{3} = 2\cos 0 = 2$
$a_{4} = 1 + 2 = 3$, $b_{4} = 2\cos \frac{\pi}{3} = 1$
$a_{5} = 3 + 1 = 4$, $b_{5} = 2\cos \pi = -2$
$a_{6} = 4 + ( -2 ) = 2$, $b_{6} = 2\cos \frac{4}{3} \pi = -1$
$a_{7} = 2 + ( -1 ) = 1$, $b_{7} = 2\cos \frac{2}{3} \pi = -1$
$a_{8} = 1 + ( -1 ) = 0$, $b_{8} = 2\cos \frac{\pi}{3} = 1$
$\vdots$
두 수열 $\{ a_{n} \}$, $\{ b_{n} \}$은 모든 자연수 $n$에 대하여 $a_{n+6} = a_{n}$, $b_{n+6} = b_{n}$이 성립한다.
$2021 = 6 \times 336 + 5$이므로
$a_{2021} - b_{2021} = a_{5} - b_{5} = 4 - ( -2 ) = 6$

정답: ⑤
