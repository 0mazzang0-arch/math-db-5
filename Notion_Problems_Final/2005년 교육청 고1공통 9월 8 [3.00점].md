## 문제
$0$이 아닌 서로 다른 세 실수 $a$, $b$, $c$가 
$| a ( b-c ) | = a ( c-b )$
를 만족할 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?

**<보기>**
ㄱ. $| a ( c-b ) | = a ( b-c )$
ㄴ. $a > 0$이면 $b < c$이다.
ㄷ. $ab > 0$이면 $ac > 0$이다.

## 보기
① ㄱ   ② ㄴ   ③ ㄷ   ④ ㄱ, ㄴ   ⑤ ㄴ, ㄷ

## 풀이
ㄱ. 
$| a ( c-b ) | = | a ( b-c ) | = a ( c-b )$ 
이므로 $| a ( c-b ) | = a ( b-c )$는 거짓이다.

ㄴ. 
$| a ( b-c ) | = a ( c-b ) \iff a ( b-c ) < 0$ 
따라서 $a > 0$이면 $b-c < 0$이므로 $b < c$이다. 

ㄷ. 
$a ( b-c ) < 0 \iff ab-ac < 0 \iff ab < ac$ 
따라서 $ab > 0$이면 $ac > ab > 0$이므로 $ac > 0$이다. 

이상에서 옳은 것은 ㄴ, ㄷ이다.
