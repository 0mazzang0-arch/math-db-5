## 문제
삼각방정식 $\sqrt{6} \sin x - \sqrt{2} \cos x - 2 = 0$ 의 모든 실근의 합을 $\frac{q}{p} \pi$ 라 할 때, $p + q$ 의 값을 구하시오. (단, $0 \le x \le 2 \pi$ 이고, $p$ 와 $q$ 는 서로소인 자연수이다.)

## 풀이
$\sqrt{6} \sin x - \sqrt{2} \cos x = 2 \sqrt{2} \sin \left( x - \frac{\pi}{6} \right)$ 이므로
방정식 $2 \sqrt{2} \sin \left( x - \frac{\pi}{6} \right) = 2$ 에서
$\sin \left( x - \frac{\pi}{6} \right) = \frac{\sqrt{2}}{2}$
$-\frac{\pi}{6} \le x - \frac{\pi}{6} \le \frac{11}{6} \pi$ 이므로
$x - \frac{\pi}{6} = \frac{\pi}{4}$ 또는 $x - \frac{\pi}{6} = \frac{3}{4} \pi$
즉, $x = \frac{5}{12} \pi$ 또는 $x = \frac{11}{12} \pi$
따라서, 모든 실근의 합은
$\frac{5}{12} \pi + \frac{11}{12} \pi = \frac{4}{3} \pi$
이므로 $p = 3$, $q = 4$ 이다.
따라서 $p + q = 3 + 4 = 7$ 이다.
