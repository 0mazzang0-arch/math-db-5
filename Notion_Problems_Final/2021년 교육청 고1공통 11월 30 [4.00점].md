## 문제

이차함수 $f(x) = a(x-1)^2 - 10$ ($a$는 양의 상수)와 실수 $k$에 대하여 $k-1 \le x \le k+1$에서 함수 $|f(x)|$의 최댓값을 $g(k)$라 할 때, 함수 $g(k)$가 다음 조건을 만족시킨다.

> $g(k) = 10$을 만족시키는 실수 $k$의 최댓값은 $\sqrt{10}$이다.

함수 $g(k)$가 $k=b$와 $k=c$에서 최솟값 $m$을 가질 때, $b^2 + c^2 + m^2$의 값을 구하시오. ($b, c$는 서로 다른 상수이다.)



> [!example] 📸 (그림/그래프)

## 풀이

방정식 $f(x) = 0$의 두 실근을 $\alpha, \beta$ ($\alpha < \beta$)라 하면 

$$ |f(x)| = \begin{cases} f(x) & (x < \alpha \text{ 또는 } x > \beta) \\ -f(x) & (\alpha \le x \le \beta) \end{cases} $$

이고 함수 $y = |f(x)|$의 그래프와 직선 $y = 10$이 만나는 서로 다른 세 점 중 $x$ 좌표가 $1$이 아닌 두 점의 $x$ 좌표를 각각 $p, q$ ($p < q$)라 하면 함수 $y = |f(x)|$의 그래프는 다음과 같다. 

$1$은 방정식 $f(k-1) = f(k+1)$의 한 실근이고 $q-p < 2$이면 $(k+1) - (k-1) > q-p$이므로 함수 $y = |f(x)|$의 그래프를 고려할 때:

$k < 1$이면 $g(k) = f(k-1) > 10$, 
$k \ge 1$이면 $g(k) = f(k+1) > 10$이므로 
$g(k) = 10$을 만족시키는 실수 $k$는 존재하지 않는다. 

따라서 $q-p \ge 2$이어야 한다. 
$k+1 = q$이면 $k-1 = q-2 \ge p$이고 
$g(k) = |f(k+1)| = f(k+1) = f(q) = 10$이다. 

$k+1 > q$이면 $g(k) = |f(k+1)| = f(k+1) > 10$이므로 $g(k) = 10$을 만족시키는 실수 $k$의 최댓값은 $q-1$이다. 
조건에서 $q-1 = \sqrt{10}$, $q = \sqrt{10} + 1$이므로 
$f(q) = f(\sqrt{10} + 1) = 10$이고 
$f(\sqrt{10} + 1) = a(\sqrt{10} + 1 - 1)^2 - 10 = 10a - 10 = 10$에서 $a = 2$이다. 

그러므로 $f(x) = 2(x-1)^2 - 10 = 2x^2 - 4x - 8$이다. 
방정식 $f(x) = 0$의 실근은 $\alpha = 1 - \sqrt{5}$, $\beta = 1 + \sqrt{5}$이다. 
함수 $|f(x)|$에서 두 실수 $x_1, x_2$ ($x_1 < x_2$)에 대하여 

$x_1 < x_2 < 1 - \sqrt{5}$이면 $|f(x_1)| > |f(x_2)|$ $\cdots$ ㉠ 
$1 - \sqrt{5} \le x_1 < x_2 < 1$이면 $|f(x_1)| < |f(x_2)|$ $\cdots$ ㉡ 
$1 \le x_1 < x_2 < 1 + \sqrt{5}$이면 $|f(x_1)| > |f(x_2)|$ $\cdots$ ㉢ 
$1 + \sqrt{5} \le x_1 < x_2$이면 $|f(x_1)| < |f(x_2)|$ $\cdots$ ㉣ 
이다. 

$k-1 \le x \le k+1$에서 함수 $|f(x)|$의 최댓값 $g(k)$는 다음과 같다. 

(ⅰ) $k+1 < 1 - \sqrt{5}$일 때:
$k < - \sqrt{5}$이고 ㉠에서 $g(k) = |f(k-1)|$

(ⅱ) $k-1 < 1 - \sqrt{5} \le k+1$일 때:
$-\sqrt{5} \le k < 2 - \sqrt{5}$이고 $g(k)$의 값은 $|f(k-1)|$과 $|f(k+1)|$ 중 큰 값이다. 
$|f(k-1)| = f(k-1) = 2k^2 - 8k - 2$
$|f(k+1)| = -f(k+1) = -2k^2 + 10$
$|f(k-1)| > |f(k+1)|$에서 $2k^2 - 8k - 2 > -2k^2 + 10$
$4(k-3)(k+1) > 0 \Rightarrow k < -1$ 또는 $k > 3$
$|f(k-1)| = |f(k+1)|$에서 $k = -1$ 또는 $k = 3$
$|f(k-1)| < |f(k+1)|$에서 $-1 < k < 3$
따라서 $-\sqrt{5} \le k < -1$일 때 $g(k) = |f(k-1)|$, $-1 \le k < 2 - \sqrt{5}$일 때 $g(k) = |f(k+1)|$이다. 

(ⅲ) $1 - \sqrt{5} \le k-1 < k+1 < 1$일 때:
$2 - \sqrt{5} \le k < 0$이고 ㉡에서 $g(k) = |f(k+1)|$

(ⅳ) $k-1 < 1 \le k+1$일 때:
$0 \le k < 2$이고 $g(k) = 10$

(ⅴ) $1 \le k-1 < k+1 < 1 + \sqrt{5}$일 때:
$2 \le k < \sqrt{5}$이고 ㉢에서 $g(k) = |f(k-1)|$

(ⅵ) $k-1 < 1 + \sqrt{5} \le k+1$일 때:
$\sqrt{5} \le k < 2 + \sqrt{5}$이고 $g(k)$의 값은 $|f(k-1)|$과 $|f(k+1)|$ 중 큰 값이다. 
$|f(k-1)| = -f(k-1) = -2k^2 + 8k + 2$
$|f(k+1)| = f(k+1) = 2k^2 - 10$
$|f(k-1)| > |f(k+1)|$에서 $-2k^2 + 8k + 2 > 2k^2 - 10$
$4(k-3)(k+1) < 0 \Rightarrow -1 < k < 3$
$|f(k-1)| = |f(k+1)|$에서 $k = -1$ 또는 $k = 3$
$|f(k-1)| < |f(k+1)|$에서 $k < -1$ 또는 $k > 3$
따라서 $\sqrt{5} \le k < 3$일 때 $g(k) = |f(k-1)|$, $3 \le k < 2 + \sqrt{5}$일 때 $g(k) = |f(k+1)|$이다. 

(ⅶ) $1 + \sqrt{5} \le k-1$일 때:
$2 + \sqrt{5} \le k$이고 ㉣에서 $g(k) = |f(k+1)|$

(ⅰ) ~ (ⅶ)에서 $g(k)$를 정리하면 다음과 같다. 

$$ g(k) = \begin{cases} |f(k-1)| = 2(k-2)^2 - 10 & (k < -1) \\ |f(k+1)| = -2k^2 + 10 & (-1 \le k < 0) \\ 10 & (0 \le k < 2) \\ |f(k-1)| = -2(k-2)^2 + 10 & (2 \le k < 3) \\ |f(k+1)| = 2k^2 - 10 & (k \ge 3) \end{cases} $$

함수 $g(k)$는 $k = -1$ 과 $k = 3$ 에서 최솟값 $8$ 을 가지므로 $b = -1, c = 3, m = 8$ 이다. 

$$ b^2 + c^2 + m^2 = (-1)^2 + 3^2 + 8^2 = 1 + 9 + 64 = 74 $$
