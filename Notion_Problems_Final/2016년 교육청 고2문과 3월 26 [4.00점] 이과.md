## 문제
전체집합 $U = \{ 1, 2, 3, 4, 5, 6, 7 \}$의 두 부분집합 $A = \{ 1, 2, 3 \}$, $B = \{ 2, 3, 4, 5 \}$에 대하여 집합 $P$를 $P = ( A \cup B ) \cap ( A \cap B )^{c}$이라 하자. $P \subseteq X \subseteq U$를 만족시키는 집합 $X$의 개수를 구하시오.

## 풀이
$A \cup B = \{ 1, 2, 3, 4, 5 \}$, $A \cap B = \{ 2, 3 \}$이므로 
$P = ( A \cup B ) \cap ( A \cap B )^{c}$
$= ( A \cup B ) - ( A \cap B )$
$= \{ 1, 4, 5 \}$
따라서 집합 $X$는 전체집합 $U = \{ 1, 2, 3, 4, 5, 6, 7 \}$의 부분집합 중 원소 $1$, $4$, $5$를 반드시 포함하는 집합이다.
그러므로 구하는 집합 $X$의 개수는
$2^{7-3} = 2^{4} = 16$

정답: $16$
