## 문제
$\log x$의 정수부분이 $4$이고 $\log y$의 정수부분이 $1$일 때, 
$\left( \log \frac{x}{y} \right) \left( \log \frac{y}{x} \right)$의 값 중에서 정수의 개수를 구하시오.

## 보기

## 풀이
$4 \le \log x < 5$, $1 \le \log y < 2$ 이고
$\left( \log \frac{x}{y} \right) \left( \log \frac{y}{x} \right) = -\left( \log \frac{x}{y} \right)^2 = -(\log x - \log y)^2$ 이다.
이때, $4 - 2 < \log x - \log y < 5 - 1$ 이므로
$2 < \log x - \log y < 4$
$4 < (\log x - \log y)^2 < 16$
$-16 < -(\log x - \log y)^2 < -4$
따라서 정수의 개수는 $-15$부터 $-5$까지이므로 $11$개이다.

정답: $11$
