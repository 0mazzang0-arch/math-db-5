## 문제
함수 $f(x) = \log_{a} x$, $g(x) = \log_{b} x$ 가 $0 < x < 1$ 에서 $f(x) > g(x)$ 가 성립하기 위한 조건으로 <보기>에서 옳은 것을 모두 고른 것은? 

$$ \begin{array}{|c|} \hline \text{<보 기>} \\ \text{ㄱ. } 1 < b < a \\ \text{ㄴ. } 0 < a < b < 1 \\ \text{ㄷ. } 0 < a < 1 < b \\ \hline \end{array} $$

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄷ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
$0 < x < 1$ 인 범위에서 $\log_{a} x > \log_{b} x$ 이기 위해서는 $\log_{a} x - \log_{b} x > 0$ 이어야 한다. 

따라서 성립하는 조건의 범위는 다음과 같다.
$1 < b < a$
$0 < a < 1 < b$
$0 < b < a < 1$

따라서 <보기> 중 옳은 것은 ㄱ, ㄷ이다.
