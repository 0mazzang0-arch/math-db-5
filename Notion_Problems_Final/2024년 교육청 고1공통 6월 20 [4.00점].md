## 문제
$x$에 대한 삼차방정식 $x^{3} - (a^{2} + a - 1)x^{2} - a(a - 3)x + 4a = 0$이 서로 다른 세 실근 $\alpha, \beta, \gamma$ ($\alpha < \beta < \gamma$)를 가질 때, $\alpha \gamma = -4$가 되도록 하는 모든 실수 $a$의 값의 합은?

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
$x^{3} - (a^{2} + a - 1)x^{2} - a(a - 3)x + 4a = (x + 1) \{ x^{2} - a(a + 1)x + 4a \} = 0$이므로 $x = -1$은 주어진 삼차방정식의 한 실근이다.

(ⅰ) $\alpha = -1$인 경우
$-1 \times \gamma = -4$에서 $\gamma = 4$이므로 $-1 < \beta < 4$이다.
$x^{2} - a(a + 1)x + 4a = 0$의 두 근이 $\beta, 4$이므로 근과 계수의 관계에 의해 $4 \beta = 4a$에서 $\beta = a$이다.
$\gamma = 4$를 $x^{2} - a(a + 1)x + 4a = 0$에 대입하면 $16 - 4a^{2} = 0$에서 $a = \pm 2$이다.
① $a = -2$인 경우
$\beta = -2$이므로 $-1 < \beta < 4$를 만족시키지 않는다.
② $a = 2$인 경우
$\beta = 2$이므로 $-1 < \beta < 4$를 만족시킨다.
①, ②에 의해 $a = 2$이다.

(ⅱ) $\beta = -1$인 경우
$\alpha, \gamma$는 이차방정식 $x^{2} - a(a + 1)x + 4a = 0$의 두 근이므로 근과 계수의 관계에 의해 $\alpha \gamma = 4a = -4$에서 $a = -1$이다.
$a = -1$을 $x^{2} - a(a + 1)x + 4a = 0$에 대입하면 $x^{2} = 4, x = \pm 2$이다.
$\alpha = -2, \gamma = 2$이므로 $\alpha < \beta < \gamma$를 만족시킨다.

(ⅲ) $\gamma = -1$인 경우
$\alpha \times (-1) = -4$에서 $\alpha = 4$이므로 $\alpha < \beta < \gamma$를 만족시키지 않는다.

따라서 (ⅰ), (ⅱ), (ⅲ)에 의해 $a = 2$ 또는 $a = -1$이므로 모든 $a$의 값의 합은 $1$이다.

정답: ①
