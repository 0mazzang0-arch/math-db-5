## 문제
다음은 임의의 양수 $x$ 에 대하여 $\log x$ 의 정수부분과 $\log \sqrt{10} x$ 의 정수부분의 합이 $\log x^{2}$ 의 정수부분과 같음을 보인 것이다. 

$$ \begin{array}{|l|} \hline \log x = n + \alpha \text{ ( } n \text{ 은 정수, } 0 \le \alpha < 1 \text{ ) 라 하면} \\ \log \sqrt{10} x = (가) \text{, } \log x^{2} = 2 n + 2 \alpha \text{ 이다.} \\ \text{(ⅰ) } 0 \le \alpha < \frac{1}{2} \text{ 이면} \\ \log x \text{ 의 정수부분과 } \log \sqrt{10} x \text{ 의 정수부분의 합은 } (나) \text{ 이고, 이것은 } \\ \log x^{2} \text{ 의 정수부분과 같다.} \\ \text{(ⅱ) } \frac{1}{2} \le \alpha < 1 \text{ 이면} \\ \log x \text{ 의 정수부분과 } \log \sqrt{10} x \text{ 의 정수부분의 합은 } (다) \text{ 이고, 이것은 } \\ \log x^{2} \text{ 의 정수부분과 같다.} \\ \text{따라서 (ⅰ), (ⅱ)에 의해 } \log x \text{ 의 정수부분과 } \log \sqrt{10} x \text{ 의 정수부분의 } \\ \text{합은 } \log x^{2} \text{ 의 정수부분과 같다.} \\ \hline \end{array} $$

위의 과정에서 $(가), (나), (다)$ 에 알맞은 것은? 

## 보기
$$ \begin{array}{cccc} & (가) & (나) & (다) \\ ① & n + \alpha + \frac{1}{2} & 2 n & 2 n \\ ② & n + \alpha + \frac{1}{2} & 2 n & 2 n + 1 \\ ③ & n + \alpha + \frac{1}{2} & 2 n + 1 & 2 n + 1 \\ ④ & n + \alpha + 1 & 2 n & 2 n + 2 \\ ⑤ & n + \alpha + 1 & 2 n + 1 & 2 n + 2 \end{array} $$

## 풀이
정답: ②

$\log x = n + \alpha$ ($n$ 은 정수, $0 \le \alpha < 1$) 라 하면 
$\log \sqrt{10} x = \log ( 10^{\frac{1}{2}} \cdot x ) = n + \alpha + \frac{1}{2}$, 
$\log x^{2} = 2 n + 2 \alpha$ 이다.

(ⅰ) $0 \le \alpha < \frac{1}{2}$ 이면:
$\log x$ 의 정수부분은 $n$ 이고, 
$\frac{1}{2} \le \alpha + \frac{1}{2} < 1$ 이므로 $\log \sqrt{10} x = n + ( \alpha + \frac{1}{2} )$ 의 정수부분은 $n$ 이다.
따라서 이들의 합은 $n + n = 2 n$ 이다.
이때 $0 \le 2 \alpha < 1$ 이므로 $\log x^{2} = 2 n + 2 \alpha$ 의 정수부분 $2 n$ 과 같다.

(ⅱ) $\frac{1}{2} \le \alpha < 1$ 이면:
$\log x$ 의 정수부분은 $n$ 이고,
$1 \le \alpha + \frac{1}{2} < \frac{3}{2}$ 이므로 $\log \sqrt{10} x = n + \alpha + \frac{1}{2}$ 의 정수부분은 $n + 1$ 이다.
따라서 이들의 합은 $n + ( n + 1 ) = 2 n + 1$ 이다.
이때 $1 \le 2 \alpha < 2$ 이므로 $\log x^{2} = 2 n + 2 \alpha = ( 2 n + 1 ) + ( 2 \alpha - 1 )$ 의 정수부분 $2 n + 1$ 과 같다.

따라서 (ⅰ), (ⅱ)에 의해 $(가) = n + \alpha + \frac{1}{2}$, $(나) = 2 n$, $(다) = 2 n + 1$ 이다.
