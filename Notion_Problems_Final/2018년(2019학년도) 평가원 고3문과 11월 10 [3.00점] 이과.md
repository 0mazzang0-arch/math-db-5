## 문제
연속확률변수 $X$ 가 갖는 값의 범위는 $0 \le X \le 2$ 이고, $X$ 의 확률밀도함수의 그래프가 그림과 같을 때, $P(\frac{1}{3} \le X \le a)$ 의 값은? (단, $a$ 는 상수이다.)

> [!example] 📸 (그림/그래프)

## 보기
① $\frac{11}{16}$   ② $\frac{5}{8}$   ③ $\frac{9}{16}$   ④ $\frac{1}{2}$   ⑤ $\frac{7}{16}$

## 풀이
$0 \le X \le 2$ 에서 확률밀도함수의 그래프와 $x$ 축으로 둘러싸인 부분의 넓이는 $1$ 이다.

$\frac{1}{2} \times \frac{1}{3} \times \frac{3}{4} + ( a - \frac{1}{3} ) \times \frac{3}{4} + \frac{1}{2} \times ( 2 - a ) \times \frac{3}{4} = 1$

$\frac{3}{8} a = \frac{3}{8}$ 에서 $a = 1$

따라서
$P(\frac{1}{3} \le X \le a) = P(\frac{1}{3} \le X \le 1)$
$= ( 1 - \frac{1}{3} ) \times \frac{3}{4} = \frac{1}{2}$
