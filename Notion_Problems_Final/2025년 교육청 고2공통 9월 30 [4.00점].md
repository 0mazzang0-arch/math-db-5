## 문제
최고차항의 계수가 양수인 이차함수 $f(x)$와 함수 $g(x) = \frac{1}{3}x + a$에 대하여 함수 
$$ h(x) = \begin{cases} g(x) + f(x) & (f(x) \ge g(x)) \\ g(x) - f(x) & (f(x) < g(x)) \end{cases} $$
가 다음 조건을 만족시킨다.

(가) $\lim_{x \to \alpha^-} h(x) \ne \lim_{x \to \alpha^+} h(x)$ 를 만족시키는 실수 $\alpha$의 값은 $2$ 뿐이다.
(나) 모든 실수 $k$에 대하여 $\lim_{x \to k} |h(x) - 1|$의 값이 존재한다.

$h(0) = \frac{7}{3}$일 때, $h(5)$의 값을 구하시오. (단, $a$는 상수이다.)

## 풀이
모든 실수 $x$에 대하여 $f(x) \ge g(x)$이면 $h(x) = g(x) + f(x)$이므로 조건 (가)를 만족시키지 않는다. 
그러므로 두 함수 $f(x)$, $g(x)$의 그래프가 서로 다른 두 점에서 만난다. 
방정식 $f(x) = g(x)$의 서로 다른 두 실근을 $x_1, x_2$ ($x_1 < x_2$)라 하자. 

$x \le x_1$일 때, $f(x) \ge g(x)$
$x_1 < x < x_2$일 때, $f(x) < g(x)$
$x \ge x_2$일 때, $f(x) \ge g(x)$를 만족시킨다. 

$$ h(x) = \begin{cases} g(x) + f(x) & (x \le x_1 \text{ 또는 } x \ge x_2) \\ g(x) - f(x) & (x_1 < x < x_2) \end{cases} $$
함수 $h(x)$는 $x = x_1$과 $x = x_2$를 제외한 모든 실수에서 극한값을 가지므로 조건 (가)에 의하여 $x_1 = 2$ 또는 $x_2 = 2$이다.

(ⅰ) $x_1 = 2$인 경우 
$\lim_{x \to x_2^-} h(x) = \lim_{x \to x_2^+} h(x)$이다. 
$\lim_{x \to x_2^-} h(x) = g(x_2) - f(x_2) = 0$, 
$\lim_{x \to x_2^+} h(x) = g(x_2) + f(x_2) = 2g(x_2)$이므로 
$g(x_2) = 0$이다. 
직선 $y = g(x)$의 기울기가 양수이므로 $g(x_1) = g(2) < 0$이다. 
$\lim_{x \to x_1^-} h(x) = g(x_1) + f(x_1) = 2g(x_1) = 2g(2)$, 
$\lim_{x \to x_1^+} h(x) = g(x_1) - f(x_1) = 0$이므로 
$\lim_{x \to x_1^-} |h(x) - 1| = |2g(2) - 1| > 1$, 
$\lim_{x \to x_1^+} |h(x) - 1| = |0 - 1| = 1$이다. 
$\lim_{x \to x_1} |h(x) - 1|$의 값이 존재하지 않으므로 조건 (나)를 만족시키지 않는다. 

(ⅱ) $x_2 = 2$인 경우 
$\lim_{x \to x_1^-} h(x) = \lim_{x \to x_1^+} h(x)$이다. 
$\lim_{x \to x_1^-} h(x) = g(x_1) + f(x_1) = 2g(x_1)$, 
$\lim_{x \to x_1^+} h(x) = g(x_1) - f(x_1) = 0$이므로 
$g(x_1) = 0$이다. 
직선 $y = g(x)$의 기울기가 양수이므로 $g(x_2) = g(2) > 0$이다. 
$\lim_{x \to x_2^-} h(x) = g(x_2) - f(x_2) = 0$, 
$\lim_{x \to x_2^+} h(x) = g(x_2) + f(x_2) = 2g(x_2) = 2g(2)$ 
이므로 
$\lim_{x \to x_2^-} |h(x) - 1| = |0 - 1| = 1$, 
$\lim_{x \to x_2^+} |h(x) - 1| = |2g(2) - 1|$이다. 
조건 (나)에 의하여 $|2g(2) - 1| = 1$이고 $g(2) > 0$이므로 $g(2) = 1$이다. 

(ⅰ), (ⅱ)에 의하여 $x_2 = 2$이다. 
$g(2) = \frac{2}{3} + a = 1$에서 $a = \frac{1}{3}$이므로 
$g(x) = \frac{1}{3}x + \frac{1}{3}$이다. 
$g(x_1) = \frac{1}{3}x_1 + \frac{1}{3} = 0$에서 $x_1 = -1$이다. 
$f(x_1) = f(-1) = g(-1) = 0$ 
$f(x_2) = f(2) = g(2) = 1$ 
상수 $p$에 대하여 $f(x) - g(x) = p(x + 1)(x - 2)$라 하면 
$x_1 < 0 < x_2$이므로 
$h(0) = g(0) - f(0) = -\{f(0) - g(0)\} = -p(1)(-2) = 2p = \frac{7}{3}$ 
$p = \frac{7}{6}$이다. 
$f(x) = \frac{7}{6}(x + 1)(x - 2) + g(x)$ 
$x_2 < 5$이므로 
$h(5) = g(5) + f(5) = g(5) + \left\{\frac{7}{6}(6)(3) + g(5)\right\} = 2g(5) + 21$ 
$g(5) = \frac{5}{3} + \frac{1}{3} = 2$이므로 
$h(5) = 2(2) + 21 = 25$이다.

정답: $25$
