## 문제
$A$는 세 자리의 자연수이고, $B$는 $900$보다 큰 세 자리의 자연수이다. $\log B$의 소수부분이 $\log A$의 소수부분의 $2$배일 때, 자연수 $A$의 값을 구하시오.

## 보기

## 풀이
$100 \le A < 1000$, $900 < B < 1000$이고, $\log B$의 소수부분이 $\log A$의 소수부분의 $2$배이므로 
$\log A = 2 + \alpha$, $\log B = 2 + 2\alpha$ ($0 \le \alpha < \frac{1}{2}$) 
$2\log A - \log B = \log \frac{A^{2}}{B} = 2$이므로 $\frac{A^{2}}{B} = 100$ 
$\therefore 100B = A^{2}$ 

$A$는 $10$의 배수이므로 $A = 10C$ ($C$는 자연수)라 하면 
$100B = (10C)^{2} = 100C^{2}$ 
$\therefore B = C^{2}$ 
따라서 $B$는 $900$보다 큰 세 자리의 수이면서 완전제곱수이다. 

$30^{2} = 900$, $31^{2} = 961$, $32^{2} = 1024$이므로 
$B = 31^{2}$, $A^{2} = 100B = 10^{2} \cdot 31^{2} = (310)^{2}$ 
$\therefore A = 310$

정답: $310$
