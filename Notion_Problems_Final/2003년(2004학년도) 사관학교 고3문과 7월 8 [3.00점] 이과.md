## 문제
함수 $f(x)=4^x$ 과 $f(x)$ 의 역함수 $g(x)$ 에 대하여 방정식 $f(x)= f( g( 2^x +6 ) )$ 의 실근을 $\alpha$ 라 할 때, 다음 중 옳은 것은?

## 보기
① $-2 \le \alpha < -1$   ② $-1 \le \alpha < 0$   ③ $0 \le \alpha < 1$   ④ $1 \le \alpha < 2$   ⑤ $2 \le \alpha < 3$

## 풀이
$f(x)= f( g(2^x+6 ))$ 에서 $g(2^x +6 )=a$ 라 하면 방정식 $x=a$ 의 근이 $\alpha$ 이다. $g(x)$ 가 $f(x)$ 의 역함수이므로 $f(a)=2^x +6=4^a$ 의 관계를 만족한다. 따라서, 실근 $\alpha$ 는 방정식 $2^x +6=4^x$ 의 해이다. 

$2^x =t$ ($t>0$) 라 하면 $t^2 -t-6=0$ 이므로 $(t-3)(t+2)=0$ 에서 $t=-2$ 또는 $t=3$ 이다. 

$t>0$ 이므로 $t=3=2^x$ 에서 $x=\log_{2} 3$ 이다. 
$\alpha = \log_{2} 3$ 이고 $2^1 < 3 < 2^2$ 이므로 $1 < \log_{2} 3 < 2$ 이다.
그러므로 $1 \le \alpha < 2$ 이다.

정답: ④
