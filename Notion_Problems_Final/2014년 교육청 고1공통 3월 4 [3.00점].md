## 문제
두 집합 $A = \{ x \mid x \text{ 는 } 12 \text{ 의 약수 } \}, B = \{ x \mid x \text{ 는 짝수 } \}$ 에 대하여 집합 $A \cap B$ 의 원소의 개수는?

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
$A = \{ x \mid x \text{ 는 } 12 \text{ 의 약수 } \} = \{ 1, 2, 3, 4, 6, 12 \}$
$B = \{ x \mid x \text{ 는 짝수 } \} = \{ 2, 4, 6, 8, 10, 12, 14, \dots \}$
$A \cap B = \{ 2, 4, 6, 12 \}$ 이므로 원소의 개수는 $4$ 이다.

정답: ④
