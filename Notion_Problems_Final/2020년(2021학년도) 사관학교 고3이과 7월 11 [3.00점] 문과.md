## 문제
함수 $f(x) = \frac{e^{x}}{\sin x + \cos x}$ 에 대하여 $-\frac{\pi}{4} < x < \frac{3\pi}{4}$ 에서 방정식 $f(x) - f'(x) = 0$ 의 실근은?

## 보기
① $-\frac{\pi}{6}$   ② $\frac{\pi}{6}$   ③ $\frac{\pi}{4}$   ④ $\frac{\pi}{3}$   ⑤ $\frac{\pi}{2}$

## 풀이
$f'(x) = \frac{e^{x}(\sin x + \cos x) - e^{x}(\cos x - \sin x)}{(\sin x + \cos x)^{2}}$
$= \frac{2 e^{x} \sin x}{(\sin x + \cos x)^{2}}$

$f(x) = f'(x)$ 이므로
$\frac{e^{x}}{\sin x + \cos x} = \frac{2 e^{x} \sin x}{(\sin x + \cos x)^{2}}$
$2 \sin x = \sin x + \cos x$
$\sin x = \cos x$
$\tan x = 1$
$-\frac{\pi}{4} < x < \frac{3\pi}{4}$ 이므로
$\therefore x = \frac{\pi}{4}$

정답은 ③입니다.
