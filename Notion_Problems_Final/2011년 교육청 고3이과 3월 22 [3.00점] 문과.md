## 문제
$\sqrt[n]{2} \times \sqrt[n]{8} = \sqrt[8]{2}$ 를 만족시키는 자연수 $n$ 의 값을 구하시오.

## 보기

## 풀이
$\sqrt[n]{2} \times \sqrt[n]{8} = \sqrt[8]{2}$ 에서 
$\sqrt[n]{16} = \sqrt[32]{16}$ 

$\therefore n = 32$ 

[다른 풀이] 
$\sqrt[n]{2} \times \sqrt[n]{8} = \sqrt[8]{2}$ 에서 
$2^{\frac{1}{n}} \times 2^{\frac{3}{n}} = 2^{\frac{1}{8}}$ 

$2^{\frac{4}{n}} = 2^{\frac{1}{8}}$, 
$\frac{4}{n} = \frac{1}{8}$ 

$\therefore n = 32$

정답: $32$
