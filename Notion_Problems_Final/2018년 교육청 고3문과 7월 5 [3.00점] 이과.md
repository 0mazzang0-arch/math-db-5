## 문제
실수 $x$ 에 대한 두 조건
$p : k \le x \le k+3$,
$q : (x-3)(x-10) \le 0$
에 대하여 $p$ 가 $q$ 이기 위한 충분조건이 되도록 하는 실수 $k$ 의 최댓값은?

## 보기
① $4$  ② $5$  ③ $6$  ④ $7$  ⑤ $8$

## 풀이
두 조건 $p$, $q$ 의 진리집합을 각각 $P$, $Q$ 라 하면
$P = \{ x \mid k \le x \le k+3 \}$
$Q = \{ x \mid 3 \le x \le 10 \}$
$p$ 가 $q$ 이기 위한 충분조건이므로 $P \subset Q$ 이어야 한다.
즉, $3 \le k$ 이고 $k+3 \le 10$ 이다.
$3 \le k$ 이고 $k \le 7$ 이므로
$3 \le k \le 7$
따라서 실수 $k$ 의 최댓값은 $7$ 이다.

정답: ④
