## 문제
다음 보기에서 항상 옳은 것을 모두 고른 것은?

ㄱ. $1 < a < b$ 이고 $0 < \log_{a} c < 1$ 이면 $\log_{b} c > \log_{b} a$ 이다.
ㄴ. $0 < a < 1 < b$ 이고 $0 < \log_{a} c < 1$ 이면 $\log_{b} a < \log_{b} c$ 이다.
ㄷ. $0 < a < b < 1$ 이고 $\log_{a} c < 0$ 이면 $\log_{a} b < \log_{c} b$ 이다.

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄷ$   ④ $ㄱ, ㄴ$   ⑤ $ㄴ, ㄷ$

## 풀이
[정답] ② 

ㄱ) $0 < \log_{a} c < 1$ 이고 $a > 1$ 이므로 $1 < c < a$ 를 만족시킨다.
양변에 밑이 $b$ 인 로그를 취하면 $b > 1$ 이므로 부동호의 방향이 바뀌지 않아 $\log_{b} c < \log_{b} a$ 를 만족한다. (거짓)

ㄴ) $0 < \log_{a} c < 1$ 이고 $0 < a < 1$ 이므로 $a < c < 1$ 를 만족시킨다.
$b > 1$ 이므로 양변에 밑이 $b$ 인 로그를 취하면 $\log_{b} a < \log_{b} c$ 가 성립한다. (참)

ㄷ) $0 < a < 1$ 이고 $\log_{a} c < 0$ 이므로 $c > 1$ 이다. 
$0 < a < b < 1$ 이므로 $\log_{a} b > 0$ 이고, $c > 1$ 이며 $0 < b < 1$ 이므로 $\log_{c} b < 0$ 이다. 
그러므로 $\log_{a} b > \log_{c} b$ 가 성립한다. (거짓)

$\therefore$ $ㄴ$만 옳다.
