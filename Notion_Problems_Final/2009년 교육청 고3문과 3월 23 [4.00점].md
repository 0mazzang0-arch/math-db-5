## 문제
$\log _{3} n$ 의 정수부분과 $\log _{4} n$ 의 정수부분이 같도록 하는 두 자리의 자연수 $n$ 의 최댓값을 구하시오.

## 보기

## 풀이
$10 \le n < 16$ 일 때 $2 < \log _{3} n < 3$, $1 < \log _{4} n < 2$ 이므로 $\log _{3} n$ 과 $\log _{4} n$ 의 정수부분은 각각 $2$, $1$ 이다.

$16 \le n < 27$ 일 때 $2 < \log _{3} n < 3$, $2 \le \log _{4} n < 3$ 이므로 $\log _{3} n$ 과 $\log _{4} n$ 의 정수부분은 $2$ 로 같다.

$27 \le n < 64$ 일 때 $3 \le \log _{3} n < 4$, $2 < \log _{4} n < 3$ 이므로 $\log _{3} n$ 과 $\log _{4} n$ 의 정수부분은 각각 $3$, $2$ 이다.

$64 \le n < 81$ 일 때 $3 < \log _{3} n < 4$, $3 \le \log _{4} n < 4$ 이므로 $\log _{3} n$ 과 $\log _{4} n$ 의 정수부분은 $3$ 으로 같다.

$81 \le n < 100$ 일 때 $4 \le \log _{3} n < 5$, $3 < \log _{4} n < 4$ 이므로 $\log _{3} n$ 과 $\log _{4} n$ 의 정수부분은 각각 $4$, $3$ 이다.

따라서 구하는 두 자리의 자연수 $n$ 의 최댓값은 $80$ 이다.

정답: $80$
