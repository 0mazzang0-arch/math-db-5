## 문제
최고차항의 계수가 $1$인 이차함수 $f(x)$에 대하여 함수 $g(x)$가
$$g(x) = \int_{0}^{x} \frac{t}{f(t)} dt$$
일 때, 함수 $g(x)$는 다음 조건을 만족시킨다.

(가) 모든 실수 $x$에 대하여 $g'(-x) = -g'(x)$이다.
(나) 점 $(1, g(1))$은 곡선 $y=g(x)$의 변곡점이다.

$g(1)$의 값은?

## 보기
① $\frac{1}{5} \ln 2$   ② $\frac{1}{4} \ln 2$   ③ $\frac{1}{3} \ln 2$   ④ $\frac{1}{2} \ln 2$   ⑤ $\ln 2$

## 풀이
$g'(x) = \frac{x}{f(x)}$이고, $g'(-x) = -g'(x)$이므로
$\frac{-x}{f(-x)} = -\frac{x}{f(x)}$에서 $f(-x) = f(x)$이다.
$f(x)$는 최고차항의 계수가 $1$인 이차함수이므로
$f(x) = x^2 + k$ ($k$는 상수)
점 $(1, g(1))$은 곡선 $y=g(x)$의 변곡점이므로 $g''(1) = 0$이다.
$g''(x) = \frac{f(x) - x f'(x)}{\{f(x)\}^2}$
$g''(1) = \frac{f(1) - f'(1)}{\{f(1)\}^2} = \frac{(1 + k) - 2}{(1 + k)^2} = \frac{k - 1}{(1 + k)^2} = 0$
$k = 1$이므로 $f(x) = x^2 + 1$이다.
따라서
$g(1) = \int_{0}^{1} \frac{t}{t^2 + 1} dt$
$= \frac{1}{2} [\ln (t^2 + 1)]_{0}^{1}$
$= \frac{1}{2} \ln 2$

정답: ④
