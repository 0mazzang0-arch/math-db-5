## 문제
$\lim_{n \to \infty} n (\sqrt{n^{2} + 1} - n)$의 값은?

## 보기
① $\frac{1}{4}$   ② $\frac{1}{3}$   ③ $\frac{1}{2}$   ④ $1$   ⑤ $2$

## 풀이
$\lim_{n \to \infty} n (\sqrt{n^{2} + 1} - n)$
$= \lim_{n \to \infty} \frac{n(\sqrt{n^{2} + 1} - n)(\sqrt{n^{2} + 1} + n)}{\sqrt{n^{2} + 1} + n}$
$= \lim_{n \to \infty} \frac{n(n^{2} + 1 - n^{2})}{\sqrt{n^{2} + 1} + n}$
$= \lim_{n \to \infty} \frac{n}{\sqrt{n^{2} + 1} + n}$
분모, 분자를 $n$으로 나누면
$= \lim_{n \to \infty} \frac{1}{\sqrt{1 + \frac{1}{n^{2}}} + 1} = \frac{1}{1 + 1} = \frac{1}{2}$

정답은 ③입니다.
