## 문제
모든 실수 $x$ 에 대하여 함수 $f(x)$ 는 다음 조건을 만족시킨다. 

$$\int_{12}^{x} f(t) dt = -x^{3} + x^{2} + \int_{0}^{1} x f(t) dt$$ 

$\int_{0}^{1} f(x) dx$ 의 값을 구하시오.

## 풀이
정적분의 성질에 의해 $x = 12$ 를 대입하면 $\int_{12}^{12} f(t) dt = 0$ 이므로 
$$-12^{3} + 12^{2} + \int_{0}^{1} 12 f(t) dt = 0$$ 
따라서 
$$\int_{0}^{1} f(x) dx = 132$$ 

(다른 풀이)
$$\int_{12}^{x} f(t) dt = -x^{3} + x^{2} + \int_{0}^{1} x f(t) dt = -x^{3} + x^{2} + x \int_{0}^{1} f(t) dt$$ 
$\int_{0}^{1} f(t) dt = c$ 라 하고 위 등식의 양변을 $x$ 에 관하여 미분하면 
$$f(x) = -3x^{2} + 2x + c$$ 
$\int_{12}^{x} f(t) dt = -x^{3} + x^{2} + \int_{0}^{1} x f(t) dt$ 에 대입하면 
$$\int_{12}^{x} (-3t^{2} + 2t + c) dt = [-t^{3} + t^{2} + ct]_{12}^{x}$$ 
$$= -x^{3} + x^{2} + cx - (-12^{3} + 12^{2} + 12c)$$ 
$$= -x^{3} + x^{2} + cx - 12(-132 + c)$$ 
이 식은 $-x^{3} + x^{2} + cx$ 와 같아야 하므로 
$$-132 + c = 0$$ 
따라서 
$$\int_{0}^{1} f(x) dx = 132$$
