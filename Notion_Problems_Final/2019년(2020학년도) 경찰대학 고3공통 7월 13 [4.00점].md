## 문제
한 개의 주사위를 $3$번 던질 때, 나온 눈의 수를 차례로 $a$, $b$, $c$라 하고, 함수 $f(x)$를
$$f(x) = (a - 3)(x^{2} + 2bx + c)$$
로 정의하자. 함수
$$g(x) = \begin{cases} 1 & (x > 0) \\ 0 & (x \le 0) \end{cases}$$
에 대하여 합성함수 $(g \circ f)(x)$가 실수 전체의 집합에서 연속일 확률은?

## 보기
① $\frac{17}{72}$   ② $\frac{7}{24}$   ③ $\frac{25}{72}$   ④ $\frac{29}{72}$   ⑤ $\frac{11}{24}$

## 풀이
$(g \circ f)(x)$가 실수 전체의 집합에서 연속이려면 $f(x)$의 함숫값이 $g(x)$의 불연속점인 $0$을 기준으로 한 영역에만 머물러야 한다. 즉, 모든 실수 $x$에 대하여 $f(x) > 0$이거나 $f(x) \le 0$이어야 한다.

주사위의 눈 $a$, $b$, $c$에 따른 경우는 다음과 같다.

$$ \begin{array}{c|l|c} \hline \text{경우} & \text{조건 및 내용} & \text{확률} \\ \hline a = 3 & f(x) = 0 \text{이므로 } (g \circ f)(x) = 0 \text{ (상수함수)} & \frac{1}{6} \\ \hline a < 3 & f(x) \le 0 \text{이어야 하므로 } x^2 + 2bx + c \ge 0 \implies D/4 = b^2 - c \le 0 & \frac{2}{6} \times \frac{9}{36} = \frac{1}{12} \\ \hline a > 3 & f(x) > 0 \text{이어야 하므로 } x^2 + 2bx + c > 0 \implies D/4 = b^2 - c < 0 & \frac{3}{6} \times \frac{7}{36} = \frac{7}{72} \\ \hline \end{array} $$

$1$) $a = 3$일 때:
나온 눈의 수 $a$가 $3$인 경우이므로 확률은 $\frac{1}{6}$이다.

$2$) $a < 3$ ($a \in \{1, 2\}$)일 때:
$a-3 < 0$이므로 $f(x) \le 0$이 되려면 $x^2 + 2bx + c \ge 0$이어야 한다.
판별식 $D/4 = b^2 - c \le 0$을 만족하는 $(b, c)$의 개수는 다음과 같다.
- $b = 1$일 때: $c \ge 1$ $\implies$ $c = 1, 2, 3, 4, 5, 6$ ($6$개)
- $b = 2$일 때: $c \ge 4$ $\implies$ $c = 4, 5, 6$ ($3$개)
따라서 $(b, c)$의 쌍은 총 $9$가지이며, 이 경우의 확률은 $\frac{2}{6} \times \frac{9}{36} = \frac{1}{3} \times \frac{1}{4} = \frac{1}{12}$이다.

$3$) $a > 3$ ($a \in \{4, 5, 6\}$)일 때:
$a-3 > 0$이므로 $f(x) > 0$이 되려면 $x^2 + 2bx + c > 0$이어야 한다.
판별식 $D/4 = b^2 - c < 0$을 만족하는 $(b, c)$의 개수는 다음과 같다.
- $b = 1$일 때: $c > 1$ $\implies$ $c = 2, 3, 4, 5, 6$ ($5$개)
- $b = 2$일 때: $c > 4$ $\implies$ $c = 5, 6$ ($2$개)
따라서 $(b, c)$의 쌍은 총 $7$가지이며, 이 경우의 확률은 $\frac{3}{6} \times \frac{7}{36} = \frac{1}{2} \times \frac{7}{36} = \frac{7}{72}$이다.

$\therefore$ 구하는 전체 확률은
$$\frac{1}{6} + \frac{1}{12} + \frac{7}{72} = \frac{12}{72} + \frac{6}{72} + \frac{7}{72} = \frac{25}{72}$$

정답은 ③이다.
