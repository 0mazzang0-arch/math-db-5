## 문제
$0 \le x \le 2 \pi$ 일 때, 삼각방정식 $\sin 2x = \cos x$ 의 모든 해의 합은?

## 보기
① $\frac{3}{2} \pi$   ② $2 \pi$   ③ $\frac{5}{2} \pi$   ④ $3 \pi$   ⑤ $\frac{7}{2} \pi$

## 풀이
정답 ④

$\sin 2x = \cos x$
$2 \sin x \cos x = \cos x$
$(2 \sin x - 1) \cos x = 0$
이므로
$\sin x = \frac{1}{2}$ 또는 $\cos x = 0$

(ⅰ) $\sin x = \frac{1}{2}$ 일 때
$0 \le x \le 2 \pi$ 이므로
$x = \frac{\pi}{6}$ 또는 $x = \frac{5}{6} \pi$

(ⅱ) $\cos x = 0$ 일 때
$0 \le x \le 2 \pi$ 이므로
$x = \frac{\pi}{2}$ 또는 $x = \frac{3}{2} \pi$

(ⅰ), (ⅱ)에 의하여 모든 해의 합은
$\frac{\pi}{6} + \frac{5}{6} \pi + \frac{\pi}{2} + \frac{3}{2} \pi = 3 \pi$
