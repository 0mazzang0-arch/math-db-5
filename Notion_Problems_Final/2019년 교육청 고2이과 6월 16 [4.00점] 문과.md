## 문제
$0 < \theta < 2 \pi$ 에서 $3 + 2 \sin^{2} \theta + \frac{1}{3 - 2 \cos^{2} \theta}$ 의 최솟값을 구하는 과정이다.

$3 + 2 \sin^{2} \theta = t$ 로 놓으면
$3 + 2 \sin^{2} \theta + \frac{1}{3 - 2 \cos^{2} \theta} = t + \frac{1}{(가)}$
이다. $0 < \theta < 2 \pi$ 에서 $t \ge 3$ 이므로 $(가) > 0$ 이다.
$t + \frac{1}{(가)} = t - 2 + \frac{1}{(가)} + 2 \ge 4$
이다. (단, 등호는 $t = (나)$ 일 때 성립한다.)
따라서 $3 + 2 \sin^{2} \theta + \frac{1}{3 - 2 \cos^{2} \theta}$ 은 $\theta = (다)$ 에서 최솟값 $4$ 를 갖는다.
위의 $(가)$ 에 알맞은 식을 $f(t)$, $(나)$ 와 $(다)$ 에 알맞은 수를 각각 $p$, $q$ 라 할 때, $f(p) + \tan^{2} ( q + \frac{\pi}{3} )$ 의 값은?

## 보기
① $4$   ② $5$   ③ $6$   ④ $7$   ⑤ $8$

## 풀이
$3 + 2 \sin^{2} \theta = t$ 로 놓으면
$3 - 2 \cos^{2} \theta = 3 - 2 ( 1 - \sin^{2} \theta ) = 1 + 2 \sin^{2} \theta$ 이다.
$t = 3 + 2 \sin^{2} \theta$ 에서 $2 \sin^{2} \theta = t - 3$ 이므로
$1 + 2 \sin^{2} \theta = 1 + t - 3 = t - 2$
이다. 따라서 $(가)$ 에 알맞은 식은 $f(t) = t - 2$ 이다.

$0 < \theta < 2 \pi$ 에서 $0 \le \sin^{2} \theta \le 1$ 이므로 $3 \le t \le 5$ 이다.
$t \ge 3$ 이므로 $t - 2 > 0$ 이다.
산술$\cdot$기하평균의 관계에 의하여
$t + \frac{1}{t - 2} = t - 2 + \frac{1}{t - 2} + 2 \ge 2 \sqrt{(t - 2) \cdot \frac{1}{t - 2}} + 2 = 4$ 이다.
단, 등호는 $t - 2 = \frac{1}{t - 2}$ 일 때 성립한다.
$(t - 2)^{2} = 1$ 에서 $t - 2 = 1$ ($\because t \ge 3$)
$t = 3$ 일 때 성립하므로 $(나)$ 에 알맞은 수 $p = 3$ 이다.

$t = 3 + 2 \sin^{2} \theta = 3$ 일 때, $2 \sin^{2} \theta = 0$ 이므로 $\sin \theta = 0$ 이다.
$0 < \theta < 2 \pi$ 에서 $\theta = \pi$ 이므로 $(다)$ 에 알맞은 수 $q = \pi$ 이다.

따라서
$f(p) + \tan^{2} ( q + \frac{\pi}{3} ) = f(3) + \tan^{2} ( \pi + \frac{\pi}{3} )$
$= ( 3 - 2 ) + \tan^{2} \frac{\pi}{3}$
$= 1 + ( \sqrt{3} )^{2} = 1 + 3 = 4$
