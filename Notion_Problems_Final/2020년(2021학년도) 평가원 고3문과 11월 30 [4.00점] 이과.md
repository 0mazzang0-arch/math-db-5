## 문제
함수 $f(x)$는 최고차항의 계수가 $1$인 삼차함수이고, 함수 $g(x)$는 일차함수이다. 함수 $h(x)$를 
$$h(x) = \begin{cases} |f(x) - g(x)| & (x < 1) \\ f(x) + g(x) & (x \ge 1) \end{cases}$$ 
이라 하자. 함수 $h(x)$가 실수 전체의 집합에서 미분가능하고, $h(0) = 0$, $h(2) = 5$일 때, $h(4)$의 값을 구하시오.

## 풀이
$y = |f(x) - g(x)|$가 $x = 0$에서 미분가능하고 $h(0) = |f(0) - g(0)| = 0$이므로 $h'(0) = 0$이다.
즉, $f'(0) = g'(0)$이므로 $y = f(x) - g(x) = x^{2}(x - \beta)$의 형태이며, 또한 $x < 1$에서 $h(x) = |f(x) - g(x)|$가 미분가능하므로 $y = f(x) - g(x) = x^{2}(x - \beta) \le 0$이다.

> [!example] 📸 (그림/그래프)

즉, $h(x) = \begin{cases} -f(x) + g(x) & (x < 1) \\ f(x) + g(x) & (x \ge 1) \end{cases}$이다.
$x = 1$에서 미분가능하므로 연속이고
$-f(1) + g(1) = f(1) + g(1)$
$-f'(1) + g'(1) = f'(1) + g'(1)$
이 성립한다. 즉, $f(1) = 0, f'(1) = 0$이다.
그러므로 $f(x) = (x - \alpha)(x - 1)^{2}$의 꼴이며 $g(x)$는 $x = 0$에서 $f(x)$와 접한다.
$f(0) = -\alpha$, $f'(x) = (x - 1)^{2} + 2(x - \alpha)(x - 1)$에서 $f'(0) = 1 + 2\alpha$이다.
$x = 0$에서의 접선의 식 $y + \alpha = (1 + 2\alpha)x$로부터 $g(x) = (1 + 2\alpha)x - \alpha$이다.
$h(2) = f(2) + g(2) = (2 - \alpha) + 2 + 4\alpha - \alpha = 5$로부터 $2\alpha = 1$, 즉 $\alpha = \frac{1}{2}$이다.
그러므로 $f(x) = (x - \frac{1}{2})(x - 1)^{2}$, $g(x) = 2x - \frac{1}{2}$이다.
$h(4) = f(4) + g(4) = \frac{7}{2} \times 3^{2} + \frac{15}{2} = \frac{78}{2} = 39$이다.

**[다른풀이]**
$h(x) = \begin{cases} |f(x) - g(x)| & (x < 1) \\ f(x) + g(x) & (x \ge 1) \end{cases}$이 실수 전체에서 미분가능하므로 $x = 1$에서 연속이다.
(ⅰ) $f(1) \ge g(1)$일 때,
$f(1) - g(1) = f(1) + g(1)$이므로 $g(1) = 0$이다. (연속)
$f'(1) - g'(1) = f'(1) + g'(1)$이므로 $g'(1) = 0$이다. (미분가능)
이때 $g(x)$는 일차함수이므로 $g'(1) \neq 0$이어야 하므로 모순이다.

(ⅱ) $f(1) < g(1)$일 때,
$-f(1) + g(1) = f(1) + g(1)$이므로 $f(1) = 0$이다. (연속)
$-f'(1) + g'(1) = f'(1) + g'(1)$이므로 $f'(1) = 0$이다. (미분가능)
(ⅰ), (ⅱ)에 의하여 $f(x) = (x - 1)^{2}(x - \alpha)$이다.
조건에 의해 $h(0) = 0$이므로 $|f(0) - g(0)| = 0$이다.
$\therefore f(0) = g(0)$
$f(0) = -\alpha$이므로 $g(x) = px - \alpha$라 하자. ($p$는 $0$이 아닌 실수)
$h(2) = f(2) + g(2) = 2 - \alpha + 2p - \alpha = 5$
$\therefore 2p - 2\alpha = 3$
$\therefore p = \frac{3}{2} + \alpha$
따라서 $x < 1$일 때 $h(x) = |f(x) - g(x)| = |x^{3} - (2 + \alpha)x^{2} + (\alpha - \frac{1}{2})x|$이다.
절댓값 함수가 $x = 0$에서 해를 가지므로 실수 전체에서 미분가능하기 위해서는 반드시 중근을 가져야 한다.
따라서 일차항의 계수가 $0$이어야 하므로 $\alpha = \frac{1}{2}$이다.
$\therefore h(4) = f(4) + g(4) = 9 \times \frac{7}{2} + \frac{15}{2} = 39$이다.

정답: $39$
