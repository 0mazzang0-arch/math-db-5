## 문제

이차정사각행렬 $A$, $B$에 대하여 $AB=C$일 때, 이를 [그림 $1$]과 같이 나타내기로 하자.

> [!example] 📸 (그림/그래프)
> [그림 $1$]

이와 같은 방법으로 [그림 $2$]의 이차정사각행렬 $C_{1}$, $C_{2}$, $C_{3}$, $C_{4}$와 $D_{1}$, $D_{2}$, $D_{3}$을 정의할 때, <보기>에서 항상 옳은 것을 모두 고른 것은?

> [!example] 📸 (그림/그래프)
> [그림 $2$]

**<보기>**
ㄱ. $C_{1}=O$ 이면 $A=O$ 이다.
ㄴ. $C_{2}=C_{3}$ 이면 $D_{2}=D_{3}$ 이다.
ㄷ. $D_{2}=E$ 이면 $D_{3}=E$ 이다.

(단, $O$는 영행렬, $E$는 단위행렬이다.)

## 보기

① $ㄴ$   ② $ㄷ$   ③ $ㄱ, ㄴ$   ④ $ㄱ, ㄷ$   ⑤ $ㄴ, ㄷ$

## 풀이

ㄱ. 거짓

$A = \begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix}$이라 하면

$C_{1} = A^{2} = \begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix} \begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix} = \begin{pmatrix} 0 & 0 \\ 0 & 0 \end{pmatrix}$

$C_{1}=O$ 이지만 $A \neq O$ 이다.

ㄴ. 참

$C_{2}=AB$, $C_{3}=BA$, $C_{4}=AB$ 에서

$C_{2}=C_{3}$ 이면 $AB=BA$ 이므로

$D_{2} = C_{2}C_{3} = (AB)(BA) = (AB)(AB) = ABAB$

$D_{3} = C_{3}C_{4} = (BA)(AB) = (AB)(AB) = ABAB$

$\therefore D_{2}=D_{3}$

ㄷ. 참

$D_{2} = ABBA = E$ 이면

$(AB)^{-1} = BA$, $(BA)^{-1} = AB$ 이므로

$D_{3} = BAAB = (AB)^{-1} AB = E$

그러므로 항상 옳은 것은 $ㄴ, ㄷ$이다.
