## 문제
$3000$ 보다 작은 네 자리 자연수 중 각 자리의 수의 합이 $10$ 이 되는 모든 자연수의 개수를 구하시오.

## 풀이
$9$ 이하의 음이 아닌 정수 $a$, $b$, $c$, $d$ 에 대하여 네 자리 이하의 자연수를 $a \times 10^{3} + b \times 10^{2} + c \times 10 + d$ 라 하자.
$3000$ 보다 작은 네 자리 자연수 중 각 자리의 수의 합이 $10$ 이므로 $1 \le a \le 2$ 이고 $a + b + c + d = 10$ 이다.

(ⅰ) $a = 1$ 인 경우
$b + c + d = 9$ 를 만족시키는 음이 아닌 세 정수 $b$, $c$, $d$ 의 순서쌍 $(b, c, d)$ 의 개수는
${}_{3}H_{9} = {}_{11}C_{9} = 55$

(ⅱ) $a = 2$ 인 경우
$b + c + d = 8$ 을 만족시키는 음이 아닌 세 정수 $b$, $c$, $d$ 의 순서쌍 $(b, c, d)$ 의 개수는
${}_{3}H_{8} = {}_{10}C_{8} = 45$

(ⅰ), (ⅱ)에 의하여 경우의 수는 $55 + 45 = 100$ 이다.

정답: $100$
