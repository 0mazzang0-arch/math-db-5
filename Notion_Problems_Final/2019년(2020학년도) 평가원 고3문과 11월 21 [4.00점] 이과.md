## 문제
수열 $\{a_{n}\}$이 모든 자연수 $n$에 대하여 다음 조건을 만족시킨다.

(가) $a_{2n} = a_{n} - 1$
(나) $a_{2n+1} = 2a_{n} + 1$

$a_{20} = 1$일 때, $\sum_{n=1}^{63} a_{n}$의 값은?

## 보기
① $704$   ② $712$   ③ $720$   ④ $728$   ⑤ $736$

## 풀이
$a_{20} = a_{10} - 1$에서 $a_{20} = 1$이므로 $a_{10} = 2$이다.
또, $a_{10} = a_{5} - 1$에서 $a_{5} = 3$이다.
$a_{5} = 2a_{2} + 1$에서 $a_{2} = 1$이다.
$a_{2} = a_{1} - 1$에서 $a_{1} = 2$이다.

한편, $a_{2n} + a_{2n+1} = (a_{n} - 1) + (2a_{n} + 1) = 3a_{n}$이므로 다음과 같은 관계가 성립한다.
$$ \begin{array}{l} a_{2} + a_{3} = 3a_{1} \\ a_{4} + a_{5} + a_{6} + a_{7} = 3(a_{2} + a_{3}) = 3^{2} a_{1} \\ a_{8} + a_{9} + \cdots + a_{15} = 3(a_{4} + a_{5} + a_{6} + a_{7}) = 3^{3} a_{1} \\ a_{16} + a_{17} + \cdots + a_{31} = 3(a_{8} + \cdots + a_{15}) = 3^{4} a_{1} \\ a_{32} + a_{33} + \cdots + a_{63} = 3(a_{16} + \cdots + a_{31}) = 3^{5} a_{1} \end{array} $$

따라서 구하고자 하는 값은 다음과 같다.
$$ \begin{array}{l} \sum_{n=1}^{63} a_{n} \\ = a_{1} + (a_{2} + a_{3}) + (a_{4} + \cdots + a_{7}) + (a_{8} + \cdots + a_{15}) + (a_{16} + \cdots + a_{31}) + (a_{32} + \cdots + a_{63}) \\ = a_{1} (1 + 3 + 3^{2} + 3^{3} + 3^{4} + 3^{5}) \\ = 2 \times \frac{3^{6} - 1}{3 - 1} \\ = 2 \times \frac{728}{2} \\ = 728 \end{array} $$
정답은 ④이다.
