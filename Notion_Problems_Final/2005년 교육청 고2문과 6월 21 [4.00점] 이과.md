## 문제
서로 다른 두 자연수 $x$, $y$가 다음 세 조건을 모두 만족할 때, $xy$의 값은?

(가) 자연수 $x$, $y$의 자리수는 같다.
(나) $x$와 $\frac{1}{y}$의 상용로그의 소수 부분은 같다.
(다) $x^{4} y^{3}$의 상용로그의 정수 부분은 $10$이다.

## 보기
① $10$   ② $10^{2}$   ③ $10^{3}$   ④ $10^{4}$   ⑤ $10^{5}$

## 풀이
[정답] ③

(가) $x$, $y$의 자릿수가 같으므로 $\log x$와 $\log y$의 정수 부분은 같다.
$\log x = n + \alpha$, $\log y = n + \beta$ ($n$은 정수, $0 \le \alpha < 1, 0 \le \beta < 1$)라고 하면
(나)에서 $x$와 $\frac{1}{y}$의 상용로그의 소수 부분이 같으므로
$\log x - \log \frac{1}{y} = \log x + \log y = \text{정수}$이다.
$(n + \alpha) + (n + \beta) = 2n + \alpha + \beta$가 정수이어야 하므로 $\alpha + \beta$는 정수이다.
$0 \le \alpha < 1, 0 \le \beta < 1$이므로 $\alpha + \beta$는 $0$ 또는 $1$이다.
$x$, $y$가 서로 다른 자연수이므로 $\alpha + \beta = 1$이다.

(다) $\log x^{4} y^{3} = 4 \log x + 3 \log y$
$= 4(n + \alpha) + 3(n + \beta)$
$= 7n + 3(\alpha + \beta) + \alpha$
$= 7n + 3 + \alpha$
$\log x^{4} y^{3}$의 정수 부분이 $10$이므로
$7n + 3 = 10$
$7n = 7$
$n = 1$

구하는 값은 $\log xy$이므로
$\log xy = \log x + \log y$
$= n + \alpha + n + \beta$
$= 2n + (\alpha + \beta)$
$= 2(1) + 1 = 3$
따라서 $xy = 10^{3}$이다.
