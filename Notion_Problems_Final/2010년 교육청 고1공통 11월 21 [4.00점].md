## 문제
임의의 두 실수 $a$, $b$에 대하여 두 연산 $◎$와 $*$를 각각 
$a ◎ b = a b - ( a + b ) + 2$, 
$a * b = ( a + 1 ) ( b + 1 ) - 1$
로 정의할 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?

$$
\begin{array}{|l|}
\hline
\text{ㄱ. } ( 2 * 3 ) ◎ 4 = 31 \\
\text{ㄴ. 연산 } ◎ \text{ 에 대한 항등원은 } 2 \text{ 이다.} \\
\text{ㄷ. 연산 } * \text{ 에 대한 } 1 \text{ 의 역원은 } - \frac{1}{2} \text{ 이다.} \\
\hline
\end{array}
$$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
ㄱ. 
$( 2 * 3 ) ◎ 4$ 
$= 11 ◎ 4$ 
$= 44 - ( 11 + 4 ) + 2$ 
$= 31$ (참) 

ㄴ. 임의의 실수 $a$ 에 대하여 연산 $◎$ 에 대한 항등원을 $e$ 라 하면, 
$a ◎ e = a$ 
$\Leftrightarrow a e - ( a + e ) + 2 = a$ 
$\Leftrightarrow a e - e - 2 a + 2 = 0$ 
$\Leftrightarrow ( a - 1 ) ( e - 2 ) = 0$ 이므로 $e = 2$ 이다. 
그러므로 연산 $◎$ 에 대한 항등원은 $2$ 이다. (참) 

ㄷ. 임의의 실수 $b$ 에 대하여 연산 $*$ 에 대한 항등원을 $e$ 라 하면, 
$b * e = b$ 
$\Leftrightarrow ( b + 1 ) ( e + 1 ) - 1 = b$ 
$\Leftrightarrow b e + b + e = b$ 
$\Leftrightarrow ( b + 1 ) e = 0$ 이므로 $e = 0$ 이다. 
이때, 연산 $*$ 에 대한 $1$ 의 역원을 $x$ 라 하면, 
$1 * x = 0$ 
$\Leftrightarrow 2 ( x + 1 ) - 1 = 0$ 
$\Leftrightarrow 2 x = - 1$ 이므로 $x = - \frac{1}{2}$ 이다. 
그러므로 연산 $*$ 에 대한 $1$ 의 역원은 $- \frac{1}{2}$ 이다. (참) 

따라서 옳은 것은 ㄱ, ㄴ, ㄷ이다.
