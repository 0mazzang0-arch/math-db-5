## 문제
두 조건 $p$, $q$의 진리집합을 각각 $P$, $Q$라 하고 
$P = \{ x | (x+4)(x-5) \leq 0 \}$, $Q = \{ x | |x| > a \}$
일 때, 명제 $\sim p \to q$가 참이기 위한 자연수 $a$의 개수는?

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
명제 $\sim p \to q$가 참이기 위해서는 $P^{c} \subset Q$를 만족해야 한다.
조건 $p$에서 $P = \{ x | -4 \leq x \leq 5 \}$이므로
$P^{c} = \{ x | x < -4 \text{ 또는 } x > 5 \}$이다.
조건 $q$에서 $Q = \{ x | x < -a \text{ 또는 } x > a \}$이다. ($a$는 자연수)

> [!example] 📸 (그림/그래프)

$P^{c} \subset Q$가 성립하기 위해서는 다음의 조건을 만족해야 한다.
$-4 \leq -a$ 이고 $a \leq 5$
즉, $a \leq 4$ 이고 $a \leq 5$ 이므로 공통 범위는 $a \leq 4$이다.
따라서 자연수 $a$는 $1, 2, 3, 4$로 그 개수는 $4$이다.
