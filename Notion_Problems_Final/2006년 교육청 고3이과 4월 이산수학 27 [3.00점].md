## 문제
함수 $y = \cos x - \cos \left( x + \frac{\pi}{3} \right)$ 가 $x = \alpha$ 에서 최댓값을 가질 때, $\tan \alpha$ 의 값은?
(단, $0 \le x < 2 \pi$)

## 보기
① $0$   ② $\frac{1}{3}$   ③ $\frac{\sqrt{3}}{3}$   ④ $1$   ⑤ $\sqrt{3}$

## 풀이
$y = \cos x - \cos \left( x + \frac{\pi}{3} \right)$
$= \cos x - \left( \cos x \cos \frac{\pi}{3} - \sin x \sin \frac{\pi}{3} \right)$
$= \frac{\sqrt{3}}{2} \sin x + \frac{1}{2} \cos x = \sin \left( x + \frac{\pi}{6} \right)$

$y$ 는 $x + \frac{\pi}{6} = \frac{\pi}{2}$ 일 때 최댓값을 갖는다.
$\therefore \alpha = \frac{\pi}{3}$, $\tan \alpha = \tan \frac{\pi}{3} = \sqrt{3}$

[다른 풀이]
$y = \cos x - \cos \left( x + \frac{\pi}{3} \right) = -2 \sin \left( x + \frac{\pi}{6} \right) \sin \left( - \frac{\pi}{6} \right)$
$= \sin \left( x + \frac{\pi}{6} \right)$

정답: ⑤
