## 문제
연속확률변수 $X$ 는 평균이 $20$ , 표준편차가 $4$ 인 정규분포를 따른다. 함수 $f(k)$ 를 $f(k) = P(k-8 \le X \le k)$ 로 정의할 때, <보기>에서 $f(k)$ 에 대한 설명으로 옳은 것만을 있는 대로 고른 것은?

## 보기
ㄱ. $f(12) = f(36)$
ㄴ. 함수 $f(k)$ 는 $k=24$ 일 때 최댓값을 갖는다.
ㄷ. 임의의 실수 $k$ 에 대하여 $f(k) = f(24-k)$ 이다.

① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
ㄱ. $f(12) = P(4 \le X \le 12)$ 이다. 확률밀도함수의 대칭성에 의해
$P(4 \le X \le 12) = P(20+8 \le X \le 20+16) = P(28 \le X \le 36) = f(36)$ (참)

ㄴ. $f(k)$ 가 최대일 때, 구간 $[k-8, k]$ 의 중점이 정규분포의 평균인 $20$ 과 일치해야 하므로
$\frac{(k-8)+k}{2} = 20$ 에서
$k-4 = 20$ 이므로 $k=24$ 이다. (참)

ㄷ. ㄴ에서 $f(k)$ 는 $k=24$ 에 대하여 대칭임을 알 수 있다. 즉, 임의의 실수 $x$ 에 대하여 $f(24-x) = f(24+x)$ 가 성립하며, 이는 $f(k) = f(48-k)$ 를 의미한다.
$f(k) = f(24-k)$ 는 함수 $f(k)$ 가 $k=12$ 에 대하여 대칭임을 의미하므로 옳지 않다. (거짓)

따라서 옳은 것은 ㄱ, ㄴ이다.
