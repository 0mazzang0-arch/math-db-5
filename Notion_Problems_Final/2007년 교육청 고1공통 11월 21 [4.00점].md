## 문제
방정식 $[x]^{2} + [y^{2}] = 1$ 을 만족시키는 점 $(x, y)$ 가 좌표평면 위에 나타내는 영역의 넓이는? (단, $[x]$ 는 $x$ 보다 크지 않은 최대의 정수이다.)

## 보기
① $2(\sqrt{2}-1)$   ② $2(2-\sqrt{2})$   ③ $4(\sqrt{2}-1)$   ④ $2(1+\sqrt{2})$   ⑤ $2(2+\sqrt{2})$

## 풀이
[정답] ④

$[x]^{2} + [y^{2}] = 1$ 에서 $[x]^{2}$ 과 $[y^{2}]$ 은 정수이므로
$[x]^{2} = 0$ 이고 $[y^{2}] = 1$ 또는 $[x]^{2} = 1$ 이고 $[y^{2}] = 0$ 인 경우이다.

즉,
$[x] = 0$ 이고 $[y^{2}] = 1$
또는 $[x] = 1$ 이고 $[y^{2}] = 0$
또는 $[x] = -1$ 이고 $[y^{2}] = 0$ 인 경우이다.

(i) $[x] = 0$ 이고 $[y^{2}] = 1$ 인 경우
$0 \le x < 1$ 이고 $1 \le y^{2} < 2$ 이므로
$0 \le x < 1$ 이고 $-\sqrt{2} < y \le -1$ 또는 $1 \le y < \sqrt{2}$ $\cdots$ ㉠

(ii) $[x] = 1$ 이고 $[y^{2}] = 0$ 인 경우
$1 \le x < 2$ 이고 $0 \le y^{2} < 1$ 이므로
$1 \le x < 2$ 이고 $-1 < y < 1$ $\cdots$ ㉡

(iii) $[x] = -1$ 이고 $[y^{2}] = 0$ 인 경우
$-1 \le x < 0$ 이고 $0 \le y^{2} < 1$ 이므로
$-1 \le x < 0$ 이고 $-1 < y < 1$ $\cdots$ ㉢

㉠, ㉡, ㉢을 좌표평면에 나타내면 다음과 같다.

> [!example] 📸 (그림/그래프)

따라서 구하는 영역의 넓이는
$2 \{ 1 \times 2 + 1 \times (\sqrt{2}-1) \} = 2(1 + \sqrt{2})$
