## 문제
함수 $f(x) = x - [x]$와 $\{x \mid 1 \le x \le 4\}$에서 정의된 세 함수 $g_1(x) = x$, $g_2(x) = x^2$, $g_3(x) = \log(1 + x^2)$이 있다. 합성함수 $y = f(g_i(x))$ $(i = 1, 2, 3)$의 불연속점의 개수를 $a_i$라 할 때, $a_1, a_2, a_3$의 대소 관계를 옳게 나타낸 것은? (단, $[x]$는 $x$보다 크지 않은 최대의 정수이다.)

## 보기
① $a_1 < a_2 < a_3$   ② $a_1 < a_3 < a_2$   ③ $a_2 < a_1 < a_3$   ④ $a_3 < a_2 < a_1$   ⑤ $a_3 < a_1 < a_2$

## 풀이
(ⅰ) $y = f(g_1(x)) = x - [x]$ ($1 \le x \le 4$)의 그래프는 다음과 같다.

> [!example] 📸 (그림/그래프)

불연속점은 $x = 2$, $3$, $4$의 $3$개이므로 $a_1 = 3$

(ⅱ) $y = f(g_2(x)) = x^2 - [x^2]$ ($1 \le x \le 4$)
$1 \le x^2 < 2$, 즉 $1 \le x < \sqrt{2}$일 때 $y = x^2 - 1$
$2 \le x^2 < 3$, 즉 $\sqrt{2} \le x < \sqrt{3}$일 때 $y = x^2 - 2$
$3 \le x^2 < 4$, 즉 $\sqrt{3} \le x < 2$일 때 $y = x^2 - 3$
$\vdots$
$15 \le x^2 < 16$, 즉 $\sqrt{15} \le x < 4$일 때 $y = x^2 - 15$
$x^2 = 16$, 즉 $x = 4$일 때 $y = 16 - 16 = 0$

> [!example] 📸 (그림/그래프)

불연속점은 $x = \sqrt{2}, \sqrt{3}, \dots, \sqrt{16}$의 $15$개이므로 $a_2 = 15$

(ⅲ) $y = f(g_3(x)) = \log(x^2 + 1) - [\log(x^2 + 1)]$ ($1 \le x \le 4$)
$0 \le \log(x^2 + 1) < 1$에서 $1 \le x^2 + 1 < 10$이므로 $0 \le x^2 < 9$이다.
$1 \le x < 3$일 때 $y = \log(x^2 + 1)$
$1 \le \log(x^2 + 1) < 2$에서 $10 \le x^2 + 1 < 100$이므로 $9 \le x^2 < 99$이다.
따라서 $3 \le x \le 4$일 때 $y = \log(x^2 + 1) - 1 = \log \frac{x^2 + 1}{10}$

> [!example] 📸 (그림/그래프)

불연속점은 $x = 3$의 $1$개이므로 $a_3 = 1$

(ⅰ), (ⅱ), (ⅲ)에서 $a_3 < a_1 < a_2$

정답: ⑤
