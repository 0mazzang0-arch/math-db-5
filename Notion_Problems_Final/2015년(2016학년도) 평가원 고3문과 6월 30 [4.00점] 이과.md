## 문제

$2$ 이상의 자연수 $n$에 대하여 다음 조건을 만족시키는 자연수 $a$, $b$의 모든 순서쌍 $(a, b)$의 개수가 $300$ 이상이 되도록 하는 가장 작은 자연수 $k$의 값을 $f(n)$이라 할 때, $f(2) \times f(3) \times f(4)$의 값을 구하시오.

(가) $a < n^k$ 이면 $b \le \log_{n} a$ 이다.
(나) $a \ge n^k$ 이면 $b \le -(a - n^k)^2 + k^2$ 이다.

## 보기


## 풀이

자연수 $n$에 대하여
$g_n(x) = \log_n x$, 
$h_n(x) = -(x - n^k)^2 + k^2$ ($k$는 상수)
이라 하자.

$g_n(n^k) = \log_n n^k = k$,
$h_n(n^k) = k^2$ 이므로
$x < n^k$ 에서 $y = g_n(x)$의 그래프와
$x \ge n^k$ 에서 $y = h_n(x)$의 그래프는 그림과 같다.

> [!example] 📸 (그림/그래프)

($ⅰ$) 조건 (가)를 만족시키는 자연수 $a, b$의 순서쌍 $(a, b)$의 개수는 부등식
$1 \le x < n^k, y \le g_n(x)$
를 만족하는 $x, y$의 순서쌍 $(x, y)$의 개수와 같다.

$1 \le x < n$ 일 때, $0$ (개)
$n \le x < n^2$ 일 때, $1 \times (n^2 - n) = n^2 - n$ (개)
$n^2 \le x < n^3$ 일 때, $2 \times (n^3 - n^2) = 2n^3 - 2n^2$ (개)
$n^3 \le x < n^4$ 일 때, $3 \times (n^4 - n^3) = 3n^4 - 3n^3$ (개)
$\cdots \cdots$
$n^{k-1} \le x < n^k$ 일 때,
$(k-1)(n^k - n^{k-1}) = (k-1)n^k - (k-1)n^{k-1}$ (개)

이들의 총합은
$(k-1)n^k - (n + n^2 + n^3 + \cdots + n^{k-1})$
$= (k-1)n^k - \frac{n(n^{k-1}-1)}{n-1}$
$= (k-1)n^k - \frac{n^k - n}{n-1}$ (개)

($ⅱ$) 조건 (나)를 만족시키는 자연수 $a, b$의 순서쌍 $(a, b)$의 개수는 부등식
$n^k \le x < n^k + k, y \le h_n(x)$
를 만족하는 $x, y$의 순서쌍 $(x, y)$의 개수와 같다.

$x = n^k$ 일 때, $k^2$ (개)
$x = n^k + 1$ 일 때, $k^2 - 1^2$ (개)
$x = n^k + 2$ 일 때, $k^2 - 2^2$ (개)
$\cdots \cdots$
$x = n^k + k-1$ 일 때, $k^2 - (k-1)^2$ (개)

이들의 총합은
$k \times k^2 - \{1^2 + 2^2 + 3^2 + \cdots + (k-1)^2\}$
$= k^3 - \frac{(k-1)k(2k-1)}{6}$
$= \frac{k(k+1)(4k-1)}{6}$ (개)

두 조건 (가), (나)를 만족시키는 자연수 $a, b$의 순서쌍 $(a, b)$의 개수를 $A(n)$이라 하면 ($ⅰ$), ($ⅱ$)에 의해
$A(n) = (k-1)n^k - \frac{n^k - n}{n-1} + \frac{k(k+1)(4k-1)}{6}$
이므로 $f(2), f(3), f(4)$의 값은 각각 다음과 같다.

$①$ $n=2$ 일 때,
$A(2) = (k-1)2^k - (2^k - 2) + \frac{k(k+1)(4k-1)}{6} = (k-2)2^k + 2 + \frac{k(k+1)(4k-1)}{6}$
$k=5$ 이면 $A(2) = 96 + 2 + 95 = 193 < 300$
$k=6$ 이면 $A(2) = 256 + 2 + 161 = 419 > 300$
이므로 $A(2) \ge 300$ 을 만족시키는 자연수 $k$의 최솟값은 $6$이다.
$\therefore f(2) = 6$

$②$ $n=3$ 일 때,
$A(3) = (k-1)3^k - \frac{3^k - 3}{2} + \frac{k(k+1)(4k-1)}{6}$
$k=4$ 이면 $A(3) = 243 - 39 + 50 = 254 < 300$
$k=5$ 이면 $A(3) = 972 - 120 + 95 = 947 > 300$
이므로 $A(3) \ge 300$ 을 만족시키는 자연수 $k$의 최솟값은 $5$이다.
$\therefore f(3) = 5$

$③$ $n=4$ 일 때,
$A(4) = (k-1)4^k - \frac{4^k - 4}{3} + \frac{k(k+1)(4k-1)}{6}$
$k=3$ 이면 $A(4) = 128 - 20 + 22 = 130 < 300$
$k=4$ 이면 $A(4) = 768 - 84 + 50 = 734 > 300$
이므로 $A(4) \ge 300$ 을 만족시키는 자연수 $k$의 최솟값은 $4$이다.
$\therefore f(4) = 4$

$\therefore f(2) \times f(3) \times f(4) = 6 \times 5 \times 4 = 120$

정답: $120$
