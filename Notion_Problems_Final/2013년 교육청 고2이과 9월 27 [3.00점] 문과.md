## 문제
다음 조건을 만족시키는 수열 $\{a_{n}\}$에 대하여 $a_{9}$의 값을 구하시오.

(가) $a_{1} = 1$
(나) $a_{n+1} = a_{n} + 2^{n}$ ($n = 1, 2, 3, \dots$)

## 풀이
주어진 조건에서 
$a_{1} = 1$, 
$a_{2} = a_{1} + 2^{1}$, 
$a_{3} = a_{2} + 2^{2}, \dots$, 
$a_{9} = a_{8} + 2^{8}$ 
식을 각 변끼리 더하면 
$a_{1} + a_{2} + a_{3} + \dots + a_{9} = (a_{1} + a_{2} + \dots + a_{8}) + (1 + 2^{1} + 2^{2} + \dots + 2^{8})$ 
$\therefore a_{9} = 1 + 2^{1} + 2^{2} + \dots + 2^{8}$ 
$= \frac{2^{9} - 1}{2 - 1}$ 
$= 511$

정답: $511$
