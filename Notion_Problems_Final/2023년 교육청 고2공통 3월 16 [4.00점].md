## 문제
집합 $X = \{ x | 0 \le x \le 4 \}$ 에 대하여 $X$ 에서 $X$ 로의 함수
$$ f(x) = \begin{cases} ax^2 + b & (0 \le x < 3) \\ x - 3 & (3 \le x \le 4) \end{cases} $$
가 일대일대응일 때, $f(1)$ 의 값은? (단, $a$, $b$ 는 상수이다.)

## 보기
① $\frac{7}{3}$   ② $\frac{8}{3}$   ③ $3$   ④ $\frac{10}{3}$   ⑤ $\frac{11}{3}$

## 풀이
집합 $\{ x | 3 \le x \le 4 \}$ 에서 정의된 함수 $y = x - 3$ 의 치역은 $\{ y | 0 \le y \le 1 \}$ 이므로 함수 $f$ 가 일대일대응이 되기 위해서는 집합 $\{ x | 0 \le x < 3 \}$ 에서 정의된 함수 $y = ax^2 + b$ 의 치역이 $\{ y | 1 < y \le 4 \}$ 이어야 하고 함수 $y = f(x)$ 의 그래프는 그림과 같아야 한다.

> [!example] 📸 (그림/그래프)

따라서 이차함수 $g(x)$ 를 $g(x) = ax^2 + b$ 라 할 때, $g(0) = 4$, $g(3) = 1$ 이다.

$g(0) = 4$ 에서 $b = 4$

$g(3) = 1$ 에서 $9a + b = 1$
즉, $a = -\frac{1}{3}$

$$ f(x) = \begin{cases} -\frac{1}{3}x^2 + 4 & (0 \le x < 3) \\ x - 3 & (3 \le x \le 4) \end{cases} $$
이므로

$f(1) = -\frac{1}{3} \times 1^2 + 4 = \frac{11}{3}$

**[참고]**
위의 풀이에서 이차함수 $g(x) = ax^2 + b$ 에 대하여 $g(0) = 1$, $g(3) = 4$ 인 경우에는 함수 $y = f(x)$ 의 그래프가 그림과 같다.

> [!example] 📸 (그림/그래프)

이 경우에는 $f(0) = f(4) = 1$ 이므로 함수 $f(x)$ 는 일대일대응이 아니다. 또한, 공역의 원소 $4$ 가 치역에 속하지 않으므로 함수 $f(x)$ 는 일대일대응이 아니다.
