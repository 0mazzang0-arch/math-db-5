## 문제
자연수 $n$ 에 대하여 집합 $A_{n} = \{ x \mid \log_{2}x^{n}=4, \text{$x$ 는 정수} \}$, $B_{n} = \{ x \mid n \log_{2}x=4, \text{$x$ 는 정수} \}$ 일 때, 집합 $(A_{1} \cup A_{2} \cup A_{4}) - (B_{1} \cup B_{2} \cup B_{4})$ 의 모든 원소의 합은?

## 보기
① $-6$   ② $-2$   ③ $0$   ④ $2$   ⑤ $6$

## 풀이
로그의 정의를 이용하면 $\log_{2} x^{n} = 4$ 를 만족하는 $x$ 는 $x^{n} = 2^{4}$ 의 정수해이므로,
$A_{1} = \{ 16 \}$, $A_{2} = \{ -4, 4 \}$, $A_{4} = \{ -2, 2 \}$ 이다.
또한 $n \log_{2} x = 4$ 를 만족하는 $x$ 는 $x = 2^{\frac{4}{n}}$ 의 정수해이므로,
$B_{1} = \{ 16 \}$, $B_{2} = \{ 4 \}$, $B_{4} = \{ 2 \}$ 이다.

따라서
$(A_{1} \cup A_{2} \cup A_{4}) - (B_{1} \cup B_{2} \cup B_{4})$
$= \{ -4, -2, 2, 4, 16 \} - \{ 2, 4, 16 \}$
$= \{ -4, -2 \}$
그러므로 모든 원소의 합은 $-4 + (-2) = -6$ 이다.

정답: ①
