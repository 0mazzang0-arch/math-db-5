## 문제
두 다항식 $P(x) = 3x^3 + x + 11$, $Q(x) = x^2 - x + 1$에 대하여 다항식 $P(x) + 4x$를 다항식 $Q(x)$로 나눈 나머지가 $5x + a$일 때, 상수 $a$의 값은?

## 보기
① $5$   ② $6$   ③ $7$   ④ $8$   ⑤ $9$

## 풀이
$P(x) + 4x = (3x^3 + x + 11) + 4x = 3x^3 + 5x + 11$이다.
이 식을 $Q(x) = x^2 - x + 1$로 나누면 다음과 같다.
$3x^3 + 5x + 11 = (x^2 - x + 1)(3x + 3) + 5x + 8$
따라서 나머지가 $5x + 8$이므로 상수 $a$의 값은 $8$이다.
