## 문제
다음 부등식의 해는?
$$(\log_{2} x)^{2} + \log_{2} x - 2 \geq 0$$

## 보기
① $\frac{1}{4} \leq x \leq 2$   ② $\frac{1}{2} \leq x \leq 4$   ③ $2 \leq x \leq 4$   ④ $0 < x \leq \frac{1}{4}$ 또는 $x \geq 2$   ⑤ $0 < x \leq \frac{1}{2}$ 또는 $x \geq 4$

## 풀이
진수는 양수이므로 
$$x > 0 \cdots \cdots ㉠$$ 

$$(\log_{2} x)^{2} + \log_{2} x - 2 \geq 0 \text{ 에서}$$ 

$$(\log_{2} x + 2)(\log_{2} x - 1) \geq 0$$ 
$$\Leftrightarrow \log_{2} x \leq -2 \text{ 또는 } \log_{2} x \geq 1$$ 

$$\log_{2} x \leq \log_{2} \frac{1}{4} \text{ 또는 } \log_{2} x \geq \log_{2} 2$$ 

$\therefore x \leq \frac{1}{4}$ 또는 $x \geq 2 \cdots \cdots ㉡$ 

$㉠$, $㉡$에서 
$0 < x \leq \frac{1}{4}$ 또는 $x \geq 2$
