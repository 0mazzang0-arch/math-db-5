## 문제
두 자리의 자연수 $N$ 에 대하여 $\log N$ 의 소수부분이 $\alpha$ 일 때,
$$\frac{1}{2} + \log N = \alpha + \log_{4} \frac{N}{8}$$
을 만족시키는 $N$ 의 값을 구하시오.

## 보기

## 풀이
$N$ 은 두 자리의 자연수이므로
$$\log N = 1 + \alpha$$
주어진 식 $\frac{1}{2} + \log N = \alpha + \log_{4} \frac{N}{8}$ 에 대입하면
$$\frac{1}{2} + 1 + \alpha = \alpha + \frac{1}{2} \left( \frac{1 + \alpha}{\log 2} - 3 \right)$$
$$\frac{3}{2} = \frac{1}{2} \left( \frac{1 + \alpha}{\log 2} - 3 \right)$$
$$3 = \frac{1 + \alpha}{\log 2} - 3$$
$$6 = \frac{1 + \alpha}{\log 2}$$
$$\log N = 1 + \alpha = 6 \log 2 = \log 2^{6} = \log 64$$
따라서 $N = 64$ 이다.

정답: $64$
