## 문제
등비수열 $\{ a_{n} \}$이 $2 ( a_{1} + a_{4} + a_{7} ) = a_{4} + a_{7} + a_{10} = 6$을 만족시킬 때, $a_{10}$의 값은?

## 보기
① $\frac{22}{7}$   ② $\frac{24}{7}$   ③ $\frac{26}{7}$   ④ $\frac{30}{7}$   ⑤ $\frac{32}{7}$

## 풀이
수열 $\{ a_{n} \}$이 등비수열이므로 
$a_{4} = a_{1} r^{3}$ 
$a_{7} = a_{4} r^{3}$ 
$a_{10} = a_{7} r^{3}$ 

따라서,
$a_{4} + a_{7} + a_{10} = a_{1} r^{3} + a_{4} r^{3} + a_{7} r^{3} = r^{3} ( a_{1} + a_{4} + a_{7} )$ 

$2 ( a_{1} + a_{4} + a_{7} ) = a_{4} + a_{7} + a_{10}$이므로 
$r^{3} = 2$ 

$r^{3} = 2$이므로 
$a_{4} = a_{1} r^{3} = 2 a_{1}$ 
$a_{7} = a_{4} r^{3} = 4 a_{1}$ 
$a_{10} = a_{7} r^{3} = 8 a_{1}$ 

$a_{4} + a_{7} + a_{10} = 6$에서 
$2 a_{1} + 4 a_{1} + 8 a_{1} = 6$, $14 a_{1} = 6$ 

따라서 $a_{1} = \frac{6}{14} = \frac{3}{7}$ 

$a_{10} = a_{1} r^{9} = a_{1} \times ( r^{3} )^{3}$이므로 
$a_{10} = \frac{3}{7} \times 2^{3} = \frac{24}{7}$

정답: ②
