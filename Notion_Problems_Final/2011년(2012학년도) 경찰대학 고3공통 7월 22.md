## 문제
각 자릿수의 계승의 합이 자신과 같은 수의 집합을 $M$이라 할 때, <보기>에서 옳은 것만을 있는 대로 고른 것은? (예를 들어, $1!+2!+3!=9$는 $123$과 다르므로 $123$은 $M$의 원소가 아니다.)

## 보기
ㄱ. 두 자리의 자연수는 $M$의 원소가 될 수 없다.
ㄴ. $M$의 원소인 세 자리 자연수의 각 자릿수는 $7$보다 작다.
ㄷ. $M$에는 $8$자리 이상의 자연수가 존재하지 않는다.

① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄱ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
[정답] ⑤

$0! = 1! = 1$, 
$2! = 2$, 
$3! = 6$, 
$4! = 24$, 
$5! = 120$, $6! = 720$, 
$7! = 5040$, 
$8! = 40320$, 
$9! = 362880$ 

ㄱ. $5! = 120 > 99$이므로 두 자리 자연수가 $M$의 원소가 되려면 각 자릿수는 $4$ 이하여야 한다. 이때, $3$ 이하의 숫자 조합으로 이루어진 두 자리 자연수의 경우 각 자릿수의 계승의 합이 $2$자리인 경우는 $33$인데, $3!+3! = 12 \neq 33$이므로 $M$의 원소가 되지 못한다. 
$4$를 하나 이상 포함한 두 자리의 자연수의 경우 
$0!+4! = 25$, 
$1!+4! = 25$, 
$2!+4! = 26$, 
$3!+4! = 30$, 
$4!+4! = 48$ 
이므로 모두 $M$의 원소가 되지 못한다. 
따라서 두 자리의 자연수는 $M$의 원소가 될 수 없다. (참)

ㄴ. $7! = 5040 > 1000$이므로 $M$의 원소인 세 자리 자연수의 각 자릿수는 $7$보다 작아야 한다. (참) 

ㄷ. $9! = 362880 < 10^{6}$이므로 $8 \times 9! < 8 \times 10^{6} < 10^{7}$이다. 
$9! \times x$와 $10^{x-1}$의 그래프의 개형을 생각할 때, $x=8$일 때 $9! \times x < 10^{x-1}$를 만족하므로 결국 $8$ 이상의 모든 자연수 $x$에 대하여 $9! \times x < 10^{x-1}$이 성립한다. 
자릿수가 $8$ 이상이면 각 자릿수가 모두 가장 큰 수인 $9$라 가정하더라도 각 자릿수의 계승의 합이 해당 자릿수의 가장 작은 수($10^{x-1}$)에 미치지 못한다. 
따라서 $M$에는 $8$자리 이상의 자연수가 존재하지 않는다. (참) 

따라서 ㄱ, ㄴ, ㄷ 모두 옳다.
