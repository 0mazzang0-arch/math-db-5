## 문제
좌표평면 위에 $5$개의 점 $P_{1}(-2, 1)$, $P_{2}(-1, 2)$, $P_{3}(0, 3)$, $P_{4}(1, 2)$, $P_{5}(2, 4)$가 있다. 점 $P_{i}$ $(i=1, 2, 3, 4, 5)$의 $x$좌표를 $x_{i}$, $y$좌표를 $y_{i}$라 할 때, $\sum_{i=1}^{5} (ax_{i} + b - y_{i})^{2}$의 값이 최소가 되도록 하는 두 실수 $a$, $b$에 대하여 $a+b$의 값을 구하시오.

## 풀이
$P_{1}(-2, 1)$, $P_{2}(-1, 2)$, $P_{3}(0, 3)$, $P_{4}(1, 2)$, $P_{5}(2, 4)$를 주어진 식에 대입하면
$\sum_{i=1}^{5} (ax_{i} + b - y_{i})^{2}$
$= (-2a + b - 1)^{2} + (-a + b - 2)^{2} + (b - 3)^{2} + (a + b - 2)^{2} + (2a + b - 4)^{2}$
$= 10a^{2} + 5b^{2} - 12a - 24b + 34$
$= 10(a - \frac{3}{5})^{2} + 5(b - \frac{12}{5})^{2} + \frac{8}{5}$

따라서 $a = \frac{3}{5}$, $b = \frac{12}{5}$일 때 최솟값을 갖는다.
$\therefore a + b = \frac{3}{5} + \frac{12}{5} = 3$
