## 문제
수열 $\{ a_{n} \}$은 $a_{1}=12$이고,
$\frac{a_{n+1}}{n} = \frac{2 a_{n}}{n+1} + \frac{2^{n+1}}{n+1} \ (n \geq 1)$
을 만족시킨다. 다음은 일반항 $a_{n}$을 구하는 과정이다.

주어진 식에 의하여
$(n+1) a_{n+1}=2 n a_{n}+n \cdot 2^{n+1}$
이다. $b_{n}=\frac{n}{2^{n}} a_{n}$이라 하면
$b_{n+1} = b_{n} + (가) \ (n \geq 1)$
이고 $b_{1} = (나)$이므로
$b_{n} = (다) \ (n \geq 1)$
이다. 그러므로
$a_{n} = \frac{2^{n}}{n} \times (다) \ (n \geq 1)$
이다.

위의 $(가)$, $(다)$에 알맞은 식을 각각 $f(n)$, $g(n)$이라 하고, $(나)$에 알맞은 수를 $p$라 할 때, $f(p)+g(p)$의 값은?

## 보기
① $15$   ② $18$   ③ $21$   ④ $24$   ⑤ $27$

## 풀이
주어진 식 $(n+1) a_{n+1}=2 n a_{n}+n \cdot 2^{n+1}$의 양변을 $2^{n+1}$로 나누면
$\frac{(n+1) a_{n+1}}{2^{n+1}} = \frac{n a_{n}}{2^{n}} + n$
이때 $b_{n} = \frac{n}{2^{n}} a_{n}$이라 하면
$b_{n+1} = b_{n} + n \ (n \geq 1)$
이므로 $f(n) = n$이다.

또한 $b_{1} = \frac{1}{2^{1}} a_{1} = \frac{12}{2} = 6$이므로 $p = 6$이다.

$n \geq 2$일 때,
$b_{n} = b_{1} + \sum_{k=1}^{n-1} k = 6 + \frac{n(n-1)}{2} = \frac{n^{2}-n+12}{2}$
이 식은 $n=1$일 때도 $b_{1} = \frac{1-1+12}{2} = 6$이 되어 성립한다.
따라서 $g(n) = \frac{n^{2}-n+12}{2}$이다.

구하고자 하는 값은
$f(p)+g(p) = f(6)+g(6)$
$f(6) = 6$
$g(6) = \frac{6^{2}-6+12}{2} = \frac{42}{2} = 21$
따라서 $f(6)+g(6) = 6+21 = 27$이다.
