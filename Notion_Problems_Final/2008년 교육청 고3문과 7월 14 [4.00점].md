## 문제
다음은 $n \geq 2$ 인 모든 자연수 $n$ 에 대하여 
$$(1+\frac{1}{1^{3}} ) (1+\frac{1}{2^{3}} ) (1+\frac{1}{3^{3}} ) \cdots (1+\frac{1}{n^{3}} ) < 3 - \frac{1}{n} \cdots \text{㉠}$$
이 성립함을 수학적귀납법으로 증명하는 과정이다.

$$ \begin{array}{|l|} \hline \\ (i) \ n=2 \text{ 일 때} \\ (\text{좌변}) = (1+\frac{1}{1^{3}} ) (1+\frac{1}{2^{3}} ) = \frac{9}{4} , \\ (\text{우변}) = 3 - \frac{1}{2} = \frac{5}{2} \\ \text{따라서 } n=2 \text{ 일 때 ㉠이 성립한다.} \\ \\ (ii) \ n=k(k \geq 2) \text{ 일 때 ㉠이 성립한다고 가정하면} \\ (1+\frac{1}{1^{3}} ) (1+\frac{1}{2^{3}} ) (1+\frac{1}{3^{3}} ) \cdots (1+\frac{1}{k^{3}} ) < 3 - \frac{1}{k} \cdots \text{㉡} \\ \text{㉡의 양변에 } (가) \text{ 를 곱하면} \\ (1+\frac{1}{1^{3}} ) (1+\frac{1}{2^{3}} ) (1+\frac{1}{3^{3}} ) \cdots (1+\frac{1}{k^{3}} ) ( (가) ) < (3 - \frac{1}{k} ) ( (가) ) \cdots \text{㉢} \\ \text{㉢의 우변을 정리하면} \\ = 3 - \frac{(나)}{k(k+1)^{3}} \\ \text{이때, } \frac{(나)}{k(k+1)^{3}} - \frac{1}{k+1} \ (다) \ 0 \text{ 이므로} \\ (3 - \frac{1}{k} ) ( (가) ) < 3 - \frac{1}{k+1} \\ \text{따라서 } n=k+1 \text{ 일 때도 ㉠이 성립한다.} \\ \\ \text{그러므로 (i), (ii)에 의하여 } n \geq 2 \text{ 인 모든 자연수 } n \text{ 에 대하여 } \\ \text{주어진 부등식은 성립한다.} \\ \hline \end{array} $$

위의 증명 과정에서 $(가)$, $(나)$, $(다)$에 알맞은 것은?

## 보기
① $(가): 1+\frac{1}{(k+1)^{3}}, (나): k^{3}+3k^{2}+2, (다): <$   ② $(가): 1+\frac{1}{(k+1)^{3}}, (나): k^{3}+3k^{2}+2, (다): >$   ③ $(가): 1+\frac{1}{(k+1)^{3}}, (나): k^{3}-3k^{2}+2, (다): <$   ④ $(가): \frac{1}{(k+1)^{3}}, (나): k^{3}-3k^{2}+2, (다): >$   ⑤ $(가): \frac{1}{(k+1)^{3}}, (나): k^{3}-3k^{2}+2, (다): <$

## 풀이
(i) $n=2$ 일 때 
$(\text{좌변}) = (1+\frac{1}{1^{3}} ) (1+\frac{1}{2^{3}} ) = \frac{9}{4}$
$(\text{우변}) = 3 - \frac{1}{2} = \frac{5}{2}$ 이므로 성립한다.

(ii) $n=k$ 일 때 ㉠이 성립한다고 가정하면 
$(1+\frac{1}{1^{3}} ) (1+\frac{1}{2^{3}} ) \cdots (1+\frac{1}{k^{3}} ) < 3 - \frac{1}{k} \cdots \text{㉡}$
㉡의 양변에 $1+\frac{1}{(k+1)^{3}}$ 를 곱하면 
$(1+\frac{1}{1^{3}} ) (1+\frac{1}{2^{3}} ) \cdots (1+\frac{1}{k^{3}} ) (1+\frac{1}{(k+1)^{3}} ) < (3 - \frac{1}{k} ) (1+\frac{1}{(k+1)^{3}} ) \cdots \text{㉢}$
따라서 $(가) = 1+\frac{1}{(k+1)^{3}}$ 이다.

㉢의 우변을 정리하면 
$(\text{우변}) = (3 - \frac{1}{k} ) (1+\frac{1}{(k+1)^{3}} ) = 3 + \frac{3}{(k+1)^{3}} - \frac{1}{k} - \frac{1}{k(k+1)^{3}}$
$= 3 - ( \frac{1}{k} + \frac{1}{k(k+1)^{3}} - \frac{3}{(k+1)^{3}} ) = 3 - \frac{(k+1)^{3} + 1 - 3k}{k(k+1)^{3}}$
$= 3 - \frac{k^{3}+3k^{2}+3k+1+1-3k}{k(k+1)^{3}} = 3 - \frac{k^{3}+3k^{2}+2}{k(k+1)^{3}}$
따라서 $(나) = k^{3}+3k^{2}+2$ 이다.

이때, 
$\frac{k^{3}+3k^{2}+2}{k(k+1)^{3}} - \frac{1}{k+1} = \frac{k^{3}+3k^{2}+2 - k(k+1)^{2}}{k(k+1)^{3}}$
$= \frac{k^{3}+3k^{2}+2 - k(k^{2}+2k+1)}{k(k+1)^{3}} = \frac{k^{2}-k+2}{k(k+1)^{3}} > 0 \ (\because k \geq 2)$
따라서 $(다) = >$ 이다.

그러므로 $(3 - \frac{1}{k} ) (1+\frac{1}{(k+1)^{3}} ) < 3 - \frac{1}{k+1}$ 이 되어 $n=k+1$ 일 때도 ㉠이 성립한다.
정답은 ② 이다.
