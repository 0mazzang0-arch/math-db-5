## 문제
닫힌구간 $[0, 1]$ 에서 $0 < f(x) < 1$ 를 만족시키는 다항함수 $f(x)$ 에 대하여 옳은 것만을 <보기>에서 있는 대로 고른 것은?

$$ \begin{array}{|l|} \hline \text{<보기>} \\ \text{ㄱ. } f(a) = a \text{ 인 실수 } a \text{ 가 열린구간 } (0, 1) \text{ 에 적어도 하나 존재한다.} \\ \text{ㄴ. } f'(b) < 1 \text{ 인 실수 } b \text{ 가 열린구간 } (0, 1) \text{ 에 적어도 하나 존재한다.} \\ \text{ㄷ. 열린구간 } (0, 1) \text{ 의 모든 } x \text{ 에 대하여 } \int_{0}^{x} f(t) dt < x \text{ 이다.} \\ \hline \end{array} $$

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
ㄱ. $g(x) = f(x) - x$ 라 하면 $f(x)$ 가 다항함수이므로 $g(x)$ 도 다항함수이고 모든 실수에서 연속이다.
$g(0) = f(0)$ 이므로 $0 < g(0) < 1$ 을 만족한다.
$g(1) = f(1) - 1$ 이고 $0 < f(1) < 1$ 이므로 $-1 < g(1) < 0$ 을 만족한다.
따라서 중간값 정리에 의해 방정식 $g(x) = f(x) - x = 0$ 의 실근이 열린구간 $(0, 1)$ 에 적어도 하나 존재한다. (참)

ㄴ. $g(x)$ 는 닫힌구간 $[0, 1]$ 에서 연속이고, 열린구간 $(0, 1)$ 에서 미분가능이므로 평균값 정리에 의해
$$ \frac{g(1) - g(0)}{1 - 0} = g'(b) = f'(b) - 1 $$
(단, $0 < b < 1$)을 만족하는 $b$ 가 적어도 하나 존재한다.
여기서 $g(1) < 0$ 이고 $g(0) > 0$ 이므로 $\frac{g(1) - g(0)}{1 - 0} < 0$ 이다.
따라서 $f'(b) - 1 < 0$ 이므로 $f'(b) < 1$ 을 만족하는 실수 $b$ 가 열린구간 $(0, 1)$ 에 적어도 하나 존재한다. (참)

ㄷ. $h(x) = \int_{0}^{x} f(t) dt - x$ 라 놓으면
$h'(x) = f(x) - 1$ 이다.
$0 < f(x) < 1$ 이므로 $-1 < f(x) - 1 < 0$ 이다.
즉, 열린구간 $(0, 1)$ 에서 $h'(x)$ 는 항상 음의 값이므로 함수 $h(x)$ 는 단조감소함수이다.
이때 $h(0) = 0$ 이므로 $0 < x < 1$ 인 모든 $x$ 에 대하여 $h(x) < h(0) = 0$ 이다.
$$ h(x) = \int_{0}^{x} f(t) dt - x < 0 $$
$$ \therefore \int_{0}^{x} f(t) dt < x $$
(단, $0 < x < 1$) (참)

정답: ⑤
