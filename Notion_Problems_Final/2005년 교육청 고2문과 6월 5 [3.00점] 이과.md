## 문제
양수 $A$에 대하여 $\log_{10} A = n + \alpha$ ($n$은 정수, $\frac{1}{2} < \alpha < 1$)일 때, $\left[ \log_{10} \frac{1}{A^{2}} \right]$의 값은? (단, $[x]$는 $x$를 넘지 않는 최대정수)

## 보기
① $-2n-2$   ② $-2n-1$   ③ $-2n$   ④ $-2n+1$   ⑤ $-2n+2$

## 풀이
$\log_{10} A = n + \alpha$ ($n$은 정수, $\frac{1}{2} < \alpha < 1$)라 하면
$\log_{10} \frac{1}{A^{2}} = -2 \log_{10} A = -2(n + \alpha) = -2n - 2\alpha$
이때 조건 $\frac{1}{2} < \alpha < 1$ 로부터
$-2 < -2\alpha < -1$
이다. 따라서 정수 부분을 구하기 위해 식을 변형하면
$\log_{10} \frac{1}{A^{2}} = (-2n - 2) + (2 - 2\alpha)$
여기서 $0 < 2 - 2\alpha < 1$ 이므로 $\log_{10} \frac{1}{A^{2}}$의 정수 부분은 $-2n - 2$이다.
따라서 $\left[ \log_{10} \frac{1}{A^{2}} \right] = -2n - 2$

정답: ①
