## 문제

함수 $f(x) = x|x| + px + q$ ($p$, $q$는 실수)에 대하여 옳은 내용만을 <보기>에서 있는 대로 고른 것은?

$$ \begin{array}{|l|} \hline \text{ㄱ. } f(-x) = -f(x)\text{는 } q = 0 \text{이기 위한 필요충분조건이다.} \\ \text{ㄴ. } p = 0 \text{이면 방정식 } f(x) = 0 \text{의 실근은 } 1 \text{개이다.} \\ \text{ㄷ. } 방정식 } f(x) = 0 \text{의 실근은 많아야 } 3 \text{개이다.} \\ \hline \end{array} $$

## 보기

① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이

[정답] ⑤ 

ㄱ. $f(x) = x|x| + px + q$에 대하여 
$f(-x) = -f(x)$
$\iff -x|-x| - p(-x) + q = -(x|x| + px + q)$
$\iff -x|x| - px + q = -(x|x| + px + q)$
$\iff q = 0$
$f(-x) = -f(x)$는 $q = 0$이기 위한 필요충분조건이다.

ㄴ. $p = 0$이면 방정식 $f(x) = 0$에서 $x|x| + q = 0$
$\therefore x|x| = -q$
$y = x|x| = \begin{cases} -x^{2} & (x < 0) \\ x^{2} & (x \geq 0) \end{cases}$ 이므로 
$y = x|x|$의 그래프와 직선 $y = -q$는 항상 한 점에서 만나므로 주어진 방정식의 실근은 $1$개다. 

ㄷ. 방정식 $f(x) = 0$에서 $x|x| + px + q = 0$
$\therefore x|x| = -px - q$
$y = x|x|$의 그래프와 직선 $y = -px - q$의 교점은 최대 $3$개이므로 주어진 방정식의 실근은 많아야 $3$개다. 

이상에서 ㄱ, ㄴ, ㄷ 모두 옳다.
