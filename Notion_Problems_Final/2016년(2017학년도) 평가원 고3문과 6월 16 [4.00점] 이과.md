## 문제
실수 $x$ 에 대한 세 조건
$p: |x| > 4$,
$q: x^{2} - 9 \leq 0$,
$r: x \leq 3$
에 대하여 <보기>에서 참인 명제만을 있는 대로 고른 것은?

$$ \begin{array}{|l|} \hline \text{ㄱ. } q \rightarrow r \\ \text{ㄴ. } p \rightarrow \sim q \\ \text{ㄷ. } r \rightarrow \sim p \\ \hline \end{array} $$

## 보기
① $ㄱ$   ② $ㄱ, ㄴ$   ③ $ㄱ, ㄷ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
세 조건 $p, q, r$ 의 진리집합을 각각 $P, Q, R$ 라 하면 

$P = \{ x | x < -4 \text{ 또는 } x > 4 \}$ 

$Q = \{ x | -3 \leq x \leq 3 \}$ 

$R = \{ x | x \leq 3 \}$ 

ㄱ. $Q \subset R$ 이므로 $q \rightarrow r$ 는 참이다. 

ㄴ. $Q^{c} = \{ x | x < -3 \text{ 또는 } x > 3 \}$ 
$P \subset Q^{c}$ 이므로 $p \rightarrow \sim q$ 는 참이다. 

ㄷ. $P^{c} = \{ x | -4 \leq x \leq 4 \}$ 
$R \not\subset P^{c}$ 이므로 $r \rightarrow \sim p$ 는 거짓이다. 

이상에서 참인 명제는 $ㄱ, ㄴ$이다. 

정답: ②
