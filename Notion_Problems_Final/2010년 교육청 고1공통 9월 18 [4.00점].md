## 문제
계수가 실수인 $x$에 대한 두 이차방정식 $ax^{2} + bx + c = 0$, $ax^{2} + 2bx + c = 0$의 근에 대한 설명으로 옳은 것만을 <보기>에서 있는 대로 고른 것은?

ㄱ. 두 이차방정식에서 각각의 두 근의 곱은 서로 같다. 
ㄴ. $ac > 0$ 이면 두 이차방정식은 실수인 공통근을 갖지 않는다. 
ㄷ. $ax^{2} + 2bx + c = 0$ 이 허근을 가지면 $ax^{2} + bx + c = 0$ 도 허근을 가진다. 

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
ㄱ. 이차방정식 $ax^{2} + bx + c = 0$의 두 근을 $\alpha$, $\beta$, $ax^{2} + 2bx + c = 0$의 두 근을 $\alpha'$, $\beta'$라 하면 
$\alpha \beta = \frac{c}{a}$, $\alpha' \beta' = \frac{c}{a}$ 
$\alpha \beta = \alpha' \beta'$ 

ㄴ. 두 이차방정식 $ax^{2} + bx + c = 0$, $ax^{2} + 2bx + c = 0$의 공통인 실근을 $\alpha$라 하면 
$a \alpha^{2} + b \alpha + c = 0$ $\cdots$ ㉠ 
$a \alpha^{2} + 2b \alpha + c = 0$ $\cdots$ ㉡ 
㉡ $-$ ㉠을 하면 $b \alpha = 0$ 
$\therefore$ $b = 0$ 또는 $\alpha = 0$ 
$ac > 0$에서 $\alpha \neq 0$이므로 $b = 0$ 
$a \alpha^{2} + c = 0$ 
$\therefore$ $\alpha^{2} = - \frac{c}{a} < 0$ ($\because ac > 0$) 
$\alpha$는 순허수이다. 그런데 $\alpha$는 실수이어야 하므로 주어진 조건을 만족시키지 않는다. 

ㄷ. $ax^{2} + 2bx + c = 0$의 판별식을 $D_{1}$이라 하면 
$\frac{D_{1}}{4} = b^{2} - ac < 0$ $\cdots$ ㉠ 
$ax^{2} + bx + c = 0$의 판별식을 $D_{2}$라 하면 
$D_{2} = b^{2} - 4ac$ 
$a$, $b$, $c$에 대하여 
$D_{2} = b^{2} - 4ac \leq 4b^{2} - 4ac < 0$ 
$ax^{2} + bx + c = 0$도 허근을 갖는다. 

이상에서 $ㄱ$, $ㄴ$, $ㄷ$ 모두 옳다.
