## 문제
자연수 $n$에 대하여 $\sqrt{n}$이 무리수일 때, $\sqrt{n} = a+b$ ($a$는 자연수, $0 < b < 1$)라 하자. 다음은 $a^{3} - 9ab + b^{3} = 0$을 만족시키는 자연수 $n$의 값을 구하는 과정이다.

$\sqrt{n} = a+b$ ($a$는 자연수, $0 < b < 1$)이므로 $b = \sqrt{n} - a$
$a^{3} - 9ab + b^{3} = a^{3} - 9a(\sqrt{n} - a) + (\sqrt{n} - a)^{3}$
$= (3a^{2} - 9a + n)\sqrt{n} + 9a^{2} + ((가)) \times a = 0$
무리수가 서로 같을 조건에 의하여
$3a^{2} - 9a + n = 0$ $\cdots \cdots$ ㉠
$9a^{2} + ((가)) \times a = 0$ $\cdots \cdots$ ㉡
㉠, ㉡에서 $n$을 소거하면
$9a^{3} + ((나)) = 0$
$\therefore a = 2$
따라서 $n = 6$

위의 (가), (나)에 알맞은 식을 각각 $f(n)$, $g(a)$라 할 때, $\frac{g(2)}{f(2)}$의 값은?

## 보기
① $12$   ② $13$   ③ $14$   ④ $15$   ⑤ $16$

## 풀이
$\sqrt{n} = a+b$ ($a$는 자연수, $0 < b < 1$)이므로 $b = \sqrt{n} - a$이다.
이를 $a^{3} - 9ab + b^{3}$에 대입하여 전개하면 다음과 같다.
$a^{3} - 9a(\sqrt{n} - a) + (\sqrt{n} - a)^{3}$
$= a^{3} - 9a\sqrt{n} + 9a^{2} + (n\sqrt{n} - 3an + 3a^{2}\sqrt{n} - a^{3})$
$= (3a^{2} - 9a + n)\sqrt{n} + 9a^{2} - 3an = 0$
따라서 $f(n) = -3n$이다.

무리수가 서로 같을 조건에 의하여
$3a^{2} - 9a + n = 0$ $\cdots \cdots$ ㉠
$9a^{2} + (-3n) \times a = 0$ $\cdots \cdots$ ㉡

㉠에서 $n = 9a - 3a^{2}$이므로 이를 ㉡에 대입하면
$9a^{2} - 3a(9a - 3a^{2}) = 0$
$9a^{2} - 27a^{2} + 9a^{3} = 0$
$9a^{3} - 18a^{2} = 0$
따라서 $g(a) = -18a^{2}$이다.

$f(2) = -3 \times 2 = -6$
$g(2) = -18 \times 2^{2} = -72$
그러므로 $\frac{g(2)}{f(2)} = \frac{-72}{-6} = 12$이다.

정답은 ①이다.
