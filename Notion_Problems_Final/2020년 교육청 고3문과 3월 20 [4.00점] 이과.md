## 문제
최고차항의 계수가 $1$ 인 삼차함수 $f(x)$ 에 대하여 함수 $g(x)$ 를
$$g(x)= \int_{0}^{x} f(t) dt + f(x)$$
라 할 때, 함수 $g(x)$ 는 다음 조건을 만족시킨다.

$$
\begin{array}{|l|}
\hline
\text{(가) 함수 } g(x) \text{ 는 } x=0 \text{ 에서 극댓값 } 0 \text{ 을 갖는다.} \\
\text{(나) 함수 } g(x) \text{ 의 도함수 } y=g'(x) \text{ 의 그래프는 원점에 대하여 대칭이다.} \\
\hline
\end{array}
$$

$f(2)$ 의 값은?

## 보기
① $-5$   ② $-4$   ③ $-3$   ④ $-2$   ⑤ $-1$

## 풀이
$g(x)= \int_{0}^{x} f(t) dt + f(x)$ 에서
$g'(x) = f(x) + f'(x)$
$g(0) = \int_{0}^{0} f(t) dt + f(0) = 0 + f(0)$
$g'(0) = f(0) + f'(0)$
조건 (가)에 의해
$g(0) = f(0) = 0$
$g'(0) = f(0) + f'(0) = 0 + f'(0) = 0$ 이므로
$f'(0) = 0$
그러므로 $x^2$ 은 $f(x)$ 의 인수이다.
$f(x) = x^2(x-k)$ ($k$ 는 상수)라 하면
$g'(x) = x^3 - kx^2 + 3x^2 - 2kx$
$= x^3 + (3-k)x^2 - 2kx$
조건 (나)에 의해 모든 실수 $x$ 에 대하여 $g'(-x) = -g'(x)$ 가 성립한다.
즉, $-x^3 + (3-k)x^2 + 2kx = -x^3 - (3-k)x^2 + 2kx$,
$2(3-k)x^2 = 0$ 에서 $k=3$
그러므로 $f(x) = x^2(x-3)$
따라서 $f(2) = 2^2(2-3) = -4$

**[다른 풀이]**
$f(x) = x^3 + ax^2 + bx + c$ 라고 놓으면
$f'(x) = 3x^2 + 2ax + b$
조건 (가)에 의해 $f(0) = 0$ 이므로 $c=0$, $f'(0) = 0$ 이므로 $b=0$
즉, $f(x) = x^3 + ax^2$
$g'(x) = f(x) + f'(x)$
$= x^3 + ax^2 + 3x^2 + 2ax$
$= x^3 + (a+3)x^2 + 2ax$
조건 (나)에 의해 함수 $y=g'(x)$ 의 그래프는 원점에 대하여 대칭이므로 $x^2$ 의 계수는 $0$ 이다. 
즉, $a+3=0$ 에서 $a=-3$
따라서 $f(x) = x^3 - 3x^2$ 에서
$f(2) = 8 - 12 = -4$

정답: ②
