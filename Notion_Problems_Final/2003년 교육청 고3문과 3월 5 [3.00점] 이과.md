## 문제
두 조건 $p : |x - a| \le 2$, $q : |x - 1| \le 3$에 대하여 $p$가 $q$이기 위한 충분조건이 되도록 상수 $a$의 값을 정할 때, $a$의 최댓값은?



> [!example] 📸 (그림/그래프)

## 보기
① $-2$   ② $-1$   ③ $0$   ④ $1$   ⑤ $2$

## 풀이
두 조건 $p$, $q$의 진리집합을 각각 $P$, $Q$라 하면 $p$가 $q$이기 위한 충분조건이 되기 위하여 $P \subset Q$이어야 한다.
이때,
$|x - a| \le 2 \Rightarrow a - 2 \le x \le a + 2$ 이므로
$P = \{ x | a - 2 \le x \le a + 2 \}$
$|x - 1| \le 3 \Rightarrow -3 \le x - 1 \le 3 \Rightarrow -2 \le x \le 4$ 이므로
$Q = \{ x | -2 \le x \le 4 \}$
$P \subset Q$를 만족하려면
$a - 2 \ge -2$ 이고 $a + 2 \le 4$ 이어야 한다.
즉, $a \ge 0$ 이고 $a \le 2$ 이므로
$0 \le a \le 2$
따라서 $a$의 최댓값은 $2$이다.
