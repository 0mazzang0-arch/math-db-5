## 문제
두 실수 $a$, $b$ 가 $a \log _{3} 2 = 4$, $\log _{3} b = 1 - \log _{3} ( \log _{2} 3 )$ 을 만족시킬 때, $ab$ 의 값을 구하시오.

## 보기

## 풀이
$a \log _{3} 2 = 4$ 에서
$a = \frac{4}{\log _{3} 2} = 4 \log _{2} 3$
$\log _{3} b = 1 - \log _{3} ( \log _{2} 3 )$ 에서
$\log _{3} b = \log _{3} 3 - \log _{3} ( \log _{2} 3 ) = \log _{3} \left( \frac{3}{\log _{2} 3} \right)$
$\log _{3} b = \log _{3} ( 3 \log _{3} 2 )$ 이므로
$b = 3 \log _{3} 2$
따라서
$ab = ( 4 \log _{2} 3 ) \times ( 3 \log _{3} 2 ) = 12 ( \log _{2} 3 \times \log _{3} 2 ) = 12 \times 1 = 12$

정답 $12$
