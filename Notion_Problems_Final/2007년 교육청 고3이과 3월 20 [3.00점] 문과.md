## 문제
$x > 0$ 일 때, 자연수 $n$ 에 대하여 함수 $f_{n}(x)$ 가 다음 조건을 만족한다.

(가) $f_{1}(x) = \log_{2} x$
(나) $f_{n+1}(x) = f_{n}(x^{2}) + f_{n}(x)$

$f_{2007}(8) = a$ 라 할 때, $\log_{27} a$ 의 값을 구하시오.

## 보기

## 풀이
$f_{1}(x) = \log_{2} x$
$f_{2}(x) = f_{1}(x^{2}) + f_{1}(x) = \log_{2} x^{2} + \log_{2} x = 2 \log_{2} x + \log_{2} x = 3 \log_{2} x$
$f_{3}(x) = f_{2}(x^{2}) + f_{2}(x) = 3 \log_{2} x^{2} + 3 \log_{2} x = 6 \log_{2} x + 3 \log_{2} x = 3^{2} \log_{2} x$
$f_{4}(x) = f_{3}(x^{2}) + f_{3}(x) = 9 \log_{2} x^{2} + 9 \log_{2} x = 18 \log_{2} x + 9 \log_{2} x = 3^{3} \log_{2} x$
$\vdots$
이상에서 $f_{n}(x) = 3^{n-1} \log_{2} x$ 임을 알 수 있다.

$a = f_{2007}(8) = 3^{2006} \log_{2} 8 = 3^{2006} \times 3 = 3^{2007}$

$\therefore \log_{27} a = \log_{3^{3}} 3^{2007} = \frac{2007}{3} = 669$

정답: $669$
