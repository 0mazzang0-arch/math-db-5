## 문제
실수 전체의 집합에서 미분가능한 함수 $f(x)$에 대하여 함수 $g(x)$를
$$g(x) = \frac{f(x) \cos x}{e^{x}}$$
라 하자. $g'(\pi) = e^{\pi} g(\pi)$일 때, $\frac{f'(\pi)}{f(\pi)}$의 값은? (단, $f(\pi) \neq 0$)

## 보기
① $e^{-2 \pi}$   ② $1$   ③ $e^{-\pi} + 1$   ④ $e^{\pi} + 1$   ⑤ $e^{2 \pi}$

## 풀이
$g(x) = e^{-x} \{ f(x) \cos x \} \cdots$ ㉠
의 양변을 $x$로 미분하면
$g'(x) = -e^{-x} \{ f(x) \cos x \} + e^{-x} \{ f'(x) \cos x - f(x) \sin x \}$
$= e^{-x} \{ -f(x) \cos x + f'(x) \cos x - f(x) \sin x \} \cdots$ ㉡

㉡에 $x = \pi$를 대입하면
$g'(\pi) = e^{-\pi} \{ -f(\pi) \cos \pi + f'(\pi) \cos \pi - f(\pi) \sin \pi \}$

그런데, 조건에서 $g'(\pi) = e^{\pi} g(\pi)$이므로
$e^{\pi} g(\pi) = e^{-\pi} \{ -f(\pi) \cos \pi + f'(\pi) \cos \pi - f(\pi) \sin \pi \}$
$\therefore e^{\pi} g(\pi) = e^{-\pi} \{ f(\pi) - f'(\pi) \} \cdots$ ㉢

㉠에 $x = \pi$를 대입하면
$g(\pi) = \frac{f(\pi) \cos \pi}{e^{\pi}}$
$\therefore g(\pi) = -\frac{f(\pi)}{e^{\pi}} \cdots$ ㉣

㉣을 ㉢에 대입하면
$-f(\pi) = e^{-\pi} (f(\pi) - f'(\pi))$

여기서 $f(\pi) \neq 0$이므로 양변을 $f(\pi)$로 나누면
$-e^{\pi} = 1 - \frac{f'(\pi)}{f(\pi)}$

$\therefore \frac{f'(\pi)}{f(\pi)} = 1 + e^{\pi}$

정답: ④
