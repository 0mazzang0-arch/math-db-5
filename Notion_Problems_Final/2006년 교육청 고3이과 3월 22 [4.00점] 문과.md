## 문제
양수 $x$ 에 대하여 $\langle x \rangle$ 는 $x$ 보다 크거나 같은 최소의 정수를 나타내기로 한다. 예를 들면, $\langle 2 \rangle = 2$, $\langle 2.5 \rangle = 3$ 이다. 수열 $\{ a_{n} \}$ 을 $a_{1} = 10$, $a_{n} = a_{\langle \frac{n}{2} \rangle} + 1$ ($n = 2, 3, 4, \dots$) 로 정의할 때, $a_{50}$ 의 값을 구하시오.

## 보기
(주관식 문항으로 보기가 없습니다.)

## 풀이
정의에 따라 $a_{50}$ 의 값을 역추적하면 다음과 같다.
$a_{50} = a_{\langle \frac{50}{2} \rangle} + 1 = a_{25} + 1$
$a_{25} = a_{\langle \frac{25}{2} \rangle} + 1 = a_{13} + 1$
$a_{13} = a_{\langle \frac{13}{2} \rangle} + 1 = a_{7} + 1$
$a_{7} = a_{\langle \frac{7}{2} \rangle} + 1 = a_{4} + 1$
$a_{4} = a_{\langle \frac{4}{2} \rangle} + 1 = a_{2} + 1$
$a_{2} = a_{\langle \frac{2}{2} \rangle} + 1 = a_{1} + 1$

이를 대입하여 계산하면 다음과 같다.
$a_{50} = a_{25} + 1 = ( a_{13} + 1 ) + 1 = a_{13} + 2$
$= ( a_{7} + 1 ) + 2 = a_{7} + 3$
$= ( a_{4} + 1 ) + 3 = a_{4} + 4$
$= ( a_{2} + 1 ) + 4 = a_{2} + 5$
$= ( a_{1} + 1 ) + 5 = 10 + 6 = 16$

따라서 $a_{50} = 16$ 이다.
