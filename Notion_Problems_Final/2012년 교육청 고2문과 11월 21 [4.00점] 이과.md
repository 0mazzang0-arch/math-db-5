## 문제

$0 < a < 1$인 실수 $a$에 대하여 곡선 $y = \log_{a} x$가 두 점 $(b, d)$, $(c, b)$를 지나고, 직선 $y = x$와 점 $(p, p)$에서 만날 때, <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $0 < b < p < c < 1$)

$$
\begin{array}{|l|}
\hline
\text{ㄱ. } p = \frac{1}{2} \text{이면 } a = \frac{1}{4} \text{이다.} \\
\text{ㄴ. } a^{b+d} = bc \\
\text{ㄷ. } \frac{p-b}{p-a^{c}} < \frac{c-b}{c-a^{c}} \\
\hline
\end{array}
$$



> [!example] 📸 (그림/그래프)

## 보기

① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이

ㄱ. 곡선 $y = \log_{a} x$가 직선 $y = x$와 점 $(p, p)$에서 만나므로 $p = \log_{a} p$이다. $p = \frac{1}{2}$이면 $\frac{1}{2} = \log_{a} \frac{1}{2}$이므로 $a^{\frac{1}{2}} = \frac{1}{2}$이다. 따라서 $a = \frac{1}{4}$이다. (참)

ㄴ. 곡선 $y = \log_{a} x$가 두 점 $(b, d)$, $(c, b)$를 지나므로 $\log_{a} b = d$, $\log_{a} c = b$이다. 즉, $a^{d} = b$, $a^{b} = c$이므로 $a^{b+d} = a^{b} \cdot a^{d} = c \cdot b = bc$이다. (참)

ㄷ. $\frac{p-b}{p-a^{c}}$는 두 점 $(p, p)$, $(a^{c}, b)$를 지나는 직선의 기울기이고, $\frac{c-b}{c-a^{c}}$는 두 점 $(c, c)$, $(a^{c}, b)$를 지나는 직선의 기울기이다. (거짓)

따라서 옳은 것은 ㄱ, ㄴ이다.
