## 문제
자연수 $n$과 두 실수 $x$, $y$에 대하여 $C(n) = x^{2} + y^{2} - n$이라 하자. 네 부등식 $C(1) \leq 0$, $C(1)C(2) \leq 0$, $C(1)C(2)C(3) \leq 0$, $C(1)C(2)C(3)C(4) \leq 0$의 영역을 같은 좌표평면 위에 나타낸 것을 각각 $D_{1}$, $D_{2}$, $D_{3}$, $D_{4}$라 할 때, <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, 경계선은 영역에 포함된다.)

---
**<보기>**
ㄱ. $D_{1}$의 넓이는 $\pi$이다.
ㄴ. $D_{2}$는 $D_{4}$에 포함된다.
ㄷ. $D_{3}$과 $D_{4}$의 넓이는 같다.
---



> [!example] 📸 (그림/그래프)

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
[정답] ⑤

각 부등식의 영역을 좌표평면 위에 나타내면 다음과 같다.
$C(n) = x^{2} + y^{2} - n$이라 하고, $x^{2} + y^{2} = r$이라 하면 각 영역의 범위는 다음과 같다.

$$
\begin{array}{l|l}
\text{영역} & \text{범위 } (x^{2} + y^{2}) \\ \hline
D_{1} & 0 \leq x^{2} + y^{2} \leq 1 \\
D_{2} & 1 \leq x^{2} + y^{2} \leq 2 \\
D_{3} & 0 \leq x^{2} + y^{2} \leq 1 \text{ 또는 } 2 \leq x^{2} + y^{2} \leq 3 \\
D_{4} & 1 \leq x^{2} + y^{2} \leq 2 \text{ 또는 } 3 \leq x^{2} + y^{2} \leq 4
\end{array}
$$

ㄱ. $D_{1}$의 넓이는 반지름의 길이가 $1$인 원의 내부이므로 $\pi \times 1^{2} = \pi$이다. (참)

ㄴ. $D_{2}$의 영역은 $1 \leq x^{2} + y^{2} \leq 2$이고, $D_{4}$의 영역은 $1 \leq x^{2} + y^{2} \leq 2$ 또는 $3 \leq x^{2} + y^{2} \leq 4$이다. 따라서 $D_{2} \subset D_{4}$가 성립한다. (참)

ㄷ. 각 영역의 넓이를 구하면 다음과 같다.
$D_{3}$의 넓이: $\pi \times 1^{2} + (\pi \times 3 - \pi \times 2) = \pi + \pi = 2\pi$
$D_{4}$의 넓이: $(\pi \times 2 - \pi \times 1) + (\pi \times 4 - \pi \times 3) = \pi + \pi = 2\pi$
따라서 $D_{3}$과 $D_{4}$의 넓이는 같다. (참)

그러므로 ㄱ, ㄴ, ㄷ 모두 옳다.
