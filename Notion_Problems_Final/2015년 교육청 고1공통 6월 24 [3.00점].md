## 문제
세 실수 $a$, $b$, $c$에 대하여 
$a^{2} + b^{2} + 4c^{2} = 44$, $ab + 2bc + 2ca = 28$일 때, $(a + b + 2c)^{2}$의 값을 구하시오.

## 보기

## 풀이
$a^{2} + b^{2} + 4c^{2} = 44$, $ab + 2bc + 2ca = 28$이므로 
$(a + b + 2c)^{2}$ 
$= a^{2} + b^{2} + (2c)^{2} + 2ab + 2b(2c) + 2(2c)a$ 
$= a^{2} + b^{2} + 4c^{2} + 2(ab + 2bc + 2ca)$ 
$= 44 + 2 \times 28$ 
$= 100$

정답: $100$
