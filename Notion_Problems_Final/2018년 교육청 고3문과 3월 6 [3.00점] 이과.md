## 문제
전체집합 $U = \{ x | x \text{는 } 6 \text{ 이하의 자연수} \}$의 두 부분집합 $A, B$에 대하여 $A = \{ x | x \text{는 } 5 \text{ 이하의 자연수} \}$, $B = \{ x | x \in U, x \text{는 홀수} \}$일 때, 집합 $A \cap B^{C}$의 모든 원소의 합은?

## 보기
① $2$   ② $3$   ③ $4$   ④ $5$   ⑤ $6$

## 풀이
$U = \{ 1, 2, 3, 4, 5, 6 \}$이고 $A = \{ 1, 2, 3, 4, 5 \}$, $B = \{ 1, 3, 5 \}$이므로
$B^{C} = \{ 2, 4, 6 \}$
$A \cap B^{C} = \{ 1, 2, 3, 4, 5 \} \cap \{ 2, 4, 6 \} = \{ 2, 4 \}$
따라서 집합 $A \cap B^{C}$의 모든 원소의 합은 $2 + 4 = 6$

정답: ⑤

[다른 풀이]
$A = \{ 1, 2, 3, 4, 5 \}$, $B = \{ 1, 3, 5 \}$이므로
$A \cap B^{C} = A - B$
$= \{ 1, 2, 3, 4, 5 \} - \{ 1, 3, 5 \}$
$= \{ 2, 4 \}$
따라서 집합 $A \cap B^{C}$의 모든 원소의 합은 $2 + 4 = 6$
