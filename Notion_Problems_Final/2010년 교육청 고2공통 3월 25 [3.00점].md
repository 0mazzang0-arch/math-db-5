## 문제
다항식 $x^{3} - 2x^{2} + ax + b$ 를 $x^{2} - 1$ 로 나눈 나머지가 $3x + 5$ 일 때, 상수 $a$, $b$ 의 곱 $ab$ 의 값을 구하시오.

## 보기


## 풀이
$x^{3} - 2x^{2} + ax + b$ 를 $x^{2} - 1$ 로 나누었을 때의 몫을 $Q(x)$ 라 하면 나머지가 $3x + 5$ 이므로 
$x^{3} - 2x^{2} + ax + b = (x^{2} - 1)Q(x) + 3x + 5$ 
위 식의 양변에 $x = -1$ 을 대입하면 
$-a + b = 5$ 
위 식의 양변에 $x = 1$ 을 대입하면 
$a + b = 9$ 
위 두 식을 연립하면 
$a = 2$, $b = 7$ 

$\therefore ab = 14$

**[다른 풀이]**
$x^{3} - 2x^{2} + ax + b$ 를 $x^{2} - 1$ 로 나누면 몫은 $x - 2$, 나머지는 $(a + 1)x + (b - 2)$ 이다. 따라서 
$(a + 1)x + (b - 2) = 3x + 5$ 
이므로 
$a + 1 = 3$, $b - 2 = 5$ 
$\therefore a = 2$, $b = 7$ 

$\therefore ab = 14$
