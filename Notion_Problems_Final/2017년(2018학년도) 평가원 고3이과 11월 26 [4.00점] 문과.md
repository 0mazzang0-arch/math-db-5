## 문제
확률변수 $X$ 가 평균이 $m$, 표준편차가 $\sigma$ 인 정규분포를 따르고 
$P(X \le 3) = P(3 \le X \le 80) = 0.3$ 일 때, $m + \sigma$ 의 값을 구하시오. (단, $Z$ 가 표준정규분포를 따르는 확률변수일 때, $P(0 \le Z \le 0.25) = 0.1$, $P(0 \le Z \le 0.52) = 0.2$ 로 계산한다.)

## 풀이
확률변수 $X$ 가 정규분포 $N(m, \sigma^2)$ 을 따르므로 확률변수 $Z = \frac{X-m}{\sigma}$ 은 표준정규분포 $N(0, 1)$ 을 따른다. 

$P(X \le 3) = P(Z \le \frac{3-m}{\sigma}) = 0.3$
즉, $0.5 - P(0 \le Z \le \frac{m-3}{\sigma}) = 0.3$ 이므로 
$P(0 \le Z \le \frac{m-3}{\sigma}) = 0.2$ 에서 
$\frac{m-3}{\sigma} = 0.52$
즉, $m = 3 + 0.52\sigma \cdots \cdots$ ㉠ 
이다. 이때 
$P(3 \le X \le 80) = P(\frac{3-m}{\sigma} \le Z \le \frac{80-m}{\sigma})$
$= P(\frac{3-m}{\sigma} \le Z \le 0) + P(0 \le Z \le \frac{80-m}{\sigma})$
$= 0.2 + P(0 \le Z \le \frac{80-m}{\sigma}) = 0.3$
에서 $P(0 \le Z \le \frac{80-m}{\sigma}) = 0.1$ 이므로 
$\frac{80-m}{\sigma} = 0.25$
즉, $m = 80 - 0.25\sigma \cdots \cdots$ ㉡ 
㉠, ㉡에서 $3 + 0.52\sigma = 80 - 0.25\sigma$, 
$0.77\sigma = 77$, 
$\sigma = 100$
따라서 $m = 3 + 0.52 \times 100 = 55$ 이므로 
$m + \sigma = 55 + 100 = 155$

[정답] $155$
