## 문제
두 집합 $A = \{ x | ( \frac{1}{3} )^{x+2} < ( \frac{1}{3} )^{x^{2}} \}$, $B = \{ x | 2^{| x-2 |} \leq 2^{a} \}$ 에 대하여 $A \cap B = A$ 가 성립하도록 하는 실수 $a$ 의 최솟값은?

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
집합 $A$에서 밑이 $0 < \frac{1}{3} < 1$ 이므로 부등호의 방향이 바뀐다.
$A = \{ x | ( \frac{1}{3} )^{x+2} < ( \frac{1}{3} )^{x^{2}} \} = \{ x | x+2 > x^{2} \}$
$x^{2} - x - 2 < 0$
$(x+1)(x-2) < 0$
$A = \{ x | -1 < x < 2 \}$

집합 $B$에서 밑이 $2 > 1$ 이므로 부등호의 방향이 유지된다.
$B = \{ x | 2^{|x-2|} \leq 2^{a} \} = \{ x | |x-2| \leq a \}$
$-a \leq x-2 \leq a$
$2-a \leq x \leq 2+a$
$B = \{ x | 2-a \leq x \leq 2+a \}$

$A \cap B = A$ 이므로 $A \subset B$ 이다.
그러므로 $2-a \leq -1$ 이고 $2+a \geq 2$ 이어야 한다.
$2-a \leq -1$ 에서 $a \geq 3$
$2+a \geq 2$ 에서 $a \geq 0$
두 조건을 동시에 만족하는 실수 $a$ 의 범위는 $a \geq 3$ 이다.
따라서 $a$ 의 최솟값은 $3$ 이다.
