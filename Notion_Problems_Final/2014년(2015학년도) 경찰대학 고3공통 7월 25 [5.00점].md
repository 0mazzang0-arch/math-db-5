## 문제
직선 $l$이 함수 $f(x)=x^4 -2x^2 -2x+3$의 그래프와 서로 다른 두 점에서 접할 때, 직선 $l$과 곡선 $y=f(x)$로 둘러싸인 영역의 넓이가 $A$이다. $30A$의 값을 구하시오.

## 풀이
직선 $l$의 방정식을 $y=g(x)$라 하자. ($g(x)$는 $1$차 이하의 다항함수)
직선 $l$과 곡선 $y=f(x)$가 서로 다른 두 점에서 접하므로 이 두 점의 $x$좌표를 $\alpha$, $\beta$ ($\alpha < \beta$)라 하면 $f(x)-g(x) = (x-\alpha)^2(x-\beta)^2$이 성립한다. 

> [!example] 📸 (그림/그래프)

그런데 $g(x)$는 $1$차 이하의 다항함수이므로 다항식 $f(x)-g(x)$의 $x^3$과 $x^2$ 항의 계수는 $f(x)$와 같다. 
$f(x)-g(x)$의 $x^3$의 계수는 $0$이므로 근과 계수와의 관계에서 방정식 $f(x)-g(x)=0$의 중근을 포함한 모든 근의 합은 $2(\alpha+\beta) = 0$이다. 
$\therefore \beta = -\alpha$

$f(x)-g(x) = (x-\alpha)^2 (x+\alpha)^2$
$= (x^2 - \alpha^2)^2$
$= x^4 -2\alpha^2 x^2 + \alpha^4$

$f(x)-g(x)$의 $x^2$의 계수는 $-2$이므로 
$-2\alpha^2 = -2$
$\therefore \alpha = -1$, $\beta = 1$ ($\because \alpha < \beta$)
$\therefore f(x)-g(x) = x^4 -2x^2 + 1$

$\therefore A = \int_{-1}^{1} \{ f(x)-g(x) \} dx$
$= \int_{-1}^{1} (x^4 - 2x^2 + 1) dx$
$= 2 \left[ \frac{x^5}{5} - \frac{2}{3}x^3 + x \right]_0^1$
$= 2 \left( \frac{1}{5} - \frac{2}{3} + 1 \right) = \frac{16}{15}$

$\therefore 30A = 30 \times \frac{16}{15} = 32$

정답: $32$
