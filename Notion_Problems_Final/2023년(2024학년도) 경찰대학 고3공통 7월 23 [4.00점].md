## 문제
방정식 $3^{x} + 3^{-x} - 2 ( \sqrt{3^{x}} + \sqrt{3^{-x}} ) - |k-2| + 7 = 0$ 이 실근을 갖지 않도록 하는 정수 $k$ 의 개수를 구하시오.

## 풀이
$\sqrt{3^{x}} > 0$, $\sqrt{3^{-x}} > 0$ 이므로 산술평균과 기하평균의 관계에 의하여 
$\sqrt{3^{x}} + \sqrt{3^{-x}} \ge 2 \sqrt{3^{x} \times 3^{-x}} = 2$ 
(등호는 $3^{x} = 3^{-x}$ 일 때 성립한다.) 

$\sqrt{3^{x}} + \sqrt{3^{-x}} = t$ ($t \ge 2$)라 하면 주어진 방정식은 
$t^{2} - 2t + 5 - |k-2| = 0$, 
$t^{2} - 2t + 5 = |k-2|$ 

$f(t) = t^{2} - 2t + 5$ 라 하면 
$f(t) = (t-1)^{2} + 4$ ($t \ge 2$) 
즉 $t \ge 2$ 에서 $f(t) \ge 5$ 이므로 주어진 방정식이 실근을 갖지 않으려면 
$|k-2| < 5$ 가 성립해야 한다. 

$\therefore -5 < k-2 < 5$, 
$-3 < k < 7$ 
따라서 조건을 만족시키는 정수 $k$ 는 $-2, -1, 0, \dots, 6$ 이므로 그 개수는 $9$ 이다.
