## 문제
$f(x) = \begin{cases} kx^2 + 2kx + 2 & (x \geq -2) \\ -3x - 4 & (x < -2) \end{cases}$, $g(x) = -x + a$가 있다. 양의 실수 $a$에 대하여 방정식 $f(x) = g(x)$의 모든 실근의 합을 $h(a)$라 할 때, 함수 $h(a)$가 항상 연속이 되도록 하는 상수 $k$의 최솟값을 $p$라 하자. $120 \times \frac{1}{p^2}$의 값을 구하시오.

[정답] $480$

## 풀이
$x < -2$일 때 모든 양의 실수 $a$에 대하여 방정식 $f(x) = g(x)$의 근을 구하면 $-3x - 4 = -x + a$에서 $x = \frac{-a - 4}{2}$이다.

$x \geq -2$일 때 방정식 $f(x) = g(x)$의 근은 아래와 같이 $3$가지의 경우로 나누어서 구할 수 있다.

(ⅰ) $k > 0$일 때
> [!example] 📸 (그림/그래프)

$x \geq -2$에서 곡선 $y = kx^2 + 2kx + 2$의 그래프와 직선 $y = -x + a$가 만나는 점의 $x$좌표는 다음과 같이 구할 수 있다.
$kx^2 + 2kx + 2 = -x + a$
$kx^2 + (2k + 1)x + 2 - a = 0$
$x = \frac{-(2k + 1) + \sqrt{(2k + 1)^2 - 4k(2 - a)}}{2k}$
$x = \frac{-(2k + 1) + \sqrt{(2k - 1)^2 + 4ak}}{2k}$
$a$에 대하여
$h(a) = \frac{-a - 4}{2} + \frac{-(2k + 1) + \sqrt{4ka + (2k - 1)^2}}{2k}$
$h(a)$는 연속이다.

(ⅱ) $k = 0$일 때
> [!example] 📸 (그림/그래프)

$x \geq -2$에서 직선 $y = 2$의 그래프와 직선 $y = -x + a$가 만나는 점의 $x$좌표는 $-x + a = 2$에서 $x = a - 2$이다.
$a$에 대하여
$h(a) = \frac{-a - 4}{2} + (a - 2) = \frac{a}{2} - 4$
$h(a)$는 연속이다.

(ⅲ) $k < 0$일 때
$x > -2$인 부분에서 곡선 $y = f(x)$와 직선 $y = -x + a$가 접할 때 판별식을 이용하여 $a$의 값을 $k$에 관하여 나타내면 다음과 같다.
> [!example] 📸 (그림/그래프)

$kx^2 + 2kx + 2 = -x + a$
$kx^2 + (2k + 1)x + 2 - a = 0$ $\cdots\cdots (\ast)$
$D = (2k + 1)^2 - 4k(2 - a) = 0$에서
$a = 2 - \frac{(2k + 1)^2}{4k} = 1 - k - \frac{1}{4k}$
$y = f(x)$와 직선 $y = -x + a$의 위치 관계는 다음 그림과 같다.

$0 < a \leq 1 - k - \frac{1}{4k}$일 때
> [!example] 📸 (그림/그래프)

$x \geq -2$에서 방정식 $f(x) = g(x)$는 실근을 갖고 $(\ast)$에서 두 실근의 합은 $-\frac{2k + 1}{k}$이다.

$a > 1 - k - \frac{1}{4k}$일 때
> [!example] 📸 (그림/그래프)

$x \geq -2$에서 방정식 $f(x) = g(x)$는 실근을 갖지 않는다.

$a$에 대하여 $h(a)$는 다음과 같다.
$h(a) = \begin{cases} \frac{-a - 4}{2} - \frac{2k + 1}{k} & (0 < a \leq 1 - k - \frac{1}{4k}) \\ \frac{-a - 4}{2} & (a > 1 - k - \frac{1}{4k}) \end{cases}$

$h(a)$가 모든 양의 실수 $a$에 대하여 연속이기 위해서는 함숫값과 극한값이 같아야 하므로 $\frac{2k + 1}{k} = 0$에서 $k = -\frac{1}{2}$

(ⅰ), (ⅱ), (ⅲ)에 의해 모든 양의 실수 $a$에 대하여 $h(a)$가 항상 연속이 되도록 하는 상수 $k$의 값의 범위는 $k = -\frac{1}{2}$ 또는 $k \geq 0$이다. 따라서 최솟값은 $p = -\frac{1}{2}$이므로 $p^2 = \frac{1}{4}$이다.
그러므로 $120 \times \frac{1}{p^2} = 120 \times 4 = 480$이다.
