## 문제
두 집합 $A = \{ x | x < 3 \}$ 와 $B = \{ x | 2^{2x} + 3 \cdot 2^{x+1} - a < 0 \}$ 에 대하여 $A = B$ 가 되도록 하는 실수 $a$ 의 값을 구하시오.

## 보기

## 풀이
$2^{2x} + 3 \cdot 2^{x+1} - a < 0$ 에서 $2^x = t$ ($t > 0$)라 하면 
$t^2 + 6t - a < 0$ 이다. 

집합 $A = \{ x | x < 3 \}$ 에서 $x < 3$ 일 때, $2^x < 2^3 = 8$ 이므로 
$0 < t < 8$ 이다. 

집합 $A = B$ 가 되기 위해서는 $t$ 에 대한 이차부등식 $t^2 + 6t - a < 0$ 의 해가 $0 < t < 8$ 이 되어야 한다. 
$t^2 + 6t - a = (t - 8)(t + 14) < 0$
$t^2 + 6t - a = t^2 + 6t - 112 < 0$

상수항을 비교하면 
$-a = -112$
$\therefore a = 112$

정답: $112$
