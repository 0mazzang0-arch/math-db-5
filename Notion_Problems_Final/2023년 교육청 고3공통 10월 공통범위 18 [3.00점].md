## 문제
두 수열 $\{ a_{n} \}$, $\{ b_{n} \}$에 대하여 
$\sum_{k=1}^{10} ( a_{k} - b_{k} + 2 ) = 50$, $\sum_{k=1}^{10} ( a_{k} - 2b_{k} ) = -10$
일 때, $\sum_{k=1}^{10} ( a_{k} + b_{k} )$의 값을 구하시오.

## 보기

## 풀이
$\sum_{k=1}^{10} ( a_{k} - b_{k} + 2 ) = 50$에서 
$\sum_{k=1}^{10} a_{k} - \sum_{k=1}^{10} b_{k} = 30$ 
$\cdots \cdots$ ㉠ 

$\sum_{k=1}^{10} ( a_{k} - 2b_{k} ) = -10$에서 
$\sum_{k=1}^{10} a_{k} - 2 \sum_{k=1}^{10} b_{k} = -10$ 
$\cdots \cdots$ ㉡ 

㉠, ㉡에서 
$\sum_{k=1}^{10} a_{k} = 70$, $\sum_{k=1}^{10} b_{k} = 40$ 
따라서 
$\sum_{k=1}^{10} ( a_{k} + b_{k} ) = \sum_{k=1}^{10} a_{k} + \sum_{k=1}^{10} b_{k} = 110$
