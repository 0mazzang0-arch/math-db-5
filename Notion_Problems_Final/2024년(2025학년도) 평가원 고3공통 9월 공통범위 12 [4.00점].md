## 문제
수열 $\{ a_{n} \}$ 은 등차수열이고, 수열 $\{ b_{n} \}$ 은 모든 자연수 $n$ 에 대하여 
$$b_{n}= \sum_{k=1}^{n}(-1)^{k+1} a_{k}$$
를 만족시킨다. $b_{2} = -2$, $b_{3} + b_{7} = 0$ 일 때, 수열 $\{ b_{n} \}$ 의 첫째항부터 제 $9$ 항까지의 합은?

## 보기
① $-22$   ② $-20$   ③ $-18$   ④ $-16$   ⑤ $-14$

## 풀이
$b_{2} = -2$ 에서
$a_{1} - a_{2} = -2 \cdots \cdots$ ㉠

$b_{3} + b_{7} = 0$ 에서
$( a_{1} - a_{2} + a_{3} ) + ( a_{1} - a_{2} + a_{3} - a_{4} + a_{5} - a_{6} + a_{7} ) = 0 \cdots \cdots$ ㉡

등차수열 $\{ a_{n} \}$ 의 첫째항을 $a$, 공차를 $d$ 라 하면
㉠에서 $-d = -2$ 이므로 $d = 2$
㉡에서 $( a + d ) + ( a + 3d ) = 0$
$\therefore a = -4$

$b_{n} = \sum_{k=1}^{n} ( -1 )^{k+1} a_{k}$ 에서
$b_{1} = a_{1} = a$
$b_{2} = a_{1} - a_{2} = -d$
$b_{3} = a_{1} - a_{2} + a_{3} = a + d$
$\vdots$
$b_{9} = a_{1} - a_{2} + a_{3} - a_{4} + a_{5} - a_{6} + a_{7} - a_{8} + a_{9} = a + 4d$

$b_{2} + b_{3} = b_{4} + b_{5} = b_{6} + b_{7} = b_{8} + b_{9} = a$ 이므로
$\sum_{k=1}^{9} b_{k} = b_{1} + (b_{2} + b_{3}) + (b_{4} + b_{5}) + (b_{6} + b_{7}) + (b_{8} + b_{9})$
$= a + a + a + a + a$
$= 5a$
$= 5 \times (-4) = -20$

정답 : ②
