## 문제
반지름의 길이가 $4$ 인 원에 내접하는 삼각형 $ABC$ 가 있다. 이 삼각형의 둘레의 길이가 $12$ 일 때, $\sin A + \sin B + \sin ( A + B )$ 의 값은?

## 보기
① $\frac{3}{2}$   ② $\frac{8}{5}$   ③ $\frac{17}{10}$   ④ $\frac{9}{5}$   ⑤ $\frac{19}{10}$

## 풀이
삼각형의 세 각 $A$, $B$, $C$ 에 대응하는 변의 길이를 각각 $a$, $b$, $c$ 라 하자.
$A + B + C = \pi$ 이므로
$\sin ( A + B ) = \sin ( \pi - C ) = \sin C$ 이다.
외접원의 반지름의 길이가 $R = 4$ 이므로 사인법칙에 의해
$\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R = 8$ 이 성립한다.
따라서 $\sin A = \frac{a}{8}$, $\sin B = \frac{b}{8}$, $\sin C = \frac{c}{8}$ 이다.
구하고자 하는 값은
$\sin A + \sin B + \sin ( A + B )$
$= \sin A + \sin B + \sin C$
$= \frac{a}{8} + \frac{b}{8} + \frac{c}{8} = \frac{a + b + c}{8}$
이때 삼각형의 둘레의 길이가 $12$ 이므로 $a + b + c = 12$ 이다.
$= \frac{12}{8} = \frac{3}{2}$

정답: ①
