## 문제
두 양수 $a$, $b$에 대하여 $\log_{ 2 } ab = 8$, $\log_{ 2 } \frac{ a }{ b } = 2$일 때, $\log_{ 2 } ( a+4b )$의 값은?

## 보기
① $3$   ② $4$   ③ $5$   ④ $6$   ⑤ $7$

## 풀이
$ab = 2^{ 8 }$, $\frac{ a }{ b } = 2^{ 2 }$  
$a = 2^{ 2 } b$이므로  
$2^{ 2 } b^{ 2 } = 2^{ 8 }$  
따라서 $b = 2^{ 3 }$이고 $a = 2^{ 5 }$이다.  
$\log_{ 2 } ( a+4b ) = \log_{ 2 } ( 2^{ 5 } + 4 \times 2^{ 3 } ) = \log_{ 2 } ( 32 + 32 ) = \log_{ 2 } 64 = 6$

정답: ④
