## 문제
수열 $\{ a_{n} \}$ 이 $a_{1} = 0$, $a_{n} + a_{n+1} = n$ 을 만족시킨다. 다음은 두 자연수 $m$, $n$ 에 대하여 $\sum_{k=n-m+1}^{n+m} a_{k}$ 의 값을 구하는 과정이다. (단, $m < n$ 이다.)

$$ \begin{array}{l} \sum_{k=n-m+1}^{n+m} a_{k} \\ = a_{n-m+1} + a_{n-m+2} + \cdots + a_{n+m-1} + a_{n+m} \\ = ( n-m+1 ) + ( n-m+3 ) + \cdots + ( n+m-3 ) + (가) \\ = \dfrac{(나) \{ ( n-m+1 ) + (가) \}}{2} \\ = (다) \end{array} $$

위 과정에서 $(가)$, $(나)$, $(다)$에 알맞은 것은?

## 보기
① $(가): n+m-1, (나): m, (다): mn$  ② $(가): n+m-1, (나): m, (다): n^{2}$  ③ $(가): n+m-1, (나): n, (다): n^{2}$  ④ $(가): n+m, (나): m-1, (다): mn$  ⑤ $(가): n+m, (나): n-1, (다): n^{2}$

## 풀이
$a_{n} + a_{n+1} = n$ 이므로 $a_{n+m-1} + a_{n+m} = n+m-1$ 이다.
따라서 $\sum_{k=n-m+1}^{n+m} a_{k}$ 는 다음과 같이 두 항씩 짝을 지어 합을 구할 수 있다.
$$ \begin{array}{l} \sum_{k=n-m+1}^{n+m} a_{k} \\ = ( a_{n-m+1} + a_{n-m+2} ) + ( a_{n-m+3} + a_{n-m+4} ) + \cdots + ( a_{n+m-1} + a_{n+m} ) \\ = ( n-m+1 ) + ( n-m+3 ) + \cdots + ( n+m-1 ) \end{array} $$
이때, 위 식은 첫째항이 $n-m+1$, 공차가 $2$ 인 등차수열의 합이다.
$n+m-1 = ( n-m+1 ) + 2 ( m-1 )$ 이므로 항의 개수는 $m$ 개이다.
따라서 이 등차수열의 합은
$$ \frac{m \{ ( n-m+1 ) + ( n+m-1 ) \}}{2} = \frac{m \cdot 2n}{2} = mn $$
그러므로 $(가)$, $(나)$, $(다)$에 알맞은 식은 차례대로 $n+m-1$, $m$, $mn$ 이다.

정답: ①
