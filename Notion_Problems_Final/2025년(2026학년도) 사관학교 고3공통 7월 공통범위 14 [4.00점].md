## 문제
이차함수 $f(x)$에 대하여 함수 
$$g(x) = \begin{cases} -x+4 & (x \le 0 \text{ 또는 } x \ge 6) \\ f(x) & (0 < x < 6) \end{cases}$$
가 다음 조건을 만족시킬 때, $g(f(2))$의 값은?

(가) 함수 $g(x)$는 실수 전체의 집합에서 연속이다. 
(나) $x$에 대한 방정식 $|g(x)| = k$의 서로 다른 양의 실근의 개수가 $3$이 되도록 하는 양수 $k$의 개수는 $1$이다. 

## 보기
① $4$   ② $\frac{9}{2}$   ③ $5$   ④ $\frac{11}{2}$   ⑤ $6$

## 풀이
(ⅰ) 이차함수 $f(x)$의 최고차항의 계수가 음수일 때

> [!example] 📸 (그림/그래프)

방정식 $|g(x)| = k$의 서로 다른 양의 실근의 개수가 $3$이 되도록 하는 양수 $k$의 개수는 무수히 많거나 존재하지 않는다. 

(ⅱ) 이차함수 $f(x)$의 최고차항의 계수가 양수일 때 

> [!example] 📸 (그림/그래프)

> [!example] 📸 (그림/그래프)

위의 그림과 같이 함수 $y = |g(x)|$의 극댓값이 $4$가 아닌 $2$보다 큰 실수인 경우 방정식 $|g(x)| = k$의 서로 다른 양의 실근의 개수가 $3$이 되도록 하는 양수 $k$의 개수는 $2$개 이상이다. 

> [!example] 📸 (그림/그래프)

따라서 조건을 만족하는 함수 $y = |g(x)|$의 그래프는 위의 그림과 같이 극댓값이 $4$이므로 
$f(x) = ax^2 + bx + c$ (단, $a$, $b$, $c$는 실수) 

$f(0) = 4$에서 $c = 4$ 
$f(6) = -2$에서 
$36a + 6b + c = -2$, $36a + 6b + 4 = -2$ 
$6a + b = -1 \cdots \cdots$ ㉠ 

$f(-\frac{b}{2a}) = -4$에서 
$\frac{b^2}{4a} - \frac{b^2}{2a} + c = -4$, $\frac{b^2}{4a} - \frac{b^2}{2a} + 4 = -4$ 
$b^2 = 32a \cdots \cdots$ ㉡ 

㉠, ㉡을 연립하면 
$(-6a-1)^2 = 32a$, $36a^2 - 20a + 1 = 0$ 
$(18a-1)(2a-1) = 0$, $a = \frac{1}{18}$ 또는 $a = \frac{1}{2}$ 

이때 함수 $f(x)$의 꼭짓점의 $x$좌표가 $0$과 $6$ 사이에 있어야 하므로 
$a = \frac{1}{18}$일 때 $x = -\frac{b}{2a} = \frac{1+6a}{2a} = 12$ 이므로 성립하지 않는다. 
$a = \frac{1}{2}$일 때 $x = -\frac{b}{2a} = \frac{1+6a}{2a} = 4$ 이므로 조건을 만족시킨다. 

$\therefore f(x) = \frac{x^2}{2} - 4x + 4$ 
$\therefore g(f(2)) = g(-2) = 6$
