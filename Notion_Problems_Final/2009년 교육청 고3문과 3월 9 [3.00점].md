## 문제
$\log x = 5.65$, $\log y = -1.35$를 만족시키는 두 양수 $x$, $y$를 
$x = a \times 10^m$ ($m$은 정수, $1 \le a < 10$) 
$y = b \times 10^n$ ($n$은 정수, $1 \le b < 10$) 
으로 나타낼 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?

$$\begin{array}{|l|} \hline
\text{ㄱ. } \frac{x}{y} \text{의 값은 자연수이다.} \\
\text{ㄴ. } m + n = 4 \\
\text{ㄷ. } ab > 10 \\ \hline
\end{array}$$

## 보기
① $ㄱ$   ② $ $   ③ $ $   ④ $ㄱ, ㄷ$   ⑤ $ $

## 풀이
ㄱ. $\log \frac{x}{y} = \log x - \log y = 5.65 - (-1.35) = 7$이므로 
$\frac{x}{y} = 10^7$이다. 
따라서 $\frac{x}{y}$의 값은 자연수이다. (참)

ㄴ. $\log x = \log(a \times 10^m) = m + \log a = 5 + 0.65$에서 
$m = 5$, $\log a = 0.65$이다. 
$\log y = \log(b \times 10^n) = n + \log b = -2 + 0.65$에서 
$n = -2$, $\log b = 0.65$이다. 
$m + n = 5 + (-2) = 3$이다. (거짓)

ㄷ. ㄴ에서 $\log ab = \log a + \log b = 0.65 + 0.65 = 1.3$이므로 
$ab = 10^{1.3} > 10^1$이다. 
즉, $ab > 10$이다. (참)

이상에서 옳은 것은 $ㄱ, ㄷ$이다.
