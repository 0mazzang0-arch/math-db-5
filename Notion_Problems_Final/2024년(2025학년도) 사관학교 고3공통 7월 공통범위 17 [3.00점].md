## 문제
공차가 $0$ 이 아닌 등차수열 $\{a_n\}$ 이 $a_{12} = 5$, $|a_5| = |a_{13}|$ 을 만족시킬 때, $a_{24}$ 의 값을 구하시오.

## 보기


## 풀이
등차수열 $\{a_n\}$ 의 첫째항을 $a$, 공차를 $d$ 라 하면
$a_n = a + (n-1)d$
$a_{12} = 5$ 이므로
$a + 11d = 5$ $\cdots \cdots$ ㉠

$|a_5| = |a_{13}|$ 이고 공차가 $0$ 이 아니므로
$a_5 = -a_{13}$
$a + 4d = -(a + 12d)$
$a = -8d$ $\cdots \cdots$ ㉡

㉡을 ㉠에 대입하면
$-8d + 11d = 5$
$\therefore d = \frac{5}{3}$

$d = \frac{5}{3}$ 를 ㉡에 대입하면
$a = -\frac{40}{3}$
$a_n = -\frac{40}{3} + (n - 1) \times \frac{5}{3}$
$\therefore a_{24} = -\frac{40}{3} + 23 \times \frac{5}{3} = 25$

정답: $25$
