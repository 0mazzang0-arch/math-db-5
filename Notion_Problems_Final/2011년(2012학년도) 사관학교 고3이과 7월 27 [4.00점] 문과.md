## 문제

삼차항의 계수가 $1$인 삼차함수 $f(x)$에 대하여 두 집합 $A$, $B$를 각각
$$A = \left\{ x \mid \frac{(x-2)^{2}}{(x-4)(x-6)} \le 0 \right\}$$
$$B = \{ x \mid (x-6)f(x) \ge 0 \}$$
라 하면 두 집합 $A$와 $B$는 다음 조건을 만족시킨다.

(가) $A \cap B = \{ 2 \} \cup \{ x \mid 5 \le x < 6 \}$
(나) $A^{c} \cap B^{c} = \{ x \mid 3 < x \le 4 \}$

$f(10)$의 값을 구하시오.



> [!example] 📸 (그림/그래프)

## 풀이

집합 $A$에서 
$$\frac{(x-2)^{2}}{(x-4)(x-6)} \le 0$$
$$\Leftrightarrow (x-2)^{2}(x-4)(x-6) \le 0, x \neq 4, x \neq 6$$
$$\Leftrightarrow x=2 \text{ 또는 } 4 < x < 6$$
$$\therefore A = \{ x \mid x=2 \text{ 또는 } 4 < x < 6 \}$$

조건 (가)에서 
$$A \cap B = \{ 2 \} \cup \{ x \mid 5 \le x < 6 \}$$
조건 (나)에서 
$A^{c} \cap B^{c} = \{ x \mid 3 < x \le 4 \}$ 이므로 드모르간의 법칙에 의해
$$(A \cup B)^{c} = \{ x \mid 3 < x \le 4 \}$$
즉, $A \cup B = \{ x \mid x \le 3 \text{ 또는 } x > 4 \}$
집합 $A$와 위 조건들을 종합하면 집합 $B$는
$$B = \{ x \mid x \le 3 \text{ 또는 } x \ge 5 \}$$
이어야 한다.

그런데 $B = \{ x \mid (x-6)f(x) \ge 0 \}$이므로 $f(x)$는 $(x-3)$, $(x-5)$를 인수로 갖는다.
또한, $f(x)$가 $(x-6)$이라는 인수를 갖지 않으면 $x=6$을 기준으로 $(x-6)f(x)$의 부호가 바뀌게 되므로, $B$의 범위가 $x \ge 5$에서 연속적으로 나타나기 위해서는 $f(x)$가 $(x-6)$을 인수로 가져야 한다.
문제에서 $f(x)$의 최고차항의 계수가 $1$이라고 했으므로
$$f(x) = (x-3)(x-5)(x-6)$$
이다.

따라서 
$$\therefore f(10) = (10-3)(10-5)(10-6) = 7 \cdot 5 \cdot 4 = 140$$

정답: $140$
