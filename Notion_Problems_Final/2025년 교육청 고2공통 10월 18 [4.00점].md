## 문제
닫힌구간 $[0, \pi]$ 에서 함수 $f(x) = \sin(x+a)$ 의 최댓값과 최솟값을 각각 $M$, $m$ 이라 할 때, $2|M| = |m|$ 이다. $3\pi$ 보다 작은 모든 양수 $a$ 의 값의 합은?



> [!example] 📸 (그림/그래프)

## 보기
① $\frac{25}{6}\pi$   ② $\frac{13}{3}\pi$   ③ $\frac{9}{2}\pi$   ④ $\frac{14}{3}\pi$   ⑤ $\frac{29}{6}\pi$

## 풀이
닫힌구간 $[0, \pi]$ 에서 함수 $f(x) = \sin(x+a)$ 의 최댓값, 최솟값은 각각 닫힌구간 $[a, a+\pi]$ 에서 함수 $y = \sin x$ 의 최댓값, 최솟값과 같다. 

(ⅰ) $0 < a \le \frac{\pi}{2}$ 일 때
$M = 1$ 에서 $2|M| = 2$ 이고, $m = \sin(a+\pi) = -\sin a$ 에서 $|m| \le 1$ 이므로 $2|M| = |m|$ 을 만족시키지 않는다.

(ⅱ) $\frac{\pi}{2} < a \le \pi$ 일 때
$M = \sin a$, $m = -1$ 이고 $2|M| = |m|$ 이므로
$2\sin a = 1$, $\sin a = \frac{1}{2}$
따라서 $a = \frac{5}{6}\pi$

(ⅲ) $\pi < a \le \frac{3}{2}\pi$ 일 때
$M = \sin(a+\pi) = -\sin a$, $m = -1$ 이고 $2|M| = |m|$ 이므로
$-2\sin a = 1$, $\sin a = -\frac{1}{2}$
따라서 $a = \frac{7}{6}\pi$

(ⅳ) $\frac{3}{2}\pi < a \le 2\pi$ 일 때
$M = 1$ 에서 $2|M| = 2$ 이고, $m = \sin a$ 에서 $|m| \le 1$ 이므로 $2|M| = |m|$ 을 만족시키지 않는다.

(ⅴ) $2\pi < a \le \frac{5}{2}\pi$ 일 때
$M = 1$ 에서 $2|M| = 2$ 이고, $m = \sin(a+\pi) = -\sin a$ 에서 $|m| \le 1$ 이므로 $2|M| = |m|$ 을 만족시키지 않는다.

(ⅵ) $\frac{5}{2}\pi < a \le 3\pi$ 일 때
$M = \sin a$, $m = -1$ 이고 $2|M| = |m|$ 이므로
$2\sin a = 1$, $\sin a = \frac{1}{2}$
따라서 $a = \frac{17}{6}\pi$

(ⅰ) ~ (ⅵ)에 의하여 구하는 모든 양수 $a$ 의 값의 합은
$\frac{5}{6}\pi + \frac{7}{6}\pi + \frac{17}{6}\pi = \frac{29}{6}\pi$
