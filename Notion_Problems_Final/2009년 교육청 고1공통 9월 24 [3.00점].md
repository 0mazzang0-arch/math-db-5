## 문제
집합 $S = \{ 1, 2, 3, 4, 5 \}$에 대하여 집합 $X$를 다음과 같이 정의한다. 
$$ X = \{ ( p, q ) \mid p \in S, q \in S, p \text{ 는 } q \text{ 의 약수} \} $$
이 때, 집합 $X$의 원소의 개수를 구하시오.

## 보기

## 풀이
집합 $S = \{ 1, 2, 3, 4, 5 \}$의 원소 $p, q$에 대하여 $p$가 $q$의 약수인 순서쌍 $( p, q )$를 구하면 다음과 같다.
- $q = 1$일 때: $( 1, 1 )$
- $q = 2$일 때: $( 1, 2 ), ( 2, 2 )$
- $q = 3$일 때: $( 1, 3 ), ( 3, 3 )$
- $q = 4$일 때: $( 1, 4 ), ( 2, 4 ), ( 4, 4 )$
- $q = 5$일 때: $( 1, 5 ), ( 5, 5 )$

따라서 조건에 알맞은 집합 $X$의 원소는
$( 1, 1 ), ( 1, 2 ), ( 2, 2 ), ( 1, 3 ), ( 3, 3 ), ( 1, 4 ), ( 2, 4 ), ( 4, 4 ), ( 1, 5 ), ( 5, 5 )$
이며, 원소의 개수는 $10$이다.

정답: $10$
