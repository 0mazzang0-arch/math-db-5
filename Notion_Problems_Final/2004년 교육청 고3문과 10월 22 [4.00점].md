## 문제
수열 $\{ a_n \}$ 이 $a_1 = 2$, $a_n + a_{n+1} = 3n$ ($n = 1, 2, 3, \dots$) 으로 정의된다. 이때, 두 수 $P = a_1 + a_3 + a_5 + \dots + a_{49}$, $Q = a_2 + a_4 + a_6 + \dots + a_{50}$ 에 대하여 $P - Q$ 의 값을 구하시오.

## 보기

## 풀이
$a_n + a_{n+1} = 3n$ 의 $n$ 에 $1, 2, 3, \dots$ 을 차례로 대입하면
$a_1 + a_2 = 3$ 에서 $2 + a_2 = 3, a_2 = 1$
$a_2 + a_3 = 6$ 에서 $1 + a_3 = 6, a_3 = 5$
$a_3 + a_4 = 9$ 에서 $5 + a_4 = 9, a_4 = 4$
$a_4 + a_5 = 12$ 에서 $4 + a_5 = 12, a_5 = 8$
$a_5 + a_6 = 15$ 에서 $8 + a_6 = 15, a_6 = 7$
$\vdots$
이때 홀수 번째 항과 짝수 번째 항의 일반항은 다음과 같다.
$a_{2n-1} = 2 + (n - 1) \times 3 = 3n - 1$
$a_{2n} = 1 + (n - 1) \times 3 = 3n - 2$

$P - Q$ 의 값을 구하면
$P - Q = (a_1 + a_3 + a_5 + \dots + a_{49}) - (a_2 + a_4 + a_6 + \dots + a_{50})$
$= (a_1 - a_2) + (a_3 - a_4) + \dots + (a_{49} - a_{50})$
위의 일반항에서 $a_{2n-1} - a_{2n} = (3n - 1) - (3n - 2) = 1$ 이므로
$= 1 \times 25$
$= 25$

정답: $25$
