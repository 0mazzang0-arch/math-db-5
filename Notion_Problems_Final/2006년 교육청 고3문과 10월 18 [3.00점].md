## 문제
$\sqrt{\frac{9^{7} + 3^{10}}{9^{4} + 3^{4}}}$ 의 값을 구하시오.

## 풀이
[정답] **$27$**

$\sqrt{\frac{3^{14} + 3^{10}}{3^{8} + 3^{4}}} = \sqrt{\frac{3^{10} ( 3^{4} + 1 )}{3^{4} ( 3^{4} + 1 )}} = 3^{3} = 27$
따라서 $\sqrt{\frac{9^{7} + 3^{10}}{9^{4} + 3^{4}}} = \sqrt{\frac{(3^{2})^{7} + 3^{10}}{(3^{2})^{4} + 3^{4}}} = \sqrt{\frac{3^{14} + 3^{10}}{3^{8} + 3^{4}}} = 27$ 이다.
