## 문제
$\sin \alpha = \frac{3}{5}$, $\sin \beta = \frac{5}{13}$ 일 때, $\tan \left( \frac{\alpha + \beta}{2} \right) = \frac{q}{p}$ 라 하자. 이 때, $p + q$ 의 값을 구하시오.
(단, $0 \le \alpha \le \frac{\pi}{2}$, $0 \le \beta \le \frac{\pi}{2}$ 이고, $p$, $q$ 는 서로소인 자연수이다.)

## 보기

## 풀이
$\cos \alpha = \frac{4}{5}$ 이고,
$\tan^{2} \frac{\alpha}{2} = \frac{\frac{1}{5}}{\frac{9}{5}} = \frac{1}{9}$ 이므로
$\tan \frac{\alpha}{2} = \frac{1}{3}$ $\cdots$ ㉠

$\cos \beta = \frac{12}{13}$ 이고
$\tan^{2} \frac{\beta}{2} = \frac{\frac{1}{13}}{\frac{25}{13}} = \frac{1}{25}$ 이므로
$\tan \frac{\beta}{2} = \frac{1}{5}$ $\cdots$ ㉡

따라서, ㉠, ㉡에 의하여
$\tan \left( \frac{\alpha}{2} + \frac{\beta}{2} \right) = \frac{\frac{1}{3} + \frac{1}{5}}{1 - \frac{1}{15}} = \frac{8}{14} = \frac{4}{7}$ 이므로
$p + q = 11$ 이다.

정답: $11$
