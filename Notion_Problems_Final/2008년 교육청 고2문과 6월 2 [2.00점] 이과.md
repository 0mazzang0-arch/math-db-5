## 문제
전체집합 $U = \{ x \mid x \text{ 는 } 20 \text{ 이하의 자연수 } \}$의 세 부분집합 $A = \{ x \mid x \text{ 는 } 18 \text{ 의 약수 } \}$, $B = \{ x \mid x \text{ 는 } 12 \text{ 의 약수 } \}$, $C = \{ x \mid x \text{ 는 } 6 \text{ 의 약수 } \}$에 대하여 다음 중 옳은 것은?

## 보기
① $A \cup C = U$   ② $A \cap B = C$   ③ $A \cup B^{C} = C$   ④ $A - C = B$   ⑤ $B - C = \emptyset$

## 풀이
전체집합 $U = \{ 1, 2, 3, \dots, 20 \}$에 대하여 각 집합의 원소를 나열하면 다음과 같다.
$A = \{ 1, 2, 3, 6, 9, 18 \}$
$B = \{ 1, 2, 3, 4, 6, 12 \}$
$C = \{ 1, 2, 3, 6 \}$

각 보기를 확인하면 다음과 같다.
① $A \cup C = \{ 1, 2, 3, 6, 9, 18 \} = A$
② $A \cap B = \{ 1, 2, 3, 6 \} = C$ (참)
③ $A \cup B^{C} = U - \{ 4, 12 \}$
④ $A - C = \{ 9, 18 \}$
⑤ $B - C = \{ 4, 12 \}$
따라서 옳은 것은 ②이다.

정답: ②
