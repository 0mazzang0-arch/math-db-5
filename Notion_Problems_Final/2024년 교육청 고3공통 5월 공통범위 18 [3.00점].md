## 문제
최고차항의 계수가 $3$ 인 이차함수 $f(x)$ 가 모든 실수 $x$ 에 대하여 
$$\int_{0}^{x} f(t)dt = 2x^{3} + \int_{0}^{-x} f(t)dt$$
를 만족시킨다. $f(1) = 5$ 일 때, $f(2)$ 의 값을 구하시오.

## 보기

## 풀이
$f(x) = 3x^{2} + ax + b$ 라 하자.
$\int_{0}^{x} f(t)dt = 2x^{3} + \int_{0}^{-x} f(t)dt$ 에서
$2x^{3} = -\int_{0}^{-x} f(t)dt + \int_{0}^{x} f(t)dt$
$= \int_{-x}^{0} f(t)dt + \int_{0}^{x} f(t)dt$
$= \int_{-x}^{x} f(t)dt$
$= [t^{3} + \frac{a}{2} t^{2} + bt]_{-x}^{x}$
$= 2x^{3} + 2bx$
모든 실수 $x$ 에 대하여 $2x^{3} = 2x^{3} + 2bx$ 이므로 $b = 0$ 이고,
$f(1) = 3 + a + b = 5$ 에서 $a = 2$ 이다.
따라서 $f(2) = 12 + 2a + b = 16$ 이다.
