## 문제
수열 $\{ a_{n} \}$ 이 모든 자연수 $n$ 에 대하여 부등식
$3 n^{2} + 2 n < a_{n} < 3 n^{2} + 3 n$
을 만족시킬 때, $\lim_{n \to \infty} \frac{5 a_{n}}{n^{2} + 2 n}$ 의 값을 구하시오.

## 보기

## 풀이
$3 n^{2} + 2 n < a_{n} < 3 n^{2} + 3 n$ 에서 각 변을 $n^{2}$ 으로 나누면
$3 + \frac{2}{n} < \frac{a_{n}}{n^{2}} < 3 + \frac{3}{n}$
이 때, $\lim_{n \to \infty} ( 3 + \frac{2}{n} ) = \lim_{n \to \infty} ( 3 + \frac{3}{n} ) = 3$ 이므로
$\lim_{n \to \infty} \frac{a_{n}}{n^{2}} = 3$
따라서
$\lim_{n \to \infty} \frac{5 a_{n}}{n^{2} + 2 n} = \lim_{n \to \infty} \frac{5 \times \frac{a_{n}}{n^{2}}}{1 + \frac{2}{n}} = 5 \times 3 = 15$

정답: $15$
