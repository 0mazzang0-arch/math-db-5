## 문제

두 다항함수 $f(x)$, $g(x)$에 대하여 $f(x)$의 한 부정적분을 $F(x)$라 하고 $g(x)$의 한 부정적분을 $G(x)$라 할 때, 이 함수들은 모든 실수 $x$에 대하여 다음 조건을 만족시킨다.

$$ \begin{array}{l} \text{(가) } \int_{1}^{x} f(t) dt = xf(x) - 2x^{2} - 1 \\ \text{(나) } f(x)G(x) + F(x)g(x) = 8x^{3} + 3x^{2} + 1 \end{array} $$

$\int_{1}^{3} g(x) dx$의 값을 구하시오.

## 풀이

조건 (가)의 
$$ \int_{1}^{x} f(t) dt = xf(x) - 2x^{2} - 1 \quad \dots \quad ㉠ $$
에서 양변을 미분하면 
$f(x) = f(x) + xf'(x) - 4x$, 
$xf'(x) = 4x$

$x \neq 0$인 모든 실수에 대하여 
$f'(x) = 4$

$f'(x)$는 다항함수이므로 모든 실수에 대하여 
$f'(x) = 4$

$\therefore f(x) = 4x + C_{1}$ ($C_{1}$은 적분상수) 

㉠의 양변에 $x = 1$을 대입하면 
$f(1) = 3$

$\therefore f(x) = 4x - 1$

$F(x) = \int f(x) dx = 2x^{2} - x + a$ ($a$는 적분상수) 

조건 (나)에서 
$f(x)G(x) + F(x)g(x) = F'(x)G(x) + F(x)G'(x) = \{F(x)G(x)\}'$
$\{F(x)G(x)\}' = 8x^{3} + 3x^{2} + 1$
이므로 
$F(x)G(x) = 2x^{4} + x^{3} + x + C_{2}$ ($C_{2}$는 적분상수) 

$F(x)$, $G(x)$가 모두 다항함수이고 $F(x)$가 최고차항의 계수가 $2$인 이차함수이므로 $G(x)$는 최고차항의 계수가 $1$인 이차함수이다. 

$G(x) = x^{2} + bx + c$라 하면 
$(2x^{2} - x + a)(x^{2} + bx + c) = 2x^{4} + x^{3} + x + C_{2}$

양변의 $3$차항의 계수를 비교하면 
$-1 + 2b = 1$에서 $b = 1$

양변의 $2$차항의 계수를 비교하면 
$a - b + 2c = 0$에서 $a = 1 - 2c$

양변의 $1$차항의 계수를 비교하면 
$ab - c = 1$에서 $a = 1 + c$

$\therefore a = 1$, $c = 0$, $G(x) = x^{2} + x$

$\therefore \int_{1}^{3} g(x) dx = G(3) - G(1) = 12 - 2 = 10$

정답: $10$
