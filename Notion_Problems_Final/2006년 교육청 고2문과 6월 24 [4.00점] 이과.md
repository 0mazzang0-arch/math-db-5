## 문제
집합 $A = \left\{ x \mid x = \sqrt[2n]{\frac{2^{11} ( 3^{4} + 3^{2} + 1 )}{3^{6} - 1}}, n \text{과 } x \text{는 양의 정수} \right\}$ 의 모든 원소들의 합을 구하시오.

## 보기


## 풀이
$x = \left\{ \frac{2^{11} ( 3^{4} + 3^{2} + 1 )}{( 3^{2} - 1 ) ( 3^{4} + 3^{2} + 1 )} \right\}^{\frac{1}{2n}}$
$x = \left( \frac{2^{11}}{3^{2} - 1} \right)^{\frac{1}{2n}} = \left( \frac{2^{11}}{8} \right)^{\frac{1}{2n}} = ( 2^{8} )^{\frac{1}{2n}} = 2^{\frac{4}{n}}$
양의 정수 $n$ 에 대하여 $x$ 가 자연수가 되기 위한 $n$ 은 $1$, $2$, $4$ 이다.
$n = 1$ 일 때, $x = 2^{4} = 16$
$n = 2$ 일 때, $x = 2^{2} = 4$
$n = 4$ 일 때, $x = 2^{1} = 2$
따라서, $A = \{ 2, 4, 16 \}$
그러므로 집합 $A$ 의 원소의 합은 $2 + 4 + 16 = 22$ 이다.

정답: $22$
