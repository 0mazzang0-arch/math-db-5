## 문제
$0 < x < 2\pi$ 에서 삼각방정식 
$3\sin x + 3\sin x \cos 2x - 6\sin x \cos x - \cos x + 1 = 0$ 
의 모든 실근의 합은? 

## 보기
① $\frac{5}{2}\pi$   ② $3\pi$   ③ $\frac{7}{2}\pi$   ④ $4\pi$   ⑤ $\frac{9}{2}\pi$

## 풀이
[정답] ② 

$\cos 2x = 2\cos^2 x - 1$ 이므로 
(준식) 
$= 6\sin x \cos^2 x - 6\sin x \cos x - \cos x + 1$ 
$= 6\sin x \cos x (\cos x - 1) - (\cos x - 1)$ 
$= (\cos x - 1) (6\sin x \cos x - 1)$ 
$= (\cos x - 1) (3\sin 2x - 1)$ 
$= 0$ 

$0 < x < 2\pi$ 이므로 
$\cos x \neq 1$ 
$\therefore \sin 2x = \frac{1}{3}$ 

> [!example] 📸 (그림/그래프)

위의 그림에서 
$\sin 2x = \frac{1}{3}$ 를 만족하는 실근은 
$0 < x < 2\pi$ 범위에서 
$4$ 개이고, 이 중 두 근은 
$x = \frac{\pi}{4}$ 에 대칭이고 남은 두 근은 
$x = \pi + \frac{\pi}{4}$ 에 대하여 대칭이므로 구하는 모든 실근의 합은 
$2 \{ \frac{\pi}{4} + (\pi + \frac{\pi}{4}) \} = 3\pi$
