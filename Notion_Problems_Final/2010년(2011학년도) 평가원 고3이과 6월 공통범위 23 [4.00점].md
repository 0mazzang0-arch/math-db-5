## 문제

최고차항의 계수가 $1$이 아닌 다항함수 $f(x)$가 다음 조건을 만족시킬 때, $f'(1)$의 값을 구하시오.

(가) $\lim_{x \to \infty} \frac{\{f(x)\}^2 - f(x^2)}{x^3 f(x)} = 4$
(나) $\lim_{x \to 0} \frac{f'(x)}{x} = 4$

## 풀이

(가)에서 $f(x)$의 최고차항을 $ax^{n}$ ($a \neq 1$)이라 하면 $\{f(x)\}^2 - f(x^2)$의 최고차항은
$a^2 x^{2n} - ax^{2n} = a(a-1)x^{2n}$ $\cdots \cdots$ ㉠
또, $x^3 f(x)$의 최고차항은
$ax^{n+3}$ $\cdots \cdots$ ㉡

$0$이 아닌 극한값이 존재하려면 ㉠과 ㉡에서
$2n = n+3$ $\therefore n = 3$

이때, 극한값이 $4$이므로
$\frac{a(a-1)}{a} = 4$ $\therefore a = 5$

$f(x) = 5x^3 + bx^2 + cx + d$로 놓으면
$f'(x) = 15x^2 + 2bx + c$

(나)에서
$\lim_{x \to 0} \frac{f'(x)}{x} = \lim_{x \to 0} \frac{15x^2 + 2bx + c}{x}$

이때, $x \to 0$일 때, (분모) $\to 0$이고 극한값이 존재하므로 (분자) $\to 0$에서
$\lim_{x \to 0} (15x^2 + 2bx + c) = c = 0$

이 값을 대입하면
$\lim_{x \to 0} \frac{15x^2 + 2bx}{x} = \lim_{x \to 0} (15x + 2b) = 2b = 4$
$\therefore b = 2$

따라서 $f'(x) = 15x^2 + 4x$이므로
$f'(1) = 15 + 4 = 19$
