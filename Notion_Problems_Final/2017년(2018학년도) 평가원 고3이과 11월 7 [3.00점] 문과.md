## 문제
$0 \le x \le 2 \pi$ 일 때, 방정식 $\cos^{2} x = \sin^{2} x - \sin x$ 의 모든 해의 합은?

## 보기
① $2 \pi$   ② $\frac{5}{2} \pi$   ③ $3 \pi$   ④ $\frac{7}{2} \pi$   ⑤ $4 \pi$

## 풀이
$\cos^{2} x = 1 - \sin^{2} x$ 이므로 방정식 $\cos^{2} x = \sin^{2} x - \sin x$ 에서
$1 - \sin^{2} x = \sin^{2} x - \sin x$
$2 \sin^{2} x - \sin x - 1 = 0$
$( 2 \sin x + 1 ) ( \sin x - 1 ) = 0$
$\sin x = - \frac{1}{2}$ 또는 $\sin x = 1$
이때, $0 \le x < 2 \pi$ 이므로
(ⅰ) $\sin x = - \frac{1}{2}$ 에서 $x = \frac{7}{6} \pi$ 또는 $x = \frac{11}{6} \pi$
(ⅱ) $\sin x = 1$ 에서 $x = \frac{\pi}{2}$
(ⅰ), (ⅱ)에서 주어진 방정식의 모든 해의 합은
$\frac{7}{6} \pi + \frac{11}{6} \pi + \frac{\pi}{2} = \frac{7}{2} \pi$

정답은 ④ 이다.
