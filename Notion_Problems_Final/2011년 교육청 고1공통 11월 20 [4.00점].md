## 문제
꼭짓점의 좌표가 $(\frac{1}{2}, \frac{9}{2})$ 인 이차함수 $f(x)=ax^{2}+bx+c$ 의 그래프가 점 $(0, 4)$ 를 지날 때, 무리함수 $g(x)=a\sqrt{x+b}+c$ 에 대하여 <보기>에서 옳은 것만을 있는 대로 고른 것은?

$$
\begin{array}{|l|}
\hline
\text{<보기>} \\
\text{ㄱ. 정의역은 } \{x \mid x \ge -2\} \text{ 이고 치역은 } \{y \mid y \le 4\} \text{ 이다.} \\
\text{ㄴ. 함수 } y=g(x) \text{ 의 그래프는 제 } 3 \text{ 사분면을 지난다.} \\
\text{ㄷ. 방정식 } f(x)=0 \text{ 의 두 근을 } \alpha, \beta (\alpha < \beta) \text{ 라 할 때, } \alpha \le x \le \beta \text{ 에서} \\
\text{\quad\ 함수 } g(x) \text{ 의 최댓값은 } 2 \text{ 이다.} \\
\hline
\end{array}
$$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄷ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
이차함수 $f(x)=ax^{2}+bx+c$ 의 꼭짓점의 좌표가 $(\frac{1}{2}, \frac{9}{2})$ 이므로 
$f(x)=a(x-\frac{1}{2})^{2}+\frac{9}{2}$

$y=f(x)$ 의 그래프가 점 $(0, 4)$ 를 지나므로
$4=a(0-\frac{1}{2})^{2}+\frac{9}{2}$
$4=\frac{1}{4}a+\frac{9}{2}$
$\frac{1}{4}a=-\frac{1}{2}$
$a=-2$

$f(x)=-2(x-\frac{1}{2})^{2}+\frac{9}{2}=-2x^{2}+2x+4$ 이므로 
$b=2$, $c=4$

무리함수 $g(x)=-2\sqrt{x+2}+4$ 에 대하여 
ㄱ. 정의역은 $\{x \mid x \ge -2\}$ 이고 치역은 $\{y \mid y \le 4\}$ 이다. (참)

ㄴ. $g(0)=-2\sqrt{2}+4 > 0$ 이고 함수 $y=g(x)$ 는 시작점이 $(-2, 4)$ 이며 $x$ 값이 증가함에 따라 $y$ 값이 감소하므로 제 $3$ 사분면을 지나지 않는다. (거짓)

ㄷ. 방정식 $f(x)=-2(x+1)(x-2)=0$ 의 두 근이 $-1$ 또는 $2$ 이므로 
$\alpha=-1$, $\beta=2$ 

$-1 \le x \le 2$ 에서 함수 $g(x)$ 는 감소하므로 최댓값은 $x=-1$ 일 때이다.
$g(-1)=-2\sqrt{-1+2}+4=2$ (참) 

따라서 옳은 것은 ㄱ, ㄷ이다.
