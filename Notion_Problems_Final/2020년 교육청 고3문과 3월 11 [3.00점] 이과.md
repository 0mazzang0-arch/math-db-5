## 문제
등차수열 $\{ a _{n} \}$ , 등비수열 $\{ b _{n} \}$ 에 대하여 $a _{1} = b _{1} = 3$ 이고 
$b _{3} = -a _{2}$ , $a _{2} + b _{2} = a _{3} + b _{3}$ 
일 때, $a _{3}$ 의 값은?

## 보기
① $-9$   ② $-3$   ③ $0$   ④ $3$   ⑤ $9$

## 풀이
등차수열 $\{ a _{n} \}$ 의 공차를 $d$ , 등비수열 $\{ b _{n} \}$ 의 공비를 $r$ 라 하면 
$a _{n} = 3 + ( n - 1 ) d$ , $b _{n} = 3r ^{n - 1}$ 
$b _{3} = -a _{2}$ 를 $a _{2} + b _{2} = a _{3} + b _{3}$ 에 대입하면 
$a _{2} + b _{2} = a _{3} - a _{2} = d$ 
그러므로 $3 + d + 3r = d$ , $3r = -3$ 에서 
$r = -1$ $\cdots$ ㉠ 

$b _{3} = -a _{2}$ 에서 $3r ^{2} = -( 3 + d )$ $\cdots$ ㉡ 
㉡에 ㉠을 대입하면 $3 \times ( -1 ) ^{2} = -3 - d$ 에서 $d = -6$ 
따라서 $a _{3} = 3 + 2 \times ( -6 ) = -9$

정답: ①
