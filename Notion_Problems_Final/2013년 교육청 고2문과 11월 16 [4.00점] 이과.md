## 문제
두 자연수 $A$, $B$는 다음 조건을 만족시킨다.

$$ \begin{array}{|l|} \hline \text{(가) } \log \frac{10 A}{B} = 3 \\ \text{(나) } \log AB \text{의 정수부분은 } 4 \text{이다.} \\ \hline \end{array} $$

<보기>에서 옳은 것만을 있는 대로 고른 것은?

$$ \begin{array}{|l|} \hline \text{ㄱ. } A = 100 B \\ \text{ㄴ. } A \text{는 네 자리 자연수이다.} \\ \text{ㄷ. } \log B \text{의 소수부분을 } \alpha \text{라 하면 } 0 \leq \alpha < \frac{1}{2} \\ \hline \end{array} $$

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄷ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
ㄱ. 조건 (가)에서 $\log \frac{10 A}{B} = 3$이므로
$\frac{10 A}{B} = 10^{3} = 1000$
$10 A = 1000 B$
따라서 $A = 100 B$이다. (참)

ㄴ. $B = \frac{A}{100}$이고 조건 (나)에서 $\log AB$의 정수부분이 $4$이므로
$4 \leq \log AB < 5$
$4 \leq \log (A \cdot \frac{A}{100}) < 5$
$4 \leq \log \frac{A^{2}}{100} < 5$
$4 \leq 2 \log A - 2 < 5$
$6 \leq 2 \log A < 7$
$3 \leq \log A < \frac{7}{2}$
$\log A$의 정수부분이 $3$이므로 $A$는 네 자리 자연수이다. (참)

ㄷ. $A = 100 B$이고 $4 \leq \log AB < 5$이므로
$4 \leq \log (100 B \cdot B) < 5$
$4 \leq \log 100 B^{2} < 5$
$4 \leq 2 + 2 \log B < 5$
$2 \leq 2 \log B < 3$
$1 \leq \log B < \frac{3}{2}$
$\log B = 1 + \alpha$ ($0 \leq \alpha < 1$)라 하면
$1 \leq 1 + \alpha < \frac{3}{2}$
$0 \leq \alpha < \frac{1}{2}$이다. (참)

따라서 옳은 것은 ㄱ, ㄴ, ㄷ이다.
