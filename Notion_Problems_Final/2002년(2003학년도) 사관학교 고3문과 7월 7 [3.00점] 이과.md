## 문제
두 함수 $f(x) = \log_2 x$, $g(x) = \sqrt{2-x}$에 대하여
$$h_1(x) = f(x) + g(x)$$
$$h_2(x) = \frac{f(x)}{g(x)}$$
$$h_3(x) = (g \circ f)(x)$$
의 정의역을 각각 $D_1$, $D_2$, $D_3$라 할 때, 정의역의 포함관계가 옳은 것은?

## 보기
① $D_1 \subset D_2 \subset D_3$   ② $D_2 \subset D_1 \subset D_3$   ③ $D_2 \subset D_3 \subset D_1$   ④ $D_3 \subset D_1 \subset D_2$   ⑤ $D_3 \subset D_2 \subset D_1$

## 풀이
$f(x) = \log_2 x$, $g(x) = \sqrt{2-x}$이므로

ⅰ) $h_1 (x) = f(x) + g(x) = \log_2 x + \sqrt{2-x}$
$x > 0$이고 실수조건에서 $2 - x \ge 0$이다.
$D_1 = \{ x \mid 0 < x \le 2 \}$

ⅱ) $h_2 (x) = \frac{f(x)}{g(x)} = \frac{\log_2 x}{\sqrt{2-x}}$이다. 
분모가 $0$이 아니어야 하므로 $2 - x > 0$이다.
$D_2 = \{ x \mid 0 < x < 2 \}$

ⅲ) $h_3 (x) = (g \circ f)(x) = \sqrt{2 - \log_2 x}$
$2 - \log_2 x \ge 0$이고, 진수조건에 의해 $x > 0$이다.
$\log_2 x \le 2$에서 $x \le 2^2 = 4$이다.
$D_3 = \{ x \mid 0 < x \le 4 \}$

따라서 $D_2 \subset D_1 \subset D_3$이다.

정답: ②
