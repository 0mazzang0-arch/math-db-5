## 문제
$x$에 대한 연립부등식 
$$\begin{cases} \log_{2} x + \log_{2} (10 - x) \le 4 \\ x^2 - ax < 0 \end{cases}$$
을 만족시키는 $x$의 값 중 정수가 $2$개가 되도록 하는 모든 자연수 $a$의 값의 합을 구하시오.

## 보기

## 풀이
(ⅰ) $\log_{2} x + \log_{2} (10 - x) \le 4$에서 진수 조건에 의해 $0 < x < 10$이고 
$x^2 - 10x + 16 \ge 0$이므로 
$\therefore 0 < x \le 2, 8 \le x < 10$ $\cdots$ ㉠ 

(ⅱ) $x^2 - ax < 0$에서 
$x(x - a) < 0$ 
$\therefore 0 < x < a$ $\cdots$ ㉡ 

> [!example] 📸 (그림/그래프)

㉠, ㉡에서 $x$의 값 중 정수가 $2$개가 되도록 하는 $a$의 값의 범위는 $2 < a \le 8$이다. 
따라서 만족하는 자연수 $a$는 $3, 4, 5, 6, 7, 8$이므로 그 합은 $3 + 4 + 5 + 6 + 7 + 8 = 33$이다.

**[정답]** $33$
