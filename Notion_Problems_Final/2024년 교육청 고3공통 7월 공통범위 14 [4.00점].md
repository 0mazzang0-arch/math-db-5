## 문제
양수 $a$에 대하여 함수 $f(x)$는
$$f(x) = \begin{cases} -2(x+1)^2 + 4 & (x \le 0) \\ a(x-5) & (x > 0) \end{cases}$$
이다. 함수 $f(x)$와 최고차항의 계수가 $1$인 삼차함 $g(x)$에 대하여 $f(k)=g(k)$를 만족시키는 서로 다른 모든 실수 $k$의 값이 $-2$, $0$, $2$일 때, $g(2a)$의 값은?

## 보기
① $14$   ② $18$   ③ $22$   ④ $26$   ⑤ $30$

## 풀이
$f(-2) = g(-2) = 2$, $f(0) = g(0) = 2$이므로 삼차방정식 $g(x) = 2$의 서로 다른 세 실근을 $-2$, $0$, $t$라 하면
$$g(x) - 2 = x(x+2)(x-t)$$
$$g(x) = x(x+2)(x-t) + 2 = x^3 + (2-t)x^2 - 2tx + 2$$
$f(k) = g(k)$를 만족시키는 양수 $k$의 값은 $2$뿐이므로, 이를 만족시키는 두 함수 $y=f(x)$, $y=g(x)$의 그래프의 개형은 다음과 같다.

> [!example] 📸 (그림/그래프)

$x > 0$에서 함수 $y=f(x)$의 그래프는 곡선 $y=g(x)$ 위의 점 $(2, g(2))$에서의 접선과 일치한다.
즉, $f(2) = g(2)$이고 $f'(2) = g'(2)$이다.

$f(2) = a(2-5) = -3a$
$g(2) = 8 + 4(2-t) - 4t + 2 = 18 - 8t$
$-3a = 18 - 8t \cdots$ ㉠

$f'(2) = g'(2)$이므로
$f'(x) = a$ ($x > 0$)에서 $f'(2) = a$
$g'(x) = 3x^2 + 2(2-t)x - 2t$에서
$g'(2) = 12 + 4(2-t) - 2t = 20 - 6t$
$a = 20 - 6t \cdots$ ㉡

두 식 ㉠, ㉡을 연립하면
$-3(20 - 6t) = 18 - 8t$
$-60 + 18t = 18 - 8t$
$26t = 78$
따라서 $t = 3$, $a = 2$이다.

$g(x) = x^3 - x^2 - 6x + 2$이므로
$g(2a) = g(4) = 4^3 - 4^2 - 6(4) + 2 = 64 - 16 - 24 + 2 = 26$
정답은 ④이다.
