## 문제
$0 < a < b < c < 1$ 을 만족하는 세 실수 $a$, $b$, $c$ 에 대하여 
$A = a^{a} b^{b} c^{c}$, $B = a^{a} b^{c} c^{b}$, $C = a^{b} b^{c} c^{a}$ 이라고 하자. 이때, $A$, $B$, $C$ 의 대소 관계로 옳은 것은?

## 보기
① $C < B < A$   ② $B < C < A$   ③ $C < A < B$   ④ $A < C < B$   ⑤ $B < A < C$

## 풀이
($1$) $\frac{A}{B} = \frac{a^{a} b^{b} c^{c}}{a^{a} b^{c} c^{b}} = \frac{b^{b-c}}{c^{b-c}} = \left( \frac{b}{c} \right)^{b-c}$ 에서 
$0 < \frac{b}{c} < 1$, $b - c < 0$ 이므로 
$\frac{A}{B} > 1$ $\therefore B < A$ 

($2$) $\frac{B}{C} = \frac{a^{a} b^{c} c^{b}}{a^{b} b^{c} c^{a}} = \frac{a^{a-b}}{c^{a-b}} = \left( \frac{a}{c} \right)^{a-b}$ 에서 
$0 < \frac{a}{c} < 1$, $a - b < 0$ 이므로 
$\frac{B}{C} > 1$ $\therefore C < B$ 

($1$), ($2$)에서 
$C < B < A$ 

정답: ①
