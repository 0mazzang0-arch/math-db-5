## 문제
영벡터가 아닌 두 공간벡터 $\vec{a}= (a_{1}, a_{2}, a_{3})$ 와 $\vec{b}= (b_{1}, b_{2}, b_{3})$ 에 대하여 $x$축, $y$축, $z$축에 대한 $\vec{a}$ 의 방향코사인을 차례대로 $\cos \alpha_{1}$, $\cos \beta_{1}$, $\cos \gamma_{1}$ 이라 하고, $\vec{b}$ 의 방향코사인을 차례대로 $\cos \alpha_{2}$, $\cos \beta_{2}$, $\cos \gamma_{2}$ 라 할 때, <보기>에서 옳은 것을 모두 고른 것은?

<보기>
ㄱ. $\cos \alpha_{1} \cos \alpha_{2} + \cos \beta_{1} \cos \beta_{2} + \cos \gamma_{1} \cos \gamma_{2} = 1$ 이다.
ㄴ. 벡터 $\vec{a}$ 의 각 성분에 대하여 $a_{1} < a_{2} < a_{3}$ 이면 $\gamma_{1} < \beta_{1} < \alpha_{1}$ 이다.
ㄷ. 벡터 $\vec{a}$ 의 성분 중 $a_{3} = -5$ 이면 $\frac{\pi}{2} < \gamma_{1} < \pi$ 이다.

## 보기
① $ㄱ$ ② $ㄴ$ ③ $ㄴ, ㄷ$ ④ $ㄱ, ㄷ$ ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
(ⅰ) $\vec{a} = (a_{1}, a_{2}, a_{3})$ 의 방향코사인은
$\Rightarrow \cos \alpha_{1} = \frac{a_{1}}{|\vec{a}|}$, $\cos \beta_{1} = \frac{a_{2}}{|\vec{a}|}$, $\cos \gamma_{1} = \frac{a_{3}}{|\vec{a}|}$

(ⅱ) $\vec{b} = (b_{1}, b_{2}, b_{3})$ 의 방향코사인은
$\Rightarrow \cos \alpha_{2} = \frac{b_{1}}{|\vec{b}|}$, $\cos \beta_{2} = \frac{b_{2}}{|\vec{b}|}$, $\cos \gamma_{2} = \frac{b_{3}}{|\vec{b}|}$

ㄱ. $\cos \alpha_{1} \cos \alpha_{2} + \cos \beta_{1} \cos \beta_{2} + \cos \gamma_{1} \cos \gamma_{2} = \frac{a_{1} b_{1} + a_{2} b_{2} + a_{3} b_{3}}{|\vec{a}| |\vec{b}|}$
위 식의 결과는 두 벡터가 이루는 각 $\theta$ 에 대한 $\cos \theta$ 의 값이므로 항상 $1$이라 할 수 없다. (거짓)

ㄴ. 벡터 $\vec{a}$ 의 각 성분이 $a_{1} < a_{2} < a_{3}$ 이면 $\cos \alpha_{1} < \cos \beta_{1} < \cos \gamma_{1}$ 이고, 구간 $[0, \pi]$ 에서 $y = \cos x$ 는 감소함수이므로 $\alpha_{1} > \beta_{1} > \gamma_{1}$ 이다. 즉, $\gamma_{1} < \beta_{1} < \alpha_{1}$ 이다. (참)

ㄷ. 벡터 $\vec{a}$ 의 성분 중 $a_{3} = -5$ 이면 $\cos \gamma_{1} = -\frac{5}{|\vec{a}|} < 0$ 이므로 $\frac{\pi}{2} < \gamma_{1} < \pi$ 이다. (참)

따라서 옳은 것은 ㄴ, ㄷ이다.
