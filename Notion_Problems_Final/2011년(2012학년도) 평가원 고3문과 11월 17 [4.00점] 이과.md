## 문제
첫째항이 $1$인 수열 $\{ a_{n} \}$에 대하여 $S_{n}= \sum_{k=1}^{n} a_{k}$라 할 때,
$n S_{n+1}=(n+2) S_{n}+(n+1)^{3}$ $(n \geq 1)$
이 성립한다. 다음은 수열 $\{ a_{n} \}$의 일반항을 구하는 과정의 일부이다.

> 자연수 $n$에 대하여 $S_{n+1}=S_{n}+a_{n+1}$이므로
> $n a_{n+1}=2 S_{n}+(n+1)^{3}$ $\cdots\cdots$ ㉠
> 이다. $2$ 이상의 자연수 $n$에 대하여
> $(n-1) a_{n}=2 S_{n-1}+n^{3}$ $\cdots\cdots$ ㉡
> 이고, ㉠에서 ㉢을 뺀 식으로부터
> $n a_{n+1}=(n+1) a_{n} + (가)$
> 를 얻는다. 양변을 $n(n+1)$로 나누면
> $\frac{a_{n+1}}{n+1}=\frac{a_{n}}{n}+\frac{(가)}{n(n+1)}$
> 이다. $b_{n}=\frac{a_{n}}{n}$이라 하면,
> $b_{n+1}=b_{n}+3+(나)$ $(n \geq 2)$
> 이므로
> $b_{n}=b_{2}+(다)$ $(n \geq 3)$
> 이다.
> $\vdots$

위의 $(가)$, $(나)$, $(다)$에 들어갈 식을 각각 $f(n)$, $g(n)$, $h(n)$이라 할 때, $\frac{f(3)}{g(3) h(6)}$의 값은?

## 보기
① $30$   ② $36$   ③ $42$   ④ $48$   ⑤ $54$

## 풀이
[정답] ② 

자연수 $n$에 대하여 $S_{n+1} = S_n + a_{n+1}$이므로 $n S_{n+1} = (n+2)S_n + (n+1)^3$에 대입하면
$n(S_n + a_{n+1}) = (n+2)S_n + (n+1)^3$
$n S_n + n a_{n+1} = n S_n + 2S_n + (n+1)^3$
$n a_{n+1} = 2S_n + (n+1)^3$ $\cdots\cdots$ ㉠
이다. $2$ 이상의 자연수 $n$에 대하여
$(n-1) a_n = 2S_{n-1} + n^3$ $\cdots\cdots$ ㉡
이고, ㉠에서 ㉡을 빼면
$n a_{n+1} - (n-1) a_n = 2(S_n - S_{n-1}) + (n+1)^3 - n^3$
$n a_{n+1} - (n-1) a_n = 2 a_n + (3n^2 + 3n + 1)$
$n a_{n+1} = (n+1) a_n + (3n^2 + 3n + 1)$
양변을 $n(n+1)$로 나누면
$\frac{a_{n+1}}{n+1} = \frac{a_n}{n} + \frac{3n^2 + 3n + 1}{n(n+1)}$
이다. $b_n = \frac{a_n}{n}$이라 하면
$b_{n+1} = b_n + \frac{3n(n+1)+1}{n(n+1)} = b_n + 3 + \frac{1}{n(n+1)}$ $(n \geq 2)$
이므로
$b_n = b_2 + \sum_{k=2}^{n-1} \left( 3 + \frac{1}{k(k+1)} \right)$
$b_n = b_2 + 3(n-2) + \sum_{k=2}^{n-1} \left( \frac{1}{k} - \frac{1}{k+1} \right)$
$b_n = b_2 + 3(n-2) + \frac{1}{2} - \frac{1}{n}$ $(n \geq 3)$
이다.
따라서 $f(n) = 3n^2 + 3n + 1$, $g(n) = \frac{1}{n(n+1)}$, $h(n) = 3(n-2) + \frac{1}{2} - \frac{1}{n}$이므로
$\frac{f(3)}{g(3)h(6)} = \frac{37}{\frac{1}{12} \times \frac{37}{3}} = 36$
