## 문제
$0 \le x < 2 \pi$ 일 때, 함수 $f(x) = 2 \cos^{2} x - |1 + 2 \sin x| - 2 |\sin x| + 2$ 에 대하여 집합 $A = \{ x \mid f(x) \text{의 값은 } 0 \text{ 이하의 정수} \}$ 라 하자. 집합 $A$ 의 원소의 개수는?

## 보기
① $6$   ② $7$   ③ $8$   ④ $9$   ⑤ $10$

## 풀이
$f(x) = 2 \cos^{2} x - |1 + 2 \sin x| - 2 |\sin x| + 2$
$= 2(1 - \sin^{2} x) - |1 + 2 \sin x| - 2 |\sin x| + 2$
$= -2 \sin^{2} x - |1 + 2 \sin x| - 2 |\sin x| + 4$

$\sin x = t$ $(-1 \le t \le 1)$ 이라 하고, $g(t) = f(x)$ 라 하면
$$ g(t) = -2t^2 - |1 + 2t| - 2|t| + 4 $$
$$ g(t) = \left\{ \begin{array}{ll} -2t^2 + 4t + 5 & (-1 \le t < -\frac{1}{2}) \\ -2t^2 + 3 & (-\frac{1}{2} \le t < 0) \\ -2t^2 - 4t + 3 & (0 \le t \le 1) \end{array} \right. $$

함수 $y = g(t)$ 의 그래프를 좌표평면에 나타내면 그림과 같다.

> [!example] 📸 (그림/그래프)

즉 $g(t)$ 의 값이 $0$ 이하의 정수가 되도록 하는 $t$ 의 값은 다음과 같다.

$g(t) = 0$ 일 때, $t = a$, $t = b$ $(-1 < a < -\frac{1}{2}, 0 < b < 1)$
$g(t) = -1$ 일 때, $t = -1$, $t = c$ $(b < c < 1)$
$g(t) = -2$ 일 때, $t = d$ $(c < d < 1)$
$g(t) = -3$ 일 때, $t = 1$

$0 \le x < 2 \pi$ 에서 함수 $y = \sin x$ 의 그래프와 직선 $y = t$ 를 좌표평면에 나타내면 그림과 같다.

> [!example] 📸 (그림/그래프)

따라서 조건을 만족시키는 집합 $A$ 의 원소의 개수는 다음과 같다.
- $\sin x = a$ 일 때, $x$ 의 개수는 $2$
- $\sin x = b$ 일 때, $x$ 의 개수는 $2$
- $\sin x = -1$ 일 때, $x$ 의 개수는 $1$
- $\sin x = c$ 일 때, $x$ 의 개수는 $2$
- $\sin x = d$ 일 때, $x$ 의 개수는 $2$
- $\sin x = 1$ 일 때, $x$ 의 개수는 $1$

그러므로 원소의 개수는 $2 + 2 + 1 + 2 + 2 + 1 = 10$ 이다.
