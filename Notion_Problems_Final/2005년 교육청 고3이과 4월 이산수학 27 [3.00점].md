## 문제
함수 $f(x) = 4 \sin x + 3 \cos x$가 $x = \theta$에서 최댓값을 가질 때, $3 \tan \theta$의 값은? (단, $0 \le x < 2\pi$)

## 보기
① $3$   ② $4$   ③ $5$   ④ $6$   ⑤ $7$

## 풀이
정답: ②

함수 $f(x) = 4 \sin x + 3 \cos x$를 합성하면
$f(x) = 5 \sin (x + \alpha)$
(단, $\cos \alpha = \frac{4}{5}$, $\sin \alpha = \frac{3}{5}$)
$0 \le x < 2\pi$인 범위에서 $x + \alpha = \frac{\pi}{2}$일 때 최댓값을 가지므로 $\theta + \alpha = \frac{\pi}{2}$이다.
한편,
$\tan \theta = \tan \left( \frac{\pi}{2} - \alpha \right) = \cot \alpha$
이고 $\cot \alpha = \frac{4}{3}$이므로
$3 \tan \theta = 3 \cdot \frac{4}{3} = 4$
