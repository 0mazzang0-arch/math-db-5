## 문제
양수 $x$ 에 대하여 $\log x$ 의 정수부분을 $f(x)$ 라 할 때 
$$f(ab) = f(a)f(b) + 2$$
를 만족시키는 $20$ 이하의 두 자연수 $a$, $b$ 의 순서쌍 $(a, b)$ 에 대하여 $a+b$ 의 최솟값은?

## 보기
① $19$   ② $20$   ③ $21$   ④ $22$   ⑤ $23$

## 풀이
조건을 만족시키는 $20$ 이하의 두 자연수 $a, b$ 에 대하여 $a+b$ 의 최솟값을 다음과 같이 $4$ 가지로 구분하여 구할 수 있다. 

(ⅰ) $1 \le a \le 9, 1 \le b \le 9$ 인 경우 
$f(a) = f(b) = 0$ 이므로 
$f(ab) = f(a)f(b) + 2 = 0 \times 0 + 2 = 2$ 
$1 \le ab \le 81$ 이므로 $f(ab) = 2$ 인 자연수 $a, b$ 는 존재하지 않는다. 

(ⅱ) $1 \le a \le 9, 10 \le b \le 20$ 인 경우 
$f(a) = 0, f(b) = 1$ 이므로 
$f(ab) = f(a)f(b) + 2 = 0 \times 1 + 2 = 2$ 
$10 \le ab \le 180$ 이므로 $f(ab) = 2$ 이려면 $100 \le ab \le 180$ 이어야 한다. 
$(a, b)$ 는 
$(5, 20),$ 
$(6, 17), (6, 18), (6, 19), (6, 20),$ 
$(7, 15), (7, 16), \dots, (7, 20),$ 
$(8, 13), (8, 14), \dots, (8, 20),$ 
$(9, 12), (9, 13), \dots, (9, 20)$ 
$a+b$ 의 최솟값은 $21$ 이다. 

(ⅲ) $10 \le a \le 20, 1 \le b \le 9$ 인 경우 
$a+b$ 의 최솟값은 $21$ 이다. 

(ⅳ) $10 \le a \le 20, 10 \le b \le 20$ 인 경우 
$f(a) = f(b) = 1$ 이므로 
$f(ab) = f(a)f(b) + 2 = 1 \times 1 + 2 = 3$ 
$100 \le ab \le 400$ 이므로 $f(ab) = 3$ 인 자연수 $a, b$ 는 존재하지 않는다. 

(ⅰ)~(ⅳ)에서 구하는 $a+b$ 의 최솟값은 $21$ 이다. 

정답: ③
