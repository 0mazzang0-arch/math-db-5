## 문제

그림과 같이 삼각형 $ABC$에서 선분 $AB$ 위에 $\overline{AD} : \overline{DB} = 3 : 2$인 점 $D$를 잡고, 점 $A$를 중심으로 하고 점 $D$를 지나는 원을 $O$, 원 $O$와 선분 $AC$가 만나는 점을 $E$라 하자. $\sin A : \sin C = 8 : 5$이고, 삼각형 $ADE$와 삼각형 $ABC$의 넓이의 비가 $9 : 35$이다. 삼각형 $ABC$의 외접원의 반지름의 길이가 $7$일 때, 원 $O$ 위의 점 $P$에 대하여 삼각형 $PBC$의 넓이의 최댓값은? (단, $\overline{AB} < \overline{AC}$)

> [!example] 📸 (그림/그래프)

## 보기

① $18+15\sqrt{3}$   ② $24+20\sqrt{3}$   ③ $30+25\sqrt{3}$   ④ $36+30\sqrt{3}$   ⑤ $42+35\sqrt{3}$

## 풀이

[정답] ④

주어진 그림에서 $\overline{AD} : \overline{DB} = 3 : 2$이므로 $\overline{AD} = 3t$ ($t > 0$)이라 하면 $\overline{DB} = 2t$, $\overline{AB} = 5t$이다.
이때 삼각형 $ABC$의 외접원의 반지름의 길이가 $7$이므로 사인법칙에 의하여
$\frac{\overline{BC}}{\sin A} = \frac{\overline{AB}}{\sin C} = 14$, $\sin A : \sin C = \overline{BC} : \overline{AB}$이다.
$\sin A : \sin C = 8 : 5$이므로 $\overline{BC} : \overline{AB} = 8 : 5$에서 $\overline{BC} = 8t$이다.
두 삼각형 $ADE$, $ABC$의 넓이의 비가 $9 : 35$이므로
$\frac{1}{2} \times 3t \times \overline{AE} \times \sin A : \frac{1}{2} \times 5t \times \overline{AC} \times \sin A = 9 : 35$
$3\overline{AE} : 5\overline{AC} = 9 : 35$, $\overline{AE} = \frac{3}{7}\overline{AC}$이다.
즉 $\overline{AE} : \overline{AC} = 3 : 7$이고 $\overline{AD} = \overline{AE}$이므로 $\overline{AE} = 3t$, $\overline{AC} = 7t$라 할 수 있다.
삼각형 $ABC$에서 코사인법칙에 의하여
$\cos(\angle ABC) = \frac{(5t)^2 + (8t)^2 - (7t)^2}{2 \times 5t \times 8t} = \frac{40t^2}{80t^2} = \frac{1}{2}$
$\therefore \angle ABC = 60^\circ$

> [!example] 📸 (그림/그래프)

삼각형 $ABC$의 외접원의 반지름의 길이가 $7$이므로 사인법칙에 의하여
$\frac{\overline{AC}}{\sin(\angle ABC)} = 14$, $7t = 14 \times \sin 60^\circ = 7\sqrt{3}$
즉 $t = \sqrt{3}$이므로 $\overline{BC} = 8t = 8\sqrt{3}$, $\overline{AD} = \overline{AE} = 3\sqrt{3}$이다.
이때 점 $A$에서 변 $BC$에 내린 수선의 발을 $H$라 하면
$\overline{AH} = \overline{AB} \times \sin 60^\circ = 5\sqrt{3} \times \frac{\sqrt{3}}{2} = \frac{15}{2}$이다.
따라서 원 $O$ 위의 점 $P$에 대하여 삼각형 $PBC$의 넓이의 최댓값은
$\frac{1}{2} \times (3\sqrt{3} + \frac{15}{2}) \times 8\sqrt{3} = 36 + 30\sqrt{3}$이다.
