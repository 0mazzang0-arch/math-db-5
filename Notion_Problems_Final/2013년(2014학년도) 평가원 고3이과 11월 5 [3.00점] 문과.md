## 문제
두 사건 $A$, $B$에 대하여 $P(A^{C} \cup B^{C}) = \frac{4}{5}$, $P(A \cap B^{C}) = \frac{1}{4}$일 때, $P(A^{C})$의 값은? (단, $A^{C}$은 $A$의 여사건이다.)

## 보기
① $\frac{1}{2}$   ② $\frac{11}{20}$   ③ $\frac{3}{5}$   ④ $\frac{13}{20}$   ⑤ $\frac{7}{10}$

## 풀이
$A^{c} \cup B^{c} = (A \cap B)^{c}$이므로 
$P(A^{c} \cup B^{c}) = P((A \cap B)^{c}) = 1 - P(A \cap B) = \frac{4}{5}$에서 
$P(A \cap B) = 1 - \frac{4}{5} = \frac{1}{5}$
$P(A \cap B^{C}) = P(A) - P(A \cap B) = P(A) - \frac{1}{5} = \frac{1}{4}$에서 
$P(A) = \frac{1}{5} + \frac{1}{4} = \frac{9}{20}$
따라서 
$P(A^{c}) = 1 - P(A) = 1 - \frac{9}{20} = \frac{11}{20}$

[다른 풀이] 
일어날 수 있는 모든 경우의 집합을 $S$라 하면 
$(A^{c} \cup B^{c})^{c} \cup (A \cap B^{c}) = (A \cap B) \cup (A \cap B^{c}) = A \cap (B \cup B^{c}) = A \cap S = A$
$(A^{c} \cup B^{c})^{c} \cap (A \cap B^{c}) = (A \cap B) \cap (A \cap B^{c}) = A \cap B \cap B^{c} = \emptyset$
$P(A) = P((A^{c} \cup B^{c})^{c} \cup (A \cap B^{c})) = P((A^{c} \cup B^{c})^{c}) + P(A \cap B^{c})$
$= 1 - P(A^{c} \cup B^{c}) + P(A \cap B^{c}) = 1 - \frac{4}{5} + \frac{1}{4} = \frac{9}{20}$
따라서 
$P(A^{c}) = 1 - P(A) = 1 - \frac{9}{20} = \frac{11}{20}$
