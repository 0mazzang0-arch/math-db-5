## 문제
좌표공간에서 직선 $\frac{x-2}{2} = \frac{y-2}{3} = z-1$ 에 수직이고 점 $(1, -5, 2)$ 를 지나는 평면의 방정식을 $2x + ay + bz + c = 0$ 이라 할 때, $a+b+c$ 의 값을 구하시오.

## 보기

## 풀이
직선에 수직이므로 평면의 법선벡터 $\vec{h} = (2, 3, 1)$ 이다. 
따라서 평면의 방정식은 
$2(x-1) + 3(y+5) + (z-2) = 0$ 
$2x + 3y + z + 11 = 0$ 
$\therefore a = 3, b = 1, c = 11$ 
$a + b + c = 15$ 

정답: $15$
