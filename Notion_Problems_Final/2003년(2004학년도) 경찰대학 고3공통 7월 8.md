## 문제
$x^{10} + 3x^{3} - 2$를 $x^{2} - x + 1$로 나눈 나머지를 $p(x)$라고 할 때, $y = p(x)$에 대하여 다음 설명 중 옳은 것은?

## 보기
① 제 $1$, $2$, $3$ 사분면만을 지난다.   ② 제 $1$, $2$, $4$ 사분면만을 지난다.   ③ 제 $1$, $3$, $4$ 사분면만을 지난다.   ④ 제 $2$, $3$, $4$ 사분면만을 지난다.   ⑤ 제 $3$, $4$ 사분면만을 지난다.

## 풀이
[정답] ④

$x^{10} + 3x^{3} - 2$를 $x^{2} - x + 1$로 나누었을 때의 몫을 $Q(x)$, 나머지를 $p(x) = ax + b$ ($a$, $b$는 상수)라 하면
$x^{10} + 3x^{3} - 2 = (x^{2} - x + 1)Q(x) + ax + b$ $\cdots \cdots$ ㉠

$x^{2} - x + 1 = 0$의 한 허근을 $\omega$라 하면
$\omega^{2} - \omega + 1 = 0$, $\omega^{3} = -1$, $\omega^{6} = 1$
따라서 ㉠의 양변에 $x = \omega$를 대입하면
$\omega^{10} + 3\omega^{3} - 2 = a\omega + b$
$\therefore -\omega - 5 = a\omega + b$

$a$, $b$는 실수이므로 복소수가 서로 같을 조건에 의하여
$a = -1$, $b = -5$
따라서 $p(x) = -x - 5$이므로 일차함수 $y = -x - 5$의 그래프는 제 $2$, $3$, $4$ 사분면만을 지난다.
