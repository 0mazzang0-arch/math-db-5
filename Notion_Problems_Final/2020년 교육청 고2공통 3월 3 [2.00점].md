## 문제
부등식 $|x-2| < 3$을 만족시키는 정수 $x$의 개수는?

## 보기
① $1$   ② $3$   ③ $5$   ④ $7$   ⑤ $9$

## 풀이
$|x-2| < 3$에서 
$-3 < x-2 < 3$ 
각 변에 $2$를 더하면 
$-1 < x < 5$ 
따라서 조건을 만족시키는 정수 $x$는 $0$, $1$, $2$, $3$, $4$이므로 그 개수는 $5$이다. 

[다른 풀이] 
$|x-2| < 3$에서 
$-3 < x-2 < 3$ 

$-3 < x-2$에서 
$x > -1 \cdots$ ㉠ 

$x-2 < 3$에서 
$x < 5 \cdots$ ㉡ 
㉠, ㉡을 모두 만족시키는 $x$의 값의 범위는 
$-1 < x < 5$
