## 문제
$-1 \le x \le 2$에서 함수 $f(x) = a \times 2^{2-x} + b$의 최댓값이 $5$, 최솟값이 $-2$일 때, $f(0)$의 값은? (단, $a > 0$이고, $a$와 $b$는 상수이다.)

## 보기
① $1$ ② $\frac{3}{2}$ ③ $2$ ④ $\frac{5}{2}$ ⑤ $3$

## 풀이
함수 $f(x) = a \times 2^{2-x} + b = 4a \times \left( \frac{1}{2} \right)^{x} + b$는 $a > 0$이고 밑이 $1$보다 작으므로, $x$의 값이 증가하면 $y$의 값은 감소한다.
따라서 함수 $f(x)$는 $x = -1$일 때 최댓값 $5$를 갖고, $x = 2$일 때 최솟값 $-2$를 갖는다.

$f(-1) = a \times 2^{3} + b = 8a + b = 5$
$f(2) = a \times 2^{0} + b = a + b = -2$

두 식을 연립하면 $7a = 7$이므로 $a = 1$, $b = -3$이다.
따라서 $f(0) = 4a + b = 4 - 3 = 1$이다.

정답: ①
