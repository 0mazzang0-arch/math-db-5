## 문제
$a_{11} + a_{21} = 82$ , $a_{11} - a_{21} = 6$ 일 때, 집합 $A = \{ a_{n} \mid a_{n} \text{은 자연수} \}$ 의 모든 원소의 합을 구하시오.

## 풀이
등차수열 $\{ a_{n} \}$ 의 첫째항을 $a$, 공차를 $d$ 로 놓으면 $a_{11} + a_{21} = 82$ 에서 
$2a + 30d = 82$ 
$a_{11} - a_{21} = 6$ 에서 
$-10d = 6$ 즉, $d = -\frac{3}{5}$ 
$a = 41 - 15d = 41 - 15 \times ( -\frac{3}{5} ) = 50$ 
$\therefore a_{n} = 50 + ( n - 1 ) \times ( -\frac{3}{5} )$ 

$a_{n}$ 이 자연수이려면 $n - 1 = 5k$, 즉 $n = 5k + 1$ ($k$ 는 음이 아닌 정수)의 형태이어야 하고, 
$a_{n} = 50 + ( n - 1 ) \times ( -\frac{3}{5} ) > 0$ 에서 
$n < \frac{253}{3} = 84.3 \dots$ 
이므로 $n$ 이 가질 수 있는 값은 $1$, $6$, $11$, $\dots$, $81$ 이고 그 개수는 $17$ 이다. 

수열 $a_{1}$, $a_{6}$, $a_{11}$, $\dots$, $a_{81}$ 은 첫째항이 $50$ 이고 제 $17$ 항이 $2$ 인 등차수열이므로 그 합은 
$\frac{17 ( 50 + 2 )}{2} = 442$
