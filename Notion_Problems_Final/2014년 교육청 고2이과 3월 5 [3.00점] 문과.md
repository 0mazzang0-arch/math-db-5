## 문제
$-2 \le x \le 2$ 에서 정의된 이차함수 $f(x) = x^2 - 2x + a$ 의 최댓값과 최솟값의 합이 $21$ 일 때, 상수 $a$ 의 값은?

## 보기
① $6$   ② $7$   ③ $8$   ④ $9$   ⑤ $10$

## 풀이
$f(x) = x^2 - 2x + a = (x-1)^2 + a-1$
($-2 \le x \le 2$)이므로 

> [!example] 📸 (그림/그래프)

위의 그림에서 함수 $f(x)$ 의 최솟값은 $f(1) = a-1$ 이고, 최댓값은 $f(-2) = a+8$ 이다. 최댓값과 최솟값의 합이 $21$ 이므로 
$(a+8) + (a-1) = 21$
$\therefore a=7$

정답: ②
