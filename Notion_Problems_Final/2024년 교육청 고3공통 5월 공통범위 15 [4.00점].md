## 문제
첫째항이 자연수인 수열 $\{ a_{n} \}$ 이 모든 자연수 $n$ 에 대하여 
$$ a_{n+1} = \begin{cases} \frac{a_{n}}{3} & (a_{n} \text{ 이 } 3\text{의 배수인 경우}) \\ \frac{a_{n}^{2} + 5}{3} & (a_{n} \text{ 이 } 3\text{의 배수가 아닌 경우}) \end{cases} $$
를 만족시킬 때, $a_{4} + a_{5} = 5$ 가 되도록 하는 모든 $a_{1}$ 의 값의 합은?

## 보기
① $63$   ② $66$   ③ $69$   ④ $72$   ⑤ $75$

## 풀이
[정답] ④

$a_{n}$ 이 자연수라 하자. 자연수 $k$ 에 대하여 
$a_{n} = 3k - 2$ 이면 
$a_{n+1} = \frac{(3k - 2)^{2} + 5}{3} = \frac{9k^{2} - 12k + 9}{3} = 3k^{2} - 4k + 3$ , 

$a_{n} = 3k - 1$ 이면 
$a_{n+1} = \frac{(3k - 1)^{2} + 5}{3} = \frac{9k^{2} - 6k + 6}{3} = 3k^{2} - 2k + 2$ , 

$a_{n} = 3k$ 이면 
$a_{n+1} = \frac{3k}{3} = k$ 
이므로 $a_{n}$ 이 자연수이면 $a_{n+1}$ 도 자연수이다. 

$a_{1}$ 이 자연수이므로 모든 자연수 $n$ 에 대하여 $a_{n}$ 은 자연수이다. 
$\cdots$ $\cdots$ ㉠ 

$a_{4}$ 가 $3$ 의 배수이면 $a_{5} = \frac{a_{4}}{3}$ 이므로 
$a_{4} + a_{5} = 5$ 에서 $a_{4} + \frac{a_{4}}{3} = 5$ , 
$a_{4} = \frac{15}{4}$ 가 되어 ㉠을 만족시키지 않는다. 
그러므로 $a_{4}$ 는 $3$ 의 배수가 아니다. 

$a_{5} = \frac{a_{4}^{2} + 5}{3}$ 이므로 $a_{4} + a_{5} = 5$ 에서 
$a_{4} + \frac{a_{4}^{2} + 5}{3} = 5$ 
$a_{4}^{2} + 3a_{4} - 10 = (a_{4} + 5)(a_{4} - 2) = 0$ 
㉠에 의하여 $a_{4} = 2$ 

(ⅰ) $a_{3}$ 이 $3$ 의 배수인 경우 
$a_{4} = \frac{a_{3}}{3} = 2$ 이므로 $a_{3} = 6$ 

$a_{2}$ 의 값을 구하면 
ⓐ $a_{2}$ 가 $3$ 의 배수인 경우 
$a_{3} = \frac{a_{2}}{3} = 6$ 이므로 $a_{2} = 18$ 
ⓑ $a_{2}$ 가 $3$ 의 배수가 아닌 경우 
$a_{3} = \frac{a_{2}^{2} + 5}{3} = 6$ 이므로 $a_{2}^{2} = 13$ 이 되어 ㉠을 만족시키지 않는다. 
그러므로 $a_{2} = 18$ 

$a_{1}$ 의 값을 구하면 
ⓐ $a_{1}$ 이 $3$ 의 배수인 경우 
$a_{2} = \frac{a_{1}}{3} = 18$ 이므로 $a_{1} = 54$ 
ⓑ $a_{1}$ 이 $3$ 의 배수가 아닌 경우 
$a_{2} = \frac{a_{1}^{2} + 5}{3} = 18$ 이므로 $a_{1}^{2} = 49$ 
$a_{1} = 7$ 

(ⅱ) $a_{3}$ 이 $3$ 의 배수가 아닌 경우 
$a_{4} = \frac{a_{3}^{2} + 5}{3} = 2$ 이므로 $a_{3}^{2} = 1$ 
$a_{3} = 1$ 

$a_{2}$ 의 값을 구하면 
ⓐ $a_{2}$ 가 $3$ 의 배수인 경우 
$a_{3} = \frac{a_{2}}{3} = 1$ 이므로 $a_{2} = 3$ 
ⓑ $a_{2}$ 가 $3$ 의 배수가 아닌 경우 
$a_{3} = \frac{a_{2}^{2} + 5}{3} = 1$ 이므로 $a_{2}^{2} = -2$ 가 되어 ㉠을 만족시키지 않는다. 
그러므로 $a_{2} = 3$ 

$a_{1}$ 의 값을 구하면 
ⓐ $a_{1}$ 이 $3$ 의 배수인 경우 
$a_{2} = \frac{a_{1}}{3} = 3$ 이므로 $a_{1} = 9$ 
ⓑ $a_{1}$ 이 $3$ 의 배수가 아닌 경우 
$a_{2} = \frac{a_{1}^{2} + 5}{3} = 3$ 이므로 $a_{1}^{2} = 4$ 
$a_{1} = 2$ 

(ⅰ), (ⅱ)에 의하여 모든 $a_{1}$ 의 값의 합은 
$54 + 7 + 9 + 2 = 72$
