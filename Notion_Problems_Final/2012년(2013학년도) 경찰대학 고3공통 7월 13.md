## 문제
$f ( x^{2} )$ $=$ $f ( x ) f ( -x )$를 만족시키는 이차식 $f ( x )$의 개수는?

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
$f ( x )$가 이차식이므로 
$f ( x ) = ax^{2} + bx + c$ 
$( a \neq 0 )$라 놓으면 

$f ( x^{2} ) = f ( x ) f ( -x )$ 

$ax^{4} + bx^{2} + c = ( ax^{2} + bx + c ) ( ax^{2} - bx + c )$ 

$\therefore ax^{4} + bx^{2} + c = a^{2} x^{4} + ( 2ac - b^{2} ) x^{2} + c^{2}$ 

위 등식이 $x$에 대한 항등식이므로 

$a = a^{2}$, $2ac - b^{2} = b$, $c = c^{2}$ 

이때, $a \neq 0$이므로 $a = 1$ 

따라서 조건을 만족시키는 순서쌍 $( a, b, c )$는 

$( 1, 1, 1 )$, $( 1, -2, 1 )$, $( 1, 0, 0 )$, $( 1, -1, 0 )$ 

의 $4$개다.
