## 문제
곡선 $y = \log_{2}x$ 위에 서로 다른 두 점 $A$, $B$가 있다. 점 $A$에서 직선 $y = x$에 내린 수선의 발을 $P$라 하고, 점 $B$를 직선 $y = x$에 대하여 대칭이동한 점을 $Q$라 할 때, 네 점 $A$, $B$, $P$, $Q$가 다음 조건을 만족시킨다.

$$ \begin{array}{|l|} \hline \text{(가) (직선 } AP\text{의 } y\text{절편)} - \text{(직선 } BQ\text{의 } y\text{절편)} = \frac{13}{2} \\ \text{(나) 직선 } AB\text{의 기울기는 } \frac{6}{7} \text{이다.} \\ \hline \end{array} $$

사각형 $APQB$의 넓이가 $\frac{q}{p}$일 때, $p + q$의 값을 구하시오. (단, $p$와 $q$는 서로소인 자연수이다.)

## 풀이
함수 $y = \log_{2}x$의 역함수는 $y = 2^{x}$이고 두 곡선은 직선 $y = x$에 대하여 대칭이다.

> [!example] 📸 (그림/그래프)

곡선 $y = \log_{2}x$ 위의 서로 다른 두 점을 $A(a, \log_{2}a)$, $B(b, \log_{2}b)$라 하자.
점 $A(a, \log_{2}a)$에서 직선 $y = x$에 내린 수선의 발이 $P$이므로 직선 $AP$의 기울기는 $-1$이다.
직선 $AP$의 방정식은
$y - \log_{2}a = -(x - a)$
$y = -x + a + \log_{2}a$
따라서 직선 $AP$의 $y$절편은 $a + \log_{2}a$이다.

점 $B(b, \log_{2}b)$를 직선 $y = x$에 대하여 대칭이동한 점이 $Q$이므로 $Q(\log_{2}b, b)$이다.
직선 $BQ$는 두 점 $B(b, \log_{2}b)$, $Q(\log_{2}b, b)$를 지나므로 기울기는 $\frac{b - \log_{2}b}{\log_{2}b - b} = -1$이다.
직선 $BQ$의 방정식은
$y - \log_{2}b = -(x - b)$
$y = -x + b + \log_{2}b$
따라서 직선 $BQ$의 $y$절편은 $b + \log_{2}b$이다.

조건 (가)에 의하여
$(a + \log_{2}a) - (b + \log_{2}b) = \frac{13}{2}$
$(a - b) + (\log_{2}a - \log_{2}b) = \frac{13}{2}$
$(a - b) + \log_{2}\frac{a}{b} = \frac{13}{2} \cdots \cdots$ ㉠

조건 (나)에서 직선 $AB$의 기울기는 $\frac{6}{7}$이므로
$\frac{\log_{2}a - \log_{2}b}{a - b} = \frac{6}{7}$
$\log_{2}a - \log_{2}b = \frac{6}{7}(a - b)$
$\log_{2}\frac{a}{b} = \frac{6}{7}(a - b) \cdots \cdots$ ㉡

㉡을 ㉠에 대입하면
$(a - b) + \frac{6}{7}(a - b) = \frac{13}{2}$
$\frac{13}{7}(a - b) = \frac{13}{2}$
$a - b = \frac{7}{2}$

이것을 다시 ㉡에 대입하면
$\log_{2}\frac{a}{b} = \frac{6}{7} \times \frac{7}{2} = 3$
$\frac{a}{b} = 2^{3} = 8$
$\therefore a = 8b$

$a - b = 8b - b = 7b = \frac{7}{2}$에서
$b = \frac{1}{2}$, $a = 8 \times \frac{1}{2} = 4$
따라서 두 점의 좌표는 $A(4, 2)$, $B(\frac{1}{2}, -1)$이다.

점 $P$는 직선 $AP$와 직선 $y = x$의 교점이고 직선 $AP$의 방정식은 $y = -x + 4 + 2$, 즉 $y = -x + 6$이므로
$x = -x + 6$, $x = 3$
$\therefore P(3, 3)$

점 $Q$의 좌표는 $Q(-1, \frac{1}{2})$이다.
직선 $AP$와 직선 $BQ$의 기울기는 모두 $-1$이므로 두 직선은 평행하므로 사각형 $APQB$는 사다리꼴이다.

선분 $AP$의 길이는
$\sqrt{(4 - 3)^{2} + (2 - 3)^{2}} = \sqrt{1^{2} + (-1)^{2}} = \sqrt{2}$

선분 $BQ$의 길이는
$\sqrt{(\frac{1}{2} - (-1))^{2} + (-1 - \frac{1}{2})^{2}} = \sqrt{(\frac{3}{2})^{2} + (-\frac{3}{2})^{2}} = \frac{3\sqrt{2}}{2}$

사다리꼴의 높이는 평행한 두 직선 $AP$와 $BQ$ 사이의 거리와 같다.
직선 $AP$의 방정식은 $y = -x + 6$, $x + y - 6 = 0$
직선 $BQ$의 방정식은 $y = -x - \frac{1}{2}$, $x + y + \frac{1}{2} = 0$
두 직선 사이의 거리를 $h$라 하면 직선 $x + y - 6 = 0$ 위의 한 점 $A(4, 2)$와 직선 $x + y + \frac{1}{2} = 0$ 사이의 거리와 같으므로
$h = \frac{|4 + 2 + \frac{1}{2}|}{\sqrt{1^{2} + 1^{2}}} = \frac{|\frac{13}{2}|}{\sqrt{2}} = \frac{13}{2\sqrt{2}}$

사각형 $APQB$의 넓이는
$\frac{1}{2} \times (\overline{AP} + \overline{BQ}) \times h$
$= \frac{1}{2} \times (\sqrt{2} + \frac{3\sqrt{2}}{2}) \times \frac{13}{2\sqrt{2}}$
$= \frac{1}{2} \times \frac{5\sqrt{2}}{2} \times \frac{13}{2\sqrt{2}}$
$= \frac{65}{8}$

따라서 $p = 8$, $q = 65$이므로
$p + q = 8 + 65 = 73$

정답: $73$
