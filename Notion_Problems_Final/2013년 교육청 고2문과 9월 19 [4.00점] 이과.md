## 문제
자연수 $m$에 대하여 $\log m$의 정수부분과 $\log 10 m^{2}$의 정수부분의 차가 $3$이 되도록 하는 자연수 $m$의 최댓값과 최솟값의 합은? (단, $3.162 < 10^{\frac{1}{2}} < 3.163$)

## 보기
① $340$   ② $342$   ③ $344$   ④ $346$   ⑤ $348$

## 풀이
$\log m = n + \alpha$ ($n$은 정수, $0 \le \alpha < 1$)라 하면
$\log 10 m^{2} = 1 + 2 \log m = 1 + 2(n + \alpha) = 2 n + 1 + 2 \alpha$

i) $0 \le \alpha < \frac{1}{2}$일 때,
$0 \le 2 \alpha < 1$이므로 $\log 10 m^{2}$의 정수부분은 $2 n + 1$이다.
두 정수부분의 차가 $3$이므로
$(2 n + 1) - n = 3$
$n + 1 = 3$
$\therefore n = 2$
$n = 2$이고 $0 \le \alpha < \frac{1}{2}$이므로
$2 \le \log m < 2 + \frac{1}{2}$
$10^{2} \le m < 10^{2.5}$
$100 \le m < 100 \times 10^{\frac{1}{2}}$
$100 \le m < 100 \times 3.162...$
$100 \le m < 316.2...$
따라서 만족하는 자연수 $m$은 $100, 101, \dots, 316$이다.

ii) $\frac{1}{2} \le \alpha < 1$일 때,
$1 \le 2 \alpha < 2$이므로 $\log 10 m^{2}$의 정수부분은 $2 n + 2$이다.
두 정수부분의 차가 $3$이므로
$(2 n + 2) - n = 3$
$n + 2 = 3$
$\therefore n = 1$
$n = 1$이고 $\frac{1}{2} \le \alpha < 1$이므로
$1 + \frac{1}{2} \le \log m < 1 + 1$
$1.5 \le \log m < 2$
$10^{1.5} \le m < 10^{2}$
$10 \times 10^{\frac{1}{2}} \le m < 100$
$31.62... \le m < 100$
따라서 만족하는 자연수 $m$은 $32, 33, \dots, 99$이다.

i), ii)에 의해 자연수 $m$의 최댓값은 $316$, 최솟값은 $32$이다.
따라서 최댓값과 최솟값의 합은
$316 + 32 = 348$

정답: ⑤
