## 문제
두 집합 $A = \{ x \mid x^{2} - x - 12 = 0 \}$, $B = \{ a, b \}$에 대하여 $A = B$일 때, $a^{2} + b^{2}$의 값을 구하시오.

## 보기

## 풀이
이차방정식 $x^{2} - x - 12 = 0$을 풀면
$$ (x + 3)(x - 4) = 0 $$
$$ x = -3 \text{ 또는 } x = 4 $$
따라서 집합 $A = \{ -3, 4 \}$이다.
$A = B$일 때, 집합 $B = \{ a, b \}$의 원소는 $-3$과 $4$이므로
$$ a^{2} + b^{2} = (-3)^{2} + 4^{2} = 25 $$
정답: $25$
