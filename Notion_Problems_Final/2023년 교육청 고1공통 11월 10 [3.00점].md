## 문제
다항식 $(x^{2} + 4)^{2} - 3x(x^{2} + 4) - 4x^{2}$ 이 $(x + a)^{2}(x^{2} + bx + c)$ 로 인수분해될 때, 세 정수 $a$, $b$, $c$ 에 대하여 $a + b + c$ 의 값은?

## 보기
① $3$   ② $5$   ③ $7$   ④ $9$   ⑤ $11$

## 풀이
$x^{2} + 4 = t$ 라 하면 
$t^{2} - 3xt - 4x^{2}$ 
$= (t - 4x)(t + x)$ 
$= (x^{2} - 4x + 4)(x^{2} + x + 4)$ 
$= (x - 2)^{2}(x^{2} + x + 4)$ 
에서 
$a = -2$, $b = 1$, $c = 4$ 
따라서 
$a + b + c = 3$

정답: ①
