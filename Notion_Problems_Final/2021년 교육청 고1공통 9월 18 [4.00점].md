## 문제

그림과 같이 좌표평면 위에 두 점 $A(0, 1)$, $B(1, 0)$ 이 있다. 양수 $n$ 과 원점 $O$ 에 대하여 선분 $OA$ 를 $1:n$ 으로 내분하는 점을 $P$, 선분 $OB$ 를 $1:n$ 으로 내분하는 점을 $Q$, 선분 $AQ$ 와 선분 $BP$ 가 만나는 점을 $R$ 라 하자. 다음은 사각형 $POQR$ 의 넓이가 $\frac{1}{42}$ 일 때, $n$ 의 값을 구하는 과정이다.

$$ \begin{array}{|l|} \hline \text{점 } P \text{의 좌표는 } (0, \frac{1}{n+1}), \text{ 점 } Q \text{의 좌표는 } (\frac{1}{n+1}, 0) \text{ 이다.} \\ \text{직선 } AQ \text{의 방정식은 } y = -(n+1)x+1, \\ \text{직선 } BP \text{의 방정식은 } y = (가) \times x + \frac{1}{n+1} \text{ 이다.} \\ \text{두 직선 } AQ, BP \text{가 만나는 점 } R \text{의 } x \text{ 좌표는 } (나) \text{ 이고} \\ \text{삼각형 } POR \text{의 넓이는 } \frac{1}{2} \times \frac{1}{n+1} \times (나) \text{ 이다.} \\ \text{두 삼각형 } POR \text{과 삼각형 } QOR \text{에서 선분 } OR \text{가 공통이고} \\ \overline{OP} = \overline{OQ}, \angle POR = \angle QOR \text{ 이므로 삼각형 } POR \text{과 삼각형 } QOR \text{는 합동이다.} \\ \text{따라서 사각형 } POQR \text{의 넓이는 삼각형 } POR \text{의 넓이의 } 2 \text{ 배이므로 } n = (다) \text{ 이다.} \\ \hline \end{array} $$

위의 (가), (나)에 알맞은 식을 각각 $f(n)$, $g(n)$ 이라 하고, (다)에 알맞은 수를 $k$ 라 할 때, $\frac{g(k)}{f(k)}$ 의 값은?



> [!example] 📸 (그림/그래프)

## 보기

① $-\frac{5}{7}$   ② $-\frac{6}{7}$   ③ $-1$   ④ $-\frac{8}{7}$   ⑤ $-\frac{9}{7}$

## 풀이

점 $P$ 의 좌표는 $(0, \frac{1}{n+1})$, 점 $Q$ 의 좌표는 $(\frac{1}{n+1}, 0)$ 이다.
직선 $AQ$ 의 방정식은
$y = -(n+1)x+1$ $\cdots \cdots$ ㉠
직선 $BP$ 의 방정식은
$y = -\frac{1}{n+1} \times x + \frac{1}{n+1}$ $\cdots \cdots$ ㉡
㉠, ㉡에서 점 $R$ 의 $x$ 좌표는 $\frac{1}{n+2}$ 이고, 삼각형 $POR$ 의 넓이는 $\frac{1}{2} \times \frac{1}{n+1} \times \frac{1}{n+2}$ 이다.
두 삼각형 $POR$ 과 삼각형 $QOR$ 는 합동이고, 사각형 $POQR$ 의 넓이는 삼각형 $POR$ 의 넓이의 $2$ 배이므로
$2 \times \frac{1}{2} \times \frac{1}{n+1} \times \frac{1}{n+2} = \frac{1}{42}$ 에서 $n = 5$ 이다.
그러므로
$f(n) = -\frac{1}{n+1}$, $g(n) = \frac{1}{n+2}$, $k = 5$
따라서
$\frac{g(5)}{f(5)} = \frac{\frac{1}{7}}{-\frac{1}{6}} = -\frac{6}{7}$

정답: ②
