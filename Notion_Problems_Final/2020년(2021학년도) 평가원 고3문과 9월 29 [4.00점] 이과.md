## 문제
흰 공 $4$개와 검은 공 $6$개를 세 상자 $A$, $B$, $C$에 남김없이 나누어 넣을 때, 각 상자에 공이 $2$개 이상씩 들어가도록 나누어 넣는 경우의 수를 구하시오.

## 풀이
세 상자에 서로 같은 흰 공 $4$개를 나누어 넣는 경우는 다음과 같다.
$$(4, 0, 0), (3, 1, 0), (2, 2, 0), (2, 1, 1)$$

(ⅰ) 흰 공을 $(4, 0, 0)$으로 나누어 넣는 경우
흰 공 $4$개를 넣을 상자를 선택한 후 검은 공 $6$개를 나누어 넣는 경우는 $x+y+z=6$에서 흰 공이 들어가지 않은 상자에 $2$개 이상씩 넣어야 하므로
$y = y' + 2, z = z' + 2$ (단, $y', z' \ge 0$)
$x + y' + z' = 2$
${}_{3}H_{2} = {}_{4}C_{2} = 6$이므로 구하는 경우의 수는
${}_{3}C_{1} \times {}_{3}H_{2} = 3 \times 6 = 18$

(ⅱ) 흰 공을 $(3, 1, 0)$으로 나누어 넣는 경우
상자 $A, B, C$에 나누어 넣고, 검은 공 $6$개를 나누어 넣는 경우는 $x+y+z=6$에서 흰 공 $1$개 넣은 상자에 $1$개 이상, 흰 공 $0$개 넣은 상자에 $2$개 이상 넣는 경우이므로
$y = y' + 1, z = z' + 2$ (단, $y', z' \ge 0$)
$x + y' + z' = 3$
${}_{3}H_{3} = {}_{5}C_{2} = 10$이므로 구하는 경우의 수는
$3! \times {}_{3}H_{3} = 6 \times 10 = 60$

(ⅲ) 흰 공을 $(2, 2, 0)$으로 나누어 넣는 경우
흰 공 $0$개인 상자를 선택한 후 검은 공 $6$개를 나누어 넣는 경우는 $x+y+z=6$에서 흰 공이 안 들어간 상자에 $2$개 이상 넣는 경우이므로
$z = z' + 2$ (단, $z' \ge 0$)
$x + y + z' = 4$
${}_{3}H_{4} = {}_{6}C_{2} = 15$이므로 구하는 경우의 수는
${}_{3}C_{1} \times {}_{3}H_{4} = 3 \times 15 = 45$

(ⅳ) 흰 공을 $(2, 1, 1)$으로 나누어 넣는 경우
흰 공 $2$개 들어가는 상자를 선택한 후 검은 공 $6$개를 나누어 넣는 경우는 $x+y+z=6$에서 흰 공 $1$개씩 들어간 상자에 $1$개 이상 넣는 경우이므로
$y = y' + 1, z = z' + 1$ (단, $y', z' \ge 0$)
$x + y' + z' = 4$
${}_{3}H_{4} = {}_{6}C_{2} = 15$이므로 구하는 경우의 수는
${}_{3}C_{1} \times {}_{3}H_{4} = 3 \times 15 = 45$

(ⅰ)∼(ⅳ)에 의해서
$18 + 60 + 45 + 45 = 168$

정답: $168$
