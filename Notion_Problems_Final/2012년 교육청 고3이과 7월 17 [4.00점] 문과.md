## 문제
수열 $\{ a_{n} \}$ 을 $a_{1}=1$, $a_{2}=2$, $a_{n+2}=a_{n+1}+\frac{a_{n}}{n+1}$ 으로 정의할 때, 다음은 수열 $\{ a_{n} \}$ 의 일반항을 구하는 과정이다.

$$ \begin{array}{|l|} \hline \\ b_{n}=\frac{a_{n}}{n+1} \text{ 이라 놓으면 } a_{n}=(n+1) b_{n} \text{ 이므로} \\ (n+3) b_{n+2} = ( (가) )b_{n+1} +b_{n} \\ (n+3) (b_{n+2}-b_{n+1} ) = - (b_{n+1}-b_{n} ) \cdots \cdots (\star) \\ \text{식 } (\star) \text{ 에 } n=1, 2, \cdots, m-1 (m \geq 2) \text{ 를 대입하면} \\ 4 (b_{3}-b_{2} ) = - (b_{2}-b_{1} ) \\ 5 (b_{4}-b_{3} ) = - (b_{3}-b_{2} ) \\ \vdots \\ (m+2) (b_{m+1}-b_{m} ) = - (b_{m}-b_{m-1} ) \\ \text{좌변과 우변을 각각 곱하여 정리하면,} \\ b_{m+1}-b_{m}= (-\frac{1}{4} ) (-\frac{1}{5} ) \cdots (-\frac{1}{m+2} ) (b_{2}-b_{1} ) \\ b_{n}=b_{1}+ \sum_{k=1}^{n-1} (b_{k+1}-b_{k} ) (n \geq 2) \\ \text{따라서 } a_{1}=1, a_{n}=(n+1) (\frac{1}{2}+ \sum_{k=1}^{n-1} (나) ) (n \geq 2) \text{ 이다.} \\ \hline \end{array} $$

위의 $(가)$, $(나)$ 에 들어갈 식을 각각 $f(n)$, $g(k)$ 라 할 때, $f(1) g(3)$ 의 값은?

## 보기
① $\frac{1}{240}$   ② $\frac{1}{180}$   ③ $\frac{1}{40}$   ④ $\frac{1}{30}$   ⑤ $\frac{1}{24}$

## 풀이
$b_{n}=\frac{a_{n}}{n+1}$ 이라 놓으면 $a_{n}=(n+1) b_{n}$ 이므로
$(n+3) b_{n+2}=(n+2) b_{n+1}+b_{n}$ 에서 $(가)$ 에 알맞은 식은 $f(n)=n+2$ 이다.
$(n+3) (b_{n+2}-b_{n+1} )=-(b_{n+1}-b_{n} ) \cdots \cdots (\star)$
식 $(\star)$ 에 $n=1, 2, \cdots, m-1 (m \geq 2)$ 를 대입하면
$4(b_{3}-b_{2} )=-(b_{2}-b_{1} )$
$5(b_{4}-b_{3} )=-(b_{3}-b_{2} )$
$\vdots$
$(m+2) (b_{m+1}-b_{m} )=- (b_{m}-b_{m-1} )$
좌변과 우변을 각각 곱하여 정리하면,
$b_{m+1}-b_{m}= (-\frac{1}{4} ) (-\frac{1}{5} ) \cdots (-\frac{1}{m+2} ) (b_{2}-b_{1} )$
이때 $a_{1}=1, a_{2}=2$ 이므로 $b_{1}=\frac{1}{2}, b_{2}=\frac{2}{3}$ 이고 $b_{2}-b_{1}=\frac{1}{6}$ 이다.
$b_{m+1}-b_{m} = \frac{(-1)^{m-1} \cdot 3!}{(m+2)!} \cdot \frac{1}{6} = \frac{(-1)^{m-1}}{(m+2)!}$
따라서 $b_{k+1}-b_{k} = \frac{(-1)^{k-1}}{(k+2)!}$ 이므로 $(나)$ 에 알맞은 식은 $g(k)=\frac{(-1)^{k-1}}{(k+2)!}$ 이다.
$b_{n}=b_{1}+ \sum_{k=1}^{n-1} (b_{k+1}-b_{k} ) (n \geq 2)$
$\therefore a_{n}=(n+1) (\frac{1}{2}+ \sum_{k=1}^{n-1} \frac{(-1)^{k-1}}{(k+2)!} ) (n \geq 2)$
$f(1)=1+2=3$
$g(3)=\frac{(-1)^{3-1}}{(3+2)!} = \frac{1}{120}$
$\therefore f(1) g(3)=3 \times \frac{1}{120} = \frac{1}{40}$

정답은 ③입니다.
