## 문제
두 실수 $a$, $b$에 대하여 이차방정식 $x^{2} + ax + b = 0$의 한 근이 $2 + 3i$일 때, $a^{2} + b^{2}$의 값을 구하시오. (단, $i = \sqrt{-1}$)

## 보기

## 풀이
이차방정식 $x^{2} + ax + b = 0$의 한 근이 $2 + 3i$이므로
$(2 + 3i)^{2} + a(2 + 3i) + b = (2a + b - 5) + (3a + 12)i = 0$
이때 $a$, $b$는 실수이므로
$2a + b - 5 = 0$, $3a + 12 = 0$
$a = -4$, $b = 13$
따라서 $a^{2} + b^{2} = (-4)^{2} + 13^{2} = 16 + 169 = 185$

정답: $185$
