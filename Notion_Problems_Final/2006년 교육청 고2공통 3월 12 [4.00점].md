## 문제
연립방정식 $\begin{cases} x^{2} - 4xy + 3y^{2} = 0 \\ 2x^{2} + xy + 3y^{2} = 24 \end{cases}$ 의 해를 $\begin{cases} x = \alpha_{i} \\ y = \beta_{i} \end{cases}$ ($i = 1, 2, 3, 4$)라 할 때, $\alpha_{i} \beta_{i}$ 의 최댓값은?

## 보기
① $9$   ② $8$   ③ $6$   ④ $4$   ⑤ $3$

## 풀이
$\begin{cases} x^{2} - 4xy + 3y^{2} = 0 & \cdots ㉠ \\ 2x^{2} + xy + 3y^{2} = 24 & \cdots ㉡ \end{cases}$

㉠ 에서
$x^{2} - 4xy + 3y^{2} = 0 \iff (x - y)(x - 3y) = 0$
$\therefore x = y$ 또는 $x = 3y$

$i$) $x = y$ 를 ㉡ 에 대입하면
$2y^{2} + y^{2} + 3y^{2} = 24$
$6y^{2} = 24$
$\therefore x = \pm 2$, $y = \pm 2$ (복부호동순)

$ii$) $x = 3y$ 를 ㉡ 에 대입하면
$2(3y)^{2} + (3y)y + 3y^{2} = 24$
$18y^{2} + 3y^{2} + 3y^{2} = 24$
$24y^{2} = 24$
$\therefore x = \pm 3$, $y = \pm 1$ (복부호동순)

따라서 주어진 연립방정식의 해의 순서쌍 $(\alpha_{i}, \beta_{i})$ 는
$(2, 2)$, $(-2, -2)$, $(3, 1)$, $(-3, -1)$ 이므로
$\alpha_{i} \beta_{i}$ 의 최댓값은 $x = 2$, $y = 2$ 또는 $x = -2$, $y = -2$ 일 때 $4$ 이다.
