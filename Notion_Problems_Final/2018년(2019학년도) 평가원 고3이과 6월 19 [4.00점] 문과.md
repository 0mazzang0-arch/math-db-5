## 문제
$0$이 아닌 실수 $p$에 대하여 좌표평면 위의 두 포물선 $x^2 = 2y$와 $( y + \frac{1}{2} )^2 = 4px$에 동시에 접하는 직선의 개수를 $f(p)$라 하자. $\lim_{p \to k^+} f(p) > f(k)$를 만족시키는 실수 $k$의 값은?



> [!example] 📸 (그림/그래프)

## 보기
① $-\frac{\sqrt{3}}{3}$ ② $-\frac{2\sqrt{3}}{9}$ ③ $-\frac{\sqrt{3}}{9}$ ④ $\frac{2\sqrt{3}}{9}$ ⑤ $\frac{\sqrt{3}}{3}$

## 풀이
[그림1]과 같이 두 포물선이 두 점에서 만나면 두 포물선에 동시에 접하는 직선의 개수는 $1$, 
[그림2]와 같이 두 포물선이 한 점에서 만나면 두 포물선에 동시에 접하는 직선의 개수는 $2$, 
[그림3]과 같이 두 포물선이 만나지 않으면 두 포물선에 동시에 접하는 직선의 개수는 $3$이다. 
두 포물선이 한 점에서만 만날 때, 교점의 좌표 $(a, b)$를 구하자. 

(i) 점 $(a, b)$가 두 곡선 위의 점이므로 
$a^2 = 2b$ 이고 $(b + \frac{1}{2})^2 = 4pa$ 
따라서 $(\frac{a^2}{2} + \frac{1}{2})^2 = 4pa$ 이므로 
$(a^2 + 1)^2 = 16pa$ $\cdots$ ㉠ 

(ii) 두 곡선 위의 점 $(a, b)$에서의 접선의 기울기가 서로 같아야 한다. 
$x^2 = 2y$, 즉 $y = \frac{x^2}{2}$에서 $\frac{dy}{dx} = x$ $\cdots$ ㉡ 

$(y + \frac{1}{2})^2 = 4px$의 양변을 $x$로 미분하면 
$2(y + \frac{1}{2}) \frac{dy}{dx} = 4p$ 
$\frac{dy}{dx} = \frac{4p}{2y + 1}$ $\cdots$ ㉢ 

$x = a$, $y = b$일 때의 ㉡, ㉢의 값이 서로 같아야 하므로 
$a = \frac{4p}{2b + 1}$ 이때 $2b = a^2$이므로 $4p = a(a^2 + 1)$ $\cdots$ ㉣ 

㉠, ㉣에서 $(a^2 + 1)^2 = 4a^2(a^2 + 1)$ 
이때 $a^2 + 1 > 0$이므로 $a^2 + 1 = 4a^2$ 
$a = \pm \frac{\sqrt{3}}{3}$ 

㉣에서 $a = \frac{\sqrt{3}}{3}$ 일 때, $p = \frac{\sqrt{3}}{9}$, 
$a = -\frac{\sqrt{3}}{3}$ 일 때, $p = -\frac{\sqrt{3}}{9}$ 
이므로 함수 $f(p) (p \neq 0)$는 다음과 같다. 
$p < -\frac{\sqrt{3}}{9}$ 이면 $f(p) = 1$, 
$p = -\frac{\sqrt{3}}{9}$ 이면 $f(p) = 2$, 
$-\frac{\sqrt{3}}{9} < p < 0$ 이면 $f(p) = 3$, 
$0 < p < \frac{\sqrt{3}}{9}$ 이면 $f(p) = 3$, 
$p = \frac{\sqrt{3}}{9}$ 이면 $f(p) = 2$, 
$p > \frac{\sqrt{3}}{9}$ 이면 $f(p) = 1$ 

따라서 $\lim_{p \to k^+} f(p) > f(k)$ 를 만족시키는 실수 $k$ 의 값은 $-\frac{\sqrt{3}}{9}$ 이다.
