## 문제
$x$에 대한 연립부등식 
$$ \begin{cases} x^{2} - a^{2} x \ge 0 \\ x^{2} - 4ax + 4a^{2} - 1 < 0 \end{cases} $$ 
을 만족시키는 정수 $x$의 개수가 $1$이 되기 위한 모든 실수 $a$ ($0 < a < \sqrt{2}$)의 값의 합은?

## 보기
① $\frac{3}{2}$   ② $\frac{25}{16}$   ③ $\frac{13}{8}$   ④ $\frac{27}{16}$   ⑤ $\frac{7}{4}$

## 풀이
[정답] ① 

$x^{2} - a^{2} x = x ( x - a^{2} ) \ge 0$에서 $x \le 0$ 또는 $x \ge a^{2}$이고, 
$x^{2} - 4ax + 4a^{2} - 1 = ( x - ( 2a - 1 ) ) ( x - ( 2a + 1 ) ) < 0$에서 $2a - 1 < x < 2a + 1$이다. 

(ⅰ) $0 < a < \frac{1}{2}$일 때, 연립부등식의 해는 
$-1 < 2a - 1 < x \le 0$ 또는 $a^{2} \le x < 2a + 1 < 2$이다.
$0 < a^{2} < \frac{1}{4}$이고 $1 < 2a + 1 < 2$이므로 $x = 0, 1$의 $2$개 정수해가 존재한다. 

> [!example] 📸 (그림/그래프)

(ⅱ) $a = \frac{1}{2}$일 때, 연립부등식의 해는 
$\frac{1}{4} = a^{2} \le x < 2a + 1 = 2$이다.
$x = 1$의 $1$개 정수해가 존재한다. 

(ⅲ) $\frac{1}{2} < a < 1$일 때, 연립부등식의 해는 
$a^{2} \le x < 2a + 1$이다.
$\frac{1}{4} < a^{2} < 1$이고 $2 < 2a + 1 < 3$이므로 $x = 1, 2$의 $2$개 정수해가 존재한다. 

> [!example] 📸 (그림/그래프)

(ⅳ) $a = 1$일 때, 연립부등식의 해는 
$1 = a^{2} = 2a - 1 < x < 2a + 1 = 3$이다.
$x = 2$의 $1$개 정수해가 존재한다. 

(ⅴ) $1 < a < \sqrt{2}$일 때, 연립부등식의 해는 
$a^{2} \le x < 2a + 1$이다.
$1 < a^{2} < 2$이고 $3 < 2a + 1 < 1 + 2\sqrt{2} < 4$이므로 $x = 2, 3$의 $2$개 정수해가 존재한다. 

> [!example] 📸 (그림/그래프)

그러므로 (ⅰ)∼(ⅴ)에 의해 $a = \frac{1}{2}$ 또는 $a = 1$일 때, $1$개 정수해가 존재한다. 
따라서 모든 실수 $a$의 값의 합은 $\frac{3}{2}$이다.
