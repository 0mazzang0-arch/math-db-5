## 문제
$sin ( \frac{\pi}{2} + \theta ) = \frac{3}{5}$ 이고 $sin \theta cos \theta < 0$ 일 때, $sin \theta + 2cos \theta$ 의 값은?

## 보기
① $-\frac{2}{5}$   ② $-\frac{1}{5}$   ③ $0$   ④ $\frac{1}{5}$   ⑤ $\frac{2}{5}$

## 풀이
$sin ( \frac{\pi}{2} + \theta ) = cos \theta = \frac{3}{5}$
$sin^{2} \theta = 1 - cos^{2} \theta = 1 - ( \frac{3}{5} )^{2} = \frac{16}{25}$
$sin \theta cos \theta < 0$ 이고 $cos \theta = \frac{3}{5} > 0$ 이므로 $sin \theta < 0$ 이다.
따라서 $sin \theta = -\frac{4}{5}$
그러므로 $sin \theta + 2cos \theta = ( -\frac{4}{5} ) + 2 ( \frac{3}{5} ) = -\frac{4}{5} + \frac{6}{5} = \frac{2}{5}$

정답: ⑤
