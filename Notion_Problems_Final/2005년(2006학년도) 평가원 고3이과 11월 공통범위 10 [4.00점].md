## 문제
좌표공간에서 $xy$ 평면, $yz$ 평면, $zx$ 평면은 공간을 $8$ 개의 부분으로 나눈다. 이 $8$ 개의 부분 중에서 구 $(x+2)^2 +(y-3)^2 +(z-4)^2 = 24$ 가 지나는 부분의 개수는?



> [!example] 📸 (그림/그래프)

## 보기
① $8$   ② $7$   ③ $6$   ④ $5$   ⑤ $4$

## 풀이
구의 중심이 $(-2, 3, 4)$ 이고 반지름 $r = \sqrt{24}$ 인 구이다.
$xy$ 평면, $yz$ 평면, $zx$ 평면까지의 거리를 각각 $d_{xy}$, $d_{yz}$, $d_{zx}$ 라 하면
$d_{xy} = 4 < r$
$d_{yz} = 2 < r$
$d_{zx} = 3 < r$
이므로 중심이 있는 영역의 바로 맞은편 영역을 지난다.
또한 $x$ 축, $y$ 축, $z$ 축과의 거리를 각각 $d_x$, $d_y$, $d_z$ 라 하면
$d_x = \sqrt{3^2 + 4^2} = 5 > r$
$d_y = \sqrt{(-2)^2 + 4^2} = \sqrt{20} < r$
$d_z = \sqrt{(-2)^2 + 3^2} = \sqrt{13} < r$
이므로 $x$ 축의 맞은편 영역은 지나지 않고 $y$ 축, $z$ 축의 맞은편 영역은 지난다.
따라서 지나는 영역은 $6$ 개이다.

**[다른 풀이]**
$(x+2)^2 + (y-3)^2 + (z-4)^2 = 24$ 에서 $y \le 0$, $z \le 0$ 을 동시에 만족하는 경우에는 $x$ 의 양, 음에 관계없이
(좌변) $\ge 3^2 + 4^2 = 25$
(우변) $= 24$
이므로 조건을 만족하는 점이 존재하지 않는다.
$\therefore$ $x \ge 0, y \le 0, z \le 0$ 인 영역과 $x < 0, y \le 0, z \le 0$ 인 영역은 지나지 않는다.
따라서 지나는 영역의 개수는 $8 - 2 = 6$ 개이다.
