## 문제

아래 증명은 조건 '$n$은 $\sqrt{n}$보다 작은 모든 자연수의 배수이다.'가 참이 되는 자연수 $n$ 중에서 가장 큰 수는 $24$임을 증명한 내용의 일부이다.

$$ \begin{array}{|l|} \hline \text{먼저 } 24 \text{가 } \sqrt{24} \text{보다 작은 모든 자연수의 배수임을 보이자.} \\ k < \sqrt{24} \text{인 자연수는 } k=1, 2, 3, 4 \text{이므로 } 24 \text{는 } k \text{의 배수이다.} \\ \text{이제 } 24 \text{보다 큰 자연수 } n \text{은 } \sqrt{n} \text{보다 작은 어떤 자연수의 배수가 되지 않음을} \\ \text{귀류법을 사용하여 보이자.} \\ \\ n \geq \text{(가)} \text{인 자연수 } n \text{이 } \sqrt{n} \text{보다 작은 모든 자연수의 배수라고 가정하자.} \\ n \text{은 } 1, 2, 3, 4 \text{의 배수이므로 } n \text{은 } 12 \text{의 배수이다.} \\ \text{따라서 } n \geq \text{(나)} \text{이고 } \sqrt{n} > 5 \text{이다.} \\ \text{그러므로 } n \text{은 } 5 \text{의 배수이고, } n \text{은 } 60 \text{의 배수이다.} \\ \text{따라서 } \sqrt{n} > 7 \text{이고 } n \text{은 (다)의 배수임을 알 수 있다.} \\ \dots \text{(생략)} \\ \hline \end{array} $$

위의 증명에서 (가), (나), (다)에 알맞은 것은?

## 보기

① $(가): 25, (나): 36, (다): 420$   ② $(가): 25, (나): 48, (다): 240$   ③ $(가): 25, (나): 36, (다): 240$   ④ $(가): 24, (나): 36, (다): 420$   ⑤ $(가): 24, (나): 48, (다): 240$

## 풀이

$24$보다 큰 자연수 $n$이 $\sqrt{n}$보다 작은 어떤 자연수의 배수라고 가정하자. 
즉, $n \geq 25$인 자연수 $n$이 $\sqrt{n}$보다 작은 모든 자연수의 배수라고 가정하면 $n$은 $1, 2, 3, 4$의 배수이므로 $n$은 $12$의 배수이다. 그러므로 $(가) = 25$이다.

따라서, $25$ 이상의 $12$의 배수 중 가장 작은 수는 $36$이므로 $n \geq 36$이고, $\sqrt{n} \geq \sqrt{36} = 6 > 5$이다. 그러므로 $(나) = 36$이다.

$n$은 $\sqrt{n}$보다 작은 $5$의 배수이고, 이미 $12$의 배수이므로 $n$은 $5$와 $12$의 최소공배수인 $60$의 배수이다.
따라서 $\sqrt{n} \geq \sqrt{60} > \sqrt{49} = 7$이고, $n$은 $7$보다 작은 모든 자연수의 배수이므로 $1, 2, 3, 4, 5, 6, 7$의 배수여야 한다.
$n$은 $60$의 배수이면서 $7$의 배수이므로 $n$은 $420$의 배수임을 알 수 있다. 그러므로 $(다) = 420$이다.

따라서 $(가) : 25, (나) : 36, (다) : 420$이다.
