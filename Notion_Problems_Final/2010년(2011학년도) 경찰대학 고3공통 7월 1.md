## 문제
$a = \log_{9}(7 - 4\sqrt{3})$ 일 때, $3^{a} + 3^{-a}$ 의 값은?

## 보기
① $4$   ② $\frac{10}{3}$   ③ $\frac{5}{2}$   ④ $2$   ⑤ $\frac{3}{2}$

## 풀이
$a = \log_{9}(7 - 4\sqrt{3})$ 이므로
$9^{a} = 3^{2a} = (3^{a})^{2} = 7 - 4\sqrt{3}$
$\therefore 3^{a} = \sqrt{7 - 4\sqrt{3}} = \sqrt{7 - 2\sqrt{12}} = \sqrt{(2 - \sqrt{3})^{2}} = 2 - \sqrt{3}$
$3^{-a} = \frac{1}{2 - \sqrt{3}} = 2 + \sqrt{3}$
$\therefore 3^{a} + 3^{-a} = 4$

[다른 풀이]
$9^{a} = 7 - 4\sqrt{3}$,
$9^{-a} = \frac{1}{7 - 4\sqrt{3}} = 7 + 4\sqrt{3}$ 이므로
$(3^{a} + 3^{-a})^{2} = 9^{a} + 2 + 9^{-a} = (7 - 4\sqrt{3}) + 2 + (7 + 4\sqrt{3}) = 16$
$\therefore 3^{a} + 3^{-a} = 4$ ($\because 3^{a} > 0$)

정답: ①
