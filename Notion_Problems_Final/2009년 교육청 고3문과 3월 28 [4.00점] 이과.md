## 문제
수열 $\{ $ $a_{n}$ $ \}$이 $a_{1} = 1$, $p a_{n+1} = q a_{n} + 1$ ($n = 1, 2, 3, \dots$)일 때, <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $p, q$는 $0$이 아닌 실수이다.)

$$ \begin{array}{|l|} \hline \text{<보 기>} \\ \text{ㄱ. } p = q \text{ 일 때, 수열 } \{ a_{n} \} \text{ 은 등차수열이다.} \\ \text{ㄴ. } p \neq q \text{ 일 때, 수열 } \{ a_{n} - \frac{1}{p-q} \} \text{ 은 등비수열이다.} \\ \text{ㄷ. } -1 < \frac{q}{p} < 1 \text{ 일 때, 수열 } \{ a_{n} \} \text{ 은 수렴한다.} \\ \hline \end{array} $$

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
ㄱ. $p = q$ 일 때 주어진 식의 양변을 $p$로 나누면 $a_{n+1} = a_{n} + \frac{1}{p}$이다. 
따라서 수열 $\{ a_{n} \}$은 공차가 $\frac{1}{p}$인 등차수열이다. (참)

ㄴ. $a_{n+1} = \frac{q}{p} a_{n} + \frac{1}{p}$이므로 $a_{n+1} - \alpha = \frac{q}{p} (a_{n} - \alpha)$라 하면, 
$\alpha = \frac{q}{p} \alpha + \frac{1}{p}$에서 $\alpha ( 1 - \frac{q}{p} ) = \frac{1}{p}$, 즉 $\alpha ( \frac{p-q}{p} ) = \frac{1}{p}$이므로 $\alpha = \frac{1}{p-q}$이다. 
따라서 수열 $\{ a_{n} - \frac{1}{p-q} \}$은 첫째항이 $a_{1} - \frac{1}{p-q}$이고, 공비가 $\frac{q}{p}$인 등비수열이다. (참)

ㄷ. ㄴ에서 $a_{n} = ( \frac{q}{p} )^{n-1} (a_{1} - \alpha) + \alpha$이므로 $-1 < \frac{q}{p} < 1$이면 $\lim_{n \to \infty} a_{n} = \alpha$이다. (참) 

이상에서 옳은 것은 ㄱ, ㄴ, ㄷ이다.
