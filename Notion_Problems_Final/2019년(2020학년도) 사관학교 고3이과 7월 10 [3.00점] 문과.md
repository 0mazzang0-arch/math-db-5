## 문제
함수 $f(x) = \frac{6x^{3}}{x^{2} + 1}$ 의 역함수를 $g(x)$ 라 할 때, $g'(3)$ 의 값은?

## 보기
① $\frac{1}{6}$   ② $\frac{1}{3}$   ③ $\frac{1}{2}$   ④ $\frac{2}{3}$   ⑤ $\frac{5}{6}$

## 풀이
$f(x) = \frac{6x^{3}}{x^{2} + 1}$ 에서
$f'(x) = \frac{18x^{2}(x^{2} + 1) - 6x^{3}(2x)}{(x^{2} + 1)^{2}} = \frac{18x^{4} + 18x^{2} - 12x^{4}}{(x^{2} + 1)^{2}} = \frac{6x^{4} + 18x^{2}}{(x^{2} + 1)^{2}}$
$f(1) = \frac{6 \times 1^{3}}{1^{2} + 1} = \frac{6}{2} = 3$ 이므로 $g(3) = 1$ 이다.
역함수의 미분법에 의하여
$g'(3) = \frac{1}{f'(1)}$
이때 $f'(1) = \frac{6(1)^{4} + 18(1)^{2}}{(1^{2} + 1)^{2}} = \frac{6 + 18}{2^{2}} = \frac{24}{4} = 6$
따라서 $g'(3) = \frac{1}{6}$ 이다.

정답: ①
