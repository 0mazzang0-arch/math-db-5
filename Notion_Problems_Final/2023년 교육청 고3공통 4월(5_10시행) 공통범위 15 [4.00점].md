## 문제
다음 조건을 만족시키는 모든 수열 $\{a_{n}\}$에 대하여 $a_{1}$의 최댓값을 $M$, 최솟값을 $m$이라 할 때, $\log_{2} \frac{M}{m}$의 값은?

(가) 모든 자연수 $n$에 대하여
$$a_{n+1} = \begin{cases} 2^{n-2} & (a_{n} < 1) \\ \log_{2} a_{n} & (a_{n} \geq 1) \end{cases}$$
이다.
(나) $a_{5} + a_{6} = 1$

## 보기
① $12$   ② $13$   ③ $14$   ④ $15$   ⑤ $16$

## 풀이
조건 (가)에서 자연수 $n$에 대하여
$a_{n} < 1$이면 $a_{n+1} = 2^{n-2} > 0$이고,
$a_{n} \geq 1$이면 $a_{n+1} = \log_{2} a_{n} \geq 0$이므로
$2$ 이상의 모든 자연수 $n$에 대하여 $a_{n} \geq 0$이다.

조건 (가), (나)에서 $a_{5}$, $a_{6}$의 값을 구하면
(ⅰ) $0 \leq a_{5} < 1$일 때
$a_{6} = 2^{5-2} = 2^{3} = 8$에서 $a_{5} + a_{6} \geq 8$이므로 $a_{5} + a_{6} = 1$을 만족시키지 않는다.
(ⅱ) $a_{5} \geq 1$일 때
$a_{6} = \log_{2} a_{5} \geq 0$에서 $a_{5} + a_{6} = a_{5} + \log_{2} a_{5} = 1$을 만족시키려면 $a_{5} = 1$, $a_{6} = 0$이다.
그러므로 $a_{5} = 1, a_{6} = 0$

$a_{4}$의 값을 구하면
(ⅰ) $0 \leq a_{4} < 1$일 때
$a_{5} = 2^{4-2} = 4$이므로 $a_{5} = 1$을 만족시키지 않는다.
(ⅱ) $a_{4} \geq 1$일 때
$a_{5} = \log_{2} a_{4} = 1$이므로 $a_{4} = 2$이다.
그러므로 $a_{4} = 2$

$a_{1}, a_{2}, a_{3}$의 값을 구하면
(ⅰ) $0 \leq a_{3} < 1$일 때
$a_{4} = 2^{3-2} = 2$이므로 $0 \leq a_{3} < 1$의 조건을 만족한다.
(a) $0 \leq a_{2} < 1$일 때
$a_{3} = 2^{2-2} = 1$이므로 $0 \leq a_{3} < 1$을 만족시키지 않는다.
(b) $a_{2} \geq 1$일 때
$a_{3} = \log_{2} a_{2}$에서 $0 \leq \log_{2} a_{2} < 1$이므로 $1 \leq a_{2} < 2$이다.
$a_{1} < 1$이면 $a_{2} = 2^{1-2} = \frac{1}{2}$이므로 $1 \leq a_{2} < 2$를 만족시키지 않는다.
$a_{1} \geq 1$이면 $a_{2} = \log_{2} a_{1}$에서 $1 \leq \log_{2} a_{1} < 2$이므로 $2 \leq a_{1} < 4$이다.

(ⅱ) $a_{3} \geq 1$일 때
$a_{4} = \log_{2} a_{3} = 2$에서 $a_{3} = 2^{2} = 4$이다.
(a) $0 \leq a_{2} < 1$일 때
$a_{3} = 2^{2-2} = 1$이므로 $a_{3} = 4$를 만족시키지 않는다.
(b) $a_{2} \geq 1$일 때
$a_{3} = \log_{2} a_{2} = 4$에서 $a_{2} = 2^{4} = 16$이다.
$a_{1} < 1$이면 $a_{2} = 2^{1-2} = \frac{1}{2}$이므로 $a_{2} = 16$을 만족시키지 않는다.
$a_{1} \geq 1$이면 $a_{2} = \log_{2} a_{1} = 16$에서 $a_{1} = 2^{16}$이다.

따라서 $a_{1}$의 값의 범위는 $2 \leq a_{1} < 4$ 또는 $a_{1} = 2^{16}$이므로
최댓값 $M = 2^{16}$, 최솟값 $m = 2$이다.
$\log_{2} \frac{M}{m} = \log_{2} \frac{2^{16}}{2} = \log_{2} 2^{15} = 15$

정답: ④
