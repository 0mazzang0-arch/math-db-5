## 문제
$\omega$는 $x^{2} + x + 1 = 0$의 한 허근이고, $f(x) = x + \frac{1}{x}$라 할 때, $f(\omega) f(\omega^{2}) f(\omega^{2^{2}}) f(\omega^{2^{3}}) \cdots f(\omega^{2^{2016}})$의 값은?

## 보기
① $-1$   ② $1$   ③ $\omega$   ④ $\frac{1}{\omega}$   ⑤ $-\omega - 1$

## 풀이
[정답] ① 

$\omega$는 $x^{2} + x + 1 = 0$의 한 허근이므로 
$\omega^{2} + \omega + 1 = 0 \cdots \cdots$ ㉠ 
㉠의 양변을 $\omega$로 나누어 정리하면 
$\omega + \frac{1}{\omega} = -1$ 
㉠의 양변에 $\omega - 1$을 곱하여 정리하면 
$\omega^{3} = 1$ 
따라서 
$f(\omega) = \omega + \frac{1}{\omega} = -1$ 
$f(\omega^{2}) = \omega^{2} + \frac{1}{\omega^{2}} = \left( \omega + \frac{1}{\omega} \right)^{2} - 2 = -1$ 
$f(\omega^{3}) = \omega^{3} + \frac{1}{\omega^{3}} = 1 + \frac{1}{1} = 2$ 
$f(\omega^{4}) = \omega^{4} + \frac{1}{\omega^{4}} = \omega + \frac{1}{\omega} = f(\omega)$ 
$\vdots$ 
이므로 자연수 $n$에 대하여 $f(\omega^{n})$은 $-1$, $-1$, $2$가 이 순서대로 반복된다. 
따라서 $n$이 $3$의 배수가 아닐 때 $f(\omega^{n}) = -1$이고, 음이 아닌 정수 $k$에 대하여 $2^{k}$은 $3$의 배수가 아니므로 모든 $k$에 대하여 $f(\omega^{2^{k}}) = -1$이다.
그러므로 구하는 값은 다음과 같다.
$f(\omega) f(\omega^{2}) f(\omega^{2^{2}}) f(\omega^{2^{3}}) \cdots f(\omega^{2^{2016}}) = (-1)^{2017} = -1$
