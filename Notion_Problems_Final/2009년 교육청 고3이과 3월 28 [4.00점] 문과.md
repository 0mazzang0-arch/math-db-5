## 문제
두 수열 $\{ a_{n} \}$, $\{ b_{n} \}$에 대하여
$$ a_{n} + b_{n} = 2 + \frac{1}{n} \quad (n = 1, 2, 3, \dots) $$
일 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?

$$ \begin{array}{|l|} \hline \text{ㄱ. } \lim_{n \to \infty} ( a_{n} + b_{n} ) = 2 \\ \text{ㄴ. 수열 } \{ a_{n} \} \text{ 이 수렴하면 수열 } \{ b_{n} \} \text{ 도 수렴한다.} \\ \text{ㄷ. } \sum_{n=1}^{\infty} a_{n} \text{ 이 수렴하면 } \sum_{n=1}^{\infty} b_{n} \text{ 도 수렴한다.} \\ \hline \end{array} $$

## 보기
① ㄱ   ② ㄱ, ㄴ   ③ ㄱ, ㄷ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
ㄱ. $\lim_{n \to \infty} ( a_{n} + b_{n} ) = \lim_{n \to \infty} \left( 2 + \frac{1}{n} \right) = 2$이다. (참)

ㄴ. $\lim_{n \to \infty} a_{n} = \alpha$라고 하면 $b_{n} = 2 + \frac{1}{n} - a_{n}$에서
$\lim_{n \to \infty} b_{n} = \lim_{n \to \infty} \left( 2 + \frac{1}{n} - a_{n} \right) = 2 - \alpha$이다. (참)

ㄷ. $\sum_{n=1}^{\infty} a_{n}$이 수렴하면 $\lim_{n \to \infty} a_{n} = 0$이므로 ㄴ에 의해 $\lim_{n \to \infty} b_{n} = 2$이다.
따라서 $\sum_{n=1}^{\infty} b_{n}$은 발산한다. (거짓)

이상에서 옳은 것은 ㄱ, ㄴ이다.
