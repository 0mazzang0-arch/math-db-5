## 문제
첫째항이 $a$ 이고 공차가 $-4$ 인 등차수열 $\{ a_{n} \}$ 의 첫째항부터 제 $n$ 항까지의 합을 $S_{n}$ 이라 하자. 모든 자연수 $n$ 에 대하여 $S_{n} < 200$ 일 때, 자연수 $a$ 의 최댓값을 구하시오.

## 보기

## 풀이
[정답] $37$

첫째항이 $a$ 이고 공차가 $-4$ 인 등차수열 $\{ a_{n} \}$ 의 첫째항부터 제 $n$ 항까지의 합 $S_{n}$ 은 
$$ S_{n} = \frac{n \{ 2 a - 4 ( n - 1 ) \} }{2} = -2 n^{2} + ( a + 2 ) n $$ 
모든 자연수 $n$ 에 대하여 $S_{n} < 200$ 이므로 
$$ -2 n^{2} + ( a + 2 ) n < 200 $$ 
$$ 2 n^{2} + 200 > ( a + 2 ) n $$ 
$$ 2 n + \frac{200}{n} > a + 2 \cdots \text{㉠} $$ 
이때 $n > 0$ 이므로 산술·기하 평균의 관계에 의하여
$$ 2 n + \frac{200}{n} \geq 2 \sqrt{ 2 n \times \frac{200}{n} } = 2 \sqrt{400} = 40 $$ 

단, 등호는 $n = 10$ 일 때 성립.

따라서 모든 자연수 $n$ 에 대하여 ㉠이 성립하려면 $a + 2 < 40$ 이어야 하므로 자연수 $a$ 의 최댓값은 $37$ 이다. 

[다른 풀이] 
$S_{n} = \frac{ n \{ 2 a - 4 ( n - 1 ) \} }{2} = -2 n^{2} + ( a + 2 ) n$ 에서 
모든 자연수 $n$ 에 대하여 $S_{n} < 200$ 이므로 
$$ -2 n^{2} + ( a + 2 ) n < 200 $$ 
$$ 2 n^{2} - ( a + 2 ) n > -200 $$ 
$f ( n ) = 2 n^{2} - ( a + 2 ) n$ 이라 하면 
$$ f ( n ) = 2 \left( n^{2} - \frac{ a + 2 }{2} n \right) = 2 \left\{ \left( n - \frac{ a + 2 }{4} \right)^{2} - \frac{ ( a + 2 )^{2} }{16} \right\} = 2 \left( n - \frac{ a + 2 }{4} \right)^{2} - \frac{ ( a + 2 )^{2} }{8} $$ 
이때 $a$ 는 자연수이므로 $f ( n )$ 이 최소가 되게 하는 $n$ 은 $\frac{a}{4}$, $\frac{a+1}{4}$, $\frac{a+2}{4}$, $\frac{a+3}{4}$ 중의 하나이다. 따라서 모든 자연수 $n$ 에 대하여 $f ( n ) > -200$ 이 성립하려면 네 부등식 $f \left( \frac{a}{4} \right) > -200$, $f \left( \frac{a+1}{4} \right) > -200$, $f \left( \frac{a+2}{4} \right) > -200$, $f \left( \frac{a+3}{4} \right) > -200$ 이 모두 성립해야 한다. 

$$ \begin{array}{l} f \left( \frac{a}{4} \right) = 2 \times \frac{1}{4} - \frac{ ( a + 2 )^{2} }{8} > -200 \implies ( a + 2 )^{2} < 1604 \quad \cdots \text{㉠} \\ f \left( \frac{a+1}{4} \right) = 2 \times \frac{1}{16} - \frac{ ( a + 2 )^{2} }{8} > -200 \implies ( a + 2 )^{2} < 1601 \quad \cdots \text{㉡} \\ f \left( \frac{a+2}{4} \right) = - \frac{ ( a + 2 )^{2} }{8} > -200 \implies ( a + 2 )^{2} < 1600 \quad \cdots \text{㉢} \\ f \left( \frac{a+3}{4} \right) = 2 \times \frac{1}{16} - \frac{ ( a + 2 )^{2} }{8} > -200 \implies ( a + 2 )^{2} < 1601 \quad \cdots \text{㉣} \end{array} $$ 

㉠, ㉡, ㉢, ㉣이 모두 성립하려면 $( a + 2 )^{2} < 1600$ 이어야 한다. 
$$ \therefore a + 2 < 40 $$ 
따라서 자연수 $a$ 의 최댓값은 $37$ 이다.
