## 문제
함수 $f(x) = x^4 - 6x^3 + 12x^2 - 8x + 1$과 이차함수 $g(x)$는 어떤 실수 $\alpha$에 대하여 다음 조건을 만족시킨다.

(가) $f(\alpha) = g(\alpha)$, $f'(\alpha) = g'(\alpha)$
(나) $f(\alpha + 1) = g(\alpha + 1)$, $f'(\alpha + 1) = g'(\alpha + 1)$

두 곡선 $y = f(x)$와 $y = g(x)$로 둘러싸인 영역의 넓이를 $S_1$, 곡선 $y = g(x)$와 $x$축으로 둘러싸인 영역의 넓이를 $S_2$라 할 때, $\frac{S_2}{S_1}$의 값은?

## 보기
① $20$   ② $25$   ③ $30$   ④ $35$   ⑤ $40$

## 풀이
$g(x) = ax^2 + bx + c$라 하자. 
이 때, (가), (나)에 의하여 
$f(x) - g(x) = (x - \alpha)^2 (x - \alpha - 1)^2$ 
따라서 $S_1 = \frac{1}{30}$이다. 또한 
$f(x) - g(x) = x^4 - 6x^3 + (12 - a)x^2 + (-8 - b)x + 1 - c$ 
$= (x - \alpha)^2 (x - \alpha - 1)^2$ 
이므로 $\alpha = 1$이다. 
따라서 $a = -1$, $b = 4$, $c = -3$이면, 
$S_2 = \frac{4}{3}$이다. 
따라서 $\frac{S_2}{S_1} = 40$이다.
