## 문제

실수 $a$ ($a \neq 0$)와 $2$보다 큰 자연수 $n$에 대하여 집합 $\{x | x \neq 2 \text{인 실수}\}$에서 정의된 함수 $f(x)$를 

$$f(x) = \begin{cases} \frac{ax-an}{x-2}-n & (x < 2 \text{ 또는 } 2 < x < n) \\ -a\sqrt{x-n}-n & (x \ge n) \end{cases}$$

이라 하자. 실수 $t$에 대하여 $x$에 대한 방정식 $|f(x)| = t$의 서로 다른 실근의 개수를 $g(t)$라 할 때, 함수 $g(t)$가 다음 조건을 만족시키도록 하는 모든 $n$의 값의 합을 구하시오.

(가) $g(t)=2$를 만족시키는 실수 $t$의 최솟값은 $0$, 최댓값은 $\frac{3}{2}n$이다.
(나) $g(|f(5)|) \times g(n) = 6$

## 풀이

[정답] $68$

[해설] 
$$f(x) = \begin{cases} \frac{ax-an}{x-2}-n & (x < 2 \text{ 또는 } 2 < x < n) \\ -a\sqrt{x-n}-n & (x \ge n) \end{cases}$$

$$\frac{ax-an}{x-2}-n = \frac{a(x-2)+2a-an}{x-2}-n = \frac{a(2-n)}{x-2}+a-n$$
이므로

$$f(x) = \begin{cases} \frac{a(2-n)}{x-2}+a-n & (x < 2 \text{ 또는 } 2 < x < n) \\ -a\sqrt{x-n}-n & (x \ge n) \end{cases}$$

(ⅰ) $0 < a < n$ 인 경우 

> [!example] 📸 (그림/그래프)

$y = |f(x)|$ 의 그래프는 다음 그림과 같고, $g(0) = 1$ 이므로 조건 (가)를 만족시키지 않는다.

(ⅱ) $a > n$ 인 경우 

> [!example] 📸 (그림/그래프)

$y = |f(x)|$ 의 그래프는 위 그림과 같고, $g(0) = 0$ 이므로 조건 (가)를 만족시키지 않는다.

(ⅲ) $a < 0$ 인 경우 

> [!example] 📸 (그림/그래프)

$y = |f(x)|$ 의 그래프는 다음 그림과 같고, 이때

$$g(t) = \begin{cases} 0 & (t < 0) \\ 2 & (t = 0 \text{ 또는 } n < t \le n-a) \\ 3 & (t = n \text{ 또는 } t > n-a) \\ 4 & (0 < t < n) \end{cases}$$
$\cdots \cdots$ ㉠

$g(t) = 2$ 를 만족시키는 실수 $t$의 최솟값은 $0$, 최댓값은 $n-a$이다. 조건 (가)에서 $n-a = \frac{3}{2}n$ 이므로 $a = -\frac{1}{2}n$ 이고

$$f(x) = \begin{cases} \frac{n^2-2n}{2x-4} - \frac{3}{2}n & (x < 2 \text{ 또는 } 2 < x < n) \\ \frac{1}{2}n\sqrt{x-n}-n & (x \ge n) \end{cases}$$

$g(n) = 3$ 이므로 $g(|f(5)|) = 2$

$f(5) < 0$ 인 경우 $-n < f(5) < 0$ 이므로 ㉠에서 $g(|f(5)|) = 4$ 가 되어 조건 (나)를 만족시키지 않는다. 그러므로 $f(5) \ge 0$

$f(5) = 0$ 또는 $n < f(5) \le \frac{3}{2}n$

① $f(5) = 0$ 인 경우
$2 < n \le 5$ 이면 $f(5) = \frac{1}{2}n\sqrt{5-n}-n = 0$
$2n = n\sqrt{5-n}$
$2 = \sqrt{5-n}$
$n = 1$ 이므로 조건을 만족시키지 않는다.

$n > 5$ 이면 $f(5) = \frac{n^2-2n}{6} - \frac{3}{2}n = \frac{1}{6}n(n-11) = 0$ 이므로 $n$의 값은 $11$

② $n < f(5) \le \frac{3}{2}n$ 인 경우
$2 < n \le 5$ 이면 $f(5) = \frac{1}{2}n\sqrt{5-n}-n$ 이므로
$n < \frac{1}{2}n\sqrt{5-n}-n \le \frac{3}{2}n$
$1 < \frac{1}{2}\sqrt{5-n}-1 \le \frac{3}{2}$
$4 < \sqrt{5-n} \le 5$
$2 < n \le 5$ 에서 $\sqrt{5-n} < \sqrt{3}$ 이므로 조건을 만족시키지 않는다.

$n > 5$ 이면 $f(5) = \frac{n^2-2n}{6} - \frac{3}{2}n = \frac{1}{6}n^2 - \frac{11}{6}n$
$n < \frac{1}{6}n^2 - \frac{11}{6}n \le \frac{3}{2}n$
$1 < \frac{1}{6}n - \frac{11}{6} \le \frac{3}{2}$
$6 < n-11 \le 9$
$17 < n \le 20$
$n$의 값은 $18, 19, 20$

이상에서 조건을 만족시키는 모든 $n$의 값은 $11, 18, 19, 20$ 이고 그 합은 $68$이다.
