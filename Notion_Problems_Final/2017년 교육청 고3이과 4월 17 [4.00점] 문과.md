## 문제
두 집합 $A = \{ x \mid x^{2} - 5x + 4 \leq 0 \}$, $B = \{ x \mid (\log_{2} x)^{2} - 2k \log_{2} x + k^{2} - 1 \leq 0 \}$에 대하여 $A \cap B \neq \emptyset$을 만족시키는 정수 $k$의 개수는?

## 보기
① $5$   ② $6$   ③ $7$   ④ $8$   ⑤ $9$

## 풀이
[정답] ①

집합 $A = \{ x \mid 1 \leq x \leq 4 \}$
집합 $B$에서 $(\log_{2} x - k + 1)(\log_{2} x - k - 1) \leq 0$
$k - 1 \leq \log_{2} x \leq k + 1$
$\therefore 2^{k-1} \leq x \leq 2^{k+1}$
$A \cap B \neq \emptyset$이 되려면
$2^{k+1} \geq 1$이고 $2^{k-1} \leq 4$이어야 한다.
$2^{k+1} \geq 2^{0}$에서 $k + 1 \geq 0$, 즉 $k \geq -1$
$2^{k-1} \leq 2^{2}$에서 $k - 1 \leq 2$, 즉 $k \leq 3$
따라서 $-1 \leq k \leq 3$
이를 만족하는 정수 $k$는 $-1$, $0$, $1$, $2$, $3$이므로 개수는 $5$이다.
