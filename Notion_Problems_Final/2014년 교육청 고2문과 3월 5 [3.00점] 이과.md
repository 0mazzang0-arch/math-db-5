## 문제
두 복소수 $\alpha = 3+i$, $\beta = 1-2i$에 대하여 $(\alpha - \beta)(\bar{\alpha} - \bar{\beta})$의 값은?
(단, $i = \sqrt{-1}$이고, $\bar{\alpha}$, $\bar{\beta}$는 각각 $\alpha$, $\beta$의 켤레복소수이다.)

## 보기
① $11$   ② $13$   ③ $15$   ④ $17$   ⑤ $19$

## 풀이
$\alpha = 3+i$, $\beta = 1-2i$에 대하여 $\bar{\alpha} = 3-i$, $\bar{\beta} = 1+2i$

$\alpha - \beta = (3+i) - (1-2i) = 2+3i$

$\bar{\alpha} - \bar{\beta} = (3-i) - (1+2i) = 2-3i$

이므로
$(\alpha - \beta)(\bar{\alpha} - \bar{\beta}) = (2+3i)(2-3i) = 4+9 = 13$

[다른 풀이]
$\alpha - \beta = (3+i) - (1-2i) = 2+3i$이므로
$\bar{\alpha} - \bar{\beta} = \overline{\alpha - \beta} = 2-3i$

$\therefore (\alpha - \beta)(\bar{\alpha} - \bar{\beta}) = (2+3i)(2-3i) = 4+9 = 13$

정답: ②
