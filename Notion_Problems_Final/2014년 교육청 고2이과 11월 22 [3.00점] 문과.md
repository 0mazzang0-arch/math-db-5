## 문제
등차수열 $\{ a_{n} \}$ 이 $a_{2} + a_{6} = 14$, $a_{3} + a_{7} = 20$ 을 만족시킬 때, $a_{10}$ 의 값을 구하시오.

## 보기

## 풀이
공차를 $d$ 라 하면 
$a_{3} = a_{2} + d$, $a_{7} = a_{6} + d$ 

$2d = 20 - 14 = 6$ 이므로 $d = 3$ 

$a_{1} + d + a_{1} + 5d = 14$ 이므로 
$a_{1} = -2$ 

따라서 
$a_{10} = a_{1} + 9d = -2 + 27 = 25$

정답: $25$
