## 문제
$x$ 에 대한 연립부등식 
$$ \begin{cases} x^{2} - 11x + 24 < 0 \\ x^{2} - 2kx + k^{2} - 9 > 0 \end{cases} $$
의 해가 $\alpha < x < \beta$ 일 때, $\beta - \alpha = 2$ 를 만족시키는 모든 실수 $k$ 의 값의 합을 구하시오.

## 풀이
$x^{2} - 11x + 24 = (x - 3)(x - 8) < 0$ 이므로 
$3 < x < 8$ 이다. 

$x^{2} - 2kx + k^{2} - 9 = x^{2} - 2kx + (k - 3)(k + 3) = \{ x - (k - 3) \}\{ x - (k + 3) \} > 0$ 이므로 
$x < k - 3$ 또는 $x > k + 3$ 이다. 

(ⅰ) $3 < k - 3 < 8$ 인 경우 
$k > 6$ 이므로 $k + 3 > 9$ 이다. 

> [!example] 📸 (그림/그래프)

$3 < x < k - 3$ 이므로 
$(k - 3) - 3 = 2$, 
$k = 8$ 이다. 

(ⅱ) $3 < k + 3 < 8$ 인 경우 
$k < 5$ 이므로 $k - 3 < 2$ 이다. 

> [!example] 📸 (그림/그래프)

$k + 3 < x < 8$ 이므로 
$8 - (k + 3) = 2$, 
$k = 3$ 이다. 

따라서 (ⅰ), (ⅱ)에 의해 모든 $k$ 의 값의 합은 
$8 + 3 = 11$ 이다. 

**정답: $11$**
