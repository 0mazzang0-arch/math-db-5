## 문제
$1 \le x \le 8$ 인 $x$ 에 대하여 양수 $N$ 이 다음 조건을 만족시킨다.

(가) $\log N = \log_{2} x$
(나) $\log N$ 의 소수부분과 $\log \frac{1}{N}$ 의 소수부분이 같다.

$N$ 의 모든 값의 곱이 $10^{\frac{n}{m}}$ 일 때, $m + n$ 의 값을 구하시오. (단, $m$, $n$ 은 서로소인 자연수이다.)

## 풀이
$1 \le x \le 8$ 이므로 $0 \le \log_{2} x \le 3$ 이다.
(가)에서 $0 \le \log N \le 3$ 이다.
(나)에서 $\log N$ 의 소수부분과 $\log \frac{1}{N}$ 의 소수부분이 같으므로 $\log N - \log \frac{1}{N} = 2\log N$ 은 정수이다.
$0 \le 2\log N \le 6$ 이므로
$2\log N = 0, 1, 2, 3, 4, 5, 6$
$\log N = 0, \frac{1}{2}, \frac{2}{2}, \frac{3}{2}, \frac{4}{2}, \frac{5}{2}, \frac{6}{2}$
따라서 $N = 1, 10^{\frac{1}{2}}, 10^{\frac{2}{2}}, 10^{\frac{3}{2}}, 10^{\frac{4}{2}}, 10^{\frac{5}{2}}, 10^{\frac{6}{2}}$ 이다.
$N$ 의 모든 값의 곱은
$1 \times 10^{\frac{1}{2}} \times 10^{\frac{2}{2}} \times \dots \times 10^{\frac{6}{2}} = 10^{\frac{1}{2} + \frac{2}{2} + \dots + \frac{6}{2}} = 10^{\frac{21}{2}}$
따라서 $\frac{n}{m} = \frac{21}{2}$ 이고 $m = 2$, $n = 21$ 이므로
$m + n = 23$

정답: $23$
