## 문제
두 실수 $a$, $b$ ($b > 0$)에 대하여 함수
$$ f(x) = \begin{cases} x^{2}+ax+b & (x \le 0) \\ -x^{2}+ax-b & (x > 0) \end{cases} $$
가 다음 조건을 만족시킬 때, $f(2) = p+q\sqrt{2}$이다. $p-q$의 값은? (단, $p$, $q$는 유리수이다.)

(가) $x$에 대한 방정식 $f(x)=t$의 서로 다른 실근의 개수가 $2$가 되도록 하는 실수 $t$의 개수는 $1$이다.
(나) 모든 정수 $k$에 대하여 $f(k)f(k+1) \ge 0$이다.

## 보기
① $-1$   ② $3$   ③ $7$   ④ $11$   ⑤ $15$

## 풀이
$-x^{2}+ax-b = -\{(-x)^{2}+a(-x)+b\}$이므로 함수 $y=x^{2}+ax+b$ ($x<0$)의 그래프와 함수 $y=-x^{2}+ax-b$ ($x>0$)의 그래프는 원점에 대하여 대칭이다.

$x^{2}+ax+b = (x+\frac{a}{2})^{2}-\frac{a^{2}}{4}+b$에서 함수 $y=f(x)$의 그래프의 개형은 다음과 같다.

(ⅰ) $a \le 0$인 경우
> [!example] 📸 (그림/그래프)

$x$에 대한 방정식 $f(x)=t$의 서로 다른 실근의 개수가 $2$가 되도록 하는 실수 $t$가 존재하지 않으므로 조건 (가)를 만족시키지 않는다.

(ⅱ) $a > 0$인 경우
① $-\frac{a^{2}}{4}+b > 0$인 경우
> [!example] 📸 (그림/그래프)

$x$에 대한 방정식 $f(x)=t$의 서로 다른 실근의 개수가 $2$가 되도록 하는 실수 $t$가 무수히 많이 존재하므로 조건 (가)를 만족시키지 않는다.

② $-\frac{a^{2}}{4}+b = 0$인 경우
> [!example] 📸 (그림/그래프)

$x$에 대한 방정식 $f(x)=t$의 서로 다른 실근의 개수가 $2$가 되도록 하는 실수 $t$가 무수히 많이 존재하므로 조건 (가)를 만족시키지 않는다.

③ $-b < -\frac{a^{2}}{4}+b < 0$인 경우
> [!example] 📸 (그림/그래프)

$x$에 대한 방정식 $f(x)=t$의 서로 다른 실근의 개수가 $2$가 되도록 하는 실수 $t$가 무수히 많이 존재하므로 조건 (가)를 만족시키지 않는다.

④ $-\frac{a^{2}}{4}+b = -b$인 경우
> [!example] 📸 (그림/그래프)

$x$에 대한 방정식 $f(x)=t$의 서로 다른 실근의 개수가 $2$가 되도록 하는 실수 $t$는 $t=-b$ 하나만 존재하므로 조건 (가)를 만족시킨다.

⑤ $-\frac{a^{2}}{4}+b < -b$인 경우
> [!example] 📸 (그림/그래프)

$x$에 대한 방정식 $f(x)=t$의 서로 다른 실근의 개수가 $2$가 되도록 하는 실수 $t$는 두 개 존재하므로 조건 (가)를 만족시키지 않는다.

(ⅰ), (ⅱ)에서 $-\frac{a^{2}}{4}+b = -b$
$b = \frac{a^{2}}{8} \cdots$ ㉠

조건 (나)에서 $f(0)f(1) \ge 0$, $f(-1)f(0) \ge 0$이어야 한다.
$f(0) = b > 0$이므로 $f(1) \ge 0$, $f(-1) \ge 0$이어야 하고 $f(-1) = -f(1)$이므로 $f(1) = 0$이 되어야 한다.

$f(1) = -1+a-b = 0$에서 $b = a-1 \cdots$ ㉡
㉠을 ㉡에 대입하면 $\frac{a^{2}}{8} = a-1$
$a^{2}-8a+8 = 0$, $a = 4 \pm 2\sqrt{2}$

$x > 0$일 때, $f(x) = -x^{2}+ax-b = -x^{2}+ax-a+1$이므로 방정식 $-x^{2}+ax-a+1 = -(x-1)(x-a+1) = 0$의 두 근은 $1$, $a-1$이다.

$a = 4+2\sqrt{2}$ 즉, $a-1 = 3+2\sqrt{2}$이면 $5 < 3+2\sqrt{2} < 6$이고 $x > 0$에서 $f(x) = -(x-1)(x-3-2\sqrt{2})$이므로
$f(5) = -(5-1)\{5-(3+2\sqrt{2})\} > 0$
$f(6) = -(6-1)\{6-(3+2\sqrt{2})\} < 0$이다.
따라서 $f(5)f(5+1) < 0$이 되어 조건 (나)를 만족시키지 않는다.

$a = 4-2\sqrt{2}$ 즉, $a-1 = 3-2\sqrt{2}$이면 $0 < 3-2\sqrt{2} < 1$이고 $x > 0$에서 $f(x) = -(x-1)(x-3+2\sqrt{2})$이므로 $f(x) < 0$을 만족시키는 양수 $x$의 값의 범위는 $0 < x < 3-2\sqrt{2}$ 또는 $x > 1$이다.
따라서 $k > 1$인 모든 정수 $k$에 대하여 $f(k) < 0$이므로 $f(k)f(k+1) > 0$이다.
또한 $f(1) = 0$이므로
$k=0$이면 $f(k)f(k+1) = f(0)f(1) = 0$,
$k=1$이면 $f(k)f(k+1) = f(1)f(2) = 0$이다.

한편, $x < 0$에서 $f(x) = (x+1)(x+3-2\sqrt{2})$이므로 $f(x) > 0$을 만족시키는 음수 $x$의 값의 범위는 $x < -1$ 또는 $-3+2\sqrt{2} < x < 0$이다.
따라서 $k+1 < -1$ 즉, $k < -2$인 모든 정수 $k$에 대하여 $f(k+1) > 0$이므로 $f(k)f(k+1) > 0$이다.
또한 $f(-1) = 0$이므로
$k = -1$이면 $f(k)f(k+1) = f(-1)f(0) = 0$이다.
$k = -2$이면 $f(k)f(k+1) = f(-2)f(-1) = 0$이다.
그러므로 $a = 4-2\sqrt{2}$일 때 모든 정수 $k$에 대하여 $f(k)f(k+1) \ge 0$이므로 조건을 만족시킨다.

$f(2) = -4+2a-b = -4+2a-(a-1) = a-3 = (4-2\sqrt{2})-3 = 1-2\sqrt{2}$
따라서 $p = 1$, $q = -2$이고 $p-q = 1-(-2) = 3$이다.

정답: ②
