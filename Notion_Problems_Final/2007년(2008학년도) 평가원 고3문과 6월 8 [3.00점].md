## 문제
$1 \le \log n < 3$ 인 자연수 $n$ 에 대하여 $\log _{2} n$ 이 정수가 되도록 하는 $n$ 의 개수는?

## 보기
① $3$   ② $4$   ③ $5$   ④ $6$   ⑤ $7$

## 풀이
$1 \le \log n < 3$ 에서 
$10 \le n < 1000$ 이다.

$\log _{2} n$ 이 정수이려면 
$n = 2^{k}$ ($k$ 는 정수)의 꼴이어야 한다. 

$10 \le n < 1000$ 이므로 $10 \le 2^{k} < 1000$ 에서 
$k = 4, 5, 6, 7, 8, 9$ 이다.

따라서, 자연수 $n$ 은 $2^{4}, 2^{5}, 2^{6}, 2^{7}, 2^{8}, 2^{9}$ 의 $6$ 개이다.
