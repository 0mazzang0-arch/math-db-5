## 문제
$n$이 자연수일 때, 함수 $f(x) = \frac{x+2n}{2x-p}$이 $f(1) < f(5) < f(3)$을 만족시키도록 하는 자연수 $p$의 최솟값을 $m$이라 하자. 자연수 $n$에 대하여 $p=m$일 때의 함수 $f(x)$와 함수 $g(x) = \frac{2x+n}{x+q}$이 $g(f(5)) < g(f(3)) < g(f(1))$을 만족시키도록 하는 자연수 $q$의 개수를 $a_n$이라 하자. $\sum_{k=1}^{20} a_k$의 값을 구하시오.

## 보기

## 풀이
$f(x) = \frac{x+2n}{2x-p} = \frac{\frac{1}{2}(2x-p) + \frac{p}{2} + 2n}{2x-p} = \frac{1}{2} + \frac{\frac{p}{2} + 2n}{2x-p}$이고, $\frac{p}{2} + 2n > 0$이므로 $f(1) < f(5) < f(3) \cdots ㉠$이 성립하려면 함수 $y = f(x)$의 그래프는 다음과 같아야 한다.

> [!example] 📸 (그림/그래프)

$1 < \frac{p}{2} < 3$이어야 하므로 $2 < p < 6$에서 자연수 $p$의 최솟값 $m$은 $3$이다.
함수 $g(x) = \frac{2x+n}{x+q} = \frac{2(x+q) + n - 2q}{x+q} = 2 + \frac{n-2q}{x+q}$이므로 곡선 $y = g(x)$의 두 점근선의 방정식은 $x = -q$, $y = 2$이다.

$p = 3$일 때 $f(x) = \frac{x+2n}{2x-3}$에 대하여
$x_1 = f(1) = -2n - 1$
$x_2 = f(5) = \frac{2n+5}{7}$
$x_3 = f(3) = \frac{2n+3}{3}$
이라 하면 $㉠$으로부터 $x_1 < x_2 < x_3$이다.
이때 문제의 조건에서 $g(x_2) < g(x_3) < g(x_1)$이 성립해야 하므로

> [!example] 📸 (그림/그래프)

함수 $y = g(x)$의 그래프는 다음과 같아야 한다.
그러므로 $x_1 < -q < x_2$이고 $n - 2q < 0$이어야 한다.
즉, $-2n - 1 < -q < \frac{2n+5}{7}$이고 $q > \frac{n}{2}$이어야 하므로
$\frac{n}{2} < q < 2n + 1$이다.

(ⅰ) $n = 2l - 1$ ($l$은 자연수)일 때
$\frac{2l-1}{2} < q < 2(2l-1) + 1$에서 $l - \frac{1}{2} < q < 4l - 1$이므로
$q = l, l + 1, \cdots, 4l - 2$이다.
$q$의 개수는 $3l - 1$이다.

(ⅱ) $n = 2l$ ($l$은 자연수)일 때
$\frac{2l}{2} < q < 2 \times 2l + 1$에서 $l < q < 4l + 1$이므로
$q = l + 1, l + 2, \cdots, 4l$이다.
$q$의 개수는 $3l$이다.

(ⅰ), (ⅱ)에 의하여 $a_{2l-1} = 3l - 1$, $a_{2l} = 3l$이다.
$\sum_{k=1}^{20} a_k = \sum_{l=1}^{10} (a_{2l-1} + a_{2l})$
$= \sum_{l=1}^{10} (3l - 1 + 3l) = \sum_{l=1}^{10} (6l - 1)$
$= 6 \sum_{l=1}^{10} l - \sum_{l=1}^{10} 1$
$= 6 \times \frac{10 \times 11}{2} - 10$
$= 330 - 10 = 320$

[정답] $320$
