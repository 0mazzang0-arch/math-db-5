## 문제
두 이차정사각행렬 $A$, $B$ 가 
$$AB = A - B$$, $$2BA + 2B = A^2$$
을 만족시킬 때, <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $E$ 는 단위행렬이다.)

**<보 기>**
ㄱ. $AB = BA$
ㄴ. $A - 3E$ 의 역행렬이 존재한다.
ㄷ. $(A + B)^2 = 16B^2$

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
ㄱ. $AB = A - B$ 에서 
$AB - A + B - E = -E$ 
$(A + E)(B - E) = -E$ 
$\therefore (A + E)^{-1} = E - B$ 
$(A + E)(E - B) = E$, $(E - B)(A + E) = E$ 이므로 
$(A + E)(E - B) = (E - B)(A + E)$ 에서 
$AB = BA$ (참) 

ㄴ. $AB = A - B$ 이고 $AB = BA$ 이므로 
$2BA + 2B = A^2$ 에서 
$2(A - B) + 2B = A^2$, 즉 $A^2 = 2A$ 
$A^2 - 2A - 3E = -3E$ 에서 
$(A + E)(A - 3E) = -3E$ 
$\therefore (A - 3E)^{-1} = -\frac{1}{3}(A + E)$ 
$A - 3E$ 의 역행렬이 존재한다. (참) 

ㄷ. $(A + E)(A - 3E) = -3E$ 에서 
$(A + E)^{-1} = \frac{1}{3}(3E - A)$ 
$(A + E)^{-1} = E - B$ 이므로 
$\frac{1}{3}(3E - A) = E - B$ 
$\therefore A = 3B$ 
$\therefore (A + B)^2 = (4B)^2 = 16B^2$ (참) 

따라서 ㄱ, ㄴ, ㄷ 모두 옳다.
