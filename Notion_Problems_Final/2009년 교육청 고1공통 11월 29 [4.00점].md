## 문제
이차함수 $y = 2x^{2}$ 의 그래프와 원 $x^{2} + (y + 1)^{2} = 1$ 에 동시에 접하는 직선이 $y = ax + b$ 일 때, $a^{2} + b$ 의 값을 구하시오. (단, $a$, $b$ 는 상수이고 $b < 0$ 이다.)

## 풀이
다음 그림과 같이 

> [!example] 📸 (그림/그래프)

직선 $y = ax + b$ 가 이차함수 $y = 2x^{2}$ 의 그래프에 접하므로 
$2x^{2} = ax + b$, $2x^{2} - ax - b = 0$
위 방정식의 판별식을 $D$ 라 하면 
$D = a^{2} + 8b = 0$, $a^{2} = -8b$ $\cdots$ ㉠ 

직선 $y = ax + b$ 가 원 $x^{2} + (y + 1)^{2} = 1$ 에 접하므로 
$\frac{|1 + b|}{\sqrt{a^{2} + 1}} = 1$, $|1 + b| = \sqrt{a^{2} + 1}$ 
$a^{2} + 1 = b^{2} + 2b + 1$, $-8b + 1 = b^{2} + 2b + 1$ ($\because$ ㉠) 
$b^{2} + 10b = 0$, $b(b + 10) = 0$ 

$\therefore b = -10$ ($\because b < 0$) 
㉠에 대입하면 $a^{2} = 80$ 

$\therefore a^{2} + b = 70$
