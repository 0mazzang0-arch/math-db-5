## 문제
두 이차정사각행렬 $A$, $B$에 대하여 $A - B = -E$, $A = B^{2}$일 때, <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $E$는 단위행렬이다.)

$$ \begin{array}{|l|} \hline \text{ㄱ. } AB = BA \\ \text{ㄴ. } B^{-1} = -B^{2} \\ \text{ㄷ. } B^{2011} = -B \\ \hline \end{array} $$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
ㄱ. $A - B = -E$에서 $B = A + E$이므로
$AB = A(A + E) = A^{2} + A$
$BA = (A + E)A = A^{2} + A$
$\therefore AB = BA$ (참)

ㄴ. $A - B = -E$에서 $A = B - E$이고 $A = B^{2}$이므로
$B^{2} - B = -E$
$B(B - E) = (B - E)B = -E$
$B(E - B) = (E - B)B = E$
$\therefore B^{-1} = E - B$
이때 $E - B = -A$이고 $A = B^{2}$이므로
$B^{-1} = -A = -B^{2}$ (참)

ㄷ. ㄴ에 의하여 $B^{-1} = -B^{2}$이므로 양변에 $B$를 곱하면
$E = -B^{3}$, 즉 $B^{3} = -E$이다.
$B^{6} = (B^{3})^{2} = (-E)^{2} = E$이므로
$B^{2011} = (B^{6})^{335} B = E^{335} B = B$
따라서 $B^{2011} = -B$는 거짓이다. (거짓)

따라서 옳은 것은 $ㄱ$, $ㄴ$이다.
