## 문제
곡선 $y= | x^{2}-2x |$ 와 직선 $y=3$ 으로 둘러싸인 부분의 넓이는?



> [!example] 📸 (그림/그래프)

## 보기
① $7$   ② $8$   ③ $9$   ④ $10$   ⑤ $11$

## 풀이
곡선 $y= | x^{2}-2x |$ 와 직선 $y=3$ 의 교점의 $x$ 좌표를 구하면 $| x^{2}-2x | =3$ 에서
(ⅰ) $x^{2}-2x=3$ 일 때 
$x^{2}-2x-3=0$ , $(x+1 ) (x-3 )=0$ 
$\therefore x=-1$ 또는 $x=3$

(ⅱ) $x^{2}-2x=-3$ 일 때 
$x^{2}-2x+3=0$ 
판별식을 $D$ 라 하면 
$\frac{D}{4}= (-1 )^{2}-1 \times 3 =-2 < 0$ 

(ⅰ), (ⅱ)에서 교점의 $x$ 좌표는 $-1$, $3$ 이다. 

따라서 구하는 넓이는 
$\int_{-1}^{3} (3- | x^{2}-2x | ) dx$ 
$= \int_{-1}^{0} \{ 3- ( x^{2} -2x ) \} dx+ \int_{0}^{2} \{ 3- ( -x^{2} +2x ) \} dx + \int_{2}^{3} \{ 3- ( x^{2} -2x ) \} dx$ 
$= [ -\frac{1}{3}x^{3}+x^{2}+3x ]_{-1}^{0} + [ \frac{1}{3} x^{3} -x^{2} +3x ]_{0}^{2} + [ -\frac{1}{3} x^{3} +x^{2} +3x ]_{2}^{3}$ 
$= 0 - ( \frac{1}{3}+1-3 ) + ( \frac{8}{3} -4+6 ) -0 + ( -9+9+9 ) - ( -\frac{8}{3} +4+6 )$ 
$= \frac{5}{3} + \frac{14}{3} + \frac{5}{3}$ 
$= 8$
