## 문제

정수 $a$ ($a \neq 0$)에 대하여 함수 $f(x)$를
$$f(x) = x^3 - 2ax^2$$
이라 하자. 다음 조건을 만족시키는 모든 정수 $k$의 값의 곱이 $-12$가 되도록 하는 $a$에 대하여 $f'(10)$의 값을 구하시오.

$$ \frac{f(x_1) - f(x_2)}{x_1 - x_2} \times \frac{f(x_2) - f(x_3)}{x_2 - x_3} < 0 $$
을 만족시키는 세 실수 $x_1$, $x_2$, $x_3$이 열린구간 $(k, k + \frac{3}{2})$에 존재한다.

## 풀이

세 점 $A(x_1, f(x_1))$, $B(x_2, f(x_2))$, $C(x_3, f(x_3))$에 대하여 직선 $AB$, 직선 $BC$, 직선 $CA$의 기울기를 각각 $m_1$, $m_2$, $m_3$이라 하면, 문제의 조건으로부터 $m_1$, $m_2$, $m_3$ 중 양수와 음수가 적어도 하나씩은 각각 존재해야 한다.

$x_1$, $x_2$, $x_3$가 모두 열린구간 $(k, k + \frac{3}{2})$에 있을 때, 열린구간 $(k, k + \frac{3}{2})$이 함수 $f(x)$가 극값을 가지는 $x$의 값을 포함하지 않는다면, $m_1$, $m_2$, $m_3$의 부호가 모두 같게 된다. 반면에 열린구간 $(k, k + \frac{3}{2})$이 함수 $f(x)$가 극값을 가지는 $x$의 값을 적어도 하나는 포함하면 $m_1$, $m_2$, $m_3$ 중 부호가 다른 두 개가 반드시 존재한다.

$f(x) = x^3 - 2ax^2$에서
$f'(x) = 3x^2 - 4ax = 3x(x - \frac{4a}{3})$
함수 $f(x)$는 $x = 0$과 $x = \frac{4a}{3}$에서 극값을 가지므로
$0 \in (k, k + \frac{3}{2})$ 또는 $\frac{4a}{3} \in (k, k + \frac{3}{2})$이다.

$0 \in (k, k + \frac{3}{2})$일 때
$k < 0 < k + \frac{3}{2}$, $-\frac{3}{2} < k < 0$
$k$는 정수이므로 $\therefore k = -1$

$\frac{4a}{3} \in (k, k + \frac{3}{2})$일 때
$k < \frac{4a}{3} < k + \frac{3}{2}$, $\frac{4a}{3} - \frac{3}{2} < k < \frac{4a}{3}$
조건을 만족시키는 모든 정수 $k$의 값의 곱이 $-12$이므로 부등식 $\frac{4a}{3} - \frac{3}{2} < k < \frac{4a}{3}$을 만족하는 정수 $k$의 값의 곱은 $12$이다. 곱해서 $12$가 되는 서로 다른 연속된 정수들은 $-4$과 $-3$ 또는 $3$과 $4$뿐이다.

(ⅰ) $-4$와 $-3$이 $k$에 대한 부등식 $\frac{4a}{3} - \frac{3}{2} < k < \frac{4a}{3}$을 만족하는 정수해일 때
$-5 \le \frac{4a}{3} - \frac{3}{2} < -4$, $-3 < \frac{4a}{3} \le -2$이어야 한다.
$a$의 공통범위를 구하면 $-\frac{9}{4} < a < -\frac{15}{8}$
$a$는 정수이므로 $a = -2$

(ⅱ) $3$과 $4$가 $k$에 대한 부등식 $\frac{4a}{3} - \frac{3}{2} < k < \frac{4a}{3}$을 만족하는 정수해일 때
$2 \le \frac{4a}{3} - \frac{3}{2} < 3$, $4 < \frac{4a}{3} \le 5$이어야 한다.
$a$의 공통범위를 구하면 $3 < a < \frac{27}{8}$
$a$는 정수이므로 이 경우에는 만족하는 $a$가 존재하지 않는다.

이상에서 $a = -2$, $f(x) = x^3 + 4x^2$, $f'(x) = 3x^2 + 8x$
$\therefore f'(10) = 300 + 80 = 380$

정답: $380$
