## 문제
세 수 $A = \sqrt[3]{\sqrt{10}}$, $B = \sqrt{5}$, $C = \sqrt[3]{\sqrt{28}}$ 의 대소 관계를 바르게 나타낸 것은?

## 보기
① $A < B < C$   ② $A < C < B$   ③ $B < A < C$   ④ $B < C < A$   ⑤ $C < A < B$

## 풀이
$A = \sqrt[3]{\sqrt{10}} = \sqrt[6]{10}$
$B = \sqrt{5} = \sqrt[6]{5^{3}} = \sqrt[6]{125}$
$C = \sqrt[3]{\sqrt{28}} = \sqrt[6]{28}$

$10 < 28 < 125$ 이므로 $\sqrt[6]{10} < \sqrt[6]{28} < \sqrt[6]{125}$ 이다.
따라서 $A < C < B$

정답: ②
