## 문제
등차수열 $\{ a_{n} \}$ 이 $a_{10} + a_{6} = 6$ , $a_{10} - a_{6} = - 12$ 를 만족시킬 때, $a_{2}$ 의 값을 구하시오.

## 풀이
등차수열 $\{ a_{n} \}$ 의 첫째항을 $a$ , 공차를 $d$ 라 하면
$a_{10} + a_{6} = ( a + 9 d ) + ( a + 5 d ) = 2 a + 14 d = 6$ $\cdots$ ㉠
$a_{10} - a_{6} = ( a + 9 d ) - ( a + 5 d ) = 4 d = - 12$ $\cdots$ ㉡
㉠, ㉡에서 $a = 24$ , $d = - 3$
$\therefore a_{2} = a + d = 24 + ( - 3 ) = 21$

정답: $21$
