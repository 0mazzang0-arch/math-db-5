## 문제
한 변의 길이가 $4$인 정사각형 $ABCD$의 내부에 한 변의 길이가 $2$인 정사각형 $EFGH$를 $\overline{AB}$와 $\overline{EF}$가 평행하도록 그린 것이다. 네 사다리꼴 $\text{ABFE}$, $\text{BCGF}$, $\text{CDHG}$, $\text{DAEH}$의 넓이를 각각 $S_{1}$, $S_{2}$, $S_{3}$, $S_{4}$라 하자. <보기>에서 항상 옳은 것만을 있는 대로 고른 것은?

> [!example] 📸 (그림/그래프)

**<보 기>**
ㄱ. $\overline{AE}^{2} + \overline{CG}^{2} = \overline{BF}^{2} + \overline{DH}^{2}$
ㄴ. $\overline{AE} = \overline{BF}$이면 $\overline{CG} = \overline{DH}$이다.
ㄷ. $S_{1} + S_{3} = S_{2} + S_{4}$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄱ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
사각형 $ABCD$를 점 $B$를 원점으로 하는 좌표평면 위에 놓으면,
$A(0, 4)$, $C(4, 0)$, $D(4, 4)$이고 점 $F$의 좌표를 $(a, b)$라 하면
(단, $0 < a < 2$, $0 < b < 2$)
$E(a, b+2)$, $G(a+2, b)$, $H(a+2, b+2)$이다.

ㄱ. $\overline{AE}^{2} + \overline{CG}^{2} = \{a^{2} + (b-2)^{2}\} + \{(a-2)^{2} + b^{2}\}$
$\overline{BF}^{2} + \overline{DH}^{2} = (a^{2} + b^{2}) + \{(a-2)^{2} + (b-2)^{2}\}$
$\therefore \overline{AE}^{2} + \overline{CG}^{2} = \overline{BF}^{2} + \overline{DH}^{2}$
따라서 참

ㄴ. $\overline{AE} = \overline{BF}$에서 $\overline{AE}^{2} = \overline{BF}^{2}$
$a^{2} + (b-2)^{2} = a^{2} + b^{2}$
$4b = 4 \implies b = 1$
$\overline{CG}^{2} = (a-2)^{2} + b^{2} = (a-2)^{2} + 1^{2}$
$\overline{DH}^{2} = (a-2)^{2} + (b-2)^{2} = (a-2)^{2} + (1-2)^{2} = (a-2)^{2} + 1$
$\therefore \overline{CG} = \overline{DH}$
따라서 참

ㄷ. $S_{1} + S_{3} = \frac{1}{2} (4 + 2) \times \{a + (2 - a)\} = 3 \times 2 = 6$
$S_{2} + S_{4} = \frac{1}{2} (4 + 2) \times \{b + (2 - b)\} = 3 \times 2 = 6$
$\therefore S_{1} + S_{3} = S_{2} + S_{4}$
따라서 참

정답: ⑤
