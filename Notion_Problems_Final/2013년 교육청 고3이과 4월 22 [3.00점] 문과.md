## 문제
로그부등식 $\log_{2} (x-1) < 2 \log_{4} (7-x)$의 해가 $\alpha < x < \beta$일 때, $\alpha^{2} + \beta^{2}$의 값을 구하시오.

## 보기

## 풀이
$\log_{2} (x-1) < \log_{2} (7-x)$에서 로그의 정의에 의하여 
$1 < x < 7$ 
$\cdots$ ㉠ 

$x-1 < 7-x$에서 
$x < 4$ 
$\cdots$ ㉡ 

㉠, ㉡에서 
$1 < x < 4$이므로 
$\alpha = 1$, $\beta = 4$ 

$\alpha^{2} + \beta^{2} = 17$ 

정답: $17$
