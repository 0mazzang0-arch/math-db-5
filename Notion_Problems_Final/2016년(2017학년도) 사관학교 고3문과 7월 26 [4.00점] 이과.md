## 문제
사관학교에서는 사관생도들에게 세 국가 $A$, $B$, $C$에서 해외 파견 교육을 받을 수 있도록 하고 있다. 해외 파견 교육 대상 사관생도를 선발하기 위해 희망자를 조사하였더니 하나 이상의 국가를 신청한 사관생도의 수가 $70$명이었고, 그 결과는 다음과 같았다.

$$ \begin{array}{l} \text{(가) } A \text{ 또는 } B \text{를 신청한 사관생도는 } 43\text{명이다.} \\ \text{(나) } B \text{ 또는 } C \text{를 신청한 사관생도는 } 51\text{명이다.} \\ \text{(다) } A\text{와 } C\text{를 동시에 신청한 사관생도는 없다.} \end{array} $$

$B$를 신청한 사관생도의 수를 구하시오.

## 보기

## 풀이
세 국가 $A$, $B$, $C$를 신청한 사관생도의 집합을 각각 $A$, $B$, $C$라 하면
$n(A \cup B \cup C) = 70$
$n(A \cup B) = 43$, $n(B \cup C) = 51$, $A \cap C = \emptyset$이다.

$n(A \cup B \cup C) = n(A \cup B) + n(C - (A \cup B))$에서
$70 = 43 + n(C - (A \cup B))$
$\therefore n(C - (A \cup B)) = 27$

이때 $A \cap C = \emptyset$이므로 $C - (A \cup B) = C - B$이다.
$n(C - B) = n(B \cup C) - n(B) = 27$에서
$51 - n(B) = 27$
$\therefore n(B) = 24$

정답: $24$
