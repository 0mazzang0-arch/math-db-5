## 문제
양수 $x$ 에 대하여 $\log x$ 의 정수부분과 소수부분을 각각 $f(x)$, $g(x)$ 라 하자. 두 부등식 
$$f(n) \le f(54), \quad g(n) \le g(54)$$
를 만족시키는 자연수 $n$ 의 개수는?

## 보기
① $42$   ② $44$   ③ $46$   ④ $48$   ⑤ $50$

## 풀이
$\log 54 = 1 + \log 5.4$ 이므로 
$f(54) = 1$, $g(54) = \log 5.4$ 이다.
따라서 $f(n) \le f(54)$ 에서 $n$ 은 자연수이므로 $f(n) = 0$ 또는 $f(n) = 1$ 이다.

(ⅰ) $f(n) = 0$ 일 때, $g(n) \le g(54)$ 에서 
$0 \le \log n \le \log 5.4$ 
$1 \le n \le 5.4$ 
$n$ 은 $1, 2, 3, 4, 5$ 의 $5$ 개이다. 

(ⅱ) $f(n) = 1$ 일 때, $g(n) \le g(54)$ 에서 
$0 \le \log n - 1 \le \log 5.4$ 
$1 \le \log n \le \log 54$ 
$10 \le n \le 54$ 
$n$ 은 $10, 11, \cdots, 54$ 의 $45$ 개이다. 

(ⅰ), (ⅱ)에서 구하는 자연수 $n$ 의 개수는 $5 + 45 = 50$ 이다.

정답: ⑤
