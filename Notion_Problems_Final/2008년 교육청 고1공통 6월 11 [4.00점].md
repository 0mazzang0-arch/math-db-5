## 문제
다음 <보기>의 집합 중 곱셈에 대하여 닫혀 있는 것을 모두 고른 것은?

$$ \begin{array}{l} \text{ㄱ. } \{ x \mid |x| \le 1, x \text{ 는 정수} \} \\ \text{ㄴ. } \{ x \mid x = a \sqrt{2}, a \text{ 는 유리수} \} \\ \text{ㄷ. } \{ x \mid x = m^2 - n^2, m, n \text{ 은 정수} \} \end{array} $$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄷ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
[정답] ③ 

ㄱ. $\{ -1, 0, 1 \}$ 이므로 곱셈에 대하여 닫혀 있다. 

ㄴ. (반례) $x = 2 \sqrt{2}$, $y = 3 \sqrt{2}$ 라 하면, $x y = 12$
$\therefore$ 곱셈에 대하여 닫혀 있지 않다. ( $12$ 는 $a \sqrt{2}$ ($a$ 는 유리수) 꼴로 나타낼 수 없다. )

ㄷ. $x = a^2 - b^2$, $y = c^2 - d^2$ 이라 하면 ($a, b, c, d$ 는 정수), 
$x y = (a^2 - b^2) (c^2 - d^2)$ 
$= (a^2 c^2 + b^2 d^2) - (a^2 d^2 + b^2 c^2)$ 
$= (a^2 c^2 + b^2 d^2) - (a^2 d^2 + b^2 c^2) + 2 a b c d - 2 a b c d$ 
$= (a c + b d)^2 - (a d + b c)^2$ 
이때, $a c + b d$ 와 $a d + b c$ 도 정수이므로 $x y$ 는 다시 두 정수의 제곱의 차의 형태로 표현된다.
$\therefore$ 곱셈에 대하여 닫혀 있다.
