## 문제
두 곡선 $y = \ln x + 3$, $y = \ln \frac{1}{x} + 3$ 과 직선 $x = e$ 로 둘러싸인 부분을 $x$ 축의 둘레로 회전시킬 때 생기는 회전체의 부피는 $V$ 이다. $\frac{V}{\pi}$ 의 값을 구하시오.

## 보기

## 풀이
$\ln x + 3 = \ln \frac{1}{x} + 3$ 에서
$2 \ln x = 0 \therefore x = 1$
따라서 두 곡선은 $(1, 3)$ 에서 만난다.

$\therefore V = \pi \int_{1}^{e} \{ (\ln x + 3)^{2} - (\ln \frac{1}{x} + 3)^{2} \} dx$
$= 12 \pi \int_{1}^{e} \ln x dx$
$= 12 \pi [ x \ln x - x ]_{1}^{e} = 12 \pi$

$\therefore \frac{V}{\pi} = 12$

정답: $12$
