## 문제
모든 항이 양수인 수열 $\{ a_{n} \}$ 의 첫째항부터 제 $n$ 항까지의 합을 $S_{n}$ 이라 하자. 다음은 $2 S_{n} = a_{n} + \frac{9}{a_{n}}$ ($n = 1, 2, 3, \cdots$) 일 때, $S_{100}$ 을 구하는 과정이다.

$$ \begin{array}{|l|} \hline 
(\text{i}) \ n = 1 \text{ 일 때, } a_{1} = (가) \\
(\text{ii}) \ n \ge 2 \text{ 일 때, } a_{n} = S_{n} - S_{n-1} \text{ 이므로} \\
2 S_{n} = S_{n} - S_{n-1} + \frac{9}{S_{n} - S_{n-1}} \text{ 이다.} \\
\text{따라서 } ( S_{n} )^{2} - ( S_{n-1} )^{2} = (나) \text{ 이다.} \\
\therefore S_{100} = (다) \\ \hline 
\end{array} $$

위 과정에서 $(가)$, $(나)$, $(다)$에 들어갈 모든 수의 합은?

## 보기
① $39$   ② $40$   ③ $41$   ④ $42$   ⑤ $43$

## 풀이
(i) $n = 1$ 일 때, $S_{1} = a_{1}$ 이므로 주어진 식에 대입하면
$2 a_{1} = a_{1} + \frac{9}{a_{1}}$
$a_{1} = \frac{9}{a_{1}}$
$a_{1}^{2} = 9$
모든 항이 양수이므로 $a_{1} = 3$ 이다.
따라서 $(가) = 3$ 이다.

(ii) $n \ge 2$ 일 때, $a_{n} = S_{n} - S_{n-1}$ 을 대입하면
$2 S_{n} = ( S_{n} - S_{n-1} ) + \frac{9}{S_{n} - S_{n-1}}$
$S_{n} + S_{n-1} = \frac{9}{S_{n} - S_{n-1}}$
$( S_{n} + S_{n-1} ) ( S_{n} - S_{n-1} ) = 9$
$( S_{n} )^{2} - ( S_{n-1} )^{2} = 9$
따라서 $(나) = 9$ 이다.

수열 $\{ ( S_{n} )^{2} \}$ 은 첫째항이 $( S_{1} )^{2} = a_{1}^{2} = 9$ 이고, 공차가 $9$ 인 등차수열이다.
$( S_{n} )^{2} = 9 + ( n - 1 ) \times 9 = 9n$
$S_{n} = \sqrt{9n} = 3 \sqrt{n}$ ($n \ge 1$) 이다.
그러므로 $S_{100} = 3 \sqrt{100} = 3 \times 10 = 30$ 이다.
따라서 $(다) = 30$ 이다.

위 과정에서 구한 (가), (나), (다)의 수의 합은
$3 + 9 + 30 = 42$ 이다.

정답은 ④ 이다.
