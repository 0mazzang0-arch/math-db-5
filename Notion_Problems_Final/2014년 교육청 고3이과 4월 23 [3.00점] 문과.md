## 문제
등차수열 $\{ a_{n} \}$ 이 $a_{1} + a_{2} + a_{3} = 21$, $a_{7} + a_{8} + a_{9} = 75$ 를 만족시킬 때, $a_{10} + a_{11} + a_{12}$ 의 값을 구하시오.

## 보기

## 풀이
등차수열 $\{ a_{n} \}$ 의 첫째항을 $a$, 공차를 $d$ 라 하자.
$a_{1} + a_{2} + a_{3} = 3a + 3d = 21$
$a_{7} + a_{8} + a_{9} = 3a + 21d = 75$
$\therefore a = 4, d = 3$
따라서 $a_{10} + a_{11} + a_{12} = 3a + 30d = 102$

정답: $102$
