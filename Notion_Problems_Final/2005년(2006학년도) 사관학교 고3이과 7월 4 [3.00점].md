## 문제
이차 이하의 모든 다항함수 $f(x)$에 대하여 등식
$$\int_0^2 f(x) dx = a f(0) + b f(1) + c f(2)$$
이 항상 성립하도록 하는 상수 $a$, $b$, $c$의 곱 $abc$의 값은?

## 보기
① $\frac{4}{81}$   ② $\frac{8}{81}$   ③ $\frac{4}{27}$   ④ $\frac{8}{27}$   ⑤ $\frac{8}{9}$

## 풀이
$f(x)$는 이차 이하의 다항식이므로 
$$f(x) = px^2 + qx + r$$
이라 하면 주어진 조건에 의해 
$$\frac{8}{3} p + 2q + 2r = ar + b(p+q+r) + c(4p+2q+r)$$
$$(3b + 12c - 8)p + (3b + 6c - 6)q + (3a + 3b + 3c - 6)r = 0 \cdots (1)$$
식 $(1)$이 임의의 $p$, $q$, $r$에 대하여 성립하기 위한 조건은 
$$3b + 12c - 8 = 0 \cdots (2)$$
$$3b + 6c - 6 = 0 \cdots (3)$$
$$3a + 3b + 3c - 6 = 0 \cdots (4)$$
을 만족해야 한다. 식 $(2) \sim (4)$를 연립해서 해를 구하면 
$$a = \frac{1}{3}, b = \frac{4}{3}, c = \frac{1}{3}$$
따라서 
$$abc = \frac{4}{27}$$
정답은 ③이다.
