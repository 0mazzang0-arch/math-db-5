## 문제
$5$ 이하의 자연수 $a$, $b$, $c$, $d$에 대하여 부등식 
$$a \le b+1 \le c \le d$$
를 만족시키는 모든 순서쌍 $(a, b, c, d)$의 개수를 구하시오.

## 풀이
$c$ 가 $5$ 이하의 자연수이므로 $1 \le b \le 4$ 이다.

(ⅰ) $b=1$ 인 경우 
$a \le 2 \le c \le d$ 에서 $a$ 를 택하는 경우의 수는 ${}_{2}\mathrm{C}_{1}$ 이고, $c$, $d$ 를 택하는 경우의 수는 ${}_{4}\mathrm{H}_{2}$ 이다. 
$${}_{2}\mathrm{C}_{1} \times {}_{4}\mathrm{H}_{2} = {}_{2}\mathrm{C}_{1} \times {}_{5}\mathrm{C}_{2} = 2 \times 10 = 20$$

(ⅱ) $b=2$ 인 경우 
$a \le 3 \le c \le d$ 에서 $a$ 를 택하는 경우의 수는 ${}_{3}\mathrm{C}_{1}$ 이고, $c$, $d$ 를 택하는 경우의 수는 ${}_{3}\mathrm{H}_{2}$ 이다. 
$${}_{3}\mathrm{C}_{1} \times {}_{3}\mathrm{H}_{2} = {}_{3}\mathrm{C}_{1} \times {}_{4}\mathrm{C}_{2} = 3 \times 6 = 18$$

(ⅲ) $b=3$ 인 경우 
$a \le 4 \le c \le d$ 에서 $a$ 를 택하는 경우의 수는 ${}_{4}\mathrm{C}_{1}$ 이고, $c$, $d$ 를 택하는 경우의 수는 ${}_{2}\mathrm{H}_{2}$ 이다. 
$${}_{4}\mathrm{C}_{1} \times {}_{2}\mathrm{H}_{2} = {}_{4}\mathrm{C}_{1} \times {}_{3}\mathrm{C}_{2} = 4 \times 3 = 12$$

(ⅳ) $b=4$ 인 경우 
$a \le 5 \le c \le d$ 에서 $a$ 를 택하는 경우의 수는 ${}_{5}\mathrm{C}_{1}$ 이고, $c$, $d$ 를 택하는 경우의 수는 $1$ 이다. 
$${}_{5}\mathrm{C}_{1} \times 1 = 5$$

(ⅰ) ~ (ⅳ)에 의하여 구하는 경우의 수는 
$$20+18+12+5=55$$

정답: $55$
