## 문제

두 곡선 $y=2^{-x}$ 과 $y=|\log_{2}x|$ 가 만나는 두 점을 $(x_{1}, y_{1})$, $(x_{2}, y_{2})$ 라 하자. $x_{1} < x_{2}$ 일 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?

$$
\begin{array}{|l|}
\hline
\text{<보 기>} \\
ㄱ. \frac{1}{2} < x_{1} < \frac{\sqrt{2}}{2} \\
ㄴ. \sqrt[3]{2} < x_{2} < \sqrt{2} \\
ㄷ. y_{1} - y_{2} < \frac{3\sqrt{2}-2}{6} \\
\hline
\end{array}
$$

## 보기

① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이

$y=2^{-x}$, $y=|\log_{2}x|$, $y=x$ 의 그래프는 다음과 같다.

> [!example] 📸 (그림/그래프)

$ㄱ.$ $0 < x < 1$ 일 때, $y=2^{-x}$, $y=-\log_{2}x$ 의 교점은 직선 $y=x$ 위에 있으므로 $x_{1}=y_{1}$ 이고 $x_{1} < 1$, $y_{1} < 1$ 이다.
$y=2^{-x}$ 은 감소함수이므로
$2^{-1} < 2^{-x_{1}} = y_{1}$
$\frac{1}{2} < y_{1} = x_{1}$
$-\log_{2}\frac{\sqrt{2}}{2} = \frac{1}{2} < y_{1} = -\log_{2}x_{1}$ 이고
$y=-\log_{2}x$ 는 감소함수이므로 $x_{1} < \frac{\sqrt{2}}{2}$ 이다.
따라서 $\frac{1}{2} < x_{1} < \frac{\sqrt{2}}{2}$ 이다. (참)

$ㄴ.$ $2^{-\sqrt[3]{2}} = \frac{1}{2^{\sqrt[3]{2}}}$ 이고 $\log_{2}\sqrt[3]{2} = \frac{1}{3}$ 이다.
$8 < 9$ 이므로 $2^{3/2} < 3$ $\cdots$ ㉠
$\sqrt[3]{2}$ 와 $\frac{3}{2}$ 을 각각 세제곱하면 $(\sqrt[3]{2})^{3} < (\frac{3}{2})^{3}$ 이므로
$\sqrt[3]{2} < \frac{3}{2}$ 즉, $2^{\sqrt[3]{2}} < 2^{3/2}$ $\cdots$ ㉡
㉠, ㉡에 의해 $2^{\sqrt[3]{2}} < 2^{3/2} < 3$ 이므로
$\frac{1}{3} < \frac{1}{2^{\sqrt[3]{2}}}$ 즉, $\log_{2}\sqrt[3]{2} < 2^{-\sqrt[3]{2}}$ 이다.
따라서 $\sqrt[3]{2} < x_{2}$ 이다.
$\log_{2}\sqrt{2} = \frac{1}{2}$, $2^{-\sqrt{2}} = \frac{1}{2^{\sqrt{2}}}$ 이고
$\frac{1}{2} > \frac{1}{2^{\sqrt{2}}}$ 이므로 $\log_{2}\sqrt{2} > 2^{-\sqrt{2}}$ 이다.
따라서 $x_{2} < \sqrt{2}$ 이다.
종합하면 $\sqrt[3]{2} < x_{2} < \sqrt{2}$ 이다. (참)

$ㄷ.$ $y_{1} = x_{1}$ 이므로 $ㄱ$에서 $\frac{1}{2} < y_{1} < \frac{\sqrt{2}}{2}$ 이다.
$y_{2} = \log_{2}x_{2}$ 이고 $\sqrt[3]{2} < x_{2} < \sqrt{2}$ 이므로
$\log_{2}\sqrt[3]{2} < \log_{2}x_{2} < \log_{2}\sqrt{2}$ 에서
$\frac{1}{3} < y_{2} < \frac{1}{2}$ 이다.
따라서 $y_{1} - y_{2} < \frac{\sqrt{2}}{2} - \frac{1}{3} = \frac{3\sqrt{2}-2}{6}$ 이다. (참)

따라서 옳은 것은 $ㄱ, ㄴ, ㄷ$ 이다.
