## 문제
$n$이 $2$ 이상의 자연수일 때 $\sum_{k=1}^{n} k ({}_{n}C_{k})^{2}$의 값을 구하는 과정이다.

$$ \begin{array}{|l|} \hline \text{두 다항식의 곱 } (a_{0} + a_{1}x + \cdots + a_{n-1}x^{n-1})(b_{0} + b_{1}x + \cdots + b_{n}x^{n}) \text{에서 } x^{n-1}\text{의 계수는} \\ a_{0}b_{n-1} + a_{1}b_{n-2} + \cdots + a_{n-1}b_{0} \cdots (*) \text{이다.} \\ (1+x)^{2n-1} = (1+x)^{n-1}(1+x)^{n}\text{의 좌변에서 } x^{n-1}\text{의 계수는 } (\text{가}) \text{ 이고,} \\ (*) \text{을 이용하여 우변에서 } x^{n-1}\text{의 계수를 구하면 } \sum_{k=1}^{n} ({}_{n-1}C_{k-1} \times (\text{나})) \text{이다.} \\ \text{따라서 } (\text{가}) = \sum_{k=1}^{n} ({}_{n-1}C_{k-1} \times (\text{나})) \text{이다.} \\ 1 \le k \le n \text{일 때, } k \times {}_{n}C_{k} = n \times {}_{n-1}C_{k-1} \text{이므로} \\ \sum_{k=1}^{n} k ({}_{n}C_{k})^{2} = \sum_{k=1}^{n} (n \times {}_{n-1}C_{k-1} \times (\text{나})) \\ = n \times \sum_{k=1}^{n} ({}_{n-1}C_{k-1} \times (\text{나})) = (\text{다}) \text{이다.} \\ \hline \end{array} $$

위의 과정에서 $(\text{가})$, $(\text{나})$, $(\text{다})$에 알맞은 것은?

## 보기
① $(\text{가}) \ {}_{2n}C_{n}, \ (\text{나}) \ {}_{n}C_{n-k+1}, \ (\text{다}) \ \frac{n}{2} \times {}_{2n}C_{n+1}$   ② $(\text{가}) \ {}_{2n-1}C_{n-1}, \ (\text{나}) \ {}_{n}C_{n-k+1}, \ (\text{다}) \ \frac{n}{2} \times {}_{2n}C_{n}$   ③ $(\text{가}) \ {}_{2n-1}C_{n-1}, \ (\text{나}) \ {}_{n}C_{n-k}, \ (\text{다}) \ \frac{n}{2} \times {}_{2n}C_{n}$   ④ $(\text{가}) \ {}_{2n}C_{n}, \ (\text{나}) \ {}_{n}C_{n-k+1}, \ (\text{다}) \ n \times {}_{2n}C_{n+1}$   ⑤ $(\text{가}) \ {}_{2n-1}C_{n-1}, \ (\text{나}) \ {}_{n}C_{n-k}, \ (\text{다}) \ n \times {}_{2n}C_{n}$

## 풀이
[정답] ③

두 다항식의 곱 $(a_{0} + a_{1}x + \cdots + a_{n-1}x^{n-1})(b_{0} + b_{1}x + \cdots + b_{n}x^{n})$에서 $x^{n-1}$의 계수는 $a_{0}b_{n-1} + a_{1}b_{n-2} + \cdots + a_{n-1}b_{0} \cdots (*)$이다.
등식 $(1+x)^{2n-1} = (1+x)^{n-1}(1+x)^{n}$의 좌변에서 $x^{n-1}$의 계수는 $(\text{가}) = {}_{2n-1}C_{n-1}$이다.
$(*)$을 이용하여 우변에서 $x^{n-1}$의 계수를 구하면, $(1+x)^{n-1}$의 전개식에서 $x^{k-1}$의 계수와 $(1+x)^{n}$의 전개식에서 $x^{(n-1)-(k-1)} = x^{n-k}$의 계수의 곱의 합이므로 $(\text{나}) = {}_{n}C_{n-k}$이다.
따라서 ${}_{2n-1}C_{n-1} = \sum_{k=1}^{n} ({}_{n-1}C_{k-1} \times {}_{n}C_{n-k})$이다.
한편 $1 \le k \le n$일 때, $k \times {}_{n}C_{k} = n \times {}_{n-1}C_{k-1}$이므로
$\sum_{k=1}^{n} k ({}_{n}C_{k})^{2} = \sum_{k=1}^{n} k \times {}_{n}C_{k} \times {}_{n}C_{k}$
$= \sum_{k=1}^{n} (n \times {}_{n-1}C_{k-1} \times {}_{n}C_{n-k})$
$= n \times \sum_{k=1}^{n} ({}_{n-1}C_{k-1} \times {}_{n}C_{n-k})$
$= n \times {}_{2n-1}C_{n-1}$
$= n \times \frac{(2n-1)!}{n!(n-1)!}$
$= \frac{n}{2} \times \frac{(2n)!}{n!n!}$
$= \frac{n}{2} \times {}_{2n}C_{n} = (\text{다})$이다.
