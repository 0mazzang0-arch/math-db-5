## 문제
두 상수 $a$, $b$ ($a < b$)에 대하여 함수 $f(x)$를
$$ f(x) = (x-a)(x-b)^2 $$
이라 하자. 함수 $g(x) = x^3 + x + 1$의 역함수 $g^{-1}(x)$에 대하여 합성함수 $h(x) = (f \circ g^{-1})(x)$가 다음 조건을 만족시킬 때, $f(8)$의 값을 구하시오.

$$ \begin{array}{|l|} \hline \text{(가) 함수 } (x-1)|h(x)| \text{ 가 실수 전체의 집합에서 미분가능하다.} \\ \text{(나) } h'(3) = 2 \\ \hline \end{array} $$

## 보기

## 풀이
$g^{-1}(x) = k(x)$라 하면 
$h(x) = (f \circ g^{-1})(x) = f(k(x)) = (k(x)-a)(k(x)-b)^2$
이때, 조건 (가)에서 함수 $(x-1)|h(x)|$ 즉, 
$(x-1)|(k(x)-a)(k(x)-b)^2| = (x-1)(k(x)-b)^2 |k(x)-a|$
절댓값 안의 값이 $(x-1)$의 인수를 가져야 실수 전체의 집합에서 미분가능하므로 
$k(1)-a = 0$
한편, $y = k(x)$는 $x = \{k(x)\}^3 + k(x) + 1$을 만족하므로 
$1 = \{k(1)\}^3 + k(1) + 1$
$k(1) = 0$
즉, $a = 0$
따라서 함수 $f(x)$는 
$f(x) = x(x-b)^2$ $\cdots$ ㉠ 
이 성립한다. 
또, 조건 (나)에서 $h'(3) = 2$를 만족하므로 $h(x) = f(k(x))$의 양변을 미분하면 
$h'(x) = f'(k(x)) \times k'(x)$
식에 $x=3$을 대입하면 
$f'(k(3)) \times k'(3) = 2$ $\cdots$ ㉠ 
그런데, $3 = \{k(3)\}^3 + k(3) + 1$에서 
$(k(3)-1)(\{k(3)\}^2 + k(3) + 2) = 0$
$\therefore k(3) = 1$ $\cdots$ ㉡ 
㉠을 미분하면 $f'(x) = (x-b)^2 + 2x(x-b)$이므로 
$f'(k(3)) = f'(1) = (1-b)^2 + 2(1-b) = (1-b)(3-b)$
또, $x = \{k(x)\}^3 + k(x) + 1$의 양변을 $x$에 대하여 미분하면 
$1 = (3 \{k(x)\}^2 + 1)k'(x)$
$x=3$을 대입하면 $1 = (3 \{k(3)\}^2 + 1)k'(3)$
㉡을 대입하면 $k'(3) = \frac{1}{4}$
㉠에 대입하면 $(1-b)(3-b) \times \frac{1}{4} = 2$
$b^2 - 4b + 3 = 8$, $b^2 - 4b - 5 = 0$
$(b+1)(b-5) = 0$
이때, 조건에서 $a < b$, 즉, $b > 0$ ($\because a = 0$)이므로 
$b = 5$
따라서 $f(x) = x(x-5)^2$이므로 $x=8$을 대입하면 
$f(8) = 8 \times 3^2 = 72$

[정답] $72$
