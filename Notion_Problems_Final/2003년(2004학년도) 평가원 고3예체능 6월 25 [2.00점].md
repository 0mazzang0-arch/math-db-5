## 문제
전체집합 $U = \{ 1, 2, 3, 4, 5, 6, 7, 8 \}$의 세 부분집합 $A$, $B$, $C$가 다음과 같다.
$A = \{ 1, 2, 3, 4 \}$
$B = \{ 3, 4, 5, 6 \}$
$C = \{ 5, 6, 7, 8 \}$
이때, 집합 $(A \cup B^c) - C$에 속하는 모든 원소의 합을 구하시오.

## 풀이
$B^c = \{ 1, 2, 7, 8 \}$이므로 
$A \cup B^c = \{ 1, 2, 3, 4, 7, 8 \}$이고, 
$(A \cup B^c) - C = \{ 1, 2, 3, 4 \}$이다. 
따라서 구하는 모든 원소의 합은 
$1 + 2 + 3 + 4 = 10$

[정답] $10$
