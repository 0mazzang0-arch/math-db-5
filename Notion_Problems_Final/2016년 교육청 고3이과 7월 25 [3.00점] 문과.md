## 문제
좌표평면 위의 곡선 $y = \frac{1}{3} x \sqrt{x}$ ($0 \le x \le 12$)에 대하여 $x = 0$에서 $x = 12$까지의 곡선의 길이를 $l$이라 할 때, $3l$의 값을 구하시오.

## 보기

## 풀이
$y' = \frac{1}{2} \sqrt{x}$ 이므로 
$l = \int_{0}^{12} \sqrt{1 + (y')^2} dx = \int_{0}^{12} \sqrt{1 + \frac{x}{4}} dx$

$\sqrt{1 + \frac{x}{4}} = t$라 놓으면 
$1 + \frac{x}{4} = t^2$, $\frac{1}{4} \frac{dx}{dt} = 2t$

$x = 0$일 때 $t = 1$, 
$x = 12$일 때 $t = 2$이므로 

$l = \int_{1}^{2} 8t^2 dt = \left[ \frac{8}{3} t^3 \right]_{1}^{2} = \frac{64}{3} - \frac{8}{3} = \frac{56}{3}$

따라서 $3l = 56$

정답: $56$
