## 문제
삼차함수 $P(x) = ax^3 + bx^2 + cx + d$가 $0 \leq x \leq 1$인 모든 실수 $x$에 대하여 $|P'(x)| \leq 1$을 만족할 때, $a$의 최댓값은? (단, $a$, $b$, $c$, $d$는 실수이다.)

## 보기
① $\frac{4}{3}$   ② $\frac{5}{3}$   ③ $2$   ④ $\frac{7}{3}$   ⑤ $\frac{8}{3}$

## 풀이
$P(x) = ax^3 + bx^2 + cx + d$라 하면,
$P'(x) = 3ax^2 + 2bx + c$이다.

각 함숫값을 구하면 다음과 같다.
$P'(0) = c$
$P'(1) = 3a + 2b + c$
$P'(\frac{1}{2}) = \frac{3}{4}a + b + c$

위의 식들을 이용하여 $a$를 $P'(x)$에 관한 식으로 나타내면 다음과 같다.
$a = \frac{2}{3} P'(1) + \frac{2}{3} P'(0) - \frac{4}{3} P'(\frac{1}{2})$

> [!example] 📸 (그림/그래프)

$0 \leq x \leq 1$에서 $|P'(x)| \leq 1$을 만족할 때, $a$가 최대가 되기 위해서는
$P'(0) = 1$, $P'(1) = 1$, $P'(\frac{1}{2}) = -1$이어야 한다.

따라서 $a$의 최댓값은
$\frac{2}{3} + \frac{2}{3} + \frac{4}{3} = \frac{8}{3}$이다.
