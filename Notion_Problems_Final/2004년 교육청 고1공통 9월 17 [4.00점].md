## 문제
두 집합 $A$, $B$ 가 
$A = \{ x \mid |x+1| \leq 2, x \text{ 는 실수 } \}$, 
$B = \{ x \mid (x+1)(x-2) \geq 0, x \text{ 는 실수 } \}$ 
일 때, $A \cap B = \{ x \mid x^2 + ax + b \leq 0 \}$ 이다. 
이때, 상수 $a$, $b$ 에 대하여 $a+b$ 의 값은?

## 보기
① $-4$   ② $-1$   ③ $2$   ④ $5$   ⑤ $7$

## 풀이
$|x+1| \leq 2$ 를 풀면, 
$-2 \leq x+1 \leq 2$ 
$\therefore -3 \leq x \leq 1 \cdots$ ㉠ 

또, 
$(x+1)(x-2) \geq 0$ 을 풀면, 
$x \leq -1$ 또는 $x \geq 2 \cdots$ ㉡ 

㉠, ㉡을 동시에 만족하는 $x$ 의 값의 범위는 
$-3 \leq x \leq -1$ 
이고, 이것을 해로 갖는 이차부등식은 
$(x+3)(x+1) \leq 0$ 
즉, $x^{2} + 4x + 3 \leq 0$ 

따라서 $a=4$, $b=3$ 이므로
$\therefore a+b = 4+3 = 7$

정답: ⑤
