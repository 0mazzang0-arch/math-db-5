## 문제
집합 $X = \{ x \mid 1 \le x \le 100, x \text{는 자연수} \}$, $Y = \{ y \mid y = x^{2} + x + 1, x \in X \}$ 에 대하여 집합 $Z = \{ z \mid z \in Y, z \text{는 } 3 \text{의 배수} \}$ 가 있다. 집합 $X - Z$ 의 원소의 개수를 구하시오.

## 보기

## 풀이
집합 $X$ 의 각 원소 $x$ 에 대하여 집합 $Y$ 의 원소를 차례로 구해보면 다음과 같다.
$x = 1 : 1^{2} + 1 + 1 = 3$
$x = 2 : 2^{2} + 2 + 1 = 7$
$x = 3 : 3^{2} + 3 + 1 = 13$
$x = 4 : 4^{2} + 4 + 1 = 21$
$x = 5 : 5^{2} + 5 + 1 = 31$
$x = 6 : 6^{2} + 6 + 1 = 43$
$x = 7 : 7^{2} + 7 + 1 = 57$
$x = 8 : 8^{2} + 8 + 1 = 73$
$x = 9 : 9^{2} + 9 + 1 = 91$
$x = 10 : 10^{2} + 10 + 1 = 111$
$\vdots$
$\therefore Y = \{ 3, 7, 13, 21, 31, 43, 57, 73, 91, 111, \dots, 10101 \}$
집합 $Y$ 의 원소 중 $3$ 의 배수인 것을 구해보면
$Z = \{ 3, 21, 57, 111, \dots, 10101 \}$
$\therefore X \cap Z = \{ 3, 21, 57 \}$
$\therefore n(X - Z) = n(X) - n(X \cap Z) = 100 - 3 = 97$

**[다른 풀이]**
$f(x) = x^{2} + x + 1 = (x + 2)(x - 1) + 3$ 으로 놓으면
$(x + 2)(x - 1)$ 이 $3$ 의 배수일 때, $f(x)$ 는 $3$ 의 배수이다.
즉, $x + 2$ 가 $3$ 의 배수일 때, $f(x)$ 는 $3$ 의 배수이다.
$x + 2$ 가 $3$ 의 배수인 경우는 $x = 1, 4, 7, 10, \dots$ 이고 이때 $f(x)$ 의 값은 $3, 21, 57, 111, \dots$ 임을 알 수 있다.
집합 $X$ 의 원소 중 집합 $Z$ 에 포함되는 원소는 $3, 21, 57$ 이다.
$\therefore n(X - Z) = 100 - 3 = 97$

정답: $97$
