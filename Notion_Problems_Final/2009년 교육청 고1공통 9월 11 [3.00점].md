## 문제
임의의 실수 $a$ 에 대하여 $a \times 0 = 0$ 임을 증명하는 과정이다. $(가)$, $(나)$, $(다)$에 알맞은 법칙을 고르시오.

$a \times 0 = a \times (0 + 0)$
$a \times 0 = a \times 0 + a \times 0$
양변에 $-a \times 0$ 을 더하면
$a \times 0 + (-a \times 0) = (a \times 0 + a \times 0) + (-a \times 0)$
$\dots (가)$
$a \times 0 + (-a \times 0) = a \times 0 + \{ a \times 0 + (-a \times 0) \}$
$\dots (나)$
$0 = a \times 0 + 0$
$\dots (다)$
$0 = a \times 0$

## 보기
① $분배법칙, 곱셈에 대한 역원, 덧셈에 대한 항등원$   ② $분배법칙, 덧셈에 대한 역원, 곱셈에 대한 항등원$   ③ $덧셈에 대한 결합법칙, 분배법칙, 덧셈에 대한 역원$   ④ $덧셈에 대한 결합법칙, 덧셈에 대한 역원, 덧셈에 대한 항등원$   ⑤ $덧셈에 대한 결합법칙, 곱셈에 대한 역원, 곱셈에 대한 항등원$

## 풀이
임의의 실수 $a$ 에 대하여 $a \times 0 = 0$ 임을 증명하는 과정에서 각 단계에 적용된 성질은 다음과 같다.

$(가)$ $(a \times 0 + a \times 0) + (-a \times 0) = a \times 0 + \{ a \times 0 + (-a \times 0) \}$ 이므로 **덧셈에 대한 결합법칙**이 이용되었다.
$(나)$ $a \times 0 + (-a \times 0) = 0$ 이므로 **덧셈에 대한 역원**의 성질이 이용되었다.
$(다)$ $a \times 0 + 0 = a \times 0$ 이므로 **덧셈에 대한 항등원**의 성질이 이용되었다.

따라서 $(가)$, $(나)$, $(다)$에 알맞은 법칙을 순서대로 나열한 것은 ④이다.

정답: ④
