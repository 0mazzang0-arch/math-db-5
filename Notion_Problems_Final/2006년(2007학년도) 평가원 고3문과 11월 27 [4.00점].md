## 문제
$0 < a < 1$ 인 $a$ 에 대하여 $10^a$ 을 $3$ 으로 나눌 때, 몫이 정수이고 나머지가 $2$ 가 되는 모든 $a$ 의 값의 합은?

## 보기
① $3 \log 2$   ② $6 \log 2$   ③ $1 + 3 \log 2$   ④ $1 + 6 \log 2$   ⑤ $2 + 3 \log 2$

## 풀이
[정답] ③ 

$10^a = 3 \cdot Q + 2$ ($Q$ 는 정수, $0 < a < 1$) 
$0 < a = \log (3Q + 2) < 1$ 
$1 < 3Q + 2 < 10$ 
따라서 $Q = 0, 1, 2$ 
그러므로 $a = \log 2, \log 5, \log 8$ 
따라서 $a$ 의 합 
$= \log 2 + \log 5 + \log 8 = 1 + 3 \log 2$
