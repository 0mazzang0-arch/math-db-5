## 문제
집합 $X = \{ 0, 2, 4 \}$에 대하여 $X$에서 $X$로의 함수 
$$ f(x) = \begin{cases} 3x + 2 & (x < 2) \\ x^2 + ax + b & (x \geq 2) \end{cases} $$
가 상수함수일 때, $a + b$의 값은? (단, $a$, $b$는 상수이다.)

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
함수 $f(x)$는 상수함수이므로 $f(0) = f(2) = f(4)$이다.
$f(0) = 3 \times 0 + 2 = 2$
$f(2) = 2^2 + 2a + b = 4 + 2a + b$
$f(4) = 4^2 + 4a + b = 16 + 4a + b$

$f(0) = f(2)$에서 $2 = 4 + 2a + b$이므로
$2a + b = -2$ $\dots$ ㉠ 

$f(0) = f(4)$에서 $2 = 16 + 4a + b$이므로
$4a + b = -14$ $\dots$ ㉡ 

㉠, ㉡을 연립하여 풀면
$a = -6$, $b = 10$
따라서 $a + b = -6 + 10 = 4$이다.

정답: ④
