## 문제
실수 $a$, $b$, $c$, $d$에 대하여 연산 $◎$을 다음과 같이 정의하자. 

$$(a, b) ◎ (c, d) = (ac + bd, ad + bc)$$

다음 중 참인 것을 모두 고르면? 

가. 연산 $◎$은 교환법칙이 성립한다. 
나. 연산 $◎$에 대한 항등원이 존재한다. 
다. 연산 $◎$에 대하여 $(0, 0)$이 아닌 모든 원소에 대한 역원이 존재한다. 

## 보기
① 가   ② 가, 나   ③ 가, 다   ④ 나, 다   ⑤ 가, 나, 다

## 풀이
가. 
$$(c, d) ◎ (a, b) = (ca + db, cb + da) = (ac + bd, ad + bc)$$ 
따라서 교환법칙이 성립한다. (참)

나. 항등원을 $(p, q)$라 하면 
$$(a, b) ◎ (p, q) = (ap + bq, aq + bp) = (a, b)$$ 
$ap + bq = a$, $aq + bp = b$를 만족해야 하므로 
$\therefore (p, q) = (1, 0)$ (참)

다. $(a, b)$의 역원을 $(m, n)$이라 하면 
$$(a, b) ◎ (m, n) = (am + bn, an + bm) = (1, 0)$$ 
$am + bn = 1 \cdots (1)$
$an + bm = 0 \cdots (2)$

$(1)$과 $(2)$를 연립하여 풀면 
$abm + b^{2}n = b \cdots (1')$
$abm + a^{2}n = 0 \cdots (2')$

$(1') - (2')$을 하면 
$(b^{2} - a^{2})n = b$
$(b + a)(b - a)n = b$
$\therefore n = \frac{b}{(b + a)(b - a)}$

마찬가지로 $m$을 구하면 
$\therefore m = \frac{a}{(a + b)(a - b)}$

이때, $(a + b)(a - b) = 0$, 즉 $a = b$ 또는 $a = -b$인 경우에는 역원이 존재하지 않는다. 
따라서 $(0, 0)$이 아닌 모든 원소에 대하여 역원이 존재하는 것은 아니다. (거짓)
