## 문제
$3^{n} = 31381059609$ 를 만족하는 양의 정수 $n$ 의 값을 구하시오.
(단, $0.47 < \log 3 < 0.48$)

## 보기

## 풀이
$3^{n} = 31381059609$ ($11$ 자리 정수) 이므로 지표의 성질에서 
$10 < \log 3^{n} < 11 \Leftrightarrow \frac{10}{\log 3} < n < \frac{11}{\log 3}$ 

$\therefore 21. \dots < n < 22. \dots$ 

$\therefore n = 22$ 

정답: $22$
