## 문제
방정식 $\sin^{2} x - \sin x = 1 - k$ 가 실근을 갖도록 하는 상수 $k$ 의 최댓값을 $M$, 최솟값을 $m$ 이라 할 때, $20M + m$ 의 값을 구하시오. (단, $0 \le x < 2\pi$ 이다.)

## 보기


## 풀이
$\sin x = t$ ($-1 \le t \le 1$)라 하면 
이차방정식 $t^{2} - t = 1 - k$ 가 $-1 \le t \le 1$ 에서 실근을 가져야 한다. 
$f(t) = t^{2} - t$ ($-1 \le t \le 1$)라 하면 그림에서 

> [!example] 📸 (그림/그래프)

$-\frac{1}{4} \le 1 - k \le 2$ 
$-1 \le k \le \frac{5}{4}$ 이므로 
$M = \frac{5}{4}$, $m = -1$ 

$\therefore 20M + m = 24$

정답: $24$
