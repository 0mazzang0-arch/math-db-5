## 문제
자연수 $n$이 $2 \le n \le 11$일 때, $-n^{2} + 9n - 18$의 $n$ 제곱근 중에서 음의 실수가 존재하도록 하는 모든 $n$의 값의 합은?

## 보기
① $31$   ② $33$   ③ $35$   ④ $37$   ⑤ $39$

## 풀이
$-n^{2} + 9n - 18$의 $n$ 제곱근 중에서 음의 실수가 존재하는 경우는 다음과 같다.

(ⅰ) $n = 2k + 1$ ($n$이 홀수)일 때,
$-n^{2} + 9n - 18 < 0$이어야 하므로
$-n^{2} + 9n - 18$
$= -(2k + 1)^{2} + 9(2k + 1) - 18$
$= -2(2k^{2} - 7k + 5) < 0$
$(k - 1)(2k - 5) > 0$에서
$\therefore k < 1, k > \frac{5}{2}$
$2 \le n \le 11$을 만족하므로 $2 \le 2k + 1 \le 11$에서
$\frac{1}{2} \le k \le 5$
$k$는 자연수이므로 $k = 3, 4, 5$
따라서 $n = 7, 9, 11$

(ⅱ) $n = 2k$ ($n$이 짝수)일 때,
$-n^{2} + 9n - 18 > 0$이어야 하므로
$-n^{2} + 9n - 18$
$= -(2k)^{2} + 9(2k) - 18$
$= -2(2k^{2} - 9k + 9) > 0$
$(2k - 3)(k - 3) < 0$에서
$\therefore \frac{3}{2} < k < 3$
$2 \le n \le 11$을 만족하므로 $2 \le 2k \le 11$에서
$1 \le k \le \frac{11}{2}$
$k$는 자연수이므로 $k = 2$
따라서 $n = 4$

(ⅰ), (ⅱ)에서 만족하는 모든 $n$의 값은 $4, 7, 9, 11$이므로 그 합은
$4 + 7 + 9 + 11 = 31$

정답: ①
