## 문제
좌표평면 위의 점 $(x, y)$ 를 행렬 $\begin{pmatrix} \begin{array}{c} x \\ y \end{array} \end{pmatrix}$ 로 나타내기로 하자. 영역 $D = \{ (x, y) | 0 \le x \le 1, 0 \le y \le 1 \}$ 에 속하는 임의의 두 점 $(a, b)$, $(c, d)$ 에 대응하는 행렬 $A = \begin{pmatrix} \begin{array}{c} a \\ b \end{array} \end{pmatrix}$, $B = \begin{pmatrix} \begin{array}{c} c \\ d \end{array} \end{pmatrix}$ 에 대하여 $A+B = \begin{pmatrix} \begin{array}{c} p \\ q \end{array} \end{pmatrix}$ 라 할 때, 점 $(p, q)$ 가 나타내는 영역의 넓이는?

## 보기
① $2$   ② $\frac{5}{2}$   ③ $3$   ④ $\frac{7}{2}$   ⑤ $4$

## 풀이
[정답] ⑤ 

$A+B= \begin{pmatrix} \begin{array}{c} a+c \\ b+d \end{array} \end{pmatrix} = \begin{pmatrix} \begin{array}{c} p \\ q \end{array} \end{pmatrix}$ 

$0 \le a \le 1$, $0 \le b \le 1$, $0 \le c \le 1$, $0 \le d \le 1$ 이므로 
$0 \le a+c \le 2$, $0 \le b+d \le 2$ 

$\therefore 0 \le p \le 2$, $0 \le q \le 2$ 

따라서 구하는 영역의 넓이는 $2 \times 2 = 4$ 이다.
