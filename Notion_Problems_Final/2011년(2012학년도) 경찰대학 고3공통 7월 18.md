## 문제
자연수 $n$을 삼진법으로 나타내어 $(a_{k} a_{k-1} \cdots a_{1})_{(3)}$이라 할 때, $b_{n} = \sum_{j=1}^{k} \frac{a_{j}}{3^{j}}$이라 하자. $\sum_{n=1}^{80} b_{n}$의 값은?

## 보기
① $\frac{81}{4}$   ② $\frac{80}{3}$   ③ $27$   ④ $40$   ⑤ $\frac{81}{2}$

## 풀이
[정답] ④

$n=1$ 부터 $n=80$ 까지의 자연수를 $3$ 진법으로 나타내면 $4$ 자리 이하의 수가 된다.
이때, 삼진법의 수 $1_{(3)}$, $12_{(3)}$, $120_{(3)}$ 등을 $0001_{(3)}$, $0012_{(3)}$, $0120_{(3)}$ 이라 생각하더라도 $j$ 가 $4$ 이하이면 $b_{n}= \sum_{j=1}^{k} \frac{a_{j}}{3^{j}} = \sum_{j=1}^{4} \frac{a_{j}}{3^{j}} = \frac{a_{1}}{3} + \frac{a_{2}}{3^{2}} + \frac{a_{3}}{3^{3}} + \frac{a_{4}}{3^{4}}$ 가 성립하므로 $n=1$ 부터 $n=80$ 까지의 자연수를 $a_{1} a_{2} a_{3} a_{4(3)}$ 라 나타내자. 


단, $a_{n} (n=1, 2, 3, 4)$ 는 $0$ 또는 $1$ 또는 $2$ 의 값을 가지며 모두 $0$ 인 경우는 제외한다.

이때, $n$ 중 $a_{1}=1$ 을 만족하는 것의 개수는 $a_{2}$, $a_{3}$, $a_{4}$ 를 정하는 방법의 수가 각각 $3$ 가지씩이므로 총 $3^{3}$ 가지 존재한다.
마찬가지로 $a_{1}=2$ 를 만족하는 $n$ 의 개수도 $3^{3}$ 가지 존재하고, $a_{2}=1, 2$, $a_{3}=1, 2$, $a_{4}=1, 2$ 를 만족하는 $n$ 의 개수도 각각 모두 $3^{3}$ 가지씩 존재한다.

$\therefore \sum_{n=1}^{80} b_{n}$
$= \sum_{n=1}^{80} \left( \frac{a_{1}}{3} + \frac{a_{2}}{3^{2}} + \frac{a_{3}}{3^{3}} + \frac{a_{4}}{3^{4}} \right)$
$= 3^{3} \times (1+2) \times \left( \frac{1}{3} + \frac{1}{3^{2}} + \frac{1}{3^{3}} + \frac{1}{3^{4}} \right)$
$= 81 \times \left( \frac{27+9+3+1}{81} \right)$
$= 27+9+3+1 = 40$
