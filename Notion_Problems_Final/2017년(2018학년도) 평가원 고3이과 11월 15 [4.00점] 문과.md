## 문제
함수 $f(x)$가 $f(x) = \int_{0}^{x} \frac{1}{1+e^{-t}} dt$일 때, $(f \circ f)(a) = \ln 5$를 만족시키는 실수 $a$의 값은?

## 보기
① $\ln 11$   ② $\ln 13$   ③ $\ln 15$   ④ $\ln 17$   ⑤ $\ln 19$

## 풀이
$f(x) = \int_{0}^{x} \frac{1}{1+e^{-t}} dt = \int_{0}^{x} \frac{e^{t}}{1+e^{t}} dt$에서 $1+e^{t} = s$로 놓으면 $e^{t} = \frac{ds}{dt}$이고, $t=0$ 일 때 $s=2$ 이고, $t=x$ 일 때 $s=1+e^{x}$이므로
$f(x) = \int_{2}^{1+e^{x}} \frac{1}{s} ds = [\ln s]_{2}^{1+e^{x}} = \ln(1+e^{x}) - \ln 2 = \ln \frac{1+e^{x}}{2}$

$(f \circ f)(a) = f(f(a)) = f\left(\ln \frac{1+e^{a}}{2}\right) = \ln \frac{1+e^{\ln \frac{1+e^{a}}{2}}}{2}$
이때, $e^{\ln \frac{1+e^{a}}{2}} = \frac{1+e^{a}}{2}$이므로
$(f \circ f)(a) = \ln \frac{1 + \frac{1+e^{a}}{2}}{2} = \ln \frac{3+e^{a}}{4}$

한편, $(f \circ f)(a) = \ln 5$이므로
$\ln \frac{3+e^{a}}{4} = \ln 5$
이때, $y=\ln x$는 일대일 함수이므로
$\frac{3+e^{a}}{4} = 5$, $e^{a} = 17$
따라서 $a = \ln 17$

정답: ④
