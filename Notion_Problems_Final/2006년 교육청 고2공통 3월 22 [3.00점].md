## 문제
$a$, $b$가 실수일 때, 이차방정식 $x^{2} + ax + b = 0$의 한 근이 $4 + 3i$이다. $a + b$의 값을 구하시오. (단, $i = \sqrt{-1}$)

## 풀이
이차방정식 $x^{2} + ax + b = 0$의 계수가 실수이므로 한 근이 $4 + 3i$이면 다른 한 근은 $4 - 3i$이다. 근과 계수와의 관계에 의하여

$-a = (4 + 3i) + (4 - 3i) = 8$
$\therefore a = -8$

$b = (4 + 3i)(4 - 3i) = 25$
따라서 $a + b = 17$
