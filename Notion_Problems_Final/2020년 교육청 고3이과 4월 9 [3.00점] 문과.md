## 문제
$0 < x \le 2\pi$ 일 때, 방정식 $\sin^{2} x = \cos^{2} x + \cos x$ 와 부등식 $\sin x > \cos x$ 를 동시에 만족시키는 모든 $x$ 의 값의 합은?

## 보기
① $\frac{4}{3}\pi$   ② $\frac{5}{3}\pi$   ③ $2\pi$   ④ $\frac{7}{3}\pi$   ⑤ $\frac{8}{3}\pi$

## 풀이
$\sin^{2} x = \cos^{2} x + \cos x$ 에서
$1 - \cos^{2} x = \cos^{2} x + \cos x$
$2\cos^{2} x + \cos x - 1 = 0$
$(2\cos x - 1)(\cos x + 1) = 0$ 이므로
$\cos x = \frac{1}{2}$ 또는 $\cos x = -1$

$0 < x \le 2\pi$ 이므로
$\cos x = \frac{1}{2}$ 에서 $x = \frac{\pi}{3}$ 또는 $x = \frac{5}{3}\pi$
$\cos x = -1$ 에서 $x = \pi$

이때 부등식 $\sin x > \cos x$ 를 확인하면
$x = \frac{\pi}{3}$ 일 때, $\sin \frac{\pi}{3} > \cos \frac{\pi}{3}$ (참)
$x = \frac{5}{3}\pi$ 일 때, $\sin \frac{5}{3}\pi < \cos \frac{5}{3}\pi$ (거짓)
$x = \pi$ 일 때, $\sin \pi > \cos \pi$ (참)

따라서 구하는 모든 $x$ 의 값의 합은
$\frac{\pi}{3} + \pi = \frac{4}{3}\pi$

정답: ①
