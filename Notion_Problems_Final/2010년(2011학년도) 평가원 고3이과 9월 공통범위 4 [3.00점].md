## 문제
좌표평면 위의 점 $( -1, 0 )$ 에서 쌍곡선 $x^{2} - y^{2} = 2$ 에 그은 접선의 방정식을 $y = mx + n$ 이라 할 때, $m^{2} + n^{2}$ 의 값은? (단, $m$, $n$ 은 상수이다.)

## 보기
① $\frac{5}{2}$   ② $3$   ③ $\frac{7}{2}$   ④ $4$   ⑤ $\frac{9}{2}$

## 풀이
접선 $y = mx + n$ 이 점 $( -1, 0 )$ 을 지나므로 $0 = -m + n$ 에서 $m = n$ 이다. 
직선 $y = mx + m$ 과 쌍곡선 $x^{2} - y^{2} = 2$ 가 한 점에서 만나므로 방정식 $x^{2} - ( mx + m )^{2} = 2$ 에서
$( 1 - m^{2} ) x^{2} - 2m^{2} x - m^{2} - 2 = 0$
이 이차방정식의 판별식을 $D$ 라 하면
$\frac{D}{4} = m^{4} + ( 1 - m^{2} ) ( m^{2} + 2 ) = 0$
$m^{4} + m^{2} + 2 - m^{4} - 2m^{2} = 0$
$m^{2} = 2$
$\therefore m^{2} + n^{2} = m^{2} + m^{2} = 2 + 2 = 4$

따라서 정답은 ④이다.
