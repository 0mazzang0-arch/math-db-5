## 문제
전체집합 $U$의 임의의 세 부분집합 $A$, $B$, $C$에 대하여 $A \subset B$인 것은 $A - B = \emptyset$이기 위한 $(가)$ 조건이고, $A \cup B = A \cup C$인 것은 $B = C$이기 위한 $(나)$ 조건이다. 이때, $(가)$, $(나)$에 알맞은 것은? (단, $U \neq \emptyset$이다.)

## 보기
① $(가): \text{충분}, (나): \text{충분}$   ② $(가): \text{충분}, (나): \text{필요}$   ③ $(가): \text{필요}, (나): \text{충분}$   ④ $(가): \text{필요}, (나): \text{필요충분}$   ⑤ $(가): \text{필요충분}, (나): \text{필요}$

## 풀이
$i)$ $A \subset B \iff A - B = \emptyset$ 이므로 $(가)$는 필요충분조건이다.

$ii)$ $B = C$ 이면 $A \cup B = A \cup C$ 가 성립한다. 
그러나 $A = \{ 1, 2, 3 \}$, $B = \{ 2, 3 \}$, $C = \{ 3 \}$ 일 때, $A \cup B = \{ 1, 2, 3 \}$, $A \cup C = \{ 1, 2, 3 \}$ 이 되어 $A \cup B = A \cup C$ 는 성립하지만 $B \neq C$ 이다. 
그러므로 $A \cup B = A \cup C$ 인 것은 $B = C$ 이기 위한 필요조건이다. 
즉, $(나)$는 필요조건이다.

따라서 $(가)$는 필요충분조건, $(나)$는 필요조건이다.

정답: ⑤
