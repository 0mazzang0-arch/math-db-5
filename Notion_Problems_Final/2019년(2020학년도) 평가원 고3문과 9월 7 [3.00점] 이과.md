## 문제
등차수열 $\{ a_{n} \}$ 에 대하여 
$a_{1} = a_{3} + 8$, $2a_{4} - 3a_{6} = 3$ 
일 때, $a_{k} < 0$ 을 만족시키는 자연수 $k$ 의 최솟값은? 

## 보기
① $8$   ② $10$   ③ $12$   ④ $14$   ⑤ $16$

## 풀이
$a_{n} = a + (n - 1)d$ 라 하자.
$a_{1} = a_{3} + 8$ 은
$a = a + 2d + 8$ 이고
$d = -4$ (공차)

$2a_{4} - 3a_{6} = 3$ 은
$2(a + 3d) - 3(a + 5d) = 3$
$2a + 6d - 3a - 15d = 3$
$-a - 9d = 3$ 이 된다.

$d = -4$ 이므로
$-a - 9(-4) = 3$
$-a + 36 = 3$
$a = 33$ 임을 알 수 있다. 

즉, 일반항 $a_{n}$ 은
$a_{n} = 33 - 4(n - 1) = 37 - 4n$

$a_{k} < 0$ 에서
$37 - 4k < 0$
$\frac{37}{4} < k$
$9.25 < k$

이를 만족시키는 자연수 $k$ 의 최솟값은 $10$ 이다.

정답: ②
