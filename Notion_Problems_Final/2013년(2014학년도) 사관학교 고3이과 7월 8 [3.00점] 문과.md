## 문제
$a_{n} = \int_{0}^{n} f(x) dx$, $b_{n} = \int_{n-1}^{n} g(x) dx$ 이고 $f(x) = g(x)$, $b_{n} = 2n + 3$ 일 때, $a_{10}$ 의 값은?

## 보기
① $110$   ② $120$   ③ $130$   ④ $140$   ⑤ $150$

## 풀이
$a_{n} - a_{n-1} = \int_{0}^{n} f(x) dx - \int_{0}^{n-1} f(x) dx = \int_{n-1}^{n} f(x) dx$
이므로 $f(x) = g(x)$ 일 때, $a_{1} = b_{1}$ 이고 $a_{n} - a_{n-1} = b_{n}$ 이 성립한다.

$\therefore a_{10} = a_{1} + \sum_{k=1}^{9} (a_{k+1} - a_{k})$
$= b_{1} + \sum_{k=1}^{9} b_{k+1} = \sum_{k=1}^{10} b_{k}$
$= \sum_{k=1}^{10} (2k + 3)$ (이유: $b_{n} = 2n + 3$)
$= \frac{10(5 + 23)}{2} = 140$

정답: ④
