## 문제
구간 $[0, a]$에서 정의된 확률변수 $X$의 확률밀도함수가 연속이다. 확률변수 $X$가 다음 조건을 만족시킬 때, 상수 $k$의 값은?

$$ \begin{array}{l} \text{(가) } 0 \le x \le a \text{인 모든 } x \text{에 대하여 } P(0 \le X \le x) = kx^2 \text{이다.} \\ \text{(나) } E(X) = 1 \end{array} $$

## 보기
① $\frac{9}{16}$   ② $\frac{4}{9}$   ③ $\frac{1}{4}$   ④ $\frac{1}{9}$   ⑤ $\frac{1}{16}$

## 풀이
$P(0 \le X \le a) = 1$이므로
$ka^2 = 1$ $\cdots$ ㉠
확률변수 $X$의 확률밀도함수를 $f(x)$라 하면
$P(0 \le X \le x) = \int_{0}^{x} f(t)dt = kx^2$
등식의 양변을 $x$에 대하여 미분하면
$f(x) = 2kx$
$E(X) = \int_{0}^{a} xf(x)dx = \int_{0}^{a} 2kx^2 dx = \left[ \frac{2}{3}kx^3 \right]_{0}^{a} = \frac{2}{3}ka^3 = 1$
$\therefore ka^3 = \frac{3}{2}$ $\cdots$ ㉡
㉡ $\div$ ㉠에서
$a = \frac{3}{2}$
㉠에서 $\left( \frac{3}{2} \right)^2 k = 1$
$\frac{9}{4}k = 1 \therefore k = \frac{4}{9}$

정답: ②
