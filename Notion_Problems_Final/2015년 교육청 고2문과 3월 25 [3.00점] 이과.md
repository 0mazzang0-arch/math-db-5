## 문제
등차수열 $\{ a_{n} \}$ 에서 $a_{1} + 2 a_{10} = 34$, $a_{1} - a_{10} = -14$ 일 때, 첫째항부터 제 $10$ 항까지의 합을 구하시오.

## 보기

## 풀이
$a_{1} + 2 a_{10} = 34 \cdots \cdots$ ㉠
$a_{1} - a_{10} = -14 \cdots \cdots$ ㉡
㉠ $-$ ㉡에서 $3 a_{10} = 48$ 이므로
$a_{10} = 16$
$a_{1} = 2$
따라서 구하는 수열의 합은
$\frac{10 ( a_{1} + a_{10} )}{2} = \frac{10 \times 18}{2} = 90$

**[다른 풀이]**
등차수열 $\{ a_{n} \}$ 의 첫째항을 $a$, 공차를 $d$ 라 하면
$a_{1} - a_{10} = -14$ 에서
$-9 d = -14$
$\therefore d = \frac{14}{9}$
$a_{1} + 2 a_{10} = 34$ 에서
$a + 2 ( a + 9 d ) = 3 a + 18 d = 3 a + 28 = 34$
$\therefore a = 2$
따라서 구하는 수열의 합은
$\frac{10 ( 2 \times 2 + 9 \times \frac{14}{9} )}{2} = \frac{10 \times 18}{2} = 90$

정답: $90$
