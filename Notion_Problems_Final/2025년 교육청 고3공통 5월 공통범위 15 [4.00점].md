## 문제
최고차항의 계수가 $1$ 이고 $\lim_{x \to 0} \frac{f(x)}{x} = 1$ 인 사차함수 $f(x)$ 와 실수 전체의 집합에서 연속인 함수 $g(x)$ 가 모든 실수 $x$ 에 대하여
$$\{g(x) - x\}\{g(x) - f(x)\} = 0$$
을 만족시킨다. 함수 $g(x)$ 가 다음 조건을 만족시킬 때, 모든 $\frac{g(-2)}{g(3)}$ 의 값의 합은?

(가) $\lim_{x \to 2} \frac{g(x) - g(2)}{x - 2}$ 의 값은 존재하지 않는다.
(나) $x \ge a$ 인 모든 실수 $x$ 에 대하여 $g(-x) = -g(x)$ 를 만족시키는 실수 $a$ 의 최솟값은 $4$ 이다.

## 보기
① $-\frac{41}{3}$   ② $-13$   ③ $-\frac{37}{3}$   ④ $-\frac{35}{3}$   ⑤ $-11$

## 풀이
두 함수 $f(x)$, $g(x)$ 가 모든 실수 $x$ 에 대하여 $\{g(x) - x\}\{g(x) - f(x)\} = 0$ 이므로 $g(x) = x$ 또는 $g(x) = f(x)$ 이다.
함수 $g(x)$ 는 실수 전체의 집합에서 연속이므로 $x = 2$ 에서 연속이다.
$g(2) = 2$ 또는 $g(2) = f(2)$ 에서 $f(2) \neq 2$ 이면 곡선 $y = f(x)$ 와 직선 $y = x$ 가 $x = 2$ 에서 만나지 않으므로
$\lim_{x \to 2} \frac{g(x) - g(2)}{x - 2} = \lim_{x \to 2} \frac{x - 2}{x - 2} = 1$ 또는 $\lim_{x \to 2} \frac{g(x) - g(2)}{x - 2} = \lim_{x \to 2} \frac{f(x) - f(2)}{x - 2} = f'(2)$ 가 되어 조건 (가)를 만족시키지 않는다.
그러므로 $f(2) = 2$ 이다.

$\lim_{x \to 0} \frac{f(x)}{x} = 1$ 에서 $f(0) = 0$, $f'(0) = 1$ 이므로 곡선 $y = f(x)$ 위의 점 $(0, 0)$ 에서의 접선의 방정식은 $y = x$ 이다.
$h(x) = f(x) - x$ 라 하면 함수 $h(x)$ 는 최고차항의 계수가 $1$ 인 사차함수이고 $h(0) = h'(0) = h(2) = 0$ 이므로
$h(x) = x^{2}(x - 2)(x - b)$ ($b$ 는 상수)
$h(b) = 0$ 이므로 $f(b) = b$ 이다.

(ⅰ) $0 \le b \le 2$ 일 때
> [!example] 📸 (그림/그래프)

조건 (나)에 의하여 $g(-4) = -g(4)$ 이므로 $g(4) = 4$, $g(-4) = -4$ 이다.
$g(x)$ 는 실수 전체의 집합에서 연속이므로 $x \le 0$ 또는 $x \ge 2$ 인 모든 실수 $x$ 에 대하여 $g(x) = x$ 이다.
$x \ge 2$ 인 모든 실수 $x$ 에 대하여 $g(-x) = -g(x)$ 이므로 조건 (나)의 최솟값이 $4$ 라는 것에 모순이다.

(ⅱ) $b > 2$ 일 때
> [!example] 📸 (그림/그래프)

(ⅰ)과 같은 방법으로 함수 $g(x)$ 를 구하면 $x \le 0$ 또는 $x \ge b$ 인 모든 실수 $x$ 에 대하여 $g(x) = x$ 이다.
$2 \le x \le b$ 인 모든 실수 $x$ 에 대하여 $g(x) = x$ 이면 $x \ge 2$ 인 모든 실수 $x$ 에 대하여 $g(-x) = -g(x)$ 이므로 조건 (나)를 만족하지 않는다.
따라서 $2 \le x \le b$ 인 모든 실수 $x$ 에 대하여 $g(x) = f(x)$ 이다.
만약 $0 \le x \le 2$ 인 모든 실수 $x$ 에 대하여 $g(x) = f(x)$ 이면 $\lim_{x \to 2} \frac{g(x) - g(2)}{x - 2} = f'(2)$ 가 되어 조건 (가)를 만족시키지 않는다.
따라서 $0 \le x \le 2$ 인 모든 실수 $x$ 에 대하여 $g(x) = x$ 이다.
$x \ge b$ 인 모든 실수 $x$ 에 대하여 $g(-x) = -g(x)$ 이므로 조건 (나)에 의하여 $b = 4$ 이다.
$f(x) = x^{2}(x - 2)(x - 4) + x$
$$ g(x) = \begin{cases} x & (x < 2 \text{ 또는 } x > 4) \\ x^2(x - 2)(x - 4) + x & (2 \le x \le 4) \end{cases} $$
$\frac{g(-2)}{g(3)} = \frac{-2}{3^{2}(3 - 2)(3 - 4) + 3} = \frac{-2}{-9 + 3} = \frac{-2}{-6} = \frac{1}{3}$

(ⅲ) $b < 0$ 일 때
> [!example] 📸 (그림/그래프)

(ⅱ)와 같은 방법으로 함수 $g(x)$ 를 구하면 $x \le b$ 또는 $x \ge 2$ 인 모든 실수 $x$ 에 대하여 $g(x) = x$ 이고, $b < x < 2$ 인 모든 실수 $x$ 에 대하여 $g(x) = f(x)$ 이다.
$-2 \le b < 0$ 이면 $x \ge 2$ 인 모든 실수 $x$ 에 대하여 $g(-x) = -g(x)$ 이므로 조건 (나)의 최솟값이 $4$ 라는 것에 모순이다.
따라서 $b < -2$ 이다.
$x \ge -b$ 인 모든 실수 $x$ 에 대하여 $g(-x) = -g(x)$ 이므로 조건 (나)에 의하여 $-b = 4$, 즉 $b = -4$ 이다.
$f(x) = x^{2}(x - 2)(x + 4) + x$
$$ g(x) = \begin{cases} x & (x \le -4 \text{ 또는 } x \ge 2) \\ x^{2}(x - 2)(x + 4) + x & (-4 < x < 2) \end{cases} $$
$\frac{g(-2)}{g(3)} = \frac{(-2)^{2}(-2 - 2)(-2 + 4) + (-2)}{3} = \frac{4(-4)(2) - 2}{3} = \frac{-32 - 2}{3} = -\frac{34}{3}$

(ⅰ), (ⅱ), (ⅲ)에 의하여 모든 $\frac{g(-2)}{g(3)}$ 의 값의 합은
$\frac{1}{3} + \left( -\frac{34}{3} \right) = -11$
