## 문제
두 집합 $A = \{ 2, 2a + 3 \}$, $B = \{ 5, 7, a^2 - 2 \}$ 에 대하여 $A \cap B = A$ 를 만족시키는 실수 $a$ 의 값은?

## 보기
① $-4$   ② $-2$   ③ $0$   ④ $2$   ⑤ $4$

## 풀이
$A \cap B = A$ 이면 $A \subset B$ 이다.
$2 \in A$ 이므로 $2 \in B$ 이어야 한다.
$B = \{ 5, 7, a^2 - 2 \}$ 에서 $a^2 - 2 = 2$ 이어야 하므로
$a^2 = 4$
$a = -2$ 또는 $a = 2$

(ⅰ) $a = -2$ 일 때
$A = \{ 2, -1 \}$, $B = \{ 5, 7, 2 \}$ 이므로
$A \cap B \neq A$

(ⅱ) $a = 2$ 일 때
$A = \{ 2, 7 \}$, $B = \{ 5, 7, 2 \}$ 이므로
$A \cap B = A$

따라서 $a = 2$ 이다.

정답: ④
