## 문제
그림과 같이 집합 $\{ x | x \geq 2 \}$ 에서 정의된 무리함수 $y = -\sqrt{2x + a} + 3$ 의 그래프가 점 $(2, b)$ 를 지날 때, 두 상수 $a$, $b$ 에 대하여 $a + b$ 의 값은?

> [!example] 📸 (그림/그래프)

## 보기
① $-2$   ② $-1$   ③ $0$   ④ $1$   ⑤ $2$

## 풀이
[정답] ②

$y = -\sqrt{2x + a} + 3 = -\sqrt{2 \left( x + \frac{a}{2} \right)} + 3$ 이므로 함수 $y = -\sqrt{2x + a} + 3$ 의 그래프는 함수 $y = -\sqrt{2x}$ 의 그래프를 $x$ 축의 방향으로 $-\frac{a}{2}$ 만큼, $y$ 축의 방향으로 $3$ 만큼 평행이동시킨 그래프이다.
주어진 그림에 의하여 $-\frac{a}{2} = 2$, $b = 3$ 이다.
따라서 $a = -4$ 이므로 $a + b = (-4) + 3 = -1$ 이다.
