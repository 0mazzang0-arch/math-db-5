## 문제
행렬 $A$ 는 $2$차정사각행렬이고 $A^{2}-3 A+E=O$ 를 만족한다. 위의 설명 중 옳은 것을 모두 고른 것은?
$$ \begin{array}{|l|} \hline \text{ㄱ. 임의의 } 2\text{차정사각행렬 } B, C \text{ 에 대하여 } A B=A C \text{ 이면 } B=C \text{ 이다.} \\ \text{ㄴ. } A-3 E \text{ 의 역행렬이 존재한다.} \\ \text{ㄷ. } A \begin{pmatrix} x \\ y \end{pmatrix} = \begin{pmatrix} 0 \\ 0 \end{pmatrix} \text{ 은 } x=0, y=0 \text{ 이외의 다른 해를 가질 수 있다.} \\ \hline \end{array} $$
(단, $E$ 는 $2$차단위행렬, $O$ 는 $2$차영행렬이다.)

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄱ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
$A^{2}-3 A+E=O$ 을 변형하면 $E=A(3 E-A)$ 이므로 $A$ 는 역행렬이 존재한다. 

ㄱ. $A B=A C$ 이면 $B=C$ 이다. $\because A$ 의 역행렬이 존재하므로 양변의 왼쪽에 $A^{-1}$를 곱하면 $B=C$이다. (참)

ㄴ. $E=(-A)(A-3 E)$ 이므로 $A-3 E$ 의 역행렬은 $-A$ 이다. 즉, 역행렬이 존재한다. (참)

ㄷ. $A$ 의 역행렬이 존재하므로 방정식 $A \begin{pmatrix} x \\ y \end{pmatrix} = \begin{pmatrix} 0 \\ 0 \end{pmatrix}$ 의 양변의 왼쪽에 $A^{-1}$를 곱하면 $\begin{pmatrix} x \\ y \end{pmatrix} = A^{-1} \begin{pmatrix} 0 \\ 0 \end{pmatrix} = \begin{pmatrix} 0 \\ 0 \end{pmatrix}$ 이 되어 해는 $x=0, y=0$ 만 가진다. (거짓)

따라서 옳은 것은 ㄱ, ㄴ이다.
