## 문제
$p < q$ 인 두 소수 $p$, $q$ 에 대하여 $p^{2} q < n \le p q^{2}$ 을 만족시키는 자연수 $n$ 의 개수가 $308$ 일 때, $p + q$ 의 값을 구하시오.

## 풀이
$p^{2} q < n \le p q^{2}$ 을 만족시키는 자연수 $n$ 의 개수는 $p q^{2} - p^{2} q$ 이므로 
$p q^{2} - p^{2} q = p q ( q - p ) = 308$ 

$p < q$ 이므로 $q - p > 0$ 이고 $p$, $q$ 가 자연수이므로 $q - p$ 도 자연수이다. 

$p < q$ 이고 $q - p < q$ 이므로 세 자연수 $p$, $q$, $q - p$ 중 $q$ 가 가장 큰 자연수이다. 

$308$ 을 소인수분해하면 
$308 = 2^{2} \times 7 \times 11$ 

$q$ 는 $308$ 의 가장 큰 소인수이므로 $q = 11$ 

$p$ 는 $308$ 의 소인수이고 $p < q$ 이므로 $p = 2$ 또는 $p = 7$ 

(ⅰ) $p = 2$ 인 경우 
$p q ( q - p ) = 2 \times 11 \times ( 11 - 2 ) = 198$ 

(ⅱ) $p = 7$ 인 경우 
$p q ( q - p ) = 7 \times 11 \times ( 11 - 7 ) = 308$ 

(ⅰ), (ⅱ)에 의하여 $p q ( q - p ) = 308$ 일 때 $p = 7$, $q = 11$ 
따라서 $p + q = 18$
