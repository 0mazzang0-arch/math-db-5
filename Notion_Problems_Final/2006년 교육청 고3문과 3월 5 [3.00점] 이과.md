## 문제
집합 $A = \left\{ x \mid x = \left( \frac{1}{256} \right)^{\frac{1}{n}}, n \text{은 } 0 \text{이 아닌 정수} \right\}$의 원소 중 자연수인 것의 개수는?

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
$\left( \frac{1}{256} \right)^{\frac{1}{n}} = \left( 2^{-8} \right)^{\frac{1}{n}} = 2^{-\frac{8}{n}}$에서 $2^{-\frac{8}{n}}$이 자연수이려면 $n = -1, -2, -4, -8$이어야 하고, 이 때 $2^{-\frac{8}{n}}$의 값은 각각 $256, 16, 4, 2$이다. 따라서 구하는 자연수의 개수는 $4$이다.
