## 문제
네 정수 $a$, $b$, $c$, $d$ 에 대하여 $a^{2} + b^{2} + c^{2} + d^{2} = 17$ 을 만족하는 $a$, $b$, $c$, $d$ 의 모든 순서쌍 $(a, b, c, d)$ 의 개수는?

## 보기
① $124$ ② $144$ ③ $164$ ④ $184$ ⑤ $204$

## 풀이
$a^2 + b^2 + c^2 + d^2 = 17$ 을 만족하는 정수의 제곱수들의 조합을 찾으면 다음과 같다.

$1)$ $16 + 1 + 0 + 0 = 17$ 인 경우:
$\{a^2, b^2, c^2, d^2\} = \{16, 1, 0, 0\}$
이때 $a, b, c, d$ 의 값은 $(\pm 4, \pm 1, 0, 0)$ 의 순열이다.
순서쌍의 개수는 다음과 같다.
$$\frac{4!}{2!} \times 2^2 = 48$$

$2)$ $9 + 4 + 4 + 0 = 17$ 인 경우:
$\{a^2, b^2, c^2, d^2\} = \{9, 4, 4, 0\}$
이때 $a, b, c, d$ 의 값은 $(\pm 3, \pm 2, \pm 2, 0)$ 의 순열이다.
순서쌍의 개수는 다음과 같다.
$$\frac{4!}{2!} \times 2^3 = 96$$

$\therefore 48 + 96 = 144$

정답: ②
