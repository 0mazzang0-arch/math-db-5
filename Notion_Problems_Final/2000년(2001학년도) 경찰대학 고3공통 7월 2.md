## 문제
$a = -b$ 일 때, $|a| = |b|$ 이지만 $a \neq b$ 이다. $a$, $b$ 는 실수이다. $a = b$ 이기 위한 충분조건이지만 필요조건이 아닌 것은?

## 보기
① $a^{2} + b^{2} = 0$   ② $a^{3} + b^{3} = 0$   ③ $a^{2} - b^{2} = 0$   ④ $a^{3} - b^{3} = 0$   ⑤ $|a| = |b|$

## 풀이
정답: ①

① $a^{2} + b^{2} = 0 \Rightarrow a = b$ (참), $a = b \Rightarrow a^{2} + b^{2} = 0$ (거짓)
$a = b = 1$ 일 때 $a^{2} + b^{2} \neq 0$ 이다.
따라서 충분조건이지만 필요조건은 아니다.

② $a^{3} + b^{3} = 0 \Rightarrow a = b$ (거짓), $a^{3} + b^{3} = 0 \Leftarrow a = b$ (거짓)
$a = 1$, $b = -1$ 일 때, $a^{3} + b^{3} = 0$ 이지만 $a \neq b$ 이다.

③ $a^{2} - b^{2} = 0 \Rightarrow a = b$ (거짓), $a^{2} - b^{2} = 0 \Leftarrow a = b$ (참)
$a = -b$ 인 경우 $a^{2} - b^{2} = 0$ 이지만 $a \neq b$ 이다. (필요조건)

④ $a^{3} - b^{3} = 0 \Rightarrow a = b$ (참), $a^{3} - b^{3} = 0 \Leftarrow a = b$ (참)
(필요충분조건)

⑤ $|a| = |b| \Rightarrow a = b$ (거짓), $|a| = |b| \Leftarrow a = b$ (참)
(필요조건)
