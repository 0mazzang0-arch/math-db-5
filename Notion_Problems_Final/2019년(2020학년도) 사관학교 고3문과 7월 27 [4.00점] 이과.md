## 문제
다항함수 $f(x)$ 가 모든 실수 $x$ 에 대하여 
$$\int_{1}^{x}(2x - 1)f(t)dt = x^{3} + ax + b$$ 
일 때, $40 \times f(1)$ 의 값을 구하시오. (단, $a$, $b$ 는 상수이다.)

## 보기

## 풀이
등식을 만족시키는 다항식 $f(x)$ 는 일차식이다. 
$f(x) = cx + d$ 라 하면 
$$\int_{1}^{x} (2x - 1) f(t) dt = x^{3} + ax + b$$ 
양변을 $x$ 에 대하여 미분하면 
$$2 \int_{1}^{x} f(t) dt + (2x - 1) f(x) = 3x^{2} + a$$ 
다시 $x$ 에 대하여 미분하면 
$$2f(x) + 2f(x) + (2x - 1) f'(x) = 6x$$ 
$$4f(x) + (2x - 1) f'(x) = 6x$$ 
이 식에 $f(x) = cx + d$ 를 대입하면 
$$4(cx + d) + c(2x - 1) = 6x$$ 
$$6cx + 4d - c = 6x$$ 
따라서 $6c = 6$, $4d - c = 0$ 이므로 
$c = 1$, $d = \frac{1}{4}$ 
$f(x) = x + \frac{1}{4}$ 
따라서 $40 \times f(1) = 40 \times \frac{5}{4} = 50$ 

정답: $50$
