## 문제
> [!example] 📸 (그림/그래프)

두 구 $S_{1} : x^{2} + y^{2} + z^{2} = 81$과 $S_{2} : x^{2} + (y - 5)^{2} + z^{2} = 56$이 있다. 두 구 $S_{1}$, $S_{2}$가 만나서 생기는 원 위의 한 점을 $P$라 하고, 점 $P$의 $xy$평면 위로의 정사영을 $P'$이라 하자. 구 $S_{1}$과 $y$축이 만나는 점을 각각 $Q$, $R$라 할 때, 사면체 $PQP'R$의 부피의 최댓값을 구하시오.

## 보기

## 풀이
두 구의 방정식을 각각 다음과 같이 놓자.
$x^{2} + y^{2} + z^{2} = 81 \cdots$ ㉠
$x^{2} + (y - 5)^{2} + z^{2} = 56 \cdots$ ㉡

㉠ $-$ ㉡에서
$y^{2} - (y - 5)^{2} = 81 - 56 = 25$
$10y - 25 = 25$
$10y = 50$, $y = 5$

이를 ㉠에 대입하면 두 구가 만나서 생기는 원의 방정식은 다음과 같다.
$x^{2} + z^{2} = 56$ ($y = 5$)

구 $S_{1}$과 $y$축의 교점 $Q$, $R$은 $x = 0$, $z = 0$을 ㉠에 대입하면 얻을 수 있다.
$y^{2} = 81 \Rightarrow y = \pm 9$
따라서 $Q(0, -9, 0)$, $R(0, 9, 0)$이다.

점 $P$를 원 위의 임의의 점 $(x, 5, z)$라 하면, 점 $P$의 $xy$평면 위로의 정사영 $P'$은 $(x, 5, 0)$이다.

삼각형 $\triangle P'QR$의 넓이는
$\frac{1}{2} \times |9 - (-9)| \times |x| = 9|x|$

사면체 $PQP'R$의 부피 $V$는 다음과 같다.
$V = \left| \frac{1}{3} \times 9x \times z \right| = 3|xz|$

산술·기하 평균의 관계에 의하여
$\frac{x^{2} + z^{2}}{2} \ge \sqrt{x^{2}z^{2}} = |xz|$
$\frac{56}{2} = 28 \ge |xz|$ (단, 등호는 $x^{2} = z^{2} = 28$일 때 성립)

따라서 부피 $V$의 최댓값은 다음과 같다.
$3 \times 28 = 84$

정답: $84$
