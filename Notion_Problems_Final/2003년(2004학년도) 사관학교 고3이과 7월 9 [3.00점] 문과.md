## 문제
$f(x)$의 부정적분 $F(x)$와 $g(x)$의 부정적분 $G(x)$가 다음 관계식을 만족한다.
$$ f(x) = \frac{G(x) + g(x)}{2}, g(x) = \frac{F(x) + f(x)}{2} $$
$f(0) = 0$, $g(0) = 2$일 때, $f(1) + g(1)$의 값은? (단, $e$는 자연로그의 밑이다.)

## 보기
① $-2$   ② $2$   ③ $1+e$   ④ $2+e$   ⑤ $2e$

## 풀이
주어진 조건에서 $F'(x) = f(x)$, $G'(x) = g(x)$이다.
두 식 $f(x) = \frac{G(x) + g(x)}{2}$와 $g(x) = \frac{F(x) + f(x)}{2}$를 변변 더하여 정리하면
$$ 2(f(x) + g(x)) = F(x) + G(x) + f(x) + g(x) $$
$F(x) + G(x) = f(x) + g(x) \cdots (1)$
식 $(1)$의 양변을 $x$에 관하여 미분하면
$F'(x) + G'(x) = (f(x) + g(x))'$
$\therefore (f(x) + g(x))' = f(x) + g(x)$
$f(x) + g(x) = ae^x$라 하고, 주어진 조건 $f(0) = 0$, $g(0) = 2$를 이용하면
$f(0) + g(0) = a$, $a = 2$
$\therefore f(x) + g(x) = 2e^x$
따라서, $f(1) + g(1) = 2e$

정답: ⑤
