## 문제
함수 $f(x) = \sin x + a \cos x$ 에 대하여 $\lim_{x \to \frac{\pi}{2}} \frac{f(x) - 1}{x - \frac{\pi}{2}} = 3$ 일 때, $f(\frac{\pi}{4})$ 의 값은? (단, $a$ 는 상수이다.)

## 보기
① $-2\sqrt{2}$   ② $-\sqrt{2}$   ③ $0$   ④ $\sqrt{2}$   ⑤ $2\sqrt{2}$

## 풀이
$f(x) = \sin x + a \cos x$ 에서 
$f(\frac{\pi}{2}) = \sin \frac{\pi}{2} + a \cos \frac{\pi}{2} = 1$ 이므로 
$\lim_{x \to \frac{\pi}{2}} \frac{f(x) - 1}{x - \frac{\pi}{2}} = \lim_{x \to \frac{\pi}{2}} \frac{f(x) - f(\frac{\pi}{2})}{x - \frac{\pi}{2}} = f'(\frac{\pi}{2})$ 
$f'(x) = \cos x - a \sin x$ 에서 
$f'(\frac{\pi}{2}) = \cos \frac{\pi}{2} - a \sin \frac{\pi}{2} = -a = 3$ 
따라서 $a = -3$ 이므로 
$f(x) = \sin x - 3 \cos x$ 
$f(\frac{\pi}{4}) = \sin \frac{\pi}{4} - 3 \cos \frac{\pi}{4} = \frac{\sqrt{2}}{2} - 3 \times \frac{\sqrt{2}}{2} = -\sqrt{2}$ 

정답: ②
