## 문제
모든 항이 자연수인 수열 $\{ a_{n} \}$ 이 다음 조건을 만족시킨다.

(가) 모든 자연수 $n$ 에 대하여 
$a_{n+1} = \begin{cases} \frac{1}{2} a_{n} + 2n & (a_{n} \text{ 이 } 4\text{ 의 배수인 경우}) \\ a_{n} + 2n & (a_{n} \text{ 이 } 4\text{ 의 배수가 아닌 경우}) \end{cases}$

(나) $a_{3} > a_{5}$

$50 < a_{4} + a_{5} < 60$ 이 되도록 하는 $a_{1}$ 의 최댓값과 최솟값을 각각 $M$, $m$ 이라 할 때, $M+m$ 의 값은?

## 보기
① $224$   ② $228$   ③ $232$   ④ $236$   ⑤ $240$

## 풀이
[정답] ②

조건 (나)에서 $a_{3} > a_{5}$ 이므로 $a_{3}$ 이 $4$ 의 배수인 경우와 $4$ 의 배수가 아닌 경우로 나누어 생각하자.

(ⅰ) $a_{3}$ 이 $4$ 의 배수인 경우
$a_{3} = 4k$ ($k$ 는 자연수)라 하면 $a_{4} = 2k + 6$
$k$ 가 홀수일 때 $a_{4}$ 는 $4$ 의 배수이고
$a_{5} = k + 11$, $a_{4} + a_{5} = 3k + 17$ 이므로
$50 < 3k + 17 < 60$, $a_{3} > a_{5}$ 에서 $k > \frac{11}{3}$
$k$ 는 홀수이므로 $k = 13$ 이고 $a_{3} = 52$

$k$ 가 짝수일 때 $a_{4}$ 는 $4$ 의 배수가 아니고
$a_{5} = 2k + 14$, $a_{4} + a_{5} = 4k + 20$ 이므로
$50 < 4k + 20 < 60$, $a_{3} > a_{5}$ 에서 $k > 7$
$k$ 는 짝수이므로 $k = 8$ 이고 $a_{3} = 32$

$a_{3} = 52$ 또는 $a_{3} = 32$
$a_{3} = 52$ 인 경우 $a_{2} = 96$ 이고
$a_{1} = 94$ 또는 $a_{1} = 188$
$a_{3} = 32$ 인 경우 $a_{2} = 56$ 이고
$a_{1} = 54$ 또는 $a_{1} = 108$

(ⅱ) $a_{3}$ 이 $4$ 의 배수가 아닌 경우
$a_{3} = 4k - 1$ 또는 $a_{3} = 4k - 3$ ($k$ 는 자연수)일 때
$a_{3}$, $a_{4}$, $a_{5}$ 는 모두 홀수이고 $a_{5} = a_{4} + 8 = a_{3} + 14 > a_{3}$ 이므로 조건 (나)에 모순이다.

$a_{3} = 4k - 2$ ($k$ 는 자연수) 일 때 $a_{4} = 4k + 4$, $a_{5} = 2k + 10$ 이고 $a_{4} + a_{5} = 6k + 14$ 이므로
$50 < 6k + 14 < 60$, $a_{3} > a_{5}$ 에서 $k > 6$, 이때 $k = 7$ 이므로 $a_{3} = 26$
$a_{2} = 22$ 또는 $a_{2} = 44$ 이다.
$a_{2} = 22$ 인 경우 $a_{1} = 40$
$a_{2} = 44$ 인 경우 $a_{1} = 42$ 또는 $a_{1} = 84$

(ⅰ), (ⅱ)에서 $M = 188$, $m = 40$ 이고 $M + m = 228$
