## 문제
그림과 같이 이차함수 $y = a x^{2} + b x + c$의 그래프가 $x$축과 두 점 $(-1, 0)$, $(3, 0)$에서 만난다. 다음 <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $a$, $b$, $c$는 상수이다.)

$$ \begin{array}{|l|} \hline \text{ㄱ. } ab > 0 \\ \text{ㄴ. } a + b + c > 0 \\ \text{ㄷ. 이 함수의 그래프는 점 } (2, c)\text{를 지난다.} \\ \hline \end{array} $$



> [!example] 📸 (그림/그래프)

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄷ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
[정답] ④

ㄱ. 그래프가 위로 볼록한 포물선이므로 $a < 0$이다.
대칭축의 방정식은 $x = -\frac{b}{2a}$이고, $x$축과 두 점 $(-1, 0)$, $(3, 0)$에서 만나므로 대칭축은 $x = \frac{-1 + 3}{2} = 1$이다.
$x = -\frac{b}{2a} = 1 > 0$에서 $a < 0$이므로 $b > 0$이다.
따라서 $ab < 0$이다. (거짓)

ㄴ. $x = 1$일 때의 함숫값은 $a(1)^{2} + b(1) + c = a + b + c$이다.
대칭축 $x = 1$은 두 절편 $-1$과 $3$ 사이에 있으며, 위로 볼록한 그래프의 꼭짓점의 $y$좌표에 해당하므로 양수이다.
따라서 $a + b + c > 0$이다. (참)

ㄷ. 이차함수 $y = a x^{2} + b x + c$의 그래프는 대칭축 $x = 1$에 대하여 대칭이다.
점 $(0, c)$는 $y$절편이며, $x = 0$은 대칭축 $x = 1$로부터 왼쪽으로 $1$만큼 떨어진 지점이다.
이때 대칭축으로부터 오른쪽으로 $1$만큼 떨어진 지점의 $x$좌표는 $2$이며, 대칭성에 의해 이 점의 $y$좌표 역시 $c$가 된다.
따라서 그래프는 점 $(2, c)$를 지난다. (참)

따라서 옳은 것은 ㄴ, ㄷ이다.
