## 문제
두 함수 $f(x)=x^{2}+ax+b$, $g(x)=\sin x$ 가 다음 조건을 만족시킬 때, $f(2)$ 의 값은? (단, $a$, $b$ 는 상수이고, $0 \leq a \leq 2$ 이다.)

$$ \begin{array}{|l|} \hline \text{[조건]} \\ \text{(가) } \{g(a\pi)\}^{2}=1 \\ \text{(나) } 0 \leq x \leq 2\pi \text{ 일 때, 방정식 } f(g(x))=0 \text{ 의 모든 해의 합은 } \frac{5}{2}\pi \text{ 이다.} \\ \hline \end{array} $$



> [!example] 📸 (그림/그래프)

## 보기
① $3$   ② $\frac{7}{2}$   ③ $4$   ④ $\frac{9}{2}$   ⑤ $5$

## 풀이
(가)에서 $\{g(a\pi)\}^{2}=1$ 이므로 $g(a\pi)=-1$ 또는 $g(a\pi)=1$ 이다.
$\sin(a\pi)=-1$ 에서 $a=\frac{3}{2}$, $\sin(a\pi)=1$ 에서 $a=\frac{1}{2}$ 이다.

(나)에서 방정식 $f(g(x))=0$ 의 해가 존재하므로 $-1 \leq t \leq 1$ 이고 $f(t)=0$ 인 실수 $t$ 가 존재한다.

$0 \leq x \leq 2\pi$ 에서 방정식 $g(x)=t$ 의 모든 해의 합은
$t=-1$ 일 때 $\frac{3}{2}\pi$,
$-1 < t \leq 0$ 일 때 $3\pi$,
$0 < t < 1$ 일 때 $\pi$,
$t=1$ 일 때 $\frac{\pi}{2}$ 이다.

$0 \leq x \leq 2\pi$ 일 때, 방정식 $f(g(x))=0$ 의 모든 해의 합이 $\frac{5}{2}\pi$ 이므로 방정식 $f(x)=0$ 은 두 실근 $-1$, $\alpha$ 를 가지고 $0 < \alpha < 1$ 이다. ($\because \frac{3}{2}\pi + \pi = \frac{5}{2}\pi$)

(ⅰ) $a=\frac{3}{2}$ 인 경우
$f(x)=x^{2}+\frac{3}{2}x+b$ 에서 $f(-1)=0$ 이므로
$f(-1)=b-\frac{1}{2}=0$ 즉, $b=\frac{1}{2}$
$f(x)=x^{2}+\frac{3}{2}x+\frac{1}{2}=(x+1)(x+\frac{1}{2})$ 에서
$f(x)=0$ 의 두 근은 $x=-1$ 또는 $x=-\frac{1}{2}$ 이므로 조건을 만족시키지 못한다.

(ⅱ) $a=\frac{1}{2}$ 인 경우
$f(x)=x^{2}+\frac{1}{2}x+b$ 에서 $f(-1)=0$ 이므로
$f(-1)=b+\frac{1}{2}=0$ 즉, $b=-\frac{1}{2}$
$f(x)=x^{2}+\frac{1}{2}x-\frac{1}{2}=(x+1)(x-\frac{1}{2})$ 에서
$f(x)=0$ 의 두 근은 $x=-1$ 또는 $x=\frac{1}{2}$ 이므로 조건을 만족시킨다.

(ⅰ), (ⅱ)에서 $f(x)=x^{2}+\frac{1}{2}x-\frac{1}{2}$ 이고,
$f(2)=2^{2}+\frac{1}{2} \cdot 2 - \frac{1}{2} = 4+1-\frac{1}{2}=\frac{9}{2}$ 이다.
