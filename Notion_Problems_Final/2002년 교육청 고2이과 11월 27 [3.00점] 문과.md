## 문제
양의 실수 $a$를 소수점 아래 첫째 자리에서 반올림했을 때의 정수 값을 $\langle a \rangle$로 나타내자.
방정식 $\langle 3x-1 \rangle=25$를 만족하는 $x$에 대하여 $\langle x \rangle$의 값을 구하시오.

## 보기

## 풀이
$\langle 3x-1 \rangle=25$의 정의에 의해
$24.5 \leq 3x-1 < 25.5$
$25.5 \leq 3x < 26.5$
$\therefore 8.5 \leq x < 8.83 \cdots$

따라서 $8.5 \leq x < 8.83 \cdots$ 범위를 만족하는 $x$를 소수점 아래 첫째 자리에서 반올림한 값은
$\langle x \rangle=9$

정답: $9$
