## 문제
양수 $x$에 대하여 $\log x$의 정수부분을 $f(x)$, 소수부분을 $g(x)$라 하자. 양수 $a$, $b$에 대하여 옳은 것을 <보기>에서 모두 고른 것은?

$$ \begin{array}{l} \text{<보 기>} \\ \text{ㄱ. } f(a^2) = 2f(a) \\ \text{ㄴ. } f(a^2) + g(a^2) = 2f(a) + 2g(a) \\ \text{ㄷ. } g(a) + g(b) = 1\text{ 이면 } ab\text{는 정수이다.} \end{array} $$

## 보기
① $ㄴ$   ② $ㄷ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
[정답] ① 

ㄱ. [반례] $a = 10^{\frac{3}{2}}$ 이면 
$f(a) = 1$, 
$f(a^2) = 3$ (거짓) 

ㄴ. $\log a = f(a) + g(a)$, 
$\log a^2 = 2 \log a = 2f(a) + 2g(a)$ 
$\log a^2 = f(a^2) + g(a^2)$ (참) 

ㄷ. [반례] $a = b = 10^{-\frac{1}{2}}$ 이면 
$g(a) + g(b) = 1$, 
$ab = \frac{1}{10}$ (거짓)
