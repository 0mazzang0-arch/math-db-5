## 문제
두 수 $a$, $b$가 다음 조건을 만족시킬 때, $\frac{b}{a}$의 값을 구하시오.

$$
\begin{array}{l}
\text{(가) } \log_{a} b = \frac{4}{3} \\
\text{(나) } \log_{9} a + \log_{3} b = 11
\end{array}
$$

## 보기


## 풀이
$\log_{a} b = \frac{4}{3}$에서 로그의 정의에 의하여 

$b = a^{\frac{4}{3}}$ $\dots$ $㉠$ 

$\log_{9} a + \log_{3} b$ 
$= \log_{9} a + \log_{9} b^{2}$ 
$= \log_{9} ab^{2} = 11$ 
에서 

$ab^{2} = 9^{11}$ $\dots$ $㉡$ 

$㉠$을 $㉡$에 대입하면 

$a ( a^{\frac{4}{3}} )^{2} = a^{\frac{11}{3}} = 9^{11}$ 
$\therefore a = 9^{3}$ 

$㉠$에서 
$b = a^{\frac{4}{3}} = ( 9^{3} )^{\frac{4}{3}} = 9^{4}$ 
$\therefore \frac{b}{a} = \frac{9^{4}}{9^{3}} = 9$
