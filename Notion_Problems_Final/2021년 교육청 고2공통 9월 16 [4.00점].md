## 문제
수열 $\{ a_{n} \}$을 $a_{n} = \sum_{k=1}^{n} \frac{1}{k}$이라 할 때, 다음은 모든 자연수 $n$에 대하여 등식
$a_{1} + 2a_{2} + 3a_{3} + \dots + na_{n} = \frac{n(n+1)}{4} ( 2a_{n+1} - 1 ) \dots (*)$
이 성립함을 수학적 귀납법으로 증명한 것이다.

---
(ⅰ) $n=1$ 일 때,
(좌변) $= a_{1}$, (우변) $= a_{2} - ( \text{가} ) = 1 = a_{1}$
$(*)$이 성립한다.

(ⅱ) $n=m$ 일 때, $(*)$이 성립한다고 가정하면
$a_{1} + 2a_{2} + 3a_{3} + \dots + ma_{m} = \frac{m(m+1)}{4} ( 2a_{m+1} - 1 )$ 이다.
$n=m+1$ 일 때, $(*)$이 성립함을 보이자.
$a_{1} + 2a_{2} + 3a_{3} + \dots + ma_{m} + (m+1) a_{m+1}$
$= \frac{m(m+1)}{4} ( 2a_{m+1} - 1 ) + (m+1) a_{m+1}$
$= (m+1) a_{m+1} ( ( \text{나} ) + 1 ) - \frac{m(m+1)}{4}$
$= \frac{(m+1)(m+2)}{2} a_{m+1} - \frac{m(m+1)}{4}$
$= \frac{(m+1)(m+2)}{2} ( a_{m+2} - ( \text{다} ) ) - \frac{m(m+1)}{4}$
$= \frac{(m+1)(m+2)}{4} ( 2a_{m+2} - 1 )$
$n=m+1$ 일 때도 $(*)$이 성립한다.

(ⅰ), (ⅱ)에 의하여 모든 자연수 $n$에 대하여
$a_{1} + 2a_{2} + 3a_{3} + \dots + na_{n} = \frac{n(n+1)}{4} ( 2a_{n+1} - 1 )$
이 성립한다.
---

위의 (가)에 알맞은 수를 $p$, (나), (다)에 알맞은 식을 각각 $f(m), g(m)$이라 할 때, $p + \frac{f(5)}{g(3)}$의 값은?

## 보기
① $9$   ② $10$   ③ $11$   ④ $12$   ⑤ $13$

## 풀이
(ⅰ) $n=1$ 일 때,
(좌변) $= a_{1}$, (우변) $= a_{2} - \frac{1}{2} = ( 1 + \frac{1}{2} ) - \frac{1}{2} = 1 = a_{1}$
그러므로 $p = \frac{1}{2}$이다.

(ⅱ) $n=m$ 일 때,
$a_{1} + 2a_{2} + 3a_{3} + \dots + ma_{m} + (m+1) a_{m+1}$
$= \frac{m(m+1)}{4} ( 2a_{m+1} - 1 ) + (m+1) a_{m+1}$
$= \frac{m(m+1)}{2} a_{m+1} - \frac{m(m+1)}{4} + (m+1) a_{m+1}$
$= (m+1) a_{m+1} ( \frac{m}{2} + 1 ) - \frac{m(m+1)}{4}$
따라서 $f(m) = \frac{m}{2}$이다.

계속해서 식을 정리하면,
$= \frac{(m+1)(m+2)}{2} a_{m+1} - \frac{m(m+1)}{4}$
$= \frac{(m+1)(m+2)}{2} ( a_{m+2} - \frac{1}{m+2} ) - \frac{m(m+1)}{4}$
$= \frac{(m+1)(m+2)}{2} a_{m+2} - \frac{m+1}{2} - \frac{m(m+1)}{4}$
$= \frac{(m+1)(m+2)}{2} a_{m+2} - \frac{2(m+1) + m(m+1)}{4}$
$= \frac{(m+1)(m+2)}{2} a_{m+2} - \frac{(m+1)(m+2)}{4}$
$= \frac{(m+1)(m+2)}{4} ( 2a_{m+2} - 1 )$
따라서 $g(m) = \frac{1}{m+2}$이다.

$p = \frac{1}{2}$, $f(5) = \frac{5}{2}$, $g(3) = \frac{1}{3+2} = \frac{1}{5}$ 이므로
$p + \frac{f(5)}{g(3)} = \frac{1}{2} + \frac{\frac{5}{2}}{\frac{1}{5}} = \frac{1}{2} + \frac{25}{2} = \frac{26}{2} = 13$

정답은 ⑤이다.
