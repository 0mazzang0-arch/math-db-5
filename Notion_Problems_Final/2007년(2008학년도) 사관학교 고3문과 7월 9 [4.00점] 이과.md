## 문제
원 $x^{2} + y^{2} = 1$ 위의 점 $P(x, y)$에 대하여 $\log_{\frac{1}{2}} (y+1) - \log_{\frac{1}{2}} (x+3)$의 최솟값을 $m$이라 할 때, $2^{m}$의 값은? (단, $y \neq -1$이다.)

## 보기
① $3$   ② $\frac{3}{2}$   ③ $1$   ④ $\frac{4}{3}$   ⑤ $\frac{2}{3}$

## 풀이
주어진 식은 로그의 성질에 의해 다음과 같다.
$$\log_{\frac{1}{2}} \left( \frac{y+1}{x+3} \right)$$
이 식의 최솟값을 구해야 한다. 밑이 $0 < \frac{1}{2} < 1$이므로, 진수인 $\frac{y+1}{x+3}$이 최대일 때 전체 식은 최솟값 $m$을 갖는다.

$\frac{y+1}{x+3}$의 기하학적 의미는 $\frac{y-(-1)}{x-(-3)}$이므로, 원 위의 점 $P(x, y)$와 점 $(-3, -1)$을 이은 직선의 기울기로 해석할 수 있다.

이 기울기가 최대가 될 때는 점 $(-3, -1)$을 지나고 원 $x^{2} + y^{2} = 1$에 접하는 직선 중 기울기가 양수인 경우이다. 기울기를 $k$라 하면 직선의 방정식은 다음과 같다.
$$y+1 = k(x+3) \Rightarrow kx - y + 3k - 1 = 0$$

원의 중심 $(0, 0)$으로부터 이 직선까지의 거리가 반지름의 길이인 $1$과 같아야 하므로:
$$\frac{|k \cdot 0 - 0 + 3k - 1|}{\sqrt{k^{2} + (-1)^{2}}} = 1$$
$$\frac{|3k - 1|}{\sqrt{k^{2} + 1}} = 1$$
양변을 제곱하여 정리하면:
$$(3k - 1)^{2} = k^{2} + 1$$
$$9k^{2} - 6k + 1 = k^{2} + 1$$
$$8k^{2} - 6k = 0$$
$$2k(4k - 3) = 0$$
$\therefore k = \frac{3}{4}$ ($\because y \neq -1$이므로 $k \neq 0$)

따라서 $\frac{y+1}{x+3}$의 최댓값은 $\frac{3}{4}$이다.
최솟값 $m$은 다음과 같다.
$$m = \log_{\frac{1}{2}} \frac{3}{4}$$
$2^{m}$의 값을 구하면:
$$2^{m} = 2^{\log_{\frac{1}{2}} \frac{3}{4}} = 2^{-\log_{2} \frac{3}{4}} = 2^{\log_{2} \left( \frac{3}{4} \right)^{-1}} = \left( \frac{3}{4} \right)^{-1} = \frac{4}{3}$$
