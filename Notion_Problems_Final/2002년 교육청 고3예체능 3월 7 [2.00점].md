## 문제
$-1 \le x \le 3$에서 정의된 함수 $f(x) = -2x^2+8x+a$의 최댓값이 $5$가 되도록 상수 $a$의 값을 정할 때, 함수 $f(x)$의 최솟값은?

## 보기
① $-11$   ② $-12$   ③ $-13$   ④ $-14$   ⑤ $-15$

## 풀이
$f(x) = -2x^2+8x+a = -2(x-2)^2+a+8$이므로 최댓값은
$f(2) = a+8 = 5$
따라서 $a = -3$이다.
이때, 최솟값은 $x = -1$일 때 발생하므로
$f(-1) = -2-8+a = -13$이다.
따라서 정답은 ③이다.
