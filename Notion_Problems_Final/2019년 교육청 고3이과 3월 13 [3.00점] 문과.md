## 문제
$0 \le x \le \pi$ 에서 정의된 함수
$$ f(x) = \begin{cases} 2 \cos x \tan x + a & (x \neq \frac{\pi}{2}) \\ 3a & (x = \frac{\pi}{2}) \end{cases} $$
가 $x = \frac{\pi}{2}$ 에서 연속일 때, 함수 $f(x)$ 의 최댓값과 최솟값의 합은? (단, $a$ 는 상수이다.)

## 보기
① $\frac{5}{2}$   ② $3$   ③ $\frac{7}{2}$   ④ $4$   ⑤ $\frac{9}{2}$

## 풀이
$\lim_{x \to \frac{\pi}{2}} f(x)$
$= \lim_{x \to \frac{\pi}{2}} (2 \cos x \tan x + a)$
$= \lim_{x \to \frac{\pi}{2}} (2 \sin x + a)$
$= 2 + a$

함수 $f(x)$ 가 $x = \frac{\pi}{2}$ 에서 연속이므로 
$f(\frac{\pi}{2}) = \lim_{x \to \frac{\pi}{2}} f(x)$
$3a = 2 + a$, $a = 1$

$0 \le x \le \pi$ 에서 함수 $f(x) = 2 \sin x + 1$ 이고, 
$0 \le \sin x \le 1$ 이므로 
함수 $f(x)$ 의 최댓값은 $3$, 최솟값은 $1$ 이다. 
따라서 구하는 값은 $3 + 1 = 4$

정답: ④
