## 문제
연립방정식 
$$ \begin{cases} 2^{x} ( 2^{x} + 2^{y} ) = 192 \\ 2^{y} ( 2^{x} + 2^{y} ) = 384 \end{cases} $$
를 만족시키는 $x$, $y$에 대하여 $2^{x+y}$의 값을 구하시오.

## 보기

## 풀이
$2^{x} = X$, $2^{y} = Y$라 하면 

$X ( X + Y ) = 192 \cdots ①$
$Y ( X + Y ) = 384 \cdots ②$

$① \div ②$를 하면 
$\frac{X}{Y} = \frac{1}{2}$, $Y = 2X$를 $①$에 대입하면 
$3X^{2} = 192$
$\therefore X = 8, Y = 16$
$\therefore 2^{x+y} = 2^{x} \cdot 2^{y} = 8 \times 16 = 128$

정답: $128$
