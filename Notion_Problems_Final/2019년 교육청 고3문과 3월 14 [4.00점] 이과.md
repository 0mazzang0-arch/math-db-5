## 문제
실수 $x$에 대한 두 조건 $p$, $q$가 다음과 같다.
$p : x \le -5$ 또는 $x > 3$
$q : x = \frac{2a+1}{3}$
$\sim p$가 $q$이기 위한 필요조건이 되도록 하는 정수 $a$의 최솟값과 최댓값의 합은?

## 보기
① $-6$   ② $-5$   ③ $-4$   ④ $-3$   ⑤ $-2$

## 풀이
두 조건 $p$, $q$의 진리집합을 각각 $P$, $Q$라 하면 
$P = \{ x | x \le -5 \text{ 또는 } x > 3 \}$, $Q = \left\{ \frac{2a+1}{3} \right\}$ 이고 
$\sim p$의 진리집합은 $P^{c} = \{ x | -5 < x \le 3 \}$이다. 

$\sim p$가 $q$이기 위한 필요조건이 되기 위해서는 $q \to \sim p$, 즉 $Q \subseteq P^{c}$이어야 하므로 
$-5 < \frac{2a+1}{3} \le 3$
$-15 < 2a+1 \le 9$
$-16 < 2a \le 8$
$-8 < a \le 4$

따라서 정수 $a$의 최댓값은 $4$, 최솟값은 $-7$이므로 그 합은 $4 + (-7) = -3$이다.
