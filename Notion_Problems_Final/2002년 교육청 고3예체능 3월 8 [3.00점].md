## 문제

다음 함수의 그래프를 평행이동하였을 때, $y = \frac{x}{x - 2}$ 의 그래프와 겹쳐질 수 있는 것만을 <보기>에서 있는 대로 고른 것은?

$$ \begin{array}{l} \text{ㄱ. } y = \frac{2}{x + 1} \\ \text{ㄴ. } y = \frac{2x + 1}{x + 2} \\ \text{ㄷ. } y = \frac{3x + 5}{x + 1} \end{array} $$

## 보기

① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄷ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이

$y = \frac{x}{x - 2} = \frac{x - 2 + 2}{x - 2} = \frac{2}{x - 2} + 1$ 이므로 $y = \frac{x}{x - 2}$ 의 그래프는 $y = \frac{2}{x}$ 의 그래프를 $x$ 축의 방향으로 $2$ 만큼, $y$ 축의 방향으로 $1$ 만큼 평행이동한 그래프이다.

따라서 주어진 분수함수를 $y = \frac{k}{x - p} + q$ 의 꼴로 정리하였을 때, $k = 2$ 인 분수함수의 그래프는 평행이동을 통하여 $y = \frac{x}{x - 2}$ 의 그래프와 겹쳐질 수 있다.

ㄱ. $y = \frac{2}{x + 1}$ 에서 $k = 2$
ㄴ. $y = \frac{2x + 1}{x + 2} = \frac{2(x + 2) - 3}{x + 2} = \frac{-3}{x + 2} + 2$ 에서 $k = -3$
ㄷ. $y = \frac{3x + 5}{x + 1} = \frac{3(x + 1) + 2}{x + 1} = \frac{2}{x + 1} + 3$ 에서 $k = 2$

따라서 평행이동을 통하여 $y = \frac{x}{x - 2}$ 의 그래프와 겹쳐질 수 있는 분수함수는 ㄱ, ㄷ 이다.
