## 문제
전체집합 $U = \{ x | x \text{는 } 20 \text{ 이하의 자연수} \}$ 의 두 부분집합 
$A = \{ x | x \text{는 } 4 \text{의 배수} \}$, $B = \{ x | x \text{는 } 20 \text{의 약수} \}$ 에 대하여 집합 $( A^{c} \cup B )^{c}$ 의 모든 원소의 합을 구하시오.

## 풀이
$A = \{ 4, 8, 12, 16, 20 \}$
$B = \{ 1, 2, 4, 5, 10, 20 \}$
드모르간의 법칙에 의하여 
$( A^{c} \cup B )^{c} = ( A^{c} )^{c} \cap B^{c} = A \cap B^{c} = A - B$
$A - B = \{ 8, 12, 16 \}$ 이므로 집합 $A - B$ 의 모든 원소의 합은 
$8 + 12 + 16 = 36$ 이다.

정답: $36$
