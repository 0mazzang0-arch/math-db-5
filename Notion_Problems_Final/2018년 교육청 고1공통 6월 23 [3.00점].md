## 문제
$x$에 대한 이차방정식 $x^{2} + ax + b = 0$의 두 근이 $3$, $4$일 때, 두 상수 $a$, $b$에 대하여 $a + b$의 값을 구하시오.

## 풀이
이차방정식의 근과 계수의 관계에 의하여 
$3 + 4 = -a$, 
$3 \times 4 = b$ 
이므로 
$a + b = -7 + 12 = 5$이다. 

[다른 풀이] 
이차방정식 $x^{2} + ax + b$의 두 근이 $3$, $4$이므로 
$3^{2} + 3a + b = 0$, 
$4^{2} + 4a + b = 0$ 
이다. 연립방정식 
$$\begin{cases} 3a + b = -9 \\ 4a + b = -16 \end{cases}$$ 
에서 $a = -7$, $b = 12$이다. 
따라서 $a + b = -7 + 12 = 5$이다.

**정답: $5$**
