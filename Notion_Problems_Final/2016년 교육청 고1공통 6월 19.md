## 문제
세 유리수 $a$, $b$, $c$에 대하여 $x$에 대한 이차방정식 $ax^{2} + \sqrt{3}bx + c = 0$의 한 근이 $\alpha = 2 + \sqrt{3}$이다. 다른 한 근을 $\beta$라 할 때, $\alpha + \frac{1}{\beta}$의 값은?

## 보기
① $-4$   ② $-2\sqrt{3}$   ③ $0$   ④ $2\sqrt{3}$   ⑤ $4$

## 풀이
$2 + \sqrt{3}$은 방정식 $ax^{2} + \sqrt{3}bx + c = 0$의 한 근이므로
$$ a(2 + \sqrt{3})^{2} + \sqrt{3}b(2 + \sqrt{3}) + c = 0 $$
$$ (7a + 3b + c) + (4a + 2b)\sqrt{3} = 0 $$
$a$, $b$, $c$가 유리수이므로 무리수가 서로 같을 조건에 의하여
$$ 7a + 3b + c = 0, \quad 4a + 2b = 0 $$
$\therefore b = -2a$, $c = -a$
이것을 주어진 방정식에 대입하면
$$ ax^{2} - 2a\sqrt{3}x - a = 0 \quad (a \ne 0) $$
$$ a(x^{2} - 2\sqrt{3}x - 1) = 0 $$
$\therefore x = \sqrt{3} \pm 2$
따라서 $\alpha = 2 + \sqrt{3}$, $\beta = -2 + \sqrt{3}$이므로
$$ \alpha + \frac{1}{\beta} = 2 + \sqrt{3} + \frac{1}{-2 + \sqrt{3}} = 0 $$

**[다른 풀이]**
$t = \sqrt{3}x$라 놓으면 주어진 방정식은
$$ \frac{a}{3}t^{2} + bt + c = 0 $$
$\therefore at^{2} + 3bt + 3c = 0 \quad \cdots \cdots$ ㉠
계수가 유리수인 방정식 ㉠의 한 근은 $t = \sqrt{3}\alpha = \sqrt{3}(2 + \sqrt{3}) = 3 + 2\sqrt{3}$이므로 다른 한 근은
$$ t = \sqrt{3}\beta = 3 - 2\sqrt{3} $$
$\therefore \beta = \frac{3 - 2\sqrt{3}}{\sqrt{3}} = \sqrt{3} - 2 = -2 + \sqrt{3}$
따라서
$$ \alpha + \frac{1}{\beta} = 2 + \sqrt{3} + \frac{1}{-2 + \sqrt{3}} = 0 $$

**[다른 풀이]**
$\alpha = 2 + \sqrt{3}$에서 $\alpha - \sqrt{3} = 2$
양변을 제곱하여 정리하면
$$ \alpha^{2} - 2\sqrt{3}\alpha - 1 = 0 $$
따라서 $\alpha$는 이차방정식 $a(x^{2} - 2\sqrt{3}x - 1) = 0$의 근이고, 다른 한 근이 $\beta$이므로 근과 계수의 관계에 의하여
$$ \alpha + \beta = 2\sqrt{3} $$
$$ (2 + \sqrt{3}) + \beta = 2\sqrt{3} $$
$\therefore \beta = -2 + \sqrt{3}$
따라서
$$ \alpha + \frac{1}{\beta} = 2 + \sqrt{3} + \frac{1}{-2 + \sqrt{3}} = 0 $$
