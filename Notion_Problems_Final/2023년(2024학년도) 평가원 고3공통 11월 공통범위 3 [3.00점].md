## 문제
$\frac{3}{2} \pi < \theta < 2 \pi$ 인 $\theta$ 에 대하여 $\sin ( -\theta ) = \frac{1}{3}$ 일 때, $\tan \theta$ 의 값은?

## 보기
① $-\frac{\sqrt{2}}{2}$   ② $-\frac{\sqrt{2}}{4}$   ③ $-\frac{1}{4}$   ④ $\frac{1}{4}$   ⑤ $\frac{\sqrt{2}}{4}$

## 풀이
$\sin ( -\theta ) = -\sin \theta = \frac{1}{3}$ 에서
$\sin \theta = -\frac{1}{3}$
$\frac{3}{2} \pi < \theta < 2 \pi$ 이므로
$\cos \theta = \sqrt{1 - \sin^2 \theta}$
$= \sqrt{1 - \left( -\frac{1}{3} \right)^2}$
$= \frac{2\sqrt{2}}{3}$

$\therefore \tan \theta = \frac{\sin \theta}{\cos \theta} = \frac{-\frac{1}{3}}{\frac{2\sqrt{2}}{3}} = -\frac{1}{2\sqrt{2}} = -\frac{\sqrt{2}}{4}$

정답은 ②입니다.
