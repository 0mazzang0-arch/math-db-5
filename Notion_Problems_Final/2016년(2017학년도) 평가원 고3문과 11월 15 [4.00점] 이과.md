## 문제
공차가 양수인 등차수열 $\{a_n\}$이 다음 조건을 만족시킬 때, $a_2$의 값은?

$$
\begin{array}{l}
\text{(가) } a_6 + a_8 = 0 \\
\text{(나) } |a_6| = |a_7| + 3
\end{array}
$$

## 보기
① $-15$   ② $-13$   ③ $-11$   ④ $-9$   ⑤ $-7$

## 풀이
등차수열의 첫째항을 $a$, 공차를 $d$ ($d > 0$)라 하면
조건 (가)에서
$(a + 5d) + (a + 7d) = 0$
$2a + 12d = 0$
$a = -6d$ $\cdots$ ㉠

조건 (나)에서
$|a_6| = |a_7| + 3$이므로
$|a + 5d| = |a + 6d| + 3$

㉠을 대입하면
$|-6d + 5d| = |-6d + 6d| + 3$
$|-d| = 0 + 3$
$|d| = 3$

$d > 0$이므로 $d = 3$

따라서
$a_2 = a + d$
$= -6d + d = -5d$
$= -5 \times 3 = -15$

정답: ①
