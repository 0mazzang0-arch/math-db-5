## 문제

최고차항의 계수가 $\frac{1}{2}$ 인 삼차함수 $f(x)$ 와 실수 $t$ 에 대하여 방정식 $f'(x) = 0$ 이 닫힌구간 $[t, t+2]$ 에서 갖는 실근의 개수를 $g(t)$ 라 할 때, 함수 $g(t)$ 는 다음 조건을 만족시킨다.

(가) 모든 실수 $a$ 에 대하여 $\lim_{t \to a^+} g(t) + \lim_{t \to a^-} g(t) \le 2$ 이다.
(나) $g(f(1)) = g(f(4)) = 2$, $g(f(0)) = 1$

$f(5)$ 의 값을 구하시오.

## 풀이

이차방정식 $f'(x) = 0$ 이 근을 갖지 않는 경우에는 $g(t) = 0$ 이다. 이는 조건 (나)에서 $g(t)$ 가 함숫값 $1$ 또는 $2$ 를 갖는 것에 모순이다. 또한, 이차방정식 $f'(x) = 0$ 이 중근 $\alpha$ 를 갖는 경우에는 $y = g(x)$ 의 그래프 개형상 $g(t)$ 가 함숫값 $2$ 를 갖는 것에 모순이다. 
그러므로 이차방정식 $f'(x) = 0$ 은 서로 다른 두 실근 $\alpha, \beta$ ($\alpha < \beta$) 를 갖는다.

(ⅰ) $\beta = \alpha + 2$ 일 때,
$y = g(t)$ 의 그래프는 다음과 같다.

> [!example] 📸 (그림/그래프)

(ⅱ) $\beta > \alpha + 2$ 일 때,
$y = g(t)$ 의 그래프는 다음과 같다.

> [!example] 📸 (그림/그래프)

이 경우 $g(t)$ 가 함숫값 $2$ 를 가질 수 없으므로 모순이다.

(ⅲ) $\beta < \alpha + 2$ 일 때,
$y = g(t)$ 의 그래프는 다음과 같다.

> [!example] 📸 (그림/그래프)

이 경우 $\beta - 2 \le a \le \alpha$ 인 $a$ 에 대하여 조건 (가)를 만족시키지 못한다.
따라서 위에서 조건을 만족시키는 것은 (ⅰ)의 경우이다.

한편, 함수 $f(x)$ 의 최고차항의 계수가 $\frac{1}{2}$ 이므로 함수 $f'(x)$ 의 최고차항의 계수는 $\frac{3}{2}$ 이다. 그러므로 $f'(x)$ 를 다음과 같이 놓을 수 있다.
$$
\begin{aligned}
f'(x) &= \frac{3}{2} (x - \alpha) \{x - (\alpha + 2)\} \\
&= \frac{3}{2} \{x^2 - (2\alpha + 2)x + \alpha^2 + 2\alpha\}
\end{aligned}
$$
이를 부정적분하면 ($C$ 는 적분상수)
$f(x) = \frac{1}{2}x^3 - \frac{3}{2}(\alpha + 1)x^2 + \frac{3}{2}(\alpha^2 + 2\alpha)x + C \quad \cdots \text{㉠}$

조건 (나)에서 $g(f(1)) = g(f(4)) = 2$ 이고, (ⅰ)의 그래프에서 $g(t)$ 의 함숫값이 $2$ 인 $t$ 의 값은 $t = \alpha$ 하나뿐이므로
$f(1) = f(4) = \alpha$ 이어야 한다.

㉠에서 $f(1) = f(4)$ 임을 이용하면
$\frac{1}{2} - \frac{3}{2}(\alpha + 1) + \frac{3}{2}(\alpha^2 + 2\alpha) + C = 32 - 24(\alpha + 1) + 6(\alpha^2 + 2\alpha) + C$
$\frac{1}{2} - \frac{3}{2}(\alpha + 1) + \frac{3}{2}(\alpha^2 + 2\alpha) = 32 - 24(\alpha + 1) + 6(\alpha^2 + 2\alpha)$
양변에 $2$ 를 곱하여 정리하면
$1 - 3(\alpha + 1) + 3(\alpha^2 + 2\alpha) = 64 - 48(\alpha + 1) + 12(\alpha^2 + 2\alpha)$
$3\alpha^2 + 3\alpha - 2 = 12\alpha^2 - 24\alpha + 16$
$9\alpha^2 - 27\alpha + 18 = 0$
$\alpha^2 - 3\alpha + 2 = 0$
$(\alpha - 1)(\alpha - 2) = 0$
따라서 $\alpha = 1$ 또는 $\alpha = 2$ 이다.

((ⅰ)-①) $\alpha = 1$ 일 때,
$f(x) = \frac{1}{2}x^3 - 3x^2 + \frac{9}{2}x + C$
$f(1) = \alpha = 1$ 이므로
$\frac{1}{2} - 3 + \frac{9}{2} + C = 1 \Rightarrow 2 + C = 1 \Rightarrow C = -1$
이때 $f(0) = -1$ 이고, $g(f(0)) = g(-1)$ 이다.
$\alpha = 1, \beta = 3$ 이므로 $g(t)$ 는 구간 $[t, t+2]$ 에 포함된 $\{1, 3\}$ 의 개수이다.
$g(-1)$ 은 구간 $[-1, 1]$ 에 포함된 실근의 개수이므로 $g(-1) = 1$ 이 되어 조건 (나)를 만족시킨다.

((ⅰ)-②) $\alpha = 2$ 일 때,
$f(x) = \frac{1}{2}x^3 - \frac{9}{2}x^2 + 12x + C$
$f(1) = \alpha = 2$ 이므로
$\frac{1}{2} - \frac{9}{2} + 12 + C = 2 \Rightarrow 8 + C = 2 \Rightarrow C = -6$
이때 $f(0) = -6$ 이고, $g(f(0)) = g(-6)$ 이다.
$\alpha = 2, \beta = 4$ 이므로 구간 $[-6, -4]$ 에 포함된 실근의 개수는 $g(-6) = 0$ 이다.
이는 $g(f(0)) = 1$ 이라는 조건 (나)에 모순이다.

따라서 ((ⅰ)-①)의 경우에 의해 함수 $f(x)$ 는
$f(x) = \frac{1}{2}x^3 - 3x^2 + \frac{9}{2}x - 1$
$f(5) = \frac{1}{2} \times 5^3 - 3 \times 25 + \frac{9}{2} \times 5 - 1$
$f(5) = \frac{125}{2} - 75 + \frac{45}{2} - 1 = \frac{170}{2} - 76 = 85 - 76 = 9$

[정답] $9$
