## 문제
$\log_{a} b = \frac{3}{2}$, $\log_{c} d = \frac{3}{4}$을 만족시키는 자연수 $a$, $b$, $c$, $d$에 대하여 $a - c = 19$일 때, $b - d$의 값을 구하시오.

## 보기

## 풀이
$b = a^{\frac{3}{2}}$, $d = c^{\frac{3}{4}}$ 즉 $a = b^{\frac{2}{3}}$, $c = d^{\frac{4}{3}}$이고 $a, b, c, d$는 자연수이므로 $a = x^2$ 꼴, $c = y^4$ 꼴이다.
따라서 $b = x^3$, $d = y^3$ 꼴이다.

$a - c = 19$에서 $x^2 - y^4 = 19$이므로 $(x + y^2)(x - y^2) = 19$이다.
즉 만족하는 경우는 다음 $4$가지의 경우이다.

(ⅰ) $x + y^2 = 1$, $x - y^2 = 19$
$x$, $y$가 없다.
(ⅱ) $x + y^2 = -1$, $x - y^2 = -19$
$x$, $y$가 없다.
(ⅲ) $x + y^2 = -19$, $x - y^2 = -1$
$x$, $y$가 없다.
(ⅳ) $x + y^2 = 19$, $x - y^2 = 1$
$x = 10$, $y = 3$

(ⅰ)∼(ⅳ)에서 $a = 100$, $b = 1000$, $c = 81$, $d = 27$
$\therefore b - d = 973$
