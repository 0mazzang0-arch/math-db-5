## 문제
복소수 $z = a + bi$ ($a$, $b$는 실수)에 대하여 $\ll z \gg$를 $\ll z \gg = a^2 + b^2$으로 정의할 때, <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $\bar{z}$는 $z$의 켤레복소수이고, $i = \sqrt{-1}$이다.)

---
**<보 기>**
ㄱ. $\ll 3 + 4i \gg = 25$
ㄴ. $\ll z + \bar{z} \gg = \ll z \gg + \ll \bar{z} \gg$
ㄷ. $\ll \frac{1}{z} \gg = \frac{1}{\ll z \gg}$ (단, $z \neq 0$)
---

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄱ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
$z = a + bi$, $\bar{z} = a - bi$ ($a$, $b$는 실수)라 하자.

ㄱ. $\ll 3 + 4i \gg = 3^2 + 4^2 = 9 + 16 = 25$ (참)

ㄴ. $z + \bar{z} = (a + bi) + (a - bi) = 2a$이므로
$\ll z + \bar{z} \gg = (2a)^2 + 0^2 = 4a^2$
$\ll z \gg + \ll \bar{z} \gg = (a^2 + b^2) + (a^2 + (-b)^2) = 2(a^2 + b^2)$
따라서 $\ll z + \bar{z} \gg = \ll z \gg + \ll \bar{z} \gg$는 항상 성립하지 않는다. (거짓)

ㄷ. $\frac{1}{z} = \frac{1}{a + bi} = \frac{a - bi}{(a + bi)(a - bi)} = \frac{a}{a^2 + b^2} - \frac{b}{a^2 + b^2}i$이므로
$\ll \frac{1}{z} \gg = \left( \frac{a}{a^2 + b^2} \right)^2 + \left( \frac{-b}{a^2 + b^2} \right)^2$
$= \frac{a^2 + b^2}{(a^2 + b^2)^2} = \frac{1}{a^2 + b^2} = \frac{1}{\ll z \gg}$ (참)

이상에서 옳은 것은 ㄱ, ㄷ이다.
