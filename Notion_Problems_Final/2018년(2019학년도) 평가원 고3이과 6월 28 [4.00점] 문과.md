## 문제
자연수 $n$ ($n \ge 3$)에 대하여 집합 $A$를
$$A = \{ (x, y) | 1 \le x \le y \le n, x \text{ 와 } y \text{ 는 자연수} \}$$
라 하자. 집합 $A$에서 임의로 선택된 한 개의 원소 $(a, b)$에 대하여 $b$가 $3$의 배수일 때, $a = b$일 확률이 $\frac{1}{9}$이 되도록 하는 모든 자연수 $n$의 값의 합을 구하시오.

## 보기
(주관식 문항으로 보기가 없습니다.)

## 풀이
$n = 3k$ 또는 $n = 3k + 1$ 또는 $n = 3k + 2$ ($k$는 자연수)일 때 $b$가 $3$의 배수인 사건의 수는 
$$3 + 6 + 9 + \cdots + 3k = 3(1 + 2 + 3 + \cdots + k) = \frac{3}{2}k(k + 1)$$
이 중에서 $a = b$인 사건의 수는 $k$이므로 조건부 확률은 다음과 같다.
$$\frac{k}{\frac{3}{2}k(k + 1)} = \frac{1}{9}$$
$$\frac{2}{3(k + 1)} = \frac{1}{9}$$
$$\frac{2}{k + 1} = \frac{1}{3}$$
$$k + 1 = 6$$
$$k = 5$$
따라서 $k = 5$일 때 가능한 모든 자연수 $n$의 값은 $15$, $16$, $17$이며, 그 합은 다음과 같다.
$$15 + 16 + 17 = 48$$
