## 문제
다항식 $( x^{2} -x ) ( x^{2} -x-1 ) -2$의 인수인 것은?

## 보기
① $x-2$   ② $x-1$   ③ $x$   ④ $x^{2}+1$   ⑤ $x^{2}+x+1$

## 풀이
$x^{2} -x=A$로 놓으면 
$$( x^{2} -x ) ( x^{2} -x-1 ) -2$$ 
$$=A ( A-1 ) -2$$ 
$$=A^{2} -A-2$$ 
$$=( A-2 ) ( A+1 )$$ 
$$=( x^{2} -x-2 ) ( x^{2} -x+1 )$$ 
$$=( x-2 ) ( x+1 ) ( x^{2} -x+1 )$$ 
따라서 $x-2$는 주어진 식의 인수이다.
