## 문제
양수 $x$에 대하여 $\log x$의 정수부분을 $f(x)$라 하자. 정수 부분이 네 자리인 양수 $t$에 대하여 
$$\log t = \frac{1}{4} f(t^{2}) - \frac{1}{2} f\left(\frac{1}{t}\right)$$
을 만족시키는 모든 실수 $t$의 곱을 $A$라 할 때, $4 \log A$의 값을 구하시오.

## 보기

## 풀이
정수 부분이 네 자리인 양수 $t$에 대하여 $1000 \le t < 10000$이므로
$\log t = 3 + \alpha$ ($0 \le \alpha < 1$)라 하면 $f(t) = 3$이다.
또한 $\log t^{2} = 2 \log t = 6 + 2\alpha$이고, $\log \frac{1}{t} = -\log t = -3 - \alpha$이다.

ⅰ) $\alpha = 0$일 때,
$f(t^{2}) = 6$, $f\left(\frac{1}{t}\right) = -3$이므로
$\log t = \frac{1}{4} \cdot 6 - \frac{1}{2} \cdot (-3) = \frac{3}{2} + \frac{3}{2} = 3$
이는 $\log t = 3 + 0$을 만족시키므로
$t = 10^{3}$

ⅱ) $0 < \alpha < \frac{1}{2}$일 때,
$f(t^{2}) = 6$, $f\left(\frac{1}{t}\right) = -4$이므로
$3 + \alpha = \frac{1}{4} \cdot 6 - \frac{1}{2} \cdot (-4) = \frac{7}{2}$
$\alpha = \frac{1}{2}$
그런데 $0 < \alpha < \frac{1}{2}$이므로 조건을 만족시키는 $\alpha$는 존재하지 않는다.

ⅲ) $\frac{1}{2} \le \alpha < 1$일 때,
$f(t^{2}) = 7$, $f\left(\frac{1}{t}\right) = -4$이므로
$3 + \alpha = \frac{1}{4} \cdot 7 - \frac{1}{2} \cdot (-4) = \frac{15}{4}$
$\alpha = \frac{3}{4}$
이는 $\frac{1}{2} \le \alpha < 1$을 만족시키므로
$\log t = 3 + \frac{3}{4} = \frac{15}{4}$, 즉 $t = 10^{\frac{15}{4}}$

모든 실수 $t$의 곱 $A$는
$A = 10^{3} \times 10^{\frac{15}{4}} = 10^{\frac{27}{4}}$
따라서
$4 \log A = 4 \cdot \frac{27}{4} = 27$
