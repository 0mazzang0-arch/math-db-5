## 문제
수열 $\{ a_{n} \}$ 은 $a_{1} = 3$ 이고
$n a_{n+1} - 2 n a_{n} + \frac{n + 2}{n + 1} = 0 (n \geq 1)$
을 만족시킨다. 다음은 일반항 $a_{n}$ 을 구하는 과정이다.

$$
\begin{array}{|l|}
\hline
a_{n+1} - 2 a_{n} + \frac{n + 2}{n(n + 1)} = 0 \text{ 에서} \\
a_{n} - 2 a_{n-1} + \frac{n + 1}{n(n - 1)} = 0 (n \geq 2) \text{ 이므로} \\
a_{n+1} - a_{n} - 2 (a_{n} - a_{n-1} ) + \frac{1}{n(n + 1)} - (가) = 0 (n \geq 2) \text{ 이다.} \\
b_{n} = a_{n+1} - a_{n} (n \geq 1) \text{ 이라 놓으면 } b_{1} = \frac{3}{2} \text{ 이고,} \\
b_{n} + \frac{1}{n(n + 1)} = 2 b_{n-1} + (가) (n \geq 2) \text{ 이다. 따라서} \\
b_{n} + \frac{1}{n(n + 1)} = 2^{n} (n \geq 1) \text{ 이다. 즉,} \\
b_{n} = 2^{n} - \frac{1}{n(n + 1)} (n \geq 1) \text{ 이므로} \\
a_{n} = a_{1} + \sum_{k=1}^{n-1} b_{k} = (나) (n \geq 2) \text{ 이다.} \\
n = 1 \text{ 일 때에도 이 식을 만족시키므로 모든 자연수 } n \text{ 에 대하여 } a_{n} = (나) \text{ 이다.} \\
\hline
\end{array}
$$

위의 (가), (나)에 알맞은 식을 각각 $f(n)$, $g(n)$ 이라 할 때, $g(6) - f(4)$ 의 값은?

## 보기
① $64$   ② $66$   ③ $68$   ④ $70$   ⑤ $72$

## 풀이
$n a_{n+1} - 2 n a_{n} + \frac{n + 2}{n + 1} = 0$ 의 양변을 $n$ 으로 나누면
$a_{n+1} - 2 a_{n} + \frac{n + 2}{n(n + 1)} = 0$ 이다.
또한 $n \geq 2$ 일 때, $a_{n} - 2 a_{n-1} + \frac{n + 1}{n(n - 1)} = 0$ 이 성립한다.
두 식을 변끼리 빼면
$a_{n+1} - a_{n} - 2(a_{n} - a_{n-1}) + \frac{n + 2}{n(n + 1)} - \frac{n + 1}{n(n - 1)} = 0$ 이고,
$\frac{n + 2}{n(n + 1)} - \frac{n + 1}{n(n - 1)} = \frac{n + 1 + 1}{n(n + 1)} - \frac{n - 1 + 2}{n(n - 1)}$
$= \left( \frac{1}{n} + \frac{1}{n(n + 1)} \right) - \left( \frac{1}{n} + \frac{2}{n(n - 1)} \right)$
$= \frac{1}{n(n + 1)} - \frac{2}{n(n - 1)}$ 이므로
$a_{n+1} - a_{n} - 2(a_{n} - a_{n-1}) + \frac{1}{n(n + 1)} - \frac{2}{n(n - 1)} = 0$ 이다.
$(가) = f(n) = \frac{2}{n(n - 1)}$ 이라 하면, $b_{n} = a_{n+1} - a_{n}$ 에 대하여
$b_{n} + \frac{1}{n(n + 1)} = 2 \left( b_{n-1} + \frac{1}{n(n - 1)} \right) (n \geq 2)$ 이다.
$b_{1} + \frac{1}{1 \cdot 2} = \frac{3}{2} + \frac{1}{2} = 2$ 이므로
$b_{n} + \frac{1}{n(n + 1)} = 2 \cdot 2^{n-1} = 2^{n}$ 에서
$b_{n} = 2^{n} - \frac{1}{n(n + 1)} (n \geq 1)$ 이다.
$n \geq 2$ 일 때,
$a_{n} = a_{1} + \sum_{k=1}^{n-1} b_{k} = 3 + \sum_{k=1}^{n-1} \left( 2^{k} - \frac{1}{k(k + 1)} \right)$
$= 3 + \frac{2(2^{n-1} - 1)}{2 - 1} - \sum_{k=1}^{n-1} \left( \frac{1}{k} - \frac{1}{k + 1} \right)$
$= 3 + 2^{n} - 2 - \left( 1 - \frac{1}{n} \right) = 2^{n} + \frac{1}{n}$ 이다.
$(나) = g(n) = 2^{n} + \frac{1}{n}$ 이고, 이는 $n = 1$ 일 때에도 $a_{1} = 2^{1} + \frac{1}{1} = 3$ 을 만족한다.
따라서 $f(n) = \frac{2}{n(n - 1)}$, $g(n) = 2^{n} + \frac{1}{n}$ 이므로
$g(6) - f(4) = \left( 2^{6} + \frac{1}{6} \right) - \frac{2}{4 \times 3} = 64 + \frac{1}{6} - \frac{1}{6} = 64$
