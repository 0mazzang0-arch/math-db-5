## 문제
실수 전체의 집합에서 정의된 두 함수
$$ f(x) = \sin^{2} x + a \cos x $$
$$ g(x) = \left\{ \begin{array}{ll} 0 & (x < -\frac{\pi}{2}) \\ x & (-\frac{\pi}{2} \le x < \pi) \\ bx & (x \ge \pi) \end{array} \right. $$
에 대하여 <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $a$, $b$는 실수이다.)

## 보기
ㄱ. $\lim_{x \to -\frac{\pi}{2}^{-}} g(x) = 0$
ㄴ. $a=2$ 이면 합성함수 $(f \circ g)(x)$는 $x=-\frac{\pi}{2}$에서 연속이다.
ㄷ. $a$의 값에 관계없이 합성함수 $(f \circ g)(x)$가 $x=\pi$에서 연속이면 $b=2n-1$ ($n$은 정수)이다.

① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄷ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
ㄱ. 함수 $g(x)$는 $x < -\frac{\pi}{2}$에서 $0$이므로
$\lim_{x \to -\frac{\pi}{2}^{-}} g(x) = \lim_{x \to -\frac{\pi}{2}^{-}} 0 = 0$ (참)

ㄴ. $a=2$ 이므로 $f(x) = \sin^{2} x + 2 \cos x$이다.
$f(x)$는 실수 전체의 집합에서 연속이므로 임의의 실수 $\alpha$에 대하여
$\lim_{x \to \alpha^{-}} f(x) = \lim_{x \to \alpha^{+}} f(x) = f(\alpha)$를 만족한다.

좌극한: $\lim_{x \to -\frac{\pi}{2}^{-}} (f \circ g)(x) = f(0) = \sin^{2} 0 + 2 \cos 0 = 2$
우극한: $\lim_{x \to -\frac{\pi}{2}^{+}} (f \circ g)(x) = \lim_{x \to -\frac{\pi}{2}^{+}} f(x) = \sin^{2} \left(-\frac{\pi}{2}\right) + 2 \cos \left(-\frac{\pi}{2}\right) = 1$
$\lim_{x \to -\frac{\pi}{2}^{-}} (f \circ g)(x) \neq \lim_{x \to -\frac{\pi}{2}^{+}} (f \circ g)(x)$ 이므로
$(f \circ g)(x)$는 $x=-\frac{\pi}{2}$에서 불연속이다. (거짓)

ㄷ. 좌극한: $\lim_{x \to \pi^{-}} (f \circ g)(x) = \lim_{x \to \pi^{-}} f(x) = \sin^{2} \pi + a \cos \pi = -a$
우극한: $\lim_{x \to \pi^{+}} (f \circ g)(x) = \lim_{x \to \pi^{+}} f(bx) = \sin^{2} b\pi + a \cos b\pi$
함숫값: $(f \circ g)(\pi) = f(b\pi) = \sin^{2} b\pi + a \cos b\pi$

$(f \circ g)(x)$가 $x=\pi$에서 연속이므로
$\lim_{x \to \pi^{-}} (f \circ g)(x) = \lim_{x \to \pi^{+}} (f \circ g)(x) = (f \circ g)(\pi)$
$\sin^{2} b\pi + a \cos b\pi = -a$
$1 - \cos^{2} b\pi + a \cos b\pi + a = 0$
$1 - \cos^{2} b\pi + a(1 + \cos b\pi) = 0 \cdots \cdots ㉠$

$a$에 대한 항등식이므로
$1 - \cos^{2} b\pi = 0$ 이고 $\cos b\pi + 1 = 0$ 이다.
$\cos b\pi = -1$ 이므로
$b\pi = (2n-1)\pi$ ($n$은 정수)
$b = 2n-1$ ($n$은 정수) (참)

따라서 옳은 것은 $ㄱ, ㄷ$이다.
