## 문제
함수 $f(x) = 8 \sin x + 4 \cos 2x + 1$ 의 최댓값은?

## 보기
① $6$   ② $7$   ③ $8$   ④ $9$   ⑤ $10$

## 풀이
$$f(x) = 8 \sin x + 4 \cos 2x + 1$$
$$= 8 \sin x + 4 ( 1 - 2 \sin^2 x ) + 1$$
$$= -8 \sin^2 x + 8 \sin x + 5$$
$$= -8 \left( \sin x - \frac{1}{2} \right)^2 + 7$$

실수 $x$ 에 대하여 $\sin x$ 는 $-1$ 과 $1$ 사이의 값을 갖는다. 즉, 
$$-1 \le \sin x \le 1$$
이다. 

$f(x)$ 는 $\sin x = \frac{1}{2}$ 일 때, 즉, $x = n \pi + (-1)^n \frac{\pi}{6}$ 일 때 최댓값 $7$ 을 갖는다. 

정답: ②
