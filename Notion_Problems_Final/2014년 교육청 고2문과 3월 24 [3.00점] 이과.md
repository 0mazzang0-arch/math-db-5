## 문제
다음은 어느 회사에서 신제품 $A$의 가격을 정하기 위하여 시장 조사를 한 결과이다.

$$ \begin{array}{|l|} \hline \text{(가) } A \text{의 가격을 } 100 \text{만 원으로 정하면 판매량은 } 2400 \text{대이다.} \\ \text{(나) } A \text{의 가격을 만 원 인상할 때마다 판매량은 } 20 \text{대씩 줄어든다.} \\ \hline \end{array} $$

신제품 $A$를 판매하여 얻은 전체 판매 금액이 최대가 되도록 하는 $A$의 가격은 $a$만 원이다. $a$의 값을 구하시오. (단, $A$의 가격은 $100$만 원 이상이다.)

## 보기

## 풀이
$A$의 가격이 $100$만 원 이상이고 만 원 인상될 때마다 판매량이 $20$대씩 줄어든다. 인상되는 가격을 $x$ (만 원), 전체 판매 금액을 $y$ (만 원)이라 하면

$y = ( 100+x ) ( 2400-20x ) = -20 ( x-10 )^{2} + 242000$

따라서 가격을 $10$만 원 올렸을 때 전체 판매 금액이 최대이므로 $A$의 가격은 $110$만 원이다.

$\therefore a = 110$

정답: $110$
