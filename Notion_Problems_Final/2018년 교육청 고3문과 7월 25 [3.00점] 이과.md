## 문제
등차수열 $\{ a_{n} \}$ 의 첫째항부터 제 $n$ 항까지의 합을 $S_{n}$ 이라 하자.
$a_{2} = 7$, $S_{7} - S_{5} = 50$ 일 때, $a_{11}$ 의 값을 구하시오.

## 풀이
$a_{2} = a_{1} + d = 7$ $\cdots$ ㉠  
$S_{7} - S_{5} = a_{7} + a_{6} = 2a_{1} + 11d = 50$ $\cdots$ ㉡  
두 식 ㉠, ㉡을 연립하면  
$a_{1} = 3$, $d = 4$  
$a_{11} = a_{1} + 10d = 43$  

정답: $43$
