## 문제
$x$에 대한 연립부등식 
$$ \begin{cases} x^{2} + ax + b \geq 0 \\ x^{2} + cx + d \leq 0 \end{cases} $$
의 해가 $1 \leq x \leq 3$ 또는 $x = 4$일 때, $a + b + c + d$의 값은? 



> [!example] 📸 (그림/그래프)

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
부등식 $x^{2} + ax + b \geq 0$와 $x^{2} + cx + d \leq 0$의 해를 수직선에 나타내면 그림과 같다. 

그림에서 $x^{2} + ax + b \geq 0$의 해는 $x \leq 3$ 또는 $x \geq 4$이므로 $(x - 3)(x - 4) \geq 0$이다. 
$x^{2} - 7x + 12 \geq 0$에서 $a = -7$, $b = 12$

$x^{2} + cx + d \leq 0$의 해는 $1 \leq x \leq 4$이므로 $(x - 1)(x - 4) \leq 0$이다. 
$x^{2} - 5x + 4 \leq 0$에서 $c = -5$, $d = 4$

따라서 $a + b + c + d = -7 + 12 - 5 + 4 = 4$

정답: ④
