## 문제
수직선 위의 두 점 $A(3)$, $B(7)$에 대하여 점 $P(x)$가 $\overline{AP} + \overline{BP} \leq 8$을 만족시킬 때, 선분 $OP$의 길이의 최댓값과 최솟값의 합은? (단, $O$는 원점이다.)

## 보기
① $7$   ② $8$   ③ $9$   ④ $10$   ⑤ $11$

## 풀이
점 $P$의 좌표를 $x$라 하면 
$\overline{AP} = |x-3|$, $\overline{BP} = |x-7|$ 

$\overline{AP} + \overline{BP} = |x-3| + |x-7|$이므로 
$|x-3| + |x-7| \leq 8$을 풀면 

(ⅰ) $x < 3$일 때 
$-(x-3) - (x-7) \leq 8$에서 $x \geq 1$ 
$\therefore 1 \leq x < 3$ 

(ⅱ) $3 \leq x < 7$일 때 
$(x-3) - (x-7) \leq 8$에서 $0 \leq 4$이므로 $3 \leq x < 7$ 

(ⅲ) $x \geq 7$일 때 
$(x-3) + (x-7) \leq 8$에서 $x \leq 9$ 
$\therefore 7 \leq x \leq 9$ 

이상에서 $1 \leq x \leq 9$이다. 

$\therefore 1 \leq \overline{OP} \leq 9$ 

따라서 선분 $OP$의 길이의 최댓값과 최솟값의 합은 $10$이다. 

정답: ④
