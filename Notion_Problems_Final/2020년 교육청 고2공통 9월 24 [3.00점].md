## 문제
두 수열 $\{$ $a_{n}$ $\}$, $\{$ $b_{n}$ $\}$ 에 대하여
$\sum_{n=1}^{5} ( a_{n} - b_{n} ) = 10$, $\sum_{n=1}^{6} ( 2 a_{n} - 2 b_{n} ) = 56$
일 때, $a_{6} - b_{6}$ 의 값을 구하시오.

## 보기

## 풀이
$2 \sum_{n=1}^{6} ( a_{n} - b_{n} ) = 56$ 이므로
$\sum_{n=1}^{6} ( a_{n} - b_{n} ) = 28$ 이다.

수열의 합과 일반항 사이의 관계에 의하여
$a_{6} - b_{6} = \sum_{n=1}^{6} ( a_{n} - b_{n} ) - \sum_{n=1}^{5} ( a_{n} - b_{n} )$
$= 28 - 10 = 18$

정답: $18$
