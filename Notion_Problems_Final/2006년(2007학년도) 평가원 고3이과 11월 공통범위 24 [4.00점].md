## 문제
> [!example] 📸 (그림/그래프)

그림과 같이 평면 $\alpha$ 위에 한 변의 길이가 $3$ 인 정삼각형 $ABC$ 가 있고, 반지름의 길이가 $2$ 인 구 $S$ 는 점 $A$ 에서 평면 $\alpha$ 에 접한다. 구 $S$ 위의 점 $D$ 에 대하여 선분 $AD$ 가 구 $S$ 의 중심 $O$ 를 지날 때, $|\vec{AB} + \vec{DC}|^{2}$ 의 값을 구하시오.

## 보기


## 풀이
$A$ 를 원점, $\overline{AB}$ 가 $x$ 축, $\alpha$ 를 $xy$ 평면, $\overline{AD}$ 를 $z$ 축으로 하는 좌표축을 도입하면 
$A(0, 0, 0)$, $B(3, 0, 0)$, $C \left( \frac{3}{2}, \frac{3\sqrt{3}}{2}, 0 \right)$, $D(0, 0, 4)$ 

$\therefore |\vec{AB} + \vec{DC}|^2$ 
$= |(3, 0, 0) + \left( \frac{3}{2}, \frac{3\sqrt{3}}{2}, -4 \right)|^2$ 
$= \left| \left( \frac{9}{2}, \frac{3\sqrt{3}}{2}, -4 \right) \right|^2$ 
$= \frac{81}{4} + \frac{27}{4} + 16 = 27 + 16 = 43$

정답: $43$
