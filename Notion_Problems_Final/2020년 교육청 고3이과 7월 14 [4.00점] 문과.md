## 문제
확률변수 $X$는 정규분포 $N(m, 2^2)$, 확률변수 $Y$는 정규분포 $N(2m, \sigma^2)$을 따른다.
$P(X \le 8) + P(Y \le 8) = 1$을 만족시키는 $m$과 $\sigma$에 대하여 $P(Y \le m+4) = 0.3085$일 때, $P(X \le \sigma)$의 값을 다음 표준정규분포표를 이용하여 구한 것은?

$$
\begin{array}{c|c}
z & P(0 \le Z \le z) \\ \hline
0.5 & 0.1915 \\
1.0 & 0.3413 \\
1.5 & 0.4332 \\
2.0 & 0.4772
\end{array}
$$

## 보기
① $0.0228$   ② $0.0668$   ③ $0.1359$   ④ $0.1587$   ⑤ $0.2857$

## 풀이
$P(Y \le m+4) = P(Z \le \frac{(m+4)-2m}{\sigma}) = P(Z \le \frac{4-m}{\sigma}) = 0.3085$
표준정규분포표에서 $P(0 \le Z \le 0.5) = 0.1915$이므로 $P(Z \le -0.5) = 0.5 - 0.1915 = 0.3085$이다.
따라서 $\frac{4-m}{\sigma} = - \frac{1}{2}$ ⋯ ㉠

$P(X \le 8) + P(Y \le 8) = 1$에서
$P(Z \le \frac{8-m}{2}) + P(Z \le \frac{8-2m}{\sigma}) = 1$
이때 $P(Z \le k) + P(Z \le -k) = 1$이 성립하므로
$\frac{8-m}{2} = - \frac{8-2m}{\sigma} = \frac{2m-8}{\sigma}$ 이다.

㉠에서 $8-2m = -\sigma$이므로 $\frac{2m-8}{\sigma} = \frac{-(8-2m)}{\sigma} = \frac{\sigma}{\sigma} = 1$이다.
즉, $\frac{8-m}{2} = 1$에서
$m = 6$
$m = 6$을 ㉠에 대입하면 $\frac{4-6}{\sigma} = - \frac{1}{2}$이므로
$\sigma = 4$

따라서 구하고자 하는 값은
$P(X \le \sigma) = P(X \le 4)$
$= P(Z \le \frac{4-6}{2})$
$= P(Z \le -1)$
$= 0.5 - P(0 \le Z \le 1.0)$
$= 0.5 - 0.3413 = 0.1587$

정답은 ④이다.
