## 문제
두 함수 $f(x)=2^x$, $g(x)=x-[x]$에 대하여 합성함수 $y=(g \circ f)(x)$의 그래프와 직선 $y=-\frac{1}{n}x+1$ ($n=1, 2, 3, \cdots$)이 만나는 점의 개수를 $a_n$이라 하자. $a_1 + a_2 + a_3$의 값은? ($[x]$는 $x$보다 크지 않은 최대의 정수이다.)

## 보기
① $6$   ② $8$   ③ $10$   ④ $12$   ⑤ $14$

## 풀이
$f(x) = 2^x$, $g(x) = x - [x]$ 이므로 
$g(f(x)) = f(x) - [f(x)] = 2^x - [2^x]$

ⅰ) $x < 0$ 일 때, 
$0 < 2^x < 1$ 이므로 $[2^x] = 0$ 
즉, $g(f(x)) = 2^x - [2^x] = 2^x$

ⅱ) $0 \le x < \log_2 2$ 일 때, 
$1 \le 2^x < 2$ 이므로 $[2^x] = 1$ 
즉, $g(f(x)) = 2^x - [2^x] = 2^x - 1$ 이므로 
$y=g(f(x))$ 의 그래프는 $y=2^x$ 의 그래프를 $y$ 축 방향으로 $-1$ 만큼 평행이동 시킨 것이다. 

ⅲ) $\log_2 2 \le x < \log_2 3$ 일 때, 
$2 \le 2^x < 3$ 이므로 $[2^x] = 2$ 
즉, $g(f(x)) = 2^x - [2^x] = 2^x - 2$ 이므로 
$y=g(f(x))$ 의 그래프는 $y=2^x$ 의 그래프를 $y$ 축 방향으로 $-2$ 만큼 평행이동 시킨 것이다. 

ⅳ) $\log_2 3 \le x < \log_2 4$ 일 때, 
$3 \le 2^x < 4$ 이므로 $[2^x] = 3$ 
즉, $g(f(x)) = 2^x - [2^x] = 2^x - 3$ 이므로 
$y=g(f(x))$ 의 그래프는 $y=2^x$ 의 그래프를 $y$ 축 방향으로 $-3$ 만큼 평행이동 시킨 것이다. 

다음의 그래프는 각각의 $x$ 의 범위에 따라 $y=2^x$ 의 그래프를 평행이동하는 과정을 보여주는 것이다. 

> [!example] 📸 (그림/그래프)

그러므로 $y=(g \circ f)(x)=2^{x} - [2^{x}]$ 의 그래프와 $n=1, 2, 3$ 일 때 $y = - \frac{1}{n}x + 1$ 의 그래프를 그리면 그림과 같다. 

> [!example] 📸 (그림/그래프)

그림을 참고하여 $n=1, 2, 3$ 일 때 $y = - \frac{1}{n}x + 1$ 과 $y= 2^x - [2^x]$ 의 그래프가 만나는 점의 개수 $a_n$ 을 구하면 

$n=1$ 일 때, $y = - \frac{1}{n}x + 1$ 의 $x$ 절편이 $1 = \log_2 2$ 이므로 교점의 개수 $a_1 = 2$ 
$n=2$ 일 때, $y = - \frac{1}{n}x + 1$ 의 $x$ 절편이 $2 = \log_2 2^2$ 이므로 교점의 개수 $a_2 = 2^2 = 4$ 
$n=3$ 일 때, $y = - \frac{1}{n}x + 1$ 의 $x$ 절편이 $3 = \log_2 2^3$ 이므로 교점의 개수 $a_3 = 2^3 = 8$ 

따라서 $a_1 + a_2 + a_3 = 2 + 4 + 8 = 14$ 

(참고) 자연수 $k$ 에 대해 $\log_2 k \le x < \log_2 (k+1)$ 일 때 
$k \le 2^x < k+1$ 이므로 $[2^x] = k$ 
즉, $g(f(x)) = 2^x - [2^x] = 2^x - k$ 이므로 
$y=g(f(x))$ 의 그래프는 $y=2^x$ 의 그래프를 $y$ 축 방향으로 $-k$ 만큼 평행이동 시킨 것이다. 
$n=k$ ($k$ 는 자연수)일 때, 직선 $y = - \frac{1}{n}x + 1$ 의 $x$ 절편이 $k = \log_2 2^k$ 이므로 교점의 개수는 $a_k = 2^k$ 이다.
