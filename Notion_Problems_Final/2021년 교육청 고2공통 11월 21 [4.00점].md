## 문제
수열 $\{ a_n \}$이 다음 조건을 만족시킨다.

(가) $a_1$은 $1$이 아닌 양수이다.
(나) 모든 자연수 $n$에 대하여 $a_{2n-1} + a_{2n} = 1$이고 $a_{2n} \cdot a_{2n+1} = 1$이다.

$\sum_{n=1}^{14} ( | a_n | - a_n ) = 10$이 되도록 하는 모든 $a_1$의 값의 합은?

## 보기
① $\frac{10}{3}$   ② $4$   ③ $\frac{14}{3}$   ④ $\frac{16}{3}$   ⑤ $6$

## 풀이
조건 (가), (나)에 의해 
$$ a_2 = 1 - a_1 $$
$$ a_3 = \frac{1}{a_2} = \frac{1}{1 - a_1} $$
$$ a_4 = 1 - a_3 = 1 - \frac{1}{1 - a_1} = -\frac{a_1}{1 - a_1} $$
$$ a_5 = \frac{1}{a_4} = -\frac{1 - a_1}{a_1} = 1 - \frac{1}{a_1} $$
$$ a_6 = 1 - a_5 = 1 - \left( 1 - \frac{1}{a_1} \right) = \frac{1}{a_1} $$
$$ a_7 = \frac{1}{a_6} = a_1 $$
$$ a_8 = 1 - a_7 = 1 - a_1 = a_2 $$
$\vdots$
그러므로 수열 $\{ a_n \}$은 모든 자연수 $n$에 대하여 $a_{n+6} = a_n$을 만족시킨다.

$$ |a_n| - a_n = \begin{array}{ll} 0 & (a_n \ge 0) \\ -2a_n & (a_n < 0) \end{array} $$
이므로 $\sum_{n=1}^{14} ( |a_n| - a_n ) = 10$이 되기 위해서는 $a_1, a_2, \dots, a_{14}$ 중에서 음수인 모든 항의 합이 $-5$이어야 한다.

(ⅰ) $0 < a_1 < 1$일 때
$a_1 > 0, a_2 > 0, a_3 > 0, a_4 < 0, a_5 < 0, a_6 > 0$ 이므로
음수인 항은 $a_4, a_5, a_{10}, a_{11}$이다.
$$ a_4 + a_5 + a_{10} + a_{11} = -5 $$
$a_4 = s$라 하면 $a_5 = \frac{1}{s}$이므로
$$ a_4 + a_5 + a_{10} + a_{11} = s + \frac{1}{s} + s + \frac{1}{s} = 2s + \frac{2}{s} = -5 $$
$$ 2s^2 + 5s + 2 = 0, \quad (2s + 1)(s + 2) = 0 $$
위의 식에서 $s = -\frac{1}{2}$ 또는 $s = -2$이다.
$s = -\frac{a_1}{1 - a_1}$이므로
$a_1 = \frac{1}{3}$ 또는 $a_1 = \frac{2}{3}$이다.

(ⅱ) $a_1 > 1$일 때
$a_1 > 1, a_2 < 0, a_3 < 0, a_4 > 0, a_5 > 0, a_6 > 0$ 이므로
음수인 항은 $a_2, a_3, a_8, a_9, a_{14}$이다.
$$ a_2 + a_3 + a_8 + a_9 + a_{14} = -5 $$
$a_2 = t$라 하면 $a_3 = \frac{1}{t}$이므로
$$ a_2 + a_3 + a_8 + a_9 + a_{14} = t + \frac{1}{t} + t + \frac{1}{t} + t = 3t + \frac{2}{t} = -5 $$
$$ 3t^2 + 5t + 2 = 0, \quad (3t + 2)(t + 1) = 0 $$
위의 식에서 $t = -\frac{2}{3}$ 또는 $t = -1$이다.
$t = 1 - a_1$이므로
$a_1 = \frac{5}{3}$ 또는 $a_1 = 2$이다.

따라서 모든 $a_1$의 값의 합은
$$ \frac{1}{3} + \frac{2}{3} + \frac{5}{3} + 2 = \frac{14}{3} $$
