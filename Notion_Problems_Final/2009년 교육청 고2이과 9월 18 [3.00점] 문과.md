## 문제
$\lim _{ n \to \infty } \frac { ( a ^{ 2 } -2a-3 ) n ^{ 2 } + ( a+1 ) n+5 } { n+2 } =b$ ($b \neq 0$)를 만족시키는 두 실수 $a$, $b$에 대하여 $a+b$의 값은? (단, $n$은 자연수이다.)

## 보기
① $3$   ② $4$   ③ $5$   ④ $6$   ⑤ $7$

## 풀이
$\lim _{ n \to \infty } \frac { ( a ^{ 2 } -2a-3 ) n ^{ 2 } + ( a+1 ) n+5 } { n+2 }$
$= \lim _{ n \to \infty } \frac { (a-3)(a+1)n^2 + (a+1)n + 5 } { n+2 } =b$ 이려면 분자의 최고차항인 $n^2$의 계수가 $0$이어야 하므로
$(a-3)(a+1)=0$ 이고, $b = \frac{a+1}{1} = a+1$ ($b \neq 0$)이어야 한다.
$(a-3)(a+1)=0$ 에서 $a=3$ 또는 $a=-1$ 이다.
$a=-1$ 이면 $b = -1+1 = 0$ 이 되어 $b \neq 0$ 이라는 조건에 어긋난다.
따라서 $a=3$ 이고, 이때 $b = 3+1 = 4$ 이다.
따라서 $a+b = 3+4 = 7$ 이다.

정답: ⑤
