## 문제
두 자연수 $a$, $b$에 대하여 닫힌구간 $[0, 2\pi]$에서 정의된 함수 
$$f(x) = |2\sin ax + b|$$ 
가 있다. 함수 $y = f(x)$의 그래프와 직선 $y = 1$이 만나는 점의 개수가 $10$이 되도록 하는 $a$, $b$의 모든 순서쌍 $(a, b)$에 대하여 $a+b$의 최댓값과 최솟값을 각각 $M$, $m$이라 할 때, $M+m$의 값은?

## 보기
① $14$   ② $15$   ③ $16$   ④ $17$   ⑤ $18$

## 풀이
자연수 $a$, $b$에 대하여 $g(x) = 2\sin ax + b$라 하면 함수 $g(x)$의 최댓값, 최솟값은 $b+2$, $b-2$, 주기는 $\frac{2\pi}{a}$이다. 
자연수 $b$에 대하여 $b+2 \ge 3$이므로 함수 $y = f(x)$의 그래프와 직선 $y = 1$이 만나기 위해서는 
$$|b-2| \le 1, \text{ 즉 } 1 \le b \le 3$$ 
을 만족해야 한다. 

(ⅰ) $b = 1$일 때 
함수 $f(x) = |2\sin ax + 1|$의 그래프와 직선 $y = 1$의 그래프의 교점의 개수가 $10$이므로 함수 $g(x) = 2\sin ax + 1$의 그래프와 직선 $y = 1$ 또는 $y = -1$의 그래프의 교점의 개수가 $10$이어야 한다. 
$0 \le x < \frac{2\pi}{a}$에서 교점의 개수가 $3$이므로 다음 그림과 같이 교점의 개수가 $10$이 되기 위해서는 
$$\frac{6\pi}{a} \le 2\pi < \frac{7\pi}{a}, \text{ 즉 } 3 \le a < \frac{7}{2}$$ 
$a$의 값은 $3$이다. 

> [!example] 📸 (그림/그래프)

(ⅱ) $b = 2$일 때 
$f(x) = |2\sin ax + 2| = 2\sin ax + 2$이므로 함수 $y = f(x)$의 그래프와 직선 $y = 1$의 교점의 개수가 $10$이어야 한다. 
$0 \le x < \frac{2\pi}{a}$에서 교점의 개수가 $2$이고 교점의 $x$좌표는 
$$2\sin ax + 2 = 1, \sin ax = -\frac{1}{2}$$ 
$$ax = \frac{7}{6}\pi \text{ 또는 } ax = \frac{11}{6}\pi$$ 
$$\therefore x = \frac{7}{6a}\pi \text{ 또는 } x = \frac{11}{6a}\pi$$ 
따라서 함수 $y = f(x)$의 그래프와 직선 $y = 1$의 교점의 개수가 $10$이 되기 위해서는 다음 그림과 같이 
$$\frac{8}{a}\pi + \frac{11}{6a}\pi \le 2\pi < \frac{10}{a}\pi + \frac{7}{6a}\pi, \text{ 즉 } \frac{59}{12} \le a < \frac{67}{12}$$ 
$a$의 값은 $5$이다. 

> [!example] 📸 (그림/그래프)

(ⅲ) $b = 3$일 때 
$f(x) = |2\sin ax + 3| = 2\sin ax + 3$이므로 함수 $y = f(x)$의 그래프와 직선 $y = 1$의 교점의 개수가 $10$이어야 한다. 
$0 \le x < \frac{2\pi}{a}$에서 교점의 개수가 $1$이고, 교점의 $x$좌표는 
$$2\sin ax + 3 = 1, \sin ax = -1$$ 
$$ax = \frac{3}{2}\pi$$ 
$$\therefore x = \frac{3}{2a}\pi$$ 
따라서 함수 $y = f(x)$의 그래프와 직선 $y = 1$의 교점의 개수가 $10$이 되기 위해서는 다음 그림과 같이 
$$\frac{18}{a}\pi + \frac{3}{2a}\pi \le 2\pi < \frac{20}{a}\pi + \frac{3}{2a}\pi, \text{ 즉 } \frac{39}{4} \le a < \frac{43}{4}$$ 
$a$의 값은 $10$이다. 

> [!example] 📸 (그림/그래프)

이상에서 자연수 $a$, $b$의 순서쌍 $(a, b)$는 $(3, 1)$, $(5, 2)$, $(10, 3)$이므로 
$a+b$의 최댓값은 $13$, 최솟값은 $4$이다. 
$M = 13, m = 4$이므로 $M+m = 17$
