## 문제
두 다항식 $A = x^{2} + y^{2}$, $B = 2x^{2} + xy - y^{2}$에 대하여 $A + B$를 간단히 하면?

## 보기
① $x^{2} - xy$   ② $x^{2} + xy$   ③ $3x^{2} - 2y^{2}$   ④ $3x^{2} - xy$   ⑤ $3x^{2} + xy$

## 풀이
두 다항식 $A = x^{2} + y^{2}$, $B = 2x^{2} + xy - y^{2}$에서 
$A + B = (x^{2} + y^{2}) + (2x^{2} + xy - y^{2}) = 3x^{2} + xy$
따라서 정답은 ⑤이다.
