## 문제
열린구간 $( - \frac{\pi}{2} , \frac{\pi}{2} )$에서 정의된 함수 
$f(x) = \ln ( \frac{\sec x + \tan x}{a} )$
의 역함수를 $g(x)$라 하자. 
$\lim_{x \to -2} \frac{g(x)}{x+2} = b$일 때, 두 상수 $a, b$의 곱 $ab$의 값은? (단, $a > 0$)

## 보기
① $\frac{e^{2}}{4}$   ② $\frac{e^{2}}{2}$   ③ $e^{2}$   ④ $2e^{2}$   ⑤ $4e^{2}$

## 풀이
$\lim_{x \to -2} \frac{g(x)}{x+2} = b$에서 $x \to -2$일 때 (분모) $\to 0$이고 극한값이 존재하므로 
$\lim_{x \to -2} g(x) = 0$
즉, $g(-2) = 0$이고 $g(x)$는 $f(x)$의 역함수이므로 
$f(0) = -2$
이때 
$f(0) = \ln ( \frac{\sec 0 + \tan 0}{a} ) = \ln ( \frac{1+0}{a} ) = \ln \frac{1}{a}$
이므로 
$\ln \frac{1}{a} = -2$
에서 $\frac{1}{a} = e^{-2} = \frac{1}{e^{2}}$, 즉 $a = e^{2}$
또, 
$b = \lim_{x \to -2} \frac{g(x)}{x+2} = \lim_{x \to -2} \frac{g(x) - g(-2)}{x - (-2)} = g'(-2)$
한편 
$f(x) = \ln ( \frac{\sec x + \tan x}{a} ) = \ln ( \frac{\sec x + \tan x}{e^{2}} )$
에서 
$f'(x) = \frac{e^{2}}{\sec x + \tan x} \times \frac{\sec x \tan x + \sec^{2} x}{e^{2}} = \sec x$
이므로 
$g'(-2) = \frac{1}{f'(g(-2))} = \frac{1}{f'(0)} = \frac{1}{\sec 0} = 1$
따라서 $a = e^{2}$, $b = 1$이므로 
$ab = e^{2} \times 1 = e^{2}$
