## 문제
함수 $f(x) = x \ln x$ 에 대하여 등식 
$$f(e^{2}) - f(e) = e(e-1) f'(c)$$
를 만족시키는 $c$ 가 열린구간 $(e, e^{2})$ 에 존재한다. $\ln c$ 의 값은?

## 보기
① $\frac{3}{e}$   ② $\frac{e+2}{e}$   ③ $\frac{2}{e-1}$   ④ $\frac{e}{e-1}$   ⑤ $\frac{2e}{e+1}$

## 풀이
$f(x) = x \ln x$ 에서 $f'(x) = \ln x + 1$ 이므로
$$f'(c) = \ln c + 1$$
주어진 등식에서
$$\frac{f(e^{2}) - f(e)}{e(e-1)} = \frac{2e^{2} - e}{e(e-1)} = \frac{2e-1}{e-1} = 1 + \frac{e}{e-1}$$
$f'(c) = \frac{f(e^{2}) - f(e)}{e(e-1)}$ 이므로
$$\ln c + 1 = 1 + \frac{e}{e-1}$$
∴ $$\ln c = \frac{e}{e-1}$$

정답: ④
