## 문제

자연수 $N$을 음이 아닌 정수 $m$과 홀수 $p$에 대하여 $N = 2^{m} \times p$로 나타낼 때, $f(N) = m$이라 하자. 
예를 들어, $40 = 2^{3} \times 5$이므로 $f(40) = 3$이다. 
다음은 모든 자연수 $n$에 대하여 

$$ f(3^{2n-1} + 1) = 2 \quad \cdots \cdots (*) $$

임을 수학적 귀납법을 이용하여 증명한 것이다.

$$
\begin{array}{|l|}
\hline
(i) \ n = 1 \text{일 때,} \\
3^{1} + 1 = 2^{2} \times 1 \text{이므로 } f(3^{1} + 1) = 2 \text{이다.} \\
\text{따라서 } n = 1 \text{일 때 } (*) \text{이 성립한다.} \\
\\
(ii) \ n = k \text{일 때 } (*) \text{이 성립한다고 가정하면} \\
f(3^{2k-1} + 1) = 2 \text{이다.} \\
\text{음이 아닌 정수 } m \text{과 홀수 } p \text{에 대하여 } 3^{2k-1} + 1 = 2^{m} \times p \text{라 하면} \\
m = 2 \text{이므로} \\
3^{2k-1} + 1 = \boxed{\text{ (가) }} \times p \\
\text{이다. } 3^{2(k+1)-1} + 1 = 9 \times 3^{2k-1} + 1 \text{이므로} \\
3^{2(k+1)-1} + 1 = 2^{2} \times ( \boxed{\text{ (나) }} ) \\
p \text{는 홀수이므로 } \boxed{\text{ (나) }} \text{도 홀수이다.} \\
\text{따라서 } f(3^{2(k+1)-1} + 1) = 2 \text{이다.} \\
n = k + 1 \text{일 때도 } (*) \text{이 성립한다.} \\
\\
(i), (ii) \text{에 의하여 모든 자연수 } n \text{에 대하여 } f(3^{2n-1} + 1) = 2 \text{이다.} \\
\hline
\end{array}
$$

위의 (가)에 알맞은 수를 $a$, (나)에 알맞은 식을 $g(p)$라 할 때, $a + g(7)$의 값은?

## 보기

① $65$   ② $67$   ③ $69$   ④ $71$   ⑤ $73$

## 풀이

$(i)$ $n = 1$일 때,
$3^{1} + 1 = 2^{2} \times 1 = 4$이므로 $f(3^{1} + 1) = 2$이다.

$(ii)$ $n = k$일 때, $f(3^{2k-1} + 1) = 2$라 가정하면
$3^{2k-1} + 1 = 2^{2} \times p = 4p$ ($p$는 홀수)이다.
따라서 (가)에 알맞은 수 $a = 4$이다.

이때 $3^{2k-1} = 4p - 1$이므로
$3^{2(k+1)-1} + 1$
$= 3^{2k+1} + 1$
$= 9 \times 3^{2k-1} + 1$
$= 9(4p - 1) + 1$
$= 36p - 9 + 1$
$= 36p - 8$
$= 2^{2} \times (9p - 2)$

$p$가 홀수이면 $9p$도 홀수이고, 홀수에서 $2$를 뺀 $9p - 2$도 홀수이다.
따라서 (나)에 알맞은 식 $g(p) = 9p - 2$이다.

$a = 4$, $g(p) = 9p - 2$이므로
$a + g(7) = 4 + (9 \times 7 - 2)$
$= 4 + (63 - 2)$
$= 4 + 61$
$= 65$

정답은 ①이다.
