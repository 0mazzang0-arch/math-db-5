## 문제
다음은 $11^{15} < 55^{9} < 11^{17} < 33^{13} < 11^{19}$ 임을 이용하여 세 수 $A = 3^{273}$, $B = 5^{189}$, $C = 11^{126}$ 의 크기를 비교하는 과정이다. 

먼저 $A$ 와 $B$ 의 크기를 비교해 보자. 

$189$ 와 $273$ 의 최대공약수는 $(가)$ 이므로 
$$\frac{A}{B} = \frac{3^{273}}{5^{189}} = \left( \frac{3^{13}}{5^{9}} \right)^{21}$$
이다. 

$11^{15} < 55^{9} < 11^{17} < 33^{13} < 11^{19}$ 을 이용하면 
$3^{13}$ $(나)$ $5^{9}$ 이므로 
$\frac{A}{B}$ $(나)$ $1$ 이다. 
$\therefore A$ $(나)$ $B$ 

같은 방법으로 $A$ 와 $C$, $B$ 와 $C$ 의 크기를 비교할 수 있다. 
따라서 $A, B, C$ 의 대소관계는 $(다)$ 이다. 

이 과정에서 $(가), (나), (다)$ 에 알맞은 것은?

## 보기
① $(가): 21, (나): >, (다): B < A < C$ ② $(가): 21, (나): <, (다): A < C < B$ ③ $(가): 21, (나): <, (다): C < A < B$ ④ $(가): 57, (나): >, (다): B < C < A$ ⑤ $(가): 57, (나): <, (다): C < A < B$

## 풀이
[정답] ②

[해설]
먼저 $(가)$ 에 들어갈 $189$ 와 $273$ 의 최대공약수를 구하면 다음과 같다.
$189 = 3^{3} \times 7$
$273 = 3 \times 7 \times 13$
$\therefore (가) = 3 \times 7 = 21$

주어진 부등식 $11^{15} < 55^{9} < 11^{17} < 33^{13} < 11^{19}$ 에서
$11^{15} < 55^{9} = (5 \times 11)^{9} = 5^{9} \times 11^{9}$ 이므로 $11^{6} < 5^{9}$
$33^{13} = (3 \times 11)^{13} = 3^{13} \times 11^{13} < 11^{19}$ 이므로 $3^{13} < 11^{6}$
따라서 $3^{13} < 11^{6} < 5^{9}$ 이 되어 $3^{13} < 5^{9}$ 이다.
$\therefore (나) = <$

마지막으로 $A, B, C$ 의 크기를 비교하면
$$ \begin{array}{l} A = 3^{273} = (3^{13})^{21} \\ B = 5^{189} = (5^{9})^{21} \\ C = 11^{126} = (11^{6})^{21} \end{array} $$
위에서 구한 $3^{13} < 11^{6} < 5^{9}$ 에 의해 $A < C < B$ 이다.
$\therefore (다) = A < C < B$

이를 정리하면 다음과 같다.
$$ \begin{array}{c|c|c|c} \hline & (가) & (나) & (다) \\ \hline \text{값} & 21 & < & A < C < B \\ \hline \end{array} $$
