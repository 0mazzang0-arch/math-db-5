## 문제
상수 $k$ ($k > 1$)에 대하여 다음 조건을 만족시키는 수열 $\{a_{n}\}$이 있다.

$$
\begin{array}{|l|}
\hline
\text{모든 자연수 } n \text{에 대하여 } a_{n} < a_{n+1} \text{ 이고} \\
\text{곡선 } y=2^{x} \text{ 위의 두 점 } P_{n}(a_{n}, 2^{a_{n}}), P_{n+1}(a_{n+1}, 2^{a_{n+1}}) \text{을 지나는} \\
\text{직선의 기울기는 } k \times 2^{a_{n}} \text{이다.} \\
\hline
\end{array}
$$

> [!example] 📸 (그림/그래프)

점 $P_{n}$을 지나고 $x$축에 평행한 직선과 점 $P_{n+1}$을 지나고 $y$축에 평행한 직선이 만나는 점을 $Q_{n}$이라 하고 삼각형 $P_{n}Q_{n}P_{n+1}$의 넓이를 $A_{n}$이라 하자. 다음은 $a_{1}=1$, $\frac{A_{3}}{A_{1}}=16$일 때, $A_{n}$을 구하는 과정이다.

$$
\begin{array}{|l|}
\hline
\text{두 점 } P_{n}, P_{n+1} \text{을 지나는 직선의 기울기가 } k \times 2^{a_{n}} \text{이므로} \\
\quad 2^{a_{n+1}-a_{n}} = k(a_{n+1}-a_{n}) + 1 \\
\text{이다. 즉, 모든 자연수 } n \text{에 대하여 } a_{n+1}-a_{n} \text{은 방정식} \\
\quad 2^{x} = kx+1 \\
\text{의 해이다. } k>1 \text{이므로 방정식 } 2^{x} = kx+1 \text{은 오직 하나의 양의} \\
\text{실근 } d \text{를 갖는다. 따라서 모든 자연수 } n \text{에 대하여} \\
\quad a_{n+1}-a_{n} = d \\
\text{이고, 수열 } \{a_{n}\} \text{은 공차가 } d \text{인 등차수열이다.} \\
\text{점 } Q_{n} \text{의 좌표가 } (a_{n+1}, 2^{a_{n}}) \text{이므로} \\
\quad A_{n} = \frac{1}{2}(a_{n+1}-a_{n})(2^{a_{n+1}}-2^{a_{n}}) \\
\text{이다. } \frac{A_{3}}{A_{1}}=16 \text{이므로 } d \text{의 값은 (가) 이고,} \\
\text{수열 } \{a_{n}\} \text{의 일반항은 (나) 이다.} \\
\text{따라서 모든 자연수 } n \text{에 대하여 } A_{n} = \text{(다) 이다.} \\
\hline
\end{array}
$$

위의 (가)에 알맞은 수를 $p$, (나)와 (다)에 알맞은 식을 각각 $f(n)$, $g(n)$이라 할 때, $p + \frac{g(4)}{f(2)}$의 값은?

## 보기
① $118$   ② $121$   ③ $124$   ④ $127$   ⑤ $130$

## 풀이
$\frac{A_{3}}{A_{1}} = 16$ 이므로 
$\frac{\frac{1}{2} \cdot d \cdot (2^{a_{4}} - 2^{a_{3}})}{\frac{1}{2} \cdot d \cdot (2^{a_{2}} - 2^{a_{1}})} = \frac{2^{a_{3}+d} - 2^{a_{3}}}{2^{a_{1}+d} - 2^{a_{1}}} = 16$ 이다.
$a_{n}$은 공차가 $d$인 등차수열이므로 $a_{n} = 1 + (n-1)d$ 이다.
$\frac{2^{1+2d}(2^{d}-1)}{2^{1}(2^{d}-1)} = 2^{2d} = 16 = 2^{4}$
따라서 $d=2$ 이다. $\therefore p = 2$ (가)

$a_{1} = 1$ 이고 $d=2$ 이므로 
$a_{n} = 1 + (n-1) \cdot 2 = 2n-1$
$\therefore f(n) = 2n-1$ (나)
$f(2) = 2(2)-1 = 3$

$A_{n} = \frac{1}{2} \cdot d \cdot (2^{a_{n+1}} - 2^{a_{n}})$
$A_{n} = \frac{1}{2} \cdot 2 \cdot (2^{2n+1} - 2^{2n-1}) = 2^{2n-1}(2^{2} - 1) = 3 \times 2^{2n-1}$
$\therefore g(n) = 3 \times 2^{2n-1}$ (다)
$g(4) = 3 \times 2^{2(4)-1} = 3 \times 2^{7}$

방정식 $2^{x} = kx+1$ 에서 $d=2$ 가 근이므로 
$2^{2} = 2k+1 \implies 4 = 2k+1 \implies k = \frac{3}{2}$

최종 구하는 값은:
$p + \frac{g(4)}{f(2)} = 2 + \frac{3 \times 2^{7}}{3} = 2 + 128 = 130$

정답은 ⑤ 이다.
