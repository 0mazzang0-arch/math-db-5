## 문제
실수에서 정의된 함수 $f(x)$를 $f(x)=x-[x]$라고 할 때, <보기>에서 옳은 것을 모두 고르면? (단, $[x]$는 $x$를 넘지 않는 최대 정수이다.)

ㄱ. $f(-3.14)=f(3.14)$
ㄴ. 모든 $x$에 대하여, $\{ y | y=[f(x)] \} = \{ -1, 0 \}$
ㄷ. 모든 $x$에 대하여, $f(x)=f(x+1)$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄷ$   ④ $ㄱ, ㄷ$   ⑤ $ㄴ, ㄷ$

## 풀이
[정답] ③ 

함수 $f(x)$를 $f(x)=x-[x]$라 하면 그래프는 
(ⅰ) $-1 \le x < 0$ 일 때, $f(x)=x+1$ 
(ⅱ) $0 \le x < 1$ 일 때, $f(x)=x$ 
(ⅲ) $1 \le x < 2$ 일 때, $f(x)=x-1$ 
(ⅳ) $2 \le x < 3$ 일 때, $f(x)=x-2$ 와 같이 되므로 

> [!example] 📸 (그림/그래프)

ㄱ. $f(-3.14) = -3.14-[-3.14] = -3.14-(-4) = 0.86$
$f(3.14)=3.14-[3.14]=3.14-3=0.14$ 
$\therefore f(-3.14) \neq f(3.14)$ 

ㄴ. 함수 $f(x)$의 치역은 $\{ f(x) | 0 \le f(x) < 1 \}$ 이므로 
$\{ y | y=[f(x)] \} = \{ 0 \}$ 이다. 

ㄷ. 위의 그림에서 함수 $f(x)$는 주기가 $1$인 주기 함수이므로 
$\therefore f(x)=f(x+1)$ 

따라서 옳은 것은 $ㄷ$이다.
