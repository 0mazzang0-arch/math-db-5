## 문제
다음은 명제 `$a$, $b$가 자연수일 때 $a^{2}+b^{2}$이 $3$으로 나누어 떨어지면 $a^{n}+b^{n}$ ($n=3, 4, 5, \cdots$)도 $3$으로 나누어 떨어진다`를 증명하는 과정이다.

$$ 
\begin{array}{|l|} 
\hline 
a, b \text{를 } 3 \text{으로 나누었을 때의 몫을 각각 } q_{1}, q_{2} \text{라 하고 나머지를 각각 } r_{1}, r_{2} \text{라 하면} \\ 
a=3 q_{1}+r_{1}, ~ b=3 q_{2}+r_{2} \\ 
\therefore a^{2}+b^{2}= (3 q_{1}+r_{1} )^{2}+ (3 q_{2}+r_{2} )^{2} \\ 
=3 ( 3q _{1}^{2} +3q _{2}^{2} + (가) ) +r _{1}^{2} +r _{2}^{2} \\ 
a^{2}+b^{2} \text{이 } 3 \text{으로 나누어 떨어지므로} \\ 
r _{1} = (나), ~ r _{2} = (다) \\ 
\text{따라서, } a ^{n} +b ^{n} ( n=3, 4, 5, \cdots ) \text{은 } 3 \text{으로 나누어 떨어진다.} \\ 
\hline 
\end{array} 
$$

위의 $(가)$, $(나)$, $(다)$에 알맞은 것을 차례로 나열한 것은?

## 보기
① $(가) : 2 q_{1} r_{1}+2 q_{2} r_{2}$, $(나) : 1$, $(다) : 1$
② $(가) : 2 q_{1} r_{1}+2 q_{2} r_{2}$, $(나) : 0$, $(다) : 0$
③ $(가) : 2 q_{1} r_{1}+2 q_{2} r_{2}$, $(나) : 2$, $(다) : 2$
④ $(가) : 2 q_{1} q_{2}+2 r_{1} r_{2}$, $(나) : 1$, $(다) : 1$
⑤ $(가) : 2 q_{1} q_{2}+2 r_{1} r_{2}$, $(나) : 0$, $(다) : 0$

## 풀이
$$a ^{2} +b ^{2} =3 ( 3q _{1} ^{2} +3q _{2} ^{2} + 2q _{1} r _{1} +2q _{2} r _{2} ) +r _{1} ^{2} +r _{2} ^{2}$$ 
에서 $a^{2}+b^{2}$이 $3$으로 나누어 떨어지려면 $r_{1}^{2}+r_{2}^{2}$이 $3$으로 나누어 떨어져야 한다. 그런데 $r_{1}$, $r_{2}$는 $a$, $b$를 $3$으로 나누었을 때의 나머지이므로 $r_{1}=0, 1, 2$, $r_{2}=0, 1, 2$이다. 
따라서 $r_{1}^{2}+r_{2}^{2}$이 $3$으로 나누어 떨어지려면 $r _{1} = 0$, $r _{2} = 0$이어야 한다. 

$\therefore a=3 q_{1}, b=3 q_{2}$ 
$\therefore a^{n}+b^{n}=3^{n} (q_{1} ^ {n}+q_{2} ^ {n} )$이므로 
$a^{n}+b^{n}$ ($n=3, 4, 5, \cdots$)은 $3$으로 나누어 떨어진다. 

그러므로 $(가) : 2 q_{1} r_{1}+2 q_{2} r_{2}$, $(나) : 0$, $(다) : 0$이다.
