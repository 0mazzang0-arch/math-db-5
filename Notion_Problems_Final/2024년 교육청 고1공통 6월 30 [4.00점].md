## 문제

두 이차함수 $f(x)$, $g(x)$가 다음 조건을 만족시킨다.

$$
\begin{array}{|l|}
\hline
\text{(가) 모든 실수 } x \text{에 대하여 } f(x) \leq 0 \leq g(x) \text{이다.} \\
\text{(나) } k-2 \leq x \leq k+2 \text{에서 함수 } f(x) \text{의 최댓값과 } k-2 \leq x \leq k+2 \text{에서 함수 } g(x) \text{의} \\
\quad \text{최솟값이 같게 되도록 하는 실수 } k \text{의 최솟값은 } 0 \text{, 최댓값은 } 1 \text{이다.} \\
\text{(다) 방정식 } f(x)=f(0) \text{의 모든 실근의 합은 음수이다.} \\
\hline
\end{array}
$$

$f(1)=-2$, $g(1)=2$일 때, $f(3)+g(11)$의 값을 구하시오.

## 풀이

조건 (가)에서 모든 실수 $x$에 대하여 $f(x) \leq 0$이므로 함수 $f(x)$는 최고차항의 계수가 음수이고 최댓값은 $0$보다 작거나 같다.
조건 (가)에서 모든 실수 $x$에 대하여 $g(x) \geq 0$이므로 함수 $g(x)$는 최고차항의 계수가 양수이고 최솟값은 $0$보다 크거나 같다.
조건 (나)에서 함수 $f(x)$의 최댓값과 함수 $g(x)$의 최솟값이 같아지기 위해서는 함수 $f(x)$의 최댓값과 함수 $g(x)$의 최솟값이 모두 $0$이어야 한다. 그러므로
$$ \begin{array}{ll} f(x)=a(x-b)^{2} \ (a<0) & \cdots ㉠ \\ g(x)=c(x-d)^{2} \ (c>0) & \cdots ㉡ \end{array} $$
이라 하자.

(ⅰ) $b=d$인 경우

> [!example] 📸 (그림/그래프)

$k-2 \leq b \leq k+2$에서 $b-2 \leq k \leq b+2$이므로 $k$의 최솟값이 $0$, 최댓값이 $1$이 되도록 하는 실수 $b$는 존재하지 않는다.

(ⅱ) $b<d$인 경우

> [!example] 📸 (그림/그래프)

$k$의 최솟값이 $0$, 최댓값이 $1$이고 $k-2 \leq b$, $d \leq k+2$에서 $d-2 \leq k \leq b+2$이므로 $b=-1$, $d=2$이다.
$b=-1$을 대입하고 ㉡에 $d=2$를 대입하면 $f(x)=a(x+1)^{2}$, $g(x)=c(x-2)^{2}$이다.
$f(x)=f(0)$은 $a(x+1)^{2}=a$이고 $(x+1)^{2}=1$, $x^{2}+2x=0$에서 모든 실근의 합은 $-2$이므로 조건 (다)를 만족시킨다.
$f(1)=4a=-2$에서 $a=-\frac{1}{2}$이고 $g(1)=c=2$이다.
$f(x)=-\frac{1}{2}(x+1)^{2}$, $g(x)=2(x-2)^{2}$이므로 $f(3)=-8$, $g(11)=162$이다.
$f(3)+g(11)=154$이다.

(ⅲ) $d<b$인 경우

> [!example] 📸 (그림/그래프)

$k$의 최솟값이 $0$, 최댓값이 $1$이고 $k-2 \leq d$, $b \leq k+2$에서 $b-2 \leq k \leq d+2$이므로 $b=2$, $d=-1$이다.
$b=2$를 대입하고 ㉡에 $d=-1$을 대입하면 $f(x)=a(x-2)^{2}$, $g(x)=c(x+1)^{2}$이다.
$f(x)=f(0)$은 $a(x-2)^{2}=4a$이고 $(x-2)^{2}=4$, $x^{2}-4x=0$에서 모든 실근의 합은 $4$이므로 조건 (다)를 만족시키지 않는다.

따라서 (ⅰ), (ⅱ), (ⅲ)에 의해 $f(3)+g(11)=154$이다.
