## 문제
모든 항이 양수인 수열 $\{ a_n \}$ 은 $a_1 = 10$ 이고
$( a_{n+1} )^{n+1} = \frac{ a_1 + ( a_2 )^2 + ( a_3 )^3 + \cdots + ( a_n )^n }{ n } \quad ( n \geq 1 )$
을 만족시킨다. 다음은 일반항 $a_n$ 을 구하는 과정의 일부이다.

$b_n = ( a_n )^n$ 이라 하면 $b_1 = 10$ 이고 주어진 식으로부터
$b_{n+1} = \frac{ b_1 + b_2 + \cdots + b_n }{ n } \quad ( n \geq 1 )$
이다. $S_n = \sum_{k=1}^n b_k$ 라 하면
$S_{n+1} = (가) \times S_n$
이다.
$S_1 = 10$, $S_n = S_1 \times \frac{ S_2 }{ S_1 } \times \frac{ S_3 }{ S_2 } \times \cdots \times \frac{ S_n }{ S_{n-1} } \quad ( n \geq 2 )$
를 이용하여 $S_n$ 을 구하면
$S_n = (나) \quad ( n \geq 1 )$
이다.
$\vdots$

위의 (가), (나)에 알맞은 식을 각각 $f(n)$, $g(n)$ 이라 할 때, $f(5) \times g(6)$ 의 값은?

## 보기
① $72$   ② $76$   ③ $80$   ④ $84$   ⑤ $88$

## 풀이
$b_n = ( a_n )^n$ 이라 하면 $b_1 = 10$ 이고 주어진 식으로부터
$b_{n+1} = \frac{ b_1 + b_2 + \cdots + b_n }{ n } \quad ( n \geq 1 )$
이다. $S_n = \sum_{k=1}^n b_k$ 라 하면 $b_{n+1} = \frac{ S_n }{ n }$ 에서
$S_n = n b_{n+1} = n ( S_{n+1} - S_n ) = n S_{n+1} - n S_n$
$( n + 1 ) S_n = n S_{n+1}$
$S_{n+1} = \frac{ n + 1 }{ n } \times S_n$ 이다.
그러므로 $(가)$ 에 알맞은 식은 $f(n) = \frac{ n + 1 }{ n }$ 이다.

$S_1 = 10$ 이고, $\frac{ S_{n+1} }{ S_n } = \frac{ n + 1 }{ n }$ 이므로 $n \geq 2$ 일 때,
$S_n = S_1 \times \frac{ S_2 }{ S_1 } \times \frac{ S_3 }{ S_2 } \times \cdots \times \frac{ S_n }{ S_{n-1} }$
$S_n = 10 \times \frac{ 2 }{ 1 } \times \frac{ 3 }{ 2 } \times \cdots \times \frac{ n }{ n - 1 } = 10 n$
$S_1 = 10$ 이므로 $S_n = 10 n \quad ( n \geq 1 )$ 이다.
그러므로 $(나)$ 에 알맞은 식은 $g(n) = 10 n$ 이다.

따라서 $f(n) = \frac{ n + 1 }{ n }$, $g(n) = 10 n$ 이므로
$f(5) \times g(6) = \frac{ 6 }{ 5 } \times 60 = 72$ 이다.

정답: ①
