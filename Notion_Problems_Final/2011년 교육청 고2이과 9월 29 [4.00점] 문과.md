## 문제
등차수열 $\{ a_{n} \}$, $\{ b_{n} \}$ 에 대하여 $a_{1} = 6$ 이고 
$$\sum_{k=1}^{n} a_{k} = \frac{2 n + 1}{n + 3} \sum_{k=1}^{n} b_{k}$$
를 만족시킬 때, $b_{11}$ 의 값을 구하시오.

## 보기


## 풀이
$\sum_{k=1}^{n} a_{k} = S_{n}$, $\sum_{k=1}^{n} b_{k} = T_{n}$ 이라 하면, 
$$\frac{S_{n}}{T_{n}} = \frac{2 n + 1}{n + 3}$$
수열 $\{ a_{n} \}$, $\{ b_{n} \}$ 이 등차수열이므로
$S_{n} = k n ( 2 n + 1 )$, $T_{n} = k n ( n + 3 )$ ($k$ 는 상수)
$S_{1} = 3 k = a_{1} = 6$ 이므로 $k = 2$
$T_{n} = 2 n ( n + 3 )$, $b_{1} = 8$
수열 $\{ b_{n} \}$ 은 $b_{1} = 8$, 공차가 $4$ 인 등차수열이다. 
$\therefore b_{11} = 48$

정답: $48$
