## 문제
두 자연수 $m$, $n$에 대하여 부등식 
$$\left| \log_{3} \frac{m}{15} \right| + \log_{3} \frac{n}{3} \le 0$$
을 만족시키는 순서쌍 $(m, n)$의 개수를 구하시오.

## 보기

## 풀이
(ⅰ) $\log_{3} \frac{m}{15} > 0 \implies m > 15$ 일 때, 
$$\left| \log_{3} \frac{m}{15} \right| + \log_{3} \frac{n}{3} \le 0 \implies 15 < m \le \frac{45}{n}$$
이며, 
$n=1$ 일 때, $15 < m \le 45$ 이므로 $30$ 개 
$n=2$ 일 때, $15 < m \le 22.5$ 이므로 $7$ 개 
따라서 $37$ 개다. 

(ⅱ) $\log_{3} \frac{m}{15} = 0 \implies m = 15$ 일 때, 
$$\left| \log_{3} \frac{m}{15} \right| + \log_{3} \frac{n}{3} \le 0 \implies \log_{3} \frac{n}{3} \le 0 \implies n \le 3$$
이며, 
$1, 2, 3$ 으로 $3$ 개다. 

(ⅲ) $\log_{3} \frac{m}{15} < 0 \implies 0 < m < 15$ 일 때, 
$$\left| \log_{3} \frac{m}{15} \right| + \log_{3} \frac{n}{3} \le 0 \implies 5n \le m < 15$$
이며, 
$n=1$ 일 때, $5 \le m < 15$ 이므로 $10$ 개 
$n=2$ 일 때, $10 \le m < 15$ 이므로 $5$ 개 
따라서 $15$ 개다. 

총 $37 + 3 + 15 = 55$ 개다.
