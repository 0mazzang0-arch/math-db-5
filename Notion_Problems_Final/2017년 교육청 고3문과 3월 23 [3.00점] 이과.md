## 문제
등차수열 $\{ a_{n} \}$ 에 대하여 $a_{2} = 2$, $a_{5} - a_{3} = 6$ 일 때, $a_{6}$ 의 값을 구하시오.

## 보기

## 풀이
등차수열 $\{ a_{n} \}$ 의 첫째항을 $a$, 공차를 $d$ 라 하면 일반항 $a_{n}$ 은 
$a_{n} = a + ( n - 1 ) d$ 이므로
$a_{5} - a_{3} = ( a + 4 d ) - ( a + 2 d ) = 2 d = 6$
$d = 3$
$a_{2} = a + d = a + 3 = 2$, $a = -1$
따라서 
$a_{6} = a + 5 d = ( -1 ) + 5 \times 3 = 14$

정답: $14$
