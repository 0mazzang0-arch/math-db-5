## 문제
최고차항의 계수가 $k$ ($k>0$)인 이차함수 $f(x)$에 대하여 $f(0)=f(-2)$, $f(0) \neq 0$이다. 함수 $g(x) = (ax+b)e^{f(x)}$ ($a<0$)이 다음 조건을 만족시킨다.

$$ \begin{array}{l} 
\text{(가) 모든 실수 } x \text{에 대하여 } (x+1)\{g(x)-mx-m\} \leq 0 \text{을 만족시키는 실수 } m \text{의 최솟값은 } -2 \text{이다.} \\ 
\text{(나) } \int_{0}^{1} g(x) dx = \int_{-2f(0)}^{1} g(x) dx = \frac{e-e^4}{k} 
\end{array} $$

$f(ab)$의 값을 구하시오. (단, $a, b$는 상수이다.)

## 보기

## 풀이
$f(x) = kx^2 + px + q$ ($p, q$는 상수)라 하면 
$f(0) = f(-2)$이므로 $q = 4k - 2p + q$, $p = 2k$ 
$f(0) \neq 0$이므로 $q \neq 0$ 
따라서 $f(x) = kx^2 + 2kx + q$ ($k>0, q \neq 0$) 

조건 (가)에서 $(x+1)\{g(x) - m(x+1)\} \leq 0$ 
$x \geq -1$일 때, $g(x) \leq m(x+1)$ 
$x < -1$일 때, $g(x) \geq m(x+1)$이고, $g(x)$는 연속함수이므로 
$g(-1) = 0$ 
$(-a+b)e^{f(-1)} = 0$에서 $b = a$ 
$g(x) = (ax+a)e^{kx^2 + 2kx + q}$에서 

$g'(x) = a\{1+2k(x+1)^2\}e^{kx^2 + 2kx + q}$ 
$g''(x) = 2ak(x+1)\{3+2k(x+1)^2\}e^{kx^2 + 2kx + q}$ 

$a<0, k>0$이므로 모든 실수 $x$에 대하여 
$g'(x) < 0$이고, $x < -1$이면 $g''(x) > 0$, $x > -1$이면 $g''(x) < 0$이다. 
조건 (가)에서 $m$의 최솟값이 $-2$이므로 $g'(-1) = -2$ 
$ae^{-k+q} = -2$ $\cdots \cdots$ ㉠ 

조건 (나)의 $\int_{0}^{1} g(x) dx = \frac{e-e^4}{k}$에서 $kx^2 + 2kx + q = t$라 하면 
$\int_{0}^{1} g(x) dx = \int_{q}^{3k+q} \frac{a}{2k} e^t dt$ 
$= \left[ \frac{a}{2k} e^t \right]_{q}^{3k+q}$ 
$= \frac{a}{2k}(e^{3k+q} - e^q)$ 

㉠을 대입하면 
$\frac{-2e^k}{2k}(e^{3k} - 1) = \frac{e-e^4}{k}$, $-e^{4k} + e^k = e - e^4$ 
$e^{4k} - e^4 - e^k + e = 0$ 
$(e^k - e)\{(e^{2k} + e^2)(e^k + e) - 1\} = 0$ 
$(e^{2k} + e^2)(e^k + e) - 1 > 0$이므로 $e^k - e = 0$, 즉 $k=1$ 

조건 (나)에서 
$\int_{-2f(0)}^{1} g(x) dx - \int_{0}^{1} g(x) dx = \int_{-2f(0)}^{0} g(x) dx = 0$ 
$k=1$이므로 $x^2 + 2x + q = t$라 하면 
$\int_{-2f(0)}^{0} g(x) dx = \int_{4q^2-3q}^{q} \frac{a}{2} e^t dt$ 
$= \left[ \frac{a}{2} e^t \right]_{4q^2-3q}^{q}$ 
$= \frac{a}{2}(e^q - e^{4q^2-3q}) = 0$ 

$a \neq 0$에서 $q = 4q^2 - 3q$이고 $q \neq 0$이므로 $q = 1$ 
㉠에 대입하면 $ae^{-1+1} = -2$, $a = -2$ 
따라서 $a = -2, b = -2$, $f(x) = x^2 + 2x + 1$이므로 
$f(ab) = f(4) = 16 + 8 + 1 = 25$
