## 문제
수열 $\{ a_{n} \}$ 에 대하여 $\sum_{k=1}^{10} a_{k} = 4$, $\sum_{k=1}^{10} ( a_{k} + 2 )^{2} = 67$ 일 때, $\sum_{k=1}^{10} ( a_{k} )^{2}$ 의 값은?

## 보기
① $7$   ② $8$   ③ $9$   ④ $10$   ⑤ $11$

## 풀이
$\sum_{k=1}^{10} ( a_{k} + 2 )^{2} = \sum_{k=1}^{10} ( a_{k}^{2} + 4a_{k} + 4 ) = \sum_{k=1}^{10} a_{k}^{2} + 4 \sum_{k=1}^{10} a_{k} + \sum_{k=1}^{10} 4$

$\sum_{k=1}^{10} ( a_{k} + 2 )^{2} = 67$, $\sum_{k=1}^{10} a_{k} = 4$, $\sum_{k=1}^{10} 4 = 40$ 이므로 식에 대입하면

$67 = \sum_{k=1}^{10} a_{k}^{2} + 4 \times 4 + 40$

$67 = \sum_{k=1}^{10} a_{k}^{2} + 16 + 40$

$67 = \sum_{k=1}^{10} a_{k}^{2} + 56$

$\sum_{k=1}^{10} a_{k}^{2} = 11$

따라서 정답은 ⑤이다.
