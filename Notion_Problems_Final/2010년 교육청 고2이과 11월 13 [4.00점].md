## 문제
함수 $f(x) = \frac{2^{[x]} + 2^{[-x]}}{2}$ 에 대하여 <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $[x]$ 는 $x$ 보다 크지 않은 최대의 정수이다.)

<보기>
ㄱ. $f \left( \frac{1}{2} \right) = \frac{3}{4}$
ㄴ. $x$ 가 정수이면 $f(x) \ge 1$ 이다.
ㄷ. 방정식 $f(x) = \frac{9}{8}$ 의 해의 개수는 $2$ 이다.

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
ㄱ. $\left[ \frac{1}{2} \right] = 0$, $\left[ -\frac{1}{2} \right] = -1$ 이므로 
$f \left( \frac{1}{2} \right) = \frac{2^0 + 2^{-1}}{2} = \frac{1 + \frac{1}{2}}{2} = \frac{3}{4}$ (참)

ㄴ. $x$ 가 정수이면 $[x] = x$, $[-x] = -x$ 이다. $2^x > 0$, $2^{-x} > 0$ 이므로 산술·기하 평균의 관계에 의하여
$f(x) = \frac{2^x + 2^{-x}}{2} \ge \sqrt{2^x \cdot 2^{-x}} = 1$ (단, 등호는 $x = 0$ 일 때, 성립한다.) (참)

ㄷ. $1 < x < 2$ 일 때 $[x] = 1, [-x] = -2$ 이고, $-2 < x < -1$ 일 때 $[x] = -2, [-x] = 1$ 이다.
해당 범위의 모든 실수 $x$ 에 대하여
$f(x) = \frac{2^1 + 2^{-2}}{2} = \frac{2 + \frac{1}{4}}{2} = \frac{9}{8}$ 이다. 
따라서 방정식 $f(x) = \frac{9}{8}$ 의 해는 무수히 많다. (거짓)

따라서 옳은 것은 ㄱ, ㄴ이다.
