## 문제
등식 $a b c = 1024$를 만족시키는 세 자연수 $a$, $b$, $c$의 순서쌍 $(a, b, c)$의 개수는?

## 보기
① $42$   ② $48$   ③ $54$   ④ $60$   ⑤ $66$

## 풀이
$a b c = 1024 = 2^{10}$이므로 $a$, $b$, $c$는 모두 $2^k$ ($k = 0, 1, 2, \dots$) 꼴로 나타낼 수 있다.
$a = 2^x$, $b = 2^y$, $c = 2^z$라 할 때, $a b c = 2^{x+y+z} = 2^{10}$이므로 $(a, b, c)$의 개수는 $x + y + z = 10$을 만족시키는 음이 아닌 정수 $x$, $y$, $z$의 순서쌍 $(x, y, z)$의 개수와 같다.
따라서 구하는 순서쌍의 개수는 중복조합을 이용하여 다음과 같이 계산할 수 있다.
${}_{3}H_{10} = {}_{3+10-1}C_{10} = {}_{12}C_{10} = {}_{12}C_{2} = \frac{12 \times 11}{2 \times 1} = 66$

정답: ⑤
