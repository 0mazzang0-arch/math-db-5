## 문제
등차수열 $\{ a_{n} \}$ 에 대하여 $a_{3} + a_{5} = 36$ 이고 $a_{2} a_{4} = 180$ 일 때, $a_{n} < 100$ 을 만족시키는 $n$ 의 최댓값을 구하시오.

## 보기

## 풀이
등차수열 $\{ a_{n} \}$ 의 첫째항을 $a$, 공차를 $d$ 라 하면
$a_{3} + a_{5} = 36$ 에서
$(a + 2d) + (a + 4d) = 36$
$2a + 6d = 36$
$a + 3d = 18 \dots$ ①

$a_{2} a_{4} = 180$ 에서
$(a + d)(a + 3d) = 180$
①의 결과를 대입하면
$(a + d) \times 18 = 180$
$a + d = 10 \dots$ ②

①, ②를 연립하여 풀면
$2d = 8$ 이므로 $d = 4$, $a = 6$

일반항 $a_{n}$ 은
$a_{n} = 6 + (n - 1) \times 4 = 4n + 2$
$a_{n} < 100$ 에서
$4n + 2 < 100$
$4n < 98$
$n < 24.5$

따라서 $n$ 의 최댓값은 $24$ 이다.
정답: $24$
