## 문제
실수 전체의 집합에서 연속인 함수 $f(x)$가 모든 실수 $x$에 대하여 
$$x \int_{0}^{x} f(t)dt - \int_{0}^{x} tf(t)dt = ae^{2x} - 4x + b$$ 
를 만족시킬 때, $f(a)f(b)$의 값을 구하시오. (단, $a$, $b$는 상수이다.)

## 풀이
$$x \int_{0}^{x} f(t)dt - \int_{0}^{x} tf(t)dt = ae^{2x} - 4x + b \dots \text{㉠}$$
㉠의 양변에 $x=0$을 대입하면 
$$0 = a + b \dots \text{㉡}$$
㉠의 양변을 $x$에 대하여 미분하면 
$$\int_{0}^{x} f(t)dt + xf(x) - xf(x) = 2ae^{2x} - 4$$ 
즉, 
$$\int_{0}^{x} f(t)dt = 2ae^{2x} - 4 \dots \text{㉢}$$
㉢의 양변에 $x=0$을 대입하면 
$$0 = 2a - 4$$, 즉 $a = 2$
이므로 ㉡에서 $b = -2$이다. 
$$\int_{0}^{x} f(t)dt = 4e^{2x} - 4$$의 양변을 $x$에 대하여 미분하면 
$f(x) = 8e^{2x}$이므로 
$$f(a)f(b) = f(2)f(-2)$$ 
$$= (8e^{4}) \times (8e^{-4})$$ 
$$= 64e^{4-4}$$ 
$$= 64$$
