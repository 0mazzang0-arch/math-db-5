## 문제
다음은 세 실수 $a$, $b$, $c$에 대하여 $a+b+c>0$, $ab+bc+ca>0$, $abc>0$이면 $a$, $b$, $c$는 모두 양수임을 증명한 것이다.

$$ \begin{array}{|l|} \hline 
abc>0 \text{이므로 } a, b, c \text{ 중 어느 것도 } 0 \text{이 아니다.} \\ 
\text{이제 } a<0 \text{이라고 가정하자.} \\ 
0<abc=a(bc) \text{이므로} \\ 
bc \text{ (가) } 0 \cdots \cdots ㉠ \\ 
\text{한편, 조건에 의하면 } ab+bc+ca>0 \text{이므로} \\ 
-bc<ab+ca \cdots \cdots ㉡ \\ 
㉠, ㉡ \text{ 에 의하여} \\ 
ab+ca \text{ (나) } 0 \\ 
\therefore b+c \text{ (다) } 0 \\ 
\therefore a+b+c<0 \\ 
\text{그런데, 이는 조건 } a+b+c>0 \text{에 모순이다.} \\ 
\therefore a>0 \\ 
\text{마찬가지 방법으로 } b>0, c>0 \text{임을 알 수 있다.} \\ 
\text{따라서 } a, b, c \text{는 모두 양수이다.} \\ \hline
\end{array} $$

위의 증명에서 (가), (나), (다)에 알맞은 것은?

## 보기
① (가) $<$, (나) $>$, (다) $>$   ② (가) $<$, (나) $>$, (다) $<$   ③ (가) $<$, (나) $<$, (다) $>$   ④ (가) $>$, (나) $<$, (다) $>$   ⑤ (가) $>$, (나) $<$, (다) $<$

## 풀이
[정답] ② 

$a<0$ 이라고 가정하자. 
그러면, $0<abc=a(bc)$ 이고 $a<0$ 이므로 
$bc<0 \cdots \cdots$ (가) $ \cdots \cdots$ ㉠ 
한편, 조건에 의하면 $ab+bc+ca>0$ 이므로 
$-bc<ab+ca \cdots \cdots$ ㉡ 
㉠, ㉡ 에 의하여 $ab+ca>0 \cdots \cdots$ (나) 
$ab+ca=a(b+c)$ 이므로 가정 $a<0$ 에 의하여 
$b+c<0 \cdots \cdots$ (다) 
그러므로 옳은 것은 ②이다.
