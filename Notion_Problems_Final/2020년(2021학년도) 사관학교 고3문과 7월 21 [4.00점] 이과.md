## 문제

두 곡선 $y = | 2^x - 4 |$, $y = \log_2 x$ 가 만나는 두 점의 $x$ 좌표를 $x_1$, $x_2$ ($x_1 < x_2$)라 할 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?

$$
\begin{array}{|l|}
\hline
\text{<보 기>} \\
ㄱ. \log_2 3 < x_1 < x_2 < \log_2 6 \\
ㄴ. (x_2 - x_1) ( 2^{x_2} - 2^{x_1} ) < 3 \\
ㄷ. 2^{x_1} + 2^{x_2} > 8 + \log_2 ( \log_3 6 ) \\
\hline
\end{array}
$$



> [!example] 📸 (그림/그래프)

## 보기
① $ㄱ$   ② $ㄱ, ㄴ$   ③ $ㄱ, ㄷ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
[정답] ② 

ㄱ. 그림에서 
$$\log_2 3 < x_1 < x_2 < \log_2 6$$
이 성립한다. 

ㄴ. 그림에서 색칠한 사다리꼴의 넓이에서 
$$\frac{1}{2} ( x_2 - x_1 ) ( 2^{x_2} - 2^{x_1} ) < \frac{3}{2}$$ 
$$(x_2 - x_1) ( 2^{x_2} - 2^{x_1} ) < 3$$
이 성립하므로 $ㄴ$은 옳다.

ㄷ. 그림에서 빨간 직각삼각형의 높이에서 
$$( 2^{x_2} - 4 ) - ( 4 - 2^{x_1} ) < \log_2 ( \log_2 6 ) - \log_2 ( \log_2 3 )$$ 
$$2^{x_1} + 2^{x_2} - 8 < \log_2 \left( \frac{\log_2 6}{\log_2 3} \right)$$
$$2^{x_1} + 2^{x_2} < 8 + \log_2 ( \log_3 6 )$$ 
따라서 $ㄷ$은 옳지 않다.

그러므로 옳은 것은 $ㄱ, ㄴ$이다.
