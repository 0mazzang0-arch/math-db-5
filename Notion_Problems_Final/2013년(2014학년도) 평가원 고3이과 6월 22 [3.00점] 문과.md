## 문제
등차수열 $\{a_n\}$ 에 대하여 $a_3 = 10$, $a_2 + a_5 = 24$ 일 때, $a_6$ 의 값을 구하시오.

## 풀이
등차수열 $\{a_n\}$ 의 첫째항을 $a$, 공차를 $d$ 라 하면
$a_3 = 10$ 에서
$a + 2d = 10$ $\cdots$ ㉠
또, $a_2 + a_5 = 24$ 에서
$(a + d) + (a + 4d) = 24$
$2a + 5d = 24$ $\cdots$ ㉡
㉠, ㉡을 연립하여 풀면
$a = 2$, $d = 4$
따라서 구하는 $a_6$ 의 값은
$a_6 = a + 5d = 2 + 5 \times 4 = 22$
