## 문제
두 상수 $p$ ($p > 0$), $q$에 대하여 이차함수 $f(x) = \frac{1}{2}(x-p)^2 + q$가 있다. 함수 $f(x)$와 양수 $m$에 대하여 함수 $g(x)$를
$$g(x) = -f(x-m)$$
이라 할 때, 방정식 $f(x) = g(x)$가 서로 다른 두 실근 $\alpha$, $\beta$ ($\alpha < \beta$)를 갖는다. 함수 $h(x)$를
$$h(x) = \begin{cases} f(x) & (x \le \alpha \text{ 또는 } x \ge \beta) \\ g(x) & (\alpha < x < \beta) \end{cases}$$
라 할 때, 함수 $h(x)$는 다음 조건을 만족시킨다.

> $x$에 대한 방정식 $h(x) = t$의 서로 다른 실근의 개수가 $2$ 이상이면서 서로 다른 모든 실근의 합이 $4p+2m$이 되도록 하는 모든 실수 $t$의 값의 범위는 $g(p) < t < 5$이다.

$f(m) + g(m) = -4$일 때, $m \times (p-q)$의 값을 구하시오.

## 풀이
$q \ge 0$이면 모든 실수 $x$에 대하여 $f(x) \ge 0$, $g(x) \le 0$이 되어 방정식 $f(x) = g(x)$는 서로 다른 두 실근을 갖지 않으므로 $q < 0$이다.
방정식 $f(x) = g(x)$가 서로 다른 두 실근 $\alpha$, $\beta$를 가지므로 두 함수 $y = f(x)$, $y = g(x)$의 그래프는 서로 다른 두 점 $P(\alpha, f(\alpha))$, $Q(\beta, f(\beta))$에서 만난다.

(ⅰ) $\alpha \ge p$일 때 

> [!example] 📸 (그림/그래프)

$t > q$인 모든 실수 $t$에 대하여 방정식 $h(x) = t$의 서로 다른 실근의 개수는 $2$이다. 
$t > q$인 실수 $t$에 대하여 $y = h(x)$의 그래프와 직선 $y = t$가 만나는 두 교점의 $x$좌표를 $x_{1}$, $x_{2}$, $y = f(x)$의 그래프와 직선 $y = t$가 만나는 두 교점의 $x$좌표를 $x_{3}$, $x_{4}$라 하자. 
$t > q$인 모든 실수 $t$에 대하여 함수 $y = f(x)$의 그래프와 직선 $y = t$가 만나는 두 교점의 $x$좌표의 합은 $2p$이므로 $h(x) = t$의 서로 다른 두 실근의 합 $x_{1} + x_{2}$는 
$$x_{1} + x_{2} \le x_{3} + x_{4} = 2p$$
$p > 0$, $m > 0$이므로 $x_{1} + x_{2} < 4p + 2m$이 되어 조건을 만족시키지 않는다. 

(ⅱ) $\alpha < p$일 때 

> [!example] 📸 (그림/그래프)

$h(\alpha) < t < h(\beta)$ 또는 $t > -q$인 모든 실수 $t$에 대하여 방정식 $h(x) = t$의 서로 다른 실근의 개수는 $2$이고, $t = h(\beta)$ 또는 $t = -q$인 실수 $t$에 대하여 방정식 $h(x) = t$의 서로 다른 실근의 개수는 $3$이고, $h(\beta) < t < -q$인 모든 실수 $t$에 대하여 방정식 $h(x) = t$의 서로 다른 실근의 개수는 $4$이다. 

(a) $h(\alpha) < t < h(\beta)$ 또는 $t > -q$일 때 
$h(x) = t$의 서로 다른 두 실근의 합은 $2p$보다 작거나 같으므로 조건을 만족시키지 않는다. 

(b) $t = h(\beta)$ 또는 $t = -q$일 때 
$h(x) = t$의 서로 다른 세 실근의 합은 $3p + m$보다 작거나 같으므로 조건을 만족시키지 않는다. 

(c) $h(\beta) < t < -q$일 때 
$y = h(x)$의 그래프와 직선 $y = t$가 만나는 네 교점의 $x$좌표를 작은 수부터 차례로 $x_{5}$, $x_{6}$, $x_{7}$, $x_{8}$이라 하면 함수 $y = f(x)$의 그래프와 직선 $y = t$가 만나는 두 교점의 $x$좌표는 $x_{5}$, $x_{8}$이고 함수 $y = g(x)$의 그래프와 직선 $y = t$가 만나는 두 교점의 $x$좌표는 $x_{6}$, $x_{7}$이다. 
$x_{5} + x_{8} = 2p$, $x_{6} + x_{7} = 2(p+m) = 2p + 2m$이므로 방정식 $h(x) = t$의 서로 다른 네 실근의 합 $x_{5} + x_{6} + x_{7} + x_{8}$은 
$$x_{5} + x_{6} + x_{7} + x_{8} = 4p + 2m$$
이때 $t$의 값의 범위는 $h(\beta) < t < -q$이다. 
조건에서 범위가 $g(p) < t < 5$이므로 $-q = 5$, 즉 $q = -5$이다. 

> [!example] 📸 (그림/그래프)

$h(\beta) = g(p)$이고 $h(\beta) = f(\beta) = g(\beta)$이므로 $g(p) = f(\beta)$이다. 
$y = g(x)$의 그래프는 $x = p + m$에 대하여 대칭이므로 
$$p + \beta = 2(p + m) \Rightarrow \beta = p + 2m$$
$f(\beta) = f(p + 2m) = \frac{1}{2}(2m)^{2} - 5 = 2m^{2} - 5$이고 
$g(p) = -\frac{1}{2}(p - (p + m))^{2} + 5 = -\frac{1}{2}m^{2} + 5$
$2m^{2} - 5 = -\frac{1}{2}m^{2} + 5$에서 $\frac{5}{2}m^{2} = 10$, $m^{2} = 4$이므로 $m = 2$이다. 
$f(x) = \frac{1}{2}(x - p)^{2} - 5$, $g(x) = -\frac{1}{2}(x - 2 - p)^{2} + 5$
$f(m) + g(m) = f(2) + g(2) = \{ \frac{1}{2}(2 - p)^{2} - 5 \} + \{ -\frac{1}{2}(2 - 2 - p)^{2} + 5 \}$
$= \frac{1}{2}(4 - 4p + p^{2}) - 5 - \frac{1}{2}p^{2} + 5 = -2p + 2$
$-2p + 2 = -4$이므로 $p = 3$이다. 

(ⅰ), (ⅱ)에 의하여 
$$m \times (p - q) = 2 \times \{ 3 - (-5) \} = 16$$
