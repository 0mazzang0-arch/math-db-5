## 문제
표는 $0$ 이 아닌 실수 $a$ 와 $2$ 이상의 자연수 $n$ 에 대한 두 종류의 변환장치를 설명한 것이다.

$$
\begin{array}{|c|l|}
\hline
\text{변환장치} & \multicolumn{1}{c|}{\text{설 명}} \\
\hline
\text{장치 1} & \begin{array}{l} \text{(ⅰ) } n \text{ 이 홀수인 경우 } k = ( a \text{ 의 } n \text{ 제곱근 중 실수} ) \\ \text{(ⅱ) } n \text{ 이 짝수인 경우 } k = ( a \text{ 의 } n \text{ 제곱근 중 양의 실수} ) \end{array} \\
\hline
\text{장치 2} & m = ( a \text{ 의 } n \text{ 제곱근 중 실수의 개수} ) \\
\hline
\end{array}
$$

그림과 같이 연결된 변환장치로 네 수를 변환했을 때, 세 수 $x$, $y$, $z$ 에 대하여 $x^{2}+y^{2}+z^{2}$ 의 값을 구하시오.



> [!example] 📸 (그림/그래프)

## 풀이
정답: $14$

해설: 
$\therefore x^{2}+y^{2}+z^{2}=(-2)^{2}+3^{2}+1^{2}=14$
