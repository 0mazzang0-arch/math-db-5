## 문제
$10 \le x < 1000$ 인 실수 $x$ 에 대하여 $\log x^{3} - \log \frac{1}{x^{2}}$ 의 값이 자연수가 되도록 하는 모든 $x$ 의 개수를 구하시오.

## 보기

## 풀이
$\log x^{3} - \log \frac{1}{x^{2}} = 3 \log x - ( -2 \log x ) = 5 \log x$

$10 \le x < 1000$ 에서 
$1 \le \log x < 3$ 이므로
$5 \le 5 \log x < 15$

따라서 $5 \log x$ 의 값이 자연수가 되도록 하는 $x$ 의 개수는 $10$ 이다.

**[정답]** 
$10$

**[보충설명]** 
$5 \log x$ 의 값이 자연수가 되도록 하는 $x$ 의 값을 구하면 
$x = 10, 10^{\frac{6}{5}}, 10^{\frac{7}{5}}, 10^{\frac{8}{5}}, \dots, 10^{\frac{14}{5}}$
