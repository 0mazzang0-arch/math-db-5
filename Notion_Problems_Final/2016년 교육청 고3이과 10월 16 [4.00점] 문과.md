## 문제
함수 $f(x) = \frac{3^x}{3^x + 3}$에 대하여 점 $(p, q)$가 곡선 $y = f(x)$ 위의 점이면 실수 $p$의 값에 관계없이 점 $(2a - p, a - q)$도 항상 곡선 $y = f(x)$ 위의 점이다. 다음은 상수 $a$의 값을 구하는 과정이다.

점 $(2a - p, a - q)$가 곡선 $y = f(x)$ 위의 점이므로
$\frac{3^{2a-p}}{3^{2a-p} + 3} = a - (가) \cdots \cdots$ ㉠
이다. ㉠은 실수 $p$의 값에 관계없이 항상 성립하므로
$p = 0$일 때, $\frac{3^{2a}}{3^{2a} + 3} = a - \frac{1}{4} \cdots \cdots$ ㉡
이고, $p = 1$일 때, $\frac{3^{2a}}{3^{2a} + (나)} = a - \frac{1}{2} \cdots \cdots$ ㉢
이다. ㉡, ㉢에서
$(3^{2a} + 3)(3^{2a} + (나)) = 24 \times 3^{2a}$ 이므로
$a = \frac{1}{2}$ 또는 $a = (다)$
이다. 이때, ㉢에서 좌변이 양수이므로 $a > \frac{1}{2}$이다.
따라서 $a = (다)$이다.

위의 (가)에 알맞은 식을 $g(p)$라 하고 (나)와 (다)에 알맞은 수를 각각 $m$, $n$이라 할 때, $(m - n) \times g(2)$의 값은?

## 보기
① $4$   ② $\frac{9}{2}$   ③ $5$   ④ $\frac{11}{2}$   ⑤ $6$

## 풀이
점 $(2a - p, a - q)$가 곡선 $y = f(x)$ 위의 점이므로
$f(p) = q$, $f(2a - p) = a - q$에서
$\frac{3^{2a-p}}{3^{2a-p} + 3} = a - q$
$= a - \frac{3^p}{3^p + 3} \cdots \cdots$ ㉠
이다. ㉠은 실수 $p$의 값에 관계없이 항상 성립하므로
$p = 0$일 때,
$\frac{3^{2a}}{3^{2a} + 3} = a - \frac{1}{4} \cdots \cdots$ ㉡
이고, $p = 1$일 때,
$\frac{3^{2a-1}}{3^{2a-1} + 3} = a - \frac{3}{3 + 3} = a - \frac{1}{2}$
좌변의 분자, 분모에 $3$을 곱하면
$\frac{3^{2a}}{3^{2a} + 9} = a - \frac{1}{2} \cdots \cdots$ ㉢
이다. ㉡, ㉢에서 ㉡ $-$ ㉢을 정리하면
$\frac{3^{2a}}{3^{2a} + 3} - \frac{3^{2a}}{3^{2a} + 9} = (a - \frac{1}{4}) - (a - \frac{1}{2}) = \frac{1}{4}$
$\frac{3^{2a}(3^{2a} + 9) - 3^{2a}(3^{2a} + 3)}{(3^{2a} + 3)(3^{2a} + 9)} = \frac{6 \times 3^{2a}}{(3^{2a} + 3)(3^{2a} + 9)} = \frac{1}{4}$
$(3^{2a} + 3)(3^{2a} + 9) = 24 \times 3^{2a}$
이므로 $3^{2a} = X$ ($X > 0$)이라 하면
$(X + 3)(X + 9) = 24X$
$X^2 - 12X + 27 = 0$
$(X - 3)(X - 9) = 0$
$X = 3$ 또는 $X = 9$
$3^{2a} = 3$ 또는 $3^{2a} = 9$
$a = \frac{1}{2}$ 또는 $a = 1$
이다. 이때, ㉢에서 좌변이 양수이므로 $a > \frac{1}{2}$이다.
따라서 $a = 1$이다.
그러므로
$g(p) = \frac{3^p}{3^p + 3}$, $m = 9$, $n = 1$
따라서
$(m - n) \times g(2) = (9 - 1) \times \frac{3^2}{3^2 + 3}$
$= 8 \times \frac{9}{12} = 8 \times \frac{3}{4} = 6$
정답은 ⑤이다.
