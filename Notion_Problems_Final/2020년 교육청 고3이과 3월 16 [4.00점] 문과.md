## 문제
함수 $f(x)$ 가 모든 실수 $x$ 에 대하여 
$$ f(x)=x^{3}-4 x \int_{0}^{1}|f(t)| d t $$
를 만족시킨다. $f(1)>0$ 일 때, $f(2)$ 의 값은? 

## 보기
① $6$   ② $7$   ③ $8$   ④ $9$   ⑤ $10$

## 풀이
$a = \int_{0}^{1}|f(t)| d t$ 라 하면 $a > 0$ 이고 
$f(x)=x^{3}-4 a x$ 이다.
$f(1)=1-4 a>0$ 에서 $a < \frac{1}{4}$ 이므로 
$0 < a < \frac{1}{4}$ 이다. 

$f(x)=x (x^{2}-4 a )=0$ 에서 
$x=0$ 또는 $x = \pm 2 \sqrt{a}$ 이다.

$0 < x < 2 \sqrt{a}$ 일 때 $f(x) < 0$ 이고 
$x \ge 2 \sqrt{a}$ 일 때 $f(x) \ge 0$ 이다. 

$0 < a < \frac{1}{4}$ 에서 $2 \sqrt{a} < 1$ 이므로 
$a = \int_{0}^{2 \sqrt{a}} \{ -f(t) \} dt + \int_{2 \sqrt{a}}^{1} f(t) dt$ 
$a = \int_{0}^{2 \sqrt{a}} (-t^{3}+4 a t ) d t+ \int_{2 \sqrt{a}}^{1} (t^{3}-4 a t ) d t$ 
$a = \left[ -\frac{1}{4} t^{4}+2 a t^{2} \right]_{0}^{2 \sqrt{a}} + \left[ \frac{1}{4} t^{4}-2 a t^{2} \right]_{2 \sqrt{a}}^{1}$ 
$a = 8 a^{2}-2 a + \frac{1}{4}$ 

$8 a^{2}-3 a + \frac{1}{4}=0$ 에서 
$32 a^{2}-12 a+1=0$, 
$(4 a-1)(8 a-1)=0$ 

$0 < a < \frac{1}{4}$ 이므로 $a = \frac{1}{8}$ 이고 
$f(x)=x^{3}-\frac{1}{2} x$ 이다.
이때, $f(2)=2^{3}-\frac{1}{2} \times 2 = 7$ 이다.

정답: ②
