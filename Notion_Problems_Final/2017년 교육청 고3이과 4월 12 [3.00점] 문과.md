## 문제
집합 $X = \{ 1, 2 \}$에서 집합 $Y = \{ 1, 2, 3, 4, 5, 6 \}$으로의 함수 $f$ 중에서 $f ( 1 ) + f ( 2 )$가 $4$의 배수가 되도록 하는 함수 $f$의 개수는?

## 보기
① $8$   ② $9$   ③ $10$   ④ $11$   ⑤ $12$

## 풀이
$f ( 1 ) = a$, $f ( 2 ) = b$라 하자.
조건을 만족시키는 순서쌍 $( a, b )$는
ⅰ) $a + b = 4$일 때
$( 1, 3 )$, $( 2, 2 )$, $( 3, 1 )$
$\therefore 3$가지
ⅱ) $a + b = 8$일 때
$( 2, 6 )$, $( 3, 5 )$, $( 4, 4 )$, $( 5, 3 )$, $( 6, 2 )$
$\therefore 5$가지
ⅲ) $a + b = 12$일 때
$( 6, 6 )$
$\therefore 1$가지
ⅰ), ⅱ), ⅲ)에 의하여 함수 $f$의 개수는 $3 + 5 + 1 = 9$이다.

정답: ②
