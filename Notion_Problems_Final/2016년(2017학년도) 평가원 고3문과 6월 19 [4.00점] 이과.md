## 문제
한 개의 주사위를 두 번 던질 때 나오는 눈의 수를 차례로 $a$, $b$라 하자. 다음은 이차함수 $f(x) = x^2 - 7x + 12$에 대하여 $f(a)f(b) = 0$이 성립할 확률을 구하는 과정이다.

$$ \begin{array}{|l|} \hline \text{첫 번째 던져서 나오는 주사위의 눈의 수를 } a \text{라 할 때 } f(a) = 0 \text{이 되는 사건을 } A \text{라 하고,} \\ \text{두 번째 던져서 나오는 주사위의 눈의 수를 } b \text{라 할 때 } f(b) = 0 \text{이 되는 사건을 } B \text{라 하자.} \\ \text{이차방정식 } f(x) = 0 \text{의 해는 } x = 3 \text{ 또는 } x = 4 \text{이므로} \\ P(A) = \text{(가)}, P(B) = \text{(가)} \text{ 이다.} \\ \text{구하는 확률 } P(A \cup B) \text{는} \\ P(A \cup B) = P(A) + P(B) - P(A \cap B) \\ \text{이고, 두 사건 } A \text{와 } B \text{는 서로 독립이므로} \\ P(A \cap B) = \text{(나)} \text{ 이다.} \\ \text{그러므로 } P(A \cup B) = \text{(다)} \text{ 이다.} \\ \hline \end{array} $$

위의 (가), (나), (다)에 알맞은 수를 각각 $m$, $n$, $k$라 할 때, $m \times n \times k$의 값은?

## 보기
① $\frac{1}{81}$   ② $\frac{5}{243}$   ③ $\frac{7}{243}$   ④ $\frac{1}{27}$   ⑤ $\frac{11}{243}$

## 풀이
첫 번째 던져서 나오는 주사위의 눈의 수를 $a$라 할 때 $f(a)=0$이 되는 사건을 $A$라 하고, 두 번째 던져서 나오는 주사위의 눈의 수를 $b$라 할 때 $f(b)=0$이 되는 사건을 $B$라 하자.
이차방정식 $f(x)=0$의 해는 $x=3$ 또는 $x=4$이므로
$P(A) = \frac{2}{6} = \frac{1}{3}$, $P(B) = \frac{2}{6} = \frac{1}{3}$이다.
구하는 확률 $P(A \cup B)$는
$P(A \cup B) = P(A) + P(B) - P(A \cap B)$
이고, 두 사건 $A$와 $B$는 서로 독립이므로
$P(A \cap B) = P(A)P(B) = \frac{1}{3} \times \frac{1}{3} = \frac{1}{9}$이다.
그러므로
$P(A \cup B) = \frac{1}{3} + \frac{1}{3} - \frac{1}{9} = \frac{5}{9}$이다.
따라서 (가), (나), (다)에 알맞은 수는 각각 $m = \frac{1}{3}$, $n = \frac{1}{9}$, $k = \frac{5}{9}$이므로
$m \times n \times k = \frac{1}{3} \times \frac{1}{9} \times \frac{5}{9} = \frac{5}{243}$이다.

정답: ②
