## 문제
다음은 두 홀수 $a, b$에 대하여 두 수 $A, B$를 $A = \frac{ab - 1}{2}, B = \frac{a - 1}{2} + \frac{b - 1}{2}$라 할 때, $(나)$임을 증명한 것이다.

$a = 2m + 1, b = 2n + 1$ ($m, n$은 정수)라 하면 

$$ A = \frac{ab - 1}{2} = (가) + m + n $$
$$ B = \frac{a - 1}{2} + \frac{b - 1}{2} = m + n $$

$\therefore A - B = (가)$

따라서, $(나)$이다.

이 때, $(나)$에 알맞은 것은?

## 보기
① $A, B$는 모두 홀수   ② $A, B$는 모두 짝수   ③ $A, B$는 모두 짝수이거나 모두 홀수   ④ $A, B$ 중 하나는 짝수이고 다른 하나는 홀수   ⑤ $A, B$는 서로 다른 정수

## 풀이
[정답] ③ 

[해설] 
$a = 2m + 1, b = 2n + 1$ ($m, n$은 정수)라 하면 
$$ A = \frac{ab - 1}{2} = \frac{(2m + 1)(2n + 1) - 1}{2} = \frac{4mn + 2m + 2n}{2} = 2mn + m + n \cdots (가) $$
$$ B = \frac{a - 1}{2} + \frac{b - 1}{2} = m + n $$
$\therefore A - B = 2mn \cdots (가)$

이때 $A - B = 2mn$은 항상 짝수이므로, $A$와 $B$의 차가 짝수라는 것은 두 수의 기우성(홀짝 성질)이 같음을 의미한다. 
따라서, $A, B$는 모두 짝수이거나 모두 홀수이다. $\cdots (나)$
