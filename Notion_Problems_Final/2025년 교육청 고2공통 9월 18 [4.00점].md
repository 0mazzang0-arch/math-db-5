## 문제
두 양수 $a$, $b$ 에 대하여 함수 $f(x)=a \cos (bx-\frac{\pi}{4})$ 가 $0 \le x \le \pi$ 에서 최댓값 $4$, 최솟값 $-2 \sqrt{2}$ 를 가질 때, $a+b$ 의 값은?

## 보기
① $\frac{7}{2}$   ② $4$   ③ $\frac{9}{2}$   ④ $5$   ⑤ $\frac{11}{2}$

## 풀이
$a \cos (bx-\frac{\pi}{4})=a \cos b (x-\frac{\pi}{4b})$ 이므로 함수 $f(x)=a \cos (bx-\frac{\pi}{4})$ 의 그래프는 함수 $y=a \cos bx$ 의 그래프를 $x$ 축의 방향으로 $\frac{\pi}{4b}$ 만큼 평행이동한 그래프와 일치한다. 
함수 $y=a \cos bx$ 의 주기는 $\frac{2 \pi}{b}$ 이므로 함수 $f(x)$ 의 그래프의 개형은 다음과 같다. 

> [!example] 📸 (그림/그래프)

$0 \le x \le \pi$ 에서 함수 $f(x)$ 의 최솟값이 음수이므로 
$\frac{3}{4b} \pi < \pi \dots \dots$ ㉠ 
함수 $f(x)$ 의 최댓값이 $4$ 이므로 $a=4$ 

$\frac{5}{4b} \pi \le \pi$ 일 때, 함수 $f(x)$ 의 최솟값이 $-4$ $(\ne -2 \sqrt{2})$ 이므로 
$\pi < \frac{5}{4b} \pi \dots \dots$ ㉡ 

㉠, ㉡에 의하여 $0 \le x \le \pi$ 에서 함수 $f(x)$ 가 $x=\pi$ 일 때, 최솟값 $-2 \sqrt{2}$ 를 갖는다. 
$\frac{3}{4b} \pi < \pi < \frac{5}{4b} \pi$ 이므로 $\frac{\pi}{2} < b \pi - \frac{\pi}{4} < \pi$ 

$f(\pi)=4 \cos (b \pi - \frac{\pi}{4})=-2 \sqrt{2}$ 
$\cos (b \pi - \frac{\pi}{4})=-\frac{\sqrt{2}}{2}$ 
$b \pi - \frac{\pi}{4}=\frac{3}{4} \pi$, $b=1$ 

따라서 $a+b=4+1=5$
