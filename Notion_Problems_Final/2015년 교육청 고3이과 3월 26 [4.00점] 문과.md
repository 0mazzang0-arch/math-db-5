## 문제
양의 실수 $x$ 에 대하여 $\log x$ 의 소수부분을 $f(x)$ 라 할 때, $\sum_{k=1}^{10} f(2^k) = m \log 2 - n$ 이다. 두 자연수 $m$, $n$ 에 대하여 $m+n$ 의 값을 구하시오.

## 풀이
ⅰ) $1 \le k \le 3$ 일 때, $1 \le 2^k < 10$ 이므로 $\log 2^k$ 의 정수부분은 $0$ 이고, 
$f(2^k) = \log 2^k = k \log 2$ 

ⅱ) $4 \le k \le 6$ 일 때, $10 \le 2^k < 10^2$ 이므로 $\log 2^k$ 의 정수부분은 $1$ 이고, 
$f(2^k) = \log 2^k - 1 = k \log 2 - 1$ 

ⅲ) $7 \le k \le 9$ 일 때, $10^2 \le 2^k < 10^3$ 이므로 $\log 2^k$ 의 정수부분은 $2$ 이고, 
$f(2^k) = \log 2^k - 2 = k \log 2 - 2$ 

ⅳ) $k = 10$ 일 때, $10^3 \le 2^k < 10^4$ 이므로 $\log 2^k$ 의 정수부분은 $3$ 이고, 
$f(2^k) = \log 2^k - 3 = k \log 2 - 3$ 

따라서 
$\sum_{k=1}^{10} f(2^k)$ 
$= \sum_{k=1}^{10} k \log 2 - (1 \times 3 + 2 \times 3 + 3 \times 1)$ 
$= 55 \log 2 - 12$ 

이므로 $m = 55$, $n = 12$ 
따라서 $m + n = 67$
