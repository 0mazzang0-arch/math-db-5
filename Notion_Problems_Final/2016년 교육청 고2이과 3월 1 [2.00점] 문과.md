## 문제
두 다항식 $A = 3x^{2} - x + 5$, $B = x^{2} - 2x - 3$에 대하여 $A - B$를 간단히 하면?

## 보기
① $x^{2} - 3x + 8$   ② $x^{2} - x + 2$   ③ $2x^{2} - 3x + 2$   ④ $2x^{2} - x + 2$   ⑤ $2x^{2} + x + 8$

## 풀이
두 다항식 $A = 3x^{2} - x + 5$, $B = x^{2} - 2x - 3$에 대하여 

$A - B$
$= ( 3x^{2} - x + 5 ) - ( x^{2} - 2x - 3 )$
$= 3x^{2} - x + 5 - x^{2} + 2x + 3$
$= 2x^{2} + x + 8$

정답은 ⑤이다.
