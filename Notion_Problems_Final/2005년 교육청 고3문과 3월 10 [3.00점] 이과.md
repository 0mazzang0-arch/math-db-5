## 문제
두 양수 $A$, $\frac{1}{A}$의 상용로그에서 정수부분의 합은 $a$이고, 소수부분의 합은 $b$이다. 이때, $a^{2} + b^{2}$의 값은? (단, $\log A$의 소수부분은 $0$이 아니다.)

## 보기
① $\frac{1}{2}$   ② $1$   ③ $\frac{3}{2}$   ④ $2$   ⑤ $\frac{5}{2}$

## 풀이
$\log A = n + \alpha$ ($n$은 정수, $0 < \alpha < 1$)라 하면
$\log \frac{1}{A} = -\log A = -n - \alpha = (-n - 1) + (1 - \alpha)$
이때 $0 < \alpha < 1$이므로 $0 < 1 - \alpha < 1$이다.
따라서 $\log A$의 정수부분은 $n$, 소수부분은 $\alpha$이고
$\log \frac{1}{A}$의 정수부분은 $-n - 1$, 소수부분은 $1 - \alpha$이다.
정수부분의 합 $a = n + (-n - 1) = -1$
소수부분의 합 $b = \alpha + (1 - \alpha) = 1$
$\therefore a^{2} + b^{2} = (-1)^{2} + 1^{2} = 1 + 1 = 2$

정답은 ④이다.
