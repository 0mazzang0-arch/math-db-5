## 문제
등차수열 $\{ a _{n} \}$ 에 대하여 $a _{2} = 6, a _{4} + a _{6} = 36$ 일 때, $a _{10}$ 의 값은?

## 보기
① $30$   ② $32$   ③ $34$   ④ $36$   ⑤ $38$

## 풀이
등차수열 $\{ a _{n} \}$ 의 공차를 $d$ 라 하면 
$a _{2} = a _{1} + d = 6$ $\cdots$ ㉠ 

$a _{4} + a _{6} = 36$ 에서 
$( a _{1} + 3d ) + ( a _{1} + 5d ) = 36$ 
$2a _{1} + 8d = 36$ 
$a _{1} + 4d = 18$ $\cdots$ ㉡ 

㉠, ㉡에서 
$a _{1} = 2, d = 4$ 

따라서 
$a _{10} = 2 + 9 \times 4 = 38$ 

[정답] ⑤
