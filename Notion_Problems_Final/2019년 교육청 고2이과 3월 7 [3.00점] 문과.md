## 문제
이차부등식 $x^{2}-8x+a \le 0$ 의 해가 $b \le x \le 6$ 일 때, $a+b$ 의 값은? (단, $a, b$ 는 상수이다.)

## 보기
① $14$   ② $15$   ③ $16$   ④ $17$   ⑤ $18$

## 풀이
이차부등식 $x^{2}-8x+a \le 0$ 의 해가 $b \le x \le 6$ 이므로
$x^{2}-8x+a = (x-b)(x-6)$
$= x^{2}-(b+6)x+6b$
$8 = b+6, a = 6b$
따라서 $b = 2, a = 12$
따라서 $a+b = 12+2 = 14$

[다른 풀이]
$x = 6$ 일 때, $x^{2}-8x+a = 0$ 이므로
$36-48+a = 0$
$\therefore a = 12$
$x^{2}-8x+12 \le 0$
$(x-2)(x-6) \le 0$
$2 \le x \le 6$
$\therefore b = 2$
따라서 $a+b = 12+2 = 14$
