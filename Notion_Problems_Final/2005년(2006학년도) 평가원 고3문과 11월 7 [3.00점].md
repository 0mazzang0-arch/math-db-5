## 문제
수열 $\{ a_{n} \}$ 이 모든 자연수 $n$ 에 대하여 $n < a_{n} < n+1$ 을 만족시킬 때, $\lim_{n \to \infty} \frac{n^{2}}{a_{1} + a_{2} + \dots + a_{n}}$ 의 값은?

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
$n < a_{n} < n+1$ 에서 
$\sum_{k=1}^{n} k < \sum_{k=1}^{n} a_{k} < \sum_{k=1}^{n} ( k+1 )$ 

$\therefore \frac{n(n+1)}{2} < \sum_{k=1}^{n} a_{k} < \frac{(n+1)(n+2)}{2} - 1$ 

$\therefore \frac{2n^{2}}{n^{2} + 3n} < \frac{n^{2}}{a_{1} + a_{2} + \dots + a_{n}} < \frac{2n^{2}}{n(n+1)}$ 

$\lim_{n \to \infty} \frac{2n^{2}}{n^{2} + 3n} = \lim_{n \to \infty} \frac{2n^{2}}{n^{2} + n} = 2$ 이므로 

$\lim_{n \to \infty} \frac{n^{2}}{a_{1} + a_{2} + \dots + a_{n}} = 2$

정답 ②
