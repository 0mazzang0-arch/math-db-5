## 문제
$0$이 아닌 세 실수 $a$, $b$, $c$가 다음 조건을 만족시킨다.

(가) $b + c < a$
(나) $\frac{\sqrt{b}}{\sqrt{a}} = -\sqrt{\frac{b}{a}}$

세 수 $a$, $b$, $c$의 대소 관계로 옳은 것은?

## 보기
① $a < c < b$   ② $b < a < c$   ③ $b < c < a$   ④ $c < a < b$   ⑤ $c < b < a$

## 풀이
조건 (가)에서
$b + c < a$ $\cdots \cdots$ ㉠

조건 (나)에서
$\frac{\sqrt{b}}{\sqrt{a}} = -\sqrt{\frac{b}{a}}$

$\therefore a < 0$, $b > 0$, 즉 $a < b$ $\cdots \cdots$ ㉡

㉠에서 $b > 0$이므로
$c < b + c < a$ $\cdots \cdots$ ㉢

㉡, ㉢에 의하여
$c < a < b$

정답: ④
