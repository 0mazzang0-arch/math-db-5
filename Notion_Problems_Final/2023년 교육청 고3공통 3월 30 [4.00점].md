## 문제
집합 $X = \{ 1, 2, 3, 4, 5 \}$에 대하여 다음 조건을 만족시키는 함수 $f : X \to X$의 개수를 구하시오.

(가) 집합 $X$의 임의의 두 원소 $x_{1}$, $x_{2}$에 대하여 $x_{1} < x_{2}$이면 $f(x_{1}) \leq f(x_{2})$이다.
(나) $f(2) \neq 1$이고 $f(4) \times f(5) < 20$이다.

## 풀이
조건 (가)를 만족시키는 함수 $f$의 개수는
${}_{5}\mathrm{H}_{5} = {}_{5+5-1}\mathrm{C}_{5} = {}_{9}\mathrm{C}_{5} = 126$
조건 (나)의 부정은
$f(2) = 1$ 또는 $f(4) \times f(5) \geq 20$ $\cdots \cdots$ ㉠
이다.

(ⅰ) $f(2) = 1$인 경우
$f(1) = 1$이고 $1 \leq f(3) \leq f(4) \leq f(5) \leq 5$이므로
$f(3)$, $f(4)$, $f(5)$의 값을 정하는 경우의 수는
${}_{5}\mathrm{H}_{3} = {}_{5+3-1}\mathrm{C}_{3} = {}_{7}\mathrm{C}_{3} = 35$

(ⅱ) $f(4) \times f(5) \geq 20$인 경우
$f(4) = 4$, $f(5) = 5$일 때
$1 \leq f(1) \leq f(2) \leq f(3) \leq 4$이므로
$f(1)$, $f(2)$, $f(3)$의 값을 정하는 경우의 수는
${}_{4}\mathrm{H}_{3} = {}_{4+3-1}\mathrm{C}_{3} = {}_{6}\mathrm{C}_{3} = 20$

$f(4) = 5$, $f(5) = 5$일 때
$1 \leq f(1) \leq f(2) \leq f(3) \leq 5$이므로
$f(1)$, $f(2)$, $f(3)$의 값을 정하는 경우의 수는
${}_{5}\mathrm{H}_{3} = {}_{5+3-1}\mathrm{C}_{3} = {}_{7}\mathrm{C}_{3} = 35$
그러므로 이 경우 구하는 함수 $f$의 개수는
$20 + 35 = 55$

(ⅲ) $f(2) = 1$이고 $f(4) \times f(5) \geq 20$인 경우
$f(1) = 1$이고 $f(4) = 4$, $f(5) = 5$일 때
$1 \leq f(3) \leq 4$에서 $f(3)$의 값을 정하는 경우의 수는
${}_{4}\mathrm{C}_{1} = 4$

$f(1) = 1$이고 $f(4) = 5$, $f(5) = 5$일 때
$1 \leq f(3) \leq 5$에서 $f(3)$의 값을 정하는 경우의 수는
${}_{5}\mathrm{C}_{1} = 5$
그러므로 이 경우 구하는 함수 $f$의 개수는
$4 + 5 = 9$

(ⅰ), (ⅱ), (ⅲ)에 의하여 ㉠을 만족시키는 함수 $f$의 개수는
$35 + 55 - 9 = 81$

따라서 구하는 함수 $f$의 개수는
$126 - 81 = 45$

정답: $45$
