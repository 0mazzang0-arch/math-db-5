## 문제
함수 $f(x) = | 2a \cos \frac{b}{2} x - (a-2)(b-2) |$ 가 다음 조건을 만족시키도록 하는 $10$ 이하의 자연수 $a, b$ 의 모든 순서쌍 $(a, b)$ 의 개수는?

(가) 함수 $f(x)$ 는 주기가 $\pi$ 인 주기함수이다.
(나) $0 \le x \le 2\pi$ 에서 함수 $y=f(x)$ 의 그래프와 직선 $y=2a-1$ 의 교점의 개수는 $4$ 이다.

## 보기
① $11$   ② $13$   ③ $15$   ④ $17$   ⑤ $19$

## 풀이
(ⅰ) $(a-2)(b-2)=0$ 일 때,
$f(x)= | 2a \cos \frac{b}{2}x|$ 의 주기는 $\frac{\pi}{\frac{b}{2}} = \frac{2\pi}{b}$ 이므로 조건 (가)에서
$\frac{2\pi}{b} = \pi \implies b=2$

$| 2a \cos x| = 2a-1$ 에서
$2a \cos x = 2a-1$ 또는 $2a \cos x = -2a+1$
$\cos x = 1-\frac{1}{2a}$ 또는 $\cos x = -1+\frac{1}{2a}$

> [!example] 📸 (그림/그래프)

$a$ 에 대하여 $0 < 1-\frac{1}{2a} < 1$, $-1 < -1+\frac{1}{2a} < 0$ 이므로 $0 \le x \le 2\pi$ 에서 $\cos x = 1-\frac{1}{2a}$ 와 $\cos x = -1+\frac{1}{2a}$ 은 각각 $2$ 개의 근을 가지므로 $10$ 이하의 자연수 $a$ 에 대하여 항상 $4$ 개의 실근을 갖는다.
$(a, b)$ 의 개수는 $10$ 이다.

(ⅱ) $(a-2)(b-2) \ne 0$ 일 때,
$f(x) = | 2a \cos \frac{b}{2}x - (a-2)(b-2) |$ 의 주기는 $\frac{2\pi}{\frac{b}{2}} = \pi$ 에서 $b=4$
$f(x) = | 2a \cos 2x - 2(a-2) |$ 이므로
$| 2a \cos 2x - 2(a-2) | = 2a-1$ 에서
$2a \cos 2x - 2a + 4 = -2a+1$ 또는 $2a \cos 2x - 2a + 4 = 2a-1$
$\therefore 2a \cos 2x = -3$ 또는 $2a \cos 2x = 4a-5 \cdots \cdots$ ㉠

(1) $a=1$ 일 때,
$\cos 2x = -\frac{3}{2}$ 또는 $\cos 2x = -\frac{1}{2}$

> [!example] 📸 (그림/그래프)

$\cos 2x = -\frac{3}{2}$ 을 만족하는 $x$ 의 값은 존재하지 않고, $\cos 2x = -\frac{1}{2}$ 을 만족하는 $x$ 의 값은 $4$ 개 존재하므로 조건을 만족한다.

(2) $a \ge 3$ 일 때,
$\cos 2x = -\frac{3}{2a}$ 또는 $\cos 2x = \frac{4a-5}{2a}$

> [!example] 📸 (그림/그래프)

$4a-5 > 2a$ 에서 $\frac{4a-5}{2a} > 1$ 이므로 $\cos 2x = \frac{4a-5}{2a}$ 를 만족하는 $x$ 의 값은 존재하지 않는다.
$2a > 3$ 에서 $-1 < -\frac{3}{2a} < 0$ 이므로 $\cos 2x = -\frac{3}{2a}$ 를 만족하는 $x$ 의 값은 $4$ 개 존재한다.
$4$ 개의 실근을 가지므로 조건을 만족한다.

따라서 $b=4$ 이고 $a=1, 3, 4, 5, \cdots, 10$ 이므로 순서쌍 $(a, b)$ 의 개수는 $9$ 개다.

(ⅰ), (ⅱ)에서 자연수 $a, b$ 의 모든 순서쌍 $(a, b)$ 의 개수는 $10+9=19$ 개다.
