## 문제
$\sin \alpha = \frac{2}{3}$ ($0 < \alpha < \frac{\pi}{2}$), $\cos \beta = \frac{1}{2}$ ($0 < \beta < \frac{\pi}{2}$) 이고 $\sin (\alpha + \beta)$, $\sin (\alpha - \beta)$ 를 두 근으로 하는 이차방정식이 $x^{2} + \frac{a}{3} x + \frac{b}{36} = 0$ 일 때, 상수 $a$, $b$ 의 곱 $ab$ 의 값은?

## 보기
① $18$   ② $19$   ③ $20$   ④ $21$   ⑤ $22$

## 풀이
$\sin \alpha = \frac{2}{3}$ ($0 < \alpha < \frac{\pi}{2}$) 이므로 $\cos \alpha = \sqrt{1 - \sin^{2} \alpha} = \sqrt{1 - \left( \frac{2}{3} \right)^{2}} = \frac{\sqrt{5}}{3}$
$\cos \beta = \frac{1}{2}$ ($0 < \beta < \frac{\pi}{2}$) 이므로 $\sin \beta = \sqrt{1 - \cos^{2} \beta} = \sqrt{1 - \left( \frac{1}{2} \right)^{2}} = \frac{\sqrt{3}}{2}$

이때,
$\sin (\alpha + \beta) = \sin \alpha \cos \beta + \cos \alpha \sin \beta = \frac{2}{3} \cdot \frac{1}{2} + \frac{\sqrt{5}}{3} \cdot \frac{\sqrt{3}}{2} = \frac{2 + \sqrt{15}}{6}$
$\sin (\alpha - \beta) = \sin \alpha \cos \beta - \cos \alpha \sin \beta = \frac{2}{3} \cdot \frac{1}{2} - \frac{\sqrt{5}}{3} \cdot \frac{\sqrt{3}}{2} = \frac{2 - \sqrt{15}}{6}$

이차방정식 $x^{2} + \frac{a}{3} x + \frac{b}{36} = 0$ 에서 근과 계수의 관계에 의하여
두 근의 합:
$-\frac{a}{3} = \frac{2 + \sqrt{15}}{6} + \frac{2 - \sqrt{15}}{6} = \frac{4}{6} = \frac{2}{3}$
$\therefore a = -2$

두 근의 곱:
$\frac{b}{36} = \left( \frac{2 + \sqrt{15}}{6} \right) \left( \frac{2 - \sqrt{15}}{6} \right) = \frac{4 - 15}{36} = -\frac{11}{36}$
$\therefore b = -11$

따라서,
$ab = (-2) \cdot (-11) = 22$
