## 문제
정규분포 $N(80, 5^2)$을 따르는 확률변수 $X$와 정규분포를 따르는 확률변수 $Y$가 $2X+Y=a$를 만족시킨다. 
$P(b \le X \le 75) = 0.1359$, $P(a-160 \le Y \le b) = 0.4332$일 때, 다음 표준정규분포표를 이용하여 $a+b$의 값을 구하시오. (단, $a, b$는 상수이다.)

$$ \begin{array}{|c|c|} \hline z & P(0 \le Z \le z) \\ \hline 0.5 & 0.1915 \\ \hline 1.0 & 0.3413 \\ \hline 1.5 & 0.4332 \\ \hline 2.0 & 0.4772 \\ \hline \end{array} $$

## 풀이
확률변수 $X$가 정규분포 $N(80, 5^2)$을 따르므로, $Z$가 표준정규분포를 따르는 확률변수일 때 
$P(b \le X \le 75) = P\left( \frac{b-80}{5} \le Z \le \frac{75-80}{5} \right)$
$= P\left( \frac{b-80}{5} \le Z \le -1 \right)$
$= P\left( 1 \le Z \le \frac{80-b}{5} \right)$
$= P\left( 0 \le Z \le \frac{80-b}{5} \right) - P(0 \le Z \le 1)$
$= P\left( 0 \le Z \le \frac{80-b}{5} \right) - 0.3413 = 0.1359$

$P\left( 0 \le Z \le \frac{80-b}{5} \right) = 0.4772$
주어진 표준정규분포표를 이용하면 $P(0 \le Z \le 2) = 0.4772$이므로
$\frac{80-b}{5} = 2$, $b = 70$

$Y = -2X + a$이므로
$P(a-160 \le Y \le b) = P(a-160 \le -2X + a \le 70)$
$= P(-160 \le -2X \le 70-a)$
$= P\left( \frac{a-70}{2} \le X \le 80 \right)$
$= P\left( \frac{\frac{a-70}{2}-80}{5} \le Z \le \frac{80-80}{5} \right)$
$= P\left( \frac{a-230}{10} \le Z \le 0 \right)$
$= P\left( 0 \le Z \le \frac{230-a}{10} \right) = 0.4332$

주어진 표준정규분포표를 이용하면 $P(0 \le Z \le 1.5) = 0.4332$이므로
$\frac{230-a}{10} = 1.5$, $a = 215$
따라서 $a + b = 215 + 70 = 285$

**[다른 풀이]**
확률변수 $X$가 정규분포 $N(80, 5^2)$을 따르므로, $Y = -2X + a$에서
$E(Y) = E(-2X + a) = -2E(X) + a = -2 \times 80 + a = -160 + a$
$\sigma(Y) = \sigma(-2X + a) = |-2|\sigma(X) = 2 \times 5 = 10$
이므로 확률변수 $Y = -2X + a$는 정규분포 $N(a-160, 10^2)$을 따른다.

$P(a-160 \le Y \le b) = P(a-160 \le Y \le 70)$
$= P\left( \frac{(a-160)-(a-160)}{10} \le Z \le \frac{70-(a-160)}{10} \right)$
$= P\left( 0 \le Z \le \frac{230-a}{10} \right) = 0.4332$

주어진 표준정규분포표를 이용하면 $P(0 \le Z \le 1.5) = 0.4332$이므로
$\frac{230-a}{10} = 1.5$, $a = 215$
따라서 $a + b = 215 + 70 = 285$
