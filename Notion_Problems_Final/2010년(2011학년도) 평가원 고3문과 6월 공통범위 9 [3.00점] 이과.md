## 문제
어느 세라믹 재료의 열전도 계수 ($k$)는 적절한 실험 조건에서 일정하고, 다음과 같이 계산된다고 한다.
$$k=C \frac{\log t_2 - \log t_1}{T_2 - T_1}$$
(단, $C$는 $0$보다 큰 상수, $T_1$($^\circ\mathrm{C}$), $T_2$($^\circ\mathrm{C}$)는 실험을 시작한 후 각각 $t_1$(초), $t_2$(초)일 때 세라믹 재료의 측정 온도이다.)
이 세라믹 재료의 열전도 계수를 측정하는 실험에서 실험을 시작한 후 $10$초일 때와 $20$초일 때의 측정 온도가 각각 $200^\circ\mathrm{C}$, $202^\circ\mathrm{C}$이었다. 실험을 시작한 후 $x$초일 때 측정 온도가 $206^\circ\mathrm{C}$가 되었다. $x$의 값은?

## 보기
① $70$   ② $80$   ③ $90$   ④ $100$   ⑤ $110$

## 풀이
$t_1 = 10$일 때, $T_1 = 200$
$t_2 = 20$일 때, $T_2 = 202$이므로
$k = C \times \frac{\log 20 - \log 10}{202 - 200} = C \times \frac{\log 2}{2}$ $\cdots$ ㉠

$t_3 = x$일 때, $T_3 = 206$이므로
$k = C \times \frac{\log x - \log 20}{206 - 202} = C \times \frac{\log x - \log 20}{4}$ $\cdots$ ㉡

㉠, ㉡에서
$\frac{\log x - \log 20}{4} = \frac{\log 2}{2}$
$\log x - \log 20 = 2 \log 2$
$\log x = \log 20 + 2 \log 2 = \log 20 + \log 4 = \log 80$
$x = 80$

정답: ②
