## 문제
함수 $f(x)=2x-[2x]$에 대하여 $f \left( \frac{1}{9} \right)$, $f \left( f \left( \frac{1}{9} \right) \right)$, $f \left( f \left( f \left( \frac{1}{9} \right) \right) \right)$, $\cdots$ 의 집합을 $S$라고 할 때, 집합 $S$의 원소의 개수는? (단, $[x]$는 $x$를 넘지 않는 최대의 정수이다. 예를 들면 $[2]=2$, $[2.6]=2$이다.)

## 보기
① $2$   ② $3$   ③ $4$   ④ $5$   ⑤ $6$

## 풀이
$f \left( \frac{1}{9} \right) = \frac{2}{9} - \left[ \frac{2}{9} \right] = \frac{2}{9}$
$f \left( f \left( \frac{1}{9} \right) \right) = f \left( \frac{2}{9} \right) = \frac{4}{9} - \left[ \frac{4}{9} \right] = \frac{4}{9}$
$f \left( f \left( f \left( \frac{1}{9} \right) \right) \right) = f \left( \frac{4}{9} \right) = \frac{8}{9} - \left[ \frac{8}{9} \right] = \frac{8}{9}$
$f \left( \frac{8}{9} \right) = \frac{16}{9} - \left[ \frac{16}{9} \right] = \frac{7}{9}$
$f \left( \frac{7}{9} \right) = \frac{14}{9} - \left[ \frac{14}{9} \right] = \frac{5}{9}$
$f \left( \frac{5}{9} \right) = \frac{10}{9} - \left[ \frac{10}{9} \right] = \frac{1}{9}$
$\vdots$

즉, $6$개의 수 $\frac{2}{9}$, $\frac{4}{9}$, $\frac{8}{9}$, $\frac{7}{9}$, $\frac{5}{9}$, $\frac{1}{9}$이 반복된다. 
따라서 $S = \left\{ \frac{1}{9}, \frac{2}{9}, \frac{4}{9}, \frac{5}{9}, \frac{7}{9}, \frac{8}{9} \right\}$이므로 집합 $S$의 원소의 개수는 $6$이다.
