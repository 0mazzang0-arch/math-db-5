## 문제
자연수 $n$ 에 대하여 $2^{n} \leq x \leq 2^{n+10}$ 에서 $| \log _{2} x-2 n |$ 의 최댓값을 $a_{n}$ 이라 할 때, $\sum_{k=1}^{10} a_{k}$ 의 값을 구하시오.

## 보기

## 풀이
부등식 $2^{n} \leq x \leq 2^{n+10}$ 에 밑을 $2$ 로 하는 로그를 취해주면 
$n \leq \log _{2} x \leq n+10 \Rightarrow -n \leq \log _{2} x-2 n \leq 10-n$

$n<5$ 일 때는 $|10-n|$ 의 절댓값이 더 크므로 
$a_{n}=10-n$ 

$n=5$ 일 때는 $|-n|$ 과 $|10-n|$ 의 절댓값이 같으므로 
$a_{n}=5$ 

$n>5$ 일 때는 $|-n|$ 의 절댓값이 더 크므로 
$a_{n}=n$ 

$\therefore \sum_{k=1}^{10} a_{k}=9+8+7+6+5+6+7+8+9+10=75$
