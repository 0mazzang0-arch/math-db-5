## 문제
공차가 양수인 등차수열 $\{ a_{n} \}$ 과 공차가 음수인 등차수열 $\{ b_{n} \}$ 의 첫째 항부터 $n$ 째 항까지의 합을 각각 $S_{n}$ 과 $T_{n}$ 이라 하자. 다음이 성립할 때, $a_{20}$ 과 $b_{20}$ 의 곱 $a_{20} b_{20}$ 의 값은?
$$ \begin{cases} a_{1} = b_{1} + 1 \\ S_{n}^{2} - T_{n}^{2} = n^{2} (n + 1) & (n = 1, 2, 3, \cdots) \end{cases} $$

## 보기
① $-108$   ② $-105$   ③ $-102$   ④ $-99$   ⑤ $-96$

## 풀이
수열 $\{ a_{n} \}$, $\{ b_{n} \}$ 이 등차수열이므로 수열 $\{ a_{n} + b_{n} \}$, $\{ a_{n} - b_{n} \}$ 도 등차수열이다.

$\sum_{k=1}^{n} (a_{k} + b_{k}) = \sum_{k=1}^{n} a_{k} + \sum_{k=1}^{n} b_{k} = S_{n} + T_{n}$,

$\sum_{k=1}^{n} (a_{k} - b_{k}) = \sum_{k=1}^{n} a_{k} - \sum_{k=1}^{n} b_{k} = S_{n} - T_{n}$ 이고

$S_{n}^{2} - T_{n}^{2} = (S_{n} + T_{n}) (S_{n} - T_{n})$ 이므로

$(S_{n} + T_{n}) (S_{n} - T_{n}) = n^{2} (n + 1)$ $\cdots \cdots$ ㉠

등차수열의 공차가 $0$ 일 때, 이 수열의 합은 $n$ 에 대한 일차식이고, 등차수열의 공차가 $0$ 이 아닐 때, 이 수열의 합은 상수항이 없는 $n$ 에 대한 이차식이다.
그런데 두 등차수열의 합 $S_{n} + T_{n}$, $S_{n} - T_{n}$ 을 곱한 값이 $n^{2} (n + 1)$ 이므로 수열 $\{ a_{n} + b_{n} \}$ 의 공차는 $0$ 이다.
($\because$ 수열 $\{ a_{n} \}$, $\{ b_{n} \}$ 의 공차는 서로 다른 부호)
즉, $a_{n} + b_{n}$ 은 일정한 값을 갖는다.
㉠의 양변에 $n = 1$ 을 대입하면

$(S_{1} + T_{1}) (S_{1} - T_{1}) = (a_{1} + b_{1}) (a_{1} - b_{1}) = 2$ 이고
$a_{1} - b_{1} = 1$ 이므로

$a_{1} + b_{1} = 2$

$\therefore a_{n} + b_{n} = 2$, $S_{n} + T_{n} = 2 n$ $\cdots \cdots$ ㉡

㉡을 ㉠에 대입하여 정리하면 $S_{n} - T_{n} = \frac{n(n + 1)}{2}$

$\therefore a_{n} - b_{n} = n$

$a_{20} - b_{20} = 20$, $a_{20} + b_{20} = 2$ 이므로
$a_{20} b_{20} = \frac{2^{2} - 20^{2}}{4} = -99$
