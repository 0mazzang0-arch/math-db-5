## 문제
두 이차정사각행렬 $A$, $B$가 $AB + A + B = 2E$, $A^3 + E = O$를 만족시킬 때, <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $E$는 단위행렬이고, $O$는 영행렬이다.)

$$
\begin{array}{|l|}
\hline
ㄱ. \ A + E \text{의 역행렬이 존재한다.} \\
ㄴ. \ AB = BA \\
ㄷ. \ A + B = -E \\
\hline
\end{array}
$$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄱ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
[정답] ③

ㄱ. $AB + A + B = 2E$에서
$(A + E)(B + E) = 3E$
$(A + E) \{ \frac{1}{3}(B + E) \} = E$
$\therefore (A + E)^{-1} = \frac{1}{3}(B + E)$
따라서 $A + E$의 역행렬이 존재한다. (참)

ㄴ. $(A + E)(B + E) = 3E$에서
$(B + E)(A + E) = 3E$이므로
$(A + E)(B + E) = (B + E)(A + E)$
$AB + A + B + E = BA + A + B + E$
$\therefore AB = BA$ (참)

ㄷ. $A^3 + E = O$에서
$(A + E)(A^2 - A + E) = O$
$(A + E)^{-1}(A + E)(A^2 - A + E) = O$
$A^2 - A + E = O$
$\therefore A^2 = A - E$

$AB + A + B = 2E$의 양변의 왼쪽에 행렬 $A$를 곱하면
$A^2 B + A^2 + AB = 2A$
$(A - E)B + (A - E) + AB = 2A$
$2AB - B + A - E = 2A$
$2AB = A + B + E$

$AB = 2E - A - B$이므로 이를 대입하면
$2(2E - A - B) = A + B + E$
$4E - 2A - 2B = A + B + E$
$3A + 3B = 3E$
$\therefore A + B = E$ (거짓)

따라서 옳은 것은 ㄱ, ㄴ이다.
