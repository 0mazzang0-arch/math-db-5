## 문제
$0 \le x < 2\pi$ 일 때, 방정식 $|\sin 2x| = \frac{1}{2}$ 의 모든 실근의 합은?



> [!example] 📸 (그림/그래프)

## 보기
① $4\pi$   ② $6\pi$   ③ $8\pi$   ④ $10\pi$   ⑤ $12\pi$

## 풀이
정답: ③

$|\sin 2x| = \frac{1}{2}$ 을 풀면 $\sin 2x = \frac{1}{2}$ 또는 $\sin 2x = -\frac{1}{2}$ 이다.
따라서 그래프와 같이 조건을 만족하는 교점은 $8$ 개이다.
그런데 이 근들은 서로 $x = \pi$ 에 대하여 대칭이므로, 제일 작은 근을 $\alpha$ 라 하면 $8$ 개의 근은 다음과 같다.
$\alpha, \frac{\pi}{2} - \alpha, \frac{\pi}{2} + \alpha, \pi - \alpha, \pi + \alpha, \frac{3}{2}\pi - \alpha, \frac{3}{2}\pi + \alpha, 2\pi - \alpha$
따라서 모든 실근의 합을 구하면 다음과 같다.
$\alpha + (\frac{\pi}{2} - \alpha) + (\frac{\pi}{2} + \alpha) + (\pi - \alpha) + (\pi + \alpha) + (\frac{3}{2}\pi - \alpha) + (\frac{3}{2}\pi + \alpha) + (2\pi - \alpha) = 8\pi$
즉, 만족하는 실근의 합은 $8\pi$ 이다.
