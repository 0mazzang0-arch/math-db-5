## 문제
부등식 $(\log_{3} x)(\log_{3} 3x) \le 20$ 을 만족시키는 자연수 $x$ 의 최댓값을 구하시오.

## 보기

## 풀이
$(\log_{3} x)(\log_{3} 3x) \le 20$ 에서
$(\log_{3} x)(\log_{3} 3 + \log_{3} x) \le 20$
$(\log_{3} x)(1 + \log_{3} x) \le 20$
$\log_{3} x = t$ 로 놓으면
$t(1 + t) \le 20$
$t^{2} + t - 20 \le 0$
$(t + 5)(t - 4) \le 0$
$\therefore -5 \le t \le 4$
$-5 \le \log_{3} x \le 4$
$3^{-5} \le x \le 3^{4}$
$\therefore \frac{1}{243} \le x \le 81$
따라서, 자연수 $x$ 의 최댓값은 $81$ 이다.

정답: $81$
