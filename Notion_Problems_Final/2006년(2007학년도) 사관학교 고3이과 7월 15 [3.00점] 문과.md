## 문제

함수 $f(x)$의 그래프가 그림과 같을 때, 함수 $g(x)$를
$$g(x)= \int_{0}^{x} f(t)dt$$
라 하자. <보기>에서 옳은 것을 모두 고른 것은? (단, 두 함수 $f(x)$, $g(x)$의 정의역은 $\{ x | 0 \le x \le 7 \}$이다.)

> [!example] 📸 (그림/그래프)
> (그래프 생략)

ㄱ. $g(x)$는 $x=5$에서 극댓값을 갖는다.
ㄴ. $g(x)$는 $x=1$에서 최솟값을 갖는다.
ㄷ. $g(5)=g(1) - \left| \int_{1}^{3} f(t)dt \right| + \left| \int_{3}^{5} f(t)dt \right|$

## 보기

① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄷ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이

[정답] ③

주어진 식의 양 변을 $x$에 관하여 미분하면
$$g'(x)=f(x)$$

ㄱ. $x=5$일 때, $g'(5)=0$이고 $g'(x)$의 부호가 양 $\rightarrow$ 음으로 바뀌므로 $y=g(x)$는 극대이다. (참)

ㄴ. 함수 $g(x)$는 주어진 구간 $(0, 7)$에서 미분가능하므로 연속이다. 따라서, 최솟값은 극소 또는 경계에서 나타난다.
$g(0)=\alpha$라고 가정하면
$g(3)=g(0)+\int_{0}^{3} f(x)dx = g(0)+ \text{면적 } A - \text{면적 } B$
$g(5)=g(0)+\int_{0}^{5} f(x)dx = g(0)+ \text{면적 } A - \text{면적 } B + \text{면적 } C - \text{면적 } D$
$\text{면적 } A < \text{면적 } B < \text{면적 } C < \text{면적 } D$이므로 $g(x)$는 $x=5$에서 최솟값을 갖는다. (거짓)

ㄷ. $g(5)=g(1) - \left| \int_{1}^{3} f(t)dt \right| + \left| \int_{3}^{5} f(t)dt \right|$
$= g(1) + \int_{1}^{3} f(t)dt + \int_{3}^{5} f(t)dt$
($\because \int_{1}^{3} f(t)dt < 0, \int_{3}^{5} f(t)dt > 0$)
$= g(1) + \int_{1}^{5} f(t)dt$
$= g(1) + [g(t)]_{1}^{5}$
$= g(1) + (g(5) - g(1))$
$= g(5)$ (참)
