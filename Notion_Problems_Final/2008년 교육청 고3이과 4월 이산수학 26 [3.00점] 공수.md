## 문제

두 집합 $X = \{ 1, 2, 3, 4, 5 \}$, $Y = \{ 1, 2, 3, 4, 5, 6, 7 \}$에 대하여 다음 조건을 모두 만족하는 함수 $f : X \to Y$의 개수는?

$$ \begin{array}{l} (가) \ f(4) = 5 \\ (나) \ \text{집합 } X \text{의 임의의 두 원소 } x_{1}, x_{2} \text{에 대하여 } x_{1} < x_{2} \text{이면 } f(x_{1}) < f(x_{2}) \text{이다.} \end{array} $$

## 보기

① $8$ ② $9$ ③ $10$ ④ $11$ ⑤ $12$

## 풀이

조건 (나)에 의하여 원소의 순서가 정해지면 함숫값이 자동으로 결정되므로 조합을 이용한다.

$1.$ $f(1), f(2), f(3)$의 값을 결정하는 방법:
$x_{1} < x_{2} < x_{3} < 4$이므로 $f(x_{1}) < f(x_{2}) < f(x_{3}) < f(4) = 5$를 만족해야 한다.
따라서 $\{ 1, 2, 3, 4 \}$에서 $3$개를 선택하는 방법의 수는 $_{4}C_{3} = 4$이다.

$2.$ $f(5)$의 값을 결정하는 방법:
$4 < 5$이므로 $5 = f(4) < f(5)$를 만족해야 한다.
따라서 $\{ 6, 7 \}$에서 $1$개를 선택하는 방법의 수는 $_{2}C_{1} = 2$이다.

따라서 구하는 함수의 개수는 곱의 법칙에 의해
$\therefore _{4}C_{3} \times _{2}C_{1} = 4 \times 2 = 8$

정답: ①
