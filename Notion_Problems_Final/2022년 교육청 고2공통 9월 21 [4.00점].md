## 문제
양수 $a$ 와 $0$ 이 아닌 실수 $d$ 에 대하여 첫째항이 모두 $a$ 이고, 공차가 각각 $d$, $-2d$ 인 두 등차수열 $\{ a_{n} \}$ 과 $\{ b_{n} \}$ 이 다음 조건을 만족시킨다.

(가) $| a_{1} | = | b_{7} |$
(나) $S_{n} = \sum_{k=1}^{n} ( | a_{k} | - | b_{k} | )$ 라 할 때, 모든 자연수 $n$ 에 대하여 $S_{n} \leq 108$ 이고, $S_{p} = 108$ 인 자연수 $p$ 가 존재한다.

$S_{n} \geq 0$ 을 만족시키는 자연수 $n$ 의 최댓값을 $m$ 이라 할 때, $a_{m}$ 의 값은?

## 보기
① $46$   ② $50$   ③ $54$   ④ $58$   ⑤ $62$

## 풀이
$a_{n} = a + ( n - 1 ) d$, 
$b_{n} = a + ( n - 1 ) ( -2d )$

조건 (가)에서 
$| a | = | a - 12d |$

$a = a - 12d$ 또는 
$a = -a + 12d$

$d \neq 0$ 이므로 
$a = -a + 12d$ 에서 
$a = 6d$

$a_{n} = 6d + ( n - 1 ) d = ( n + 5 ) d$
$b_{n} = 6d - 2 ( n - 1 ) d = ( -2n + 8 ) d$

$a$ 는 양수이므로 $d > 0$ 이고, 모든 자연수 $n$ 에 대하여 $a_{n} > 0$ 이다.
$1 \leq n \leq 3$ 일 때 $b_{n} > 0$ 이고, $n \geq 4$ 일 때 $b_{n} \leq 0$ 이므로 수열 $\{ c_{n} \}$ 을 $c_{n} = | a_{n} | - | b_{n} |$ 이라 하면

$c_{n} = \begin{cases} ( n + 5 ) d - ( -2n + 8 ) d & ( 1 \leq n \leq 3 ) \\ ( n + 5 ) d - ( 2n - 8 ) d & ( n \geq 4 ) \end{cases}$

$c_{n} = \begin{cases} 3 ( n - 1 ) d & ( 1 \leq n \leq 3 ) \\ ( 13 - n ) d & ( n \geq 4 ) \end{cases}$

$1 \leq n \leq 13$ 일 때 $c_{n} \geq 0$ 이고 $c_{13} = 0$ 이며, $n \geq 14$ 일 때 $c_{n} < 0$ 이다.
$S_{n} = \sum_{k=1}^{n} ( | a_{k} | - | b_{k} | ) = \sum_{k=1}^{n} c_{k}$ 의 값이 최대가 되는 $n$ 은 $12$ 또는 $13$ 이다.

$S_{12} = S_{13} = \sum_{n=1}^{13} c_{n} = \sum_{n=1}^{3} 3 ( n - 1 ) d + \sum_{n=4}^{13} ( 13 - n ) d$
$= 9d + 45d = 54d = 108$
$d = 2$

따라서 수열 $\{ c_{n} \}$ 은
$c_{n} = \begin{cases} 6 ( n - 1 ) & ( 1 \leq n \leq 3 ) \\ 2 ( 13 - n ) & ( n \geq 4 ) \end{cases}$

$c_{1} = 0, c_{2} = 6, c_{3} = 12$
$c_{4} = -c_{22}, c_{5} = -c_{21}, c_{6} = -c_{20}, \dots, c_{12} = -c_{14}, c_{13} = 0$
$c_{23} = -20$

$S_{22} = ( c_{1} + c_{2} + c_{3} ) + ( c_{4} + c_{5} + \dots + c_{22} ) = ( 0 + 6 + 12 ) + 0 = 18$
$S_{23} = S_{22} + c_{23} = 18 + ( -20 ) = -2$

$n \geq 23$ 일 때 $S_{n} < 0$ 이므로 $S_{n} \geq 0$ 을 만족시키는 자연수 $n$ 의 최댓값 $m = 22$ 이다.
따라서 $a_{m} = a_{22} = ( 22 + 5 ) \times 2 = 54$ 이다.

정답: ③
