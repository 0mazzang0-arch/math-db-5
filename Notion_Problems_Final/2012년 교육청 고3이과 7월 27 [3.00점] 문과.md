## 문제
$0 \le x \le \pi$ 일 때, 방정식 $2 \sin x \cos x - \cos 2x + 1 = 0$ 을 만족시키는 모든 실근의 합은 $\frac{q}{p} \pi$ 이다. $p + q$ 의 값을 구하시오. (단, $p$ 와 $q$ 는 서로소인 자연수이다.)

## 풀이
$2 \sin x \cos x + 2 \sin^{2} x = 0$

$2 \sin x ( \sin x + \cos x ) = 0$

$2 \sqrt{2} \sin x \sin \left( x + \frac{\pi}{4} \right) = 0$

$\sin x = 0$ 또는 $\sin \left( x + \frac{\pi}{4} \right) = 0$

$0 \le x \le \pi$ 이므로

$x = 0, \frac{3}{4} \pi, \pi$

$\therefore$ 모든 실근의 합은 $\frac{7}{4} \pi$

$\frac{q}{p} \pi = \frac{7}{4} \pi$ 이고 $p, q$ 는 서로소인 자연수이므로
$p = 4, q = 7$
$p + q = 11$
