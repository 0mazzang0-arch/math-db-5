## 문제
$0 < x < 2 \pi$ 일 때, 방정식 $(\cos 2x - \cos x) \sin x = 0$ 을 만족시키는 모든 해의 합은 $k \pi$ 이다. $10k$ 의 값을 구하시오.

## 풀이
$\cos 2x = 2 \cos^2 x - 1$ 이므로
$(\cos 2x - \cos x) \sin x = (2 \cos^2 x - \cos x - 1) \sin x = (\cos x - 1)(2 \cos x + 1) \sin x = 0$

(ⅰ) $\sin x = 0$ 에서 $x = \pi$
(ⅱ) $\cos x = 1$ 을 만족시키는 $x$ 는 $0 < x < 2 \pi$ 에 존재하지 않는다.
(ⅲ) $\cos x = -\frac{1}{2}$ 에서 $x = \frac{2}{3} \pi$ 또는 $x = \frac{4}{3} \pi$

따라서 구하는 모든 해의 합은 $\pi + \frac{2}{3} \pi + \frac{4}{3} \pi = 3 \pi$
따라서 $k = 3$ 이므로 $10k = 30$

정답: $30$

**[다른 풀이]**
위의 (ⅱ), (ⅲ) 경우는 $\cos 2x - \cos x = -2 \sin \frac{3}{2} x \sin \frac{x}{2}$ 임을 이용하여 풀 수도 있다.
$0 < \frac{3}{2} x < 3 \pi$ 이고, $0 < \frac{x}{2} < \pi$ 이므로 $\cos 2x - \cos x = 0$ 의 해는 $\frac{3}{2} x = \pi$ 또는 $\frac{3}{2} x = 2 \pi$ 이고 $x = \frac{2}{3} \pi$ 또는 $x = \frac{4}{3} \pi$ 이다.
