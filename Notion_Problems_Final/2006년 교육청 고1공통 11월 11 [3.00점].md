## 문제
$x$에 대한 두 다항식 $x^{3} - x$, $x^{2} + (a + 1)x + a$의 최대공약수가 이차식이 되도록 하는 모든 $a$ 값의 합은?

## 보기
① $-3$   ② $-1$   ③ $1$   ④ $3$   ⑤ $5$

## 풀이
다항식 $x^{3} - x = x(x - 1)(x + 1)$, $x^{2} + (a + 1)x + a = (x + 1)(x + a)$에서 최대공약수가 이차식이 되려면 $x^{2} + (a + 1)x + a$ 자체가 최대공약수이어야 한다.

즉, $(x + 1)(x + a)$가 $x(x - 1)(x + 1)$의 약수이어야 하므로
$x + a = x$ 또는 $x + a = x - 1$이다.

$x + a = x \Rightarrow a = 0$
$x + a = x - 1 \Rightarrow a = -1$

따라서 모든 $a$의 값의 합은 $0 + (-1) = -1$이다.

정답: ②
