## 문제
두 실수 $x$, $y$ ($x > y$)가 $x + y = 1$, $xy = -1$을 만족시킬 때, 수열 $\{ a_n \}$을 
$a_n = \sum_{k=1}^{n} x^{n-k} y^{k-1}$ ($n = 1, 2, 3, \dots$)으로 정의하자. 다음은 수열 $\{ a_n \}$의 제 $n$항을 구하는 과정이다.

$$
\begin{array}{|l|}
\hline
x + y = 1, xy = -1\text{에서 두 실수 } x, y\text{는 방정식 } t^2 - t + (가) = 0\text{의 두 근이다.} \\
\text{한편 } a_n = \sum_{k=1}^{n} x^{n-k} y^{k-1} = x^{n-1} + x^{n-2} y + \dots + y^{n-1} \dots (*) \\
(*)\text{은 첫째항이 } x^{n-1}\text{이고 공비가 } \frac{y}{x}\text{인 등비수열의 첫째항부터 제 } n\text{항까지의 합이므로} \\
a_n = \frac{(나)}{\sqrt{5}} \\
\hline
\end{array}
$$

위의 과정에서 $(가)$에 들어갈 수를 $m$, $(나)$에 알맞은 식을 $f(n)$이라 할 때, $m + \{ f(3) \}^2$의 값은?

## 보기
① $17$   ② $19$   ③ $21$   ④ $23$   ⑤ $25$

## 풀이
$(t - x)(t - y) = t^2 - (x + y) t + xy$ 이고 
$x + y = 1$, $xy = -1 \dots ㉠$ 이므로
$m = xy = -1$

$a_n = \frac{x^{n-1} \left\{ 1 - \left( \frac{y}{x} \right)^n \right\}}{1 - \frac{y}{x}} = \frac{x^n - y^n}{x - y}$

㉠에서
$(x - y)^2 = (x + y)^2 - 4xy = 1^2 - 4(-1) = 5$
$\therefore x - y = \sqrt{5}$ ($\because x > y$)

$\therefore a_n = \frac{x^n - y^n}{\sqrt{5}}$
$\therefore f(n) = x^n - y^n$

$\therefore f(3) = x^3 - y^3$
$= (x - y)^3 + 3xy(x - y)$
$= (\sqrt{5})^3 + 3(-1)(\sqrt{5})$
$= 5\sqrt{5} - 3\sqrt{5} = 2\sqrt{5}$

$\therefore m + \{ f(3) \}^2 = -1 + (2\sqrt{5})^2 = -1 + 20 = 19$

정답: ②
