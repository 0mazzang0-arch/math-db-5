## 문제
$\log _{x}(8-x-y)$ 가 정의되기 위한 자연수 $x$, $y$ 의 순서쌍 $(x, y)$ 의 개수를 구하시오.



> [!example] 📸 (그림/그래프)

## 보기

## 풀이
밑의 조건에서 
$x>0$, $x \neq 1 \cdots$ ㉠ 
진수 조건에서 
$8-x-y>0 \cdots$ ㉡ 
㉠, ㉡을 동시에 만족하는 자연수 $x$, $y$ 의 순서쌍 $(x, y)$ 의 개수는 다음과 같다.

$x=2$ 일 때, $x+y < 8$ 에서 $y < 6$ 이므로 $y=1, 2, 3, 4, 5$ 의 $5$ 개
$x=3$ 일 때, $x+y < 8$ 에서 $y < 5$ 이므로 $y=1, 2, 3, 4$ 의 $4$ 개
$x=4$ 일 때, $x+y < 8$ 에서 $y < 4$ 이므로 $y=1, 2, 3$ 의 $3$ 개
$x=5$ 일 때, $x+y < 8$ 에서 $y < 3$ 이므로 $y=1, 2$ 의 $2$ 개
$x=6$ 일 때, $x+y < 8$ 에서 $y < 2$ 이므로 $y=1$ 의 $1$ 개

따라서 구하는 자연수 $x$, $y$ 의 순서쌍 $(x, y)$ 의 개수는
$5+4+3+2+1=15$
개이다.

정답: $15$
