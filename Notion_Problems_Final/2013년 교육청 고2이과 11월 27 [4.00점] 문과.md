## 문제
자연수 $n$ 에 대하여 $\log n$ 의 정수부분과 소수부분을 각각 $f(n)$, $g(n)$ 이라 하자. 두 집합 $A$, $B$ 를 
$$A = \{ n \mid f(3n) = f(n) + 1, n \text{ 은 } 50 \text{ 이하의 자연수} \}$$ 
$$B = \{ g(n) \mid n \in A \}$$ 
라 할 때, 집합 $B$ 의 원소의 개수를 구하시오.

## 보기


## 풀이
i) $1 \leq n \leq 9$ 일 때, $f(n) = 0$ 이므로 
$f(3n) = f(n) + 1 = 1$ 이기 위해서는 
$10 \leq 3n \leq 99$ 즉, $\frac{10}{3} \leq n \leq 33$ 
$\therefore 4 \leq n \leq 9$ 

ii) $10 \leq n \leq 50$ 일 때, $f(n) = 1$ 이므로 
$f(3n) = f(n) + 1 = 2$ 이기 위해서는 
$100 \leq 3n \leq 999$ 즉, $\frac{100}{3} \leq n \leq 333$ 
$\therefore 34 \leq n \leq 50$ 

따라서 $A = \{ n \mid 4 \leq n \leq 9 \text{ 또는 } 34 \leq n \leq 50, n \text{ 은 자연수} \}$ 이다. 

$g(40) = \log 40 - f(40) = (1 + \log 4) - 1 = \log 4 = g(4)$ 
$g(50) = \log 50 - f(50) = (1 + \log 5) - 1 = \log 5 = g(5)$ 
인 경우 이외에는 $g(n)$ 은 서로 다른 값을 갖는다. 

$n(A)$ 의 원소의 개수는 $(9 - 4 + 1) + (50 - 34 + 1) = 6 + 17 = 23$ 이므로 
$n(B) = n(A) - 2 = 23 - 2 = 21$ 

따라서 집합 $B$ 의 원소의 개수는 $21$ 이다.

**[정답]** $21$
