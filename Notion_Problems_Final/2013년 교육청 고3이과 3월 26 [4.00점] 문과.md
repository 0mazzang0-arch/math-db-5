## 문제
공기 중의 암모니아 농도가 $C$일 때 냄새의 세기 $I$는 다음 식을 만족시킨다고 한다. 

$$I = k \log C + a$$

(단, $k$와 $a$는 상수이다.) 
공기 중의 암모니아 농도가 $40$일 때 냄새의 세기는 $5$이고, 공기 중의 암모니아 농도가 $10$일 때 냄새의 세기는 $4$이다. 공기 중의 암모니아 농도가 $p$일 때 냄새의 세기는 $2.5$이다. $100 p$의 값을 구하시오. (단, 암모니아 농도의 단위는 $\text{ppm}$이다.)

## 보기
(해당 문제는 주관식 문항으로 보기가 없습니다.)

## 풀이
$C = 40$일 때 $I = 5$이므로 
$5 = k \log 40 + a$ $\cdots$ ㉠ 

$C = 10$일 때 $I = 4$이므로 
$4 = k \log 10 + a$ $\cdots$ ㉡ 

㉠ $-$ ㉡에서 
$1 = k(\log 40 - \log 10) = k \log 4$ 

$\therefore k = \frac{1}{\log 4} = \log_{4} 10$ 

㉡에 대입하면 
$a = 4 - \log_{4} 10$ 

$\therefore I = (\log_{4} 10) \log C + 4 - \log_{4} 10$ 

$C = p$일 때 $I = 2.5$이므로 
$2.5 = \log_{4} 10 \times \log p + 4 - \log_{4} 10$ 
$-1.5 = (\log p - 1) \log_{4} 10$ 

$\therefore \log \frac{p}{10} = -1.5 \log 4 = \log 4^{-\frac{3}{2}} = \log \frac{1}{8}$ 

따라서 $\frac{p}{10} = \frac{1}{8}$이므로 
$p = 1.25$ 

$\therefore 100 p = 125$

정답: $125$
