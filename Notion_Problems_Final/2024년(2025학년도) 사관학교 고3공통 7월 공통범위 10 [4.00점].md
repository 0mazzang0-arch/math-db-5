## 문제
$-\frac{1}{2} < t < 0$인 실수 $t$에 대하여 직선 $x=t$가 두 곡선 $y= \log _{2} (x+1)$, $y= \log _{\frac{1}{2}} (-x)+1$과 만나는 점을 각각 $A$, $B$라 하고, 점 $B$를 지나고 $x$축에 평행한 직선이 곡선 $y= \log _{2} (x+1)$과 만나는 점을 $C$라 하자. $\overline{ AB } = \log _{2} 9$일 때, 선분 $BC$의 길이는?



> [!example] 📸 (그림/그래프)

## 보기
① $4$   ② $\frac{13}{3}$   ③ $\frac{14}{3}$   ④ $5$   ⑤ $\frac{16}{3}$

## 풀이
점 $A$의 $x$좌표를 $t$라 하면 $A(t, \log _{2} (t+1))$, $B(t, \log _{\frac{1}{2}} (-t)+1)$이다.
$\overline{ AB } = \log _2 9$이므로 
$\log _{\frac{1}{2}} (-t) + 1 - \log _2 (t+1) = \log _2 9$
$-\log _2 (-t) + \log _2 2 - \log _2 (t+1) = \log _2 9$
$\log _2 (-t) + \log _2 (t+1) = \log _2 2 - \log _2 9 = \log _2 \frac{2}{9}$
$-t(t+1) = \frac{2}{9}$
$9t^2 + 9t + 2 = 0$
$(3t+2)(3t+1) = 0$
따라서 $t = -\frac{2}{3}$ 또는 $t = -\frac{1}{3}$이다.
$-\frac{1}{2} < t < 0$이므로 $t = -\frac{1}{3}$이다.
점 $B$의 $y$좌표가 $\log _{\frac{1}{2}} (\frac{1}{3}) + 1 = \log _2 3 + 1$이므로 점 $C$의 $x$좌표는
$\log _2 3 + 1 = \log _2 (x+1)$
$\log _2 6 = \log _2 (x+1)$
따라서 $x=5$이다.
선분 $BC$의 길이는 $5 - (-\frac{1}{3}) = \frac{16}{3}$이다.
정답은 ⑤이다.
