## 문제
두 수열 $\{ a_{n} \}$, $\{ b_{n} \}$ 이 모든 자연수 $n$ 에 대하여 다음 조건을 만족시킨다.

$$ \begin{array}{l} \text{(가) } 4^{n} < a_{n} < 4^{n} + 1 \\ \text{(나) } 2 + 2^{2} + 2^{3} + \dots + 2^{n} < b_{n} < 2^{n+1} \end{array} $$

$\lim_{n \to \infty} \frac{4 a_{n} + b_{n}}{2 a_{n} + 2^{n} b_{n}}$ 의 값은?

## 보기
① $\frac{1}{4}$   ② $\frac{1}{2}$   ③ $1$   ④ $2$   ⑤ $4$

## 풀이
[정답] ③

조건 (가)에서 양변을 $4^{n}$ 으로 나누면
$1 < \frac{a_{n}}{4^{n}} < 1 + \frac{1}{4^{n}}$
$\lim_{n \to \infty} 1 = \lim_{n \to \infty} \left( 1 + \frac{1}{4^{n}} \right) = 1$ 이므로
$\lim_{n \to \infty} \frac{a_{n}}{4^{n}} = 1$ … ㉠

조건 (나)에서
$2 + 2^{2} + 2^{3} + \dots + 2^{n} = \frac{2 ( 2^{n} - 1 )}{2 - 1} = 2^{n+1} - 2$
$2^{n+1} - 2 < b_{n} < 2^{n+1}$ 에서 양변을 $2^{n}$ 으로 나누면
$2 - \frac{2}{2^{n}} < \frac{b_{n}}{2^{n}} < 2$
$\lim_{n \to \infty} \left( 2 - \frac{2}{2^{n}} \right) = \lim_{n \to \infty} 2 = 2$ 이므로
$\lim_{n \to \infty} \frac{b_{n}}{2^{n}} = 2$ … ㉡

㉠, ㉡에 의하여
$\lim_{n \to \infty} \frac{4 a_{n} + b_{n}}{2 a_{n} + 2^{n} b_{n}} = \lim_{n \to \infty} \frac{\frac{4 a_{n} + b_{n}}{4^{n}}}{\frac{2 a_{n} + 2^{n} b_{n}}{4^{n}}}$
$= \lim_{n \to \infty} \frac{4 \times \frac{a_{n}}{4^{n}} + \frac{b_{n}}{2^{n}} \times \frac{1}{2^{n}}}{2 \times \frac{a_{n}}{4^{n}} + \frac{b_{n}}{2^{n}}}$
$= \frac{4 \times 1 + 2 \times 0}{2 \times 1 + 2} = \frac{4}{4} = 1$
