## 문제
최고차항의 계수가 $1$인 사차함수 $f(x)$에 대하여 함수 $g(x)$가 다음 조건을 만족시킨다.

$$ \begin{array}{l} \text{(가) } -1 \le x < 1 \text{ 일 때, } g(x) = f(x) \text{이다.} \\ \text{(나) 모든 실수 } x \text{ 에 대하여 } g(x+2) = g(x) \text{이다.} \end{array} $$

옳은 것만을 <보기>에서 있는 대로 고른 것은?

<보기>
ㄱ. $f(-1) = f(1)$ 이고 $f'(-1) = f'(1)$ 이면, $g(x)$는 실수 전체의 집합에서 미분가능하다.
ㄴ. $g(x)$가 실수 전체의 집합에서 미분가능하면, $f'(0)f'(1) < 0$이다.
ㄷ. $g(x)$가 실수 전체의 집합에서 미분가능하고 $f'(1) > 0$ 이면, 구간 $(-\infty, -1)$에 $f'(c) = 0$인 $c$가 존재한다.



> [!example] 📸 (그림/그래프)

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄷ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
ㄱ. $g(x)$는 $-1 < x < 1$에서 미분가능하므로 $x = -1$과 $x = 1$에서 미분가능성만 확인하면 된다.
(ⅰ) $x = -1$일 때
$g'(-1) = \lim_{h \to 0} \frac{g(-1+h) - g(-1)}{h}$
$\lim_{h \to 0+} \frac{g(-1+h) - g(-1)}{h} = \lim_{h \to 0+} \frac{f(-1+h) - f(-1)}{h} = f'(-1)$
$\lim_{h \to 0-} \frac{g(-1+h) - g(-1)}{h} = \lim_{h \to 0-} \frac{f(1+h) - f(1)}{h} = f'(1)$
$f'(-1) = f'(1)$이므로 $g'(-1)$은 존재한다.

(ⅱ) $x = 1$일 때
$g'(1) = \begin{cases} \lim_{h \to 0+} \frac{g(1+h) - g(1)}{h} = \lim_{h \to 0+} \frac{f(-1+h) - f(-1)}{h} = f'(-1) \\ \lim_{h \to 0-} \frac{g(1+h) - g(1)}{h} = \lim_{h \to 0-} \frac{f(1+h) - f(1)}{h} = f'(1) \end{cases}$
$\therefore g'(1)$은 존재한다.
$\therefore g(x)$는 모든 실수에서 미분가능하다.
$\therefore$ 참

ㄴ. ㄱ과 마찬가지 방법으로 $g(x)$가 실수 전체에서 미분가능하면 $f(1) = f(-1)$, $f'(1) = f'(-1)$이 되어야 한다.
$f(x) - f(1) = (x+1)(x-1)(x^2 + ax + b)$가 된다.
$f'(x) = 2x(x^2 + ax + b) + (x^2 - 1)(2x + a)$
$f'(1) = 2(1 + a + b)$
$f'(-1) = -2(1 - a + b)$
$f'(1) = f'(-1)$이므로
$2 + 2a + 2b = -2 + 2a - 2b$
$4 + 4b = 0$
$\therefore b = -1$
$\therefore f'(0) = -a$, $f'(1) = 2(1 + a - 1) = 2a$
$\therefore f'(0) \times f'(1) = -2a^2 \le 0$
$a = 0$일 때 $f(x) = (x^2 - 1)^2 + f(1)$인 경우 $f'(0) \times f'(1) = 0$이 된다.
$\therefore$ 거짓

ㄷ. ㄴ에서 $f(x) = (x^2 - 1)(x^2 + ax - 1) + f(1)$이 되는데 $f'(1) = 2a > 0$이면 $f'(-1) = 2a > 0$이 된다.
최고차항 계수가 $1$인 $4$차 함수는 $x \to -\infty$일 때 $f(x) \to \infty$가 되어야 하므로 $f'(x)$는 $x < -1$일 때 $f'(x) < 0$이 되는 $x$가 존재한다.
$f'(-1) > 0$이고 $x < -1$인 어떤 지점에서 $f'(x) < 0$이므로, 사잇값 정리에 의해 $f'(c) = 0$이 되는 $c$가 $(-\infty, -1)$에 적어도 $1$개 존재한다.
$\therefore$ 참

따라서 옳은 것은 ㄱ, ㄷ이다.
