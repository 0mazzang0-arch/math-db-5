## 문제
함수 
$$ f(x) = \begin{cases} \log_{\frac{1}{3}}x & (0 < x \le 1) \\ \log_{3}x & (x > 1) \end{cases} $$
에 대하여 방정식 $f(x) + f(3x) = 3$ 의 모든 실근의 합은?

## 보기
① $\frac{8}{3}$   ② $\frac{26}{9}$   ③ $\frac{28}{9}$   ④ $\frac{10}{3}$   ⑤ $\frac{32}{9}$

## 풀이
(ⅰ) $0 < x \le \frac{1}{3}$ 인 경우 
$0 < x < 3x \le 1$ 이므로 
$f(x) + f(3x)$ 
$= \log_{\frac{1}{3}}x + \log_{\frac{1}{3}}3x$ 
$= \log_{\frac{1}{3}}3x^{2} = 3$ 
$3x^{2} = \frac{1}{27}$ 
$x = -\frac{1}{9}$ 또는 $x = \frac{1}{9}$ 
$0 < x \le \frac{1}{3}$ 이므로 $x = \frac{1}{9}$ 

(ⅱ) $\frac{1}{3} < x \le 1$ 인 경우 
$\frac{1}{3} < x \le 1 < 3x$ 이므로 
$f(x) + f(3x)$ 
$= \log_{\frac{1}{3}}x + \log_{3}3x$ 
$= -\log_{3}x + \log_{3}3 + \log_{3}x = 1$ 
이므로 방정식 $f(x) + f(3x) = 3$ 의 실근은 존재하지 않는다. 

(ⅲ) $x > 1$ 인 경우 
$1 < x < 3x$ 이므로 
$f(x) + f(3x)$ 
$= \log_{3}x + \log_{3}3x$ 
$= \log_{3}3x^{2} = 3$ 
$3x^{2} = 27$ 
$x = -3$ 또는 $x = 3$ 
$x > 1$ 이므로 $x = 3$ 

(ⅰ) $\sim$ (ⅲ)에 의하여 방정식 $f(x) + f(3x) = 3$ 의 모든 실근은 $\frac{1}{9}$, $3$ 이고 그 합은 $\frac{1}{9} + 3 = \frac{28}{9}$ 이다.

정답: ③
