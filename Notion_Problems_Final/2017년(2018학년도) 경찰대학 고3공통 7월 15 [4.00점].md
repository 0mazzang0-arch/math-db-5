## 문제
방정식 $2x^2 = x + 3[x]$ 의 실근의 개수를 $p$, 모든 실근의 합을 $q$ 라 할 때, $pq$ 의 값은? (단, $[x]$ 는 $x$ 를 넘지 않는 최대 정수이다.)

## 보기
① $12$   ② $13$   ③ $14$   ④ $15$   ⑤ $16$

## 풀이
$n \le x < n + 1$ 이면 $[x] = n$ 이므로 $2x^2 = x + 3n$ 인데, 양변의 그래프는 그림과 같다.

> [!example] 📸 (그림/그래프)

따라서 실근은 
$0 \le x < 1 : 2x^2 = x \implies x = 0, x = \frac{1}{2}$
$1 \le x < 2 : 2x^2 = x + 3 \implies x = \frac{3}{2}$
$2 \le x < 3 : 2x^2 = x + 6 \implies x = 2$

$\therefore p = 4$
$q = 0 + \frac{1}{2} + \frac{3}{2} + 2 = 4$
$pq = 16$

[정답] ⑤
