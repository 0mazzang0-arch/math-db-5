## 문제
삼차다항식 $P(x)$가 다음 조건을 만족시킨다.

(가) $(x-1)P(x-2) = (x-7)P(x)$
(나) $P(x)$를 $x^{2}-4x+2$로 나눈 나머지는 $2x-10$이다.

$P(4)$의 값은?

## 보기
① $-6$   ② $-3$   ③ $0$   ④ $3$   ⑤ $6$

## 풀이
정답: ① 

[해설] 
조건 (가)에서 
$$(x-1)P(x-2) = (x-7)P(x)$$ 
위 식의 양변에 $x=1$, $x=7$을 각각 대입하면 
$P(1)=0$, $P(5)=0$ 
$\cdots \cdots$ ㉠ 

$P(x)$는 삼차다항식이므로 조건 (나)에서 $P(x)$를 $x^{2}-4x+2$로 나누었을 때의 몫을 $ax+b$ ($a$, $b$는 상수)라 하면 나머지는 $2x-10$이므로 
$$P(x) = (x^{2}-4x+2)(ax+b) + 2x-10$$ 
위 식의 양변에 $x=1$, $x=5$를 각각 대입하면 
$P(1) = -a-b-8=0$, $P(5) = 35a+7b=0$ ($\because$ ㉠) 
위 두 식을 연립하면 $a=2$, $b=-10$ 
따라서 $P(x) = (x^{2}-4x+2)(2x-10) + 2x-10$이므로 
$P(4) = -6$ 

[다른 풀이] 
조건 (가)에서 
$$(x-1)P(x-2) = (x-7)P(x)$$ 
$\cdots \cdots$ ㉠ 
위 식의 양변에 $x=1$, $x=7$을 각각 대입하면 
$P(1)=0$, $P(5)=0$ 
이때, $P(x)$는 삼차다항식이므로 상수 $a$, $k$에 대하여 
$$P(x) = a(x-1)(x-5)(x-k) \quad (a \ne 0)$$ 
로 놓을 수 있다. 이것을 ㉠에 대입하면 
$$a(x-1)(x-3)(x-7)(x-k-2) = a(x-7)(x-1)(x-5)(x-k)$$ 
위 등식은 $x$에 대한 항등식이므로 
$$(x-3)(x-k-2) = (x-5)(x-k)$$ 
$\therefore k=3$ 
따라서 $P(x) = a(x-1)(x-3)(x-5)$ 
이때, 조건 (나)에서 $P(x)$를 $x^{2}-4x+2$로 나누었을 때의 몫을 $Q(x)$라 하면 나머지는 $2x-10$이므로 
$$a(x-1)(x-3)(x-5) = (x^{2}-4x+2)Q(x) + 2x-10$$ 
$$\therefore a(x^{2}-4x+3)(x-5) = (x^{2}-4x+2)Q(x) + 2x-10$$ 
$\cdots \cdots$ ㉡ 
방정식 $x^{2}-4x+2=0$의 한 근을 $\alpha$라 하면 $\alpha^{2}-4\alpha+2=0$이므로 ㉡의 양변에 $x=\alpha$를 대입하면 
$$a(\alpha-5) = 2\alpha-10$$ 
$\therefore a=2$ 
따라서 $P(x) = 2(x-1)(x-3)(x-5)$이므로 
$P(4) = -6$ 

[다른 풀이] 
조건 (가)에서 
$$(x-1)P(x-2) = (x-7)P(x)$$ 
$\cdots \cdots$ ㉠ 
조건 (나)에서 삼차다항식 $P(x)$를 $x^{2}-4x+2$로 나누었을 때의 몫을 일차식 $Q(x)$라 하면 나머지는 $2x-10$이므로 
$$P(x) = (x^{2}-4x+2)Q(x) + 2x-10$$ 
따라서 
$$(x-1)P(x-2) = (x-1) [ \{(x-2)^{2}-4(x-2)+2\}Q(x-2) + 2(x-2)-10 ]$$ 
$$= (x-1) \{ (x^{2}-8x+14)Q(x-2) + 2(x-7) \}$$ 
이고, 
$$(x-7)P(x) = (x-7) \{ (x^{2}-4x+2)Q(x) + 2(x-5) \}$$ 
이므로 ㉠에서 
$$(x-1) \{ (x^{2}-8x+14)Q(x-2) + 2(x-7) \} = (x-7) \{ (x^{2}-4x+2)Q(x) + 2(x-5) \}$$ 
따라서 좌변은 $x-7$을 인수로 가져야 하므로 $Q(x-2)$가 $x-7$을 인수로 가진다. 삼차다항식 $P(x)$의 최고차항의 계수를 $a$라 하면 
$$Q(x-2) = a(x-7)$$ 
$\therefore Q(x) = a(x-5)$ 
따라서 우변은 
$$(x-7) \{ a(x^{2}-4x+2)(x-5) + 2(x-5) \}$$ 
이고, 마찬가지로 $x-1$을 인수로 가져야 하므로 
$$(1-7) \{ a(1^{2}-4+2)(1-5) + 2(1-5) \} = 0$$ 
$\therefore a=2$ 
따라서 $P(x) = 2(x^{2}-4x+2)(x-5) + 2(x-5)$이므로 
$P(4) = -6$
