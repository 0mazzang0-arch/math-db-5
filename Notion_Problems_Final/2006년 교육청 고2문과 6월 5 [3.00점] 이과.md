## 문제
세 수 $A = 2^{100}$, $B = 3^{70}$, $C = 5^{40}$의 대소 관계를 옳게 나타낸 것은?

## 보기
① $A < B < C$   ② $A < C < B$   ③ $B < C < A$   ④ $C < A < B$   ⑤ $C < B < A$

## 풀이
$A = 2^{100} = (2^{10})^{10} = 1024^{10}$
$B = 3^{70} = (3^{7})^{10} = 2187^{10}$
$C = 5^{40} = (5^{4})^{10} = 625^{10}$
따라서 $C < A < B$이다.

**[다른 풀이]**
$A$, $B$, $C$에 각각 상용로그를 취하면
$\log A = \log 2^{100} = 100 \log 2 = 30.1$
$\log B = \log 3^{70} = 70 \log 3 = 33.397$
$\log C = \log 5^{40} = 40 \log 5 = 27.96$
따라서 $\log C < \log A < \log B$이므로 $C < A < B$이다.

정답: ④
