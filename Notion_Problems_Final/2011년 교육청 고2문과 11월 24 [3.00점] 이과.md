## 문제
로그방정식 $\log_{3} ( \log_{5} 8 ) + \log_{3} ( \log_{2} x ) = 2$의 해를 $\alpha$라 할 때, $\alpha$의 값을 구하시오.

## 보기

## 풀이
$\log_{3} ( \log_{5} 8 ) + \log_{3} ( \log_{2} x ) = \log_{3} 9$
$\Leftrightarrow ( \log_{5} 8 ) ( \log_{2} x ) = 9$
$\log_{2} x = 9 \log_{8} 5 = 3 \log_{2} 5$
따라서 $\alpha = 5^{3} = 125$

정답: $125$
