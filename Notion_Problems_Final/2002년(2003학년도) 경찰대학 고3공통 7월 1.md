## 문제
다음 중에서 조건 $p$, $q$, $r$ 에 대한 설명으로 옳은 것은? (단, $x$, $y$ 는 실수)
$$
\begin{array}{l}
p : xy \neq 0 \\
q : x^{2} + y^{2} \neq 0 \\
r : xy > 0
\end{array}
$$

## 보기
① $p$ 는 $q$ 이기 위한 필요조건   ② $p$ 는 $r$ 이기 위한 충분조건   ③ $r$ 는 $p$ 이기 위한 충분조건   ④ $p$ 는 $q$ 이기 위한 필요충분조건   ⑤ $q$ 는 $r$ 이기 위한 필요충분조건

## 풀이
[정답] ③ 

명제 $p$, $q$, $r$ 의 진리집합을 $P$, $Q$, $R$ 이라 하면 
$P = \{ (x, y) | x \neq 0 \text{ 그리고 } y \neq 0 \}$ 
$Q = \{ (x, y) | x \neq 0 \text{ 또는 } y \neq 0 \}$ 
$R = \{ (x, y) | x > 0 \text{ 그리고 } y > 0 \} \cup \{ (x, y) | x < 0 \text{ 그리고 } y < 0 \}$ 

진리집합의 포함관계는 $R \subset P \subset Q$ 이다. 
따라서, 「 $r$ 는 $p$ 이기 위한 충분조건이다 」 가 참이다.
