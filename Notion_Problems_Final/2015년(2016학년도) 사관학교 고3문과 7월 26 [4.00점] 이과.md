## 문제
수직선 위의 원점에 있는 두 점 $A$, $B$를 다음의 규칙에 따라 이동시킨다.

$$
\begin{array}{|l|}
\hline
(\text{가}) \text{ 주사위를 던져 } 5 \text{ 이상의 눈이 나오면 } A\text{를 양의 방향으로 } 2\text{만큼, } B\text{를 음의 방향으로 } 1\text{만큼 이동시킨다.} \\
(\text{나}) \text{ 주사위를 던져 } 4 \text{ 이하의 눈이 나오면 } A\text{를 음의 방향으로 } 2\text{만큼, } B\text{를 양의 방향으로 } 1\text{만큼 이동시킨다.} \\
\hline
\end{array}
$$

주사위를 $5$번 던지고 난 후 두 점 $A$, $B$ 사이의 거리가 $3$ 이하가 될 확률이 $\frac{q}{p}$일 때, $p+q$의 값을 구하시오. (단, $p$와 $q$는 서로소인 자연수이다.)

## 풀이
$5$ 이상의 눈이 나오는 횟수를 $X$라 하면 $4$ 이하의 눈이 나오는 횟수는 $5-X$이므로 점 $A$의 위치는 $2X-2(5-X) = 4X-10$이고 점 $B$의 위치는 $-X+(5-X) = 5-2X$이다. 
이때, 두 점 $A$, $B$ 사이의 거리는 
$$|(4X-10)-(5-2X)| = |6X-15|$$
이다.

$|6X-15| \le 3$에서 $-3 \le 6X-15 \le 3$, 즉 $2 \le X \le 3$이다.
주사위를 던져 $5$ 이상의 눈이 나올 확률은 $\frac{2}{6} = \frac{1}{3}$이므로 확률변수 $X$는 이항분포 $B(5, \frac{1}{3})$를 따른다.
따라서 두 점 $A$, $B$ 사이의 거리가 $3$ 이하가 될 확률은 
$P(2 \le X \le 3) = P(X=2) + P(X=3)$
$= {}_{5}C_{2} \left( \frac{1}{3} \right)^2 \left( \frac{2}{3} \right)^3 + {}_{5}C_{3} \left( \frac{1}{3} \right)^3 \left( \frac{2}{3} \right)^2$
$= \frac{80}{243} + \frac{40}{243} = \frac{120}{243} = \frac{40}{81}$이다.

따라서 $p=81$, $q=40$이므로
$p+q = 81+40 = 121$

[정답] $121$
