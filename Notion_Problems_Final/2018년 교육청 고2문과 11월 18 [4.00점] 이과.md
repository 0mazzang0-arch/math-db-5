## 문제
다음은 $\sum_{k=1}^{14} \log_{2} \{ \log_{k+1} ( k+2 ) \}$의 값을 구하는 과정이다.

자연수 $n$에 대하여 
$\log_{n+1} ( n+2 ) = \frac{(가)}{\log_{2} ( n+1 )}$이므로 
$\sum_{k=1}^{14} \log_{2} \{ \log_{k+1} ( k+2 ) \} = \log_{2} \left( \frac{(나)}{\log_{2} 2} \right)$
따라서 
$\sum_{k=1}^{14} \log_{2} \{ \log_{k+1} ( k+2 ) \} = (다)$

위의 (가)에 알맞은 식을 $f(n)$이라 하고, (나), (다)에 알맞은 수를 각각 $p$, $q$라 할 때, $f(p+q)$의 값은?

## 보기
① $3$   ② $4$   ③ $5$   ④ $6$   ⑤ $7$

## 풀이
자연수 $n$에 대하여 
$\log_{n+1} ( n+2 ) = \frac{\log_{2} ( n+2 )}{\log_{2} ( n+1 )}$이므로 

$\sum_{k=1}^{14} \log_{2} \{ \log_{k+1} ( k+2 ) \}$
$= \log_{2} \left( \frac{\log_{2} 3}{\log_{2} 2} \right) + \log_{2} \left( \frac{\log_{2} 4}{\log_{2} 3} \right) + \dots + \log_{2} \left( \frac{\log_{2} 16}{\log_{2} 15} \right)$
$= \log_{2} \left( \frac{\log_{2} 3}{\log_{2} 2} \times \frac{\log_{2} 4}{\log_{2} 3} \times \dots \times \frac{\log_{2} 16}{\log_{2} 15} \right)$
$= \log_{2} \left( \frac{\log_{2} 16}{\log_{2} 2} \right) = \log_{2} 4 = 2$

$\therefore f(n) = \log_{2} ( n+2 )$, 
$p = \log_{2} 16 = 4$, 
$q = 2$

따라서 
$f(p+q) = f(4+2) = f(6) = \log_{2} ( 6+2 ) = \log_{2} 8 = 3$

정답: ①
