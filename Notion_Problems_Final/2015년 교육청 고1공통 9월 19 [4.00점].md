## 문제
이차 이상의 다항식 $f(x)$를 $(x-a)(x-b)$로 나눈 나머지를 $R(x)$라 할 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?
(단, $a$, $b$는 서로 다른 $2$개의 실수이다.)

## 보기
$$ \begin{array}{|l|} \hline \text{ㄱ. } f(a) - R(a) = 0 \\ \text{ㄴ. } f(a) - R(b) = f(b) - R(a) \\ \text{ㄷ. } af(b) - bf(a) = (a - b)R(0) \\ \hline \end{array} $$
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄷ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
$f(x)$를 $(x-a)(x-b)$로 나누었을 때의 몫을 $Q(x)$라 하면 나머지는 $R(x)$이므로
$f(x) = (x-a)(x-b)Q(x) + R(x)$ $\cdots$ ㉠

$ㄱ$. ㉠의 양변에 $x = a$를 대입하면 
$f(a) = R(a)$ 
$\therefore f(a) - R(a) = 0$

$ㄴ$. [반례] $f(x) = (x-a)(x-b) + x$라 하면 $R(x) = x$이고 
$f(a) - R(b) = a - b$, $f(b) - R(a) = b - a$
$a \neq b$이므로 
$f(a) - R(b) \neq f(b) - R(a)$

$ㄷ$. $R(x) = px + q$ ($p, q$는 상수)라 하면 
$f(a) = pa + q$, $f(b) = pb + q$
$af(b) - bf(a) = a(pb + q) - b(pa + q) = abp + aq - abp - bq = (a - b)q$
$R(0) = q$이므로 
$af(b) - bf(a) = (a - b)R(0)$

이상에서 옳은 것은 $ㄱ, ㄷ$이다.
