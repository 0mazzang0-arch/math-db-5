## 문제
두 상수 $a$ 와 $b$ 에 대하여 부등식 $x^{2}+a x+b \le 0$ 의 해가 $-1 \le x \le 3$ 일 때, 부등식 $x^{2}-a x+b \le 0$ 의 해는?

## 보기
① $-3 \le x \le -1$   ② $-2 \le x \le 2$   ③ $-3 \le x \le 1$   ④ $-1 \le x \le 2$   ⑤ $1 \le x \le 3$

## 풀이
해가 $-1 \le x \le 3$ 이고, $x^{2}$ 의 계수가 $1$ 인 이차부등식은 
$(x+1)(x-3) \le 0$
$x^{2}-2x-3 \le 0$
이므로 $a=-2$, $b=-3$ 이다.
따라서 부등식 $x^{2}-ax+b \le 0$ 에 $a$, $b$ 의 값을 대입하면
$x^{2}-(-2)x+(-3) \le 0$
$x^{2}+2x-3 \le 0$
$(x+3)(x-1) \le 0$
$\therefore -3 \le x \le 1$

정답: ③
