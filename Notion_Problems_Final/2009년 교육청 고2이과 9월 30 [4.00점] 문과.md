## 문제
다음 두 조건을 만족시키는 모든 정수 $r$ 의 값의 합을 구하시오.

(가) 무한급수 $\sum_{n=1}^{\infty} \left( \frac{r-5}{8} \right)^n$ 이 수렴한다.
(나) $\lim_{n \to \infty} \frac{r^{n+1} - 7^n + 2}{r^n + 7^{n+1} + 2^{n-1}} = -\frac{1}{7}$

## 보기
(본 문제는 주관식으로 보기가 없습니다.)

## 풀이
무한급수 $\sum_{n=1}^{\infty} \left( \frac{r-5}{8} \right)^n$ 이 수렴하려면 
$-1 < \frac{r-5}{8} < 1$ 이어야 하므로
$-8 < r-5 < 8$
$-3 < r < 13$

조건 (나)의 극한값을 구하기 위해 범위를 나누어 조사하면 다음과 같다.

(ⅰ) $-3 < r < 7$ 일 때,
$\lim_{n \to \infty} \frac{r^{n+1} - 7^n + 2}{r^n + 7^{n+1} + 2^{n-1}} = \lim_{n \to \infty} \frac{r \left( \frac{r}{7} \right)^n - 1 + \frac{2}{7^n}}{\left( \frac{r}{7} \right)^n + 7 + \frac{1}{2} \left( \frac{2}{7} \right)^n} = -\frac{1}{7}$
이는 조건 (나)를 만족시킨다.

(ⅱ) $r = 7$ 일 때,
$\lim_{n \to \infty} \frac{7^{n+1} - 7^n + 2}{7^n + 7^{n+1} + 2^{n-1}} = \lim_{n \to \infty} \frac{6 \times 7^n + 2}{8 \times 7^n + \frac{1}{2} \times 2^n} = \frac{6}{8} = \frac{3}{4}$
이는 조건 (나)를 만족시키지 않는다.

(ⅲ) $7 < r < 13$ 일 때,
$\lim_{n \to \infty} \frac{r - \left( \frac{7}{r} \right)^n + \frac{2}{r^n}}{1 + 7 \left( \frac{7}{r} \right)^n + \frac{1}{2} \left( \frac{2}{r} \right)^n} = r$
이는 조건 (나)를 만족시키지 않는다.

따라서 (ⅰ), (ⅱ), (ⅲ)에서 구하는 $r$ 의 범위는 $-3 < r < 7$ 이다.
이를 만족시키는 모든 정수 $r$ 의 값은 $-2, -1, 0, 1, 2, 3, 4, 5, 6$ 이다.
따라서 모든 정수 $r$ 의 값의 합은
$(-2) + (-1) + 0 + 1 + 2 + 3 + 4 + 5 + 6 = 18$ 이다.
