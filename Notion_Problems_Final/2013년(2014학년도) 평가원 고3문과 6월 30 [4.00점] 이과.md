## 문제
자연수 $k$에 대하여 $\log k$의 정수 부분과 소수 부분을 각각 $x$ 좌표와 $y$ 좌표로 갖는 점을 $P_{k}$라 하자. 다음 조건을 만족시키는 자연수 $m$, $n$의 모든 순서쌍 ($m$, $n$)의 개수를 구하시오.

(가) $1 \le m < n < 100$
(나) $\overline{P_{m} P_{n}} = \sqrt{1 + (\log 2)^{2}}$

## 풀이
자연수 $m$의 정수 부분과 소수 부분을 각각 $n_{1}$, $\alpha_{1}$, 자연수 $n$의 정수 부분과 소수 부분을 각각 $n_{2}$, $\alpha_{2}$라 하면 점의 좌표는 $P_{m}(n_{1}, \alpha_{1})$, $P_{n}(n_{2}, \alpha_{2})$이다.

조건 (나)에서 $\overline{P_{m} P_{n}} = \sqrt{1 + (\log 2)^{2}}$이므로
$\overline{P_{m} P_{n}}^{2} = 1 + (\log 2)^{2}$
$(n_{2} - n_{1})^{2} + (\alpha_{2} - \alpha_{1})^{2} = 1 + (\log 2)^{2}$
이 때, $n_{1}$, $n_{2}$은 정수이고 $0 \le \alpha_{1} < 1$, $0 \le \alpha_{2} < 1$이므로
$|n_{2} - n_{1}| = 1$, $|\alpha_{2} - \alpha_{1}| = \log 2$이다.
한편, $m < n$이므로 $n_{2} = n_{1} + 1$이다.

($i$) $\alpha_{2} > \alpha_{1}$일 때,
$\alpha_{2} = \alpha_{1} + \log 2$이므로
$\log m = n_{1} + \alpha_{1}$ $\cdots$ ㉠
$\log n = n_{1} + 1 + \alpha_{1} + \log 2$ $\cdots$ ㉡
㉡에서 ㉠을 변변 빼면
$\log \frac{n}{m} = 1 + \log 2 = \log 20$
$\therefore \frac{n}{m} = 20$, 즉 $n = 20m$
이 조건을 만족하는 순서쌍 ($m$, $n$)은 ($1$, $20$), ($2$, $40$), ($3$, $60$), ($4$, $80$)로 $4$개이다.

($ii$) $\alpha_{2} < \alpha_{1}$일 때,
$\alpha_{2} = \alpha_{1} - \log 2$이므로
$\log m = n_{1} + \alpha_{1}$ $\cdots$ ㉢
$\log n = n_{1} + 1 + \alpha_{1} - \log 2$ $\cdots$ ㉣
㉣에서 ㉢을 변변 빼면
$\log \frac{n}{m} = 1 - \log 2 = \log 5$
$\therefore \frac{n}{m} = 5$, 즉 $n = 5m$
이 조건을 만족하는 순서쌍 ($m$, $n$)은 ($2$, $10$), ($3$, $15$), ($4$, $20$), ($5$, $25$), ($6$, $30$), ($7$, $35$), ($8$, $40$), ($9$, $45$)로 $8$개이다.

따라서 만족하는 모든 순서쌍 ($m$, $n$)의 개수는 $4 + 8 = 12$개이다.

[정답] $12$
