## 문제
$\lim _{n \to \infty} \frac{1}{\sqrt{n^{2} + n + 1} - \sqrt{n^{2} - n + 1}}$ 의 값은?

## 보기
① $1$   ② $\frac{3}{2}$   ③ $2$   ④ $\frac{5}{2}$   ⑤ $3$

## 풀이
$$ \lim _{n \to \infty} \frac{1}{\sqrt{n^{2} + n + 1} - \sqrt{n^{2} - n + 1}} $$
$$ = \lim _{n \to \infty} \frac{\sqrt{n^{2} + n + 1} + \sqrt{n^{2} - n + 1}}{(\sqrt{n^{2} + n + 1} - \sqrt{n^{2} - n + 1})(\sqrt{n^{2} + n + 1} + \sqrt{n^{2} - n + 1})} $$
$$ = \lim _{n \to \infty} \frac{\sqrt{n^{2} + n + 1} + \sqrt{n^{2} - n + 1}}{(n^{2} + n + 1) - (n^{2} - n + 1)} $$
$$ = \lim _{n \to \infty} \frac{\sqrt{n^{2} + n + 1} + \sqrt{n^{2} - n + 1}}{2n} $$
$$ = \lim _{n \to \infty} \frac{\sqrt{1 + \frac{1}{n} + \frac{1}{n^{2}}} + \sqrt{1 - \frac{1}{n} + \frac{1}{n^{2}}}}{2} = \frac{1 + 1}{2} = 1 $$

정답: ①
