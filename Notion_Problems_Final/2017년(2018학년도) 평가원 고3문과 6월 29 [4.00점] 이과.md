## 문제

공차가 $0$이 아닌 등차수열 $\{a_n\}$이 있다. 수열 $\{b_n\}$은 $b_1 = a_1$이고, $2$ 이상의 자연수 $n$에 대하여
$$ b_n = \begin{cases} b_{n-1} + a_n & (n\text{이 } 3\text{의 배수가 아닌 경우}) \\ b_{n-1} - a_n & (n\text{이 } 3\text{의 배수인 경우}) \end{cases} $$
이다. $b_{10} = a_{10}$일 때, $\frac{b_8}{b_{10}} = \frac{q}{p}$이다. $p+q$의 값을 구하시오. (단, $p$와 $q$는 서로소인 자연수이다.)

## 풀이

등차수열 $\{a_n\}$의 공차를 $d$ ($d \neq 0$), 첫째항부터 제 $n$항까지의 합을 $S_n$이라 하자.

$b_1 = a_1 = S_1$
$b_2 = b_1 + a_2 = S_2$
$b_3 = b_2 - a_3 = S_3 - 2a_3$
$b_4 = b_3 + a_4 = S_4 - 2a_3$
$b_5 = b_4 + a_5 = S_5 - 2a_3$
$b_6 = b_5 - a_6 = S_6 - 2(a_3 + a_6)$
$b_7 = b_6 + a_7 = S_7 - 2(a_3 + a_6)$
$b_8 = b_7 + a_8 = S_8 - 2(a_3 + a_6)$
$b_9 = b_8 - a_9 = S_9 - 2(a_3 + a_6 + a_9)$ $\cdots$ ㉠

한편 $b_{10} = b_9 + a_{10}$이고 $b_{10} = a_{10}$이므로
$b_9 = 0$ $\cdots$ ㉡

㉠, ㉡에서
$0 = S_9 - 2(a_3 + a_6 + a_9)$, 즉 $S_9 = 2(a_3 + a_6 + a_9)$이므로
$\frac{9(2a_1 + 8d)}{2} = 2(3a_1 + 15d)$
$3(a_1 + 4d) = 2(a_1 + 5d)$
$a_1 = -2d$

또한 $b_9 = b_8 - a_9$이므로 $b_8 = a_9$이다.
따라서
$\frac{b_8}{b_{10}} = \frac{a_9}{a_{10}} = \frac{a_1 + 8d}{a_1 + 9d} = \frac{-2d + 8d}{-2d + 9d} = \frac{6}{7}$
이므로 $p = 7$, $q = 6$이고
$p + q = 7 + 6 = 13$

정답: $13$
