## 문제
양수 $x$에 대하여 $\log x$의 소수부분을 $f(x)$라 할 때, 
$f(2x) \leq f(x)$
를 만족시키는 $100$보다 작은 자연수 $x$의 개수는?

## 보기
① $55$   ② $57$   ③ $59$   ④ $61$   ⑤ $63$

## 풀이
[정답] ①

(i) $1 \leq x < 5$ 일 때, $2 \leq 2x < 10$ 이므로 
$f(x) = \log x$, 
$f(2x) = \log 2x = \log 2 + \log x$ 
$f(2x) > f(x)$ 이므로 주어진 부등식을 만족하는 $x$ 는 존재하지 않는다. 

(ii) $5 \leq x < 10$ 일 때, $10 \leq 2x < 20$ 이므로 
$f(x) = \log x$ 
$f(2x) = -1 + \log 2x = -1 + \log 2 + \log x$ 
$f(2x) \leq f(x)$ 에서 
$-1 + \log 2 + \log x \leq \log x$ 
$\therefore \log 2 < 1$ 
따라서 주어진 부등식은 항상 성립한다. 

(iii) $10 \leq x < 50$ 일 때, $20 \leq 2x < 100$ 이므로 
$f(x) = -1 + \log x$ 
$f(2x) = -1 + \log 2x = -1 + \log 2 + \log x$ 
따라서 $f(2x) > f(x)$ 이므로 주어진 부등식을 만족하는 $x$ 는 존재하지 않는다. 

(iv) $50 \leq x < 100$ 일 때, $100 \leq 2x < 200$ 이므로 
$f(x) = -1 + \log x$ 
$f(2x) = -2 + \log 2x = -2 + \log 2 + \log x$ 
$f(2x) \leq f(x)$ 에서 
$-2 + \log 2 + \log x \leq -1 + \log x$ 
$\therefore \log 2 < 1$ 
따라서 주어진 부등식은 항상 성립한다. 

위의 (i) $\sim$ (iv)에서 $f(2x) \leq f(x)$ 를 만족하는 자연수는 $5, 6, 7, 8, 9$ 와 $50, 51, 52, \cdots, 99$ 로 모두 $55$ 개다.
