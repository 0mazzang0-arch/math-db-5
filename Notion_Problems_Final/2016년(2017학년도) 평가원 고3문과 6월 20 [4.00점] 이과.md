## 문제
첫째항이 $a$인 수열 $\{ a_{n} \}$은 모든 자연수 $n$에 대하여 

$$ a_{n+1} = \begin{cases} a_{n} + ( -1 )^{n} \times 2 & ( n \text{이 } 3 \text{의 배수가 아닌 경우} ) \\ a_{n} + 1 & ( n \text{이 } 3 \text{의 배수인 경우} ) \end{cases} $$

를 만족시킨다. $a_{15} = 43$일 때, $a$의 값은?

## 보기
① $35$   ② $36$   ③ $37$   ④ $38$   ⑤ $39$

## 풀이
$a_{1} = a$ 
$a_{2} = a + ( -1 )^{1} \times 2 = a - 2$ 
$a_{3} = ( a - 2 ) + ( -1 )^{2} \times 2 = a$ 
$a_{4} = a + 1$ 
$a_{5} = ( a + 1 ) + ( -1 )^{4} \times 2 = a + 3$ 
$a_{6} = ( a + 3 ) + ( -1 )^{5} \times 2 = a + 1$ 
$a_{7} = ( a + 1 ) + 1 = a + 2$ 
$\vdots$ 
이므로 
$a_{15} = a + 4$ 
따라서 $a + 4 = 43$에서 
$a = 39$이다.

정답: ⑤
