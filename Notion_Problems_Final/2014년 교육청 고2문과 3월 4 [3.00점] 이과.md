## 문제
두 조건 $p: x^{2} \leq 9$, $q: x \leq a$에 대하여 명제 $p \rightarrow q$가 참이 되도록 하는 실수 $a$의 최솟값은?

## 보기
① $1$  ② $2$  ③ $3$  ④ $4$  ⑤ $5$

## 풀이
조건 $p: x^{2} \leq 9$에서
$(x+3)(x-3) \leq 0$
$\therefore -3 \leq x \leq 3$

두 조건 $p$, $q$의 진리집합은 각각
$P = \{ x | -3 \leq x \leq 3 \}$
$Q = \{ x | x \leq a \}$
이므로 명제 $p \rightarrow q$가 참이 되려면
$P \subset Q$

> [!example] 📸 (그림/그래프)

따라서 $a \geq 3$이므로 실수 $a$의 최솟값은 $3$이다.
