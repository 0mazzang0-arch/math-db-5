## 문제
연립부등식 
$$ \left\{ \begin{array}{l} x^{4}-50 x^{2}+49 \leq 0 \\ \frac{(x-5)(x+1)}{x-3} \geq 0 \end{array} \right. $$
를 만족시키는 모든 정수 $x$ 의 합을 구하시오.

## 풀이
부등식 $x^{4}-50 x^{2}+49 \leq 0$ 을 풀면, 
$(x+7)(x+1)(x-1)(x-7) \leq 0$ 이므로 
$-7 \leq x \leq -1$ 또는 $1 \leq x \leq 7$ 이다. 

부등식 $\frac{(x-5)(x+1)}{x-3} \geq 0$ 을 풀면, 
$-1 \leq x < 3$ 또는 $x \geq 5$ 이다. 

그러므로 두 부등식을 모두 만족하는 범위는 
$x = -1$ 또는 $1 \leq x < 3$ 또는 $5 \leq x \leq 7$ 이므로 
정수 $x = -1$, $1$, $2$, $5$, $6$, $7$ 이다. 

따라서, 모든 정수 $x$ 의 합은 $20$ 이다.
