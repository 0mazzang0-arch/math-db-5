## 문제
두 실수 $x$, $y$에 대한 연립방정식 
$$ \left\{ \begin{array}{l} 3^{x} = 9^{y} \\ ( \log_{2} 8x ) ( \log_{2} 4y ) = -1 \end{array} \right. $$
의 해를 $x = \alpha$, $y = \beta$라 할 때, $\frac{1}{\alpha\beta}$의 값을 구하시오.

## 보기

## 풀이
$3^{x} = 3^{2y}$에서 
$x = 2y$ $\cdots$ ㉠ 

$( \log_{2} 8x ) ( \log_{2} 4y ) = -1$에서 
$( 3 + \log_{2} x ) ( 2 + \log_{2} y ) = -1$ $\cdots$ ㉡ 

㉠을 ㉡에 대입하여 정리하면 
$( \log_{2} 2y + 3 ) ( 2 + \log_{2} y ) = -1$
$( 1 + \log_{2} y + 3 ) ( 2 + \log_{2} y ) = -1$
$( \log_{2} y + 4 ) ( \log_{2} y + 2 ) = -1$
$( \log_{2} y )^{2} + 6 \log_{2} y + 8 = -1$
$( \log_{2} y + 3 )^{2} = 0$이므로 
$y = \frac{1}{8}$, $x = \frac{1}{4}$이다. 

따라서 $\alpha = \frac{1}{4}$, $\beta = \frac{1}{8}$이므로 
$\frac{1}{\alpha\beta} = 32$
