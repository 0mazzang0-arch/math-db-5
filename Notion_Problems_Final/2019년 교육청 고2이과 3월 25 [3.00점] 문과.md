## 문제
자연수 $n$에 대하여 자연수 전체 집합의 부분집합 $A_{n}$을 다음과 같이 정의하자.
$$A_{n} = \{ x \mid x \text{는 } \sqrt{n} \text{ 이하의 홀수} \}$$
$A_{n} \subseteq A_{25}$를 만족시키는 $n$의 최댓값을 구하시오.

## 보기

## 풀이
$\sqrt{25} = 5$이므로 
$A_{25} = \{ 1, 3, 5 \}$
$1 \le \sqrt{n} < 7$이면 $A_{n} \subseteq A_{25}$이므로 
$1 \le n < 49$
따라서 자연수 $n$의 최댓값은 $48$이다.

정답: $48$
