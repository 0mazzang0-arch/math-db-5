## 문제
$\tan 2 \alpha = \frac{5}{12}$ 일 때, $\tan \alpha = p$ 이다. $60p$ 의 값을 구하시오. (단, $0 < \alpha < \frac{\pi}{4}$ 이다.)

## 보기


## 풀이
$\tan 2 \alpha = \frac{2 \tan \alpha}{1 - \tan^{2} \alpha} = \frac{5}{12}$ 에서 $\tan \alpha = p$ 로 치환하면 

$24p = 5 - 5p^{2}$ 
$\Leftrightarrow 5p^{2} + 24p - 5 = 0$ 
$\therefore p = \frac{1}{5}, p = -5$ 

$0 < \alpha < \frac{\pi}{4}$ 이므로 
$\tan \alpha = \frac{1}{5}$ 
$\therefore 60p = 12$ 

$\tan 2 \alpha = \frac{5}{12}$ 일 때, $\tan \alpha = p$ 이다. $60p$ 의 값을 구하면 $12$ 이다.
