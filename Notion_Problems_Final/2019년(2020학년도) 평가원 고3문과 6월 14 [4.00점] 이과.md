## 문제
$( x^{2} - \frac{1}{x} ) ( x + \frac{a}{x^{2}} )^{4}$ 의 전개식에서 $x^{3}$ 의 계수가 $7$ 일 때, 상수 $a$ 의 값은?

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
[정답] ② 

$( x + \frac{a}{x^{2}} )^{4}$ 의 전개식의 일반항은 
$${}_{4}C_{r} x^{4-r} \left( \frac{a}{x^{2}} \right)^{r} = {}_{4}C_{r} a^{r} \frac{x^{4-r}}{x^{2r}} = {}_{4}C_{r} a^{r} x^{4-3r} \cdots ㉠$$ 

$( x^{2} - \frac{1}{x} ) ( x + \frac{a}{x^{2}} )^{4} = x^{2} ( x + \frac{a}{x^{2}} )^{4} - \frac{1}{x} ( x + \frac{a}{x^{2}} )^{4}$ 이므로 전개식에서 $x^{3}$ 의 계수는 다음과 같다. 

(ⅰ) $x^{2} ( x + \frac{a}{x^{2}} )^{4}$ 에서 $x^{3}$ 의 계수는 $㉠$에서 $x^{1}$ 의 계수를 구하면 되므로 $4-3r=1$, 즉 $r=1$ 일 때이다. 
$${}_{4}C_{1} a^{1} = 4a$$ 

(ⅱ) $\frac{1}{x} ( x + \frac{a}{x^{2}} )^{4}$ 에서 $x^{3}$ 의 계수는 $㉠$에서 $x^{4}$ 의 계수를 구하면 되므로 $4-3r=4$, 즉 $r=0$ 일 때이다. 
$${}_{4}C_{0} a^{0} = 1$$ 

따라서 (ⅰ), (ⅱ)에 의해 구하는 $x^{3}$ 의 계수는 $4a-1$ 이다. 
따라서 $4a - 1 = 7$ 
$\therefore a = 2$
