## 문제
최고차항의 계수가 $1$인 삼차함수 $f(x)$가 다음 조건을 만족시킬 때, $f(3)$의 값을 구하시오.

(가) 모든 실수 $x$에 대하여 $f(-x)=-f(x)$이다.  
(나) $\int_{-2}^{2} xf(x)dx = \frac{144}{5}$

## 풀이
최고차항의 계수가 $1$인 삼차함수 $f(x)$가 조건 (가)에 의하여 모든 실수 $x$에 대하여 $f(-x)=-f(x)$이므로  
$f(x) = x^3 + ax$ ($a$는 상수)  
라 하면 조건 (나)에서 $\int_{-2}^{2} xf(x)dx = \frac{144}{5}$이므로  

$\int_{-2}^{2} x(x^3 + ax)dx$  
$= \int_{-2}^{2} (x^4 + ax^2)dx$  
$= 2 \int_{0}^{2} (x^4 + ax^2)dx$  
$= 2 \left[ \frac{1}{5}x^5 + \frac{a}{3}x^3 \right]_0^2$  
$= 2 \left( \frac{32}{5} + \frac{8a}{3} \right)$  
$= \frac{192 + 80a}{15}$  

$\frac{192 + 80a}{15} = \frac{144}{5}$에서 $a = 3$  

$f(x) = x^3 + 3x$이므로  
$f(3) = 3^3 + 3 \times 3 = 36$
