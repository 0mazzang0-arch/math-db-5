## 문제
함수 $f(x) = x^3 + ax^2 + bx + 1$은 $x = -1$에서 극대이고, $x = 3$에서 극소이다. 함수 $f(x)$의 극댓값은? (단, $a$, $b$는 상수이다.)

## 보기
① $0$   ② $3$   ③ $6$   ④ $9$   ⑤ $12$

## 풀이
$f(x) = x^3 + ax^2 + bx + 1$에서
$f'(x) = 3x^2 + 2ax + b$
함수 $f(x)$가 $x = -1$, $x = 3$에서 극값을 가지므로, 이차방정식 $3x^2 + 2ax + b = 0$의 서로 다른 두 실근이 $-1$, $3$이다. 
따라서 이차방정식의 근과 계수의 관계에 의하여
$-1 + 3 = -\frac{2a}{3}$, $-1 \times 3 = \frac{b}{3}$
$\therefore a = -3$, $b = -9$
$f(x) = x^3 - 3x^2 - 9x + 1$이므로 극댓값은 $x = -1$일 때,
$f(-1) = (-1)^3 - 3(-1)^2 - 9(-1) + 1 = 6$
이다.
