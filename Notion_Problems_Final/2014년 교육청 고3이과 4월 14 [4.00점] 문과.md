## 문제
부등식 $\frac{(f \circ g)(x)}{g(x)} \leq 1$ 을 만족시키는 모든 정수 $x$ 의 개수는?

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
부등식 $\frac{(f \circ g)(x)}{g(x)} \leq 1$ 에서
$\frac{(f \circ g)(x) - g(x)}{g(x)} \leq 0$
$g(x) \{ f(g(x)) - g(x) \} \leq 0$ (단, $g(x) \neq 0$)

i) $g(x) > 0$ 일 때
$f(g(x)) \leq g(x)$
$g(x) = t$ ($t > 0$)라 하면
$f(t) \leq t$
$0 < t \leq 3$ ($\because t > 0$)
$0 < g(x) \leq 3$ 이므로
$-1 \leq x < 0$ 또는 $2 < x \leq 3$
$\therefore$ 정수 $x$ 는 $-1$, $3$

ii) $g(x) < 0$ 일 때
$f(g(x)) \geq g(x)$
$g(x) = t$ ($t < 0$)라 하면
$f(t) \geq t$
$-2 \leq t < 0$ ($\because t < 0$)
$-2 \leq g(x) < 0$ 이므로
$0 < x < 2$
$\therefore$ 정수 $x$ 는 $1$

따라서 만족하는 모든 정수 $x$ 는 $-1, 1, 3$ 으로 개수는 $3$ 이다.

정답: ③
