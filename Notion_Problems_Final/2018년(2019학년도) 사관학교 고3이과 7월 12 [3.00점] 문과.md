## 문제
실수 전체의 집합에서 미분가능한 함수 $f(x)$ 가 모든 실수 $x$ 에 대하여 
$$x f(x)=x^{2} e^{-x}+ \int_{1}^{x} f(t) d t$$ 
를 만족시킬 때, $f(2)$ 의 값은? 

## 보기
① $\frac{1}{e}$   ② $\frac{e+1}{e^{2}}$   ③ $\frac{e+2}{e^{2}}$   ④ $\frac{e+3}{e^{2}}$   ⑤ $\frac{e+4}{e^{2}}$

## 풀이
$$x f(x)=x^{2} e^{-x}+ \int_{1}^{x} f(t) d t$$ 
에 $x=1$ 을 대입하면 
$$f(1)=e^{-1}$$ 
양변을 $x$ 에 대하여 미분하면 
$$f(x)+x f'(x)=2 x e^{-x}-x^{2} e^{-x}+f(x)$$ 
$$x f'(x)=2 x e^{-x}-x^{2} e^{-x}$$ 
$$f'(x)=2 e^{-x}-x e^{-x}$$ 
$f(x)$ 는 $f'(x)$ 의 부정적분이므로 
$$f(x) = \int (2 e^{-x}-x e^{-x} ) d x$$ 
$$= -2 e^{-x} + x e^{-x} - \int e^{-x} d x$$ 
$$= -2 e^{-x} + x e^{-x} + e^{-x} + C = (x-1) e^{-x} + C$$ 
$f(1)=e^{-1}$ 이므로 
$$(1-1) e^{-1} + C = e^{-1} \implies C = e^{-1}$$ 
따라서 
$$f(x) = (x-1) e^{-x} + e^{-1}$$ 
이다. 
$$\therefore f(2) = (2-1) e^{-2} + e^{-1} = e^{-2} + e^{-1} = \frac{e+1}{e^{2}}$$

정답: ②
