## 문제
함수 $f(x) = 2\cos \left( x + \frac{\pi}{2} \right) + 3$의 최솟값은?

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
모든 실수 $x$에 대하여 
$-1 \le \cos x \le 1$ 이므로
$-1 \le \cos \left( x + \frac{\pi}{2} \right) \le 1$ 이다.
양변에 $2$를 곱하면
$-2 \le 2\cos \left( x + \frac{\pi}{2} \right) \le 2$
양변에 $3$을 더하면
$1 \le 2\cos \left( x + \frac{\pi}{2} \right) + 3 \le 5$ 이다.
따라서 함수 $f(x) = 2\cos \left( x + \frac{\pi}{2} \right) + 3$의 최솟값은 $1$이다.

정답: ①
