## 문제
좌표공간에서 세 직선 $x = -y = \frac{z}{2}$, $x = y = \frac{z}{2a}$, $x = -\frac{y}{2} = \frac{z}{a}$가 같은 평면 위에 있을 때, $20a$의 값을 구하시오. (단, $a \neq 0$이다.)

## 보기

## 풀이
세 직선을 
$l : x = -y = \frac{z}{2}$
$m : x = y = \frac{z}{2a}$
$n : x = -\frac{y}{2} = \frac{z}{a}$
라 하고 세 직선이 모두 점 $(0, 0, 0)$을 지나므로 세 직선을 포함하는 평면의 방정식을 $\alpha x + \beta y + \gamma z = 0$으로 놓을 수 있다. 세 직선이 같은 평면에 있으므로 세 직선의 방향벡터와 평면의 법선벡터는 수직이다.

$\alpha - \beta + 2\gamma = 0 \cdots$ ㉠
$\alpha + \beta + 2a\gamma = 0 \cdots$ ㉡
$\alpha - 2\beta + a\gamma = 0 \cdots$ ㉢

㉡ $-$ ㉢을 하면 $3\beta + a\gamma = 0$
$\therefore \beta = -\frac{1}{3}a\gamma \cdots$ ㉣

㉡ $\times 2 +$ ㉢을 하면 $3\alpha + 5a\gamma = 0$
$\therefore \alpha = -\frac{5}{3}a\gamma \cdots$ ㉤

㉣, ㉤을 ㉠에 대입하면
$-\frac{5}{3}a\gamma + \frac{1}{3}a\gamma + 2\gamma = 0$
$-\frac{4}{3}a = -2$
$\therefore a = \frac{3}{2}$
$\therefore 20a = 30$
