## 문제
사차함수 $f(x) = \frac{1}{4}x^{4} + \frac{1}{3}(a+1)x^{3} - ax$가 $x = \alpha$, $\gamma$에서 극소, $x = \beta$에서 극대일 때, 실수 $a$의 값의 범위는? (단, $\alpha < 0 < \beta < \gamma < 3$)

## 보기
① $-\frac{9}{2} < a < -4$   ② $-4 < a < -\frac{7}{2}$   ③ $-\frac{7}{2} < a < -3$   ④ $-3 < a < -\frac{5}{2}$   ⑤ $-\frac{5}{2} < a < -2$

## 풀이
$f'(x) = x^{3} + (a+1)x^{2} - a = (x+1)(x^{2} + ax - a) = 0$ 의 서로 다른 세 실근이 $\alpha$, $\beta$, $\gamma$이다.
따라서 $\alpha = -1$이고, $x^{2} + ax - a = 0$의 서로 다른 두 실근이 $\beta$, $\gamma$이다.
$g(x) = x^{2} + ax - a$라 하면 $0 < \beta < \gamma < 3$이므로
(1) $D = a^{2} + 4a > 0$ 에서 $a < -4$ 또는 $a > 0$
(2) 대칭축: $0 < -\frac{a}{2} < 3$ 에서 $-6 < a < 0$
(3) $g(0) = -a > 0$ 에서 $a < 0$
(4) $g(3) = 9 + 3a - a = 2a + 9 > 0$ 에서 $a > -\frac{9}{2}$
이여야 하므로 만족하는 $a$의 범위는 $-\frac{9}{2} < a < -4$이다.

정답: ①
