## 문제
전체집합 $U = \{ x | x$는 $9$ 이하의 자연수 $\}$의 두 부분집합 $A$, $B$에 대하여 $A \cap B = \{ 1, 2 \}$, $A^{c} \cap B = \{ 3, 4, 5 \}$, $A^{c} \cap B^{c} = \{ 8, 9 \}$일 때, 집합 $A$의 모든 원소의 합은?

## 보기
① $8$   ② $10$   ③ $12$   ④ $14$   ⑤ $16$

## 풀이
전체집합 $U = \{ 1, 2, 3, 4, 5, 6, 7, 8, 9 \}$이고, 주어진 조건에 따라 각 영역에 속하는 원소를 구하면 다음과 같다.
$A \cap B = \{ 1, 2 \}$
$A^{c} \cap B = B - A = \{ 3, 4, 5 \}$
$A^{c} \cap B^{c} = (A \cup B)^{c} = \{ 8, 9 \}$

주어진 집합들을 벤다이어그램으로 나타내면 다음과 같다. 

> [!example] 📸 (그림/그래프)

이때 $A \cup B = U - (A \cup B)^{c} = \{ 1, 2, 3, 4, 5, 6, 7 \}$이므로
$A = (A \cup B) - (B - A) = \{ 1, 2, 3, 4, 5, 6, 7 \} - \{ 3, 4, 5 \} = \{ 1, 2, 6, 7 \}$이다.

따라서 집합 $A$의 모든 원소의 합은
$1 + 2 + 6 + 7 = 16$

정답: ⑤
