## 문제
두 실수 $a$, $b$에 대하여 함수 
$$f(x) = \lim_{n \to \infty} \frac{-a|x| - |x|^{n} + b}{|x|^{n} + 1}$$ 
가 모든 실수 $x$에서 연속일 때, <보기>에서 옳은 것만을 있는 대로 고른 것은? 

ㄱ. $a - b = 1$
ㄴ. 함수 $f(x)$의 최솟값은 $-1$이다.
ㄷ. $a < 1$일 때, 함수 $f(x)$의 그래프는 $x$축과 만나지 않는다.



> [!example] 📸 (그림/그래프)

## 보기
① $ㄴ$   ② $ㄱ, ㄴ$   ③ $ㄱ, ㄷ$   ④ $ㄱ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
정답: ④ 

함수 $f(x)$를 구간별로 정의하면 다음과 같다.
$$f(x) = \begin{cases} -a|x| + b & (|x| < 1) \\ \frac{-a-1+b}{2} & (|x| = 1) \\ -1 & (|x| > 1) \end{cases}$$

ㄱ. 함수 $f(x)$가 모든 실수 $x$에서 연속이므로 $x = 1$에서 연속이어야 한다.
따라서 $\lim_{x \to 1} f(x) = f(1)$이 성립해야 한다.
$\lim_{x \to 1^{-}} (-a|x| + b) = -a + b$
$\lim_{x \to 1^{+}} (-1) = -1$
즉, $-a + b = -1$ 이므로
$\therefore a - b = 1$ (참)

ㄴ. [반례] $a = -1$, $b = -2$일 때, $a - b = 1$을 만족한다.
이때 $|x| < 1$에서 $f(x) = |x| - 2$이므로, $x = 0$일 때 함수 $f(x)$의 최솟값은 $-2$이다. (거짓)

ㄷ. $a < 1$일 때, $a - b = 1$에서 $b = a - 1 < 0$이다.
$|x| < 1$에서 $f(x) = -a|x| + b$의 그래프를 고려하면:
$1)$ $a \ge 0$인 경우: $f(x) = -a|x| + b \le b < 0$이므로 $x$축과 만나지 않는다.
$2)$ $a < 0$인 경우: $f(x)$는 $|x|$가 커질수록 증가하며, 최댓값은 $\lim_{|x| \to 1} (-a|x| + b) = -a + b = -1$이다.
따라서 모든 실수 $x$에 대하여 $f(x) \le -1$이므로 함수 $f(x)$의 그래프는 $x$축과 만나지 않는다. (참)

따라서 옳은 것은 ㄱ, ㄷ이다.
