## 문제
두 이차정사각행렬 $A$, $B$ 가 
$$A + B A = 2 E, A B + B A = - A + B$$
를 만족시킬 때, <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $E$ 는 단위행렬이다.)

---
**<보 기>**
ㄱ. $A^{-1}$ 이 존재한다. 
ㄴ. $(A + B)(A - B) = A^{2} - B^{2}$ 
ㄷ. $A + B = 4 E$

## 보기
① $ㄱ$  ② $ㄴ$  ③ $ㄱ, ㄴ$  ④ $ㄴ, ㄷ$  ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
ㄱ. $(E + B) A = 2 E$ 이므로 $A^{-1} = \frac{1}{2} (E + B)$ 이다. (참) 

ㄴ. $(E + B) A = A (E + B) = 2 E$ 이므로 $A + B A = A + A B$ 에서 $A B = B A$ 이다. 
$A B = B A$ 이면 $(A + B)(A - B) = A^{2} - A B + B A - B^{2} = A^{2} - B^{2}$ 이다. (참) 

ㄷ. $A B = B A$ 이므로 $A B + B A = 2 A B = - A + B$ 이다. 
$(E + B) A = 2 E$ 에서 $A B = 2 E - A$ 이므로 이를 대입하면 
$2 (2 E - A) = - A + B$ 
$4 E - 2 A = - A + B$ 
따라서 $A + B = 4 E$ 이다. (참) 

정답은 ⑤ 이다.
