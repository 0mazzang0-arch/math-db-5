## 문제
이차함수 $f(x)$가 $f(0) = 0$이고 
$$\lim_{x \to 0} \frac{f(x)}{x} = \lim_{x \to 1} \frac{f(x) - x}{x - 1}$$
일 때, $60 \times f'(0)$의 값을 구하시오.

## 풀이
$f(0) = 0$이고 주어진 극한의 식에서 분모가 $0$으로 갈 때 극한값이 존재하므로 $f(1) - 1 = 0$, 즉 $f(1) = 1$이다.
또한 극한의 정의에 의해
$f'(0) = f'(1) - 1$
이 성립한다.
$f(x) = a x^{2} + b x + c$라 하면 $f'(x) = 2 a x + b$이다.
$f(0) = 0$에서 $c = 0$이고,
$f(1) = a + b + c = 1$에서 $a + b = 1$이다.
$f'(0) = f'(1) - 1$에서
$b = (2 a + b) - 1$
$2 a = 1$ 이므로 $\therefore a = \frac{1}{2}$이다.
$a + b = 1$에 대입하면 $b = \frac{1}{2}$이다.
따라서 $f'(x) = x + \frac{1}{2}$이므로
$60 \times f'(0) = 60 \times \frac{1}{2} = 30$이다.
