## 문제
두 수 $2$와 $4$ 사이에 $n$개의 수 $a_{1}$, $a_{2}$, $a_{3}$, $\cdots$, $a_{n}$을 넣어 만든 $(n+2)$개의 수 $2$, $a_{1}$, $a_{2}$, $a_{3}$, $\cdots$, $a_{n}$, $4$가 이 순서대로 등차수열을 이룬다. 집합 $A_{n} = \{ 2, a_{1}, a_{2}, a_{3}, \cdots, a_{n}, 4 \}$에 대하여 <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $n$은 자연수이다.)

$$ \begin{array}{|l|} \hline \text{<보기>} \\ \text{ㄱ. } n \text{이 홀수이면 } 3 \in A_{n} \\ \text{ㄴ. 모든 자연수 } n \text{에 대하여 } A_{n} \subset A_{2n+1} \\ \text{ㄷ. } A_{2n+1} - A_{n} \text{의 모든 원소의 합을 } S_{n} \text{이라 할 때, } S_{6} + S_{13} = 63 \text{이다.} \\ \hline \end{array} $$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
[정답] ⑤

ㄱ. $n$이 홀수이면 집합 $A_{n}$의 항의 개수 $(n+2)$는 홀수이다. 이때 $3$은 양 끝 항인 $2$와 $4$의 등차중항($\frac{2+4}{2}=3$)이므로, 항상 집합의 원소로 포함된다. 따라서 $3 \in A_{n}$ (참)

ㄴ. 집합 $A_{n}$에서 $4 = 2 + (n+1)d_{1}$이라 하면, 공차 $d_{1} = \frac{2}{n+1}$이다.
집합 $A_{n}$의 임의의 원소는 $x_{k} = 2 + k \cdot \frac{2}{n+1}$ ($k=0, 1, \cdots, n+1$)이다.
집합 $A_{2n+1}$에서 $4 = 2 + (2n+2)d_{2}$라 하면, 공차 $d_{2} = \frac{2}{2n+2} = \frac{1}{n+1}$이다.
집합 $A_{2n+1}$의 임의의 원소는 $y_{m} = 2 + m \cdot \frac{1}{n+1}$ ($m=0, 1, \cdots, 2n+2$)이다.
$x_{k} = 2 + \frac{2k}{n+1} = y_{2k}$이므로, $A_{n}$의 모든 원소는 $A_{2n+1}$에 포함된다.
즉, $A_{n} \subset A_{2n+1}$ (참)

ㄷ. ㄴ에 의하여 $A_{2n+1} - A_{n}$은 집합 $A_{2n+1}$의 원소 중 $m$이 홀수인 원소들의 집합이다.
$A_{2n+1} - A_{n} = \{ 2 + \frac{1}{n+1}, 2 + \frac{3}{n+1}, \cdots, 2 + \frac{2n+1}{n+1} \}$
이 집합의 원소의 개수는 $(n+1)$개이며, 이는 등차수열을 이룬다.
$S_{n} = \frac{n+1}{2} \left( \left( 2 + \frac{1}{n+1} \right) + \left( 2 + \frac{2n+1}{n+1} \right) \right)$
$S_{n} = \frac{n+1}{2} \left( 4 + \frac{2n+2}{n+1} \right) = \frac{n+1}{2} (4 + 2) = 3(n+1)$
$S_{6} = 3(6+1) = 21$
$S_{13} = 3(13+1) = 42$
$S_{6} + S_{13} = 21 + 42 = 63$ (참)

따라서 옳은 것은 ㄱ, ㄴ, ㄷ이다.
