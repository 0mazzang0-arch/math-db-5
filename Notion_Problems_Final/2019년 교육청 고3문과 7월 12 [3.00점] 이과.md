## 문제
`$1$`보다 큰 두 실수 `$a$`, `$b$`에 대하여 $\log _{a} \frac{a^{3}}{b^{2}} = 2$가 성립할 때, $\log _{a} b + 3 \log _{b} a$의 값은?

## 보기
① $\frac{9}{2}$   ② $5$   ③ $\frac{11}{2}$   ④ $6$   ⑤ $\frac{13}{2}$

## 풀이
$\log _{a} \frac{a^{3}}{b^{2}} = \log _{a} a^{3} - \log _{a} b^{2} = 3 - 2 \log _{a} b = 2$
$2 \log _{a} b = 1$이므로 $\log _{a} b = \frac{1}{2}$
$\log _{b} a = \frac{1}{\log _{a} b} = 2$
따라서
$\log _{a} b + 3 \log _{b} a = \frac{1}{2} + 3 \times 2 = \frac{1}{2} + 6 = \frac{13}{2}$

정답: ⑤
