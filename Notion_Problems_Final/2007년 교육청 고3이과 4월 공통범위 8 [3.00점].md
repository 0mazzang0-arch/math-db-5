## 문제
확률변수 $X$ 가 취하는 모든 값이 $1$, $2$, $3$, $\dots$, $99$ 일 때, $X=k$ 일 확률은
$$P(X=k) = \frac{a}{\sqrt{k+1} + \sqrt{k}} (k=1, 2, 3, \dots, 99)$$
이다.
$$P(X=16) + P(X=17) + P(X=18) + \dots + P(X=99) = b$$
라 할 때, $a+b$ 의 값은?

## 보기
① $\frac{5}{9}$   ② $\frac{7}{9}$   ③ $1$   ④ $\frac{11}{9}$   ⑤ $\frac{13}{9}$

## 풀이
$P(X=k) = \frac{a}{\sqrt{k+1} + \sqrt{k}} \cdot \frac{\sqrt{k+1} - \sqrt{k}}{\sqrt{k+1} - \sqrt{k}}$
$= a(\sqrt{k+1} - \sqrt{k})$

확률의 총합은 $1$ 이므로
$\sum_{k=1}^{99} P(X=k) = a \{ (\sqrt{2} - 1) + (\sqrt{3} - \sqrt{2}) + \dots + (\sqrt{100} - \sqrt{99}) \}$
$= a(\sqrt{100} - 1) = 9a = 1$
따라서 $a = \frac{1}{9}$

$b = 1 - \{ P(X=1) + P(X=2) + \dots + P(X=15) \}$
$= 1 - \frac{1}{9} \{ (\sqrt{2} - 1) + (\sqrt{3} - \sqrt{2}) + \dots + (\sqrt{16} - \sqrt{15}) \}$
$= 1 - \frac{1}{9} (4 - 1) = \frac{2}{3}$

그러므로
$a + b = \frac{1}{9} + \frac{2}{3} = \frac{7}{9}$

정답: ②
