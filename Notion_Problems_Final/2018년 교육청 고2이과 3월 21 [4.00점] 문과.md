## 문제
다음 조건을 만족시키는 이차함수 $f(x)$에 대하여 $f(3)$의 최댓값을 $M$, 최솟값을 $m$이라 할 때, $M-m$의 값은?

(가) 부등식 $f \left( \frac{1-x}{4} \right) \le 0$의 해가 $-7 \le x \le 9$이다. 
(나) 모든 실수 $x$에 대하여 부등식 $f(x) \ge 2x - \frac{13}{3}$이 성립한다. 

## 보기
① $\frac{7}{4}$   ② $\frac{11}{6}$   ③ $\frac{23}{12}$   ④ $2$   ⑤ $\frac{25}{12}$

## 풀이
조건 (가)에서 $\frac{1-x}{4} = t$라 하면 $x = 1 - 4t$이고, 부등식 $f \left( \frac{1-x}{4} \right) \le 0$의 해가 $-7 \le x \le 9$이므로 
$-7 \le 1 - 4t \le 9$ 
$\therefore -2 \le t \le 2$ 
따라서 $f(t) = k(t-2)(t+2)$ ($k > 0$)에서 
$f(x) = k(x-2)(x+2) = k(x^2 - 4) \cdots\cdots$ ㉠ 
로 놓을 수 있다. 조건 (나)에서 이차부등식 $f(x) \ge 2x - \frac{13}{3}$, 즉 $kx^2 - 2x - 4k + \frac{13}{3} \ge 0$이 항상 성립해야 하므로 이차방정식 $kx^2 - 2x - 4k + \frac{13}{3} = 0$의 판별식을 $D$라 하면 
$\frac{D}{4} = 1 - k \left( -4k + \frac{13}{3} \right) \le 0$, 
$4k^2 - \frac{13}{3}k + 1 \le 0$ 
$12k^2 - 13k + 3 \le 0$, 
$(4k - 3)(3k - 1) \le 0$ 
$\therefore \frac{1}{3} \le k \le \frac{3}{4}$ 
이때, ㉠에서 $f(3) = 5k$이므로 
$\frac{5}{3} \le f(3) \le \frac{15}{4}$ 
따라서 $M = \frac{15}{4}$, $m = \frac{5}{3}$이므로 
$M - m = \frac{15}{4} - \frac{5}{3} = \frac{25}{12}$ 

**[다른 풀이]** 
$0$이 아닌 실수 $k$와 두 상수 $a$, $b$에 대하여 $f(x) = k(x-a)(x-b)$라 하면 조건 (가)에서 부등식 $f \left( \frac{1-x}{4} \right) \le 0$의 해가 $-7 \le x \le 9$이므로 
$k \left( \frac{1-x}{4} - a \right) \left( \frac{1-x}{4} - b \right) \le 0$ 
$k \left( \frac{1-4a-x}{4} \right) \left( \frac{1-4b-x}{4} \right) \le 0$ 
$k(x + 4a - 1)(x + 4b - 1) \le 0$ 
의 해가 $-7 \le x \le 9$이다. 따라서 
$k > 0$, $-4a + 1 = -7$, $-4b + 1 = 9$ 
$\therefore k > 0, a = 2, b = -2$ 
따라서 
$f(x) = k(x - 2)(x + 2) = k(x^2 - 4)$ ($k > 0$) $\cdots\cdots$ ㉠ 
조건 (나)에서 이차부등식 $f(x) \ge 2x - \frac{13}{3}$, 즉 $kx^2 - 2x - 4k + \frac{13}{3} \ge 0$이 항상 성립해야 하므로 이차방정식 $kx^2 - 2x - 4k + \frac{13}{3} = 0$의 판별식을 $D$라 하면 
$\frac{D}{4} = 1 - k \left( -4k + \frac{13}{3} \right) \le 0$, 
$4k^2 - \frac{13}{3}k + 1 \le 0$ 
$12k^2 - 13k + 3 \le 0$, 
$(4k - 3)(3k - 1) \le 0$ 
$\therefore \frac{1}{3} \le k \le \frac{3}{4}$ 
이때, ㉠에서 $f(3) = 5k$이므로 
$\frac{5}{3} \le f(3) \le \frac{15}{4}$ 
따라서 $M = \frac{15}{4}$, $m = \frac{5}{3}$이므로 
$M - m = \frac{15}{4} - \frac{5}{3} = \frac{25}{12}$
