## 문제
함수 $f(x) = (x^{2} + 2) e^{-x}$ 에 대하여 함수 $g(x)$ 가 미분가능하고 
$g \left( \frac{x + 8}{10} \right) = f^{-1}(x)$, $g(1) = 0$
을 만족시킬 때, $|g'(1)|$ 의 값을 구하시오.

## 보기


## 풀이
$g \left( \frac{x + 8}{10} \right) = f^{-1}(x)$ 에서 $f \left( g \left( \frac{x + 8}{10} \right) \right) = x$ 이다.
양변을 $x$ 에 대하여 미분하면 
$f' \left( g \left( \frac{x + 8}{10} \right) \right) \times g' \left( \frac{x + 8}{10} \right) \times \frac{1}{10} = 1$
이 식에 $x = 2$ 를 대입하면
$f' (g(1)) \times g' (1) \times \frac{1}{10} = 1$
$f'(0) \times g'(1) = 10$
한편, $f'(x) = 2x e^{-x} - (x^{2} + 2)e^{-x} = (2x - x^{2} - 2) e^{-x}$ 이므로
$f'(0) = -2$
따라서 $(-2) \times g'(1) = 10$ 에서 $g'(1) = -5$
즉, $|g'(1)| = |-5| = 5$

정답: $5$
