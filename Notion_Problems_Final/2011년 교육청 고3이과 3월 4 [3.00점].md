## 문제
$0 \le x < 2 \pi$ 일 때, 방정식 $2 \cos 2 x = 1 + \cos x$ 의 모든 실근의 합은?



> [!example] 📸 (그림/그래프)

## 보기
① $\pi$   ② $\frac{3}{2} \pi$   ③ $2 \pi$   ④ $\frac{5}{2} \pi$   ⑤ $3 \pi$

## 풀이
방정식 $2 \cos 2 x = 1 + \cos x$ 에서 코사인배각공식에 의해
$$2(2 \cos^{2} x - 1) = 1 + \cos x$$
$$4 \cos^{2} x - \cos x - 3 = 0$$
$$(4 \cos x + 3)(\cos x - 1) = 0$$
따라서 $\cos x = - \frac{3}{4}$ 또는 $\cos x = 1$ 이다.

이때 $\cos \alpha = \frac{3}{4}$ $(0 < \alpha < \frac{\pi}{2})$ 라 하면
$\cos x = - \frac{3}{4}$ 의 해는 $x = \pi - \alpha$, $x = \pi + \alpha$ 이다.
$\cos x = 1$ 의 해는 $x = 0$ 이다.

따라서 모든 실근 $x$ 의 값의 합은
$0 + ( \pi - \alpha ) + ( \pi + \alpha ) = 2 \pi$
이다.
