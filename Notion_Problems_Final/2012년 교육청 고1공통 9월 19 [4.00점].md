## 문제
$x$에 대한 이차 다항식 $f(x)$가 다음 조건을 만족한다.

$$ \begin{array}{l} \text{(가) } x^{3} + 3x^{2} + 4x + 2 \text{를 } f(x) \text{로 나눈 나머지는 } g(x) \text{이다.} \\ \text{(나) } x^{3} + 3x^{2} + 4x + 2 \text{를 } g(x) \text{로 나눈 나머지는 } f(x) - x^{2} - 2x \text{이다.} \end{array} $$

이때, $g(1)$의 값은?

## 보기
① $3$   ② $4$   ③ $5$   ④ $6$   ⑤ $7$

## 풀이
[정답] ② 

조건 (가)에서 $x^{3} + 3x^{2} + 4x + 2$를 $f(x)$로 나누었을 때의 몫을 $Q(x)$라 하면 나머지가 $g(x)$이므로 
$$x^{3} + 3x^{2} + 4x + 2 = f(x) Q(x) + g(x) \cdots \cdots ㉠$$ 

조건 (나)에서 $x^{3} + 3x^{2} + 4x + 2$를 $g(x)$로 나누었을 때의 몫을 $Q(x)$라 하면 나머지가 $f(x) - x^{2} - 2x$이므로 
$$x^{3} + 3x^{2} + 4x + 2 = g(x) Q(x) + f(x) - x^{2} - 2x \cdots \cdots ㉡$$ 

㉠에서 $f(x)$는 이차다항식이므로 $g(x)$는 일차 이하의 다항식이다. 따라서 ㉡에서 $f(x) - x^{2} - 2x$는 상수이어야 하므로 
$$f(x) - x^{2} - 2x = a \text{ (} a \text{는 상수)}$$ 

이것을 ㉠에 대입하여 정리하면 
$$g(x) = (2 - a)(x + 1)$$ 

이것을 ㉡에 대입하여 정리하면 
$$x^{3} + 3x^{2} + 4x + 2 = (2 - a)(x + 1) Q(x) + a$$ 

위 식의 양변에 $x = -1$을 대입하면 
$$a = 0$$ 

따라서 $g(x) = 2(x + 1)$이므로 
$$g(1) = 4$$
