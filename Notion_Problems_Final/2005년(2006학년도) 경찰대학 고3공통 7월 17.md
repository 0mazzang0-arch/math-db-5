## 문제
수열 $\{ a_{n} \}$과 $\{ b_{n} \}$은 $n \geq 2$일 때, $a_{n} = 2 a_{n-1} + b_{n-1}$과 $b_{n} = -3 a_{n-1} - b_{n-1}$을 만족하고 $a_{1} = b_{1} = 1$이다. $a_{100} + b_{100}$의 값은?

## 보기
① $-2$   ② $-1$   ③ $0$   ④ $1$   ⑤ $2$

## 풀이
$a_{n} = 2 a_{n-1} + b_{n-1} \cdots (1)$
$b_{n} = -3 a_{n-1} - b_{n-1} \cdots (2)$
두 식을 더하면
$a_{n} + b_{n} = -a_{n-1} \cdots (3)$
이 된다. 식 $(1)$과 $(3)$을 정리하면
$a_{n} = a_{n-1} - a_{n-2} \ (n \geq 3)$
$a_{1} = b_{1} = 1$이므로
$a_{1} = 1$
$a_{2} = 2 a_{1} + b_{1} = 3$
$a_{3} = a_{2} - a_{1} = 3 - 1 = 2$
$a_{4} = a_{3} - a_{2} = 2 - 3 = -1$
$a_{5} = a_{4} - a_{3} = (-1) - 2 = -3$
$a_{6} = a_{5} - a_{4} = (-3) - (-1) = -2$
$a_{7} = a_{6} - a_{5} = -2 - (-3) = 1$
$a_{8} = a_{7} - a_{6} = 1 - (-2) = 3$
$a_{9} = a_{8} - a_{7} = 3 - 1 = 2$
$\vdots$
따라서, $a_{100} + b_{100} = -a_{99} = -2$ 이다.
(수열 $\{ a_{n} \}$은 주기가 $6$인 수열이므로 $a_{99} = a_{3} = 2$이다.)

정답: ①
