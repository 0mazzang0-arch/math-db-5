## 문제
세 조건 $p$, $q$, $r$이 
$$ \begin{array}{l} p : a^{2} + b^{2} + c^{2} = 0 \\ q : |a| + |b| + |c| = 0 \\ r : (a-b)^{2} + (b-c)^{2} + (c-a)^{2} = 0 \end{array} $$
일 때, $p$는 $q$이기 위한 $(가)$ 조건, $p$는 $r$이기 위한 $(나)$ 조건이다. $(가)$, $(나)$에 들어갈 내용을 바르게 짝지은 것은? (단, $a$, $b$, $c$는 실수이다.)

## 보기
① $(가)$ 필요, $(나)$ 충분   ② $(가)$ 충분, $(나)$ 필요   ③ $(가)$ 충분, $(나)$ 필요충분   ④ $(가)$ 필요충분, $(나)$ 충분   ⑤ $(가)$ 필요충분, $(나)$ 필요

## 풀이
세 조건 $p$, $q$, $r$에 대하여 $a$, $b$, $c$가 실수일 때:
$p : a^{2} + b^{2} + c^{2} = 0 \iff a = 0, b = 0, c = 0$
$q : |a| + |b| + |c| = 0 \iff a = 0, b = 0, c = 0$
$r : (a-b)^{2} + (b-c)^{2} + (c-a)^{2} = 0 \iff a = b$, $b = c$, $c = a \iff a = b = c$

이때, 
$p \iff q$ 이므로 $p$는 $q$이기 위한 필요충분조건이다. $\therefore (가) : 필요충분$
$p \implies r$ 이지만 $r \not\implies p$ (예: $a = b = c = 1$) 이므로 $p$는 $r$이기 위한 충분조건이다. $\therefore (나) : 충분$

따라서 $(가)$ 필요충분, $(나)$ 충분이다.

정답: ④
