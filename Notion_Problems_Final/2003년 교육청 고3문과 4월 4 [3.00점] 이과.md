## 문제
집합 $A_{n}$을 다음과 같이 정의한다.
$$A_{n} = \{ m \mid [ \frac{m}{n} ] = 0, m, n \text{ 은 서로소인 자연수} \}$$
다음 중 옳지 않은 것은?
(단, $[x]$는 $x$를 넘지 않는 최대 정수이다.)

## 보기
① $A_{1} = \emptyset$   ② $A_{2} \cap A_{3} = A_{2}$   ③ $A_{2} \cup A_{3} = A_{3}$   ④ $A_{3} \cup A_{4} = A_{4}$   ⑤ $A_{3} \cap A_{4} = A_{2}$

## 풀이
$A_{n} = \{ m \mid [ \frac{m}{n} ] = 0, m, n \text{ 은 서로소인 자연수} \}$에서
$[ \frac{m}{n} ] = 0$이 성립하려면 $0 \le \frac{m}{n} < 1$이어야 한다.
$m$, $n$은 자연수이므로 $1 \le m < n$을 만족해야 한다.
즉, 집합 $A_{n}$은 $n$보다 작고 $n$과 서로소인 자연수의 집합이다.

각 보기의 집합을 구하면 다음과 같다.
- $A_{1}$ : $1 \le m < 1$을 만족하는 자연수 $m$은 존재하지 않으므로 $A_{1} = \emptyset$
- $A_{2}$ : $1 \le m < 2$이고 $\text{gcd}(m, 2) = 1$인 $m = 1 \Rightarrow A_{2} = \{ 1 \}$
- $A_{3}$ : $1 \le m < 3$이고 $\text{gcd}(m, 3) = 1$인 $m = 1, 2 \Rightarrow A_{3} = \{ 1, 2 \}$
- $A_{4}$ : $1 \le m < 4$이고 $\text{gcd}(m, 4) = 1$인 $m = 1, 3 \Rightarrow A_{4} = \{ 1, 3 \}$

이제 각 보기를 검토한다.
① $A_{1} = \emptyset$ (참)
② $A_{2} \cap A_{3} = \{ 1 \} \cap \{ 1, 2 \} = \{ 1 \} = A_{2}$ (참)
③ $A_{2} \cup A_{3} = \{ 1 \} \cup \{ 1, 2 \} = \{ 1, 2 \} = A_{3}$ (참)
④ $A_{3} \cup A_{4} = \{ 1, 2 \} \cup \{ 1, 3 \} = \{ 1, 2, 3 \}$
   이때 $A_{4} = \{ 1, 3 \}$이므로 $A_{3} \cup A_{4} \neq A_{4}$ (거짓)
⑤ $A_{3} \cap A_{4} = \{ 1, 2 \} \cap \{ 1, 3 \} = \{ 1 \} = A_{2}$ (참)

따라서 옳지 않은 것은 ④이다.

정답: ④
