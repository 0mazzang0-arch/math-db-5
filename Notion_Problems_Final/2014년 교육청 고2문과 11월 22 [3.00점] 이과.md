## 문제
등차수열 $\{ a_{n} \}$ 에서 $a_{2} = 4$, $a_{5} = 13$ 일 때, $a_{1} + a_{2} + a_{3} + a_{4} + a_{5}$ 의 값을 구하시오.

## 풀이
등차수열 $\{ a_{n} \}$ 의 첫째항을 $a_{1}$, 공차를 $d$ 라 하면
$a_{2} = a_{1} + d = 4 \cdots$ ㉠
$a_{5} = a_{1} + 4d = 13 \cdots$ ㉡
㉠, ㉡에서 $a_{1} = 1$, $d = 3$
$\therefore a_{1} + a_{2} + a_{3} + a_{4} + a_{5} = \frac{5 \{ 2 \times 1 + (5 - 1) \times 3 \}}{2} = 35$

정답: $35$
