## 문제
$\lim _{n \to \infty} \frac{1}{n} \left( \sum_{k=1}^{n} \sin \frac{k}{2} \pi \right)$ 의 값은? 

## 보기
① $0$   ② $\frac{1}{2}$   ③ $\frac{1}{\sqrt{2}}$   ④ $\frac{\sqrt{3}}{2}$   ⑤ $1$

## 풀이
$\sin \frac{n}{2} \pi = a_{n}$ 이라 하면 

$S_{n} = \sum_{k=1}^{n} \sin \frac{k}{2} \pi$ 

$= \sin \frac{\pi}{2} + \sin \frac{2 \pi}{2} + \sin \frac{3 \pi}{2} + \cdots + \sin \frac{n \pi}{2}$ 

$= 1 + 0 + (-1) + 0 + 1 + 0 + (-1) + 0 + \cdots$ 

i) $n = 4l + 1$ ($l = 0, 1, 2, \dots$) 일 때 
$\sum_{k=1}^{n} \sin \frac{k \pi}{2} = 1 \Rightarrow \lim _{n \to \infty} \frac{S_{4 l+1}}{n} = \frac{1}{\infty} = 0$ 

ii) $n = 4l + 2$ ($l = 0, 1, 2, \dots$) 일 때 
$\sum_{k=1}^{n} \sin \frac{k \pi}{2} = 1 \Rightarrow \lim _{n \to \infty} \frac{S_{4 l+2}}{n} = \frac{1}{\infty} = 0$ 

iii) $n = 4l + 3$ ($l = 0, 1, 2, \dots$) 일 때 
$\sum_{k=1}^{n} \sin \frac{k \pi}{2} = 0 \Rightarrow \lim _{n \to \infty} \frac{S_{4 l+3}}{n} = \frac{0}{\infty} = 0$ 

iv) $n = 4l + 4$ ($l = 0, 1, 2, \dots$) 일 때 
$\sum_{k=1}^{n} \sin \frac{k \pi}{2} = 0 \Rightarrow \lim _{n \to \infty} \frac{S_{4 l+4}}{n} = \frac{0}{\infty} = 0$ 

따라서 $\lim _{n \to \infty} \frac{1}{n} \left( \sum_{k=1}^{n} \sin \frac{k}{2} \pi \right) = 0$ 이다.

정답: ①
