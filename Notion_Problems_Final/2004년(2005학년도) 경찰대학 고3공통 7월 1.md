## 문제
다항식 $x^{3} - 3x^{2}b + 3xb^{2} - c^{3}$ 이 $(x - a)(x - b)$로 나누어떨어질 때 세 실수 $a$, $b$, $c$의 관계는?

## 보기
① $a = b$   ② $b = c$   ③ $a = c$   ④ $a^{2} + b^{2} = c^{2}$   ⑤ $a = b = c$

## 풀이
다항식 $x^{3} - 3x^{2}b + 3xb^{2} - c^{3}$ 이 $(x - a)(x - b)$로 나누어떨어지므로 그때의 몫을 $Q(x)$라 하면 
$x^{3} - 3x^{2}b + 3xb^{2} - c^{3} = (x - a)(x - b)Q(x) \cdots \cdots$ ㉠ 

㉠의 양변에 $x = a$를 대입하면 
$a^{3} - 3a^{2}b + 3ab^{2} - c^{3} = 0 \cdots \cdots$ ㉡ 

㉠의 양변에 $x = b$를 대입하면 
$b^{3} - 3b^{3} + 3b^{3} - c^{3} = 0$, $b^{3} = c^{3}$ 

$\therefore b = c$ ($\because b$, $c$는 실수) $\cdots \cdots$ ㉢ 

이것을 ㉡에 대입하면 
$a^{3} - 3a^{2}b + 3ab^{2} - b^{3} = 0$ 
$(a - b)^{3} = 0$ 

$\therefore a = b$ 

따라서 $a = b = c$이다.

정답: ⑤
