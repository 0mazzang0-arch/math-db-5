## 문제
두 집합
$A = \{ x \mid x \text{는 } 12 \text{의 양의 약수} \}$
$B = \{ x \mid x \text{는 } \square \text{의 양의 약수} \}$
에 대하여 $A \cap B = \{ 1, 2 \}$일 때, $\square$에 들어갈 수 있는 $20$보다 작은 자연수들의 합을 구하시오.

## 보기

## 풀이
$A = \{ 1, 2, 3, 4, 6, 12 \}$이므로 $A \cap B = \{ 1, 2 \}$가 되기 위해서 $\square$에 들어갈 수 있는 수는 $3$, $4$를 약수로 갖지 않는 $2$의 배수이다.
$20$보다 작은 자연수 중 이를 만족하는 것은 $2, 10, 14$이다.
따라서 이들의 합은 $2 + 10 + 14 = 26$이다.

정답: $26$
