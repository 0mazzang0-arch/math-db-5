## 문제
$a > 1$ 일 때, 함수 $f(x) = 2x^3 - 3(a+1)x^2 + 6ax - 4a + 2$ 에 대하여 방정식 $f(x) = 0$ 의 한 실근을 $b$ 라 하자. 다음은 두 수 $a, b$ 의 크기를 비교하는 과정이다.

$$ \begin{array}{|l|} \hline f'(x) = (가) \text{ 이고 } a > 1 \text{ 이므로 } f(x) \text{ 는 } x = 1 \text{ 에서 } \\ (나) \text{ 을 가진다. 그런데 } f(1) < 0 \text{ 이고 } f(b) = 0 \text{ 이므로 } \\ a \ (다) \ b \text{ 이다. } \\ \hline \end{array} $$

위의 과정에서 $(가), (나), (다)$ 에 알맞은 것은?

## 보기
① $(가): 6(x+a)(x+1), (나): \text{극솟값}, (다): >$   ② $(가): 6(x+a)(x+1), (나): \text{극솟값}, (다): <$   ③ $(가): 6(x-a)(x-1), (나): \text{극솟값}, (다): >$   ④ $(가): 6(x-a)(x-1), (나): \text{극댓값}, (다): <$   ⑤ $(가): 6(x-a)(x-1), (나): \text{극댓값}, (다): >$

## 풀이
[정답] ④

$f(x) = 2x^3 - 3(a+1)x^2 + 6ax - 4a + 2$ 에서
$f'(x) = 6x^2 - 6(a+1)x + 6a$
$= 6\{x^2 - (a+1)x + a\}$
$= 6(x-a)(x-1)$

이때, $a > 1$ 이므로 $x = 1$ 에서 극댓값을 가진다.
그런데, $f(1) = 2 - 3(a+1) + 6a - 4a + 2 = 1 - a$ 이고, $a > 1$ 이므로 $f(1) < 0$ 이다.
$b$ 는 $f(x) = 0$ 의 한 근이므로 $f(b) = 0$.

따라서 그래프의 개형을 그려 보면 다음과 같다.
> [!example] 📸 (그림/그래프)

그래프의 개형에 의해 극댓값인 $f(1)$ 이 $0$ 보다 작으므로 실근 $b$ 는 극솟값을 갖는 지점인 $x = a$ 보다 큰 범위에서 나타난다.
따라서 $a < b$ 이다.

정리하면 다음과 같다.
$(가): 6(x-a)(x-1)$
$(나): \text{극댓값}$
$(다): <$
