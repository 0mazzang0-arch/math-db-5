## 문제
연립부등식 $\begin{cases} x^{2}-3 x-4 \leq 0 \\ -x^{2}+x+2 < 0 \end{cases}$ 의 해가 $\alpha < x \leq \beta$ 일 때, $\alpha + \beta$ 의 값은?

## 보기
① $1$   ② $3$   ③ $6$   ④ $8$   ⑤ $9$

## 풀이
$\begin{cases} x^{2}-3 x-4 \leq 0 \Rightarrow -1 \leq x \leq 4 \\ -x^{2}+x+2 < 0 \Rightarrow x < -1 \text{ 또는 } x > 2 \end{cases}$
따라서, 해는 $2 < x \leq 4$ 이므로 $\alpha = 2$, $\beta = 4$ 이다.
$\therefore \alpha + \beta = 6$
