## 문제
모든 실수 $x$ 에 대하여 연속인 함수 $f(x)$ 는 $f(x+4) = f(x)$ 를 만족시키고, 닫힌구간 $[0, 4]$ 에서 다음과 같이 정의된다.
$$ f(x) = \begin{cases} 3x & (0 \le x < 1) \\ x^2 + ax + b & (1 \le x \le 4) \end{cases} $$
이 때, $f(10)$ 의 값은?

## 보기
① $-1$   ② $0$   ③ $1$   ④ $2$   ⑤ $3$

## 풀이
[정답] ②

함수 $f(x)$ 가 모든 실수 $x$ 에 대하여 연속이려면 $x=1$ 에서 연속이어야 하고, 주기가 $4$ 인 주기함수이므로 $f(0)=f(4)$ 를 만족해야 한다.

(ⅰ) $x=1$ 에서의 연속성:
$\lim_{x \to 1-} f(x) = \lim_{x \to 1+} f(x) = f(1)$ 이므로
$3(1) = 1^2 + a(1) + b$
$3 = 1 + a + b$
$a + b = 2 \dots ①$

(ⅱ) 주기에 따른 끝점의 함숫값 일치:
$f(0) = f(4)$ 이므로
$3(0) = 4^2 + 4a + b$
$0 = 16 + 4a + b$
$4a + b = -16 \dots ②$

$①, ②$ 를 연립하여 풀면:
$3a = -18$ 에서 $a = -6$
$b = 8$

따라서 $1 \le x \le 4$ 에서 $f(x) = x^2 - 6x + 8$ 이고, $f(x+4) = f(x)$ 이므로
$f(10) = f(10 - 4 \times 2) = f(2)$
$f(2) = 2^2 - 6(2) + 8 = 4 - 12 + 8 = 0$
$\therefore f(10) = 0$
