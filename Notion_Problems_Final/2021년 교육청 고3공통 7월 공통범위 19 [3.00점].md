## 문제
두 다항함수 $f(x)$, $g(x)$가 
$$\lim_{x \to 2} \frac{f(x) - 4}{x^2 - 4} = 2, \quad \lim_{x \to 2} \frac{g(x) + 1}{x - 2} = 8$$
을 만족시킨다. 함수 $h(x) = f(x)g(x)$에 대하여 $h'(2)$의 값을 구하시오.

## 보기

## 풀이
$\lim_{x \to 2} \frac{f(x) - 4}{x^2 - 4} = 2$에서 $x \to 2$일 때 (분모) $\to 0$이고 극한값이 존재하므로 (분자) $\to 0$이다. 

$\lim_{x \to 2} \{f(x) - 4\} = 0$이므로 
$f(2) = 4$ 

$\lim_{x \to 2} \frac{f(x) - 4}{x^2 - 4} = \lim_{x \to 2} \left\{ \frac{1}{x + 2} \times \frac{f(x) - f(2)}{x - 2} \right\} = \frac{1}{4} f'(2) = 2$ 
따라서 $f'(2) = 8$ 이므로 

$\lim_{x \to 2} \frac{g(x) + 1}{x - 2} = 8$에서 $x \to 2$일 때 (분모) $\to 0$이고 극한값이 존재하므로 (분자) $\to 0$이다. 

$\lim_{x \to 2} \{g(x) + 1\} = 0$이므로 
$g(2) = -1$ 

$\lim_{x \to 2} \frac{g(x) + 1}{x - 2} = \lim_{x \to 2} \frac{g(x) - g(2)}{x - 2} = g'(2) = 8$ 

$h'(x) = f'(x)g(x) + f(x)g'(x)$ 
따라서 
$h'(2) = f'(2)g(2) + f(2)g'(2) = 8 \times (-1) + 4 \times 8 = 24$ 

정답: $24$
