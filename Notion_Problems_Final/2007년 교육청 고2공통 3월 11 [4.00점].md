## 문제
$0 \le x \le 2\pi$ 에서 두 함수 $y = \sin x$ 와 $y = -\sin x + a$ 의 그래프가 만나는 점의 개수를 $N(a)$ 라 할 때, <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $a$ 는 실수이다.)

$$
\begin{array}{|l|}
\hline
\text{ㄱ. } N(0) = 3 \\
\text{ㄴ. } |a| > 2 \text{ 이면 } N(a) = 0 \\
\text{ㄷ. } N(a) = 2 \text{ 이면 } N(-a) = 2 \\
\hline
\end{array}
$$



> [!example] 📸 (그림/그래프)

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
$\sin x = -\sin x + a$ 
$\Leftrightarrow 2\sin x = a$ 
따라서 $N(a)$ 는 $0 \le x \le 2\pi$ 에서 $y = 2\sin x$ 와 $y = a$ 의 그래프가 만나는 점의 개수와 같다. 

ㄱ. $a = 0$ 일 때,
$2\sin x = 0$
$x = 0, \pi, 2\pi$
따라서 $N(0) = 3$ 이므로 참이다.

ㄴ. $-2 \le 2\sin x \le 2$ 이므로 $|a| > 2$ 이면 $y = 2\sin x$ 와 $y = a$ 의 그래프는 만나지 않는다.
따라서 $N(a) = 0$ 이므로 참이다.

ㄷ. 아래 $y = 2\sin x$ 의 그래프에서와 같이
$N(a) = 2$ 이면 $-2 < a < 0$ 또는 $0 < a < 2$ 이다.
이때 $-a$ 의 범위 또한 $0 < -a < 2$ 또는 $-2 < -a < 0$ 가 되므로
$N(-a) = 2$ 이다. 그러므로 참이다.

따라서 옳은 것은 ㄱ, ㄴ, ㄷ 이다.
