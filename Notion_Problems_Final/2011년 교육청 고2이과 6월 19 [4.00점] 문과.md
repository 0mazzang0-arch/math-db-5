## 문제

세 이차정사각행렬 $A$, $B$, $C$가 다음 조건을 만족시킨다.

$$ \begin{array}{|l|} \hline \text{(가) } A B = B C \\ \text{(나) } B \text{의 역행렬이 존재한다.} \\ \hline \end{array} $$

<보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $E$는 단위행렬이다.)

$$ \begin{array}{l} \text{ㄱ. } A = E \text{이면 } C = E \text{이다.} \\ \text{ㄴ. } A \text{의 역행렬이 존재하면 } C \text{의 역행렬도 존재한다.} \\ \text{ㄷ. } A^{7} B = B C^{7} \end{array} $$

## 보기

① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이

[정답] ⑤

ㄱ. $A = E$ 이면 (가)에서 $B = B C$ 이다.
$B$의 역행렬이 존재하므로 $B^{-1} B = B^{-1} B C$
$\therefore C = B^{-1} B = E$ (참)

ㄴ. 행렬 $A$의 역행렬이 존재하면 $A B = B C$에서 $C = B^{-1} A B$이다.
$C^{-1} = (B^{-1} A B)^{-1} = B^{-1} A^{-1} B$ 이므로 행렬 $C$의 역행렬이 존재한다. (참)

ㄷ. (가), (나)에서 $A = B C B^{-1}$ 이므로
$A^{7} = (B C B^{-1})^{7}$
$= (B C B^{-1}) (B C B^{-1}) \cdots (B C B^{-1})$
$= B C^{7} B^{-1}$
$\therefore A^{7} B = B C^{7}$ (참)

따라서 옳은 것은 $ㄱ$, $ㄴ$, $ㄷ$이다.

[다른풀이]
ㄷ. $A^{7} B$
$= A A \cdots A A B$
$= A A \cdots A B C$
$= A A \cdots B C C$
$\vdots$
$= B C \cdots C C C$
$= B C^{7}$ (참)
