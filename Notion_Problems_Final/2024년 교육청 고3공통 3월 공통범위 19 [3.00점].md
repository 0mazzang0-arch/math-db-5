## 문제
실수 $a$에 대하여 함수 $f(x) = x^{3} - \frac{5}{2}x^{2} + ax + 2$이다. 곡선 $y = f(x)$ 위의 두 점 $A(0, 2)$, $B(2, f(2))$에서의 접선을 각각 $l$, $m$이라 하자. 두 직선 $l$, $m$이 만나는 점이 $x$축 위에 있을 때, $60 \times |f(2)|$의 값을 구하시오.



> [!example] 📸 (그림/그래프)

## 보기
(주관식 문항으로 보기가 없습니다.)

## 풀이
$f(0) = 2$, $f(2) = 2a$

$f'(x) = 3x^{2} - 5x + a$에서
$f'(0) = a$, $f'(2) = a + 2$

직선 $l$의 방정식은
$y = f'(0)x + f(0)$
$y = ax + 2$ $\cdots$ ㉠

직선 $m$의 방정식은
$y = f'(2)(x - 2) + f(2)$
$y = (a + 2)(x - 2) + 2a$
$y = (a + 2)x - 4$ $\cdots$ ㉡

㉠, ㉡에서 두 직선 $l$, $m$이 만나는 점의 좌표는 $(3, 3a + 2)$이고 이 점이 $x$축 위에 있으므로
$3a + 2 = 0$

$a = -\frac{2}{3}$이므로
$f(2) = 2 \times \left(-\frac{2}{3}\right) = -\frac{4}{3}$

따라서
$60 \times |f(2)| = 60 \times |-\frac{4}{3}| = 80$
