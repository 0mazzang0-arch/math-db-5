## 문제
양수 $x$ 에 대하여 $\log x$ 의 정수부분과 소수부분을 각각 $f(x)$, $g(x)$ 라 할 때, 다음 조건을 만족시키는 모든 $x$ 의 값의 곱은?

(가) $f(x) + 3g(x)$ 의 값은 정수이다.
(나) $f(x) + f(x^{2}) = 6$

## 보기
① $10^{4}$   ② $10^{\frac{13}{3}}$   ③ $10^{\frac{14}{3}}$   ④ $10^{5}$   ⑤ $10^{\frac{16}{3}}$

## 풀이
$\log x = n + \alpha$ ($n$ 은 정수, $0 \le \alpha < 1$)이라 하면 $f(x) = n$, $g(x) = \alpha$ 이다.
(가)에서 $f(x) + 3g(x) = n + 3\alpha$ 의 값이 정수이어야 하므로 $3\alpha$ 가 정수이어야 한다.
따라서 $\alpha = 0$ 또는 $\alpha = \frac{1}{3}$ 또는 $\alpha = \frac{2}{3}$ 이다.

(i) $\alpha = 0$ 일 때,
$\log x^{2} = 2 \log x = 2n$ 이므로 $f(x^{2}) = 2n$ 이다.
$f(x) + f(x^{2}) = n + 2n = 6$
$3n = 6$
$n = 2$
$\log x = 2 + 0$ 이므로 $x = 10^{2}$ 이다.

(ii) $\alpha = \frac{1}{3}$ 일 때,
$\log x^{2} = 2 \log x = 2n + \frac{2}{3}$ 이므로 $f(x^{2}) = 2n$ 이다.
$f(x) + f(x^{2}) = n + 2n = 6$
$3n = 6$
$n = 2$
$\log x = 2 + \frac{1}{3}$ 이므로 $x = 10^{2 + \frac{1}{3}} = 10^{\frac{7}{3}}$ 이다.

(iii) $\alpha = \frac{2}{3}$ 일 때,
$\log x^{2} = 2 \log x = 2n + \frac{4}{3} = 2n + 1 + \frac{1}{3}$ 이므로 $f(x^{2}) = 2n + 1$ 이다.
$f(x) + f(x^{2}) = n + 2n + 1 = 6$
$3n = 5$
$n = \frac{5}{3}$
$n$ 은 정수가 아니므로 모순이다.

따라서 주어진 조건을 만족시키는 모든 $x$ 의 값의 곱은
$10^{2} \times 10^{2 + \frac{1}{3}} = 10^{4 + \frac{1}{3}} = 10^{\frac{13}{3}}$

정답: ②
