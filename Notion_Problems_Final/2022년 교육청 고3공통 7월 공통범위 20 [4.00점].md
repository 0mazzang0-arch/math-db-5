## 문제
최고차항의 계수가 $3$인 이차함수 $f(x)$에 대하여 함수 
$$g(x)=x^{2} \int_{0}^{x} f(t) dt - \int_{0}^{x} t^{2} f(t) dt$$ 
가 다음 조건을 만족시킨다.

(가) 함수 $g(x)$는 극값을 갖지 않는다.
(나) 방정식 $g'(x)=0$의 모든 실근은 $0$, $3$이다.

$\int_{0}^{3} |f(x)| dt$의 값을 구하시오.

## 풀이
$g'(x) = 2x \int_{0}^{x} f(t) dt + x^{2} f(x) - x^{2} f(x) = 2x \int_{0}^{x} f(t) dt$
$h(x) = \int_{0}^{x} f(t) dt$라 하면 $h(0)=0$
조건 (나)에 의하여 방정식 $g'(x) = 2xh(x) = 0$의 모든 실근은 $0$과 $3$이므로 $h(x)=0$의 실근은 $0$과 $3$이다.

(ⅰ) $h(x) = ax^{2}(x-3)$ ($a$는 상수)라 하면 
$g'(x) = 2ax^{3}(x-3)$이고 함수 $g(x)$는 $x=0$, $x=3$에서 극값을 가지므로 조건 (가)에 모순이다.

(ⅱ) $h(x) = ax(x-3)^{2}$ ($a$는 상수)라 하면 
$g'(x) = 2ax^{2}(x-3)^{2}$이므로 $g'(x)$의 부호가 변하지 않아 함수 $g(x)$는 극값을 갖지 않는다.

$h'(x) = f(x)$이므로 
$f(x) = \frac{d}{dx} \{ a(x^{3}-6x^{2}+9x) \} = a(3x^{2}-12x+9) = 3a(x-1)(x-3)$
$f(x)$의 최고차항의 계수가 $3$이므로 $3a=3$에서 $a=1$이다.
따라서 $f(x) = 3(x-1)(x-3)$이다.

구하는 값은 
$\int_{0}^{3} |f(x)| dx = 3 \int_{0}^{3} |(x-1)(x-3)| dx$
$= 3 \int_{0}^{1} (x-1)(x-3) dx - 3 \int_{1}^{3} (x-1)(x-3) dx$
$= 3 \left[ \frac{1}{3}x^{3}-2x^{2}+3x \right]_{0}^{1} - 3 \left[ \frac{1}{3}x^{3}-2x^{2}+3x \right]_{1}^{3}$
$= 3 \left( \frac{1}{3}-2+3 \right) - 3 \left( 9-18+9 - \left( \frac{1}{3}-2+3 \right) \right)$
$= 4 - 3 \left( -\frac{4}{3} \right) = 4 + 4 = 8$

정답: $8$
