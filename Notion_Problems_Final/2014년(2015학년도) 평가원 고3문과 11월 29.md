## 문제
두 다항함수 $f(x)$ 와 $g(x)$ 가 모든 실수 $x$ 에 대하여 
$$g(x)=(x^3 +2) f(x)$$ 
를 만족시킨다. $g(x)$ 가 $x=1$ 에서 극솟값 $24$ 를 가질 때, $f(1)-f'(1)$ 의 값을 구하시오.

## 보기

## 풀이
$g(x) = (x^3 + 2) f(x)$ 에서 
$g'(x) = 3x^2 f(x) + (x^3 + 2) f'(x)$ 
$g(x)$ 가 $x=1$ 에서 극솟값 $24$ 를 가지므로 
$g(1) = 24$, $g'(1) = 0$ 
$g(1) = 3f(1) = 24$ 에서 
$f(1) = 8$ 
$g'(1) = 3f(1) + 3f'(1) = 0$ 에서 
$3 \times 8 + 3f'(1) = 0$, 
$f'(1) = -8$ 
$\therefore f(1) - f'(1) = 8 - (-8) = 16$
