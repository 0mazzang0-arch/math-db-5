## 문제
$\sin \theta + \cos \theta = \sin \theta \cos \theta$ 일 때, $\sin \theta \cos \theta$ 의 값은 $a + b \sqrt{2}$ 이다. $10a - b$ 의 값을 구하시오. (단, $a$, $b$ 는 유리수이다.)

## 풀이
$\sin \theta + \cos \theta = \sin \theta \cos \theta$ 에서 양변을 제곱하면 
$1 + 2 \sin \theta \cos \theta = (\sin \theta \cos \theta)^2$ 이다. 
따라서 $(\sin \theta \cos \theta)^2 - 2 \sin \theta \cos \theta - 1 = 0$
$\therefore \sin \theta \cos \theta = 1 - \sqrt{2}$ ($\because -1 \le \sin \theta \cos \theta \le 1$)
$a = 1, b = -1$ 이므로 
$10a - b = 11$ 이다.
정답: $11$
