## 문제
공차가 음의 정수인 등차수열 $\{ a_{n} \}$ 에 대하여 $a_{6} = -2$, $\sum_{k=1}^{8} | a_{k} | = \sum_{k=1}^{8} a_{k} + 42$ 일 때, $\sum_{k=1}^{8} a_{k}$ 의 값은?

## 보기
① $40$   ② $44$   ③ $48$   ④ $52$   ⑤ $56$

## 풀이
등차수열 $\{ a_{n} \}$ 의 공차를 $d$ ($d < 0$) 이라 하자. 

$a_{6}$, $d$ 가 모두 정수이므로 등차수열 $\{ a_{n} \}$ 의 모든 항은 정수이다. 

$d = a_{6} - a_{5} = -2 - a_{5}$ 이고 $d < 0$ 이므로 $a_{5} > -2$
즉, $a_{5} = -1$ 또는 $a_{5}$ 는 음이 아닌 정수이다. 

(ⅰ) $a_{5} = -1$ 일 때 
$d = -2 - a_{5} = -1$ 이므로 $a_{n} = -n + 4$ 
$\sum_{k=1}^{8} a_{k} = -4$, $\sum_{k=1}^{8} | a_{k} | = 16$ 이므로 
$\sum_{k=1}^{8} | a_{k} | = \sum_{k=1}^{8} a_{k} + 42 \cdots$ ㉠ 
이 성립하지 않는다. 

(ⅱ) $a_{5}$ 는 음이 아닌 정수일 때 
$n \le 5$ 일 때 $a_{n} \ge 0$ 이고 $| a_{n} | = a_{n}$ 
$n \ge 6$ 일 때 $a_{n} < 0$ 이고 $| a_{n} | = -a_{n}$ 
㉠에서 $-a_{6} - a_{7} - a_{8} = a_{6} + a_{7} + a_{8} + 42$ 
$a_{6} + a_{7} + a_{8} = -21$ 
$a_{6} + ( a_{6} + d ) + ( a_{6} + 2d ) = -21$, $a_{6} + d = -7$ 
$a_{6} = -2$ 이므로 $d = -5$ 

(ⅰ), (ⅱ)에서 $d = -5$ 이고 $a_{1} = a_{6} - 5d = -2 + 25 = 23$ 이다. 
따라서 
$\sum_{k=1}^{8} a_{k} = \frac{8 \times \{ 2 \times 23 + 7 \times ( -5 ) \}}{2} = 44$
