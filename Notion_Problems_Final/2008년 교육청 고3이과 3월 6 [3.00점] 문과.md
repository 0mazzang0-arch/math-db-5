## 문제
자연수 $n$ 에 대하여 $\sqrt{n^{2} + 1}$ 의 정수 부분을 $a_{n}$ 이라 할 때, $\lim_{n \to \infty} a_{n} ( \sqrt{n^{2} + 1} - a_{n} )$ 의 값은?

## 보기
① $0$   ② $\frac{1}{2}$   ③ $\frac{\sqrt{2}}{2}$   ④ $1$   ⑤ $\sqrt{2}$

## 풀이
$n < \sqrt{n^{2} + 1} < n + 1$ 에서 $\sqrt{n^{2} + 1}$ 의 정수 부분이 $n$ 이므로
$a_{n} = n$

$\therefore \lim_{n \to \infty} a_{n} ( \sqrt{n^{2} + 1} - a_{n} )$
$= \lim_{n \to \infty} n ( \sqrt{n^{2} + 1} - n )$
$= \lim_{n \to \infty} \frac{n}{\sqrt{n^{2} + 1} + n}$
$= \lim_{n \to \infty} \frac{1}{\sqrt{1 + \frac{1}{n^{2}}} + 1}$
$= \frac{1}{2}$

정답: ②
