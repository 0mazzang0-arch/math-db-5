## 문제
다음은 모든 자연수 $n$ 에 대하여 
$$\sum_{k=1}^{n}(5 k-3) \left(\frac{1}{k}+\frac{1}{k+1}+\frac{1}{k+2}+ \cdots+\frac{1}{n} \right)=\frac{n(5 n+3)}{4}$$ 
이 성립함을 수학적귀납법으로 증명한 것이다. 

(1) $n=1$ 일 때, (좌변) $=2$ , (우변) $=2$ 이므로 주어진 등식은 성립한다. 
(2) $n=m$ 일 때 성립한다고 가정하면 
$$\sum_{k=1}^{m}(5 k-3) \left(\frac{1}{k}+\frac{1}{k+1}+\frac{1}{k+2}+ \cdots+\frac{1}{m} \right) = \frac{m(5 m+3)}{4}$$ 
$n=m+1$ 일 때 성립함을 보이자. 
$$\sum_{k=1}^{m+1}(5 k-3) \left(\frac{1}{k}+\frac{1}{k+1}+ \cdots+\frac{1}{m+1} \right)$$ 
$$= \sum_{k=1}^{m}(5 k-3) \left(\frac{1}{k}+\frac{1}{k+1}+ \cdots+\frac{1}{m+1} \right) + \frac{(가)}{m+1}$$ 
$$= \sum_{k=1}^{m}(5 k-3) \left(\frac{1}{k}+\frac{1}{k+1}+ \cdots+\frac{1}{(나)} \right) + \frac{1}{m+1} \sum_{k=1}^{m}(5 k-3) + \frac{(가)}{m+1}$$ 
$$= \frac{m(5 m+3)}{4} + \frac{1}{m+1} \sum_{k=1}^{m}((다)) + \frac{(가)}{m+1}$$ 
$$= \frac{(m+1)(5 m+8)}{4}$$ 
그러므로 $n=m+1$ 일 때도 성립한다. 
따라서 모든 자연수 $n$ 에 대하여 주어진 등식은 성립한다. 
위의 증명에서 $(가)$, $(나)$, $(다)$에 알맞은 것은?

## 보기
① $(가) \ 5m-3, (나) \ m, (다) \ 5k+2$   ② $(가) \ 5m-3, (나) \ m+1, (다) \ 5k+2$   ③ $(가) \ 5m+2, (나) \ m, (다) \ 5k-3$   ④ $(가) \ 5m+2, (나) \ m, (다) \ 5k+2$   ⑤ $(가) \ 5m+2, (나) \ m+1, (다) \ 5k-3$

## 풀이
[정답] ③

$n=m+1$ 일 때,
$$
\begin{array}{l}
\sum_{k=1}^{m+1} (5k-3) \left( \frac{1}{k} + \frac{1}{k+1} + \cdots + \frac{1}{m} + \frac{1}{m+1} \right) \\
= \sum_{k=1}^{m} (5k-3) \left( \frac{1}{k} + \frac{1}{k+1} + \cdots + \frac{1}{m+1} \right) + \{ 5(m+1)-3 \} \frac{1}{m+1} \\
= \sum_{k=1}^{m} (5k-3) \left( \frac{1}{k} + \frac{1}{k+1} + \cdots + \frac{1}{m} + \frac{1}{m+1} \right) + \frac{5m+2}{m+1} \\
\text{따라서 } (가) = 5m+2 \text{ 이다.} \\
\\
= \sum_{k=1}^{m} (5k-3) \left( \frac{1}{k} + \cdots + \frac{1}{m} \right) + \frac{1}{m+1} \sum_{k=1}^{m} (5k-3) + \frac{5m+2}{m+1} \\
\text{따라서 } (나) = m \text{ 이다.} \\
\\
= \frac{m(5m+3)}{4} + \frac{1}{m+1} \left( \sum_{k=1}^{m} (5k-3) + (5m+2) \right) \\
= \frac{m(5m+3)}{4} + \frac{1}{m+1} \sum_{k=1}^{m+1} (5k-3) \\
\text{따라서 } (다) = 5k-3 \text{ 이다.} \\
\\
= \frac{m(5m+3)}{4} + \frac{5m+4}{2} = \frac{5m^2+3m+10m+8}{4} = \frac{(m+1)(5m+8)}{4}
\end{array}
$$
그러므로 알맞은 조합은 ③이다.
