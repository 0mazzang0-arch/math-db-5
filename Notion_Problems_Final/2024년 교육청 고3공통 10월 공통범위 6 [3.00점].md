## 문제
공비가 양수인 등비수열 $\{ a_{n} \}$ 의 첫째항부터 제 $n$ 항까지의 합을 $S_{n}$ 이라 하자. 
$4 ( S_{4} - S_{2} ) = S_{6} - S_{4}$, $a_{3} = 12$
일 때, $S_{3}$ 의 값은?

## 보기
① $18$   ② $21$   ③ $24$   ④ $27$   ⑤ $30$

## 풀이
등비수열 $\{ a_{n} \}$ 의 공비를 $r$ ($r > 0$) 이라 하면 
$S_{4} - S_{2} = a_{3} + a_{4} = a_{1} r^{2} ( 1 + r )$ 
$S_{6} - S_{4} = a_{5} + a_{6} = a_{1} r^{4} ( 1 + r )$ 

$4 ( S_{4} - S_{2} ) = S_{6} - S_{4}$ 이므로 
$4 a_{1} r^{2} ( 1 + r ) = a_{1} r^{4} ( 1 + r )$ 

$a_{1} \neq 0$ 이고 $r^{2} = 4$ 이므로 
$r = 2$ 

$a_{3} = 12$ 에서 
$a_{1} \times 2^{2} = 12$, 
$a_{1} = 3$ 

따라서 
$S_{3} = a_{1} + a_{2} + a_{3} = 3 + 3 \times 2 + 3 \times 2^{2} = 21$

정답: ②
