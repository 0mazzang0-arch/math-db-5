## 문제
무한수열 $a_{1}$, $2a_{2}$, $2^{2} a_{3}$, $\cdots$, $2^{n-1} a_{n}$, $\cdots$의 첫째항부터 제 $n$항까지의 합이 $5n$일 때, $\sum_{n=1}^{\infty} a_{n}$의 값을 구하시오.

## 풀이
$b_{n} = 2^{n-1} a_{n}$이라 하고, 
$S_{n} = a_{1} + 2a_{2} + 2^{2} a_{3} + \cdots + 2^{n-1} a_{n} = 5n$이라 할 때, 
$b_{n} = S_{n} - S_{n-1} = 5$ ($n \ge 2$), $b_{1} = S_{1} = a_{1} = 5$ 
즉, $a_{n} = \frac{5}{2^{n-1}}$ ($n \ge 1$) 
따라서 $\sum_{n=1}^{\infty} a_{n} = \frac{5}{1 - \frac{1}{2}} = 10$

[정답] $10$
