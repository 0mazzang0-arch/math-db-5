## 문제
수열 $\{ a_{n} \}$에 대하여
$$\sum_{k=1}^{7} ( a_{k} + k ) = 50, \sum_{k=1}^{7} ( a_{k} + 2 )^{2} = 300$$
일 때, $$\sum_{k=1}^{7} a_{k}^{2}$$의 값을 구하시오.

## 풀이
$$\sum_{k=1}^{7} ( a_{k} + k ) = 50$$에서 
$$\sum_{k=1}^{7} a_{k} + \sum_{k=1}^{7} k = 50$$ 
$$\therefore \sum_{k=1}^{7} a_{k} = 50 - \sum_{k=1}^{7} k = 50 - \frac{7 \times 8}{2} = 22$$ 

$$\sum_{k=1}^{7} ( a_{k} + 2 )^{2} = 300$$에서 
$$\sum_{k=1}^{7} ( a_{k}^{2} + 4a_{k} + 4 ) = 300$$ 
$$\sum_{k=1}^{7} a_{k}^{2} + 4 \sum_{k=1}^{7} a_{k} + \sum_{k=1}^{7} 4 = 300$$ 
$$\therefore \sum_{k=1}^{7} a_{k}^{2} = 300 - 4 \times 22 - 7 \times 4 = 184$$

[정답] $184$
