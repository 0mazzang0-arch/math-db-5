## 문제
삼차함수 $f(x)$에 대하여 함수 $g(x)$를 
$$ g(x) = \begin{cases} -f(x) & (x < 0) \\ |f(x)| - |2x^{2}-8| & (x \ge 0) \end{cases} $$
이라 하자. 함수 $g(x)$가 실수 전체의 집합에서 미분가능할 때, $f(-5)$의 값을 구하시오.

## 풀이
함수 $g(x)$가 $x=0$에서 연속이므로 
$\lim_{x \to 0^{-}} g(x) = -f(0)$, $\lim_{x \to 0^{+}} g(x) = |f(0)| - 8$
$-f(0) = |f(0)| - 8$
$f(0) = 4 \quad \cdots$ ㉠ 
$g(0) = |f(0)| - 8 = -4$

함수 $g(x)$가 $x=0$에서 미분가능하므로 
$\lim_{h \to 0^{-}} \frac{g(0+h) - g(0)}{h} = \lim_{h \to 0^{-}} \frac{-f(0+h) + f(0)}{h} = -f'(0)$
$\lim_{h \to 0^{+}} \frac{g(0+h) - g(0)}{h} = \lim_{h \to 0^{+}} \frac{|f(0+h)| + 2h^{2} - 8 + 4}{h}$
$= \lim_{h \to 0^{+}} \frac{|f(0+h)| - f(0)}{h} + \lim_{h \to 0^{+}} \frac{2h^{2}}{h}$
$= \lim_{h \to 0^{+}} \frac{f(0+h) - f(0)}{h} + 0 = f'(0)$
$-f'(0) = f'(0)$
$f'(0) = 0 \quad \cdots$ ㉡ 

함수 $g(x)$가 $x=2$에서 미분가능하므로 
$\lim_{h \to 0^{-}} \frac{g(2+h) - g(2)}{h} = \lim_{h \to 0^{-}} \frac{|f(2+h)| + 2(2+h)^{2} - 8 - |f(2)|}{h}$
$= \lim_{h \to 0^{-}} \frac{|f(2+h)| - |f(2)|}{h} + \lim_{h \to 0^{-}} \frac{2h^{2} + 8h}{h} = \lim_{h \to 0^{-}} \frac{|f(2+h)| - |f(2)|}{h} + 8$
$\lim_{h \to 0^{+}} \frac{g(2+h) - g(2)}{h} = \lim_{h \to 0^{+}} \frac{|f(2+h)| - 2(2+h)^{2} + 8 - |f(2)|}{h}$
$= \lim_{h \to 0^{+}} \frac{|f(2+h)| - |f(2)|}{h} - \lim_{h \to 0^{+}} \frac{2h^{2} + 8h}{h} = \lim_{h \to 0^{+}} \frac{|f(2+h)| - |f(2)|}{h} - 8$
따라서, $\lim_{h \to 0^{-}} \frac{|f(2+h)| - |f(2)|}{h} + 8 = \lim_{h \to 0^{+}} \frac{|f(2+h)| - |f(2)|}{h} - 8 \quad \cdots$ ㉢ 

즉 함수 $|f(x)|$가 $x=2$에서 미분가능하지 않다. 
$f(2) = 0 \quad \cdots$ ㉣ 
$\lim_{h \to 0^{+}} \frac{|f(2+h)| - |f(2)|}{h} = |f'(2)|$ 와 $\lim_{h \to 0^{-}} \frac{|f(2+h)| - |f(2)|}{h} = -|f'(2)|$ 를 ㉢에 대입하면 
$|f'(2)| = 8 \quad \cdots$ ㉤ 

㉠, ㉡, ㉣, ㉤을 모두 만족시키는 함수 $f(x)$는 
$f(x) = 3x^{3} - 7x^{2} + 4$ 또는 $f(x) = -x^{3} + x^{2} + 4$

(ⅰ) $f(x) = 3x^{3} - 7x^{2} + 4$ 인 경우 
$3x^{3} - 7x^{2} + 4 = (3x+2)(x-1)(x-2)$ 이므로 $g(x)$가 $x=1$에서 미분가능하지 않다. 

(ⅱ) $f(x) = -x^{3} + x^{2} + 4$ 인 경우 
$-x^{3} + x^{2} + 4 = -(x-2)(x^{2} + x + 2)$ 이므로 $g(x)$는 실수 전체의 집합에서 미분가능하다. 

(ⅰ), (ⅱ)에서 $f(x) = -x^{3} + x^{2} + 4$ 이다. 
따라서 $f(-5) = 125 + 25 + 4 = 154$ 이다.
