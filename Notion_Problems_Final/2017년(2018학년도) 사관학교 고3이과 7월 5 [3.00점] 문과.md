## 문제
좌표공간의 두 점 $A(1, 2, -1)$, $B(3, 1, -2)$에 대하여 선분 $AB$를 $2:1$로 외분하는 점의 좌표는?

## 보기
① $(5, 0, -3)$   ② $(5, 3, -4)$   ③ $(4, 0, -3)$   ④ $(4, 3, -3)$   ⑤ $(3, 0, -4)$

## 풀이
좌표공간의 두 점 $A(1, 2, -1)$, $B(3, 1, -2)$에 대하여 선분 $AB$를 $2:1$로 외분하는 점의 좌표는 다음과 같다.
$$ 2B - A = 2(3, 1, -2) - (1, 2, -1) = (5, 0, -3) $$
따라서 구하는 점의 좌표는 $(5, 0, -3)$이다.
