## 문제
두 실수 $a$, $b$에 대하여 두 함수 
$f(x)=a x+b$
$g(x)=\frac{1}{a x+b-2}+3$
이 다음 조건을 만족시키도록 하는 두 실수 $a$, $b$의 순서쌍 $(a, b)$를 좌표평면에 나타낸 영역을 $R$라 하자.

(가) $x>0$일 때, $1<g(x)<3$
(나) 두 함수 $y=f(x)$와 $y=\frac{1}{x-2}+3$의 그래프의 교점이 제 $4$사분면 위에는 있지 않다.

영역 $R$에 속하는 점 $(a, b)$에 대하여 $a^{2}+b^{2}$의 최댓값을 $M$이라 할 때, $100 M$의 값을 구하시오. (단, $a \neq 0$)

## 풀이
$g(x) = \frac{1}{ax+b-2} + 3 = \frac{\frac{1}{a}}{x-\frac{2-b}{a}} + 3$이므로 점근선은 두 직선 
$x = \frac{2-b}{a}$, $y=3$이다.

(ⅰ) $a>0$, $b \geq 2$일 때 
$\frac{2-b}{a} \leq 0$이고 $\frac{1}{a} > 0$이므로 
$y=g(x)$의 그래프는 그림과 같다. 

> [!example] 📸 (그림/그래프)

$x>0$에서 $g(x) > 3$이므로 $a>0$, $b \geq 2$일 때 조건 (가)를 만족시키지 않는다.

(ⅱ) $a>0$, $b<2$일 때 
$\frac{2-b}{a} > 0$이고 $\frac{1}{a} > 0$이므로 
$y=g(x)$의 그래프는 그림과 같다. 

> [!example] 📸 (그림/그래프)

$x > \frac{2-b}{a}$에서 $g(x) > 3$이므로 
$a>0$, $b<2$일 때 조건 (가)를 만족시키지 않는다.

(ⅲ) $a<0$, $b \geq 2$일 때 
$\frac{2-b}{a} \leq 0$이고 $\frac{1}{a} < 0$이므로 
$y=g(x)$의 그래프는 그림과 같다. 

> [!example] 📸 (그림/그래프)

$0 < x < \frac{2-b}{a}$에서 $g(x) > 3$이므로 
$a<0$, $b \geq 2$일 때 조건 (가)를 만족시키지 않는다.
(※ 입력 데이터에 따라 $b \geq 2$일 때 $\frac{2-b}{a} \geq 0$이므로 해당 범위를 검토함)

(ⅳ) $a<0$, $b<2$일 때 
$\frac{2-b}{a} < 0$이고 $\frac{1}{a} < 0$이므로 
$y=g(x)$의 그래프는 그림과 같다. 

> [!example] 📸 (그림/그래프)

$y=g(x)$의 그래프에서 
$x>0$일 때, $\frac{1}{b-2}+3 < g(x) < 3$이므로 
$1 \leq \frac{1}{b-2} + 3$ 
$b-2 < 0$이므로 $-\frac{1}{2} \geq b-2$ 
$\therefore b \leq \frac{3}{2}$ 

$a<0$, $b \leq \frac{3}{2}$일 때 조건 (가)를 만족한다.

함수 $y = \frac{1}{x-2} + 3$의 그래프의 $x$절편은 $\frac{5}{3}$, $y$절편은 $\frac{5}{2}$이므로 조건 (나)를 만족시키는 두 함수 $y=f(x)$와 $y=\frac{1}{x-2}+3$의 그래프는 그림과 같다. 

> [!example] 📸 (그림/그래프)

조건 (나)에서 $f(\frac{5}{3}) \geq 0$, $b>0$이므로 
$\frac{5}{3}a + b \geq 0$ 
$\therefore b \geq -\frac{5}{3}a$ 

따라서 $a<0$, $0 < b \leq \frac{3}{2}$, $b \geq -\frac{5}{3}a$에서 조건 (가), (나)를 만족시키므로 영역 $R$를 좌표평면에 나타내면 그림과 같다. 

> [!example] 📸 (그림/그래프)

양수 $k$에 대하여 $x^{2} + y^{2} = k^{2}$이라 하면 $k$는 중심이 $(0, 0)$인 원의 반지름의 길이이다. 

> [!example] 📸 (그림/그래프)

점 $(a, b)$는 영역 $R$에 속하는 점이고 그림에서 $x = -\frac{9}{10}$, $y = \frac{3}{2}$일 때, $k$가 최대이므로 $a^{2} + b^{2}$의 최댓값은 
$M = (-\frac{9}{10})^{2} + (\frac{3}{2})^{2} = \frac{81}{100} + \frac{225}{100} = \frac{306}{100}$ 
따라서 $100M = 306$

[정답] $306$
