## 문제
좌표평면 위의 세 점 $A(-2, 0)$, $B(4, 0)$, $C(1, 2)$를 지나는 원이 있다. 이 원의 중심의 좌표를 $(p, q)$라 할 때, $p+q$의 값은?

## 보기
① $-\frac{3}{4}$   ② $-\frac{5}{8}$   ③ $-\frac{1}{2}$   ④ $-\frac{3}{8}$   ⑤ $-\frac{1}{4}$

## 풀이
세 점 $A(-2, 0)$, $B(4, 0)$, $C(1, 2)$를 지나는 원의 방정식을 
$x^{2} + y^{2} + ax + by + c = 0$
라 놓으면 이 원이 세 점 $A$, $B$, $C$를 지나므로 
$4 - 2a + c = 0$
$16 + 4a + c = 0$
$5 + a + 2b + c = 0$
위의 세 식을 연립하면 
$a = -2$, $b = \frac{5}{2}$, $c = -8$
따라서 이 원의 방정식은 
$x^{2} + y^{2} - 2x + \frac{5}{2}y - 8 = 0$
$\therefore (x - 1)^{2} + (y + \frac{5}{4})^{2} = (\frac{13}{4})^{2}$
따라서 $p = 1$, $q = -\frac{5}{4}$이므로 
$p + q = -\frac{1}{4}$

정답: ⑤
