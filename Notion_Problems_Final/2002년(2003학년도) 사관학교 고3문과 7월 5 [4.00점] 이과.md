## 문제
집합 $A = \{ 0, 1 \}$, $B = \{ 1, 4(x - 1), x + 2 \}$에 대하여 $A \otimes B = \{ ab \mid a \in A, b \in B \}$라 할 때, 집합 $A \otimes B$의 진부분집합의 개수가 $7$이다. 이때, 실수 $x$의 최댓값과 최솟값의 합은?

## 보기
① $0$   ② $\frac{1}{4}$   ③ $1$   ④ $-\frac{3}{4}$   ⑤ $-1$

## 풀이
집합 $A \otimes B$의 원소는 다음과 같다.
$$ \begin{array}{c|c|c|c} \otimes & 1 & 4(x - 1) & x + 2 \\ \hline 0 & 0 & 0 & 0 \\ \hline 1 & 1 & 4(x - 1) & x + 2 \end{array} $$
따라서 $A \otimes B = \{ 0, 1, 4(x - 1), x + 2 \}$이다. 
집합 $A \otimes B$의 진부분집합의 개수가 $7$이므로 $2^n - 1 = 7$에서 $n(A \otimes B) = 3$이다. 즉, 집합 $A \otimes B$의 서로 다른 원소의 개수는 $3$개여야 한다.

이미 $0$과 $1$은 확보되었으므로, $4(x - 1)$과 $x + 2$ 중 하나가 $0$ 또는 $1$이거나, 두 값이 서로 같아야 한다.

(ⅰ) $4(x - 1) = x + 2$일 때:
$3x = 6 \Rightarrow x = 2$
$A \otimes B = \{ 0, 1, 4, 4 \} = \{ 0, 1, 4 \}$ (원소 $3$개)

(ⅱ) $4(x - 1) = 0$일 때:
$x = 1$
$A \otimes B = \{ 0, 1, 0, 3 \} = \{ 0, 1, 3 \}$ (원소 $3$개)

(ⅲ) $4(x - 1) = 1$일 때:
$x = \frac{5}{4}$
$A \otimes B = \{ 0, 1, 1, \frac{13}{4} \} = \{ 0, 1, \frac{13}{4} \}$ (원소 $3$개)

(ⅳ) $x + 2 = 0$일 때:
$x = -2$
$A \otimes B = \{ 0, 1, -12, 0 \} = \{ 0, 1, -12 \}$ (원소 $3$개)

(ⅴ) $x + 2 = 1$일 때:
$x = -1$
$A \otimes B = \{ 0, 1, -8, 1 \} = \{ 0, 1, -8 \}$ (원소 $3$개)

가능한 모든 $x$의 값은 $2, 1, \frac{5}{4}, -2, -1$이다.
따라서 $x$의 최댓값은 $2$, 최솟값은 $-2$이다.
최댓값과 최솟값의 합은 $2 + (-2) = 0$이다.

정답: ①
