## 문제
첫째항이 $0$이 아닌 등차수열 $\{ a_{n} \}$의 첫째항부터 제 $n$항까지의 합 $S_{n}$에 대하여 $S_{9} = S_{18}$이다. 집합 $T_{n}$을 $T_{n} = \{ S_{k} \mid k = 1, 2, 3, \dots, n \}$이라 하자. 집합 $T_{n}$의 원소의 개수가 $13$이 되도록 하는 모든 자연수 $n$의 값의 합을 구하시오.

## 보기

## 풀이
수열 $\{ a_{n} \}$의 첫째항을 $a$ ($a \neq 0$), 공차를 $d$라 하면 $S_{9} = S_{18}$이므로 

$$\frac{9(2a + 8d)}{2} = \frac{18(2a + 17d)}{2}$$ 

$a = -13d$ 

$$S_{n} = \frac{n \{ -26d + (n - 1)d \}}{2} = \frac{d}{2} n ( n - 27 )$$ 

$S_{1} = S_{26} = -13d$, 
$S_{2} = S_{25} = -25d$, 
$S_{3} = S_{24} = -36d$, 
$\vdots$ 
$S_{13} = S_{14} = -91d$, 
$S_{27} = 0$, 
$S_{28} = 14d$, 
$S_{29} = 29d$, 
$\dots$ 

집합 $T_{n}$의 원소의 개수가 $13$이 되도록 하는 자연수 $n$의 값은 $13, 14, \dots, 26$이다.

따라서 모든 자연수 $n$의 값의 합은 

$13 + 14 + 15 + \dots + 26 = 273$

[정답] $273$
