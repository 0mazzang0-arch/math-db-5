## 문제
로그부등식 $(1 + \log_{3} x)(a - \log_{3} x) > 0$ 의 해가 $\frac{1}{3} < x < 9$ 일 때, 상수 $a$ 의 값은?

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
$\frac{1}{3} < x < 9$ 에서
$\log_{3} \frac{1}{3} < \log_{3} x < \log_{3} 9$
$-1 < \log_{3} x < 2 \cdots$ ㉠

로그부등식 $(1 + \log_{3} x)(a - \log_{3} x) > 0$ 에서
$(\log_{3} x + 1)(\log_{3} x - a) < 0$
이 부등식의 해가 ㉠이 되기 위해서는
$-1 < \log_{3} x < a$ 이므로
$a = 2$

정답: ②
