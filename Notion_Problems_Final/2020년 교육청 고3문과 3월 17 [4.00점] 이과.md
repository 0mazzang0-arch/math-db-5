## 문제

등차수열 $\{ a_n \}$의 첫째항부터 제 $n$항까지의 합을 $S_n$이라 하자. $a_3 = 42$일 때, 다음 조건을 만족시키는 $4$ 이상의 자연수 $k$의 값은?

$$ \begin{array}{l} \text{(가) } a_{k-3} + a_{k-1} = -24 \\ \text{(나) } S_k = k^2 \end{array} $$

## 보기

① $13$   ② $14$   ③ $15$   ④ $16$   ⑤ $17$

## 풀이

[정답] ③

[해설]
$a_{k-3}$, $a_{k-2}$, $a_{k-1}$은 이 순서대로 등차수열을 이루므로 $a_{k-2}$는 $a_{k-3}$과 $a_{k-1}$의 등차중항이다.
즉,
$$ a_{k-2} = \frac{a_{k-3} + a_{k-1}}{2} = \frac{-24}{2} = -12 $$
$$ S_k = \frac{k(a_1 + a_k)}{2} = \frac{k(a_3 + a_{k-2})}{2} $$
$$ S_k = \frac{k\{42 + (-12)\}}{2} = 15k $$
따라서 $k^2 = 15k$이고 $k \neq 0$이므로 $k = 15$이다.

[다른 풀이 $1$]
수열 $\{ a_n \}$의 첫째항을 $a$, 공차를 $d$라 하면
$a_3 = a + 2d = 42 \cdots$ ㉠
$a_{k-2} = \frac{a_{k-3} + a_{k-1}}{2}$에서
$a + (k-3)d = -12 \cdots$ ㉡
$S_k = \frac{k\{2a + (k-1)d\}}{2} = k^2$이고, $k \neq 0$이므로
$2a + (k-1)d = 2k \cdots$ ㉢
㉢에서 ㉠을 빼면
$a + (k-3)d = 2k - 42 \cdots$ ㉣
㉡, ㉣에서 $2k - 42 = -12$이므로 $k = 15$이다.

[다른 풀이 $2$]
수열 $\{ a_n \}$의 첫째항을 $a$, 공차를 $d$라 하면
$a_3 = a + 2d = 42$, $a = 42 - 2d \cdots$ ㉠
$a_{k-3} + a_{k-1} = a + (k-4)d + a + (k-2)d = -24$이므로
$a + (k-3)d = -12 \cdots$ ㉡
㉠을 ㉡에 대입하면
$42 - 2d + kd - 3d = -12$
$kd - 5d = -54 \cdots$ ㉢
$S_k = \frac{k\{2a + (k-1)d\}}{2} = k^2$이고, $k \neq 0$이므로
$2a + (k-1)d = 2k \cdots$ ㉣
㉠을 ㉣에 대입하면
$84 - 4d + kd - d = 2k$, $kd - 5d = 2k - 84 \cdots$ ㉤
㉢, ㉤에서 $2k - 84 = -54$이므로 $k = 15$이다.
