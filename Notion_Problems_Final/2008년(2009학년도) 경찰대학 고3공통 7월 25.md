## 문제
두 원 $C_{1}: x^{2}+y^{2}=1$ 과 $C_{r}: x^{2}+y^{2}=r^{2}$ 이 있다. (단, $r>1$) 다음 조건에 따라 $C_{r}$ 위의 점 $P_{k}$ 를 차례로 잡자. ($k=1, 2, 3, \dots$)

(ⅰ) $P_{1}=(r, 0)$
(ⅱ) 점 $P_{k+1}$ 은 점 $P_{k}$ 에서 $C_{1}$ 에 그은 접선이 $C_{r}$ 와 만나는 점이다.
(ⅲ) 선분 $P_{1} P_{2}$ 는 제 $1$ 사분면을 지난다.
(ⅳ) 선분 $P_{k+1} P_{k+2}$ 와 선분 $P_{k} P_{k+1}$ 은 다른 선분이다.

이때, <보기>에서 옳은 것만을 있는 대로 고른 것은?



> [!example] 📸 (그림/그래프)

## 보기
ㄱ. $r>\sqrt{2}$ 이면 $\angle P_{1} P_{2} P_{3} < 90^{\circ}$ 이다.
ㄴ. $r=\frac{2\sqrt{3}}{3}$ 이면 $P_{5}$ 의 좌표는 $P_{5}\left(-1, -\frac{\sqrt{3}}{3}\right)$ 이다.
ㄷ. $\angle P_{1} P_{2} P_{3} = 100^{\circ}$ 이면 $P_{1}=P_{10}$ 이다.

① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄱ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
그림에서 $\angle OP_{1}H = \theta$ 라고 하면 $\angle P_{n} P_{n+1} P_{n+2}=2\theta$, $\sin \theta = \frac{\overline{OH}}{\overline{OP_{1}}} = \frac{1}{r}$ 이다.

ㄱ. $r>\sqrt{2}$ 이면 $\sin \theta = \frac{1}{r} < \frac{1}{\sqrt{2}}$ 이므로 $\theta < 45^{\circ}$ 이다.
$\therefore \angle P_{1} P_{2} P_{3} = 2\theta < 90^{\circ}$ (참)

ㄴ. $r = \frac{2\sqrt{3}}{3}$ 이면 $\sin \theta = \frac{1}{r} = \frac{\sqrt{3}}{2}$ 에서 $\theta = 60^{\circ}$ 이다.
이때 $\angle OP_{n} P_{n+1} = 60^{\circ}$, $\angle P_{n} P_{n+1} P_{n+2} = 120^{\circ}$ 이다.
한 내각의 크기가 $120^{\circ}$ 이므로 $P_{k}$ 는 정육각형을 이룬다.
$P_{5}$ 의 좌표는 $P_{2}$ 의 좌표의 원점 대칭이다.
$P_{2}(x, y) = \left( \frac{2\sqrt{3}}{3} \cos 60^{\circ}, \frac{2\sqrt{3}}{3} \sin 60^{\circ} \right) = \left( \frac{\sqrt{3}}{3}, 1 \right)$
$\therefore P_{5} \left( -\frac{\sqrt{3}}{3}, -1 \right)$ (거짓)

ㄷ. $\angle P_{1} P_{2} P_{3} = 100^{\circ}$ 이면 $2\theta = 100^{\circ}$ 이므로 $\theta = 50^{\circ}$ 이다.
중심각 $\angle P_{n} O P_{n+1} = 180^{\circ} - 2\theta = 180^{\circ} - 100^{\circ} = 80^{\circ}$ 이다.
점 $P_{n}$ 이 다시 $P_{1}$ 으로 돌아오려면 $80^{\circ} \times n = 360^{\circ} \times m$ 을 만족해야 한다.
이를 만족하는 최소의 양의 정수 $m, n$ 은 각각 $n=9, m=2$ 이다.
따라서 $P_{1} = P_{1+9} = P_{10}$ 이다. (참)

따라서 옳은 것은 ㄱ, ㄷ이다.
