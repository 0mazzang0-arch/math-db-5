## 문제
다항함수 $f(x)$ 가 모든 실수 $x$ 에 대하여 
$$f(x) = \frac{3}{4} x^{2} + \left( \int_{0}^{1} f(x) dx \right)^{2}$$ 
을 만족시킬 때, $\int_{0}^{2} f(x) dx$ 의 값은? 

## 보기
① $\frac{9}{4}$   ② $\frac{5}{2}$   ③ $\frac{11}{4}$   ④ $3$   ⑤ $\frac{13}{4}$

## 풀이
$\int_{0}^{1} f(x) dx = a$ 라 하면, 
$$f(x) = \frac{3}{4} x^{2} + a^{2}$$ 

$$a = \int_{0}^{1} \left( \frac{3}{4} x^{2} + a^{2} \right) dx = \left[ \frac{1}{4} x^{3} + a^{2} x \right]_{0}^{1} = a^{2} + \frac{1}{4}$$ 

$$a^{2} - a + \frac{1}{4} = \left( a - \frac{1}{2} \right)^{2} = 0$$ 
따라서 $a = \frac{1}{2}$ 이고, 
$$f(x) = \frac{3}{4} x^{2} + \frac{1}{4}$$ 

$$\int_{0}^{2} f(x) dx = \int_{0}^{2} \left( \frac{3}{4} x^{2} + \frac{1}{4} \right) dx = \left[ \frac{1}{4} x^{3} + \frac{1}{4} x \right]_{0}^{2} = \frac{8}{4} + \frac{2}{4} = \frac{5}{2}$$

정답: ②
