## 문제

$\frac{12}{5} < k \leq 4$ 인 상수 $k$ 와 자연수 $n$ 에 대하여 수열 $\{ a_n \}$ 이 다음 조건을 만족시킨다.

(가) $n$ 이 짝수이면 $a_n$ 은 $0 \leq x \leq 2$ 에서 직선 $y = -\frac{k}{2n}$ 와 $y = 2 \sin (n \pi x + \frac{\pi}{2}) + |k \sin^2 (n \pi x) - (k-1)|$ 이 만나는 서로 다른 점의 개수와 같다.
(나) $n$ 이 홀수이면 $a_n$ 은 $0 \leq x \leq 2$ 에서 직선 $y = \frac{k+1}{n}$ 과 $y = 2 \sin (n \pi x + \frac{\pi}{2}) + |k \sin^2 (n \pi x) - (k-1)|$ 이 만나는 서로 다른 점의 개수와 같다.

$0 < a_2 < 6$ 일 때, $\sum_{n=1}^{5} a_n$ 의 값을 구하시오.

## 풀이

[정답] $53$

$y = 2 \sin (n \pi x + \frac{\pi}{2}) + |k \sin^2 (n \pi x) - (k-1)|$
$= 2 \cos (n \pi x) + |k \{ 1 - \cos^2 (n \pi x) \} - (k-1)|$
$= 2 \cos (n \pi x) + |1 - k \cos^2 (n \pi x)|$

$0 \leq x \leq 2$ 에서 $t = \cos (n \pi x)$ 라 할 때, 함수 $f(t) = 2t + |1 - k t^2| (-1 \leq t \leq 1)$ 이라 하자.
$1 - k t^2 = 0$ 에서 $t = -\frac{1}{\sqrt{k}}$ 또는 $t = \frac{1}{\sqrt{k}}$ 이므로

$$ f(t) = \begin{cases} -kt^2 + 2t + 1 & ( -\frac{1}{\sqrt{k}} \leq t \leq \frac{1}{\sqrt{k}} ) \\ kt^2 + 2t - 1 & ( -1 \leq t < -\frac{1}{\sqrt{k}} \text{ 또는 } \frac{1}{\sqrt{k}} < t \leq 1 ) \end{cases} $$
$$ = \begin{cases} -k ( t - \frac{1}{k} )^2 + ( 1 + \frac{1}{k} ) & ( -\frac{1}{\sqrt{k}} \leq t \leq \frac{1}{\sqrt{k}} ) \\ k ( t + \frac{1}{k} )^2 - ( 1 + \frac{1}{k} ) & ( -1 \leq t < -\frac{1}{\sqrt{k}} \text{ 또는 } \frac{1}{\sqrt{k}} < t \leq 1 ) \end{cases} $$

$f(-1) = k - 3$, $f(-\frac{1}{\sqrt{k}}) = -\frac{2}{\sqrt{k}}$, $f(\frac{1}{\sqrt{k}}) = \frac{2}{\sqrt{k}}$, $f(1) = k + 1$
$k \leq 4$ 이므로 $1 \leq \frac{2}{\sqrt{k}}$ 이다. 함수 $y = f(t)$ 의 그래프의 개형은 다음과 같다.

> [!example] 📸 (그림/그래프)

조건 (가)에 의하여 $a_2$ 는 $f(t_0) = -\frac{k}{4}$ 인 실수 $t_0$ ($-1 \leq t_0 \leq 1$) 에 대하여 곡선 $t = \cos 2 \pi x$ 와 직선 $t = t_0$ 이 만나는 서로 다른 점의 개수와 같다.
$\frac{12}{5} < k \leq 4$ 에서 $-\frac{2}{\sqrt{k}} \leq -\frac{k}{4} < k - 3$ 이다.

(I) $-\frac{2}{\sqrt{k}} < -\frac{k}{4}$ 인 경우
$f(t) = -\frac{k}{4}$ 인 실수 $t$ 를 $t_1, t_2$ ($-1 < t_1 < t_2 < 0$) 이라 하면

> [!example] 📸 (그림/그래프)

$0 \leq x \leq 2$ 에서 주기가 $1$ 인 곡선 $t = \cos 2 \pi x$ 와 두 직선 $t = t_1, t = t_2$ 가 만나는 서로 다른 점의 개수는 $a_2 = (2 + 2) \times 2 = 8$ 이다. 이는 $0 < a_2 < 6$ 을 만족시키지 않는다.

(II) $-\frac{2}{\sqrt{k}} = -\frac{k}{4}$ 인 경우
$f(t) = -\frac{k}{4}$ 인 실수 $t$ 를 $t_3 = -\frac{1}{\sqrt{k}}$ ($-1 < t_3 < 0$) 이라 하면

> [!example] 📸 (그림/그래프)

$0 \leq x \leq 2$ 에서 주기가 $1$ 인 곡선 $t = \cos 2 \pi x$ 와 직선 $t = t_3$ 이 만나는 서로 다른 점의 개수는 $a_2 = 2 \times 2 = 4 \cdots \cdots$ ㉠

(I), (II)에 의하여 $0 < a_2 < 6$ 을 만족시키는 $k$ 의 값은 $-\frac{2}{\sqrt{k}} = -\frac{k}{4}$ 에서 $k = 4$ 이다.
따라서 모든 자연수 $n$ 에 대하여 $0 \leq x \leq 2$ 에서 $t = \cos (n \pi x) (-1 \leq t \leq 1)$ 일 때,

$$ f(t) = \begin{cases} -4 (t - \frac{1}{4})^2 + \frac{5}{4} & ( -\frac{1}{2} \leq t \leq \frac{1}{2} ) \\ 4 (t + \frac{1}{4})^2 - \frac{5}{4} & ( -1 \leq t < -\frac{1}{2}, \frac{1}{2} < t \leq 1 ) \end{cases} $$

함수 $y = f(t)$ 의 그래프의 개형은 다음과 같다.

> [!example] 📸 (그림/그래프)

(i) $n = 1$ 인 경우
조건 (나)에 의하여 $f(t) = 5$ 인 실수 $t$ 는 $1$ 뿐이다.

> [!example] 📸 (그림/그래프)

$0 \leq x \leq 2$ 에서 주기가 $2$ 인 곡선 $t = \cos \pi x$ 와 직선 $t = 1$ 이 만나는 서로 다른 점의 개수 $a_1 = 2$

(ii) $n = 2$ 인 경우
㉠에 의하여 $a_2 = 4$

(iii) $n = 3$ 인 경우
조건 (나)에 의하여 $f(t) = \frac{5}{3}$ 인 실수 $t$ 를 $t_4$ ($0 < t_4 < 1$) 이라 하면

> [!example] 📸 (그림/그래프)

$0 \leq x \leq 2$ 에서 주기가 $\frac{2}{3}$ 인 곡선 $t = \cos 3 \pi x$ 와 직선 $t = t_4$ 가 만나는 서로 다른 점의 개수 $a_3 = 2 \times 3 = 6$

(iv) $n = 4$ 인 경우
조건 (가)에 의하여 $f(t) = -\frac{1}{2}$ 인 실수 $t$ 를 $t_5, t_6$ ($-1 < t_5 < t_6 < 0$) 이라 하면

> [!example] 📸 (그림/그래프)

$0 \leq x \leq 2$ 에서 주기가 $\frac{1}{2}$ 인 곡선 $t = \cos 4 \pi x$ 와 두 직선 $t = t_5, t = t_6$ 이 만나는 서로 다른 점의 개수 $a_4 = (2 \times 4) \times 2 = 16$

(v) $n = 5$ 인 경우
조건 (나)에 의하여 $f(t) = 1$ 인 실수 $t$ 는 $-1, 0, \frac{1}{2}$ 이다.

> [!example] 📸 (그림/그래프)

$0 \leq x \leq 2$ 에서 주기가 $\frac{2}{5}$ 인 곡선 $t = \cos 5 \pi x$ 와 세 직선 $t = -1, t = 0, t = \frac{1}{2}$ 이 만나는 서로 다른 점의 개수 $a_5 = 5 + (2 \times 5) \times 2 = 25$

(i)～(v)에 의하여
$\sum_{n=1}^{5} a_n = 2 + 4 + 6 + 16 + 25 = 53$
