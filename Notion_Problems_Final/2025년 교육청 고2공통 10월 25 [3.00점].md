## 문제
등차수열 $\{ a_{n} \}$ 의 첫째항부터 제 $n$ 항까지의 합을 $S_{n}$ 이라 하자. $S_{11} = 88$ 이고 $a_{5} = 3$ 일 때, $a_{7}$ 의 값을 구하시오.

## 풀이
등차수열 $\{ a_{n} \}$ 의 공차를 $d$ 라 하자.

$S_{11} = \frac{11 \{ 2a_{1} + ( 11 - 1 ) d \} }{2} = 11 ( a_{1} + 5d ) = 88$
에서 $a_{1} + 5d = 8$ 이고,
$a_{5} = 3$ 에서 $a_{1} + 4d = 3$ 이다.
두 식을 연립하여 계산하면 $d = 5$, $a_{1} = -17$
따라서 $a_{7} = -17 + 6 \times 5 = 13$

**[다른 풀이]**
$S_{11} = \frac{11 ( a_{1} + a_{11} ) }{2} = 88$ 에서 $\frac{ a_{1} + a_{11} }{2} = 8$
$a_{1}$ 과 $a_{11}$ 의 등차중항은 $a_{6}$ 이므로 $a_{6} = 8$ 이고,
$a_{5} = 3$ 이므로 등차수열 $\{ a_{n} \}$ 의 공차는 $5$ 이다.
따라서 $a_{7} = 8 + 5 = 13$

정답: $13$
