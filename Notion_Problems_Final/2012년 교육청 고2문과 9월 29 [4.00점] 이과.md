## 문제
양수 $x$에 대하여 $\log x$의 정수부분을 $f(x)$, 소수부분을 $g(x)$라 하자. 다음 조건을 만족시키는 양수 $a$에 대하여 $f(a^{6})$의 값을 구하시오.

(가) $6 \{ g(a) \}^{2} - 5 g(a) + 1 = 0$
(나) $f(a) + f(a^{2}) + f(a^{3}) = 14$

## 보기

## 풀이
$f(a) = n$이라 하자.

$6 \{ g(a) \}^{2} - 5 g(a) + 1 = 0$
$(3 g(a) - 1)(2 g(a) - 1) = 0$
$\therefore g(a) = \frac{1}{3}$ 또는 $g(a) = \frac{1}{2}$

(i) $g(a) = \frac{1}{3}$일 때
$\log a = n + \frac{1}{3}$
$\log a^{2} = 2 \log a = 2n + \frac{2}{3}$
$\log a^{3} = 3 \log a = 3n + 1$
$f(a) + f(a^{2}) + f(a^{3}) = n + 2n + (3n + 1) = 6n + 1$
$6n + 1 = 14$를 만족하는 정수 $n$이 존재하지 않으므로 $g(a)$는 $\frac{1}{3}$이 아니다.

(ii) $g(a) = \frac{1}{2}$일 때
$\log a = n + \frac{1}{2}$
$\log a^{2} = 2 \log a = 2n + 1$
$\log a^{3} = 3 \log a = 3n + \frac{3}{2}$
$f(a) + f(a^{2}) + f(a^{3}) = n + (2n + 1) + (3n + 1) = 6n + 2$
$6n + 2 = 14$를 만족하는 정수 $n$은 $2$이다.

따라서
$\log a = 2 + \frac{1}{2}$
$\log a^{6} = 6 \log a = 6 \times (2 + \frac{1}{2}) = 15$
$\therefore f(a^{6}) = 15$

정답: $15$
