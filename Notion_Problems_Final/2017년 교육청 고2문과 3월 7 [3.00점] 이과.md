## 문제
두 집합 $A = \{ 1, 2, 3, 6 \}$, $B = \{ 2, 4, 6, 8 \}$에 대하여 집합 $( A \cup B ) - ( A \cap B )$의 모든 원소의 합은?

## 보기
① $12$ ② $14$ ③ $16$ ④ $18$ ⑤ $20$

## 풀이
$A \cup B = \{ 1, 2, 3, 4, 6, 8 \}$, $A \cap B = \{ 2, 6 \}$이므로
$( A \cup B ) - ( A \cap B ) = \{ 1, 3, 4, 8 \}$이다.
따라서 집합 $( A \cup B ) - ( A \cap B )$의 모든 원소의 합은
$1 + 3 + 4 + 8 = 16$이다.

정답: ③
