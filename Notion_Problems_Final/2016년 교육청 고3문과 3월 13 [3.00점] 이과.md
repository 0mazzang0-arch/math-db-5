## 문제
점 $Q_{n}$의 $y$ 좌표를 $a_{n}$이라 할 때, $f^{-1} (a_{2} ) \cdot f^{-1} (a_{9} )$의 값은? (단, $a_{n} = 3n + 1$이고, $f(x) = x^{2} (x \ge 0)$이다.)

## 보기
① $\frac{7\sqrt{2}}{2}$   ② $7$   ③ $7\sqrt{2}$   ④ $7\sqrt{3}$   ⑤ $14$

## 풀이
점 $Q_{n}$의 $y$ 좌표는 점 $P_{n}$의 $y$ 좌표와 같다.
$a_{n} = 3n + 1$이므로
$a_{2} = 3 \times 2 + 1 = 7$,
$a_{9} = 3 \times 9 + 1 = 28$이다.
$f(x) = x^{2} (x \ge 0)$의 역함수는 $f^{-1}(x) = \sqrt{x} (x \ge 0)$이다.
그러므로
$f^{-1} (a_{2} ) = \sqrt{7}$,
$f^{-1} (a_{9} ) = \sqrt{28}$
따라서
$f^{-1} (a_{2} ) \cdot f^{-1} (a_{9} ) = \sqrt{7} \times \sqrt{28}$
$= \sqrt{7 \times 28} = \sqrt{196} = 14$

정답: ⑤
