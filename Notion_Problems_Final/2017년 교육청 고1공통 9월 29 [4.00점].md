## 문제
$2 \le x \le 4$에서 이차함수 $y = x^{2} -4ax + 4a^{2} + b$의 최솟값이 $4$가 되도록 하는 두 실수 $a$, $b$에 대하여 $2a + b$의 최댓값을 $M$이라 하자. $4M$의 값을 구하시오.

## 보기


## 풀이
$f(x) = x^{2} -4ax + 4a^{2} + b$라 하면 
$f(x) = (x-2a)^{2} + b$ 
이므로 $y = f(x)$의 그래프의 꼭짓점의 좌표는 $(2a, b)$이다. 꼭짓점의 $x$좌표와 주어진 범위에 따라 경우를 나누면 

(ⅰ) $a < 1$일 때 
$f(x)$는 $x = 2$에서 최솟값 $(2-2a)^{2} + b$를 가지므로 
$(2-2a)^{2} + b = 4$ 
$\therefore b = -4(a-1)^{2} + 4$ 

(ⅱ) $1 \le a < 2$일 때 
$f(x)$는 $x = 2a$에서 최솟값 $b$를 가지므로 
$b = 4$ 

(ⅲ) $a \ge 2$일 때 
$f(x)$는 $x = 4$에서 최솟값 $(4-2a)^{2} + b$를 가지므로 
$(4-2a)^{2} + b = 4$ 
$\therefore b = -4(a-2)^{2} + 4$ 

이상에서 
$b = \begin{cases} -4(a-1)^{2} + 4 & (a < 1) \\ 4 & (1 \le a < 2) \\ -4(a-2)^{2} + 4 & (a \ge 2) \end{cases}$

$2a + b = k$라 하면 $b = -2a + k$ 

> [!example] 📸 (그림/그래프)

$b = -4(a-2)^{2} + 4$의 그래프와 직선 $b = -2a + k$가 접할 때 $k$는 최댓값을 갖는다. 방정식 $-4(a-2)^{2} + 4 = -2a + k$, 즉 $4a^{2} - 18a + 12 + k = 0$의 판별식을 $D$라 하면 
$\frac{D}{4} = (-9)^{2} - 4(12 + k) = 0$ 
$\therefore k = \frac{33}{4}$ 

따라서 $M = \frac{33}{4}$이므로 $4M = 33$

정답: $33$
