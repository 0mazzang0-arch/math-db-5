## 문제

$\overline{AB}=8$, $\overline{AC}=6$, $\angle A=90^{\circ}$인 직각삼각형 $ABC$에 내접하는 직사각형을 만들 때, [그림 1]과 같이 직사각형의 두 변이 삼각형의 변 위에 존재하는 경우와 [그림 2]와 같이 직사각형의 한 변만이 삼각형의 변 위에 존재하는 경우가 있다. 

![그림 1]
![그림 2]

[그림 1]과 [그림 2]의 경우에 내접하는 직사각형의 넓이를 각각 $S_{1}$, $S_{2}$라 하자. <보기>에서 옳은 것을 모두 고른 것은?



> [!example] 📸 (그림/그래프)

## 보기

ㄱ. $S_{1}$의 최댓값은 $12$이다.
ㄴ. $S_{1}$이 최대일 때, 직사각형의 둘레의 길이는 $14$이다.
ㄷ. $S_{2}$의 최댓값과 $S_{1}$의 최댓값은 같다.

① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이

ㄱ. 그림과 같이 직사각형의 두 변의 길이를 $x$, $y$라고 하면 $S_{1}=xy$이고, 닮음비에 의해 
$\overline{BC} = \frac{5}{3}x + \frac{5}{4}y = 10$
$\therefore 4x + 3y = 24$
산술·기하평균에서 
$24 = 4x + 3y \ge 2\sqrt{4x \times 3y}$
(단, 등호는 $4x = 3y$일 때 성립)
$12 \ge \sqrt{12xy}$, $144 \ge 12xy$
$S_{1} = xy \le 12$
따라서 $S_{1}$의 최댓값은 $12$이다. (참)

ㄴ. ㄱ에서 $4x = 3y = 12$, 즉 $x = 3$, $y = 4$일 때 $S_{1}$이 최대이다.
따라서 $S_{1}$이 최대일 때 직사각형의 둘레의 길이는 $2(x+y) = 2(3+4) = 14$이다. (참)

ㄷ. 그림과 같이 직사각형의 두 변의 길이를 $x$, $y$라고 하면 $S_{2} = xy$이고, 닮음비에 의해 
$\frac{4}{3}x + y + \frac{3}{4}x = 10$에서
$16x + 12y + 9x = 120$, $25x + 12y = 120$
$120 = 25x + 12y \ge 2\sqrt{25x \times 12y} = 10\sqrt{12xy}$
(단, 등호는 $25x = 12y$일 때 성립)
$12 \ge \sqrt{12xy}$, $144 \ge 12xy$
$S_{2} = xy \le 12$
따라서 $S_{1}$의 최댓값과 $S_{2}$의 최댓값은 $12$로 같다. (참)

따라서 옳은 것은 ㄱ, ㄴ, ㄷ이다.
