## 문제
다음 조건을 만족시키는 모든 자연수의 개수를 구하시오.

(가) 네 자리의 홀수이다.
(나) 각 자리의 수의 합이 $8$ 보다 작다.

## 보기


## 풀이
[정답] $80$

[해설]
조건을 만족시키는 자연수는 각 자리의 수의 합이 $8$ 보다 작은 네 자리의 홀수이므로 일의 자리의 수는 $1$, $3$, $5$ 이다.
이 네 자리의 자연수를 $a \times 10^{3} + b \times 10^{2} + c \times 10 + d$ ($a \neq 0$) 이라 하자.

(ⅰ) $d=1$ 인 경우
$a+b+c \le 6$ ($a \ge 1$) 을 만족시키는 음이 아닌 정수 $a$, $b$, $c$ 의 순서쌍 ($a, b, c$) 의 개수와 같으므로
$a+b+c=6$ ($a \ge 1$) 일 때, ${}_{3}H_{5}$
$a+b+c=5$ ($a \ge 1$) 일 때, ${}_{3}H_{4}$
$\vdots$
$a+b+c=1$ ($a \ge 1$) 일 때, ${}_{3}H_{0}$
${}_{3}H_{5} + {}_{3}H_{4} + \cdots + {}_{3}H_{0} = {}_{7}C_{5} + {}_{6}C_{4} + \cdots + {}_{2}C_{0} = 56$

(ⅱ) $d=3$ 인 경우
$a+b+c \le 4$ ($a \ge 1$) 을 만족시키는 음이 아닌 정수 $a$, $b$, $c$ 의 순서쌍 ($a, b, c$) 의 개수와 같으므로 (ⅰ)과 같은 방법으로
${}_{3}H_{3} + {}_{3}H_{2} + {}_{3}H_{1} + {}_{3}H_{0} = {}_{5}C_{3} + {}_{4}C_{2} + {}_{3}C_{1} + {}_{2}C_{0} = 20$

(ⅲ) $d=5$ 인 경우
$a+b+c \le 2$ ($a \ge 1$) 을 만족시키는 음이 아닌 정수 $a$, $b$, $c$ 의 순서쌍 ($a, b, c$) 의 개수와 같으므로 (ⅰ)과 같은 방법으로
${}_{3}H_{1} + {}_{3}H_{0} = {}_{3}C_{1} + {}_{2}C_{0} = 4$

따라서 조건을 만족시키는 자연수의 개수는 $56 + 20 + 4 = 80$

[다른 풀이]
(ⅰ) $d=1$ 인 경우
$a+b+c \le 6$ ($a \ge 1$) 을 만족시키는 음이 아닌 정수 $a$, $b$, $c$ 의 순서쌍 $a, b, c$ 의 개수는 $a+b+c+e = 6$ ($a \ge 1$) 을 만족시키는 음이 아닌 정수 $a, b, c, e$ 의 순서쌍 ($a, b, c, e$) 의 개수와 같다.
${}_{4}H_{5} = {}_{8}C_{5} = 56$

(ⅱ) $d=3$ 인 경우
$a+b+c \le 4$ ($a \ge 1$) 을 만족시키는 음이 아닌 정수 $a, b, c$ 의 순서쌍 ($a, b, c$) 의 개수는 $a+b+c+e = 4$ ($a \ge 1$) 을 만족시키는 음이 아닌 정수 $a, b, c, e$ 의 순서쌍 ($a, b, c, e$) 의 개수와 같다.
${}_{4}H_{3} = {}_{6}C_{3} = 20$

(ⅲ) $d=5$ 인 경우
$a+b+c \le 2$ ($a \ge 1$) 을 만족시키는 음이 아닌 정수 $a, b, c$ 의 순서쌍 ($a, b, c$) 의 개수는 $a+b+c+e = 2$ ($a \ge 1$) 을 만족시키는 음이 아닌 정수 $a, b, c, e$ 의 순서쌍 ($a, b, c, e$) 의 개수와 같다.
${}_{4}H_{1} = {}_{4}C_{1} = 4$

따라서 조건을 만족시키는 자연수의 개수는 $56 + 20 + 4 = 80$
