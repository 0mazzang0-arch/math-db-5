## 문제
복소수 $z=a+bi$ ($a$, $b$는 실수)에 대하여 등식
$$2z+ \bar{z } = 3+5i$$
가 성립할 때, $a+b$의 값은?
(단, $i= \sqrt { -1}$이고, $\bar{z}$는 $z$의 켤레복소수이다.)

## 보기
① $6$   ② $7$   ③ $8$   ④ $9$   ⑤ $10$

## 풀이
등식 $2z+ \bar{z } = 3+5i$ 에 $z=a+bi$, $\bar{z}=a-bi$ 를 대입하면
$2( a+bi ) + ( a-bi ) = 3+5i$
$3a+bi=3+5i$
$\therefore a=1$, $b=5$
따라서 $a+b=6$이다.

정답: ①
