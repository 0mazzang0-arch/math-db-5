## 문제
연속확률변수 $X$ 가 갖는 값의 범위가 $0 \leq X \leq 3$ 이고, 확률밀도함수의 그래프는 다음과 같다.

> [!example] 📸 (그림/그래프)

$P(m \leq X \leq 2) = P(2 \leq X \leq 3)$ 일 때, $m$ 의 값은? (단, $0 < m < 2$ 이다.)

## 보기
① $\frac{\sqrt{2}}{2}$   ② $\frac{\sqrt{3}}{2}$   ③ $1$   ④ $\sqrt{2}$   ⑤ $\sqrt{3}$

## 풀이
주어진 확률밀도함수의 그래프에서 $P(2 \leq X \leq 3)$ 의 값은 삼각형의 넓이와 같으므로
$P(2 \leq X \leq 3) = \frac{1}{2} \times 1 \times \frac{2}{3} = \frac{1}{3}$

$0 \leq X \leq 2$ 구간에서 확률밀도함수의 식은 $f(x) = \frac{1}{3}x$ 이다.
$P(m \leq X \leq 2) = P(0 \leq X \leq 2) - P(0 \leq X \leq m)$ 이므로
$P(m \leq X \leq 2) = \left( \frac{1}{2} \times 2 \times \frac{2}{3} \right) - \left( \frac{1}{2} \times m \times \frac{m}{3} \right) = \frac{1}{3}$
$\frac{2}{3} - \frac{m^2}{6} = \frac{1}{3}$

$\therefore \frac{m^2}{6} = \frac{1}{3}$
$m^2 = 2$
$0 < m < 2$ 이므로 $m = \sqrt{2}$
