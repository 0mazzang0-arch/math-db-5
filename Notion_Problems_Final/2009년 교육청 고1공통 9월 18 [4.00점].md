## 문제
$S = \{ a, b, c \}$ 의 부분집합을 원소로 갖는 집합 $X$ 가 다음 두 조건을 만족한다.

(가) $A \in X$ 이면 $S - A \in X$
(나) $A \in X, B \in X$ 이면 $A \cup B \in X$

이 때, 집합 $X$ 의 개수는? (단, $X \neq \emptyset$)

## 보기
① $2$   ② $3$   ③ $4$   ④ $5$   ⑤ $6$

## 풀이
$X$ 는 $\emptyset$, $\{ a \}$, $\{ b \}$, $\{ c \}$, $\{ a, b \}$, $\{ a, c \}$, $\{ b, c \}$, $S$ 의 일부를 원소로 하고 주어진 조건을 만족하는 집합은 다음과 같다.

- $\{ S, \emptyset \}$
- $\{ \{ a \}, \{ b, c \}, S, \emptyset \}$
- $\{ \{ b \}, \{ a, c \}, S, \emptyset \}$
- $\{ \{ c \}, \{ a, b \}, S, \emptyset \}$
- $\{ \{ a \}, \{ b \}, \{ c \}, \{ a, b \}, \{ a, c \}, \{ b, c \}, S, \emptyset \}$

그러므로 만족하는 집합 $X$ 의 개수는 $5$ 이다.

정답: ④
