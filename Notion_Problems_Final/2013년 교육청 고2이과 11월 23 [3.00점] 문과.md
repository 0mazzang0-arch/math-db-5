## 문제
수열 $\{ a_{n} \}$ 의 일반항이 $a_{n} = 2 n^{2} + n$ 일 때, 수열 $\{ a_{n} \}$ 의 계차수열을 $\{ b_{n} \}$ 이라 하자. $\sum_{n=1}^{10} b_{n}$ 의 값을 구하시오.

## 보기

## 풀이
$b_{n} = a_{n+1} - a_{n}$
$= 2 (n+1)^{2} + (n+1) - 2 n^{2} - n = 4 n + 3$
따라서 $\sum_{n=1}^{10} b_{n} = \sum_{n=1}^{10} (4 n + 3) = 250$

[다른 풀이]
$a_{11} = a_{1} + \sum_{n=1}^{10} b_{n}$ 에서
$\sum_{n=1}^{10} b_{n} = a_{11} - a_{1} = 250$

정답: $250$
