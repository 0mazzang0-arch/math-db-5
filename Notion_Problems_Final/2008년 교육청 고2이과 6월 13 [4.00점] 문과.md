## 문제
실수 전체의 집합 $R$에서 연산 $\odot$를 $a \odot b = a b + 2 ( a + b ) + 2$로 정의한다. 연산 $\odot$에 대한 실수 $x$의 역원을 $f ( x )$라 할 때, $y = f ( x )$의 그래프의 개형은?

## 풀이
연산 $\odot$에 대한 항등원을 $e$라 하면, 모든 실수 $a$에 대하여 $a \odot e = e \odot a = a$를 만족하는 항등원은 $e = - 1$이다. 
따라서 $x \odot f ( x ) = f ( x ) \odot x = - 1$
$x f ( x ) + 2 x + 2 f ( x ) + 2 = - 1$
$( x + 2 ) f ( x ) = - 2 x - 3$
$x \neq - 2$일 때, 
$f ( x ) = \frac{- 2 x - 3}{x + 2} = \frac{- 2 ( x + 2 ) + 1}{x + 2} = \frac{1}{x + 2} - 2$
따라서 $y = f ( x )$는 $x = - 2$, $y = - 2$를 점근선으로 하는 유리함수이다.
