## 문제
$0 \le x < 2\pi$ 일 때, 방정식 $\sin 4x = \frac{1}{2}$ 의 서로 다른 실근의 개수는?

## 보기
① $2$   ② $4$   ③ $6$   ④ $8$   ⑤ $10$

## 풀이
$y = \sin 4x$ 의 주기는 $\frac{2\pi}{|4|} = \frac{\pi}{2}$ 이다.
좌표평면에 직선 $y = \frac{1}{2}$ 과 $0 \le x < 2\pi$ 의 범위에서 함수 $y = \sin 4x$ 의 그래프를 그리면 다음과 같다.

> [!example] 📸 (그림/그래프)

따라서 구하는 서로 다른 실근의 개수는 $8$ 이다.
