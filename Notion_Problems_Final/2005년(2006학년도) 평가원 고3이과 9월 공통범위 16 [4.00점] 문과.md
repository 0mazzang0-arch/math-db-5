## 문제
세 확률변수 $X$, $Y$, $W$는 각각 다음과 같다.
$$ \begin{array}{l} 
(가) \text{ } X \text{는 이항분포 } B \left( 100, \frac{1}{5} \right) \text{을 따른다.} \\ 
(나) \text{ } Y \text{는 이항분포 } B \left( 225, \frac{1}{5} \right) \text{을 따른다.} \\ 
(다) \text{ } W \text{는 이항분포 } B \left( 400, \frac{1}{5} \right) \text{을 따른다.} 
\end{array} $$
위의 내용에 대하여 <보기>에서 옳은 것을 모두 고른 것은?
$$ \begin{array}{l} 
ㄱ. \text{ } P \left( \left| \frac{X}{100} - \frac{1}{5} \right| < \frac{1}{10} \right) < P \left( \left| \frac{W}{400} - \frac{1}{5} \right| < \frac{1}{10} \right) \\ 
ㄴ. \text{ } P \left( \left| \frac{X}{100} - \frac{1}{5} \right| < \frac{1}{10} \right) < P \left( \left| \frac{Y}{225} - \frac{1}{5} \right| < \frac{1}{25} \right) \\ 
ㄷ. \text{ } P \left( \left| \frac{Y}{225} - \frac{1}{5} \right| < \frac{1}{25} \right) < P \left( \left| \frac{W}{400} - \frac{1}{5} \right| < \frac{1}{25} \right) 
\end{array} $$

## 보기
① $ㄱ$   ② $ㄷ$   ③ $ㄱ, ㄷ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
세 확률변수 $X$, $Y$, $W$의 분포와 정규분포 근사를 정리하면 다음과 같다.

$1$. 확률변수 $X$는 이항분포 $B \left( 100, \frac{1}{5} \right)$를 따르므로,
$E(X) = 100 \times \frac{1}{5} = 20$, $V(X) = 100 \times \frac{1}{5} \times \frac{4}{5} = 4^2$이다.
따라서 $X$는 근사적으로 정규분포 $N(20, 4^2)$을 따른다.
$P \left( \left| \frac{X}{100} - \frac{1}{5} \right| < \frac{1}{10} \right) = P(|X - 20| < 10) = P \left( \left| \frac{X-20}{4} \right| < \frac{10}{4} \right) = P(|Z| < 2.5)$

$2$. 확률변수 $Y$는 이항분포 $B \left( 225, \frac{1}{5} \right)$를 따르므로,
$E(Y) = 225 \times \frac{1}{5} = 45$, $V(Y) = 225 \times \frac{1}{5} \times \frac{4}{5} = 6^2$이다.
따라서 $Y$는 근사적으로 정규분포 $N(45, 6^2)$을 따른다.
$P \left( \left| \frac{Y}{225} - \frac{1}{5} \right| < \frac{1}{25} \right) = P(|Y - 45| < 9) = P \left( \left| \frac{Y-45}{6} \right| < \frac{9}{6} \right) = P(|Z| < 1.5)$

$3$. 확률변수 $W$는 이항분포 $B \left( 400, \frac{1}{5} \right)$를 따르므로,
$E(W) = 400 \times \frac{1}{5} = 80$, $V(W) = 400 \times \frac{1}{5} \times \frac{4}{5} = 8^2$이다.
따라서 $W$는 근사적으로 정규분포 $N(80, 8^2)$을 따른다.
$P \left( \left| \frac{W}{400} - \frac{1}{5} \right| < \frac{1}{25} \right) = P(|W - 80| < 16) = P \left( \left| \frac{W-80}{8} \right| < \frac{16}{8} \right) = P(|Z| < 2)$
$P \left( \left| \frac{W}{400} - \frac{1}{5} \right| < \frac{1}{10} \right) = P(|W - 80| < 40) = P \left( \left| \frac{W-80}{8} \right| < \frac{40}{8} \right) = P(|Z| < 5)$

이제 <보기>의 각 문항을 비교하면 다음과 같다.
ㄱ. $P(|Z| < 2.5) < P(|Z| < 5)$ 이므로 참이다.
ㄴ. $P(|Z| < 2.5) < P(|Z| < 1.5)$ 이므로 거짓이다.
ㄷ. $P(|Z| < 1.5) < P(|Z| < 2)$ 이므로 참이다.

따라서 옳은 것은 ㄱ, ㄷ이다.
