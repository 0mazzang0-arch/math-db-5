## 문제
두 수열 $\{ a_{n} \}$, $\{ b_{n} \}$을 모든 자연수 $n$에 대하여 
$$ a_{n} = \log_{2} ( n ! ) $$ 
$$ b_{n} = a_{n+1} - a_{n} $$ 
으로 정의할 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?

---
**<보기>**
ㄱ. $b_{15} = 4$
ㄴ. $9 < \sum_{k=1}^{5} b_{k} < 10$
ㄷ. $n$이 짝수일 때 $b_{n}$의 값은 무리수이다.
---

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
$b_{n} = a_{n+1} - a_{n} = \log_{2} \frac{( n + 1 ) ! }{ n ! } = \log_{2} ( n + 1 )$

ㄱ. $b_{15} = \log_{2} ( 15 + 1 ) = \log_{2} 16 = 4$ (참)

ㄴ. $\sum_{k=1}^{5} b_{k} = \log_{2} 2 + \log_{2} 3 + \log_{2} 4 + \log_{2} 5 + \log_{2} 6 = \log_{2} ( 2 \times 3 \times 4 \times 5 \times 6 ) = \log_{2} 720$
$2^{9} = 512$, $2^{10} = 1024$이므로
$\log_{2} 2^{9} < \log_{2} 720 < \log_{2} 2^{10}$
$9 < \log_{2} 720 < 10$ (참)

ㄷ. $n$이 짝수이면 $n + 1$은 $3$ 이상의 홀수이다.
$b_{n} = \log_{2} ( n + 1 )$에서 $n + 1$이 홀수이면 $b_{n}$은 유리수가 될 수 없다. ($b_{n} = \frac{p}{q}$라 하면 $2^{\frac{p}{q}} = n + 1 \Rightarrow 2^{p} = ( n + 1 )^{q}$가 되어 모순)
따라서 $b_{n}$은 무리수이다. (참)

ㄱ, ㄴ, ㄷ 모두 옳으므로 정답은 ⑤이다.
