## 문제
공차가 정수인 등차수열 $\{a_{n}\}$이 다음 조건을 만족시킨다.

(가) $a_{7} = 37$
(나) 모든 자연수 $n$에 대하여 $\sum_{k=1}^{n} a_{k} \leq \sum_{k=1}^{13} a_{k}$이다.

$\sum_{k=1}^{21} | a_{k} |$의 값은?

## 보기
① $681$   ② $683$   ③ $685$   ④ $687$   ⑤ $689$

## 풀이
등차수열 $\{a_{n}\}$의 공차를 $d$라 하면
조건 (가)에 의하여 $a_{7} = a_{1} + 6d = 37$
조건 (나)에 의하여 $a_{13} \geq 0$이고, $a_{14} \leq 0$이다.

$a_{1} + 12d \geq 0$, $37 + 6d \geq 0$, $-\frac{37}{6} \leq d$
$a_{1} + 13d \leq 0$, $37 + 7d \leq 0$, $d \leq -\frac{37}{7}$

$-\frac{37}{6} \leq d \leq -\frac{37}{7}$이고 $d$는 정수이므로
$d = -6$, $a_{1} = 73$
따라서

$\sum_{k=1}^{21} | a_{k} | = | a_{1} | + | a_{2} | + \cdots + | a_{21} |$
$= a_{1} + a_{2} + \cdots + a_{13} + ( -a_{14} ) + ( -a_{15} ) + \cdots + ( -a_{21} )$
$= ( a_{1} + a_{2} + \cdots + a_{13} ) - ( a_{14} + a_{15} + \cdots + a_{21} )$
$= \sum_{k=1}^{13} a_{k} - ( \sum_{k=1}^{21} a_{k} - \sum_{k=1}^{13} a_{k} )$
$= 2 \sum_{k=1}^{13} a_{k} - \sum_{k=1}^{21} a_{k}$
$= 2 \times \frac{13 \{ 2 \times 73 + 12 \times ( -6 ) \}}{2} - \frac{21 \{ 2 \times 73 + 20 \times ( -6 ) \}}{2}$
$= 689$
