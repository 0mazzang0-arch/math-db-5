## 문제
양의 정수 $n$에 대하여 $\log n$의 정수부분을 $f(n)$, 소수부분을 $g(n)$이라 할 때, 다음 조건을 만족시키는 양의 정수 $n$의 개수는?

(가) $f(3) < f(n) < f(2011)$
(나) $\{g(n)\}^2 - g(n) + \log 2 \times \log 5 < 0$

## 보기
① $326$   ② $328$   ③ $330$   ④ $332$   ⑤ $334$

## 풀이
(가)에서 
$3$은 한 자리의 양의 정수이므로 $f(3) = 0$, 
$2011$은 네 자리의 양의 정수이므로 $f(2011) = 3$이다.
즉, $0 < f(n) < 3$이므로
$f(n) = 1$ 또는 $f(n) = 2$이다.

(나)에서 주어진 식의 좌변을 인수분해하면 
$\{g(n) - \log 2\}\{g(n) - \log 5\} < 0$
$\therefore \log 2 < g(n) < \log 5$

이때 $\log n = f(n) + g(n)$이므로 
ⅰ) $f(n) = 1$일 때 
$1 + \log 2 < f(n) + g(n) < 1 + \log 5$
$\therefore \log 20 < \log n < \log 50$
따라서 양의 정수 $n$은 $21, 22, \dots, 49$로 $29$개다. 

ⅱ) $f(n) = 2$일 때 
$2 + \log 2 < f(n) + g(n) < 2 + \log 5$
$\therefore \log 200 < \log n < \log 500$
따라서 양의 정수 $n$은 $201, 202, \dots, 499$로 $299$개다. 

ⅰ), ⅱ)에 의하여 양의 정수 $n$의 개수는 
$29 + 299 = 328$이다.

정답은 ②이다.
