## 문제
두 집합 $A = \{ 1, a^3 - 3a \}$, $B = \{ a + 2, a^2 - a \}$에 대하여 $A \cap B = \{ 2 \}$가 되도록 상수 $a$의 값을 정할 때, 집합 $A \cup B$의 모든 원소의 합은?

## 보기
① $3$   ② $4$   ③ $5$   ④ $6$   ⑤ $7$

## 풀이
$A \cap B = \{ 2 \}$이므로 $2 \in A$이다.
$A = \{ 1, a^3 - 3a \}$에서
$a^3 - 3a = 2$
$a^3 - 3a - 2 = 0$
$(a + 1)^2 (a - 2) = 0$
따라서 $a = -1$ 또는 $a = 2$

(i) $a = -1$일 때,
$B = \{ a + 2, a^2 - a \} = \{ 1, 2 \}$
이때 $A = \{ 1, 2 \}$이므로 $A \cap B = \{ 1, 2 \}$가 되어 조건에 적당하지 않다.

(ii) $a = 2$일 때,
$B = \{ a + 2, a^2 - a \} = \{ 4, 2 \}$
이때 $A = \{ 1, 2 \}$이므로 $A \cap B = \{ 2 \}$가 되어 조건에 적당하다.

(i), (ii)에서 $a = 2$이고 $A \cup B = \{ 1, 2, 4 \}$이다.
따라서 $A \cup B$의 모든 원소의 합은 $1 + 2 + 4 = 7$이다.

정답: ⑤
