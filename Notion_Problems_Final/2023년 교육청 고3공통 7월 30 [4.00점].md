## 문제
집합 $X = \{ 1, 2, 3, 4, 5, 6, 7 \}$ 에 대하여 다음 조건을 만족시키는 함수 $f : X \to X$ 의 개수를 구하시오.

(가) $f(7) - f(1) = 3$
(나) $5$ 이하의 모든 자연수 $n$ 에 대하여 $f(n) \le f(n + 2)$ 이다.
(다) $\frac{1}{3} | f(2) - f(1) |$ 과 $\frac{1}{3} \sum_{k=1}^{4} f(2k - 1)$ 의 값은 모두 자연수이다.

## 보기

## 풀이
[정답] $150$

조건 (가)에 의하여 순서쌍 $(f(1), f(7))$ 은
$(1, 4)$, $(2, 5)$, $(3, 6)$, $(4, 7)$
조건 (나)에 의하여
$f(1) \le f(3) \le f(5) \le f(7)$
이고
$f(2) \le f(4) \le f(6)$
조건 (다)에 의하여 $| f(2) - f(1) |$ 과 $f(1) + f(3) + f(5) + f(7)$ 의 값은 모두 $3$ 의 배수인 자연수이다.

(ⅰ) $f(1) = 1$, $f(7) = 4$ 인 경우
$f(3) + f(5) = 4$ 또는 $f(3) + f(5) = 7$
$(f(3), f(5))$ 는 $(1, 3)$, $(2, 2)$, $(3, 4)$ ($3$ 가지)
$f(1) = 1$ 이므로 $f(2) = 4$ 또는 $f(2) = 7$
$f(2) = 4$ 이면 $(f(4), f(6))$ 의 개수는 ${}_{4}\text{H}_{2} = 10$
$f(2) = 7$ 이면 $(f(4), f(6))$ 의 개수는 ${}_{1}\text{H}_{2} = 1$
${}_{4}\text{H}_{2} + {}_{1}\text{H}_{2} = 10 + 1 = 11$
$3 \times 11 = 33$

(ⅱ) $f(1) = 2$, $f(7) = 5$ 인 경우
$f(3) + f(5) = 5$ 또는 $f(3) + f(5) = 8$
$(f(3), f(5))$ 는 $(2, 3)$, $(3, 5)$, $(4, 4)$ ($3$ 가지)
$f(1) = 2$ 이므로 $f(2) = 5$
$(f(4), f(6))$ 의 개수는 ${}_{3}\text{H}_{2} = 6$
$3 \times 6 = 18$

(ⅲ) $f(1) = 3$, $f(7) = 6$ 인 경우
$f(3) + f(5) = 6$ 또는 $f(3) + f(5) = 9$ 또는 $f(3) + f(5) = 12$
$(f(3), f(5))$ 는 $(3, 3)$, $(3, 6)$, $(4, 5)$, $(6, 6)$ ($4$ 가지)
$f(1) = 3$ 이므로 $f(2) = 6$
$(f(4), f(6))$ 의 개수는 ${}_{2}\text{H}_{2} = 3$
$4 \times 3 = 12$

(ⅳ) $f(1) = 4$, $f(7) = 7$ 인 경우
$f(3) + f(5) = 10$ 또는 $f(3) + f(5) = 13$
$(f(3), f(5))$ 는 $(4, 6)$, $(5, 5)$, $(6, 7)$ ($3$ 가지)
$f(1) = 4$ 이므로 $f(2) = 1$ 또는 $f(2) = 7$
$f(2) = 1$ 이면 $(f(4), f(6))$ 의 개수는 ${}_{7}\text{H}_{2} = 28$
$f(2) = 7$ 이면 $(f(4), f(6))$ 의 개수는 ${}_{1}\text{H}_{2} = 1$
${}_{7}\text{H}_{2} + {}_{1}\text{H}_{2} = 28 + 1 = 29$
$3 \times 29 = 87$

따라서 (ⅰ), (ⅱ), (ⅲ), (ⅳ)에 의하여
$33 + 18 + 12 + 87 = 150$
