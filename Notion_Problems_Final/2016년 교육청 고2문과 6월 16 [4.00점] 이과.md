## 문제
등차수열 $\{ a_{n} \}$에 대하여 $a_{1} = 1$, $a_{2} + a_{4} = 18$이다. $S_{n} = \sum_{k=1}^{n} a_{k}$라 할 때, $\lim_{n \to \infty} ( \sqrt{S_{n+1}} - \sqrt{S_{n}} )$의 값은?

## 보기
① $\sqrt{2}$   ② $2\sqrt{2}$   ③ $3\sqrt{2}$   ④ $4\sqrt{2}$   ⑤ $5\sqrt{2}$

## 풀이
$a_{1} = 1$,
$a_{2} + a_{4} = ( a_{1} + d ) + ( a_{1} + 3d ) = 2a_{1} + 4d = 18$에서
$d = 4$이다.

$S_{n} = \frac{n \{ 2a_{1} + ( n - 1 ) d \}}{2} = \frac{n \{ 2 + 4 ( n - 1 ) \}}{2} = 2n^{2} - n$이고,
$S_{n+1} = 2 ( n + 1 )^{2} - ( n + 1 ) = 2n^{2} + 3n + 1$이다.

$\lim_{n \to \infty} ( \sqrt{S_{n+1}} - \sqrt{S_{n}} )$
$= \lim_{n \to \infty} ( \sqrt{2n^{2} + 3n + 1} - \sqrt{2n^{2} - n} )$
$= \lim_{n \to \infty} \frac{( \sqrt{2n^{2} + 3n + 1} - \sqrt{2n^{2} - n} ) ( \sqrt{2n^{2} + 3n + 1} + \sqrt{2n^{2} - n} )}{\sqrt{2n^{2} + 3n + 1} + \sqrt{2n^{2} - n}}$
$= \lim_{n \to \infty} \frac{4n + 1}{\sqrt{2n^{2} + 3n + 1} + \sqrt{2n^{2} - n}}$
$= \frac{4}{2\sqrt{2}}$
$= \sqrt{2}$

[다른 풀이]
공차 $d$를 다음과 같이 구할 수도 있다.
$2a_{3} = a_{2} + a_{4} = 18$이므로 $a_{3} = 9$이다.
$a_{3} = a_{1} + 2d = 1 + 2d = 9$
$d = 4$이다.
