## 문제
모든 항이 실수이고 공비가 $0$이 아닌 등비수열 $\{ a_{n} \}$에 대하여 옳은 것만을 <보기>에서 있는 대로 고른 것은?

$$
\begin{array}{|l|}
\hline
\text{<보 기>} \\
\text{ㄱ. } a_{1} < 0 \text{ 이면 } a_{101} < 0 \text{ 이다.} \\
\text{ㄴ. } a_{1} < a_{2} < 1 \text{ 이면 } \lim_{n \to \infty} a_{n} = 0 \text{ 이다.} \\
\text{ㄷ. } a_{1} < a_{2} < 0 \text{ 이면 } \sum_{n=1}^{\infty} a_{n} < a_{1} \text{ 이다.} \\
\hline
\end{array}
$$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄷ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
등비수열 $\{ a_{n} \}$의 공비를 $r$ ($r \neq 0$)라 하면

ㄱ. $a_{101} = a_{1} r^{100}$이다. $a_{1} < 0$이고 $r^{100} > 0$이므로 $a_{101} < 0$이다. (참)

ㄴ. (반례) $a_{n} = 2^{n-3}$이라 하면 $a_{1} = \frac{1}{4}$, $a_{2} = \frac{1}{2}$이므로 $a_{1} < a_{2} < 1$을 만족하지만 $\lim_{n \to \infty} a_{n} = \infty$이다. (거짓)

ㄷ. $a_{1} < a_{2} < 0$이므로 $a_{1} < a_{1} r < 0$이다. $a_{1} < 0$이므로 양변을 $a_{1}$로 나누면 부동호 방향이 바뀌어 $1 > r > 0$이다.
$0 < r < 1$이므로 $\frac{1}{1-r} > 1$이다.
이때 급수의 합 $\sum_{n=1}^{\infty} a_{n} = \frac{a_{1}}{1-r}$에서 $a_{1} < 0$이고 $\frac{1}{1-r} > 1$이므로 $\frac{a_{1}}{1-r} < a_{1}$이다.
따라서 $\sum_{n=1}^{\infty} a_{n} < a_{1}$이다. (참)

따라서 옳은 것은 ㄱ, ㄷ이다.
