## 문제
$x$에 대한 연립이차부등식 
$$ \left\{ \begin{array}{l} x^{2} - 10x + 21 \le 0 \\ x^{2} - 2(n - 1)x + n^{2} - 2n \ge 0 \end{array} \right. $$
을 만족시키는 정수 $x$의 개수가 $4$가 되도록 하는 모든 자연수 $n$의 값의 합을 구하시오.

## 보기

## 풀이
연립이차부등식 
$$ \left\{ \begin{array}{l} x^{2} - 10x + 21 \le 0 \\ x^{2} - 2(n - 1)x + n^{2} - 2n \ge 0 \end{array} \right. $$
$x^{2} - 10x + 21 \le 0$에서
$(x - 3)(x - 7) \le 0$
$3 \le x \le 7$

$x^{2} - 2(n - 1)x + n^{2} - 2n \ge 0$에서
$\{x - (n - 2)\}(x - n) \ge 0$
$x \le n - 2$ 또는 $x \ge n$

$$ \begin{array}{l} (ⅰ) \text{ } 1 \le n \le 3 \text{ 인 경우 } x \text{ 의 개수는 } 5 \\ (ⅱ) \text{ } 4 \le n \le 8 \text{ 인 경우 } x \text{ 의 개수는 } 4 \\ (ⅲ) \text{ } n \ge 9 \text{ 인 경우 } x \text{ 의 개수는 } 5 \end{array} $$

따라서 연립이차부등식을 만족시키는 정수 $x$의 개수가 $4$가 되도록 하는 모든 자연수 $n$은 $4, 5, 6, 7, 8$이다.
그 값의 합은
$4 + 5 + 6 + 7 + 8 = 30$

정답: $30$
