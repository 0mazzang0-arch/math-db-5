## 문제
부등식 $(\log_{\frac{1}{2}} x - 2) \log_{\frac{1}{4}} x < 4$를 만족시키는 자연수 $x$의 개수는?

## 보기
① $1$   ② $3$   ③ $5$   ④ $7$   ⑤ $9$

## 풀이
로그의 진수 조건에서 $x > 0$이다.
$(\log_{\frac{1}{2}} x - 2) \log_{\frac{1}{4}} x < 4$에서 $\log_{\frac{1}{2}} x = t$라 하면 $\log_{\frac{1}{4}} x = \frac{1}{2} \log_{\frac{1}{2}} x = \frac{1}{2} t$이므로
$\frac{1}{2} t (t - 2) < 4$
$t^2 - 2t < 8$
$t^2 - 2t - 8 < 0$
$(t + 2) (t - 4) < 0$
$-2 < t < 4$이다.
$t = \log_{\frac{1}{2}} x$를 대입하면
$-2 < \log_{\frac{1}{2}} x < 4$
$(\frac{1}{2})^4 < x < (\frac{1}{2})^{-2}$
$\frac{1}{16} < x < 4$이다.
이를 만족시키는 자연수 $x$는 $1$, $2$, $3$이므로 그 개수는 $3$이다.

정답: ②
