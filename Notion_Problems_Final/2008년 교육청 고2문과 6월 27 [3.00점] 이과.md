## 문제
이차항의 계수가 $1$인 이차함수 $y=f(x)$와 직선 $y=k$의 교점이 $(1, k)$, $(11, k)$이다. 부등식 $f(x) < f(2) - 2$의 해가 $\alpha < x < \beta$일 때, $\alpha \beta$의 값을 구하시오.

## 풀이
$f(1) = f(11) = k$이므로 
$f(x) = (x - 1)(x - 11) + k = x^{2} - 12x + 11 + k$
따라서 부등식 $f(x) < f(2) - 2$에서 
$x^{2} - 12x + 11 + k < -11 + k$
$\therefore x^{2} - 12x + 22 < 0$
이 부등식의 해가 $\alpha < x < \beta$이므로 방정식 $x^{2} - 12x + 22 = 0$의 두 근이 $\alpha$, $\beta$이다. 근과 계수의 관계에 의하여 $\alpha \beta = 22$이다.

정답: $22$
