## 문제
정수 $n$에 대하여 점 $(a, 0)$에서 곡선 $y=(x-n)e^{x}$에 그은 접선의 개수를 $f(n)$이라 하자. <보기>에서 옳은 것만을 있는 대로 고른 것은?

$$ \begin{array}{|l|} \hline \text{<보기>} \\ \text{ㄱ. } a=0\text{일 때, } f(4)=1\text{이다.} \\ \text{ㄴ. } f(n)=1\text{인 정수 } n\text{의 개수가 } 1\text{인 정수 } a\text{가 존재한다.} \\ \text{ㄷ. } \sum_{n=1}^{5} f(n)=5\text{를 만족시키는 정수 } a\text{의 값은 } -1\text{ 또는 } 3\text{이다.} \\ \hline \end{array} $$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄷ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
[정답] ③ 

점 $(a, 0)$에서 그은 접선이 곡선 $y=(x-n)e^{x}$과 만나는 점의 좌표를 $(t, (t-n)e^{t})$라 하자. 

$y' = e^{x} + (x-n)e^{x} = (x-n+1)e^{x}$이므로 점 $(t, (t-n)e^{t})$에서 이 곡선에 그은 접선의 방정식은 
$y = (t-n+1)e^{t}(x-t) + (t-n)e^{t}$ 
이 직선이 점 $(a, 0)$을 지나므로 
$0 = (t-n+1)e^{t}(a-t) + (t-n)e^{t}$ 
$t^{2} - (n+a)t + an+n-a = 0$ 
이 방정식의 판별식을 $D$라 하면 
$D = (n+a)^{2} - 4(an+n-a) = (n-a)(n-a-4)$ 

ㄱ. $a=0$일 때 $n=4$이면 $D=0$이므로 점 $(0, 0)$에서 $y=(x-4)e^{x}$에 그은 접선의 개수는 $1$이다. 
$f(4)=1$

ㄴ. $D = (n-a)(n-a-4) = 0$에서 $n=a$ 또는 $n=a+4$이므로 $f(n)=1$인 정수 $n$의 개수는 항상 $2$이다. 

ㄷ. $a$에 대하여 $f(n)$은 
$$ \begin{array}{l} f(n) = \left\{ \begin{array}{ll} 0 & (a < n < a+4) \\ 1 & (n = a \text{ 또는 } n = a+4) \\ 2 & (n < a \text{ 또는 } n > a+4) \end{array} \right. \end{array} $$ 
$f(n)$이 가질 수 있는 값은 $0$, $1$, $2$뿐이다. 
$\sum_{n=1}^{5} f(n)=5$이므로 가능한 경우는 다음과 같다. 
(ⅰ) $f(1)=0$, $f(2)=0$, $f(3)=1$, $f(4)=2$, $f(5)=2$인 경우는 $3=a+4$, $a=-1$ 
(ⅱ) $f(1)=2$, $f(2)=2$, $f(3)=1$, $f(4)=0$, $f(5)=0$인 경우는 $3=a$, $a=3$ 
따라서 $a=-1$ 또는 $a=3$ 

이상에서 옳은 것은 ㄱ, ㄷ이다.
