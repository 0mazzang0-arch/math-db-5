## 문제
두 실수 $a$, $b$와 최고차항의 계수가 $1$인 삼차함수 $f(x)$에 대하여 함수 $g(x)$를 
$$ g(x) = \left\{ \begin{array}{ll} a & (x < -1) \\ |f(x)| & (-1 \le x \le 5) \\ b & (x > 5) \end{array} \right. $$
라 하자. $g(x)$가 $x = -1$, $x = 5$에서 미분가능할 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?



> [!example] 📸 (그림/그래프)

## 보기
ㄱ. $f(x)$는 $x = -1$에서 극댓값을 갖는다.
ㄴ. $f(9) = 0$이면 $a > b$이다.
ㄷ. $a = b$이면 $f(0) = 46$이다.

① ㄱ   ② ㄴ   ③ ㄱ, ㄷ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
정답: ③

ㄱ. $x = -1$, $x = 5$에서 $g(x)$가 미분가능하다면 상수함수와 만나는 점에서의 미분계수는 $0$이어야 하므로 $g'(-1) = g'(5) = 0$이다.
즉, $f'(-1) = f'(5) = 0$이며, 최고차항의 계수가 $1$인 삼차함수 $f(x)$의 도함수는 $f'(x) = 3(x+1)(x-5)$이다.
도함수의 부호가 $x = -1$ 좌우에서 양($+$)에서 음($-$)으로 변하므로, $f(x)$는 $x = -1$에서 극댓값을 갖는다. (참)

ㄴ. $f'(x) = 3(x+1)(x-5) = 3x^2 - 12x - 15$를 적분하면
$f(x) = x^3 - 6x^2 - 15x + C$이다.
$f(9) = 0$이면 $9^3 - 6 \cdot 9^2 - 15 \cdot 9 + C = 729 - 486 - 135 + C = 108 + C = 0$에서 $C = -108$이다.
$f(x) = x^3 - 6x^2 - 15x - 108$이므로
$f(-1) = -1 - 6 + 15 - 108 = -100$
$f(5) = 125 - 150 - 75 - 108 = -208$
미분가능 조건에 의해 $a = |f(-1)| = 100$, $b = |f(5)| = 208$이다.
따라서 $a < b$이므로 ㄴ은 거짓이다. (거짓)

ㄷ. $f(x) = x^3 - 6x^2 - 15x + C$에서
$f(-1) = 8 + C$, $f(5) = -100 + C$이다.
$g(x)$가 미분가능하기 위해서는 극대, 극소 지점에서 꺾이지 않고 상수함수와 연결되어야 하므로 $a = |f(-1)|$, $b = |f(5)|$이다.
삼차함수의 그래프 개형상 $a = f(-1) = 8 + C$, $b = -f(5) = -(-100 + C) = 100 - C$가 성립해야 한다.
$a = b$이면 $8 + C = 100 - C$
$2C = 92$
$\therefore C = 46$
따라서 $f(0) = C = 46$이다. (참)
