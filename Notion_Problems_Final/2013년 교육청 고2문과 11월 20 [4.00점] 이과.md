## 문제
두 이차정사각행렬 $A$, $B$ 가
$$ A B + B = A, \quad A B A - A^{2} = E $$
를 만족시킬 때, <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $E$ 는 단위행렬이다.)

$$ \begin{array}{|l|} \hline \text{ㄱ. } A B = B A \\ \text{ㄴ. } A^{3} B^{3} = E \\ \text{ㄷ. } (A - E)^{30} = -3^{15} E \\ \hline \end{array} $$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄷ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
[정답] ③

ㄱ. $A B A - A^{2} = E$ 에서
$(A B - A) A = E \cdots ㉠$
$A (B A - A) = E$ 이고 양변의 왼쪽에 $(A B - A)$ 를 곱하면
$(A B - A) A (B A - A) = (A B - A) E$
$E (B A - A) = A B - A$
$B A - A = A B - A$
$\therefore A B = B A$ (참)

ㄴ. $A B + B = A$ 이므로 $A B - A = -B$
$㉠$ 에서 $(-B) A = E$
ㄱ에 의해 $A B = B A$ 이므로 $A B = -E$
$A^{3} B^{3} = (A B)^{3} = (-E)^{3} = -E$ (거짓)

ㄷ. $A B = -E$ 이므로 $A B A - A^{2} = E$ 에서
$(-E) A - A^{2} = E$
$A^{2} + A + E = O \cdots ㉡$
양변에 $(A - E)$ 를 곱하면
$(A - E)(A^{2} + A + E) = O$
$A^{3} - E = O \quad \therefore A^{3} = E$
$㉡$ 에서 $A^{2} = -A - E$ 이므로
$(A - E)^{2} = A^{2} - 2 A + E = (-A - E) - 2 A + E = -3 A$
$(A - E)^{30} = \{ (A - E)^{2} \}^{15} = (-3 A)^{15} = -3^{15} A^{15}$
$A^{3} = E$ 이므로 $A^{15} = (A^{3})^{5} = E^{5} = E$
$\therefore (A - E)^{30} = -3^{15} E$ (참)

따라서 옳은 것은 $ㄱ, ㄷ$ 이다.
