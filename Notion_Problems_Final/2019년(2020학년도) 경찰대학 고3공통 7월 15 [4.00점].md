## 문제
두 곡선 $y = x^{3} + 4x^{2} - 6x + 5$, $y = x^{3} + 5x^{2} - 9x + 6$이 만나는 점의 $x$ 좌표를 $\alpha$, $\beta$ ($\alpha < \beta$)라 할 때, 곡선 $y = 6x^{5} + 4x^{3} + 1$과 두 직선 $x = \alpha$, $x = \beta$와 $x$축으로 둘러싸인 부분의 넓이는 $a\sqrt{5}$이다. 자연수 $a$의 값은?

## 보기
① $160$   ② $162$   ③ $164$   ④ $166$   ⑤ $168$

## 풀이
두 곡선의 교점의 $x$ 좌표를 구하면 다음과 같다.
$$x^{3} + 4x^{2} - 6x + 5 = x^{3} + 5x^{2} - 9x + 6$$
$$x^{2} - 3x + 1 = 0$$

이 방정식의 두 근이 $\alpha$, $\beta$이므로 근과 계수의 관계에 의해
$$\alpha + \beta = 3, \alpha \beta = 1$$
$$(\beta - \alpha)^{2} = (\beta + \alpha)^{2} - 4 \alpha \beta = 3^{2} - 4(1) = 5$$
$\alpha < \beta$이므로
$$\beta - \alpha = \sqrt{5}$$

한편, $x^{2} - 3x + 1 = 0$에서 $x^{2} = 3x - 1$이므로 차수를 낮추면
$$x^{4} = (3x - 1)^{2} = 9x^{2} - 6x + 1 = 9(3x - 1) - 6x + 1 = 21x - 8$$
$$x^{6} = x^{2} \cdot x^{4} = (3x - 1)(21x - 8) = 63x^{2} - 45x + 8 = 63(3x - 1) - 45x + 8 = 144x - 55$$

따라서 $\alpha, \beta$에 대하여 다음이 성립한다.
$$\alpha^{2} = 3\alpha - 1, \alpha^{4} = 21\alpha - 8, \alpha^{6} = 144\alpha - 55$$
$$\beta^{2} = 3\beta - 1, \beta^{4} = 21\beta - 8, \beta^{6} = 144\beta - 55$$

구하는 넓이 $S$는 다음과 같다.
$$S = \int_{\alpha}^{\beta} (6x^{5} + 4x^{3} + 1) dx$$
$$= [x^{6} + x^{4} + x]_{\alpha}^{\beta}$$
$$= (\beta^{6} - \alpha^{6}) + (\beta^{4} - \alpha^{4}) + (\beta - \alpha)$$
$$= \{ (144\beta - 55) - (144\alpha - 55) \} + \{ (21\beta - 8) - (21\alpha - 8) \} + (\beta - \alpha)$$
$$= 144(\beta - \alpha) + 21(\beta - \alpha) + (\beta - \alpha)$$
$$= 166(\beta - \alpha)$$
$$= 166\sqrt{5}$$

$\therefore a = 166$
