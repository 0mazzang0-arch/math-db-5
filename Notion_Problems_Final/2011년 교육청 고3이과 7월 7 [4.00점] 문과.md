## 문제

자연수 $k$에 대하여 집합 $A_{k}$를 
$$ A_{k} = \{ x | \log x - [\log x] = \frac{1}{k}, 1 \le x \le 10^{5} \} $$
라 할 때, <보기>에서 옳은 것만을 있는 대로 고른 것은? ($[x]$는 $x$보다 크지 않은 최대의 정수이다.)

$$
\begin{array}{|l|}
\hline
\text{ㄱ. } \sqrt{10} \in A_{2} \\
\text{ㄴ. } n(A_{3}) = n(A_{5}) \\
\text{ㄷ. } A_{m} \cap A_{n} \neq \emptyset \text{를 만족하는 서로 다른 자연수 } m, n \text{이 존재한다.} \\
\hline
\end{array}
$$

## 보기

① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이

$\log x - [\log x] = \frac{1}{k}$에서 $\log x = [\log x] + \frac{1}{k}$이다.
이때 $[\log x]$는 정수이므로 $\log x = (\text{정수}) + \frac{1}{k}$ 꼴이다.
또한 $1 \le x \le 10^{5}$에서 $0 \le \log x \le 5$이다.

$k = 1$일 때,
$\log x = (\text{정수}) + 1$이 되어 소수 부분이 $0$이 되어야 하므로 조건 $\log x - [\log x] = 1$을 만족하는 $x$는 존재하지 않는다.
즉, $A_{1} = \emptyset$

$k \neq 1$일 때,
$0 < \frac{1}{k} < 1$이므로 정수 부분 $[\log x]$가 될 수 있는 값은 $0, 1, 2, 3, 4$이다.
($[\log x] = 5$이면 $\log x = 5 + \frac{1}{k} > 5$가 되어 범위를 벗어난다.)
따라서 $A_{k} = \{ 10^{\frac{1}{k}}, 10^{1+\frac{1}{k}}, 10^{2+\frac{1}{k}}, 10^{3+\frac{1}{k}}, 10^{4+\frac{1}{k}} \}$이다.

ㄱ. $A_{2}$의 원소는 $\log x$의 소수 부분이 $\frac{1}{2}$인 수들이다.
$\log \sqrt{10} = \frac{1}{2}$이므로 $[\log \sqrt{10}] = 0$이고 소수 부분은 $\frac{1}{2}$이다.
따라서 $\sqrt{10} \in A_{2}$ (참)

ㄴ. $2$ 이상의 자연수 $k$에 대하여 집합 $A_{k}$의 원소의 개수는 항상 $5$개이다.
따라서 $n(A_{3}) = 5, n(A_{5}) = 5$이므로 $n(A_{3}) = n(A_{5})$ (참)

ㄷ. 서로 다른 자연수 $m, n$에 대하여 $A_{m} \cap A_{n} \neq \emptyset$이려면
$\log x = a + \frac{1}{m} = b + \frac{1}{n}$ ($a, b$는 정수)를 만족해야 한다.
$\frac{1}{m} - \frac{1}{n} = b - a$
$m, n$이 $2$ 이상의 서로 다른 자연수일 때, $0 < | \frac{1}{m} - \frac{1}{n} | < 1$이므로 좌변은 $0$이 아닌 소수이지만 우변 $b - a$는 정수이다.
이를 만족하는 정수 $b - a$는 존재하지 않는다.
따라서 $A_{m} \cap A_{n} = \emptyset$이다. (거짓)

그러므로 옳은 것은 ㄱ, ㄴ이다.
