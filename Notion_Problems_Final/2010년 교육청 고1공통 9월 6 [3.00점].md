## 문제
전체집합 $U = \{ 1, 2, 3, 4, 5 \}$의 부분집합 $X$에 대하여 $S(X)$를 집합 $X$의 모든 원소의 합이라 하자. 집합 $U$의 공집합이 아닌 두 부분집합 $A$, $B$에 대한 설명 중에서 옳은 것만을 <보기>에서 있는 대로 고른 것은?

ㄱ. $1 \leq S(A) \leq 15$
ㄴ. $A \cup B = B$ 이면 $S(A) < S(B)$ 이다.
ㄷ. $S(A \cap B) = \frac{S(U)}{5}$ 이고 $S(A \cup B) = S(U)$ 이면 $S(A) + S(B) = 18$ 이다.

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄱ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
[정답] ④

ㄱ. $A = \{ 1 \}$ 일 때, $S(A) = 1$ 이고 $A = U$ 일 때, $S(A) = 15$ 이므로 $1 \leq S(A) \leq 15$ 이다. (참)
ㄴ. $A \cup B = B$ 이면 $A \subset B$ 이다. 이때 $A = B$ 인 경우 $S(A) = S(B)$ 이므로 $S(A) < S(B)$ 는 항상 성립하지 않는다. (거짓)
ㄷ. $S(U) = 1 + 2 + 3 + 4 + 5 = 15$ 이므로 $S(A \cap B) = \frac{S(U)}{5} = \frac{15}{5} = 3$ 이다.
$S(A \cup B) = S(U) = 15$ 일 때,
$S(A \cup B) = S(A) + S(B) - S(A \cap B)$ 에서
$15 = S(A) + S(B) - 3$ 이므로
$S(A) + S(B) = 18$ 이다. (참)

따라서 옳은 것은 ㄱ, ㄷ이다.
