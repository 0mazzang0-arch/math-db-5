## 문제
$a$ $>$ $1$, $b$ $>$ $1$ 일 때, $\log_{ a^{ 3 } } b^{ 2 } + \log_{ b^{ 4 } } a^{ 3 }$ 의 최솟값은?

## 보기
① $1$   ② $\sqrt{ 2 }$   ③ $\sqrt{ 3 }$   ④ $2$   ⑤ $\sqrt{ 5 }$

## 풀이
$a$ $>$ $1$, $b$ $>$ $1$ 에서 $\log_{ a } b$ $>$ $0$, $\log_{ b } a$ $>$ $0$ 이다.
산술기하평균의 관계에 의하여
$\log_{ a^{ 3 } } b^{ 2 } + \log_{ b^{ 4 } } a^{ 3 }$
$= \frac{ 2 }{ 3 } \log_{ a } b + \frac{ 3 }{ 4 } \log_{ b } a$
$\ge 2 \sqrt{ \frac{ 2 }{ 3 } \log_{ a } b \cdot \frac{ 3 }{ 4 } \log_{ b } a }$
$= 2 \sqrt{ \frac{ 1 }{ 2 } } = 2 \cdot \frac{ 1 }{ \sqrt{ 2 } } = \sqrt{ 2 }$
따라서 구하는 최솟값은 $\sqrt{ 2 }$ 이다.

정답: ②
