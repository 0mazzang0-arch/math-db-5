## 문제
함수 $f(x) = (x - 2)^{3}$과 두 실수 $m$, $n$에 대하여 함수 $g(x)$를
$$ g(x) = \begin{cases} f(x) & (|x| < a) \\ mx + n & (|x| \ge a) \end{cases} (a > 0) $$
이라 하자. 함수 $g(x)$가 실수 전체의 집합에서 연속일 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?

ㄱ. $a = 1$일 때, $m = 13$이다.
ㄴ. 함수 $g(x)$가 $x = a$에서 미분가능할 때, $m = 48$이다.
ㄷ. $f(a) - 2af'(a) > n - ma$를 만족시키는 자연수 $a$의 개수는 $5$이다.

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
ㄱ. $g(x)$가 실수 전체의 집합에서 연속이려면 $x = a$와 $x = -a$에서 연속이어야 한다.
$$ f(a) = ma + n, \quad f(-a) = -ma + n $$
두 식을 빼면 $f(a) - f(-a) = 2ma$이므로 $m = \frac{f(a) - f(-a)}{2a}$이다.
$a = 1$일 때, $f(1) = (1-2)^3 = -1$, $f(-1) = (-1-2)^3 = -27$이므로
$m = \frac{-1 - (-27)}{1 - (-1)} = \frac{26}{2} = 13$ (참)

ㄴ. 함수 $g(x)$가 $x = a$에서 미분가능하려면 $f'(a) = m$이어야 한다.
$f'(x) = 3(x-2)^2$이므로 $m = 3(a-2)^2$이다.
ㄱ의 식 $f(a) - f(-a) = 2ma$에 대입하면
$(a-2)^3 - (-a-2)^3 = 2a \cdot 3(a-2)^2$
$(a-2)^3 + (a+2)^3 = 6a(a-2)^2$
$(a^3 - 6a^2 + 12a - 8) + (a^3 + 6a^2 + 12a + 8) = 6a(a^2 - 4a + 4)$
$2a^3 + 24a = 6a^3 - 24a^2 + 24a$
$4a^3 - 24a^2 = 0$
$4a^2(a-6) = 0$
$a > 0$이므로 $a = 6$이다. $\therefore a = 6$
이때 $m = f'(6) = 3(6-2)^2 = 3 \cdot 16 = 48$ (참)

$$ \begin{array}{c} \text{> [!example] 📸 (그림/그래프)} \end{array} $$

ㄷ. $x = a$에서 접선의 방정식은 $y = f'(a)(x-a) + f(a)$이다.
이 직선의 $x = -a$에서의 $y$값은 $f(a) - 2af'(a)$이다.
또한 $g(x) = mx + n$에서 $x = -a$일 때 $y = -ma + n$이다.
ㄴ의 결과에서 $a = 6$일 때 $f(a) - 2af'(a) = n - ma$가 성립하며,
함수 $f(x)$의 그래프 개형을 고려할 때 $f(a) - 2af'(a) > n - ma$를 만족시키는 범위는 $0 < a < 6$이다.
따라서 이를 만족시키는 자연수 $a$는 $1, 2, 3, 4, 5$로 개수는 $5$이다. (참)
