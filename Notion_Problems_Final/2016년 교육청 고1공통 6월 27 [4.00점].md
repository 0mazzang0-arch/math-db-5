## 문제
이차함수 $f(x)$가 다음 조건을 만족시킨다.

$$
\begin{array}{|l|}
\hline
(가) \text{ } x \text{에 대한 방정식 } f(x) = 0 \text{의 두 근은 } -2 \text{와 } 4 \text{이다.} \\
(나) \text{ } 5 \le x \le 8 \text{에서 이차함수 } f(x) \text{의 최댓값은 } 80 \text{이다.} \\
\hline
\end{array}
$$

$f(-5)$의 값을 구하시오.



> [!example] 📸 (그림/그래프)

## 풀이
조건 (가)에서 $f(x) = a(x+2)(x-4)$라 두면
$f(x) = a(x-1)^2 - 9a$이다. ($a$는 상수)

조건 (나)에서 $5 \le x \le 8$에서 함수 $f(x)$의 최댓값이 $80$이므로

(ⅰ) $a > 0$일 때
$5 \le x \le 8$에서 함수 $f(x)$의 최댓값은 $f(8)$이므로
$f(8) = 80$, $40a = 80$
$\therefore a = 2$

(ⅱ) $a < 0$일 때
$5 \le x \le 8$에서 함수 $f(x)$의 최댓값은 $f(5)$이므로
$f(5) = 80$, $7a = 80$
$\therefore a = \frac{80}{7}$
$a < 0$이므로 주어진 조건을 만족시키지 않는다.

(ⅰ), (ⅱ)에서 $a = 2$
따라서 $f(x) = 2(x+2)(x-4)$이므로
$f(-5) = 2 \times (-3) \times (-9) = 54$

정답: $54$
