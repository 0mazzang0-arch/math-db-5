## 문제
그림과 같이 길이가 $60$ $cm$ 인 철사를 길이가 $a$ ($cm$), $b$ ($cm$), $c$ ($cm$) ($a < b < c$) 인 세 조각으로 잘랐다. 이 조각들로 만든 세 변의 길이가 각각 $a$, $b$, $c$ 인 삼각형이 다음 조건을 만족시킨다.

$$ \begin{array}{|l|} \hline \text{(가) 두 변의 길이의 차의 최댓값은 } 16 \text{ 이다.} \\ \text{(나) 두 변의 길이의 차의 최솟값은 } 2 \text{ 이다.} \\ \hline \end{array} $$

$3a - b + c$ 의 값을 구하시오. (단, 철사의 두께는 고려하지 않고, 철사를 세 조각으로 자를 때 남는 조각은 없다.)

## 풀이
$a < b < c$ 이므로 두 변의 길이의 차의 최댓값은 $c - a$ 이다.
그러므로 (가)에 의해 $c - a = 16$ 이다.
또한 (나)에 의해 $b - a = 2$ 또는 $c - b = 2$ 이다.

(ⅰ) $b - a = 2$ 인 경우
$b - a = 2$ 이고 $c - a = 16$ 이므로 두 식을 더하면
$-2a + b + c = 18$ $\cdots \cdots$ ㉠
세 변의 길이의 합이 $60$ $cm$ 이므로
$a + b + c = 60$ $\cdots \cdots$ ㉡
㉡ $-$ ㉠을 하면 $3a = 42$ 이다.
따라서 $a = 14$, $b = 16$, $c = 30$ 이다.
이때 $c = a + b$ 이므로 삼각형의 결정조건에 위배된다.

(ⅱ) $c - b = 2$ 인 경우
$c - b = 2$ 이고 $c - a = 16$ 이므로 두 식을 더하면
$-a - b + 2c = 18$ $\cdots \cdots$ ㉢
세 변의 길이의 합이 $60$ $cm$ 이므로
$a + b + c = 60$ $\cdots \cdots$ ㉣
㉢ $+$ ㉣를 하면 $3c = 78$ 이다.
따라서 $a = 10$, $b = 24$, $c = 26$ 이고 $c^{2} = a^{2} + b^{2}$ 이므로 이 삼각형은 직각삼각형이다.
그러므로 $3a - b + c = 3 \times 10 - 24 + 26 = 32$ 이다.

정답: $32$
