## 문제
자연수 전체의 집합의 부분집합 $X$가 상수 $p$에 대하여 다음 조건을 만족시킨다.

(가) $n(X) = 3$
(나) $x \in X$일 때,
$x$가 홀수이면 $\frac{x+p}{2} \in X$이고,
$x$가 짝수이면 $\frac{x}{2} \in X$이다.

$5 \in X$일 때, 모든 자연수 $p$의 값의 합을 구하시오.

## 보기

## 풀이
자연수 전체의 집합의 부분집합 $X$가 $n(X) = 3$이고 $5 \in X$이므로 $X = \{ 5, \alpha, \beta \}$이다.
대응 $f : X \to X$를 $x \in X$가 홀수이면 $f(x) = \frac{x+p}{2}$, $x \in X$가 짝수이면 $f(x) = \frac{x}{2}$로 정의하면 $f$는 함수이다.

> [!example] 📸 (그림/그래프)

$\frac{5+p}{2}$가 자연수가 되어야 하므로 $p$는 홀수이다.

ⅰ) $f(5) = 5$, 즉 $p = 5$일 때

> [!example] 📸 (그림/그래프)

$f(\alpha) = 5$, $f(\alpha) = \alpha$, $f(\alpha) = \beta$인 경우를 각각 생각할 수 있다.
$f(\alpha) = 5$일 때, $\alpha$가 짝수이면 $\frac{\alpha}{2} = 5$, $\alpha = 10$이다.
$\beta$가 홀수이면 $\beta = 15$이고, $\beta$가 짝수이면 $\beta = 20$이다.
따라서 $X = \{ 5, 10, 15 \}$ 또는 $X = \{ 5, 10, 20 \}$이다.
$p = 5$이고 $\alpha$가 짝수인 경우 가능한 집합 $X$가 존재하므로, $p = 5$일 때 $\alpha$가 홀수인 경우와 $f(\alpha) = \alpha$, $f(\alpha) = \beta$인 경우는 다루지 않는다.

ⅱ) $f(5) = \alpha$일 때
$\alpha \neq 5$이므로 $p \neq 5$이다.
$p < 5$인 홀수인 경우에는 $n(X) = 3$인 경우가 존재하지 않음을 쉽게 확인할 수 있다.
따라서 $p > 5$인 경우만 생각하면 된다.
$5 < p$이면 $5 < \alpha < p$이다.
$f(\alpha) = 5$ 또는 $f(\alpha) = \alpha$ 또는 $f(\alpha) = \beta$를 생각해 볼 수 있다.

ⅱ-1) $f(5) = \alpha$, $f(\alpha) = 5$일 때

> [!example] 📸 (그림/그래프)

$\alpha$가 짝수이면 $\frac{\alpha}{2} = 5$, $\alpha = 10$이고, $\frac{5+p}{2} = 10$이므로 $p = 15$이다.
이때 $\beta$의 값은 $\beta = 15$와 $\beta = 20$이 가능하다.

ⅱ-2) $f(5) = \alpha, f(\alpha) = \alpha$일 때
$\alpha \neq 5$이면 $p \neq 5$이므로 $\frac{\alpha}{2} \neq \alpha$이고 $\frac{\alpha+p}{2} \neq \alpha$이다.

ⅱ-3) $f(5) = \alpha$, $f(\alpha) = \beta$일 때

> [!example] 📸 (그림/그래프)

$f(\beta) = 5$ 또는 $f(\beta) = \alpha$ 또는 $f(\beta) = \beta$의 경우를 생각해 볼 수 있다.

ⅱ-3-a) $f(\beta) = 5$일 때,
ⅱ-3-a-ㄱ) $\alpha$가 짝수이면 $\frac{\alpha}{2} = \beta$이다.
$\beta$가 짝수이면 $\frac{\beta}{2} = 5$, $\beta = 10$이고, $\alpha = 20$이다.
$\frac{5+p}{2} = 20$이므로 $p = 35$이다.
이때 $X = \{ 5, 10, 20 \}$이다.
$\beta$가 홀수이면 $f(\beta) = f(f(\alpha)) = f(f(f(5)))$에서
$\frac{\beta+p}{2} = \frac{\alpha+2p}{4} = \frac{5+5p}{8} = 5$이고, $p = 7$이다.
이때 $X = \{ 3, 5, 6 \}$이다.

ⅱ-3-a-ㄴ) $p > 5$이면 $5, \alpha, p, \frac{\alpha+p}{2}$의 대소 관계는 다음 그림과 같다.

> [!example] 📸 (그림/그래프)

$\alpha$가 홀수이면 $\frac{\alpha+p}{2} = \beta$이다.
$\beta$가 홀수이면 $\alpha < \beta < \frac{\beta+p}{2} < p$이므로 $n(X) \ge 4$가 되어 모순이 생긴다.
$\beta$가 짝수이면 $\frac{\beta}{2} = 5$이다.
$f(\beta) = \frac{\beta}{2} = 5$, 즉 $\beta = 10$일 때,
$\frac{\alpha+p}{2} = \frac{\frac{5+p}{2}+p}{2} = 10$에서 $p = \frac{35}{3}$이므로 모순이다.

ⅱ-3-b) $f(\beta) = \alpha$일 때,
ⅱ-3-b-ㄱ) $\alpha$가 짝수이고 $\beta$가 짝수이면 $f(\beta) = \frac{\beta}{2} = \frac{\alpha}{4} = \alpha$이므로 $\alpha = 0$이 되어 모순이다.
ⅱ-3-b-ㄴ) $\alpha$가 짝수이고 $\beta$가 홀수이면 $\alpha = f(\beta) = f(f(f(5)))$이므로
$\frac{5+p}{2} = \frac{5+5p}{8}$, 즉 $p = 15$이고 $\beta = 5$가 되어 모순이다.
ⅱ-3-b-ㄷ) $\alpha$가 홀수이면 $\beta$는 짝수이고 이 경우 $\frac{\beta}{2} = \alpha$가 된다.
$\frac{5+3p}{8} = \frac{5+p}{2}$에서 $p < 0$이 되어 모순이다.

ⅱ-3-c) $f(\beta) = \beta$일 때,
$\beta < p$이므로 $\frac{\beta}{2} \neq \beta$이고 $\frac{\beta+p}{2} \neq \beta$이므로 $f(\beta) \neq \beta$이다.

ⅰ), ⅱ)에서 조건을 만족시키는 모든 자연수 $p$의 값의 합은
$5 + 7 + 15 + 35 = 62$이다.
