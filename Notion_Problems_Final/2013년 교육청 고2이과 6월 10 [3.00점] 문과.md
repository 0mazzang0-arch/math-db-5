## 문제
자연수 $n$ 에 대하여 $\log n$ 의 정수부분을 $a_{n}$ 이라 할 때, $\sum_{k=1}^{200}(-1)^{a_{k}}$ 의 값은?

## 보기
① $16$   ② $18$   ③ $20$   ④ $22$   ⑤ $24$

## 풀이
(i) $1 \le n < 10$ 일 때,
$\log 1 \le \log n < \log 10$
$0 \le \log n < 1$
$a_{n} = 0$

(ii) $10 \le n < 100$ 일 때,
$\log 10 \le \log n < \log 100$
$1 \le \log n < 2$
$a_{n} = 1$

(iii) $100 \le n \le 200$ 일 때,
$\log 100 \le \log n \le \log 200 < \log 1000$
$2 \le \log n < 3$
$a_{n} = 2$

$\therefore \sum_{k=1}^{200}(-1)^{a_{k}}$
$= (-1)^{a_{1}} + (-1)^{a_{2}} + (-1)^{a_{3}} + \cdots + (-1)^{a_{200}}$
$= (-1)^{0} \times 9 + (-1)^{1} \times 90 + (-1)^{2} \times 101$
$= 9 - 90 + 101$
$= 20$
