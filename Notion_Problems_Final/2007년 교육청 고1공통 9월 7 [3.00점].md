## 문제
이차부등식 $ax^{2} + bx + c \ge 0$의 해가 $x = 2$ 뿐일 때, 다음 <보기>에서 옳은 내용을 모두 고른 것은?

$$
\begin{array}{|l|}
\hline
\text{ㄱ. } a < 0 \\
\text{ㄴ. } b^{2} - 4ac = 0 \\
\text{ㄷ. } a + b + c < 0 \\
\hline
\end{array}
$$

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
이차부등식 $ax^{2} + bx + c \ge 0$의 해가 $x = 2$ 뿐이려면 위로 볼록한 포물선이 $x$축과 $x = 2$에서 접해야 하므로 $a < 0$이다.
해가 $x = 2$이고 $x^{2}$의 계수가 $1$인 이차부등식은 $(x - 2)^{2} \le 0$이다.
$\therefore x^{2} - 4x + 4 \le 0$
양변에 $a$ ($a < 0$)를 곱하면 부동호의 방향이 바뀌어
$ax^{2} - 4ax + 4a \ge 0$
이 부등식이 $ax^{2} + bx + c \ge 0$와 같으므로
$b = -4a$, $c = 4a$ $\cdots \cdots$ ㉠
따라서 $a < 0$, $b > 0$, $c < 0$ $\cdots \cdots$ ㉡

ㄱ. ㉡에서 $a < 0$ (참)
ㄴ. 방정식 $ax^{2} + bx + c = 0$이 $x = 2$를 중근으로 가지므로, 이 이차방정식의 판별식을 $D$라 하면 $D = b^{2} - 4ac = 0$ (참)
ㄷ. ㉠, ㉡에서 $a + b + c = a + (-4a) + 4a = a < 0$ (참)

따라서 ㄱ, ㄴ, ㄷ 모두 옳다.
