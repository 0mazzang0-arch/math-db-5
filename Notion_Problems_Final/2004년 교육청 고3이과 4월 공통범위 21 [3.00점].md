## 문제
로그방정식 $\log _{2} x^{2} + \log _{2} y^{2} = \log _{\sqrt{2}} (x + y + 3)$ 을 만족시키는 양의 정수 $x$, $y$ 에 대하여 $x^{2} + 2 y^{2}$ 의 최솟값을 구하시오.

## 풀이
로그방정식 $\log _{2} x^{2} + \log _{2} y^{2} = \log _{\sqrt{2}} (x + y + 3)$ 에서
$2 \log _{2} (x^{2} y^{2}) = 2 \log _{2} (x + y + 3)^{2}$
$x^{2} y^{2} = (x + y + 3)^{2}$
$x, y$ 는 양의 정수이므로
$x y = x + y + 3$
$x y - x - y + 1 = 3 + 1$
$(x - 1)(y - 1) = 4$
이를 만족시키는 양의 정수 $x, y$ 의 순서쌍은
$$ \begin{array}{l} \left\{ \begin{array}{l} x = 2 \\ y = 5 \end{array} \right. \text{ 또는 } \left\{ \begin{array}{l} x = 5 \\ y = 2 \end{array} \right. \text{ 또는 } \left\{ \begin{array}{l} x = 3 \\ y = 3 \end{array} \right. \end{array} $$
각 경우에 대하여 $x^{2} + 2 y^{2}$ 의 값을 구하면 다음과 같다.
$1)$ $x = 2, y = 5$ 일 때,
$2^{2} + 2 \times 5^{2} = 4 + 50 = 54$
$2)$ $x = 5, y = 2$ 일 때,
$5^{2} + 2 \times 2^{2} = 25 + 8 = 33$
$3)$ $x = 3, y = 3$ 일 때,
$3^{2} + 2 \times 3^{2} = 9 + 18 = 27$
따라서 $x^{2} + 2 y^{2}$ 의 최솟값은 $27$ 이다.
