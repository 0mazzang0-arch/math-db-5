## 문제
$f(x) = \sqrt{x}$, $g(x) = f(x) + 1$일 때, $a_{3} + b_{4}$의 값은?

## 보기
① $5$   ② $\frac{16}{3}$   ③ $\frac{17}{3}$   ④ $6$   ⑤ $\frac{19}{3}$

## 풀이
$$a_{3} = \int_{0}^{3} \sqrt{x} dx = \left[ \frac{2}{3} x \sqrt{x} \right]_{0}^{3} = 2 \sqrt{3}$$
$$b_{4} = \int_{3}^{4} (\sqrt{x} + 1) dx = \left[ \frac{2}{3} x \sqrt{x} + x \right]_{3}^{4} = \frac{19}{3} - 2 \sqrt{3}$$
$$\therefore a_{3} + b_{4} = \frac{19}{3}$$

정답: ⑤
