## 문제
함수 $f(x) = x^3 + ax$ 에서 $x$ 의 값이 $1$ 에서 $3$ 까지 변할 때의 평균변화율이 $f'(a)$ 의 값과 같게 되도록 하는 양수 $a$ 에 대하여 $3a^2$ 의 값을 구하시오.

## 보기

## 풀이
$f'(x) = 3x^2 + a$ 에서 $x$ 의 값이 $1$ 에서 $3$ 까지 변할 때의 함수 $f(x)$ 의 평균변화율이 $f'(a)$ 의 값과 같으므로

$$\frac{f(3) - f(1)}{3 - 1} = f'(a)$$

$$\frac{\left(3^{3} + 3a\right) - \left(1^{3} + a\right)}{2} = 3a^2 + a$$

$$\frac{26 + 2a}{2} = 3a^2 + a$$

$13 + a = 3a^2 + a$

따라서 $3a^2 = 13$ 이다.

**정답: $13$**
