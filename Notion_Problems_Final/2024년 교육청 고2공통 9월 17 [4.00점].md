## 문제
수열 $\{ a_{n} \}$이 다음 조건을 만족시킨다.

(가) $a_{12} - a_{10} = 5$
(나) 모든 자연수 $n$에 대하여 $\sum_{k=1}^{n} a_{2k} = \sum_{k=1}^{n} a_{2k-1} + n^2$이다.

$a_{9} = 16$일 때, $a_{11}$의 값은?

## 보기
① $17$   ② $18$   ③ $19$   ④ $20$   ⑤ $21$

## 풀이
$n \ge 2$인 자연수 $n$에 대하여
$\sum_{k=1}^{n} a_{2k} - \sum_{k=1}^{n-1} a_{2k} = ( \sum_{k=1}^{n} a_{2k-1} + n^2 ) - \{ \sum_{k=1}^{n-1} a_{2k-1} + (n-1)^2 \}$
$a_{2n} = a_{2n-1} + 2n - 1$

$n=6$일 때,
$a_{12} = a_{11} + 11$

$n=5$일 때,
$a_{10} = a_{9} + 9$

$a_{12} - a_{10} = ( a_{11} + 11 ) - ( a_{9} + 9 ) = a_{11} - a_{9} + 2$

조건 (가)에서 $a_{12} - a_{10} = 5$이고 $a_{9} = 16$이므로
$5 = a_{11} - 16 + 2$
따라서 $a_{11} = 19$

정답: ③
