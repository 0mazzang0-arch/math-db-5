## 문제
$f(0)=0$ 인 삼차함수 $f(x)$ 의 도함수 $y=f'(x)$ 의 그래프가 그림과 같다.

> [!example] 📸 (그림/그래프)

실수 $k$ 에 대하여 함수 $g(x)$ 를
$$ g(x) = \begin{cases} (x-k) + f(k) & (x \le k) \\ f(x) & (x > k) \end{cases} $$
라 하자. $x \le k$ 에서 두 함수 $y=f(x)$ 와 $y=g(x)$ 의 그래프가 만나는 서로 다른 점의 개수를 $h(k)$ 라 할 때, $\sum_{k=1}^{7} h(k)$ 의 값은? (단, $f'(0)=1$, $f'(1)=f'(3)=0$)

## 보기
① $10$   ② $11$   ③ $12$   ④ $13$   ⑤ $14$

## 풀이
$f(0)=0$ 이고 $f'(x) = \frac{1}{3}(x-1)(x-3)$ 이므로
$f(x) = \frac{1}{9}x(x-3)^{2}$

$x \le k$ 에서 함수 $y=g(x)$ 의 그래프는 기울기가 $1$ 인 직선의 일부이다. 그러므로 곡선 $y=f(x)$ 위의 점 $(k, f(k))$ 에서의 접선의 기울기가 $1$ 이 되는 $k$ 의 값을 찾으면
$f'(k) = \frac{1}{3}(k-1)(k-3) = 1$
$k(k-4) = 0$
$k=0$ 또는 $k=4$

(ⅰ) $k=0$ 일 때
$f(0)=0$ 이므로 $x \le 0$ 에서 함수 $g(x)=x+f(0)=x$ 이고 $f'(0)=1$ 이므로 $x=0$ 에서 곡선 $y=f(x)$ 에 직선 $y=x$ 가 접한다.
$\frac{1}{9}x(x-3)^{2} = x$, $x(x-3)^{2} = 9x$
$x=0$ 또는 $x=6$ $\cdots\cdots$ ㉠
$x \le 0$ 에서 $y=f(x)$, $y=g(x)$ 의 그래프가 만나는 점의 개수는 $h(0)=1$ 이다.

> [!example] 📸 (그림/그래프)

(ⅱ) $k=4$ 일 때
$x \le 4$ 에서 함수 $g(x)=(x-4)+f(4)$ 이고, $f'(4)=1$ 이므로 $x=4$ 에서 곡선 $y=f(x)$ 에 직선 $y=(x-4)+f(4)$ 가 접한다. 그러므로 $x \le 4$ 에서 두 함수 $y=f(x)$, $y=g(x)$ 의 그래프가 만나는 서로 다른 점의 개수는 $h(4)=2$ 이다.

> [!example] 📸 (그림/그래프)

(ⅲ) $k=6$ 일 때
$f(6)=6$ 이므로 $x \le 6$ 에서 $g(x)=(x-6)+f(6)=x$ 이다.
$x \le 6$ 에서 두 함수 $y=f(x)$, $y=g(x)$ 의 그래프가 만나는 서로 다른 점의 개수는 $h(6)=2$ 이다.

> [!example] 📸 (그림/그래프)

(ⅰ), (ⅱ), (ⅲ)에 의하여
$0 < k \le 4$ 일 때, $h(k)=2$
$4 < k < 6$ 일 때, $h(k)=3$
$k=6$ 일 때, $h(k)=2$
$k > 6$ 일 때, $h(k)=1$

따라서 $\sum_{k=1}^{7} h(k) = h(1)+h(2)+h(3)+h(4)+h(5)+h(6)+h(7)$
$= 2+2+2+2+3+2+1 = 14$
