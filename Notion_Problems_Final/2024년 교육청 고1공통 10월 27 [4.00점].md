## 문제
두 자연수 $a$, $b$ ($b \le 20$)에 대하여 전체집합 $U = \{ x | x \text{는 } 20 \text{ 이하의 자연수} \}$의 두 부분집합
$$ A = \{ x | x \text{는 } a \text{의 배수}, x \in U \}, $$
$$ B = \{ x | x \text{는 } b \text{의 약수}, x \in U \} $$
가 다음 조건을 만족시킨다.

(가) $\{ 3, 6 \} \subset A \cap B$
(나) $n(B - A) = 2$

집합 $A - B$의 모든 원소의 합의 최솟값을 구하시오.

## 보기

## 풀이
$A \cap B \subset A$이므로 조건 (가)에서 $\{ 3, 6 \} \subset A$이다.
$3$, $6$이 모두 $a$의 배수이므로 $a = 1$ 또는 $a = 3$이다.

$a = 1$이면 $A = U$가 되어 $B - A = \emptyset$이므로 조건 (나)를 만족시키지 않는다.
그러므로 $a = 3$이고, $A = \{ 3, 6, 9, 12, 15, 18 \}$이다.

또한 $A \cap B \subset B$이므로 조건 (가)에서 $\{ 3, 6 \} \subset B$이다.
$3$, $6$이 모두 $b$의 약수이므로 $b$는 $6$의 배수여야 한다. $b \le 20$이므로 가능한 $b$의 값은 다음과 같다.
$b = 6$ 또는 $b = 12$ 또는 $b = 18$

(ⅰ) $b = 6$일 때
$B = \{ 1, 2, 3, 6 \}$이므로 $B - A = \{ 1, 2 \}$가 되어 $n(B - A) = 2$이다. 즉, 조건 (나)를 만족시킨다.
이때 $A - B = \{ 9, 12, 15, 18 \}$이므로 모든 원소의 합은
$9 + 12 + 15 + 18 = 54$

(ⅱ) $b = 12$일 때
$B = \{ 1, 2, 3, 4, 6, 12 \}$이므로 $B - A = \{ 1, 2, 4 \}$가 되어 $n(B - A) = 3$이다. 즉, 조건 (나)를 만족시키지 않는다.

(ⅲ) $b = 18$일 때
$B = \{ 1, 2, 3, 6, 9, 18 \}$이므로 $B - A = \{ 1, 2 \}$가 되어 $n(B - A) = 2$이다. 즉, 조건 (나)를 만족시킨다.
이때 $A - B = \{ 12, 15 \}$이므로 모든 원소의 합은
$12 + 15 = 27$

따라서 (ⅰ), (ⅱ), (ⅲ)에 의하여 집합 $A - B$의 모든 원소의 합의 최솟값은 $27$이다.

[정답] $27$
