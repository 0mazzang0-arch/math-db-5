## 문제
다음은 $3$이 아닌 양수 $p$에 대하여 

$$\sum_{n=1}^{\infty} \frac{p^{n} + 3p^{n-1} + 3^{2} p^{n-2} + \cdots + 3^{n-1} p + 3^{n}}{(p+3)^{n}} = \frac{p^{2} + 3p + (다)}{3p}$$ 

가 성립함을 보이는 과정이다.

---

$p^{n} + 3p^{n-1} + 3^{2} p^{n-2} + \cdots + 3^{n-1} p + 3^{n}$은 $p^{n}$, 공비가 $(가)$인 등비수열의 첫째항부터 제 $(n+1)$항까지의 합이고, $p \neq 3$이므로 

$p^{n} + 3p^{n-1} + 3^{2} p^{n-2} + \cdots + 3^{n-1} p + 3^{n} = \frac{p^{n+1} - 3^{n+1}}{(나)}$이다. 

$0 < \frac{p}{p+3} < 1$, $0 < \frac{3}{p+3} < 1$이므로 

$$\sum_{n=1}^{\infty} \frac{p^{n} + 3p^{n-1} + 3^{2} p^{n-2} + \cdots + 3^{n-1} p + 3^{n}}{(p+3)^{n}}$$ 

$$= \sum_{n=1}^{\infty} \frac{p^{n+1} - 3^{n+1}}{(나) \times (p+3)^{n}}$$ 

$$= \frac{1}{(나)} \left\{ p \sum_{n=1}^{\infty} \left( \frac{p}{p+3} \right)^{n} - 3 \sum_{n=1}^{\infty} \left( \frac{3}{p+3} \right)^{n} \right\}$$ 

$$= \frac{p^{2} + 3p + (다)}{3p}$$ 
이다.

---

위의 $(가)$, $(나)$에 알맞은 식을 각각 $f(p)$, $g(p)$라 하고, $(다)$에 알맞은 수를 $k$라 할 때, $f(k) \times g(k)$의 값은?

## 보기
① $2$   ② $4$   ③ $6$   ④ $8$   ⑤ $10$

## 풀이
$p^{n} + 3p^{n-1} + 3^{2} p^{n-2} + \cdots + 3^{n-1} p + 3^{n}$은 첫째항이 $p^{n}$, 공비가 $\frac{3}{p}$인 등비수열의 첫째항부터 제 $(n+1)$항까지의 합이므로 
$f(p) = \frac{3}{p}$이다.

또한 $p \neq 3$일 때, 이 합은
$$\frac{p^{n} \left\{ 1 - \left( \frac{3}{p} \right)^{n+1} \right\}}{1 - \frac{3}{p}} = \frac{p^{n} \times \frac{p^{n+1} - 3^{n+1}}{p^{n+1}}}{\frac{p-3}{p}} = \frac{p^{n+1} - 3^{n+1}}{p-3}$$
이므로 $g(p) = p-3$이다.

무한등비급수의 합을 구하면
$$\sum_{n=1}^{\infty} \frac{p^{n+1} - 3^{n+1}}{(p-3)(p+3)^{n}} = \frac{1}{p-3} \left\{ p \sum_{n=1}^{\infty} \left( \frac{p}{p+3} \right)^{n} - 3 \sum_{n=1}^{\infty} \left( \frac{3}{p+3} \right)^{n} \right\}$$
$$= \frac{1}{p-3} \left( p \times \frac{\frac{p}{p+3}}{1 - \frac{p}{p+3}} - 3 \times \frac{\frac{3}{p+3}}{1 - \frac{3}{p+3}} \right)$$
$$= \frac{1}{p-3} \left( p \times \frac{p}{3} - 3 \times \frac{3}{p} \right) = \frac{1}{p-3} \left( \frac{p^{2}}{3} - \frac{9}{p} \right)$$
$$= \frac{1}{p-3} \times \frac{p^{3} - 27}{3p} = \frac{(p-3)(p^{2} + 3p + 9)}{(p-3)3p} = \frac{p^{2} + 3p + 9}{3p}$$
따라서 $k = 9$이다.

$f(k) \times g(k) = f(9) \times g(9) = \frac{3}{9} \times (9-3) = \frac{1}{3} \times 6 = 2$

정답은 ①이다.
