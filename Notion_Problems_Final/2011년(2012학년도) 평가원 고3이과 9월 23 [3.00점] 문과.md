## 문제
연속확률변수 $X$의 확률밀도함수가 $f(x) = ax+b$ ($0 \le x \le 1$)이다. $E(X) = \frac{7}{12}$일 때, 두 상수 $a$, $b$에 대하여 $80ab$의 값을 구하시오.

## 풀이
$\int_{0}^{1} f(x) dx = 1$이므로
$\int_{0}^{1} (ax+b) dx = \left[ \frac{a}{2}x^2 + bx \right]_{0}^{1} = \frac{a}{2} + b = 1$
$\therefore a+2b = 2 \cdots ㉠$

$E(X) = \int_{0}^{1} xf(x) dx = \frac{7}{12}$이므로
$\int_{0}^{1} (ax^2 + bx) dx = \left[ \frac{a}{3}x^3 + \frac{b}{2}x^2 \right]_{0}^{1} = \frac{a}{3} + \frac{b}{2} = \frac{7}{12}$
$\therefore 4a+6b = 7 \cdots ㉡$

$㉠, ㉡$을 연립하여 풀면
$a=1, b=\frac{1}{2}$
$\therefore 80ab = 80 \times 1 \times \frac{1}{2} = 40$

정답: $40$
