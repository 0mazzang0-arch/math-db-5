## 문제
두 자연수 $a$, $b$ ($a < b$)에 대하여 분수부등식 
$$\frac{x}{x-a} + \frac{x}{x-b} \leq 0$$
을 만족시키는 정수 $x$가 $2$개가 되도록 하는 순서쌍 $(a, b)$의 개수는?

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
[정답] ③ 

분모를 통분시키면 
$$\frac{x(x-b) + x(x-a)}{(x-a)(x-b)} \leq 0$$ 

$$\therefore x(2x - a - b)(x - a)(x - b) \leq 0$$
(단, $x \neq a$, $x \neq b$) 

> [!example] 📸 (그림/그래프)

이때, $0 < a < \frac{a+b}{2} < b$ 이므로 
$0 \leq x < a$ 또는 $\frac{a+b}{2} \leq x < b$ 

(ⅰ) $a = 1$ 일 때, 
$0 \leq x < 1$ 또는 $\frac{1+b}{2} \leq x < b$ 
$x$가 $2$개이므로 $b = 3$, $4$ 

(ⅱ) $a = 2$ 일 때, 
$0 \leq x < 2$ 또는 $\frac{2+b}{2} \leq x < b$ 
$x$가 $2$개이므로 $b = 3$ 

$\therefore$ 순서쌍 $(a, b) = (1, 3), (1, 4), (2, 3)$으로 총 $3$개이다.
