## 문제
닫힌구간 $[ -1, 1 ]$ 에서 정의된 연속확률변수 $X$ 의 확률밀도함수 $f ( x )$ 가 다음 조건을 만족시킨다.

(가) $f ( -x ) = f ( x )$
(나) $\int_{ 0 }^{ 1 } x^{ 2 } f ( x ) dx = \frac{ 1 }{ 10 }$

$V ( 10X + 3 )$ 의 값을 구하시오.

## 보기

## 풀이
(가)에서 $f ( -x ) = f ( x )$ 이므로 $-x f ( -x ) = -x f ( x )$ 이다. 즉, $x f ( x )$ 는 기함수이므로
$\int_{ -1 }^{ 1 } x f ( x ) dx = 0$
마찬가지로 $x^{ 2 } f ( -x ) = x^{ 2 } f ( x )$ 이므로 $x^{ 2 } f ( x )$ 는 우함수이다.
$\int_{ -1 }^{ 1 } x^{ 2 } f ( x ) dx = 2 \int_{ 0 }^{ 1 } x^{ 2 } f ( x ) dx = 2 \times \frac{ 1 }{ 10 } = \frac{ 1 }{ 5 }$

$\therefore V ( X ) = \int_{ -1 }^{ 1 } x^{ 2 } f ( x ) dx - \left\{ \int_{ -1 }^{ 1 } x f ( x ) dx \right\}^{ 2 }$
$= \frac{ 1 }{ 5 } - 0 = \frac{ 1 }{ 5 }$

$\therefore V ( 10X + 3 ) = 10^{ 2 } V ( X ) = 100 \times \frac{ 1 }{ 5 } = 20$

정답: $20$
