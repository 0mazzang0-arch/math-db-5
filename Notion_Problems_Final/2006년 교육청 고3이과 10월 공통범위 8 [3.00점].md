## 문제

쌍곡선 $\frac{x^{2}}{2} - \frac{y^{2}}{18} = 1$ 과 직선 $y = ax + b$ ($a$, $b$ 는 상수)의 교점의 개수에 대한 설명 중 옳은 내용을 <보기>에서 모두 고른 것은?

$$ \begin{array}{|l|} \hline \text{<보기>} \\ \text{ㄱ. } a = -4 \text{ 이고 } b = 0 \text{ 일 때 교점은 없다.} \\ \text{ㄴ. } a = 3 \text{ 이고 } b > 0 \text{ 일 때 교점은 } 1 \text{ 개이다.} \\ \text{ㄷ. } a = \frac{1}{3} \text{ 이고 } b < 0 \text{ 일 때 교점은 } 2 \text{ 개이다.} \\ \hline \end{array} $$

## 보기

① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이

[정답] ⑤ 

쌍곡선 $\frac{x^{2}}{2} - \frac{y^{2}}{18} = 1$ 의 점근선의 방정식은 $y = \pm 3x$ 이다. 

> [!example] 📸 (그림/그래프)

ㄱ. $a = -4$, $b = 0$ 이면 교점 $0$ 개 
ㄴ. $a = 3$, $b > 0$ 이면 점근선과 평행이므로 교점 $1$ 개 
ㄷ. $a = \frac{1}{3}$, $b < 0$ 이면 교점이 $2$ 개이다. 
이상에서 옳은 것은 $ㄱ, ㄴ, ㄷ$이다.
