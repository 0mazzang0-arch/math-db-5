## 문제
실수 $a$, $b$, $c$ 가
$$\log \frac{ab}{2} = (\log a)(\log b)$$
$$\log \frac{bc}{2} = (\log b)(\log c)$$
$$\log (ca) = (\log c)(\log a)$$
를 만족시킬 때, $a+b+c$ 의 값을 구하시오. (단, $a$, $b$, $c$ 는 모두 $10$ 보다 크다.)

## 보기


## 풀이
$\log a = A$, $\log b = B$, $\log c = C$ 라 하고 준 식을 정리하면
$A+B-\log 2 = AB$ $\cdots$ ㉠
$B+C-\log 2 = BC$ $\cdots$ ㉡
$C+A = CA$ $\cdots$ ㉢
(단, $A > 1$, $B > 1$, $C > 1$)

㉠ $-$ ㉡에서 $A-C = B(A-C)$ 이므로 $A=C$ ($\because B > 1$)
㉢에 대입하면 $2C = C^2$ 즉, $C=2$ ($\because C > 1$)
$\therefore A=2$

㉠에 대입하면 $2+B-\log 2 = 2B$
$\therefore B= 2-\log 2 = \log 100 - \log 2 = \log 50$

$\log a = 2$ 에서 $a=100$
$\log b = \log 50$ 에서 $b=50$
$\log c = 2$ 에서 $c=100$

$\therefore a+b+c = 100+50+100 = 250$

**[다른 풀이]**
$\log a = A$, $\log b = B$, $\log c = C$ 라 하면
$\log \frac{ab}{2} = (\log a)(\log b)$ 에서
$A+B-\log 2=AB$, $(A-1)(B-1)=\log 5$ $\cdots$ ㉠

$\log \frac{bc}{2} = (\log b)(\log c)$ 에서
$B+C-\log 2=BC$, $(B-1)(C-1)=\log 5$ $\cdots$ ㉡

$\log (ca) = (\log c)(\log a)$ 에서
$C+A=CA$, $(C-1)(A-1)=1$ $\cdots$ ㉢

㉠, ㉡, ㉢을 변끼리 곱하면
$\{(A-1)(B-1)(C-1)\}^2 = (\log 5)^2$
$A > 1$, $B > 1$, $C > 1$ 이므로
$(A-1)(B-1)(C-1)=\log 5$ $\cdots$ ㉣

㉣ $\div$ ㉠을 하면 $C-1=1$, $\log c=2$
$\therefore c=100$

㉣ $\div$ ㉡을 하면 $A-1=1$, $\log a=2$
$\therefore a=100$

㉣ $\div$ ㉢을 하면 $B-1=\log 5$, $\log b=\log 50$
$\therefore b=50$

이상에서 $a+b+c=250$
