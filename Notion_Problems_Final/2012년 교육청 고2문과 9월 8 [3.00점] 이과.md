## 문제
부등식 $3 \leq [ \log_{3} n ] \leq 4$ 을 만족시키는 자연수 $n$ 의 개수는? (단, $[x]$ 는 $x$ 보다 크지 않은 최대의 정수이다.)

## 보기
① $186$   ② $196$   ③ $206$   ④ $216$   ⑤ $226$

## 풀이
$[ \log_{3} n ] = k$ ($k$ 는 정수)라 할 때, $k \leq \log_{3} n < k + 1$ 이 성립한다.
주어진 부등식 $3 \leq [ \log_{3} n ] \leq 4$ 에 의하여 $[ \log_{3} n ]$ 이 가질 수 있는 값은 $3$ 또는 $4$ 이다.
따라서 $3 \leq \log_{3} n < 5$ 가 성립한다.
로그의 정의에 의해
$3^{3} \leq n < 3^{5}$
$27 \leq n < 243$ 이다.
이를 만족시키는 자연수 $n$ 의 개수는
$243 - 27 = 216$
따라서 구하는 자연수 $n$ 의 개수는 $216$ 이다.

정답: ④
