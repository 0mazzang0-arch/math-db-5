## 문제
다항함수 $f(x)$가 상수 $a$ ($a > 0$)와 모든 실수 $x$에 대하여 
$$xf(x) - f(x) = x^{2} + x \int_{0}^{a} f(t) dt + \frac{3}{2} a$$
를 만족시킬 때, $f(10)$의 값을 구하시오.

## 풀이
$xf(x) - f(x) = x^{2} + x \int_{0}^{a} f(t) dt + \frac{3}{2} a$ 에서 
$\int_{0}^{a} f(t) dt = k$ ($k$는 상수)라 하면 
$(x - 1) f(x) = x^{2} + kx + \frac{3}{2} a$ 

$f(x)$는 다항함수이므로 다항식 $x^{2} + kx + \frac{3}{2} a$는 $x - 1$을 인수로 가져야 한다. 
$1 + k + \frac{3}{2} a = 0$, $k = -1 - \frac{3}{2} a$ $\cdots$ $\cdots$ ㉠ 

$f(x) = \frac{x^{2} + kx + \frac{3}{2} a}{x - 1} = \frac{x^{2} + (-1 - \frac{3}{2} a)x + \frac{3}{2} a}{x - 1}$ 
$= \frac{(x - 1) (x - \frac{3}{2} a)}{x - 1} = x - \frac{3}{2} a$ 

$k = \int_{0}^{a} f(t) dt$ 이므로 
$k = \int_{0}^{a} (t - \frac{3}{2} a) dt = \left[ \frac{1}{2} t^{2} - \frac{3}{2} at \right]_{0}^{a}$ 
$= \frac{1}{2} a^{2} - \frac{3}{2} a^{2} = -a^{2}$ $\cdots$ $\cdots$ ㉡ 

㉠, ㉡에서 
$-a^{2} = -1 - \frac{3}{2} a$, $a^{2} - \frac{3}{2} a - 1 = 0$ 
$2 a^{2} - 3 a - 2 = 0$, $(2 a + 1) (a - 2) = 0$ 

$a > 0$ 이므로 $a = 2$ 
따라서 $f(x) = x - \frac{3}{2} \times 2 = x - 3$ 이므로 
$f(10) = 10 - 3 = 7$ 

**[다른 풀이]** 
모든 실수 $x$에 대하여 
$xf(x) - f(x) = x^{2} + x \int_{0}^{a} f(t) dt + \frac{3}{2} a$ $\cdots$ $\cdots$ ㉠ 
가 성립한다. 따라서 다항함수 $f(x)$는 최고차항의 계수가 $1$인 일차함수이다. 

$f(x) = x + k$ ($k$는 상수)라 하면 
$xf(x) - f(x) = x(x + k) - (x + k) = x^{2} + (k - 1)x - k$ 
이므로 ㉠에서 
$x^{2} + (k - 1)x - k = x^{2} + x \int_{0}^{a} f(t) dt + \frac{3}{2} a$ 
계수를 비교하면 
$k - 1 = \int_{0}^{a} f(t) dt$, $-k = \frac{3}{2} a$ 

$\int_{0}^{a} f(t) dt = \int_{0}^{a} (t + k) dt = \left[ \frac{1}{2} t^{2} + kt \right]_{0}^{a} = \frac{1}{2} a^{2} + ka$ 이므로 
$k - 1 = \frac{1}{2} a^{2} + ka$ 

$k = -\frac{3}{2} a$ 를 대입하면 
$-\frac{3}{2} a - 1 = \frac{1}{2} a^{2} - \frac{3}{2} a^{2}$ 
$2 a^{2} - 3 a - 2 = 0$ 
$(a - 2) (2 a + 1) = 0$ 

$a > 0$ 이므로 $a = 2$, $k = -3$ 
따라서 $f(x) = x - 3$ 이므로 $f(10) = 7$
