## 문제
두 집합
$A = \{ x | x^{3} + a x^{2} + b x + c = 0 \}$
$B = \{ x | x^{2} + x - 2 = 0 \}$
에 대하여 $A = B$가 성립한다. 다항식 $x^{3} + a x^{2} + b x + c$를 $x^{2} + x - 2$로 나눈 몫을 $Q(x)$라 할 때, $Q(100)$의 값을 구하시오. (단, $a$, $b$, $c$는 상수이고, $c > 0$이다.)

## 풀이
이차방정식 $x^{2} + x - 2 = 0$의 해는 $x = -2$ 또는 $x = 1$이므로
$B = \{ 1, -2 \}$
따라서 $x^{3} + a x^{2} + b x + c = 0$의 해가 $x = -2$ 또는 $x = 1$이어야 하므로 $x^{3} + a x^{2} + b x + c$는 $(x + 2)^{2}(x - 1)$이거나 $(x + 2)(x - 1)^{2}$이다.
이때 $c > 0$이므로
$x^{3} + a x^{2} + b x + c = (x + 2)(x - 1)^{2}$
따라서 $Q(x) = x - 1$이므로
$Q(100) = 100 - 1 = 99$
