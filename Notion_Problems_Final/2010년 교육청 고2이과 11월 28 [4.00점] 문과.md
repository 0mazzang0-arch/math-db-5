## 문제
자연수 $n$에 대하여 
$$f(n)=(\log n \text{의 정수 부분})$$ 
일 때, 다음 두 조건을 만족시키는 $n$의 최댓값과 최솟값의 합을 구하시오. 

(가) $n$은 두 자리의 자연수이다. 
(나) $\log 3n - f(2n) < \log 2$

## 보기

## 풀이
$n$은 두 자리의 자연수이므로 $f(n)=1$이고 
$\log n=1+ \alpha (0 \le \alpha < 1)$라 하면 
$\log 2n=1+ \log 2+ \alpha$이다. 

(ⅰ) $0 \le \log 2+ \alpha < 1$인 경우 
$f(2n)=1$이므로 $10 \le n \le 49$이고, 
$\log 3n - 1 < \log 2$에서 $3n < 20$이다. 따라서 두 자리의 자연수 $n$은 없다. 

(ⅱ) $1 \le \log 2+ \alpha$인 경우 
$f(2n)=2$이므로 $50 \le n \le 99$이고 
$\log 3n - 2 < \log 2$에서 $3n < 200$이다. 
$50 \le n < \frac{200}{3}$이 되어 만족시키는 자연수 $n$의 범위는 $50 \le n \le 66$이다. 

그러므로 최솟값은 $50$, 최댓값은 $66$이며, 그 합은 $116$이다.

정답: $116$
