## 문제
원점을 동시에 출발하여 수직선 위를 움직이는 두 점 $P$, $Q$의 시각 $t$에서의 속도가 각각 
$v_1(t) = \frac{1}{2}t^2 - 3t$, $v_2(t) = -\frac{1}{2}t^2 + t$이다. 다음은 두 점 $P$, $Q$가 출발 후 처음으로 만날 때까지 두 점 $P$, $Q$ 사이의 거리의 최댓값을 구하는 과정이다.

$$
\begin{array}{|l|}
\hline
\text{두 점 } P, Q \text{의 시각 } t \text{에서의 위치를 각각 } x_1(t), x_2(t) \text{라 하면} \\
x_1(t) = \frac{1}{6}t^3 - \frac{3}{2}t^2, \quad x_2(t) = \text{ (가) } \\
\text{출발 후 처음으로 두 점 } P, Q \text{가 만나는 시각은 } t = 6 \text{이다.} \\
0 < t \leq 6 \text{에서 두 점 } P, Q \text{ 사이의 거리를 } l(t) \text{라 하면} \\
l(t) \text{는 } t = \text{ (나) } \text{일 때 극대이면서 최대이므로} \\
l(t) \text{의 최댓값은 } \text{ (다) } \text{이다.} \\
\hline
\end{array}
$$

위의 (가)에 알맞은 식을 $f(t)$라 하고, (나), (다)에 알맞은 수를 각각 $a$, $b$라 할 때, $\frac{a \times b}{f(2)}$의 값은?

## 보기
① $60$   ② $62$   ③ $64$   ④ $66$   ⑤ $68$

## 풀이
두 점 $P$, $Q$의 시각 $t$에서의 위치를 각각 $x_1(t)$, $x_2(t)$라 하면 
$x_1(t) = \int_{0}^{t} v_1(t) dt = \frac{1}{6}t^3 - \frac{3}{2}t^2$, 
$x_2(t) = \int_{0}^{t} v_2(t) dt = -\frac{1}{6}t^3 + \frac{1}{2}t^2$이다.

출발 후 처음으로 두 점 $P$, $Q$가 만나는 시각은 $t = 6$이다. 
$0 < t \leq 6$에서 두 점 $P$, $Q$ 사이의 거리를 $l(t)$라 하면 
$l(t) = | \frac{1}{6}t^3 - \frac{3}{2}t^2 - ( -\frac{1}{6}t^3 + \frac{1}{2}t^2 ) | = | \frac{1}{3}t^3 - 2t^2 |$이다.

$g(t) = \frac{1}{3}t^3 - 2t^2$이라 하면 $g'(t) = t^2 - 4t = 0$에서 $t = 0$, $4$이다.
따라서 $l(t)$는 $t = 4$일 때 극대이면서 최대이므로 
$l(t)$의 최댓값은 $\frac{32}{3}$이다.

(가)에 알맞은 식은 $f(t) = -\frac{1}{6}t^3 + \frac{1}{2}t^2$이고, 
(나), (다)에 알맞은 수는 각각 $a = 4$, $b = \frac{32}{3}$이다. 
따라서 
$\frac{a \times b}{f(2)} = \frac{4 \times \frac{32}{3}}{f(2)}$이다.
$f(2) = -\frac{1}{6} \times 2^3 + \frac{1}{2} \times 2^2 = -\frac{4}{3} + 2 = \frac{2}{3}$이므로
$\frac{4 \times \frac{32}{3}}{\frac{2}{3}} = 64$이다.

정답은 ③이다.
