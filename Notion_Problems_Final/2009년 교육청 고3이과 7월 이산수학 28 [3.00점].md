## 문제
함수 $f(x) = \frac{\sin x + \cos x}{\sin x + \cos x - 2}$ 에 대하여 <보기>에서 옳은 것만을 있는 대로 고른 것은?

ㄱ. 최솟값은 $-1 - \sqrt{2}$ 이다.
ㄴ. $x = \frac{\pi}{4}$ 에서 최댓값을 갖는다.
ㄷ. $x = \frac{5}{4}\pi$ 에서 극댓값을 갖는다.

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄷ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
[정답] ③ 

$t = \sin x + \cos x = \sqrt{2} \sin \left( x + \frac{\pi}{4} \right)$ 라 놓으면 
$-\sqrt{2} \le t \le \sqrt{2}$ 
$g(t) = \frac{t}{t - 2} = 1 + \frac{2}{t - 2}$ 

ㄱ. $t = \sqrt{2}$ 일 때, 최솟값 
$1 + \frac{2}{\sqrt{2} - 2} = 1 + \frac{2(\sqrt{2} + 2)}{2 - 4} = 1 - (\sqrt{2} + 2) = -1 - \sqrt{2}$ (참) 

ㄴ. $x = \frac{\pi}{4}$ 일 때, 
$t = \sqrt{2}$ 에서 최솟값을 갖는다. (거짓) 

ㄷ. $f(x) = 1 + \frac{2}{\sqrt{2} \sin \left( x + \frac{\pi}{4} \right) - 2}$ 이다. 
$f'(x) = -\frac{2 \sqrt{2} \cos \left( x + \frac{\pi}{4} \right)}{\left( \sqrt{2} \sin \left( x + \frac{\pi}{4} \right) - 2 \right)^{2}}$ 이므로 
$x = 2n \pi + \frac{\pi}{4}$ ($n$ 은 정수)에서 극솟값을 갖고 
$x = 2n \pi + \frac{5}{4}\pi$ ($n$ 은 정수)에서 극댓값을 갖는다. (참)
