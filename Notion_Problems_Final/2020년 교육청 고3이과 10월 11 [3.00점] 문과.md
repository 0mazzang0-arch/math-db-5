## 문제
$0 \le x < 2 \pi$ 일 때, 방정식 $\sin x = \sqrt{3} ( 1 + \cos x )$ 의 모든 해의 합은?

## 보기
① $\frac{\pi}{3}$   ② $\frac{2}{3} \pi$   ③ $\pi$   ④ $\frac{4}{3} \pi$   ⑤ $\frac{5}{3} \pi$

## 풀이
$\sin x = \sqrt{3} ( 1 + \cos x )$ 이고, $\sin^{2} x = 1 - \cos^{2} x$ 이므로
$1 - \cos^{2} x = 3 ( 1 + \cos x )^{2}$
$2 ( 1 + \cos x ) ( 2 \cos x + 1 ) = 0$

(ⅰ) $\cos x = -1$ 일 때,
$\sin x = 0$ 이고 $x = \pi$

(ⅱ) $\cos x = - \frac{1}{2}$ 일 때,
$\sin x = \frac{\sqrt{3}}{2}$ 이고 $x = \frac{2}{3} \pi$

(ⅰ), (ⅱ)에서 방정식의 모든 해의 합은
$\pi + \frac{2}{3} \pi = \frac{5}{3} \pi$ 이다.

정답은 ⑤이다.
