## 문제
실수 $x$ 에 대하여 두 조건 $p$, $q$ 가 
$p : |x - 2| < 2$, $q : 5 - k < x < k$
일 때, 명제 $p \rightarrow q$ 가 참이 되도록 하는 실수 $k$ 의 최솟값은?

## 보기
① $3$   ② $5$   ③ $7$   ④ $9$   ⑤ $11$

## 풀이
[정답] ② 

조건 $p$ 가 $|x - 2| < 2$ 이므로 
$-2 < x - 2 < 2$ 에서
$0 < x < 4$
조건 $p$ 의 진리집합은 $P = \{ x | 0 < x < 4 \}$
조건 $q$ 가 $5 - k < x < k$ 이므로 
조건 $q$ 의 진리집합은 $Q = \{ x | 5 - k < x < k \}$
명제 $p \rightarrow q$ 가 참이 되려면 $P \subset Q$ 이어야 한다.

> [!example] 📸 (그림/그래프)

$5 - k \le 0$ 이고 $k \ge 4$ 에서 
$k \ge 5$ 이고 $k \ge 4$
$\therefore k \ge 5$ 
따라서 $k$ 의 최솟값은 $5$ 이다.
