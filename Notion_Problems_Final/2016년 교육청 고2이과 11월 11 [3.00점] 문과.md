## 문제
$\int _{0} ^{1} ( 4x-3 ) dx + \int _{1} ^{k} ( 4x-3 ) dx = 0$ 일 때, 양수 $k$ 의 값은?

## 보기
① $\frac{3}{2}$   ② $2$   ③ $\frac{5}{2}$   ④ $3$   ⑤ $\frac{7}{2}$

## 풀이
$\int _{0} ^{1} ( 4x-3 ) dx + \int _{1} ^{k} ( 4x-3 ) dx$
$= \int _{0} ^{k} ( 4x-3 ) dx$
$= [ 2x^{2} - 3x ] _{0} ^{k}$
$= 2k^{2} - 3k = 0$
$k ( 2k-3 ) = 0$
따라서 $k = \frac{3}{2}$ ($k > 0$)

정답: ①
