## 문제
부등식 $\log |x-1| + \log (x+2) \le 1$을 만족시키는 모든 정수 $x$의 값의 합을 구하시오.

## 보기

## 풀이
$|x-1|$과 $x+2$는 로그의 진수이므로 
$|x-1| > 0$, $x+2 > 0$ 
$\therefore x \ne 1$, $x > -2$

(ⅰ) $-2 < x < 1$ 인 경우 
$\log (-x+1) + \log (x+2) \le 1$ 
$(-x+1)(x+2) \le 10$, $x^{2} + x - 2 \ge -10$ 
$x^{2} + x + 8 = (x + \frac{1}{2})^{2} + \frac{31}{4} \ge 0$ 
$-2 < x < 1$ 인 모든 실수 $x$에 대하여 항상 성립한다. 
$x$는 정수이므로 $x = -1$ 또는 $x = 0$

(ⅱ) $x > 1$ 인 경우 
$\log (x-1) + \log (x+2) \le 1$ 
$(x-1)(x+2) \le 10$, $(x+4)(x-3) \le 0$ 
$-4 \le x \le 3$ 
$x > 1$ 이므로 $1 < x \le 3$ 
$x$는 정수이므로 $x = 2$ 또는 $x = 3$

(ⅰ), (ⅱ)에 의하여 모든 정수 $x$의 값의 합은 
$(-1) + 0 + 2 + 3 = 4$

[정답] $4$
