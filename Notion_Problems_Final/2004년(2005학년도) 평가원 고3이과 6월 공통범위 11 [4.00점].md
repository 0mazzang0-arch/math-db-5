## 문제
연속확률변수 $X$의 확률밀도함수 $f(x)$가 모든 실수 $x$에 대하여 $f(2+x)=f(2-x)$를 만족시킨다. $a$, $b$ ($a < b$)에 대하여 $P(2-a \leq X \leq 2+b) = p_1$, $P(2+a \leq X \leq 2+b) = p_2$일 때, 확률 $P(2-b \leq X \leq 2+b)$를 $p_1$과 $p_2$로 나타낸 것은? (단, $p_1 > 0$, $p_2 > 0$이다.)



> [!example] 📸 (그림/그래프)

## 보기
① $p_1 + p_2$   ② $\frac{p_1 + p_2}{2}$   ③ $\frac{p_1 - p_2}{2}$   ④ $p_1 - p_2$   ⑤ $p_2 - p_1$

## 풀이
[정답] ①

$f(x)$가 $f(2+x)=f(2-x)$를 만족하므로 $X$의 확률밀도함수 $f(x)$는 $x=2$에 대하여 대칭이다.

$P(2-a \leq X \leq 2+b) = p_1$ 에서
$P(2-a \leq X \leq 2) + P(2 \leq X \leq 2+a) + P(2+a \leq X \leq 2+b) = p_1$ 이고,
$P(2+a \leq X \leq 2+b) = p_2$ 이므로
$P(2-a \leq X \leq 2) + P(2 \leq X \leq 2+a) + p_2 = p_1$ 이다.

이때 $x=2$에 대한 대칭성에 의해 $P(2-a \leq X \leq 2) = P(2 \leq X \leq 2+a)$ 이므로
$2 \times P(2 \leq X \leq 2+a) + p_2 = p_1$
$2 \times P(2 \leq X \leq 2+a) = p_1 - p_2$

구하고자 하는 확률 $P(2-b \leq X \leq 2+b)$는 대칭성에 의해
$P(2-b \leq X \leq 2+b) = 2 \times P(2 \leq X \leq 2+b)$
$= 2 \times \{ P(2 \leq X \leq 2+a) + P(2+a \leq X \leq 2+b) \}$
$= 2 \times P(2 \leq X \leq 2+a) + 2 \times P(2+a \leq X \leq 2+b)$
$= (p_1 - p_2) + 2p_2$
$= p_1 + p_2$
