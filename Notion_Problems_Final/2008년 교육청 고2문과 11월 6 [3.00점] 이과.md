## 문제
세 수 $A = \sqrt[3]{2}$, $B = \sqrt[4]{3}$, $C = \sqrt[6]{5}$ 의 대소 관계를 바르게 나타낸 것은?

## 보기
① $A < B < C$   ② $A < C < B$   ③ $B < C < A$   ④ $C < B < A$   ⑤ $C < A < B$

## 풀이
세 수 $A, B, C$의 각 항을 $12$제곱하면 다음과 같다.
$A^{12} = (\sqrt[3]{2})^{12} = (\sqrt[3]{2^{12}}) = 2^{\frac{12}{3}} = 16$
$B^{12} = (\sqrt[4]{3})^{12} = (\sqrt[4]{3^{12}}) = 3^{\frac{12}{4}} = 27$
$C^{12} = (\sqrt[6]{5})^{12} = (\sqrt[6]{5^{12}}) = 5^{\frac{12}{6}} = 25$

$16 < 25 < 27$이므로 $A^{12} < C^{12} < B^{12}$이다.
따라서 $\therefore A < C < B$

정답: ②
