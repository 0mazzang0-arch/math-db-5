## 문제
$x$에 대한 이차방정식 $x^{2} - px + p + 3 = 0$이 허근 $\alpha$를 가질 때, $\alpha^{3}$이 실수가 되도록 하는 모든 실수 $p$의 값의 곱은?



> [!example] 📸 (그림/그래프)

## 보기
① $-2$   ② $-3$   ③ $-4$   ④ $-5$   ⑤ $-6$

## 풀이
$\alpha = a + bi$ ($a$, $b$는 실수)라 하면 $\bar{\alpha} = a - bi$
계수가 실수인 이차방정식 $x^{2} - px + p + 3 = 0$의 한 근이 $a + bi$이면 다른 한 근은 $a - bi$이다.
따라서 근과 계수의 관계에 의하여
$(a + bi) + (a - bi) = 2a = p$, $(a + bi)(a - bi) = a^{2} + b^{2} = p + 3$
$\therefore a = \frac{p}{2}$, $b^{2} = -a^{2} + p + 3 = -\frac{p^{2}}{4} + p + 3$ $\cdots$ ㉠

이때,
$\alpha^{3} = (a + bi)^{3} = a^{3} + 3a^{2}bi - 3ab^{2} - b^{3}i = (a^{3} - 3ab^{2}) + (3a^{2}b - b^{3})i$
에서 $\alpha^{3}$이 실수이므로 $3a^{2}b - b^{3} = 0$
그런데 $b \neq 0$이므로 $b^{2} = 3a^{2}$ $\cdots$ ㉡

㉠을 ㉡에 대입하면 $-\frac{p^{2}}{4} + p + 3 = 3(\frac{p}{2})^{2}$
$\therefore p^{2} - p - 3 = 0$
따라서 근과 계수의 관계에 의하여 모든 실수 $p$의 곱은 $-3$이다.

**[다른 풀이 1]**
이차방정식 $x^{2} - px + p + 3 = 0$이 허근을 가지므로 이 이차방정식의 판별식을 $D$라 하면
$D = p^{2} - 4(p + 3) < 0$, $p^{2} - 4p - 12 < 0$
$(p + 2)(p - 6) < 0$
$\therefore -2 < p < 6$ $\cdots$ ㉠

이차방정식 $x^{2} - px + p + 3 = 0$의 한 허근이 $\alpha$이므로
$\alpha^{2} - p\alpha + p + 3 = 0$
$\therefore \alpha^{2} = p\alpha - p - 3$

위 식의 양변에 $\alpha$를 곱하면
$\alpha^{3} = \alpha^{2} \times \alpha = p\alpha^{2} - (p + 3)\alpha = p(p\alpha - p - 3) - (p + 3)\alpha = (p^{2} - p - 3)\alpha - p(p + 3)$
이때, $\alpha^{3}$은 실수, $p(p + 3)$은 실수, $\alpha$는 허수이므로
$p^{2} - p - 3 = 0$

$f(p) = p^{2} - p - 3$이라 하면 $f(p) = (p - \frac{1}{2})^{2} - \frac{13}{4}$
이때, ㉠에서 $-2 < p < 6$이어야 한다.
(ⅰ) $y = f(p)$의 그래프의 대칭축 $p = \frac{1}{2}$은 $-2$와 $6$ 사이에 존재한다.
(ⅱ) $f(-2) = 3 > 0$, $f(6) = 27$이다.
(ⅲ) $f(\frac{1}{2}) < 0$이므로 실근이 존재한다.
이상에서 $p^{2} - p - 3 = 0$의 두 실근은 $-2$와 $6$ 사이에 존재한다. 따라서 $\alpha^{3}$이 실수가 되도록 하는 모든 실수 $p$의 값의 곱은 $-3$이다.

**[다른 풀이 2]**
이차방정식 $x^{2} - px + p + 3 = 0$이 허근을 가지므로 이 이차방정식의 판별식을 $D$라 하면
$D = p^{2} - 4(p + 3) < 0$, $p^{2} - 4p - 12 < 0$
$(p + 2)(p - 6) < 0$
$\therefore -2 < p < 6$ $\cdots$ ㉠

이차방정식 $x^{2} - px + p + 3 = 0$의 한 허근이 $\alpha$이므로
$\alpha^{2} - p\alpha + p + 3 = 0$
$\therefore \alpha^{2} - p\alpha + p^{2} = p^{2} - p - 3$

위 식의 양변에 $\alpha + p$를 곱하면
$\alpha^{3} + p^{3} = (\alpha + p)(p^{2} - p - 3)$
$\therefore \alpha^{3} = (p^{2} - p - 3)\alpha + p(p^{2} - p - 3) - p^{3} = (p^{2} - p - 3)\alpha - p(p + 3)$
이때, $\alpha^{3}$이 실수이므로 $p^{2} - p - 3 = 0$
따라서 근과 계수의 관계에 의하여 모든 실수 $p$의 곱은 $-3$이다.
