## 문제
함수 $y = \log_{\frac{1}{2}} (x-a) + b$ 가 닫힌구간 $[2, 5]$ 에서 최댓값 $3$, 최솟값 $1$ 을 갖는다. $a+b$ 의 값은? (단, $a, b$ 는 상수이다.)

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
함수 $y = \log_{\frac{1}{2}} (x-a) + b$ 는 $x$ 의 값이 증가하면 $y$ 의 값은 감소하므로 $x=2$ 일 때 최댓값 $3$, $x=5$ 일 때 최솟값 $1$ 을 갖는다.

$\log_{\frac{1}{2}} (2-a) + b = 3$
$\log_{\frac{1}{2}} (5-a) + b = 1$

두 식을 연립하여 빼면 다음과 같다.
$\log_{\frac{1}{2}} (2-a) - \log_{\frac{1}{2}} (5-a) = 2$
$\log_{\frac{1}{2}} \frac{2-a}{5-a} = 2$

로그의 정의에 의해
$\frac{2-a}{5-a} = \left(\frac{1}{2}\right)^2 = \frac{1}{4}$
$4(2-a) = 5-a$
$8-4a = 5-a$
$3a = 3$
$a = 1$

$a=1$ 을 첫 번째 식에 대입하면
$\log_{\frac{1}{2}} (2-1) + b = 3$
$0 + b = 3$
$b = 3$

따라서 $a+b = 1+3 = 4$ 이다.

정답: ④
