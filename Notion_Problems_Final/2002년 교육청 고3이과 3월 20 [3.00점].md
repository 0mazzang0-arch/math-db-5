## 문제
실수 $a$ 에 대하여 $a$ 보다 크지 않은 최대의 정수를 $[a]$, $a$ 보다 작지 않은 최소의 정수를 $\langle a \rangle$ 로 나타내기로 한다. 예를 들어 $[4.3] = 4$, $\langle 4.3 \rangle = 5$ 이다. 이때, $\langle [\frac{3x+8}{x+2}] + \frac{x}{5} \rangle = 5$ 를 만족시키는 양의 정수 $x$ 의 개수는?

## 보기
① $3$   ② $5$   ③ $7$   ④ $9$   ⑤ $11$

## 풀이
$\frac{3x+8}{x+2} = 3 + \frac{2}{x+2}$ 이고 $x$ 가 양의 정수일 때 $0 < \frac{2}{x+2} < 1$ 이므로 
$3 < \frac{3x+8}{x+2} < 4$ 이다.
따라서 $[\frac{3x+8}{x+2}] = 3$ 이다.

$\langle [\frac{3x+8}{x+2}] + \frac{x}{5} \rangle = 5 \iff \langle 3 + \frac{x}{5} \rangle = 5$
최소의 정수를 나타내는 기호 $\langle \cdot \rangle$ 의 정의에 의해
$4 < 3 + \frac{x}{5} \leq 5$
양변에서 $3$ 을 빼면
$1 < \frac{x}{5} \leq 2$
$5 < x \leq 10$

따라서 이를 만족시키는 양의 정수 $x$ 는 $6, 7, 8, 9, 10$ 으로 그 개수는 $5$ 개이다.
