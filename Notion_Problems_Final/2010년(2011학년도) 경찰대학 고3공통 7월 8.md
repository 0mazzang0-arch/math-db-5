## 문제
복소수 $z$에 대하여 $\frac{z-\bar{z}}{i} < 0$이고 $\frac{z}{1+z^{2}}$와 $\frac{z^{2}}{1+z}$가 모두 실수일 때, $z$의 값은? (단, $\bar{z}$는 $z$의 켤레복소수이고 $i$는 허수단위이다.)

## 보기
① $\frac{1}{2} - \frac{\sqrt{3}}{2}i$   ② $-\frac{1}{2} - \frac{\sqrt{3}}{2}i$   ③ $-\frac{\sqrt{3}}{2} - \frac{1}{2}i$   ④ $-\frac{1}{2} + \frac{\sqrt{3}}{2}i$   ⑤ $\frac{\sqrt{3}}{2} - \frac{1}{2}i$

## 풀이
$\frac{z-\bar{z}}{i} < 0$이므로 $z - \bar{z} \neq 0$이다.
$\frac{z}{1+z^{2}}$가 실수이므로 $\frac{z}{1+z^{2}} = \overline{\left( \frac{z}{1+z^{2}} \right)}$이다.
$\frac{z}{1+z^{2}} = \frac{\bar{z}}{1+\bar{z}^{2}}$
$z(1+\bar{z}^{2}) = \bar{z}(1+z^{2})$
$z + z\bar{z}^{2} - \bar{z} - z^{2}\bar{z} = 0$
$(z - \bar{z}) - z\bar{z}(z - \bar{z}) = 0$
$(z - \bar{z})(1 - z\bar{z}) = 0$
$\therefore z\bar{z} = 1$ ($\because z - \bar{z} \neq 0$)

또한, $\frac{z^{2}}{1+z}$가 실수이므로 $\frac{z^{2}}{1+z} = \overline{\left( \frac{z^{2}}{1+z} \right)}$이다.
$\frac{z^{2}}{1+z} = \frac{\bar{z}^{2}}{1+\bar{z}}$
$z^{2}(1+\bar{z}) = \bar{z}^{2}(1+z)$
$z^{2} + z^{2}\bar{z} - \bar{z}^{2} - z\bar{z}^{2} = 0$
$(z - \bar{z})(z + \bar{z}) + z\bar{z}(z - \bar{z}) = 0$
$(z - \bar{z})(z + \bar{z} + z\bar{z}) = 0$
$\therefore z + \bar{z} = -z\bar{z} = -1$ ($\because z - \bar{z} \neq 0, z\bar{z} = 1$)

따라서 $z$, $\bar{z}$를 두 근으로 하는 이차방정식은
$x^{2} + x + 1 = 0$
$\therefore x = \frac{-1 \pm \sqrt{3}i}{2}$

그런데 $\frac{z - \bar{z}}{i} < 0$에서 $z = a + bi$ ($a, b$는 실수)라 하면
$\frac{2bi}{i} = 2b < 0 \quad \therefore b < 0$
따라서 $z = \frac{-1 - \sqrt{3}i}{2} = -\frac{1}{2} - \frac{\sqrt{3}}{2}i$이다.

정답: ②
