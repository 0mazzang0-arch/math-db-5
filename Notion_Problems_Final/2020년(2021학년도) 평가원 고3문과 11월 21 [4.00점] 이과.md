## 문제
수열 $\{a_{n}\}$은 $0 < a_{1} < 1$이고, 모든 자연수 $n$에 대하여 다음 조건을 만족시킨다.

(가) $a_{2n} = a_{2} \times a_{n} + 1$
(나) $a_{2n+1} = a_{2} \times a_{n} - 2$

$a_{7} = 2$일 때, $a_{25}$의 값은?

## 보기
① $78$   ② $80$   ③ $82$   ④ $84$   ⑤ $86$

## 풀이
$n = 1$일 때,
(가)에서 
$a_{2} = a_{2} \times a_{1} + 1$
$a_{2} = \frac{1}{1 - a_{1}} \cdots$ ㉠

(나)에서 
$a_{3} = a_{2} \times a_{1} - 2$
$a_{3} = \frac{1}{1 - a_{1}} - 3 = \frac{-2 + 3a_{1}}{1 - a_{1}} \cdots$ ㉡

$n = 3$일 때,
(나)에서 
$a_{7} = 2 = a_{2} \times a_{3} - 2$
$a_{2} \times a_{3} = 4$

㉠, ㉡을 대입하여 정리하면
$4a_{1}^{2} - 11a_{1} + 6 = (a_{1} - 2)(4a_{1} - 3) = 0$

$0 < a_{1} < 1$이므로 
$a_{1} = \frac{3}{4}$

따라서 
$a_{2} = 4$, $a_{3} = \frac{4}{a_{2}} = 1$

$a_{2n} = 4a_{n} + 1$, $a_{2n+1} = 4a_{n} - 2$

$a_{25} = 4a_{12} - 2$
$= 4(4a_{6} + 1) - 2$
$= 16a_{6} + 2$
$= 16(4a_{3} + 1) + 2 = 64a_{3} + 18 = 64 + 18 = 82$

정답 ③
