## 문제
최고차항의 계수가 음수인 삼차함수 $f(x)$의 도함수를 $h(x)$라 하자. $f(-1) = f(1) = f(2) = 0$일 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?

$$ \begin{array}{|l|} \hline \text{ㄱ. } \int_{1}^{2} f(x) dx > 0 \\ \text{ㄴ. } h(0) < 0 \\ \text{ㄷ. } \int_{m}^{n} h(x) dx \text{의 값이 최대일 때, } m+n = \frac{4}{3} \text{이다. } (m < n) \\ \hline \end{array} $$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄷ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
ㄱ. $f(x) = a(x+1)(x-1)(x-2)$ ($a < 0$)라 하자.
$1 < x < 2$ 일 때, $f(x) > 0$ 이므로
$\int_{1}^{2} f(x) dx > 0$
$\therefore$ (참)

ㄴ. $f(x) = a(x^{3} - 2x^{2} - x + 2)$ 이므로
$h(x) = f'(x) = a(3x^{2} - 4x - 1)$
$h(0) = -a$ 이고 $a < 0$ 이므로 $h(0) > 0$
$\therefore$ (거짓)

ㄷ. 방정식 $h(x) = 0$ 의 서로 다른 두 실근을 $\alpha, \beta$ ($\alpha < \beta$)라 하자.

> [!example] 📸 (그림/그래프)

$\int_{m}^{n} h(x) dx$ 의 값이 최대가 되려면 $[m, n]$ 이 $h(x) \geq 0$ 를 만족시키는 구간이어야 하므로
$m = \alpha, n = \beta$
근과 계수의 관계에 의해
$\therefore m + n = \alpha + \beta = \frac{4}{3}$
$\therefore$ (참)

따라서 옳은 것은 $ㄱ, ㄷ$ 이다.
