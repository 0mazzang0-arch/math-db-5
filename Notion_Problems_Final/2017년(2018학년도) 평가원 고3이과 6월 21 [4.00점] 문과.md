## 문제
최고차항의 계수가 $1$인 사차함수 $f(x)$에 대하여 $F(x) = \ln |f(x)|$라 하고, 최고차항의 계수가 $1$인 삼차함수 $g(x)$에 대하여 $G(x) = \ln |g(x) \sin x|$라 하자.
$\lim_{x \to 1} (x-1) F'(x) = 3$, $\lim_{x \to 0} \frac{F'(x)}{G'(x)} = \frac{1}{4}$일 때, $f(3) + g(3)$의 값은?

## 보기
① $57$   ② $55$   ③ $53$   ④ $51$   ⑤ $49$

## 풀이
$F(x) = \ln |f(x)|$이므로 $F'(x) = \frac{f'(x)}{f(x)}$이다.
$\lim_{x \to 1} (x-1) F'(x) = 3$에서
$\lim_{x \to 1} \frac{(x-1) f'(x)}{f(x)} = 3 \cdots ㉠$
이때 $x \to 1$일 때 (분자) $\to 0$이므로 (분모) $\to 0$에서
$\lim_{x \to 1} f(x) = f(1) = 0$
또, $f'(x)$는 다항함수이므로 미분가능하다.
그러므로 $㉠$에서
$\lim_{x \to 1} \frac{f'(x)}{\frac{f(x)-f(1)}{x-1}} = 3$
이때, $f'(1) \neq 0$이라 하면 위의 식의 좌변은 $\frac{f'(1)}{f'(1)} = 1$이므로 만족시키지 않는다.
그러므로 $f'(1) = 0$이어야 한다.
이때, $f(x) = (x-1)^2 (x^2 + ax + b)$ ($1+a+b \neq 0$)라 하면 $㉠$의 좌변은
$\lim_{x \to 1} \frac{(x-1) \{2(x-1)(x^2+ax+b) + (x-1)^2 (2x+a)\}}{(x-1)^2 (x^2+ax+b)}$
$= \lim_{x \to 1} \frac{2(x^2+ax+b) + (x-1)(2x+a)}{x^2+ax+b} = 2$
그러므로 만족시키지 않는다.
$f(x) = (x-1)^3 (x+a)$ ($a+1 \neq 0$)이라 하면 $㉠$의 좌변은
$\lim_{x \to 1} \frac{(x-1) \{3(x-1)^2 (x+a) + (x-1)^3\}}{(x-1)^3 (x+a)}$
$= \lim_{x \to 1} \frac{3(x+a) + (x-1)}{x+a} = 3$
그러므로 만족시킨다.
따라서 $f(x) = (x-1)^3 (x+a)$이다.

한편 $\lim_{x \to 0} \frac{F'(x)}{G'(x)} = \frac{1}{4}$에서
$F'(x) = \frac{3(x-1)^2 (x+a) + (x-1)^3}{(x-1)^3 (x+a)} = \frac{3(x+a) + (x-1)}{(x-1)(x+a)}$이므로
$\lim_{x \to 0} \frac{\frac{3(x+a) + (x-1)}{(x-1)(x+a)}}{\frac{g'(x) \sin x + g(x) \cos x}{g(x) \sin x}}$
$= \lim_{x \to 0} \frac{\{3(x+a) + (x-1)\} g(x) \sin x}{(x-1)(x+a) \{g'(x) \sin x + g(x) \cos x\}}$
$= \lim_{x \to 0} \frac{(4x+3a-1) g(x) \sin x}{(x-1)(x+a) \{g'(x) \sin x + g(x) \cos x\}} = \frac{1}{4} \cdots ㉡$
위에서 $x \to 0$일 때 (분자) $\to 0$이고 극한값이 $0$이 아니므로 (분모) $\to 0$에서
$\lim_{x \to 0} [(x-1)(x+a) \{g'(x) \sin x + g(x) \cos x\}] = 0$
$a g(0) = 0$
$a=0$ 또는 $g(0)=0$

$a=0$일 때, $㉡$에서
$\lim_{x \to 0} \frac{(4x-1) g(x) \sin x}{x(x-1) \{g'(x) \sin x + g(x) \cos x\}}$
$= \lim_{x \to 0} \left[ \frac{g(x)}{g'(x) \sin x + g(x) \cos x} \cdot \frac{4x-1}{x-1} \cdot \frac{\sin x}{x} \right] = \frac{1}{4}$
이때, 위의 극한값이 존재하고 극한값 $\lim_{x \to 0} \frac{4x-1}{x-1} = 1, \lim_{x \to 0} \frac{\sin x}{x} = 1$이므로
$\lim_{x \to 0} \frac{g(x)}{g'(x) \sin x + g(x) \cos x} = \frac{1}{4} \cdots ㉢$이어야 한다.
$g(0) \neq 0$이라 하면 위의 식의 좌변은 $\frac{g(0)}{g(0)} = 1$이므로 만족시키지 않는다.
$g(0)=0$일 때, $g(x) = x(x^2+cx+d)$ ($d \neq 0$)라 하면 $㉢$의 좌변은
$\lim_{x \to 0} \frac{x(x^2+cx+d)}{(3x^2+2cx+d) \sin x + x(x^2+cx+d) \cos x}$
$= \lim_{x \to 0} \frac{x^2+cx+d}{(3x^2+2cx+d) \frac{\sin x}{x} + (x^2+cx+d) \cos x} = \frac{d}{d+d} = \frac{1}{2}$
이므로 만족시키지 않는다.
그러므로 $g(x) = x^2 (x+c)$ ($c \neq 0$)이라 하면 $㉢$의 좌변은
$\lim_{x \to 0} \frac{x^2 (x+c)}{(3x^2+2cx) \sin x + x^2 (x+c) \cos x}$
$= \lim_{x \to 0} \frac{x+c}{(3x+2c) \frac{\sin x}{x} + (x+c) \cos x} = \frac{c}{2c+c} = \frac{1}{3}$
$g(x) = x^3$이라 하면 $㉢$의 좌변은
$\lim_{x \to 0} \frac{x^3}{3x^2 \sin x + x^3 \cos x} = \lim_{x \to 0} \frac{1}{3 \frac{\sin x}{x} + \cos x} = \frac{1}{4}$
그러므로 위의 식을 만족시킨다. 따라서 $g(x) = x^3$이다.

만약 $a \neq 0$이고 $g(0)=0$이라 하자.
$g(x) = x(x^2+cx+d)$라 하면 $㉡$에서
$-\frac{1}{a} \lim_{x \to 0} \frac{(4x-3a+1)(x^2+cx+d) \sin x}{(3x^2+2cx+d) \frac{\sin x}{x} + (x^2+cx+d) \cos x} = \frac{1}{4}$
이때, $x \to 0$일 때 (분자) $\to 0$이고 극한값이 $0$이 아니므로 (분모) $\to 0$에서 $d=0$이다.
이때 위의 식은
$-\frac{1}{a} \lim_{x \to 0} \frac{(4x-3a+1)(x+c) \sin x}{(3x+2c) \frac{\sin x}{x} + (x+c) \cos x} = \frac{1}{4}$
$x \to 0$일 때 (분자) $\to 0$이고 극한값이 $0$이 아니므로 (분모) $\to 0$에서 $c=0$이다.
이때 위의 식은
$-\frac{1}{a} \lim_{x \to 0} \frac{(4x-3a+1) x \sin x}{3 \sin x + x \cos x} = -\frac{1}{a} \lim_{x \to 0} \frac{(4x-3a+1) \sin x}{3 \frac{\sin x}{x} + \cos x} = 0$
그러므로 조건을 만족시키지 않는다.

따라서 $f(x) = x(x-1)^3$, $g(x) = x^3$이므로
$f(3) + g(3) = 3(2)^3 + 3^3 = 24 + 27 = 51$
정답은 ④이다.
