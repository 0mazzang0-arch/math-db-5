## 문제
실수 $a$ 에 대하여 $a$ 보다 크지 않은 최대의 정수를 $[a]$, $a$ 보다 작지 않은 최소의 정수를 $\langle a \rangle$ 로 나타내기로 한다. 예를 들어 $[4.3]=4$, $\langle 4.3 \rangle=5$ 이다. 이때, $\langle [\frac{3x+8}{x+2}] + \frac{x}{5} \rangle = 5$ 를 만족시키는 양의 정수 $x$ 의 개수는?

## 보기
① $3$   ② $5$   ③ $7$   ④ $9$   ⑤ $11$

## 풀이
$\frac{3x+8}{x+2} = 3 + \frac{2}{x+2}$ 이고, $x$ 가 양의 정수이므로 $x \ge 1$ 에서 $0 < \frac{2}{x+2} \le \frac{2}{3} < 1$ 이다.
따라서 $3 < \frac{3x+8}{x+2} < 4$ 이므로
$\therefore [\frac{3x+8}{x+2}] = 3$

주어진 식에 대입하면
$\therefore \langle [\frac{3x+8}{x+2}] + \frac{x}{5} \rangle = 5 \iff \langle 3 + \frac{x}{5} \rangle = 5$

$\langle a \rangle = n$ 이면 $n-1 < a \le n$ 이므로
$4 < 3 + \frac{x}{5} \le 5$
$1 < \frac{x}{5} \le 2$
$5 < x \le 10$

이를 만족하는 양의 정수 $x$ 는 $6, 7, 8, 9, 10$ 이다.
따라서 양의 정수 $x$ 의 개수는 $5$ 개이다.

정답: ②
