## 문제
두 다항함수 $f(x)$, $g(x)$가 다음 조건을 만족시킨다.

(가) $g(x) = x^3 f(x) - 7$
(나) $\lim_{x \to 2} \frac{f(x) - g(x)}{x - 2} = 2$

곡선 $y = g(x)$ 위의 점 $(2, g(2))$에서의 접선의 방정식이 $y = ax + b$일 때, $a^2 + b^2$의 값을 구하시오. (단, $a$, $b$는 상수이다.)

## 보기

## 풀이
조건 (나)에서 $x \to 2$일 때, (분모) $\to 0$이므로 (분자) $\to 0$이어야 한다.
즉, $f(2) - g(2) = 0$이므로 $f(2) = g(2)$이다.
조건 (가)에서 $x = 2$를 대입하면
$g(2) = 2^3 f(2) - 7 = 8f(2) - 7$이므로
$g(2) = 8g(2) - 7$에서 $7g(2) = 7$, 즉 $g(2) = 1$이다.

또한 조건 (나)에서
$\lim_{x \to 2} \frac{\{f(x) - f(2)\} - \{g(x) - g(2)\}}{x - 2} = f'(2) - g'(2) = 2$이다.
조건 (가)의 양변을 $x$에 대하여 미분하면
$g'(x) = 3x^2 f(x) + x^3 f'(x)$
$x = 2$를 대입하면
$g'(2) = 12f(2) + 8f'(2)$
이때 $f(2) = 1$이고 $f'(2) = g'(2) + 2$이므로
$g'(2) = 12 \times 1 + 8 \{g'(2) + 2\} = 8g'(2) + 28$
$-7g'(2) = 28$에서 $g'(2) = -4$이다.

따라서 접선의 방정식은
$y - 1 = -4(x - 2)$
$y = -4x + 9$
이므로 $a = -4$, $b = 9$이다.
따라서 $a^2 + b^2 = (-4)^2 + 9^2 = 16 + 81 = 97$이다.

정답: $97$
