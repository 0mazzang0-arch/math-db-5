## 문제
첫째항이 양수인 등차수열 $\{ a_{n} \}$ 에 대하여 
$a_{5} = 3 a_{1}$, $a_{1}^{2} + a_{3}^{2} = 20$ 
일 때, $a_{5}$ 의 값을 구하시오.

## 보기

## 풀이
등차수열 $\{ a_{n} \}$ 의 공차를 $d$ 라 하면 
$a_{5} = 3 a_{1}$ 에서 
$a_{1} + 4 d = 3 a_{1}$, 
$a_{1} = 2 d$ $\cdots$ ㉠ 

$a_{1}^{2} + a_{3}^{2} = 20$ 에서 
$a_{1}^{2} + ( a_{1} + 2 d )^{2} = 20$ $\cdots$ ㉡ 

㉠, ㉡ 에서 
$5 a_{1}^{2} = 20$ 

$a_{1}^{2} = 4$ 에서 
$a_{1} > 0$ 이므로 
$a_{1} = 2$ 

따라서 
$a_{5} = a_{1} + 4 d = 3 a_{1} = 6$

정답: $6$
