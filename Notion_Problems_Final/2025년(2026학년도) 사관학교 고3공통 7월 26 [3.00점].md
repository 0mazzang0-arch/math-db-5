## 문제
정규분포를 따르는 확률변수 $X$에 대하여 $P(a - 3 \le X \le a + 1)$은 $a = 10$일 때 최댓값 $0.6$을 갖는다. $P(X \ge 10) + P(8 \le X \le 11)$의 값은?

## 보기
① $0.5$   ② $0.6$   ③ $0.7$   ④ $0.8$   ⑤ $0.9$

## 풀이
확률변수 $X$의 평균을 $m$이라 하면 $X$의 확률밀도함수는 $x = m$에서 최댓값을 갖고, 정규분포곡선은 직선 $x = m$에 대하여 대칭이다. 
따라서 $P(a - 3 \le X \le a + 1)$이 $a = 10$ 즉, $P(7 \le X \le 11)$에서 최댓값을 가지므로 
$m = \frac{7 + 11}{2} = 9$
확률밀도함수가 직선 $x = 9$에 대하여 대칭이므로 
$P(7 \le X \le 11) = 0.6$, $P(7 \le X \le 9) = 0.3$ 
$P(8 \le X \le 11) = P(7 \le X \le 10)$ 
$\therefore P(X \ge 10) + P(8 \le X \le 11)$ 
$= P(X \ge 10) + P(7 \le X \le 10)$ 
$= P(X \ge 7)$ 
$= P(7 \le X \le 9) + P(X \ge 9)$ 
$= 0.3 + 0.5$ 
$= 0.8$

정답: ④
