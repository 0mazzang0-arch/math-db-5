## 문제
모평균 $75$, 모표준편차 $5$인 정규분포를 따르는 모집단에서 임의 추출한 크기 $25$인 표본의 표본평균을 $\bar{X}$라 하자. 표준정규분포를 따르는 확률변수 $Z$에 대하여 양의 상수 $c$가 $P(|Z| > c) = 0.06$을 만족시킬 때, <보기>에서 옳은 것을 모두 고른 것은?

$$
\begin{array}{|l|}
\hline
\text{ㄱ. } P(Z > a) = 0.05\text{인 상수 } a\text{에 대하여 } c > a\text{이다.} \\
\text{ㄴ. } P(\bar{X} \leq c + 75) = 0.97 \\
\text{ㄷ. } P(\bar{X} > b) = 0.01\text{인 상수 } b\text{에 대하여 } c < b - 75\text{이다.} \\
\hline
\end{array}
$$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
ㄱ. $P(|Z| \leq c) = 1 - 0.06 = 0.94$이므로
$P(0 \leq Z \leq c) = \frac{1}{2} P(|Z| \leq c) = 0.47$이다.
$P(Z > a) = 0.05$이면 $P(0 \leq Z \leq a) = 0.5 - 0.05 = 0.45$이다.
이때 $P(0 \leq Z \leq a) < P(0 \leq Z \leq c)$이므로 $a < c$, 즉 $c > a$이다. (참)

ㄴ. 모평균 $m = 75$, 모표준편차 $\sigma = 5$, 표본의 크기 $n = 25$이므로 표본평균 $\bar{X}$의 평균과 분산은 다음과 같다.
$E(\bar{X}) = m = 75$
$V(\bar{X}) = \frac{\sigma^2}{n} = \frac{25}{25} = 1$
따라서 확률변수 $\bar{X}$는 정규분포 $N(75, 1^2)$을 따르므로
$P(\bar{X} \leq c + 75) = P\left(Z \leq \frac{(c + 75) - 75}{1}\right) = P(Z \leq c)$
$= 0.5 + P(0 \leq Z \leq c) = 0.5 + 0.47 = 0.97$이다. (참)

ㄷ. $P(\bar{X} > b) = P\left(Z > \frac{b - 75}{1}\right) = P(Z > b - 75) = 0.01$이므로
$P(0 \leq Z \leq b - 75) = 0.5 - 0.01 = 0.49$이다.
이때 $P(0 \leq Z \leq c) = 0.47$이므로 $P(0 \leq Z \leq c) < P(0 \leq Z \leq b - 75)$이다.
따라서 $c < b - 75$이다. (참)

따라서 보기 중 옳은 것은 ㄱ, ㄴ, ㄷ이다.
