## 문제
첫째항이 $3$인 등차수열 $\{ a_{n} \}$에 대하여 
$$\sum_{n=1}^{10} ( a_{5n} - a_{n} ) = 440$$ 
일 때, $\sum_{n=1}^{10} a_{n}$의 값을 구하시오.

## 풀이
등차수열 $\{ a_{n} \}$의 공차를 $d$라 하면 일반항은 
$$a_{n} = 3 + ( n - 1 ) d$$ 

$a_{5n} - a_{n} = 4dn$이므로 
$$\sum_{n=1}^{10} 4dn = 4d \times \frac{10 \times 11}{2} = 220d = 440$$ 

$d = 2$이므로 
$$a_{n} = 2n + 1$$ 

따라서 
$$\sum_{n=1}^{10} a_{n} = \sum_{n=1}^{10} ( 2n + 1 ) = 120$$

정답: $120$
