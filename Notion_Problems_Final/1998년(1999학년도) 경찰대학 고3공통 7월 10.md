## 문제
$\int_{1}^{x} (t - x) f(t) dt = x^3 - 3x^2 + ax + b$ 일 때, $a^2 + b^2 + \int_{0}^{1} f(x) dx$ 의 값은?

## 풀이
주어진 식은 임의의 실수 $x$ 에 관하여 성립하므로 $x$ 에 $1$ 을 대입하여 정리하면
$a + b = 2$ $\cdots$ ㉠
또한, 주어진 식의 양변을 $x$ 에 관하여 미분하면
$x f(x) - x f(x) - \int_{1}^{x} f(t) dt = 3x^2 - 6x + a$
이므로
$\int_{1}^{x} f(t) dt = -3x^2 + 6x - a$ $\cdots$ ㉡
식 ㉡ 을 다시 $x$ 에 관하여 미분하면
$f(x) = -6x + 6$
식 ㉡ 에 $x = 1$ 을 대입하여 정리하면
$3 - a = 0$ , $a = 3$
식 ㉠ 으로부터 $b = -1$ 이다. 따라서, 구하는 식의 값은
$10 + \int_{0}^{1} (-6x + 6) dx = 13$
