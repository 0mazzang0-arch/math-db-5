## 문제
$(2+6x-x^{3})^{2} = a_{0} + a_{1}x + a_{2}x^{2} + a_{3}x^{3} + a_{4}x^{4} + a_{5}x^{5} + a_{6}x^{6}$이 $x$에 대한 항등식일 때, $a_{0} + a_{2} + a_{4} + a_{6}$의 값을 구하시오.

## 풀이
주어진 등식의 양변에 $x = 1$을 대입하면
$49 = a_{0} + a_{1} + a_{2} + a_{3} + a_{4} + a_{5} + a_{6} \cdots \cdots$ ㉠
주어진 등식의 양변에 $x = -1$을 대입하면
$9 = a_{0} - a_{1} + a_{2} - a_{3} + a_{4} - a_{5} + a_{6} \cdots \cdots$ ㉡
㉠ $+$ ㉡을 하면
$58 = 2(a_{0} + a_{2} + a_{4} + a_{6})$

$\therefore a_{0} + a_{2} + a_{4} + a_{6} = 29$

**[다른 풀이]**
$(2+6x-x^{3})^{2} = 4 + 24x + 36x^{2} - 4x^{3} - 12x^{4} + x^{6}$이므로
$a_{0} = 4$, $a_{2} = 36$, $a_{4} = -12$, $a_{6} = 1$

$\therefore a_{0} + a_{2} + a_{4} + a_{6} = 4 + 36 - 12 + 1 = 29$

정답: $29$
