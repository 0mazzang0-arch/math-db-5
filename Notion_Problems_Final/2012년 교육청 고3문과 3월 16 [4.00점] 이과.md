## 문제
등식 $2^{a} = 5^{b}$ 을 만족시키는 양의 실수 $a$, $b$ 에 대하여 <보기>에서 옳은 것만을 있는 대로 고른 것은?

$$ \begin{array}{|l|} \hline \text{<보 기>} \\ \text{ㄱ. } b = \frac{1}{2} \text{ 이면 } a = \log_{4} 5 \text{ 이다.} \\ \text{ㄴ. } 2 < \frac{a}{b} < 3 \\ \text{ㄷ. } \frac{1}{a} + \frac{1}{b} \text{ 은 무리수이다.} \\ \hline \end{array} $$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
$ㄱ.$ $b = \frac{1}{2}$ 이면 $2^{a} = 5^{\frac{1}{2}}$ 에서 $a = \log_{2} \sqrt{5} = \log_{4} 5$ (참)

$ㄴ.$ $2^{a} = 5^{b}$ 에서 $2^{\frac{a}{b}} = 5$ 이므로 $\frac{a}{b} = \log_{2} 5$ 이다.
$\log_{2} 4 < \log_{2} 5 < \log_{2} 8$ 에서 $2 < \log_{2} 5 < 3$ 이므로 $2 < \frac{a}{b} < 3$ (참)

$ㄷ.$ (반례) $2^{a} = 5^{b} = 10$ 으로 놓으면 $2 = 10^{\frac{1}{a}}$, $5 = 10^{\frac{1}{b}}$ 에서 $\frac{1}{a} = \log 2$, $\frac{1}{b} = \log 5$ 이다.
따라서 $\frac{1}{a} + \frac{1}{b} = \log 2 + \log 5 = \log 10 = 1$ (유리수)이므로 거짓이다.

이상에서 옳은 것은 $ㄱ, ㄴ$ 이다.
