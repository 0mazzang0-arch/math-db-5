## 문제
두 다항식 $A = x^{2} + 2xy - y^{2}$, $B = x^{2} - xy + 2y^{2}$ 에 대하여 $( 3A + B ) - ( A + 3B )$ 를 간단히 하면?

## 보기
① $3xy - 3y^{2}$   ② $6xy - 6y^{2}$   ③ $2x^{2} + xy + y^{2}$   ④ $4x^{2} + 2xy + 2y^{2}$   ⑤ $6x^{2} + 3xy + 3y^{2}$

## 풀이
[정답] ② 

$$( 3A + B ) - ( A + 3B )$$ 
$$= 2 ( A - B )$$ 
$$= 2 \{ ( x^{2} + 2xy - y^{2} ) - ( x^{2} - xy + 2y^{2} ) \}$$ 
$$= 6xy - 6y^{2}$$
