## 문제
$x = \sqrt{3} + \sqrt{2}$, $y = \sqrt{3} - \sqrt{2}$일 때, $x^2y + xy^2 + x + y$의 값은?

## 보기
① $\sqrt{3}$   ② $2\sqrt{3}$   ③ $3\sqrt{3}$   ④ $4\sqrt{3}$   ⑤ $5\sqrt{3}$

## 풀이
$x = \sqrt{3} + \sqrt{2}$, $y = \sqrt{3} - \sqrt{2}$일 때,
$x + y = (\sqrt{3} + \sqrt{2}) + (\sqrt{3} - \sqrt{2}) = 2\sqrt{3}$
$xy = (\sqrt{3} + \sqrt{2})(\sqrt{3} - \sqrt{2}) = (\sqrt{3})^{2} - (\sqrt{2})^{2} = 3 - 2 = 1$

주어진 식을 공통인수로 묶어 인수분해하면
$x^2y + xy^2 + x + y = xy(x + y) + (x + y)$
$= (x + y)(xy + 1)$

위의 식에 $x + y = 2\sqrt{3}$, $xy = 1$을 대입하면
$(2\sqrt{3}) \times (1 + 1) = 2\sqrt{3} \times 2 = 4\sqrt{3}$

정답: ④
