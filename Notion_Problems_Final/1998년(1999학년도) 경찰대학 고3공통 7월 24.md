## 문제
$b x^{2}+2 x+a=0$ 의 두 근을 $\alpha$, $\beta$ 라 할 때, $a x^{2}-2 x+b>0$ 의 근의 범위는? (단, $0 < \alpha < \beta$)

## 보기
① $\alpha < x < \beta$   ② $-\frac{1}{\alpha} < x < -\frac{1}{\beta}$   ③ $x < -\frac{1}{\alpha}$ 또는 $x > -\frac{1}{\beta}$   ④ $x < -\beta$ 또는 $x > -\alpha$   ⑤ 근이 없다.

## 풀이
근과 계수와의 관계에 의하면 
$$\alpha + \beta = -\frac{2}{b}$$, 
$$\alpha \beta = \frac{a}{b}$$ 

$0 < \alpha < \beta$ 이므로 
$b < 0$, 
$a < 0$ 

부등식 $a x^{2}-2 x+b>0$ 의 양변을 $b$ 로 나누면 ($b < 0$ 이므로 부동호 방향이 바뀐다.) 
$\frac{a}{b} x^{2}-\frac{2}{b} x+1 < 0$ 

$\alpha \beta x^{2}+(\alpha + \beta) x+1 < 0$ 

$(\alpha x+1)(\beta x+1) < 0$ 

따라서, 부등식의 해는 
$-\frac{1}{\alpha} < x < -\frac{1}{\beta}$

정답: ②
