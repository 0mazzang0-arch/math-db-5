## 문제
이차함수 $f(x)=ax^2$ 에 대하여 구간 $[0, 2]$ 에서 정의된 연속확률변수 $X$ 의 확률밀도함수 $g(x)$ 가
$$ g(x) = \begin{cases} f(x) & (0 \le x < 1) \\ f(x-1) + f(1) & (1 \le x \le 2) \end{cases} $$
일 때, $P(a \le X \le a+1) = \frac{q}{p}$ 이다. $p+q$ 의 값을 구하시오. (단, $p$ 와 $q$ 는 서로소인 자연수이다.)

## 풀이
함수 $g(x)$ 가 구간 $[0, 2]$ 에서 정의된 연속확률변수 $X$ 의 확률밀도함수이므로
$P(0 \le X \le 2) = \int_0^2 g(x) dx = 1$ 이다.

$\int_0^2 g(x) dx = \int_0^1 g(x) dx + \int_1^2 g(x) dx$
$= \int_0^1 f(x) dx + \int_1^2 \{ f(x-1) + f(1) \} dx$
$= \int_0^1 f(x) dx + \int_0^1 \{ f(x) + f(1) \} dx$
$= 2 \int_0^1 f(x) dx + f(1)$
$= 2 \int_0^1 ax^2 dx + a$
$= 2 \left[ \frac{ax^3}{3} \right]_0^1 + a$
$= \frac{5}{3} a$

따라서 $\frac{5}{3} a = 1$ 에서 $a = \frac{3}{5}$ 이다.
$f(x) = \frac{3}{5} x^2$

$\therefore P\left(\frac{3}{5} \le X \le \frac{8}{5}\right) = \int_{3/5}^{8/5} g(x) dx$
$= \int_{3/5}^1 g(x) dx + \int_1^{8/5} g(x) dx$
$= \int_{3/5}^1 f(x) dx + \int_1^{8/5} \{ f(x-1) + f(1) \} dx$
$= \int_{3/5}^1 f(x) dx + \int_0^{3/5} \{ f(x) + f(1) \} dx$
$= \int_0^1 f(x) dx + \frac{3}{5} f(1)$
$= \int_0^1 \frac{3}{5} x^2 dx + \left( \frac{3}{5} \right)^2$
$= \left[ \frac{x^3}{5} \right]_0^1 + \frac{9}{25}$
$= \frac{1}{5} + \frac{9}{25} = \frac{14}{25}$

$\therefore p = 25, q = 14$
$\therefore p+q = 39$

[정답] $39$
