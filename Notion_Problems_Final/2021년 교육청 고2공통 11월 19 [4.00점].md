## 문제

다음은 수열 $\{ a_n \}$ 이 모든 자연수 $n$ 에 대하여 
$\sum _{ k= 1 } ^{ n }  ( \frac{1}{k} - \frac{1}{n+1}  ) a_k = n^2$ 
을 만족시킬 때, $\sum _{k=1} ^{n} a _{k}$ 를 구하는 과정이다.

$$
\begin{array}{|l|}
\hline
T _{n} = \sum _{k=1} ^{n}   ( \frac{1}{k} - \frac{1}{n+1}  ) a_k \text{ 라 하자.} \\
(ⅰ) \ T_1 = 1 \text{ 이므로 } a_1 = ( \text{가} ) \text{ 이다.} \\
(ⅱ) \ 2 \text{ 이상의 자연수 } n \text{ 에 대하여 } \\
T_n = n^2 \text{ 에서 } T_n - T_{n-1} = 2n-1 \text{ 이고 } \\
T _{n} = \sum _{k=1} ^{n}  \frac{a _{k}}{k} - \frac{1}{( \text{나} )} \times \sum _{ k=1} ^{n } a_k \text{ 에서 } \\
T _{n} -T _{n-1} = \frac{1}{( \text{다} )} \times \sum _{ k=1} ^{n } a_k \text{ 이므로 } \\
\sum _{ k= 1 } ^{ n } a_k = (2n-1 ) \times ( ( \text{다} ) ) \text{ 이다.} \\
(ⅰ), (ⅱ) \text{에 의하여 모든 자연수 } n \text{ 에 대하여 } \\
\sum _{ k= 1 } ^{ n} a_k = (2n - 1 ) \times ( ( \text{다} ) ) \text{ 이다.} \\
\hline
\end{array}
$$

(가)에 알맞은 수를 $p$, (나), (다)에 알맞은 식을 각각 $f(n)$, $g(n)$이라 할 때, $f(2p) \times g(3p)$의 값은?

## 보기

① $190$   ② $200$   ③ $210$   ④ $220$   ⑤ $230$

## 풀이

$T _{n} = \sum _{k=1} ^{n} ( \frac{1}{k} - \frac{1}{n+1} ) a_k$ 라 하자.

(ⅰ) $T _{1} = ( 1 - \frac{1}{2} ) a_1 = 1$ 이므로 $a _{1} = 2$ 이다. 따라서 $p = 2$ 이다.

(ⅱ) $2$ 이상의 자연수 $n$ 에 대하여 $T_n = n^2$ 에서
$T _{n} - T _{n-1} = n^2 - (n-1)^2 = 2n-1$ 이다.
또한 $T _{n} = \sum _{ k= 1 } ^{ n } \frac{a_k}{k} - \frac{1}{n+1} \sum _{ k= 1 } ^{ n } a_k$ 이므로 (나)에 알맞은 식 $f(n) = n+1$ 이다.

$T _{n} -T _{n-1} = ( \sum _{ k= 1 } ^{ n } \frac{a_k}{k} - \frac{1}{n+1} \sum _{ k= 1 } ^{ n} a_k ) - ( \sum _{ k= 1 } ^{ n-1 } \frac{a_k}{k} - \frac{1}{n} \sum _{ k= 1 } ^{ n-1 } a_k )$
$= \frac{a_n}{n} - \frac{1}{n+1} \sum _{ k= 1 } ^{ n } a_k + \frac{1}{n} ( \sum _{ k= 1 } ^{ n } a_k - a_n )$
$= ( \frac{1}{n} - \frac{1}{n+1} ) \sum _{ k= 1 } ^{ n } a_k = \frac{1}{n(n+1)} \sum _{k=1} ^{n} a _{k}$

$T_n - T_{n-1} = \frac{1}{n(n+1)} \sum_{k=1}^n a_k$ 이므로 (다)에 알맞은 식 $g(n) = n(n+1)$ 이다.
$\frac{1}{n(n+1)} \times \sum _{k=1} ^{n} a _{k} = 2n-1$ 에서
$\sum _{k=1} ^{n} a _{k} = (2n-1) \times n(n+1)$ 이다.

따라서 $p = 2$, $f(n) = n+1$, $g(n) = n(n+1)$ 이므로
$f(2p) \times g(3p) = f(4) \times g(6) = (4+1) \times (6 \times 7) = 5 \times 42 = 210$ 이다.

정답은 ③ 이다.
