## 문제
두 이차함수 $f(x)$, $g(x)$는 다음 조건을 만족시킨다.

(가) $f(x)g(x) = (x^2 - 4)(x^2 - 9)$
(나) $f(\alpha) = f(\alpha + 5) = 0$인 실수 $\alpha$가 존재한다.

에서 옳은 것만을 있는 대로 고른 것은?

ㄱ. $f(2) = 0$일 때, $g(3) = 0$이다.
ㄴ. $g(2) > 0$일 때, $f(\frac{5}{2}) < g(\frac{5}{2})$이다.
ㄷ. $x$에 대한 방정식 $f(x) - g(x) = 0$이 서로 다른 두 정수 $m, n$을 근으로 가질 때, $|m + n| = 5$이다.

## 보기
① ㄱ  ② ㄴ  ③ ㄱ, ㄴ  ④ ㄱ, ㄷ  ⑤ ㄱ, ㄴ, ㄷ

## 풀이
조건 (가)에 의하여 
$f(x)g(x) = (x+2)(x-2)(x+3)(x-3)$이고 조건 (나)에 의하여 
$f(x) = 0$의 두 실근이 $\alpha, \alpha+5$이다.
즉, $f(x) = 0$의 두 실근의 차가 $5$이다. 

$$ \begin{array}{l} \text{가능한 } f(x), g(x) \text{의 조합:} \\ \text{Case 1: } \begin{cases} f(x) = a(x-2)(x+3) \\ g(x) = \frac{1}{a}(x+2)(x-3) \end{cases} \\ \text{Case 2: } \begin{cases} f(x) = a(x+2)(x-3) \\ g(x) = \frac{1}{a}(x-2)(x+3) \end{cases} \\ (\text{단, } a \neq 0 \text{는 상수}) \end{array} $$

ㄱ. $f(2) = 0$이면 Case 1에 해당하므로 
$\begin{cases} f(x) = a(x-2)(x+3) \\ g(x) = \frac{1}{a}(x+2)(x-3) \end{cases}$이다. 
따라서 $g(3) = 0$이다. (참)

ㄴ. $g(2) > 0$이면 Case 1에서 $g(2) = \frac{1}{a}(4)(-1) = -\frac{4}{a} > 0$이므로 $a < 0$이다. 
$f(\frac{5}{2}) = a(\frac{1}{2})(\frac{11}{2}) = \frac{11}{4}a < 0$
$g(\frac{5}{2}) = \frac{1}{a}(\frac{9}{2})(-\frac{1}{2}) = -\frac{9}{4a} > 0$
따라서 $f(\frac{5}{2}) < 0 < g(\frac{5}{2})$이다. (참)

ㄷ. (ⅰ) $\begin{cases} f(x) = a(x-2)(x+3) \\ g(x) = \frac{1}{a}(x+2)(x-3) \end{cases}$ ($a \neq 0$)인 경우 
$f(x) - g(x) = (a - \frac{1}{a})x^2 + (a + \frac{1}{a})x - 6(a - \frac{1}{a}) = 0$
$a - \frac{1}{a} \neq 0$ 즉, $a \neq \pm 1$이어야 서로 다른 두 근을 갖는다.
양변에 $a$를 곱하여 정리하면 $(a^2 - 1)x^2 + (a^2 + 1)x - 6(a^2 - 1) = 0$이다.
근과 계수의 관계에 의해 두 정수 근 $m, n$의 곱은 $mn = -6$이다.
$$ \begin{array}{|c|} \hline (m, n) \text{의 경우의 수} \\ \hline (-6, 1), (1, -6), (-3, 2), (2, -3), (-2, 3), (3, -2), (-1, 6), (6, -1) \\ \hline \end{array} $$
총 $8$가지 경우에 대하여 합 $m+n = \frac{-(a^2+1)}{a^2-1}$을 확인한다.

① $(m, n) = (-6, 1)$ 또는 $(1, -6)$인 경우:
$m+n = -5 = \frac{-a^2-1}{a^2-1} \implies -a^2-1 = -5a^2+5 \implies 4a^2 = 6 \implies a = \pm \sqrt{\frac{3}{2}}$
이때 정수 근 $x = -6, 1$을 갖는다.

② $(m, n) = (-3, 2)$ 또는 $(2, -3)$인 경우:
$m+n = -1 = \frac{-a^2-1}{a^2-1} \implies -a^2-1 = -a^2+1$ (만족하는 $a$ 없음)

③ $(m, n) = (-2, 3)$ 또는 $(3, -2)$인 경우:
$m+n = 1 = \frac{-a^2-1}{a^2-1} \implies a^2+1 = -a^2+1 \implies a = 0$ (조건 $a \neq 0$에 위배)

④ $(m, n) = (-1, 6)$ 또는 $(6, -1)$인 경우:
$m+n = 5 = \frac{-a^2-1}{a^2-1} \implies -a^2-1 = 5a^2-5 \implies 6a^2 = 4 \implies a = \pm \sqrt{\frac{2}{3}}$
이때 정수 근 $x = -1, 6$을 갖는다.

(ⅱ) $\begin{cases} f(x) = a(x+2)(x-3) \\ g(x) = \frac{1}{a}(x-2)(x+3) \end{cases}$ ($a \neq 0$)인 경우 
$f(x) - g(x) = (a - \frac{1}{a})x^2 - (a + \frac{1}{a})x - 6(a - \frac{1}{a}) = 0$에서 위와 동일한 방식으로 $|m+n| = 5$임을 알 수 있다.

따라서 $x$에 대한 방정식 $f(x) - g(x) = 0$이 서로 다른 두 정수 $m, n$을 근으로 가지면 $|m+n| = 5$이다. (참)

정답은 ⑤이다.
