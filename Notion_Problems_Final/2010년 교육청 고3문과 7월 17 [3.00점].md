## 문제
달걀의 신선도를 결정하는 중요한 요소 중 하나가 $HU$ (호우 유니트)값이다. 농후단백의 높이(뭉쳐있는 흰자의 높이)가 $h$ ($mm$) 이고 무게가 $w$ ($g$) 일 때, $HU$ 는 다음과 같이 계산한다.
$$HU = 100 \log (h + 7.57 - 1.7 w^{0.37})$$
$HU = 90$ 이고 무게가 $50$ $g$ 일 때 농후단백의 높이 $h$ 의 값은? (단, $1.7 \times 50^{0.37} = 7.24$, $\log 2 = 0.30$ 으로 계산한다.)

## 보기
① $6.24$   ② $6.50$   ③ $6.87$   ④ $7.13$   ⑤ $7.67$

## 풀이
$HU = 100 \log (h + 7.57 - 1.7 w^{0.37})$ 에 $HU = 90$, $w = 50$ 을 대입하면
$$90 = 100 \log (h + 7.57 - 1.7 \times 50^{0.37})$$
조건에서 $1.7 \times 50^{0.37} = 7.24$ 이므로
$$0.9 = \log (h + 7.57 - 7.24)$$
$$0.9 = \log (h + 0.33)$$
이때 $0.9 = 3 \times 0.30 = 3 \log 2 = \log 2^{3} = \log 8$ 이므로
$$\log 8 = \log (h + 0.33)$$
$$8 = h + 0.33$$
$$\therefore h = 7.67$$

정답: ⑤
