## 문제
$a$가 자연수일 때, 실수 $x$에 대한 두 조건
$p : |x| \ge a$,
$q : x(x-3) \le 0$
이 있다. $p$가 $\sim q$이기 위한 충분조건이 되도록 하는 $a$의 최솟값은?

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
[정답] ④

[해설]
두 조건 $p$, $q$에 대한 진리집합을 각각 $P, Q$라 하자.
$p \rightarrow \sim q$가 참이면 이 명제의 대우인 $q \rightarrow \sim p$도 참이므로 $Q \subset P^{c}$이어야 한다.
$P = \{ x \mid |x| \ge a \}$이므로
$P^{c} = \{ x \mid |x| < a \} = \{ x \mid -a < x < a \}$
$Q = \{ x \mid x(x-3) \le 0 \} = \{ x \mid 0 \le x \le 3 \}$
그러므로 $\{ x \mid 0 \le x \le 3 \} \subset \{ x \mid -a < x < a \}$에서
$-a < 0$이고 $a > 3$이어야 하므로 $a > 3$
따라서 자연수 $a$의 최솟값은 $4$

[다른 풀이]
두 조건 $p, q$에 대한 진리집합을 각각 $P, Q$라 하면
$P = \{ x \mid |x| \ge a \} = \{ x \mid x \le -a \text{ 또는 } x \ge a \}$,
$Q = \{ x \mid x(x-3) \le 0 \} = \{ x \mid 0 \le x \le 3 \}$
$p$가 $\sim q$이기 위한 충분조건이면 $P \subset Q^{c}$이어야 한다.
$Q^{c} = \{ x \mid x < 0 \text{ 또는 } x > 3 \}$에서
$-a < 0$이고 $a > 3$이어야 하므로 $a > 3$
따라서 자연수 $a$의 최솟값은 $4$
