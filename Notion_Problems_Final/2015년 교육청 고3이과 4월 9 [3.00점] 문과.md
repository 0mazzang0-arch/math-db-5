## 문제
$0 \le x < 2\pi$ 일 때, 방정식 $\cos 2x + 6\cos^2 \frac{x}{2} = 1$ 의 서로 다른 모든 실근의 합은?

## 보기
① $\pi$   ② $\frac{3}{2}\pi$   ③ $2\pi$   ④ $\frac{5}{2}\pi$   ⑤ $3\pi$

## 풀이
방정식 $\cos 2x + 6\cos^2 \frac{x}{2} = 1$ 에서 배각의 공식과 반각의 공식을 대입하면
$$2\cos^2 x - 1 + 6 \times \frac{1 + \cos x}{2} = 1$$
$$2\cos^2 x + 3\cos x + 1 = 0$$
$$(2\cos x + 1)(\cos x + 1) = 0$$
따라서 $\cos x = -\frac{1}{2}$ 또는 $\cos x = -1$ 이다.
$0 \le x < 2\pi$ 이므로
$x = \frac{2}{3}\pi$ 또는 $x = \frac{4}{3}\pi$ 또는 $x = \pi$
따라서 모든 실근의 합은
$\frac{2}{3}\pi + \frac{4}{3}\pi + \pi = 3\pi$

정답: ⑤
