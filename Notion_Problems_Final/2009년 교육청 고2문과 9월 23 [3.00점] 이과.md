## 문제
$\log x$ 의 정수부분이 $4$ 이고 $\log x$ 와 $\log \sqrt[3]{x}$ 의 소수부분이 같을 때, $\log x = \frac{q}{p}$ 이다. 이때, $p+q$ 의 값을 구하시오. (단, $p$, $q$ 는 서로소인 자연수이다.)

## 보기

## 풀이
$\log x = 4 + \alpha$ ($0 \le \alpha < 1$)라 하면 
$\log \sqrt[3]{x} = \frac{1}{3} \log x = \frac{4}{3} + \frac{\alpha}{3} = 1 + \frac{1}{3} + \frac{\alpha}{3}$ 

$0 \le \frac{\alpha}{3} < \frac{1}{3}$ 이므로 
$\log \sqrt[3]{x}$ 의 소수부분은 $\frac{1}{3} + \frac{\alpha}{3}$ 이다. 

$\log x$ 와 $\log \sqrt[3]{x}$ 의 소수부분이 같으므로 
$\alpha = \frac{1}{3} + \frac{\alpha}{3}$ 에서 
$\alpha = \frac{1}{2}$ 

$\therefore \log x = 4 + \frac{1}{2} = \frac{9}{2}$ 
따라서 $p = 2$, $q = 9$ 이므로 
$p + q = 11$

정답: $11$
