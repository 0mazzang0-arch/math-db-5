## 문제
다음은 두 자연수 $x$와 $y$에 대하여 $x^2 + xy + y^2$이 $10$의 배수이면 $x$와 $y$가 어떤 성질을 가짐을 증명한 것이다.

$$ \begin{array}{|l|} \hline N = x^2 + xy + y^2 \text{이라 하고 } N \text{이 } 10 \text{의 배수라 하자.} \\ N \text{이 } 10 \text{의 배수이므로 } x, y \text{는 모두 짝수이다. 만일 } x, y \text{ 중 하나만 } 10 \text{의 배수이면 } \\ N \text{은 } 10 \text{의 배수가 아니다.} \\ \text{한편, } x, y \text{가 모두 } 10 \text{의 배수가 아니면 } x^2 \text{과 } y^2 \text{의 일의 자리의 수는 } 4 \text{ 또는 } 6 \text{이다.} \\ (ⅰ) \text{ } x^2 \text{의 일의 자리의 수가 } 4 \text{이고 } y^2 \text{의 일의 자리의 수가 } 6 \text{인 경우에 } x \text{의 일의 자리의 수가 } \\ \text{될 수 있는 것은 } 2 \text{ 또는 } 8 \text{ 뿐이고, } y \text{의 일의 자리의 수가 될 수 있는 것은 } 4 \text{ 또는 } 6 \text{ 뿐이다.} \\ \text{따라서 } xy \text{의 일의 자리의 수가 될 수 있는 것은 } (가) \text{ 뿐이므로 } N \text{은 } 10 \text{의 배수가 아니다.} \\ (ⅱ) \text{ } x^2 \text{의 일의 자리의 수가 } 6 \text{이고 } y^2 \text{의 일의 자리의 수가 } 4 \text{인 경우에는 (ⅰ)의 경우와 } \\ \text{마찬가지로 } N \text{은 } 10 \text{의 배수가 아니다.} \\ (ⅲ) \text{ } x^2 \text{과 } y^2 \text{의 일의 자리의 수가 모두 } 4 \text{이거나 모두 } 6 \text{인 경우에 (ⅰ)의 경우처럼 하면 } \\ xy \text{의 일의 자리의 수가 될 수 있는 것은 } (나) \text{ 뿐이므로 } N \text{은 } 10 \text{의 배수가 아니다.} \\ \text{따라서 } x^2 + xy + y^2 \text{이 } 10 \text{의 배수이면 } (다) \text{ } 10 \text{의 배수이다.} \\ \hline \end{array} $$

위의 빈칸 $(가)$, $(나)$, $(다)$에 알맞은 것은?

## 보기
① $(가): 4 \text{ 또는 } 6, (나): 2 \text{ 또는 } 8, (다): x, y \text{ 는 모두}$
② $(가): 4 \text{ 또는 } 6, (나): 4 \text{ 또는 } 6, (다): x, y \text{ 는 모두}$
③ $(가): 2 \text{ 또는 } 8, (나): 4 \text{ 또는 } 6, (다): x, y \text{ 는 모두}$
④ $(가): 2 \text{ 또는 } 8, (나): 2 \text{ 또는 } 8, (다): x, y \text{ 중 하나만}$
⑤ $(가): 2 \text{ 또는 } 4, (나): 6 \text{ 또는 } 8, (다): x, y \text{ 중 하나만}$

## 풀이
$(ⅰ)$ $x^2$의 일의 자리의 수가 $4$이고 $y^2$의 일의 자리의 수가 $6$인 경우:
$x$의 일의 자리의 수는 $2$ 또는 $8$이고, $y$의 일의 자리의 수는 $4$ 또는 $6$이다.
$xy$의 일의 자리의 수를 구하면 다음과 같다.
$2 \times 4 = 8$
$2 \times 6 = 12$
$8 \times 4 = 32$
$8 \times 6 = 48$
따라서 $xy$의 일의 자리의 수가 될 수 있는 것은 $2$ 또는 $8$ 뿐이므로 $N$은 $10$의 배수가 아니다. 그러므로 $(가)$는 $2$ 또는 $8$이다.

$(ⅱ)$ $x^2$과 $y^2$의 일의 자리의 수가 모두 $4$인 경우:
$x, y$의 일의 자리의 수는 모두 $2$ 또는 $8$이다. 이때 $xy$의 일의 자리는 $2 \times 2 = 4$, $2 \times 8 = 16$, $8 \times 8 = 64$에서 $4$ 또는 $6$이다.
$x^2$과 $y^2$의 일의 자리의 수가 모두 $6$인 경우:
$x, y$의 일의 자리의 수는 모두 $4$ 또는 $6$이다. 이때 $xy$의 일의 자리는 $4 \times 4 = 16$, $4 \times 6 = 24$, $6 \times 6 = 36$에서 $4$ 또는 $6$이다.
따라서 $xy$의 일의 자리의 수가 될 수 있는 것은 $4$ 또는 $6$ 뿐이므로 $N$은 $10$의 배수가 아니다. 그러므로 $(나)$는 $4$ 또는 $6$이다.

따라서 $x^2 + xy + y^2$이 $10$의 배수이면 $x, y$ 중 하나만 $10$의 배수인 경우도 아니고, $x, y$가 모두 $10$의 배수가 아닌 경우도 아니므로 $x, y$는 모두 $10$의 배수이다. 그러므로 $(다)$는 $x, y$는 모두이다.

정답은 ③이다.
