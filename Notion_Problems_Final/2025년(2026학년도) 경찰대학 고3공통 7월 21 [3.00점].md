## 문제
등차수열 $\{a_n\}$ 에 대하여 $a_1 + a_2 = 5$, $a_3 + a_4 + a_5 = 30$ 일 때, $a_6 + a_7 + a_8 + a_9$ 의 값을 구하시오.

## 풀이
등차수열 $\{a_n\}$ 의 공차를 $d$ 라 하면 주어진 조건 $a_1 + a_2 = 5$ 에서
$a_1 + (a_1 + d) = 2a_1 + d = 5$ $\cdots$ ㉠
주어진 조건 $a_3 + a_4 + a_5 = 30$ 에서
$(a_1 + 2d) + (a_1 + 3d) + (a_1 + 4d) = 3a_1 + 9d = 30$
$a_1 + 3d = 10$ $\cdots$ ㉡
㉠, ㉡을 연립하면 $d = 3$, $a_1 = 1$
따라서 등차수열 $\{a_n\}$ 의 일반항은
$a_n = 1 + (n - 1) \times 3 = 3n - 2$

$\therefore a_6 + a_7 + a_8 + a_9 = 16 + 19 + 22 + 25 = 82$

정답: $82$
