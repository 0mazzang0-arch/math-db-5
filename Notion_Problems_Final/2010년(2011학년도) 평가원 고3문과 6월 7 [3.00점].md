## 문제
로그부등식 $\log_{2}(x^{2} + x - 2) < \log_{2}(-2x + 2)$의 해가 $\alpha < x < \beta$일 때, $\alpha\beta$의 값은?

## 보기
① $2$   ② $4$   ③ $6$   ④ $8$   ⑤ $10$

## 풀이
$\log_{2}(x^{2} + x - 2) < \log_{2}(-2x + 2)$에서
진수 조건에 의해
$x^{2} + x - 2 > 0$
$(x + 2)(x - 1) > 0$
$\therefore x < -2$ 또는 $x > 1$ $\cdots$ ㉠

$-2x + 2 > 0$
$\therefore x < 1$ $\cdots$ ㉡

㉠, ㉡에서
$x < -2$ $\cdots$ ㉢

주어진 로그부등식의 밑이 $2$이므로
$x^{2} + x - 2 < -2x + 2$
$x^{2} + 3x - 4 < 0$
$(x + 4)(x - 1) < 0$
$\therefore -4 < x < 1$ $\cdots$ ㉣

㉢, ㉣에서
$-4 < x < -2$
따라서 $\alpha = -4$, $\beta = -2$
$\therefore \alpha\beta = 8$

정답: ④
