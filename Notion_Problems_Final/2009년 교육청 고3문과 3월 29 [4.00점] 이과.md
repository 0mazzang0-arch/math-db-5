## 문제
그림은 함수 $f(x)=2 \left| x-\frac{1}{2} \right|$ ($0 \le x \le 1$)의 그래프이다.

> [!example] 📸 (그림/그래프)

자연수 $n$에 대하여 집합 $A_n$을 $A_n = \{ x | f^n (x)=1, 0 \le x \le 1 \}$이라 할 때, 집합 $A_n$의 원소의 개수를 $a_n$이라 하자. 예를 들어 $A_1 = \{ 0, 1 \}$, $A_2 = \{ 0, \frac{1}{2}, 1 \}$이므로 $a_1 = 2$, $a_2 = 3$이다. $\lim_{n \to \infty} \frac{a_n}{a_{n+1}}$의 값은? (단, $f^1 = f$, $f^{n+1} = f \circ f^n$ ($n=1, 2, 3, \cdots$)이다.)

## 보기
① $\frac{1}{4}$   ② $\frac{1}{2}$   ③ $\frac{2}{3}$   ④ $\frac{3}{4}$   ⑤ $1$

## 풀이
방정식 $f(x)=1$의 해는 함수 $y=f(x)$의 그래프와 직선 $y=1$의 교점의 $x$좌표와 같다.
$\therefore A_1 = \{ 0, 1 \}$ … ㉠

㉠에서 $f(f(x))=1 \iff f(x)=0$ 또는 $f(x)=1$
따라서 방정식 $f^2(x)=1$의 해는 함수 $y=f(x)$의 그래프와 두 직선 $y=0$, $y=1$의 교점의 $x$좌표와 같다.
$\therefore A_2 = \{ 0, \frac{1}{2}, 1 \}$ … ㉡

㉡에서 $f(f(f(x)))=1 \iff f(x)=0$ 또는 $f(x)=\frac{1}{2}$ 또는 $f(x)=1$
따라서 방정식 $f^3(x)=1$의 해는 함수 $y=f(x)$의 그래프와 세 직선 $y=0$, $y=\frac{1}{2}$, $y=1$의 교점의 $x$좌표와 같다.
$\therefore A_3 = \{ 0, \frac{1}{4}, \frac{1}{2}, \frac{3}{4}, 1 \}$

이와 같은 방법으로 $a_1 = 2$, $a_{n+1} = 2a_n - 1$ ($n=1, 2, 3, \cdots$)이 성립함을 알 수 있다.
이때, $(a_{n+1} - 1) = 2(a_n - 1)$이므로
$a_n - 1 = (a_1 - 1) \times 2^{n-1} = 1 \times 2^{n-1}$
$\therefore a_n = 2^{n-1} + 1$ ($n=1, 2, 3, \cdots$)

$\therefore \lim_{n \to \infty} \frac{a_n}{a_{n+1}} = \lim_{n \to \infty} \frac{2^{n-1} + 1}{2^n + 1} = \frac{1}{2}$

정답: ②
