## 문제
다음 보기 중 옳은 것만을 있는 대로 고른 것은?

$ㄱ$. $\left[ \frac{[3x]}{3} \right] = [x]$
$ㄴ$. $[x] + [x+2y] = [2x+2y]$
$ㄷ$. $[x] + \left[ x+\frac{1}{3} \right] + \left[ x+\frac{2}{3} \right] = [3x]$

## 보기
① $ㄱ$   ② $ㄷ$   ③ $ㄱ, ㄴ$   ④ $ㄱ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
$x=n+\alpha$, $y=m+\beta$ (단, $0 \le \alpha < 1$, $0 \le \beta < 1$)라고 하면

$ㄱ$. $[3x] = [3n+3\alpha] = 3n+[3\alpha]$ ($0 \le 3\alpha < 3$)이므로
$\left[ \frac{[3x]}{3} \right] = \left[ n + \frac{3\alpha}{3} \right] = n$, $[x]=n$ (참)

$ㄴ$. $[x]=n$, $[x+2y] = [n+2m+\alpha+2\beta] = n+2m+[\alpha+2\beta]$
$\therefore [x]+[x+2y] = 2n+2m+[\alpha+2\beta]$
$[2x+2y] = [2n+2m+2\alpha+2\beta] = 2n+2m+[2\alpha+2\beta]$
$\frac{1}{2} \le \alpha < 1$ 인 경우에 성립하지 않는다. (거짓)

$ㄷ$. $[x] + \left[ x+\frac{1}{3} \right] + \left[ x+\frac{2}{3} \right] = [3x]$
($i$) $0 \le \alpha < \frac{1}{3}$ 인 경우, 좌변 $= 3n$, 우변 $= 3n$ (성립)
($ii$) $\frac{1}{3} \le \alpha < \frac{2}{3}$ 인 경우, 좌변 $= 3n+1$, 우변 $= 3n+1$ (성립)
($iii$) $\frac{2}{3} \le \alpha < 1$ 인 경우, 좌변 $= 3n+2$, 우변 $= 3n+2$ (성립) (참)
