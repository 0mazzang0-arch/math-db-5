## 문제

두 이차함수 $f(x)$, $g(x)$에 대하여 실수 전체의 집합에서 정의된 함수 $h(x)$가 $0 \le x < 4$에서

$$ h(x) = \left\{ \begin{array}{ll} x & (0 \le x < 2) \\ f(x) & (2 \le x < 3) \\ g(x) & (3 \le x < 4) \end{array} \right. $$

이고, 다음 조건을 만족시킨다.

(가) 모든 실수 $x$에 대하여 $h(x) = h(x - 4) + k$ ($k$는 상수)이다.  
(나) 함수 $h(x)$는 실수 전체의 집합에서 미분가능하다.  
(다) $\int_{0}^{4} h(x) dx = 6$

$h(\frac{13}{2}) = \frac{q}{p}$일 때, $p + q$의 값을 구하시오. (단, $p$와 $q$는 서로소인 자연수이다.)

## 풀이

[정답] 
$21$

[해설]
함수 $h(x)$는 실수 전체의 집합에서 미분가능하므로 $x = 2$, $x = 3$, $x = 4$에서 연속이고 미분가능해야 한다.

그림과 같은 모양이어야 한다. 

$f(x) = a(x - \frac{5}{2})^{2} + b$라 하면 
$f'(x) = 2a(x - \frac{5}{2})$ 
$f'(2) = 2a(2 - \frac{5}{2}) = -a = 1$ ($\because 0 \le x < 2$에서 $h'(x) = 1$)
$\therefore a = -1$

$f(2) = a(2 - \frac{5}{2})^{2} + b = 2$ 
$-\frac{1}{4} + b = 2$, $b = \frac{9}{4}$ 

즉, 
$f(x) = -(x - \frac{5}{2})^{2} + \frac{9}{4}$
$x = 3$에서 미분가능 조건에 의해 $g(3) = f(3) = 2$, $g'(3) = f'(3) = -1$
$x = 4$에서 미분가능 조건에 의해 $h(4) = h(0) + k = k$, $h'(4) = h'(0) = 1$
따라서 $g(4) = k$, $g'(4) = 1$

위 조건들을 만족하는 $g(x)$를 구하면
$g(x) = (x - \frac{7}{2})^{2} + \frac{7}{4}$
그리고 $k = g(4) = (4 - \frac{7}{2})^2 + \frac{7}{4} = 2$이다. 

$h(\frac{13}{2}) = h(\frac{13}{2} - 4) + k = h(\frac{5}{2}) + 2$
$h(\frac{5}{2}) = f(\frac{5}{2}) = \frac{9}{4}$이므로
$h(\frac{13}{2}) = \frac{9}{4} + 2 = \frac{17}{4}$

따라서 $p = 4$, $q = 17$이므로
$\therefore p + q = 4 + 17 = 21$
