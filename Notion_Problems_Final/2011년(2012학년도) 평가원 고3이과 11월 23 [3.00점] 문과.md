## 문제
방정식 $3 \cos 2x + 17 \cos x = 0$을 만족시키는 $x$에 대하여 $\tan^2 x$의 값을 구하시오.

## 보기

## 풀이
$3 \cos 2x + 17 \cos x = 0$
$3(2 \cos^2 x - 1) + 17 \cos x = 0$
$6 \cos^2 x + 17 \cos x - 3 = 0$
$(6 \cos x - 1)(\cos x + 3) = 0$

$\therefore \cos x = \frac{1}{6}$ ($\because -1 \le \cos x \le 1$)

$\sin^2 x = 1 - \cos^2 x = 1 - \frac{1}{36} = \frac{35}{36}$

$\therefore \tan^2 x = \frac{\sin^2 x}{\cos^2 x} = \frac{\frac{35}{36}}{\frac{1}{36}} = 35$

정답: $35$
