## 문제
다음 부등식의 해는?
$$(\log_{2} x)^{2} + \log_{2} x - 2 \ge 0$$

## 보기
① $\frac{1}{4} \le x \le 2$   ② $\frac{1}{2} \le x \le 4$   ③ $2 \le x \le 4$   ④ $0 < x \le \frac{1}{4}$ 또는 $x \ge 2$   ⑤ $0 < x \le \frac{1}{2}$ 또는 $x \ge 4$

## 풀이
진수는 양수이므로
$$x > 0 \cdots \cdots ㉠$$
$$(\log_{2} x)^{2} + \log_{2} x - 2 \ge 0$$에서
$$(\log_{2} x + 2)(\log_{2} x - 1) \ge 0$$
$$\leftrightarrow \log_{2} x \le -2 \text{ 또는 } \log_{2} x \ge 1$$
$$\log_{2} x \le \log_{2} \frac{1}{4} \text{ 또는 } \log_{2} x \ge \log_{2} 2$$
$$\therefore x \le \frac{1}{4} \text{ 또는 } x \ge 2 \cdots \cdots ㉡$$
$㉠$, $㉡$에서
$$0 < x \le \frac{1}{4} \text{ 또는 } x \ge 2$$
따라서 정답은 ④입니다.
