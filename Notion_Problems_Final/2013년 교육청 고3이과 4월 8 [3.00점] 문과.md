## 문제
삼각방정식 $3\sin^{2} x+4\cos^{2} \frac{x}{2} - 4 = 0$의 서로 다른 실근의 개수는? (단, $0 < x < 4\pi$)



> [!example] 📸 (그림/그래프)

## 보기
① $4$   ② $5$   ③ $6$   ④ $7$   ⑤ $8$

## 풀이
$\cos^{2} \frac{x}{2} = \frac{1 + \cos x}{2}$이므로 주어진 식에 대입하면
$$3(1 - \cos^{2} x) + 2(1 + \cos x) - 4 = 0$$
$$3 - 3\cos^{2} x + 2 + 2\cos x - 4 = 0$$
$$3\cos^{2} x - 2\cos x - 1 = 0$$
$$(3\cos x + 1)(\cos x - 1) = 0$$
따라서 $\cos x = -\frac{1}{3}$ 또는 $\cos x = 1$이다.

$0 < x < 4\pi$에서 $y = \cos x$의 그래프와 두 직선 $y = -\frac{1}{3}$, $y = 1$의 교점의 개수를 확인하면 다음과 같다.

1. $y = \cos x$와 $y = -\frac{1}{3}$의 교점:
$0 < x < 2\pi$에서 $2$개, $2\pi < x < 4\pi$에서 $2$개이므로 총 $4$개이다.

2. $y = \cos x$와 $y = 1$의 교점:
$0 < x < 4\pi$ 범위에서 $x = 2\pi$일 때 $1$개이다. (단, 양 끝점 $0$과 $4\pi$는 포함되지 않는다.)

따라서 서로 다른 실근의 개수는 $4 + 1 = 5$이다.

정답: ②
