## 문제
양수 $a$와 두 실수 $b$, $c$에 대하여 함수 
$f(x) = (ax^{2} + bx + c)e^{x}$
은 다음 조건을 만족시킨다.

(가) $f(x)$는 $x = -\sqrt{3}$과 $x = \sqrt{3}$에서 극값을 갖는다.
(나) $0 \le x_{1} < x_{2}$인 임의의 두 실수 $x_{1}$, $x_{2}$에 대하여 $f(x_{2}) - f(x_{1}) + x_{2} - x_{1} \ge 0$이다.

세 수 $a$, $b$, $c$의 곱 $abc$의 최댓값을 $\frac{k}{e^{3}}$라 할 때, $60k$의 값을 구하시오.

## 보기

## 풀이
$f(x) = (ax^{2} + bx + c)e^{x}$에서 

$f'(x) = (2ax + b)e^{x} + (ax^{2} + bx + c)e^{x} = \{ax^{2} + (2a + b)x + b + c\}e^{x}$ 

$f'(x) = 0$에서 
$ax^{2} + (2a + b)x + b + c = 0$ $\cdots$ ㉠ 

조건 (가)에서 방정식 ㉠의 두 근이 $-\sqrt{3}$, $\sqrt{3}$이므로 근과 계수의 관계에 의하여 

$2a + b = 0$, $\frac{b + c}{a} = -3$ 

$\therefore b = -2a$, $c = -a$ 

조건 (나)에서 $0 \le x_{1} < x_{2}$일 때, $f(x_{2}) - f(x_{1}) + x_{2} - x_{1} \ge 0$이므로 

$\frac{f(x_{2}) - f(x_{1})}{x_{2} - x_{1}} + 1 \ge 0$ $\cdots$ ㉡ 

함수 $f(x)$는 닫힌구간 $[x_{1}, x_{2}]$에선 연속이고, 열린구간 $(x_{1}, x_{2})$에서 미분가능하므로 평균값 정리에 의하여 

$\frac{f(x_{2}) - f(x_{1})}{x_{2} - x_{1}} = f'(c)$ 

인 $c$가 $x_{1}$과 $x_{2}$ 사이에 존재한다. 

$0 \le x_{1} < x_{2}$인 임의의 두 실수 $x_{1}$, $x_{2}$에 대하여 부등식 ㉡이 성립하므로 임의의 양수 $c$에 대하여 

$f'(c) + 1 \ge 0$ 
이 성립한다. 

$f'(x) = (ax^{2} - 3a)e^{x}$이므로 
$x > 0$일 때, 부등식 $(ax^{2} - 3a)e^{x} + 1 \ge 0$이 항상 성립한다. 
$a > 0$이므로 

$(x^{2} - 3)e^{x} \ge -\frac{1}{a}$ $\cdots$ ㉢ 

이고, $g(x) = (x^{2} - 3)e^{x}$이라 하면 

$g'(x) = 2xe^{x} + (x^{2} - 3)e^{x} = (x^{2} + 2x - 3)e^{x} = (x + 3)(x - 1)e^{x}$ 

$g'(x) = 0$에서 $x = -3$ 또는 $x = 1$이므로 

$x > 0$일 때, $g(x)$는 $x = 1$에서 최솟값을 갖는다. 
$g(1) = -2e$이므로 부등식 ㉢이 $x > 0$에서 항상 성립하려면 $-2e \ge -\frac{1}{a}$이다. 
즉, $a \le \frac{1}{2e}$이므로 

$abc = a \times (-2a) \times (-a) = 2a^{3} \le 2 \times \left(\frac{1}{2e}\right)^{3} = \frac{1}{4e^{3}}$ 

따라서 $abc$의 최댓값은 $\frac{1}{4e^{3}}$이므로 $k = \frac{1}{4}$ 

$\therefore 60k = 60 \times \frac{1}{4} = 15$
