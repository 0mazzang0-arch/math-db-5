## 문제
공차가 $0$이 아닌 등차수열 $\{ a_{n} \}$이 다음 조건을 만족시킨다.

$$
\begin{array}{|l|}
\hline
\text{(가) } \sum_{n=1}^{5} a_{n} = 2 \left| \sum_{n=1}^{10} a_{n} \right| \\
\text{(나) } a_{3} a_{6} > 0 \\
\hline
\end{array}
$$

$\frac{a_{21}}{a_{1}}$의 값은?

## 보기
① $-5$   ② $-\frac{17}{4}$   ③ $-\frac{7}{2}$   ④ $-\frac{11}{4}$   ⑤ $-2$

## 풀이
등차수열 $\{ a_{n} \}$의 공차를 $d$ ($d \neq 0$)라 하면 (가)에 의하여 
$\frac{5 ( 2a_{1} + 4d )}{2} = 2 \left| \frac{10 ( 2a_{1} + 9d )}{2} \right|$
$a_{1} + 2d = 2 \left| 2a_{1} + 9d \right|$

(ⅰ) $a_{1} + 2d = 4a_{1} + 18d$ 일 때, 
$a_{1} = -\frac{16}{3} d$
$a_{3} a_{6} = \left( -\frac{10}{3} d \right) \times \left( -\frac{d}{3} \right) = \frac{10}{9} d^{2} > 0$ 이므로 (나)의 조건을 만족시킨다. 

(ⅱ) $a_{1} + 2d = -4a_{1} - 18d$ 일 때, 
$a_{1} = -4d$
$a_{3} a_{6} = ( -2d ) \times d = -2d^{2} < 0$ 이므로 (나)의 조건에 모순이다. 

따라서 
$\frac{a_{21}}{a_{1}} = \frac{-\frac{16}{3} d + 20d}{-\frac{16}{3} d} = -\frac{11}{4}$
