## 문제
부등식 $[x]^{3}-6[x]^{2}+11[x]-6 \geq 0$ 을 만족시키는 모든 실수 $x$ 의 집합은? (단, $[x]$ 는 $x$ 보다 크지 않은 최대의 정수이다.)

## 보기
① $\{x | x \geq 1\}$   ② $\{x | x \geq 3\}$   ③ $\{x | 1 \leq x \leq 3\}$   ④ $\{x | 1 \leq x < 4\}$   ⑤ $\{x | 1 \leq x < 2 \text{ 또는 } x \geq 3\}$

## 풀이
$[x] = n$ 이라 두면 (단, $n$ 은 정수)
준식은 다음과 같다.
$n^{3} - 6n^{2} + 11n - 6 \geq 0$
$\iff (n - 1)(n - 2)(n - 3) \geq 0$
$\therefore 1 \leq n \leq 2$ 또는 $n \geq 3$
이때 $n$ 은 정수이므로 $n = 1, 2, 3, \dots$ 이다.
그러므로 $[x] = 1, 2, 3, \dots$ 이며, 이를 $x$ 의 범위로 나타내면 다음과 같다.
$1 \leq x < 2$ 또는 $2 \leq x < 3$ 또는 $3 \leq x < 4 \dots$
따라서 $x \geq 1$ 이다.

정답: ①
