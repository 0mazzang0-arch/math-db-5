## 문제
첫째항이 $1$인 수열 $\{ a_{n} \}$에 대하여 $S_{n} = \sum_{k=1}^{n} a_{k}$라 할 때, 
$$ a_{n+1} = ( 2^{n} - 1 ) ( S_{n} + 1 ) ( n \geq 1 ) \dots (*) $$
이 성립한다. 다음은 일반항 $a_{n}$을 구하는 과정이다.

$$ \begin{array}{|l|} \hline \text{식 } (*) \text{의 양변에 } S_{n} \text{을 더하여 정리하면} \\ S_{n+1} + 1 = 2^{n} ( S_{n} + 1 ) \text{이다.} \\ b_{n} = \log_{2} ( S_{n} + 1 ) \text{이라 하면 } b_{1} = 1 \text{이고} \\ b_{n+1} = \text{(가)} + b_{n} \text{이다.} \\ \text{수열 } \{ b_{n} \} \text{의 일반항을 구하면} \\ b_{n} = \frac{n^{2} - n + 2}{2} ( n \geq 1 ) \text{이므로} \\ S_{n} = 2^{\frac{n^{2} - n + 2}{2}} - 1 ( n \geq 1 ) \text{이다.} \\ \text{그러므로 } a_{1} = 1 \text{이고 } n \geq 2 \text{일 때} \\ a_{n} = S_{n} - S_{n-1} \\ = 2^{\frac{n^{2} - n + 2}{2}} - 2^{\text{(나)}} \\ = 2^{\text{(나)}} \times ( 2^{n-1} - 1 ) \text{이다.} \\ \hline \end{array} $$

위의 (가)와 (나)에 알맞은 식을 각각 $f(n)$, $g(n)$이라 할 때, $f(12) - g(5)$의 값은?

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
[정답] ⑤

식 $(*)$의 양변에 $S_{n}$을 더하여 정리하면 
$$ S_{n+1} + 1 = 2^{n} ( S_{n} + 1 ) $$
이다. $b_{n} = \log_{2} ( S_{n} + 1 )$이라 하면 $b_{1} = 1$이고 
$$ \log_{2} ( S_{n+1} + 1 ) = \log_{2} \{ 2^{n} ( S_{n} + 1 ) \} $$
$$ \log_{2} ( S_{n+1} + 1 ) = n + \log_{2} ( S_{n} + 1 ) $$
$$ b_{n+1} = n + b_{n} $$
이다. 수열 $\{ b_{n} \}$의 일반항을 구하면 
$$ b_{n} = 1 + \sum_{k=1}^{n-1} k = 1 + \frac{n(n-1)}{2} = \frac{n^{2} - n + 2}{2} ( n \geq 2 ) $$
$b_{1} = 1$에서 
$$ b_{n} = \frac{n^{2} - n + 2}{2} ( n \geq 1 ) $$
이므로 
$$ S_{n} = 2^{\frac{n^{2} - n + 2}{2}} - 1 $$
이다. 그러므로 $a_{1} = 1$이고, $n \geq 2$일 때, 
$$ a_{n} = S_{n} - S_{n-1} $$
$$ = ( 2^{\frac{n^{2} - n + 2}{2}} - 1 ) - ( 2^{\frac{(n-1)^{2} - (n-1) + 2}{2}} - 1 ) $$
$$ = 2^{\frac{n^{2} - n + 2}{2}} - 2^{\frac{n^{2} - 3n + 4}{2}} $$
$$ = 2^{\frac{n^{2} - 3n + 4}{2}} \times ( 2^{n-1} - 1 ) $$
이다.
따라서 $f(n) = n$, $g(n) = \frac{n^{2} - 3n + 4}{2}$이므로 
$$ f(12) - g(5) = 12 - \frac{5^{2} - 15 + 4}{2} = 12 - 7 = 5 $$
