## 문제
$(1- \sin 2002^\circ ) (1+ \sin 2002^\circ )$ 를 간단히 하면?

## 보기
① $\cos 2^\circ$   ② $\cos^{2} 2^\circ$   ③ $\cos 22^\circ$   ④ $\cos^{2} 22^\circ$   ⑤ $-\cos^{2} 22^\circ$

## 풀이
$$(1- \sin 2002^\circ ) (1+ \sin 2002^\circ ) = 1- \sin ^{2} 2002^\circ$$
$$= \cos ^{2} 2002^\circ \cdots ㉠$$

$$\cos 2002^\circ = \cos ( 360^\circ \times 5 + 202^\circ ) = \cos 202^\circ$$
$$= \cos ( 180^\circ + 22^\circ ) = - \cos 22^\circ \cdots ㉡$$

$㉠$, $㉡$에서
$$(1- \sin 2002^\circ ) (1+ \sin 2002^\circ ) = (- \cos 22^\circ )^{2} = \cos ^{2} 22^\circ$$

따라서 정답은 ④입니다.
