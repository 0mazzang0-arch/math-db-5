## 문제
두 집합 $A = \{1, 2, 3, 4, 5\}$, $B = \{3, 4, 5\}$에 대하여 $A \cap X = X$, $(A - B) \cup X = X$를 만족하고, 모든 원소의 합이 $3$의 배수인 집합 $X$의 개수는?

## 보기
① $4$   ② $5$   ③ $6$   ④ $7$   ⑤ $8$

## 풀이
$A \cap X = X$, $(A - B) \cup X = X$에서 $(A - B) \subseteq X \subseteq A$이다.
$A - B = \{1, 2\}$이므로, $\{1, 2\} \subseteq X \subseteq \{1, 2, 3, 4, 5\}$를 만족해야 한다.

집합 $X$는 $\{1, 2\}$를 반드시 원소로 가져야 하며, 추가할 수 있는 원소는 $\{3, 4, 5\}$의 부분집합이다.
이때 $1 + 2 = 3$이므로, 추가되는 원소들의 합이 $3$의 배수(또는 $0$)가 되어야 전체 원소의 합이 $3$의 배수가 된다.

조건을 만족하는 집합 $X$는 다음과 같다.
1. 추가 원소가 없을 때: $X = \{1, 2\}$ (합: $3$)
2. 추가 원소의 합이 $3$일 때: $X = \{1, 2, 3\}$ (합: $6$)
3. 추가 원소의 합이 $9$일 때: $X = \{1, 2, 4, 5\}$ (합: $12$)
4. 추가 원소의 합이 $12$일 때: $X = \{1, 2, 3, 4, 5\}$ (합: $15$)

따라서 집합 $X$의 개수는 $4$개이다.

정답: ①
