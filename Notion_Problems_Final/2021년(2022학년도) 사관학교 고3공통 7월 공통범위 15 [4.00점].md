## 문제
다음 조건을 만족시키는 모든 수열 $\{ a_{n} \}$에 대하여 $a_{1}$의 최솟값을 $m$이라 하자.

(가) 수열 $\{ a_{n} \}$의 모든 항은 정수이다.
(나) 모든 자연수 $n$에 대하여 $a_{2n} = a_{3} \times a_{n} + 1$, $a_{2n+1} = 2a_{n} - a_{2}$

$a_{1} = m$인 수열 $\{ a_{n} \}$에 대하여 $a_{9}$의 값은?

## 보기
① $-53$   ② $-51$   ③ $-49$   ④ $-47$   ⑤ $-45$

## 풀이
조건 (나)에 $n=1$을 대입하면 다음과 같은 관계식을 얻을 수 있다.
$$ \begin{array}{l} a_{2} = a_{3} a_{1} + 1 \cdots ㉠ \\ a_{3} = 2a_{1} - a_{2} \cdots ㉡ \end{array} $$
㉡을 ㉠에 대입하면
$$ a_{2} = a_{1} ( 2a_{1} - a_{2} ) + 1 $$
$$ a_{2} = 2a_{1}^{2} - a_{1} a_{2} + 1 $$
$$ a_{2} ( a_{1} + 1 ) = 2a_{1}^{2} + 1 $$
$$ \therefore a_{2} = \frac{2a_{1}^{2} + 1}{a_{1} + 1} = \frac{2(a_{1}^{2} - 1) + 3}{a_{1} + 1} = 2 ( a_{1} - 1 ) + \frac{3}{a_{1} + 1} $$

모든 항이 정수이므로 $a_{2}$가 정수가 되기 위해서는 $a_{1} + 1$이 $3$의 약수가 되어야 한다.
$$ a_{1} + 1 = -3, -1, 1, 3 $$
$$ a_{1} = -4, -2, 0, 2 $$
따라서 $a_{1}$의 최솟값 $m$은 $m = -4$이다.

$a_{1} = -4$일 때의 $a_{2}, a_{3}$의 값을 구하면
$$ \begin{array}{l} a_{2} = 2 ( -4 - 1 ) + \frac{3}{-4 + 1} = -10 - 1 = -11 \\ a_{3} = 2 ( -4 ) - ( -11 ) = -8 + 11 = 3 \end{array} $$

이를 이용하여 $a_{9}$의 값을 구하면
$$ \begin{array}{l} a_{4} = a_{2 \times 2} = a_{3} a_{2} + 1 = 3 \times ( -11 ) + 1 = -32 \\ a_{9} = a_{2 \times 4 + 1} = 2a_{4} - a_{2} \\ a_{9} = 2 \times ( -32 ) - ( -11 ) = -64 + 11 = -53 \end{array} $$

정답은 ①이다.
