## 문제
열린구간 $(0, 5)$에서 미분가능한 두 함수 $f(x)$, $g(x)$의 그래프가 그림과 같다. 합성함수 $h(x) = (f \circ g)(x)$에 대하여 <보기>에서 옳은 것만을 있는 대로 고른 것은?

ㄱ. $h(3) = 4$
ㄴ. $h'(2) \ge 0$
ㄷ. 함수 $h(x)$는 구간 $(3, 4)$에서 감소한다.

> [!example] 📸 (그림/그래프)

## 보기
① ㄱ  ② ㄴ  ③ ㄱ, ㄴ  ④ ㄱ, ㄷ  ⑤ ㄴ, ㄷ

## 풀이
ㄱ. $h(3) = f(g(3)) = f(1) = 5$
ㄴ. $h'(2) = (f \circ g)'(2) = f'(g(2))g'(2)$
$2 < g(2) < 3$이고, $2 < x < 3$에서 $f'(x) \le 0$이므로 $f'(g(2)) \le 0$이다.
또한 $g'(2) \le 0$이므로 $h'(2) = f'(g(2))g'(2) \ge 0$이다.
ㄷ. $h'(x) = (f \circ g)'(x) = f'(g(x))g'(x)$에 대하여 구간 $(3, 4)$에서 $0 < g(x) < 1$이므로 $f'(g(x)) > 0$이고 $g'(x) < 0$이다.
$h'(x) = f'(g(x))g'(x) < 0$이므로 함수 $h(x)$는 구간 $(3, 4)$에서 감소한다.

따라서 옳은 것은 ㄴ, ㄷ이다.
