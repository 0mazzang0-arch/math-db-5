## 문제
두 이차정사각행렬 $A$, $B$가
$$A^2 + BA = E$$, $$BA + B^2 = A - E$$
를 만족시킬 때, <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $E$는 단위행렬이다.)

$$ \begin{array}{|l|} \hline \text{<보 기>} \\ \text{ㄱ. } A \text{의 역행렬이 존재한다.} \\ \text{ㄴ. } (AB)^2 = A^2 B^2 \\ \text{ㄷ. } A^3 = 2E \\ \hline \end{array} $$

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
ㄱ. $(A+B)A = E$이므로 $A$의 역행렬은 $A^{-1} = A+B$로 존재한다. (참)

ㄴ. $A(A+B) = (A+B)A = E$이므로
$A^2 + AB = A^2 + BA$
$AB = BA$
따라서 $(AB)^2 = ABAB = AABB = A^2 B^2$이다. (참)

ㄷ. $BA + B^2 = A - E$에서 $B(A+B) = A - E$이다.
ㄱ에서 $(A+B)^{-1} = A$이므로
$B = (A-E)A = A^2 - A$
이를 $A^2 + BA = E$에 대입하면
$A^2 + (A^2 - A)A = E$
$A^2 + A^3 - A^2 = E$
$A^3 = E$
따라서 $A^3 = 2E$는 거짓이다. (거짓)

따라서 옳은 것은 ㄱ, ㄴ이다.
