## 문제
수열 $\{a_n\}$은 $a_1 = 1$이고 
$a_{n+1} = \sum_{k=1}^{n} 2^{n-k} a_k$ ($n \ge 1$)
을 만족시킨다. 다음은 일반항 $a_n$을 구하는 과정이다.

$$ \begin{array}{|l|} \hline \text{주어진 식으로부터 } a_2 = \text{(가) 이다.} \\ \text{자연수 } n \text{에 대하여} \\ a_{n+2} = \sum_{k=1}^{n+1} 2^{n+1-k} a_k \\ = \sum_{k=1}^{n} 2^{n+1-k} a_k + a_{n+1} \\ = \text{(나)} \sum_{k=1}^{n} 2^{n-k} a_k + a_{n+1} \\ = \text{(다)} a_{n+1} \text{이다.} \\ \text{따라서 } a_1 = 1 \text{이고, } n \ge 2 \text{일 때, } a_n = (\text{다})^{n-2} \text{이다.} \\ \hline \end{array} $$

위의 (가), (나), (다)에 알맞은 수를 각각 $p$, $q$, $r$라 할 때, $p+q+r$의 값은?

## 보기
① $3$   ② $4$   ③ $5$   ④ $6$   ⑤ $7$

## 풀이
$a_1 = 1$이고 $a_{n+1} = \sum_{k=1}^{n} 2^{n-k} a_k$이다.
먼저 $n=1$일 때
$a_2 = \sum_{k=1}^{1} 2^{1-k} a_k = 2^{0} a_1 = a_1 = 1$
그러므로 (가) $p = 1$이다.

자연수 $n$에 대하여
$a_{n+2} = \sum_{k=1}^{n+1} 2^{n+1-k} a_k$
$= \sum_{k=1}^{n} 2^{n+1-k} a_k + a_{n+1}$
$= 2 \sum_{k=1}^{n} 2^{n-k} a_k + a_{n+1}$
($\because 2^{n+1-k} = 2 \times 2^{n-k}$)
따라서 (나) $q = 2$이다.

주어진 조건인 $a_{n+1} = \sum_{k=1}^{n} 2^{n-k} a_k$을 대입하면
$2 \sum_{k=1}^{n} 2^{n-k} a_k + a_{n+1}$
$= 2a_{n+1} + a_{n+1} = 3a_{n+1}$
그러므로 (다) $r = 3$이 된다.

따라서 $p+q+r = 1+2+3 = 6$이다.

정답은 ④이다.
