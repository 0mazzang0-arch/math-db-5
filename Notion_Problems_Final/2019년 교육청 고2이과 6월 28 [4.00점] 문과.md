## 문제
$100$ 이하의 자연수 $k$ 에 대하여 $2 \le \log_{n} k < 3$ 을 만족시키는 자연수 $n$ 의 개수를 $f(k)$ 라 하자. 예를 들어 $f(30) = 2$ 이다. $f(k) = 4$ 가 되도록 하는 $k$ 의 최댓값을 구하시오.

## 풀이
$2 \le \log_{n} k < 3$ 에서 $n^2 \le k < n^3$ 이고, 로그의 밑 조건에 의해 $n > 1$ 이다.
$1$ 보다 큰 자연수 $n$ 에 대하여 $n^2 \le k < n^3$ 을 만족시키는 $100$ 이하의 자연수 $k$ 를 구하면 다음과 같다.

(ⅰ) $n=2$ 일 때, $4 \le k < 8$ 에서 $k=4, 5, 6, 7$
(ⅱ) $n=3$ 일 때, $9 \le k < 27$ 에서 $k=9, 10, \dots, 26$
(ⅲ) $n=4$ 일 때, $16 \le k < 64$ 에서 $k=16, 17, \dots, 63$
(ⅳ) $n=5$ 일 때, $25 \le k < 125$ 에서 $k=25, 26, \dots, 100$
(ⅴ) $n=6$ 일 때, $36 \le k < 216$ 에서 $k=36, 37, \dots, 100$
(ⅵ) $n=7$ 일 때, $49 \le k < 343$ 에서 $k=49, 50, \dots, 100$
(ⅶ) $n=8$ 일 때, $64 \le k < 512$ 에서 $k=64, 65, \dots, 100$
(ⅷ) $n=9$ 일 때, $81 \le k < 729$ 에서 $k=81, 82, \dots, 100$
(ⅸ) $n=10$ 일 때, $100 \le k < 1000$ 에서 $k=100$

그러므로 $k$ 의 값에 따라 조건을 만족시키는 $f(k)$ 를 구하면 다음과 같다.

(ⅰ) $k=1, 2, 3$ 일 때, $f(k) = 0$
(ⅱ) $k=4, 5, 6, 7$ 일 때, $f(k) = 1$
(ⅲ) $k=8$ 일 때, $f(k) = 0$
(ⅳ) $k=9, 10, \dots, 15$ 일 때, $f(k) = 1$
(ⅴ) $k=16, 17, \dots, 24$ 일 때, $f(k) = 2$
(ⅵ) $k=25, 26$ 일 때, $f(k) = 3$
(ⅶ) $k=27, 28, \dots, 35$ 일 때, $f(k) = 2$
(ⅷ) $k=36, 37, \dots, 48$ 일 때, $f(k) = 3$
(ⅸ) $k=49, 50, \dots, 80$ 일 때, $f(k) = 4$
(ⅹ) $k=81, 82, \dots, 99$ 일 때, $f(k) = 5$
(ⅺ) $k=100$ 일 때, $f(k) = 6$

따라서 $f(k) = 4$ 가 되도록 하는 $k$ 의 최댓값은 $80$ 이다.

정답: $80$
