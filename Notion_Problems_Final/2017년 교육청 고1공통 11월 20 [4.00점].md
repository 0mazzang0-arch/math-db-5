## 문제
최고차항의 계수가 $1$인 이차식 $f(x)$를 $x-1$로 나누었을 때의 몫을 $Q_1(x)$라 하고, $f(x)$를 $x-2$로 나누었을 때의 몫을 $Q_2(x)$라 하면 $Q_1(x)$, $Q_2(x)$는 다음 조건을 만족시킨다.

(가) $Q_2(1) = f(2)$
(나) $Q_1(1) + Q_2(1) = 6$

$f(3)$의 값은?

## 보기
① $7$   ② $8$   ③ $9$   ④ $10$   ⑤ $11$

## 풀이
$f(x)$를 $x-1$로 나누었을 때의 몫을 $Q_1(x)$, 나머지를 $R_1$이라 하면
$f(x) = (x-1)Q_1(x) + R_1$ $\cdots$ ㉠

$f(x)$를 $x-2$로 나누었을 때의 몫을 $Q_2(x)$, 나머지를 $R_2$라 하면
$f(x) = (x-2)Q_2(x) + R_2$ $\cdots$ ㉡

㉡의 양변에 $x=2$를 대입하면 조건 (가)에서
$R_2 = f(2) = Q_2(1)$

$f(x) = (x-2)Q_2(x) + Q_2(1)$의 양변에 $x=1$을 대입하면
$f(1) = -Q_2(1) + Q_2(1) = 0$

㉠의 양변에 $x=1$을 대입하면 $f(1) = R_1 = 0$

$f(x)$는 최고차항의 계수가 $1$인 이차식이므로 $Q_1(x) = x+a$라 하면
$f(x) = (x-1)(x+a)$

$Q_1(1) = 1+a$, $f(2) = 2+a = Q_2(1)$이므로 조건 (나)에서
$Q_1(1) + Q_2(1) = (1+a) + (2+a) = 2a+3 = 6$

$\therefore a = \frac{3}{2}$

따라서 $f(x) = (x-1)\left(x + \frac{3}{2}\right)$이므로
$f(3) = (3-1)\left(3 + \frac{3}{2}\right) = 2 \times \frac{9}{2} = 9$
