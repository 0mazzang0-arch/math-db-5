## 문제
방정식 $a+b+c=9$를 만족시키는 음이 아닌 정수 $a$, $b$, $c$의 모든 순서쌍 $(a, b, c)$ 중에서 임의로 한 개를 선택할 때, 선택한 순서쌍 $(a, b, c)$가 $a < 2$ 또는 $b < 2$를 만족시킬 확률은 $\frac{q}{p}$이다. $p+q$의 값을 구하시오. (단, $p$와 $q$는 서로소인 자연수이다.)

## 보기

## 풀이
방정식 $a+b+c=9$를 만족시키는 음이 아닌 정수 $a$, $b$, $c$의 모든 순서쌍의 개수는
$${}_3\mathrm{H}_9 = {}_{3+9-1}\mathrm{C}_9 = {}_{11}\mathrm{C}_2 = 55$$
이다. 선택한 순서쌍이 $a < 2$ 또는 $b < 2$를 만족시킬 확률 $\mathrm{P}(a < 2 \text{ 또는 } b < 2)$는 다음과 같다.

$$\mathrm{P}(a < 2 \text{ 또는 } b < 2)$$
$$= \mathrm{P}(a \le 1 \text{ 또는 } b \le 1)$$
$$= \mathrm{P}(b+c \ge 8 \text{ 또는 } a+c \ge 8)$$
$$= \mathrm{P}(b+c \ge 8) + \mathrm{P}(a+c \ge 8) - \mathrm{P}(b+c \ge 8 \text{ 이고 } a+c \ge 8)$$
$$= \mathrm{P}(b+c \ge 8) + \mathrm{P}(a+c \ge 8) - \{\mathrm{P}(c=9) + \mathrm{P}(c=8) + \mathrm{P}(c=7, a=1, b=1)\}$$
$$= \frac{({}_2\mathrm{H}_8 + {}_2\mathrm{H}_9) + ({}_2\mathrm{H}_8 + {}_2\mathrm{H}_9) - ({}_2\mathrm{H}_0 + {}_2\mathrm{H}_1 + 1)}{{}_3\mathrm{H}_9}$$
$$= \frac{(9+10) + (9+10) - (1+2+1)}{55}$$
$$= \frac{19 + 19 - 4}{55} = \frac{34}{55}$$

따라서 $p = 55$, $q = 34$이며 $p$, $q$는 서로소인 자연수이다.
구하는 값은
$$p+q = 55 + 34 = 89$$

[정답] $89$
