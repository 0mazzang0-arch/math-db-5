## 문제

다음은 임의의 실수 $a$, $b$ 에 대하여 $(가) \leq |a+b| \leq (나)$ 임을 증명한 것이다.

$-|a| \leq a \leq |a|$ 이고 $-|b| \leq b \leq |b|$ 이므로 
$|a+b| \leq (나) \cdots \cdots$ ㉠
㉠을 이용하면 $|a| \leq |a+b| + (다)$ 이므로 
$|a| - (다) \leq |a+b|$ 이다.
같은 방법으로 하면 $(다) - |a| \leq |a+b|$
$\therefore (가) \leq |a+b| \cdots \cdots$ ㉡
㉠과 ㉡에 의해 $(가) \leq |a+b| \leq (나)$ 가 성립한다.

위의 증명에서 $(가)$, $(나)$, $(다)$에 알맞은 것을 순서대로 적은 것은?

## 보기

① $(가): | |a|-|b| |, (나): |a|-|b|, (다): |b|$   ② $(가): | |a|-|b| |, (나): |a|+|b|, (다): -|b|$   ③ $(가): |a+b|, (나): |a-b|, (다): -|b|$   ④ $(가): | |a|-|b| |, (나): |a|+|b|, (다): |b|$   ⑤ $(가): |a-b|, (나): |a+b|, (다): |b|$

## 풀이

$-|a| \leq a \leq |a|$, $-|b| \leq b \leq |b|$ 에서 
$-(|a| + |b|) \leq a+b \leq |a| + |b|$ 이므로 
$|a+b| \leq |a| + |b| \cdots \cdots$ ㉠
따라서 $(나) = |a| + |b|$ 이다.

㉠을 이용하면 $|a| = |(a+b) - b| \leq |a+b| + |-b| = |a+b| + |b|$ 이므로 
$|a| \leq |a+b| + |b|$ 에서 $|a| - |b| \leq |a+b|$ 이다.
따라서 $(다) = |b|$ 이다.

같은 방법으로 $|b| = |(a+b) - a| \leq |a+b| + |-a| = |a+b| + |a|$ 이므로 
$|b| - |a| \leq |a+b|$ 이다.

$|a| - |b| \leq |a+b|$ 이고 $-(|a| - |b|) \leq |a+b|$ 이므로 
$| |a| - |b| | \leq |a+b| \cdots \cdots$ ㉡
따라서 $(가) = | |a| - |b| |$ 이다.

㉠과 ㉡ 에 의해 
$| |a| - |b| | \leq |a+b| \leq |a| + |b|$ 가 성립한다.

그러므로 알맞은 조합은
$(가): | |a| - |b| |, (나): |a| + |b|, (다): |b|$

정답은 ④이다.
