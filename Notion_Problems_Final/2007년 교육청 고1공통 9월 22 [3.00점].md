## 문제
유리수 $a$, $b$에 대하여 이차방정식 $x^{2} + ax + b = 0$의 한 근이 $1 + \sqrt{5}$일 때, $a^{2} + b^{2}$의 값을 구하시오.

## 보기

## 풀이
$x^{2} + ax + b = 0$의 한 근이 $1 + \sqrt{5}$이므로
$(1 + \sqrt{5})^{2} + a(1 + \sqrt{5}) + b = 0$
$(a + b + 6) + (a + 2)\sqrt{5} = 0$
$a$, $b$가 유리수이므로 $a + b + 6 = 0$, $a + 2 = 0$
$\therefore a = -2$, $b = -4$
따라서 
$a^{2} + b^{2} = (-2)^{2} + (-4)^{2} = 20$

정답: $20$
