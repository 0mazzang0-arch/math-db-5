## 문제
$30 \le a \le 40$, $150 \le b \le 294$ 일 때, $\sqrt{a} + \sqrt[3]{b}$ 의 값이 자연수가 되도록 하는 두 자연수 $a$, $b$ 에 대하여 $a+b$ 의 값을 구하시오.

## 풀이
$30 \le a \le 40$, $150 \le b \le 294$ 인 자연수 $a$, $b$ 에 대하여 $\sqrt{a} + \sqrt[3]{b}$ 의 값이 자연수가 되려면 $a$ 는 자연수의 제곱수이고, $b$ 는 자연수의 세제곱수이다.

$5^{2} < 30 \le a \le 40 < 7^{2}$ 이므로
$a = 6^{2} = 36$

$5^{3} < 150 \le b \le 294 < 7^{3}$ 이므로
$b = 6^{3} = 216$

$\therefore a+b = 36 + 216 = 252$

정답: $252$
