## 문제
첫째항이 $a$ 이고 공차가 $a + 1$ 인 등차수열 $\{ a_{n} \}$ 이 
$a_{2} - a_{3} + a_{4} - a_{5} + a_{6} = 15$ 
를 만족시킬 때, $a_{7}$ 의 값을 구하시오.

## 풀이
$a_{2} - a_{3} + a_{4} - a_{5} + a_{6} = a_{2} + ( -a_{3} + a_{4} ) + ( -a_{5} + a_{6} )$
$= a + 3 ( a + 1 ) = 4a + 3 = 15$ 에서 
$a = 3$ 이다.

$\therefore a_{7} = a + 6 ( a + 1 ) = 7a + 6 = 27$
따라서 정답은 $27$ 이다.
