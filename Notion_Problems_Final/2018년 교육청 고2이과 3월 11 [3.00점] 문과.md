## 문제
부등식 $|3x-2| \le x+6$의 해가 $\alpha \le x \le \beta$일 때, $\alpha + \beta$의 값은?

## 보기
① $3$   ② $4$   ③ $5$   ④ $6$   ⑤ $7$

## 풀이
부등식 $|3x-2| \le x+6$에서
(ⅰ) $x \ge \frac{2}{3}$ 일 때
$3x-2 \ge 0$ 이므로
$3x-2 \le x+6$
$2x \le 8$
$\therefore x \le 4$
$x \ge \frac{2}{3}$ 이므로 $\frac{2}{3} \le x \le 4$

(ⅱ) $x < \frac{2}{3}$ 일 때
$3x-2 < 0$ 이므로
$-(3x-2) \le x+6$
$-4 \le 4x$
$\therefore x \ge -1$
$x < \frac{2}{3}$ 이므로 $-1 \le x < \frac{2}{3}$

(ⅰ), (ⅱ)에서 주어진 부등식의 해는
$-1 \le x \le 4$
따라서 $\alpha = -1$, $\beta = 4$ 이므로
$\alpha + \beta = (-1) + 4 = 3$

정답: ①
