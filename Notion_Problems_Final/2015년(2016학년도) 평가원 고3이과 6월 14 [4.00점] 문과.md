## 문제
닫힌구간 $[0, 4]$ 에서 정의된 함수 $f(x) = 2\sqrt{2} \sin \frac{\pi}{4} x$ 의 그래프가 그림과 같고 직선 $y = g(x)$ 가 $y = f(x)$ 의 그래프 위의 점 $A(1, 2)$ 를 지난다. 

> [!example] 📸 (그림/그래프)

일차함수 $g(x)$ 가 닫힌구간 $[0, 4]$ 에서 $f(x) \le g(x)$ 를 만족시킬 때, $g(3)$ 의 값은? 

## 보기
① $\pi$   ② $\pi + 1$   ③ $\pi + 2$   ④ $\pi + 3$   ⑤ $\pi + 4$

## 풀이
닫힌 구간 $[0, 4]$ 에서 $f(x) \le g(x)$ 를 만족시키기 위해서는 일차함수 $g(x)$ 의 그래프가 점 $(1, 2)$ 에서 함수 $f(x) = 2\sqrt{2} \sin \frac{\pi}{4} x$ 의 그래프와 접해야 한다. 
따라서, 
$f'(x) = 2\sqrt{2} \cos \frac{\pi}{4} x \times \frac{\pi}{4} = \frac{\sqrt{2} \pi}{2} \cos \frac{\pi}{4} x$
이고 
$f'(1) = \frac{\sqrt{2} \pi}{2} \times \frac{\sqrt{2}}{2} = \frac{\pi}{2}$
이므로 
$g(x) = \frac{\pi}{2}(x - 1) + 2 = \frac{\pi}{2} x - \frac{\pi}{2} + 2$
$\therefore g(3) = \pi + 2$

정답: ③
