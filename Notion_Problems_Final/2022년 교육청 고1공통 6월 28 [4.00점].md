## 문제
$x$ 에 대한 연립부등식 
$$ \begin{cases} x^{2} - ( a^{2} - 3 ) x - 3 a^{2} < 0 \\ x^{2} + ( a - 9 ) x - 9 a > 0 \end{cases} $$
을 만족시키는 정수 $x$ 가 존재하지 않기 위한 실수 $a$ 의 최댓값을 $M$ 이라 하자. $M^{2}$ 의 값을 구하시오. (단, $a > 2$)

## 보기

## 풀이
연립부등식 
$$ \begin{cases} x^{2} - ( a^{2} - 3 ) x - 3 a^{2} < 0 & \cdots \cdots ㉠ \\ x^{2} + ( a - 9 ) x - 9 a > 0 & \cdots \cdots ㉡ \end{cases} $$
에서 이차부등식 $㉠$ 의 해는 
$x^{2} - ( a^{2} - 3 ) x - 3 a^{2} = ( x - a^{2} ) ( x + 3 ) < 0$ 이고 
$-3 < x < a^{2}$ 이다. 

$a > 2$ 이므로 이차부등식 $㉡$ 의 해는 
$x^{2} + ( a - 9 ) x - 9 a = ( x + a ) ( x - 9 ) > 0$ 이고 
$x > 9$ 또는 $x < -a$ 이다. 

$a^{2} > 10$ 이면 연립부등식의 해에 $x = 10$ 이 포함되므로 정수 $x$ 가 존재한다. 그러므로 정수 $x$ 가 존재하지 않기 위한 $a$ 의 범위는 $a^{2} \le 10$ 에서 $2 < a \le \sqrt{10}$ 이다. 따라서 $a$ 의 최댓값 $M = \sqrt{10}$ 이므로 $M^{2} = 10$ 이다.

정답: $10$
