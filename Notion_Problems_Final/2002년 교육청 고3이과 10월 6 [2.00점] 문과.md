## 문제

위와 같은 순서로 키를 누르면, 계산기는 $\sqrt{\sqrt{2}} \times 4$를 계산하게 된다. 즉,
$$\sqrt{\sqrt{2}} \times 4 = (2^{\frac{1}{2}})^{\frac{1}{2}} \times 2^{2} = 2^{\frac{1}{4}} \times 2^{2} = 2^{\frac{1}{4} + 2} = 2^{\frac{9}{4}}$$
을 계산한 값이 화면에 나타나게 된다. 철수가 가지고 있는 계산기는 [$\sqrt{x}$] 키를 누르면 화면에 나타나 있는 양수의 양의 제곱근을 계산하여 그 값을 화면에 나타낸다. 아래와 같은 순서로 키를 눌렀을 때 화면에 나타나는 값은?

> [!example] 📸 (그림/그래프)
> [ $2$ ] [ $\sqrt{x}$ ] [ $\sqrt{x}$ ] [ $\times$ ] [ $4$ ] [ $=$ ]

## 보기

① $2^{\frac{3}{4}}$ ② $2^{\frac{5}{4}}$ ③ $2^{\frac{7}{4}}$ ④ $2^{\frac{9}{4}}$ ⑤ $2^{\frac{11}{4}}$

## 풀이

계산기에 입력된 순서에 따른 수식은 다음과 같다.
$1$. [ $2$ ] 키를 누르면 화면에 $2$가 나타난다.
$2$. [ $\sqrt{x}$ ] 키를 한 번 누르면 $\sqrt{2} = 2^{\frac{1}{2}}$이 된다.
$3$. [ $\sqrt{x}$ ] 키를 다시 한 번 누르면 $\sqrt{\sqrt{2}} = (2^{\frac{1}{2}})^{\frac{1}{2}} = 2^{\frac{1}{4}}$이 된다.
$4$. [ $\times$ ] [ $4$ ] [ $=$ ] 키를 누르면 앞에 계산된 값에 $4$($2^{2}$)를 곱하게 된다.

따라서 최종적으로 계산되는 값은 다음과 같다.
$$2^{\frac{1}{4}} \times 2^{2} = 2^{\frac{1}{4} + 2} = 2^{\frac{9}{4}}$$

정답은 ④이다.
