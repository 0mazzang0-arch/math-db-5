## 문제
$0 < \alpha < \beta < \frac{\pi}{2}$ 인 두 수 $\alpha$, $\beta$ 가 
$\sin \alpha \sin \beta = \frac{\sqrt{3}+1}{4}$, $\cos \alpha \cos \beta = \frac{\sqrt{3}-1}{4}$
을 만족시킬 때, $\cos (3\alpha + \beta)$ 의 값은?

## 보기
① $-1$   ② $-\frac{\sqrt{3}}{2}$   ③ $-\frac{\sqrt{2}}{2}$   ④ $-\frac{1}{2}$   ⑤ $0$

## 풀이
$0 < \alpha < \beta < \frac{\pi}{2}$ 이므로 
$0 < \alpha + \beta < \pi$, $0 < \beta - \alpha < \frac{\pi}{2}$ 이다. 

$\cos (\alpha + \beta) = \cos \alpha \cos \beta - \sin \alpha \sin \beta = \frac{\sqrt{3}-1}{4} - \frac{\sqrt{3}+1}{4} = -\frac{1}{2}$ 이고 
$\cos (\beta - \alpha) = \cos \alpha \cos \beta + \sin \alpha \sin \beta = \frac{\sqrt{3}-1}{4} + \frac{\sqrt{3}+1}{4} = \frac{\sqrt{3}}{2}$ 이므로 
$\alpha + \beta = \frac{2}{3} \pi$, $\beta - \alpha = \frac{\pi}{6}$ 이다.

이때 $2\alpha = (\alpha + \beta) - (\beta - \alpha) = \frac{2}{3} \pi - \frac{\pi}{6} = \frac{\pi}{2}$ 이므로
$\therefore \cos (3\alpha + \beta) = \cos (2\alpha + (\alpha + \beta)) = \cos \left( \frac{\pi}{2} + \frac{2}{3} \pi \right) = \cos \frac{7}{6} \pi = -\cos \frac{\pi}{6} = -\frac{\sqrt{3}}{2}$

정답: ②
