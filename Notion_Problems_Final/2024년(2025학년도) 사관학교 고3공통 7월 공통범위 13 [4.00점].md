## 문제
$-6 \le t \le 2$ 인 실수 $t$ 와 함수 $f(x)=2x(2-x)$ 에 대하여 $x$ 에 대한 방정식 $\{f(x)-t\}\{f(x-1)-t\}=0$ 의 실근 중에서 집합 $\{x | 0 \le x \le 3\}$ 에 속하는 가장 큰 값과 가장 작은 값의 차를 $g(t)$ 라 할 때, 함수 $g(t)$ 는 $t=a$ 에서 불연속이다. $\lim_{t \to a-} g(t) + \lim_{t \to a+} g(t)$ 의 값은? (단, $a$ 는 $-6 < a < 2$ 인 상수이다.)

## 보기
① $3$   ② $\frac{7}{2}$   ③ $4$   ④ $\frac{9}{2}$   ⑤ $5$

## 풀이
$-6 \le t \le 2$ 이고, 함수 $f(x)=2x(2-x)$ 이다. 방정식 $\{f(x)-t\}\{f(x-1)-t\}=0$ 에서
$f(x)=t$ 또는 $f(x-1)=t \cdots \cdots$ ㉠
방정식 $f(x)=t$ 의 두 실근을 $\alpha, \beta$ ($\alpha < \beta$)라 하면, 방정식 $f(x-1)=t$ 의 두 실근은 $\alpha+1, \beta+1$ 이다.
방정식 $f(x)=t$ 에서
$2x(2-x)=t$, $2x^{2}-4x+t=0$
근과 계수의 관계에 의하여
$\alpha + \beta = 2$, $\alpha\beta = \frac{t}{2}$
이므로
$(\alpha-\beta)^{2} = 4-2t$
$\therefore \beta-\alpha = \sqrt{4-2t}$

$0 \le x \le 3$ 에서 ㉠을 만족시키는 실수 $x$ 의 최댓값과 최솟값의 차가 $g(t)$ 이다.

(ⅰ) $0 \le t \le 2$ 일 때
> [!example] 📸 (그림/그래프)

$0 \le t < 2$ 인 실수 $t$ 에 대하여 $0 \le x \le 3$ 에서 ㉠을 만족시키는 실수 $x$ 는
$x=\alpha$ 또는 $x=\alpha+1$ 또는 $x=\beta$ 또는 $x=\beta+1$
($\alpha < \alpha+1 < \beta < \beta+1$ 또는 $\alpha < \beta \le \alpha+1 < \beta+1$)
$g(t) = (\beta+1) - \alpha = \sqrt{4-2t} + 1$
$t=2$ 이면 $0 \le x \le 3$ 에서 ㉠을 만족시키는 실수 $x$ 는
$x=1$ 또는 $x=2$, $g(2)=1$
$0 \le t \le 2$ 인 모든 실수 $t$ 에 대하여
$g(t) = \sqrt{4-2t} + 1$

(ⅱ) $-6 \le t < 0$ 일 때
> [!example] 📸 (그림/그래프)

$0 \le x \le 3$ 에서 ㉠을 만족시키는 실수 $x$ 는
$x=\alpha+1$ 또는 $x=\beta$ ($\alpha+1 < \beta$)
$g(t) = \beta - (\alpha+1) = \sqrt{4-2t} - 1$

(ⅰ), (ⅱ)에 의하여
$g(t) = \begin{cases} \sqrt{4-2t}-1 & (-6 \le t < 0) \\ \sqrt{4-2t}+1 & (0 \le t \le 2) \end{cases}$
이다.
$\lim_{t \to 0-} g(t) = \lim_{t \to 0-} (\sqrt{4-2t}-1) = 1$
$\lim_{t \to 0+} g(t) = \lim_{t \to 0+} (\sqrt{4-2t}+1) = 3$
$\lim_{t \to 0-} g(t) \neq \lim_{t \to 0+} g(t)$ 이므로 함수 $g(t)$ 는 $t=0$ 에서 불연속이다.
$\therefore a=0$
따라서
$\lim_{t \to a-} g(t) + \lim_{t \to a+} g(t) = \lim_{t \to 0-} g(t) + \lim_{t \to 0+} g(t) = 1+3=4$

정답: ③
