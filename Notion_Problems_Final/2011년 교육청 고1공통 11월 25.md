## 문제
$-1 \le x \le 1$ 에서 이차부등식 $x^{2} - 2x + 3 \le -x^{2} + k$ 가 항상 성립할 때, 실수 $k$ 의 최솟값을 구하시오.

## 풀이
$x^2 - 2x + 3 \le -x^2 + k$ 에서 
$2x^2 - 2x + 3 - k \le 0 \cdots \cdots$ ㉠ 

$f(x) = 2x^2 - 2x + 3 - k$ 라 하면 
$f(x) = 2 \left( x - \frac{1}{2} \right)^2 + \frac{5}{2} - k$ 
($-1 \le x \le 1$) 
이므로 $-1 \le x \le 1$ 에서 부등식 ㉠을 만족시키려면 함수 $f(x)$ 의 최댓값이 $0$ 보다 작거나 같아야 한다. 

함수 $f(x)$ 는 $x = -1$ 에서 최댓값 $7 - k$ 를 가지므로 
$7 - k \le 0$ 
$\therefore k \ge 7$ 

따라서 $k$ 의 최솟값은 $7$ 이다.

정답: $7$
