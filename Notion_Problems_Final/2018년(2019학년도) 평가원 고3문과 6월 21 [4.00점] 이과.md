## 문제
상수 $a$, $b$에 대하여 삼차함수 $f(x) = x^3 + ax^2 + bx$가 다음 조건을 만족시킨다.

(가) $f(-1) > -1$
(나) $f(1) - f(-1) > 8$

<보기>에서 옳은 것만을 있는 대로 고른 것은?

ㄱ. 방정식 $f'(x) = 0$은 서로 다른 두 실근을 갖는다.
ㄴ. $-1 < x < 1$일 때, $f'(x) \ge 0$이다.
ㄷ. 방정식 $f(x) - f'(k)x = 0$의 서로 다른 실근의 개수가 $2$가 되도록 하는 모든 실수 $k$의 개수는 $4$이다.

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄷ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
조건 (가)에서 $f(-1) = -1 + a - b > -1$이므로 $a > b$
조건 (나)에서 $f(1) - f(-1) = (1 + a + b) - (-1 + a - b) > 8$
$2 + 2b > 8$이므로 $b > 3$
조건 (가), (나)에서 $a > b > 3$

ㄱ. $f'(x) = 3x^2 + 2ax + b$이므로 $3x^2 + 2ax + b = 0$의 판별식을 $D_1$이라 하면
$\frac{D_1}{4} = a^2 - 3b$
$a > b > 3$이므로 $a^2 - 3b > b^2 - 3b = b(b - 3) > 0$
따라서 $f'(x) = 0$은 서로 다른 두 실근을 갖는다. (참)

ㄴ. $f'(-1) = 3 - 2a + b = 3 - (2a - b) = 3 - \{(a - b) + a\}$
$a - b > 0$이고 $a > 3$이므로 $f'(-1) < 0$이다.
$-1 < x < 1$인 모든 실수 $x$에 대하여 $f'(x) \ge 0$이 성립하지 않는다. (거짓)

ㄷ. $f(x) - f'(k)x = 0$
$x^3 + ax^2 + bx - (3k^2 + 2ak + b)x = 0$
$x(x^2 + ax - 3k^2 - 2ak) = 0$
$x = 0$ 또는 $x^2 + ax - 3k^2 - 2ak = 0$

(ⅰ) 이차방정식 $x^2 + ax - 3k^2 - 2ak = 0$이 $x = 0$을 근으로 갖는 경우
$-3k^2 - 2ak = 0$에서 $k = 0$ 또는 $k = -\frac{2a}{3}$
이때 $f(x) - f'(k)x = 0$은 두 실근 $x = 0$, $x = -a$를 갖는다. ($a > 3$이므로 서로 다름)

(ⅱ) 이차방정식 $x^2 + ax - 3k^2 - 2ak = 0$이 $0$이 아닌 중근을 갖는 경우
이차방정식의 판별식을 $D_2$라 하면
$D_2 = a^2 - 4(-3k^2 - 2ak) = a^2 + 12k^2 + 8ak = 0$
$k$에 대한 이차방정식 $12k^2 + 8ak + a^2 = 0$에서
$k = \frac{-8a \pm \sqrt{64a^2 - 48a^2}}{24} = \frac{-8a \pm 4a}{24}$
따라서 $k = -\frac{a}{2}$ 또는 $k = -\frac{a}{6}$
이때 중근은 $x = -\frac{a}{2}$이며, $a > 3$이므로 이 근은 $0$이 아니다.

(ⅰ), (ⅱ)에서 실수 $k$의 개수는 $0, -\frac{2a}{3}, -\frac{a}{2}, -\frac{a}{6}$으로 $4$이다. (참)

이상에서 옳은 것은 ㄱ, ㄷ이다.
