## 문제
두 양수 $a$, $b$가 $\log_{b} a + \log_{a} b = \frac{26}{5}$, $ab = 27$을 만족시킬 때, $a^{2} + b^{2}$의 값은? (단, $a \neq 1$, $b \neq 1$)

## 보기
① $240$   ② $242$   ③ $244$   ④ $246$   ⑤ $248$

## 풀이
$\log a = A$, $\log b = B$라고 하면
$\frac{A}{B} + \frac{B}{A} = \frac{26}{5}$, $5A^{2} - 26AB + 5B^{2} = 0$
$\therefore A = \frac{1}{5}B$ 또는 $A = 5B$

$ab = 27$의 양변에 상용로그를 취하면
$A + B = \log 27$

(ⅰ) $A = \frac{1}{5}B$ 일 때
$A = \frac{1}{6} \log 27$, $B = \frac{5}{6} \log 27$

(ⅱ) $A = 5B$ 일 때
$A = \frac{5}{6} \log 27$, $B = \frac{1}{6} \log 27$

즉, $a^{2} + b^{2} = 27^{\frac{5}{3}} + 27^{\frac{1}{3}} = (3^{3})^{\frac{5}{3}} + (3^{3})^{\frac{1}{3}} = 3^{5} + 3^{1} = 243 + 3 = 246$

정답: ④
