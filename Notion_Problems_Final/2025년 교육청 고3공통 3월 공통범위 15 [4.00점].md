## 문제
세 실수 $a$, $p$, $q$ ($p < q$)에 대하여 함수 $f(x)$가
$$f(x) = \begin{cases} | 2^{x}-4| & (x \le p \text{ 또는 } x \ge q) \\ a+ \log_{2}x & (p < x < q) \end{cases}$$
이다. 함수 $f(x)$가 실수 전체의 집합에서 실수 전체의 집합으로의 일대일대응일 때, $f\left(\frac{p+q}{2}\right)$의 값은?



> [!example] 📸 (그림/그래프)

## 보기
① $\frac{3}{2}$   ② $2$   ③ $\frac{5}{2}$   ④ $3$   ⑤ $\frac{7}{2}$

## 풀이
[정답] ②

함수 $y= | 2^{x}-4|$ 의 치역이 $\{y| y \ge 0 \}$, 
함수 $y=a+ \log_{2}x$ 의 치역이 실수 전체의 집합이고 
함수 $f(x)$ 의 치역이 실수 전체의 집합이므로 

$p \le 0$ 
$\cdots \cdots$ ㉠ 

$p < x < q$ 에서 $f(x)=a+ \log_{2}x$ 이고 
함수 $f(x)$ 의 정의역이 실수 전체의 집합이므로 

$p \ge 0$ 
$\cdots \cdots$ ㉡ 

㉠, ㉡에서 $p=0$ 

$\{f(x)| x \le 0 \} = \{y| 3 \le y < 4 \}$ 
함수 $f(x)$ 가 일대일대응이므로 

$\{f(x)| x > 0 \} = \{y| y < 3 \text{ 또는 } y \ge 4 \}$ 이고 

$\{f(x)| 0 < x < q \} = \{y| y < a+ \log_{2}q \} = \{y| y < 3 \}$, 

$\{f(x)| x \ge q \} = \{y| y \ge 4 \}$ 이다. 

$| 2^{q}-4| = 4$ 에서 
$q=3$ 

$a+ \log_{2}3=3$, 
$a=3- \log_{2}3$ 

$f\left(\frac{p+q}{2}\right)$ 
$=f\left(\frac{3}{2}\right)=3- \log_{2}3+ \log_{2}\frac{3}{2}$ 

$=3- \log_{2}3+ \log_{2}3-1=2$
