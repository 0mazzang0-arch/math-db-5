## 문제
연속확률변수 $X$ 가 갖는 값의 범위는 $0 \le X \le a$ 이고, $X$ 의 확률밀도함수의 그래프가 그림과 같다.

> [!example] 📸 (그림/그래프)

$P(X \le b) - P(X \ge b) = \frac{1}{4}$, $P(X \le \sqrt{5}) = \frac{1}{2}$ 일 때, $a + b + c$ 의 값은? (단, $a$, $b$, $c$ 는 상수이다.)

## 보기
① $\frac{11}{2}$   ② $6$   ③ $\frac{13}{2}$   ④ $7$   ⑤ $\frac{15}{2}$

## 풀이
$P(0 \le X \le a) = 1$ 이므로 확률변수 $X$ 의 확률밀도함수의 그래프로부터
$\frac{1}{2} ac = 1$, $ac = 2 \cdots$ ㉠

한편, $P(0 \le X \le a) = P(X \le b) + P(X \ge b) = 1$ 이고
$P(X \le b) - P(X \ge b) = \frac{1}{4}$ 이므로
$P(X \le b) = \frac{5}{8}$, $P(X \ge b) = \frac{3}{8}$

따라서 확률변수 $X$ 의 확률밀도함수의 그래프로부터
$\frac{1}{2} \times b \times c = \frac{5}{8}$, $bc = \frac{5}{4} \cdots$ ㉡

$P(X \le b) > \frac{1}{2}$ 이므로 $0 < \sqrt{5} < b$
이때 두 점 $(0, 0)$, $(b, c)$ 를 지나는 직선의 방정식은
$y = \frac{c}{b} x$
이므로 $P(X \le \sqrt{5}) = \frac{1}{2}$ 에서
$\frac{1}{2} \times \sqrt{5} \times (\frac{c}{b} \times \sqrt{5}) = \frac{5c}{2b} = \frac{1}{2}$
$b = 5c \cdots$ ㉢

㉠, ㉡, ㉢을 연립하면
$a = 4$, $b = \frac{5}{2}$, $c = \frac{1}{2}$

$\therefore a + b + c = 4 + \frac{5}{2} + \frac{1}{2} = 7$
