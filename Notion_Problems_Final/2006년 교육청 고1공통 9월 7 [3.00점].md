## 문제
$x$에 대한 이차방정식 $x^2 + ax + b = 0$의 두 근이 $a$, $b$일 때, $a^2 + b^2$의 값은? (단, $ab \neq 0$)

## 보기
① $3$   ② $5$   ③ $7$   ④ $10$   ⑤ $13$

## 풀이
이차방정식 $x^2 + ax + b = 0$의 두 근이 $a$, $b$이므로 근과 계수의 관계에 의하여
$a + b = -a$, $ab = b$
두 식을 연립하면 $a = 1$, $b = -2$

$\therefore a^2 + b^2 = 5$

[참고]
주어진 방정식의 양변에 $x = a$를 대입하면
$2a^2 + b = 0$

$\therefore b = -2a^2$ $\cdots$ ㉠

$x = b$를 대입하면
$b^2 + ab + b = 0$ $\cdots$ ㉡

㉠을 ㉡에 대입하면
$4a^4 - 2a^3 - 2a^2 = 0$, $2a^2(2a^2 - a - 1) = 0$
$2a^2(2a + 1)(a - 1) = 0$

$\therefore a = 1, b = -2$ 또는 $a = -\frac{1}{2}, b = -\frac{1}{2}$
($\because ab \neq 0$)

그런데 $a = -\frac{1}{2}, b = -\frac{1}{2}$일 때, $x^2 - \frac{1}{2}x - \frac{1}{2} = 0$의 두 근은 $x = -\frac{1}{2}$과 $x = 1$이므로 $a$, $b$는 $x^2 + ax + b = 0$의 두 근이 아니다.

정답: ②
