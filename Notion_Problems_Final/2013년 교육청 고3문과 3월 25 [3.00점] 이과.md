## 문제
주어진 무한등비수열은 $-1 < x < 4$ 일 때 $0$ 으로 수렴하고, $x = 4$ 일 때 $1$ 로 수렴한다. 무한등비수열 $\left\{ \left( \frac{2x - 3}{5} \right)^{n} \right\}$ 이 수렴하도록 하는 모든 정수 $x$ 의 합을 구하시오.

## 보기

## 풀이
무한등비수열 $\left\{ \left( \frac{2x - 3}{5} \right)^{n} \right\}$ 이 수렴하려면 공비가 $-1$ 보다 크고 $1$ 보다 작거나 같아야 하므로
$-1 < \frac{2x - 3}{5} \le 1$,
$-5 < 2x - 3 \le 5$,
$-2 < 2x \le 8$
$\therefore -1 < x \le 4$
따라서 구하는 정수 $x$ 는 $x = 0$, $1$, $2$, $3$, $4$ 이고 그 합은 $10$ 이다.

정답: $10$
