## 문제
함수 $f(x) = \sum_{n=1}^{10} \frac{x^{n}}{n}$에 대하여 $f' \left( \frac{1}{2} \right) = \frac{q}{p}$일 때, $q-p$의 값은? (단, $p$와 $q$는 서로소인 자연수이다.)

## 보기
① $508$   ② $509$   ③ $510$   ④ $511$   ⑤ $512$

## 풀이
$f(x) = x + \frac{x^{2}}{2} + \frac{x^{3}}{3} + \dots + \frac{x^{10}}{10}$이므로
$f'(x) = 1 + x + x^{2} + \dots + x^{9}$이다.
$f' \left( \frac{1}{2} \right) = 1 + \frac{1}{2} + \frac{1}{2^{2}} + \frac{1}{2^{3}} + \dots + \frac{1}{2^{9}}$
$= \frac{1 - \frac{1}{2^{10}}}{1 - \frac{1}{2}}$
$= \frac{2^{10} - 1}{2^{9}}$
$= \frac{1023}{512}$
이때 $p$와 $q$는 서로소인 자연수이므로
$q = 1023, p = 512$
따라서 $q - p = 1023 - 512 = 511$이다.

정답: ④
