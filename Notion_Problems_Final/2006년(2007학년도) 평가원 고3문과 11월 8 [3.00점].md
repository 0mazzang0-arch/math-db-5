## 문제
$1$보다 큰 세 실수 $a$, $b$, $c$에 대하여 $\log_{a} c : \log_{b} c = 2 : 1$일 때, $\log_{a} b + \log_{b} a$의 값은?

## 보기
① $1$   ② $\frac{3}{2}$   ③ $2$   ④ $\frac{5}{2}$   ⑤ $3$

## 풀이
$\log_{a} c : \log_{b} c = 2 : 1$이므로
$\log_{a} c = 2 \log_{b} c$
이 때, $\frac{1}{\log_{c} a} = \frac{2}{\log_{c} b}$이므로
$\frac{\log_{c} b}{\log_{c} a} = \log_{a} b = 2$
$\therefore \log_{a} b + \log_{b} a = \log_{a} b + \frac{1}{\log_{a} b} = 2 + \frac{1}{2} = \frac{5}{2}$

[정답] ④
