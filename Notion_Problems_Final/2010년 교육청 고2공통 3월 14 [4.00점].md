## 문제
실수 전체의 집합에서 정의된 이차함수 $f(x) = ax^2 + bx$ 가 다음 조건을 만족시킨다.

(가) $f(-1) + f(1) < 0$
(나) $-1 \le x_1 < x_2 \le 1$ 이면 $-2 < f(x_1) < f(x_2) < 2$ 이다.

실수 $a$, $b$ 에 대한 설명으로 항상 옳은 것만을 <보기>에서 있는 대로 고른 것은?

> <보기>
> ㄱ. $a^2 < b^2$
> ㄴ. $2a + b \ge 0$
> ㄷ. 방정식 $f(x) = 0$ 의 두 근의 합은 $2$ 보다 작다.

## 보기
① ㄱ   ② ㄱ, ㄴ   ③ ㄱ, ㄷ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
[정답] ②

$f(-1) = a - b$, 
$f(0) = 0$, 
$f(1) = a + b$ 이고, 
조건 (가)에서 
$f(-1) + f(1) = (a - b) + (a + b) = 2a < 0$ 
$\therefore a < 0$ $\cdots \cdots$ ㉠ 

조건 (나)에서 
$-2 < a - b < 0 < a + b < 2$ $\cdots \cdots$ ㉡ 

ㄱ. ㉡에서 
$a - b < 0$ 이고 
$a + b > 0$ 이므로 
$(a + b)(a - b) < 0$ 
$\therefore a^2 < b^2$ 

ㄴ. 다음 그림과 같이 조건 (가)를 만족시키는 이차함수 $y = f(x)$ 의 그래프는 위로 볼록한 포물선이므로 $y = f(x)$ 의 그래프의 대칭축 $x = -\frac{b}{2a}$ 가 직선 $x = 1$ 보다 왼쪽에 있지 않다. 

> [!example] 📸 (그림/그래프)

$\therefore -\frac{b}{2a} \ge 1$ 
$a < 0$ 이므로 양변에 $-2a$ 를 곱하면 
$b \ge -2a$ 
$\therefore 2a + b \ge 0$ 

ㄷ. 이차방정식 $f(x) = 0$ 의 두 실근의 합은 근과 계수의 관계에 의해 $-\frac{b}{a}$ 이다. 그런데 $-\frac{b}{2a} \ge 1$ 에서 $-\frac{b}{a} \ge 2$ 이므로 이차방정식 $f(x) = 0$ 의 두 실근의 합은 $2$ 보다 크거나 같다. 

이상에서 옳은 것은 ㄱ, ㄴ이다.
