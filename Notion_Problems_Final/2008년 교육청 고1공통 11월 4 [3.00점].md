## 문제
전체집합 $U = \{1, 2, 3, 4, 5\}$의 두 부분집합 $A$, $B$에 대하여 $A \cap (A - B) = A$, $A \cup B = U$이고 집합 $A = \{1, 4, 5\}$일 때, 집합 $B$의 모든 원소의 합은?

## 보기
① $2$   ② $3$   ③ $5$   ④ $7$   ⑤ $9$

## 풀이
$A \cap (A - B) = A$이므로 $A \subseteq (A - B)$이다. 이는 $A \cap B = \emptyset$임을 의미한다.
$A \cap B = \emptyset$이고 $A \cup B = U$이므로 $B$는 $A$의 여집합 $A^c$와 같다.
$U = \{1, 2, 3, 4, 5\}$이고 $A = \{1, 4, 5\}$이므로
$B = \{2, 3\}$이다.
따라서 집합 $B$의 모든 원소의 합은 $2 + 3 = 5$이다.

정답: ③
