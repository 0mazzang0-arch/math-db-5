## 문제
이차정사각행렬 $X$, $Y$에 대하여 $*$를 
$$ X * Y = (X - Y)(X + Y) $$
라 정의하자. 이차정사각행렬 $A$, $B$에 대하여 <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $E$는 단위행렬이고, $O$는 영행렬이다.)

$$ \begin{array}{|l|} \hline \text{<보기>} \\ \text{ㄱ. } A * O = O \text{ 이면 } A = O \text{ 이다.} \\ \text{ㄴ. } A * B = A * (-B) \text{ 이면 } (AB)^{2} = A^{2} B^{2} \text{ 이다.} \\ \text{ㄷ. } A * E = A \text{ 이면 } A + E \text{ 의 역행렬이 존재한다.} \\ \hline \end{array} $$

## 보기
① ㄱ   ② ㄴ   ③ ㄷ   ④ ㄱ, ㄴ   ⑤ ㄴ, ㄷ

## 풀이
[정답] ⑤ 

ㄱ. $A * O = (A - O)(A + O) = A^{2} = O$ 이면 $A = O$ 이다. (거짓)
반례로 $A = \left( \begin{array}{cc} 0 & 0 \\ 1 & 0 \end{array} \right)$ 이면 $A^{2} = O$ 이지만 $A \neq O$ 이다.

ㄴ. $A * B = (A - B)(A + B) = A^{2} + AB - BA - B^{2}$
$A * (-B) = (A - (-B))(A + (-B)) = (A + B)(A - B) = A^{2} - AB + BA - B^{2}$
$A * B = A * (-B)$ 이면 
$A^{2} + AB - BA - B^{2} = A^{2} - AB + BA - B^{2}$ 이므로
$2AB = 2BA$, 즉 $AB = BA$ 이다.
교환법칙이 성립하므로 $(AB)^{2} = A^{2} B^{2}$ 이다. (참)

ㄷ. $A * E = (A - E)(A + E) = A^{2} - E$ 이다. (단위행렬 $E$는 모든 행렬과 교환법칙 성립)
$A * E = A$ 이므로 $A^{2} - E = A$, 즉 $A^{2} - A - E = O$ 이다.
이 식을 변형하면
$A^{2} - A - 2E + E = O$
$(A + E)(A - 2E) + E = O$
$(A + E)(2E - A) = E$
따라서 $(A + E)^{-1} = 2E - A$ 가 존재한다. (참)

따라서 옳은 것은 ㄴ, ㄷ이다.
