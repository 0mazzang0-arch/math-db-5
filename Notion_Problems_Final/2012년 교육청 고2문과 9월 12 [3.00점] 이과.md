## 문제
함수 $f(x)$의 그래프 위의 두 점 $P(a, b)$, $Q(c, d)$가 $a < c$, $a + b = c + d$를 만족시킨다. $b + c = 2\sqrt{2}$일 때, $d$의 값은?

## 보기
① $\frac{1}{4}$   ② $\frac{1}{2}$   ③ $1$   ④ $\frac{3}{2}$   ⑤ $2$

## 풀이
$a + b = c + d$는 $\frac{d - b}{c - a} = -1$이므로 점 $P$, $Q$는 $y = x$에 대하여 대칭이다. 
따라서 $a = d$, $b = c$이고 $0 < a \le 1$이다. 

$b + c = 2c = 2\sqrt{2}$에서 $c = \sqrt{2}$이다.
$Q(c, d)$가 함수 $f(x) = \log_{2} x$의 그래프 위의 점이므로 $d = \log_{2} \sqrt{2}$이다. 

$\therefore d = \frac{1}{2}$

정답: ②
