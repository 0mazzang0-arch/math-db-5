## 문제
두 함수 $f(x) = \log x$, $g(x) = 10^x$과 실수의 부분집합 $C$에 대하여 두 집합 $A$, $B$를 각각 
$$ A = \{ x \mid f(x) \in C \}, \quad B = \{ x \mid g(x) \in C \} $$
라고 하자. <보기>에서 옳은 것만을 있는 대로 고른 것은?

**<보기>**
ㄱ. $C = \{ \frac{1}{10}, 1, 10 \}$ 이면 $B = \{ -1, 0, 1 \}$ 이다.
ㄴ. $C$가 자연수 전체의 집합이면 집합 $B$는 곱셈에 대하여 닫혀있다.
ㄷ. 집합 $C$가 공집합이 아니면 $\{ g(x) \mid x \in C \} = A$ 이다.

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄷ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
ㄱ. $k \in C$이면 $g(x) = 10^x = k$를 만족하는 $x = \log k \in B$이다. 
$C = \{ \frac{1}{10}, 1, 10 \}$이면 
$B = \{ \log \frac{1}{10}, \log 1, \log 10 \} = \{ -1, 0, 1 \}$이다. 
따라서 참이다. 

ㄴ. 집합 $C$가 자연수의 집합이고, $10^{\log 2} = 2$이므로 $\log 2 \in B$이다. 
$10^{(\log 2) \times (\log 2)} = (10^{\log 2})^{\log 2} = 2^{\log 2} \notin C$
즉, $(\log 2) \times (\log 2) \notin B$이다. 
따라서 거짓이다. 

ㄷ. $k \in C$이면 $f(x) = \log x = k$를 만족하는 $x = 10^k \in A$이다. 
따라서 $\{ g(x) \mid x \in C \} = \{ 10^x \mid x \in C \} = A$이다. 
따라서 참이다. 

그러므로 옳은 것은 ㄱ, ㄷ이다.
