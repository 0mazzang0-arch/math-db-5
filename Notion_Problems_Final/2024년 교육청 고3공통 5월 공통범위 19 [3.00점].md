## 문제
집합 $U = \{ x \mid -5 \le x \le 5, x \text{ 는 정수} \}$의 공집합이 아닌 부분집합 $X$에 대하여 두 집합 $A$, $B$를
$A = \{ a \mid a \text{ 는 } x \text{ 의 실수인 네제곱근, } x \in X \}$,
$B = \{ b \mid b \text{ 는 } x \text{ 의 실수인 세제곱근, } x \in X \}$
라 하자. $n(A) = 9$, $n(B) = 7$이 되도록 하는 집합 $X$의 모든 원소의 합의 최댓값을 구하시오.

## 보기

## 풀이
집합 $X$의 원소 중 양수의 개수를 $p$, 음수의 개수를 $q$라 하자. 

$0 \notin X$이면 $n(A) = 2p$이므로 $n(A) = 9$를 만족시키지 않는다. 그러므로 $0 \in X$

$n(A) = 2p + 1 = 9$에서 $p = 4$ 

$n(B) = p + q + 1 = 7$에서 $q = 2$ 

따라서 집합 $X$의 모든 원소의 합은 $X = \{ -2, -1, 0, 2, 3, 4, 5 \}$일 때 최대이고 그 값은 
$-2 + (-1) + 0 + 2 + 3 + 4 + 5 = 11$ 

정답: $11$
