## 문제
그림과 같이 두 원 $C_{1} : x^{2} + y^{2} = 1$과 $C_{2} : x^{2} + y^{2} = 8$이 있다. 원 $C_{1}$에 접하는 직선 $l$의 방정식은 $ax + by + 1 = 0$이다. 직선 $l$에 평행하고 원 $C_{2}$에 접하는 두 직선을 각각 $l_{1}$, $l_{2}$라 하자. 점 $P_{1}(x_{1}, y_{1})$은 직선 $l_{1}$ 위에 있고, 점 $P_{2}(x_{2}, y_{2})$는 직선 $l_{2}$와 원 $C_{2}$의 접점이다. $(ax_{1} + by_{1} + 1)(ax_{2} + by_{2} + 1)$의 값은?

> [!example] 📸 (그림/그래프)

## 보기
① $-7$   ② $-4$   ③ $-1$   ④ $2$   ⑤ $5$

## 풀이
원점에서 직선 $l$까지의 거리가 $1$이므로 
$$\frac{1}{\sqrt{a^{2} + b^{2}}} = 1$$ 
이다. 
$\therefore a^{2} + b^{2} = 1$ 
점 $P_{1}(x_{1}, y_{1})$에서 직선 $l$에 이르는 수선의 길이는 
$|ax_{1} + by_{1} + 1| = 2\sqrt{2} + 1$ 이고, 점 $P_{2}(x_{2}, y_{2})$에서 직선 $l$에 이르는 수선의 길이는 
$|ax_{2} + by_{2} + 1| = 2\sqrt{2} - 1$ 이다. 따라서 
$$|(ax_{1} + by_{1} + 1)(ax_{2} + by_{2} + 1)| = (2\sqrt{2} + 1)(2\sqrt{2} - 1) = 7$$ 
그런데 직선 $l$이 두 점 $P_{1}(x_{1}, y_{1})$, $P_{2}(x_{2}, y_{2})$ 사이를 지나므로 
$$(ax_{1} + by_{1} + 1)(ax_{2} + by_{2} + 1) = -7$$ 
이다.
