## 문제
그림은 두 함수 $f : X \to X$, $g : X \to X$를 나타낸 것이다.

> [!example] 📸 (그림/그래프)

함수 $h : X \to X$ 가 $f \circ h = g$를 만족시킬 때, $( h \circ f  )  ( 3  )$의 값은?

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
$( h \circ f  )  ( 3  ) = h  ( f  ( 3  )  ) = h  ( 2  )$이다.
한편, $f \circ h = g$이므로 $( f \circ h  )  ( 2  ) = g  ( 2  )$이다.
즉, $f  ( h  ( 2  )  ) = 3$이다. 이때 $f  ( 1  ) = 3$이므로 $h  ( 2  ) = 1$이다.
따라서 $( h \circ f  )  ( 3  ) = 1$이다.
