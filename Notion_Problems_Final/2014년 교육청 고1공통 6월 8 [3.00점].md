## 문제
$( 2x+y-1 )^{2} = 3$ 을 만족시키는 $x$, $y$ 에 대하여 $4x^{2} + y^{2} + 4xy - 4x - 2y$ 의 값은?

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
**[정답]** ②

$( 2x+y-1 )^{2} = 3$ 에서
$4x^{2} + y^{2} + ( -1 )^{2} + 4xy - 2y - 4x = 3$ 이므로
$4x^{2} + y^{2} + 4xy - 4x - 2y = 2$

**[다른 풀이]**
$2x+y-1 = \pm \sqrt{3}$ 이므로
$2x+y = 1 \pm \sqrt{3}$
위 식의 양변을 제곱하면
$( 2x+y )^{2} = ( 1 \pm \sqrt{3} )^{2}$

$\therefore 4x^{2} + 4xy + y^{2} = 4 \pm 2\sqrt{3}$

따라서
$4x^{2} + y^{2} + 4xy - 2y - 4x$
$= ( 4x^{2} + 4xy + y^{2} ) - 2( 2x+y )$
$= ( 4 \pm 2\sqrt{3} ) - 2( 1 \pm \sqrt{3} )$ (복부호동순)
$= 2$
