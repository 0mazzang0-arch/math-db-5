## 문제
함수 $f( 0 ) = 0$ 인 함수 $y = f( x )$ ($ -2 \le x \le 2 $) 의 그래프는 다음과 같다.

> [!example] 📸 (그림/그래프)

함수 $y = g( x )$ ($ -2 \le x \le 2 $) 의 그래프가 다음과 같이 주어질 때, 합성함수 $y = ( g \circ f )( x )$ 가 구간 $[ -2, 2 ]$ 에서 연속이 되는 경우를 <보기>에서 있는 대로 고른 것은?

$$
\begin{array}{c}
ㄱ. \\
\text{> [!example] 📸 (그림/그래프)} \\
\\
ㄴ. \\
\text{> [!example] 📸 (그림/그래프)} \\
\\
ㄷ. \\
\text{> [!example] 📸 (그림/그래프)}
\end{array}
$$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄷ$   ④ $ㄱ, ㄷ$   ⑤ $ㄴ, ㄷ$

## 풀이
합성함수 $( g \circ f )( x )$ 가 $x = 0$ 에서 연속이면 구간 $[ -2, 2 ]$ 에서 연속이다.
따라서 $\lim_{x \to 0} g( f( x ) ) = g( f( 0 ) )$ 가 성립해야 한다.

이때,
$\lim_{x \to 0^+} g( f( x ) ) = g( -1 )$
$\lim_{x \to 0^-} g( f( x ) ) = g( 1 )$
$g( f( 0 ) ) = g( 0 )$ 이므로

$g( -1 ) = g( 1 ) = g( 0 )$ 이어야 한다.

이때, $g( -1 ) = g( 1 ) = g( 0 )$ 을 만족하는 함수 $g( x )$ 는 $ㄴ$뿐이다.
함수 $f( 0 ) = 0$ 인 함수 $y = f( x )$ ($ -2 \le x \le 2 $) 의 그래프는 다음과 같다.

> [!example] 📸 (그림/그래프)

정답: ②
