## 문제
다항식 $x^{4} + 7x^{2} + 16$이 $(x^{2} + ax + b)(x^{2} - ax + b)$로 인수분해될 때, 두 양수 $a$, $b$에 대하여 $a + b$의 값은?

## 보기
① $5$   ② $6$   ③ $7$   ④ $8$   ⑤ $9$

## 풀이
다항식 $x^{4} + 7x^{2} + 16$을 인수분해하면 
$x^{4} + 7x^{2} + 16$
$= (x^{4} + 8x^{2} + 16) - x^{2}$
$= (x^{2} + 4)^{2} - x^{2}$
$= (x^{2} + x + 4)(x^{2} - x + 4)$
이므로 $a = 1$, $b = 4$이다. 

$\therefore a + b = 5$

정답: ①
