## 문제
두 자연수 $a$, $b$ 에 대하여 함수 
$f(x) = \begin{cases} 2^{x+a} + b & (x \leq -8) \\ -3^{x-3} + 8 & (x > -8) \end{cases}$
이 다음 조건을 만족시킬 때, $a+b$ 의 값은?

집합 $\{ f(x) \mid x \leq k \}$ 의 원소 중 정수인 것의 개수가 $2$ 가 되도록 하는 모든 실수 $k$ 의 값의 범위는 $3 \leq k < 4$ 이다.

## 보기
① $11$   ② $13$   ③ $15$   ④ $17$   ⑤ $19$

## 풀이
함수 $y=2^{x+a}+b$ 는 증가함수이고, 함수 $y=2^{x+a}+b$ 의 그래프의 점근선은 $y=b$ 이다. 
$f(-8)=b+2^{a-8}$ 

> [!example] 📸 (그림/그래프)

함수 $y=-3^{x-3}+8$ 는 감소함수이고, 함수 $y=-3^{x-3}+8$ 의 그래프의 점근선은 $y=8$ 이다. 

$\lim_{x \to -8^{+}} f(x) = \lim_{x \to -8^{+}} (-3^{x-3}+8) = 8-3^{-11}$ 

$\therefore 7 < \lim_{x \to -8^{+}} f(x) < 8$ $\cdots \cdots$ ㉠ 

> [!example] 📸 (그림/그래프)

집합 $\{ f(x) \mid x \leq k \}$ 의 원소 중에서 정수인 것의 개수가 $2$ 인 실수 $k$ 의 값의 범위는 $3 \leq k < 4$ 이다. $\cdots \cdots$ ㉡ 

$f(3)=7$, $f(4)=5$ 
이때, $k \neq 4$ 이므로 $3 \leq k < 4$ 일 때, 집합 $\{ f(x) \mid x \leq k \}$ 에서 
$5 \notin \{ f(x) \mid x \leq k \}$ 

$3 < x < 4$ 에서 $f(x)=6$ 인 실수 $x$ 는 오직 하나이다. 
$f(\alpha) = 6$ ($3 < \alpha < 4$)이라 하자. 

$f(3)=7$, $f(\alpha)=6$ ($3 < \alpha < 4$)이므로 $3 \leq k < 4$ 일 때 집합 $\{ f(x) \mid x \leq k \}$ 에서 
$\{ 6, 7 \} \subset \{ f(x) \mid x \leq k \}$ 

㉡에서 $k$ 의 최솟값이 $3$ 이고, $f(3)=7$ 이므로 $x < 3$ 인 실수 $x$ 에 대하여 $f(x)$ 의 값이 정수인 것은 $6$ 뿐이다. 
㉠에 의하여 $-8 < x < 3$ 에서 $7 < f(x) < 8$ 이므로 $x \leq -8$ 에서 $f(x)=6$ 인 실수 $x$ 가 존재한다. 

$x \leq -8$ 에서 $f(x)$ 가 정수인 것은 $6$ 뿐이고, 함수 $y=f(x)$ 의 그래프의 점근선은 $y=b$ 이므로 
$b+1=6$ 
$\therefore b=5$ 

$6 \leq f(-8) < 7$, $6 \leq b+2^{a-8} < 7$ 

$6 \leq b+2^{a-8} < 7$ 에서 
$6 \leq 5+2^{a-8} < 7$, $1 \leq 2^{a-8} < 2$ 

$0 \leq a-8 < 1$ 이므로 $a=8$ 이다. 

$\therefore a+b=13$ 

> [!example] 📸 (그림/그래프)
