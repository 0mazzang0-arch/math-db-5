## 문제
최고차항의 계수가 $1$ 인 삼차함수 $f(x)$ 가 다음 조건을 만족시킨다.

(가) 함수 $f(x)$ 에 대하여 $f(k-1)f(k+1) < 0$ 을 만족시키는 정수 $k$ 는 존재하지 않는다.
(나) $f'(-\frac{1}{4}) = -\frac{1}{4}$, $f'(\frac{1}{4}) < 0$

이때, $f(8)$ 의 값을 구하시오.

## 보기


## 풀이
삼차함수 $f(x)$ 의 최고차항의 계수가 $1$ 이고 $f'(x) < 0$ 인 $x$ 가 존재하므로 함수 $f(x)$ 의 그래프는 극점을 갖는 개형이다. 
또 $f'(-\frac{1}{4}) < 0$, $f'(\frac{1}{4}) < 0$ 이므로 $f'(0) < 0$ 이다. 

(ⅰ) $f(0) > 0$ 일 때 
$f(-2) \ge 0$ 
$f(-2) \ne 0$ 이면 $f(-2) > 0$ 이므로 $f(-4) \ge 0$ 
$f(-4) \ne 0$ 이면 $f(-4) > 0$ 이므로 $f(-6) \ge 0$ 
$\vdots$ 
$\lim_{x \to -\infty} f(x) = -\infty$ 이므로 모든 자연수 $n$ 에 대하여 $f(-2n) > 0$ 일 수는 없다. 따라서 $f(-2n) = 0$ 인 자연수 $n$ 이 존재한다. 
$f'(0) < 0$ 이고 $f(-2n) = 0 < f(0)$ 이므로 $f'(2n) > 0$ 이어야 한다. 삼차함수의 증감과 개형을 생각해보면 $x = -2n$ 이 $x < 0$ 인 구간에서 함수 $f(x)$ 가 $x$ 축과 만나는 유일한 점임을 알 수 있다. 

> [!example] 📸 (그림/그래프)

$f(-2n-1) < 0 < f(-2n+1)$ 이므로 문제의 가정에 모순되므로 $f(0) > 0$ 는 참이 아니다. 

(ⅱ) $f(0) < 0$ 일 때 
$\lim_{x \to \infty} f(x) = \infty$ 이므로 (ⅰ)과 같은 방법으로 $f(2n) = 0$ 인 자연수 $n$ 이 존재함을 보일 수 있고, 같은 방법으로 모순을 이끌어 낼 수 있다. 
(ⅰ), (ⅱ)로부터 $f(0) = 0$ 이다. 

구간 $[-\frac{1}{4}, \frac{1}{4}]$ 에서 함수 $f(x)$ 가 감소하므로 $f(\frac{1}{4}) < 0 < f(-\frac{1}{4})$ 이다. 
$\lim_{x \to -\infty} f(x) = -\infty$, $\lim_{x \to \infty} f(x) = \infty$ 이므로 사잇값 정리에 의해서 함수 $f(x)$ 는 $x$ 축과 $x < 0$ 인 구간과 $x > 0$ 인 구간에서 각각 한 번씩 만난다. 
따라서 $f(x) = x(x - \alpha)(x - \gamma)$ ($\alpha < 0 < \gamma$)로 놓을 수 있다. 

> [!example] 📸 (그림/그래프)

$\alpha < -1$ 이면 $\alpha$ 보다 큰 정수들 중에서 가장 작은 정수 $m_{1}$ 이 존재하여 $m_{1}-1 \le \alpha < m_{1} \le -1$ 을 만족한다. 이때 $f(m_{1}-2) < 0 < f(m_{1})$ 이 되므로 모순이다. 
만일 $\gamma > 1$ 이면 $\gamma$ 보다 작은 정수들 중에서 가장 큰 정수 $m_{2}$ 가 존재하여 $1 \le m_{2} < \gamma \le m_{2}+1$ 을 만족한다. 이때 $f(m_{2}) < 0 < f(m_{2}+2)$ 가 되므로 역시 모순이다. 
$-1 \le \alpha < 0 < \gamma \le 1$ 
$-1 < \alpha < 0 < \gamma < 1$ 이면 $f(-1) < 0 < f(1)$ 이 되어 모순이므로 
$\alpha = -1$ 또는 $\gamma = 1$ 

(1) $\alpha = -1$ 일 때 
$f(x) = x(x+1)(x - \gamma) = x^{3} + (1 - \gamma)x^{2} - \gamma x$ 에서 
$f'(x) = 3x^{2} + 2(1 - \gamma)x - \gamma$ 
$f'(-\frac{1}{4}) = \frac{3}{16} - \frac{1}{2}(1 - \gamma) - \gamma = -\frac{5}{16} - \frac{1}{2}\gamma = -\frac{1}{4}$ 
$\gamma = -\frac{1}{8}$ 
$\gamma > 0$ 이어야 하므로 조건을 만족하지 않는다. 

(2) $\gamma = 1$ 일 때 
$f(x) = x(x - \alpha)(x-1) = x^{3} - (\alpha + 1)x^{2} + \alpha x$ 에서 
$f'(x) = 3x^{2} - 2(1 + \alpha)x + \alpha$ 
$f'(-\frac{1}{4}) = \frac{3}{16} + \frac{1}{2}(1 + \alpha) + \alpha = \frac{11}{16} + \frac{3}{2}\alpha = -\frac{1}{4}$ 
$\alpha = -\frac{5}{8}$ 

따라서 $f(x) = x(x-1)(x + \frac{5}{8})$ 이므로 
$f(8) = 8 \times 7 \times \frac{69}{8} = 483$

**[다른 풀이]**
삼차함수 $f(x)$ 의 최고차항의 계수가 $1$ 이고 $f'(x) < 0$ 인 $x$ 가 존재하므로 함수 $f(x)$ 의 그래프는 극점을 갖는 개형이다. 방정식 $f(x) = 0$ 의 실근이 $x = \alpha$ 하나뿐이면 $k-1 < \alpha < k+1$ 인 정수 $k$ 가 존재하고 $f(k-1) < 0 < f(k+1)$ 이므로 문제의 가정에 모순된다. 
방정식 $f(x) = 0$ 의 실근이 $x = \alpha$, $x = \beta$ ($\alpha < \beta$) 두 개뿐이라 하고 $x = \beta$ 가 중근이라 하자. 또 $\alpha$ 보다 작은 정수 중에서 가장 큰 정수를 $k-1$ 이라 하자. 그러면 $f(k-1) < 0$ 이다. 

> [!example] 📸 (그림/그래프)

$f(k-1)f(k+1) < 0$ 을 만족시키는 정수 $k$ 가 존재하지 않기 위해서는 $\beta = k+1$, $\alpha = k$ 이어야 한다. (그렇지 않으면 $f(k)f(k+2) < 0$ 이 된다.) 
이때 $f'(-\frac{1}{4}) < 0$, $f'(\frac{1}{4}) < 0$ 이므로 $f'(0) < 0$ 이고 따라서 $k < 0 < k+1$ 이다. 
$k$ 는 정수이므로 이것은 불가능하다. 

$x = \alpha$ 가 중근인 경우도 같은 방법으로 모순임을 보일 수 있다. 그러므로 방정식 $f(x) = 0$ 은 서로 다른 세 실근을 가져야 한다. 이제 세 실근을 $\alpha$, $\beta$, $\gamma$ ($\alpha < \beta < \gamma$)라 하자. 

(ⅰ) $\alpha$ 가 정수일 때 
$\alpha - 1$, $\alpha + 1$ 은 모두 정수이고 $f(\alpha - 1) < 0$ 이므로 $f(\alpha + 1) \le 0$ 이어야 한다. 
$\therefore \beta \le \alpha + 1 \le \gamma$ 
$\gamma \ge \alpha + 2$ 이면 $\gamma$ 보다 작은 정수들 중에서 가장 큰 정수 $m$ 에 대하여 $\alpha + 1 \le m < \gamma$ 이고 $f(m) < 0 < f(m+2)$ 이므로 문제의 가정을 만족하지 않는다. 
$\therefore \gamma < \alpha + 2$ 
$\beta < \alpha + 1 < \gamma$ 이면 $f(\alpha + 1) < 0 < f(\alpha + 2)$ 이므로 마찬가지로 모순이다. 
$\therefore \beta = \alpha + 1$ 또는 $\gamma = \alpha + 1$ 

> [!example] 📸 (그림/그래프)

$\gamma = \alpha + 1$ 이면 $f'(0) < 0$ 이므로 $\alpha < 0 < \gamma = \alpha + 1$ 이 되어 불가능하다. 따라서 $\beta = \alpha + 1$ 이고 $f'(0) < 0$ 으로부터 $\beta = \alpha + 1 = 0$ 이다. 즉 $f(x) = x(x+1)(x - \gamma)$ 
$f(x) = x(x+1)(x - \gamma) = x^{3} + (1 - \gamma)x^{2} - \gamma x$ 
$f'(x) = 3x^{2} + 2(1 - \gamma)x - \gamma$ 
$f'(-\frac{1}{4}) = \frac{3}{16} - \frac{1}{2}(1 - \gamma) - \gamma = -\frac{5}{16} - \frac{1}{2}\gamma = -\frac{1}{4}$ 
$\gamma = -\frac{1}{8}$ 
$\gamma > 0$ 이어야 하므로 모순이다. 

(ⅱ) $\alpha$ 가 정수가 아닐 때 
$\alpha$ 보다 작은 정수 중에서 가장 큰 정수를 $k-1$ 이라 하자. 그러면 $f(k-1) < 0$ 이다. 

> [!example] 📸 (그림/그래프)

$\beta \le k+1 \le \gamma$ 이어야 한다. 
$f(k-2) < 0$ 이므로 $\beta \le k \le \gamma$ 이어야 한다. 
$\gamma$ 가 정수가 아니면 $k+1 \le n-1 < \gamma < n+1$ 인 정수 $n$ 이 존재한다. $f(n-1) < 0 < f(n+1)$ 이므로 이것은 가정에 모순된다. 따라서 $\gamma$ 는 정수이다. 
$\gamma > k+1$ 이면 $\beta \le k < \gamma - 1$ 에서 $f(\gamma - 1) < 0 < f(\gamma + 1)$ 이므로 모순이다. 
$\therefore \gamma = k+1$ 
$\beta \ne k$ 이면 $f(k) < 0 < f(k+2)$ 이므로 가정에 모순된다. 
$\therefore \beta = k$ 
$f'(k-1) > 0$, $f'(k+1) > 0$, $f'(0) < 0$ 이므로 
$\therefore \beta = k = 0$ 
$\therefore f(x) = x(x-1)(x - \alpha) = x^{3} - (1 + \alpha)x^{2} + \alpha x$ 
$f'(x) = 3x^{2} - 2(1 + \alpha)x + \alpha$ 에서 
$f'(-\frac{1}{4}) = \frac{3}{16} + \frac{1}{2}(1 + \alpha) + \alpha = \frac{11}{16} + \frac{3}{2}\alpha = -\frac{1}{4}$ 
$\alpha = -\frac{5}{8}$ 

이상에서 $f(x) = x(x-1)(x + \frac{5}{8})$ 이므로 
$f(8) = 8 \times 7 \times \frac{69}{8} = 483$
