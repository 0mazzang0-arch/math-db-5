## 문제

다음은 $2006$보다 작은 자연수 $n$에 대하여 $2006$과 $n$이 서로소이면 $2006$과 $2006-n$도 서로소이다를 증명한 것이다.

$$
\begin{array}{|l|}
\hline
2006\text{과 } 2006-n\text{이 (가)라고 가정하면} \\
2006\text{과 } 2006-n\text{은 (나) 이상의 공약수가 존재한다.} \\
2006=at, 2006-n=bt \text{ (} a, b, t \text{는 자연수, } t \geq 2 \text{)라 하면} \\
n = ( \text{(다)} ) \times t \text{이므로 } t \text{는 } 2006 \text{과 } n \text{의 공약수이다.} \\
\text{이것은 } 2006 \text{과 } n \text{이 서로소이다에 모순이므로} \\
2006 \text{과 } 2006-n \text{도 서로소이다.} \\
\hline
\end{array}
$$

위의 증명에서 $(가) \sim (다)$를 바르게 짝지은 것은?

## 보기

① $(가): 서로소이다., (나): 2, (다): a+b$   ② $(가): 서로소이다., (나): 3, (다): a-b$   ③ $(가): 서로소가 아니다., (나): 2, (다): a+b$   ④ $(가): 서로소가 아니다., (나): 2, (다): a-b$   ⑤ $(가): 서로소가 아니다., (나): 3, (다): a+b$

## 풀이

정답: ④

귀류법을 이용하여 증명하고자 하므로 $2006$과 $2006-n$이 서로소가 아니다라고 가정하면 $2006$과 $2006-n$은 $2$ 이상의 공약수가 존재한다.

$2006=at, 2006-n=bt$ ($a, b, t$는 자연수, $t \geq 2$)라 하면
$n = 2006 - (2006-n) = at - bt = (a-b) \times t$이므로 $t$는 $2006$과 $n$의 공약수이다.
이것은 $2006$과 $n$이 서로소이다에 모순이므로 $2006$과 $2006-n$도 서로소이다.

따라서 
$(가): \text{서로소가 아니다.}$
$(나): 2$
$(다): a-b$
이다.
