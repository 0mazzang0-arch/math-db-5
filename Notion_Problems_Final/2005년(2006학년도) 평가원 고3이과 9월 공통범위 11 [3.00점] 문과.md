## 문제
다음은 자연수 $n$에 대하여 $\log_{2} n$이 유리수이면 $n$을 $n = 2^{k}$ (단, $k$는 $k \ge 0$인 정수)의 꼴로 나타낼 수 있음을 증명한 것이다.

$$
\begin{array}{|l|}
\hline
\text{자연수 } n \text{에 대하여 } \log_{2} n \text{이 유리수라고 하자.} \\
\\
n \text{이 자연수이므로 } n = 2^{k} \cdot m \text{을 만족시키는 } k \ge 0 \text{인 정수 } k \text{와 홀수인 자연수 } m \text{이 존재한다. 그러면} \\
\log_{2} n = (가) \\
\text{따라서 } \log_{2} n \text{이 유리수이면 } \log_{2} m \text{도 유리수이어야 하므로} \\
\log_{2} m = \frac{q}{p} \text{ (단, } p \text{는 자연수이고 } q \text{는 정수)로 놓을 수 있다. 그러면} \\
(나) \text{에서 } m \text{이 홀수이므로 } m^{p} \text{은 홀수이다. 따라서 } 2^{q} \text{도 홀수이어야 하므로} \\
(다) \text{이고 } m = 1 \text{이다. 따라서 } n \text{을 } n = 2^{k} \text{ (단, } k \text{는 } k \ge 0 \text{인 정수)의 꼴로 나타낼 수 있다.} \\
\hline
\end{array}
$$

위의 증명에서 $(가)$, $(나)$, $(다)$에 알맞은 것은?

## 보기
① $(가): k \log_{2} m, (나): m^{q} = 2^{p}, (다): q = 1$   ② $(가): k \log_{2} m, (나): m^{p} = 2^{q}, (다): q = 1$   ③ $(가): k + \log_{2} m, (나): m^{q} = 2^{p}, (다): q = 0$   ④ $(가): k + \log_{2} m, (나): m^{p} = 2^{q}, (다): q = 1$   ⑤ $(가): k + \log_{2} m, (나): m^{p} = 2^{q}, (다): q = 0$

## 풀이
$n = 2^{k} \cdot m$ 이므로 양변에 밑이 $2$인 로그를 취하면
$\log_{2} n = \log_{2} (2^{k} \cdot m) = \log_{2} 2^{k} + \log_{2} m = k + \log_{2} m$
따라서 $(가): k + \log_{2} m$

$\log_{2} m = \frac{q}{p}$ 에서 로그의 정의에 의해 $m = 2^{\frac{q}{p}}$ 이다.
양변을 $p$제곱하면 $m^{p} = (2^{\frac{q}{p}})^{p} = 2^{q}$
따라서 $(나): m^{p} = 2^{q}$

$m$이 홀수이므로 $m$의 거듭제곱인 $m^{p}$도 홀수이다.
$m^{p} = 2^{q}$ 에서 $2^{q}$이 홀수가 되기 위해서는 지수 $q$가 $0$이어야 한다. ($2^{0} = 1$로 유일한 홀수 거듭제곱)
따라서 $(다): q = 0$

그러므로 정답은 ⑤이다.
