## 문제
모든 항이 $0$이 아닌 등차수열 $\{a_{n}\}$과 $1$보다 큰 자연수 $m$에 대하여 다항식 
$P(x)=a_{m+1} x^{m} + a_{m} x^{m-1} + a_{m-1} x^{m-2} + \dots + a_{2} x + a_{1}$
이 있다. 
$P(1)=5P(-1)$을 만족시키는 다항식 $P(x)$에서 자연수 $m$의 값을 $k$라 하자. 
다항식 $a_{k+1} x^{k} + a_{k} x^{k-1} + \dots + a_{2} x + a_{1}$이 $x+2$로 나누어떨어질 때, $\frac{a_{1}}{a_{k+1}}$의 값을 구하시오.

## 풀이
등차수열 $\{a_{n}\}$의 첫째항을 $a$, 공차를 $d$라 하자. 

$P(x)=a_{m+1} x^{m} + a_{m} x^{m-1} + \dots + a_{2} x + a_{1}$에서 
$P(1)=a_{m+1} + a_{m} + \dots + a_{2} + a_{1}$이고 이는 등차수열 $\{a_{n}\}$의 첫째항부터 제 $(m+1)$항까지의 합이므로 
$P(1)= \frac{(m+1)(2a+md)}{2}$ 

$P(-1)=a_{m+1} (-1)^{m} + a_{m} (-1)^{m-1} + \dots - a_{2} + a_{1}$ 

(ⅰ) $m$이 홀수일 때 
$P(-1)=-a_{m+1} + a_{m} - \dots - a_{2} + a_{1}$ 
$=-(a_{m+1} - a_{m}) - \dots - (a_{2} - a_{1})$ 
$= - \frac{m+1}{2} d$ 

$P(1)=5P(-1)$에서 
$\frac{(m+1)(2a+md)}{2} = 5 \times \left( - \frac{m+1}{2} d \right)$ 
$m+1 \neq 0$이므로 
$2a+md = -5d$ 
$a + \frac{m+5}{2} d = 0$ 

$m$은 홀수이므로 $\frac{m+5}{2}$는 자연수이고 $a+ \frac{m+5}{2} d$는 수열 $\{a_{n}\}$의 제 $(\frac{m+7}{2})$항이다. 
$a_{\frac{m+7}{2}} = a + (\frac{m+7}{2} - 1) d = a + \frac{m+5}{2} d = 0$ 
$\{a_n\}$의 모든 항이 $0$이 아니라는 조건에 모순이다. 따라서 $P(1)=5P(-1)$을 만족시키는 홀수 $m$은 존재하지 않는다. 

(ⅱ) $m$이 짝수일 때 
$P(-1)=a_{m+1} - a_{m} + \dots - a_{2} + a_{1}$ 
$=(a_{m+1} - a_{m}) + \dots + (a_{3} - a_{2}) + a_{1}$ 
$=a+ \frac{m}{2} d = \frac{2a+md}{2}$ 

$P(1)=5P(-1)$에서 
$\frac{(m+1)(2a+md)}{2} = 5 \times \frac{2a+md}{2} \cdots (\ast)$ 

$m$은 짝수이므로 $\frac{m}{2}$은 자연수이고 $a + \frac{m}{2} d$는 수열 $\{a_{n}\}$의 제 $(\frac{m}{2} + 1)$항이다. 
$\{a_{n}\}$의 모든 항이 $0$이 아니므로 $a_{\frac{m}{2} + 1} = a + \frac{m}{2} d = \frac{2a + md}{2} \neq 0$ 

$(\ast)$에서 $m+1=5$이므로 $m=4$ 

(ⅰ), (ⅱ)에서 가능한 자연수 $m$은 $4$뿐이므로 $k=4$이다. 

다항식 $a_{5} x^{4} + a_{4} x^{3} + a_{3} x^{2} + a_{2} x + a_{1}$이 $x+2$로 나누어떨어질 때, 인수정리에 의하여 
$a_{5} (-2)^{4} + a_{4} (-2)^{3} + a_{3} (-2)^{2} + a_{2} (-2) + a_{1} = 0$ 

$a_{n} = a+(n-1)d$이므로 
$(a+4d)(-2)^{4} + (a+3d)(-2)^{3} + (a+2d)(-2)^{2} + (a+d)(-2) + a = 0$ 
$16 (a+4d) - 8(a+3d) + 4(a+2d) - 2(a+d) + a = 0$ 
$11a+46d = 0 \Rightarrow d = - \frac{11}{46} a$ 

모든 자연수 $n$에 대하여 $a_{n} \neq 0$이므로 
$\frac{a_{1}}{a_{k+1}} = \frac{a}{a_{5}} = \frac{a}{a+4d}$ 
$= \frac{a}{a - \frac{22}{23} a} = \frac{1}{\frac{1}{23}} = 23$

정답: $23$
