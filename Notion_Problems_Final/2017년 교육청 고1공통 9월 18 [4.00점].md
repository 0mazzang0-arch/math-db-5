## 문제
양수 $k$에 대하여 이차함수 $y=- \frac{x^{2}}{2} +k$의 그래프와 직선 $y=mx$가 만나는 서로 다른 두 점을 각각 $A$, $B$라 하자. 다음은 실수 $m$의 값에 관계없이 $\frac{1}{\overline{OA}} + \frac{1}{\overline{OB}}$이 일정한 값을 갖기 위한 $k$의 값을 구하는 과정이다. ($O$는 원점이다.)

두 점 $A$, $B$의 $x$좌표를 각각 $\alpha$, $\beta$ ($\alpha < 0 < \beta$)라 하면 $\alpha$, $\beta$는 이차방정식 $- \frac{x^{2}}{2} +k=mx$의 근이므로 이차방정식의 근과 계수의 관계에 의해
$\alpha + \beta = -2m$, $\alpha \beta = -2k$
두 점 $A$, $B$는 직선 $y=mx$ 위의 점이므로
$A(\alpha, m\alpha)$, $B(\beta, m\beta)$
$\overline{OA} = -\alpha \times \text{(가)}$, $\overline{OB} = \beta \times \text{(가)}$
$\frac{1}{\overline{OA}} + \frac{1}{\overline{OB}} = \frac{1}{-\alpha \times \text{(가)}} + \frac{1}{\beta \times \text{(가)}} = \frac{\alpha - \beta}{\alpha \beta \times \text{(가)}} = \frac{-\sqrt{4m^{2} + \text{(나)}}}{-2k \times \text{(가)}}$
실수 $m$의 값에 관계없이 $\frac{1}{\overline{OA}} + \frac{1}{\overline{OB}}$이 갖는 일정한 값을 $t$라 하자.
$t^{2} = \frac{4m^{2} + \text{(나)}}{(2k \times \text{(가)})^{2}}$이므로 이를 정리하면
$4(1-k^{2}t^{2})m^{2} + 4(2k-k^{2}t^{2}) = 0 \cdots \cdots$ ㉠
따라서 ㉠이 $m$에 대한 항등식이므로 $k = \text{(다)}$이다.
이때 $\frac{1}{\overline{OA}} + \frac{1}{\overline{OB}} = \frac{1}{k}$이다.

위의 (가), (나)에 알맞은 식을 각각 $f(m)$, $g(k)$라 하고 (다)에 알맞은 수를 $p$라 할 때, $f(p) \times g(p)$의 값은?

## 보기
① $2$   ② $2\sqrt{5}$   ③ $10$   ④ $10\sqrt{5}$   ⑤ $50$

## 풀이
두 점 $A$, $B$의 $x$좌표를 각각 $\alpha$, $\beta$ ($\alpha < 0 < \beta$)라 하면 $\alpha$, $\beta$는 이차방정식 $- \frac{x^{2}}{2} +k=mx$의 근이므로 이차방정식의 근과 계수의 관계에 의해
$\alpha + \beta = -2m$, $\alpha \beta = -2k$
두 점 $A$, $B$는 직선 $y=mx$ 위의 점이므로
$A(\alpha, m\alpha)$, $B(\beta, m\beta)$

$\overline{OA} = \sqrt{\alpha^{2} + (m\alpha)^{2}} = |\alpha|\sqrt{1+m^{2}} = -\alpha \sqrt{1+m^{2}}$
$\overline{OB} = \sqrt{\beta^{2} + (m\beta)^{2}} = |\beta|\sqrt{1+m^{2}} = \beta \sqrt{1+m^{2}}$

$\frac{1}{\overline{OA}} + \frac{1}{\overline{OB}} = \frac{1}{-\alpha \sqrt{1+m^{2}}} + \frac{1}{\beta \sqrt{1+m^{2}}} = \frac{\beta - \alpha}{-\alpha \beta \sqrt{1+m^{2}}}$
이때 $(\beta - \alpha)^{2} = (\alpha + \beta)^{2} - 4\alpha\beta = (-2m)^{2} - 4(-2k) = 4m^{2} + 8k$이므로
$\frac{1}{\overline{OA}} + \frac{1}{\overline{OB}} = \frac{\sqrt{4m^{2} + 8k}}{2k \sqrt{1+m^{2}}}$

실수 $m$의 값에 관계없이 $\frac{1}{\overline{OA}} + \frac{1}{\overline{OB}}$이 갖는 일정한 값을 $t$라 하자.
$t^{2} = \frac{4m^{2} + 8k}{4k^{2}(1+m^{2})}$
$4k^{2}t^{2} + 4k^{2}t^{2}m^{2} = 4m^{2} + 8k$
$4(k^{2}t^{2} - 1)m^{2} + 4(k^{2}t^{2} - 2k) = 0$ (또는 $4(1-k^{2}t^{2})m^{2} + 4(2k-k^{2}t^{2}) = 0$) $\cdots \cdots$ ㉠

㉠이 $m$에 대한 항등식이므로
$k^{2}t^{2} - 1 = 0 \Rightarrow k^{2}t^{2} = 1$
$k^{2}t^{2} - 2k = 0 \Rightarrow 1 - 2k = 0$
$\therefore k = \frac{1}{2}$

따라서 $f(m) = \sqrt{1+m^{2}}$, $g(k) = 8k$, $p = \frac{1}{2}$이다.
$f(p) \times g(p) = f\left(\frac{1}{2}\right) \times g\left(\frac{1}{2}\right) = \sqrt{1+\left(\frac{1}{2}\right)^{2}} \times \left(8 \times \frac{1}{2}\right) = \sqrt{\frac{5}{4}} \times 4 = \frac{\sqrt{5}}{2} \times 4 = 2\sqrt{5}$
