## 문제
양수 $t$에 대하여 $\log t$의 정수부분과 소수부분을 각각 $f(t)$, $g(t)$라 하자. 자연수 $n$에 대하여 $f(t) = 9n \left\{ g(t) - \frac{1}{3} \right\}^{2} - n$을 만족시키는 모든 $f(t)$의 합을 $a_{n}$이라 할 때, $\lim_{n \to \infty} \frac{a_{n}}{n^{2}}$의 값은?

## 보기
① $4$   ② $\frac{9}{2}$   ③ $5$   ④ $\frac{11}{2}$   ⑤ $6$

## 풀이
$\log t = f(t) + g(t)$ 이고, $f(t)$는 정수, $0 \le g(t) < 1$이므로
$-\frac{1}{3} \le g(t) - \frac{1}{3} < \frac{2}{3}$이다.
각 변을 제곱하면
$0 \le \left\{ g(t) - \frac{1}{3} \right\}^{2} < \frac{4}{9}$
$0 \le 9n \left\{ g(t) - \frac{1}{3} \right\}^{2} < 4n$
양변에 $n$을 빼면
$-n \le 9n \left\{ g(t) - \frac{1}{3} \right\}^{2} - n < 3n$
즉, $-n \le f(t) < 3n$이고, $f(t)$는 정수이므로 $f(t)$의 값은
$-n, -n+1, -n+2, \dots, 0, 1, 2, \dots, 3n-1$이다.

$a_{n}$은 위 정수들의 합이므로
$a_{n} = (-n) + (-n+1) + \dots + (-1) + 0 + 1 + 2 + \dots + n + (n+1) + (n+2) + \dots + (3n-1)$
$-n$부터 $n$까지의 합은 $0$이므로
$a_{n} = (n+1) + (n+2) + \dots + (3n-1)$
항의 개수는 $(3n-1) - (n+1) + 1 = 2n-1$개이다.
$a_{n} = \frac{(2n-1) \{ (n+1) + (3n-1) \}}{2}$
$a_{n} = \frac{(2n-1)(4n)}{2}$
$a_{n} = 2n(2n-1) = 4n^{2} - 2n$

따라서 구하고자 하는 극한값은
$\therefore \lim_{n \to \infty} \frac{a_{n}}{n^{2}} = \lim_{n \to \infty} \frac{2n(2n-1)}{n^{2}} = \lim_{n \to \infty} \frac{4n^{2}-2n}{n^{2}} = 4$

정답: ①
