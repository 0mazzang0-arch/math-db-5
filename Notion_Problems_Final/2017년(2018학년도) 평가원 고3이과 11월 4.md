## 문제
두 사건 $A$와 $B$는 서로 독립이고 $P(A) = \frac{2}{3}$, $P(A \cup B) = \frac{5}{6}$일 때, $P(B)$의 값은?

## 보기
① $\frac{1}{3}$   ② $\frac{5}{12}$   ③ $\frac{1}{2}$   ④ $\frac{7}{12}$   ⑤ $\frac{2}{3}$

## 풀이
$P(A \cup B) = \frac{5}{6}$에서
$P(A) + P(B) - P(A \cap B) = \frac{5}{6}$ $\dots$ ㉠
두 사건 $A$와 $B$가 서로 독립이므로
$P(A \cap B) = P(A)P(B)$ $\dots$ ㉡
㉠, ㉡에서
$P(A) + P(B) - P(A)P(B) = \frac{5}{6}$
이때, $P(A) = \frac{2}{3}$이므로
$\frac{2}{3} + P(B) - \frac{2}{3}P(B) = \frac{5}{6}$
$\frac{1}{3}P(B) = \frac{1}{6}$
따라서 $P(B) = \frac{1}{2}$이다.

정답: ③
