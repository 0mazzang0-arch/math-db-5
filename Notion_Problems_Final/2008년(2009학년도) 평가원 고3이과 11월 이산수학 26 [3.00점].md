## 문제
$0 \le x < 2\pi$ 일 때, 방정식 $\sin 2x = 2\cos x - 2\cos^2 x$ 를 만족시키는 서로 다른 모든 $x$ 의 값의 합은?

## 보기
① $\pi$   ② $\frac{5}{4}\pi$   ③ $\frac{3}{2}\pi$   ④ $\frac{7}{4}\pi$   ⑤ $2\pi$

## 풀이
$\sin 2x = 2\sin x \cos x$ 이므로 주어진 방정식은
$2\sin x \cos x = 2\cos x - 2\cos^2 x$
$\cos x (\sin x + \cos x - 1) = 0$
$\therefore \cos x = 0$ 또는 $\sin x + \cos x = 1$

(ⅰ) $\cos x = 0$ 에서
$x = \frac{\pi}{2}$ 또는 $x = \frac{3}{2}\pi$

(ⅱ) $\sin x + \cos x = 1$ 에서
$\sqrt{2} \sin \left( x + \frac{\pi}{4} \right) = 1$
$\leftrightarrow \sin \left( x + \frac{\pi}{4} \right) = \frac{1}{\sqrt{2}}$
$x + \frac{\pi}{4} = \frac{\pi}{4}$ 또는 $x + \frac{\pi}{4} = \frac{3\pi}{4}$
$x = 0$ 또는 $x = \frac{\pi}{2}$

따라서 구하는 서로 다른 모든 $x$ 의 값의 합은
$0 + \frac{\pi}{2} + \frac{3}{2}\pi = 2\pi$
