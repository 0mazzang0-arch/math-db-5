## 문제
양수 $x$ 에 대하여 $\log x$ 의 소수부분을 $f(x)$ 라 하자. $\frac{1}{100} \le x \le 100$ 일 때, 방정식 $f(x) = \log 3$ 의 서로 다른 실근의 개수를 구하시오.

## 보기

## 풀이
양수 $x$ 에 대하여 $\log x$ 의 소수부분이 $f(x)$ 이므로 $f(x) = \log 3$ 에서 $\log x$ 의 소수부분은 $\log 3$ 이다. 
따라서 $\log x = n + \log 3$ ($n$ 은 정수) $\cdots \cdots$ ㉠ 

$\frac{1}{100} \le x \le 100$ 일 때, $-2 \le \log x \le 2$ 
이것을 ㉠에 대입하면 

$-2 \le n + \log 3 \le 2$ 

$-2 - \log 3 \le n \le 2 - \log 3$ 

$0 < \log 3 < 1$ 이므로 
$-3 < n < 2$ 

$\therefore n = -2, -1, 0, 1$ 
따라서 구하는 $x$ 의 개수는 $4$ 이다.

정답: $4$
