## 문제
양의 실수 $a$ 와 $b$ 에 대하여 집합 $A$ 와 $B$ 를 다음과 같이 정의한다. 
$$A = \{ x | (x-a)(x+a) \le 0 \}$$
$$B = \{ x | |x-1| \le b \}$$
이때 $A \cap B = \emptyset$ 이기 위한 필요충분조건은?



> [!example] 📸 (그림/그래프)

## 보기
① $a-b < 1$   ② $a-b > 1$   ③ $a+b = 1$   ④ $a+b < 1$   ⑤ $a+b > 1$

## 풀이
두 집합 
$A = \{ x | -a \le x \le a \}$, 
$B = \{ x | 1-b \le x \le 1+b \}$ 에서 
$A \cap B = \emptyset$ 이므로 $2$ 가지 경우로 나누어 보면 
(ⅰ) $a < 1-b$ 이므로 $a+b < 1$ 
(ⅱ) $a$, $b$ 는 양의 실수이므로 항상 $1+b > -a$ 이다. 

따라서 (ⅰ), (ⅱ)에 의해서 $a+b < 1$ 이다. 

정답: ④
