## 문제
함수 $f(x) = 3 \sin \frac{\pi(x+a)}{2} + b$의 그래프가 그림과 같다. 두 양수 $a$, $b$에 대하여 $a \times b$의 최솟값을 구하시오.

## 풀이
함수 $f(x) = 3 \sin \frac{\pi(x+a)}{2} + b$에서 최댓값은 $3 + b = 5$이고 최솟값은 $-3 + b = -1$이므로 $b = 2$이다.
함수 $f(x) = 3 \sin \frac{\pi}{2}(x + a) + 2$의 그래프는 함수 $y = 3 \sin \frac{\pi}{2}x + 2$의 그래프를 $x$축의 방향으로 $-a$만큼 평행이동한 것이다.
모든 실수 $x$에 대하여 $f(x + p) = f(x)$를 만족하는 최소의 양수 $p$가 $4$이므로 함수 $f(x)$의 주기는 $4$이다.
함수 $f(x)$의 그래프는 함수 $y = 3 \sin \frac{\pi}{2}x + 2$의 그래프와 일치하고 함수 $f(x)$의 주기는 $4$이므로 $a = 4k$ ($k$는 정수)이다.
$a$가 양수일 때 $a$의 최솟값은 $4$이므로 $a \times b$의 최솟값은 $4 \times 2 = 8$이다.

정답: $8$
