## 문제
함수 $f(x) = 2 \cos (x - \frac{\pi}{6}) + 3 \sin x$의 최댓값은?

## 보기
① $4$   ② $\sqrt{17}$   ③ $3 \sqrt{2}$   ④ $\sqrt{19}$   ⑤ $2 \sqrt{5}$

## 풀이
$$f(x) = 2 \cos (x - \frac{\pi}{6}) + 3 \sin x$$
$$= 2 (\cos x \cos \frac{\pi}{6} + \sin x \sin \frac{\pi}{6}) + 3 \sin x$$
$$= 2 (\frac{\sqrt{3}}{2} \cos x + \frac{1}{2} \sin x) + 3 \sin x$$
$$= \sqrt{3} \cos x + \sin x + 3 \sin x$$
$$= 4 \sin x + \sqrt{3} \cos x$$
$$= \sqrt{19} (\frac{4}{\sqrt{19}} \sin x + \frac{\sqrt{3}}{\sqrt{19}} \cos x)$$
$$= \sqrt{19} \sin (x + \alpha)$$
(단, $\cos \alpha = \frac{4}{\sqrt{19}}$, $\sin \alpha = \frac{\sqrt{3}}{\sqrt{19}}$)

따라서 함수 $f(x)$의 최댓값은 $\sqrt{19}$이다.
