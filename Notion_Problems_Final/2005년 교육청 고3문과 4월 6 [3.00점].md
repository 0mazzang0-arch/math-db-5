## 문제
두 자연수 $x$, $y$에 대하여 $x^{8}$은 $25$자리의 수, $y^{5}$은 $16$자리의 수일 때, $xy$는 $n$자리의 수가 된다. 이 때, $n$의 값은?

## 보기
① $4$   ② $5$   ③ $6$   ④ $7$   ⑤ $8$

## 풀이
$x^{8}$이 $25$자리의 수이므로 
$24 \le \log_{10} x^{8} < 25$
$3 \le \log_{10} x < \frac{25}{8} = 3.125$ 

$y^{5}$이 $16$자리의 수이므로 
$15 \le \log_{10} y^{5} < 16$
$3 \le \log_{10} y < \frac{16}{5} = 3.2$ 

두 식을 더하면 
$3 + 3 \le \log_{10} x + \log_{10} y < 3.125 + 3.2$
$6 \le \log_{10} xy < 6.325$ 

따라서 $\log_{10} xy$의 정수 부분이 $6$이므로 $xy$는 $7$자리의 자연수이다. 
$n = 7$

정답: ④
