## 문제
집합 $A = \{ 1, 2, 3 \}$에 대하여 수열 $\{ a_n \}$은 집합 $A$의 원소로 이루어진 수열이다. 이 수열이 등식 
$$\sum_{n=1}^{\infty} \frac{a_n}{10^n} = \frac{104}{333}$$
를 만족시킬 때, $\sum_{n=1}^{\infty} \frac{a_n}{5^n} = \frac{q}{p}$이다. $p+q$의 값을 구하시오. (단, $p$, $q$는 서로소인 자연수이다.)

## 풀이
$\frac{104}{333} = \frac{312}{999} = 0.\dot{3}1\dot{2}$이므로 
$$\sum_{n=1}^{\infty} \frac{a_n}{10^n} = 0.\dot{3}1\dot{2}$$
수열 $\{ a_n \}$의 항은 $3, 1, 2$가 순환하는 형태이다.
$\therefore \sum_{n=1}^{\infty} \frac{a_n}{5^n} = 0.\dot{3}1\dot{2}_{(5)}$
$$= \frac{3}{5} + \frac{1}{5^2} + \frac{2}{5^3} + \frac{3}{5^4} + \cdots$$
$$= \frac{\frac{3}{5} + \frac{1}{5^2} + \frac{2}{5^3}}{1 - \frac{1}{5^3}} = \frac{3 \times 5^2 + 1 \times 5 + 2}{5^3 - 1}$$
$$= \frac{82}{124} = \frac{41}{62}$$
이때 $p = 62$, $q = 41$이며, 두 수는 서로소인 자연수이다.
따라서 $p+q = 62 + 41 = 103$

정답: $103$
