## 문제

그림과 같이 이차함수 $y = x^{2} - ( a+4 ) x+3a+3$의 그래프가 $x$축과 만나는 서로 다른 두 점을 각각 $A$, $B$라 하고, $y$축과 만나는 점을 $C$라 하자. 삼각형 $ABC$의 넓이의 최댓값은? (단, $0 < a < 2$)

> [!example] 📸 (그림/그래프)

## 보기

① $\frac{13}{4}$   ② $\frac{27}{8}$   ③ $\frac{7}{2}$   ④ $\frac{29}{8}$   ⑤ $\frac{15}{4}$

## 풀이

이차방정식 $x^{2} - ( a+4 ) x+3a+3 = ( x-3 ) \{ x- ( a+1 ) \} = 0$의 근은 $x=3$, $x=a+1$이다. 

$0<a<2$이므로 $A( a+1, 0 )$, $B( 3, 0 )$, $C( 0, 3a+3 )$이다. 

삼각형 $ABC$의 밑변의 길이 $\overline{AB} = 3 - (a+1) = 2-a$이고, 높이 $\overline{OC} = 3a+3$이다. 

삼각형 $ABC$의 넓이는 
$\frac{1}{2} ( 2-a ) ( 3a+3 )$
$= -\frac{3}{2} ( a-2 ) ( a+1 )$
$= -\frac{3}{2} ( a^{2} -a-2 )$
$= -\frac{3}{2} ( a- \frac{1}{2} )^{2} + \frac{27}{8}$
이다. 

따라서 $0<a<2$에서 삼각형 $ABC$의 넓이의 최댓값은 $a= \frac{1}{2}$일 때 $\frac{27}{8}$이다. 

정답: ②
