## 문제
$a$, $b$, $c$가 서로 다른 양수일 때, 다음 중에서 항상 성립하는 것을 모두 고르면?

ㄱ. $a + \frac{1}{a} > 2$
ㄴ. $a^2 + b^2 + c^2 > ab + bc + ca$
ㄷ. $|a + b| > |a| + |b|$

## 보기
① ㄱ   ② ㄴ   ③ ㄷ   ④ ㄱ, ㄴ   ⑤ ㄴ, ㄷ

## 풀이
ㄱ. 산술·기하평균의 관계에 의하여 
$$a + \frac{1}{a} \ge 2 \sqrt{a \times \frac{1}{a}} = 2$$
이때 등호는 $a = \frac{1}{a}$, 즉 $a = 1$일 때 성립한다. $a, b, c$가 서로 다른 양수일 때 $a = 1$일 수 있으므로 $a + \frac{1}{a} > 2$가 항상 성립하는 것은 아니다.

ㄴ. $a^2 + b^2 + c^2 - (ab + bc + ca)$
$= \frac{1}{2} \{ (a - b)^2 + (b - c)^2 + (c - a)^2 \}$
$a, b, c$가 서로 다른 양수이므로 $a \neq b, b \neq c, c \neq a$이다.
따라서 $\frac{1}{2} \{ (a - b)^2 + (b - c)^2 + (c - a)^2 \} > 0$이므로
$\therefore a^2 + b^2 + c^2 > ab + bc + ca$

ㄷ. $|a + b|^2 - (|a| + |b|)^2 = 2(ab - |ab|)$
$a, b$가 양수이므로 $ab = |ab|$이다.
즉, $2(ab - |ab|) = 2(ab - ab) = 0$
$\therefore |a + b| = |a| + |b|$
따라서 $|a + b| > |a| + |b|$는 성립하지 않는다.

따라서 항상 성립하는 것은 ㄴ뿐이다.
