## 문제
반지름의 길이가 $\sqrt{3}$인 원 $C$에 내접하는 삼각형 $ABC$에 대하여 $\angle BAC$의 이등분선이 원 $C$와 만나는 점 중 $A$가 아닌 점을 $D$라 하고, 두 선분 $BC$, $AD$의 교점을 $E$라 하자. $\overline{BD} = \sqrt{3}$일 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?



> [!example] 📸 (그림/그래프)

## 보기
ㄱ. $\sin ( \angle DBE ) = \frac{1}{2}$
ㄴ. $\overline{AB}^{2} + \overline{AC}^{2} = \overline{AB} \times \overline{AC} + 9$
ㄷ. 삼각형 $ABC$의 넓이가 삼각형 $BDE$의 넓이의 $4$배가 되도록 하는 모든 $\overline{BE}$의 값의 합은 $\frac{9}{4}$이다.

① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
$\angle DAC = \angle BAD = \theta$라 하면 두 각 $\angle DAC$, $\angle DBC$가 모두 호 $CD$에 대한 원주각이므로 $\angle DBC = \theta$이다.

ㄱ. 삼각형 $ABD$에서 사인법칙에 의하여
$\frac{\overline{BD}}{\sin ( \angle BAD )} = 2 \sqrt{3}$
$\frac{\sqrt{3}}{\sin \theta} = 2 \sqrt{3}$
$\sin \theta = \frac{1}{2}$
따라서 $\sin ( \angle DBE ) = \sin ( \angle DBC ) = \sin \theta = \frac{1}{2}$ (참)

ㄴ. $0 < \angle BAC = 2 \theta < \pi$에서 $0 < \theta < \frac{\pi}{2}$이고, $\sin \theta = \frac{1}{2}$이므로 $\theta = \frac{\pi}{6}$이다.
$\angle BAC = 2 \times \frac{\pi}{6} = \frac{\pi}{3}$
삼각형 $ABC$에서 사인법칙에 의하여
$\frac{\overline{BC}}{\sin \frac{\pi}{3}} = 2 \sqrt{3}$
$\overline{BC} = 2 \sqrt{3} \times \frac{\sqrt{3}}{2} = 3$
삼각형 $ABC$에서 코사인법칙에 의하여
$\overline{BC}^{2} = \overline{AB}^{2} + \overline{AC}^{2} - 2 \times \overline{AB} \times \overline{AC} \times \cos \frac{\pi}{3}$
$3^{2} = \overline{AB}^{2} + \overline{AC}^{2} - 2 \times \overline{AB} \times \overline{AC} \times \frac{1}{2}$
$\overline{AB}^{2} + \overline{AC}^{2} = \overline{AB} \times \overline{AC} + 9 \cdots (*)$ (참)

ㄷ. $\overline{BE} = a$ ($0 < a < 3$)이라 하면 $\overline{CE} = \overline{BC} - \overline{BE} = 3 - a$이다.
삼각형 $ABC$에서 $AD$가 $\angle BAC$의 이등분선이므로 $\overline{AB} : \overline{AC} = \overline{BE} : \overline{CE}$에서
$\overline{AB} : \overline{AC} = a : ( 3 - a )$
비례상수 $k$ ($k > 0$)에 대하여 $\overline{AB} = ak$, $\overline{AC} = ( 3 - a )k$라 하면
삼각형 $ABC$의 넓이는 $\frac{1}{2} \times \overline{AB} \times \overline{AC} \times \sin \frac{\pi}{3} = \frac{\sqrt{3}}{4} a ( 3 - a ) k^{2}$이고,
삼각형 $BDE$의 넓이는 $\frac{1}{2} \times \overline{BD} \times \overline{BE} \times \sin \frac{\pi}{6} = \frac{1}{2} \times \sqrt{3} \times a \times \frac{1}{2} = \frac{\sqrt{3}}{4} a$이다.
삼각형 $ABC$의 넓이가 삼각형 $BDE$의 넓이의 $4$배이어야 하므로
$\frac{\sqrt{3}}{4} a ( 3 - a ) k^{2} = 4 \times \frac{\sqrt{3}}{4} a$
$( 3 - a ) k^{2} = 4 \cdots (**)$
또한 (ㄴ)의 식 $(*)$에 대입하면
$( ak )^{2} + \{ ( 3 - a ) k \}^{2} = ( ak )( 3 - a ) k + 9$
$k^{2} \{ a^{2} + ( 9 - 6a + a^{2} ) \} = k^{2} ( 3a - a^{2} ) + 9$
$k^{2} ( 3a^{2} - 9a + 9 ) = 9$
$k^{2} ( a^{2} - 3a + 3 ) = 3 \cdots (***)$
$(**)$과 $(***)$을 연립하여 $k^{2}$을 소거하면
$\frac{3 - a}{a^{2} - 3a + 3} = \frac{4}{3}$
$3 ( 3 - a ) = 4 ( a^{2} - 3a + 3 )$
$9 - 3a = 4a^{2} - 12a + 12$
$4a^{2} - 9a + 3 = 0$
이차방정식의 판별식 $D = 81 - 48 = 33 > 0$이고, 두 근의 합은 근과 계수의 관계에 의해 $\frac{9}{4}$이다. (참)

따라서 옳은 것은 ㄱ, ㄴ, ㄷ이다.
