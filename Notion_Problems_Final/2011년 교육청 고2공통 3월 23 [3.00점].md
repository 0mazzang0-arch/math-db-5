## 문제
실수 $x$ 에 대한 두 조건 
$p : -1 < x < a + 1$
$q : |x - 10| \geq 1$
에 대하여, 명제 $p \rightarrow q$ 가 참이 되도록 하는 $a$ 의 최댓값을 구하시오.

## 보기


## 풀이
[정답] $8$

조건 $p$, $q$ 의 진리집합을 각각 $P$, $Q$ 라 하면 
$P = \{ x | -1 < x < a + 1 \}$, 
$Q = \{ x | x \leq 9$ 또는 $x \geq 11 \}$ 

$P \subset Q$ 이므로 

> [!example] 📸 (그림/그래프)

$a + 1 \leq 9$ 이다. 

$\therefore a$ 의 최댓값은 $8$
