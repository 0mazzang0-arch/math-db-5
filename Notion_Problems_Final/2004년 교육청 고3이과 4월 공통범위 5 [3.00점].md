## 문제
다음 보기의 방정식 중 열린구간 $(0, 1)$에서 적어도 한 개의 실근을 갖는 것을 모두 고르면?

ㄱ. $\cos \pi x - x = 0$
ㄴ. $2^{x} + x - 2 = 0$
ㄷ. $\log_{2}(x + 1) + x - 1 = 0$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
[정답] ⑤ 

ㄱ. $f(x) = \cos \pi x - x$라 하면 
$f(0) = \cos 0 - 0 = 1$, $f(1) = \cos \pi - 1 = -2$이므로 
$f(0)f(1) = -2 < 0$ 

ㄴ. $g(x) = 2^{x} + x - 2$라 하면 
$g(0) = 2^{0} + 0 - 2 = -1$, $g(1) = 2^{1} + 1 - 2 = 1$이므로 
$g(0)g(1) = -1 < 0$ 

ㄷ. $h(x) = \log_{2}(x + 1) + x - 1$이라 하면 
$h(0) = \log_{2}1 + 0 - 1 = -1$, $h(1) = \log_{2}2 + 1 - 1 = 1$이므로 
$h(0)h(1) = -1 < 0$ 

중간값의 정리에 의하여 세 함수 $f(x) = 0, g(x) = 0, h(x) = 0$은 열린구간 $(0, 1)$에서 각각 적어도 한 개의 실근을 갖는다. 
따라서 열린구간 $(0, 1)$에서 적어도 한 개의 실근을 갖는 방정식은 ㄱ, ㄴ, ㄷ이다.
