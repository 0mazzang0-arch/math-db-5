## 문제
최고차항의 계수가 $1$이고 다음 조건을 만족시키는 모든 이차함수 $f(x)$에 대하여 $f(1)$의 값의 합을 구하시오.

$$ \lim_{x \to k} \frac{f(x)}{f'(x)f(x-4)} \text{의 값이 존재하지 않는 실수 } k \text{의 값은 } p \text{ (} p < 4 \text{) 와 } 4 \text{ 뿐이다.} $$

## 풀이
이차함수 $f(x)$가 $x$축과 만나지 않으면 $\displaystyle \lim_{x \to k} \frac{f(x)}{f'(x)f(x-4)}$의 값이 존재하지 않는 실수 $k$ 개수가 $1$이므로 조건을 만족시키지 않는다.
따라서 최고차항의 계수가 $1$인 이차함수 $f(x)$에 대하여 이차방정식 $f(x)=0$의 두 실근을 $\alpha$, $\beta$라 하면
$$ f(x) = (x-\alpha)(x-\beta) $$
로 놓을 수 있다.
$$ f'(x) = 2x-(\alpha+\beta) $$
$\displaystyle \lim_{x \to k} \frac{f(x)}{f'(x)f(x-4)}$의 값이 존재하지 않아야 하므로 분모가 $0$이 되거나 분자가 $0$이 되지 않는 지점을 찾아야 한다. 즉, $f'(x)=0$ 또는 $f(x-4)=0$이고 $f(x) \neq 0$인 경우이다.

$f'(x)=0$에서 $x = \frac{\alpha+\beta}{2}$
$f(x-4)=0$에서 $x-4=\alpha$ 또는 $x-4=\beta$
$\therefore x = \alpha+4$ 또는 $x = \beta+4$

따라서 분모를 $0$으로 만드는 $k$의 값의 후보는 $\frac{\alpha+\beta}{2}, \alpha+4, \beta+4$이다.
이때 극한값이 존재하지 않는 $k$의 값이 $p$와 $4$뿐이므로 이 두 값은 집합 $\left\{ \frac{\alpha+\beta}{2}, \alpha+4, \beta+4 \right\}$에 속하면서 분자를 $0$으로 만들지 않아야 한다.
분자가 $0$이 되는 경우는 $f(k)=0$이므로 $k=\alpha$ 또는 $k=\beta$이다.

(ⅰ) 방정식 $f(x)=0$이 서로 다른 두 실근을 갖는 경우
분모를 $0$으로 만드는 값의 집합을 $D$라고 하면
$$ D = \left\{ \frac{\alpha+\beta}{2}, \alpha+4, \beta+4 \right\} $$
분자를 $0$으로 만드는 값의 집합을 $N$이라 하면
$$ N = \{ \alpha, \beta \} $$
존재하지 않는 $k$ 값의 집합은 $D-N$이고 이 집합이 $\{ p, 4 \}$가 되어야 한다.

(a) $D$의 원소 중 하나가 $N$의 원소와 같은 경우
$\frac{\alpha+\beta}{2} = \alpha$ 또는 $\frac{\alpha+\beta}{2} = \beta$이면 $\alpha = \beta$이므로 모순이다.
$\alpha+4 = \alpha$ 또는 $\beta+4 = \beta$는 성립하지 않으므로 $\alpha+4 = \beta$ 또는 $\beta+4 = \alpha$이다.
$\beta = \alpha+4$라고 하면
$$ D = \left\{ \frac{\alpha + (\alpha+4)}{2}, \alpha+4, (\alpha+4)+4 \right\} = \{ \alpha+2, \alpha+4, \alpha+8 \} $$
$$ N = \{ \alpha, \alpha+4 \} $$
$\therefore D-N = \{ \alpha+2, \alpha+8 \} = \{ p, 4 \}$
이때 $p < 4$이므로 $\alpha+2=p$이고 $\alpha+8=4$이다.
$\alpha = -4$이고 $\beta = \alpha+4 = 0$이다.
$f(x) = x(x+4) = x^2+4x$이므로 $f(1) = 1+4 = 5$이다.

(b) $D$의 원소끼리 중복되는 경우
$\frac{\alpha+\beta}{2} = \alpha+4$이면 $\beta = \alpha+8$이다.
$$ D = \left\{ \frac{\alpha + (\alpha+8)}{2}, \alpha+4, (\alpha+8)+4 \right\} = \{ \alpha+4, \alpha+12 \} $$
$$ N = \{ \alpha, \alpha+8 \} $$
$D$와 $N$은 서로소이므로 $D-N = D = \{ \alpha+4, \alpha+12 \} = \{ p, 4 \}$
이때 $p < 4$이므로 $\alpha+4 = p, \alpha+12 = 4$이다.
$\therefore \alpha = -8, \beta = \alpha+8 = 0$이다.
따라서 $f(x) = x(x+8) = x^2+8x$이므로 $f(1) = 1+8 = 9$이다.

(ⅱ) 방정식 $f(x)=0$이 중근을 가질 때
$f(x) = (x-\alpha)^2$에서 $f'(x) = 2(x-\alpha)$
$$ D = \{ \alpha, \alpha+4 \}, N = \{ \alpha \} $$
$$ D-N = \{ \alpha+4 \} $$
$k$가 하나뿐이므로 문제의 조건을 만족하지 않는다.

(ⅰ), (ⅱ)에 의해 가능한 $f(x)$는 $f(x) = x^2+4x$ 또는 $f(x) = x^2+8x$이다.
그러므로 $f(1) = 5$ 또는 $f(1) = 9$이다.
따라서 모든 이차함수 $f(x)$에 대하여 $f(1)$의 값의 합은 $5+9=14$이다.
