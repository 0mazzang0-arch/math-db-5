## 문제
최고차항의 계수가 양수인 다항식 $f(x)$ 가 모든 실수 $x$ 에 대하여 
$\{f(x)\}^3 = 4x^2 f(x) + 8x^2 + 6x + 1$
을 만족시킬 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?

## 보기
ㄱ. 다항식 $f(x)$ 를 $x$ 로 나눈 나머지는 $1$ 이다.  
ㄴ. 다항식 $f(x)$ 의 최고차항의 계수는 $4$ 이다.  
ㄷ. 다항식 $\{f(x)\}^3$ 을 $x^2 - 1$ 로 나눈 나머지는 $14x + 13$ 이다.  

① ㄱ   ② ㄴ   ③ ㄱ, ㄷ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
[정답] ③

ㄱ. $f(0)^3 = 1$ 이므로 $f(0) = 1$ 이다.  
ㄴ. $f(x)$ 의 차수를 $n$ 이라 하면 좌변의 차수는 $3n$, 우변의 차수는 $n+2$ 이므로  
$3n = n + 2$  
$\therefore n = 1$  
$f(x) = ax + b$ 라 하면 좌변의 최고차항의 계수는 $a^3$, 우변의 최고차항의 계수는 $4a$ 이므로  
$a^3 = 4a$, $a(a^2 - 4) = 0$, $a(a + 2)(a - 2) = 0$  
$\therefore a = -2$ 또는 $a = 0$ 또는 $a = 2$  
$a > 0$ 이므로 $a = 2$  
$f(x)$ 의 최고차항의 계수는 $2$ 이다.  
ㄷ. ㄱ과 ㄴ에 의해 $f(x) = 2x + 1$ 이므로  
$\{f(x)\}^3$ 을 $x^2 - 1$ 로 나누었을 때의 몫을 $Q(x)$, 나머지를 $cx + d$ 라 하면  
$\{f(x)\}^3 = (x^2 - 1)Q(x) + cx + d$  
$x = 1$ 을 대입하면  
$\{f(1)\}^3 = c + d = 27$ $\cdots$ ㉠  
$x = -1$ 을 대입하면  
$\{f(-1)\}^3 = -c + d = -1$ $\cdots$ ㉡  
㉠, ㉡을 연립하면 $c = 14$, $d = 13$  
$\{f(x)\}^3$ 을 $x^2 - 1$ 로 나누었을 때의 나머지는 $14x + 13$ 이다.  
이상에서 옳은 것은 ㄱ, ㄷ이다.
