## 문제
실수 $x$ 에 대한 두 조건 $p$ , $q$ 가 다음과 같다.
$p : a < x < 5$
$q : x^{2} - x - 2 < 0$
$\sim p$ 가 $q$ 이기 위한 필요조건이 되도록 하는 정수 $a$ 의 최솟값은? (단, $a < 5$)

## 보기
① $-2$   ② $-1$   ③ $0$   ④ $1$   ⑤ $2$

## 풀이
조건 $p$ 의 진리집합을 $P$ 라 하면 $P = \{ x \mid a < x < 5 \}$ 이고 조건 $\sim p$ 의 진리집합은 $P^{c} = \{ x \mid x \le a \text{ 또는 } x \ge 5 \}$ 이다.
조건 $q$ 의 진리집합을 $Q$ 라 하면 $x^{2} - x - 2 < 0$ 에서 $(x - 2)(x + 1) < 0$ , $-1 < x < 2$ 이므로 $Q = \{ x \mid -1 < x < 2 \}$ 이다.
이때 $\sim p$ 가 $q$ 이기 위한 필요조건이 되려면 $Q \subset P^{c}$ 이어야 한다.
따라서 $2 \le a < 5$ 이므로 정수 $a$ 의 최솟값은 $2$ 이다.

정답: ⑤
