## 문제
$18$ 이하의 자연수 $k$ 에 대하여 두 집합
$$ A = \{ x \mid x \text{는 } k \text{의 양의 약수} \}, B = \{ 2, 5, 6 \} $$
이 있다. $n ( A \cap B ) = 2$ 일 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?

ㄱ. $A \cap B = \{ 2, 5 \}$ 이면 $k = 10$ 이다.
ㄴ. $A \cap B = \{ 5, 6 \}$ 을 만족하는 $k$ 가 존재한다.
ㄷ. 집합 $A - B$ 의 모든 원소의 합이 홀수가 되는 모든 $k$ 의 값의 합은 $28$ 이다.

## 보기
① ㄱ   ② ㄷ   ③ ㄱ, ㄷ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
ㄱ. $A \cap B = \{ 2, 5 \}$ 에서 $2 \in A, 5 \in A$
$2$ 와 $5$ 가 $k$ 의 약수이므로 $k$ 는 $10$ 의 배수이다.
$k$ 는 $18$ 이하의 자연수이므로 $k = 10$ (참)

ㄴ. $A \cap B = \{ 5, 6 \}$ 에서 $5 \in A, 6 \in A$
$5$ 와 $6$ 이 $k$ 의 약수이므로 $k$ 는 $30$ 의 배수이다.
$k$ 는 $18$ 이하의 자연수이므로 존재하지 않는다. (거짓)

ㄷ. (ⅰ) $A \cap B = \{ 2, 5 \}$ 일 때,
$k = 10$ 이고 $A = \{ 1, 2, 5, 10 \}$ 이므로
$A - B = \{ 1, 10 \}$ 의 모든 원소의 합은 $11$

(ⅱ) $A \cap B = \{ 2, 6 \}$ 일 때, $2 \in A, 6 \in A$
$2$ 와 $6$ 이 $k$ 의 약수이므로 $k$ 는 $6$ 의 배수이다.
$k$ 는 $18$ 이하의 자연수이므로 가능한 $k$ 는 $6, 12, 18$ 이다.
$k = 6$ 인 경우, $A = \{ 1, 2, 3, 6 \}$ 이므로 집합 $A - B = \{ 1, 3 \}$ 의 모든 원소의 합은 $4$
$k = 12$ 인 경우, $A = \{ 1, 2, 3, 4, 6, 12 \}$ 이므로 $A - B = \{ 1, 3, 4, 12 \}$ 의 모든 원소의 합은 $20$
$k = 18$ 인 경우, $A = \{ 1, 2, 3, 6, 9, 18 \}$ 이므로 $A - B = \{ 1, 3, 9, 18 \}$ 의 모든 원소의 합은 $31$

$\therefore$ 집합 $A - B$ 의 모든 원소의 합이 홀수가 되는 모든 $k$ 의 값의 합은 $10 + 18 = 28$ (참)

따라서 옳은 것은 ㄱ, ㄷ이다.
