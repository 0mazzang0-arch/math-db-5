## 문제
함수 $f(x) = \sin x + \sqrt{3} \cos x + \sqrt{5} \cos \left( x + \frac{\pi}{3} \right)$ 의 최댓값을 구하시오.

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
$f(x) = \sin x + \sqrt{3} \cos x + \sqrt{5} \cos \left( x + \frac{\pi}{3} \right)$
$= 2 \left( \frac{1}{2} \sin x + \frac{\sqrt{3}}{2} \cos x \right) + \sqrt{5} \cos \left( x + \frac{\pi}{3} \right)$
$= 2 \sin \left( x + \frac{\pi}{3} \right) + \sqrt{5} \cos \left( x + \frac{\pi}{3} \right)$
$= 3 \left\{ \frac{2}{3} \sin \left( x + \frac{\pi}{3} \right) + \frac{\sqrt{5}}{3} \cos \left( x + \frac{\pi}{3} \right) \right\}$
$= 3 \sin \left( x + \frac{\pi}{3} + \theta \right)$ (단, $\tan \theta = \frac{\sqrt{5}}{2}$)

따라서 함수 $f(x)$ 의 최댓값은 $3$ 이다.
