## 문제
다음 조건을 만족시키는 모든 자연수 $n$의 값의 합을 구하시오.

(가) $1 < n < 10$
(나) $\log \frac{1}{n}$의 소수부분은 $\log n^{2}$의 소수부분보다 크다.

## 풀이
$\log n = \alpha$라 하면 $1 < n < 10$이므로 $0 < \alpha < 1$이다.
$\log \frac{1}{n} = -\log n = -\alpha = -1 + (1 - \alpha)$이므로 $\log \frac{1}{n}$의 소수부분은 $1 - \alpha$이다.
$\log n^{2} = 2 \log n = 2 \alpha$이므로 $2 \alpha$의 범위에 따라 소수부분을 나누어 구한다.

(ⅰ) $0 < \alpha < \frac{1}{2}$일 때,
$\log n^{2}$의 소수부분은 $2 \alpha$이다.
조건 (나)에 의해 $1 - \alpha > 2 \alpha$
$3 \alpha < 1$, 즉 $0 < \alpha < \frac{1}{3}$이다.
$0 < \log n < \frac{1}{3}$이므로 $1 < n < \sqrt[3]{10}$이다.
$2^{3} = 8$, $3^{3} = 27$이므로 $2 < \sqrt[3]{10} < 3$이다.
따라서 이를 만족시키는 자연수 $n = 2$이다.

(ⅱ) $\frac{1}{2} \le \alpha < 1$일 때,
$\log n^{2}$의 소수부분은 $2 \alpha - 1$이다.
조건 (나)에 의해 $1 - \alpha > 2 \alpha - 1$
$3 \alpha < 2$, 즉 $\frac{1}{2} \le \alpha < \frac{2}{3}$이다.
$\frac{1}{2} \le \log n < \frac{2}{3}$이므로 $\sqrt{10} \le n < \sqrt[3]{100}$이다.
$\sqrt{10} \approx 3.16$이고, $4^{3} = 64$, $5^{3} = 125$이므로 $4 < \sqrt[3]{100} < 5$이다.
따라서 이를 만족시키는 자연수 $n = 4$이다.

따라서 구하는 모든 자연수 $n$의 값의 합은
$2 + 4 = 6$
이다.
