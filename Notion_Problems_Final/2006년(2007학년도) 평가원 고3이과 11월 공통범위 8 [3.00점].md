## 문제
다음 그림은 원점을 출발하여 수직선 위를 움직이는 점 $P$의 시각 $t$ ($0 \le t \le d$)에서의 속도 $v(t)$를 나타내는 그래프이다. 
$\int_{0}^{a} |v(t)| dt = \int_{a}^{d} |v(t)| dt$ 일 때, <보기>에서 옳은 것을 모두 고른 것은? (단, $0 < a < b < c < d$이다.)

> [!example] 📸 (그림/그래프)

$$\begin{array}{|l|}
\hline
\text{ㄱ. 점 } P\text{는 출발 하고 나서 원점을 다시 지난다.} \\
\text{ㄴ. } \int_{0}^{c} v(t) dt = \int_{c}^{d} v(t) dt \\
\text{ㄷ. } \int_{0}^{b} v(t) dt = \int_{b}^{d} |v(t)| dt \\
\hline
\end{array}$$

## 보기
① ㄴ   ② ㄷ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
그림과 같이 각각의 넓이를 $S_{1}$, $S_{2}$, $S_{3}$, $S_{4}$라고 하자. 
조건에서 $\int_{0}^{a} |v(t)| dt = \int_{a}^{d} |v(t)| dt$이므로 
$S_{1} = S_{2} + S_{3} + S_{4}$이다.

> [!example] 📸 (그림/그래프)

ㄱ. $S_{1} > S_{2} + S_{3}$이므로, 점 $P$는 운동 방향을 바꾼 후 다시 원점을 지나기 전에 또다시 운동 방향을 바꾼다. 즉, 다시 원점을 지나지 않는다. (거짓)

ㄴ. $\int_{0}^{c} v(t) dt = S_{1} - S_{2} - S_{3}$
$\int_{c}^{d} v(t) dt = S_{4}$
$S_{1} = S_{2} + S_{3} + S_{4}$에서 $S_{1} - S_{2} - S_{3} = S_{4}$이므로
$\therefore \int_{0}^{c} v(t) dt = \int_{c}^{d} v(t) dt$ (참)

ㄷ. $\int_{0}^{b} v(t) dt = S_{1} - S_{2}$
$\int_{b}^{d} |v(t)| dt = S_{3} + S_{4}$
$S_{1} = S_{2} + S_{3} + S_{4}$에서 $S_{1} - S_{2} = S_{3} + S_{4}$이므로
$\therefore \int_{0}^{b} v(t) dt = \int_{b}^{d} |v(t)| dt$ (참)

따라서 옳은 것은 ㄴ, ㄷ이다.
