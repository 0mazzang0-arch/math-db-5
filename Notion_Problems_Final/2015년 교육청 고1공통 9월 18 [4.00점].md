## 문제

임의의 실수 $m$ 에 대하여 부등식 $y > x^{2}$ 의 영역에 있는 한 점 $P(a, b)$ 를 지나고 기울기가 $m$ 인 직선이 이차함수 $y = x^{2}$ 의 그래프와 만나는 두 점을 $A$, $B$ 라 할 때, 선분 $AB$ 의 중점 $M$ 이 그리는 도형의 방정식을 구하는 과정이다.

$$ \begin{array}{|l|} \hline \text{임의의 실수 } m \text{ 에 대하여 부등식 } y > x^{2} \text{ 의 영역에 있는 한 점 } P(a, b) \text{ 를 지나고} \\ \text{기울기가 } m \text{ 인 직선의 방정식은} \\ y = m(x - a) + b \cdots ㉠ \\ ㉠ \text{이 이차함수 } y = x^{2} \text{ 의 그래프와 만나는 두 점 } A, B \text{ 의 좌표를 각각 } A(x_{1}, y_{1}), \\ B(x_{2}, y_{2}), \text{ 선분 } AB \text{ 의 중점 } M \text{ 의 좌표를 } M(X, Y) \text{ 라 하면} \\ x_{1}, x_{2} \text{ 는 이차방정식 } x^{2} - m(x - a) - b = 0 \text{ 의 두 근이므로} \\ X = \frac{x_{1} + x_{2}}{2} = \text{ (가) } \cdots ㉡ \\ Y = \frac{y_{1} + y_{2}}{2} = \text{ (나) } - am + b \cdots ㉢ \\ ㉡, ㉢ \text{ 에 의하여 } Y = \text{ (다) } X^{2} - 2aX + b \\ \text{따라서 구하는 도형의 방정식은 } y = \text{ (다) } x^{2} - 2ax + b \text{ 이다.} \\ \hline \end{array} $$

위의 과정에서 (가), (나)에 알맞은 식을 각각 $f(m)$, $g(m)$ 이라 하고, (다)에 알맞은 수를 $k$ 라 할 때, $f(k) + g(k)$ 의 값은?



> [!example] 📸 (그림/그래프)

## 보기

① $1$   ② $\frac{3}{2}$   ③ $2$   ④ $\frac{5}{2}$   ⑤ $3$

## 풀이

임의의 실수 $m$ 에 대하여 부등식 $y > x^{2}$ 의 영역에 있는 한 점 $P(a, b)$ 를 지나고 기울기가 $m$ 인 직선의 방정식은
$y = m(x - a) + b \cdots$ ㉠

㉠이 이차함수 $y = x^{2}$ 의 그래프와 만나는 두 점 $A$, $B$ 의 좌표를 각각 $A(x_{1}, y_{1})$, $B(x_{2}, y_{2})$, 선분 $AB$ 의 중점 $M$ 의 좌표를 $M(X, Y)$ 라 하면 $x_{1}, x_{2}$ 는 이차방정식 $x^{2} - m(x - a) - b = 0$, 즉 $x^{2} - mx + am - b = 0$ 의 두 근이므로 근과 계수의 관계에 의해 $x_{1} + x_{2} = m$ 이다.

따라서
$X = \frac{x_{1} + x_{2}}{2} = \frac{m}{2} \cdots$ ㉡
이므로 $f(m) = \frac{m}{2}$ 이다.

또한 $y_{1} = x_{1}^{2}$, $y_{2} = x_{2}^{2}$ 이므로
$Y = \frac{y_{1} + y_{2}}{2} = \frac{x_{1}^{2} + x_{2}^{2}}{2} = \frac{(x_{1} + x_{2})^{2} - 2x_{1} x_{2}}{2}$
$Y = \frac{m^{2} - 2(am - b)}{2} = \frac{m^{2}}{2} - am + b \cdots$ ㉢
이므로 $g(m) = \frac{m^{2}}{2}$ 이다.

㉡에서 $m = 2X$ 이므로 이를 ㉢에 대입하면
$Y = \frac{(2X)^{2}}{2} - a(2X) + b = 2X^{2} - 2aX + b$
따라서 (다)에 알맞은 수 $k = 2$ 이다.

구하는 값은
$f(k) + g(k) = f(2) + g(2) = \frac{2}{2} + \frac{2^{2}}{2} = 1 + 2 = 3$

정답은 ⑤ 이다.
