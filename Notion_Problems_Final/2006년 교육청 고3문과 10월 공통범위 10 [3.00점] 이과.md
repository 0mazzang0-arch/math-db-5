## 문제
$x$보다 크지 않은 최대의 정수를 $[x]$로 나타낼 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?

$$
\begin{array}{|l|}
\hline
\text{ㄱ. } 1 < a < 10 \text{ 일 때, } [\log 100a] = 2 \text{ 이다.} \\
\text{ㄴ. } [\log x] = 3 \text{ 인 정수 } x \text{ 의 개수는 } 9 \times 10^{3} \text{ 이다.} \\
\text{ㄷ. } \text{자연수 } n \text{ 에 대하여 } [\log x] = n \text{ 이면 } [\log x^{2}] = 2n \text{ 이다.} \\
\hline
\end{array}
$$

## 보기
① $ㄱ$   ② $ㄱ, ㄴ$   ③ $ㄱ, ㄷ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
[정답] ②

ㄱ. $1 < a < 10$에서 $0 < \log a < 1$이므로
$\log 100a = 2 + \log a$
$\therefore [\log 100a] = 2$ (참)

ㄴ. $[\log x] = 3$이므로 $10^{3} \le x < 10^{4}$
따라서, 정수 $x$의 개수는
$10^{4} - 10^{3} = 10^{3}(10 - 1) = 9 \times 10^{3}$ (참)

ㄷ. $\log x = n + \alpha$ ($0 \le \alpha < 1$)
$0 \le \alpha < \frac{1}{2}$일 때,
$[\log x^{2}] = [2n + 2\alpha] = 2n$
$\frac{1}{2} \le \alpha < 1$일 때,
$[\log x^{2}] = [2n + 2\alpha] = 2n + 1$
$\therefore [\log x^{2}] = 2n$ 또는 $2n + 1$ (거짓)

따라서, 옳은 것은 ㄱ, ㄴ이다.
