## 문제

양수 $a$에 대하여 $\log a$의 정수부분과 소수부분을 각각 $f(a)$, $g(a)$라 할 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?

$$ \begin{array}{|l|} \hline \text{<보 기>} \\ \text{ㄱ. } f(80)-1=g(80)-3 \log 2 \\ \text{ㄴ. } f(a^2)=2 f(a) \text{ 이면 } f(a^3)=3 f(a) \text{ 이다.} \\ \text{ㄷ. } g(a^2)=g(a) \text{ 이면 모든 자연수 } n\text{에 대하여 } g(a^n)=0 \text{ 이다.} \\ \hline \end{array} $$

## 보기

① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄷ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이

ㄱ. (참)
$\log 80 = \log 10 + \log 8 = 1 + 3 \log 2$ 에서
$f(80) = 1$, $g(80) = 3 \log 2$
$\therefore f(80) - 1 = g(80) - 3 \log 2$

ㄴ. (거짓)
$\log a = f(a) + g(a)$, $\log a^2 = 2 f(a) + 2 g(a)$
$f(a^2) = 2 f(a)$ 이므로 $0 \le 2 g(a) < 1$
$\therefore 0 \le g(a) < \frac{1}{2}$
$\log a^3 = 3 f(a) + 3 g(a)$ 에서 $0 \le 3 g(a) < \frac{3}{2}$ 이고
$1 \le 3 g(a) < \frac{3}{2}$ 일 때는 $f(a^3) = 3 f(a) + 1$ 이다.

ㄷ. (참)
$\log a^2 = 2 f(a) + 2 g(a)$ 에서 $0 \le g(a) < 1$ 이므로 $0 \le 2 g(a) < 2$
$0 \le 2 g(a) < 1$ 일 때 $g(a^2) = 2 g(a)$ 이므로
$g(a^2) = g(a)$ 에서 $g(a) = 2 g(a) \therefore g(a) = 0$
$1 \le 2 g(a) < 2$ 일 때 $g(a^2) = 2 g(a) - 1$ 이므로
$g(a^2) = g(a)$ 에서 $g(a) = 2 g(a) - 1$
$g(a) = 1$ 이므로 모순이다.
$\therefore g(a) = 0$
$\log a^n = n \log a = n f(a) + n g(a)$ 에서 $n g(a) = 0$ 이고
$n f(a)$ 는 정수이므로 $g(a^n) = 0$ 이다.

정답: ③
