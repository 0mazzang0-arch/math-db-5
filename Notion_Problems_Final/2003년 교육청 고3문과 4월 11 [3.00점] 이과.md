## 문제
분수함수 $y = \frac{cx+d}{ax+b}$ 의 그래프가 그림과 같을 때, <보기> 중 옳은 것을 모두 고른 것은?
(단, $a$, $b$, $c$, $d$ 는 실수이다.)

$$ \begin{array}{|l|} \hline \text{<보 기>} \\ \text{ㄱ. } b = c \\ \text{ㄴ. } ab < 0 \\ \text{ㄷ. } \frac{d}{b} < 0 \\ \hline \end{array} $$



> [!example] 📸 (그림/그래프)

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
$y = \frac{cx+d}{ax+b} = \frac{c}{a} + \frac{ad-bc}{a^2 (x + \frac{b}{a})}$ 의 점근선이 $x = 1$, $y = -1$ 이므로
$-\frac{b}{a} = 1$ 

$\therefore ab < 0$, $a = -b \cdots \cdots$ ① 

$\frac{c}{a} = -1$ 에서 $a = -c \cdots \cdots$ ② 

또, ①과 ②에서 $b = c$ 

$y$ 절편이 양수이므로 $\frac{d}{b} > 0$ 이다.

[정답] ②
