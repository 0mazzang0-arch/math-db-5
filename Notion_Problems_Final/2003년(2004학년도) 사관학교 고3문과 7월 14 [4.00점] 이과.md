## 문제
자연수 $n$은 $n = p_{1}^{m_{1}} p_{2}^{m_{2}} \cdots p_{r}^{m_{r}}$ (단, $m_{1}$, $m_{2}$, $\cdots$, $m_{r}$은 음이 아닌 정수이고, $p_{1}$, $p_{2}$, $\cdots$, $p_{r}$은 모두 서로 다른 소수이다.)과 같이 유일하게 소인수분해됩니다. 이때, $f(n) = m_{1} + m_{2} + \cdots + m_{r}$로 정의합니다. 예를 들면, $f(10) = f(2^{1} \cdot 5^{1}) = 1 + 1 = 2$입니다. 다음 <보기>에서 옳은 것을 모두 고르면?

$$ \begin{array}{|l|} \hline \text{<보기>} \\ \text{ㄱ. } f(10^{4}) < f(10^{5}) \\ \text{ㄴ. } n_{1} < n_{2}\text{ 이면 } f(n_{1}) < f(n_{2})\text{ 이다.} \\ \text{ㄷ. } \text{자연수 } n\text{ 에 대하여 } f(2^{n} + 2^{n+3}) = n + 2\text{ 이다.} \\ \hline \end{array} $$

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄷ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
ㄱ. $f(10^{4}) = f(2^{4} \cdot 5^{4}) = 4 + 4 = 8$
$f(10^{5}) = f(2^{5} \cdot 5^{5}) = 5 + 5 = 10$이므로 
$\therefore f(10^{4}) < f(10^{5})$ (참)

ㄴ. (반례) $n_{1} = 6 < n_{2} = 10$ 이면 
$f(n_{1}) = f(6) = f(2 \cdot 3) = 1 + 1 = 2$ 
$f(n_{2}) = f(10) = f(2 \cdot 5) = 1 + 1 = 2$ 이므로 
$\therefore f(n_{1}) = f(n_{2})$ 이 되어 모순이다. (거짓)

ㄷ. 자연수 $n$ 에 대하여 
$f(2^{n} + 2^{n+3}) = f(2^{n}(1 + 2^{3})) = f(2^{n} \cdot 9) = f(2^{n} \cdot 3^{2}) = n + 2$ (참)

따라서 옳은 것은 ㄱ, ㄷ이다.
