## 문제
함수 $f(x) = \sin(x+\alpha) + 2\cos(x+\alpha)$에 대하여 $f' \left( \frac{\pi}{4} \right) = 0$일 때, $\tan \alpha$의 값은? (단, $\alpha$는 상수이다.)

## 보기
① $-\frac{5}{6}$   ② $-\frac{2}{3}$   ③ $-\frac{1}{2}$   ④ $-\frac{1}{3}$   ⑤ $-\frac{1}{6}$

## 풀이
정답: ④

$f(x) = \sin(x+\alpha) + 2\cos(x+\alpha)$를 미분하면
$f'(x) = \cos(x+\alpha) - 2\sin(x+\alpha)$
$x = \frac{\pi}{4}$를 대입하면
$f' \left( \frac{\pi}{4} \right) = \cos \left( \frac{\pi}{4} + \alpha \right) - 2\sin \left( \frac{\pi}{4} + \alpha \right)$
$= \cos \frac{\pi}{4} \cos \alpha - \sin \frac{\pi}{4} \sin \alpha - 2 \left( \sin \frac{\pi}{4} \cos \alpha + \cos \frac{\pi}{4} \sin \alpha \right)$
$= \frac{\sqrt{2}}{2} \cos \alpha - \frac{\sqrt{2}}{2} \sin \alpha - \sqrt{2} \cos \alpha - \sqrt{2} \sin \alpha$
$= -\frac{\sqrt{2}}{2} (\cos \alpha + 3\sin \alpha) = 0$

따라서 $\cos \alpha + 3\sin \alpha = 0$ 이므로
$\cos \alpha = -3\sin \alpha$
$-\frac{1}{3} = \frac{\sin \alpha}{\cos \alpha}$

$\therefore \tan \alpha = -\frac{1}{3}$
