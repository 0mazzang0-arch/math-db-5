## 문제
양의 실수 $x$ 에 대하여 $\log_{10} x$ 의 정수부분을 $f(x)$, 소수부분을 $g(x)$ 라 할 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?

$$ \begin{array}{|l|} \hline \text{<보 기>} \\ \text{ㄱ. } f(1234) + f(0.1234) = 2 \\ \text{ㄴ. } g(5034) - g(0.05034) = 0 \\ \text{ㄷ. } \log_{10} a - \log_{10} 0.0762 = 4 \text{ 이면 } f(a) = 2 \text{ 이다.} \\ \hline \end{array} $$

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
[정답] ⑤ 

ㄱ. 
$f(1234) = 3$, 
$f(0.1234) = -1$ 이므로 
$f(1234) + f(0.1234) = 3 + (-1) = 2$ (참)

ㄴ. 
$g(5034) = \log_{10} 5.034$, 
$g(0.05034) = \log_{10} 5.034$ 이므로 
$g(5034) - g(0.05034) = 0$ (참)

ㄷ. 
$\log_{10} a - \log_{10} 0.0762 = 4$ 에서
$\log_{10} a = \log_{10} 0.0762 + 4 = \log_{10} (0.0762 \times 10^{4}) = \log_{10} 762$ 이다.
따라서 $\log_{10} 762 = 2 + \log_{10} 7.62$ 이므로 $f(a) = 2$ 이다. (참)

따라서 ㄱ, ㄴ, ㄷ 모두 옳다.
