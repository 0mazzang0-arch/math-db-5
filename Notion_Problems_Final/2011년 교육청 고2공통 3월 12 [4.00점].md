## 문제
등식 $\sqrt{x-y+1} \sqrt{x^{2}+y^{2}-2}=-\sqrt{(x-y+1) (x^{2}+y^{2}-2 )}$를 만족하는 두 실수 $x$, $y$의 순서쌍 $(x, y)$를 좌표평면 위에 모두 나타낸 영역을 $D$라 하자. 이때, 옳은 것만을 <보기>에서 있는 대로 고른 것은?

$$
\begin{array}{|l|}
\hline
\text{ㄱ. 점 } (-1, 1) \text{은 } D \text{에 속한다.} \\
\text{ㄴ. 점 } (a, b) \text{가 } D \text{에 속하면 점 } (-b, -a) \text{도 } D \text{에 속한다.} \\
\text{ㄷ. } D \text{에 속하는 점 } (a, b) \text{에 대하여 } a^{2}+b^{2} \text{의 최솟값은 } \frac{1}{2} \text{이다.} \\
\hline
\end{array}
$$

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
[정답] ⑤ 

등식이 성립하기 위해서는 두 실수 $x$, $y$가 다음 ① 또는 ② 또는 ③을 만족해야 한다. 

$x-y+1 < 0$ 이고 $x^{2}+y^{2}-2 < 0 \cdots$ ① 
$x-y+1 = 0 \cdots$ ② 
$x^{2}+y^{2}-2 = 0 \cdots$ ③ 

①, ②, ③을 좌표평면 위에 나타내면 그림과 같다. 

> [!example] 📸 (그림/그래프)

(단, 경계선은 영역에 포함된다.) 

ㄱ. 점 $(-1, 1)$은 $x-y+1 = -1-1+1 = -1 < 0$, $x^{2}+y^{2}-2 = (-1)^{2}+1^{2}-2 = 0$ 이므로 $D$에 속한다. (참) 

ㄴ. $D$에 속하는 모든 점은 직선 $y=-x$에 대하여 대칭이므로 점 $(a, b)$가 $D$에 속하면 점 $(-b, -a)$도 $D$에 속한다. (참) 

ㄷ. $\sqrt{a^{2}+b^{2}}$의 최솟값은 원점과 직선 $y=x+1$ 사이의 거리 $\frac{1}{\sqrt{1^{2}+(-1)^{2}}}=\frac{1}{\sqrt{2}}$과 같다. 
$\therefore a^{2}+b^{2}$의 최솟값은 $\frac{1}{2}$ (참) 

따라서 옳은 것은 ㄱ, ㄴ, ㄷ이다.
