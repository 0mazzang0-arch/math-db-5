## 문제

다음은 $n$ 이 자연수일 때, $x$ 에 대한 방정식 $x^{n}=2\cos\theta+1$ 의 실근의 개수에 대한 설명이다.

$$
\begin{array}{|l|}
\hline
\text{(가) } n \text{ 이 짝수이고 } \frac{2\pi}{3} < \theta < \frac{4\pi}{3} \text{ 이면 } a \text{ 개} \\
\text{(나) } n \text{ 이 짝수이고 } \frac{3\pi}{2} < \theta < 2\pi \text{ 이면 } b \text{ 개} \\
\text{(다) } n \text{ 이 홀수이고 } \frac{\pi}{3} < \theta < \frac{3\pi}{2} \text{ 이면 } c \text{ 개} \\
\hline
\end{array}
$$

$a+2b+3c$ 의 값은?

## 보기

① $3$   ② $4$   ③ $5$   ④ $6$   ⑤ $7$

## 풀이

$n$ 이 짝수일 때, $x^{n}=k$ 의 실근의 개수는 $k<0$ 이면 $0$ 개, $k=0$ 이면 $1$ 개, $k>0$ 이면 $2$ 개이다.
또한 $n$ 이 홀수일 때, $x^{n}=k$ 의 실근의 개수는 $k$ 의 값에 관계없이 항상 $1$ 개이다.

(가) $n$ 이 짝수이고 $\frac{2\pi}{3} < \theta < \frac{4\pi}{3}$ 이면 $\cos\theta < -\frac{1}{2}$ 이므로 $2\cos\theta+1 < 0$ 이다.
따라서 실근의 개수 $a=0$ 이다.

(나) $n$ 이 짝수이고 $\frac{3\pi}{2} < \theta < 2\pi$ 이면 $\cos\theta > 0$ 이므로 $2\cos\theta+1 > 1 > 0$ 이다.
따라서 실근의 개수 $b=2$ 이다.

(다) $n$ 이 홀수이면 $\theta$ 의 값에 관계없이 실근의 개수 $c=1$ 이다.

따라서 $a+2b+3c = 0 + 2 \times 2 + 3 \times 1 = 0 + 4 + 3 = 7$ 이다.

정답: ⑤
