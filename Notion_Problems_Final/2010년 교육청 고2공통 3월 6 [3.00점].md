## 문제
$0 < a < b$ 일 때, $A$, $B$, $C$ 를 각각 
$A = \frac{a}{a + 1}$, $B = \frac{b}{b + 1}$, $C = \frac{a + b}{a + b + 2}$
라 하자. $A$, $B$, $C$ 의 대소 관계를 바르게 나타낸 것은? 

## 보기
① $A < B < C$   ② $A < C < B$   ③ $B < A < C$   ④ $B < C < A$   ⑤ $C < A < B$

## 풀이
[정답] ② 

$0 < a < b$ 에서 
$a - b < 0$ 

$A - B = \frac{a}{a + 1} - \frac{b}{b + 1} = \frac{a - b}{( a + 1 ) ( b + 1 )} < 0$ 이므로 
$A < B$ 

$A - C = \frac{a}{a + 1} - \frac{a + b}{a + b + 2} = \frac{a - b}{( a + 1 ) ( a + b + 2 )} < 0$ 이므로 
$A < C$ 

$B - C = \frac{b}{b + 1} - \frac{a + b}{a + b + 2} = \frac{b - a}{( b + 1 ) ( a + b + 2 )} > 0$ 이므로 
$B > C$ 

따라서 
$A < C < B$
