## 문제
두 집합 $X = \{ 0, 1, 2 \}$, $Y = \{ 1, 2, 3, 4 \}$ 에 대하여 두 함수 $f : X \to Y$, $g : X \to Y$ 를
$f(x) = 2x^{2} - 4x + 3$, $g(x) = a|x - 1| + b$
라 하자. 두 함수 $f$ 와 $g$ 가 서로 같도록 하는 상수 $a$, $b$ 에 대하여 $2a - b$ 의 값은?

## 보기
① $-3$   ② $-1$   ③ $1$   ④ $3$   ⑤ $5$

## 풀이
$f(0) = 3$, $f(1) = 1$, $f(2) = 3$
$g(0) = a + b$, $g(1) = b$, $g(2) = a + b$
두 함수 $f$ 와 $g$ 가 서로 같으므로
$f(0) = g(0)$, $f(1) = g(1)$, $f(2) = g(2)$
$\therefore a + b = 3$, $b = 1$ 이므로
$a = 2$, $b = 1$
따라서 $2a - b = 3$

정답: ④
