## 문제
다음은 $x$에 대한 방정식
$$(x^2 + ax + a)(x^2 + x + a) = 0$$
의 근 중 서로 다른 허근의 개수가 $2$이기 위한 실수 $a$의 값의 범위를 구하는 과정이다.

(1) $a=1$인 경우
$$(x^2 + x + 1)^2 = 0$$이다.
$x^2 + x + 1 = 0$의 근은
$$x = \frac{-1 \pm \sqrt{(가)} i}{2}$$
(단, $i = \sqrt{-1}$)이므로 방정식 $(x^2 + x + 1)^2 = 0$의 서로 다른 허근의 개수는 $2$이다.

(2) $a \neq 1$인 경우
$x^2 + ax + a = 0$의 근은
$$x = \frac{-a \pm \sqrt{(나)}}{2}$$이다.
(ⅰ) $(나) < 0$일 때, 방정식 $x^2 + x + a = 0$은 실근을 가져야 하므로 실수 $a$의 값의 범위는 $0 < a \leq \frac{1}{4}$이다.
(ⅱ) $(나) \geq 0$일 때, 방정식 $x^2 + x + a = 0$은 허근을 가져야 하므로 실수 $a$의 값의 범위는 $a \geq (다)$이다.

따라서 (1)과 (2)에 의하여 방정식 $(x^2 + ax + a)(x^2 + x + a) = 0$의 근 중 서로 다른 허근의 개수가 $2$이기 위한 실수 $a$의 값의 범위는 $0 < a \leq \frac{1}{4}$ 또는 $a = 1$ 또는 $a \geq (다)$이다.

위의 (가), (다)에 알맞은 수를 각각 $p$, $q$라 하고, (나)에 알맞은 식을 $f(a)$라 할 때, $p + q + f(5)$의 값은?

## 보기
① $8$   ② $9$   ③ $10$   ④ $11$   ⑤ $12$

## 풀이
(1) $a = 1$인 경우
$$(x^2 + x + 1)^2 = 0$$
$x^2 + x + 1 = 0$의 근은 $x = \frac{-1 \pm \sqrt{3}i}{2}$이므로 (가)에 알맞은 수 $p = 3$이다.

(2) $a \neq 1$인 경우
$x^2 + ax + a = 0$의 근은 $x = \frac{-a \pm \sqrt{a^2 - 4a}}{2} = \frac{-a \pm \sqrt{a(a - 4)}}{2}$이므로 (나)에 알맞은 식 $f(a) = a(a - 4)$이다.

(ⅰ) $a(a - 4) < 0$ (즉, $0 < a < 4$)일 때,
방정식 $x^2 + x + a = 0$이 실근을 가져야 하므로 판별식 $D = 1 - 4a \geq 0$에서 $a \leq \frac{1}{4}$이다.
따라서 $0 < a \leq \frac{1}{4}$이다.

(ⅱ) $a(a - 4) \geq 0$ (즉, $a \leq 0$ 또는 $a \geq 4$)일 때,
방정식 $x^2 + x + a = 0$이 허근을 가져야 하므로 판별식 $D = 1 - 4a < 0$에서 $a > \frac{1}{4}$이다.
따라서 $a \geq 4$이다. 그러므로 (다)에 알맞은 수 $q = 4$이다.

따라서 $p = 3$, $q = 4$, $f(a) = a(a - 4)$이므로
$$p + q + f(5) = 3 + 4 + 5(5 - 4) = 3 + 4 + 5 = 12$$
정답은 ⑤이다.
