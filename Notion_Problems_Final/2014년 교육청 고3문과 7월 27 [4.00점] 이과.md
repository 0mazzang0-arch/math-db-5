## 문제
$10 < x < 100$ 인 $x$ 에 대하여 $log \sqrt{x}$ 의 소수 부분이 $log \frac{1}{x}$ 의 소수 부분의 $5$ 배이다. $log x = \frac{q}{p}$ 일 때, $p+q$ 의 값을 구하시오. (단, $p$, $q$ 는 서로소인 자연수이다.)

## 보기

## 풀이
$log x = 1 + \alpha$ ($0 < \alpha < 1$)이라 하자.

$log \sqrt{x} = \frac{1 + \alpha}{2}$ 이고
$log \frac{1}{x} = -log x = -(1 + \alpha) = -2 + (1 - \alpha)$ 이다.

주어진 조건에 의하여
$\frac{1 + \alpha}{2} = 5 ( 1 - \alpha )$
$1 + \alpha = 10 ( 1 - \alpha )$
$1 + \alpha = 10 - 10 \alpha$
$11 \alpha = 9$
$\alpha = \frac{9}{11}$ 이므로

$log x = 1 + \frac{9}{11} = \frac{20}{11}$
따라서 $p = 11$, $q = 20$ 이므로
$p + q = 31$

정답: $31$
