## 문제
두 집합 $A = \{ x | x^{2}-x-2 < 0 \}$, $B = \{ x | \frac{a}{x+1} + \frac{1}{x-3} > 0 \}$ 에 대하여 $A \subset B$ 가 되도록 하는 양수 $a$ 의 최솟값은?

## 보기
① $3$   ② $4$   ③ $5$   ④ $6$   ⑤ $7$

## 풀이
집합 $A$ 의 부등식을 풀면 다음과 같다.
$x^{2}-x-2 < 0$
$(x-2)(x+1) < 0$
$-1 < x < 2$
$\therefore A = \{ x | -1 < x < 2 \}$

집합 $B$ 의 부등식을 정리하면 다음과 같다.
$\frac{a}{x+1} + \frac{1}{x-3} > 0$
$\frac{a(x-3) + (x+1)}{(x+1)(x-3)} > 0$
$\frac{(a+1)x - (3a-1)}{(x+1)(x-3)} > 0$

양수 $a$ 에 대하여 $a+1 > 0$ 이므로 위 식은 다음과 동치이다.
$(x - \frac{3a-1}{a+1})(x+1)(x-3) > 0$

이때 $a > 0$ 이므로 $\frac{3a-1}{a+1}$ 의 범위를 확인하면:
$-1 < \frac{3a-1}{a+1} < 3$ 이 성립한다.
따라서 집합 $B$ 는 다음과 같다.
$B = \{ x | -1 < x < \frac{3a-1}{a+1} \text{ 또는 } x > 3 \}$

$A \subset B$ 가 성립하기 위해서는 $(-1, 2) \subset (-1, \frac{3a-1}{a+1})$ 이어야 하므로
$\frac{3a-1}{a+1} \geq 2$
$3a-1 \geq 2a+2$
$a \geq 3$

따라서 양수 $a$ 의 최솟값은 $3$ 이다.

정답: ①
