## 문제
이차부등식 $a x^{2} + b x + c < 0$의 해가 $\frac{1}{8} < x < \frac{1}{2}$일 때, 이차부등식 $c x^{2} + b x + a \le 0$의 해 중에서 정수 $x$의 개수는?

## 보기
① $5$   ② $7$   ③ $9$   ④ $11$   ⑤ $13$

## 풀이
$a x^{2} + b x + c < 0$의 해가 $\frac{1}{8} < x < \frac{1}{2}$이므로 $a > 0$이다.
해가 $\frac{1}{8} < x < \frac{1}{2}$이고 $x^{2}$의 계수가 $1$인 이차부등식은
$( x - \frac{1}{8} ) ( x - \frac{1}{2} ) < 0$
$\therefore x^{2} - \frac{5}{8} x + \frac{1}{16} < 0$
양변에 $a$를 곱하면
$a x^{2} - \frac{5}{8} a x + \frac{1}{16} a < 0$
이 부등식이 $a x^{2} + b x + c < 0$과 같으므로
$b = - \frac{5}{8} a$, $c = \frac{1}{16} a$
이것을 $c x^{2} + b x + a \le 0$에 대입하면
$\frac{1}{16} a x^{2} - \frac{5}{8} a x + a \le 0$
$a > 0$이므로 양변을 $a$로 나누면
$\frac{1}{16} x^{2} - \frac{5}{8} x + 1 \le 0$
$x^{2} - 10 x + 16 \le 0$
$( x - 2 ) ( x - 8 ) \le 0$
$\therefore 2 \le x \le 8$
따라서 정수 $x$는 $2, 3, 4, 5, 6, 7, 8$의 $7$개다.
