## 문제
등식 $(i+i^{2}) + (i^{2}+i^{3}) + (i^{3}+i^{4}) + \cdots + (i^{18}+i^{19}) = a+bi$를 만족시키는 실수 $a$, $b$에 대하여 $4(a+b)^{2}$의 값을 구하시오. (단, $i = \sqrt{-1}$이다.)

## 풀이
주어진 등식에서 
$$(i+i^{2}) + (i^{2}+i^{3}) + (i^{3}+i^{4}) + \cdots + (i^{18}+i^{19})$$
$$= (i+i^{2} + \cdots + i^{18}) + (i^{2}+i^{3} + \cdots + i^{19})$$
$$= (i-1) + (-1-i) = -2$$
이때, 
$$(i+i^{2}) + (i^{2}+i^{3}) + (i^{3}+i^{4}) + \cdots + (i^{18}+i^{19}) = a+bi$$
이므로 복소수가 서로 같을 조건에 의하여 
$a = -2$, $b = 0$
$\therefore 4(a+b)^{2} = 16$

**[다른 풀이]**
$i+i^{19} = 0$이므로 
$$(i+i^{2}) + (i^{2}+i^{3}) + (i^{3}+i^{4}) + \cdots + (i^{18}+i^{19})$$
$$= i + \{(i+i^{2}) + (i^{2}+i^{3}) + (i^{3}+i^{4}) + \cdots + (i^{18}+i^{19})\} + i^{19}$$
$$= (i+i) + (i^{2}+i^{2}) + \cdots + (i^{19}+i^{19})$$
$$= 2(i+i^{2}+i^{3} + \cdots + i^{19})$$
$$= 2(i+i^{2}+i^{3} + \cdots + i^{19} + i^{20} - i^{20})$$
$$= 2(-i^{20})$$
$$= -2$$
이때, 
$$(i+i^{2}) + (i^{2}+i^{3}) + (i^{3}+i^{4}) + \cdots + (i^{18}+i^{19}) = a+bi$$
이므로 복소수가 서로 같을 조건에 의하여 
$a = -2$, $b = 0$
$\therefore 4(a+b)^{2} = 16$
