## 문제

사차함수 $f(x)$의 도함수 $y=f'(x)$의 그래프가 그림과 같고, $f'(-\sqrt{2}) = f'(0) = f'(\sqrt{2}) = 0$이다.

> [!example] 📸 (그림/그래프)

$f(0) = 1$, $f(\sqrt{2}) = -3$일 때, $f(m)f(m+1) < 0$을 만족시키는 모든 정수 $m$의 값의 합은?

## 보기

① $-2$   ② $-1$   ③ $0$   ④ $1$   ⑤ $2$

## 풀이

함수 $f'(x)$는 삼차함수이고 $f'(0) = f'(\sqrt{2}) = f'(-\sqrt{2}) = 0$이므로
$$f'(x) = kx(x+\sqrt{2})(x-\sqrt{2}) = kx(x^2-2) = kx^3-2kx$$
(단, $k$는 상수)

$f(x) = \frac{k}{4}x^4 - kx^2 + C$
(단, $C$는 적분상수)

$f(0) = C = 1$
$f(\sqrt{2}) = k - 2k + C = -k + 1 = -3$ 이므로 $k = 4$

따라서 $f(x) = x^4 - 4x^2 + 1$
함수 $y=f(x)$의 그래프는 그림과 같다.

> [!example] 📸 (그림/그래프)

$f(-2) = 1 > 0$
$f(-1) = -2 < 0$
$f(0) = 1 > 0$
$f(1) = -2 < 0$
$f(2) = 1 > 0$

$f(m)f(m+1) < 0$을 만족시키는 정수 $m$을 구하면 다음과 같다.
- $m = -2$일 때, $f(-2)f(-1) = (1)(-2) = -2 < 0$
- $m = -1$일 때, $f(-1)f(0) = (-2)(1) = -2 < 0$
- $m = 0$일 때, $f(0)f(1) = (1)(-2) = -2 < 0$
- $m = 1$일 때, $f(1)f(2) = (-2)(1) = -2 < 0$

따라서 $f(m)f(m+1) < 0$을 만족시키는 모든 정수 $m$은 $-2$, $-1$, $0$, $1$이다.
모든 정수 $m$의 값의 합은
$(-2) + (-1) + 0 + 1 = -2$

정답: ①
