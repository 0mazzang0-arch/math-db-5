## 문제
양의 실수 $x$에 대하여 $f(x) = \log x$이다. $f(n)$의 정수 부분이 $1$, 소수 부분이 $\alpha$일 때, $2\alpha$의 정수 부분이 $1$인 모든 자연수 $n$의 개수는? (단, $3.1 < \sqrt{10} < 3.2$)

## 보기
① $64$   ② $66$   ③ $68$   ④ $70$   ⑤ $72$

## 풀이
$f(n) = \log n = 1 + \alpha$ (단, $0 \le \alpha < 1$)
조건에 의해 $2\alpha$의 정수 부분이 $1$이므로
$1 \le 2\alpha < 2$ 에서
$\frac{1}{2} \le \alpha < 1$ 이다.
$1 + \frac{1}{2} \le \log n < 1 + 1$ 이므로
$\frac{3}{2} \le \log n < 2$
$10^{\frac{3}{2}} \le n < 10^{2}$
$10\sqrt{10} \le n < 100$
문제에서 $3.1 < \sqrt{10} < 3.2$ 이므로
$31 < 10\sqrt{10} < 32$
즉, $10\sqrt{10} = 31. \dots$ 이므로
$31. \dots \le n < 100$
그런데 $n$은 자연수이므로
$32 \le n < 100$
따라서 자연수 $n$의 개수는 $100 - 32 = 68$ 이다.

정답: ③
