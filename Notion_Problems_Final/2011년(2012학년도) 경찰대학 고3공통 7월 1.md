## 문제
$\frac{1}{x} + \frac{1}{y} + \frac{1}{z} = 0$, $x^{2} + y^{2} + z^{2} = 2$ 일 때, $(x + y + z)^{12}$의 값은?

## 보기
① $0$   ② $1$   ③ $16$   ④ $32$   ⑤ $64$

## 풀이
$\frac{1}{x} + \frac{1}{y} + \frac{1}{z} = 0$에서 
$\frac{xy + yz + zx}{xyz} = 0$ 

$\therefore xy + yz + zx = 0$ 

이때, $x^{2} + y^{2} + z^{2} = 2$ 이므로 
$(x + y + z)^{2} = x^{2} + y^{2} + z^{2} + 2(xy + yz + zx) = 2$ 

따라서 
$(x + y + z)^{12} = \{(x + y + z)^{2}\}^{6} = 2^{6} = 64$ 

정답은 ⑤이다.
