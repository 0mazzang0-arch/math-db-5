## 문제
수열 $\{ a_{n} \}$ 은 $15$ 와 서로소인 자연수를 작은 수부터 차례대로 모두 나열하여 만든 것이다. 예를 들면 $a_{2} = 2$, $a_{4} = 7$ 이다. $\sum_{n=1}^{16} a_{n}$ 의 값은?

## 보기
① $240$   ② $280$   ③ $320$   ④ $360$   ⑤ $400$

## 풀이
$1 \le n \le 15$ 를 만족시키는 자연수 $n$ 중 $15$ 와 서로소인 자연수 $8$ 개
$16 \le n \le 30$ 을 만족시키는 자연수 $n$ 중 $15$ 와 서로소인 자연수 $8$ 개
$\vdots$
$15(k-1)+1 \le n \le 15k$ 를 만족시키는 자연수 $n$ 중 $15$ 와 서로소인 자연수 $8$ 개 ($k=1, 2, 3, \dots$)

$a_{16}$ 은 $16 \le n \le 30$ 을 만족시키는 $15$ 와 서로소인 자연수 $n$ 중 가장 큰 수이다.
$\sum_{n=1}^{16} a_{n}$ 은 $1$ 부터 $30$ 까지 자연수 중 $15$ 와 서로소인 자연수들의 합이다.

$1$ 부터 $30$ 까지 자연수 중에는 $10$ 개의 $3$ 의 배수, $6$ 개의 $5$ 의 배수, $2$ 개의 $15$ 의 배수가 있다.
따라서 
$\sum_{n=1}^{16} a_{n} = \sum_{n=1}^{30} n - \sum_{n=1}^{10} 3n - \sum_{n=1}^{6} 5n + \sum_{n=1}^{2} 15n$
$= 240$

[다른 풀이]
$a_{1} = 1$, $a_{2} = 2$, $a_{3} = 4$, $a_{4} = 7$, $a_{5} = 8$, $a_{6} = 11$, $a_{7} = 13$, $a_{8} = 14$ 이고
$k$ ($k=1, 2, 3, \dots, 14$)가 $15$ 와 서로소이면 $15+k$ 도 $15$ 와 서로소이므로
$a_{9} = a_{1} + 15$, $a_{10} = a_{2} + 15$, $\dots$, $a_{16} = a_{8} + 15$

$\sum_{n=1}^{8} a_{n} = 1+2+4+7+8+11+13+14 = 60$
$\sum_{n=1}^{16} a_{n} = \sum_{n=1}^{8} a_{n} + \sum_{n=9}^{16} a_{n}$
$= 2\sum_{n=1}^{8} a_{n} + 15 \times 8$
$= 2 \times 60 + 120$
$= 240$

정답: ①
