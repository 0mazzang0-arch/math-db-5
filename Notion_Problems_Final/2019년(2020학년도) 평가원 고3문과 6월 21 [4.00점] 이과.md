## 문제
실수 전체의 집합에서 정의된 함수 $f(x)$가 다음 조건을 만족시킨다.

(가) 
$$ f(x) = \left\{ \begin{array}{ll} 2 & ( 0 \le x < 2 ) \\ -2x+6 & ( 2 \le x < 3 ) \\ 0 & ( 3 \le x \le 4 ) \end{array} \right. $$
(나) 모든 실수 $x$에 대하여 $f(-x) = f(x)$이고 $f(x) = f(x-8)$이다.

이때 실수 전체의 집합에서 정의된 함수 
$$ g(x) = \left\{ \begin{array}{ll} \frac{|x|}{x} + n & ( x \neq 0 ) \\ n & ( x = 0 ) \end{array} \right. $$
에 대하여 함수 $(f \circ g)(x)$가 상수함수가 되도록 하는 $60$ 이하의 자연수 $n$의 개수는?



> [!example] 📸 (그림/그래프)

## 보기
① $30$   ② $32$   ③ $34$   ④ $36$   ⑤ $38$

## 풀이
[정답] ① 

조건 (나)에서 모든 실수 $x$ 에 대하여 $f(-x)=f(x)$ 이므로 $-4 \le x \le 0$ 에서의 함수 $y=f(x)$ 의 그래프는 $0 \le x \le 4$ 에서의 함수 $y=f(x)$ 의 그래프를 $y$ 축에 대하여 대칭 이동시킨 그래프와 같다. 또한 모든 실수 $x$ 에 대하여 $f(x)=f(x-8)$ 이므로 $4 \le x \le 12$ 에서의 함수 $y=f(x)$ 의 그래프는 $-4 \le x \le 4$ 에서의 함수 $y=f(x)$ 의 그래프를 $x$ 축의 방향으로 $8$ 만큼 평행이동 시킨 그래프와 같다. 
이와 마찬가지로 정수 $k$ 에 대하여 $-4+8k \le x \le 4+8k$ 에서의 함수 $y=f(x)$ 의 그래프는 $-4 \le x \le 4$ 에서의 함수 $y=f(x)$ 의 그래프를 $x$ 축의 방향으로 $8k$ 만큼 평행 이동시킨 그래프와 같다. 

한편, $x>0$ 에서 $\frac{|x|}{x}=\frac{x}{x}=1$ 이고 $x<0$ 에서 $\frac{|x|}{x}=\frac{-x}{x}=-1$ 이므로 
$$ g(x) = \left\{ \begin{array}{ll} n+1 & (x>0) \\ n & (x=0) \\ n-1 & (x<0) \end{array} \right. $$ 
함수 $g(x)$ 의 치역은 $\{ n-1, n, n+1 \}$ 이다.
실수 전체의 집합에서 함수 $(f \circ g)(x)$ 가 상수함수이려면 $f(n-1)=f(n)=f(n+1)$ 을 만족시켜야 한다. 
즉, 연속인 세 정수에 대하여 함수 $f$ 의 값이 같은 경우는 다음과 같다. 

(ⅰ) $(f \circ g)(x)=2$ 가 되는 경우 
$-2+8k \le n-1$ 이고 $n+1 \le 2+8k$ ($k$ 는 정수) 
$8k-1 \le n \le 8k+1$ ($k$ 는 정수) 
$k<0$ 이면 $8k+1<0$ 이므로 조건을 만족시키는 자연수 $n$ 은 존재하지 않는다. 
$k=0$ 이면 $-1 \le n \le 1$ 이므로 조건을 만족시키는 자연수 $n$ 은 $1$ 이다.
$k \ge \frac{61}{8}$ 이므로 (원문 내용 유지)
$1 \le k \le 7$ 이면 $8k-1 \le n \le 8k+1$ ($k$ 는 정수)이므로 조건을 만족시키는 $60$ 이하의 자연수 $n$ 은 $8k-1$, $8k$, $8k+1$ 이다.
$k>8$ 이면 $8k-1>63$ 이므로 조건을 만족시키는 $60$ 이하의 자연수 $n$ 은 존재하지 않는다. 

(ⅱ) $(f \circ g)(x)=0$ 이 되는 경우 
$3+8t \le n-1$ 이고 $n+1 \le 5+8t$ ($t$ 는 정수) 
$4+8t \le n \le 4+8t$ 에서 $n=4+8t$ ($t$ 는 정수) 
$1 \le 4+8t \le 60$ 에서 $-\frac{3}{8} \le t \le 7$ 이므로 $0 \le t \le 7$ 이다.
$60$ 이하의 자연수 $n$ 의 개수는 $4, 12, 20, 28, 36, 44, 52, 60$ 의 $8$ 개이다. 

(ⅰ), (ⅱ)에서 조건을 만족시키는 $60$ 이하의 자연수 $n$ 의 개수는 $(1+3 \times 7)+8=30$ 이다.
