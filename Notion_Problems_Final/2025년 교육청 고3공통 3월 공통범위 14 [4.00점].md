## 문제
최고차항의 계수가 $1$ 인 사차함수 $f (x )$ 가 다음 조건을 만족시킨다.

$$ \int_{x_{1}}^{x_{2}} \{ f (t ) - f (a ) \} dt \ge \int_{x_{1}}^{x_{2}} f' (a ) (t - a ) dt $$
를 만족시키는 모든 실수 $a$ 의 값의 범위가 $a \le -1$ 또는 $a \ge 3$ 이다. ($x_{1} \le x_{2}$ 인 모든 실수 $x_{1}$, $x_{2}$ 에 대하여)

$f (1 ) = 15$, $f' (1 ) = 1$ 일 때, $f (4 )$ 의 값은?

## 보기
① $21$   ② $23$   ③ $25$   ④ $27$   ⑤ $29$

## 풀이
$\int_{x_{1}}^{x_{2}} \{ f (t ) - f (a ) \} dt \ge \int_{x_{1}}^{x_{2}} f' (a ) (t - a ) dt$ 에서
$\int_{x_{1}}^{x_{2}} f (t ) dt - \int_{x_{1}}^{x_{2}} f (a ) dt \ge \int_{x_{1}}^{x_{2}} f' (a ) (t - a ) dt$
$\int_{x_{1}}^{x_{2}} f (t ) dt \ge \int_{x_{1}}^{x_{2}} f' (a ) (t - a ) dt + \int_{x_{1}}^{x_{2}} f (a ) dt$
$\int_{x_{1}}^{x_{2}} f (t ) dt \ge \int_{x_{1}}^{x_{2}} \{ f' (a ) (t - a ) + f (a ) \} dt$

이 부등식이 $x_{1} \le x_{2}$ 인 모든 실수 $x_{1}$, $x_{2}$ 에 대하여 성립하려면, 사차함수 $f (x )$ 는 모든 실수 $x$ 에 대하여 다음을 만족해야 한다.
$f (x ) \ge f' (a ) (x - a ) + f (a )$

두 점 $(-1, f (-1 ) )$, $(3, f (3 ) )$ 을 지나는 직선의 방정식을 $y = mx + n$ ($m, n$ 은 상수)이라 하면
$m = \frac{f (3 ) - f (-1 )}{3 - (-1 )} = \frac{f (3 ) - f (-1 )}{4}$ 이다.

모든 실수 $x$ 에 대하여 $f (x ) \ge f' (a ) (x - a ) + f (a )$ 를 만족시키는 모든 실수 $a$ 의 값의 범위가 $a \le -1$ 또는 $a \ge 3$ 이므로,

$x = -1$, $a = 3$ 일 때:
$f (-1 ) \ge -4 f' (3 ) + f (3 )$
$\frac{f (3 ) - f (-1 )}{4} \le f' (3 )$ $\cdots$ ㉠

$x = 3$, $a = -1$ 일 때:
$f (3 ) \ge 4 f' (-1 ) + f (-1 )$
$f' (-1 ) \le \frac{f (3 ) - f (-1 )}{4}$ $\cdots$ ㉡

㉠, ㉡에서 $f' (-1 ) \le m \le f' (3 )$ $\cdots$ ㉢

함수 $g (x )$ 를 $g (x ) = f (x ) - (mx + n )$ 이라 하자.
$g (a ) = f (a ) - ma - n$
$g' (x ) = f' (x ) - m$ 에서 $g' (a ) = f' (a ) - m$ 이다.

모든 실수 $x$ 에 대하여 $f (x ) \ge f' (a ) (x - a ) + f (a )$ 는
$f (x ) - (mx + n ) \ge f' (a ) (x - a ) + f (a ) - (mx + n )$
$f (x ) - (mx + n ) \ge \{ f' (a ) - m \} (x - a ) + \{ f (a ) - ma - n \}$
즉, $g (x ) \ge g' (a ) (x - a ) + g (a )$ 와 같다.

이를 만족시키는 모든 실수 $a$ 의 값의 범위도 $a \le -1$ 또는 $a \ge 3$ 이다.
$g (-1 ) = g (3 ) = 0$ 이므로
$g (x ) \ge g' (-1 ) (x + 1 )$ $\cdots$ ㉣
$g (x ) \ge g' (3 ) (x - 3 )$ $\cdots$ ㉤

㉢에서 $g' (-1 ) \le 0 \le g' (3 )$ 이므로, ㉣, ㉤에서 $x \le -1$ 또는 $x \ge 3$ 일 때 $g (x ) \ge 0$ 이다.
구간 $(-\infty, -1]$ 에서 $g (x )$ 의 최솟값은 $g (-1 ) = 0$ 이고, 구간 $[3, \infty)$ 에서 $g (x )$ 의 최솟값은 $g (3 ) = 0$ 이다.

최고차항의 계수가 $1$ 인 사차함수 $g (x )$ 는 최솟값을 가지고, 모든 실수 $x$ 에 대하여 $g (x ) \ge g (c )$ 를 만족시키는 실수 $c$ 가 닫힌구간 $[-1, 3]$ 에 존재한다.
$g (c )$ 가 극솟값이므로 $g' (c ) = 0$ 이고 $g (x ) \ge g (c ) = g' (c ) (x - c ) + g (c )$ 이다.
실수 $a = c$ 가 $g (x ) \ge g' (a ) (x - a ) + g (a )$ 를 만족시키므로 $c \le -1$ 또는 $c \ge 3$ 이다.
그러므로 $c = -1$ 또는 $c = 3$ 이다.

즉 $g (-1 ) = g (3 ) = 0$, $g' (-1 ) = g' (3 ) = 0$ 이므로
$g (x ) = (x + 1 )^{2} (x - 3 )^{2}$
$f (x ) = (x + 1 )^{2} (x - 3 )^{2} + mx + n$
$f' (x ) = 4 (x + 1 ) (x - 3 ) (x - 1 ) + m$

$f' (1 ) = 1$ 이므로 $m = 1$ 이다.
$f (1 ) = 15$ 이므로 $16 + m + n = 15$, 즉 $m + n = -1$ $\cdots$ ㉥
$m = 1$ 을 ㉥에 대입하면 $n = -2$ 이다.

따라서 $f (x ) = (x + 1 )^{2} (x - 3 )^{2} + x - 2$
$f (4 ) = (5 )^{2} (1 )^{2} + 4 - 2 = 25 + 2 = 27$

정답은 ④ 이다.
