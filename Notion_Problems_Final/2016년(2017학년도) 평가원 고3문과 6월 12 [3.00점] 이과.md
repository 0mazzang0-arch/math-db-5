## 문제
등차수열 $\{ a_{n} \}$에 대하여
$a_{8} = a_{2} + 12$, $a_{1} + a_{2} + a_{3} = 15$
일 때, $a_{10}$의 값은?

## 보기
① $17$  ② $19$  ③ $21$  ④ $23$  ⑤ $25$

## 풀이
등차수열 $\{ a_{n} \}$의 첫째항을 $a$, 공차를 $d$라 하면
$a_{8} - a_{2} = 6d$이므로 $6d = 12$에서 $d = 2$
$a_{1} + a_{2} + a_{3} = 3a_{2} = 3 ( a + d ) = 3 ( a + 2 )$이므로
$3 ( a + 2 ) = 15$에서 $a = 3$
따라서 $a_{10} = a + 9d = 3 + 9 \times 2 = 21$

정답: ③
