## 문제
그림과 같이 반지름의 길이가 $3$인 두 원을 서로의 중심을 지나도록 그렸을 때, 두 원의 내부에서 겹친 부분이 나타내는 도형을 $F_{1}$이라 하자. $F_{1}$의 내부에 반지름의 길이가 같고 서로의 중심을 지나는 두 원을 $F_{1}$과 접하면서 반지름의 길이가 최대가 되도록 그렸을 때, 그려진 두 원의 내부에서 겹친 부분이 나타내는 도형을 $F_{2}$라 하자. $F_{2}$의 내부에 반지름의 길이가 같고 서로의 중심을 지나는 두 원을 $F_{2}$와 접하면서 반지름의 길이가 최대가 되도록 그렸을 때, 그려진 두 원의 내부에서 겹친 부분이 나타내는 도형을 $F_{3}$이라 하자. 

> [!example] 📸 (그림/그래프)

이와 같은 방법으로 계속하여 도형 $F_{n}$을 그려 나갈 때, $F_{n}$의 둘레의 길이를 $l_{n}$이라 하자. $\sum_{n=1}^{\infty} l_{n}$의 값은?

## 보기
① $2\pi(1+\sqrt{7})$   ② $\frac{8\pi}{3}(1+\sqrt{7})$   ③ $\frac{4\pi}{3}(2+\sqrt{7})$   ④ $2\pi(2+\sqrt{7})$   ⑤ $\frac{5\pi}{3}(2+\sqrt{7})$

## 풀이
[정답] ① 

주어진 그림의 두 번째 그림에서 작은 원의 반지름의 길이를 $r$라 하면 
$AC = 3- r$, 
$BC = \frac{r}{2}$, 
$AB = \frac{3}{2}$이고 
$\angle B = 90^\circ$이므로 
$(3-r)^2 = ( \frac{3}{2} )^2 + ( \frac{r}{2} )^2$ ; 
$4(3-r)^2 = 9+r^2$ 
$3r^2 -24r+27=0$ ; 
$r^2 -8r+9=0$ ; 
$r= \frac{12 \pm \sqrt{63}}{3} = 4 \pm \sqrt{7}$ 

$r < 3$이므로 
$r = 4 - \sqrt{7}$ 
따라서 큰 원과 작은 원의 닮음비가 $3 : 4-\sqrt{7}$이므로 공비는 $\frac{4-\sqrt{7}}{3}$이다. 한편, 오른쪽 그림의 

> [!example] 📸 (그림/그래프)

삼각형에서 세 변의 길이는 모두 $3$이다. 따라서 정삼각형이므로 도형 $F_1$의 둘레의 길이 
$l_1 = ( 3 \times \frac{\pi}{3} ) \times 4 = 4\pi$ 
$\therefore \sum_{n=1}^{\infty} l_n = \frac{4\pi}{1 - \frac{4-\sqrt{7}}{3}} = \frac{12\pi}{-1+\sqrt{7}} = 2\pi (1+\sqrt{7})$ 

[별해] 
도형 $F_1$의 두 원의 중심을 연결하는 선분과 도형 $F_2$의 두 원의 중심을 연결하는 선분은 서로 다른 것을 수직 이등분한다. 

> [!example] 📸 (그림/그래프)

도형 $F_2$의 반지름의 길이를 $x$라 하면 위 그림의 직각삼각형 $O_1 O_2 H$에서 
$(3-x)^2 = ( \frac{3}{2} )^2 + ( \frac{x}{2} )^2$, 
$3x^2 -24x +27=0$ 
$x^2 -8x+9=0$ $\therefore x = 4- \sqrt{7}$ ($\because x < 3$) 
따라서 서로 닮음인 도형 $F_1$과 $F_2$의 닮음비는 $\frac{4- \sqrt{7}}{3}$이다. 

> [!example] 📸 (그림/그래프)

위 그림에서 $l_1 = 2 \times ( 3 \times \frac{2}{3} \pi ) = 4\pi$이므로 수열 $\{l_n\}$은 첫째항이 $4\pi$이고 공비가 $\frac{4-\sqrt{7}}{3}$인 등비수열이다. 
$\therefore \sum_{n=1}^{\infty} l_n = \frac{4\pi}{1 - \frac{4-\sqrt{7}}{3}} = 2\pi (1+ \sqrt{7})$
