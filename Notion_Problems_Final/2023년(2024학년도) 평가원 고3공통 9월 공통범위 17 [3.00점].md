## 문제
두 수열 $\{ a_{n} \}$, $\{ b_{n} \}$ 에 대하여 
$\sum_{k=1}^{10} ( 2a_{k} - b_{k} ) = 34$, $\sum_{k=1}^{10} a_{k} = 10$ 
일 때, $\sum_{k=1}^{10} ( a_{k} - b_{k} )$ 의 값을 구하시오.

## 풀이
$\sum_{k=1}^{10} ( 2a_{k} - b_{k} ) = 34$, $\sum_{k=1}^{10} a_{k} = 10$ 에서 
$\sum_{k=1}^{10} ( a_{k} - b_{k} ) = \sum_{k=1}^{10} \{ ( 2a_{k} - b_{k} ) - a_{k} \}$ 
$= \sum_{k=1}^{10} ( 2a_{k} - b_{k} ) - \sum_{k=1}^{10} a_{k}$ 
$= 34 - 10 = 24$

정답: $24$
