## 문제
모든 실수 $x$ 에서 정의된 함수 $f(x) = \int_{1}^{x} (x^{2} - t) dt$ 에 대하여 직선 $y = 6x - k$ 가 곡선 $y = f(x)$ 에 접할 때, 양수 $k$ 의 값은?

## 보기
① $\frac{11}{2}$   ② $\frac{13}{2}$   ③ $\frac{15}{2}$   ④ $\frac{17}{2}$   ⑤ $\frac{19}{2}$

## 풀이
$f(x) = \int_{1}^{x} (x^{2} - t) dt = x^{2} \int_{1}^{x} dt - \int_{1}^{x} t dt$ 
$= x^{2} [ t ]_{1}^{x} - [ \frac{1}{2} t^{2} ]_{1}^{x} = x^{2} (x - 1) - \frac{1}{2} (x^{2} - 1)$ 
$= x^{3} - \frac{3}{2} x^{2} + \frac{1}{2}$

직선 $y = 6x - k$ 와 곡선 $y = f(x)$ 의 접점의 좌표를 $P(t, t^{3} - \frac{3}{2} t^{2} + \frac{1}{2})$ 라 놓으면 접선의 기울기가 $6$ 이므로 
$f'(t) = 6$
$3t^{2} - 3t = 6$
$(t - 2)(t + 1) = 0$
$\therefore t = 2, -1$
$\therefore P(2, \frac{5}{2}), P(-1, -2)$

점 $P$ 는 직선 위의 점이므로 $P(2, \frac{5}{2})$ 일 때,
$\frac{5}{2} = 6(2) - k$
$k = \frac{19}{2}$

$P(-1, -2)$ 일 때,
$-2 = 6(-1) - k$
$k = -4$

$\therefore k > 0$ 이므로 $k = \frac{19}{2}$

[별해]
$f(x) = \int_{1}^{x} (x^{2} - t) dt$
$= [ x^{2} t - \frac{1}{2} t^{2} ]_{1}^{x}$
$= x^{2} (x - 1) - \frac{1}{2} (x^{2} - 1)$
$= x^{3} - \frac{3}{2} x^{2} + \frac{1}{2}$

$\therefore f'(x) = 3x^{2} - 3x$

$f'(x) = 6$ 에서 $x^{2} - x - 2 = 0$
$\therefore x = -1, 2$

$f(-1) = -1 - \frac{3}{2} + \frac{1}{2} = -2$
$f(2) = 8 - 6 + \frac{1}{2} = \frac{5}{2}$

이므로 함수 $f(x)$ 에 접하는 기울기가 $6$ 인 직선의 방정식은 
$y = 6(x + 1) - 2 = 6x + 4$ 또는 $y = 6(x - 2) + \frac{5}{2} = 6x - \frac{19}{2}$ 이다. 
따라서 양수 $k$ 의 값은 $\frac{19}{2}$ 이다.
