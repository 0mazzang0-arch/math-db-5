## 문제
> [!example] 📸 (그림/그래프)

그림과 같이 두 곡선 $y=3^{x}$과 $y=a^{x}$ $(0 < a < 1)$의 교점을 $P$라 하고, 직선 $y=3$이 두 곡선 $y=3^{x}$, $y=a^{x}$과 만나는 점을 각각 $Q$, $R$라 하자. $\angle RPQ = 90^{\circ}$일 때, $(\frac{1}{a})^{16}$의 값을 구하시오.

## 보기

## 풀이
두 곡선 $y=3^{x}$과 $y=a^{x}$은 점 $(0, 1)$에서 만나므로 점 $P$의 좌표는 $P(0, 1)$이다. 
곡선 $y=3^{x}$ 위의 점 $Q$의 좌표를 $Q(t, 3)$이라 하면 $3^{t} = 3$에서 $t=1$이므로 $Q(1, 3)$이다.
직선 $PQ$의 기울기는 $\frac{3-1}{1-0} = 2$이고 $\angle RPQ = 90^{\circ}$이므로 직선 $PR$의 기울기는 $-\frac{1}{2}$이다.
두 점 $P$, $R$를 지나는 직선의 방정식은 $y = -\frac{1}{2}x + 1$이다. 
점 $R$의 $y$좌표가 $3$이므로 $3 = -\frac{1}{2}x + 1$에서 $2 = -\frac{1}{2}x$, $x = -4$이다. 
따라서 점 $R$의 좌표는 $R(-4, 3)$이다. 
또한 점 $R$는 곡선 $y=a^{x}$ 위의 점이므로 $3 = a^{-4}$가 성립한다.
$(\frac{1}{a})^{4} = 3$이므로, $(\frac{1}{a})^{16} = \{(\frac{1}{a})^{4}\}^{4} = 3^{4} = 81$이다.

**[별해]**
두 점 $P$, $Q$의 좌표는 $P(0, 1)$, $Q(1, 3)$이다.
이때, $\overline{PQ} = \sqrt{(1-0)^{2} + (3-1)^{2}} = \sqrt{5}$이다.
곡선 $y=a^{x}$ 위의 점 $R$의 좌표를 $R(b, 3)$이라 하면 $a^{b} = 3 \cdots (*)$이다.
이때, $\overline{RQ} = 1-b$이고 $\overline{RP} = \sqrt{(b-0)^{2} + (3-1)^{2}} = \sqrt{b^{2} + 4}$이다.
삼각형 $PQR$는 직각삼각형이므로 피타고라스의 정리에 의해
$\overline{RQ}^{2} = \overline{PQ}^{2} + \overline{RP}^{2}$
$\Leftrightarrow (1-b)^{2} = 5 + (b^{2} + 4)$
$\Leftrightarrow 1-2b+b^{2} = b^{2} + 9$
$\Leftrightarrow -2b = 8$
$\therefore b = -4$
$b = -4$를 $(*)$에 대입하면 $a^{-4} = 3$이다.
$\therefore (\frac{1}{a})^{16} = (a^{-4})^{4} = 3^{4} = 81$

정답: $81$
