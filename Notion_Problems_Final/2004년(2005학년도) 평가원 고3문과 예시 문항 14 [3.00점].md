## 문제
$a \neq 1$이므로 $log_{c} a \neq 0$이다. 다음은 $log_{a} b$를 임의의 양수 $c$($c \neq 1$)를 밑으로 하는 로그로 바꾸어 나타낼 수 있음을 증명한 것이다.

$$ \begin{array}{|l|} \hline log_{a} b = x, \ log_{c} a = y \text{라고 하면} \\ a^{x} = b, \ c^{y} = a \text{이다.} \\ \text{이때, } b = c^{(가)} \text{이므로} \\ (가) = log_{c} b \text{이다.} \\ \text{즉, } log_{a} b \cdot log_{c} a = log_{c} b \text{이다.} \\ \text{여기서 (나) 이므로 } log_{c} a \neq 0 \text{이다.} \\ \text{따라서 } log_{a} b = \frac{log_{c} b}{log_{c} a} \text{이다.} \\ \hline \end{array} $$

위의 증명에서 $(가)$와 $(나)$에 알맞은 것은?

## 보기
① $(가) \ xy, \ (나) \ a \neq 1$   ② $(가) \ xy, \ (나) \ a > 0$   ③ $(가) \ x + y, \ (나) \ a \neq 1$   ④ $(가) \ x + y, \ (나) \ a > 0$   ⑤ $(가) \ \frac{x}{y}, \ (나) \ a \neq 1$

## 풀이
[정답] ①

[해설]
$$ \begin{array}{l} a^{x} = b, \ c^{y} = a \text{ 이므로} \\ b = a^{x} = (c^{y})^{x} = c^{xy} \\ \text{그러므로 } (가) = xy \text{ 이고,} \\ xy = log_{c} b \text{ 가 성립한다.} \\ \\ log_{a} b \cdot log_{c} a = log_{c} b \text{ 에서} \\ log_{a} b = \frac{log_{c} b}{log_{c} a} \text{ 가 정의되려면 } log_{c} a \neq 0 \text{ 이어야 한다.} \\ \text{로그의 밑 조건에 의해 } a > 0, \ a \neq 1 \text{ 이므로} \\ (나) \text{ 에 알맞은 조건은 } a \neq 1 \text{ 이다.} \end{array} $$
