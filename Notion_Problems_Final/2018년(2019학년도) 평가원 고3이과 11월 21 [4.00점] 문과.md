## 문제
실수 전체의 집합에서 미분가능한 함수 $f(x)$가 다음 조건을 만족시킬 때, $f(-1)$의 값은?

$$
\begin{array}{l}
\text{(가) 모든 실수 } x \text{에 대하여 } 2 \{ f(x) \}^2 f'(x) = \{ f(2x+1) \}^2 f'(2x+1) \text{이다.} \\
\text{(나) } f(-\frac{1}{8}) = 1, f(6) = 2
\end{array}
$$

## 보기
① $\frac{\sqrt[3]{3}}{6}$   ② $\frac{\sqrt[3]{3}}{3}$   ③ $\frac{\sqrt[3]{3}}{2}$   ④ $\frac{2\sqrt[3]{3}}{3}$   ⑤ $\frac{5\sqrt[3]{3}}{6}$

## 풀이
(가)에서 $\{f(x)\}^3$을 미분하면 $3\{f(x)\}^2 \cdot f'(x)$이고, $\{f(2x+1)\}^3$을 미분하면 $3\{f(2x+1)\}^2 \cdot f'(2x+1) \cdot 2$이다.
따라서 주어진 식 $2\{f(x)\}^2 f'(x) = \{f(2x+1)\}^2 f'(2x+1)$을 적분하기 위해 양변에 $6$을 곱하면 다음과 같다.
$6\{f(x)\}^2 f'(x) = 3\{f(2x+1)\}^2 f'(2x+1) \cdot 2$
이 식을 적분하면
$2\{f(x)\}^3 = \frac{1}{2}\{f(2x+1)\}^3 + C'$
$4\{f(x)\}^3 = \{f(2x+1)\}^3 + C$ $\cdots$ ㉠

㉠에 각 값을 대입하여 식을 세우면 다음과 같다.
$$
\begin{array}{lll}
x = -\frac{1}{8} \text{ 대입:} & 4\{f(-\frac{1}{8})\}^3 = \{f(\frac{3}{4})\}^3 + C & \implies 4 = \{f(\frac{3}{4})\}^3 + C \quad (\because f(-\frac{1}{8}) = 1) \cdots ㉡ \\
x = \frac{5}{2} \text{ 대입:} & 4\{f(\frac{5}{2})\}^3 = \{f(6)\}^3 + C & \implies 4\{f(\frac{5}{2})\}^3 = 8 + C \quad (\because f(6) = 2) \cdots ㉢ \\
x = \frac{3}{4} \text{ 대입:} & 4\{f(\frac{3}{4})\}^3 = \{f(\frac{5}{2})\}^3 + C & \cdots ㉣
\end{array}
$$

㉣에 ㉡과 ㉢에서 구한 $\{f(\frac{3}{4})\}^3 = 4 - C$와 $\{f(\frac{5}{2})\}^3 = \frac{8+C}{4}$를 대입하면
$4(4 - C) = \frac{8 + C}{4} + C$
$16 - 4C = \frac{8 + C + 4C}{4}$
$64 - 16C = 8 + 5C$
$21C = 56$
$C = \frac{56}{21} = \frac{8}{3}$

따라서 관계식은 $4\{f(x)\}^3 = \{f(2x+1)\}^3 + \frac{8}{3}$이다.
이 식에 $x = -1$을 대입하면
$4\{f(-1)\}^3 = \{f(-2+1)\}^3 + \frac{8}{3}$
$4\{f(-1)\}^3 = \{f(-1)\}^3 + \frac{8}{3}$
$3\{f(-1)\}^3 = \frac{8}{3}$
$\{f(-1)\}^3 = \frac{8}{9}$
$f(-1) = \sqrt[3]{\frac{8}{9}} = \frac{2}{\sqrt[3]{9}} = \frac{2 \cdot \sqrt[3]{3}}{\sqrt[3]{9} \cdot \sqrt[3]{3}} = \frac{2\sqrt[3]{3}}{3}$

정답은 ④이다.
