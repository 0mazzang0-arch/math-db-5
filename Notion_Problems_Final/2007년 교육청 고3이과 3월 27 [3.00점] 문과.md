## 문제
두 연속확률변수 $X$, $Y$가 갖는 값의 범위는 각각 $0 \le X \le 4$, $0 \le Y \le 4$이고, $X$의 확률밀도함수 $f(x)$와 $Y$의 확률밀도함수 $g(x)$는 다음과 같다.
$$ f(x) = ax(x-4) \quad (0 \le x \le 4) $$
$$ g(x) = \begin{cases} b & (0 \le x \le 2) \\ f(x-2) + b & (2 \le x \le 4) \end{cases} $$
<보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $a$, $b$는 상수이다.)

---
**<보기>**
ㄱ. $P(2 \le X \le 4) = \frac{1}{2}$
ㄴ. $b = \frac{1}{8}$
ㄷ. $P(1 \le Y \le 4) = \frac{5}{8}$



> [!example] 📸 (그림/그래프)

## 보기
① ㄱ   ② ㄱ, ㄴ   ③ ㄱ, ㄷ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
[정답] ②

ㄱ. 이차곡선 $f(x) = ax(x-4)$는 $x = 2$에 대하여 대칭이므로
$P(0 \le X \le 2) = P(2 \le X \le 4) = \frac{1}{2}$ (참)

ㄴ. 곡선 $y = f(x-2) + b$는 곡선 $y = f(x)$를 $x$축의 방향으로 $2$만큼, $y$축의 방향으로 $b$만큼 평행 이동한 것이다. 확률밀도함수의 성질에 의해 $\int_{0}^{4} g(x) dx = 1$이므로
$P(0 \le Y \le 4) = \int_{0}^{2} b dx + \int_{2}^{4} \{f(x-2) + b\} dx$
$= 2b + \int_{0}^{2} f(t) dt + 2b = 1$ (단, $t = x-2$)
$4b + P(0 \le X \le 2) = 1$
$4b + \frac{1}{2} = 1$
$\therefore b = \frac{1}{8}$ (참)

ㄷ. $P(1 \le Y \le 4) = 1 - P(0 \le Y \le 1)$
$0 \le x \le 2$에서 $g(x) = b = \frac{1}{8}$이므로
$P(0 \le Y \le 1) = \int_{0}^{1} \frac{1}{8} dx = \frac{1}{8}$
$P(1 \le Y \le 4) = 1 - \frac{1}{8} = \frac{7}{8}$ (거짓)

[참고]
곡선 $y = f(x)$와 $x$축 사이의 넓이와 곡선 $y = g(x)$와 $x$축 사이의 넓이는 모두 $1$이다. $P(\alpha \le X \le \beta)$는 구간 $\alpha \le x \le \beta$에서 $y = f(x)$와 $x$축 사이의 넓이이다.
