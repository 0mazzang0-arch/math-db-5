## 문제

이차함수 $f(x) = k(x-1)^2 - 4k+2$ 에 대하여 곡선 $y=f(x)$ 의 꼭짓점을 $A$ 라 하고, 이 곡선이 $y$ 축과 만나는 점을 $B$ 라 하자. <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $O$ 는 원점이다.)

$$ \begin{array}{|l|} \hline \text{<보기>} \\ \text{ㄱ. } k=1 \text{ 일 때, } \overline{OA} = \sqrt{5} \text{ 이다.} \\ \text{ㄴ. } 0 \text{ 이 아닌 실수 } k \text{ 의 값에 관계없이 곡선 } y=f(x) \text{ 가 항상 지나는 점은 } 2 \text{ 개이다.} \\ \text{ㄷ. } 0 \text{ 이 아닌 실수 } k \text{ 의 값에 관계없이 직선 } AB \text{ 는 항상 점 } (-3, 2) \text{ 를 지난다.} \\ \hline \end{array} $$

## 보기

① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이

ㄱ. (참) 
$k=1$ 일 때 
$f(x) = (x-1)^2 - 2$ 이므로 
$A(1, -2)$ 
$\therefore \overline{OA} = \sqrt{1^2 + (-2)^2} = \sqrt{5}$ 

ㄴ. (참) 
$y = k(x-1)^2 - 4k + 2$ 를 $k$ 에 대하여 정리하면 
$k \{(x-1)^2 - 4\} + 2 - y = 0$ 
$0$ 이 아닌 실수 $k$ 의 값에 관계없이 성립하려면 
$(x-1)^2 = 4$ 
$2 - y = 0$ 
$x-1 = \pm 2$ 에서 $x=3$ 또는 $x=-1$ 
$y = 2$ 
따라서 $y=f(x)$ 는 $0$ 이 아닌 실수 $k$ 의 값에 관계없이 항상 두 점 $(3, 2)$, $(-1, 2)$ 를 지난다. 

ㄷ. (참) 
$A(1, -4k+2)$, $B(0, -3k+2)$ 이고 
직선 $AB$ 의 기울기는 
$\frac{(-4k+2) - (-3k+2)}{1-0} = -k$ 
직선 $AB$ 의 방정식은 
$y = -kx - 3k + 2$ 
$k$ 에 대하여 정리하면 
$k(x+3) + y - 2 = 0$ 
$0$ 이 아닌 실수 $k$ 의 값에 관계없이 성립하려면 
$x+3 = 0$, $y-2 = 0$ 
즉, $x=-3$, $y=2$ 
따라서 직선 $AB$ 는 $0$ 이 아닌 실수 $k$ 의 값에 관계없이 항상 점 $(-3, 2)$ 를 지난다. 

이상에서 옳은 것은 ㄱ, ㄴ, ㄷ이다.
