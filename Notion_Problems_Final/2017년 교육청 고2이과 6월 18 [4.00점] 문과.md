## 문제

수열 $\{ a_{n} \}$ 의 일반항은 $a_{n}= \frac{1}{n}$ 일 때, 다음은 모든 자연수 $n$ 에 대하여 
$$\sum_{k=1}^{n} (a_{k} a_{k+1} )^{2}= \sum_{k=1}^{n} (a_{k} )^{2}+ \sum_{k=1}^{n} (a_{k+1} )^{2}+2 (a_{n+1}-1 ) \cdots \cdots (*)$$
이 성립함을 수학적 귀납법으로 증명하는 과정의 일부이다.

(i) $n=1$ 일 때,
(좌변) $= (가) =$ (우변) 이므로 $(*)$ 이 성립한다.

(ii) $n=m(m \geq 1)$ 일 때, $(*)$ 이 성립한다고 가정하면
$$\sum_{k=1}^{m} (a_{k} a_{k+1} )^{2}= \sum_{k=1}^{m} (a_{k} )^{2}+ \sum_{k=1}^{m} (a_{k+1} )^{2}+2 (a_{m+1}-1 )$$
이므로
$$\sum_{k=1}^{m+1} (a_{k} a_{k+1} )^{2}= \sum_{k=1}^{m} (a_{k} a_{k+1} )^{2}+ (a_{m+1} a_{m+2} )^{2}$$
$$= \sum_{k=1}^{m} (a_{k} )^{2}+ \sum_{k=1}^{m} (a_{k+1} )^{2}+2 (a_{m+1}-1 )+ (a_{m+1} a_{m+2} )^{2}$$
$$= \sum_{k=1}^{m} (a_{k} )^{2}+ \sum_{k=1}^{m} (a_{k+1} )^{2}+2 ( \frac{1}{m+1}-1 )+ \{ \frac{1}{(m+1)(m+2)} \}^{2}$$
$$= \sum_{k=1}^{m+1} (a_{k} )^{2}+ \sum_{k=1}^{m+1} (a_{k+1} )^{2}+2 ( \frac{1}{m+1}-1- (나) )$$
$$\vdots$$
따라서 $n=m+1$ 일 때도 $(*)$ 이 성립한다.

(i), (ii)에 의하여 모든 자연수 $n$ 에 대하여 $(*)$ 이 성립한다.

위의 과정에서 (가)에 알맞은 수를 $p$, (나)에 알맞은 식을 $f(m)$ 이라 할 때, $\frac{p}{f(14)}$ 의 값은?

## 보기

① $60$   ② $62$   ③ $64$   ④ $66$   ⑤ $68$

## 풀이

(i) $n=1$ 일 때,
(좌변) $= \sum_{k=1}^{1} (a_{k} a_{k+1} )^{2}= (a_{1} a_{2})^{2} = (1 \cdot \frac{1}{2})^{2} = \frac{1}{4}$
(우변) $= \sum_{k=1}^{1} (a_{k} )^{2}+ \sum_{k=1}^{1} (a_{k+1} )^{2}+2 (a_{2}-1 )= 1^{2} + (\frac{1}{2})^{2} + 2(\frac{1}{2}-1) = 1 + \frac{1}{4} - 1 = \frac{1}{4}$
이므로 $(*)$ 이 성립한다. 따라서 $p = \frac{1}{4}$ 이다.

(ii) $n=m(m \geq 1)$ 일 때, $(*)$ 이 성립한다고 가정하면
$$\sum_{k=1}^{m+1} (a_{k} a_{k+1} )^{2}= \sum_{k=1}^{m} (a_{k} )^{2}+ \sum_{k=1}^{m} (a_{k+1} )^{2}+2 ( \frac{1}{m+1}-1 )+ ( \frac{1}{m+1} - \frac{1}{m+2} )^{2}$$
여기서 $\sum_{k=1}^{m+1} (a_{k} )^{2} = \sum_{k=1}^{m} (a_{k} )^{2} + ( \frac{1}{m+1} )^{2}$ 이고, $\sum_{k=1}^{m+1} (a_{k+1} )^{2} = \sum_{k=1}^{m} (a_{k+1} )^{2} + ( \frac{1}{m+2} )^{2}$ 이다.
이를 대입하여 정리하면
$$\sum_{k=1}^{m+1} (a_{k} a_{k+1} )^{2} = \sum_{k=1}^{m+1} (a_{k} )^{2} - ( \frac{1}{m+1} )^{2} + \sum_{k=1}^{m+1} (a_{k+1} )^{2} - ( \frac{1}{m+2} )^{2} + 2 ( \frac{1}{m+1}-1 ) + ( \frac{1}{m+1} - \frac{1}{m+2} )^{2}$$
$$= \sum_{k=1}^{m+1} (a_{k} )^{2} + \sum_{k=1}^{m+1} (a_{k+1} )^{2} - \frac{1}{(m+1)^{2}} - \frac{1}{(m+2)^{2}} + 2 ( \frac{1}{m+1}-1 ) + \frac{1}{(m+1)^{2}} - \frac{2}{(m+1)(m+2)} + \frac{1}{(m+2)^{2}}$$
$$= \sum_{k=1}^{m+1} (a_{k} )^{2} + \sum_{k=1}^{m+1} (a_{k+1} )^{2} + 2 ( \frac{1}{m+1}-1 ) - \frac{2}{(m+1)(m+2)}$$
$$= \sum_{k=1}^{m+1} (a_{k} )^{2} + \sum_{k=1}^{m+1} (a_{k+1} )^{2} + 2 ( \frac{1}{m+1}-1 - \frac{1}{(m+1)(m+2)} )$$
따라서 $f(m) = \frac{1}{(m+1)(m+2)}$ 이다.

$p = \frac{1}{4}$ 이고 $f(14) = \frac{1}{15 \cdot 16} = \frac{1}{240}$ 이므로
$$\frac{p}{f(14)} = \frac{1/4}{1/240} = \frac{240}{4} = 60$$
정답은 ① 이다.
