## 문제
함수 $f(x) = \lim_{n \to \infty} \frac{x^{2n+1} + ax^2 + bx - 2}{x^{2n} + 1}$ 가 실수 전체의 집합에서 연속일 때, 두 상수 $a$, $b$의 곱 $ab$의 값은?

## 보기
① $-2$   ② $-1$   ③ $0$   ④ $1$   ⑤ $2$

## 풀이
(ⅰ) $|x| < 1$ 일 때, 
$\lim_{n \to \infty} x^{2n} = 0$ 이므로 
$f(x) = ax^2 + bx - 2$

(ⅱ) $|x| > 1$ 일 때, 
$\lim_{n \to \infty} \frac{1}{x^{2n}} = 0$ 이므로 
$f(x) = \lim_{n \to \infty} \frac{x + \frac{ax^2}{x^{2n}} + \frac{bx}{x^{2n}} - \frac{2}{x^{2n}}}{1 + \frac{1}{x^{2n}}} = x$

(ⅲ) $x = 1$ 일 때, 
$f(1) = \frac{a + b - 1}{2}$

(ⅳ) $x = -1$ 일 때, 
$f(-1) = \frac{a - b - 3}{2}$ 

함수 $f(x)$ 가 실수 전체의 집합에서 연속이므로 $x = -1$, $x = 1$ 에서 연속이다. 

① $x = -1$ 에서 연속이므로 
$a - b - 2 = -1 = \frac{a - b - 3}{2}$ 
$\therefore a - b = 1 \cdots$ ㉠ 

② $x = 1$ 에서 연속이므로 
$a + b - 2 = 1 = \frac{a + b - 1}{2}$ 
$\therefore a + b = 3 \cdots$ ㉡ 

㉠, ㉡에서 $a = 2$, $b = 1$ 
따라서 $ab = 2$ 

정답: ⑤
