## 문제
$A \triangle B = (A - B) \cup (B - A)$로 정의할 때, $\{ a, b, c \} \triangle X = Y$를 만족하는 집합 $X$, $Y$에 대한 다음 설명 중 옳지 않은 것은? (단, $U = \{ a, b, c, d \}$)



> [!example] 📸 (그림/그래프)

## 보기
① $X = \{ a, b, c \}$이면 $Y = \emptyset$이다.
② $X = \{ d \}$이면 $Y = U$이다.
③ $X = \{ c \}$이면 $Y = \{ a, b \}$이다.
④ $X = U$이면 $Y = \{ a, b, c \}$이다.
⑤ $X = \{ c, d \}$이면 $Y = \{ a, b, d \}$이다.

## 풀이
집합 $A \triangle B = (A - B) \cup (B - A)$은 대칭차집합이므로 즉, $A$와 $B$의 합집합에서 교집합을 뺀 집합이다.

④에서 $X = U$이면 $\{ a, b, c \} \triangle \{ a, b, c, d \} = \{ d \}$이다. 
전체집합을 $U = \{ a, b, c, d \}$라 하고, $U$의 두 부분집합 $A$, $B$에 대하여 연산 $\triangle$을 정의할 때, ④의 결과는 $Y = \{ d \}$가 되어야 하므로 옳지 않다.

정답: ④
