## 문제
두 사건 $A$, $B$에 대하여 $P(A) = 0.4$, $P(B) = 0.5$, $P(A \cup B) = 0.8$일 때, $P(A^C | B) + P(A | B^C)$의 값은?

## 보기
① $1.1$   ② $1.2$   ③ $1.3$   ④ $1.4$   ⑤ $1.5$

## 풀이
덧셈정리에 의해
$P(A \cup B) = P(A) + P(B) - P(A \cap B)$
$0.8 = 0.4 + 0.5 - P(A \cap B)$
$P(A \cap B) = 0.1$

조건부확률의 정의에 의해
$P(A^C | B) = \frac{P(A^C \cap B)}{P(B)} = \frac{P(B) - P(A \cap B)}{P(B)} = \frac{0.5 - 0.1}{0.5} = \frac{0.4}{0.5} = \frac{4}{5}$
$P(A | B^C) = \frac{P(A \cap B^C)}{P(B^C)} = \frac{P(A) - P(A \cap B)}{1 - P(B)} = \frac{0.4 - 0.1}{1 - 0.5} = \frac{0.3}{0.5} = \frac{3}{5}$

따라서 구하는 값은
$P(A^C | B) + P(A | B^C) = \frac{4}{5} + \frac{3}{5} = \frac{7}{5} = 1.4$
