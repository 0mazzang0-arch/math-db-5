## 문제
$\lim _{x \to 2} \frac{x^{2} + x - 6}{\sqrt{x + 2} - 2}$ 의 값을 구하시오.

## 풀이
$\lim _{x \to 2} \frac{x^{2} + x - 6}{\sqrt{x + 2} - 2}$
$= \lim _{x \to 2} \frac{(x^{2} + x - 6)(\sqrt{x + 2} + 2)}{(\sqrt{x + 2} - 2)(\sqrt{x + 2} + 2)}$
$= \lim _{x \to 2} \frac{(x - 2)(x + 3)(\sqrt{x + 2} + 2)}{x - 2}$
$= (2 + 3)(\sqrt{2 + 2} + 2)$
$= 5 \times 4$
$= 20$

정답: $20$
