## 문제
$a_{2} = -4$이고 공차가 $0$이 아닌 등차수열 $\{ a_{n} \}$에 대하여 수열 $\{ b_{n} \}$을 $b_{n} = a_{n} + a_{n+1}$ ($n \geq 1$)이라 하자. 두 집합 $A, B$를
$$ A = \{ a_{1}, a_{2}, a_{3}, a_{4}, a_{5} \}, B = \{ b_{1}, b_{2}, b_{3}, b_{4}, b_{5} \} $$
라 하자. $n(A \cap B) = 3$이 되도록 하는 모든 수열 $\{ a_{n} \}$에 대하여 $a_{20}$의 값의 합은?

## 보기
① $30$   ② $34$   ③ $38$   ④ $42$   ⑤ $46$

## 풀이
등차수열 $\{ a_{n} \}$의 공차를 $d$라 하면 $a_{2} = -4$이므로
$a_{1} + d = -4$ ($d \neq 0$) $\cdots \cdots$ ㉠
또한 수열 $\{ b_{n} \}$을 $b_{n} = a_{n} + a_{n+1}$ ($n \geq 1$)이라 하면 두 집합 $A, B$를
$A = \{ a_{1}, a_{1} + d, a_{1} + 2d, a_{1} + 3d, a_{1} + 4d \}$
$B = \{ 2a_{1} + d, 2a_{1} + 3d, 2a_{1} + 5d, 2a_{1} + 7d, 2a_{1} + 9d \}$
로 놓을 수 있고 수열 $\{ b_{n} \}$은 첫째항이 $2a_{1} + d$, 공차가 $2d$인 등차수열이다. 이때 $n(A \cap B) = 3$이 되려면 다음과 같이 세 경우를 생각해 볼 수 있다.

(ⅰ) $a_{1} = b_{1}$일 때
$a_{1} = 2a_{1} + d$에서 $a_{1} = -d$이므로 $a_{1} + d = 0$이 되어 조건 ㉠에 위배된다.

(ⅱ) $a_{1} = b_{2}$일 때
$a_{1} = 2a_{1} + 3d$에서 $a_{1} = -3d$
㉠에 대입하면 $-3d + d = -4$, $-2d = -4$
$\therefore d = 2, a_{1} = -6$
$\therefore a_{20} = -6 + 19 \times 2 = 32$

(ⅲ) $a_{1} = b_{3}$일 때
$a_{1} = 2a_{1} + 5d$에서 $a_{1} = -5d$
㉠에 대입하면 $-5d + d = -4$, $-4d = -4$
$\therefore d = 1, a_{1} = -5$
$\therefore a_{20} = -5 + 19 \times 1 = 14$

이상에서 조건을 만족하는 모든 수열 $\{ a_{n} \}$에 대하여 $a_{20}$의 값의 합은
$32 + 14 = 46$
