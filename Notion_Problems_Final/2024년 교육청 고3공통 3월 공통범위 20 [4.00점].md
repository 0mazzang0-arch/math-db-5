## 문제
두 함수 $f(x) = 2x^{2} + 2x - 1$, $g(x) = \cos \frac{\pi}{3}x$에 대하여 $0 \le x < 12$에서 방정식 $f(g(x)) = g(x)$를 만족시키는 모든 실수 $x$의 값의 합을 구하시오.

## 풀이
$f(g(x)) = g(x)$에서 $g(x) = t$ ($-1 \le t \le 1$)이라 하면
$f(t) = t$에서 $2t^{2} + 2t - 1 = t$,
$(2t - 1)(t + 1) = 0$
$t = \frac{1}{2}$ 또는 $t = -1$이므로 $g(x) = \frac{1}{2}$ 또는 $g(x) = -1$이다.

함수 $g(x) = \cos \frac{\pi}{3}x$의 주기는 $6$이고,
$g(1) = g(5) = \frac{1}{2}$, $g(3) = -1$이다.

> [!example] 📸 (그림/그래프)

그러므로 $0 \le x < 12$에서
$g(7) = g(11) = \frac{1}{2}$, $g(9) = -1$이다.
따라서 구하는 모든 실수 $x$의 값의 합은
$1 + 3 + 5 + 7 + 9 + 11 = 36$이다.

정답: $36$
