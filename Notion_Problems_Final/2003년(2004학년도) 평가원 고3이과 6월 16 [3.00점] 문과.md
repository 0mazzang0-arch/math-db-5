## 문제
복소평면에서 세 복소수 $z_{1}$, $z_{2}$, $z_{3}$는 다음을 만족한다.

$$
\begin{array}{l}
\text{(가) } |z_{1}-3 i |=1 \\
\text{(나) } |z_{2}-3 i |= \sqrt{2} \\
\text{(다) } |z_{3}-5 i |=1
\end{array}
$$

$\arg z_{1}$, $\arg z_{2}$, $\arg z_{3}$의 최댓값을 각각 $\theta_{1}$, $\theta_{2}$, $\theta_{3}$라고 할 때, $\theta_{1}$, $\theta_{2}$, $\theta_{3}$의 대소 관계를 옳게 나타낸 것은? 

(단, $i= \sqrt{-1}$ 이고, $\arg z_{k}$ ($k=1, 2, 3$)는 $z_{k}$의 편각을 나타내며 $0 \le \arg z_{k} < 2 \pi$로 한다.)

## 보기
① $\theta_{1}= \theta_{2}< \theta_{3}$   ② $\theta_{1}< \theta_{2}< \theta_{3}$   ③ $\theta_{2}< \theta_{3}< \theta_{1}$   ④ $\theta_{2}< \theta_{1}= \theta_{3}$   ⑤ $\theta_{3}< \theta_{1}< \theta_{2}$

## 풀이
[정답] ⑤

복소평면에서 복소수 $z_1$은 중심이 $(0, 3)$, 반지름의 길이가 $1$인 원의 자취이고, $z_2$는 중심이 $(0, 3)$, 반지름의 길이가 $\sqrt{2}$인 원의 자취를 나타내며, $z_3$은 중심이 $(0, 5)$, 반지름의 길이가 $1$인 원의 자취이다. 

따라서, 복소평면 위에 복소수 $z_1$, $z_2$, $z_3$를 나타내고 $\arg z_1$, $\arg z_2$, $\arg z_3$의 최댓값을 생각하면 원점으로부터 각 원에 그은 접선의 기울기와 관련이 있다.

각 원의 중심에서 $y$축과 이루는 각을 $\alpha_k$라 하면 $\theta_k = \frac{\pi}{2} + \alpha_k$이고 $\sin \alpha_k = \frac{r_k}{d_k}$이므로 다음과 같다.
- $\sin \alpha_1 = \frac{1}{3}$
- $\sin \alpha_2 = \frac{\sqrt{2}}{3}$
- $\sin \alpha_3 = \frac{1}{5}$

$\sin \alpha_3 < \sin \alpha_1 < \sin \alpha_2$이므로 $\alpha_3 < \alpha_1 < \alpha_2$이다.
그러므로 $\theta_1$, $\theta_2$, $\theta_3$의 대소 관계는 다음과 같다.
$$\theta_3 < \theta_1 < \theta_2$$
