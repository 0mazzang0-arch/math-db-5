## 문제
자연수 $m$부터 시작하여 $n$개의 연속한 자연수의 합을 $3^{k}$과 같이 $3$의 거듭제곱으로 나타낼 수 있을 때, $<< m, n >> = k$로 나타내기로 한다. 

$1 + 2 = 3 = 3^{1}$이므로 $<< 1, 2 >> = 1$이고, 
$2 + 3 + 4 = 9 = 3^{2}$이므로 $<< 2, 3 >> = 2$이다. 

$<< 4, a >> = 2$, $<< b, 6 >> = 3$을 만족시키는 자연수 $a$, $b$에 대하여 $10a + b$의 값을 구하시오.

## 풀이
$<< 4, a >> = 2$에서 $4$부터 시작하여 $a$개의 연속한 자연수의 합이 $3^{2} = 9$가 되어야 한다.
$4 + 5 = 9 = 3^{2}$이므로 $a = 2$이다.

$<< b, 6 >> = 3$에서 $b$부터 시작하여 $6$개의 연속한 자연수의 합이 $3^{3} = 27$이 되어야 한다.
$b + (b + 1) + (b + 2) + (b + 3) + (b + 4) + (b + 5) = 27$
$6b + 15 = 27$
$6b = 12$
$b = 2$

따라서 $10a + b = 10 \times 2 + 2 = 22$이다.
