## 문제
$4^{x} = 5$ 일 때, $\frac{8^{x} + 8^{-x}}{2^{x} + 2^{-x}} = \frac{b}{a}$ ($a$, $b$ 는 서로소인 양수)이다. 이때, $a + b$ 의 값을 구하시오.

## 보기

## 풀이
$\frac{8^{x} + 8^{-x}}{2^{x} + 2^{-x}}$
$= \frac{( 2^{x} + 2^{-x} ) ( 4^{x} - 2^{x} \cdot 2^{-x} + 4^{-x} )}{2^{x} + 2^{-x}}$
$= 4^{x} - 1 + 4^{-x}$

$4^{x} = 5$ 이므로 $4^{-x} = \frac{1}{5}$ 이다.
$5 - 1 + \frac{1}{5} = \frac{21}{5}$

$\frac{8^{x} + 8^{-x}}{2^{x} + 2^{-x}} = \frac{21}{5} = \frac{b}{a}$ 이고, $a$, $b$ 는 서로소인 양수이므로
$a = 5$, $b = 21$ 이다.
따라서 $a + b = 5 + 21 = 26$

정답: $26$
