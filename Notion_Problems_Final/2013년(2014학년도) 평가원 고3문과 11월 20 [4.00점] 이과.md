## 문제
양의 실수 $x$ 에 대하여 $\log x$ 의 정수부분과 소수부분을 각각 $f(x)$, $g(x)$ 라 하자. 자연수 $n$ 에 대하여 
$f(x) - (n+1)g(x) = n$
을 만족시키는 모든 $x$ 의 값의 곱을 $a_{n}$ 이라 할 때, $\lim _{n \to \infty} \frac{\log a_{n}}{n^{2}}$ 의 값은?

## 보기
① $1$   ② $\frac{3}{2}$   ③ $2$   ④ $\frac{5}{2}$   ⑤ $3$

## 풀이
$f(x)$ 는 정수, $0 \le g(x) < 1$ 이고 $f(x) = n + (n+1)g(x)$ 에서 
$0 \le (n+1)g(x) < n+1$ 이므로
$n \le n + (n+1)g(x) < 2n+1$ 이다.
따라서 $n \le f(x) < 2n+1$ 이다.

$f(x) = n$ 이면 $g(x) = 0$
$f(x) = n+1$ 이면 $g(x) = \frac{1}{n+1}$
$f(x) = n+2$ 이면 $g(x) = \frac{2}{n+1}$
$\cdots$
$f(x) = 2n$ 이면 $g(x) = \frac{n}{n+1}$

$\log x = f(x) + g(x)$ 이고 $a_{n}$ 은 모든 $x$ 의 곱이므로 $\log a_{n}$ 은 모든 $\log x$ 의 합과 같다.
$a_{n} = 10^{n} \times 10^{n+1 + \frac{1}{n+1}} \times \cdots \times 10^{2n + \frac{n}{n+1}}$
$\log a_{n} = \{n + (n+1) + \cdots + 2n\} + \left( \frac{1+2+\cdots+n}{n+1} \right)$
$\log a_{n} = \frac{3n(n+1)}{2} + \frac{n(n+1)}{2(n+1)}$
$\log a_{n} = \frac{3n(n+1)}{2} + \frac{n}{2}$

$\therefore \lim_{n \to \infty} \frac{\log a_{n}}{n^{2}}$
$= \lim_{n \to \infty} \frac{\frac{3n(n+1)}{2} + \frac{n}{2}}{n^{2}}$
$= \lim_{n \to \infty} \left( \frac{3}{2} + \frac{4}{2n} \right) = \frac{3}{2}$

정답: ②
