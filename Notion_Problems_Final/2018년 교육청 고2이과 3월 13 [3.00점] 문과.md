## 문제
첫째항이 $20$ 인 등차수열 $\{a_n\}$ 에 대하여 수열 $\{b_n\}$ 을 
$b_n = a_n + a_{n+1}$ ($n=1, 2, 3, \dots$)
이라 하자. $a_{10} = b_{10}$ 일 때, $b_8$ 의 값은?

## 보기
① $6$   ② $8$   ③ $10$   ④ $12$   ⑤ $14$

## 풀이
등차수열 $\{a_n\}$ 의 공차를 $d$ 라 하면 
$a_n = 20+(n-1)d$ , $a_{n+1} = 20+nd$ 
$b_n = a_n + a_{n+1}$ 이므로 
$b_n = a_n + a_{n+1}$ 
$= \{20+(n-1)d\} + (20+nd)$ 
$= 40 + (2n-1) d$ 
$a_{10} = b_{10}$ 이므로 $20 + 9d = 40 + 19d$ 에서 
$-10d = 20$ , $d = -2$ 
따라서 
$b_n = 40-2(2n-1) = 42-4n$ 
이므로 
$b_8 = 42-4 \times 8 = 42-32 = 10$ 

**[다른 풀이]** 
$b_{10} = a_{10} + a_{11}$ 이고 $a_{10} = b_{10}$ 이므로 $a_{11} = 0$ 
등차수열 $\{a_n\}$ 의 공차를 $d$ 라 하면 
$a_{11} = 20+10d = 0$ 
$d = -2$ 
따라서 $a_n = 20-2(n-1) = 22-2n$ 이므로 
$b_8 = a_8 + a_9$ 
$= (22-2 \times 8) + (22-2 \times 9)$ 
$= 6 + 4 = 10$
