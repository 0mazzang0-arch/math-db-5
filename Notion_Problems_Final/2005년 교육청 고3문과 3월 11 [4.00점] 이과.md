## 문제
세 자리의 자연수 $N$에 대하여 $[ \log 2N ] = [ \log N ] + 1$이 성립할 때, <보기>에서 옳은 것을 모두 고른 것은? (단, $\log 2 = 0.3010$이고 $[ x ]$는 $x$보다 크지 않은 최대의 정수이다.)

---
**<보기>**
ㄱ. $N^{2}$은 항상 $6$자리의 수이다.
ㄴ. $N^{3}$은 항상 $9$자리의 수이다.
ㄷ. $N^{4}$은 항상 $12$자리의 수이다.
---

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
$[ \log 2N ] = [ \log N ] + 1$에서 $\log 2N$의 정수부분이 $\log N$의 정수부분 보다 $1$만큼 크므로 $2N$의 자리수가 $N$의 자리수보다 $1$만큼 크다.
즉, $5 \times 10^{2} \le N < 10^{3}$ 이므로
$\therefore 2 + \log 5 \le \log N < 3$이다.

ㄱ. $5.3980 \le \log N^{2} < 6$에서 $\log N^{2}$의 정수부분이 항상 $5$이다. 
$\therefore N^{2}$은 항상 $6$자리의 수이다.

ㄴ. $8.0970 \le \log N^{3} < 9$에서 $\log N^{3}$의 정수부분은 항상 $8$이다. 
$\therefore N^{3}$은 항상 $9$자리의 수이다.

ㄷ. $10.7960 \le \log N^{4} < 12$에서 $\log N^{4}$의 정수부분은 $10$ 또는 $11$이다. 
$\therefore N^{4}$은 $11$자리 또는 $12$자리의 수이다.

따라서 옳은 것은 ㄱ과 ㄴ이다.
