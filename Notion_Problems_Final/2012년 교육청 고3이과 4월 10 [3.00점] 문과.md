## 문제
세 집합 $A = \{ x \mid x^{3}-6 x^{2}+11 x-6>0 \}$, $B = \{ x \mid x > \alpha \}$, $C = \{ x \mid x > \beta \}$에 대하여 $B \subset A \subset C$일 때, $\alpha - \beta$의 최솟값은?

## 보기
① $2$   ② $3$   ③ $4$   ④ $5$   ⑤ $6$

## 풀이
[정답] ① 

$x^{3}-6 x^{2}+11 x-6>0$
$(x-1)(x-2)(x-3)>0$
따라서 $1 < x < 2$ 또는 $x > 3$이다.
$A = \{ x \mid 1 < x < 2 \text{ 또는 } x > 3 \}$이고 $B = \{ x \mid x > \alpha \}$, $C = \{ x \mid x > \beta \}$일 때,
$B \subset A$를 만족하려면 $\alpha \geq 3$이어야 한다.
$A \subset C$를 만족하려면 $\beta \leq 1$이어야 한다.
따라서 $\alpha - \beta$의 최솟값은 $\alpha = 3$, $\beta = 1$일 때 $3 - 1 = 2$이다.
