## 문제
조선시대 산학서에서 중심이 같은 서로 다른 두 원으로 둘러싸인 도형을 환($\text{環}$)이라 불렀다.

> [!example] 📸 (그림/그래프)

다음은 실경과 정주라는 것을 이용하여 환의 넓이를 구하는 과정이다. 

그림에서 두 원 $C_{1}$, $C_{2}$는 모두 중심이 $O$인 원이다. $C_{1}$, $C_{2}$의 반지름의 길이를 각각 $r_{1}$, $r_{2}$ $( r_{1} >r_{2} )$라 하고, 원의 중심 $O$와 원 $C_{1}$ 위의 점 $P$를 연결한 선분이 원 $C_{2}$와 만나는 점을 $Q$라 하자. $PQ$의 중점을 $R$라 하고, 중심이 $O$이고 선분 $OR$를 반지름으로 하는 원 $D$를 그린다. 

> [!example] 📸 (그림/그래프)

이때 선분 $PQ$의 길이를 실경, 원 $D$의 둘레의 길이를 정주라 한다. 

실경은 (가), 정주는 $2 \pi \times$ (나) 이므로
(실경) $\times$ (정주) $= ( (가) ) \times ( 2 \pi \times (나) )$
$= \pi r_{1}^{2} - \pi r_{2}^{2}$
$= ($환의 넓이$)$

위의 과정에서 (가), (나)에 알맞은 식을 더하면 $p r_{1} + q r_{2}$이다. $8 ( p^{2} + q^{2} )$의 값은? (단, $p$, $q$는 상수이다.)

## 보기
① $20$   ② $24$   ③ $28$   ④ $32$   ⑤ $36$

## 풀이
두 원 $C_{1}$, $C_{2}$의 반지름의 길이가 각각 $r_{1}$, $r_{2}$ $( r_{1} > r_{2} )$이므로 
실경은 (가) $= r_{1} - r_{2}$
점 $R$는 선분 $PQ$의 중점이므로 선분 $OR$의 길이는 
$OR = r_{2} + \frac{r_{1} - r_{2}}{2} = \frac{r_{1} + r_{2}}{2}$
따라서 (나) $= \frac{r_{1} + r_{2}}{2}$
정주는 $2 \pi \times \frac{r_{1} + r_{2}}{2}$

(실경) $\times$ (정주) $= ( r_{1} - r_{2} ) \times ( 2 \pi \times \frac{r_{1} + r_{2}}{2} )$
$= \pi r_{1}^{2} - \pi r_{2}^{2}$
$= ($환의 넓이$)$

(가), (나)에 알맞은 식을 더하면
$( r_{1} - r_{2} ) + \frac{r_{1} + r_{2}}{2} = \frac{3}{2} r_{1} - \frac{1}{2} r_{2}$
따라서 $p = \frac{3}{2}$, $q = -\frac{1}{2}$

$\therefore 8 ( p^{2} + q^{2} ) = 8 ( \frac{9}{4} + \frac{1}{4} ) = 8 \times \frac{10}{4} = 20$
