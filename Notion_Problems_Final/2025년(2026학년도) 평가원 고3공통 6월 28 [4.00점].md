## 문제
공 $15$ 개와 비어 있는 세 상자 $A$, $B$, $C$ 가 있다. 한 개의 주사위를 사용하여 다음 규칙에 따라 세 상자 $A$, $B$, $C$ 에 공을 넣는 시행을 한다.

- 주사위를 한 번 던져 나온 눈의 수가 $3$ 의 배수이면 세 상자 $A$, $B$, $C$ 에 넣는 공의 개수가 각각 $1$, $2$, $0$ 이고,
- 나온 눈의 수가 $3$ 의 배수가 아니면 세 상자 $A$, $B$, $C$ 에 넣는 공의 개수가 각각 $1$, $1$, $1$ 이다.

이 시행을 $5$ 번 반복한 후 상자 $B$ 에 들어 있는 공의 개수가 홀수일 때, 상자 $A$ 에 들어 있는 공의 개수와 상자 $C$ 에 들어 있는 공의 개수의 합이 $8$ 이상일 확률은?



> [!example] 📸 (그림/그래프)

## 보기
① $\frac{44}{61}$   ② $\frac{47}{61}$   ③ $\frac{50}{61}$   ④ $\frac{53}{61}$   ⑤ $\frac{56}{61}$

## 풀이
$5$ 번 시행 후 상자 $B$ 에 들어 있는 공의 개수가 홀수인 사건을 $X$, 상자 $A$ 와 $C$ 에 들어 있는 공의 개수의 합이 $8$ 이상인 사건을 $Y$ 라 하면 구하는 확률은 $P(Y|X)$ 이다.

$5$ 번 시행 중 $3$ 의 배수가 나오는 횟수를 $x$ 라 하면 $5$ 번 시행 후 상자 $B$ 에 들어 있는 공의 개수는
$$2x + (5 - x) = x + 5$$
$5$ 번 시행 후 상자 $B$ 에 들어 있는 공의 개수가 홀수이어야 하므로
$$x = 0 \text{ 또는 } x = 2 \text{ 또는 } x = 4$$

(ⅰ) $x = 0$ 일 때
상자 $B$ 에 들어 있는 공의 개수는 $5$ 이다.
$A$, $C$ 에 들어 있는 공의 개수의 합은 $(1 + 1) \times 5 = 10$ 이다.
$5$ 번 시행 중 $3$ 의 배수가 $0$ 번 나올 확률은
$${}_{5}C_{0} \left( \frac{1}{3} \right)^{0} \left( \frac{2}{3} \right)^{5} = \frac{32}{243}$$

(ⅱ) $x = 2$ 일 때
상자 $B$ 에 들어 있는 공의 개수는 $7$ 이다.
$A$, $C$ 에 들어 있는 공의 개수의 합은 $(0 + 1) \times 2 + (1 + 1) \times 3 = 8$ 이다.
$5$ 번 시행 중 $3$ 의 배수가 $2$ 번 나올 확률은
$${}_{5}C_{2} \left( \frac{1}{3} \right)^{2} \left( \frac{2}{3} \right)^{3} = \frac{80}{243}$$

(ⅲ) $x = 4$ 일 때
상자 $B$ 에 들어 있는 공의 개수는 $9$ 이다.
$A$, $C$ 에 들어 있는 공의 개수의 합은 $(0 + 1) \times 4 + (1 + 1) \times 1 = 6$ 이다.
$5$ 번 시행 중 $3$ 의 배수가 $4$ 번 나올 확률은
$${}_{5}C_{4} \left( \frac{1}{3} \right)^{4} \left( \frac{2}{3} \right)^{1} = \frac{10}{243}$$

이상에서
$P(X) = \frac{32 + 80 + 10}{243} = \frac{122}{243}$
$P(X \cap Y) = \frac{32 + 80}{243} = \frac{112}{243}$
$\therefore P(Y|X) = \frac{P(X \cap Y)}{P(X)} = \frac{\frac{112}{243}}{\frac{122}{243}} = \frac{56}{61}$
