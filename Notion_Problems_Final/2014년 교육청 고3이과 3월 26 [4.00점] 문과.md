## 문제
두 함수 $f(x)=(x-a)(x-5)$, $g(x)=(x-1)(x-3)^{2}$이 다음 조건을 만족시킨다.

(가) 집합 $\left\{ x \mid \frac{g(x)}{f(x)}=0 \right\}$의 원소의 개수는 $1$이다.
(나) 집합 $\left\{ x \mid \frac{f(x)}{g(x)} \leq 0, x \text{는 자연수} \right\}$의 원소의 개수는 $2$이다.

$f(10)$의 값을 구하시오. (단, $a$는 상수이다.)

## 보기

## 풀이
[정답] $35$

(가)에서 집합 $\left\{ x \mid \frac{g(x)}{f(x)}=0 \right\}$의 원소의 개수는 $1$이고, $g(x)=(x-1)(x-3)^{2}$이므로 $f(1)=0$ 또는 $f(3)=0$이다.
따라서 $f(x)=(x-a)(x-5)$에서 $a=1$ 또는 $a=3$이다.

$i)$ $a=1$이면
$$ \frac{f(x)}{g(x)}=\frac{(x-1)(x-5)}{(x-1)(x-3)^{2}} \leq 0 $$
$$ \frac{x-5}{(x-3)^{2}} \leq 0 \quad (x \neq 1, x \neq 3) $$
$\therefore x<1$ 또는 $1<x<3$ 또는 $3<x \leq 5$
이때 자연수 $x$는 $2, 4, 5$이므로 원소의 개수가 $3$개가 되어 (나) 조건에 모순이다.

$ii)$ $a=3$이면
$$ \frac{f(x)}{g(x)}=\frac{(x-3)(x-5)}{(x-1)(x-3)^{2}} \leq 0 $$
$$ \frac{x-5}{(x-1)(x-3)} \leq 0 \quad (x \neq 1, x \neq 3) $$
이 부등식을 풀면 $(x-1)(x-3)(x-5) \leq 0$ ($x \neq 1, x \neq 3$)이다.
$\therefore x<1$ 또는 $3<x \leq 5$
이때 자연수 $x$는 $4, 5$이므로 원소의 개수가 $2$개가 되어 (나) 조건을 만족시킨다.

$i), ii)$에서 $a=3$이므로
$$ f(x)=(x-3)(x-5) $$
$\therefore f(10)=(10-3)(10-5)=7 \times 5=35$
