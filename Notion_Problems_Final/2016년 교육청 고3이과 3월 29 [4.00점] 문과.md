## 문제
집합 $X = \{ -3, -2, -1, 1, 2, 3 \}$ 에 대하여 $X$ 에서 $X$ 로의 함수 $f(x)$ 는 다음 조건을 만족시킨다.

$$
\begin{array}{l}
(\text{가}) \ X \text{ 의 모든 원소 } x \text{ 에 대하여 } | f(x)+f(-x) | = 1 \text{ 이다.} \\
(\text{나}) \ x > 0 \text{ 이면 } f(x) > 0 \text{ 이다.}
\end{array}
$$

함수 $f(x)$ 의 개수를 구하시오.

## 풀이
$| f(x)+f(-x) | = 1$ 에서 $f(x)+f(-x) = 1$ 또는 $f(x)+f(-x) = -1$ 이다.
$x > 0$ 인 집합 $X$ 의 원소 $x$ 에 대하여
$$
\begin{array}{l}
(\text{ⅰ}) \ f(x) = 1 \text{ 일 때, } f(-x) = -2 \\
(\text{ⅱ}) \ f(x) = 2 \text{ 일 때, } f(-x) = -3 \text{ 또는 } f(-x) = -1 \\
(\text{ⅲ}) \ f(x) = 3 \text{ 일 때, } f(-x) = -2
\end{array}
$$
이상에서 $f(x)$ 의 값에 따라 $f(-x)$ 의 값이 정해지므로 $f(1)$ 과 $f(-1)$ 의 값을 정하는 경우의 수는 $4$ 이고, $f(2)$ 와 $f(-2)$ 의 값을 정하는 경우의 수와 $f(3)$ 과 $f(-3)$ 의 값을 정하는 경우의 수도 각각 $4$ 이므로 구하는 함수 $f(x)$ 의 개수는
$4 \times 4 \times 4 = 64$

[정답] $64$
