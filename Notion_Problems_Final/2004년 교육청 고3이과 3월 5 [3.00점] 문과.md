## 문제
분수부등식 
$$\frac{3}{x-1} + \frac{4}{x-5} + \frac{12}{x^2-6x+5} \le -1$$ 
을 만족하는 정수 $x$ 의 개수는? 

## 보기
① $3$   ② $4$   ③ $5$   ④ $6$   ⑤ $7$

## 풀이
주어진 부등식을 통분하여 정리하면 
$$\frac{x^{2}+x-2}{x^{2}-6x+5} \le 0$$ 
$$\frac{(x-1)(x+2)}{(x-1)(x-5)} \le 0$$ 
이때 분모는 $0$ 이 될 수 없으므로 $x \neq 1, x \neq 5$ 이다. 
$$(x+2)(x-5) \le 0 \quad (x \neq 1, x \neq 5)$$ 
$$-2 \le x \le 5 \quad (x \neq 1, x \neq 5)$$ 
따라서 이를 만족하는 정수 $x$ 는 $-2, -1, 0, 2, 3, 4$ 이며, 그 개수는 $6$ 개이다. 

정답: ④
