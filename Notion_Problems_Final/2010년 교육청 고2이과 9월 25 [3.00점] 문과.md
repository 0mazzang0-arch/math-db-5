## 문제
$\log _{2} \sin \theta + \log _{2} \cos \theta = -4$ 일 때, 
$\log _{2} (\sin \theta + \cos \theta) = \frac{1}{2} (\log _{2} x - 4)$를 만족하는 $x$의 값을 구하시오.

## 풀이
$\log _{2} \sin \theta \cos \theta = -4$ 이므로, $\sin \theta \cos \theta = \frac{1}{16}$
$(\sin \theta + \cos \theta)^{2} = 1 + 2 \sin \theta \cos \theta = 1 + \frac{2}{16} = \frac{9}{8}$
$\log _{2} (\sin \theta + \cos \theta) = \frac{1}{2} (\log _{2} x - 4)$ 의 양변에 $2$를 곱하면
$2 \log _{2} (\sin \theta + \cos \theta) = \log _{2} x - 4$
$\log _{2} (\sin \theta + \cos \theta)^{2} = \log _{2} x - \log _{2} 16$
$\log _{2} \frac{9}{8} = \log _{2} \frac{x}{16}$ 이므로
$\frac{9}{8} = \frac{x}{16}$
$\therefore x = 18$

정답: $18$
