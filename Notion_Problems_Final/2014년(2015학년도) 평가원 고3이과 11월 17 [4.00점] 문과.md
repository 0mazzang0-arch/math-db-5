## 문제
수열 $\{a_n\}$ 은 $a_1=1$ 이고, $S_n = \sum_{k=1}^{n} a_k$ 라 할 때,
$a_{n+1}=(n+1)S_n + n!$ ($n \geq 1$)
을 만족시킨다. 다음은 일반항 $a_n$ 을 구하는 과정이다.

자연수 $n$ 에 대하여 $a_{n+1}=S_{n+1}-S_n$ 이므로 주어진 식에 의하여
$S_{n+1}=(n+2)S_n+n!$ ($n \geq 1$)
이다. 양변을 $(n+2)!$ 로 나누면
$\frac{S_{n+1}}{(n+2)!} = \frac{S_n}{(n+1)!} + \frac{1}{(n+1)(n+2)}$
이다. $b_n = \frac{S_n}{(n+1)!}$ 이라 하면 $b_1 = \frac{1}{2}$ 이고
$b_{n+1} = b_n + \frac{1}{(n+1)(n+2)}$
이다. 수열 $\{b_n\}$ 의 일반항을 구하면
$b_n = \frac{(가)}{n+1}$
이므로
$S_n = (가) \times n!$
이다. 그러므로
$a_n = (나) \times (n-1)!$ ($n \geq 1$)
이다.

위의 (가), (나)에 알맞은 식을 각각 $f(n)$, $g(n)$ 이라 할 때, $f(7)+g(6)$ 의 값은?

## 보기
① $44$   ② $41$   ③ $38$   ④ $35$   ⑤ $32$

## 풀이
$b_{n+1} = b_n + \frac{1}{(n+1)(n+2)}$ 에서
$b_{n+1} - b_n = \frac{1}{(n+1)(n+2)} = \frac{1}{n+1} - \frac{1}{n+2}$
이때, $n$ 에 $1, 2, \dots, n-1$ 을 차례대로 대입하면
$$ \begin{array}{l} b_2 - b_1 = \frac{1}{2} - \frac{1}{3} \\ b_3 - b_2 = \frac{1}{3} - \frac{1}{4} \\ \dots \\ b_n - b_{n-1} = \frac{1}{n} - \frac{1}{n+1} \end{array} $$
위의 식을 좌변은 좌변끼리 우변은 우변끼리 더하면
$b_n - b_1 = \frac{1}{2} - \frac{1}{n+1}$
$b_1 = \frac{1}{2}$ 이므로
$b_n = 1 - \frac{1}{n+1} = \frac{n}{n+1}$
따라서 $f(n) = n$ 이고,
$S_n = (n+1)! \times b_n = (n+1)! \times \frac{n}{n+1} = n \times n!$
이다.

$n \geq 2$ 일 때,
$a_n = S_n - S_{n-1}$
$a_n = n \times n! - (n-1) \times (n-1)!$
$a_n = n \times n(n-1)! - (n-1) \times (n-1)!$
$a_n = (n^2 - n + 1)(n-1)!$
따라서 $g(n) = n^2 - n + 1$ 이다.
($n=1$ 일 때 $a_1 = (1^2-1+1) \times 0! = 1$ 이므로 성립한다.)

$f(7) = 7$,
$g(6) = 6^2 - 6 + 1 = 31$
이므로
$f(7) + g(6) = 7 + 31 = 38$
