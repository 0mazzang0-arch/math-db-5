## 문제
$\sin \theta + \cos \theta = \frac{\sqrt{15}}{3}$ 일 때, $\cos 2\theta$ 의 값은? (단, $0 < \theta < \frac{\pi}{4}$)

## 보기
① $\frac{\sqrt{2}}{3}$   ② $\frac{\sqrt{3}}{3}$   ③ $\frac{2}{3}$   ④ $\frac{\sqrt{5}}{3}$   ⑤ $\frac{\sqrt{6}}{3}$

## 풀이
$(\sin \theta + \cos \theta)^{2} = 1 + 2\sin \theta \cos \theta = \frac{15}{9} = \frac{5}{3}$ 이므로
$2\sin \theta \cos \theta = \sin 2\theta = \frac{2}{3}$ 이다.
$0 < \theta < \frac{\pi}{4}$ 에서 $0 < 2\theta < \frac{\pi}{2}$ 이므로 $\cos 2\theta > 0$ 이다.
따라서
$\cos 2\theta = \sqrt{1 - \sin^{2} 2\theta} = \sqrt{1 - \frac{4}{9}} = \sqrt{\frac{5}{9}} = \frac{\sqrt{5}}{3}$
정답: ④
