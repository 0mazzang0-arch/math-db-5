## 문제
어느 지역에 서식하는 어떤 동물의 개체 수에 대한 변화를 조사한 결과, 지금으로부터 $t$ 년 후에 이 동물의 개체 수를 $N$ 이라 하면 등식
$$\log N = k + t \log \frac{4}{5}$$
(단, $k$ 는 상수)가 성립한다고 한다. 이 동물의 현재 개체 수가 $5000$ 일 때, 개체 수가 처음으로 $1000$ 보다 적어지는 때는 지금으로부터 $n$ 년 후이다. 자연수 $n$ 의 값은? (단, $\log 2 = 0.3010$ 으로 계산한다.)

## 보기
① $4$   ② $6$   ③ $8$   ④ $10$   ⑤ $12$

## 풀이
[정답] ③ 

$t = 0$ 일 때, 현재 개체 수가 $5000$ 이므로 
$\log 5000 = k + 0 \cdot \log \frac{4}{5}$ 가 성립한다. 
$\therefore k = 3 + \log 5$ 

$n$ 년 후 개체 수 $N$ 가 $1000$ 보다 작을 때, 
$\log N = k + n \log \frac{4}{5} < \log 1000$ 에서 
$3 + \log 5 + n \log \frac{4}{5} < 3$ 

$\log 2 = 0.3010$ 이므로 
$\log 5 = \log \frac{10}{2} = 1 - \log 2 = 0.699$ 
$\log \frac{4}{5} = \log \frac{8}{10} = 3 \log 2 - 1 = -0.097$ 

따라서 $0.699 - 0.097 \times n < 0$ 에서 
$n > \frac{0.699}{0.097} = 7. \dots$ 

따라서 구하는 자연수 $n$ 의 최솟값은 $8$ 이다.
