## 문제
로그함수 $f(x) = \log_{\frac{1}{3}} x$ 에 대하여 $0 < x < 1$ 에서 등식 $[f(x)] = f(x)$ 를 만족하는 모든 $x$ 의 값의 합은? (단, $[x]$ 는 $x$ 보다 크지 않은 최대의 정수이다.)

## 보기
① $1$   ② $\frac{1}{2}$   ③ $\frac{2}{3}$   ④ $\frac{3}{4}$   ⑤ $\frac{8}{9}$

## 풀이
$[f(x)] = f(x)$ $\cdots\cdots$ ㉠ 

$0 < x < 1$ 에서 $f(x) = \log_{\frac{1}{3}} x > 0$ 이므로 

(ⅰ) $0 < f(x) < 1$ 일 때, 즉 $0 < \log_{\frac{1}{3}} x < 1$ 
$\therefore \frac{1}{3} < x < 1$ $\cdots\cdots$ ㉡ 
$[f(x)] = 0$ 이므로 ㉠에서 
$0 = \log_{\frac{1}{3}} x$ $\therefore x = 1$ 

(ⅱ) $1 \le f(x) < 2$ 일 때, 즉 $1 \le \log_{\frac{1}{3}} x < 2$ 
$\therefore \frac{1}{9} < x \le \frac{1}{3}$ 
$[f(x)] = 1$ 이므로 ㉠에서 
$1 = \log_{\frac{1}{3}} x$ $\therefore x = \frac{1}{3}$ 

(ⅲ) $2 \le f(x) < 3$ 일 때, 즉 $2 \le \log_{\frac{1}{3}} x < 3$ 
$\therefore \frac{1}{27} < x \le \frac{1}{9}$ 
$[f(x)] = 2$ 이므로 ㉠에서 
$2 = \log_{\frac{1}{3}} x$ $\therefore x = \frac{1}{9}$ 

(ⅳ) $3 \le f(x) < 4$ 일 때, 즉 $3 \le \log_{\frac{1}{3}} x < 4$ 
$\therefore \frac{1}{81} < x \le \frac{1}{27}$ 
$[f(x)] = 3$ 이므로 ㉠에서 
$3 = \log_{\frac{1}{3}} x$ $\therefore x = \frac{1}{27}$ 
$\vdots$ 

따라서, 실근은 $\frac{1}{3}$, $\frac{1}{9}$, $\frac{1}{27}$, $\cdots$ 이므로 모든 실근의 합은 첫째항이 $\frac{1}{3}$ 이고, 공비가 $\frac{1}{3}$ 인 무한등비급수의 합이다. 
$\therefore \frac{1}{3} + \frac{1}{9} + \frac{1}{27} + \cdots = \frac{\frac{1}{3}}{1 - \frac{1}{3}} = \frac{1}{2}$
