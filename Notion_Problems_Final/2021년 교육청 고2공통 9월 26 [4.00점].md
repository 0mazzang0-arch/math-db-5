## 문제
다항함수 $f(x)$ 가 
$\lim_{x \to \infty} \frac{f(x)}{2x^2} = 1$, $\lim_{x \to 1} \frac{f(x)-3}{(x-1)(x-2)} = 4$
를 만족시킬 때, $f(4)$ 의 값을 구하시오.

## 보기

## 풀이
$f(x)$ 가 다항함수이고 $\lim_{x \to \infty} \frac{f(x)}{2x^2} = 1$ 이므로 
$f(x) = 2x^2 + ax + b$ ($a$, $b$ 는 실수) 

$\lim_{x \to 1} \frac{f(x)-3}{(x-1)(x-2)} = 4$ 이고 $\lim_{x \to 1} (x-1)(x-2) = 0$ 이므로 
$\lim_{x \to 1} \{ f(x)-3 \} = 0$ 

$\lim_{x \to 1} f(x) = \lim_{x \to 1} [ \{ f(x)-3 \} + 3 ]$ 
$= \lim_{x \to 1} \{ f(x)-3 \} + \lim_{x \to 1} 3$ 
$= 0 + 3 = 3$ 

$\lim_{x \to 1} f(x) = 2 + a + b = 3$ 
즉, $b = -a + 1$ 

$f(x)-3 = 2x^2 + ax - a - 2$ 
$= (x-1)(2x + a + 2)$ 

$\lim_{x \to 1} \frac{f(x)-3}{(x-1)(x-2)} = \lim_{x \to 1} \frac{(x-1)(2x + a + 2)}{(x-1)(x-2)}$ 
$= \frac{2 + a + 2}{-1} = 4$ 

따라서 $a = -8$, $b = 9$ 이므로 
$f(x) = 2x^2 - 8x + 9$ 

$\therefore f(4) = 9$
