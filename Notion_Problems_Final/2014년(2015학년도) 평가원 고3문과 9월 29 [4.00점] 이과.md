## 문제
구간 $[0, 3]$의 모든 실수의 값을 가지는 연속확률변수 $X$에 대하여 
$P(x \le X \le 3) = a(3 - x)$ $(0 \le x \le 3)$
이 성립할 때, $P(0 \le X < a) = \frac{q}{p}$이다. $p + q$의 값을 구하시오. (단, $a$는 상수이고, $p$와 $q$는 서로소인 자연수이다.)

## 보기

## 풀이
$P(0 \le X \le 3) = 1$이어야 하므로 
$P(0 \le X \le 3) = 3a = 1$
$\therefore a = \frac{1}{3}$

$\therefore P(0 \le X < a) = P\left(0 \le X < \frac{1}{3}\right)$
$= P(0 \le X \le 3) - P\left(\frac{1}{3} \le X \le 3\right)$
$= 1 - \frac{1}{3} \times \left(3 - \frac{1}{3}\right)$
$= 1 - \frac{1}{3} \times \frac{8}{3} = 1 - \frac{8}{9} = \frac{1}{9}$

따라서 $\frac{q}{p} = \frac{1}{9}$이므로 $p = 9$, $q = 1$이다.
$\therefore p + q = 10$
