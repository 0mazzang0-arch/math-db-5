## 문제
다음 등식을 만족시키는 세 실수 $a$, $b$, $c$가 있다.
$$ \left( \frac{1}{3} \right)^a = 2a, \left( \frac{1}{3} \right)^{2b} = b, \left( \frac{1}{2} \right)^{2c} = c $$
이때, 세 실수 $a$, $b$, $c$의 대소 관계를 옳게 나타낸 것은?

## 보기
① $a < b < c$   ② $a < c < b$   ③ $b < a < c$   ④ $b < c < a$   ⑤ $c < a < b$

## 풀이
[정답] ①

주어진 등식은 다음과 같이 정리할 수 있다.
$1$) $\left( \frac{1}{3} \right)^a = 2a$
$2$) $\left( \{ \left( \frac{1}{3} \right)^2 \right\}^b = b \Rightarrow \left( \frac{1}{9} \right)^b = b$
$3$) $\left( \{ \left( \frac{1}{2} \right)^2 \right\}^c = c \Rightarrow \left( \frac{1}{4} \right)^c = c$

각 방정식의 해는 함수 $y = \left( \frac{1}{3} \right)^x$와 $y = 2x$, $y = \left( \frac{1}{9} \right)^x$와 $y = x$, $y = \left( \frac{1}{4} \right)^x$와 $y = x$의 그래프의 교점의 $x$좌표이다.

그래프의 개형을 통해 교점의 위치를 비교하면 다음과 같다.
> [!example] 📸 (그림/그래프)

$\therefore a < b < c$
