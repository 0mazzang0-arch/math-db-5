## 문제
실수 전체의 집합에서 미분가능한 함수 $f(x)$가 모든 실수 $x$에 대하여
$f'(x^2 + x + 1) = \pi f(1)\sin \pi x + f(3)x + 5x^2$
을 만족시킬 때, $f(7)$의 값을 구하시오.

## 보기

## 풀이
$a = f(1), b = f(3)$이라 하자.
$f'(x^2 + x + 1) = \pi a \sin \pi x + bx + 5x^2$의 양변에 $(2x + 1)$을 곱하면
$(2x + 1)f'(x^2 + x + 1) = a\pi(2x + 1)\sin \pi x + bx + (2b + 5)x^2 + 10x^3$
이다. 위 식의 양변을 부정적분 하면
$f(x^2 + x + 1) = -a(2x + 1)\cos \pi x + \frac{2a}{\pi}\sin \pi x + \frac{b}{2}x^2 + \frac{2b + 5}{3}x^3 + \frac{5}{2}x^4 + C$
이다.  (단, $C$는 상수)

$g(x) = x^2 + x + 1$이라 하자.
$a = f(g(0)) = -a + C$ $\dots$ ㉠
$a = f(g(-1)) = -a + \frac{b}{2} - \frac{2b + 5}{3} + \frac{5}{2} + C$ $\dots$ ㉡
$b = f(g(1)) = 3a + \frac{b}{2} + \frac{2b + 5}{3} + \frac{5}{2} + C$ $\dots$ ㉢

이때, ㉡ $+$ ㉢ $\implies$ $a + b = 2a + b + 2C + 5$이고
㉠ $\implies$ $C = 2a$이다.
따라서 $a = -1, b = 5, C = -2$이고
$f(x^2 + x + 1) = (2x + 1)\cos \pi x - \frac{2}{\pi}\sin \pi x + \frac{5}{2}x^2 + 5x^3 + \frac{5}{2}x^4 - 2$
이다.

$f(x^2 + x + 1)$에 $x = 2$를 대입하면 $f(7) = 93$이다.

정답: $93$
