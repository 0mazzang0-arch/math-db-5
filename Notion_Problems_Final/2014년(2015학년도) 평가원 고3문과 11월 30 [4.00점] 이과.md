## 문제

좌표평면에서 자연수 $n$ 에 대하여 다음 조건을 만족시키는 삼각형 $OAB$ 의 개수를 $f(n)$ 이라 할 때, $f(1)+f(2)+f(3)$ 의 값을 구하시오. (단, $O$ 는 원점이다.)

$$ \begin{array}{|l|} \hline \text{(가) 점 } A \text{ 의 좌표는 } (-2, 3^n) \text{ 이다.} \\ \text{(나) 점 } B \text{ 의 좌표를 } (a, b) \text{ 라 할 때, } a \text{ 와 } b \text{ 는 자연수이고 } b \le \log_2 a \text{ 를 만족시킨다.} \\ \text{(다) 삼각형 } OAB \text{ 의 넓이는 } 50 \text{ 이하이다.} \\ \hline \end{array} $$



> [!example] 📸 (그림/그래프)

## 풀이

삼각형 $OAB$ 의 넓이는
$$ \frac{1}{2} \times (a+2) \times (3^n + b) - \frac{1}{2} \times 2 \times 3^n - \frac{1}{2} ab $$
$$ = \frac{1}{2} (a \cdot 3^n + ab + 2 \cdot 3^n + 2b) - 3^n - \frac{1}{2} ab $$
$$ = \frac{3^n}{2} a + b $$
이때, 넓이가 $50$ 이하이므로
$$ \frac{3^n}{2} a + b \le 50 $$
$$ \therefore 3^n a + 2b \le 100 \dots \dots ㉠ $$

(ⅰ) $n=1$ 일 때,
$3a+2b \le 100$ 이므로 $a$ 의 값은 $2, 3, \dots, 32$ 의 값을 가질 수 있다.
$b \le \log_2 a$ 를 만족시키므로
- $2^1 \le a < 2^2$ 일 때, $b=1$
- $2^2 \le a < 2^3$ 일 때, $b=1, 2$
- $2^3 \le a < 2^4$ 일 때, $b=1, 2, 3$
- $2^4 \le a \le 30$ 일 때, $b=1, 2, 3, 4$
- $a=31$ 일 때, $b=1, 2, 3$
- $a=32$ 일 때, $b=1, 2$

$f(1)$ 은 순서쌍 $(a, b)$ 의 개수와 같으므로
$$ f(1) = 1 \times (2^2 - 2^1) + 2 \times (2^3 - 2^2) + 3 \times (2^4 - 2^3) + 4 \times (30 - 2^4 + 1) + 3 + 2 $$
$$ = 2 + 8 + 24 + 60 + 3 + 2 = 99 $$

(ⅱ) $n=2$ 일 때,
$9a+2b \le 100$ 이므로 $a$ 의 값은 $2, 3, \dots, 10$ 의 값을 가질 수 있다.
$b \le \log_2 a$ 를 만족시키므로
- $2^1 \le a < 2^2$ 일 때, $b=1$
- $2^2 \le a < 2^3$ 일 때, $b=1, 2$
- $2^3 \le a \le 10$ 일 때, $b=1, 2, 3$

$f(2)$ 는 순서쌍 $(a, b)$ 의 개수와 같으므로
$$ f(2) = 1 \times (2^2 - 2^1) + 2 \times (2^3 - 2^2) + 3 \times 3 $$
$$ = 2 + 8 + 9 = 19 $$

(ⅲ) $n=3$ 일 때,
$27a+2b \le 100$ 이므로 $a$ 의 값은 $2, 3$ 의 값을 가질 수 있다.
$b \le \log_2 a$ 를 만족시키므로 $a=2, 3$ 일 때, $b=1$ 이다.
$f(3)$ 은 순서쌍 $(a, b)$ 의 개수와 같으므로
$$ f(3) = 2 $$

따라서 (ⅰ), (ⅱ), (ⅲ)에서
$$ f(1) + f(2) + f(3) = 99 + 19 + 2 = 120 $$

**[정답] $120$**
