## 문제
집합 $X = \{-3, -2, -1, 0, 1, 2\}$에서 실수 전체의 집합으로의 일대일함수 $f(x)$가 다음 조건을 만족시킨다.

(가) 집합 $X$의 모든 원소 $x$에 대하여 $\{f(x) + x^{2} - 5\} \times \{f(x) + 4x\} = 0$이다.
(나) $f(0) \times f(1) \times f(2) < 0$

$f(-3) + f(-2) + f(-1) + f(0) + f(1) + f(2)$의 값을 구하시오.

## 풀이
$\{f(x) + x^{2} - 5\} \times \{f(x) + 4x\} = 0$에서 $g(x) = -x^{2} + 5$, $h(x) = -4x$라 하면 집합 $X$의 모든 원소 $x$에 대하여 $f(x) = g(x)$ 또는 $f(x) = h(x)$이다.

$g(x) = h(x)$에서 $x^{2} - 4x - 5 = 0$, $(x + 1)(x - 5) = 0$이므로 $x = -1$ 또는 $x = 5$이다.
이때 $x \in X$이므로 $x = -1$이다.
$g(-1) = h(-1) = 4$이므로 $f(-1) = 4$이다.

이차함수 $y = g(x)$의 그래프는 $y$축에 대하여 대칭이므로 $g(1) = g(-1) = 4$이다.
만약 $f(1) = g(1) = 4$라 하면 $f(-1) = f(1) = 4$가 되어 함수 $f(x)$가 일대일함수라는 조건에 모순이다.
그러므로 $f(1) = h(1) = -4$이다.

$g(x) = -4$에서 $x^{2} = 9$이므로 $x = \pm 3$이다.
이때 $g(-3) = -4$이고, 만약 $f(-3) = g(-3) = -4$라 하면 $f(1) = f(-3) = -4$가 되어 함수 $f(x)$가 일대일함수라는 조건에 모순이다.
그러므로 $f(-3) = h(-3) = 12$이다.

$f(0) = h(0) = 0$이라 하면 $f(0) \times f(1) \times f(2) = 0$이 되어 조건 (나)를 만족시키지 않는다.
그러므로 $f(0) = g(0) = 5$이다.

조건 (나) $f(0) \times f(1) \times f(2) < 0$에서 $f(1) = -4$이므로 $f(0) \times f(2) > 0$이어야 한다.
$f(0) = 5 > 0$이므로 $f(2) > 0$이다.
$h(2) = -8 < 0$이므로 $f(2) = g(2) = 1$이다.

이차함수 $y = g(x)$의 그래프는 $y$축에 대하여 대칭이므로 $g(-2) = g(2) = 1$이다.
만약 $f(-2) = g(-2) = 1$이라 하면 $f(2) = f(-2) = 1$이 되어 함수 $f(x)$가 일대일함수라는 조건에 모순이다.
그러므로 $f(-2) = h(-2) = 8$이다.

따라서 구하는 값은
$f(-3) + f(-2) + f(-1) + f(0) + f(1) + f(2)$
$= 12 + 8 + 4 + 5 + (-4) + 1 = 26$

정답: $26$
