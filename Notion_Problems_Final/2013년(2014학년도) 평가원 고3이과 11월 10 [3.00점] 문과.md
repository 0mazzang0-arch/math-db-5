## 문제

그림과 같이 닫힌 구간 $[-4, 4]$에서 정의된 함수 $f(x)$와 함수 $g(x) = -\frac{1}{2}x + 1$의 그래프가 세 점에서 만나고 그 세 점의 $x$ 좌표는 $\alpha, \beta, 2$이다. 부등식 

$$\frac{g(x)}{f(x)} \leq 1$$

을 만족시키는 정수 $x$의 개수는? (단, $-4 < \alpha < -3, 0 < \beta < 1$)

> [!example] 📸 (그림/그래프)

## 보기

① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이

부등식 $\frac{g(x)}{f(x)} \leq 1$을 정리하면
$$\frac{g(x) - f(x)}{f(x)} \leq 0$$
$$\iff f(x) \{ g(x) - f(x) \} \leq 0, f(x) \neq 0$$

(ⅰ) $f(x) < 0$ 이고 $g(x) \geq f(x)$ 일 때,
그래프를 통해 확인하면
$$-2 < x < 0, 2 < x \leq 4$$
이 범위를 만족하는 정수 $x$의 값은 $-1, 3, 4$이다.

(ⅱ) $f(x) > 0$ 이고 $g(x) \leq f(x)$ 일 때,
그래프를 통해 확인하면
$$-4 \leq x \leq \alpha, \beta \leq x < 2$$
조건에서 $-4 < \alpha < -3, 0 < \beta < 1$이므로 정수 $x$의 값은 $-4, 1$이다.

(ⅰ), (ⅱ)에 의하여 구하는 정수 $x$의 개수는 $3 + 2 = 5$이다.

정답: ⑤
