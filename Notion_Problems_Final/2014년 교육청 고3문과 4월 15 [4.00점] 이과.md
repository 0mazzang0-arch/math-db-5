## 문제
$1$보다 크고 $10$보다 작은 세 자연수 $a$, $b$, $c$에 대하여 
$$\frac{\log_{c} b}{\log_{a} b} = \frac{1}{2}$$, $$\frac{\log_{b} c}{\log_{a} c} = \frac{1}{3}$$
일 때, $a+2b+3c$의 값은?

## 보기
① $21$   ② $24$   ③ $27$   ④ $30$   ⑤ $33$

## 풀이
$$\frac{\log_{c} b}{\log_{a} b} = \log_{c} b \times \frac{1}{\log_{a} b} = \frac{1}{2}$$에서 
$$\frac{\log_{a} b}{\log_{a} c} \times \frac{1}{\log_{a} b} = \frac{1}{2}$$, 
$$\frac{1}{\log_{a} c} = \frac{1}{2}$$ 
$$\log_{a} c = 2$$ 
$$\therefore c = a^{2}$$ 

$$\frac{\log_{b} c}{\log_{a} c} = \log_{b} c \times \frac{1}{\log_{a} c} = \frac{1}{3}$$에서 
$$\frac{\log_{a} c}{\log_{a} b} \times \frac{1}{\log_{a} c} = \frac{1}{3}$$, 
$$\frac{1}{\log_{a} b} = \frac{1}{3}$$ 
$$\log_{a} b = 3$$ 
$$\therefore b = a^{3}$$ 

이때 $a, b, c$는 $1$보다 크고 $10$보다 작은 자연수이므로 
$a=2, b=8, c=4$ 
$$\therefore a+2b+3c=30$$
