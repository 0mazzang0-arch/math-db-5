## 문제
$0 \le t \le 3$인 실수 $t$와 상수 $k$에 대하여 $t \le x \le t+1$에서 방정식 $\sin \frac{\pi}{2} x = k$의 모든 해의 개수를 $f(t)$라 하자. 함수 $f(t)$가 
$$ f(t) = \left\{ \begin{array}{ll} 1 & (0 \le t < a \text{ 또는 } a < t \le b) \\ 2 & (t = a) \\ 0 & (b < t \le 3) \end{array} \right. $$
일 때, $a^2 + b^2 + k^2$의 값은? (단, $a, b$는 $0 < a < b < 3$인 상수이다.) 



> [!example] 📸 (그림/그래프)

## 보기
① $2$   ② $\frac{5}{2}$   ③ $3$   ④ $\frac{7}{2}$   ⑤ $4$

## 풀이
[정답] ③ 

함수 $y = \sin \frac{\pi}{2}x$의 치역은 $\{ y \mid -1 \le y \le 1 \}$이고, 주기가 $4$이다. 
따라서 $0 \le x \le 4$에서 함수 $y = \sin \frac{\pi}{2}x$의 그래프가 직선 $y = k$와 두 점에서 만나려면 $-1 < k < 1$이어야 한다. 

$0 \le x \le 4$에서 함수 $y = \sin \frac{\pi}{2}x$의 그래프가 직선 $y = k$와 만나는 두 점의 $x$좌표를 각각 $\alpha, \beta$ ($\alpha < \beta$)라 하면 $\beta - \alpha > 1$일 때, $f(t) = 2$를 만족시키는 $t$의 값은 존재하지 않고, $\beta - \alpha < 1$일 때, $f(t) = 2$를 만족시키는 $t$의 값이 유일하지 않다. 

$\beta - \alpha = 1$일 때, $f(t) = 2$가 되기 위해서는 구간 $[t, t+1]$이 두 해 $\alpha, \beta$를 모두 포함해야 하므로 $t = \alpha$인 경우에만 가능하다. 이때 $f(0) = 1$이려면 구간 $[0, 1]$에 해가 하나 포함되어야 하므로 $\alpha = \frac{1}{2}$, $\beta = \frac{3}{2}$이다. 

$0 \le t < \frac{1}{2}$, $\frac{1}{2} < t \le \frac{3}{2}$일 때, $f(t) = 1$ 
$t = \frac{1}{2}$일 때, $f(t) = 2$ 
$\frac{3}{2} < t \le 3$일 때, $f(t) = 0$ 

따라서 $a = \frac{1}{2}$, $b = \frac{3}{2}$, $k = \sin \left( \frac{\pi}{2} \cdot \frac{1}{2} \right) = \frac{\sqrt{2}}{2}$이므로 
$a^2 + b^2 + k^2 = \left( \frac{1}{2} \right)^2 + \left( \frac{3}{2} \right)^2 + \left( \frac{\sqrt{2}}{2} \right)^2 = \frac{1}{4} + \frac{9}{4} + \frac{2}{4} = 3$
