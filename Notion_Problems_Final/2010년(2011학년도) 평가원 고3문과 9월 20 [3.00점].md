## 문제
지수방정식 $( 2 ^ { x } - 8 ) ( 3 ^ { 2 x } - 9 ) = 0$ 의 두 실근을 $\alpha$ , $\beta$ 라 할 때, $\alpha ^ { 2 } + \beta ^ { 2 }$ 의 값을 구하시오.

## 풀이
$2 ^ { x } - 8 = 0$ 에서 
$x = 3$ 

$3 ^ { 2 x } - 9 = 9 ^ { x } - 9 = 0$ 에서 
$x = 1$ 

$\therefore \alpha ^ { 2 } + \beta ^ { 2 } = 3 ^ { 2 } + 1 ^ { 2 } = 10$

**정답: $10$**
