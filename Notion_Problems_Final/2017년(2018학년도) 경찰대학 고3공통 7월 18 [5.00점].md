## 문제
함수 $f(x) = [x] + \left[ x + \frac{1}{100} \right] + \left[ x + \frac{2}{100} \right] + \cdots + \left[ x + \frac{99}{100} \right]$ 에 대하여 <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $[x]$ 는 $x$ 를 넘지 않는 최대 정수이다.)

## 보기
ㄱ. $f \left( \frac{4}{3} \right) = 133$
ㄴ. 자연수 $n$ 에 대하여 $f \left( x + \frac{n}{2} \right) = f(x) + 50n$
ㄷ. 자연수 $n$ 에 대하여 $\frac{n}{100} \le x < \frac{n+1}{100}$ 일 때, $f(f(x) - 1) = nf(x) - 1$ 을 만족시키는 자연수 $n$ 의 개수는 $1$ 이다.

① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄱ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
$[x] = n$ 이라 하고, $\frac{m}{100} \le x - [x] < \frac{m+1}{100}$ 이라 하자.
($m = 0, 1, 2, \dots, 99$)

$f(x) = [x] + \left[ x + \frac{1}{100} \right] + \left[ x + \frac{2}{100} \right] + \cdots + \left[ x + \frac{99}{100} \right]$ 에서

$[x] = \left[ x + \frac{1}{100} \right] = \left[ x + \frac{2}{100} \right] = \cdots = \left[ x + \frac{99-m}{100} \right] = n$ 이고
$\left[ x + \frac{100-m}{100} \right] = \left[ x + \frac{101-m}{100} \right] = \cdots = \left[ x + \frac{99}{100} \right] = n+1$
이므로
$f(x) = n \times (100 - m) + (n + 1) \times m = 100n + m$

한편, $n + \frac{m}{100} \le x < n + \frac{m+1}{100}$ 에서
$100n + m \le 100x < 100n + m + 1$ 이므로
$[100x] = 100n + m$
따라서 $f(x) = [100x]$ 이다.

ㄱ. $f \left( \frac{4}{3} \right) = \left[ \frac{400}{3} \right] = 133$ (참)

ㄴ. $f \left( x + \frac{n}{2} \right) = \left[ 100 \left( x + \frac{n}{2} \right) \right] = [100x + 50n] = [100x] + 50n = f(x) + 50n$ (참)

ㄷ. $\frac{n}{100} \le x < \frac{n+1}{100}$ 일 때, $f(x) = [100x] = n$ 이므로
$f(f(x) - 1) = f(n - 1) = [100(n - 1)] = 100(n - 1)$ 이고
$nf(x) - 1 = n^2 - 1$ 이다.
$100n - 100 = n^2 - 1$, $n^2 - 100n + 99 = 0$ 에서
$(n - 1)(n - 99) = 0$ 이므로 $n = 1$, $99$ 이다.
따라서 자연수 $n$ 의 개수는 $2$ 이다. (거짓)

따라서 옳은 것은 ㄱ, ㄴ이다.
