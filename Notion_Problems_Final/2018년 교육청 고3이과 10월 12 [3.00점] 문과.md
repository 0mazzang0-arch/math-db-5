## 문제
열린 구간 $(0, \pi)$ 에서 부등식 
$$ ( 2^{x} - 8 ) ( \cos x - \frac{1}{2} ) < 0 $$
의 해가 $a < x < b$ 또는 $c < x < d$ 일 때, $( b - a ) + ( d - c )$ 의 값은? (단, $b < c$)

## 보기
① $\pi - 3$   ② $\frac{7 \pi}{6} - 3$   ③ $\frac{4 \pi}{3} - 3$   ④ $3 - \frac{\pi}{3}$   ⑤ $3 - \frac{\pi}{6}$

## 풀이
(ⅰ) $2^{x} - 8 < 0$ 이고 $\cos x - \frac{1}{2} > 0$ 인 경우 
$x < 3$ 이고 $x < \frac{\pi}{3}$ 이므로 
$0 < x < \frac{\pi}{3}$ 

(ⅱ) $2^{x} - 8 > 0$ 이고 $\cos x - \frac{1}{2} < 0$ 인 경우 
$x > 3$ 이고 $x > \frac{\pi}{3}$ 이므로 
$3 < x < \pi$ 

따라서 $0 < x < \frac{\pi}{3}$ 또는 $3 < x < \pi$ 이므로 
$a = 0$, $b = \frac{\pi}{3}$, $c = 3$, $d = \pi$ 이다.
$$ ( b - a ) + ( d - c ) = ( \frac{\pi}{3} - 0 ) + ( \pi - 3 ) = \frac{4 \pi}{3} - 3 $$

정답: ③
