## 문제
삼차방정식 $x^{3} + 2x^{2} - 3x + 4 = 0$ 의 세 근을 $\alpha$, $\beta$, $\gamma$ 라 할 때, $( 3 + \alpha ) ( 3 + \beta ) ( 3 + \gamma )$ 의 값은?

## 보기
① $-5$   ② $-4$   ③ $-3$   ④ $-2$   ⑤ $-1$

## 풀이
삼차방정식 $x^{3} + 2x^{2} - 3x + 4 = 0$ 의 세 근이 $\alpha$, $\beta$, $\gamma$ 이므로 
$x^{3} + 2x^{2} - 3x + 4 = ( x - \alpha ) ( x - \beta ) ( x - \gamma )$
위 식의 양변에 $x = -3$ 을 대입하면 
$( -3 )^{3} + 2 ( -3 )^{2} - 3 ( -3 ) + 4 = ( -3 - \alpha ) ( -3 - \beta ) ( -3 - \gamma )$
$4 = - ( 3 + \alpha ) ( 3 + \beta ) ( 3 + \gamma )$
$\therefore ( 3 + \alpha ) ( 3 + \beta ) ( 3 + \gamma ) = -4$

정답: ②
