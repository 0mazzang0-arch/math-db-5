## 문제
두 등차수열 $\{ a_{n} \}$, $\{ b_{n} \}$에 대하여 $a_{1} + b_{1} = 45$, $\sum_{k=1}^{10} a_{k} + \sum_{k=1}^{10} b_{k} = 500$일 때, $a_{10} + b_{10}$의 값을 구하시오.

## 보기

## 풀이
$\{ a_{n} \}$, $\{ b_{n} \}$이 등차수열이므로 
$\sum_{k=1}^{10} a_{k} + \sum_{k=1}^{10} b_{k} = \frac{10 (a_{1} + a_{10})}{2} + \frac{10 (b_{1} + b_{10})}{2}$
$= 5 \{ (a_{1} + b_{1}) + (a_{10} + b_{10}) \} = 500$
$a_{1} + b_{1} = 45$이므로
$5 \{ 45 + (a_{10} + b_{10}) \} = 500$
$45 + (a_{10} + b_{10}) = 100$
$\therefore a_{10} + b_{10} = 55$
