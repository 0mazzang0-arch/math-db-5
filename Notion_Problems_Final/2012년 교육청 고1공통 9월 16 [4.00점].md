## 문제
세 실수 $a$, $b$, $c$가 다음 조건을 모두 만족시킬 때, $a$, $b$, $c$의 대소 관계로 옳은 것은?

(가) $c < 0$
(나) $a(a - b) < 0$
(다) $c^2 - ac - bc + ab < 0$

## 보기
① $a < b < c$   ② $a < c < b$   ③ $b < a < c$   ④ $b < c < a$   ⑤ $c < a < b$

## 풀이
정답: ④

조건 (나)에서 $a(a - b) < 0$이므로 
$a > 0$, $b > a$, $b > 0$ 또는 $a < 0$, $a > b$, $b < 0$
조건 (가)에서 $c < 0$이므로 조건 (다)에서 
$c^{2} - ac - bc + ab = (c - a)(c - b) < 0$
이므로 $a < c < b$ 또는 $b < c < a$

(ⅰ) $a < c < b$ 일 때 
$a - b < 0$이므로 $a > 0$이다. 그런데 $c < 0$이므로 주어진 조건을 만족시키지 않는다. 
(ⅱ) $b < c < a$ 일 때 
$a - b > 0$이므로 $a < 0$이다. 따라서 주어진 조건을 만족시킨다. 

(ⅰ), (ⅱ)에서 $b < c < a$
