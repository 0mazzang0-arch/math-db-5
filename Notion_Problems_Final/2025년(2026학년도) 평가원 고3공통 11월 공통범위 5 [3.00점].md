## 문제
함수 $f ( x ) = ( x + 2 ) ( 2 x^{2} - x - 2 )$ 에 대하여 $f' ( 1 )$ 의 값은?

## 보기
① $6$   ② $7$   ③ $8$   ④ $9$   ⑤ $10$

## 풀이
$f ( x ) = ( x + 2 ) ( 2 x^{2} - x - 2 )$ 에서
$f' ( x ) = ( 2 x^{2} - x - 2 ) + ( x + 2 ) ( 4 x - 1 )$
$= ( 2 x^{2} - x - 2 ) + ( 4 x^{2} + 7 x - 2 )$
$= 6 x^{2} + 6 x - 4$
따라서
$f' ( 1 ) = 6 + 6 - 4 = 8$

정답: ③
