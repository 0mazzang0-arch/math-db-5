## 문제
상용로그 $\log 2x$, $\log 4x$, $\log 6x$, $\log 8x$, $\log 10x$의 정수 부분의 합이 $12$가 되도록 하는 모든 자연수 $x$의 개수를 구하시오.

## 보기


## 풀이
$\log x$의 정수 부분을 $n$이라 하면 $\log 10x = 1 + \log x$이므로 $\log 2x$, $\log 4x$, $\log 6x$, $\log 8x$, $\log 10x$의 정수 부분은 각각 $n$ 또는 $n+1$이다.
정수 부분이 $n+1$인 것이 $m$개라고 하면 $(5-m)n + m(n+1) = 5n + m = 12$이다.
$n$은 정수이고 $0 \le m \le 5$이므로 $n = 2$, $m = 2$이다.
즉, $\log 2x$, $\log 4x$, $\log 6x$의 정수 부분은 $2$이고, $\log 8x$, $\log 10x$의 정수 부분은 $3$이어야 한다.
따라서 $2 \le \log 6x < 3$, $3 \le \log 8x < 4$이므로
$100 \le 6x < 1000$, $1000 \le 8x < 10000$이다.
이를 만족하는 $x$의 범위는 $\frac{1000}{8} \le x < \frac{1000}{6}$이다.
$125 \le x < 166.66\dots$
이를 만족하는 자연수 $x$의 개수는 $166 - 125 + 1 = 42$이다.

정답: $42$
