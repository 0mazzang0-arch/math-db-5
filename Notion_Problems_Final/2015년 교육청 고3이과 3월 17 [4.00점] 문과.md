## 문제
두 이차정사각행렬 $A$, $B$가 $A^2 + AB = 2E$, $A^2 - 2A = B^2 + 2B$를 만족시킬 때, <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $E$는 단위행렬이다.)

## 보기
$$ \begin{array}{|l|} \hline \text{ㄱ. } A\text{의 역행렬이 존재한다.} \\ \text{ㄴ. } AB = -BA \\ \text{ㄷ. } B^2 = 5E - 3A \\ \hline \end{array} $$

① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄷ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
ㄱ. 
$A^2 + AB = 2E$ 에서 
$\frac{1}{2}A(A + B) = E$ 이므로 
$A$ 의 역행렬은 존재하고 
$A^{-1} = \frac{1}{2}(A + B)$ 이다. (참) 

ㄴ. ㄱ에서 
$A$ 의 역행렬은 존재하고 역행렬의 정의에 의해 
$\frac{1}{2}A(A + B) = \frac{1}{2}(A + B)A = E$ 
$A(A + B) = (A + B)A$ 
$A^2 + AB = A^2 + BA$ 
$AB = BA$ 
$A$ 의 역행렬이 존재하므로 
$BA \neq O$ 
즉, 
$BA \neq -BA$ 
그러므로 
$AB \neq -BA$ 이다. (거짓) 

ㄷ. 
$A^2 - 2A = B^2 + 2B$ , 
$A^2 - B^2 = 2(A + B)$ 
$AB = BA$ 이므로 
$A^2 - B^2 = (A + B)(A - B)$ 
ㄱ의 풀이에서 
$\frac{1}{2}A(A + B) = E$ 이므로 
$(A + B)^{-1} = \frac{1}{2}A$ 
$(A + B)(A - B) = 2(A + B)$ 의 양변에 
$(A + B)$ 의 역행렬 
$\frac{1}{2}A$ 를 곱하면 
$A - B = 2E$ , 
$A = B + 2E$ 
$A^2 + AB = 2E$ 에 
$A = B + 2E$ 를 대입하면 
$2E = (B + 2E)^2 + (B + 2E)B$ 
$= 2B^2 + 6B + 4E$ 
그러므로 
$B^2 = -3B - E$ 
$= -3(A - 2E) - E$ 
$= -3A + 5E$ (참) 

따라서 옳은 것은 ㄱ, ㄷ이다. 

**[다른 풀이]** 
ㄷ은 다음과 같이 확인할 수도 있다. 
$A^2 = 2E - AB = B^2 + 2B + 2A$ 
$B^2 + 2B + 2A + AB = 2E$ 
$(A + B)(B + 2E) = 2E$ 
$\frac{1}{2}(B + 2E) = (A + B)^{-1}$ 
ㄱ의 풀이에서 
$\frac{1}{2}A(A + B) = E$ 이므로 
$(A + B)^{-1} = \frac{1}{2}A$ 
즉, 
$A = B + 2E$ 
$A^2 + AB = 2E$ 에 
$B = A - 2E$ 를 대입하면 
$A^2 + A(A - 2E) = 2E$ 
$2A^2 - 2A = 2E$ 
$A^2 = A + E$ 
그러므로 
$B^2 = (A - 2E)^2$ 
$= A^2 - 4A + 4E$ 
$= (A + E) - 4A + 4E$ 
$= -3A + 5E$
