## 문제
$k-1 \le x \le k+3$ 인 어떤 실수 $x$ 에 대하여 $0 \le x \le 2$ 이다. 가 참이 되게 하는 정수 $k$ 의 개수는?



> [!example] 📸 (그림/그래프)

## 보기
① $4$   ② $5$   ③ $6$   ④ $7$   ⑤ $8$

## 풀이
명제 $k-1 \le x \le k+3$ 인 어떤 실수 $x$ 에 대하여 $0 \le x \le 2$ 이다. 에서 조건 $k-1 \le x \le k+3$ 의 진리집합을 $P = \{x | k-1 \le x \le k+3\}$, 조건 $0 \le x \le 2$ 의 진리집합을 $Q = \{x | 0 \le x \le 2\}$ 라 하자. 

주어진 명제가 참이 되려면 진리집합 $P$ 에 속하는 원소 중에서 진리집합 $Q$ 에 속하는 원소가 존재한다는 것을 의미한다. 
즉, 주어진 명제가 참이 되려면 $P \cap Q \neq \emptyset$ 이어야 한다. 따라서 두 진리집합 $P$ 와 $Q$ 를 수직선 위에 나타내어 보면 다음의 $2$ 가지 경우 중 하나이다. 

(i) $k-1 \ge 0$ 인 경우 
$k-1 \le 2$, $k \le 3$ 
$\therefore 1 \le k \le 3$ 

(ii) $k-1 < 0$ 인 경우 
$0 \le k+3$, $k \ge -3$ 
$\therefore -3 \le k < 1$ 

그러므로 (i), (ii) 에서 $-3 \le k \le 3$ 이다. 
따라서 $-3 \le k \le 3$ 를 만족하는 정수는 $-3$, $-2$, $-1$, $0$, $1$, $2$, $3$ 의 $7$ 개다.
