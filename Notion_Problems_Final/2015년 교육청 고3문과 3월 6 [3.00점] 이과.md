## 문제
등차수열 $\{ a_{n} \}$ 에 대하여 $a_{4} = 9$, $a_{7} = 21$ 일 때, $a_{3} + a_{8}$ 의 값은?

## 보기
① $28$   ② $29$   ③ $30$   ④ $31$   ⑤ $32$

## 풀이
[정답] ③

$a_{n} = a + (n - 1)d$ ($a_{1} = a$, $d$ : 공차)에서
$a_{4} = a + 3d = 9$, $a_{7} = a + 6d = 21$ 이므로 위의 두 식을 연립하면
$d = 4$

$a_{8} = a_{4} + 4d$ 이고 $a_{3} = a_{4} - d$ 이므로
$a_{3} + a_{8} = 2a_{4} + 3d = 2 \times 9 + 3 \times 4 = 30$

[다른 풀이 1]
$d = 4$ 이므로 $a_{4} = a + 12 = 9$, $a = -3$
$a_{n} = -3 + (n - 1) \times 4 = 4n - 7$
$a_{3} = 5$, $a_{8} = 25$
$a_{3} + a_{8} = 30$

[다른 풀이 2]
$a_{4}$ 는 제 $4$ 항이고, $a_{7}$ 은 제 $7$ 항이다.
또, $a_{3}$ 은 제 $3$ 항이고, $a_{8}$ 은 제 $8$ 항이다.
$4 + 7 = 3 + 8$ 이므로 $a_{4} + a_{7} = a_{3} + a_{8}$
따라서 $a_{3} + a_{8} = 9 + 21 = 30$
