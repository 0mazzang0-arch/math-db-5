## 문제
$n \ge 2$인 모든 자연수 $n$에 대하여 집합 $A_{n}$을 $A_{n}=\{1, 2, \dots, n\}$이라 하자. 집합 $A_{n}$의 부분집합 중 원소가 $2$개인 각 부분집합에서 작은 원소를 뽑아 그 원소들의 평균을 $a_{n}$이라 하자. 다음은 $a_{n}=\frac{n+1}{3}$임을 수학적귀납법으로 증명한 것이다.

$$ \begin{array}{|l|} \hline (1) \text{ } n=2 \text{ 일 때, } A_{2}=\{1, 2\} \text{ 의 원소가 } 2 \text{ 개인 부분집합은 자신뿐이므로 } \\ a_{2}=1=\frac{2+1}{3} \text{ 이다.} \\ (2) \text{ } n=k \text{ (} k \ge 2 \text{) 일 때 성립한다고 가정하면 } a_{k}=\frac{k+1}{3} \text{ 이다.} \\ A_{k+1}=\{1, 2, \dots, k, k+1\} \text{ 의 부분집합 중 원소가 } 2 \text{ 개인 모든 부분집합은, } \\ A_{k} \text{ 의 부분집합 중 원소가 } 2 \text{ 개인 모든 부분집합에 } k \text{ 개의 집합 } \\ \{1, k+1\}, \{2, k+1\}, \dots, \{k, k+1\} \\ \text{을 추가한 것이다.} \\ A_{k} \text{ 의 부분집합 중 원소가 } 2 \text{ 개인 부분집합의 개수는 (가) 이므로} \\ a_{k+1} = \frac{(나) + (1+2+ \dots + k)}{{}_{k+1}\mathrm{C}_{2}} \\ = \frac{k+2}{3} = \frac{(k+1)+1}{3} \\ \text{그러므로 (1), (2)에 의하여 } n \ge 2 \text{ 인 모든 자연수 } n \text{ 에 대하여 } a_{n}=\frac{n+1}{3} \text{ 이다.} \\ \hline \end{array} $$

위 증명에서 (가), (나)에 알맞은 것은?

## 보기
① $(가) \text{ } {}_{k}\mathrm{C}_{2}, \text{ } (나) \text{ } {}_{k}\mathrm{C}_{2} \cdot \frac{k}{3}$   ② $(가) \text{ } {}_{k}\mathrm{C}_{2}, \text{ } (나) \text{ } {}_{k}\mathrm{C}_{2} \cdot \frac{k+1}{3}$   ③ $(가) \text{ } {}_{k+1}\mathrm{C}_{2}, \text{ } (나) \text{ } {}_{k+1}\mathrm{C}_{2} \cdot \frac{k}{3}$   ④ $(가) \text{ } {}_{k+1}\mathrm{C}_{2}, \text{ } (나) \text{ } {}_{k+1}\mathrm{C}_{2} \cdot \frac{k+1}{3}$   ⑤ $(가) \text{ } {}_{k+1}\mathrm{C}_{2}, \text{ } (나) \text{ } {}_{k}\mathrm{C}_{2} \cdot \frac{k}{3}$

## 풀이
집합 $n(A_{k})=k$이고 부분집합 중 원소가 $2$개인 부분집합의 개수는 ${}_{k}\mathrm{C}_{2}$이다. 평균이 $a$이고 변량의 개수가 $n$일 때 변량의 총합은 $a \times n$으로 계산된다. 평균 $a_{k}=\frac{k+1}{3}$이므로 ${}_{k}\mathrm{C}_{2}$개의 변량의 총합은 ${}_{k}\mathrm{C}_{2} \cdot \frac{k+1}{3}$이 되고, 위에서 열거한 $k$개의 집합에서 작은 원소들의 총합은 $1+2+3+ \dots + k$이므로 

$$ a_{k+1} = \frac{{}_{k}\mathrm{C}_{2} \cdot \frac{k+1}{3} + (1+2+3+ \dots + k)}{{}_{k+1}\mathrm{C}_{2}} $$

$$ = \frac{k+2}{3} = \frac{(k+1)+1}{3} $$

이다. 따라서 $(가)$에 알맞은 식은 ${}_{k}\mathrm{C}_{2}$이며, $(나)$에 알맞은 식은 ${}_{k}\mathrm{C}_{2} \cdot \frac{k+1}{3}$이다.

정답: ②
