## 문제
실수 $k$ 에 대하여 함수 $f(x) = 2\cos^2 x + 2\sin x + k$ 의 최댓값이 $\frac{15}{2}$ 일 때, 함수 $f(x)$ 의 최솟값은?

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
$\cos^2 x = 1 - \sin^2 x$ 이므로
$f(x) = 2\cos^2 x + 2\sin x + k$
$= 2(1 - \sin^2 x) + 2\sin x + k$
$= -2\sin^2 x + 2\sin x + 2 + k$

$\sin x = t$ 라 하면 $-1 \le t \le 1$ 이고
$y = -2t^2 + 2t + 2 + k$
$= -2(t^2 - t + \frac{1}{4}) + \frac{5}{2} + k$
$= -2(t - \frac{1}{2})^2 + \frac{5}{2} + k$

함수 $y = -2(t - \frac{1}{2})^2 + \frac{5}{2} + k$ 는 $t = \frac{1}{2}$ 에서 최댓값 $\frac{5}{2} + k$ 를 갖는다.
따라서 $\frac{5}{2} + k = \frac{15}{2}$, 즉 $k = 5$ 이다.

그러므로 함수 $y = -2t^2 + 2t + 7$ 은 $t = -1$ 에서 최솟값 $-2 \times (-1)^2 + 2 \times (-1) + 7 = 3$ 을 갖는다.
즉, $f(x)$ 의 최솟값은 $3$ 이다.
