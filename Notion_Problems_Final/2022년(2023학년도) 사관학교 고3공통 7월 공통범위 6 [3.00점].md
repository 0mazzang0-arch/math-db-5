## 문제
함수 $f(x) = \frac{1}{2} x^{4} + ax^{2} + b$ 가 $x=a$ 에서 극소이고, 극댓값 $a+8$ 을 가질 때, $a+b$ 의 값은? (단, $a$, $b$ 는 상수이다.)

## 보기
① $2$   ② $3$   ③ $4$   ④ $5$   ⑤ $6$

## 풀이
$f(x) = \frac{1}{2} x^{4} + ax^{2} + b$ 에서 

$f'(x) = 2x^{3} + 2ax = 2x(x^{2} + a)$ $\cdots$ ㉠ 

함수 $f(x)$ 가 극댓값을 가지기 위해서는 ㉠에서 $2x(x^{2} + a) = 0$ 이 서로 다른 세 실근을 가져야 하므로 $a < 0$ 

$x=a$ 에서 극소이므로 

$f'(a) = 2a(a^{2} + a) = 2a^{2}(a + 1) = 0$ 

$a < 0$ 이므로 $a = -1$ 

$a = -1$ 을 ㉠에 대입하면 

$f'(x) = 2x(x^{2} - 1) = 2x(x + 1)(x - 1)$ 

따라서 함수 $f(x)$ 는 $x = 0$ 에서 극댓값을 가지므로 

$f(0) = b = a + 8$

$a = -1$ 이므로 $b = -1 + 8 = 7$

$\therefore a + b = 6$

정답은 ⑤이다.
