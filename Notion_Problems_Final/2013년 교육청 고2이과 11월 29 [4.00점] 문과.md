## 문제
$x$에 대한 연립부등식 
$$ \begin{cases} (x-n)(x-2n)(x-3n) \ge 0 \\ \frac{x-20}{x} \le 0 \end{cases} $$
을 만족시키는 자연수 $x$의 개수가 $10$이 되도록 하는 모든 자연수 $n$의 값의 합을 구하시오.

## 보기


## 풀이
$\frac{x-20}{x} \le 0$ 에서 
$0 < x \le 20$ 

$(x-n)(x-2n)(x-3n) \ge 0$ 에서 
$n \le x \le 2n$ 또는 $x \ge 3n$ 

> [!example] 📸 (그림/그래프)

i) $n \le 20 < 2n$ 즉, $10 < n \le 20$ 인 경우 
$20 - n + 1 = 10$ 이므로 
$n = 11$ 

ii) $2n \le 20 < 3n$ 즉, $\frac{20}{3} < n \le 10$ 인 경우 
$2n - n + 1 = 10$ 이므로 
$n = 9$ 

iii) $3n \le 20$ 즉, $n \le \frac{20}{3}$ 인 경우 
$(2n - n + 1) + (20 - 3n + 1) = 10$ 이므로 
$22 - 2n = 10$, $n = 6$ 

따라서 모든 $n$의 값의 합은 
$11 + 9 + 6 = 26$

정답: $26$
