## 문제
두 양수 $x$, $y$ 에 대하여 등식 
$$( \log_{ 3 } x )^{ 2 } + ( \log_{ 3 } y )^{ 2 } = \log_{ 9 } x^{ 2 } + \log_{ 9 } y^{ 2 }$$
이 성립할 때, $xy$ 의 최댓값은 $M$, 최솟값은 $m$ 이다. $M + m$ 의 값을 구하시오.

## 보기

## 풀이
$\log_{ 3 } x = X$, $\log_{ 3 } y = Y$ 라 하면 
$$( \log_{ 3 } x )^{ 2 } + ( \log_{ 3 } y )^{ 2 } = 2 \log_{ 9 } x + 2 \log_{ 9 } y = \log_{ 3 } x + \log_{ 3 } y$$ 는 
$$X^{ 2 } + Y^{ 2 } = X + Y$$, 
$$\left( X - \frac{ 1 }{ 2 } \right)^{ 2 } + \left( Y - \frac{ 1 }{ 2 } \right)^{ 2 } = \frac{ 1 }{ 2 } \cdots ㉠$$ 

$\log_{ 3 } xy = \log_{ 3 } x + \log_{ 3 } y = X + Y$ 에서 직선 
$$X + Y - \log_{ 3 } xy = 0$$ 과 $㉠$이 만나야 하므로 
$$\frac{ \left| \frac{ 1 }{ 2 } + \frac{ 1 }{ 2 } - \log_{ 3 } xy \right| }{ \sqrt{ 1^{ 2 } + 1^{ 2 } } } \le \frac{ 1 }{ \sqrt{ 2 } }$$
$$| 1 - \log_{ 3 } xy | \le 1$$ 
$$0 \le \log_{ 3 } xy \le 2$$ 
$$\therefore 1 \le xy \le 9$$ 
따라서 $M = 9$, $m = 1$ 이므로
$$\therefore M + m = 9 + 1 = 10$$
