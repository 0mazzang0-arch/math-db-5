## 문제
실수 $x_1$에 대하여 함수 $f(x) = 2x(1-x)$의 그래프 위에 있는 점 $P_n(x_n, y_n)$을 다음과 같이 귀납적으로 정의한다.

$$ \begin{array}{l} P_1 = P_1(x_1, y_1) = P_1(x_1, f(x_1)) \\ P_{n+1} = P_{n+1}(x_{n+1}, y_{n+1}) = P_{n+1}(y_n, f(y_n)) \\ (n = 1, 2, 3, \cdots) \end{array} $$

이때 <보기>에서 참인 명제만을 있는 대로 고른 것은?

$$ \begin{array}{|l|} \hline \text{ㄱ. } x_1 \neq \frac{1}{2} \text{ 일 때, } a_n = \log |1 - 2x_n| \text{ 이라 하면, } a_n = 2^{n-1} \log |1 - 2x_1| \text{ 이다.} \\ \text{ㄴ. } 0 < x_1 < 1 \text{ 이고 } x_1 \neq \frac{1}{2} \text{ 일 때, } n \text{ 의 값이 커짐에 따라 점 } P_n \text{ 은 점 } \left( \frac{1}{2}, \frac{1}{2} \right) \text{ 에 한없이 가까워진다.} \\ \text{ㄷ. } x_1 < 0 \text{ 일 때 } n \text{ 의 값이 커짐에 따라 점 } P_n \text{ 은 점 } (0, 0) \text{ 에 한없이 가까워진다.} \\ \hline \end{array} $$

## 보기
① $ㄴ$   ② $ㄱ, ㄴ$   ③ $ㄱ, ㄷ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
[정답] ②

$ㄱ$. 
$$ 1 - 2x_{n+1} = 1 - 2y_n = 1 - 2f(x_n) = 1 - 4x_n(1 - x_n) = (1 - 2x_n)^2 $$
$\log |1 - 2x_{n+1}| = 2 \log |1 - 2x_n|$ 이므로 
$a_{n+1} = 2a_n$ 
$\therefore a_n = a_1 \cdot 2^{n-1} = 2^{n-1} \cdot \log |1 - 2x_1|$ ($참$)

$ㄴ$. 
$0 < x_1 < 1$ 일 때, 
$0 < |1 - 2x_1| < 1$ 이므로 
$\log |1 - 2x_1| < 0$ 
$\therefore \lim_{n \to \infty} a_n = -\infty$ ($\because a_n = 2^{n-1} \cdot \log |1 - 2x_1|$) 
$a_n = \log |1 - 2x_n|$ 이므로 $a_n \to -\infty$ 이려면 $|1 - 2x_n| \to 0$ 이어야 한다. 
$\therefore \lim_{n \to \infty} x_n = \frac{1}{2}$, 
$\lim_{n \to \infty} y_n = \lim_{n \to \infty} f(x_n) = f\left( \frac{1}{2} \right) = \frac{1}{2}$ ($참$)

$ㄷ$. 
$x_1 < 0$ 일 때, 
$1 - 2x_1 > 1$ 이므로 
$\log |1 - 2x_1| > 0$ 
$\therefore \lim_{n \to \infty} a_n = \infty$ ($\because a_n = 2^{n-1} \cdot \log |1 - 2x_1|$) 
$a_n = \log |1 - 2x_n|$ 이므로 $a_n \to \infty$ 이려면 $|1 - 2x_n| \to \infty$ 이어야 한다. 
즉, $\lim_{n \to \infty} |x_n| = \infty$ 이어야 한다. ($거짓$)
