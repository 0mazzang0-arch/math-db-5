## 문제
구간 $[0, 3]$ 의 모든 실수 값을 가지는 연속확률변수 $X$ 에 대하여 $X$ 의 확률밀도함수의 그래프는 그림과 같다. $P(0 \le X \le 2) = \frac{q}{p}$ 라 할 때, $p+q$ 의 값을 구하시오. (단, $k$ 는 상수이고, $p$ 와 $q$ 는 서로소인 자연수이다.)

> [!example] 📸 (그림/그래프)

## 풀이
$P(0 \le X \le 3) = 1$ 이므로
$3k + \frac{1}{2} \times 3 \times 2k = 1$
$6k = 1$
$\therefore k = \frac{1}{6}$

$P(0 \le X \le 2)$ 의 값은 사다리꼴의 넓이와 같으므로
$P(0 \le X \le 2) = \frac{1}{2} \times (3k + k) \times 2$
$= 4k$
$= 4 \times \frac{1}{6}$
$= \frac{2}{3}$

따라서 $\frac{q}{p} = \frac{2}{3}$ 이고, $p$ 와 $q$ 는 서로소인 자연수이므로
$p = 3, q = 2$
$\therefore p + q = 3 + 2 = 5$

정답: $5$
