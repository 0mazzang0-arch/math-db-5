## 문제
$x \ge 0$에서 정의된 연속함수 $f(x)$가 다음 조건을 만족시킨다.

(가) $$ f(x) = \left\{ \begin{array}{ll} -x(x-2) & (0 \le x < 2) \\ \frac{1}{2}x-1 & (2 \le x \le 4) \end{array} \right. $$
(나) $x \ge 0$인 모든 실수 $x$에 대하여 $f(x+4) = f(x)+1$이다.

실수 $a$가 $5 \le a \le 6$일 때, $\int_{a}^{a+2} f(x) dx$의 최솟값은?

## 보기
① $\frac{121}{48}$   ② $\frac{31}{12}$   ③ $\frac{127}{48}$   ④ $\frac{65}{24}$   ⑤ $\frac{133}{48}$

## 풀이
$g(a) = \int_{a}^{a+2} f(x) dx$라 하면 
$g'(a) = f(a+2) - f(a)$

(ⅰ) $f(a+2)$의 값을 구하면 
$5 \le a \le 6$에서 $7 \le a+2 \le 8$
$f(a+2) = f((a-2)+4) = f(a-2)+1$
$3 \le a-2 \le 4$이므로 조건 (가)에서 
$f(a-2) = \frac{1}{2}(a-2)-1 = \frac{1}{2}a-2$
$\therefore f(a+2) = (\frac{1}{2}a-2)+1 = \frac{1}{2}a-1$

(ⅱ) $f(a)$의 값을 구하면 
$5 \le a \le 6$이므로 
$f(a) = f((a-4)+4) = f(a-4)+1$
$1 \le a-4 \le 2$이므로 조건 (가)에서 
$f(a-4) = -(a-4)\{(a-4)-2\} = -(a-4)(a-6) = -a^{2}+10a-24$
$\therefore f(a) = -a^{2}+10a-24+1 = -a^{2}+10a-23$

(ⅰ), (ⅱ)에 의해 
$f(a+2) = \frac{1}{2}a-1$, $f(a) = -a^{2}+10a-23$이므로 
$g'(a) = f(a+2)-f(a) = (\frac{1}{2}a-1) - (-a^{2}+10a-23) = a^{2}-\frac{19}{2}a+22$
$g'(a) = 0$에서 $2a^{2}-19a+44=0$, $(2a-11)(a-4)=0$
$\therefore a=4$ 또는 $a=\frac{11}{2}$
이때 $5 \le a \le 6$이므로 $a=\frac{11}{2}$
따라서 $g(a)$는 $a=\frac{11}{2}$에서 극솟값이면서 최솟값을 가지므로 $g(a)$의 최솟값은 
$\int_{11/2}^{15/2} f(x) dx$

함수 $f(x)$를 $x$축의 방향으로 $-4$만큼 평행이동하면 
$\int_{11/2}^{15/2} f(x) dx = \int_{3/2}^{7/2} f(x+4) dx = \int_{3/2}^{7/2} \{f(x)+1\} dx$
$= \int_{3/2}^{7/2} f(x) dx + 2$
$= \int_{3/2}^{2} (-x^{2}+2x) dx + \int_{2}^{7/2} (\frac{1}{2}x-1) dx + 2$
$= [-\frac{1}{3}x^{3}+x^{2}]_{3/2}^{2} + [\frac{1}{4}x^{2}-x]_{2}^{7/2} + 2$
$= \frac{5}{24} + \frac{9}{16} + 2 = \frac{133}{48}$

따라서 구하는 최솟값은 $\frac{133}{48}$이다.

정답: ⑤
