## 문제
실수 $x$ 에 대하여 $f(x)= \lim _{n \to \infty} \frac{x^{2n+1} - 2x^{2n} + 1}{x^{2n+2} + x^{2n} + 1}$ 일 때, $\lim _{x \to -1-} f(x)=a$, $\lim _{x \to 1-} f(x)=b$ 라 하자. $\frac{b}{a+2}$ 의 값은?

## 보기
① $-\frac{1}{4}$   ② $-\frac{1}{2}$   ③ $\frac{1}{2}$   ④ $2$   ⑤ $4$

## 풀이
$|x| < 1$ 이면 
$f(x) = \lim_{n \to \infty} \frac{x^{2n+1} - 2x^{2n} + 1}{x^{2n+2} + x^{2n} + 1} = 1$ 

$|x| > 1$ 이면 
$f(x) = \lim_{n \to \infty} \frac{x^{2n+1} - 2x^{2n} + 1}{x^{2n+2} + x^{2n} + 1}$ 
$= \lim_{n \to \infty} \frac{x - 2 + \frac{1}{x^{2n}}}{x^2 + 1 + \frac{1}{x^{2n}}} = \frac{x-2}{x^2 + 1}$ 

$\lim_{x \to -1-} f(x) = \lim_{x \to -1-} \frac{x-2}{x^2 + 1} = -\frac{3}{2} = a$ 
$\lim_{x \to 1-} f(x) = 1 = b$ 

$\therefore \frac{b}{a+2} = \frac{1}{-\frac{3}{2} + 2} = \frac{1}{\frac{1}{2}} = 2$

정답: ④
