## 문제
$2$ 이상의 네 자연수 $a$, $b$, $c$, $d$에 대하여 
$$( 14^2 + 2 \times 14 )^2 - 18 \times ( 14^2 + 2 \times 14 ) + 45 = a \times b \times c \times d$$ 
일 때, $a+b+c+d$의 값은?

## 보기
① $56$   ② $58$   ③ $60$   ④ $62$   ⑤ $64$

## 풀이
$14 = X$라 하면 
$$( 14^2 + 2 \times 14 )^2 - 18 \times ( 14^2 + 2 \times 14 ) + 45$$ 
$$= ( X^2 + 2X )^2 - 18 ( X^2 + 2X ) + 45$$ 
$$= ( X^2 + 2X - 3 ) ( X^2 + 2X - 15 )$$ 
$$= ( X-1 ) ( X+3 ) ( X-3 ) ( X+5 )$$ 
$X = 14$를 대입하면 
$$( 14-1 ) \times ( 14+3 ) \times ( 14-3 ) \times ( 14+5 )$$ 
$$= 13 \times 17 \times 11 \times 19$$ 
$a, b, c, d$는 $2$ 이상의 자연수이므로 네 수는 $11, 13, 17, 19$이다.
따라서 $a+b+c+d = 13+17+11+19 = 60$
