## 문제
최고차항의 계수가 $1$ 인 두 삼차함수 $f (x )$, $g (x )$ 가 다음 조건을 만족시킨다.

(가) $g (2 )=0$
(나) $n=2$, $3$, $4$, $5$ 일 때, $\lim_{x \to n} \frac{f (x )}{g (x )} = (n-2 ) (n-3 )$ 이다.

$g (6 )$ 의 값은?

## 보기
① $12$   ② $13$   ③ $14$   ④ $15$   ⑤ $16$

## 풀이
조건 (나)에서 
$\lim_{x \to 2} \frac{f (x )}{g (x )} = 0$, $\lim_{x \to 3} \frac{f (x )}{g (x )} = 0$
이므로 $f (2 )=0$, $f (3 )=0$
이때 최고차항의 계수가 $1$ 인 삼차함수 $f (x )$ 를 
$f (x )= (x-2 ) (x-3 ) (x-a )$ ($a$ 는 상수) 
라 하고, 조건 (가)에서 $g (2 )=0$ 이므로, 최고차항의 계수가 $1$ 인 삼차함수 $g (x )$ 를 
$g (x )= (x-2 ) (x^{2}+px+q )$ ($p$, $q$ 는 상수) 
라 하자. 

조건 (나)에서 $n=2$ 일 때 
$\lim_{x \to 2} \frac{f (x )}{g (x )} = \lim_{x \to 2} \frac{(x-2 ) (x-3 ) (x-a )}{(x-2 ) (x^{2}+px+q )} = \frac{(2-3 ) (2-a )}{4+2p+q} = \frac{a-2}{4+2p+q} = 0$
이므로 $a=2$ ( 단, $4+2p+q \neq 0$) 
$\therefore f (x ) = (x-2 )^{2} (x-3 )$ 

또, $n=4$ 일 때 $\lim_{x \to 4} \frac{f (x )}{g (x )} = 2$ 이므로 
$\lim_{x \to 4} \frac{f (x )}{g (x )} = \frac{(4-2 )^{2} (4-3 )}{g (4 )} = \frac{4}{g (4 )} = 2$ 
$\therefore g (4 ) = 2 \cdots \cdots$ ㉠ 

또, $n=5$ 일 때 $\lim_{x \to 5} \frac{f (x )}{g (x )} = 6$ 이므로 
$\lim_{x \to 5} \frac{f (x )}{g (x )} = \frac{(5-2 )^{2} (5-3 )}{g (5 )} = \frac{18}{g (5 )} = 6$ 
$\therefore g (5 ) = 3 \cdots \cdots$ ㉡ 

조건 (가)와 ㉠, ㉡에 의하여 
$g (2 ) = 0$, $g (4 ) = 2$, $g (5 ) = 3$ 
이므로 $h (x ) = g (x ) - (x-2 )$ 라 하면, $h (x )$ 는 최고차항의 계수가 $1$ 인 삼차함수이고 
$h (2 ) = g (2 ) - (2-2 ) = 0$, 
$h (4 ) = g (4 ) - (4-2 ) = 0$, 
$h (5 ) = g (5 ) - (5-2 ) = 0$ 

따라서 $h (x ) = (x-2 ) (x-4 ) (x-5 )$ 이므로 
$g (x ) = h (x ) + (x-2 ) = (x-2 ) (x-4 ) (x-5 ) + (x-2 )$ 
$\therefore g (6 ) = 4 \times 2 \times 1 + 4 = 12$
