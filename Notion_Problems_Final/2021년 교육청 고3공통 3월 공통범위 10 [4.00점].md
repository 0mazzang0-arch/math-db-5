## 문제
자연수 $n$에 대하여 점 $A_n(n, n^2)$을 지나고 직선 $y=nx$에 수직인 직선이 $x$축과 만나는 점을 $B_n$이라 하자.

> [!example] 📸 (그림/그래프)

다음은 삼각형 $A_nOB_n$의 넓이를 $S_n$이라 할 때, $\sum_{n=1}^{8} \frac{S_n}{n^3}$의 값을 구하는 과정이다. (단, $O$는 원점이다.)

점 $A_n(n, n^2)$을 지나고 직선 $y=nx$에 수직인 직선의 방정식은
$$y = (가) \times x + n^2 + 1$$
이므로 두 점 $A_n$, $B_n$의 좌표를 이용하여 $S_n$을 구하면
$$S_n = (나)$$
따라서
$$\sum_{n=1}^{8} \frac{S_n}{n^3} = (다)$$
이다.

위의 (가), (나)에 알맞은 식을 각각 $f(n)$, $g(n)$이라 하고, (다)에 알맞은 수를 $r$라 할 때, $f(1) + g(2) + r$의 값은?

## 보기
① $105$   ② $110$   ③ $115$   ④ $120$   ⑤ $125$

## 풀이
점 $A_n(n, n^2)$을 지나고 직선 $y=nx$에 수직인 직선의 기울기는 $-\frac{1}{n}$이므로 직선의 방정식은
$$y - n^2 = -\frac{1}{n}(x - n)$$
$$y = -\frac{1}{n}x + n^2 + 1$$
즉, $(가) = -\frac{1}{n}$이다.

점 $B_n$의 좌표는 $-\frac{1}{n}x + n^2 + 1 = 0$에서 $(n^3 + n, 0)$이다.
점 $A_n$의 좌표는 $(n, n^2)$이므로 삼각형 $A_nOB_n$의 넓이 $S_n$은
$$S_n = \frac{1}{2} \times (n^3 + n) \times n^2 = \frac{n^5 + n^3}{2}$$
즉, $(나) = \frac{n^5 + n^3}{2}$이다.

따라서
$$\sum_{n=1}^{8} \frac{S_n}{n^3} = \sum_{n=1}^{8} \frac{n^5 + n^3}{2n^3} = \sum_{n=1}^{8} \frac{n^2 + 1}{2}$$
$$= \frac{1}{2} \sum_{n=1}^{8} n^2 + \frac{1}{2} \sum_{n=1}^{8} 1$$
$$= \frac{1}{2} \times \frac{8 \times 9 \times 17}{6} + \frac{1}{2} \times 1 \times 8$$
$$= 102 + 4 = 106$$
즉, $(다) = 106$이다.

(가), (나)에 알맞은 식 $f(n) = -\frac{1}{n}$, $g(n) = \frac{n^5 + n^3}{2}$와 (다)에 알맞은 수 $r = 106$에 대하여
$$f(1) + g(2) + r = -1 + \frac{2^5 + 2^3}{2} + 106$$
$$= -1 + 20 + 106 = 125$$

정답은 ⑤이다.
