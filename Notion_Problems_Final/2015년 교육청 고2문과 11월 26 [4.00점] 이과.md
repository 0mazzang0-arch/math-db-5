## 문제
두 집합 $A = \{ 1, 2, 3, 4, a \}$, $B = \{ 1, 3, 5 \}$에 대하여 집합 $X = \{ x + y \mid x \in A, y \in B \}$라 할 때, $n(X) = 10$이 되도록 하는 자연수 $a$의 최댓값을 구하시오.

## 보기

## 풀이
$X = \{ 2, 3, 4, 5, 6, 7, 8, 9, a + 1, a + 3, a + 5 \}$이므로 $n(X) = 10$이 되기 위해서는 
$( a + 3 < 2, a + 5 \geq 2 )$ 또는 $( a + 1 \leq 9, a + 3 > 9 )$이어야 한다.

$\therefore -3 \leq a < -1$ 또는 $6 < a \leq 8$

그러므로 자연수 $a$는 $7, 8$이다.
따라서 자연수 $a$의 최댓값은 $8$이다.

[정답] $8$
