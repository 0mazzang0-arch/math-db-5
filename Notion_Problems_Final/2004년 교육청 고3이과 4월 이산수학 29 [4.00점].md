## 문제
삼각형 $A B C$에서 $\sin ( A+B ) \sin ( A-B ) = \sin^{2} C$일 때, $\angle A$의 크기는?

## 보기
① $\frac{\pi}{6}$   ② $\frac{\pi}{4}$   ③ $\frac{\pi}{3}$   ④ $\frac{\pi}{2}$   ⑤ $\frac{3 \pi}{4}$

## 풀이
$\sin ( A+B ) \sin ( A-B )$
$= -\frac{1}{2} ( \cos 2 A - \cos 2 B )$
$= -\frac{1}{2} ( 1 - 2 \sin^{2} A - 1 + 2 \sin^{2} B )$
$= \sin^{2} A - \sin^{2} B = \sin^{2} C$

삼각형 $A B C$의 대변의 길이를 각각 $a$, $b$, $c$, 외접원의 반지름의 길이를 $R$라 하면
$\sin A = \frac{a}{2 R}$, $\sin B = \frac{b}{2 R}$, $\sin C = \frac{c}{2 R}$이므로
$\left( \frac{a}{2 R} \right)^{2} - \left( \frac{b}{2 R} \right)^{2} = \left( \frac{c}{2 R} \right)^{2}$
$a^{2} - b^{2} = c^{2}$, 즉 $a^{2} = b^{2} + c^{2}$이다.

따라서 삼각형 $A B C$는 $\angle A$가 직각인 직각삼각형이므로
$\angle A = \frac{\pi}{2}$이다.

정답: ④
