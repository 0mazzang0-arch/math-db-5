## 문제
$2$ 이상의 세 자연수 $p$, $q$, $r$ 에 대하여
$42 \times ( 42-1 ) \times ( 42+6 ) +5 \times 42-5=p \times q \times r$
일 때, $p+q+r$ 의 값은?

## 보기
① $131$   ② $133$   ③ $135$   ④ $137$   ⑤ $139$

## 풀이
$42=A$ 라 하면
$42 \times ( 42-1 ) \times ( 42+6 ) +5 \times 42-5$
$=A ( A-1 ) ( A+6 ) +5A-5$
$=A ( A-1 ) ( A+6 ) +5 ( A-1 )$
$= ( A-1 ) \{ A ( A+6 ) +5 \}$
$= ( A-1 ) ( A^{2} +6A+5 )$
$= ( A-1 ) ( A+1 ) ( A+5 )$
$=41 \times 43 \times 47$

따라서 $p+q+r=41+43+47=131$
정답: ①
