## 문제
두 다항함수 $f(x)$, $g(x)$ 가 
$f(x) = \int x g(x) dx$, $\frac{d}{dx} \{ f(x) - g(x) \} = 4x^{3} + 2x$
를 만족시킬 때, $g(1)$ 의 값은?

## 보기
① $10$   ② $11$   ③ $12$   ④ $13$   ⑤ $14$

## 풀이
$f'(x) = xg(x)$, 
$f'(x) - g'(x) = 4x^{3} + 2x$ 이므로 
$xg(x) - g'(x) = 4x^{3} + 2x$ 
함수 $g(x)$ 는 최고차항의 계수가 $4$ 인 
$g(x) = 4x^{2} + ax + b$ 
$g'(x) = 8x + a$ 이므로 
$x( 4x^{2} + ax + b ) - ( 8x + a ) = 4x^{3} + 2x$ 
$4x^{3} + ax^{2} + ( b - 8 ) x - a = 4x^{3} + 2x$ 에서 
$a = 0$, $b = 10$ 이므로 
$g(x) = 4x^{2} + 10$ 
따라서 $g(1) = 14$
