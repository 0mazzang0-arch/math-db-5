## 문제
두 집합 $X = \{ 1, 2, 3, 4, 5, 6, 7, 8 \}$, $Y = \{ 1, 2, 3, 4, 5 \}$에 대하여 다음 조건을 만족시키는 $X$에서 $Y$로의 함수 $f$의 개수를 구하시오.

(가) $f(4) = f(1) + f(2) + f(3)$
(나) $2f(4) = f(5) + f(6) + f(7) + f(8)$

## 보기

## 풀이
$f(k) = x_{k}$ ($k = 1, 2, \cdots, 8$)이라 하면 $x_{k}$는 $5$ 이하의 자연수이다.
구하는 함수 $f$의 개수는 두 방정식 $x_{4} = x_{1} + x_{2} + x_{3}$, $2x_{4} = x_{5} + x_{6} + x_{7} + x_{8}$을 만족시키는 $5$ 이하의 자연수 $x_{1}, x_{2}, \cdots, x_{8}$의 모든 순서쌍 $(x_{1}, x_{2}, \cdots, x_{8})$의 개수와 같다.
$x_{1} + x_{2} + x_{3} \ge 3$이므로 조건 (가)에 의하여 $3 \le x_{4} \le 5$이다.

(ⅰ) $x_{4} = 3$인 경우
$x_{1} + x_{2} + x_{3} = 3$을 만족시키는 $5$ 이하의 자연수 $x_{1}, x_{2}, x_{3}$의 순서쌍 $(x_{1}, x_{2}, x_{3})$은 $(1, 1, 1)$의 $1$가지이다.
$x_{5} + x_{6} + x_{7} + x_{8} = 6$을 만족시키는 $5$ 이하의 자연수 $x_{5}, x_{6}, x_{7}, x_{8}$의 순서쌍 $(x_{5}, x_{6}, x_{7}, x_{8})$의 개수는 서로 다른 $4$개에서 $2$개를 택하는 중복조합의 수와 같으므로 ${}_{4}H_{2} = {}_{5}C_{2} = 10$이다.
$f$의 개수는 $1 \times 10 = 10$이다.

(ⅱ) $x_{4} = 4$인 경우
$x_{1} + x_{2} + x_{3} = 4$를 만족시키는 $5$ 이하의 자연수 $x_{1}, x_{2}, x_{3}$의 순서쌍 $(x_{1}, x_{2}, x_{3})$은 $(1, 1, 2), (1, 2, 1), (2, 1, 1)$의 $3$가지이다.
$x_{5} + x_{6} + x_{7} + x_{8} = 8$을 만족시키는 $5$ 이하의 자연수 $x_{5}, x_{6}, x_{7}, x_{8}$의 순서쌍 $(x_{5}, x_{6}, x_{7}, x_{8})$의 개수는 서로 다른 $4$개에서 $4$개를 택하는 중복조합의 수와 같으므로 ${}_{4}H_{4} = {}_{7}C_{4} = 35$이다.
$f$의 개수는 $3 \times 35 = 105$이다.

(ⅲ) $x_{4} = 5$인 경우
$x_{1} + x_{2} + x_{3} = 5$를 만족시키는 $5$ 이하의 자연수 $x_{1}, x_{2}, x_{3}$의 모든 순서쌍 $(x_{1}, x_{2}, x_{3})$의 개수는 서로 다른 $3$개에서 $2$개를 택하는 중복조합의 수와 같으므로 ${}_{3}H_{2} = {}_{4}C_{2} = 6$이다.
$x_{5} + x_{6} + x_{7} + x_{8} = 10$을 만족시키는 $5$ 이하의 자연수 $x_{5}, x_{6}, x_{7}, x_{8}$의 모든 순서쌍 $(x_{5}, x_{6}, x_{7}, x_{8})$의 개수는 방정식 $x_{5} + x_{6} + x_{7} + x_{8} = 10$을 만족시키는 전체 자연수 해의 개수에서 $x_{k} \ge 6$인 $k$ ($k = 5, 6, 7, 8$)가 존재하는 경우를 제외한 개수와 같다.
$x_{5} + x_{6} + x_{7} + x_{8} = 10$을 만족시키는 자연수 순서쌍의 개수는 서로 다른 $4$개에서 $6$개를 택하는 중복조합의 수와 같으므로 ${}_{4}H_{6} = {}_{9}C_{6} = 84$이다.
$x_{5} + x_{6} + x_{7} + x_{8} = 10$을 만족시키는 $x_{5}, x_{6}, x_{7}, x_{8}$ 중 $6$ 이상의 자연수가 존재하는 모든 순서쌍 $(x_{5}, x_{6}, x_{7}, x_{8})$의 개수는 $\{ 6, 2, 1, 1 \}$을 일렬로 나열하는 경우의 수와 $\{ 7, 1, 1, 1 \}$을 일렬로 나열하는 경우의 수의 합과 같으므로 $\frac{4!}{2!} + \frac{4!}{3!} = 16$이다.
$f$의 개수는 $6 \times (84 - 16) = 408$이다.

(ⅰ), (ⅱ), (ⅲ)에 의하여 구하는 함수 $f$의 개수는 $10 + 105 + 408 = 523$이다.

정답: $523$
