## 문제
이차방정식 $2x^{2} - 2x + 1 = 0$ 의 한 근을 $\alpha$ 라 할 때, $\alpha^{4} - \alpha^{2} + \alpha$ 의 값은?

## 보기
① $\frac{1}{4}$   ② $\frac{5}{16}$   ③ $\frac{3}{8}$   ④ $\frac{7}{16}$   ⑤ $\frac{1}{2}$

## 풀이
방정식 $2x^{2} - 2x + 1 = 0$ 의 근은 $x = \frac{1 \pm i}{2}$ 이다.
이때, $\alpha = \frac{1 + i}{2}$ 라 하면 $\alpha^{2} = \frac{1}{2}i$ 이다.

$\alpha^{4} = \left( \frac{1}{2} i \right)^{2} = -\frac{1}{4}$

$\alpha^{4} - \alpha^{2} + \alpha = -\frac{1}{4} - \frac{1}{2} i + \frac{1 + i}{2} = \frac{1}{4}$

$\alpha = \frac{1 - i}{2}$ 인 경우도 마찬가지로 성립한다.
따라서 $\alpha^{4} - \alpha^{2} + \alpha = \frac{1}{4}$ 이다.

[다른 풀이]
$\alpha^{2} = \alpha - \frac{1}{2}$ 이므로
$\alpha^{4} = \left( \alpha - \frac{1}{2} \right)^{2} = \alpha^{2} - \alpha + \frac{1}{4}$
따라서 $\alpha^{4} - \alpha^{2} + \alpha = \left( \alpha^{2} - \alpha + \frac{1}{4} \right) - \alpha^{2} + \alpha = \frac{1}{4}$ 이다.

정답: ①
