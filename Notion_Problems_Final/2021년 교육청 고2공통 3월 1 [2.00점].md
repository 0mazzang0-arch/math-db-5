## 문제
두 다항식 $A = 3 x^{2} + 2 x y$, $B = - x^{2} + x y$에 대하여 $A + 2 B$를 간단히 하면?

## 보기
① $x^{2} + 3 x y$   ② $x^{2} + 4 x y$   ③ $x^{2} + 5 x y$   ④ $2 x^{2} + 4 x y$   ⑤ $2 x^{2} + 5 x y$

## 풀이
$A + 2 B$
$= ( 3 x^{2} + 2 x y ) + 2 ( - x^{2} + x y )$
$= ( 3 x^{2} + 2 x y ) + ( - 2 x^{2} + 2 x y )$
$= x^{2} + 4 x y$

정답: ②
