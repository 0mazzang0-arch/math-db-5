## 문제
자연수 $x$에 대하여 $x^4 + x^3 + x^2 + x + 1$이 어떤 자연수의 제곱이 되는 $x$의 개수를 구하는 과정이다.

$x^4 + x^3 + x^2 + x + 1 = y^2$ ($y$는 자연수)라 하자.

$4x^4 + 4x^3 + 4x^2 + 4x + 4 = (2y)^2$

$(\text{(가)})^2 + (3x^2 + 4x + 4) = (2y)^2 \cdots$ ㉠

$(2x^2 + x + 1)^2 - (x - 3)(x + 1) = (2y)^2 \cdots$ ㉡

$(x - 3)(x + 1) > 0$이면 ㉡에서 $2x^2 + x + 1 > 2y$이고 ㉠에서 $(\text{(가)}) < 2y$이므로 만족하는 자연수 $y$는 존재하지 않는다.

따라서 $(x - 3)(x + 1) \le 0$이므로 $-1 \le x \le 3$이다.

그러므로 $x^4 + x^3 + x^2 + x + 1$을 어떤 자연수의 제곱이 되게 하는 자연수 $x$의 개수는 $(\text{(나)})$개이다.

위의 과정에서 (가), (나)에 알맞은 것은?

## 보기
① $(가) \ 2x^2 + 1, (나) \ 1$   ② $(가) \ 2x^2 + 2x, (나) \ 1$   ③ $(가) \ 2x^2 + x, (나) \ 1$   ④ $(가) \ 2x^2 + 2x, (나) \ 2$   ⑤ $(가) \ 2x^2 + x, (나) \ 2$

## 풀이
$x^4 + x^3 + x^2 + x + 1 = y^2$ ($y$는 자연수)라 하면,
$4(x^4 + x^3 + x^2 + x + 1) = 4y^2$
$4x^4 + 4x^3 + 4x^2 + 4x + 4 = (2y)^2$
이때, $4x^4 + 4x^3 + 4x^2 + 4x + 4 = (2x^2 + x)^2 + 3x^2 + 4x + 4$ 이므로
$(2x^2 + x)^2 + (3x^2 + 4x + 4) = (2y)^2 \cdots$ ㉠
또한, $4x^4 + 4x^3 + 4x^2 + 4x + 4 = (2x^2 + x + 1)^2 - (x^2 - 2x - 3)$
$= (2x^2 + x + 1)^2 - (x - 3)(x + 1) = (2y)^2 \cdots$ ㉡

$(x - 3)(x + 1) > 0$ 이면,
㉡에서 $(2x^2 + x + 1)^2 > (2y)^2$ 이므로 $2x^2 + x + 1 > 2y$
㉠에서 $x$가 자연수일 때 $3x^2 + 4x + 4 > 0$ 이므로 $(2x^2 + x)^2 < (2y)^2$, 즉 $2x^2 + x < 2y$
즉, $2x^2 + x < 2y < 2x^2 + x + 1$ 이 되어 이를 만족하는 자연수 $y$는 존재하지 않는다.

따라서 $(x - 3)(x + 1) \le 0$ 이어야 하므로 $-1 \le x \le 3$ 이다.
$x$는 자연수이므로 $x \in \{1, 2, 3\}$ 이다.
$x = 1$ 일 때, $1^4 + 1^3 + 1^2 + 1 + 1 = 5$ (제곱수 아님)
$x = 2$ 일 때, $2^4 + 2^3 + 2^2 + 2 + 1 = 16 + 8 + 4 + 2 + 1 = 31$ (제곱수 아님)
$x = 3$ 일 때, $3^4 + 3^3 + 3^2 + 3 + 1 = 81 + 27 + 9 + 3 + 1 = 121 = 11^2$ (제곱수)
따라서 만족하는 자연수 $x$의 개수는 $1$개이다.

그러므로 $(가) \ 2x^2 + x, (나) \ 1$이다.
