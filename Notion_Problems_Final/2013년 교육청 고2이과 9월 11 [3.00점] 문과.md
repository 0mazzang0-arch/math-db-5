## 문제
실수 전체의 집합에서 정의된 함수 $f(x)$가 다음 조건을 만족시킨다.

(가) $0 \le x \le 2$ 일 때, $f(x) = 2^x$
(나) 모든 실수 $x$ 에 대하여 $f(x) = f(-x)$
(다) 모든 실수 $x$ 에 대하여 $f(x) = f(x+4)$

$-10 \le x \le 10$ 에서 부등식 $\left( \frac{1}{3} \right)^{f(x)} - \frac{1}{27} \ge 0$ 을 만족시키는 모든 정수 $x$ 의 개수는?



> [!example] 📸 (그림/그래프)

## 보기
① $9$   ② $11$   ③ $13$   ④ $15$   ⑤ $17$

## 풀이
부등식 $\left( \frac{1}{3} \right)^{f(x)} - \frac{1}{27} \ge 0$ 에서
$\left( \frac{1}{3} \right)^{f(x)} \ge \left( \frac{1}{3} \right)^3$
밑이 $1$보다 작으므로
$f(x) \le 3 \cdots ㉠$

조건 (가), (나)에 의해 $-2 \le x \le 2$ 에서 $f(x) \le 3$ 의 해를 구하면
$f(x) = 2^{|x|} \le 3$
$|x| \le \log_{2} 3$
$-\log_{2} 3 \le x \le \log_{2} 3 \cdots ㉡$
$1 < \log_{2} 3 < 2$ 이므로 ㉡을 만족하는 정수 $x$ 는 $-1, 0, 1$ 이다.

조건 (다)에 의해 함수 $f(x)$ 는 주기가 $4$ 인 주기함수이므로, $-10 \le x \le 10$ 에서 ㉠의 해 중 정수는 다음과 같다.
$x = -1, 0, 1$ ($-2 \le x \le 2$ 범위)
$x = 3, 4, 5$ ($2 \le x \le 6$ 범위)
$x = 7, 8, 9$ ($6 \le x \le 10$ 범위)
$x = -5, -4, -3$ ($-6 \le x \le -2$ 범위)
$x = -9, -8, -7$ ($-10 \le x \le -6$ 범위)

따라서 정수 $x$ 의 개수는 $15$ 이다.
