## 문제

삼차방정식 $x^{3} = 1$의 한 허근을 $\omega$라 할 때, <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $\bar{\omega}$는 $\omega$의 켤레복소수이다.)

$$
\begin{array}{|l|}
\hline
\text{ㄱ. } (\bar{\omega})^{3} = 1 \\
\text{ㄴ. } \frac{1}{\omega} + \left( \frac{1}{\omega} \right)^{2} = \frac{1}{\bar{\omega}} + \left( \frac{1}{\bar{\omega}} \right)^{2} \\
\text{ㄷ. } (-\omega-1)^{n} = \left( \frac{\bar{\omega}}{\omega+\bar{\omega}} \right)^{n} \text{을 만족시키는 } 100 \text{ 이하의 자연수 } n \text{의 개수는 } 50 \text{이다.} \\
\hline
\end{array}
$$

## 보기

① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이

$x^{3} = 1$에서 $x^{3} - 1 = 0$, 즉 $(x - 1)(x^{2} + x + 1) = 0$이므로 $\omega$는 $x^{2} + x + 1 = 0$의 한 허근이고, 방정식의 계수가 실수이므로 $\omega$의 켤레복소수 $\bar{\omega}$도 $x^{2} + x + 1 = 0$의 허근이다.

따라서 $\omega^{2} + \omega + 1 = 0$, $(\bar{\omega})^{2} + \bar{\omega} + 1 = 0$, $\omega^{3} = 1$, $(\bar{\omega})^{3} = 1$이다.
근과 계수의 관계에 의하여 $\omega + \bar{\omega} = -1$, $\omega\bar{\omega} = 1$이다.

ㄱ. $(\bar{\omega})^{3} = 1$ (참)

ㄴ. $\frac{1}{\omega} + \left( \frac{1}{\omega} \right)^{2} = \frac{\omega + 1}{\omega^{2}} = \frac{-\omega^{2}}{\omega^{2}} = -1$이고,
$\frac{1}{\bar{\omega}} + \left( \frac{1}{\bar{\omega}} \right)^{2} = \frac{\bar{\omega} + 1}{(\bar{\omega})^{2}} = \frac{-(\bar{\omega})^{2}}{(\bar{\omega})^{2}} = -1$이므로
$\frac{1}{\omega} + \left( \frac{1}{\omega} \right)^{2} = \frac{1}{\bar{\omega}} + \left( \frac{1}{\bar{\omega}} \right)^{2}$이다. (참)

ㄷ. $(-\omega - 1)^{n} = (\omega^{2})^{n}$이고,
$\left( \frac{\bar{\omega}}{\omega + \bar{\omega}} \right)^{n} = (-\bar{\omega})^{n} = \left( -\frac{1}{\omega} \right)^{n} = (-1)^{n} \times \left( \frac{1}{\omega} \right)^{n} = (-1)^{n} \times (\omega^{2})^{n}$이다.
$(-\omega - 1)^{n} = \left( \frac{\bar{\omega}}{\omega + \bar{\omega}} \right)^{n}$에서
$(\omega^{2})^{n} = (-1)^{n} \times (\omega^{2})^{n}$
$\therefore 1 = (-1)^{n}$
$n$은 짝수이므로 $100$ 이하의 짝수 $n$의 개수는 $50$이다. (참)

이상에서 ㄱ, ㄴ, ㄷ 모두 옳다.

정답: ⑤
