## 문제
자연수 $n$ 에 대하여 $\log 2^n$ 의 정수부분을 $f(n)$, $\log 5^n$ 의 정수부분을 $g(n)$ 이라 할 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?

$$
\begin{array}{|l|}
\hline
\text{ㄱ. } f(10) = 3 \\
\text{ㄴ. } f(n) + g(n) = n \\
\text{ㄷ. } 2^n \text{ 의 자릿수와 } 5^n \text{ 의 자릿수의 합은 } 10^n \text{ 의 자릿수와 같다.} \\
\hline
\end{array}
$$

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄷ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
ㄱ. $2^{10} = 1024$ 로 $4$ 자리 수이므로 지표(정수부분)는 $3$ 이다. 
$\therefore f(10) = 3$ (참)

ㄴ. $\log 2^n = n \log 2 = f(n) + \alpha \ (0 < \alpha < 1)$
$\log 5^n = n \log 5 = g(n) + \beta \ (0 < \beta < 1)$ 에서
$\log 2^n + \log 5^n = f(n) + g(n) + \alpha + \beta$
$\log (2^n \cdot 5^n) = f(n) + g(n) + \alpha + \beta$
$\therefore n = f(n) + g(n) + \alpha + \beta$
$n$, $f(n)$, $g(n)$ 이 모두 정수이므로 $\alpha + \beta$ 도 정수이다. 
$0 < \alpha < 1$, $0 < \beta < 1$ 이므로 $0 < \alpha + \beta < 2$ 이다.
따라서 $\alpha + \beta = 1$ 이다. 
$\therefore f(n) + g(n) = n - 1$
따라서 ㄴ은 거짓이다.

ㄷ. ㄴ에서 $f(n) + g(n) = n - 1$ 이므로
$(f(n) + 1) + (g(n) + 1) = n - 1 + 2 = n + 1$ 이다.
$2^n$ 의 자릿수는 $f(n) + 1$ 이고, $5^n$ 의 자릿수는 $g(n) + 1$ 이다. 
또한 $10^n$ 의 자릿수는 $n + 1$ 이다.
따라서 $2^n$ 의 자릿수와 $5^n$ 의 자릿수의 합은 $10^n$ 의 자릿수와 같다. 
$\therefore$ 참

따라서 옳은 것은 ㄱ, ㄷ이다.
