## 문제
자연수 $k$에 대하여 $0 \le x < 2 \pi$일 때, $x$에 대한 방정식 $\sin k x = \frac{1}{3}$의 서로 다른 실근의 개수가 $8$이다. $0 \le x < 2 \pi$일 때, $x$에 대한 방정식 $\sin k x = \frac{1}{3}$의 모든 해의 합은?

## 보기
① $5 \pi$   ② $6 \pi$   ③ $7 \pi$   ④ $8 \pi$   ⑤ $9 \pi$

## 풀이
함수 $y = \sin k x$의 주기는 $\frac{2 \pi}{k}$이다.
$0 \le x < 2 \pi$일 때, 방정식 $\sin k x = \frac{1}{3}$의 서로 다른 실근의 개수는 $0 \le x < 2 \pi$에서 곡선 $y = \sin k x$와 직선 $y = \frac{1}{3}$이 만나는 점의 개수와 같다.

$1 \le l \le k$인 자연수 $l$에 대하여 $\frac{2(l - 1)}{k} \pi \le x < \frac{2 l}{k} \pi$에서 곡선 $y = \sin k x$와 직선 $y = \frac{1}{3}$이 만나는 점의 개수는 $2$이고, $0 \le x < 2 \pi$에서 곡선 $y = \sin k x$와 직선 $y = \frac{1}{3}$이 만나는 점의 개수가 $8$이므로 $2 k = 8$에서 $k = 4$이다.

$0 \le x < 2 \pi$일 때, 방정식 $\sin 4 x = \frac{1}{3}$의 서로 다른 실근을 작은 수부터 크기순으로 나열한 것을 $\alpha_{1}, \alpha_{2}, \alpha_{3}, \dots, \alpha_{8}$이라 하자. 함수 $y = \sin 4 x$의 주기는 $\frac{\pi}{2}$이므로 대칭성에 의해 다음과 같은 관계가 성립한다.

$\alpha_{2} = \frac{\pi}{4} - \alpha_{1}$
$\alpha_{3} = \frac{\pi}{2} + \alpha_{1}$
$\alpha_{4} = \frac{3}{4} \pi - \alpha_{1}$
$\alpha_{5} = \pi + \alpha_{1}$
$\alpha_{6} = \frac{5}{4} \pi - \alpha_{1}$
$\alpha_{7} = \frac{3}{2} \pi + \alpha_{1}$
$\alpha_{8} = \frac{7}{4} \pi - \alpha_{1}$

따라서 구하는 모든 해의 합은
$\alpha_{1} + \alpha_{2} + \alpha_{3} + \alpha_{4} + \alpha_{5} + \alpha_{6} + \alpha_{7} + \alpha_{8}$
$= \alpha_{1} + (\frac{\pi}{4} - \alpha_{1}) + (\frac{\pi}{2} + \alpha_{1}) + (\frac{3}{4} \pi - \alpha_{1}) + (\pi + \alpha_{1}) + (\frac{5}{4} \pi - \alpha_{1}) + (\frac{3}{2} \pi + \alpha_{1}) + (\frac{7}{4} \pi - \alpha_{1})$
$= \frac{\pi}{4} + \frac{2 \pi}{4} + \frac{3 \pi}{4} + \frac{4 \pi}{4} + \frac{5 \pi}{4} + \frac{6 \pi}{4} + \frac{7 \pi}{4}$
$= \frac{28}{4} \pi = 7 \pi$

정답은 ③이다.
