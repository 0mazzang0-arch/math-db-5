## 문제

> [!example] 📸 (그림/그래프)

그림과 같이 $1$보다 큰 두 실수 $a$, $k$에 대하여 직선 $y=k$가 두 곡선 $y=2\log_{a}x+k$, $y=a^{x-k}$와 만나는 점을 각각 $A$, $B$라 하고, 직선 $x=k$가 두 곡선 $y=2\log_{a}x+k$, $y=a^{x-k}$와 만나는 점을 각각 $C$, $D$라 하자. $\overline{AB} \times \overline{CD} = 85$이고 삼각형 $CAD$의 넓이가 $35$일 때, $a+k$의 값을 구하시오.

## 풀이

두 점 $A$와 $B$의 $y$좌표는 모두 $k$이므로 $A(1, k)$, $B(\log_{a}k+k, k)$이다.
두 점 $C$와 $D$의 $x$좌표는 모두 $k$이므로 $C(k, 2\log_{a}k+k)$, $D(k, 1)$이다.
두 선분 $\overline{AB}$와 $\overline{CD}$가 만나는 점을 $E$라 하면 $E(k, k)$이므로
$\overline{AE} = k-1$, $\overline{BE} = \log_{a}k$, $\overline{CE} = 2\log_{a}k$, $\overline{DE} = k-1$이다.
사각형 $ADBC$의 넓이는 $\frac{1}{2} \times \overline{AB} \times \overline{CD} = \frac{85}{2}$이고,
삼각형 $CAD$의 넓이는 $35$이므로 삼각형 $CBD$의 넓이는 $\frac{85}{2} - 35 = \frac{15}{2}$이다.

$\overline{AE} = p$, $\overline{BE} = q$라 하면 두 삼각형 $CAD$, $CBD$의 넓이의 비는
$p : q = 35 : \frac{15}{2} = 14 : 3$ 즉, $q = \frac{3}{14}p$이다.
이때 $\overline{CE} = 2q$, $\overline{DE} = p$이므로 삼각형 $CAD$의 넓이는
$\frac{1}{2} \times \overline{AE} \times \overline{CD} = \frac{1}{2} \times \overline{AE} \times (\overline{CE} + \overline{DE})$
$= \frac{1}{2} \times p \times (2q + p) = \frac{p}{2} \times (\frac{3}{7}p + p)$
$= \frac{5}{7}p^{2} = 35$

$p^{2} = 49$이고 $p > 0$이므로 $p = 7$, $q = \frac{3}{2}$이다.
$k-1 = p$, $\log_{a}k = q$이므로
$k = p+1 = 8$
$q = \log_{a}k = \log_{a}8 = \frac{3}{2}$, $a^{\frac{3}{2}} = 8$ 즉, $a = 4$이다.
따라서 $a+k = 12$이다.

정답: $12$
