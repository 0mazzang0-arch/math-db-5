## 문제
함수 $f(x) = \sqrt{2x+a} + 7$ 은 $x = -2$ 일 때 최솟값 $m$ 을 갖는다. $a+m$ 의 값을 구하시오. (단, $a$ 는 상수이다.)

## 보기

## 풀이
$f(x) = \sqrt{2x+a} + 7 = \sqrt{2 \left( x + \frac{a}{2} \right)} + 7$
이므로 함수 $y = f(x)$ 의 그래프는 다음과 같다.

> [!example] 📸 (그림/그래프)

함수 $f(x)$ 는 $x = -\frac{a}{2}$ 일 때 최솟값 $7$ 을 가진다.
$-\frac{a}{2} = -2$ 에서 $a = 4$ 이고 $m = 7$ 이므로 
$a+m = 11$ 이다.

정답: $11$
