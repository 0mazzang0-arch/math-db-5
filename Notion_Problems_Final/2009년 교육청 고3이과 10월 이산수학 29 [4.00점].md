## 문제
좌표평면 위를 움직이는 점 $P$ 의 시각 $t$ 에서의 위치벡터를 $\vec{p} = (x, y)$ 라 하면 
$x = \frac{e^t + e^{-t}}{2}$, $y = \frac{e^t - e^{-t}}{2}$ 
이 성립한다. 이 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?

$$ \begin{array}{|c|} \hline \text{< 보 기 >} \\ \text{ㄱ. } t = 1 \text{ 에서 점 } P \text{ 의 속도 } \vec{v} \text{ 와 위치벡터 } \vec{p} \text{ 는 서로 수직이다.} \\ \text{ㄴ. 임의의 시각 } t \text{ 에서 점 } P \text{ 의 가속도 } \vec{a} \text{ 와 위치벡터 } \vec{p} \text{ 는 서로 같다.} \\ \text{ㄷ. 점 } P \text{ 가 } t = 0 \text{ 에서 } t = 1 \text{ 까지 움직인 거리는 } 1 \text{ 이상이다.} \\ \hline \end{array} $$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
ㄱ. 
$\vec{v} = \left( \frac{dx}{dt}, \frac{dy}{dt} \right) = \left( \frac{e^t - e^{-t}}{2}, \frac{e^t + e^{-t}}{2} \right)$
$t = 1$ 일 때, $\vec{v} \cdot \vec{p}$ 를 계산하면
$\vec{v} \cdot \vec{p} = \left( \frac{e^1 - e^{-1}}{2} \right) \left( \frac{e^1 + e^{-1}}{2} \right) + \left( \frac{e^1 + e^{-1}}{2} \right) \left( \frac{e^1 - e^{-1}}{2} \right)$
$= \frac{e^2 - e^{-2}}{4} + \frac{e^2 - e^{-2}}{4} = \frac{e^2 - e^{-2}}{2} \neq 0$ (거짓)

ㄴ. 
$\vec{a} = \left( \frac{d^2 x}{dt^2}, \frac{d^2 y}{dt^2} \right) = \left( \frac{e^t + e^{-t}}{2}, \frac{e^t - e^{-t}}{2} \right) = \vec{p}$ (참)

ㄷ. 시각 $t = 0$ 에서 $t = 1$ 까지 점 $P$ 가 움직인 거리는
$\int_{0}^{1} \sqrt{\left( \frac{dx}{dt} \right)^2 + \left( \frac{dy}{dt} \right)^2} dt = \int_{0}^{1} \sqrt{\left( \frac{e^t - e^{-t}}{2} \right)^2 + \left( \frac{e^t + e^{-t}}{2} \right)^2} dt$
$= \int_{0}^{1} \sqrt{\frac{e^{2t} - 2 + e^{-2t}}{4} + \frac{e^{2t} + 2 + e^{-2t}}{4}} dt$
$= \int_{0}^{1} \sqrt{\frac{2e^{2t} + 2e^{-2t}}{4}} dt = \int_{0}^{1} \sqrt{\frac{e^{2t} + e^{-2t}}{2}} dt$
이때 산술-기하 평균에 의하여 $\frac{e^{2t} + e^{-2t}}{2} \ge \sqrt{e^{2t} \cdot e^{-2t}} = 1$ 이므로
$\int_{0}^{1} \sqrt{\frac{e^{2t} + e^{-2t}}{2}} dt \ge \int_{0}^{1} 1 dt = [t]_0^1 = 1$ (참)

따라서 옳은 것은 ㄴ, ㄷ이다.
