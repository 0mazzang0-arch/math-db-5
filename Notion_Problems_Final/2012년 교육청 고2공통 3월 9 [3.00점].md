## 문제
두 집합 $A = \{ x | (x - 1)(x - 26) > 0 \}$, $B = \{ x | (x - a)(x - a^{2}) \leq 0 \}$ 에 대하여 $A \cap B = \emptyset$ 이 되도록 하는 정수 $a$ 의 개수는?

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
집합 $A$ 에서 $x < 1$ 또는 $x > 26$ 이므로 $A \cap B = \emptyset$ 이 되기 위해서는 $1 \leq a \leq a^{2} \leq 26$ 이다.
$\therefore 1 \leq a \leq \sqrt{26}$
따라서 정수 $a$ 의 개수는 $5$ 이다.
정답: ⑤
