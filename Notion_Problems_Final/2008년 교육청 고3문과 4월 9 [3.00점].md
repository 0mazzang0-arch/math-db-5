## 문제
무한급수 $\frac{1}{1 \times 3} + \frac{2}{3 \times 7} + \frac{3}{7 \times 13} + \frac{4}{13 \times 21} + \dots$ 의 합은?

## 보기
① $\frac{1}{2}$   ② $\frac{3}{4}$   ③ $1$   ④ $\frac{5}{4}$   ⑤ $\frac{3}{2}$

## 풀이
수열의 일반항을 $a_n$, $n$ 항까지의 합을 $S_n$ 이라 하면 
$a_n = \frac{n}{(n^2 - n + 1)(n^2 + n + 1)}$ 
$S_n = \sum_{k=1}^{n} \frac{1}{2} \left( \frac{1}{k^2 - k + 1} - \frac{1}{k^2 + k + 1} \right)$ 
$= \frac{1}{2} \left\{ \left( 1 - \frac{1}{3} \right) + \left( \frac{1}{3} - \frac{1}{7} \right) + \left( \frac{1}{7} - \frac{1}{13} \right) + \dots + \left( \frac{1}{n^2 - n + 1} - \frac{1}{n^2 + n + 1} \right) \right\}$ 
$= \frac{1}{2} \left( 1 - \frac{1}{n^2 + n + 1} \right)$ 
따라서 $\lim_{n \to \infty} S_n = \frac{1}{2}$ 이다. 

정답: ①
