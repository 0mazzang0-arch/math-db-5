## 문제
삼차방정식 $x^{3} + x - 2 = 0$의 서로 다른 두 허근을 $\alpha$, $\beta$라 할 때, $\frac{\beta}{\alpha} + \frac{\alpha}{\beta}$의 값은?

## 보기
① $-\frac{7}{2}$   ② $-\frac{5}{2}$   ③ $-\frac{3}{2}$   ④ $-\frac{1}{2}$   ⑤ $\frac{1}{2}$

## 풀이
삼차방정식 $x^{3} + x - 2 = (x - 1)(x^{2} + x + 2) = 0$이므로 $\alpha$, $\beta$는 $x^{2} + x + 2 = 0$의 두 허근이다. 
근과 계수의 관계에서 $\alpha + \beta = -1$, $\alpha\beta = 2$이므로 
$\frac{\beta}{\alpha} + \frac{\alpha}{\beta} = \frac{\alpha^{2} + \beta^{2}}{\alpha\beta} = \frac{(\alpha + \beta)^{2} - 2\alpha\beta}{\alpha\beta} = \frac{1 - 4}{2} = -\frac{3}{2}$

[다른 풀이] 
$\alpha$, $\beta$는 $x^{2} + x + 2 = 0$의 두 허근이므로 $\alpha^{2} + \alpha + 2 = 0$에서 $\alpha^{2} = -\alpha - 2$이고, 같은 방법으로 $\beta^{2} = -\beta - 2$이다. 
따라서 $\frac{\beta}{\alpha} + \frac{\alpha}{\beta} = \frac{\alpha^{2} + \beta^{2}}{\alpha\beta} = \frac{-(\alpha + \beta) - 4}{\alpha\beta} = -\frac{3}{2}$이다. 

정답: ③
