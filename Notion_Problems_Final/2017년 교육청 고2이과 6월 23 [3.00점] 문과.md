## 문제
$\lim _{n \to \infty} (\sqrt{n^{2} + 8n + 10} - n)$ 의 값을 구하시오.

## 풀이
$\lim_{n \to \infty} (\sqrt{n^{2} + 8n + 10} - n)$ 
$= \lim_{n \to \infty} \frac{(\sqrt{n^{2} + 8n + 10} - n)(\sqrt{n^{2} + 8n + 10} + n)}{\sqrt{n^{2} + 8n + 10} + n}$ 
$= \lim_{n \to \infty} \frac{8n + 10}{\sqrt{n^{2} + 8n + 10} + n}$ 
$= \lim_{n \to \infty} \frac{8 + \frac{10}{n}}{\sqrt{1 + \frac{8}{n} + \frac{10}{n^{2}}} + 1}$ 
$= 4$

정답: $4$
