## 문제
함수 $f(x)=a \log _{2} x+2 b$ 가 다음 조건을 만족한다. 

(가) $f(2)=3$ 
(나) 모든 양의 실수 $x$, $y$ 에 대하여 $f(xy)=f(x)+f(y)-2$ 

이때, $f \left( \frac{1}{16} \right)$ 의 값은? (단, $a$, $b$ 는 상수이다.) 

## 보기
① $-5$   ② $-4$   ③ $-3$   ④ $-2$   ⑤ $-1$

## 풀이
조건 (가)에서 
$f(2) = a \log_{2} 2 + 2b = a + 2b = 3 \cdots \text{㉠}$

조건 (나)에서 모든 양의 실수 $x$, $y$ 에 대하여 $f(xy)=f(x)+f(y)-2$ 이므로, 
$x=1$, $y=1$ 을 대입하면 
$f(1) = f(1) + f(1) - 2$
$f(1) = 2$ 

함수식 $f(x)=a \log _{2} x+2 b$ 에 $x=1$ 을 대입하면 
$f(1) = a \log_{2} 1 + 2b = 2b$ 이므로 
$2b = 2$ 에서 $b = 1$ 이다. 

이를 ㉠에 대입하면 
$a + 2(1) = 3$ 이므로 $a = 1$ 이다. 

따라서 $f(x) = \log_{2} x + 2$ 이므로 
$f \left( \frac{1}{16} \right) = \log_{2} \left( \frac{1}{16} \right) + 2 = -4 + 2 = -2$ 이다. 

정답: ④
