## 문제
세 집합 $X = \{ x | x > 0 \}$, $Y = \{ y | y \text{는 정수} \}$, $Z = \{ z | 0 \le z < 1 \}$에 대하여 두 함수 $f : X \to Y$, $g : X \to Z$가 다음을 만족한다.

집합 $X$의 임의의 원소 $x$에 대하여 $10^{f(x)+g(x)} = x$이다.
$f(a) = 3$을 만족하는 양수 $a$에 대하여 $g(a) + g(\sqrt{a}) = 1$이 되는 $a$의 값은?

## 보기
① $10^{\frac{13}{4}}$   ② $10^{\frac{10}{3}}$   ③ $10^{\frac{7}{2}}$   ④ $10^{\frac{11}{3}}$   ⑤ $10^{\frac{15}{4}}$

## 풀이
$10^{f(x)+g(x)} = x$의 양변에 상용로그를 취하면 $f(x) + g(x) = \log x$이다.
즉, $\log x$의 정수부분이 $f(x)$, 소수부분이 $g(x)$이다.

$f(a) = 3$이므로 $\log a$의 정수부분은 $3$이다.
$\log a = 3 + g(a)$

$\log \sqrt{a} = \frac{1}{2} \log a = \frac{3}{2} + \frac{g(a)}{2} = 1 + \frac{1}{2} + \frac{g(a)}{2} = 1 + \frac{1 + g(a)}{2}$

이때 $0 \le g(a) < 1$이므로 $\frac{1}{2} \le \frac{1 + g(a)}{2} < 1$이다.
따라서 $\log \sqrt{a}$의 소수부분은 $g(\sqrt{a}) = \frac{1 + g(a)}{2}$이다.

$g(a) + g(\sqrt{a}) = g(a) + \frac{1 + g(a)}{2} = 1$이므로
$2g(a) + 1 + g(a) = 2$
$3g(a) = 1$
$g(a) = \frac{1}{3}$

따라서 $\log a = 3 + \frac{1}{3} = \frac{10}{3}$이므로
$a = 10^{\frac{10}{3}}$이다.

정답: ②
