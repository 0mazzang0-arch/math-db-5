## 문제
$1$ 이 아닌 양의 실수 전체의 집합에서 정의된 함수 $f(x)$ 를 $f(x)=2^{\frac{1}{\log_{2} x}}$ 이라 하자. 다음은 방정식 $8 \times f(f(x))=f(x^{2})$ 의 모든 해의 곱을 구하는 과정이다.

$$
\begin{array}{l}
x \neq 1 \text{ 인 모든 양의 실수 } x \text{ 에 대하여 } \\
f(f(x))=2^{\frac{1}{\log_{2} f(x)}} \text{ 에서 } \\
8 \times f(f(x))=2^{( (가) + \frac{1}{\log_{2} f(x)} )} \text{ 이고, } \\
f(x)=2^{\frac{1}{\log_{2} x}} \text{ 에서 } \log_{2} f(x) = \frac{1}{(나)} \text{ 이다. } \\
\text{방정식 } 8 \times f(f(x))=f(x^{2}) \text{ 에서 } \\
2^{( (가) + (나) )} = 2^{\frac{1}{2 \log_{2} x}} \\
(가) + (나) = \frac{1}{2 \log_{2} x} \\
\text{그러므로 방정식 } 8 \times f(f(x))=f(x^{2}) \text{ 의 모든 해는 } \\
\text{방정식 } ( (가) + (나) ) \times 2 \log_{2} x = 1 \text{ 의 모든 해와 같다. } \\
\text{따라서 방정식 } 8 \times f(f(x))=f(x^{2}) \text{ 의 모든 해의 곱은 } (다) \text{ 이다. }
\end{array}
$$

위의 $(가)$, $(다)$에 알맞은 수를 각각 $p$, $q$라 하고, $(나)$에 알맞은 식을 $g(x)$라 할 때, $p \times q \times g(4)$의 값은?

## 보기
① $\frac{1}{4}$   ② $\frac{3}{8}$   ③ $\frac{1}{2}$   ④ $\frac{5}{8}$   ⑤ $\frac{3}{4}$

## 풀이
$x \neq 1$ 인 모든 양의 실수 $x$ 에 대하여 $f(f(x))=2^{\frac{1}{\log_{2} f(x)}}$ 이므로
$8 \times f(f(x)) = 2^{3} \times 2^{\frac{1}{\log_{2} f(x)}} = 2^{( 3 + \frac{1}{\log_{2} f(x)} )}$ 이다.
따라서 $(가)$ 에 알맞은 수 $p = 3$ 이다.

또한, $f(x)=2^{\frac{1}{\log_{2} x}}$ 에서 양변에 밑이 $2$ 인 로그를 취하면
$\log_{2} f(x) = \log_{2} \left( 2^{\frac{1}{\log_{2} x}} \right) = \frac{1}{\log_{2} x}$ 이다.
따라서 $(나)$ 에 알맞은 식 $g(x) = \log_{2} x$ 이다.

이제 $\frac{1}{\log_{2} f(x)} = \log_{2} x$ 임을 이용하면 방정식 $8 \times f(f(x))=f(x^{2})$ 은 다음과 같다.
$f(x^{2}) = 2^{\frac{1}{\log_{2} x^{2}}} = 2^{\frac{1}{2 \log_{2} x}}$ 이므로
$2^{(3 + \log_{2} x)} = 2^{\frac{1}{2 \log_{2} x}}$ 에서
$3 + \log_{2} x = \frac{1}{2 \log_{2} x}$
$(3 + \log_{2} x) \times 2 \log_{2} x = 1$
$2(\log_{2} x)^{2} + 6 \log_{2} x - 1 = 0$

$\log_{2} x = t$ 라 하면 이차방정식 $2t^{2} + 6t - 1 = 0$ 의 판별식을 $D$ 라 할 때,
$\frac{D}{4} = 3^{2} - 2 \times (-1) = 11 > 0$ 이므로 서로 다른 두 실근 $\alpha, \beta$ 를 갖는다.
근과 계수의 관계에 의해 $\alpha + \beta = -\frac{6}{2} = -3$ 이다.
방정식 $2(\log_{2} x)^{2} + 6 \log_{2} x - 1 = 0$ 의 두 실근은 $x_{1} = 2^{\alpha}$, $x_{2} = 2^{\beta}$ 이므로
모든 해의 곱 $q = 2^{\alpha} \times 2^{\beta} = 2^{\alpha + \beta} = 2^{-3} = \frac{1}{8}$ 이다.
따라서 $(다)$ 에 알맞은 수 $q = \frac{1}{8}$ 이다.

최종적으로 구하는 값은
$p \times q \times g(4) = 3 \times \frac{1}{8} \times \log_{2} 4 = 3 \times \frac{1}{8} \times 2 = \frac{3}{4}$

정답은 ⑤ 이다.
