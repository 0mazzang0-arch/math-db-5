## 문제
로그부등식 $( \log_{ \frac{1}{9} } x ) ( \log_{3} \frac{x}{9} ) \geq a$ 의 해가 $\frac{1}{9} \leq x \leq 81$ 일 때, 상수 $a$ 의 값은?

## 보기
① $-4$   ② $-3$   ③ $-2$   ④ $-1$   ⑤ $0$

## 풀이
로그부등식 $( \log_{ \frac{1}{9} } x ) ( \log_{3} \frac{x}{9} ) \geq a$ 에서 $\log_{ \frac{1}{9} } x = -\frac{1}{2} \log_{3} x$ 이고 $\log_{3} \frac{x}{9} = \log_{3} x - 2$ 이므로 주어진 식은 다음과 같다.
$( -\frac{1}{2} \log_{3} x ) ( \log_{3} x - 2 ) \geq a$

이때 $\log_{3} x = t$ 라 하면 해가 $\frac{1}{9} \leq x \leq 81$ 이므로 $t$ 의 범위는 다음과 같다.
$\log_{3} \frac{1}{9} \leq \log_{3} x \leq \log_{3} 81$
$-2 \leq t \leq 4$

치환된 부등식 $( -\frac{t}{2} ) ( t - 2 ) \geq a$ 를 정리하면
$-t^{2} + 2t \geq 2a$
$t^{2} - 2t + 2a \leq 0$

이 부등식의 해가 $-2 \leq t \leq 4$ 이므로 이차방정식 $t^{2} - 2t + 2a = 0$ 의 두 근이 $-2$ 와 $4$ 가 되어야 한다.
이차방정식의 근과 계수의 관계에 의해 두 근의 곱은
$(-2) \times 4 = 2a$
$-8 = 2a$
따라서 $a = -4$ 이다.
