## 문제
두 실수 $a$ 와 $b$ 가 $1$ 이 아닌 양수일 때, 함수 $y=a^x$ 의 그래프와 함수 $y=\log_b x$ 의 그래프가 항상 만나는 경우를 <보기>에서 모두 고른 것은?

## 보기
$$ \begin{array}{|l|} \hline \text{ㄱ. } a > 1 \text{ 이고 } b > 1 \\ \text{ㄴ. } a > 1 \text{ 이고 } 0 < b < 1 \\ \text{ㄷ. } 0 < a < 1 \text{ 이고 } 0 < b < 1 \\ \hline \end{array} $$

① $ㄱ$   ② $ㄴ$   ③ $ㄷ$   ④ $ㄱ, \ ㄴ$   ⑤ $ㄴ, \ ㄷ$

## 풀이
[정답] ⑤

각각의 경우를 그래프로 그리면 다음과 같다.

> [!example] 📸 (그림/그래프)

그림에서 두 그래프가 항상 만나는 것은 $ㄴ$, $ㄷ$이다.
