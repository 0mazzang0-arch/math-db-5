## 문제
수열 $\{a_{n}\}$ 이 다음 조건을 만족시킨다.

- $a_{1}=7$
- $2$ 이상의 자연수 $n$ 에 대하여 $\sum_{k=1}^{n}a_{k}=\frac{2}{3}a_{n}+\frac{1}{6}n^{2}-\frac{1}{6}n+10$

다음은 $\sum_{k=1}^{12}a_{k}+ \sum_{k=1}^{5}a_{2k+1}$ 의 값을 구하는 과정이다.

$$ \begin{array}{|l|} \hline 
2 \text{ 이상의 자연수 } n \text{ 에 대하여 } a_{n+1} = \sum_{k=1}^{n+1} a_{k} - \sum_{k=1}^{n} a_{k} \text{ 이므로 } \\ 
a_{n+1} = \frac{2}{3} ( a_{n+1} - a_{n} ) + \text{ (가) } \\ 
\text{이고, 이 식을 정리하면} \\ 
2a_{n} + a_{n+1} = 3 \times \text{ (가) } \cdots \cdots ㉠ \\ 
\text{이다.} \\ 
\\
\sum_{k=1}^{n} a_{k} = \frac{2}{3} a_{n} + \frac{1}{6} n^{2} - \frac{1}{6} n + 10 \text{ } ( n \ge 2 ) \\ 
\text{에서 양변에 } n=2 \text{ 를 대입하면} \\ 
a_{2} = \text{ (나) } \cdots \cdots ㉡ \\ 
\text{이다. ㉠과 ㉡에 의하여} \\ 
\sum_{k=1}^{12} a_{k} + \sum_{k=1}^{5} a_{2k+1} = a_{1} + a_{2} + \sum_{k=1}^{5} ( 2a_{2k+1} + a_{2k+2} ) \\ 
= \text{ (다) } \\ 
\text{이다.} \\ \hline \end{array} $$

위의 (가)에 알맞은 식을 $f(n)$ 이라 하고, (나), (다)에 알맞은 수를 각각 $p$, $q$ 라 할 때, $\frac{p \times q}{f(12)}$ 의 값을 구하시오.

## 풀이
$2$ 이상의 자연수 $n$ 에 대하여 $\sum_{k=1}^{n} a_{k} = \frac{2}{3} a_{n} + \frac{1}{6} n^{2} - \frac{1}{6} n + 10$ 에서 $a_{n+1} = \sum_{k=1}^{n+1} a_{k} - \sum_{k=1}^{n} a_{k}$ 이므로
$a_{n+1} = \frac{2}{3} ( a_{n+1} - a_{n} ) + \{ \frac{1}{6} ( n+1 )^{2} - \frac{1}{6} ( n+1 ) + 10 \} - ( \frac{1}{6} n^{2} - \frac{1}{6} n + 10 )$
$= \frac{2}{3} ( a_{n+1} - a_{n} ) + \frac{1}{3} n$
이 식을 정리하면 $2a_{n} + a_{n+1} = 3 \times \frac{1}{3} n \cdots \cdots ㉠$
$\therefore 2a_{n} + a_{n+1} = n$ 이다.

$\sum_{k=1}^{n} a_{k} = \frac{2}{3} a_{n} + \frac{1}{6} n^{2} - \frac{1}{6} n + 10$ ($n \ge 2$) 에서 양변에 $n=2$ 를 대입하면 $a_{1} = 7$ 이므로
$a_{1} + a_{2} = \frac{2}{3} a_{2} + \frac{1}{6} \times 4 - \frac{1}{6} \times 2 + 10$
$7 + \frac{1}{3} a_{2} = \frac{1}{3} + 10$
$a_{2} = 10 \cdots \cdots ㉡$ 이다.

㉠과 ㉡에 의하여
$\sum_{k=1}^{12} a_{k} + \sum_{k=1}^{5} a_{2k+1} = a_{1} + a_{2} + \sum_{k=1}^{5} ( 2a_{2k+1} + a_{2k+2} )$
$= 7 + 10 + \sum_{k=1}^{5} ( 2k+1 )$
$= 17 + 2 \times \frac{5 \times 6}{2} + 5 = 52$ 이다.

따라서 (가) $f(n) = \frac{1}{3} n$, (나) $p = 10$, (다) $q = 52$ 이므로
$\frac{p \times q}{f(12)} = \frac{10 \times 52}{\frac{1}{3} \times 12} = \frac{520}{4} = 130$
