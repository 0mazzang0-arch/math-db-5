## 문제
실수 $a$에 대하여 함수 $f(x)$를 
$$f(x) = \lim_{n \to \infty} \frac{(a-2)x^{2n+1} + 2x}{3x^{2n} + 1}$$
라 하자. $(f \circ f)(1) = \frac{5}{4}$가 되도록 하는 모든 $a$의 값의 합은?

## 보기
① $\frac{11}{2}$   ② $\frac{13}{2}$   ③ $\frac{15}{2}$   ④ $\frac{17}{2}$   ⑤ $\frac{19}{2}$

## 풀이
$x$의 범위에 따라서 극한값을 나누어보면 다음과 같다.

(ⅰ) $-1 < x < 1$ 일 때 
$\lim_{n \to \infty} x^{2n} = \lim_{n \to \infty} x^{2n+1} = 0$ 이므로 
$$f(x) = \lim_{n \to \infty} \frac{(a-2)x^{2n+1} + 2x}{3x^{2n} + 1} = 2x$$

(ⅱ) $x = 1$ 일 때 
$$f(1) = \frac{a}{4}$$

(ⅲ) $x = -1$ 일 때 
$$f(-1) = -\frac{a}{4}$$

(ⅳ) $x < -1, x > 1$ 일 때 
$$f(x) = \lim_{n \to \infty} \frac{(a-2)x + \frac{2}{x^{2n-1}}}{3 + \frac{1}{x^{2n}}} = \frac{a-2}{3}x$$

$(f \circ f)(1) = \frac{5}{4}$ 이고, $f(1) = \frac{a}{4}$ 이므로 
$$f\left(\frac{a}{4}\right) = \frac{5}{4}$$
$f\left(\frac{a}{4}\right) = \frac{5}{4}$ 가 될 수 있는 값을 위 범위에 따라서 구해본다.

(ⅰ) $-1 < \frac{a}{4} < 1$ 일 때 
$f(x) = 2x$ 이므로 
$$f\left(\frac{a}{4}\right) = \frac{a}{2} = \frac{5}{4}$$
되어 $a = \frac{5}{2}$이다. 이는 조건을 만족한다.

(ⅱ) $\frac{a}{4} = 1$ 일 때 
$f(x) = \frac{a}{4}$ 이므로 
$$f\left(\frac{a}{4}\right) = \frac{a}{4} = \frac{5}{4}$$
되어 $a = 5$ 이지만, 조건 $\frac{a}{4} = 1$ ($a=4$)에 모순되어 성립할 수 없다.

(ⅲ) $\frac{a}{4} = -1$ 일 때 
$f(x) = -\frac{a}{4}$ 이므로 
$$f\left(\frac{a}{4}\right) = -\frac{a}{4} = \frac{5}{4}$$
되어 $a = -5$ 이지만, 조건 $\frac{a}{4} = -1$ ($a=-4$)에 모순되어 성립할 수 없다.

(ⅳ) $\frac{a}{4} < -1, \frac{a}{4} > 1$ 일 때 
$f(x) = \frac{a-2}{3}x$ 이므로 $f\left(\frac{a}{4}\right) = \frac{5}{4}$ 를 만족시키는 $a$ 의 값을 구하면 
$$\frac{a-2}{3} \cdot \frac{a}{4} = \frac{5}{4}$$
$$a^2 - 2a - 15 = 0$$
$$(a-5)(a+3) = 0$$
$a = 5$ 또는 $a = -3$ 중에서 조건을 만족시키는 값은 $a = 5$ 이다.

따라서 모든 $a$ 의 값의 합은 $\frac{5}{2} + 5 = \frac{15}{2}$ 이다.
