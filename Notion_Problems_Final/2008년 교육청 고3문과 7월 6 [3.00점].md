## 문제
$\log_{2} x = 5.2$ 일 때, $\log \frac{1}{x}$ 의 소수부분은? (단, $\log 2 = 0.30$)

## 보기
① $0.32$   ② $0.36$   ③ $0.40$   ④ $0.44$   ⑤ $0.48$

## 풀이
$\log_{2} x = 5.2$ 이므로 밑의 변환 공식에 의해
$\frac{\log x}{\log 2} = 5.2$
$\log x = 5.2 \times \log 2 = 5.2 \times 0.30 = 1.56$
$\log \frac{1}{x} = -\log x = -1.56$
$-1.56 = -2 + 0.44$
이때 소수부분은 $0 \le (\text{소수부분}) < 1$ 을 만족해야 하므로
$\log \frac{1}{x}$ 의 소수부분은 $0.44$ 이다.

정답: ④
