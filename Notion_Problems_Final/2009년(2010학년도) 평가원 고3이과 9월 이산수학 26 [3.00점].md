## 문제
$\sqrt{3} \sin \theta + \cos \theta = \frac{1}{2}$ 일 때, $\cos \left( \theta + \frac{\pi}{6} \right)$ 의 값은? (단, $0 < \theta < \pi$)

## 보기
① $-\frac{\sqrt{5}}{8}$   ② $-\frac{\sqrt{15}}{8}$   ③ $-\frac{1}{2}$   ④ $-\frac{\sqrt{5}}{4}$   ⑤ $-\frac{\sqrt{15}}{4}$

## 풀이
$\sqrt{3} \sin \theta + \cos \theta = 2 \left( \frac{\sqrt{3}}{2} \sin \theta + \frac{1}{2} \cos \theta \right) = 2 \sin \left( \theta + \frac{\pi}{6} \right) = \frac{1}{2}$ 에서 
$\sin \left( \theta + \frac{\pi}{6} \right) = \frac{1}{4}$ 이다.

$\sin^2 \left( \theta + \frac{\pi}{6} \right) + \cos^2 \left( \theta + \frac{\pi}{6} \right) = 1$ 이므로
$\cos^2 \left( \theta + \frac{\pi}{6} \right) = 1 - \left( \frac{1}{4} \right)^2 = \frac{15}{16}$
$\therefore \cos \left( \theta + \frac{\pi}{6} \right) = \pm \frac{\sqrt{15}}{4}$

한편, $0 < \theta < \pi$ 에서 $\frac{\pi}{6} < \theta + \frac{\pi}{6} < \frac{7}{6} \pi$ 이고 
$\sin \left( \theta + \frac{\pi}{6} \right) = \frac{1}{4} > 0$ 이므로 $\theta + \frac{\pi}{6}$ 는 제 $1$ 사분면 또는 제 $2$ 사분면의 각이다. 
그런데 $\frac{\pi}{6} < \theta + \frac{\pi}{6} < \frac{7}{6} \pi$ 범위 내에서 $\sin$ 값이 $\frac{1}{4}$ 이 되려면 $\theta + \frac{\pi}{6}$ 는 $\frac{\pi}{6}$ 보다 크고 $\pi$ 보다 작은 범위에 있어야 하며, 이 조건에서 $\cos \left( \theta + \frac{\pi}{6} \right)$ 의 값은 음수여야 한다.

$\therefore \cos \left( \theta + \frac{\pi}{6} \right) = -\frac{\sqrt{15}}{4}$
