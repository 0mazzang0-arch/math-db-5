## 문제
다항함수 $f(x)$의 도함수 $f'(x)$가 $f'(x) = 4x^{3} + ax$이고 $f(0) = -2$, $f(1) = 1$일 때, $f(2)$의 값은? (단, $a$는 상수이다.)

## 보기
① $18$   ② $19$   ③ $20$   ④ $21$   ⑤ $22$

## 풀이
$f'(x) = 4x^{3} + ax$에서
$f(x) = x^{4} + \frac{a}{2} x^{2} + C$

$f(0) = -2$이므로
$f(0) = C = -2$ $\cdots$ ㉠

$f(1) = 1$에서
$f(1) = 1 + \frac{a}{2} + C = 1$ $\cdots$ ㉡

㉠, ㉡을 연립하면
$C = -2$, $a = 4$

따라서 $f(x) = x^{4} + 2x^{2} - 2$이므로
$f(2) = 16 + 2 \times 2^{2} - 2 = 22$
