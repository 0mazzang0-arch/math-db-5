## 문제
수열 $\{ a_{n} \}$에 대하여
$$\sum_{k=1}^{15} ( 3a_{k} + 2 ) = 45$$,
$$2\sum_{k=1}^{15} a_{k} = 42 + \sum_{k=1}^{14} a_{k}$$
일 때, $a_{15}$의 값을 구하시오.

## 보기

## 풀이
$$\sum_{k=1}^{15} ( 3a_{k} + 2 ) = 3\sum_{k=1}^{15} a_{k} + \sum_{k=1}^{15} 2 = 3\sum_{k=1}^{15} a_{k} + 30 = 45$$
$$3\sum_{k=1}^{15} a_{k} = 15$$, $$\sum_{k=1}^{15} a_{k} = 5$$

$$2\sum_{k=1}^{15} a_{k} = 42 + \sum_{k=1}^{14} a_{k}$$
위의 식에 $\sum_{k=1}^{15} a_{k} = 5$를 대입하면
$$10 = 42 + \sum_{k=1}^{14} a_{k}$$, $$\sum_{k=1}^{14} a_{k} = -32$$

따라서
$$a_{15} = \sum_{k=1}^{15} a_{k} - \sum_{k=1}^{14} a_{k} = 5 - ( -32 ) = 37$$

정답: $37$
