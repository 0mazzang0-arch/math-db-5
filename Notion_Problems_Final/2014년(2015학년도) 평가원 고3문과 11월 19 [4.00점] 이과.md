## 문제
두 이차정사각행렬 $A$, $B$ 가 
$$A^{2}-A B=3 E, \ A^{2} B-B^{2} A=A+B$$ 
를 만족시킬 때, <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $E$ 는 단위행렬이다.)

## 보기
ㄱ. $A$ 의 역행렬이 존재한다. 
ㄴ. $A B=B A$ 
ㄷ. $(A+2 B)^{2}=24 E$ 

① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
ㄱ. $A^2 - AB = 3E$ 에서 
$A(A - B) = 3E$ 이므로 
$A^{-1} = \frac{1}{3}(A - B)$ (참) 

ㄴ. $A \cdot \frac{1}{3}(A - B) = \frac{1}{3}(A - B) \cdot A = E$ 이므로 
$A(A - B) = (A - B)A$ 
$A^2 - AB = A^2 - BA$ 
$\therefore AB = BA$ (참) 

ㄷ. $AB = BA$ 이므로 
$A^2 B - B^2 A = A^2 B - AB^2$ 
$A^2 - AB = 3E$ 의 양변에 $B$ 를 곱하면 
$A^2 B - AB^2 = 3B$ 
$A^2 B - B^2 A = A + B$ 에서 
$3B = A + B$, $A = 2B$ 이므로 
$A^2 - AB = 3E$ 에 대입하면 
$(2B)^2 - 2B \cdot B = 3E$ 
$4B^2 - 2B^2 = 3E$, $2B^2 = 3E$ 
$\therefore B^2 = \frac{3}{2}E$ 
$\therefore (A + 2B)^2 = (2B + 2B)^2 = (4B)^2 = 16B^2$ 
$= 16 \times \frac{3}{2}E = 24E$ (참) 

따라서 옳은 것은 ㄱ, ㄴ, ㄷ 이다.
