## 문제
$80^{x} = 2$, $\left( \frac{1}{10} \right)^{y} = 4$, $a^{z} = 8$을 만족시키는 세 실수 $x$, $y$, $z$에 대하여 $\frac{1}{x} + \frac{2}{y} - \frac{1}{z} = 1$이 성립할 때, 양수 $a$의 값은?

## 보기
① $32$   ② $64$   ③ $96$   ④ $128$   ⑤ $160$

## 풀이
[정답] ② 

$80^{x} = 2$ 에서 $2^{\frac{1}{x}} = 80 \cdots$ ㉠ 
$\left( \frac{1}{10} \right)^{y} = 4 = 2^{2}$ 에서 $2^{\frac{2}{y}} = \frac{1}{10} \cdots$ ㉡ 
$a^{z} = 8 = 2^{3}$ 에서 $2^{\frac{3}{z}} = a$ 이므로 $2^{\frac{1}{z}} = a^{\frac{1}{3}} \cdots$ ㉢ 

㉠ $\times$ ㉡ $\div$ ㉢ 하면 
$2^{\frac{1}{x} + \frac{2}{y} - \frac{1}{z}} = \frac{80 \times \frac{1}{10}}{\sqrt[3]{a}}$ 
$\frac{80 \times \frac{1}{10}}{\sqrt[3]{a}} = 2^{1} = 2$, 
$\sqrt[3]{a} = 4$ 

$\therefore a = 4^{3} = 64$ 

[다른 풀이] 
$\frac{1}{x} = \log_{2} 80$, 
$\frac{1}{y} = \log_{4} \frac{1}{10}$, 
$\frac{1}{z} = \log_{8} a$ 

$\frac{1}{x} + \frac{2}{y} - \frac{1}{z} = \log_{2} 80 + 2 \log_{4} \frac{1}{10} - \log_{8} a$
$= \log_{2} 80 + \log_{2} \frac{1}{10} - \log_{2} \sqrt[3]{a}$
$= \log_{2} \frac{80 \times \frac{1}{10}}{\sqrt[3]{a}} = 1$ 

$2^{1} = \frac{80 \times \frac{1}{10}}{\sqrt[3]{a}}$, 
$\sqrt[3]{a} = 4$ 

$\therefore a = 4^{3} = 2^{6} = 64$
