## 문제
그림과 같이 함수 $f(x) = 3x^2 - 7x + 2$ 에 대하여 곡선 $y = f(x)$ 와 직선 $y = \frac{1}{3}x - \frac{2}{3}$ 및 $y$ 축으로 둘러싸인 영역을 $A$, 곡선 $y = f(x)$ 와 직선 $y = \frac{1}{3}x - \frac{2}{3}$ 로 둘러싸인 영역을 $B$, 곡선 $y = f(x)$ 와 두 직선 $y = \frac{1}{3}x - \frac{2}{3}$, $x = k$ ($k > 2$) 로 둘러싸인 영역을 $C$ 라 하자. 
($A$ 의 넓이) $+$ ($C$ 의 넓이) $=$ ($B$ 의 넓이) 일 때, 상수 $k$ 의 값은?

> [!example] 📸 (그림/그래프)

## 보기
① $\frac{29}{12}$   ② $\frac{5}{2}$   ③ $\frac{31}{12}$   ④ $\frac{8}{3}$   ⑤ $\frac{11}{4}$

## 풀이
직선 $y = \frac{1}{3}x - \frac{2}{3}$ 에서 $g(x) = \frac{1}{3}x - \frac{2}{3}$ 라 하고, 함수 $y = f(x)$ 의 그래프와 직선 $y = g(x)$ 의 두 교점의 $x$ 좌표를 각각 $\alpha$, $\beta$ ($\alpha < \beta$)라 하자. 

그림에서 세 영역 $A$, $B$, $C$ 의 넓이를 각각 $S_1$, $S_2$, $S_3$ 이라 하면 
$S_1 = \int_{0}^{\alpha} \{ f(x) - g(x) \} dx$ 
$S_2 = \int_{\alpha}^{\beta} \{ g(x) - f(x) \} dx$ 
$S_3 = \int_{\beta}^{k} \{ f(x) - g(x) \} dx$ 

($A$ 의 넓이) $+$ ($C$ 의 넓이) $=$ ($B$ 의 넓이) 이므로 
$S_1 + S_3 = S_2$, $S_1 - S_2 + S_3 = 0$ 

이때 
$S_1 - S_2 + S_3$ 
$= \int_{0}^{\alpha} \{ f(x) - g(x) \} dx - \int_{\alpha}^{\beta} \{ g(x) - f(x) \} dx + \int_{\beta}^{k} \{ f(x) - g(x) \} dx$ 
$= \int_{0}^{\alpha} \{ f(x) - g(x) \} dx + \int_{\alpha}^{\beta} \{ f(x) - g(x) \} dx + \int_{\beta}^{k} \{ f(x) - g(x) \} dx$ 
$= \int_{0}^{k} \{ f(x) - g(x) \} dx$ 

이므로 
$\int_{0}^{k} \{ f(x) - g(x) \} dx = 0$ $\cdots \cdots$ ㉠ 

$f(x) - g(x) = 3x^2 - \frac{22}{3}x + \frac{8}{3}$ 이고, 이를 ㉠에 대입하면 
$\int_{0}^{k} ( 3x^2 - \frac{22}{3}x + \frac{8}{3} ) dx = 0$ 
$\left[ x^3 - \frac{11}{3}x^2 + \frac{8}{3}x \right]_{0}^{k} = 0$ 
$k^3 - \frac{11}{3}k^2 + \frac{8}{3}k = 0$, $k( 3k^2 - 11k + 8 ) = 0$ 
$k( k - 1 )( 3k - 8 ) = 0$ 

$k > 2$ 이므로 $k = \frac{8}{3}$
