## 문제
최고차항의 계수가 양수인 삼차함수 $f(x)$에 대하여 방정식 $(f \circ f)(x) = x$의 모든 실근이 $0$, $1$, $a$, $2$, $b$이다.  
$f'(1) < 0$, $f'(2) < 0$, $f'(0) - f'(1) = 6$일 때, $f(5)$의 값을 구하시오. (단, $1 < a < 2 < b$)

## 보기
① $Value1$   ② $Value2$   ③ $Value3$   ④ $Value4$   ⑤ $Value5$  
*(참고: 제공된 입력 텍스트에 선택지가 포함되어 있지 않아 임의의 형식을 제시합니다.)*

## 풀이
$(f \circ f)(x) = x$를 만족하기 위해서는  
ⅰ) 모든 $x$에 대하여 $f(x) = x$이거나  
ⅱ) $f(p) = q, f(q) = p$가 되어야 한다.  

ⅰ)의 경우 $f(1) = 1, f(2) = 2$가 되면 $f(x)$는 증가함수가 되어 $f'(1) < 0$, $f'(2) < 0$인 조건에 만족하지 않는다.  
ⅱ)의 경우 $f(1) = 2, f(2) = 1, f(0) = 0$와 $f'(0) - f'(1) = 6$을 만족해야 한다.  

$f(x) = ax^{3} + bx^{2} + cx + d$라 하면 $f'(x) = 3ax^{2} + 2bx + c$이고, 주어진 조건에 의해 다음과 같은 식을 얻을 수 있다.

$$ \begin{array}{ll} f(1) = a + b + c + d = 2 & \cdots ① \\ f(2) = 8a + 4b + 2c + d = 1 & \cdots ② \\ f(0) = d = 0 & \cdots ③ \\ f'(0) - f'(1) = c - (3a + 2b + c) = 6 & \cdots ④ \end{array} $$

①, ②, ③, ④를 연립하면 $a = 1, b = -\frac{9}{2}, c = \frac{11}{2}$이므로  
$f(x) = x^3 - \frac{9}{2}x^2 + \frac{11}{2}x$을 얻는다.  

따라서 $f(5) = 40$이다.
