## 문제
집합 $A_{n}$ 을 다음과 같이 정의한다. 
$$A_{n} = \{ m \mid \left[ \frac{m}{n} \right] = 0, m, n \text{ 은 서로소인 자연수} \}$$
다음 중 옳지 않은 것은?
(단, $[x]$ 는 $x$ 를 넘지 않는 최대 정수이다.)

## 보기
① $A_{1} = \emptyset$   ② $A_{2} \cap A_{3} = A_{2}$   ③ $A_{2} \cup A_{3} = A_{3}$   ④ $A_{3} \cup A_{4} = A_{4}$   ⑤ $A_{3} \cap A_{4} = A_{2}$

## 풀이
[정답] ④

조건에 의해 $\left[ \frac{m}{n} \right] = 0$ 이면 $0 \le \frac{m}{n} < 1$ 이다.
$m, n$ 은 자연수이므로 $1 \le m < n$ 이며, $m$ 과 $n$ 은 서로소여야 한다.
즉, $A_{n} = \{ m \mid 1 \le m < n, \gcd(m, n) = 1 \}$ 이다.

각 집합을 구하면 다음과 같다.
- $n = 1$ 일 때, $1 \le m < 1$ 을 만족하는 자연수 $m$ 은 존재하지 않는다. $\therefore A_{1} = \emptyset$
- $n = 2$ 일 때, $1 \le m < 2$ 이고 $\gcd(m, 2) = 1$ 인 자연수 $m = 1$ 이다. $\therefore A_{2} = \{ 1 \}$
- $n = 3$ 일 때, $1 \le m < 3$ 이고 $\gcd(m, 3) = 1$ 인 자연수 $m = 1, 2$ 이다. $\therefore A_{3} = \{ 1, 2 \}$
- $n = 4$ 일 때, $1 \le m < 4$ 이고 $\gcd(m, 4) = 1$ 인 자연수 $m = 1, 3$ 이다. $\therefore A_{4} = \{ 1, 3 \}$

보기를 검토하면 다음과 같다.
① $A_{1} = \emptyset$ (참)
② $A_{2} \cap A_{3} = \{ 1 \} \cap \{ 1, 2 \} = \{ 1 \} = A_{2}$ (참)
③ $A_{2} \cup A_{3} = \{ 1 \} \cup \{ 1, 2 \} = \{ 1, 2 \} = A_{3}$ (참)
④ $A_{3} \cup A_{4} = \{ 1, 2 \} \cup \{ 1, 3 \} = \{ 1, 2, 3 \} \neq A_{4}$ (거짓)
⑤ $A_{3} \cap A_{4} = \{ 1, 2 \} \cap \{ 1, 3 \} = \{ 1 \} = A_{2}$ (참)

따라서 옳지 않은 것은 ④ 이다.
