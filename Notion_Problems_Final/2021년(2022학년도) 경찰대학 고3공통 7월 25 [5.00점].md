## 문제
두 집합 $X$, $Y$를
$X = \{ \{ a_{n} \} \mid \{ a_{n} \} \text{은 모든 항이 자연수인 수열이고, } \log a_{n} + \log a_{n+1} = 2n \}$, 
$Y = \{ a_{4} \mid \{ a_{n} \} \in X \}$
라 하자. 집합 $Y$의 모든 원소의 합이 $p \times 100$일 때, $p$의 값을 구하시오.

## 보기

## 풀이
$\log a_{n} + \log a_{n+1} = 2n$에서 $a_{n} a_{n+1} = 10^{2n}$
$a_{1} a_{2} = 10^{2}$, $a_{2} a_{3} = 10^{4}$, $a_{3} a_{4} = 10^{6}$에서
$a_{2} = 10^{2} \times \frac{1}{a_{1}}$, 
$a_{3} = 10^{4} \times \frac{1}{a_{2}} = 10^{2} \times a_{1}$, 
$a_{4} = 10^{6} \times \frac{1}{a_{3}} = 10^{4} \times \frac{1}{a_{1}}$
조건에서 $\{ a_{n} \}$는 모든 항이 자연수이므로 $a_{2}$가 자연수가 되려면 $a_{1}$은 $100 = 2^{2} \times 5^{2}$의 약수이므로 가능한 $a_{1}$은 총 $9$가지다. 

(ⅰ) $a_{1} = 1$일 때 $a_{4} = 10^{4} \times 1 = 10000$
(ⅱ) $a_{1} = 2$일 때 $a_{4} = 10^{4} \times \frac{1}{2} = 5000$
(ⅲ) $a_{1} = 2^{2}$일 때 $a_{4} = 10^{4} \times \frac{1}{2^{2}} = 2500$
(ⅳ) $a_{1} = 5$일 때 $a_{4} = 10^{4} \times \frac{1}{5} = 2000$
(ⅴ) $a_{1} = 2 \times 5$일 때 $a_{4} = 10^{4} \times \frac{1}{10} = 1000$
(ⅵ) $a_{1} = 2^{2} \times 5$일 때 $a_{4} = 10^{4} \times \frac{1}{20} = 500$
(ⅶ) $a_{1} = 5^{2}$일 때 $a_{4} = 10^{4} \times \frac{1}{25} = 400$
(ⅷ) $a_{1} = 2 \times 5^{2}$일 때 $a_{4} = 10^{4} \times \frac{1}{50} = 200$
(ⅸ) $a_{1} = 2^{2} \times 5^{2}$일 때 $a_{4} = 10^{4} \times \frac{1}{100} = 100$

(ⅰ)∼(ⅸ)에서 만족하는 값을 모두 더하면 $21700$
즉, $p \times 100 = 217 \times 100$에서 $p = 217$

정답: $217$
