## 문제
함수 $f(x) = x^{3} + x^{2} + ax + b$ ($a$, $b$는 상수)에 대하여 함수 $g(x)$를
$$g(x) = f(x) + (x - 1)f'(x)$$
라 하자. <보기>에서 옳은 것만을 있는 대로 고른 것은?

<보기>
ㄱ. 함수 $h(x) = (x - 1)f(x)$ 이면 $h'(x) = g(x)$ 이다.
ㄴ. 함수 $f(x)$ 가 $x = -1$ 에서 극값 $0$ 을 가지면 $\int_{0}^{1} g(x) dx = -1$ 이다.
ㄷ. $f(0) = 0$ 이면 방정식 $g(x) = 0$ 은 열린구간 $(0, 1)$ 에서 적어도 하나의 실근을 갖는다.

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄱ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
ㄱ. $h(x) = (x - 1)f(x)$ 에서 곱의 미분법을 이용하면
$h'(x) = (x - 1)'f(x) + (x - 1)f'(x) = f(x) + (x - 1)f'(x)$
$h'(x) = g(x)$ (참)

ㄴ. 함수 $f(x)$ 가 $x = -1$ 에서 극값 $0$ 을 가지므로
$f'(-1) = 0$, $f(-1) = 0$
$f'(x) = 3x^{2} + 2x + a$ 이므로
$f'(-1) = 3 - 2 + a = 0$ 에서 $a = -1$
$f(-1) = (-1)^{3} + (-1)^{2} - (-1) + b = -1 + 1 + 1 + b = 0$ 에서 $b = -1$
$f(x) = x^{3} + x^{2} - x - 1$ 이므로 $f(0) = -1$ 이다.
$\int_{0}^{1} g(x) dx = \int_{0}^{1} h'(x) dx = [h(x)]_{0}^{1} = [ (x - 1)f(x) ]_{0}^{1}$
$= (1 - 1)f(1) - (0 - 1)f(0) = 0 - ( -f(0) ) = f(0) = -1$ (참)

ㄷ. $f(0) = 0$ 이면
$h(0) = (0 - 1)f(0) = -1 \times 0 = 0$
$h(1) = (1 - 1)f(1) = 0 \times f(1) = 0$
함수 $h(x)$ 가 닫힌 구간 $[0, 1]$ 에서 연속이고 열린 구간 $(0, 1)$ 에서 미분가능하므로, 평균값의 정리(또는 롤의 정리)에 의해
$\frac{h(1) - h(0)}{1 - 0} = h'(c)$ 를 만족하는 $c$ 가 열린 구간 $(0, 1)$ 에서 적어도 하나 존재한다.
$h'(c) = \frac{0 - 0}{1} = 0$
ㄱ에서 $h'(x) = g(x)$ 이므로 $g(c) = 0$ 이다.
따라서 방정식 $g(x) = 0$ 은 열린 구간 $(0, 1)$ 에서 적어도 하나의 실근을 갖는다. (참)

이상에서 옳은 것은 ㄱ, ㄴ, ㄷ이다.
