## 문제
첫째항이 $7$인 등비수열 $\{ a_n \}$의 첫째항부터 제 $n$항까지의 합을 $S_n$이라 하자. 
$$\frac{ S_9 - S_5 }{ S_6 - S_2 } = 3$$
일 때, $a_7$의 값을 구하시오.

## 보기


## 풀이
첫째항이 $7$인 등비수열 $\{ a_n \}$의 공비를 $r$이라 하자. 
$S_9 - S_5 = a_6 + a_7 + a_8 + a_9$
$= 7r^5 + 7r^6 + 7r^7 + 7r^8$
$= 7r^5 (1 + r + r^2 + r^3) \cdots$ ㉠ 

$S_6 - S_2 = a_3 + a_4 + a_5 + a_6$
$= 7r^2 + 7r^3 + 7r^4 + 7r^5$
$= 7r^2 (1 + r + r^2 + r^3) \cdots$ ㉡ 

주어진 조건 $\frac{ S_9 - S_5 }{ S_6 - S_2 } = 3$에 ㉠, ㉡을 대입하면 
$$\frac{ 7r^5 (1 + r + r^2 + r^3) }{ 7r^2 (1 + r + r^2 + r^3) } = 3$$
$r^3 = 3 \cdots$ ㉢ 

$\therefore a_7 = 7r^6$
$= 7 \times (r^3)^2$
$= 7 \times 3^2$ ($\because$ ㉢) 
$= 7 \times 9 = 63$

정답: $63$
