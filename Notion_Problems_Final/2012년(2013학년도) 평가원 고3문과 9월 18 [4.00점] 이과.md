## 문제
이차함수 $f(x)$에 대하여 함수 $g(x)$가
$$ g(x) = \int \{ x^2 + f(x) \} dx, \quad f(x)g(x) = -2x^4 + 8x^3 $$
을 만족시킬 때, $g(1)$의 값은?

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
[정답] ②

$f(x) = ax^2 + bx + c$ ($a \neq 0$)라 하면
$$ g(x) = \int \{ (1+a)x^2 + bx + c \} dx = \frac{1+a}{3}x^3 + \frac{b}{2}x^2 + cx + C $$

(단, $C$는 적분상수)

이때 $f(x)$는 이차함수이고 $f(x)g(x)$가 사차함수이므로 $g(x)$는 이차함수이다.
$\therefore a = -1$

$f(x) = -x^2 + bx + c$, $g(x) = \frac{b}{2}x^2 + cx + C$
$$ f(x)g(x) = (-x^2 + bx + c) \left( \frac{b}{2}x^2 + cx + C \right) $$
$$ = - \frac{b}{2}x^4 + \left( \frac{b^2}{2} - c \right)x^3 + \left( \frac{3}{2}bc - C \right)x^2 + (bC + c^2)x + cC $$
주어진 조건 $f(x)g(x) = -2x^4 + 8x^3$에서 계수를 비교하면
$- \frac{b}{2} = -2 \Rightarrow b = 4$
$\frac{b^2}{2} - c = 8 \Rightarrow 8 - c = 8 \Rightarrow c = 0$
$\frac{3}{2}bc - C = 0 \Rightarrow 0 - C = 0 \Rightarrow C = 0$

따라서 $g(x) = 2x^2$이므로
$g(1) = 2$
