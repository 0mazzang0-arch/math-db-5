## 문제
이차함수 $f(x)$가 다음 조건을 만족시킨다.

$$ \begin{array}{|l|} \hline \text{(가) } f(1) = 0 \\ \text{(나) 모든 실수 } x \text{에 대하여 } f(x) \geq f(3) \text{이다.} \\ \hline \end{array} $$

<보기>에서 옳은 것만을 있는 대로 고른 것은?

$$ \begin{array}{|l|} \hline \text{<보 기>} \\ \text{ㄱ. } f(5) = 0 \\ \text{ㄴ. } f(2) < f(\frac{1}{2}) < f(6) \\ \text{ㄷ. } f(0) = k \text{라 할 때, } x \text{에 대한 방정식 } f(x) = kx \text{의 두 실근의 합은 } 11 \text{이다.} \\ \hline \end{array} $$



> [!example] 📸 (그림/그래프)

## 보기
① $ㄱ$   ② $ㄷ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
조건 (나)에서 모든 실수 $x$에 대하여 $f(x) \geq f(3)$이므로 $y = f(x)$의 대칭축은 직선 $x = 3$이고, 아래로 볼록한 그래프이다.

ㄱ. $y = f(x)$의 대칭축이 직선 $x = 3$이고, 조건 (가)에서 $f(1) = 0$이므로 대칭성에 의해
$f(5) = f(1) = 0$이다. (참)

ㄴ. $y = f(x)$의 그래프는 아래로 볼록하고 대칭축이 $x = 3$이다. $x$의 값과 대칭축 $x = 3$ 사이의 거리를 비교하면
$|3 - 2| = 1$
$|3 - \frac{1}{2}| = 2.5$
$|3 - 6| = 3$
이다. 대칭축에서 멀어질수록 함숫값이 커지므로
$f(2) < f(\frac{1}{2}) < f(6)$
이다. (참)

ㄷ. 방정식 $f(x) = 0$의 두 실근이 $1$, $5$이므로
$f(x) = a(x - 1)(x - 5) \quad (a > 0)$
라 할 수 있다.
$f(0) = k$이므로
$k = a(0 - 1)(0 - 5) = 5a$
방정식 $f(x) = kx$에서
$a(x - 1)(x - 5) = 5ax$
$a(x^{2} - 6ax + 5a) = 5ax$
$x^{2} - 6x + 5 = 5x$ ($\because a > 0$)
$x^{2} - 11x + 5 = 0$
이차방정식의 근과 계수의 관계에 의하여 두 실근의 합은 $11$이다. (참)

이상에서 ㄱ, ㄴ, ㄷ 모두 옳다.
