## 문제
수열 $\{ a_{n} \}$ 이 $a_{1} = 2$ 이고, 모든 자연수 $n$ 에 대하여 
$a_{n+1} = 2 ( a_{n} + 2 )$
를 만족시킨다. $a_{5}$ 의 값을 구하시오.

## 풀이
$a_{n+1} = 2 ( a_{n} + 2 )$ 의 $n$ 에 $1$, $2$, $3$, $4$ 를 차례로 대입하면 
$a_{2} = 2 ( a_{1} + 2 ) = 2 \times ( 2 + 2 ) = 8$ 
$a_{3} = 2 ( a_{2} + 2 ) = 2 \times ( 8 + 2 ) = 20$ 
$a_{4} = 2 ( a_{3} + 2 ) = 2 \times ( 20 + 2 ) = 44$ 
$a_{5} = 2 ( a_{4} + 2 ) = 2 \times ( 44 + 2 ) = 92$ 
따라서 $a_{5} = 92$ 이다.

정답: $92$
