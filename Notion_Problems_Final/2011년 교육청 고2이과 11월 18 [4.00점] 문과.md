## 문제
$0 < a < b < 1$인 두 실수 $a$, $b$에 대하여 옳은 것만을 <보기>에서 있는 대로 고른 것은?

$$
\begin{array}{|l|}
\hline
\text{<보기>} \\
\text{ㄱ. } \log_{a} b < 1 \\
\text{ㄴ. } \log_{(b+1)} (a+1) > 1 \\
\text{ㄷ. 서로 다른 두 양수 } c, d\text{에 대하여 } \log_{a} c = \log_{b} d\text{이면 } c < d\text{이다.} \\
\hline
\end{array}
$$

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
[정답] ①

ㄱ. $a < b$ 이고 $a < 1$ 이므로 $\log_{a} b < \log_{a} a \iff \log_{a} b < 1$ (참)
ㄴ. (반례) $a = \frac{1}{4}, b = \frac{9}{16}$ (거짓)
ㄷ. (반례) $a = \frac{1}{4}, b = \frac{1}{2}, c = 4, d = 2$ (거짓)

따라서 옳은 것은 ㄱ이다.
