## 문제
곡선 $y = \sin^{2} x$ ($0 \le x \le \pi$) 의 두 변곡점을 각각 $A$, $B$ 라 할 때, 점 $A$ 에서의 접선과 점 $B$ 에서의 접선이 만나는 점의 $y$ 좌표는 $p + q \pi$ 이다. $40(p + q)$ 의 값을 구하시오. (단, $p$, $q$ 는 유리수이다.)

## 보기

## 풀이
$y = \sin^{2} x$ 에서
$y' = 2 \sin x \cos x = \sin 2x$
$y'' = 2 \cos 2x = 0$ 이므로 $x = \frac{\pi}{4}$ 또는 $x = \frac{3\pi}{4}$
$x = \frac{\pi}{4}$ 일 때, 점 $A$ 의 좌표는 $\left( \frac{\pi}{4}, \frac{1}{2} \right)$ 이고, $y'|_{x = \frac{\pi}{4}} = 1$ 이다.
따라서 접선 $l_{1}$ 의 방정식은
$l_{1} : y = 1 \left( x - \frac{\pi}{4} \right) + \frac{1}{2} = x - \frac{\pi}{4} + \frac{1}{2} \cdots ①$
$x = \frac{3\pi}{4}$ 일 때, 점 $B$ 의 좌표는 $\left( \frac{3\pi}{4}, \frac{1}{2} \right)$ 이고, $y'|_{x = \frac{3\pi}{4}} = -1$ 이다.
따라서 접선 $l_{2}$ 의 방정식은
$l_{2} : y = -1 \left( x - \frac{3\pi}{4} \right) + \frac{1}{2} = -x + \frac{3\pi}{4} + \frac{1}{2} \cdots ②$
$①, ②$ 를 연립하면
$2y = \frac{\pi}{2} + 1$
$y = \frac{1}{2} + \frac{1}{4} \pi$
점의 $y$ 좌표가 $p + q \pi$ 이므로 $p = \frac{1}{2}$, $q = \frac{1}{4}$ 이다.
$\therefore 40(p + q) = 40 \left( \frac{1}{2} + \frac{1}{4} \right) = 40 \times \frac{3}{4} = 30$
