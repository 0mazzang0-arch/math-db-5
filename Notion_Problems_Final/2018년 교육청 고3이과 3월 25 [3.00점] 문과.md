## 문제
함수 $f(x) = \sin^{2} x + \sin \left( x + \frac{\pi}{2} \right) + 1$의 최댓값을 $M$이라 할 때, $4M$의 값을 구하시오.

## 풀이
$\sin^{2} x = 1 - \cos^{2} x$, $\sin \left( x + \frac{\pi}{2} \right) = \cos x$이므로
$$ f(x) = \sin^{2} x + \sin \left( x + \frac{\pi}{2} \right) + 1 $$
$$ = 1 - \cos^{2} x + \cos x + 1 $$
$$ = - \left( \cos x - \frac{1}{2} \right)^{2} + \frac{9}{4} $$
이때 $-1 \leq \cos x \leq 1$이므로 함수 $f(x)$는 $\cos x = \frac{1}{2}$일 때 최댓값 $\frac{9}{4}$를 갖는다.
따라서 $M = \frac{9}{4}$이므로
$$ 4M = 9 $$
