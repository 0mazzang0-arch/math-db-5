## 문제
곡선 $y = \sin^{2} x \cos x$ ($0 \leq x \leq \frac{\pi}{2}$)와 $x$축으로 둘러싸인 도형의 넓이는?



> [!example] 📸 (그림/그래프)

## 보기
① $\frac{1}{4}$   ② $\frac{1}{3}$   ③ $\frac{1}{2}$   ④ $1$   ⑤ $2$

## 풀이
$0 \leq x \leq \frac{\pi}{2}$에서 $\sin^{2} x \cos x = 0$의 해를 구하면 $x = 0$ 또는 $x = \frac{\pi}{2}$이다.
$0 \leq x \leq \frac{\pi}{2}$일 때 $\sin^{2} x \cos x \geq 0$이므로 구하는 넓이는
$\int_{0}^{\frac{\pi}{2}} |\sin^{2} x \cos x| dx = \int_{0}^{\frac{\pi}{2}} \sin^{2} x \cos x dx$이다.
$\sin x = t$로 놓고 양변을 $x$에 대하여 미분하면 $\cos x = \frac{dt}{dx}$이다.
$x = 0$이면 $t = 0$이고, $x = \frac{\pi}{2}$이면 $t = 1$이다.
따라서 $\int_{0}^{\frac{\pi}{2}} \sin^{2} x \cos x dx = \int_{0}^{1} t^{2} dt = \left[ \frac{1}{3} t^{3} \right]_{0}^{1} = \frac{1}{3}$이다.

정답: ②
