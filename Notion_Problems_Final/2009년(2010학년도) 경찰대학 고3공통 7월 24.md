## 문제
함수 $f(x) = \log_{2} (x^{2} + x + 1) - \log_{2} x$ 에 대하여 $[f(1)] + [f(2)] + [f(3)] + \cdots + [f(1022)]$ 의 값은? (단, $[x]$ 는 $x$ 보다 크지 않은 최대의 정수이다.)

## 보기
① $2^{11} + 2$   ② $2^{12} - 2$   ③ $2^{12} + 2$   ④ $2^{13} - 2$   ⑤ $2^{13} + 2$

## 풀이
$f(x) = \log_{2} (x^{2} + x + 1) - \log_{2} x$
$= \log_{2} \frac{x^{2} + x + 1}{x}$
$= \log_{2} (x + 1 + \frac{1}{x})$
에서 $x \ge 1$ 일 때 $x + 1 + \frac{1}{x} \ge 3$ 이므로
$f(x) = \log_{2} (x + 1 + \frac{1}{x}) > 1$
따라서 $[f(x)] = [\log_{2} (x + 1 + \frac{1}{x})] = k$ ($k$ 는 자연수) 라 하면
$k \le \log_{2} (x + 1 + \frac{1}{x}) < k + 1$
$2^{k} \le x + 1 + \frac{1}{x} < 2^{k+1}$
$\therefore 2^{k} - 1 \le x + \frac{1}{x} < 2^{k+1} - 1$

$k = 1$ 일 때, $1 \le x + \frac{1}{x} < 3$ 이므로 자연수 $x$ 는 $1$, $2$ 의 $2$ 개
$k = 2$ 일 때, $3 \le x + \frac{1}{x} < 7$ 이므로 자연수 $x$ 는 $3$, $4$, $5$, $6$ 의 $4$ 개
$k = 3$ 일 때, $7 \le x + \frac{1}{x} < 15$ 이므로 자연수 $x$ 는 $7$, $8$, $\cdots$, $14$ 의 $8$ 개
$\vdots$
$k = 9$ 일 때, $511 \le x + \frac{1}{x} < 1023$ 이므로 자연수 $x$ 는 $511$, $512$, $\cdots$, $1022$ 의 $512$ 개

이때, $k = [f(x)]$ 이므로
$[f(1)] + [f(2)] + \cdots + [f(1022)]$
$= 1 \times 2 + 2 \times 2^{2} + 3 \times 2^{3} + \cdots + 9 \times 2^{9}$

$S = 1 \times 2 + 2 \times 2^{2} + 3 \times 2^{3} + \cdots + 9 \times 2^{9} \cdots$ ㉠
이라 하면
$2 S = 1 \times 2^{2} + 2 \times 2^{3} + 3 \times 2^{4} + \cdots + 8 \times 2^{9} + 9 \times 2^{10} \cdots$ ㉡
이므로 ㉠ $-$ ㉡을 하면
$-S = 2 + 2^{2} + 2^{3} + \cdots + 2^{9} - 9 \times 2^{10} = \frac{2 (2^{9} - 1)}{2 - 1} - 9 \times 2^{10}$
$\therefore S = -\frac{2 (2^{9} - 1)}{2 - 1} + 9 \times 2^{10}$
$= 9 \times 2^{10} - 2^{10} + 2 = 8 \times 2^{10} + 2 = 2^{13} + 2$

정답: ⑤
