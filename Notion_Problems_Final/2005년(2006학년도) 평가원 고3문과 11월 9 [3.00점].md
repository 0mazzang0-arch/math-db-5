## 문제
부등식 $a^m < a^n < b^n < b^m$ 을 만족시키는 양수 $a$, $b$ 와 자연수 $m$, $n$ 에 대하여 옳은 것은?

## 보기
① $a < 1 < b$, $m > n$   ② $a < 1 < b$, $m < n$   ③ $a < b < 1$, $m < n$   ④ $1 < a < b$, $m > n$   ⑤ $1 < a < b$, $m < n$

## 풀이
각 변에 상용로그를 취하면 
$m \log a < n \log a < n \log b < m \log b$ 이므로 

ⅰ) $n$ 이 자연수이고 $n \log a < n \log b$ 이므로 
$\therefore a < b$ 

ⅱ) $(m - n) \log a < 0$, $(m - n) \log b > 0$ 이므로 
$\log a$ 와 $\log b$ 는 부호가 반대이다. 따라서 
$\log a < 0$, $\log b > 0$ 이다. 

$\therefore a < 1 < b$, $n < m$

따라서 옳은 것은 $a < 1 < b$, $m > n$ 이다.
