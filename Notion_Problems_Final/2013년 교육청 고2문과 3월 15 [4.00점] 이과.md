## 문제
표는 양의 실수 $x$, $y$에 대하여 두 종류의 변환장치를 설명한 것이다.

> [!example] 📸 (그림/그래프)

예를 들면 

> [!example] 📸 (그림/그래프)

를 변환하였을 때 $\frac{4 x y}{2 x + y}$이다.

> [!example] 📸 (그림/그래프)

을 변환한 것을 $\frac{(가)}{x+y}$라 할 때, $(가)$에 알맞은 식은?

## 보기
① $x y$   ② $2 x y$   ③ $2 x y + 1$   ④ $2 x y + 2$   ⑤ $4 x y + 2$

## 풀이
변환장치를 이용하여 계산하면 
$$\frac{2 x y}{x + y} + \frac{2 \times \frac{1}{x y}}{\frac{1}{x} + \frac{1}{y}}$$ 
$$= \frac{2 x y}{x + y} + \frac{\frac{2}{x y}}{\frac{x + y}{x y}}$$ 
$$= \frac{2 x y}{x + y} + \frac{2}{x + y}$$ 
$$= \frac{2 x y + 2}{x + y}$$ 
이므로 $(가)$에 알맞은 식은 $2 x y + 2$이다.
