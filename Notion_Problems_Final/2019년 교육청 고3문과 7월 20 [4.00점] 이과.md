## 문제
최고차항의 계수가 양수인 사차함수 $f(x) = ax^4 + bx^2 + c$ ($a$, $b$, $c$는 상수)가 다음 조건을 만족시킨다.

(가) 방정식 $f(x) = 0$의 모든 실근이 $\alpha$, $\beta$, $\gamma$이다. (단, $\alpha < \beta < \gamma$)
(나) $f(1) = -\frac{3}{4}$, $f'(-1) = 1$

보기에서 옳은 것만을 있는 대로 고른 것은?

## 보기
ㄱ. $f(0) = 0$
ㄴ. $f'(\alpha) = -4$
ㄷ. 방정식 $|f(x)| = k(x-\alpha)$의 서로 다른 실근의 개수가 $3$이 되도록 하는 양수 $k$의 범위는 $\frac{8}{27} < k < 4$이다.

① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
$f(x) = f(-x)$이고 방정식 $f(x) = 0$의 서로 다른 실근의 개수가 $3$이므로 $f(0) = c = 0$이다.

$f(1) = a + b = -\frac{3}{4}$, $f'(-1) = -4a - 2b = 1$
위의 두 식을 연립하면 $a = \frac{1}{4}$, $b = -1$

ㄱ. $f(0) = 0$ (참)

ㄴ. $f(x) = \frac{1}{4}x^4 - x^2 = \frac{1}{4}x^2(x+2)(x-2)$
따라서 $\alpha = -2$, $\beta = 0$, $\gamma = 2$
$f'(x) = x^3 - 2x$
$f'(\alpha) = f'(-2) = -4$ (참)

ㄷ. 
> [!example] 📸 (그림/그래프)

$y = -f(x)$ 위의 점 $(-2, 0)$에서의 접선 $l_1$의 기울기는
$-f'(-2) = -(-4) = 4$

$(-2, 0)$에서 곡선 $y = -f(x)$에 그은 접선 중 하나를 $l_2$라 할 때, 접점을 $(t, -\frac{1}{4}t^4 + t^2)$ ($t \neq -2, 0$)이라 하자.
$l_2$의 기울기는 $-f'(t) = -(t^3 - 2t) = -t^3 + 2t$

$l_2$의 방정식은 $y = (-t^3 + 2t)(x-t) - \frac{1}{4}t^4 + t^2$
$l_2$가 점 $(-2, 0)$을 지나므로
$0 = (-t^3 + 2t)(-2-t) - \frac{1}{4}t^4 + t^2$
$t(3t-4)(t+2)^2 = 0$에서 $t = \frac{4}{3}$

$-f'(\frac{4}{3}) = -(\frac{4}{3})^3 + 2 \times \frac{4}{3} = \frac{8}{27}$

$y = |f(x)|$의 그래프와 직선 $y = k(x+2)$의 교점의 개수는
$0 < k < \frac{8}{27}$일 때, $5$개
$k = \frac{8}{27}$일 때, $4$개
$\frac{8}{27} < k < 4$일 때, $3$개
$k \geq 4$일 때, $2$개

따라서 $|f(x)| = k(x+2)$의 서로 다른 실근의 개수가 $3$이 되도록 하는 양수 $k$의 범위는 $\frac{8}{27} < k < 4$이다. (참)

이상에서 ㄱ, ㄴ, ㄷ 모두 옳다.
