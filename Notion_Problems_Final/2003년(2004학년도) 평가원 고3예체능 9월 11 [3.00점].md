## 문제
이차함수 $f(x) = x^2 + 2ax + a + 2$ 의 꼭짓점이 제 $2$ 사분면에 있을 때, $a$ 값의 범위는?

## 보기
① $-2 < a < 3$   ② $-2 < a < 2$   ③ $-1 < a < 0$   ④ $-1 < a < 1$   ⑤ $0 < a < 2$

## 풀이
이차함수 $f(x) = x^2 + 2ax + a + 2$ 의 꼭짓점이 제 $2$ 사분면에 있기 위해서는 
$f(x) = x^2 + 2ax + a^2 - a^2 + a + 2$ 
$f(x) = (x + a)^2 - a^2 + a + 2$ 
에서 $f(x)$ 의 꼭짓점의 $x$ 좌표가 음수이고, $y$ 좌표가 양수이어야 한다. 즉, 

(ⅰ) $-a < 0$ 
$\therefore a > 0$ 

(ⅱ) $-a^2 + a + 2 > 0$ 
$a^2 - a - 2 < 0$ 
$(a + 1)(a - 2) < 0$ 
$\therefore -1 < a < 2$ 

따라서 (ⅰ), (ⅱ)에 의하여 $0 < a < 2$ 이다.

정답: ⑤
