## 문제
양수 $A$에 대하여 $\log_{10} A = n + \alpha$ ($n$은 정수, $\frac{1}{2} < \alpha < 1$)일 때, $[\log_{10} \frac{1}{A^{2}}]$의 값은? (단, $[x]$는 $x$를 넘지 않는 최대정수)

## 보기
① $-2 n-2$   ② $-2 n-1$   ③ $-2 n$   ④ $-2 n+1$   ⑤ $-2 n+2$

## 풀이
$\log_{10} A = n + \alpha$ ($n$은 정수, $\frac{1}{2} < \alpha < 1$)라 하면
$\log_{10} \frac{1}{A^{2}} = -2 \log_{10} A = -2(n + \alpha) = -2n - 2\alpha$
이때 조건에서 $\frac{1}{2} < \alpha < 1$이므로 각 변에 $-2$를 곱하면
$-2 < -2 \alpha < -1$
상용로그의 지표(정수 부분)와 가수(소수 부분)의 성질을 이용하기 위해 식을 다음과 같이 변형한다.
$\log_{10} \frac{1}{A^{2}} = -2n - 2\alpha = (-2n - 2) + (2 - 2\alpha)$
위에서 구한 $-2 < -2\alpha < -1$의 각 변에 $2$를 더하면
$0 < 2 - 2 \alpha < 1$
따라서 $\log_{10} \frac{1}{A^{2}}$의 정수 부분은 $-2n - 2$이다.
$[\log_{10} \frac{1}{A^{2}}] = -2 n - 2$

정답: ①
