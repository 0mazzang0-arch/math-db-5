## 문제
연립부등식 
$$ \begin{cases} 1 < [ \log_{4} x ]^{2} + [ \log_{4} y ]^{2} < 4 \\ x + y < 20 \end{cases} $$
을 만족하는 순서쌍 $(x, y)$ 중에서 $x$, $y$ 가 모두 정수인 순서쌍의 개수는? (단, $[x]$ 는 $x$ 보다 크지 않은 최대의 정수이다.)

## 보기
① $66$   ② $78$   ③ $84$   ④ $87$   ⑤ $91$

## 풀이
$\log _{4} x = X$, $\log _{4} y = Y$ 라 하면 
$1 < [X]^{2} + [Y]^{2} < 4$ 
즉, $[X]^{2} + [Y]^{2} = 2$, $3$ 이다. 

(ⅰ) $[X]^{2} + [Y]^{2} = 2$ 인 경우 : 
$x$, $y$ 가 정수이므로 가능한 경우는 $[X] = 1$, $[Y] = 1$ 인 경우뿐이다. 따라서, 
$1 \le \log _{4} x < 2$, $1 \le \log _{4} y < 2$ 
$4 \le x < 16$, $4 \le y < 16$ 
이때 $x + y < 20$ 을 만족해야 하므로 $(x, y)$ 의 개수는 
$\frac{1}{2} \cdot 12 \cdot 13 = 78$ (가지) 

(ⅱ) $[X]^{2} + [Y]^{2} = 3$ 인 경우: 
이를 만족하는 정수 $[X]$, $[Y]$ 는 존재하지 않는다. 

따라서 구하는 순서쌍의 개수는 $78$ 이다.

정답: ②
