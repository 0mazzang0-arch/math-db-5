## 문제
등차수열 $\{ a_{n} \}$에서 $a_{1} = 6$, $a_{10} = -12$일 때, $|a_{1}| + |a_{2}| + |a_{3}| + \dots + |a_{20}|$의 값은?

## 보기
① $280$   ② $284$   ③ $288$   ④ $292$   ⑤ $296$

## 풀이
[정답] ② 

등차수열이므로 
$$a_{10} = a_{1} + 9d = 6 + 9d = -12 \quad \therefore d = -2$$ 
$$\therefore a_{n} = 6 + (n - 1)(-2) = -2n + 8$$ 
$$\therefore a_{n} \ge 0 \text{인 } n \text{은 } -2n + 8 \ge 0 \quad \therefore n \le 4 \text{이므로}$$ 
$$|a_{1}| + |a_{2}| + \dots + |a_{20}|$$ 
$$= (a_{1} + a_{2} + a_{3} + a_{4}) - (a_{5} + a_{6} + \dots + a_{20})$$ 
$$= -S_{20} + 2(a_{1} + a_{2} + a_{3} + a_{4})$$ 
$$= -\frac{20 \{ 2 \times 6 + 19 \times (-2) \}}{2} + 2 \times \frac{4 \{ 2 \times 6 + 3 \times (-2) \}}{2}$$ 
$$= 284$$
