## 문제
공차가 $0$이 아닌 등차수열 $\{a_n\}$에 대하여 수열 $\{b_n\}$은 $b_n = a_{3n-2} + 2a_{3n-1} + a_{3n}$이다. $A_n = \sum_{k=1}^{n} a_k$, $B_n = \sum_{k=1}^{n} b_k$일 때, $\lim_{n \to \infty} \frac{B_n}{A_n}$의 값을 구하시오.

## 보기

## 풀이
$a_n = an+b$ ($a \neq 0$)라 하면,
$b_n = a_{3n-2} + 2a_{3n-1} + a_{3n}$
$= 4a_{3n-1} = 12an - 4a + 4b$

$A_n = \frac{an(n+1)}{2} + bn$
$B_n = \frac{12an(n+1)}{2} - (4a - 4b)n$

따라서 
$\lim_{n \to \infty} \frac{B_n}{A_n} = \lim_{n \to \infty} \frac{6an^2 + \dots}{\frac{1}{2}an^2 + \dots} = \frac{6a}{\frac{1}{2}a} = 12$
