## 문제
함수 $f(x) = \frac{x}{x^{2} + x + 8}$ 에 대하여 부등식 $f'(x) > 0$ 의 해가 $\alpha < x < \beta$ 일 때, $\alpha^{2} + \beta^{2}$ 의 값을 구하시오.

## 보기

## 풀이
$f'(x) = \frac{(x^{2} + x + 8) - x(2x + 1)}{(x^{2} + x + 8)^{2}} = \frac{-x^{2} + 8}{(x^{2} + x + 8)^{2}} > 0$
$(x^{2} + x + 8)^{2} > 0$ 이므로 
$-x^{2} + 8 > 0$
$x^{2} < 8$
$-2\sqrt{2} < x < 2\sqrt{2}$
$\therefore \alpha = -2\sqrt{2}, \beta = 2\sqrt{2}$
따라서 $\alpha^{2} + \beta^{2} = (-2\sqrt{2})^{2} + (2\sqrt{2})^{2} = 8 + 8 = 16$

정답: $16$
