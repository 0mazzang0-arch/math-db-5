## 문제
두 수 $n$, $\alpha$가 다음 두 조건을 만족시킨다.

(가) $n$은 자연수이고, $0 < \alpha < \frac{1}{2}$이다.
(나) $\log_{4} n = 1 - \alpha$

등식 $\log_{2} m^{2} = n + \alpha$를 만족시키는 실수 $m$에 대하여 $3 m^{4}$의 값을 구하시오.

## 보기


## 풀이
조건 (가)에서 $0 < \alpha < \frac{1}{2}$이므로
$\frac{1}{2} < 1 - \alpha < 1$이다.
조건 (나)에서 $1 - \alpha = \log_{4} n$이므로
$\frac{1}{2} < \log_{4} n < 1$
$4^{\frac{1}{2}} < n < 4^{1}$
$2 < n < 4$
$n$은 자연수이므로 $n = 3$이다.

그러면 $\log_{4} 3 = 1 - \alpha$에서
$\alpha = 1 - \log_{4} 3 = \log_{4} 4 - \log_{4} 3 = \log_{4} \frac{4}{3}$
등식 $\log_{2} m^{2} = n + \alpha$에 $n = 3$과 $\alpha = \log_{4} \frac{4}{3}$를 대입하면
$\log_{2} m^{2} = 3 + \log_{4} \frac{4}{3}$
$\log_{4} m^{4} = \log_{4} 4^{3} + \log_{4} \frac{4}{3}$
$\log_{4} m^{4} = \log_{4} \left( 4^{3} \times \frac{4}{3} \right)$
$\log_{4} m^{4} = \log_{4} \frac{4^{4}}{3}$
$m^{4} = \frac{4^{4}}{3} = \frac{256}{3}$

따라서
$3 m^{4} = 3 \times \frac{256}{3} = 256$

정답: $256$
