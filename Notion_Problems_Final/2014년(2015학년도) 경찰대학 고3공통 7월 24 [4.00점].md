## 문제
두 개의 주사위를 던져 나오는 눈의 수 중 크거나 같은 수를 확률변수 $X$라 할 때, $E(6X) = \frac{p}{q}$이다. $p+q$의 값을 구하시오. (단, $p$, $q$는 서로소인 자연수이다.)

## 보기

## 풀이
[정답] $167$

두 주사위에서 나오는 눈의 수를 각각 $a$, $b$라 하자.
ⅰ) $X = 1$인 경우는 $(a, b) = (1, 1)$로 $1$가지이다.
ⅱ) $X = k$ ($k = 2, 3, \dots, 6$)인 경우의 수는 $a, b$ 모두 $k$ 이하인 순서쌍 $(a, b)$의 개수에서 $a, b$ 모두 $k-1$ 이하인 순서쌍 $(a, b)$의 개수를 뺀 값 $k^2 - (k-1)^2 = 2k-1$이다.
ⅰ), ⅱ)에서 $X = k$ ($k = 1, 2, \dots, 6$)인 경우의 수는 $2k-1$개다.
전체 경우의 수는 $6^2 = 36$이므로 $P(X = k) = \frac{2k-1}{36}$이다.

$\therefore E(X) = \sum_{k=1}^{6} k \cdot P(X=k) = \sum_{k=1}^{6} \frac{2k^2 - k}{36} = \frac{161}{36}$
$\therefore E(6X) = 6E(X) = \frac{161}{6}$
$\therefore p = 161$, $q = 6$
$\therefore p + q = 167$
