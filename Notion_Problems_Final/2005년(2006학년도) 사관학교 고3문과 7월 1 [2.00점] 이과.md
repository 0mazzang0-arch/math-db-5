## 문제
$n = 2006$ 이고 $a = \frac{3}{4}$ 일 때, 세 수 $A$, $B$, $C$ 를 각각 
$$A = \sqrt[n]{a^{n-1}}, B = \sqrt[n]{a^{n+1}}, C = \sqrt[n+1]{a^{n}}$$
이라 하자. 다음 중 세 수 $A$, $B$, $C$ 의 대소 관계로 옳은 것은? 

## 보기
① $A < B < C$   ② $A < C < B$   ③ $B < A < C$   ④ $B < C < A$   ⑤ $C < A < B$

## 풀이
밑이 $1$ 보다 작으므로 단조감소함수이다. 즉, 지수가 클수록 작은 수가 된다. 

$$A = a^{\frac{n-1}{n}} = a^{1 - \frac{1}{n}}, B = a^{\frac{n+1}{n}} = a^{1 + \frac{1}{n}}, C = a^{\frac{n}{n+1}} = a^{1 - \frac{1}{n+1}}$$

지수의 대소는 

$$1 - \frac{1}{n} < 1 - \frac{1}{n+1} < 1 + \frac{1}{n}$$

따라서, 

$$B < C < A$$
