## 문제
모든 항이 양수인 수열 $\{ a_{n} \}$은 $a_{1} = 10$이고
$( a_{n+1} )^{n+1} = \frac{ a_{1} + ( a_{2} )^{2} + ( a_{3} )^{3} + \cdots + ( a_{n} )^{n} }{ n }$ ($n \ge 1$)
을 만족시킨다. 다음은 일반항 $a_{n}$을 구하는 과정의 일부이다.

$$ \begin{array}{l} 
b_{n} = ( a_{n} )^{n}이라 하면 \\
b_{1} = 10이고 주어진 식으로부터 \\
b_{n+1} = \frac{ b_{1} + b_{2} + \cdots + b_{n} }{ n } ( n \ge 1 ) \\
이다. \\
S_{n} = \sum_{k=1}^{n} b_{k}라 하면 \\
S_{n+1} = (가) \times S_{n} \\
이다. \\
S_{1} = 10, \\
S_{n} = S_{1} \times \frac{ S_{2} }{ S_{1} } \times \frac{ S_{3} }{ S_{2} } \times \cdots \times \frac{ S_{n} }{ S_{n-1} } ( n \ge 2 ) \\
를 이용하여 S_{n}을 구하면 \\
S_{n} = (나) ( n \ge 1 ) \\
이다. \\
\vdots
\end{array} $$

위의 (가), (나)에 알맞은 식을 각각 $f(n)$, $g(n)$이라 할 때, $f(5) \times g(6)$의 값은?

## 보기
① $72$   ② $76$   ③ $80$   ④ $84$   ⑤ $88$

## 풀이
$b_{n} = ( a_{n} )^{n}$이라 하면 $b_{1} = 10$이고 주어진 식으로부터
$b_{n+1} = \frac{ b_{1} + b_{2} + \cdots + b_{n} }{ n }$ ($n \ge 1$)이다.
$S_{n} = \sum_{k=1}^{n} b_{k}$라 하면 $b_{n+1} = \frac{ S_{n} }{ n }$에서
$S_{n+1} - S_{n} = \frac{ S_{n} }{ n }$
$n ( S_{n+1} - S_{n} ) = S_{n}$
$n S_{n+1} - n S_{n} = S_{n}$
$n S_{n+1} = ( n + 1 ) S_{n}$
$S_{n+1} = \frac{ n + 1 }{ n } S_{n}$
따라서 $f(n) = \frac{ n + 1 }{ n }$이다.

$S_{1} = 10$이고, $\frac{ S_{n+1} }{ S_{n} } = \frac{ n + 1 }{ n }$이므로 $n \ge 2$일 때,
$S_{n} = S_{1} \times \frac{ S_{2} }{ S_{1} } \times \frac{ S_{3} }{ S_{2} } \times \cdots \times \frac{ S_{n} }{ S_{n-1} }$
$= 10 \times \frac{ 2 }{ 1 } \times \frac{ 3 }{ 2 } \times \cdots \times \frac{ n }{ n - 1 }$
$= 10n$
$S_{1} = 10$이므로 $S_{n} = 10n$ ($n \ge 1$)이다.
따라서 $g(n) = 10n$이다.

$f(5) \times g(6) = \frac{ 6 }{ 5 } \times ( 10 \times 6 ) = \frac{ 6 }{ 5 } \times 60 = 72$
