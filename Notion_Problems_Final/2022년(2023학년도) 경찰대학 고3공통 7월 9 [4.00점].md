## 문제
집합 $A = \{ 1, 2, 3, 4, 5 \}$에서 $A$로의 함수 중에서 다음 조건을 만족시키는 함수 $f(x)$의 개수는?

(가) $\log f(x)$는 일대일함수가 아니다.
(나) $\log \{ f(1) + f(2) + f(3) \} = 2 \log 2 + \log 3$
(다) $\log f(4) + \log f(5) \le 1$

## 보기
① $134$   ② $140$   ③ $146$   ④ $152$   ⑤ $158$

## 풀이
조건 (가)에서 $f$가 일대일함수가 아니고, 
조건 (나)에서 
$\log \{ f(1) + f(2) + f(3) \} = \log 12$ 
$\therefore f(1) + f(2) + f(3) = 12$ $\cdots \cdots$ ㉠ 
조건 (다)에서 
$\log f(4) f(5) \le 1$ 
$\therefore f(4) f(5) \le 10$ $\cdots \cdots$ ㉡ 
㉠, ㉡을 동시에 만족하는 경우는 다음과 같다. 

(ⅰ) $f(1)$, $f(2)$, $f(3)$에 대응되는 값이 $5, 5, 2$인 경우의 수는 $\frac{3!}{2!} = 3$ 
$f(4)$, $f(5)$의 값이 대응되는 전체 경우의 수가 $5^2 = 25$ 
$f(4) f(5) > 10$을 만족하는 경우의 수는 
(1) $f(4)$, $f(5)$의 값이 모두 $4$ 이상인 경우의 수가 $2^2 = 4$ 
(2) $f(4)$, $f(5)$의 값이 $3, 5$ 또는 $4, 5$에 대응되는 경우의 수가 $2! \times 2 = 4$ 
따라서 $f(4) f(5) \le 10$을 만족하는 경우의 수는 $25 - (4 + 4) = 17$ 
이상에서 조건을 만족하는 경우의 수는 $3 \times 17 = 51$ 

(ⅱ) $f(1)$, $f(2)$, $f(3)$에 대응되는 값이 $5, 4, 3$인 경우의 수는 $3! = 6$ 
$f(4) f(5) \le 10$을 만족하는 경우의 수는 (ⅰ)에서 $f(4)$, $f(5)$가 $1, 2$에 대응되는 경우는 함수 $f$가 일대일함수이므로 조건 (가)에 어긋나므로 이 경우를 제외하면 
$25 - (4 + 4 + 2) = 15$ 
$6 \times 15 = 90$ 

(ⅲ) $f(1)$, $f(2)$, $f(3)$에 대응되는 값이 $4, 4, 4$인 경우의 수는 $1$ 
$f(4) f(5) \le 10$을 만족하는 경우의 수는 (ⅰ)에서 $17$ 
$1 \times 17 = 17$ 

(ⅰ)～(ⅲ)에서 조건을 만족하는 함수 $f$의 개수는 $51 + 90 + 17 = 158$이다.

정답: ⑤
