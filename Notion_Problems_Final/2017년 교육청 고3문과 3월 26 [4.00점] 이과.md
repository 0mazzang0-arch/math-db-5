## 문제
수열 $\{ a_{n} \}$ 이 모든 자연수 $n$ 에 대하여 
$a_{1} = 3$, $a_{n+1} = \frac{2}{3} a_{n}$
을 만족시킬 때, $\sum_{n=1}^{\infty} a_{2n-1} = \frac{q}{p}$ 이다. $p+q$ 의 값을 구하시오. (단, $p$ 와 $q$ 는 서로소인 자연수이다.)

## 보기

## 풀이
수열 $\{ a_{n} \}$ 이 $a_{1} = 3$ 이고, 모든 자연수 $n$ 에 대하여 $a_{n+1} = \frac{2}{3} a_{n}$ 이므로 수열 $\{ a_{n} \}$ 은 첫째항이 $3$ 이고 공비가 $\frac{2}{3}$ 인 등비수열이다.
따라서 $a_{n} = 3 \times \left( \frac{2}{3} \right)^{n-1}$ 이다.
$a_{2n-1} = 3 \times \left( \frac{2}{3} \right)^{(2n-1)-1} = 3 \times \left( \frac{2}{3} \right)^{2(n-1)} = 3 \times \left( \frac{4}{9} \right)^{n-1}$
$\sum_{n=1}^{\infty} a_{2n-1} = \frac{3}{1 - \frac{4}{9}} = \frac{3}{\frac{5}{9}} = \frac{27}{5}$
따라서 $\frac{q}{p} = \frac{27}{5}$ 이고, $p=5$, $q=27$ 이므로 
$p+q = 5+27 = 32$

정답: $32$
