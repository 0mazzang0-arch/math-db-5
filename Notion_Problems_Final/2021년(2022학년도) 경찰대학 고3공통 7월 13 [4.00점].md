## 문제
실수 $r = \frac{3}{\sqrt[3]{4} - \sqrt[3]{2} + 1}$ 에 대하여 $r + r^{2} + r^{3} = a \sqrt[3]{4} + b \sqrt[3]{2} + c$ 일 때, $a + b + c$ 의 값은? (단, $a, b, c$ 는 유리수이다.)

## 보기
① $7$   ② $9$   ③ $11$   ④ $13$   ⑤ $15$

## 풀이
$\sqrt[3]{2} = t$ 로 치환하면
$r = \frac{3}{t^{2} - t + 1}$
$= \frac{3(t + 1)}{(t + 1)(t^{2} - t + 1)}$
$= \frac{3(t + 1)}{t^{3} + 1}$
$= \frac{3(t + 1)}{2 + 1}$
$= t + 1$
$= \sqrt[3]{2} + 1$ ($\because \sqrt[3]{2} = t$)

즉, $r - 1 = \sqrt[3]{2}$ 이므로 $r - 1 = x$ 라 하면
$x^{2} = \sqrt[3]{4}$, $x^{3} = 2$

$r = x + 1$ 이므로 구하고자 하는 식에 대입하면
$r + r^{2} + r^{3} = (x + 1) + (x + 1)^{2} + (x + 1)^{3}$
$= x^{3} + 4x^{2} + 6x + 3$
$= 2 + 4 \sqrt[3]{4} + 6 \sqrt[3]{2} + 3$
$= 4 \sqrt[3]{4} + 6 \sqrt[3]{2} + 5$

따라서 $a = 4$, $b = 6$, $c = 5$ 이므로
$a + b + c = 4 + 6 + 5 = 15$

정답: ⑤
