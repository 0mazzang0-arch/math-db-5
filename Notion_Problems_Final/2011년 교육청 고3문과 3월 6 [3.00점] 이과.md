## 문제
$1$보다 큰 세 실수 $a$, $b$, $c$에 대하여 
$$\log _{a} 2 = \log _{b} 5 = \log _{c} 10 = \log _{abc} x$$
가 성립할 때, 실수 $x$의 값은? 

## 보기
① $\frac{1}{10}$   ② $\sqrt{10}$   ③ $10$   ④ $10 \sqrt{10}$   ⑤ $100$

## 풀이
$\log _{a} 2 = \log _{b} 5 = \log _{c} 10 = \log _{abc} x = \frac{1}{k}$이라고 하면, 
$\log _{2} a = \log _{5} b = \log _{10} c = \log _{x} abc = k$이다. 
따라서 $a = 2^{k}$, $b = 5^{k}$, $c = 10^{k}$이므로 $ab = c$이다. 
$\therefore k = \log _{x} abc = \log _{x} c^{2}$
이때, $k = \log _{x} c^{2} = \log _{x} 10^{2k} = k \log _{x} 100$이므로 
$x = 100$

[다른 풀이] 
$\frac{\log 2}{\log a} = \frac{\log 5}{\log b} = \frac{\log 10}{\log c} = \frac{\log 2 + \log 5 + \log 10}{\log a + \log b + \log c}$
$= \frac{\log 100}{\log abc} = \log_{abc} 100$
$\therefore x = 100$

정답: ⑤
