## 문제
모든 실수 $x$ 에 대하여 $f(x+2)=f(x)$ 인 함수 $f(x)$ 가
$f(x) = -2 \left| x - \frac{1}{2} \right| + 1$ ($-\frac{1}{2} \le x \le \frac{3}{2}$)
이고 함수 $g(x) = \lim_{n \to \infty} \frac{\{1+f(x)\}^{n} - 1}{\{1+f(x)\}^{n} + 1}$ 일 때, $g(10\sqrt{2}) - g(\sqrt{3})$ 의 값은?



> [!example] 📸 (그림/그래프)

## 보기
① $-2$   ② $-1$   ③ $0$   ④ $1$   ⑤ $2$

## 풀이
$y=f(x)$ 의 그래프는 다음과 같다.

> [!example] 📸 (그림/그래프: $y=f(x)$ 는 주기가 $2$ 이고, $x=k$ (정수)에서 $f(x)=0$, $x=2k+ \frac{1}{2}$ 에서 최댓값 $1$, $x=2k- \frac{1}{2}$ 에서 최솟값 $-1$ 을 갖는 톱니 모양의 그래프이다.)

함수 $y=g(x)$ 는 $1+f(x)$ 의 값의 범위에 따라 다음과 같이 결정된다.

(ⅰ) $x$ 가 정수이면 $f(x)=0$ 이므로 $1+f(x)=1$ 이다.
따라서 $g(x) = \lim_{n \to \infty} \frac{1^{n}-1}{1^{n}+1} = 0$

(ⅱ) $2k-1 < x < 2k$ ($k$ 는 정수)이면 $-1 \le f(x) < 0$ 이므로 $0 \le 1+f(x) < 1$ 이다.
따라서 $g(x) = \lim_{n \to \infty} \frac{(\text{0~1 사이})^{n}-1}{(\text{0~1 사이})^{n}+1} = \frac{-1}{1} = -1$

(ⅲ) $2k < x < 2k+1$ ($k$ 는 정수)이면 $0 < f(x) \le 1$ 이므로 $1 < 1+f(x) \le 2$ 이다.
따라서 $g(x) = \lim_{n \to \infty} \frac{(\text{1 초과})^{n}-1}{(\text{1 초과})^{n}+1} = 1$

$y=g(x)$ 의 그래프는 다음과 같다.

> [!example] 📸 (그림/그래프: 정수 지점에서는 $0$, 짝수와 홀수 사이에서는 $1$, 홀수와 짝수 사이에서는 $-1$ 을 갖는 단계 함수)

$10\sqrt{2} = \sqrt{200}$ 이고 $14^{2} = 196$, $15^{2} = 225$ 이므로 $14 < 10\sqrt{2} < 15$ 이다.
이는 (ⅲ)의 범위 ($2k < x < 2k+1$, 여기서 $k=7$)에 해당하므로 $g(10\sqrt{2}) = 1$ 이다.

$\sqrt{3} \approx 1.732$ 이므로 $1 < \sqrt{3} < 2$ 이다.
이는 (ⅱ)의 범위 ($2k-1 < x < 2k$, 여기서 $k=1$)에 해당하므로 $g(\sqrt{3}) = -1$ 이다.

따라서 $g(10\sqrt{2}) - g(\sqrt{3}) = 1 - (-1) = 2$ 이다.
