## 문제
자연수 $n$에 대하여 
$$f(n) = \sum_{k=1}^{n} ({}_{2k}C_{1} + {}_{2k}C_{3} + {}_{2k}C_{5} + \cdots + {}_{2k}C_{2k-1})$$ 
일 때, $f(5)$의 값을 구하시오.

## 보기

## 풀이
다항식 $(1+x)^{2n}$의 전개식은 
$$(1+x)^{2n} = {}_{2n}C_{0} + {}_{2n}C_{1}x + {}_{2n}C_{2}x^{2} + \cdots + {}_{2n}C_{2n}x^{2n} \cdots \cdots ㉠$$ 
등식 $㉠$의 양변에 $x=1$을 대입하면 
$$2^{2n} = {}_{2n}C_{0} + {}_{2n}C_{1} + {}_{2n}C_{2} + {}_{2n}C_{3} + \cdots + {}_{2n}C_{2n-1} + {}_{2n}C_{2n} \cdots \cdots ㉡$$ 
등식 $㉠$의 양변에 $x=-1$을 대입하면 
$$0 = {}_{2n}C_{0} - {}_{2n}C_{1} + {}_{2n}C_{2} - {}_{2n}C_{3} + \cdots - {}_{2n}C_{2n-1} + {}_{2n}C_{2n} \cdots \cdots ㉢$$ 
$㉡ - ㉢$을 하여 양변을 $2$로 나누면 
$${}_{2n}C_{1} + {}_{2n}C_{3} + {}_{2n}C_{5} + \cdots + {}_{2n}C_{2n-1} = 2^{2n-1}$$ 
이므로 
$$f(n) = \sum_{k=1}^{n} ({}_{2k}C_{1} + {}_{2k}C_{3} + {}_{2k}C_{5} + \cdots + {}_{2k}C_{2k-1})$$ 
$$= \sum_{k=1}^{n} 2^{2k-1}$$ 
이때 $f(n)$은 첫째항이 $2$이고 공비가 $4$인 등비수열의 첫째항부터 제 $n$항까지의 합이므로 
$$f(5) = \sum_{k=1}^{5} 2^{2k-1} = \frac{2(4^{5}-1)}{4-1} = 682$$
따라서 정답은 $682$이다.
