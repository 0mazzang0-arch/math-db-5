## 문제
$\sin \theta = -{1} \over {3}$ 일 때, ${ \cos \theta } \over { \tan \theta }$ 의 값은?

## 보기
① $-4$   ② $-{11} \over {3}$   ③ $-{10} \over {3}$   ④ $-3$   ⑤ $-{8} \over {3}$

## 풀이
${ \cos \theta } \over { \tan \theta } = { \cos \theta } \div { \sin \theta \over \cos \theta } = { \cos ^{2} \theta } \over { \sin \theta } = { 1 - \sin ^{2} \theta } \over { \sin \theta } = { 1 } \over { \sin \theta } - \sin \theta$
$\sin \theta = -{1} \over {3}$ 을 대입하면
${ 1 } \over { -{1} \over {3} } - ( -{1} \over {3} ) = -3 + {1} \over {3} = -{8} \over {3}$
따라서 정답은 ⑤이다.
