## 문제

$-1 \leq t \leq 1$ 인 실수 $t$ 에 대하여 $x$ 에 대한 방정식 
$$(\sin \frac{\pi x}{2} - t)(\cos \frac{\pi x}{2} - t) = 0$$
의 실근 중에서 집합 $\{x \mid 0 \leq x < 4\}$ 에 속하는 가장 작은 값을 $\alpha(t)$, 가장 큰 값을 $\beta(t)$ 라 하자. <보기>에서 옳은 것만을 있는 대로 고른 것은?

ㄱ. $-1 \leq t < 0$ 인 모든 실수 $t$ 에 대하여 $\alpha(t) + \beta(t) = 5$ 이다.
ㄴ. $\{t \mid \beta(t) - \alpha(t) = \beta(0) - \alpha(0)\} = \{t \mid 0 \leq t \leq \frac{\sqrt{2}}{2} \}$
ㄷ. $\alpha(t_1) = \alpha(t_2)$ 인 두 실수 $t_1, t_2$ 에 대하여 $t_2 - t_1 = \frac{1}{2}$ 이면 $t_1 \times t_2 = \frac{1}{3}$ 이다.

## 보기
① ㄱ   ② ㄱ, ㄴ   ③ ㄱ, ㄷ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이

ㄱ. $-1 \leq t < 0$ 인 모든 실수 $t$ 에 대하여 $\alpha(t)$ 와 $\beta(t)$ 의 위치는 아래 그림과 같다. 

> [!example] 📸 (그림/그래프)

$t$ 의 범위에서 점 $(t, \alpha(t))$ 는 곡선 $y = \sin \frac{\pi x}{2}$ 위에 있고 점 $(t, \beta(t))$ 는 곡선 $y = \cos \frac{\pi x}{2}$ 에 있다. 
$\alpha(t)$ 와 $\beta(t)$ 는 직선 $x = \frac{5}{2}$ 에 대해 대칭이므로 $\alpha(t) + \beta(t) = 5$ 이다. (참)

ㄴ. $t = 0$ 일 때, $\alpha(0) = 0$, $\beta(0) = 3$ 이다. 

> [!example] 📸 (그림/그래프)

$t > \frac{\sqrt{2}}{2}$ 일 때는 $\beta(t) - \alpha(t) > 3$ 이므로 $\beta(t) - \alpha(t) = 3$ 을 만족하는 $t$ 의 범위는 $0 \leq t \leq \frac{\sqrt{2}}{2}$ 이다. (참) 

ㄷ. $\alpha(t_1) = \alpha(t_2) = k$ 라 하면, $t_1$ 과 $t_2$ 는 $\sin \frac{k \pi}{2}$ 또는 $\cos \frac{k \pi}{2}$ 이다. 
$\sin \frac{k \pi}{2} = t_1$, $\cos \frac{k \pi}{2} = t_2$ 라 하면, 
$$t_2 - t_1 = \cos \frac{k \pi}{2} - \sin \frac{k \pi}{2} = \frac{1}{2}$$
$$\cos^2 \frac{k \pi}{2} - 2 \cos \frac{k \pi}{2} \sin \frac{k \pi}{2} + \sin^2 \frac{k \pi}{2} = \frac{1}{4}$$
$$\cos \frac{k \pi}{2} \sin \frac{k \pi}{2} = \frac{3}{8} = t_2 \times t_1$$
$t_1 \times t_2 = \frac{3}{8}$ 이다. (거짓) 
따라서 보기에서 옳은 것은 ㄱ, ㄴ이다. 

**[다른 풀이]** 

$(\sin \frac{\pi x}{2} - t)(\cos \frac{\pi x}{2} - t) = 0$ 에서 
$\sin \frac{\pi x}{2} = t$ 또는 $\cos \frac{\pi x}{2} = t$ 인 실근 중 가장 작은 값이 $\alpha(t)$, 가장 큰 값이 $\beta(t)$ 이다. 
( 단, $0 \leq x < 4$ 이므로 $0 \leq \frac{\pi x}{2} < 2\pi$ 이다.) 
이를 $t$ 값의 범위에 따라 그림을 그리면 아래와 같다. 

ㄱ. $-1 \leq t < 0$ 인 경우 

> [!example] 📸 (그림/그래프)

$\cos \frac{\pi \alpha(t)}{2} = t$ ( 단, $1 < \alpha(t) \leq 2$) 
$\sin \frac{\pi \beta(t)}{2} = t$ ( 단, $3 \leq \beta(t) < 4$) 
$\alpha(t)$ 와 $\beta(t)$ 는 $x = \frac{5}{2}$ 에 대하여 대칭이므로 
$\frac{\alpha(t) + \beta(t)}{2} = \frac{5}{2}$ 이고 $\alpha(t) + \beta(t) = 5$ 이다. (참) 

ㄴ. $0 \leq t \leq \frac{\sqrt{2}}{2}$ 인 경우 

> [!example] 📸 (그림/그래프)

$\sin \frac{\pi \alpha(t)}{2} = t$ ( 단, $0 \leq \alpha(t) \leq \frac{1}{2}$) 
$\cos \frac{\pi \beta(t)}{2} = t$ ( 단, $3 \leq \beta(t) < 4$) 
$t = 0$ 일 때, $\alpha(0) = 0$, $\beta(0) = 3$ 이므로 $\beta(0) - \alpha(0) = 3$ 이다. 
$0 \leq t \leq \frac{\sqrt{2}}{2}$ 인 모든 $t$ 에 대하여 $\beta(t) - \alpha(t) = 3$ 이다. (참) 

ㄷ. $\frac{\sqrt{2}}{2} \leq t \leq 1$ 인 경우 

> [!example] 📸 (그림/그래프)

$\cos \frac{\pi \alpha(t)}{2} = t$ ( 단, $0 \leq \alpha(t) < \frac{1}{2}$) 
$\cos \frac{\pi \beta(t)}{2} = t$ ( 단, $3 \leq \beta(t) < 4$) 
$\alpha(t_1) = \alpha(t_2)$ 이므로 $\frac{\pi \alpha(t_1)}{2} = \frac{\pi \alpha(t_2)}{2} = \theta$ 라 두면, 
$t_2 - t_1 = \frac{1}{2}$ 이므로 $t_2 > t_1$ 이고 $0 \leq \alpha(t) < \frac{1}{2}$ 이므로 서로 같은 실근이 존재할 수 있다. 
$t_2$ 는 ㄷ의 그래프에서 $\cos \frac{\pi \alpha(t_2)}{2} = \cos \theta = t_2$ ( 단, $\frac{\sqrt{2}}{2} \leq t_2 \leq 1$)이다. 
$t_1$ 은 ㄴ의 그래프에서 $\sin \frac{\pi \alpha(t_1)}{2} = \sin \theta = t_1$ ( 단, $0 \leq t_1 \leq \frac{\sqrt{2}}{2}$) 이다. 
$t_2 - t_1 = \cos \theta - \sin \theta = \frac{1}{2}$ 이고 
$\cos^2 \theta - 2 \sin \theta \cos \theta + \sin^2 \theta = \frac{1}{4}$ 
$\sin \theta \cos \theta = \frac{3}{8}$ 이므로 $t_1 \times t_2 = \frac{3}{8}$ 이다. (거짓)
