## 문제
조건 $p$가 조건 $q$이기 위한 충분조건인 것을 <보기>에서 모두 고른 것은? (단, $x$, $y$는 실수이고 $\alpha$, $\beta$는 복소수이다.)

$$ \begin{array}{|l|} \hline \text{ㄱ. } p : x^{2} - y^{2} = 0, \text{ } q : x - y = 0 \\ \text{ㄴ. } p : x^{2} + y^{2} = 0, \text{ } q : x + y = 0 \\ \text{ㄷ. } p : \alpha^{2} + \beta^{2} = 0, \text{ } q : \alpha + \beta = 0 \\ \hline \end{array} $$

## 보기
① ㄱ   ② ㄴ   ③ ㄷ   ④ ㄱ, ㄷ   ⑤ ㄴ, ㄷ

## 풀이
ㄱ. (반례) $x=1$, $y=-1$ 이면 $x^{2} - y^{2} = 0$ 이지만 $x - y \neq 0$ 이다. 따라서 명제 $p \rightarrow q$ 는 참이 아니다. 

ㄴ. $x^{2} \geq 0$, $y^{2} \geq 0$ 이므로 $x^{2} + y^{2} = 0$ 이면 $x = y = 0$ 이다. 따라서 $x + y = 0$ 이므로 명제 $p \rightarrow q$ 는 참이다. 

ㄷ. (반례) $\alpha = 1$, $\beta = i$ 이면 $\alpha^{2} + \beta^{2} = 1^{2} + i^{2} = 1 - 1 = 0$ 이지만 $\alpha + \beta = 1 + i \neq 0$ 이다. 따라서 명제 $p \rightarrow q$ 는 참이 아니다. 

이상에서 조건 $p$ 가 조건 $q$ 이기 위한 충분조건인 것은 ㄴ이다. 

정답: ②
