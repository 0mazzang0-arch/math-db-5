다음은 $n \ge 6$일 때, 검은색으로 칠할 타일 $3$개를 고르는 경우의 수 $S(n)$을 구하는 과정이다.

$$ \begin{array}{|l|} \hline \text{첫째 줄에 있는 타일 중 검은색으로 칠할 타일의 개수를 } k \\ (k=0, 1, 2, 3) \text{이라 하면} \\ ( \text{i} ) \text{ } k=0 \text{ 일 때 둘째 줄에 있는 } n \text{ 개의 타일 중에서 검은색으로 칠할 타일} \\ 3 \text{ 개를 고르는 경우의 수는 } (가) \text{ 이다.} \\ ( \text{ii} ) \text{ } k=1 \text{ 일 때 둘째 줄에 있는 } n \text{ 개의 타일 중에서 검은색으로 칠할 타일} \\ 2 \text{ 개를 고르는 경우의 수는 } {}_{3}H_{n-3} \text{ 이고, 첫째 줄에서 검은색으로 칠할 타일} \\ 1 \text{ 개를 고르는 경우의 수는 } (나) \text{ 이므로, 검은색으로 칠할 타일} \\ 3 \text{ 개를 고르는 경우의 수는 } {}_{3}H_{n-3} \times (나) \text{ 이다.} \\ ( \text{iii} ) \text{ } k=2 \text{ 일 때 (ii)와 같은 방법으로 구할 수 있다.} \\ ( \text{iv} ) \text{ } k=3 \text{ 일 때 (i)과 같은 방법으로 구할 수 있다.} \\ \text{따라서 } S(n) = \frac{2(n-2)(2n^2-8n+9)}{3} \text{ 이다.} \\ \hline \end{array} $$

위의 $(가), (나)$에 알맞은 식을 각각 $f(n), g(n)$이라 할 때, $f(10)+g(8)$의 값은?

## 보기

① $60$   ② $61$   ③ $62$   ④ $63$   ⑤ $64$

## 풀이

$( \text{i} ) $ $k=0$ 일 때
$n$ 개의 타일 중에서 검은색으로 칠할 타일 $3$ 개를 고르는 경우의 수는 다음과 같다.
검은색 타일 사이에는 $1$ 개 이상의 흰색 타일이 있어야 한다.
흰색 타일들에 의해 생기는 빈 공간이 $4$ 곳이므로 타일의 개수를 결정하는 경우의 수는 ${}_{4}H_{n-5}$ 이다.

$( \text{ii} ) $ $k=1$ 일 때
둘째 줄에서 $n$ 개의 타일 중 검은색으로 칠할 타일 $2$ 개를 고르는 경우의 수는 ${}_{3}H_{n-3}$ 이다.
첫째 줄에서 검은색으로 칠할 타일 $1$ 개를 고르는 경우의 수는 둘째 줄에서 검은색으로 칠할 타일의 바로 위쪽에 있는 타일을 제외한 나머지 $n-2$ 개의 타일 중 $1$ 개의 타일을 고르는 경우의 수와 같으므로 ${}_{n-2}C_{1}$ 이다.
그러므로 검은색으로 칠할 타일 $3$ 개를 고르는 경우의 수는 ${}_{3}H_{n-3} \times {}_{n-2}C_{1}$ 이다.

$(가)$에 알맞은 식은 ${}_{4}H_{n-5}$ 이므로
$f(n) = {}_{4}H_{n-5}$
$(나)$에 알맞은 식은 ${}_{n-2}C_{1}$ 이므로
$g(n) = {}_{n-2}C_{1}$

따라서 $f(10)+g(8)$ 의 값은
$$ \begin{aligned} f(10)+g(8) &= {}_{4}H_{10-5} + {}_{8-2}C_{1} \\ &= {}_{4}H_{5} + {}_{6}C_{1} \\ &= {}_{8}C_{5} + 6 \\ &= 56 + 6 \\ &= 62 \end{aligned} $$

정답: ③
