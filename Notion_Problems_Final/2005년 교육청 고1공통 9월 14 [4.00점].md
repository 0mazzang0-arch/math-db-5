## 문제
그림에서 사각형 $ABCD$는 한 변의 길이가 $1$인 정사각형이고, 세 점 $A$, $B$, $E$는 수직선 위에 있으며 $\overline{AC} = \overline{AE}$이다. 세 점 $A$, $B$, $E$에 대응하는 수를 각각 $p$, $q$, $r$라 할 때, 옳은 내용을 <보기>에서 모두 고른 것은?

> [!example] 📸 (그림/그래프)

$$
\begin{array}{|l|}
\hline
\text{ㄱ. } p \text{ 가 유리수이면 } q \text{ 는 유리수, } r \text{ 는 무리수이다.} \\
\text{ㄴ. } p \text{ 가 무리수이면 } q \text{ 는 무리수, } r \text{ 는 유리수이다.} \\
\text{ㄷ. } q \text{ 가 유리수이면 } r \text{ 는 무리수이다.} \\
\hline
\end{array}
$$

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄱ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
세 수 $p$, $q$, $r$ 사이의 관계는 다음과 같다.
$q = p + 1$, $r = p + \sqrt{2}$, $r = q + \sqrt{2} - 1$

ㄱ. $(\text{유리수}) + (\text{유리수}) = (\text{유리수})$이고 $(\text{유리수}) + (\text{무리수}) = (\text{무리수})$이므로 $p$가 유리수이면 $q$는 유리수, $r$는 무리수이다.

ㄴ. (반례) $p = -\sqrt{2}$일 때 $r = 0$이 되어 $r$는 유리수일 수도 있으나, 문제의 조건이나 그림의 일반적인 상황에서 $p = \sqrt{2}$와 같은 경우 $r = 2\sqrt{2}$가 되어 $r$는 무리수이다. 즉, $p$가 무리수라고 해서 $r$가 항상 유리수인 것은 아니다.

ㄷ. $r = q + (\sqrt{2} - 1)$에서 $\sqrt{2} - 1$이 무리수이므로 $q$가 유리수이면 $r$는 무리수이다.

이상에서 옳은 것은 ㄱ, ㄷ이다.
