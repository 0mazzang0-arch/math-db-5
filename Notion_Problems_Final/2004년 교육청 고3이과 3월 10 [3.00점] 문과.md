## 문제
함수 $f(x) = x + \log_{10} x$ 에 대하여 $\sum_{n=1}^{100} [f(n)]$ 의 값은? (단, $[x]$ 는 $x$ 보다 크지 않은 최대의 정수이다.)

## 보기
① $5055$   ② $5060$   ③ $5084$   ④ $5128$   ⑤ $5142$

## 풀이
$f(x) = x + \log_{10} x$ 이므로 

$$ \sum_{n=1}^{100} [f(n)] = [f(1)] + [f(2)] + \cdots + [f(100)] $$ 

$$ = [1 + \log_{10} 1] + [2 + \log_{10} 2] + \cdots + [100 + \log_{10} 100] $$ 

$n$ 이 정수일 때 $[n + x] = n + [x]$ 이므로

$$ = (1 + 2 + \cdots + 100) + ([\log_{10} 1] + [\log_{10} 2] + \cdots + [\log_{10} 100]) $$ 

이때, 
$1 \le n \le 9$ 일 때 $[\log_{10} n] = 0$ ($9$ 개)
$10 \le n \le 99$ 일 때 $[\log_{10} n] = 1$ ($90$ 개)
$n = 100$ 일 때 $[\log_{10} 100] = 2$ ($1$ 개)

따라서
$$ = 5050 + (0 \times 9 + 1 \times 90 + 2 \times 1) = 5050 + 92 = 5142 $$

정답은 ⑤ 이다.
