## 문제
두 집합 $A = \{ x | x^{2}-6x+a \leq 0 \}$, $B = \{ x | x^{2}-x+b < 0 \}$에 대하여 $A \cup B = \{ x | -1 < x \leq 4 \}$를 만족시키는 두 상수 $a$, $b$에 대하여 $a+b$의 값은?

## 보기
① $4$   ② $5$   ③ $6$   ④ $7$   ⑤ $8$

## 풀이
$A \cap B = \emptyset$ 와 $A \cup B = \{ x | -1 < x \leq 4 \}$ 이므로 
$x^{2}-6x+a = (x-4)(x-2)$ 이고 
$x^{2}-x+b = (x+1)(x-2)$ 이다. 
따라서 $a = 8$, $b = -2$ 이다. 
두 집합 $A = \{ x | x^{2}-6x+a \leq 0 \}$, $B = \{ x | x^{2}-x+b < 0 \}$에 대하여 $A \cup B = \{ x | -1 < x \leq 4 \}$를 만족시키는 두 상수 $a$, $b$에 대하여 $a+b$의 값은 
$a+b = 8+(-2) = 6$이다.

정답: ③
