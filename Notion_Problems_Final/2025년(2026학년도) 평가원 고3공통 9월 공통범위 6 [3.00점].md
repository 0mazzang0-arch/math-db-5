## 문제
$\cos ( \theta - \pi ) = \frac{3}{5}$ 이고 $\tan \theta < 0$ 일 때, $\sin \theta$ 의 값은?

## 보기
① $-\frac{4}{5}$   ② $-\frac{3}{5}$   ③ $\frac{1}{5}$   ④ $\frac{3}{5}$   ⑤ $\frac{4}{5}$

## 풀이
$\cos ( \theta - \pi ) = \cos \{ - ( \pi - \theta ) \} = \cos ( \pi - \theta ) = - \cos \theta$
$\cos ( \theta - \pi ) = \frac{3}{5}$ 이므로
$- \cos \theta = \frac{3}{5}$, $\cos \theta = - \frac{3}{5}$
$\cos \theta < 0$ 이고 $\tan \theta < 0$ 에서 각 $\theta$ 는 제 $2$ 사분면의 각이므로 $\sin \theta > 0$
$\therefore \sin \theta = \sqrt{1 - \cos^{2} \theta} = \sqrt{1 - ( - \frac{3}{5} )^{2}} = \frac{4}{5}$

정답: ⑤
