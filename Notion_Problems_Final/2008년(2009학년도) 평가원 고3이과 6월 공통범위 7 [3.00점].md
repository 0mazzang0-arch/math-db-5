## 문제
삼차함수 $f(x) = x(x - 1)(ax + 1)$의 그래프 위의 점 $P(1, 0)$을 접점으로 하는 접선을 $l$이라 하자. 직선 $l$에 수직이고 점 $P$를 지나는 직선이 곡선 $y = f(x)$와 서로 다른 세 점에서 만나도록 하는 $a$의 값의 범위는?

## 보기
① $-1 < a < - \frac{1}{3}$ 또는 $0 < a < 1$   ② $- \frac{1}{3} < a < 0$ 또는 $0 < a < 1$   ③ $-1 < a < 0$ 또는 $0 < a < \frac{1}{3}$   ④ $-1 < a < 0$ 또는 $\frac{1}{3} < a < 1$   ⑤ $-2 < a < - \frac{1}{3}$ 또는 $\frac{1}{3} < a < 2$

## 풀이
[정답] ③

$P(1, 0)$에서 법선의 방정식을 구하면 $f'(1) = a + 1$이므로
$y = - \frac{1}{a + 1} (x - 1)$ (단, $a \neq -1$) $\cdots$ ㉠
($\because f'(1) = 0$이면 법선이 $x$축에 수직이 되어 부적합)

㉠과 $y = f(x)$가 서로 다른 세 점에서 만나므로
$x(x - 1)(ax + 1) = - \frac{1}{a + 1} (x - 1)$
이것이 서로 다른 세 실근을 갖는다.
$(x - 1) \left\{ x(ax + 1) + \frac{1}{a + 1} \right\} = 0$

$ax^2 + x + \frac{1}{a + 1} = 0$이 $x = 1$인 근을 갖지 않으므로, 서로 다른 두 실근을 가지면 된다.
$a(a + 1)x^2 + (a + 1)x + 1 = 0$
$D = (a + 1)^2 - 4a(a + 1) > 0$
$(a + 1)(1 - 3a) > 0$
$(a + 1)(3a - 1) < 0$
$\therefore -1 < a < \frac{1}{3}$

이때 $a \neq 0$이어야 하므로
$-1 < a < 0$ 또는 $0 < a < \frac{1}{3}$
