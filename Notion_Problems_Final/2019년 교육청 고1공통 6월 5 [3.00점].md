## 문제
모든 실수 $x$ 에 대하여 등식 
$$x^{2} + 5x + a = (x + 4)(x + b)$$
가 성립할 때, $a + b$ 의 값은? (단, $a$, $b$ 는 상수이다.)

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
$x$ 에 대한 항등식이므로 $x = -4$ 를 대입하면 
$16 - 20 + a = 0$ 이므로 
$a = 4$ 이다. 

$x^{2} + 5x + 4 = (x + 4)(x + 1)$ 이므로 
$b = 1$ 이다. 
따라서 $a + b = 5$ 이다. 

[다른 풀이] 
$(x + 4)(x + b) = x^{2} + (4 + b)x + 4b$ 이다. 
$x^{2} + 5x + a = x^{2} + (4 + b)x + 4b$ 의 양변의 계수를 비교하면 
$5 = 4 + b$, $a = 4b$ 이다. 
따라서 $b = 1$, $a = 4$ 이므로 $a + b = 5$ 이다.
