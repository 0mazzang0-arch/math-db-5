## 문제
이차방정식 $x^{2} + x - 3 = 0$의 두 근을 $\alpha$, $\beta$라 할 때, $f(\alpha) = f(\beta) = 1$을 만족하는 이차식 $f(x)$는? (단, $f(x)$의 이차항의 계수는 $1$이다.)

## 보기
① $x^{2} + x - 2$   ② $x^{2} - x - 4$   ③ $x^{2} + x + 4$   ④ $x^{2} - 2x + 2$   ⑤ $x^{2} + 2x + 4$

## 풀이
$x^{2} + x - 3 = 0$의 두 근이 $\alpha$, $\beta$이므로
$\alpha^{2} + \alpha = 3$, $\beta^{2} + \beta = 3$
근과 계수의 관계에 의하여
$\alpha + \beta = -1$, $\alpha \beta = -3$

$f(x) = x^{2} + ax + b$라 하면
$f(\alpha) = \alpha^{2} + a\alpha + b = 1$ $\cdots \cdots$ ㉠
$f(\beta) = \beta^{2} + a\beta + b = 1$ $\cdots \cdots$ ㉡

㉠ $-$ ㉡을 하면
$\alpha^{2} - \beta^{2} + a\alpha - a\beta = 0$
$(\alpha + \beta)(\alpha - \beta) + a(\alpha - \beta) = 0$
$(\alpha - \beta)(\alpha + \beta + a) = 0$

$\therefore \alpha = \beta$ 또는 $\alpha + \beta + a = 0$
그런데 $\alpha \neq \beta$이므로 $\alpha + \beta + a = 0$
$\therefore a = -(\alpha + \beta) = -(-1) = 1$

$a = 1$을 ㉠에 대입하면
$\alpha^{2} + \alpha + b = 1$
$b = 1 - (\alpha^{2} + \alpha) = 1 - 3 = -2$

$\therefore f(x) = x^{2} + x - 2$

**[다른 풀이]**
$x^{2} + x - 3 = 0$의 두 근을 $\alpha$, $\beta$라 하면
$x^{2} + x - 3 = (x - \alpha)(x - \beta)$ $\cdots \cdots$ ㉠

$f(\alpha) = f(\beta) = 1$에서
$f(\alpha) - 1 = 0$, $f(\beta) - 1 = 0$

따라서 최고차항의 계수가 $1$인 이차식 $f(x) - 1$은 $x - \alpha$, $x - \beta$를 인수로 갖는다. 따라서
$f(x) - 1 = (x - \alpha)(x - \beta) = x^{2} + x - 3$ ($\because$ ㉠)

$\therefore f(x) = x^{2} + x - 2$

정답: ①
