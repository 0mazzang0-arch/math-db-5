## 문제
$k$가 자연수일 때 $\log k$의 정수부분 $n$과 소수부분 $\alpha$에 대하여 좌표평면 위의 점 $P_{k}$를 $P_{k}(\alpha, n)$이라 하자. 점 $P_{k}$를 곡선 $y = (\sqrt{10})^{x}$ 위에 있도록 하는 모든 $k$ 값의 합은?

## 보기
① $1210$   ② $3210$   ③ $5410$   ④ $7510$   ⑤ $9410$

## 풀이
자연수 $k$에 대하여 $\log k = n + \alpha$ ($n$은 정수, $0 \le \alpha < 1$)라 하면, 점 $P_{k}(\alpha, n)$이 곡선 $y = (\sqrt{10})^{x}$ 위에 있기 위한 조건은 다음과 같다.
$n = (\sqrt{10})^{\alpha} = 10^{\frac{\alpha}{2}}$
양변에 상용로그를 취하면
$\log n = \frac{\alpha}{2}$
$\alpha = 2 \log n = \log n^{2}$
이때 $0 \le \alpha < 1$이므로
$0 \le \log n^{2} < 1$
$1 \le n^{2} < 10$
이를 만족하는 정수 $n$의 값은 $n = 1, 2, 3$이다.
$n = 1$일 때, $\alpha = \log 1^{2} = \log 1$
$\log k = 1 + \log 1 = 1$ $\therefore k = 10^{1} = 10$
$n = 2$일 때, $\alpha = \log 2^{2} = \log 4$
$\log k = 2 + \log 4 = \log 10^{2} + \log 4 = \log 400$ $\therefore k = 400$
$n = 3$일 때, $\alpha = \log 3^{2} = \log 9$
$\log k = 3 + \log 9 = \log 10^{3} + \log 9 = \log 9000$ $\therefore k = 9000$
따라서 모든 $k$의 값의 합은
$10 + 400 + 9000 = 9410$

정답: ⑤
