## 문제
이차함수 $f(x) = a x^{2} + 2 b x + c$ 가 주어져 있다. $x \ge 0$ 인 모든 $x$ 에 대하여 $f(x) > 0$ 일 때, <보기>에서 항상 옳은 것을 모두 고르면?

$$ \begin{array}{|l|} \hline \text{ㄱ. } c > 0 \\ \text{ㄴ. } a b > 0 \\ \text{ㄷ. } b^{2} - a c < 0 \\ \hline \end{array} $$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
$f(x) = a x^{2} + 2 b x + c$ 가 $x \ge 0$ 인 모든 $x$ 에 대하여 $f(x) > 0$ 이어야 하므로 그래프의 개형은 아래로 볼록한 개형이어야 한다.
$\therefore a > 0$

ㄱ. $x = 0$ 일 때, 즉 $f(0) > 0$ 이어야 하므로 $f(0) = c > 0$
$\therefore$ 참

ㄴ, ㄷ. $f(x) = a x^{2} + 2 b x + c = a \left(x + \frac{b}{a}\right)^{2} - \frac{b^{2} - a c}{a}$ 에서 축의 위치에 따라 생각해 보면
(i) $-\frac{b}{a} > 0$ 즉, $a b < 0$ 일 때,
(최솟값) $= f \left( - \frac{b}{a} \right) = - \frac{b^{2} - a c}{a} > 0$
$\therefore b^{2} - a c < 0$ ($\because a > 0$)

(ii) $-\frac{b}{a} < 0$ 즉, $a b > 0$ 일 때,
(최솟값) $= f(0) = c > 0$

(i), (ii)에서 반드시 $a b > 0$, $b^{2} - a c < 0$ 이라고는 할 수 없다.
$\therefore$ 거짓

따라서 옳은 것은 $ㄱ$뿐이다.
