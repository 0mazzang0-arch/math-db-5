## 문제
실수 $a$ ($a > 1$)와 자연수 $n$에 대하여 직선 $x = n$이 두 함수 $y = 3 a^{x}$, $y = 3 a^{x - 1}$의 그래프와 만나는 점을 각각 $P_{n}$, $Q_{n}$이라 하자. 선분 $P_{n} Q_{n}$의 길이를 $l_{n}$, 사다리꼴 $P_{n} Q_{n} Q_{n + 2} P_{n + 2}$의 넓이를 $S_{n}$이라 하자. 두 실수 $L$, $S$에 대하여 $\sum_{k = 1}^{20} l_{k} = L$, $\sum_{k = 1}^{5} S_{4 k - 3} = S$일 때, 다음은 $\frac{S}{L} = \frac{2}{5}$를 만족시키는 $a$의 값을 구하는 과정이다.

두 점 $P_{n}$, $Q_{n}$의 좌표는 각각 $( n, 3 a^{n} )$, $( n, 3 a^{n - 1} )$
선분 $P_{n} Q_{n}$의 길이 $l_{n}$은
$l_{n} = 3 ( a - 1 ) \times a^{n - 1}$이므로
$L = \sum_{k = 1}^{20} l_{k} = 3 \times ($ $(가)$ $)$ 이다.
사다리꼴 $P_{n} Q_{n} Q_{n + 2} P_{n + 2}$의 넓이 $S_{n}$은
$S_{n} = 3 ( a - 1 ) \times ( a^{n - 1} + a^{n + 1} )$이므로
$S = \sum_{k = 1}^{5} S_{4 k - 3} = S_{1} + S_{5} + S_{9} + S_{13} + S_{17}$
$= \frac{3}{(나)} \times ($ $(가)$ $)$ 이다.
따라서 $\frac{S}{L} = \frac{\frac{3}{(나)} \times (가)}{3 \times (가)} = \frac{1}{(나)} = \frac{2}{5}$
이므로 $a =$ $(다)$ 이다.

위의 $(가)$, $(나)$에 알맞은 식을 각각 $f(a)$, $g(a)$라 하고, $(다)$에 알맞은 수를 $p$라 할 때, $\frac{f(\sqrt{2})}{g(20 p)}$의 값은?

## 보기
① $24$   ② $27$   ③ $30$   ④ $33$   ⑤ $36$

## 풀이
두 점 $P_{n}$, $Q_{n}$의 좌표는 각각 $( n, 3 a^{n} )$, $( n, 3 a^{n - 1} )$ 이다.
선분 $P_{n} Q_{n}$의 길이 $l_{n}$은
$l_{n} = 3 a^{n} - 3 a^{n - 1} = 3 ( a - 1 ) \times a^{n - 1}$이므로
$L = \sum_{k = 1}^{20} l_{k} = 3 ( a - 1 ) \sum_{k = 1}^{20} a^{k - 1}$
$= 3 ( a - 1 ) \times \frac{a^{20} - 1}{a - 1} = 3 \times ( a^{20} - 1 )$ 이다.
따라서 $f(a) = a^{20} - 1$ 이다.

사다리꼴 $P_{n} Q_{n} Q_{n + 2} P_{n + 2}$의 높이는 $( n + 2 ) - n = 2$ 이므로 넓이 $S_{n}$은
$S_{n} = \frac{1}{2} \times 2 \times ( l_{n} + l_{n + 2} ) = l_{n} + l_{n + 2}$
$= 3 ( a - 1 ) a^{n - 1} + 3 ( a - 1 ) a^{n + 1} = 3 ( a - 1 ) ( a^{n - 1} + a^{n + 1} )$ 이다.
$S = \sum_{k = 1}^{5} S_{4 k - 3} = S_{1} + S_{5} + S_{9} + S_{13} + S_{17}$
$= 3 ( a - 1 ) \{ ( a^{0} + a^{2} ) + ( a^{4} + a^{6} ) + ( a^{8} + a^{10} ) + ( a^{12} + a^{14} ) + ( a^{16} + a^{18} ) \}$
$= 3 ( a - 1 ) ( 1 + a^{2} + a^{4} + \cdots + a^{18} )$
$= 3 ( a - 1 ) \times \frac{( a^{2} )^{10} - 1}{a^{2} - 1} = 3 ( a - 1 ) \times \frac{a^{20} - 1}{( a - 1 ) ( a + 1 )}$
$= \frac{3}{a + 1} \times ( a^{20} - 1 )$ 이다.
따라서 $g(a) = a + 1$ 이다.

$\frac{S}{L} = \frac{\frac{3}{a + 1} ( a^{20} - 1 )}{3 ( a^{20} - 1 )} = \frac{1}{a + 1} = \frac{2}{5}$
이므로 $2 a + 2 = 5$, 즉 $a = \frac{3}{2}$ 이다.
따라서 $p = \frac{3}{2}$ 이다.

위의 결과에서
$f( \sqrt{2} ) = ( \sqrt{2} )^{20} - 1 = 2^{10} - 1 = 1023$
$g( 20 p ) = g( 20 \times \frac{3}{2} ) = g( 30 ) = 30 + 1 = 31$
따라서 $\frac{f( \sqrt{2} )}{g( 20 p )} = \frac{1023}{31} = 33$ 이다.
