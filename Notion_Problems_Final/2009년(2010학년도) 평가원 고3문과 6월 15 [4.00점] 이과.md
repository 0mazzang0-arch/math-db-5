## 문제
$n$ 이 $2$ 이상의 자연수일 때 $\sum_{k=1}^{n} k ( {}_n \mathrm{C}_k )^2$ 의 값을 구하는 과정이다.

$$ \begin{array}{|l|} \hline \text{두 다항식의 곱 } (a_0 + a_1 x + \cdots + a_{n-1} x^{n-1} ) (b_0 + b_1 x + \cdots + b_n x^n ) \text{ 에서 } \\ x^{n-1} \text{ 의 계수는 } a_0 b_{n-1} + a_1 b_{n-2} + \cdots + a_{n-1} b_0 \quad \cdots (*) \text{ 이다.} \\ \\ (1+x)^{2n-1} = (1+x)^{n-1} (1+x)^n \text{ 의 좌변에서 } x^{n-1} \text{ 의 계수는 (가) 이고,} \\ (*) \text{ 을 이용하여 우변에서 } x^{n-1} \text{ 의 계수를 구하면 } \\ \sum_{k=1}^{n} ( {}_{n-1} \mathrm{C}_{k-1} \times (나) ) \text{ 이다. } \\ \text{따라서 (가) } = \sum_{k=1}^{n} ( {}_{n-1} \mathrm{C}_{k-1} \times (나) ) \text{ 이다.} \\ \\ 1 \le k \le n \text{ 일 때, } k \times {}_n \mathrm{C}_k = n \times {}_{n-1} \mathrm{C}_{k-1} \text{ 이므로} \\ \sum_{k=1}^{n} k ( {}_n \mathrm{C}_k )^2 = \sum_{k=1}^{n} ( n \times {}_{n-1} \mathrm{C}_{k-1} \times (나) ) \\ = n \times \sum_{k=1}^{n} ( {}_{n-1} \mathrm{C}_{k-1} \times (나) ) = (다) \text{ 이다.} \\ \hline \end{array} $$

위의 과정에서 (가), (나), (다)에 알맞은 것은?

## 보기
① (가) ${}_{2n} \mathrm{C}_n$, (나) ${}_n \mathrm{C}_{n-k+1}$, (다) $\frac{n}{2} \times {}_{2n} \mathrm{C}_{n+1}$   ② (가) ${}_{2n-1} \mathrm{C}_{n-1}$, (나) ${}_n \mathrm{C}_{n-k+1}$, (다) $\frac{n}{2} \times {}_{2n} \mathrm{C}_n$   ③ (가) ${}_{2n-1} \mathrm{C}_{n-1}$, (나) ${}_n \mathrm{C}_{n-k}$, (다) $\frac{n}{2} \times {}_{2n} \mathrm{C}_n$   ④ (가) ${}_{2n} \mathrm{C}_n$, (나) ${}_n \mathrm{C}_{n-k+1}$, (다) $n \times {}_{2n} \mathrm{C}_{n+1}$   ⑤ (가) ${}_{2n-1} \mathrm{C}_{n-1}$, (나) ${}_n \mathrm{C}_{n-k}$, (다) $n \times {}_{2n} \mathrm{C}_n$

## 풀이
두 다항식의 곱 $(a_0 + a_1 x + \cdots + a_{n-1} x^{n-1})(b_0 + b_1 x + \cdots + b_n x^n)$ 에서 $x^{n-1}$ 의 계수는 $a_0 b_{n-1} + a_1 b_{n-2} + \cdots + a_{n-1} b_0$ 이다. 

등식 $(1+x)^{2n-1} = (1+x)^{n-1} (1+x)^n$ 의 좌변에서 $x^{n-1}$ 의 계수는 ${}_{2n-1} \mathrm{C}_{n-1}$ 이다. 
따라서 (가) $= {}_{2n-1} \mathrm{C}_{n-1}$ 이다.

우변 $(1+x)^{n-1} (1+x)^n$ 에서 $x^{n-1}$ 의 계수를 구하기 위해 $(1+x)^{n-1} = \sum_{k=1}^{n} {}_{n-1} \mathrm{C}_{k-1} x^{k-1}$ 와 $(1+x)^n = \sum_{j=0}^{n} {}_n \mathrm{C}_j x^j$ 를 이용한다. 
$x^{n-1}$ 이 되기 위해서는 $x^{k-1}$ 와 $x^{n-k}$ 가 곱해져야 하므로, 계수는 $\sum_{k=1}^{n} ( {}_{n-1} \mathrm{C}_{k-1} \times {}_n \mathrm{C}_{n-k} )$ 이다.
따라서 (나) $= {}_n \mathrm{C}_{n-k}$ 이다.

$1 \le k \le n$ 일 때, $k \times {}_n \mathrm{C}_k = n \times {}_{n-1} \mathrm{C}_{k-1}$ 임을 이용하면 
$$ \begin{aligned} \sum_{k=1}^{n} k ( {}_n \mathrm{C}_k )^2 &= \sum_{k=1}^n (n \times {}_{n-1} \mathrm{C}_{k-1} \times {}_n \mathrm{C}_{n-k}) \\ &= n \times \sum_{k=1}^n ( {}_{n-1} \mathrm{C}_{k-1} \times {}_n \mathrm{C}_{n-k} ) \\ &= n \times {}_{2n-1} \mathrm{C}_{n-1} \end{aligned} $$
이다. 이를 정리하면
$$ n \times {}_{2n-1} \mathrm{C}_{n-1} = n \times \frac{(2n-1)!}{(n-1)! n!} = \frac{n}{2n} \times \frac{(2n)!}{(n-1)! n!} \times \frac{2n}{n} = \frac{n}{2} \times \frac{(2n)!}{n! n!} = \frac{n}{2} \times {}_{2n} \mathrm{C}_n $$
따라서 (다) $= \frac{n}{2} \times {}_{2n} \mathrm{C}_n$ 이다.

정답은 ③ 이다.
