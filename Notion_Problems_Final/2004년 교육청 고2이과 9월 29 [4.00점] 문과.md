## 문제
수열 $\{ a_{n} \}$을 다음과 같이 정의한다.
$$ a_{1} = \frac{1}{2}, a_{n+1} = \frac{1}{1 - a_{n}} (n = 1, 2, 3, \cdots) $$
이때, $a_{1} + a_{2} + a_{3} + \cdots + a_{k} = 6$을 만족하는 자연수 $k$의 값을 구하시오.

## 풀이
$a_{1} = \frac{1}{2}$
$a_{2} = \frac{1}{1 - a_{1}} = \frac{1}{1 - \frac{1}{2}} = 2$
$a_{3} = \frac{1}{1 - a_{2}} = \frac{1}{1 - 2} = -1$
$a_{4} = \frac{1}{1 - a_{3}} = \frac{1}{1 - (-1)} = \frac{1}{2}$
$\vdots$
따라서 수열 $\{ a_{n} \}$은 $\frac{1}{2}$, $2$, $-1$이 반복된다.

한 주기의 합은 $a_{1} + a_{2} + a_{3} = \frac{1}{2} + 2 + (-1) = \frac{3}{2}$이고
$6 = \frac{3}{2} \times 4$이므로
$k = 3 \times 4 = 12$이다.

정답: $12$
