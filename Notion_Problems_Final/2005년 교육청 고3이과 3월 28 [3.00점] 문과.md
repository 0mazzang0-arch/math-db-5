## 문제
어떤 음료 회사는 사은행사로 음료수를 구입할 때 경품을 주기로 하고, 컵 $1$ 개, 컵 $2$ 개, 다음 기회에 중 하나의 문구를 병뚜껑의 안쪽에 써 넣었다. 컵 $1$ 개가 나올 확률은 $\frac{p}{10}$, 컵 $2$ 개가 나올 확률은 $\frac{p}{100}$, 다음 기회에가 나올 확률은 $p$ 이다. 이와 같은 행사에서 음료수 $3$ 병을 구입하였을 때, 경품으로 $3$ 개의 컵을 받을 확률은?

## 보기
① $\frac{3}{1000} p^{3}$   ② $\frac{7}{1000} p^{3}$   ③ $\frac{9}{1000} p^{3}$   ④ $\frac{11}{1000} p^{3}$   ⑤ $\frac{13}{1000} p^{3}$

## 풀이
[정답] ② 

컵이 $1$ 개, $2$ 개, $0$ 개인 개수를 각각 $x, y, z$ 로 두면 $1 \times x + 2 \times y + 0 \times z = 3$ 이다. 
$x, y, z$ 는 $0$ 이상인 정수이므로 
(i) $(x, y, z) = (3, 0, 0)$ 일 때, 
${}_{3}C_{3} \left( \frac{p}{10} \right)^{3} = \frac{p^{3}}{1000}$ 
(ii) $(x, y, z) = (1, 1, 1)$ 일 때 
${}_{3}C_{1} \left( \frac{p}{10} \right)^{1} \times {}_{2}C_{1} \left( \frac{p}{100} \right)^{1} \times {}_{1}C_{1} p^{1} = \frac{6}{1000} p^{3}$ 
따라서 $\left( \frac{1}{1000} + \frac{6}{1000} \right) p^{3} = \frac{7}{1000} p^{3}$
