## 문제
부등식 $(x-1)(x-2) \le 0$과 해가 같은 것을 <보기>에서 모두 고른 것은?

$$ 
\begin{array}{l} 
ㄱ.\ (x-2)(x+1)^{2}(x-1)^{3} \le 0 \\ 
ㄴ.\ \frac{x-2}{x-1} \le 0 \\ 
ㄷ.\ \frac{(x-1)(x-2)}{x^{2}} \le 0 
\end{array} 
$$

## 보기
① $ㄱ$   ② $ㄷ$   ③ $ㄴ$   ④ $ㄱ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
부등식 $(x-1)(x-2) \le 0$의 해는 $1 \le x \le 2$이므로

ㄱ. $(x-2)(x+1)^{2}(x-1)^{3} \le 0$에서
(ⅰ) $x = -1$일 때 등호가 성립한다.
(ⅱ) $x \ne -1$일 때 $(x+1)^{2} > 0$이므로 $(x-2)(x-1)^{3} \le 0 \iff 1 \le x \le 2$
(ⅰ), (ⅱ)에 의하여 $x = -1$ 또는 $1 \le x \le 2$

ㄴ. $\frac{x-2}{x-1} \le 0$의 양변에 $(x-1)^{2}$을 곱하면 
$(x-1)(x-2) \le 0$, $x \ne 1$
$\therefore 1 < x \le 2$

ㄷ. $x^{2} > 0$이므로 $\frac{(x-1)(x-2)}{x^{2}} \le 0 \iff (x-1)(x-2) \le 0$
$\therefore 1 \le x \le 2$

따라서 주어진 부등식과 해가 같은 것은 $ㄷ$뿐이다.

정답: ②
