## 문제
두 복소수 $z_1 = a + b i$, $z_2 = c + d i$에 대하여 $a$, $b$, $c$, $d$는 자연수이고 $z_1 \overline{z_1} = 10$일 때, <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $i = \sqrt{-1}$이고, $\overline{z}$는 복소수 $z$의 켤레복소수이다.)

## 보기
ㄱ. $a^2 + b^2 = 10$
ㄴ. $z_1 + \overline{z_2} = 3$이면 $c + d = 5$이다.
ㄷ. $(z_1 + z_2)(\overline{z_1 + z_2}) = 41$이면 $z_2 \overline{z_2}$의 최댓값은 $17$이다.

① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
[정답] ⑤

ㄱ. 
$z_1 \overline{z_1} = (a + bi)(a - bi) = a^2 + b^2 = 10$ (참)

ㄴ. 
$a^2 + b^2 = 10$에서 $a$, $b$가 자연수일 때,
$a = 1$이면 $b = 3$이다.
$a = 2$이면 $b^2 = 6$인 자연수 $b$는 존재하지 않는다.
$a = 3$이면 $b = 1$이다.
$a \ge 4$이면 $a^2 \ge 16$이므로 자연수 $b$는 존재하지 않는다.

$z_1 + \overline{z_2} = (a + bi) + (c - di) = (a + c) + (b - d)i = 3$
이때 $a, c$가 자연수이므로 $a + c = 3$, $b - d = 0$이다.
$a + c = 3$에서 $a$는 $3$보다 작은 자연수이어야 하므로 $a = 1$ 또는 $a = 2$이다.
위의 조건에서 $a = 1$일 때만 $b$가 자연수로 존재하므로
$a = 1$, $c = 2$, $b = 3$이 된다.
$b = d$이므로 $d = 3$이다.
따라서 $c + d = 2 + 3 = 5$ (참)

ㄷ. 
$(z_1 + z_2)(\overline{z_1 + z_2})$
$= \{(a + c) + (b + d)i\}\{(a + c) - (b + d)i\}$
$= (a + c)^2 + (b + d)^2 = 41$
$a, b, c, d$가 자연수이므로 $a + c \ge 2$, $b + d \ge 2$이다.
제곱수의 합이 $41$이 되는 경우는 $16 + 25 = 41$뿐이다.

$a + c = 2$이면 $(b + d)^2 = 37$인 자연수 $b + d$는 존재하지 않는다.
$a + c = 3$이면 $(b + d)^2 = 32$인 자연수 $b + d$는 존재하지 않는다.
$a + c = 4$이면 $b + d = 5$이다.
$a + c = 5$이면 $b + d = 4$이다.
$a + c = 6$이면 $(b + d)^2 = 5$인 자연수 $b + d$는 존재하지 않는다.
$a + c \ge 7$이면 $(a + c)^2 \ge 49$이므로 자연수 $b + d$는 존재하지 않는다.

(ⅰ) $a + c = 4$, $b + d = 5$일 때
$a = 1$이면 $b = 3$이므로 $c = 3$, $d = 2$이다. 이때 $z_2 \overline{z_2} = 3^2 + 2^2 = 13$
$a = 3$이면 $b = 1$이므로 $c = 1$, $d = 4$이다. 이때 $z_2 \overline{z_2} = 1^2 + 4^2 = 17$

(ⅱ) $a + c = 5$, $b + d = 4$일 때
$a = 1$이면 $b = 3$이므로 $c = 4$, $d = 1$이다. 이때 $z_2 \overline{z_2} = 4^2 + 1^2 = 17$
$a = 3$이면 $b = 1$이므로 $c = 2$, $d = 3$이다. 이때 $z_2 \overline{z_2} = 2^2 + 3^2 = 13$

(ⅰ), (ⅱ)에 의하여 $z_2 \overline{z_2} = c^2 + d^2$의 값은 $13$ 또는 $17$이므로
$z_2 \overline{z_2}$의 최댓값은 $17$이다. (참)

따라서 옳은 것은 ㄱ, ㄴ, ㄷ이다.
