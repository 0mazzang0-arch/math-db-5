## 문제
$\int_{2 \pi}^{3 \pi} x \sin x dx$ 의 값은?

## 보기
① $\pi$   ② $2 \pi$   ③ $3 \pi$   ④ $4 \pi$   ⑤ $5 \pi$

## 풀이
부분적분법에 의해 
$\int x \sin x dx = -x \cos x - \int -\cos x dx$
$= -x \cos x + \sin x + C$
(단, $C$ 는 적분상수)

아래끝, 위끝을 대입하여 정적분을 계산하면
$\therefore \int_{2 \pi}^{3 \pi} x \sin x dx = [-x \cos x + \sin x]_{2 \pi}^{3 \pi}$
$= -3 \pi \cdot (-1) + 0 - (-2 \pi) \cdot 1 - 0 = 5 \pi$

정답: ⑤
