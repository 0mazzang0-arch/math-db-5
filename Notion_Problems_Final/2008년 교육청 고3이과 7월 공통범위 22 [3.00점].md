## 문제
두 집합 $A = \{ x \mid (x - 2)(x - 4)(x + 1)^2 < 0 \}$, $B = \left\{ x \mid \dfrac{(x + 1)^3}{x - a} > 0 \right\}$ 에 대하여 $A \subset B$ 일 때, 상수 $a$ 의 최댓값을 $M$ 이라 하자. $7M$ 의 값을 구하시오.

## 보기

## 풀이
$A = \{ x \mid 2 < x < 4 \}$ 이고
$i)$ $a \le -1$ 이면 
$B = \{ x \mid x < a \text{ 또는 } x > -1 \}$
$\therefore A \subset B$
$ii)$ $a > -1$ 이면 
$B = \{ x \mid x < -1 \text{ 또는 } x > a \}$
$A \subset B$ 가 되려면 $a \le 2$ 이어야 하므로
$-1 < a \le 2$

$i)$, $ii)$ 에 의해 $a \le 2$ 이므로 상수 $a$ 의 최댓값 $M = 2$ 이다.
따라서 $7M = 7 \times 2 = 14$

[정답] $14$
