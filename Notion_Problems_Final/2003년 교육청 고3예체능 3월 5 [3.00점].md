## 문제
두 조건 $p : |x - a| \leq 2$, $q : |x - 1| \leq 3$ 에 대하여 $p$ 가 $q$ 이기 위한 충분조건이 되도록 상수 $a$ 의 값을 정할 때, $a$ 의 최댓값은?

## 보기
① $-2$   ② $-1$   ③ $0$   ④ $1$   ⑤ $2$

## 풀이
조건 $p$, $q$ 의 진리집합을 각각 $P$, $Q$ 라 하면 $p$ 가 $q$ 이기 위한 충분조건이 되기 위하여 $P \subset Q$ 이어야 한다. 
이때, $P = \{ x \mid a - 2 \leq x \leq a + 2 \}$, $Q = \{ x \mid -2 \leq x \leq 4 \}$ 이므로 

> [!example] 📸 (그림/그래프)

$a - 2 \geq -2$, $a + 2 \leq 4$ 
즉, $a \geq 0$, $a \leq 2 \therefore 0 \leq a \leq 2$ 
따라서 $a$ 의 최댓값은 $2$ 이다.
