## 문제
등차수열 $\{ a_n \}$ 에 대하여 $a_1 + a_{10} = 22$ 일 때, $\sum_{k=2}^{9} a_k$ 의 값을 구하시오.

## 풀이
등차수열 $\{ a_n \}$ 의 공차를 $d$ 라 하면 
$a_1 + a_{10} = 22$ 에서 
$a_1 + (a_1 + 9d) = 2a_1 + 9d = 22$ 
따라서 
$\sum_{k=2}^{9} a_k = \sum_{k=1}^{9} a_k - a_1$ 
$= \frac{9(2a_1 + 8d)}{2} - a_1$ 
$= \frac{18a_1 + 72d - 2a_1}{2}$ 
$= \frac{16a_1 + 72d}{2}$ 
$= \frac{8(2a_1 + 9d)}{2}$ 
$= \frac{8 \times 22}{2}$ 
$= 88$ 

**[다른 풀이]** 
수열 $\{ a_n \}$ 이 등차수열이므로 
$a_1 + a_{10} = a_2 + a_9 = a_3 + a_8 = a_4 + a_7 = a_5 + a_6$ 
따라서 조건에서 $a_1 + a_{10} = 22$ 이므로 
$\sum_{k=2}^{9} a_k = (a_2 + a_9) + (a_3 + a_8) + (a_4 + a_7) + (a_5 + a_6)$ 
$= 22 + 22 + 22 + 22$ 
$= 88$
