## 문제
다음은 $r$가 자연수일 때 $n \ge r$인 모든 자연수 $n$에 대하여 등식 
$$\sum_{i=r}^{n} {}_{i}C_{r} = {}_{n+1}C_{r+1}$$
이 성립함을 수학적귀납법으로 증명하는 과정이다.

($1$) $n=r$일 때 
(좌변) $= {}_{r}C_{r} =$ (가), (우변) $= {}_{r+1}C_{r+1} =$ (가)

($2$) $n=k$ ($k \ge r$)일 때 주어진 등식이 성립한다고 가정하면 
$$\sum_{i=r}^{k} {}_{i}C_{r} = {}_{k+1}C_{r+1}$$
이다. $n=k+1$일 때 성립함을 보이자.

$$\sum_{i=r}^{k+1} {}_{i}C_{r} = {}_{k+1}C_{r+1} + \text{(나)}$$
$$= \text{(다)} + \frac{(k+1)!}{(k+1-r)!r!} = \frac{(k+2)!}{(k+1-r)!(r+1)!}$$
$$= {}_{k+2}C_{r+1}$$

따라서 $n=k+1$일 때 주어진 등식이 성립한다.
($1$), ($2$)에 의하여 $n \ge r$인 모든 자연수 $n$에 대하여 주어진 등식이 성립한다.

위 증명에서 (가), (나), (다)에 알맞은 것은?

## 보기
① (가) : $1$, (나) : ${}_{k+1}C_{r}$, (다) : $\frac{(k+1)!}{(k-r)!(r+1)!}$   ② (가) : $1$, (나) : ${}_{k+1}C_{r}$, (다) : $\frac{(k+1)!}{(k+1-r)!r!}$   ③ (가) : $1$, (나) : ${}_{k+1}C_{r+1}$, (다) : $\frac{(k+1)!}{(k-r)!(r+1)!}$   ④ (가) : $r$, (나) : ${}_{k+1}C_{r+1}$, (다) : $\frac{(k+1)!}{(k+1-r)!r!}$   ⑤ (가) : $r$, (나) : ${}_{k+1}C_{r}$, (다) : $\frac{(k+1)!}{(k+1-r)!r!}$

## 풀이
정답 ①

(가) 
$${}_{r}C_{r} = {}_{r+1}C_{r+1} = 1$$

(나) 
$$\sum_{i=r}^{k+1} {}_{i}C_{r} = \sum_{i=r}^{k} {}_{i}C_{r} + {}_{k+1}C_{r} = {}_{k+1}C_{r+1} + {}_{k+1}C_{r}$$ 
이므로 (나) $= {}_{k+1}C_{r}$

(다) 
$${}_{n}C_{r} = \frac{n!}{(n-r)!r!}$$
이므로
$${}_{k+1}C_{r+1} + {}_{k+1}C_{r} = \frac{(k+1)!}{((k+1)-(r+1))!(r+1)!} + \frac{(k+1)!}{(k+1-r)!r!}$$
$$= \frac{(k+1)!}{(k-r)!(r+1)!} + \frac{(k+1)!}{(k+1-r)!r!}$$
따라서 (다) $= \frac{(k+1)!}{(k-r)!(r+1)!}$

그러므로 
(가) $= 1$, (나) $= {}_{k+1}C_{r}$, (다) $= \frac{(k+1)!}{(k-r)!(r+1)!}$
