## 문제
수열 $\{ a_{n} \}$ 에 대하여 $\sum_{k=1}^{10} a_{2k-1} = 12$, $\sum_{k=1}^{10} a_{2k} = 20$ 일 때, $\sum_{k=1}^{20} ( -1 )^{k} a_{k}$ 의 값을 구하시오.

## 풀이
$\sum_{k=1}^{20} ( -1 )^{k} a_{k}$
$= \sum_{k=1}^{10} \{ ( -1 )^{2k-1} a_{2k-1} + ( -1 )^{2k} a_{2k} \}$
$= \sum_{k=1}^{10} ( -a_{2k-1} + a_{2k} )$
$= - \sum_{k=1}^{10} a_{2k-1} + \sum_{k=1}^{10} a_{2k}$
$= ( -12 ) + 20$
$= 8$

정답: $8$
