## 문제
두 함수 $f(x)$, $g(x)$를 $f(x)=x^{2}-6x+3$, $g(x)=a^{x}$ ($a>0$, $a \neq 1$)이라 하자. $1 \le x \le 4$에서 함수 $(g \circ f)(x)$의 최댓값은 $27$, 최솟값은 $m$이다. $m$의 값은?

## 보기
① $\frac{1}{27}$   ② $\frac{1}{3}$   ③ $\frac{\sqrt{3}}{3}$   ④ $3$   ⑤ $3\sqrt{3}$

## 풀이
두 함수 $f(x)$, $g(x)$를 $f(x)=x^{2}-6x+3$, $g(x)=a^{x}$ ($a>0$, $a \neq 1$)이라 하자.
$f(x) = (x-3)^{2}-6$이므로 $1 \le x \le 4$에서 $f(x)$의 값의 범위는 다음과 같다.
$x=3$일 때 최솟값 $f(3)=-6$
$x=1$일 때 $f(1)=1-6+3=-2$
$x=4$일 때 $f(4)=16-24+3=-5$
즉, $-6 \le f(x) \le -2$이다.

합성함수 $(g \circ f)(x) = a^{f(x)}$의 최댓값이 $27$이므로
(i) $a>1$인 경우
$f(x)=-2$일 때 최댓값 $a^{-2}=27$이어야 한다. 그러나 $a>1$이면 $a^{-2} < 1$이므로 $27$이 될 수 없어 모순이다.

(ii) $0 < a < 1$인 경우
$f(x)=-6$일 때 최댓값 $a^{-6}=27$이다.
$a^{-6} = (a^{-2})^{3} = 3^{3}$에서 $a^{-2}=3$이다.
이때 최솟값 $m$은 $f(x)=-2$일 때의 값이므로
$m = a^{-2} = 3$이다.

따라서 구하는 최솟값 $m$의 값은 $3$이다.
