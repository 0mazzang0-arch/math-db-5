## 문제
다음 함수 중에서 그 그래프 위의 서로 다른 세 점 $A(a, p)$, $B(b, q)$, $C(c, r)$를 선택하되, $x$ 좌표 $a, b, c$는 차례로 등차수열을 이루고, $y$ 좌표 $p, q, r$는 차례로 등비수열을 이루게 할 수 있는 것만을 <보기>에서 있는 대로 고른 것은?

## 보기
$$ 
\begin{array}{|l|} 
\hline 
ㄱ. f(x) = x \\ 
ㄴ. g(x) = \frac{1}{x} \\ 
ㄷ. h(x) = 2^{x} \\ 
\hline 
\end{array} 
$$

① $ㄱ$   ② $ㄴ$   ③ $ㄷ$   ④ $ㄱ, ㄴ$   ⑤ $ㄴ, ㄷ$

## 풀이
[정답] ③ 

세 점 $A(a, p)$, $B(b, q)$, $C(c, r)$에서 $a, b, c$는 차례로 등차수열을 이루고, $p, q, r$는 차례로 등비수열을 이루므로 
$$2b = a + c \quad \cdots \cdots ㉠$$ 
$$q^2 = pr \quad \cdots \cdots ㉡$$ 

ㄱ. $f(x) = x$ 위의 세 점을 잡으면 $A(a, a)$, $B(b, b)$, $C(c, c)$이므로 ㉡은 
$$b^2 = ac \quad \cdots \cdots ㉢$$ 
㉠의 양변을 제곱하면 
$$4b^2 = a^2 + 2ac + c^2$$ 
㉢을 대입하면 
$$4ac = a^2 + 2ac + c^2$$ 
$$(a - c)^2 = 0$$ 
$\therefore a = c$ 
그런데, 문제에서 $a \neq c$이므로 조건을 만족하는 함수가 아니다. 

ㄴ. $g(x) = \frac{1}{x}$ 위의 세 점을 잡으면 $A(a, \frac{1}{a})$, $B(b, \frac{1}{b})$, $C(c, \frac{1}{c})$이므로 ㉡은 
$$\frac{1}{b^2} = \frac{1}{ac} \quad \cdots \cdots ㉢$$ 
즉, $b^2 = ac$이므로 $a = c$ ($\because ㉠$)가 되어 문제의 조건을 만족하는 함수가 아니다. 

ㄷ. $h(x) = 2^x$ 위의 세 점을 잡으면 $A(a, 2^a)$, $B(b, 2^b)$, $C(c, 2^c)$이므로 ㉡은 
$$(2^b)^2 = 2^a \cdot 2^c$$, 즉 
$$2^{2b} = 2^{a+c}$$ 
$\therefore 2b = a + c$ 
따라서, ㉠과 같은 조건이 되므로 $2b = a + c$를 만족하는 모든 실수 $a, b, c$에 대하여 문제의 조건이 성립한다. 
(예. $a = 1, b = 2, c = 3$) 
그러므로 조건을 만족하는 것은 ㄷ뿐이다.
