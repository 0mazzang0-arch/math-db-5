## 문제
$A = \cos \frac{2}{5} \pi$, $B = \sin \frac{2}{5} \pi$, $C = \cos ( \sin \frac{2}{5} \pi )$, $D = \sin ( \sin \frac{2}{5} \pi )$ 일 때, $A$, $B$, $C$, $D$ 의 대소 관계를 나타내면?

## 보기
① $A < C < D < B$   ② $A < D < C < B$   ③ $C < A < B < D$   ④ $D < C < B < A$   ⑤ $C < D < A < B$

## 풀이
> [!example] 📸 (그림/그래프)

위의 그림에서 알 수 있듯이 각도가 $\frac{\pi}{4} < x < \frac{\pi}{2}$ 인 경우 $y = \sin x$ 는 증가하고, $y = \cos x$ 는 감소한다. 또한 $\sin x$ 값이 $\cos x$ 의 값보다 항상 크다.

$\sin \frac{2}{5} \pi$ 는 $\frac{2}{5} \pi$ 가 $\frac{\pi}{3}$ 와 $\frac{\pi}{2}$ 사이의 각이므로
$\sin \frac{\pi}{3} \left( = \frac{\sqrt{3}}{2} \right) < \sin \frac{2}{5} \pi < \sin \frac{\pi}{2} (= 1)$
즉, $\frac{\sqrt{3}}{2} < B < 1$ 이다.

마찬가지로 $\cos \frac{2}{5} \pi$ 에서
$\cos \frac{\pi}{2} (= 0) < \cos \frac{2}{5} \pi < \cos \frac{\pi}{3} \left( = \frac{1}{2} \right)$
즉, $0 < A < \frac{1}{2}$ 이다.

$C = \cos B$, $D = \sin B$ 에서 $\frac{\pi}{4} < \frac{\sqrt{3}}{2} < B < 1 < \frac{\pi}{2}$ 이므로
$\sin B > \cos B$ 에서 $D > C$ 이다.

또한, $B < 1 < \frac{\pi}{3}$ 이고 $\cos$ 함수는 $0 < x < \frac{\pi}{2}$ 에서 감소하므로
$\cos B > \cos \frac{\pi}{3} = \frac{1}{2}$
따라서 $C > \frac{1}{2}$ 이 되어 $C > A$ 가 성립한다.

마지막으로 $D = \sin B < B$ 이므로 전체적인 대소 관계는 다음과 같다.
$A < C < D < B$
