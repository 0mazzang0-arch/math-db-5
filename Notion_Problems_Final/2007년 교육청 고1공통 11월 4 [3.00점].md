## 문제
두 집합 $A = \{ a, b, c \}$, $B = \{ a, b, c, d \}$ 에 대하여 $B \cup X = B$, $n(A \cap X) = 2$ 를 만족시키는 집합 $X$ 의 개수는? (단, $n(A)$ 는 집합 $A$ 의 원소의 개수이다.)

## 보기
① $4$   ② $6$   ③ $8$   ④ $10$   ⑤ $12$

## 풀이
$B \cup X = B$ 이므로 $X \subset B$ 이다.
$n(A \cap X) = 2$ 이므로 집합 $X$ 는 집합 $A$ 의 원소 $\{ a, b, c \}$ 중 $2$ 개를 포함해야 한다.
이때, 집합 $B$ 의 원소 중 $A$ 에 속하지 않는 원소는 $\{ d \}$ 이다.
조건을 만족시키는 집합 $X$ 는 다음과 같다.
$\{ a, b \}$, $\{ a, c \}$, $\{ b, c \}$, $\{ a, b, d \}$, $\{ a, c, d \}$, $\{ b, c, d \}$
$\therefore 6$ 개
