## 문제
이차함수 $f(x) = x^{2} + 1$과 두 실수 $x_{1}$, $x_{2}$에 대하여 옳은 것만을 <보기>에서 있는 대로 고른 것은?

$$
\begin{array}{|l|}
\hline
\text{<보기>} \\
ㄱ. \ 0 < x_{1} < x_{2} \text{ 이면 } 2^{f(x_{1})} < 2^{f(x_{2})} \text{ 이다.} \\
ㄴ. \ x_{1} < x_{2} < 0 \text{ 이면 } \log_{2} f(x_{1}) > \log_{2} f(x_{2}) \text{ 이다.} \\
ㄷ. \ x_{1} < 0 < x_{2} \text{ 이면 } \log_{\frac{1}{2}} f(x_{1}) < \log_{\frac{1}{2}} f(x_{2}) \text{ 이다.} \\
\hline
\end{array}
$$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
ㄱ. $0 < x_{1} < x_{2}$ 이면 $f(x_{1}) < f(x_{2})$ 이므로 $2^{f(x_{1})} < 2^{f(x_{2})}$ (참)

ㄴ. $x_{1} < x_{2} < 0$ 이면 $f(x_{1}) > f(x_{2}) > 1$ 이므로 $\log_{2} f(x_{1}) > \log_{2} f(x_{2})$ (참)

ㄷ. (반례) $x_{1} = -2$, $x_{2} = 3$ 이면 $f(x_{1}) = 5$, $f(x_{2}) = 10$ 이다.
밑이 $1$보다 작은 로그함수 $y = \log_{\frac{1}{2}} x$는 감소함수이므로 $f(x_{1}) < f(x_{2})$ 일 때 $\log_{\frac{1}{2}} f(x_{1}) > \log_{\frac{1}{2}} f(x_{2})$ 이다.
즉, $\log_{\frac{1}{2}} f(-2) > \log_{\frac{1}{2}} f(3)$ 이므로 (거짓)

정답: ③
