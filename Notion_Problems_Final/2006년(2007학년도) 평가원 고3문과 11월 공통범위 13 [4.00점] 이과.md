## 문제
정수 $n$에 대하여 두 집합 $A(n)$, $B(n)$이 
$$ A(n) = \{ x | \log_{2} x \le n \} $$
$$ B(n) = \{ x | \log_{4} x \le n \} $$
일 때, 다음 <보기>에서 옳은 것만을 있는 대로 고른 것은?

---
**<보 기>**
ㄱ. $A(1) = \{ x | 0 < x \le 1 \}$
ㄴ. $A(4) = B(2)$
ㄷ. $A(n) \subset B(n)$일 때, $B(-n) \subset A(-n)$이다.
---

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄱ, ㄷ$   ⑤ $ㄴ, ㄷ$

## 풀이
두 집합 $A(n)$, $B(n)$을 조건제시법의 정의에 따라 정리하면 다음과 같다.
$A(n) = \{ x | 0 < x \le 2^{n} \}$, 
$B(n) = \{ x | 0 < x \le 4^{n} \}$ 이다.

ㄱ. $2^{1} = 2$이므로 $A(1) = \{ x | 0 < x \le 2 \}$이다. 따라서 $A(1) = \{ x | 0 < x \le 1 \}$은 옳지 않다. (거짓)

ㄴ. $2^{4} = 16$이고 $4^{2} = 16$이므로,
$A(4) = \{ x | 0 < x \le 16 \}$
$B(2) = \{ x | 0 < x \le 16 \}$
즉, $A(4) = B(2)$이다. (참)

ㄷ. $A(n) \subset B(n)$이면 $2^{n} \le 4^{n}$이 성립해야 하므로 $n \ge 0$이다.
이때 $B(-n)$의 원소의 범위는 $0 < x \le 4^{-n}$이고, $A(-n)$의 원소의 범위는 $0 < x \le 2^{-n}$이다.
$n \ge 0$일 때 $4^{-n} \le 2^{-n}$이므로 $B(-n) \subset A(-n)$이 성립한다. (참)

이상에서 옳은 것은 $ㄴ, ㄷ$이다.
