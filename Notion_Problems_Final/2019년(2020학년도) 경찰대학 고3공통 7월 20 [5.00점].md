## 문제
두 양수 $a$, $b$가 $\frac{1}{a} + \frac{1}{b} \le 4$, $(a - b)^{2} = 16(ab)^{3}$을 만족할 때, $a + b$의 값은?

## 보기
① $1$   ② $\sqrt{2}$   ③ $2$   ④ $2\sqrt{2}$   ⑤ $4$

## 풀이
[정답] ③

$\frac{1}{a} + \frac{1}{b} = \frac{a+b}{ab} \le 4$에서 $a+b \le 4ab$, 즉 $(a+b)^2 \le 16(ab)^2$
$(a+b)^2 = (a-b)^2 + 4ab = 16(ab)^3 + 4(ab) \le 16(ab)^2$
$ab = t$라 두면 $16t^3 + 4t \le 16t^2$, $4t^2 + 1 \le 4t$, 즉 $(2t - 1)^2 \le 0$
따라서 $ab = t = \frac{1}{2}$, $(a+b)^2 = 16t^3 + 4t = 2 + 2 = 4$
$\therefore a+b = 2$
실제로 $ab = \frac{1}{2}$, $a+b = 2$라고 하면 $x^2 - 2x + \frac{1}{2} = 0$으로부터 $a$, $b$는 $1 + \frac{1}{\sqrt{2}}$, $1 - \frac{1}{\sqrt{2}}$이고, 주어진 조건을 모두 만족시킨다.
