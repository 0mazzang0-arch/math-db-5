## 문제
$0 \le x < 2\pi$ 일 때, 방정식 $\cos 2x + 3\sin x = 2$ 의 모든 실근의 합은?

## 보기
① $\frac{\pi}{2}$   ② $\pi$   ③ $\frac{3}{2}\pi$   ④ $2\pi$   ⑤ $\frac{5}{2}\pi$

## 풀이
$\cos 2x + 3\sin x = 2$
$(1 - 2\sin^2 x) + 3\sin x = 2$
$2\sin^2 x - 3\sin x + 1 = 0$
$(2\sin x - 1)(\sin x - 1) = 0$

$0 \le x < 2\pi$ 이므로
$\sin x = \frac{1}{2}$ 또는 $\sin x = 1$
따라서 
$x = \frac{\pi}{6}$ 또는 $x = \frac{\pi}{2}$ 또는 $x = \frac{5}{6}\pi$

모든 실근의 합은 
$\frac{\pi}{6} + \frac{\pi}{2} + \frac{5}{6}\pi = \frac{3}{2}\pi$

정답: ③
