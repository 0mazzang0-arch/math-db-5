## 문제
$\tan \alpha = -\frac{5}{12}$ ($\frac{3}{2} \pi < \alpha < 2 \pi$)이고 $0 \le x < \frac{\pi}{2}$일 때, 부등식 
$\cos x \le \sin (x + \alpha) \le 2 \cos x$
를 만족시키는 $x$에 대하여 $\tan x$의 최댓값과 최솟값의 합은?

## 보기
① $\frac{31}{12}$   ② $\frac{37}{12}$   ③ $\frac{43}{12}$   ④ $\frac{49}{12}$   ⑤ $\frac{55}{12}$

## 풀이
$\frac{3}{2} \pi < \alpha < 2 \pi$에서 $\tan \alpha = -\frac{5}{12}$이므로
$\sin \alpha = -\frac{5}{13}$, $\cos \alpha = \frac{12}{13}$이다.
$\sin (x + \alpha)$를 전개하면
$\sin (x + \alpha) = \sin x \cos \alpha + \cos x \sin \alpha = \frac{12}{13} \sin x - \frac{5}{13} \cos x$
주어진 부등식 $\cos x \le \sin (x + \alpha) \le 2 \cos x$에 대입하면
$\cos x \le \frac{12}{13} \sin x - \frac{5}{13} \cos x \le 2 \cos x$
$0 \le x < \frac{\pi}{2}$일 때 $\cos x > 0$이므로 양변을 $\cos x$로 나누면
$1 \le \frac{12}{13} \tan x - \frac{5}{13} \le 2$
각 변에 $\frac{5}{13}$를 더하면
$\frac{18}{13} \le \frac{12}{13} \tan x \le \frac{31}{13}$
$\frac{18}{12} \le \tan x \le \frac{31}{12}$
$\frac{3}{2} \le \tan x \le \frac{31}{12}$
따라서 $\tan x$의 최댓값은 $\frac{31}{12}$, 최솟값은 $\frac{3}{2}$이다.
최댓값과 최솟값의 합은
$\frac{31}{12} + \frac{3}{2} = \frac{31 + 18}{12} = \frac{49}{12}$

정답: ④
