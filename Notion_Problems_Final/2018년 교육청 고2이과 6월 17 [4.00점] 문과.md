## 문제
$a_{1} = 1$, $a_{2} = -1$, $a_{3} = 4$ 인 수열 $\{ a_{n} \}$ 이 모든 자연수 $n$ 에 대하여
$$n ( n - 2 ) a_{n+1} = \sum_{i=1}^{n} a_{i}$$
를 만족시킨다. 다음은
$$a_{n} = \frac{8}{( n - 1 ) ( n - 2 )} \quad ( n \ge 3 )$$
임을 수학적 귀납법으로 증명한 것이다.

$$
\begin{array}{|l|}
\hline
(ⅰ) \ n = 3 \text{ 일 때, } a_{3} = 4 = \frac{8}{( 3 - 1 ) ( 3 - 2 )} \text{ 이므로 성립한다.} \\
(ⅱ) \ n = k \ ( k \ge 3 ) \text{ 일 때, 성립한다고 가정하면} \\
\quad a_{k} = \frac{8}{( k - 1 ) ( k - 2 )} \\
\quad k ( k - 2 ) a_{k+1} = \sum_{i=1}^{k} a_{i} = a_{k} + \sum_{i=1}^{k-1} a_{i} \\
\quad = a_{k} + ( k - 1 ) ( k - 3 ) a_{k} \\
\quad = a_{k} \times ( 가 ) \\
\quad = \frac{8}{( k - 1 ) ( k - 2 )} \times ( 가 ) \\
\quad = \frac{( 나 )}{k - 1} \text{ 이다. 그러므로} \\
\quad a_{k+1} = \frac{1}{k ( k - 2 )} \times \frac{( 나 )}{k - 1} = \frac{8}{( 다 )} \\
\quad n = k + 1 \text{ 일 때 성립한다.} \\
(ⅰ), (ⅱ) \text{에 의하여 } n \ge 3 \text{ 인 모든 자연수 } n \text{ 에 대하여} \\
a_{n} = \frac{8}{( n - 1 ) ( n - 2 )} \text{ 이다.} \\
\hline
\end{array}
$$

위의 (가), (나), (다)에 알맞은 식을 각각 $f ( k )$, $g ( k )$, $h ( k )$ 라 할 때, $\frac{f ( 13 ) \times g ( 14 )}{h ( 12 )}$ 의 값은?

## 보기
① $88$   ② $96$   ③ $104$   ④ $112$   ⑤ $120$

## 풀이
(ⅰ) $n = 3$ 일 때, $a_{3} = 4 = \frac{8}{( 3 - 1 ) ( 3 - 2 )}$ 이므로 성립한다.
(ⅱ) $n = k$ ($k \ge 3$) 일 때, 성립한다고 가정하면 $a_{k} = \frac{8}{( k - 1 ) ( k - 2 )}$ 이다.
$( k - 1 ) ( k - 3 ) a_{k} = \sum_{i=1}^{k-1} a_{i}$ 임을 이용하면
$$k ( k - 2 ) a_{k+1} = a_{k} + ( k - 1 ) ( k - 3 ) a_{k}$$
$$= a_{k} \{ 1 + ( k - 1 ) ( k - 3 ) \}$$
$$= a_{k} ( k^{2} - 4k + 4 ) = a_{k} ( k - 2 )^{2}$$
이므로 $f ( k ) = ( k - 2 )^{2}$ 이다.

또한,
$$a_{k} \times ( k - 2 )^{2} = \frac{8}{( k - 1 ) ( k - 2 )} \times ( k - 2 )^{2} = \frac{8 ( k - 2 )}{k - 1}$$
이므로 $g ( k ) = 8 ( k - 2 )$ 이다.

마지막으로
$$a_{k+1} = \frac{1}{k ( k - 2 )} \times \frac{8 ( k - 2 )}{k - 1} = \frac{8}{k ( k - 1 )}$$
이므로 $h ( k ) = k ( k - 1 )$ 이다.

따라서 구하고자 하는 값은
$$\frac{f ( 13 ) \times g ( 14 )}{h ( 12 )} = \frac{( 13 - 2 )^{2} \times 8 ( 14 - 2 )}{12 ( 12 - 1 )} = \frac{11^{2} \times 8 \times 12}{12 \times 11} = 11 \times 8 = 88$$
정답은 ①이다.
