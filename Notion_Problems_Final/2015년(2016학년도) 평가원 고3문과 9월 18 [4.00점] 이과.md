## 문제
두 이차정사각행렬 $A$, $B$가 $A B - B^{2} = E$, $B^{3} + B = E$를 만족시킬 때, <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $E$는 단위행렬이다.)

$$
\begin{array}{|l|}
\hline
\text{ㄱ. } A B^{2} = E \\
\text{ㄴ. } A B = B A \\
\text{ㄷ. } A - B^{2} = B - E \\
\hline
\end{array}
$$

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
ㄱ. $A B - B^{2} = E$의 양변의 오른쪽에 $B$를 곱하면 $A B^{2} - B^{3} = B$이므로 
$A B^{2} = B^{3} + B$이다. 
조건에서 $B^{3} + B = E$이므로 $A B^{2} = E$이다. (참)

ㄴ. $A B - B^{2} = (A - B) B = E$이므로 $B$의 역행렬 $B^{-1}$이 존재하고, $B^{-1} = A - B$이다.
역행렬의 정의에 의해 $B (A - B) = E$이므로 $B A - B^{2} = E$이다.
$A B - B^{2} = B A - B^{2}$에서 $A B = B A$이다. (참)

ㄷ. $A B - B^{2} = E$와 $B^{3} + B = E$에서 $A B - B^{2} = B^{3} + B$가 성립한다.
ㄴ에서 $B^{-1}$이 존재하므로 양변의 오른쪽에 $B^{-1}$을 곱하면
$(A B - B^{2}) B^{-1} = (B^{3} + B) B^{-1}$
$A B B^{-1} - B^{2} B^{-1} = B^{3} B^{-1} + B B^{-1}$
$A - B = B^{2} + E$
따라서 $A - B^{2} = B + E$이다.
그러므로 $A - B^{2} = B - E$는 거짓이다. (거짓)

따라서 옳은 것은 ㄱ, ㄴ이다.
