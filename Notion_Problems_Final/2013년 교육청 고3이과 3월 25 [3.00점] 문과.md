## 문제
두 이차정사각행렬 $A$, $B$ 가 $A+B=8E$, $AB=2E$ 를 만족시킬 때, 행렬 $A^{-1} (A^{2}+B^{2} ) B^{-1}$ 의 모든 성분의 합을 구하시오. (단, $E$ 는 단위행렬이다.)

## 보기

## 풀이
$AB=BA$ 이므로 
$(A+B)^{2}=A^{2}+2AB+B^{2}$ 
따라서 $64E=A^{2}+4E+B^{2}$ 이므로 
$A^{2}+B^{2}=60E$ 
$A^{-1} (A^{2}+B^{2} ) B^{-1}$ 
$=A^{-1}(60E) B^{-1}$ 
$=60(BA)^{-1}$ 
$=60(2E)^{-1}$ 
$=30E$ 
따라서 구하는 행렬의 모든 성분의 합은 $60$ 이다. 

**[다른 풀이]**
$B=8E-A$ 를 $AB=2E$ 에 대입하면 
$A(8E-A)=2E$, $8A-A^{2}=2E$ 
$\therefore A^{2}=8A-2E$ 
마찬가지로 $B^{2}=8B-2E$ 
$\therefore A^{2}+B^{2}=(8A-2E)+(8B-2E)$ 
$=8(A+B)-4E$ 
$=8 \times (8E)-4E$ 
$=60E$ 
또, $AB=2E$ 에서 $AB=BA$ 이고 
$A^{-1}=\frac{1}{2}B$, $B^{-1}=\frac{1}{2}A$ 이므로 
$A^{-1} ( A^{2} + B^{2} ) B^{-1}$ 
$=\frac{1}{4}B (A^{2}+B^{2} ) A$ 
$= \frac{1}{4}B(60E)A$ 
$=15BA$ 
$=30E$ 
따라서 구하는 합은 $60$ 이다.
