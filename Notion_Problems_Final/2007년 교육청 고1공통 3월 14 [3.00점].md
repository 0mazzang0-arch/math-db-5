## 문제
다음은 $n$이 자연수일 때 $\sqrt{78-3n}$이 자연수가 되게 하는 $n$의 값을 구하는 과정이다.

$$
\begin{array}{|l|}
\hline
\text{자연수 } k \text{에 대하여 } \sqrt{78-3n}=k \text{라 하면 } 78-3n=k^{2} \text{이므로} \\
n=\frac{78-k^{2}}{3} \text{이다. 이 때, } n \text{은 자연수이므로 } 78-k^{2} \text{은 } 3 \text{의 배수이다.} \\
\text{따라서 } k \text{는 (가)의 배수이다.} \\
\text{또, } 78-k^{2}>0 \text{이므로 } k=\text{(나) 또는 } k=6 \text{이다.} \\
\text{그러므로 } n=\text{(다) 또는 } n=14 \text{이다.} \\
\hline
\end{array}
$$

위의 과정에서 (가), (나), (다)에 들어갈 수의 합은?

## 보기
① $27$   ② $29$   ③ $31$   ④ $33$   ⑤ $35$

## 풀이
자연수 $k$에 대하여 $\sqrt{78-3n}=k$라 하면 $78-3n=k^{2}$이므로 $n=\frac{78-k^{2}}{3}$이다. 
이때, $n$은 자연수이므로 $78-k^{2}$은 $3$의 배수이다. 
그런데 $78$은 $3$의 배수이므로 $k^{2}$도 $3$의 배수이다. 
따라서 $k$는 $3$의 배수이다. 
$\therefore$ (가) $= 3$

또, $78-k^{2}>0$이므로 $k^{2}<78$이다. 
$k$는 $3$의 배수인 자연수이므로 $k=3$ 또는 $k=6$이다. 
$\therefore$ (나) $= 3$

그러므로 구하는 자연수 $n$의 값은 
$k=3$일 때, $n=\frac{78-3^{2}}{3}=23$
$k=6$일 때, $n=\frac{78-6^{2}}{3}=14$
$\therefore$ (다) $= 23$

따라서 (가), (나), (다)에 들어갈 수의 합은 
$3+3+23=29$이다.

정답: ②
