## 문제
$\int_{0}^{\pi} x \cos (\pi - x) dx$의 값을 구하시오.

## 풀이
$\cos (\pi - x) = -\cos x$이므로
$\int_{0}^{\pi} x \cos (\pi - x) dx = \int_{0}^{\pi} x (-\cos x) dx$
이때, $u(x) = x$, $v'(x) = -\cos x$, $u'(x) = 1$, $v(x) = -\sin x$라 하면
$\int_{0}^{\pi} x \cos (\pi - x) dx = \int_{0}^{\pi} x (-\cos x) dx$
$= [x(-\sin x)]_{0}^{\pi} + \int_{0}^{\pi} \sin x dx$
$= [-\cos x]_{0}^{\pi}$
$= -\cos \pi + \cos 0$
$= 2$

정답: $2$
