## 문제
함수 $f(x)$를 $f(x) = \log x - [\log x]$라 하자. $10 < a < 100$인 실수 $a$에 대하여 $\{f(a)\}^2 + \{f(\frac{1}{a})\}^2$의 최솟값은? (단, $[x]$는 $x$보다 크지 않은 최대의 정수이다.)

## 보기
① $\frac{1}{4}$   ② $\frac{1}{2}$   ③ $\frac{3}{4}$   ④ $1$   ⑤ $\frac{3}{2}$

## 풀이
$\log a = 1 + \alpha$ ($0 < \alpha < 1$)라 하면 
$f(a) = \alpha$
$\log \frac{1}{a} = -\log a = - (1 + \alpha) = -2 + (1 - \alpha)$
이므로
$f(\frac{1}{a}) = 1 - \alpha$
$\{f(a)\}^2 + \{f(\frac{1}{a})\}^2 = \alpha^2 + (1 - \alpha)^2 = 2(\alpha - \frac{1}{2})^2 + \frac{1}{2}$
$0 < \alpha < 1$에서 $\alpha = \frac{1}{2}$일 때 최솟값 $\frac{1}{2}$을 갖는다.

정답: ②
