## 문제
두 실수 $x$, $y$에 대하여 $x * y = x + y + xy$라 하자. $(1 * a) * b = 3$을 만족시키는 정수 $a$, $b$의 순서쌍 $(a, b)$의 개수는?

## 보기
① $2$   ② $4$   ③ $6$   ④ $8$   ⑤ $10$

## 풀이
$(1 * a) * b = 3$에서
$(1 + a + a) * b = 3$
$(1 + 2a) * b = 3$
$(1 + 2a) + b + (1 + 2a)b = 3$
$1 + 2a + b + b + 2ab = 3$
$1 + 2a + 2b + 2ab = 3$
$2ab + 2a + 2b = 2$
$ab + a + b = 1$
양변에 $1$을 더하면
$ab + a + b + 1 = 2$
$(a + 1)(b + 1) = 2$

이때 $a$, $b$는 정수이므로 $a + 1$, $b + 1$도 정수이다.
곱해서 $2$가 되는 두 정수 $(a + 1, b + 1)$의 쌍은
$(1, 2)$, $(2, 1)$, $(-1, -2)$, $(-2, -1)$이다.

이를 만족시키는 정수 $a$, $b$의 순서쌍 $(a, b)$는
$(0, 1)$, $(1, 0)$, $(-2, -3)$, $(-3, -2)$
의 $4$개다.

정답: ②
