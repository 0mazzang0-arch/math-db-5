## 문제
$(17.8)^{n}$ 의 정수부분이 $9$ 자리의 자연수가 되도록 하는 자연수 $n$ 의 값은? (단, $\log 1.78=0.25$ 로 계산한다.)

## 보기
① $6$   ② $7$   ③ $8$   ④ $9$   ⑤ $10$

## 풀이
$(17.8)^{n}$ 이 $9$ 자리수가 되기 위해서 $8 \le \log (17.8)^{n} < 9$ 이여야 한다. 
$\log 17.8 = \log (1.78 \times 10) = 1.25$ 이므로 

$\frac{8}{\log 17.8} \le n < \frac{9}{\log 17.8}$ 

$\frac{32}{5} \le n < \frac{36}{5}$ 

$6.4 \le n < 7.2$

따라서 자연수 $n$ 의 값은 $7$ 이다.

정답: ②
