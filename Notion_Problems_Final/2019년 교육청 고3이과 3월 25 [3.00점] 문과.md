## 문제
$2 \le x \le 3$ 에서 함수 $f(x) = \left( \frac{1}{3} \right)^{2x-a}$ 의 최댓값은 $27$, 최솟값은 $m$ 이다. $a \times m$ 의 값을 구하시오. (단, $a$ 는 상수이다.)

## 보기

## 풀이
함수 $f(x) = \left( \frac{1}{3} \right)^{2x-a}$ 은 밑이 $1$ 보다 작으므로 감소함수이다.
따라서 $2 \le x \le 3$ 에서 $x = 2$ 일 때 최댓값을 갖는다.

$f(2) = \left( \frac{1}{3} \right)^{4-a} = 27$
$3^{a-4} = 3^{3}$
$a = 7$

$f(x) = \left( \frac{1}{3} \right)^{2x-7}$
함수 $f(x)$ 는 $2 \le x \le 3$ 에서 $x = 3$ 일 때 최솟값 $m$ 을 가지므로
$m = f(3) = \left( \frac{1}{3} \right)^{6-7} = \left( \frac{1}{3} \right)^{-1} = 3$

따라서 $a \times m = 7 \times 3 = 21$

정답: $21$
