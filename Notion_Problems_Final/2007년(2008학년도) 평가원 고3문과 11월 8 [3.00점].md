## 문제
연속확률변수 $X$가 갖는 값의 범위는 $0 \le X \le 3$이고, 확률 $P(X \le 1)$과 확률 $P(X \le 2)$의 값이 이차방정식 $6x^{2} - 5x + 1 = 0$의 두 근일 때, 확률 $P(1 < X \le 2)$의 값은?

## 보기
① $\frac{1}{12}$   ② $\frac{1}{6}$   ③ $\frac{1}{4}$   ④ $\frac{1}{3}$   ⑤ $\frac{5}{12}$

## 풀이
$6x^2 - 5x + 1 = 0$에서
$(2x - 1)(3x - 1) = 0$
$\therefore x = \frac{1}{2}$ 또는 $x = \frac{1}{3}$

$P(X \le 1) < P(X \le 2)$이므로
$P(X \le 1) = \frac{1}{3}$, $P(X \le 2) = \frac{1}{2}$

$\therefore P(1 < X \le 2) = P(X \le 2) - P(X \le 1)$
$= \frac{1}{2} - \frac{1}{3}$
$= \frac{1}{6}$

정답: ②
