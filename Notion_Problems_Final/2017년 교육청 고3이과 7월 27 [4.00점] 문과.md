## 문제
함수 $f(x)$ 가 
$$f(x) = e^{x} + \int_{0}^{1} t f(t) dt$$
를 만족시킬 때, $f(\ln 10)$ 의 값을 구하시오.

## 보기

## 풀이
$\int_{0}^{1} t f(t) dt = a$ 라 하면 
$f(x) = e^{x} + a$ 

$a = \int_{0}^{1} t ( e^{t} + a ) dt$ 에서 

$u'(t) = e^{t} + a, v(t) = t$ 라 하면 

$u(t) = e^{t} + at, v'(t) = 1$ 이므로 

$$ \begin{array}{l} a = \int_{0}^{1} t ( e^{t} + a ) dt \\ = [ t ( e^{t} + at ) ]_{0}^{1} - \int_{0}^{1} ( e^{t} + at ) dt \\ = e + a - [ e^{t} + \frac{1}{2} a t^{2} ]_{0}^{1} \\ = \frac{1}{2} a + 1 \end{array} $$

$a = 2$ 이므로 
$f(x) = e^{x} + 2$ 

따라서 
$f(\ln 10) = 12$

정답: $12$
