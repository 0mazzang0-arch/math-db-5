## 문제
좌표평면에서 원 $(x-a)^{2}+(y-b)^{2}=r^{2}$ 이 $x$ 축과 두 점 $(3+\sqrt{21}, 0)$, $(3-\sqrt{21}, 0)$ 에서 만나고, $y$ 축과 두 점 $(0, 6)$, $(0, -2)$ 에서 만나도록 상수 $a$, $b$, $r$ 의 값을 정할 때, $a+b+r$ 의 값을 구하시오. (단, $r>0$)

## 풀이
주어진 네 점을 지나는 원의 방정식을 $x^{2}+y^{2}+Ax+By+C=0$ 이라 놓으면 $y=0$ 일 때 이차방정식 $x^{2}+Ax+C=0$ 의 두 근이 $3+\sqrt{21}$, $3-\sqrt{21}$ 이다.

$\therefore A = -(3+\sqrt{21}+3-\sqrt{21}) = -6$
$C = (3+\sqrt{21})(3-\sqrt{21}) = -12$

또, $x=0$ 일 때 이차방정식 $y^{2}+By+C=0$ 의 두 근이 $6$, $-2$ 이므로
$B = -(6-2) = -4$
$C = 6 \times (-2) = -12$

$\therefore x^{2} +y^{2} -6x-4y-12=0$
$(x-3)^{2} + (y-2)^{2} = 5^{2}$

$\therefore a=3$, $b=2$, $r=5$
$\therefore a+b+r=10$

**[다른 풀이]**
$x$ 축과 만나는 두 점을 이은 선분의 수직이등분선 위에 원의 중심이 있으므로
$a = \frac{3+\sqrt{21}+3-\sqrt{21}}{2} = 3$

또, $y$ 축과 만나는 두 점을 이은 선분의 수직이등분선 위에 원의 중심이 있으므로
$b = \frac{6+(-2)}{2} = 2$

이때, 중심 $(3, 2)$ 에서 점 $(0, 6)$ 까지의 거리가 반지름의 길이이므로
$r = \sqrt{3^{2}+(2-6)^{2}} = 5$

$\therefore a+b+r=10$
