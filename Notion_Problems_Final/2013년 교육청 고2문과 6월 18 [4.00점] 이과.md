## 문제
이차정사각행렬 $A$, $B$에 대하여 $AB + A^{2} = E$, $AB + BA = A^{2} + B^{2}$이 성립할 때, <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $E$는 단위행렬이다.)

$$\begin{array}{|l|}
\hline
\text{ㄱ. } A \text{의 역행렬이 존재한다.} \\
\text{ㄴ. } AB = BA \\
\text{ㄷ. } B^{-1} = pA + qB \text{이면 } p + q = 3 \text{이다. (단, } p, q \text{는 상수이다.)} \\
\hline
\end{array}$$

## 보기
① $ㄱ$   ② $ㄱ, ㄴ$   ③ $ㄱ, ㄷ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
[정답] ② 

ㄱ. 
$AB + A^{2} = A(B + A) = E$ 
$A^{-1} = B + A$ 
$A$의 역행렬이 존재한다. (참) 

ㄴ. 
$AB + A^{2} = E$에서 
$AB = E - A^{2} \cdots$ ㉠ 
$A^{-1} = B + A$이므로 
$A(B + A) = (B + A)A = E$ 
$BA + A^{2} = E$ 
$BA = E - A^{2} \cdots$ ㉡ 
$AB = BA$ (참) 

ㄷ. 
ㄴ에서 $AB = BA$이므로 
$AB + BA = A^{2} + B^{2}$에서 
$2AB = A^{2} + B^{2} \cdots$ ㉢ 
$AB + A^{2} = E$에서 
$A^{2} = E - AB \cdots$ ㉣ 
$2AB = E - AB + B^{2}$ 
$3AB - B^{2} = E$ 
$(3A - B)B = E$ 
$\therefore B^{-1} = 3A - B$ 
$p = 3$, $q = -1$이므로 $p + q = 2$ (거짓) 

따라서 옳은 것은 ㄱ, ㄴ이다.
