## 문제
두 수열 $\{ a_{n} \}$ , $\{ b_{n} \}$ 에 대하여 $\sum_{n=1}^{7} ( a_{n} - 2 ) ( b_{n} - 2 ) = 60$ , $\sum_{n=1}^{7} ( a_{n} + b_{n} ) = 44$ 일 때, $\sum_{n=1}^{7} a_{n} b_{n}$ 의 값을 구하시오.

## 보기

## 풀이
$\sum_{n=1}^{7} ( a_{n} - 2 ) ( b_{n} - 2 ) = \sum_{n=1}^{7} \{ a_{n} b_{n} - 2 ( a_{n} + b_{n} ) + 4 \}$
$= \sum_{n=1}^{7} a_{n} b_{n} - 2 \sum_{n=1}^{7} ( a_{n} + b_{n} ) + 4 \times 7$
$= \sum_{n=1}^{7} a_{n} b_{n} - 88 + 28 = 60$

따라서
$\sum_{n=1}^{7} a_{n} b_{n} = 120$

**[정답]** $120$
