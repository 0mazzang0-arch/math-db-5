## 문제
좌표평면 위의 점 $A$, $B$, $C$, $D$, $E$가 한 직선 위에 있고, 다음 조건을 만족시킨다.

$$
\begin{array}{l}
\text{(가) 점 } B \text{의 좌표는 } (-1, 3) \text{이고, 점 } D \text{의 좌표는 } (3, -1) \text{이다.} \\
\text{(나) } B \text{는 선분 } AC \text{의 중점이다.} \\
\text{(다) } C \text{는 선분 } AD \text{를 } 2:1 \text{로 내분한다.} \\
\text{(라) } E \text{는 선분 } CD \text{를 } 3:2 \text{로 외분한다.}
\end{array}
$$

이때 $\overline{AE}^{2}$의 값을 구하시오.

## 풀이
$B(-1, 3)$, $D(3, -1)$을 지나는 직선의 방정식은
$y = -x + 2$
점 $A$, $B$, $C$, $D$, $E$가 한 직선 위에 있으므로
$A(a, -a + 2)$, $C(c, -c + 2)$, $E(e, -e + 2)$로 놓으면
점 $B$는 선분 $AC$의 중점이고 $C$는 선분 $AD$를 $2:1$로 내분하는 점이므로 $C$는 선분 $BD$의 중점이다.

$\therefore C(1, 1)$

$B$는 선분 $AC$의 중점이므로
$\frac{a + 1}{2} = -1$

$\therefore a = -3$
따라서 $A(-3, 5)$이므로 점 $E$는 선분 $CD$를 $3:2$로 외분하는 점이다.

$\therefore e = \frac{3 \times 3 - 2 \times 1}{3 - 2} = 7$
따라서 $E(7, -5)$이므로
$\overline{AE}^{2} = (7 - (-3))^{2} + (-5 - 5)^{2} = 10^{2} + (-10)^{2} = 200$이다.

정답: $200$
