## 문제
이차함수 $f(x) = \frac{3x-x^{2}}{2}$ 에 대하여 구간 $[0, \infty)$ 에서 정의된 함수 $g(x)$ 가 다음 조건을 만족시킨다.

(가) $0 \le x < 1$ 일 때, $g(x) = f(x)$ 이다.
(나) $n \le x < n+1$ 일 때, $g(x) = \frac{1}{2^{n}} \{ f(x-n) - (x-n) \} + x$ 이다. ($n$ 은 자연수이다.)

어떤 자연수 $k$ ($k \ge 6$) 에 대하여 함수 $h(x)$ 는
$$ h(x) = \left\{ \begin{array}{ll} g(x) & (0 \le x < 5 \text{ 또는 } x \ge k) \\ 2x - g(x) & (5 \le x < k) \end{array} \right. $$
이다. 수열 $\{ a_{n} \}$ 을 $a_{n} = \int_{0}^{n} h(x) dx$ 라 할 때, $\lim_{n \to \infty} (2a_{n} - n^{2}) = \frac{241}{768}$ 이다. $k$ 의 값을 구하시오.

## 풀이
직선 $y=x$ 와 $x$ 축 및 직선 $x=n$ 으로 둘러싸인 삼각형의 넓이가 $\frac{n^{2}}{2}$ 이다.
$2 \lim_{n \to \infty} (a_{n} - \frac{n^{2}}{2}) = \frac{241}{768}$ 이므로
$F(x) = h(x) - x$
$$ F(x) = \left\{ \begin{array}{ll} g(x) - x & (0 \le x < 5, x \ge k) \\ x - g(x) & (5 \le x < k) \end{array} \right. $$
로 놓으면 $2 \lim_{n \to \infty} \int_{0}^{n} F(x) dx = \frac{241}{768}$ 이다.
$n \le x < n+1$ 일 때
$g(x) - x = \frac{1}{2^{n}} \{ f(x-n) - (x-n) \}$
$= \frac{1}{2^{n}} \{ \frac{3}{2}(x-n) - \frac{1}{2}(x-n)^{2} - (x-n) \}$
$= \frac{1}{2^{n}} \{ \frac{1}{2}(x-n) - \frac{1}{2}(x-n)^{2} \}$
$= \frac{1}{2^{n+1}} (x-n) (1+n-x)$
$= -\frac{1}{2^{n+1}} (x-n) (x-(n+1))$
이므로
$\int_{n}^{n+1} \{ g(x) - x \} dx = \int_{n}^{n+1} \{ -\frac{1}{2^{n+1}} (x-n) (x-(n+1)) \} dx$
$= \int_{0}^{1} \{ -\frac{1}{2^{n+1}} x (x-1) \} dx$
$= -\frac{1}{2^{n+1}} \int_{0}^{1} x (x-1) dx$
$= \frac{1}{2^{n+1}} \times \frac{1}{6}$ …… ㉠

한편,
$\int_{0}^{n} F(x) dx = \int_{0}^{1} (g(x)-x) dx + \cdots + \int_{4}^{5} (g(x)-x) dx$
$+ \int_{5}^{6} (x-g(x)) dx + \cdots + \int_{k-1}^{k} (x-g(x)) dx$
$+ \int_{k}^{k+1} (g(x)-x) dx + \cdots + \int_{n-1}^{n} (g(x)-x) dx$

이때, ㉠은 $n=0$ 일 때도 성립하므로 ㉠에서
$2 \int_{0}^{n} F(x) dx = \frac{1}{3} \times \{ \frac{1}{2} + (\frac{1}{2})^{2} + \cdots + (\frac{1}{2})^{5} \}$
$- \frac{1}{3} \times \{ (\frac{1}{2})^{6} + \cdots + (\frac{1}{2})^{k} \}$
$+ \frac{1}{3} \times \{ (\frac{1}{2})^{k+1} + \cdots + (\frac{1}{2})^{n} \}$
$= \frac{1}{3} \times \{ \frac{1}{2} + (\frac{1}{2})^{2} + \cdots + (\frac{1}{2})^{n} \} - \frac{2}{3} \times \{ (\frac{1}{2})^{6} + \cdots + (\frac{1}{2})^{k} \}$
$= \frac{1}{3} \times \frac{\frac{1}{2} \{ 1 - (\frac{1}{2})^{n} \}}{1 - \frac{1}{2}} - \frac{2}{3} \times \frac{(\frac{1}{2})^{6} \{ 1 - (\frac{1}{2})^{k-5} \}}{1 - \frac{1}{2}}$
$= \frac{1}{3} \{ 1 - (\frac{1}{2})^{n} \} - \frac{2}{3} \times (\frac{1}{2})^{5} \{ 1 - (\frac{1}{2})^{k-5} \}$
$= \frac{1}{3} \{ 1 - (\frac{1}{2})^{n} \} - \frac{1}{48} \{ 1 - (\frac{1}{2})^{k-5} \}$

따라서 $\lim_{n \to \infty} (\frac{1}{2})^{n} = 0$ 이므로
$2 \lim_{n \to \infty} \int_{0}^{n} F(x) dx = \frac{1}{3} - \frac{1}{48} \{ 1 - (\frac{1}{2})^{k-5} \}$
$= \frac{15}{48} + \frac{1}{48} \times (\frac{1}{2})^{k-5}$
즉, $\frac{15}{48} + \frac{1}{48} \times (\frac{1}{2})^{k-5} = \frac{241}{768}$,
$(\frac{1}{2})^{k-5} = \frac{1}{16}$
이므로 $k-5=4$ 에서 $k=9$ 이다.
