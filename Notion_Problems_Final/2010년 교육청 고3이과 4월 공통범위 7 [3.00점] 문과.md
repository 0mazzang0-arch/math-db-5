## 문제
다음은 $a$, $b$가 $1$이 아닌 양의 실수일 때, 
$\log _{a} b = \log _{b} a$이면 $\frac{a^{2} + 1}{b^{2} + 1} = \frac{a}{b} \cdots \cdots (\ast)$ 가 성립함을 증명한 것이다.

$$ \begin{array}{|l|} \hline \\ \log _{b} a = \frac{1}{(\text{가})} \text{ 이고 가정에서 } \log _{a} b = \log _{b} a \text{ 이므로} \\ \log _{a} b = 1 \text{ 또는 } \log _{a} b = -1 \text{ 이다.} \\ (\text{ⅰ}) \log _{a} b = 1 \text{ 일 때, } \frac{a^{2} + 1}{b^{2} + 1} = (\text{나}) \text{ 이고 } \frac{a}{b} = (\text{나}) \text{ 이다.} \\ (\text{ⅱ}) \log _{a} b = -1 \text{ 일 때, } \frac{a^{2} + 1}{b^{2} + 1} = (\text{다}) \text{ 이고 } \frac{a}{b} = (\text{다}) \text{ 이다.} \\ \text{따라서, (ⅰ), (ⅱ)에 의하여 } \frac{a^{2} + 1}{b^{2} + 1} = \frac{a}{b} \text{ 이므로 } (\ast) \text{ 가 성립한다.} \\ \\ \hline \end{array} $$

위 증명에서 $(\text{가})$, $(\text{나})$, $(\text{다})$에 알맞은 것은?

## 보기
① $(\text{가}): \log _{a} b, (\text{나}): -1, (\text{다}): b^{2}$   ② $(\text{가}): \log _{b} \frac{1}{a}, (\text{나}): -1, (\text{다}): ab$   ③ $(\text{가}): \log _{a} b, (\text{나}): 1, (\text{다}): a^{2}$   ④ $(\text{가}): \log _{b} \frac{1}{a}, (\text{나}): -1, (\text{다}): a^{2}$   ⑤ $(\text{가}): \log _{a} b, (\text{나}): 1, (\text{다}): b^{2}$

## 풀이
가정에서 $\log _{a} b = \log _{b} a$이고 $\log _{b} a = \frac{1}{\log _{a} b}$이므로 $(\text{가})$에 알맞은 식은 $\log _{a} b$이다.
$\log _{a} b = \frac{1}{\log _{a} b}$에서 $(\log _{a} b)^{2} = 1$이므로 $\log _{a} b = 1$ 또는 $\log _{a} b = -1$이다.

(ⅰ) $\log _{a} b = 1$일 때, $a = b$이므로 
$\frac{a^{2} + 1}{b^{2} + 1} = \frac{a^{2} + 1}{a^{2} + 1} = 1$, $\frac{a}{b} = \frac{a}{a} = 1$이다. 따라서 $(\text{나})$에 알맞은 값은 $1$이다.

(ⅱ) $\log _{a} b = -1$일 때, $b = a^{-1} = \frac{1}{a}$이므로 
$\frac{a^{2} + 1}{b^{2} + 1} = \frac{a^{2} + 1}{(\frac{1}{a})^{2} + 1} = \frac{a^{2} + 1}{\frac{1 + a^{2}}{a^{2}}} = a^{2}$, $\frac{a}{b} = \frac{a}{\frac{1}{a}} = a^{2}$이다. 따라서 $(\text{다})$에 알맞은 식은 $a^{2}$이다.

위 내용을 종합하면 
$(\text{가}): \log _{a} b$, $(\text{나}): 1$, $(\text{다}): a^{2}$
정답은 ③이다.
