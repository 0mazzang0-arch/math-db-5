## 문제
이차정사각행렬 $A$, $B$에 대하여 <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $E$는 단위행렬이고, $k$는 실수이다.)

## 보기
ㄱ. $A^{5} = A^{7} = E$이면 $A = E$이다.
ㄴ. $(A + B)^{2} = A^{2} + AB + BA + B^{2}$
ㄷ. $(kA)B = A(kB) = k(AB)$

① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
ㄱ, ㄴ, ㄷ 모두 참이다.

ㄱ. $A^{7} = A^{5} A^{2} = E$이므로 $A^{5} = E$를 대입하면 $E A^{2} = E$가 되어 $A^{2} = E$이다.
이때 $A^{5} = (A^{2})^{2} A = E$이므로 $E^{2} A = E$에서 $A = E$이다. (참)

ㄴ. 행렬의 곱셈에 대한 분배법칙에 의하여 
$(A + B)^{2} = (A + B)(A + B) = A^{2} + AB + BA + B^{2}$
이 성립한다. (참)

ㄷ. 행렬의 실수배에 대한 성질에 의하여
$(kA)B = A(kB) = k(AB)$
가 성립한다. (참)

따라서 ㄱ, ㄴ, ㄷ 모두 옳다.

정답: ⑤
