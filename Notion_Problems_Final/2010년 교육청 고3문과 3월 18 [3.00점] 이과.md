## 문제
행렬 $A = \left( \begin{array}{cc} a & b \\ c & d \end{array} \right)$ 에 대하여
$$ A \left( \begin{array}{c} 2 \\ 3 \end{array} \right) = \left( \begin{array}{c} 3 \\ 4 \end{array} \right), A^{2} \left( \begin{array}{c} 2 \\ 3 \end{array} \right) = \left( \begin{array}{c} 5 \\ 7 \end{array} \right) $$
일 때, $a b c d$ 의 값을 구하시오.

## 보기

## 풀이
$$ A \left( \begin{array}{c} 2 \\ 3 \end{array} \right) = \left( \begin{array}{cc} a & b \\ c & d \end{array} \right) \left( \begin{array}{c} 2 \\ 3 \end{array} \right) = \left( \begin{array}{c} 2 a + 3 b \\ 2 c + 3 d \end{array} \right) = \left( \begin{array}{c} 3 \\ 4 \end{array} \right) $$

$$ A^{2} \left( \begin{array}{c} 2 \\ 3 \end{array} \right) = A \left\{ A \left( \begin{array}{c} 2 \\ 3 \end{array} \right) \right\} = A \left( \begin{array}{c} 3 \\ 4 \end{array} \right) = \left( \begin{array}{cc} a & b \\ c & d \end{array} \right) \left( \begin{array}{c} 3 \\ 4 \end{array} \right) = \left( \begin{array}{c} 3 a + 4 b \\ 3 c + 4 d \end{array} \right) = \left( \begin{array}{c} 5 \\ 7 \end{array} \right) $$

$2 a + 3 b = 3$, $3 a + 4 b = 5$
$\therefore a = 3$, $b = - 1$

$2 c + 3 d = 4$, $3 c + 4 d = 7$
$\therefore c = 5$, $d = - 2$

따라서 $a b c d = 3 \times ( - 1 ) \times 5 \times ( - 2 ) = 30$ 이다.

정답: $30$
