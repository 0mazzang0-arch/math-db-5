## 문제
등식 $\frac{1}{i} + \frac{3}{i^2} + \frac{5}{i^3} + \frac{7}{i^4} = a + bi$를 만족시키는 실수 $a$, $b$의 합 $a + b$의 값은? (단, $i = \sqrt{-1}$)

## 보기
① $2$   ② $4$   ③ $6$   ④ $8$   ⑤ $10$

## 풀이
$\frac{1}{i^2} = -1$, $\frac{1}{i^3} = i$, $\frac{1}{i^4} = 1$이므로
$\frac{1}{i} + \frac{3}{i^2} + \frac{5}{i^3} + \frac{7}{i^4} = -i - 3 + 5i + 7 = 4 + 4i$

따라서 $4 + 4i = a + bi$
복소수가 서로 같을 조건에 의하여 $a = 4$, $b = 4$
따라서 $a + b = 8$

[다른 풀이]
$\frac{1}{i} + \frac{3}{i^2} + \frac{5}{i^3} + \frac{7}{i^4} = a + bi$의 양변에 $i^4 = 1$을 곱하면
$i^3 + 3i^2 + 5i + 7 = a + bi$

따라서 $4 + 4i = a + bi$
복소수가 서로 같을 조건에 의하여 $a = 4$, $b = 4$
따라서 $a + b = 8$

정답: ④
