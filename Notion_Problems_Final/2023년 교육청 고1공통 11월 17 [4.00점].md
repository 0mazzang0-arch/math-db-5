## 문제
양수 $k$ 에 대하여 이차함수 $f(x) = -x^{2} + 4x + k + 3$ 의 그래프와 직선 $y = 2x + 3$ 이 서로 다른 두 점 $(\alpha, f(\alpha))$, $(\beta, f(\beta))$ 에서 만난다. $\alpha \leq x \leq \beta$ 에서 함수 $f(x)$ 의 최댓값이 $10$ 일 때, $\alpha \leq x \leq \beta$ 에서 함수 $f(x)$ 의 최솟값은? (단, $\alpha < \beta$)

## 보기
① $1$   ② $2$   ③ $3$   ④ $4$   ⑤ $5$

## 풀이
이차함수 $f(x) = -(x - 2)^{2} + k + 7$ 의 그래프의 꼭짓점의 좌표는 $(2, k + 7)$ 이고 직선 $y = 2x + 3$ 은 점 $(2, 7)$ 을 지난다. $f(2) = k + 7 > 7$ 이므로 함수 $y = f(x)$ 의 그래프와 직선 $y = 2x + 3$ 은 그림과 같다.

> [!example] 📸 (그림/그래프)

$\alpha < 2 < \beta$ 이므로 $\alpha \leq x \leq \beta$ 에서 함수 $f(x)$ 의 최댓값은 $f(2)$, 최솟값은 $f(\alpha)$ 이다. 

$f(2) = k + 7 = 10$ 에서 $k = 3$ 이다.

$-x^{2} + 4x + 6 = 2x + 3$ 에서 $(x + 1)(x - 3) = 0$ 이므로 $\alpha = -1$, $\beta = 3$ 이다.

따라서 $-1 \leq x \leq 3$ 에서 함수 $f(x)$ 의 최솟값은 
$f(-1) = -(-1)^{2} + 4 \times (-1) + 6 = 1$ 이다.
