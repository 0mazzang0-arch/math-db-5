## 문제
두 집합 $X = \{ x \mid -3 \leq x \leq 5 \}$, $Y = \{ y \mid |y| \leq a, a > 0 \}$에 대하여 $X$에서 $Y$로의 함수 $f(x) = 2x + b$가 일대일대응이다. 두 상수 $a$, $b$에 대하여 $a^{2} + b^{2}$의 값은?

## 보기
① $66$   ② $68$   ③ $70$   ④ $72$   ⑤ $74$

## 풀이
함수 $f(x) = 2x + b$가 일대일대응이므로 치역과 공역이 같다.
직선 $y = f(x)$의 기울기가 양수이므로
$f(-3) = -6 + b = -a$ $\cdots$ ㉠
$f(5) = 10 + b = a$ $\cdots$ ㉡
㉠, ㉡에서 $a = 8$, $b = -2$
따라서 $a^{2} + b^{2} = 8^{2} + (-2)^{2} = 68$

정답: ②
