## 문제
컴퓨터 화면에서 마우스 커서가 아이콘까지 이동하는 시간을 $T$ (초), 현재 마우스 커서의 위치로부터 아이콘의 중심까지의 거리를 $D$ ($\text{cm}$), 마우스 커서가 움직이는 방향으로 측정한 아이콘의 폭을 $W$ ($\text{cm}$)라 하면 다음과 같은 관계식이 성립한다고 한다. (단, $D>0$)

$$T=a+ \frac{1}{10} \log_{2} \left( \frac{D}{W} + 1 \right)$$ (단, $a$는 상수)

그림과 같이 컴퓨터 화면에 두 개의 아이콘 $A$, $B$가 있다. 현재 마우스 커서의 위치에서 아이콘 $A$의 방향으로 측정한 아이콘 $A$의 폭 $W_{A}$와 아이콘 $B$의 방향으로 측정한 아이콘 $B$의 폭 $W_{B}$는 모두 $1 \text{cm}$로 같다. 현재 마우스 커서의 위치로부터 아이콘 $A$의 중심까지의 거리와 아이콘 $B$의 중심까지의 거리를 각각 $D_{A}$ ($\text{cm}$), $D_{B}$ ($\text{cm}$)라 할 때, 마우스 커서가 아이콘 $A$까지 이동하는 시간 $T_{A}$, 아이콘 $B$까지 이동하는 시간 $T_{B}$는 각각 $0.71$초, $0.66$초이다. $\frac{D_{A} + 1}{D_{B} + 1}$의 값은?



> [!example] 📸 (그림/그래프)

## 보기
① $1$   ② $\sqrt{2}$   ③ $2$   ④ $2 \sqrt{2}$   ⑤ $4$

## 풀이
마우스 커서가 아이콘 $A$까지 이동하는 시간이 $0.71$초이고 $W_{A} = 1$이므로
$0.71 = a + \frac{1}{10} \log_{2} (D_{A} + 1)$ $\cdots$ ㉠

마우스 커서가 아이콘 $B$까지 이동하는 시간이 $0.66$초이고 $W_{B} = 1$이므로
$0.66 = a + \frac{1}{10} \log_{2} (D_{B} + 1)$ $\cdots$ ㉡

㉠$-$㉡에서
$0.05 = \frac{1}{10} \log_{2} (D_{A} + 1) - \frac{1}{10} \log_{2} (D_{B} + 1)$
$0.05 = \frac{1}{10} \log_{2} \frac{D_{A} + 1}{D_{B} + 1}$

양변에 $10$을 곱하면
$0.5 = \log_{2} \frac{D_{A} + 1}{D_{B} + 1}$

따라서 로그의 정의에 의해
$\frac{D_{A} + 1}{D_{B} + 1} = 2^{0.5} = \sqrt{2}$

정답: ②
