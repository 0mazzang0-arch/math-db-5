## 문제
자연수 전체의 집합 $U$의 두 부분집합 $A = \{ 1, 2, 4, 8, 16 \}$, $B = \{ x \mid x^{2} - 4x + 3 = 0 \}$에 대하여 $n(X) = 2$, $X - (A - B) = \emptyset$을 만족시키는 $U$의 모든 부분집합 $X$의 개수를 구하시오.

## 보기

## 풀이
$x^{2} - 4x + 3 = (x - 3)(x - 1) = 0$이므로 
$x = 1$ 또는 $x = 3$이다.
즉, $B = \{ 1, 3 \}$이므로 
$A - B = \{ 2, 4, 8, 16 \}$이다. 
$X - (A - B) = \emptyset$이므로 $X \subset (A - B)$이다. 
$n(X) = 2$이므로 집합 $X$는 집합 $A - B$의 원소의 개수가 $2$인 부분집합이다. 
즉, 조건을 만족시키는 집합 $X$의 개수는 집합 $A - B$의 원소 $2, 4, 8, 16$ 중 $2$개의 원소를 택하는 경우의 수 ${}_{4}C_{2} = \frac{4 \times 3}{2} = 6$과 같다. 
따라서 조건을 만족시키는 집합 $X$의 개수는 $6$이다.
