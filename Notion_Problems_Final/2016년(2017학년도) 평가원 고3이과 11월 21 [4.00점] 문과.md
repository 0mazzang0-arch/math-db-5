## 문제
닫힌 구간 $[0, 1]$ 에서 증가하는 연속함수 $f(x)$ 가 
$\int_{0}^{1} f(x) dx = 2$, $\int_{0}^{1} |f(x)| dx = 2 \sqrt{2}$ 를 만족시킨다. 함수 $F(x)$ 가 
$F(x) = \int_{0}^{x} |f(t)| dt$ ($0 \le x \le 1$) 일 때, $\int_{0}^{1} f(x) F(x) dx$ 의 값은?

## 보기
① $4 - \sqrt{2}$   ② $2 + \sqrt{2}$   ③ $5 - \sqrt{2}$   ④ $1 + 2 \sqrt{2}$   ⑤ $2 + 2 \sqrt{2}$

## 풀이
정답 ④

함수 $y = f(x)$ 의 그래프가 닫힌 구간 $[0, 1]$ 에서 $x$ 축과 만나는 점의 $x$ 좌표를 $k$ 라 하자. 
곡선 $y = f(x)$ 와 $x$ 축, $y$ 축으로 둘러싸인 부분의 넓이를 $S_{1}$, 곡선 $y = f(x)$ 와 $x$ 축 및 직선 $x = 1$ 로 둘러싸인 부분의 넓이를 $S_{2}$ 라 하면 

$\int_{0}^{1} f(x) dx = 2$ 에서 
$-S_{1} + S_{2} = 2 \cdots \cdots$ ㉠ 

$\int_{0}^{1} |f(x)| dx = 2 \sqrt{2}$ 에서 
$S_{1} + S_{2} = 2 \sqrt{2} \cdots \cdots$ ㉡ 

㉠, ㉡에서 
$S_{1} = \sqrt{2} - 1, S_{2} = \sqrt{2} + 1$ 

(ⅰ) $0 \le x \le k$ 인 경우 
$F(x) = \int_{0}^{x} (-f(t)) dt$ 이므로 
$F'(x) = -f(x)$ 

$\int_{0}^{k} f(x) F(x) dx$ 에서 
$F(x) = s$ 로 놓으면 
$x = 0$ 일 때 $s = 0$, 
$x = k$ 일 때 $s = \sqrt{2} - 1$ 이고, 
$F'(x) \frac{dx}{ds} = 1$ 이므로 

$\int_{0}^{k} f(x) F(x) dx = \int_{0}^{\sqrt{2}-1} (-s) ds = [-\frac{1}{2} s^{2}]_{0}^{\sqrt{2}-1} = -\frac{1}{2} (\sqrt{2}-1)^{2}$ 

(ⅱ) $k \le x \le 1$ 인 경우 
$F(x) = (\sqrt{2} - 1) + \int_{k}^{x} f(t) dt$ 이므로 
$F'(x) = f(x)$ 

$\int_{k}^{1} f(x) F(x) dx$ 에서 
$F(x) = s$ 로 놓으면 
$x = k$ 일 때 $s = \sqrt{2} - 1$, 
$x = 1$ 일 때 $s = 2 \sqrt{2}$ 이고, 
$F'(x) \frac{dx}{ds} = 1$ 이므로 

$\int_{k}^{1} f(x) F(x) dx = \int_{\sqrt{2}-1}^{2 \sqrt{2}} s ds = [\frac{1}{2} s^{2}]_{\sqrt{2}-1}^{2 \sqrt{2}} = 4 - \frac{1}{2} (\sqrt{2}-1)^{2}$ 

(ⅰ), (ⅱ)에서 
$\int_{0}^{1} f(x) F(x) dx = \int_{0}^{k} f(x) F(x) dx + \int_{k}^{1} f(x) F(x) dx$ 
$= -\frac{1}{2} (\sqrt{2}-1)^{2} + 4 - \frac{1}{2} (\sqrt{2}-1)^{2}$ 
$= 4 - (\sqrt{2}-1)^{2}$ 
$= 4 - (3 - 2 \sqrt{2})$ 
$= 1 + 2 \sqrt{2}$
