## 문제
등비수열 $\{ a_{n} \}$ 에 대하여 
$a_{1} = \alpha$, $a_{3} = \alpha + 12$, $a_{5} = \alpha + 72$
일 때, $a_{7}$ 의 값을 구하시오.

## 풀이
$a_{3} = \alpha r^{2} = \alpha + 12$
$a_{5} = \alpha r^{4} = \alpha + 72$
$\alpha r^{4} - \alpha = \alpha ( r^{2} + 1 ) ( r^{2} - 1 ) = 12 ( r^{2} + 1 ) = 72$
따라서 $r^{2} = 5$, $\alpha = 3$ 이다.
$a_{7} = \alpha r^{6} = 3 \times 5^{3} = 375$

정답: $375$
