## 문제
이차함수 $f(x)$ 는 다음 두 조건을 만족시킨다.

$$ \begin{array}{|l|} \hline \text{(가) 모든 실수 } x \text{ 에 대하여 } f(3-x)=f(3+x) \text{ 이다.} \\ \text{(나) } y=f(x) \text{ 의 그래프는 두 점 } (-1, 2) \text{, } (4, 17) \text{ 을 지난다.} \\ \hline \end{array} $$

옳은 것만을 <보기>에서 있는 대로 고른 것은?

$$ \begin{array}{|l|} \hline \text{ㄱ. } y=f(x) \text{ 의 그래프는 직선 } x=3 \text{ 에 대하여 대칭이다.} \\ \text{ㄴ. } 1 \leq x \leq 8 \text{ 에서 이차함수 } f(x) \text{ 의 최솟값은 } -7 \text{ 이다.} \\ \text{ㄷ. } g(x)=f(x+3) \text{ 일 때, 모든 실수 } x \text{ 에 대하여 } g(-x)=-g(x) \text{ 이다.} \\ \hline \end{array} $$

## 보기
① ㄱ  ② ㄴ  ③ ㄱ, ㄴ  ④ ㄴ, ㄷ  ⑤ ㄱ, ㄴ, ㄷ

## 풀이
ㄱ. 모든 실수 $x$ 에 대하여 $f(3-x)=f(3+x)$ 이므로 $y=f(x)$ 의 그래프는 직선 $x=3$ 에 대하여 대칭이다. (참)

ㄴ. $f(x)=a(x-3)^{2}+b$ 라 놓으면, $y=f(x)$ 가 두 점 $(-1, 2)$, $(4, 17)$ 을 지나므로
$$ \begin{array}{ll} 16a+b=2 & \cdots \cdots \text{㉠} \\ a+b=17 & \cdots \cdots \text{㉡} \end{array} $$
이때, ㉠ 과 ㉡을 연립하여 풀면, $a=-1$, $b=18$
그러므로 $f(x)=-(x-3)^{2}+18$ 은 $1 \leq x \leq 8$ 에서 $x=8$ 일 때, 이차함수 $f(x)$ 는 최솟값 $f(8)=-(8-3)^{2}+18 = -25+18 = -7$ 을 갖는다. (참)

ㄷ. $g(x)=f(x+3)=-( (x+3)-3)^{2}+18 = -x^{2}+18$ 이므로 $y=g(x)$ 의 그래프는 $y$ 축에 대하여 대칭이다. 그러므로 모든 실수 $x$ 에 대하여 $g(-x)=g(x)$ 이다. 
$\therefore g(-x) \neq -g(x)$ (거짓)

따라서 옳은 것은 ㄱ, ㄴ이다.
