## 문제
다음 세 조건을 동시에 만족시키는 양의 실수 $x$, $y$가 있다.

$$ \begin{array}{l} \text{Ⅰ. } \log_{10} x^{2} y^{3} = 12.5 \\ \text{Ⅱ. } x \text{와 } y \text{의 상용로그의 정수부분은 같다.} \\ \text{Ⅲ. } x \text{와 } \frac{1}{y} \text{의 상용로그의 소수부분은 같다.} \end{array} $$

이때, $\frac{x}{y}$의 값을 구하시오.

## 풀이
Ⅱ에서 $\log x = n + \alpha$, $\log y = n + \beta$라 하면 ($n$은 정수, $0 \le \alpha < 1, 0 \le \beta < 1$)
Ⅲ에 의해서 $\alpha + \beta = 1$이다. (단, $\beta \neq 0$)
한편 Ⅰ에서 
$2(n + \alpha) + 3(n + \beta) = 12.5$
$5n + 2\alpha + 3\beta = 12.5$
$5n + 2(\alpha + \beta) + \beta = 12.5$
$5n + 2 + \beta = 12.5$이므로
$n = 2$, $\beta = \frac{1}{2} = \alpha$이다.

$x = 10^{\frac{5}{2}}$, $y = 10^{\frac{5}{2}}$
$\therefore \frac{x}{y} = 1$

정답: $1$
