## 문제
다항함수 $f(x)$ 에 대하여 $f'(x) = 9x^2 + 4x$ 이고 $f(1) = 6$ 일 때, $f(2)$ 의 값을 구하시오.

## 보기

## 풀이
$f(x) = \int (9x^2 + 4x)dx = 3x^3 + 2x^2 + C$ (단, $C$ 는 적분상수)
$f(1) = 3 + 2 + C = 6$ 이므로
$5 + C = 6$ 에서 $C = 1$ 이다.
따라서 $f(x) = 3x^3 + 2x^2 + 1$ 이므로
$f(2) = 3 \times 2^3 + 2 \times 2^2 + 1 = 24 + 8 + 1 = 33$
정답: $33$
