## 문제
삼각방정식 $2 \sin x - 4 \sin x \cos^2 x - \cos 2x + 1 = 0$ 을 만족시키는 모든 근의 합은? (단, $0 \leq x < 2 \pi$)

## 보기
① $\frac{5}{2} \pi$   ② $\frac{11}{4} \pi$   ③ $3 \pi$   ④ $\frac{13}{4} \pi$   ⑤ $\frac{7}{2} \pi$

## 풀이
$\cos^2 x = 1 - \sin^2 x$, $1 - \cos 2x = 2 \sin^2 x$ 이므로 주어진 방정식에 대입하면 다음과 같다.
$2 \sin x - 4 \sin x (1 - \sin^2 x) + 2 \sin^2 x = 0$
$4 \sin^3 x + 2 \sin^2 x - 2 \sin x = 0$
$2 \sin x (2 \sin^2 x + \sin x - 1) = 0$
$2 \sin x (\sin x + 1)(2 \sin x - 1) = 0$

따라서 $\sin x = 0$ 또는 $\sin x = -1$ 또는 $\sin x = \frac{1}{2}$ 이다.
$0 \leq x < 2 \pi$ 의 범위에서 각 방정식의 해를 구하면
$x = 0, \pi$ (단, $\sin x = 0$ 일 때)
$x = \frac{3}{2} \pi$ (단, $\sin x = -1$ 일 때)
$x = \frac{\pi}{6}, \frac{5}{6} \pi$ (단, $\sin x = \frac{1}{2}$ 일 때)

따라서 구하는 모든 근의 합은
$\frac{0 + 6 + 9 + 1 + 5}{6} \pi = \frac{21}{6} \pi = \frac{7}{2} \pi$
