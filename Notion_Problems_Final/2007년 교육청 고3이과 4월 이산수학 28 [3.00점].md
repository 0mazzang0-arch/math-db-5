## 문제
$y = \cos x + 2 \sin (x + \frac{\pi}{6})$ 의 최댓값을 $M$, 최솟값을 $m$ 이라 할 때, $M - m$ 의 값은?

## 보기
① $\sqrt{7}$   ② $2\sqrt{7}$   ③ $3\sqrt{7}$   ④ $4\sqrt{7}$   ⑤ $5\sqrt{7}$

## 풀이
삼각함수의 덧셈정리를 이용하여 식을 정리하면 다음과 같다.
$$y = \cos x + 2 \left( \sin x \cos \frac{\pi}{6} + \cos x \sin \frac{\pi}{6} \right)$$
$$= \cos x + 2 \left( \frac{\sqrt{3}}{2} \sin x + \frac{1}{2} \cos x \right)$$
$$= \cos x + \sqrt{3} \sin x + \cos x$$
$$= \sqrt{3} \sin x + 2 \cos x$$

삼각함수의 합성을 이용하면
$$y = \sqrt{(\sqrt{3})^2 + 2^2} \sin (x + \theta) = \sqrt{7} \sin (x + \theta)$$
(단, $\sin \theta = \frac{2}{\sqrt{7}}, \cos \theta = \frac{\sqrt{3}}{\sqrt{7}}$)

함수 $y = \sqrt{7} \sin (x + \theta)$ 의 최댓값 $M = \sqrt{7}$, 최솟값 $m = -\sqrt{7}$ 이므로
$$M - m = \sqrt{7} - (-\sqrt{7}) = 2\sqrt{7}$$

정답: ②
