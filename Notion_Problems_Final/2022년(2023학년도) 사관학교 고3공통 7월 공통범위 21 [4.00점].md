## 문제
등차수열 $\{ a_{n} \}$이 다음 조건을 만족시킨다.

(가) $a_{6} + a_{7} = - \frac{1}{2}$
(나) $a_{l} + a_{m} = 1$이 되도록 하는 두 자연수 $l$, $m$ ($l < m$)의 모든 순서쌍 $(l, m)$의 개수는 $6$이다.

등차수열 $\{ a_{n} \}$의 첫째항부터 제 $14$항까지의 합을 $S$라 할 때, $2S$의 값을 구하시오.

## 보기
(해당 문항은 주관식으로 보기가 없습니다.)

## 풀이
등차수열의 성질에 의해 $a_{1} + a_{n} = a_{2} + a_{n-1} = \cdots$를 만족하고, 조건 (나)에서 $a_{l} + a_{m} = 1$을 만족하는 $l$, $m$의 순서쌍 $(l, m)$의 개수가 $6$이므로
$a_{1} + a_{n} = a_{2} + a_{n-1} = \cdots = a_{6} + a_{n-5}$에서 $n-5 = 7$ 또는 $n-5 = 8$이다.

$n-5 = 7$일 때는 조건 (가)와 모순이므로 $n-5 = 8$이다.

$\therefore a_{1} + a_{13} = a_{6} + a_{8} = 1 \cdots$ ㉠

조건 (가)에서 $a_{6} + a_{7} = - \frac{1}{2} \cdots$ ㉡

수열 $\{ a_{n} \}$의 공차를 $d$라 하면 ㉠ $-$ ㉡에서 $d = \frac{3}{2}$이다.

$\therefore 2S = 2 \times \frac{14(a_{1} + a_{14})}{2}$
$= 14(a_{1} + a_{13} + d)$
$= 14 \left( 1 + \frac{3}{2} \right)$
$= 35$

정답: $35$
