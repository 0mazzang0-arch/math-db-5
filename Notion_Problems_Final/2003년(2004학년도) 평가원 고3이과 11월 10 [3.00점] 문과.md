## 문제
$z$와 $w$가 복소수일 때, <보기>에서 옳은 것을 모두 고른 것은? (단, $i = \sqrt{-1}$이고, $\bar{w}$는 $w$의 켤레복소수이다.)

$$ \begin{array}{|l|} \hline \text{<보기>} \\ ㄱ. \ z = \bar{w} \text{ 이면 } |z+w| \le 2|z| \text{ 이다.} \\ ㄴ. \ z = iw \text{ 이면 } |z-w| = \sqrt{2}|z| \text{ 이다.} \\ ㄷ. \ z = -\bar{w} \text{ 이면 } |z+w| \le |z|^{2} \text{ 이다.} \\ \hline \end{array} $$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄱ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
[정답] ③

$w = a+bi$ (단, $a, b$는 실수)라 하자.

$ㄱ$. $z = \bar{w} = a-bi$
$\therefore |z+w| = |2a| = 2|a|$
$|z| = \sqrt{a^{2} + b^{2}}$이고, $2|a| \le 2\sqrt{a^{2} + b^{2}}$ (등호는 $b = 0$일 때 성립)이므로
$|z+w| \le 2|z|$ (참)

$ㄴ$. $z = iw = -b+ai$
$\therefore |z-w| = |(-a-b)+(a-b)i|$
$= \sqrt{(-a-b)^{2} + (a-b)^{2}} = \sqrt{2(a^{2} + b^{2})}$
$= \sqrt{2}\sqrt{a^{2} + b^{2}} = \sqrt{2}|z|$ (참)

$ㄷ$. (반례) $w = i$이면, $z = -\bar{w} = i$이다.
$|z+w| = |2i| = 2 \ge 1 = |z|^{2}$ (거짓)

이상에서 옳은 것은 $ㄱ$과 $ㄴ$이다.

[다른 풀이]
$ㄱ$. $z = \bar{w}$에서 $|z| = |\bar{w}| = |w|$이므로
$|z+w| \le |z| + |w| = 2|z|$

$ㄴ$. $z = iw$에서 $|z| = |iw| = |w|$이므로
$|z-w| = |iw-w| = |w(i-1)| = |w||i-1| = \sqrt{2}|z|$
