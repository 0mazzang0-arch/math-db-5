## 문제
두 실수 $a$, $b$에 대하여 
$2^{a} + 2^{b} = 2$, $2^{-a} + 2^{-b} = \frac{9}{4}$ 
일 때, $2^{a+b}$의 값은 $\frac{q}{p}$이다. 
$p+q$의 값을 구하시오. (단, $p$와 $q$는 서로소인 자연수이다.)

## 보기

## 풀이
$2^{-a} + 2^{-b} = \frac{1}{2^{a}} + \frac{1}{2^{b}} = \frac{2^{a} + 2^{b}}{2^{a+b}} = \frac{9}{4} \dots$ ㉠ 
그런데 $2^{a} + 2^{b} = 2$이므로 이 값을 ㉠에 대입하면 
$\frac{2}{2^{a+b}} = \frac{9}{4}$ 
$2^{a+b} = 2 \times \frac{4}{9} = \frac{8}{9}$ 
따라서 $p = 9$, $q = 8$이므로 
$p+q = 9+8 = 17$

[정답] $17$
