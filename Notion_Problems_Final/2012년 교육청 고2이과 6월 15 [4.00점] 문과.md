## 문제
수열 $\{ S_{n} \}$ 에서 $S_{n} = p n^{2} + 2 n$ 일 때, 옳은 것만을 <보기>에서 있는 대로 고른 것은?
$$ \begin{array}{|l|} \hline \text{<보기>} \\ \text{ㄱ. } p = 0 \text{ 일 때 수열 } \{ S_{n} \} \text{ 은 공차가 } 2 \text{ 인 등차수열이다.} \\ \text{ㄴ. } p = 1 \text{ 일 때 수열 } \{ S_{n} \} \text{ 의 계차수열은 공차가 } 2 \text{ 인 등차수열이다.} \\ \text{ㄷ. } p = 2 \text{ 일 때 } S_{n} \text{ 은 첫째항이 } 4 , \text{ 공차가 } 4 \text{ 인 등차수열의 제 } 1 \text{ 항부터 제 } n \text{ 항까지의 합이다.} \\ \quad ( n = 1 , 2 , 3 , \cdots ) \\ \hline \end{array} $$

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄱ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
ㄱ. $p = 0$ 일 때,
$S_{n} = 2 n = 2 + 2 ( n - 1 )$
$\{ S_{n} \}$ 은 첫째항이 $2$ 이고 공차가 $2$ 인 등차수열이다. (참)

ㄴ. $p = 1$ 일 때,
$S_{n} = n^{2} + 2 n$
$\{ S_{n} \}$ 의 계차수열을 $\{ b_{n} \}$ 이라 하면
$b_{n} = S_{n+1} - S_{n}$
$= \{ ( n + 1 )^{2} + 2 ( n + 1 ) \} - \{ n^{2} + 2 n \}$
$= 2 n + 3$
$= 5 + 2 ( n - 1 )$
$\{ b_{n} \}$ 은 첫째항이 $5$ 이고 공차가 $2$ 인 등차수열이다. 그러므로 수열 $\{ S_{n} \}$ 의 계차수열은 공차가 $2$ 인 등차수열이다. (참)

ㄷ. $p = 2$ 일 때,
제 $1$ 항부터 제 $n$ 항까지의 합이 $S_{n} = 2 n^{2} + 2 n$ 인 수열의 일반항 $a_{n}$ 을 구하자.
$n \geq 2$ 일 때,
$a_{n} = S_{n} - S_{n-1}$
$= 2 n^{2} + 2 n - \{ 2 ( n - 1 )^{2} + 2 ( n - 1 ) \}$
$= 4 n$
이므로 $a_{n} = 4 n ( n \geq 2 ) \cdots \cdots$ ㉠
㉠에 $n = 1$ 을 대입하면 $a_{1} = 4 \times 1 = 4$ 이고,
$a_{1} = S_{1} = 2 \times 1^{2} + 2 \times 1 = 4$
그러므로 수열 $\{ a_{n} \}$ 은 첫째항이 $4$ 이고 공차가 $4$ 인 등차수열이다.
따라서 $S_{n}$ 은 첫째항이 $4$ 이고 공차가 $4$ 인 등차수열의 제 $1$ 항부터 제 $n$ 항까지의 합을 의미한다. (참)
따라서 ㄱ, ㄴ, ㄷ 모두 참이다.

**[다른풀이]**
ㄷ. $p = 2$ 일 때,
$S_{n} = 2 n^{2} + 2 n = 2 n ( n + 1 ) = \frac{n ( 4 n + 4 )}{2} = \frac{n \{ 2 \times 4 + ( n - 1 ) 4 \}}{2}$
즉, $S_{n}$ 은 첫째항이 $4$, 공차가 $4$ 인 등차수열의 제 $1$ 항부터 제 $n$ 항까지의 합을 의미한다.

정답: ⑤
