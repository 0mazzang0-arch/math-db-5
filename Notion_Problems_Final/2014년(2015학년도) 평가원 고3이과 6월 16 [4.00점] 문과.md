## 문제
두 이차정사각행렬 $A$, $B$가 $A^{2} = -A$, $A^{2} + B^{2} = A + E$를 만족시킬 때, <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $E$는 단위행렬이다.)

ㄱ. $A^{3} = A$
ㄴ. $A B^{2} = B^{2} A$
ㄷ. $B$의 역행렬이 존재한다.

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄴ$   ④ $ㄱ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
ㄱ. $A^{2} = -A$ 이므로 
$A^{3} = A \cdot A^{2} = A(-A) = -A^{2} = -(-A) = A$ (참)

ㄴ. $A^{2} + B^{2} = A + E$ 에서 $B^{2} = -A^{2} + A + E$
$A^{2} = -A$ 이므로 
$B^{2} = -(-A) + A + E = 2A + E$
이므로
$A B^{2} = A(2A + E) = 2A^{2} + A = 2(-A) + A = -A$
$B^{2} A = (2A + E)A = 2A^{2} + A = 2(-A) + A = -A$
따라서 $A B^{2} = B^{2} A$ (참)

ㄷ. ㄴ에서 $B^{2} = 2A + E$ 이므로 양변을 제곱하면
$B^{4} = (2A + E)^{2} = 4A^{2} + 4A + E$
$A^{2} = -A$를 대입하면,
$B^{4} = 4(-A) + 4A + E = E$
$B \cdot B^{3} = B^{3} \cdot B = E$ 이므로
행렬 $B$의 역행렬은 $B^{3}$이다. (참)

그러므로 <보기> 중 옳은 것은 ㄱ, ㄴ, ㄷ이다.
