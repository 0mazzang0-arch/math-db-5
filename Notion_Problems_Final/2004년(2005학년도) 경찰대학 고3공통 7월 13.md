## 문제
연립부등식 $\begin{cases}x^{2}-7 x+12 \geq 0 \\ (x-2)(x+a)<0\end{cases}$ 에서 해집합이 $\{ x \mid 2 < x \leq 3 \}$ 일 때, $a$ 의 최솟값은?

## 보기
① $4$   ② $3$   ③ $-2$   ④ $-3$   ⑤ $-4$

## 풀이
부등식 $x^{2}-7 x+12=(x-4)(x-3) \geq 0$ 의 해집합을 $P$ 라 하면 
$P= \{ x \mid x \geq 4 \text{ 또는 } x \leq 3 \}$ 
부등식 $(x-2)(x+a)<0$ 의 해집합을 $Q$ 라 하면 
$P \cap Q= \{ x \mid 2 < x \leq 3 \}$ 
해집합의 관계를 수직선에 나타내면 다음과 같다. 

> [!example] 📸 (그림/그래프)

그림에서 알 수 있듯이 집합 $Q$ 는 영역 $2 < x < 3$ 을 반드시 포함해야 한다. 
(ⅰ) $a=-3$ 인 경우 
$Q= \{ 2 < x < 3 \}$ 이고 $P \cap Q= \{ 2 < x < 3 \}$ 이므로 조건을 만족하지 않는다. 
(ⅱ) $a=-4$ 인 경우 
$Q= \{ 2 < x < 4 \}$ 이고 $P \cap Q= \{ 2 < x \leq 3 \}$ 이므로 조건을 만족한다. 
따라서, 조건을 만족하는 $a$ 값의 범위는 
$3 < -a \leq 4$ , $\therefore -4 \leq a < -3$ 

$a$ 값의 최솟값은 $-4$ 이다.
