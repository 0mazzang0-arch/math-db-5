## 문제
다음은 모든 자연수 $n$에 대하여 등식 
$$\sum_{k=1}^{2 n+1} (n^{2}+k )=n^{3}+(n+1)^{3}$$ 
이 성립함을 수학적귀납법으로 증명한 것이다. 

(ⅰ) $n=1$ 일 때, 
(좌변) $= (가)$, (우변) $= (가)$ 이므로 주어진 등식은 성립한다. 

(ⅱ) $n=m \ (m \ge 1)$ 일 때, 주어진 등식이 성립한다고 가정하자. 
$$\sum_{k=1}^{2(m+1)+1} \{ (m+1)^{2}+k \}$$ 
$$= \sum_{k=1}^{2 m+1} \{ (m+1)^{2}+k \} + (나)$$ 
$$= \sum_{k=1}^{2 m+1} (m^{2}+k )+ \sum_{k=1}^{2 m+1}(2 m+1) + (나)$$ 
$$= (다) + (2 m+1)^{2} + (나)$$ 
$$= (m+1)^{3}+(m+2)^{3}$$ 
그러므로 $n=m+1$ 일 때에도 성립한다. 

(ⅰ), (ⅱ)에 의하여 모든 자연수 $n$에 대하여 주어진 등식은 성립한다. 

위의 증명에서 $(가)$, $(나)$, $(다)$에 알맞은 것은?

## 보기
① $(가) \ 8, \ (나) \ 2 m^{2}+8 m+6, \ (다) \ (m-1)^{3}+(m+1)^{3}$   ② $(가) \ 8, \ (나) \ 2 m^{2}+8 m+7, \ (다) \ (m-1)^{3}+(m+1)^{3}$   ③ $(가) \ 9, \ (나) \ 2 m^{2}+8 m+7, \ (다) \ (m-1)^{3}+(m+1)^{3}$   ④ $(가) \ 9, \ (나) \ 2 m^{2}+8 m+7, \ (다) \ m^{3}+(m+1)^{3}$   ⑤ $(가) \ 9, \ (나) \ 2 m^{2}+8 m+6, \ (다) \ m^{3}+(m+1)^{3}$

## 풀이
[정답] ④ 

(가) $9$ 
(나) $2 m^{2}+8 m+7$ 
(다) $m^{3}+(m+1)^{3}$
