## 문제
함수 $f(x)$를 $f(x) = \lim_{n \to \infty} \frac{x^n + 3}{x^n + 1}$으로 정의할 때, $f(-3) + f\left(\frac{1}{4}\right) + f(1)$의 값을 구하시오.

## 보기

## 풀이
$f(-3) = \lim_{n \to \infty} \frac{(-3)^n + 3}{(-3)^n + 1} = \lim_{n \to \infty} \frac{1 + 3 \left( -\frac{1}{3} \right)^n}{1 + \left( -\frac{1}{3} \right)^n} = 1$
마찬가지 방법으로 구하면 
$f\left(\frac{1}{4}\right) = 3$, $f(1) = 2$

$\therefore f(-3) + f\left(\frac{1}{4}\right) + f(1) = 1 + 3 + 2 = 6$

정답: $6$
