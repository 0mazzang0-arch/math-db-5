## 문제
두 자연수 $a$, $b$에 대하여 실수 $x$에 대한 두 조건
$p : x^{2} - 4x + a + 2 \le 0$
$q : 0 < |x - b| \le 4$
의 진리집합을 각각 $P$, $Q$라 하자. 
$P \ne \emptyset$, $P \subset Q$가 되도록 하는 $a$, $b$의 모든 순서쌍 $(a, b)$의 개수는?

## 보기
① $5$   ② $6$   ③ $7$   ④ $8$   ⑤ $9$

## 풀이
$P \ne \emptyset$ 이려면 $x^{2} - 4x + a + 2 \le 0$을 만족시키는 실수 $x$가 존재해야 한다.
이차방정식 $x^{2} - 4x + a + 2 = 0$의 판별식을 $D$라 하면 
$D = (-4)^{2} - 4(a + 2)$이고 $D \ge 0$이어야 하므로 
$(-4)^{2} - 4(a + 2) \ge 0$에서 $a \le 2$이다.
$P \ne \emptyset$가 되도록 하는 자연수 $a$의 값은 $1$, $2$이다.

또한 $0 < |x - b| \le 4$에서 
$Q = \{x \mid b - 4 \le x < b \text{ 또는 } b < x \le b + 4\}$이다.

(ⅰ) $a = 1$일 때
$x^{2} - 4x + 3 = (x - 1)(x - 3) \le 0$에서 
$P = \{x \mid 1 \le x \le 3\}$이므로 $P \subset Q$이려면 
$P \subset \{x \mid b - 4 \le x < b\}$이거나 $P \subset \{x \mid b < x \le b + 4\}$이어야 한다.

(a) $P \subset \{x \mid b - 4 \le x < b\}$일 때
> [!example] 📸 (그림/그래프)
$b - 4 \le 1$, $3 < b$에서 $3 < b \le 5$이므로 
$P \subset Q$가 되도록 하는 자연수 $b$의 값은 $4$, $5$이다.
$P \ne \emptyset$, $P \subset Q$가 되도록 하는 $a$, $b$의 모든 순서쌍 $(a, b)$는 $(1, 4)$, $(1, 5)$이다.

(b) $P \subset \{x \mid b < x \le b + 4\}$일 때
> [!example] 📸 (그림/그래프)
$b < 1$, $3 \le b + 4$에서 $-1 \le b < 1$이므로 
$P \subset Q$가 되도록 하는 자연수 $b$의 값은 존재하지 않는다.

(ⅱ) $a = 2$일 때
$x^{2} - 4x + 4 = (x - 2)^{2} \le 0$에서 $P = \{2\}$이므로 
$P \subset Q$이려면 $b - 4 \le 2 < b$ 또는 $b < 2 \le b + 4$이어야 한다.
$2 < b \le 6$ 또는 $-2 \le b < 2$이다.
$P \subset Q$가 되도록 하는 자연수 $b$의 값은 $1$, $3$, $4$, $5$, $6$이다.
$P \ne \emptyset$, $P \subset Q$가 되도록 하는 $a$, $b$의 모든 순서쌍 $(a, b)$는 
$(2, 1)$, $(2, 3)$, $(2, 4)$, $(2, 5)$, $(2, 6)$이다.

따라서 (ⅰ), (ⅱ)에 의하여 구하는 모든 순서쌍 $(a, b)$의 개수는 $7$이다.

정답: ③
