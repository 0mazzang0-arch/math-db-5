## 문제
두 집합 $X = \{1, 2, 3, 4, 5, 6\}$, $Y = \{1, 2, 3, 4, 5\}$에 대하여 다음 조건을 만족시키는 $X$에서 $Y$로의 함수 $f$의 개수는?

$$ \begin{array}{|l|} \hline \text{(가) } \sqrt{f(1) \times f(2) \times f(3)} \text{ 의 값은 자연수이다.} \\ \text{(나) 집합 } X \text{ 의 임의의 두 원소 } x_1, x_2 \text{ 에 대하여 } x_1 < x_2 \text{ 이면 } f(x_1) \leq f(x_2) \text{ 이다.} \\ \hline \end{array} $$

## 보기
① $84$   ② $87$   ③ $90$   ④ $93$   ⑤ $96$

## 풀이
조건 (가)에서 $\sqrt{f(1) \times f(2) \times f(3)}$의 값이 자연수인 경우는 세 수 $f(1)$, $f(2)$, $f(3)$ 중 하나의 수가 $1$ 또는 $4$이고 나머지 두 수가 서로 같은 경우이다.
조건 (나)에 의하여

(i) $f(3) = 1$인 경우
$(f(1), f(2), f(3))$은 $(1, 1, 1)$
$(f(4), f(5), f(6))$의 개수는 ${}_{5}H_{3}$
$1 \times {}_{5}H_{3} = {}_{7}C_{3} = 35$

(ii) $f(3) = 2$인 경우
$(f(1), f(2), f(3))$은 $(1, 2, 2)$
$(f(4), f(5), f(6))$의 개수는 ${}_{4}H_{3}$
$1 \times {}_{4}H_{3} = {}_{6}C_{3} = 20$

(iii) $f(3) = 3$인 경우
$(f(1), f(2), f(3))$은 $(1, 3, 3)$
$(f(4), f(5), f(6))$의 개수는 ${}_{3}H_{3}$
$1 \times {}_{3}H_{3} = {}_{5}C_{3} = 10$

(iv) $f(3) = 4$인 경우
$(f(1), f(2), f(3))$은 $(1, 1, 4)$, $(1, 4, 4)$, $(2, 2, 4)$, $(3, 3, 4)$, $(4, 4, 4)$
$(f(4), f(5), f(6))$의 개수는 ${}_{2}H_{3}$
$5 \times {}_{2}H_{3} = 5 \times {}_{4}C_{3} = 20$

(v) $f(3) = 5$인 경우
$(f(1), f(2), f(3))$은 $(1, 5, 5)$, $(4, 5, 5)$
$(f(4), f(5), f(6))$의 개수는 ${}_{1}H_{3}$
$2 \times {}_{1}H_{3} = 2 \times {}_{3}C_{3} = 2$

따라서 (i) ~ (v)에 의하여 함수 $f$의 개수는
$35 + 20 + 10 + 20 + 2 = 87$

정답: ②
