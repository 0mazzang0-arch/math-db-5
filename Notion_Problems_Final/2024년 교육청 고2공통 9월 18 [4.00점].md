## 문제
$f(x) = \begin{cases} -2^{x} + 2 & (x < 1) \\ \log_{2} x & (x \geq 1) \end{cases}$ 에 대하여 $a-1 \leq x \leq a+1$ 에서 함수 $f(x)$ 의 최댓값과 최솟값의 차가 $1$ 이 되도록 하는 모든 실수 $a$ 의 값의 합은?



> [!example] 📸 (그림/그래프)

## 보기
① $3$   ② $\log_{2} \frac{32}{3}$   ③ $\log_{2} \frac{40}{3}$   ④ $4$   ⑤ $\log_{2} \frac{56}{3}$

## 풀이
$a-1 \leq x \leq a+1$ 에서 함수 $f(x)$ 의 최댓값을 $M$, 최솟값을 $m$ 이라 하자.

(ⅰ) $a < 0$ 인 경우
$a-1 < a+1 < 1$ 이므로
$M = f(a-1) = -2^{a-1} + 2$
$m = f(a+1) = -2^{a+1} + 2$
$M - m = (-2^{a-1} + 2) - (-2^{a+1} + 2)$
$= -2^{a-1} + 2^{a+1} = -\frac{1}{2} \cdot 2^{a} + 2 \cdot 2^{a} = \frac{3}{2} \cdot 2^{a} = 1$
$2^{a} = \frac{2}{3}$ 이므로 $a = \log_{2} \frac{2}{3}$

(ⅱ) $0 \leq a \leq 2$ 인 경우
$a-1 \leq 1 \leq a+1$ 이므로
① 최댓값이 $f(a-1)$ 인 경우
$M = f(a-1) = -2^{a-1} + 2$
$m = f(1) = \log_{2} 1 = 0$
$M - m = (-2^{a-1} + 2) - 0 = 1$
$2^{a-1} = 1$ 에서 $a-1 = 0$
$a = 1$

② 최댓값이 $f(a+1)$ 인 경우
$M = f(a+1) = \log_{2} (a+1)$
$m = f(1) = \log_{2} 1 = 0$
$M - m = \log_{2} (a+1) - 0 = 1$
$a+1 = 2$ 에서 $a = 1$
①, ②에 의하여 $a = 1$

(ⅲ) $a > 2$ 인 경우
$1 < a-1 < a+1$ 이므로
$M = f(a+1) = \log_{2} (a+1)$
$m = f(a-1) = \log_{2} (a-1)$
$M - m = \log_{2} (a+1) - \log_{2} (a-1) = 1$
$\log_{2} \frac{a+1}{a-1} = 1$
$\frac{a+1}{a-1} = 2$, $a+1 = 2a - 2$
$a = 3$

(ⅰ) ~ (ⅲ) 에 의하여 모든 실수 $a$ 의 값은 $\log_{2} \frac{2}{3}$, $1$, $3$ 이고 그 합은
$\log_{2} \frac{2}{3} + 1 + 3 = \log_{2} \frac{2}{3} + 4 = \log_{2} \frac{2}{3} + \log_{2} 16 = \log_{2} \frac{32}{3}$
