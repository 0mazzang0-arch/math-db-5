## 문제
$n \geq 2$인 자연수일 때, 부등식 
$$\sum_{i=1}^{n} \left(\frac{1}{2i-1} - \frac{1}{2i}\right) < \frac{1}{4} \left(3 - \frac{1}{n}\right) \cdots ㉠$$
을 수학적 귀납법으로 증명하는 과정이다.

(ⅰ) $n=2$일 때, 
$(\text{좌변}) = \left(1 - \frac{1}{2}\right) + \left(\frac{1}{3} - \frac{1}{4}\right) = \frac{7}{12}$
$(\text{우변}) = \frac{1}{4} \left(3 - \frac{1}{2}\right) = \frac{5}{8}$
이므로 성립한다. 

(ⅱ) $n=k$ ($k \geq 2$인 자연수)일 때, $㉠$이 성립한다고 가정하면 
$$\sum_{i=1}^{k} \left(\frac{1}{2i-1} - \frac{1}{2i}\right) < \frac{1}{4} \left(3 - \frac{1}{k}\right)$$
$(가)$를 더하면 
$$\sum_{i=1}^{k+1} \left(\frac{1}{2i-1} - \frac{1}{2i}\right) < \frac{1}{4} \left(3 - \frac{1}{k}\right) + (가)$$
$\frac{1}{4} \left(3 - \frac{1}{k}\right) + (가)$에서 $2(2k+1) > 4k$임을 이용하여 
$$\frac{1}{4} \left(3 - \frac{1}{k}\right) + (가) < \frac{1}{4} \left(3 - \frac{1}{k}\right) + (나) = \frac{3}{4} - \frac{1}{4(k+1)}$$
따라서 $n=k+1$일 때도 $㉠$이 성립한다. 

(ⅰ), (ⅱ)에 의하여 부등식 $㉠$은 $n \geq 2$인 모든 자연수에 대하여 성립한다. 
위의 과정에서 $(가)$, $(나)$에 알맞은 것은?

## 보기
① $(가): \frac{1}{2(2k+1)(k+1)}, (나): \frac{1}{2k(k+1)}$   ② $(가): \frac{1}{2(2k+1)(k+1)}, (나): \frac{1}{4k(k+1)}$   ③ $(가): \frac{1}{2k(2k-1)}, (나): \frac{1}{2k(k+1)}$   ④ $(가): \frac{1}{2k(2k-1)}, (나): \frac{1}{4k(k+1)}$   ⑤ $(가): \frac{1}{2k(2k-1)}, (나): \frac{1}{k(k+1)}$

## 풀이
(ⅰ) $n=2$일 때,
$(\text{좌변}) = \left(1-\frac{1}{2}\right) + \left(\frac{1}{3}-\frac{1}{4}\right) = \frac{1}{2} + \frac{1}{12} = \frac{7}{12}$
$(\text{우변}) = \frac{1}{4} \left(3-\frac{1}{2}\right) = \frac{1}{4} \cdot \frac{5}{2} = \frac{5}{8}$
$\frac{7}{12} < \frac{5}{8}$이므로 성립한다.

(ⅱ) $n=k$ ($k \geq 2$)일 때, $\sum_{i=1}^{k} \left(\frac{1}{2i-1} - \frac{1}{2i}\right) < \frac{1}{4} \left(3 - \frac{1}{k}\right)$가 성립한다고 가정하면, $n=k+1$일 때의 좌변을 만들기 위해 $(k+1)$번째 항을 더한다.
$(가) = \frac{1}{2(k+1)-1} - \frac{1}{2(k+1)} = \frac{1}{2k+1} - \frac{1}{2k+2} = \frac{1}{2(2k+1)(k+1)}$

가정된 부등식의 양변에 $(가)$를 더하면
$$\sum_{i=1}^{k+1} \left(\frac{1}{2i-1} - \frac{1}{2i}\right) < \frac{1}{4} \left(3 - \frac{1}{k}\right) + \frac{1}{2(2k+1)(k+1)}$$
이때, $2(2k+1) > 4k$이므로 $\frac{1}{2(2k+1)} < \frac{1}{4k}$이다. 양변에 $\frac{1}{k+1}$을 곱하면
$\frac{1}{2(2k+1)(k+1)} < \frac{1}{4k(k+1)}$ 이므로
$(나) = \frac{1}{4k(k+1)}$이다.

이를 정리하면
$\frac{1}{4} \left(3 - \frac{1}{k}\right) + \frac{1}{4k(k+1)} = \frac{3}{4} - \frac{1}{4k} + \frac{1}{4k(k+1)} = \frac{3}{4} - \frac{(k+1)-1}{4k(k+1)} = \frac{3}{4} - \frac{1}{4(k+1)}$
따라서 $n=k+1$일 때도 성립한다.

정답은 ②이다.
