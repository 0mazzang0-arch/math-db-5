## 문제
$7$ 이하의 두 자연수 $m$, $n$에 대하여 함수 
$f (x )= | 2^{m} \cos x-2^{n}|$
이 있다. $0 \le x \le 2 \pi$에서 방정식 
$\{f (x ) \}^{2}- (2^{5}+2^{4} )f (x )+2^{9}=0$
의 서로 다른 실근의 개수가 $6$이 되도록 하는 $m$, $n$의 모든 순서쌍 $(m,n )$의 개수는?

## 보기
① $2$   ② $4$   ③ $6$   ④ $8$   ⑤ $10$

## 풀이
$\{f (x ) \}^{2}- (2^{5}+2^{4} )f (x )+2^{9}$ 
$= (f (x )-2^{5} ) (f (x )-2^{4} )=0$ 이므로 
$f (x )=2^{5}$, $f (x )=2^{4}$이다. 
방정식 $\{f (x ) \}^{2}- (2^{5}+2^{4} )f (x )+2^{9}=0$의 서로 다른 실근의 개수는 함수 $y=f (x )$의 그래프가 두 직선 $y=2^{4}$, $y=2^{5}$과 각각 만나는 점의 개수의 합과 같다. 

(ⅰ) $m < n$ 일 때 

> [!example] 📸 (그림/그래프)

(ⅱ) $m=n$ 일 때 

> [!example] 📸 (그림/그래프)

(ⅰ)과 (ⅱ)에서 실근의 개수가 최대 $4$이므로 조건을 만족시키지 않는다. 

(ⅲ) $m > n$ 일 때 

> [!example] 📸 (그림/그래프)

서로 다른 실근의 개수가 $6$이 되려면 $16 \le 2^{m}-2^{n} < 32$, $2^{m}+2^{n} > 32$이므로 $2^{n}+16 \le 2^{m} < 2^{n}+32$이고 $2^{m} > 32-2^{n}$이다. 

$n=1$이면 $30 < 2^{m} < 34$이므로 $m=5$이다. 

$n=2$이면 $28 < 2^{m} < 36$이므로 $m=5$이다. 

$n=3$이면 $24 < 2^{m} < 40$이므로 $m=5$이다. 

$n=4$이면 $32 \le 2^{m} < 48$이므로 $m=5$이다. 

⑤ $n=5$이면 $48 \le 2^{m} < 64$이므로 부등식을 만족시키는 자연수 $m$은 존재하지 않는다. 

⑥ $n=6$이면 $80 \le 2^{m} < 96$이므로 부등식을 만족시키는 자연수 $m$은 존재하지 않는다. 

따라서 $m$, $n$의 모든 순서쌍 $(m,n )$의 개수는 $4$이다.
