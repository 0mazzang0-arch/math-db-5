## 문제
최고차항의 계수가 양수인 삼차함수 $f(x)$가 다음 조건을 만족시킨다.

(가) 함수 $f(x)$는 $x=0$에서 극댓값, $x=k$에서 극솟값을 가진다. (단, $k$는 상수이다.)
(나) $1$보다 큰 모든 실수 $t$에 대하여 $\int_{0}^{t} | f'(x) | dx = f(t) + f(0)$

위의 내용에서 옳은 것만을 <보기>에서 있는 대로 고른 것은?

## 보기
ㄱ. $\int_{0}^{k} f'(x) dx < 0$
ㄴ. $0 < k \le 1$
ㄷ. 함수 $f(x)$의 극솟값은 $0$이다.

① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
ㄱ. 조건 (가)에서 최고차항의 계수가 양수인 삼차함수 $f(x)$가 $x=0$에서 극대, $x=k$에서 극소이므로
$$f'(x) = ax(x-k) \quad (a>0, k>0)$$
구간 $[0, k]$에서 $f'(x) \le 0$이므로
$$\int_{0}^{k} f'(x) dx < 0$$
이다.

ㄴ. 조건 (나)에서
$$\int_{0}^{t} | f'(x) | dx = f(t) + f(0)$$
양변을 $t$에 대하여 미분하면
$$| f'(t) | = f'(t) \cdots ㉠$$
이 식은 $t > 1$인 모든 실수 $t$에 대하여 성립하므로
$$f'(t) \ge 0 \quad (t > 1)$$
$f(x)$는 $x=0$에서 극댓값, $x=k$에서 극솟값을 가지므로 도함수의 그래프 개형상 $k \le 1$이어야 한다.
따라서 $0 < k \le 1$이다.

ㄷ. $f'(x) = ax(x-k) = ax^2 - akx$에서
$$\int_{0}^{t} | f'(x) | dx$$
$$= -\int_{0}^{k} (ax^2 - akx) dx + \int_{k}^{t} (ax^2 - akx) dx$$
$$= -\left[ \frac{a}{3}x^3 - \frac{ak}{2}x^2 \right]_{0}^{k} + \left[ \frac{a}{3}x^3 - \frac{ak}{2}x^2 \right]_{k}^{t}$$
$$= -\left( \frac{ak^3}{3} - \frac{ak^3}{2} \right) + \left( \frac{at^3}{3} - \frac{akt^2}{2} - \frac{ak^3}{3} + \frac{ak^3}{2} \right)$$
$$= \frac{ak^3}{6} + \left( \frac{at^3}{3} - \frac{akt^2}{2} + \frac{ak^3}{6} \right)$$
$$= \frac{at^3}{3} - \frac{akt^2}{2} + \frac{ak^3}{3} \cdots ㉡$$
또한 $f(x) = \int (ax^2 - akx) dx = \frac{a}{3}x^3 - \frac{ak}{2}x^2 + C$ ($C$는 적분상수)라 하면
$$f(t) + f(0) = \left( \frac{a}{3}t^3 - \frac{ak}{2}t^2 + C \right) + C = \frac{a}{3}t^3 - \frac{ak}{2}t^2 + 2C \cdots ㉢$$
㉡, ㉢이 같아야 하므로 $2C = \frac{ak^3}{3}$에서 $C = \frac{ak^3}{6}$이다.
$f(x) = \frac{a}{3}x^3 - \frac{ak}{2}x^2 + \frac{ak^3}{6}$이므로 극솟값 $f(k)$는
$$f(k) = \frac{ak^3}{3} - \frac{ak^3}{2} + \frac{ak^3}{6} = 0$$
따라서 옳은 것은 ㄱ, ㄴ, ㄷ이다.
