## 문제
$a < 0$ 일 때, $x$ 에 대한 연립부등식 
$$ \left\{ \begin{array}{l} (x-a)^{2} < a^{2} \\ x^{2} + a < (a+1)x \end{array} \right. $$
의 해가 $b < x < b+1$ 이다. $a+b$ 의 값은? (단, $a, b$ 는 상수이다.)



> [!example] 📸 (그림/그래프)

## 보기
① $2$   ② $1$   ③ $0$   ④ $-1$   ⑤ $-2$

## 풀이
$a < 0$ 이므로 
$(x-a)^{2} < a^{2}$ 에서 
$x^{2} - 2ax + a^{2} < a^{2}$ 
$x(x-2a) < 0$ 
$2a < x < 0$ $\cdots$ ㉠ 

$x^{2} + a < (a+1)x$ 에서 
$x^{2} - (a+1)x + a < 0$ 
$(x-1)(x-a) < 0$ 
$a < x < 1$ $\cdots$ ㉡ 

㉠, ㉡을 모두 만족시키는 $x$ 의 값의 범위는 
$a < x < 0$ 
주어진 연립부등식의 해가 $b < x < b+1$ 이므로 
$a = b$, $b+1 = 0$ 에서 
$a = -1$, $b = -1$ 
따라서 $a+b = (-1) + (-1) = -2$ 

정답: ⑤
