## 문제
자연수 $n$에 대하여 함수 $y = \log_{c} |x|$의 그래프와 직선 $y = n$의 교점의 $x$좌표를 각각 $a_{n}$, $b_{n}$ ($a_{n} > b_{n}$)이라 할 때, 수열 $\{a_{n}\}$, $\{b_{n}\}$에 대하여 옳은 것만을 <보기>에서 있는 대로 고른 것은?

$$ \begin{array}{|l|} \hline \text{<보기>} \\ \text{ㄱ. } a_{n} + b_{n} = 0 \\ \text{ㄴ. } \lim_{n \to \infty} b_{n} = 0 \text{ 이면 } \sum_{n=1}^{\infty} a_{n} = \frac{c}{1-c} \text{ 이다.} \\ \text{ㄷ. } \sum_{n=1}^{\infty} \frac{1}{b_{n}} \text{ 이 발산하면 } \sum_{n=1}^{\infty} a_{n} \text{ 이 발산한다.} \\ \hline \end{array} $$

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
$y = \log_{c} |x|$와 $y = n$의 교점의 $x$좌표를 구하면 $|x| = c^{n}$에서 $a_{n} = c^{n}$이고 $b_{n} = -c^{n}$이다. 즉, $\{a_{n}\}$과 $\{b_{n}\}$은 공비가 $c$인 등비수열이다.

ㄱ. $a_{n} + b_{n} = c^{n} + (-c^{n}) = 0$ (참)

ㄴ. $\lim_{n \to \infty} b_{n} = 0$이면 $\lim_{n \to \infty} (-c^{n}) = 0$이므로 $0 < c < 1$이다.
따라서 $\sum_{n=1}^{\infty} a_{n} = \sum_{n=1}^{\infty} c^{n} = \frac{c}{1-c}$ (참)

ㄷ. $\sum_{n=1}^{\infty} \frac{1}{b_{n}}$이 발산하면 수열 $\{\frac{1}{b_{n}}\}$의 공비 $\frac{1}{c}$은 $\frac{1}{c} \ge 1$이다. 로그의 밑 조건에 의해 $c \neq 1$이므로 $\frac{1}{c} > 1$이고, 이는 $0 < c < 1$을 의미한다.
$0 < c < 1$이면 수열 $\{a_{n}\}$의 급수 $\sum_{n=1}^{\infty} a_{n}$은 수렴한다. (거짓)

따라서 옳은 것은 ㄱ, ㄴ이다.
