## 문제
$a < 0 < b < c$ 일 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?

$$ \begin{array}{|l|} \hline \text{<보 기>} \\ \text{ㄱ. } \frac{a}{b} < \frac{a}{c} \\ \text{ㄴ. } a + c < b \\ \text{ㄷ. } \frac{c}{a - b} < \frac{b}{a - c} \\ \hline \end{array} $$

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄷ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
ㄱ. $bc > 0$, $a < 0$, $c - b > 0$ 이므로 
$\frac{a}{b} - \frac{a}{c} = \frac{a ( c - b )}{bc} < 0$ 
$\therefore \frac{a}{b} < \frac{a}{c}$

ㄴ. [반례] $a = -1$, $b = 1$, $c = 3$ 이면 $a < 0 < b < c$ 가 성립하지만 
$a + c = -1 + 3 = 2 > b = 1$ 이 성립한다.

ㄷ. $a - b < 0$, $a - c < 0$, $b - c < 0$, $-a + b + c > 0$ 이므로 
$\frac{c}{a - b} - \frac{b}{a - c} = \frac{c ( a - c ) - b ( a - b )}{( a - b ) ( a - c )} = \frac{( b - c ) ( -a + b + c )}{( a - b ) ( a - c )} < 0$ 
$\therefore \frac{c}{a - b} < \frac{b}{a - c}$

이상에서 옳은 것은 ㄱ, ㄷ이다.
