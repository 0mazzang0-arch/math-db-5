## 문제
삼각형 $ABC$ 가 다음 조건을 만족시킨다.

(가) $\cos A = - \frac{1}{4}$
(나) $\sin B + \sin C = \frac{9}{8}$

삼각형 $ABC$ 의 넓이가 $\sqrt{15}$ 일 때, 삼각형 $ABC$ 의 외접원의 넓이는 $\frac{q}{p} \pi$ 이다. $p+q$ 의 값을 구하시오. (단, $p$ 와 $q$ 는 서로소인 자연수이다.)

## 보기

## 풀이
삼각형 $ABC$ 에서 $\overline{BC}=a$, $\overline{CA}=b$, $\overline{AB}=c$ 라 하고 삼각형 $ABC$ 의 외접원의 반지름의 길이를 $R$ 라 하자.
삼각형 $ABC$ 에서 사인법칙에 의해
$a=2R \sin A$, $b=2R \sin B$, $c=2R \sin C$ 이다.

조건 (가)에 의해
$\sin A = \sqrt{1 - \left( - \frac{1}{4} \right)^2} = \frac{\sqrt{15}}{4}$ 이므로
$a = 2R \times \frac{\sqrt{15}}{4} = \frac{\sqrt{15}}{2} R \cdots$ ㉠

조건 (나)에 의해
$b+c = 2R (\sin B + \sin C) = 2R \times \frac{9}{8} = \frac{9}{4} R \cdots$ ㉡

삼각형 $ABC$ 의 넓이가 $\sqrt{15}$ 이므로
$\frac{1}{2} bc \sin A = \frac{1}{2} bc \times \frac{\sqrt{15}}{4} = \sqrt{15}$ 에서
$bc = 8 \cdots$ ㉢

삼각형 $ABC$ 에서 코사인법칙에 의해
$a^2 = b^2 + c^2 - 2bc \cos A$
$a^2 = b^2 + c^2 - 2bc \times \left( - \frac{1}{4} \right) = b^2 + c^2 + \frac{1}{2} bc$
$a^2 = (b+c)^2 - 2bc + \frac{1}{2} bc = (b+c)^2 - \frac{3}{2} bc$

㉠, ㉡, ㉢에 의해
$\left( \frac{\sqrt{15}}{2} R \right)^2 = \left( \frac{9}{4} R \right)^2 - \frac{3}{2} \times 8$
$\frac{15}{4} R^2 = \frac{81}{16} R^2 - 12$
$12 = \left( \frac{81}{16} - \frac{60}{16} \right) R^2 = \frac{21}{16} R^2$
$R^2 = 12 \times \frac{16}{21} = \frac{64}{7}$

따라서 삼각형 $ABC$ 의 외접원의 넓이는 $\frac{64}{7} \pi$ 이므로
$p = 7$, $q = 64$ 이다.
$p+q = 7 + 64 = 71$

**정답: $71$**
