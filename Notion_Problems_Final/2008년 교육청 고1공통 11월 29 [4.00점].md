## 문제
$x$ 에 대한 일차부등식 $| x - a [ a ] | < b [ b ]$ 의 해가 $8 < x < 30$ 이다. 이를 만족하는 양수 $a$, $b$ 에 대하여 $8 a + 9 b$ 의 값을 구하시오. (단, $[ x ]$ 는 $x$ 보다 크지 않은 최대의 정수이다.)

## 풀이
$| x - a [ a ] | < b [ b ]$ 에서
$a [ a ] - b [ b ] < x < a [ a ] + b [ b ]$ 이므로 
$a [ a ] - b [ b ] = 8$ 이고 
$a [ a ] + b [ b ] = 30$ 이다. 
연립하여 풀면 
$a [ a ] = 19$, $b [ b ] = 11$ 

$[ a ] = n$ ($n$ 은 양의 정수)라 하면 
$n \le a < n + 1$ 이므로 
$n^{2} \le a [ a ] < n^{2} + n$ 
$n^{2} \le 19 < n^{2} + n$ 을 만족하는 $n = 4$ 이므로 
$a = \frac{19}{4}$ 

같은 방법으로 $[ b ] = m$ ($m$ 은 양의 정수)라 하면
$m^{2} \le b [ b ] < m^{2} + m$
$m^{2} \le 11 < m^{2} + m$ 을 만족하는 $m = 3$ 이므로
$b = \frac{11}{3}$ 

$\therefore 8 a + 9 b = 8 \times \frac{19}{4} + 9 \times \frac{11}{3} = 38 + 33 = 71$
