## 문제
$10 < a < b < 50$ 인 두 자연수 $a$, $b$ 에 대하여 $\log _{2} a$ 의 소수부분과 $\log _{2} b$ 의 소수부분이 같을 때 순서쌍 $(a, b)$ 의 개수는?

## 보기
① $15$   ② $16$   ③ $17$   ④ $18$   ⑤ $19$

## 풀이
임의의 실수 $x$ 에 대하여 $x = n + \alpha$ ($n$ 은 정수, $0 \le \alpha < 1$) 일 때, $n$ 을 $x$ 의 정수부분, $\alpha$ 를 $x$ 의 소수부분이라 하자.

$\log _{2} a$ 의 소수부분과 $\log _{2} b$ 의 소수부분이 같으므로
$\log _{2} b - \log _{2} a$ 는 정수이어야 한다.

$\log _{2} b - \log _{2} a = \log _{2} \frac{b}{a}$ 이고, $10 < a < b < 50$ 이므로
$\log _{2} 10 < \log _{2} a < \log _{2} 50$
$\log _{2} 10 < \log _{2} b < \log _{2} 50$

$\log _{2} \frac{b}{a}$ 의 범위를 구하면
$0 < \log _{2} \frac{b}{a} < \log _{2} 50 - \log _{2} 10 = \log _{2} 5$
$\log _{2} 5 = 2. \dots$ 이므로 정수인 $\log _{2} \frac{b}{a}$ 의 값은 $1$ 또는 $2$ 이다.

$1)$ $\log _{2} \frac{b}{a} = 1$ 일 때, $\frac{b}{a} = 2$ 즉, $b = 2a$
$10 < a < b < 50$ 에서 $10 < a < 2a < 50$ 이므로 $10 < a < 25$ 이다.
따라서 자연수 $a$ 는 $11, 12, \dots, 24$ 이며 그 개수는 $14$ 개이다.

$2)$ $\log _{2} \frac{b}{a} = 2$ 일 때, $\frac{b}{a} = 4$ 즉, $b = 4a$
$10 < a < b < 50$ 에서 $10 < a < 4a < 50$ 이므로 $10 < a < 12.5$ 이다.
따라서 자연수 $a$ 는 $11, 12$ 이며 그 개수는 $2$ 개이다.

따라서 구하는 순서쌍 $(a, b)$ 의 개수는
$14 + 2 = 16$

정답: ②
