## 문제
첫째항이 $16$이고 공비가 $2^{\frac{1}{10}}$인 등비수열 $\{ a_{n} \}$에 대하여 $\log a_{n}$의 소수부분을 $b_{n}$이라 하자. 
$b_{1}, b_{2}, b_{3}, \dots, b_{k-1}, b_{k}, b_{k+1} + 1$이 주어진 순서로 등차수열을 이룰 때, $k$의 값을 구하시오. (단, $\log 2 = 0.301$로 계산한다.)

## 풀이
$a_{n} = 16 \times (2^{\frac{1}{10}})^{n-1} = 2^{4} \times 2^{\frac{n-1}{10}} = 2^{\frac{n+39}{10}}$이므로 
$\log a_{n} = \log (2^{\frac{n+39}{10}}) = \left( \frac{n+39}{10} \right) \log 2$
$\log a_{n} = (0.1n + 3.9) \times 0.301$
$\log a_{n} = 0.0301n + 1.1739$
$\log a_{n} = 1 + (0.0301n + 0.1739)$

$b_{n}$은 $\log a_{n}$의 소수부분이므로 $0.0301n + 0.1739 < 1$을 만족하는 $n$의 범위를 구하면
$0.0301n < 0.8261$
$n < 27.445 \dots$

따라서 $1 \leq n \leq 27$일 때, $\log a_{n}$의 정수부분은 $1$이고 소수부분 $b_{n}$은
$b_{n} = 0.0301n + 0.1739$
이때 수열 $b_{1}, b_{2}, b_{3}, \dots, b_{27}$은 공차가 $0.0301$인 등차수열을 이룬다.

$n = 28$일 때, 
$\log a_{28} = 0.0301 \times 28 + 1.1739 = 0.8428 + 1.1739 = 2.0167$
이므로 소수부분 $b_{28} = 0.0167$이다.
$b_{28} + 1 = 1.0167$
이때 $0.0301 \times 28 + 0.1739 = 0.8428 + 0.1739 = 1.0167$이므로
$b_{28} + 1$은 앞선 소수부분의 식 $0.0301n + 0.1739$에 $n=28$을 대입한 값과 같다.

마찬가지로 $b_{29} + 1 = 0.0301 \times 29 + 0.1739$가 성립한다.
따라서 수열 $b_{1}, b_{2}, \dots, b_{27}, b_{28} + 1, b_{29} + 1$은 공차가 $0.0301$인 등차수열을 이룬다.
그러므로 수열 $b_{1}, b_{2}, \dots, b_{k}, b_{k+1} + 1$이 등차수열을 이루는 $k$의 값은 $27$이다.

정답: $27$
