## 문제

다항함수 $f(x)$가 다음 조건을 만족시킨다.

(가) $\lim_{x \to \infty} \frac{f(x)}{x^{4}} = 1$
(나) $f(1) = f'(1) = 1$

$-1 \leq n \leq 4$인 정수 $n$에 대하여 함수 $g(x)$를
$g(x) = f(x-n) + n$ ($n \leq x < n+1$)
이라 하자. 함수 $g(x)$가 열린구간 $(-1, 5)$에서 미분가능할 때, $\int_{0}^{4} g(x) dx = \frac{q}{p}$이다. $p+q$의 값을 구하시오. (단, $p$, $q$는 서로소인 자연수이다.)

## 풀이

함수 $g(x)$는
$$ \begin{array}{l} g(x) = \begin{cases} f(x+1)-1 & (-1 \leq x < 0) \\ f(x) & (0 \leq x < 1) \\ f(x-1)+1 & (1 \leq x < 2) \\ \vdots & \\ f(x-4)+4 & (4 \leq x < 5) \end{cases} \end{array} $$
이다. 함수 $g(x)$가 $x=1$에서 연속이므로
$g(1) = \lim_{x \to 1^-} g(x)$
$g(1) = f(0) + 1$이고 $\lim_{x \to 1^-} g(x) = \lim_{x \to 1^-} f(x) = f(1) = 1$이므로
$f(0) + 1 = 1$에서 $f(0) = 0$ $\cdots$ ㉠

함수 $g(x)$가 $x=1$에서 미분가능하므로
$\lim_{x \to 1^+} \frac{g(x)-g(1)}{x-1} = \lim_{x \to 1^-} \frac{g(x)-g(1)}{x-1}$이다.
$\lim_{x \to 1^+} \frac{g(x)-g(1)}{x-1} = \lim_{x \to 1^+} \frac{f(x-1)+1-g(1)}{x-1}$
$= \lim_{x \to 1^+} \frac{f(x-1)}{x-1} = \lim_{x \to 0^+} \frac{f(x)}{x} = f'(0)$
이고
$\lim_{x \to 1^-} \frac{g(x)-g(1)}{x-1} = \lim_{x \to 1^-} \frac{f(x)-f(1)}{x-1} = f'(1) = 1$
이므로
$f'(0) = 1$ $\cdots$ ㉡

$f(x)$는 최고차항의 계수가 $1$인 사차함수이므로
$f(x) = x^{4} + ax^{3} + bx^{2} + cx + d$라 하자.
㉠, ㉡에 의하여
$c = 1, d = 0$
조건 (나)에 의하여
$f(1) = 1 + a + b + 1 = 1$ $\cdots$ ㉢
$f'(1) = 4 + 3a + 2b + 1 = 1$ $\cdots$ ㉣
㉢, ㉣에 의하여
$a = -2$, $b = 1$
$f(x) = x^{4} - 2x^{3} + x^{2} + x$

$\int_{0}^{4} g(x) dx = \int_{0}^{1} g(x) dx + \int_{1}^{2} g(x) dx + \int_{2}^{3} g(x) dx + \int_{3}^{4} g(x) dx$
$= \frac{8}{15} + (\frac{8}{15} + 1) + (\frac{8}{15} + 2) + (\frac{8}{15} + 3)$
$= \frac{122}{15}$
그러므로 $p = 15$, $q = 122$
따라서 $p+q = 137$이다.
