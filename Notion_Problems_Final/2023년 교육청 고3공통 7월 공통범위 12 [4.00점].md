## 문제
모든 항이 정수이고 공차가 $5$인 등차수열 $\{a_{n}\}$과 자연수 $m$이 다음 조건을 만족시킨다.

(가) $\sum_{k=1}^{2m+1} a_{k} < 0$
(나) $|a_{m}| + |a_{m+1}| + |a_{m+2}| < 13$

$24 < a_{21} < 29$일 때, $m$의 값은?

## 보기
① $10$   ② $12$   ③ $14$   ④ $16$   ⑤ $18$

## 풀이
[정답] ③

등차수열 $\{a_{n}\}$의 첫째항을 $a$라 하자.
조건 (가)에 의하여
$\sum_{k=1}^{2m+1} a_{k} = \frac{(2m+1) \{2a + 5 \times (2m+1-1)\}}{2} = (2m+1)(a+5m) < 0$
$2m+1 > 0$이므로
$a+5m = a_{m+1} < 0$

(ⅰ) $a_{m+1} = -1$인 경우
$|a_{m}| + |a_{m+1}| + |a_{m+2}| = |-6| + |-1| + |4| = 11 < 13$이므로 조건을 만족한다.
이때 $a_{m+1} = -1$이므로
$a_{m+6} = -1 + 5 \times 5 = 24$
$a_{m+7} = -1 + 6 \times 5 = 29$
$24 < a_{21} < 29$인 등차수열의 항 $a_{21}$이 존재하지 않는다. (모든 항은 정수이며 공차가 $5$이므로 $24$ 다음 항은 $29$이다.)

(ⅱ) $a_{m+1} = -2$인 경우
$|a_{m}| + |a_{m+1}| + |a_{m+2}| = |-7| + |-2| + |3| = 12 < 13$이므로 조건을 만족한다.
이때 $a_{m+1} = -2$이므로
$a_{m+7} = -2 + 6 \times 5 = 28$
$24 < 28 < 29$이므로 $a_{21} = 28$이 될 수 있다.
따라서 $m+7 = 21$이므로 $m = 14$이다.

(ⅲ) $a_{m+1} \le -3$인 경우
$|a_{m}| + |a_{m+1}| + |a_{m+2}| = |a_{m+1}-5| + |a_{m+1}| + |a_{m+1}+5|$
$a_{m+1} \le -3$일 때, $a_{m+1}-5 \le -8$이고 $a_{m+1}+5 \le 2$이다.
만약 $a_{m+1} = -3$이면 $|-8| + |-3| + |2| = 13$이 되어 조건 (나)를 만족하지 않는다.
$a_{m+1}$의 값이 더 작아지면 절댓값의 합은 $13$보다 커지게 된다.

따라서 (ⅰ), (ⅱ), (ⅲ)에 의하여 $m = 14$이다.
