## 문제
부등식 $\log_{3} (x-3) + \log_{3} (x+1) < 1 + \log_{3} 4$의 해가 $a < x < b$일 때, $ab$의 값을 구하시오.

## 보기

## 풀이
$$\log_{3} (x-3) + \log_{3} (x+1) < 1 + \log_{3} 4$$
$$\log_{3} (x-3)(x+1) < \log_{3} 12$$ 에서
$$(x-3)(x+1) < 12$$
$$x^{2} - 2x - 15 < 0$$
$$(x-5)(x+3) < 0$$
$$-3 < x < 5$$
밑 조건에 의하여 $x > 3$ 이므로 $3 < x < 5$ 이다.
$\therefore a = 3, b = 5$ 이므로 $ab = 15$

정답: $15$
