## 문제
함수 $y = \sqrt{2} \sin x + 3\cos \left( x + \frac{\pi}{4} \right)$는 $x = \theta$일 때 최댓값을 갖는다. $\tan \theta$의 값은?

## 보기
① $-\frac{2}{3}$   ② $-\frac{1}{3}$   ③ $0$   ④ $\frac{1}{3}$   ⑤ $\frac{2}{3}$

## 풀이
주어진 식을 전개하면 다음과 같다.
$$y = \sqrt{2} \sin x + 3\cos \left( x + \frac{\pi}{4} \right)$$
$$= \sqrt{2} \sin x + 3 \left( \cos x \cos \frac{\pi}{4} - \sin x \sin \frac{\pi}{4} \right)$$
$$= \sqrt{2} \sin x + 3 \left( \frac{\sqrt{2}}{2} \cos x - \frac{\sqrt{2}}{2} \sin x \right)$$
$$= \frac{3\sqrt{2}}{2} \cos x - \frac{\sqrt{2}}{2} \sin x$$

삼각함수의 합성에 의해
$$y = \sqrt{5} \cos(x + \alpha)$$
(이 때, $\tan \alpha = \frac{\frac{\sqrt{2}}{2}}{\frac{3\sqrt{2}}{2}} = \frac{1}{3}$이다.)

주어진 함수는 $x + \alpha = 2n\pi$ ($n$은 정수)일 때 최댓값을 가지므로
$x = 2n\pi - \alpha$
즉, $\theta = 2n\pi - \alpha$일 때 최댓값을 갖는다.

따라서 구하는 값은
$$\tan \theta = \tan(2n\pi - \alpha) = -\tan \alpha = -\frac{1}{3}$$

정답: ②
