## 문제
두 복소수 $z_{1}$, $z_{2}$ 가 $\overline{z_{1}} - \overline{z_{2}} = 1+2i$, $\overline{z_{1}} \overline{z_{2}} = 4-3i$ 를 만족시킬 때, $(z_{1}-1)(z_{2}+1)$ 의 값은? (단, $i = \sqrt{-1}$ 이고, $\overline{z}$ 는 $z$ 의 켤레복소수이다.)

## 보기
① $4$   ② $4+5i$   ③ $4-5i$   ④ $4+i$   ⑤ $4-i$

## 풀이
$\overline{z_{1}} - \overline{z_{2}} = \overline{z_{1}-z_{2}} = 1+2i$ 이므로 
$z_{1}-z_{2} = 1-2i$ 

$\overline{z_{1}} \overline{z_{2}} = \overline{z_{1}z_{2}} = 4-3i$ 이므로 
$z_{1}z_{2} = 4+3i$ 

$\therefore ( z_{1} - 1 ) ( z_{2} + 1 ) = z_{1} z_{2} + z_{1} - z_{2} - 1$ 
$= ( 4+3i ) + ( 1-2i ) - 1$ 
$= 4+i$
