## 문제
방정식 $x^{3} = 1$의 한 허근을 $\omega$라고 할 때, 자연수 $n$에 대하여 함수 $f(n)$을 다음과 같이 정의한다. 

$$f(n) = \frac{\omega^{2n}}{\omega^{n} + 1}$$

이때, $f(1) + f(2) + f(3) + \dots + f(20)$의 값을 구하시오.

## 보기

## 풀이
$x^{3} - 1 = (x - 1)(x^{2} + x + 1) = 0$에서 $x^{2} + x + 1 = 0$의 한 허근이 $\omega$이므로
$\omega^{3} = 1$, $\omega^{2} + \omega + 1 = 0$, $\omega^{2} + \omega = -1$, $\omega^{2} + 1 = -\omega$이다.

$f(1) = f(4) = \dots = f(19) = \frac{\omega^{2}}{\omega + 1} = \frac{\omega^{2}}{-\omega^{2}} = -1$
$f(2) = f(5) = \dots = f(20) = \frac{\omega^{4}}{\omega^{2} + 1} = \frac{\omega}{-\omega} = -1$
$f(3) = f(6) = \dots = f(18) = \frac{\omega^{6}}{\omega^{3} + 1} = \frac{1}{1 + 1} = \frac{1}{2}$

$\therefore f(1) + f(2) + f(3) + \dots + f(20)$
$= 6 \times \left( -1 - 1 + \frac{1}{2} \right) + (-1) + (-1)$
$= 6 \times \left( -\frac{3}{2} \right) - 2$
$= -9 - 2$
$= -11$

정답: $-11$
