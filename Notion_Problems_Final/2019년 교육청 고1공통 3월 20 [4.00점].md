## 문제

좌표평면에서 두 이차함수 
$y=x^{2}-2x+1$, $y=-\frac{1}{2}x^{2}+3x-\frac{5}{2}$ 
의 그래프가 $x$축에 수직인 직선과 만나는 두 점을 각각 $A$, $B$라 하자. 다음은 점 $C(k, 0)$에 대하여 삼각형 $ABC$가 정삼각형이 되도록 하는 양수 $k$의 값을 구하는 과정이다. 

$$ \begin{array}{|l|} \hline \text{두 점 } A, B \text{를 지나는 직선의 방정식을 } x=t \text{라 하고} \\ \text{직선 } x=t \text{와 } x \text{축과의 교점을 } D \text{라 하자.} \\ \text{삼각형 } ABC \text{가 정삼각형이 되기 위해서는 직선 } CD \text{가 선분 } AB \text{를} \\ \text{수직이등분해야 한다.} \\ \text{그러므로 } \overline{AD} = \overline{BD} \text{에서} \\ t^{2} + (가) = 0 \\ t=1 \text{ 또는 } t= (나) \\ \text{이때 } t=1 \text{인 경우는 조건을 만족시키지 않고} \\ t= (나) \text{인 경우는 조건을 만족시킨다.} \\ \text{따라서 양수 } k \text{의 값은 } (다) \text{이다.} \\ \hline \end{array} $$

위의 (가)에 알맞은 식을 $f(t)$라 하고 (나), (다)에 알맞은 수를 각각 $a$, $b$라 할 때, $f(a)+b$의 값은?



> [!example] 📸 (그림/그래프)

## 보기

① $-12+16\sqrt{3}$   ② $-11+16\sqrt{3}$   ③ $-12+17\sqrt{3}$   ④ $-12+18\sqrt{2}$   ⑤ $-11+18\sqrt{2}$

## 풀이

두 점 $A$, $B$를 지나는 직선의 방정식을 $x=t$라 하면 두 점 $A$, $B$의 $x$좌표는 $t$이므로 이차함수 $y=x^{2}-2x+1$에 $x=t$를 대입하여 점 $A$의 좌표를 구하면 다음과 같다. 
$A(t, t^{2}-2t+1)$

마찬가지로 이차함수 $y=-\frac{1}{2}x^{2}+3x-\frac{5}{2}$에 $x=t$를 대입하여 점 $B$의 좌표를 구하면 다음과 같다. 
$B(t, -\frac{1}{2}t^{2}+3t-\frac{5}{2})$

삼각형 $ABC$가 정삼각형이 되기 위해서는 직선 $CD$가 선분 $AB$를 수직이등분해야 하므로 두 점 $A$, $B$에서 점 $D(t, 0)$를 잇는 두 선분의 길이가 서로 같아야 한다. 
따라서 $\overline{AD} = \overline{BD}$이다. 

선분 $\overline{AD}$의 길이는 점 $A$의 $y$좌표와 같으므로 다음과 같다. 
$\overline{AD} = t^{2}-2t+1$

선분 $\overline{BD}$의 길이는 점 $B$의 $y$좌표의 부호를 바꾼 것과 같으므로 다음과 같다. 
$\overline{BD} = -(-\frac{1}{2}t^{2}+3t-\frac{5}{2}) = \frac{1}{2}t^{2}-3t+\frac{5}{2}$

두 선분의 길이가 같아야 하므로 
$t^{2}-2t+1 = \frac{1}{2}t^{2}-3t+\frac{5}{2}$
정리하면 
$\frac{1}{2}t^{2}+t-\frac{3}{2} = 0$
양변에 $2$를 곱하면 
$t^{2}+2t-3 = 0$
$(t-1)(t+3) = 0$
따라서 $t=1$ 또는 $t=-3$이다. 

(ⅰ) $t=1$인 경우 
$\overline{AD} = 1^{2}-2+1=0$이므로 $\overline{BD} = \overline{AD} = 0$이 되어 삼각형이 만들어지지 않는다. 

(ⅱ) $t=-3$인 경우 
$\overline{AD} = (-3)^{2}-2 \times (-3)+1 = 16$이므로 삼각형 $ABC$의 한 변의 길이는 
$16 \times 2 = 32$이다. 

삼각형 $ABC$의 높이는 
$\frac{\sqrt{3}}{2} \times 32 = 16\sqrt{3}$이다. 

점 $C$의 $x$좌표인 $k$는 점 $D$의 $x$좌표 $t$에서 높이만큼 떨어진 곳에 위치하므로 
$k = -3+16\sqrt{3}$ 또는 $k = -3-16\sqrt{3}$이다. 

(ⅰ), (ⅱ)에 의해 양수 $k$의 값은 $b = -3+16\sqrt{3}$이다. 

(가)에 알맞은 식 $f(t) = 2t-3$, (나)에 알맞은 수 $a = -3$, (다)에 알맞은 수 $b = -3+16\sqrt{3}$이므로 구하는 값은 다음과 같다. 
$f(a)+b = f(-3)+b$
$= 2 \times (-3)-3+(-3+16\sqrt{3})$
$= -6-3-3+16\sqrt{3}$
$= -12+16\sqrt{3}$

따라서 정답은 ①이다.
