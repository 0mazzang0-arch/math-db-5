## 문제
부등식 $a^{x-1} < a^{2x+1}$ 의 해가 $x < -2$ 일 때, 부등식 $\log_{a}(x-2) < \log_{a}(4-x)$ 의 해는? (단, $a$ 는 $1$ 이 아닌 양수이다.)

## 보기
① $2 < x < 3$   ② $3 < x < 4$   ③ $2 < x < 4$   ④ $x < 3$   ⑤ $x > 3$

## 풀이
부등식 $a^{x-1} < a^{2x+1}$ 의 해가 $x < -2$ 이려면 지수의 대소 관계가 바뀌어야 하므로
$x-1 > 2x+1$
$\iff x < -2$
따라서 밑의 조건은 $0 < a < 1$ 이다.

부등식 $\log_{a}(x-2) < \log_{a}(4-x)$ 에서 밑이 $0 < a < 1$ 이므로
$x-2 > 4-x$
$2x > 6$
$\therefore x > 3$ $\cdots$ ㉠

진수는 양수이어야 하므로
$x-2 > 0$ 이고 $4-x > 0$
$x > 2$ 이고 $x < 4$
$\therefore 2 < x < 4$ $\cdots$ ㉡

㉠, ㉡의 공통 범위를 구하면
$3 < x < 4$ 이다.

정답: ②
