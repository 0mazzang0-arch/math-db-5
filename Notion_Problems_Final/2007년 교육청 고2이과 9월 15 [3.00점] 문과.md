## 문제

다음은 임의의 양수 $x$에 대하여 $\log x$의 정수부분과 $\log \sqrt{10} x$의 정수부분의 합이 $\log x^2$의 정수부분과 같음을 보인 것이다.

$$ \begin{array}{|l|} \hline \log x = n + \alpha \text{ (} n \text{은 정수, } 0 \leq \alpha < 1 \text{)라 하면} \\ \log \sqrt{10} x = (가) , \log x^2 = 2n + 2\alpha \text{이다.} \\ (ⅰ) \text{ } 0 \leq \alpha < \frac{1}{2} \text{이면} \\ \log x \text{의 정수부분과 } \log \sqrt{10} x \text{의 정수부분의 합은 } (나) \text{ 이고, } \\ \text{이것은 } \log x^2 \text{의 정수부분과 같다.} \\ (ⅱ) \text{ } \frac{1}{2} \leq \alpha < 1 \text{이면} \\ \log x \text{의 정수부분과 } \log \sqrt{10} x \text{의 정수부분의 합은 } (다) \text{ 이고, } \\ \text{이것은 } \log x^2 \text{의 정수부분과 같다.} \\ \text{따라서 (ⅰ), (ⅱ)에 의해 } \log x \text{의 정수부분과 } \log \sqrt{10} x \text{의 정수부분의 } \\ \text{합은 } \log x^2 \text{의 정수부분과 같다.} \\ \hline \end{array} $$

위의 과정에서 $(가), (나), (다)$에 알맞은 것은?

## 보기

① $(가): n + \alpha + \frac{1}{2}, (나): 2n, (다): 2n$
② $(가): n + \alpha + \frac{1}{2}, (나): 2n, (다): 2n + 1$
③ $(가): n + \alpha + \frac{1}{2}, (나): 2n + 1, (다): 2n + 1$
④ $(가): n + \alpha + 1, (나): 2n, (다): 2n + 2$
⑤ $(가): n + \alpha + 1, (나): 2n + 1, (다): 2n + 2$

## 풀이

$\log x = n + \alpha$ ($n$은 정수, $0 \leq \alpha < 1$)라 하면
$\log \sqrt{10} x = \log 10^{\frac{1}{2}} + \log x = n + \alpha + \frac{1}{2}$ 이므로 $(가) = n + \alpha + \frac{1}{2}$ 이다.
또한 $\log x^2 = 2n + 2\alpha$ 이다.

(ⅰ) $0 \leq \alpha < \frac{1}{2}$ 이면:
$\log x$의 정수부분은 $n$이다.
$\log \sqrt{10} x = n + (\alpha + \frac{1}{2})$에서 $0.5 \leq \alpha + \frac{1}{2} < 1$이므로 $\log \sqrt{10} x$의 정수부분은 $n$이다.
따라서 이들의 합은 $n + n = 2n$이다. $\therefore (나) = 2n$
이때 $\log x^2 = 2n + 2\alpha$에서 $0 \leq 2\alpha < 1$이므로 정수부분은 $2n$으로 서로 같다.

(ⅱ) $\frac{1}{2} \leq \alpha < 1$ 이면:
$\log x$의 정수부분은 $n$이다.
$\log \sqrt{10} x = n + (\alpha + \frac{1}{2})$에서 $1 \leq \alpha + \frac{1}{2} < 1.5$이므로 $\log \sqrt{10} x$의 정수부분은 $n + 1$이다.
따라서 이들의 합은 $n + (n + 1) = 2n + 1$이다. $\therefore (다) = 2n + 1$
이때 $\log x^2 = 2n + 2\alpha$에서 $1 \leq 2\alpha < 2$이므로 정수부분은 $2n + 1$로 서로 같다.

따라서 (ⅰ), (ⅱ)에 의해 $\log x$의 정수부분과 $\log \sqrt{10} x$의 정수부분의 합은 $\log x^2$의 정수부분과 같다.
