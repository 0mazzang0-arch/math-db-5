## 문제
좌표평면에서 직선 $y=1$ 위의 점 $P_{n}(n, 1)$에 대하여 수열 $\{a_{n}\}$을 다음과 같이 정의한다.

$a_{n} = (\text{선분 } OP_{n} \text{의 길이}) \quad (n=1, 2, 3, \dots)$

> [!example] 📸 (그림/그래프)

$\lim_{n \to \infty} \frac{[a_{1}] + [a_{2}] + [a_{3}] + \dots + [a_{n}]}{n^{2}}$의 값은?
(단, $O$는 원점이고, $[x]$는 $x$보다 크지 않은 최대의 정수이다.)

## 보기
① $\frac{1}{3}$   ② $\frac{1}{2}$   ③ $\frac{2}{3}$   ④ $1$   ⑤ $\frac{3}{2}$

## 풀이
주어진 조건에서 $a_{n} = \sqrt{n^{2} + 1}$이다.

$n < \sqrt{n^{2} + 1} < \sqrt{n^{2} + 2n + 1} = n + 1$이므로 $[a_{n}] = n$이다.

$\lim_{n \to \infty} \frac{[a_{1}] + [a_{2}] + [a_{3}] + \dots + [a_{n}]}{n^{2}}$
$= \lim_{n \to \infty} \frac{1 + 2 + 3 + \dots + n}{n^{2}}$
$= \lim_{n \to \infty} \frac{\frac{n(n + 1)}{2}}{n^{2}}$
$= \lim_{n \to \infty} \frac{n(n + 1)}{2n^{2}} = \frac{1}{2}$

정답: ②
