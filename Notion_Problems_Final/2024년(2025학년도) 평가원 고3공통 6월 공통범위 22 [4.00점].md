## 문제
수열 $\{ a_{n} \}$ 은 $a_{2} = -a_{1}$ 이고, $n \geq 2$ 인 모든 자연수 $n$ 에 대하여
$$ a_{n+1} = \left\{ \begin{array}{ll} a_{n} - \sqrt{n} a_{\sqrt{n}} & (\sqrt{n} \text{ 이 자연수이고 } a_{n} > 0 \text{ 인 경우}) \\ a_{n} + 1 & (\text{그 외의 경우}) \end{array} \right. $$
를 만족시킨다. $a_{15} = 1$ 이 되도록 하는 모든 $a_{1}$ 의 값의 곱을 구하시오.

## 풀이
$a_{1} = -\alpha$, $a_{2} = \alpha$ 라 하면
$a_{3} = \alpha + 1$, $a_{4} = \alpha + 2$

(ⅰ) $a_{4} \le 0$, $a_{9} \le 0$ 인 경우
$a_{4} = \alpha + 2$ ($\alpha \le -2$) 이므로 $a_{9} = \alpha + 7$ ($\alpha \le -7$)
$a_{15} = \alpha + 13$
$a_{15} = 1$ 이므로 $\alpha + 13 = 1$
$\alpha = -12$ 이므로 $a_{1} = 12$

(ⅱ) $a_{4} \le 0$, $a_{9} > 0$ 인 경우
$a_{4} = \alpha + 2$ ($\alpha \le -2$) 이므로 $a_{9} = \alpha + 7$
$a_{9} > 0$ 이므로 $\alpha > -7$
$a_{10} = a_{9} - 3 a_{3} = \alpha + 7 - 3 ( \alpha + 1 ) = -2\alpha + 4$
$\therefore a_{15} = -2\alpha + 9$
$a_{15} = 1$ 이므로 $-2\alpha + 9 = 1$
$\therefore \alpha = 4$
$\alpha \le -2$ 이므로 모순이다.

(ⅲ) $a_{4} > 0$, $a_{9} \le 0$ 인 경우
$a_{4} = \alpha + 2$ 이므로 $\alpha > -2$
$a_{5} = a_{4} - 2a_{2} = \alpha + 2 - 2\alpha = -\alpha + 2$
$a_{9} = -\alpha + 6$ 이므로 $\alpha \ge 6$
$\therefore a_{15} = -\alpha + 12$
$a_{15} = 1$ 이므로 $-\alpha + 12 = 1$
$\therefore \alpha = 11$
$\therefore a_{1} = -11$

(ⅳ) $a_{4} > 0$, $a_{9} > 0$ 인 경우
$a_{4} = \alpha + 2$ 에서 $\alpha > -2$
$a_{5} = a_{4} - 2a_{2} = \alpha + 2 - 2\alpha = -\alpha + 2$
$a_{9} = -\alpha + 6$ 이므로 $\alpha < 6$
$a_{10} = a_{9} - 3 a_{3} = -\alpha + 6 - 3 ( \alpha + 1 ) = -4\alpha + 3$
$\therefore a_{15} = -4\alpha + 8$
$a_{15} = 1$ 이므로 $-4\alpha + 8 = 1$
$\therefore \alpha = \frac{7}{4}$
$\therefore a_{1} = -\frac{7}{4}$

이상에서 조건을 만족시키는 $a_{1}$ 은 $12$, $-11$, $-\frac{7}{4}$ 이므로 모든 $a_{1}$ 의 값의 곱은
$12 \times ( -11 ) \times ( - \frac{7}{4} ) = 231$
