## 문제
다음은 $a$, $b$가 $1$이 아닌 양의 실수일 때,
$log_{a} b = log_{b} a$이면 $\frac{a^{2} + 1}{b^{2} + 1} = \frac{a}{b}$이다. $\cdots \cdots$ $(＊)$
가 성립함을 증명한 것이다.

---
$log_{b} a = \frac{1}{(가)}$이고 가정에서 $log_{a} b = log_{b} a$이므로
$log_{a} b = 1$ 또는 $log_{a} b = -1$이다.

$(ⅰ)$ $log_{a} b = 1$일 때,
$\frac{a^{2} + 1}{b^{2} + 1} = (나)$이고 $\frac{a}{b} = (나)$이다.

$(ⅱ)$ $log_{a} b = -1$일 때,
$\frac{a^{2} + 1}{b^{2} + 1} = (다)$이고 $\frac{a}{b} = (다)$이다.

따라서, $(ⅰ)$, $(ⅱ)$에 의하여 $\frac{a^{2} + 1}{b^{2} + 1} = \frac{a}{b}$이므로 $(＊)$가 성립한다.
---

위 증명에서 $(가)$, $(나)$, $(다)$에 알맞은 것은?

## 보기
① $(가): log_{a} b, (나): -1, (다): b^{2}$   ② $(가): log_{b} \frac{1}{a}, (나): -1, (다): ab$   ③ $(가): log_{a} b, (나): 1, (다): a^{2}$   ④ $(가): log_{b} \frac{1}{a}, (나): -1, (다): a^{2}$   ⑤ $(가): log_{a} b, (나): 1, (다): b^{2}$

## 풀이
[정답] ③

[해설]
로그의 밑변환 공식에 의해 $log_{b} a = \frac{1}{log_{a} b}$이므로
$(가): log_{a} b$

$(ⅰ)$ $log_{a} b = 1$일 때:
$a = b$이므로
$\frac{a^{2} + 1}{b^{2} + 1} = \frac{a^{2} + 1}{a^{2} + 1} = 1$, $\frac{a}{b} = \frac{a}{a} = 1$이다.
따라서 $(나): 1$

$(ⅱ)$ $log_{a} b = -1$일 때:
$b = a^{-1} = \frac{1}{a}$이므로
$\frac{a^{2} + 1}{b^{2} + 1} = \frac{a^{2} + 1}{(\frac{1}{a})^{2} + 1} = \frac{a^{2} + 1}{\frac{1 + a^{2}}{a^{2}}} = (a^{2} + 1) \cdot \frac{a^{2}}{a^{2} + 1} = a^{2}$이다.
또한 $\frac{a}{b} = \frac{a}{\frac{1}{a}} = a^{2}$이다.
따라서 $(다): a^{2}$

종합하면 다음과 같다.
$$ \begin{array}{c|c|c} (가) & (나) & (다) \\ \hline log_{a} b & 1 & a^{2} \end{array} $$
