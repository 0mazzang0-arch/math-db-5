## 문제
두 다항식 $A = x^{2} - 1$, $B = x^{2} + 2x + 7$에 대하여 $2A + B$를 간단히 하면?

## 보기
① $2x^{2} - 2x + 5$   ② $2x^{2} + 2x + 5$   ③ $3x^{2} - 2x + 5$   ④ $3x^{2} + 2x + 5$   ⑤ $3x^{2} + 4x + 5$

## 풀이
두 다항식 $A = x^{2} - 1$, $B = x^{2} + 2x + 7$에서
$2A + B$
$= 2(x^{2} - 1) + (x^{2} + 2x + 7)$
$= 2x^{2} - 2 + x^{2} + 2x + 7$
$= 3x^{2} + 2x + 5$

따라서 정답은 ④이다.
