## 문제
$1$보다 큰 세 자연수 $a$, $b$, $c$에 대하여 세 수 $\log a$, $\log b$, $\log c$가 이 순서대로 공차가 자연수인 등차수열을 이룬다. $\log abc = 15$일 때, $\log \frac{ac^{2}}{b}$의 최댓값은?

## 보기
① $11$   ② $12$   ③ $13$   ④ $14$   ⑤ $15$

## 풀이
$\log a$, $\log b$, $\log c$가 이 순서대로 등차수열을 이루므로 
$2 \log b = \log a + \log c$ 
$\log b^{2} = \log ac$, $b^{2} = ac$ 

$\log abc = \log b^{3} = 15$이므로 
$\log b = 5$ 

$\log a + \log b + \log c = 15$를 만족시키고 공차가 자연수인 등차수열 $\log a$, $\log b$, $\log c$의 순서쌍 $(\log a, \log b, \log c)$는 $a, b, c$가 $1$보다 큰 자연수이므로 $\log a, \log b, \log c > 0$이어야 한다.
따라서 가능한 순서쌍은 
$(4, 5, 6)$, $(3, 5, 7)$, $(2, 5, 8)$, $(1, 5, 9)$이다.

$\log \frac{ac^{2}}{b} = \log \frac{ac \cdot c}{b} = \log \frac{b^{2} \cdot c}{b} = \log bc$
$= \log b + \log c = 5 + \log c$

따라서 $\log c$가 최대일 때 전체 값이 최대가 되므로 $\log c = 9$일 때,
$\log \frac{ac^{2}}{b}$의 최댓값은 
$5 + 9 = 14$
