## 문제
다항식 $P(x) = x^{3} + x^{2} + x + 1$을 $x-k$로 나눈 나머지와 $x+k$로 나눈 나머지의 합이 $8$이다. $P(x)$를 $x-k^{2}$으로 나눈 나머지를 구하시오. (단, $k$는 상수이다.)

## 풀이
나머지정리에 의하여 다항식 $P(x)$를 $x-k$로 나눈 나머지는 
$P(k) = k^{3} + k^{2} + k + 1$
다항식 $P(x)$를 $x+k$로 나눈 나머지는 
$P(-k) = -k^{3} + k^{2} - k + 1$
나머지의 합이 $8$이므로 
$P(k) + P(-k) = (k^{3} + k^{2} + k + 1) + (-k^{3} + k^{2} - k + 1)$
$= 2k^{2} + 2 = 8$
$2k^{2} = 6$
$k^{2} = 3$
다항식 $P(x)$를 $x-k^{2}$으로 나눈 나머지는 $P(k^{2})$이므로
$P(k^{2}) = (k^{2})^{3} + (k^{2})^{2} + k^{2} + 1$
$= 3^{3} + 3^{2} + 3 + 1$
$= 27 + 9 + 3 + 1 = 40$
따라서 정답은 $40$이다.
