## 문제
$n$이 자연수일 때, 집합 $A_{n}$을 다음과 같이 정의한다.
$$A_{1} = \emptyset, A_{n+1} = A_{n} \cup \{ A_{n} \}$$
예를 들어 $A_{2} = \{ \emptyset \}$, $A_{3} = \{ \emptyset, \{ \emptyset \} \}$이다.
이때, <보기> 중 옳은 것을 모두 고르면? (단, $\emptyset$는 공집합)

**<보기>**
ㄱ. $A_{4} = \{ \emptyset, \{ \emptyset \}, \{ \emptyset, \{ \emptyset \} \} \}$
ㄴ. $A_{n} \in A_{n+1}$
ㄷ. $A_{n+1} = \{ A_{1}, A_{2}, A_{3}, \dots, A_{n} \}$

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
$A_{n+1} = A_{n} \cup \{ A_{n} \}$에서 

ㄱ. $A_{4} = A_{3} \cup \{ A_{3} \}$이므로 
$A_{4} = \{ \emptyset, \{ \emptyset \}, \{ \emptyset, \{ \emptyset \} \} \}$ $\therefore$ 참 

ㄴ. $A_{n+1} = A_{n} \cup \{ A_{n} \}$의 정의에 의해 $\{ A_{n} \} \subset A_{n+1}$이므로
$A_{n} \in A_{n+1}$ $\therefore$ 참 

ㄷ.
$A_{1} = \emptyset$
$A_{2} = \{ \emptyset \} = \{ A_{1} \}$
$A_{3} = \{ \emptyset, \{ \emptyset \} \} = \{ A_{1}, A_{2} \}$
$A_{4} = \{ \emptyset, \{ \emptyset \}, \{ \emptyset, \{ \emptyset \} \} \} = \{ A_{1}, A_{2}, A_{3} \}$
$\vdots$
$A_{n+1} = \{ A_{1}, A_{2}, A_{3}, \dots, A_{n} \}$ $\therefore$ 참 

따라서 ㄱ, ㄴ, ㄷ 모두 옳다.

정답: ⑤
