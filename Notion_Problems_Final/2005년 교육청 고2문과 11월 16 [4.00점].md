## 문제
정수부분이 세 자리인 양의 실수 $x$ 에 대하여 $\log _{10} x$ 의 소수부분이 $\log _{10} \frac{1}{x}$ 의 소수부분의 $2$ 배일 때, $\log _{10} x + \log _{10} x^{2} + \cdots + \log _{10} x^{9}$ 의 값은? (단, $\log _{10} x$ 의 소수부분은 $0$ 이 아니다.)

## 보기
① $60$   ② $90$   ③ $120$   ④ $150$   ⑤ $180$

## 풀이
정수부분이 세 자리인 양의 실수 $x$ 에 대하여 $\log _{10} x$ 의 정수부분은 $2$ 이므로
$\log _{10} x = 2 + \alpha$ (단, $0 < \alpha < 1$)라 하면
$\log _{10} \frac{1}{x} = -\log _{10} x = -2 - \alpha = -3 + (1 - \alpha)$ 이다.
이때, 주어진 조건으로부터 $\log _{10} x$ 의 소수부분이 $\log _{10} \frac{1}{x}$ 의 소수부분의 $2$ 배이므로
$\alpha = 2(1 - \alpha)$
$3\alpha = 2$
$\therefore \alpha = \frac{2}{3}$
따라서, $\log _{10} x = 2 + \frac{2}{3} = \frac{8}{3}$ 이므로
$($준식$)$ $= (1 + 2 + \cdots + 9) \log _{10} x = 45 \log _{10} x = 45 \times \frac{8}{3} = 120$ 이다.

정답: ③
