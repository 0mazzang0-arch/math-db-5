## 문제
그림과 같이 두 점 $F(k, 0)$, $F'(-k, 0)$을 초점으로 하는 쌍곡선 $\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$과 점 $F$를 초점으로 하는 포물선 $y^2 = 56(x+c)$가 있다. 쌍곡선 위의 임의의 점 $P$에 대하여 $|\overline{PF} - \overline{PF'}| = 10$이 성립하고, 포물선의 꼭짓점 $A$에 대하여 $\overline{AF'} : \overline{FF'} = 1:6$이 성립한다. 이때, $\frac{c^2}{a^2 - b^2}$의 값은? (단, $0 < k < c$이다.)

> [!example] 📸 (그림/그래프)

## 보기
① $\frac{53}{14}$   ② $\frac{55}{14}$   ③ $\frac{30}{7}$   ④ $\frac{32}{7}$   ⑤ $\frac{34}{7}$

## 풀이
쌍곡선 위의 임의의 점 $P$에 대하여 $|\overline{PF} - \overline{PF'}| = 10$이므로 주축의 길이 $2a = 10$이다.
따라서 $a = 5$이다.

포물선 $y^2 = 56(x+c)$의 방정식은 $y^2 = 4 \times 14(x+c)$로 나타낼 수 있으므로, 꼭짓점 $A(-c, 0)$에서 초점 $F(k, 0)$까지의 거리 $\overline{AF}$는 $14$이다.
즉, $k - (-c) = k + c = 14$ $\cdots$ ㉠

쌍곡선의 초점이 $F(k, 0)$, $F'(-k, 0)$이고 $0 < k < c$이므로, 포물선의 꼭짓점 $A$, 초점 $F, F'$의 위치 관계에 의해
$\overline{AF'} = |-k - (-c)| = c - k$
$\overline{FF'} = |k - (-k)| = 2k$
이때 $\overline{AF'} : \overline{FF'} = 1:6$이므로
$(c - k) : 2k = 1 : 6$
$6(c - k) = 2k$
$6c - 6k = 2k$
$8k = 6c$, 즉 $k = \frac{3}{4}c$ $\cdots$ ㉡

㉡을 ㉠에 대입하면
$\frac{3}{4}c + c = 14$
$\frac{7}{4}c = 14$
$c = 8$
이를 ㉡에 대입하면 $k = 6$이다.

쌍곡선에서 초점의 좌표가 $k$이므로 $k^2 = a^2 + b^2$이 성립한다.
$6^2 = 5^2 + b^2$
$36 = 25 + b^2$
$b^2 = 11$

따라서 구하고자 하는 값은
$\frac{c^2}{a^2 - b^2} = \frac{8^2}{5^2 - 11} = \frac{64}{25 - 11} = \frac{64}{14} = \frac{32}{7}$
