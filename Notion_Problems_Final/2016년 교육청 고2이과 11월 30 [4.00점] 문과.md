## 문제

최고차항의 계수가 $1$인 사차함수 $f(x)$와 이차함수 $g(x) = 2x^2 - x - 4$가 다음 조건을 만족시킨다.

(가) 곡선 $y=f(x) - g(x)$는 $x$ 좌표가 $2$인 점에서 $x$ 축에 접한다.
(나) 함수 $y= |f(x) - g(x)|$는 실수 전체의 집합에서 미분가능하다.

$f'(0) = 2$일 때, $f(1)$의 최댓값은 $\alpha$이다. $40\alpha$의 값을 구하시오.

## 풀이

조건 (가)에서 $y=f(x) - g(x)$는 $x$ 좌표가 $2$인 점에서 $x$ 축에 접하므로
$f(2) - g(2) = 0$, $f'(2) - g'(2) = 0$
따라서 $f(x) - g(x)$는 $(x-2)^2$을 인수로 갖는다. $\cdots \cdots (✽)$

$f(x) - g(x) = (x-2)^2(x^2 + ax + b)$라 하자.
조건 (나)에서 $y = |f(x) - g(x)|$는 실수 전체의 집합에서 미분가능하므로 사차함수 $y = f(x) - g(x)$의 그래프가 $x$ 축을 뚫고 지나가는 실근을 가지면 안 된다. 즉, $f(x) - g(x) \geq 0$이어야 한다.

$(x-2)^2 \geq 0$이므로 모든 실수 $x$에 대하여
$x^2 + ax + b \geq 0$
판별식 $D = a^2 - 4b \leq 0 \cdots \cdots ㉠$

$f(x) = (x-2)^2(x^2 + ax + b) + g(x)$에서
$f'(x) = 2(x-2)(x^2 + ax + b) + (x-2)^2(2x + a) + g'(x)$
$g'(x) = 4x - 1$이므로
$f'(0) = 2(-2)(b) + (-2)^2(a) + g'(0)$
$f'(0) = -4b + 4a - 1 = 2$
$b = a - \frac{3}{4} \cdots \cdots ㉡$

$㉠$, $㉡$에서
$a^2 - 4(a - \frac{3}{4}) \leq 0$
$a^2 - 4a + 3 \leq 0$
$(a-1)(a-3) \leq 0$
$\therefore 1 \leq a \leq 3$

$f(1) = (1-2)^2(1 + a + b) + g(1)$
$g(1) = 2(1)^2 - 1 - 4 = -3$이므로
$f(1) = 1 + a + b - 3 = a + b - 2$
$㉡$을 대입하면
$f(1) = a + (a - \frac{3}{4}) - 2 = 2a - \frac{11}{4}$

$f(1)$은 $a = 3$일 때 최댓값 $\alpha = 2(3) - \frac{11}{4} = \frac{13}{4}$을 갖는다.
따라서 $40\alpha = 40 \times \frac{13}{4} = 130$

**[참고]**
$(✽)$에서 $f(2) - g(2) = 0$이므로 $f(x) - g(x)$는 $(x-2)$를 인수로 갖는다.
$f(x) - g(x) = (x-2)Q(x)$라 하면
$f'(x) - g'(x) = Q(x) + (x-2)Q'(x)$
$f'(2) - g'(2) = Q(2) = 0$
$\therefore Q(x)$는 $(x-2)$를 인수로 갖는다.
따라서 $f(x) - g(x)$는 $(x-2)^2$을 인수로 갖는다.

정답: $130$
