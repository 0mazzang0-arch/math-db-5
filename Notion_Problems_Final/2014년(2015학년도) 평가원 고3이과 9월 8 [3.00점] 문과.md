## 문제
$0 \le x \le \pi$ 일 때, 삼각방정식 $\sin x = \sin 2x$ 의 모든 해의 합은?

## 보기
① $\pi$   ② $\frac{7}{6}\pi$   ③ $\frac{5}{4}\pi$   ④ $\frac{4}{3}\pi$   ⑤ $\frac{3}{2}\pi$

## 풀이
$\sin x = 2 \sin x \cos x$
$\sin x (2 \cos x - 1) = 0$
$\sin x = 0$ 또는 $\cos x = \frac{1}{2}$
$x = 0, \pi, \frac{\pi}{3}$ ($\because 0 \le x \le \pi$)
$\because$ 모든 해의 합은 $\frac{4}{3}\pi$

정답: ④
