## 문제
두 이차정사각행렬 $A$, $B$가
$$ AB - A = 2E, \quad BA^{2} - A^{2} + B = -E $$
를 만족시킬 때, <보기>에서 옳은 것만을 있는 대로 고른 것은? (단, $E$는 단위행렬이다.)

$$
\begin{array}{|l|}
\hline
\text{ㄱ. } AB = BA \\
\text{ㄴ. } 2A + B = -E \\
\text{ㄷ. } A + E \text{의 역행렬이 존재한다.} \\
\hline
\end{array}
$$

## 보기
① ㄱ   ② ㄴ   ③ ㄱ, ㄴ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
[정답] ⑤

ㄱ. $AB - A = 2E$에서 $\frac{1}{2} A(B - E) = E$이다.
역행렬의 정의에 의해 $\frac{1}{2} A(B - E) = E = (B - E) \left( \frac{1}{2} A \right)$ 이므로
$\frac{1}{2} (AB - A) = \frac{1}{2} (BA - A)$
$AB - A = BA - A$
$AB = BA$ (참)

ㄴ. $BA^{2} - A^{2} + B = -E$에서
$(BA - A)A + B = -E$
ㄱ에 의해 $AB = BA$이므로
$(AB - A)A + B = -E$
이때 $AB - A = 2E$를 대입하면
$(2E)A + B = -E$
$2A + B = -E$ (참)

ㄷ. ㄴ에 의하여 $B = -2A - E$이다.
이를 $AB - A = 2E$에 대입하면
$A(-2A - E) - A = 2E$
$-2A^{2} - A - A = 2E$
$-2A^{2} - 2A = 2E$
양변을 $-2$로 나누면
$A^{2} + A = -E$
$-A(A + E) = E$
따라서 $(A + E)$의 역행렬은 $-A$로 존재한다. (참)

따라서 옳은 것은 ㄱ, ㄴ, ㄷ이다.
