## 문제
집합 $X = \{ 1 , 2 , 3 , 4 \}$ 에 대하여 함수 $f : X \to X$ 가 그림과 같다.

> [!example] 📸 (그림/그래프)

함수 $g : X \to X$ 의 역함수가 존재하고, $g ( 2 ) = 3$ , $g^{-1} ( 1 ) = 3$ , $( g \circ f ) ( 2 ) = 2$ 일 때, $g^{-1} ( 4 ) + ( f \circ g ) ( 2 )$ 의 값을 구하시오.

## 풀이
함수 $g$ 의 역함수가 존재하므로 함수 $g$ 는 일대일대응이다.
$g ( 2 ) = 3$
$g^{-1} ( 1 ) = 3$ 이므로 $g ( 3 ) = 1$
$( g \circ f ) ( 2 ) = g ( f ( 2 ) ) = 2$ , $f ( 2 ) = 1$ 이므로 $g ( 1 ) = 2$
$\therefore g ( 4 ) = 4$ , $g^{-1} ( 4 ) = 4$
따라서 $g^{-1} ( 4 ) + ( f \circ g ) ( 2 ) = 4 + f ( g ( 2 ) ) = 4 + f ( 3 ) = 4 + 3 = 7$

**정답: $7$**
