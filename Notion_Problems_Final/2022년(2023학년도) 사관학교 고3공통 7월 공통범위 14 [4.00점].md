## 문제
최고차항의 계수가 $1$인 이차함수 $f(x)$에 대하여 함수 $g(x)$를
$$g(x) = \begin{cases} f(x) & (x < 1) \\ 2f(1) - f(x) & (x \geq 1) \end{cases}$$
이라 하자. 함수 $g(x)$에 대하여 <보기>에서 옳은 것만을 있는 대로 고른 것은?

## 보기
ㄱ. 함수 $g(x)$는 실수 전체의 집합에서 연속이다.
ㄴ. $\lim_{h \to 0+} \frac{g(-1+h) + g(-1-h) - 6}{h} = a$ ($a$는 상수)이고 $g(1) = 1$이면 $g(a) = 1$이다.
ㄷ. $\lim_{h \to 0+} \frac{g(b+h) + g(b-h) - 6}{h} = 4$ ($b$는 상수)이면 $g(4) = 1$이다.

① ㄱ   ② ㄱ, ㄴ   ③ ㄱ, ㄷ   ④ ㄴ, ㄷ   ⑤ ㄱ, ㄴ, ㄷ

## 풀이
ㄱ. $\lim_{x \to 1-} g(x) = \lim_{x \to 1-} f(x) = f(1)$,
$\lim_{x \to 1+} g(x) = \lim_{x \to 1+} \{2f(1) - f(x)\} = f(1)$,
$g(1) = 2f(1) - f(1) = f(1)$
$g(x)$는 $x = 1$에서 연속이다.
그러므로 $g(x)$는 실수 전체의 집합에서 연속이다. (참)

ㄴ. $\lim_{h \to 0+} \frac{g(-1+h) + g(-1-h) - 6}{h} = a$에서 극한값이 존재하고 (분모) $\to 0$이므로 (분자) $\to 0$이어야 한다.
$\lim_{h \to 0+} \{g(-1+h) + g(-1-h) - 6\} = 0$에서 $g(-1) = 3$
$\therefore f(-1) = 3$
$\lim_{h \to 0+} \frac{g(-1+h) + g(-1-h) - 6}{h}$
$= \lim_{h \to 0+} \frac{\{g(-1+h) - g(-1)\} - \{g(-1-h) - g(-1)\}}{h}$
$= g'(-1) - g'(-1) = 0$
$\therefore a = 0$
$g(1) = 1$에서 $f(1) = 1$
$f(x) = x^2 + px + q$라 하면
$f(-1) = 3$에서 $1 - p + q = 3$, $p - q = -2 \cdots ㉠$
$f(1) = 1$에서 $1 + p + q = 1$, $p + q = 0 \cdots ㉡$
$p = -1, q = 1$
$f(x) = x^2 - x + 1$이므로
$g(a) = g(0) = f(0) = 1$ (참)

ㄷ. $b \neq 1$이면 $\lim_{h \to 0+} \frac{g(b+h) + g(b-h) - 6}{h} = 0$이므로 $b = 1$이다.
$\lim_{h \to 0+} \frac{g(1+h) + g(1-h) - 6}{h} = 4$에서 극한값이 존재하고 (분모) $\to 0$이므로 (분자) $\to 0$이어야 한다.
$\lim_{h \to 0+} \{g(1+h) + g(1-h) - 6\} = 0$에서 $g(1) = 3$
$\therefore f(1) = 3$
$\lim_{h \to 0+} \frac{g(1+h) + g(1-h) - 6}{h}$
$= \lim_{h \to 0+} \frac{2f(1) - f(1+h) + f(1-h) - 6}{h}$
$= \lim_{h \to 0+} \frac{f(1-h) - f(1+h)}{h}$
$= -2f'(1)$
$-2f'(1) = 4$에서 $f'(1) = -2$
$f(x) = x^2 + rx + s$라 하면 $f'(x) = 2x + r$
$f(1) = 3$에서 $1 + r + s = 3$, $r + s = 2 \cdots ㉢$
$f'(1) = -2$에서 $2 + r = -2$, $r = -4$
$r = -4$를 $㉢$에 대입하면 $s = 6$
$f(x) = x^2 - 4x + 6$이므로
$g(4) = 2f(1) - f(4) = 6 - (16 - 16 + 6) = 0$ (거짓)

따라서 옳은 것은 ㄱ, ㄴ이다.
