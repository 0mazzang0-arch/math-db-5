## 문제
정수 부분이 세 자리인 두 실수 $x$, $y$가 다음 두 조건을 만족한다.

$$ \begin{array}{l} \text{(가) } \log x + \log y \text{는 정수이다.} \\ \text{(나) } \log x - \log y = 0.4 \end{array} $$

$x$의 최고 자리의 숫자를 $a$, $y$의 최고 자리의 숫자를 $b$라 할 때, $a + b$의 값을 구하시오. (단, $\log 2 = 0.3010$, $\log 3 = 0.4771$)

## 보기

## 풀이
$\log x = 2 + \alpha$, $\log y = 2 + \beta$ ($0 \leq \alpha < 1$, $0 \leq \beta < 1$)라 하자.
(가)에서 $\log x + \log y = 4 + (\alpha + \beta)$의 값이 정수이므로 $\alpha + \beta = 0$ 또는 $\alpha + \beta = 1$이다.
$\therefore \alpha = \beta = 0$ 또는 $\beta = 1 - \alpha \cdots$ ①
(나)에서 $\log x - \log y = \alpha - \beta = 0.4 \cdots$ ②
①, ②에 의해 $\alpha = 0.7$, $\beta = 0.3$이다.

$\log x = 2.7$에서 $\log 500 < \log x < \log 600$이므로 $x$의 최고 자리의 숫자는 $5$이다.
$\therefore a = 5$

$\log y = 2.3$에서 $\log 100 < \log y < \log 200$이므로 $y$의 최고 자리의 숫자는 $1$이다.
$\therefore b = 1$

$\therefore a + b = 6$

정답: $6$
