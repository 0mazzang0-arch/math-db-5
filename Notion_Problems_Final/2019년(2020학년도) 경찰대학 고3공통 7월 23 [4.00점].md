## 문제
자연수 $n$ 에 대하여 $| n - \sqrt{m - \frac{1}{2}} | < 1$ 을 만족하는 자연수 $m$ 의 개수를 $a_{n}$ 이라 하자. $\frac{1}{100} \sum_{n=1}^{100} a_{n}$ 의 값을 구하시오.

## 풀이
$| n - \sqrt{m - \frac{1}{2}} | < 1$ 에서
$n - 1 < \sqrt{m - \frac{1}{2}} < n + 1$
$(n - 1)^2 < m - \frac{1}{2} < (n + 1)^2$
$(n - 1)^2 + \frac{1}{2} < m < (n + 1)^2 + \frac{1}{2}$

이때 $m$ 은 자연수이므로
$a_{n} = (n + 1)^2 - (n - 1)^2 = 4n$

따라서
$\frac{1}{100} \sum_{n=1}^{100} a_{n} = \frac{1}{100} \sum_{n=1}^{100} 4n$
$= \frac{1}{50} \times 100 \times 101 = 202$

정답: $202$
