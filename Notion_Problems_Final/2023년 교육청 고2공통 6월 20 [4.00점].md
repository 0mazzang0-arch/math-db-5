## 문제
$1$이 아닌 두 자연수 $a$, $b$가 다음 조건을 만족시킨다.

(가) $a < b < a^2$
(나) $\log_{a} b$는 유리수이다.

$\log a < \frac{3}{2}$일 때, $a+b$의 최댓값은?

## 보기
① $250$   ② $270$   ③ $290$   ④ $310$   ⑤ $330$

## 풀이
조건 (나)에서 $\log_{a} b = \frac{n}{m}$ ($m$과 $n$은 서로소인 자연수)라 하면 $b = a^{\frac{n}{m}}$이다.

$\log a < \frac{3}{2}$에서 $a < 10^{\frac{3}{2}}$이고, 조건 (가)에서 $1 < a < a^{\frac{n}{m}} < a^2 < 1000$이다.

(ⅰ) $m=1$일 때, $1 < a < a^n < a^2 < 1000$을 만족시키는 자연수 $n$은 존재하지 않는다.
(ⅱ) $m=2$일 때, $1 < a < a^{\frac{n}{2}} < a^2 < 1000$을 만족시키는 자연수 $n$은 $3$이고, $a$, $b$는 자연수이므로 $1 < a < a^{\frac{3}{2}} < a^2 < 1000$을 만족시키는 순서쌍 $(a, b)$는 $(4, 8)$, $(9, 27)$, $(16, 64)$, $(25, 125)$이다.
(ⅲ) $m=3$일 때, $1 < a < a^{\frac{n}{3}} < a^2 < 1000$을 만족시키는 자연수 $n$은 $4$, $5$이다.
① $m=3$, $n=4$일 때, $a$, $b$는 자연수이므로 $1 < a < a^{\frac{4}{3}} < a^2 < 1000$을 만족시키는 $(a, b)$는 $(8, 16)$, $(27, 81)$이다.
② $m=3$, $n=5$일 때, $a$, $b$는 자연수이므로 $1 < a < a^{\frac{5}{3}} < a^2 < 1000$을 만족시키는 $(a, b)$는 $(8, 32)$, $(27, 243)$이다.
(ⅳ) $m=4$일 때, $1 < a < a^{\frac{n}{4}} < a^2 < 1000$을 만족시키는 자연수 $n$은 $5$, $7$이다.
① $m=4$, $n=5$일 때, $a$, $b$는 자연수이므로 $1 < a < a^{\frac{5}{4}} < a^2 < 1000$을 만족시키는 $(a, b)$는 $(16, 32)$이다.
② $m=4$, $n=7$일 때, $a$, $b$는 자연수이므로 $1 < a < a^{\frac{7}{4}} < a^2 < 1000$을 만족시키는 $(a, b)$는 $(16, 128)$이다.
(ⅴ) $m \ge 5$일 때, $1 < a < a^{\frac{n}{m}} < a^2 < 1000$에서 $a^{\frac{n}{m}}$이 자연수이므로 $a \ge 2^5$이고 $a^2 \ge 2^{10} = 1024$이다.
그러므로 $1 < a < a^{\frac{n}{m}} < a^2 < 1000$을 만족시키는 자연수 $a$는 존재하지 않는다.

따라서 (ⅰ) $\sim$ (ⅴ)에서 $a+b$의 최댓값은 $27 + 243 = 270$이다.

[다른 풀이]
$\log a < \frac{3}{2}$에서 $a^2 < 10^3$, $1 < a < b < a^2 < 10^3$
$\log_{a} b = \frac{n}{m}$ ($m, n$은 서로소인 자연수), $a^n = b^m$에서 $a = c^m$, $b = c^n$ ($c$는 $2$ 이상의 자연수)라 하자.
$a^2 < 1000$이므로 $1 < c^m < c^n < c^{2m} < 1000$, $m = 2, 3, 4$이다.
(ⅰ) $m=2$일 때, $n=3$이고 $c = 2, 3, 4, 5$이므로 $a+b = c^m + c^n \le 5^2 + 5^3 = 150$
(ⅱ) $m=3$일 때, $n = 4, 5$이고 $c = 2, 3$이므로 $a+b = c^m + c^n \le 3^3 + 3^5 = 270$
(ⅲ) $m=4$일 때, $n = 5, 7$이고 $c = 2$이므로 $a+b = c^m + c^n \le 2^4 + 2^7 = 144$
(ⅳ) $m \ge 5$일 때, $c^{2m} < 1000$인 자연수 $c$가 존재하지 않으므로 (가)와 (나)를 만족시키는 자연수 $a$와 $b$는 존재하지 않는다.

따라서 (ⅰ) $\sim$ (ⅳ)에서 $a+b$의 최댓값은 $270$이다.
