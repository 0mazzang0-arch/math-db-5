## 문제
삼각형 $ABC$ 에서 $\angle A = 105^{\circ}$, $\angle B = 30^{\circ}$ 이고 $\overline{AB} = 12$ 일 때, $\overline{AC}^{2}$ 의 값은?



> [!example] 📸 (그림/그래프)

## 보기
① $72$   ② $76$   ③ $80$   ④ $84$   ⑤ $88$

## 풀이
$\angle C = 180^{\circ} - (105^{\circ} + 30^{\circ}) = 45^{\circ}$ 이므로 사인법칙에 의하여 
$$\frac{\overline{AC}}{\sin 30^{\circ}} = \frac{12}{\sin 45^{\circ}}$$
따라서 $\overline{AC} = 12 \times \frac{\sin 30^{\circ}}{\sin 45^{\circ}} = 12 \times \frac{1/2}{\sqrt{2}/2} = \frac{12}{\sqrt{2}} = 6\sqrt{2}$ 이다.
그러므로 $\overline{AC}^{2} = (6\sqrt{2})^{2} = 72$
정답 ①

## 문제
음이 아닌 정수 전체의 집합 $U$ 의 두 부분집합 $A$, $B$ 는 다음과 같다.
$A = \{ x \mid x \text{ 는 } 3^{n} \text{ 을 } 10 \text{ 으로 나눈 나머지, } n \text{ 은 자연수} \}$
$B = \{ x \mid x \text{ 는 } 9^{n} \text{ 을 } 10 \text{ 으로 나눈 나머지, } n \text{ 은 자연수} \}$
다음 물음에 답하시오.

## 보기
① $Value1$   ② $Value2$   ③ $Value3$   ④ $Value4$   ⑤ $Value5$

## 풀이
$3^{1}=3$, $3^{2}=9$, $3^{3}=27$, $3^{4}=81$, $3^{5}=243, \dots$ 이므로 $10$ 으로 나눈 나머지의 집합 $A = \{ 1, 3, 7, 9 \}$ 이다.
$9^{1}=9$, $9^{2}=81$, $9^{3}=729, \dots$ 이므로 $10$ 으로 나눈 나머지의 집합 $B = \{ 1, 9 \}$ 이다.
