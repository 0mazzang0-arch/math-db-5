## 문제
집합 $X = \{ 1, 2, 3, 4, 5, 6 \}$에 대하여 함수 $f : X \to X$ 중에서 다음 조건을 만족시키는 함수 $f$의 개수를 구하시오.

(가) $f(3) \times f(6)$은 $3$의 배수이다.
(나) 집합 $X$의 임의의 두 원소 $x_{1}$, $x_{2}$에 대하여 $x_{1} < x_{2}$이면 $f(x_{1}) \leq f(x_{2})$이다.

## 보기

## 풀이
(ⅰ) $f(3)$이 $3$의 배수인 경우
① $f(3) = 3$인 경우
$f(1)$, $f(2)$를 선택하는 경우의 수는 ${}_{3}\mathrm{H}_{2} = {}_{4}\mathrm{C}_{2} = 6$
$f(4)$, $f(5)$, $f(6)$을 선택하는 경우의 수는 ${}_{4}\mathrm{H}_{3} = {}_{6}\mathrm{C}_{3} = 20$
$6 \times 20 = 120$

② $f(3) = 6$인 경우
$f(1)$, $f(2)$를 선택하는 경우의 수는 ${}_{6}\mathrm{H}_{2} = {}_{7}\mathrm{C}_{2} = 21$
$f(4)$, $f(5)$, $f(6)$을 선택하는 경우의 수는 ${}_{1}\mathrm{H}_{3} = {}_{3}\mathrm{C}_{3} = 1$
$21 \times 1 = 21$

①, ②에 의하여 $f(3)$이 $3$의 배수인 경우의 수는 $120 + 21 = 141$

(ⅱ) $f(6)$이 $3$의 배수인 경우
① $f(6) = 3$인 경우
$f(1)$, $f(2)$, $f(3)$, $f(4)$, $f(5)$를 선택하는 경우의 수는 ${}_{3}\mathrm{H}_{5} = {}_{7}\mathrm{C}_{5} = 21$

② $f(6) = 6$인 경우
$f(1)$, $f(2)$, $f(3)$, $f(4)$, $f(5)$를 선택하는 경우의 수는 ${}_{6}\mathrm{H}_{5} = {}_{10}\mathrm{C}_{5} = 252$

①, ②에 의하여 $f(6)$이 $3$의 배수인 경우의 수는 $21 + 252 = 273$

(ⅲ) $f(3)$, $f(6)$이 모두 $3$의 배수인 경우
① $f(3) = f(6) = 3$인 경우
$f(1)$, $f(2)$를 선택하는 경우의 수는 ${}_{3}\mathrm{H}_{2} = {}_{4}\mathrm{C}_{2} = 6$
$f(4)$, $f(5)$를 선택하는 경우의 수는 ${}_{1}\mathrm{H}_{2} = {}_{2}\mathrm{C}_{2} = 1$
$6 \times 1 = 6$

② $f(3) = 3$, $f(6) = 6$인 경우
$f(1)$, $f(2)$를 선택하는 경우의 수는 ${}_{3}\mathrm{H}_{2} = {}_{4}\mathrm{C}_{2} = 6$
$f(4)$, $f(5)$를 선택하는 경우의 수는 ${}_{4}\mathrm{H}_{2} = {}_{5}\mathrm{C}_{2} = 10$
$6 \times 10 = 60$

③ $f(3) = f(6) = 6$인 경우
$f(1)$, $f(2)$를 선택하는 경우의 수는 ${}_{6}\mathrm{H}_{2} = {}_{7}\mathrm{C}_{2} = 21$
$f(4)$, $f(5)$를 선택하는 경우의 수는 ${}_{1}\mathrm{H}_{2} = {}_{2}\mathrm{C}_{2} = 1$
$21 \times 1 = 21$

①, ②, ③에 의하여 $f(3)$, $f(6)$이 모두 $3$의 배수인 경우의 수는 $6 + 60 + 21 = 87$

따라서 (ⅰ), (ⅱ), (ⅲ)에 의하여 구하는 경우의 수는 $141 + 273 - 87 = 327$

정답: $327$
