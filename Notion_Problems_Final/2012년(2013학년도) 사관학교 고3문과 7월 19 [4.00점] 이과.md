## 문제
구간 $[0, 1]$에서 정의된 연속확률변수 $X$의 확률밀도함수가 일차함수 $f(x) = ax + b$ ($0 \leq x \leq 1$)일 때, <보기>에서 옳은 것만을 있는 대로 고른 것은?

> **<보기>**
> ㄱ. $\frac{a}{2} + b = 1$
> ㄴ. $E(X) = m$일 때, $P(0 \leq X \leq m) = P(m \leq X \leq 1)$이다.
> ㄷ. $E(X)$의 최댓값과 최솟값의 합은 $1$이다.

## 보기
① $ㄱ$   ② $ㄴ$   ③ $ㄱ, ㄷ$   ④ $ㄴ, ㄷ$   ⑤ $ㄱ, ㄴ, ㄷ$

## 풀이
[정답] ③ 

ㄱ. $\int_{0}^{1} f(x) dx = 1$이므로 
$$ \int_{0}^{1} (ax + b) dx = \left[ \frac{a}{2}x^2 + bx \right]_{0}^{1} = \frac{a}{2} + b = 1 $$
따라서 (참)이다.

ㄴ. 확률밀도함수의 그래프가 $x = m$에 대하여 대칭일 때와 같은 특별한 경우일 때만 성립한다. 
$P(0 \leq x \leq 1) = 1$이므로 
$$ P(0 \leq x \leq m) = P(m \leq x \leq 1) \iff P(0 \leq x \leq m) = \frac{1}{2} $$
$f(x) = 2x$ ($0 \leq x \leq 1$)인 경우 
$$ E(X) = \int_{0}^{1} x f(x) dx = \int_{0}^{1} 2x^2 dx = \left[ \frac{2}{3}x^3 \right]_{0}^{1} = \frac{2}{3} $$
$P(0 \leq x \leq \frac{2}{3}) = \int_{0}^{\frac{2}{3}} 2x dx = \frac{4}{9}$이므로 
$P(0 \leq x \leq m) \neq \frac{1}{2}$이다. 
$\therefore P(0 \leq x \leq m) \neq P(m \leq x \leq 1)$ (거짓) 

ㄷ. $E(X) = \int_{0}^{1} xf(x) dx = \int_{0}^{1} x(ax + b) dx$ 
$$ = \left[ \frac{a}{3}x^3 + \frac{b}{2}x^2 \right]_{0}^{1} = \frac{a}{3} + \frac{b}{2} $$ 
$0 \leq x \leq 1$인 $x$에 대하여 $f(x) \geq 0$이어야 하므로 
$f(0) = b \geq 0$이고 $f(1) = a + b \geq 0$이다. 
ㄱ에서 $\frac{a}{2} + b = 1$이므로 $b = 1 - \frac{a}{2} \geq 0$에서 $a \leq 2$ 
$a + b = \frac{a}{2} + 1 \geq 0$에서 $a \geq -2$ 
$\therefore -2 \leq a \leq 2$ 
$$ \frac{a}{3} + \frac{b}{2} = \frac{a}{3} + \frac{1 - \frac{a}{2}}{2} = \frac{a}{12} + \frac{1}{2} $$ 
$\therefore -\frac{1}{6} + \frac{1}{2} \leq E(X) \leq \frac{1}{6} + \frac{1}{2}$ 
따라서 $E(X)$의 최댓값과 최솟값의 합은 $1$이다. (참) 

[다른 풀이] 
$E(X)$의 값이 작으려면 $0$ 근처 구간에서의 확률이 커야 한다. 
따라서 $E(X)$가 최소가 되는 경우는 [그림 1]과 같다. 
비슷한 방법으로 $E(X)$가 최대가 되는 경우는 [그림 2]와 같다.
