## 문제
$\int_{-1}^{1} ( 4x^{3} + x^{2} - \frac{1}{2}x + a ) dx = 2$ 일 때, 상수 $a$의 값은?

## 보기
① $\frac{1}{3}$   ② $\frac{2}{3}$   ③ $1$   ④ $\frac{4}{3}$   ⑤ $\frac{5}{3}$

## 풀이
$\int_{-1}^{1} ( 4x^{3} + x^{2} - \frac{1}{2}x + a ) dx$
$= 2 \int_{0}^{1} ( x^{2} + a ) dx$
$= 2 [ \frac{1}{3}x^{3} + ax ]_{0}^{1}$
$= \frac{2}{3} + 2a$

$\frac{2}{3} + 2a = 2$ 이므로
$2a = \frac{4}{3}$
$a = \frac{2}{3}$

정답: ②
