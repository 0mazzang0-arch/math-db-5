## 문제
$1, 2, 3, \dots, 3n$ ($n$은 자연수)의 숫자가 하나씩 적혀 있는 $3n$장의 카드 중 임의로 꺼낸 $2$장의 카드에 적혀 있는 두 수를 각각 $a, b$ ($a < b$)라 하자. $3a < b$일 확률을 $P_n$이라 할 때, 다음은 $\lim_{n \to \infty} P_n$의 값을 구하는 과정이다.

$$ \begin{array}{|l|} \hline 3n \text{장의 카드 중 } 2 \text{장의 카드를 꺼내는 경우의 수는 } {}_{3n}C_2 \text{이다.} \\ 3a < b \text{인 경우에는 } b \le 3n \text{이므로 } 1 \le a < n \text{이다.} \\ \text{따라서 } a=k \text{라 하면 } 3a < b \text{를 만족시키는 } b \text{의 경우의 수는} \\ (가) \text{이므로 } P_n = \frac{(나)}{{}_{3n}C_2} \text{이다.} \\ \text{그러므로 } \lim_{n \to \infty} P_n = (다) \text{이다.} \\ \hline \end{array} $$

위의 과정에서 $(가), (나), (다)$에 알맞은 것은?

## 보기
① $(가): 3(n-k), (나): \frac{3}{2}n(n-1), (다): \frac{1}{3}$   ② $(가): 3(n-k), (나): \frac{3}{2}n(n-1), (다): \frac{2}{3}$   ③ $(가): 3(n-k), (나): 3n(n-1), (다): \frac{2}{3}$   ④ $(가): 3(n-k+1), (나): 3n(n-1), (다): \frac{1}{3}$   ⑤ $(가): 3(n-k+1), (나): 3n(n-1), (다): \frac{2}{3}$

## 풀이
$3n$장의 카드 중 $2$장의 카드를 꺼내는 경우의 수는 ${}_{3n}C_2$이다.

$a=k$ ($k=1, 2, 3, \dots, n-1$)이면 $3a < b$를 만족시키는 $b$는 $3k+1, 3k+2, \dots, 3n$ 중의 하나이어야 하므로 $b$의 경우의 수는 $3n - 3k = 3(n-k)$이다.
따라서 $(가) = 3(n-k)$이다.

$$ P_n = \frac{\sum_{k=1}^{n-1} 3(n-k)}{{}_{3n}C_2} $$
이때,
$$ \sum_{k=1}^{n-1} 3(n-k) = 3n(n-1) - \frac{3}{2}n(n-1) = \frac{3}{2}n(n-1) $$
이므로 $(나) = \frac{3}{2}n(n-1)$이다.

$$ P_n = \frac{\frac{3}{2}n(n-1)}{{}_{3n}C_2} $$
이때, ${}_{3n}C_2 = \frac{3n(3n-1)}{2}$이므로
$$ \lim_{n \to \infty} P_n = \lim_{n \to \infty} \frac{\frac{3n^2-3n}{2}}{\frac{9n^2-3n}{2}} = \frac{3}{9} = \frac{1}{3} $$
따라서 $(다) = \frac{1}{3}$이다.

이상에서 $(가), (나), (다)$에 알맞은 것은 $3(n-k), \frac{3}{2}n(n-1), \frac{1}{3}$이다.

정답: ①
