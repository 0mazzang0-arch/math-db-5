## 문제
곡선 $f(x) = -x^{3} - x^{2} + x + 1$과 $x$축으로 둘러싸인 영역 $A = \{ (x, y) \mid -1 \le x \le 1, 0 \le y \le f(x) \}$에서 $3x + 4y$의 최댓값은?

## 보기
① $3$   ② $4$   ③ $5$   ④ $6$   ⑤ $8$

## 풀이
$f(x) = -x^{3} - x^{2} + x + 1 = -(x - 1)(x + 1)^{2}$이고 영역 $A$는 다음 그림의 어두운 부분이다.

> [!example] 📸 (그림/그래프)

$3x + 4y = k$라 할 때, $k$는 위 그림과 같이 $-1 \le x \le 1$인 범위에서 직선 $3x + 4y = k$가 함수 $y = f(x)$에 접할 때 최대가 된다.
접점의 좌표를 $(a, f(a))$라 할 때,

$f'(a) = -3a^{2} - 2a + 1 = -\frac{3}{4}$에서

$12a^{2} + 8a - 7 = (2a - 1)(6a + 7) = 0$

$\therefore a = \frac{1}{2}$

$f(\frac{1}{2}) = -\frac{1}{8} - \frac{1}{4} + \frac{1}{2} + 1 = \frac{9}{8}$이므로
접점의 좌표는 $(\frac{1}{2}, \frac{9}{8})$이다.

$\therefore 3x + 4y \le 3a + 4f(a) = 3 \times \frac{1}{2} + 4 \times \frac{9}{8} = 6$
