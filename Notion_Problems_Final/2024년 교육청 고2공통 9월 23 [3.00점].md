## 문제
등차수열 $\{ a_{n} \}$ 에 대하여 $a_{3} + a_{5} + a_{7} = 18$ 일 때, $a_{4} + a_{6}$ 의 값을 구하시오.

## 보기


## 풀이
등차수열 $\{ a_{n} \}$ 의 첫째항을 $a$, 공차를 $d$ 라 하면
$a_{3} + a_{5} + a_{7} = (a + 2d) + (a + 4d) + (a + 6d) = 3a + 12d = 18$ 이므로
$a + 4d = 6$
따라서
$a_{4} + a_{6} = (a + 3d) + (a + 5d) = 2a + 8d = 12$

**[다른 풀이]**
세 항 $a_{3}$, $a_{5}$, $a_{7}$ 은 이 순서대로 등차수열을 이루므로
$a_{3} + a_{5} + a_{7} = 3a_{5} = 18$, $a_{5} = 6$
$a_{5}$ 는 두 항 $a_{4}$, $a_{6}$ 의 등차중항이므로
$a_{5} = \frac{a_{4} + a_{6}}{2}$
따라서
$a_{4} + a_{6} = 2a_{5} = 12$

정답: $12$
