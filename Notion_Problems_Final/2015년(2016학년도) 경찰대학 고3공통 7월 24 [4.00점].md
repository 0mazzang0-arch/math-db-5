## 문제
다항함수 $f(x) = x^3(x^3 + 1)(x^3 + 2)(x^3 + 3)$ 에 대하여 $f'(-1) = a$ 이고 $f(x)$ 의 최솟값이 $b$ 일 때, $a^2 + b^2$ 의 값을 구하시오.

## 풀이
$f'(-1) = 3(-1)(1)(2) = a$ 이므로 $a = -6$ 이며,
$t = x^3, \frac{dt}{dx} = 3x^2$ 이므로 ($t = x^3$ 은 일대일대응함수이며 증가함수이다.)
$f'(x) = (4t^3 + 18t^2 + 22t + 6) \cdot 3x^2$ 이며, 이때 $3x^2 \ge 0$ 이므로 $2t^3 + 9t^2 + 11t + 3$ 의 부호를 살펴보자.
$2t^3 + 9t^2 + 11t + 3 = (2t + 3)(t^2 + 3t + 1)$ 이므로 $t^2 + 3t + 1 = 0$ 의 한 근을 $\alpha$ 라 하면, $y = f(t)$ 는 $t = \alpha$ 일 때, 최솟값을 갖는다.
따라서 
$f(\alpha) = \alpha(\alpha + 1)(\alpha + 2)(\alpha + 3) = (\alpha^2 + 3\alpha)(\alpha^2 + 3\alpha + 2) = -1$ ($\because \alpha^2 + 3\alpha + 1 = 0$)
즉 $b = -1$ 이다.
그러므로 $a^2 + b^2 = 37$ 이다.
