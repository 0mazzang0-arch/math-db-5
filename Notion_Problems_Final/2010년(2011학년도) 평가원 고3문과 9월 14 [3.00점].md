## 문제
연속확률변수 $X$ 가 갖는 값의 범위는 $0 \le X \le 2$ 이고, $X$ 의 확률밀도함수의 그래프는 그림과 같다. 확률 $P ( a \le X \le a + \frac{1}{2} )$ 의 값이 최대가 되도록 하는 상수 $a$ 의 값은?

> [!example] 📸 (그림/그래프)

## 보기
① $\frac{3}{8}$   ② $\frac{1}{2}$   ③ $\frac{5}{8}$   ④ $\frac{3}{4}$   ⑤ $\frac{7}{8}$

## 풀이
확률 $P ( a \le X \le a + \frac{1}{2} )$ 의 값이 최대가 되려면 구간 $a \le x \le a + \frac{1}{2}$ 안에 $1$ 이 포함되어 있어야 하므로 $\frac{1}{2} < a < 1$ 인 경우를 생각한다.

$\frac{1}{2} < a < 1$ 일 때 $1 < a + \frac{1}{2} < \frac{3}{2}$ 이므로 

$P ( a \le X \le a + \frac{1}{2} ) = ( \frac{1}{2} - \frac{a^{2}}{2} ) + \frac{1}{2} - ( \frac{3 - 2a}{2} )^{2}$
$= -a^{2} + \frac{3}{2} a - \frac{1}{8}$
$= -( a - \frac{3}{4} )^{2} + \frac{7}{16}$

따라서 $a = \frac{3}{4}$ 일 때 확률 $P ( a \le X \le a + \frac{1}{2} )$ 의 값이 최대이다.
