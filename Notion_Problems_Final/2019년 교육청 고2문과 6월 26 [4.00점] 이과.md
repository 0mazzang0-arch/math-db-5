## 문제
방정식 $\left( \log _{2} \frac{x}{2} \right) \left( \log _{2} 4x \right) = 4$의 서로 다른 두 실근 $\alpha$, $\beta$에 대하여 $64 \alpha \beta$의 값을 구하시오.

## 풀이
$\left( \log _{2} \frac{x}{2} \right) \left( \log _{2} 4x \right) = 4$에서
$\left( \log _{2} x - 1 \right) \left( \log _{2} x + 2 \right) = 4$이다.
$\log _{2} x = X$로 놓으면 
$\left( X - 1 \right) \left( X + 2 \right) = 4$
$X^{2} + X - 6 = 0$
$\left( X - 2 \right) \left( X + 3 \right) = 0$
$\therefore X = 2$ 또는 $X = -3$
이므로 $\alpha \beta = 2^{2} \times 2^{-3} = 2^{-1}$이다. 
따라서 $64 \alpha \beta = 64 \times 2^{-1} = 32$이다.

정답: $32$
