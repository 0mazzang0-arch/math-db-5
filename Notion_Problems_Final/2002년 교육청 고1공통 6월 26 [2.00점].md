## 문제
두 집합 $A = \{ 0, a + 1 \}$, $B = \{ a - 3, a - 1, a^{2} - 11 \}$에 대하여 $A \cap B = \{ -2 \}$일 때, 집합 $B$의 모든 원소의 합을 구하시오.

## 풀이
$A \cap B = \{ -2 \}$이므로,
$a + 1 = -2$ $\therefore$ $a = -3$

$\therefore$ $a - 3 = -6$, $a - 1 = -4$, $a^{2} - 11 = -2$

따라서 집합 $B$의 모든 원소의 합은
$-6 + ( -4 ) + ( -2 ) = -12$
