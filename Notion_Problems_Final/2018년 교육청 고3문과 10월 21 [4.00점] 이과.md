## 문제
함수 $f(x) = \frac{k}{x} + 5$ ($k$는 양의 상수)의 그래프를 $x$축의 방향으로 $m$ ($m > 0$)만큼 평행이동시킨 그래프를 나타내는 함수를 $y = g(x)$라 하자. 두 함수 $f(x)$, $g(x)$가 다음 조건을 만족시킨다.

(가) $g(a) = b$, $g(b) = a$인 서로 다른 두 실수 $a$, $b$가 존재한다.
(나) $0 < x < m$에서 정의된 함수 $\frac{1}{f(x) - g(x)}$의 최댓값은 $\frac{5}{24}$이다.

$g(9)$의 값은?

## 보기
① $\frac{11}{2}$   ② $\frac{13}{2}$   ③ $\frac{15}{2}$   ④ $\frac{17}{2}$   ⑤ $\frac{19}{2}$

## 풀이
함수 $y = g(x)$의 그래프는 함수 $y = \frac{k}{x} + 5$의 그래프를 $x$축의 방향으로 $m$만큼 평행이동시킨 그래프이므로 
$g(x) = \frac{k}{x - m} + 5$ 

조건 (가)에서 
$g(a) = b$에서 $\frac{k}{a - m} + 5 = b$ …… ㉠ 
$g(b) = a$에서 $\frac{k}{b - m} + 5 = a$ …… ㉡ 

㉠, ㉡에서 
$k = (b - 5)(a - m) = (a - 5)(b - m)$ 
$(a - b)(m - 5) = 0$ 

$a \neq b$이므로 $m = 5$ 
따라서 $g(x) = \frac{k}{x - 5} + 5$ 

$f(x) - g(x) = (\frac{k}{x} + 5) - (\frac{k}{x - 5} + 5) = -\frac{5k}{x(x - 5)}$ 

$0 < x < 5$일 때 
$\frac{1}{f(x) - g(x)} = -\frac{1}{5k}x(x - 5) = -\frac{1}{5k} \left\{ (x - \frac{5}{2})^{2} - \frac{25}{4} \right\}$ 

이므로 함수 $\frac{1}{f(x) - g(x)}$은 $x = \frac{5}{2}$에서 최댓값 
$-\frac{1}{5k} \times (-\frac{25}{4}) = \frac{5}{4k}$를 갖는다. 

조건 (나)에서 $\frac{5}{4k} = \frac{5}{24}$이므로 $k = 6$ 

$g(9) = \frac{6}{9 - 5} + 5 = \frac{3}{2} + 5 = \frac{13}{2}$

정답: ②
