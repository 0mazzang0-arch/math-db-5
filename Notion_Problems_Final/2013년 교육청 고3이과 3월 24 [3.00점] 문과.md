## 문제
$n$ 이 자연수일 때, 집합 $A$ 를 
$$A = \left\{ x \mid \frac{x-2n}{(x-n)(x-3n)} \leq 0 \right\}$$
이라 하자. $120 \in A$ 일 때, $n$ 의 최솟값을 구하시오.

## 보기

## 풀이
$$\frac{x-2n}{(x-n)(x-3n)} \leq 0$$
$\Leftrightarrow (x-n)(x-2n)(x-3n) \leq 0$ (단, $x \neq n, x \neq 3n$)
따라서 주어진 분수부등식의 해는 
$x < n$ 또는 $2n \leq x < 3n$
그런데 $120 \in A$ 이므로 
$120 < n$ 또는 $2n \leq 120 < 3n$
따라서 자연수 $n$ 의 최솟값은 $41$ 이다.

**[다른 풀이]**
$x=120$ 을 $\frac{x-2n}{(x-n)(x-3n)} \leq 0$ 에 대입하면 
$$\frac{120-2n}{(120-n)(120-3n)} \leq 0$$
$$\frac{n-60}{(n-120)(n-40)} \geq 0$$
$\Leftrightarrow (n-40)(n-60)(n-120) \geq 0$ (단, $n \neq 40, n \neq 120$)
$40 < n \leq 60$ 또는 $n > 120$
따라서 자연수 $n$ 의 최솟값은 $41$ 이다.

정답: $41$
