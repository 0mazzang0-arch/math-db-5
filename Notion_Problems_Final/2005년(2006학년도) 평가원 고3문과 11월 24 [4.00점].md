## 문제
정의역이 $\{ x \mid 1 \le x \le 81 \}$ 인 함수 $y = (\log_{3} x) (\log_{\frac{1}{3}} x) + 2\log_{3} x + 10$의 최댓값을 $M$, 최솟값을 $m$이라 할 때, $M + m$의 값을 구하시오.

## 풀이
$\log_{3} x = t$라 두면 ($1 \le x \le 81$이므로 $0 \le t \le 4$)
$y = t(-t) + 2t + 10$
$= -t^2 + 2t + 10$
$= -(t - 1)^2 + 11$

(i) $t = 4$ 일 때, 최솟값 $m = -(4 - 1)^2 + 11 = 2$
(ii) $t = 1$ 일 때, 최댓값 $M = -(1 - 1)^2 + 11 = 11$

$\therefore M + m = 11 + 2 = 13$

정답: $13$
