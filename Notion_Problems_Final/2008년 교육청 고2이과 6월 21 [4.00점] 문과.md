## 문제
$A$ 축을 돌려서 발생한 동력을 체인을 통하여 $B$ 축에 전달하는 장치가 있다. $A$ 축에는 $A_{1}$, $A_{2}$, $A_{3}$, $B$ 축에는 $B_{1}$, $B_{2}$, $B_{3}$, $\dots$, $B_{7}$ 의 크기가 서로 다른 톱니바퀴가 달려 있다. $A$ 축을 일정한 속력으로 돌릴 때, $B$ 축의 속력은 다음과 같다.

$$
\begin{array}{|l|}
\hline
\text{Ⅰ. } A_{i+1} \text{과 } B_{j} \text{를 연결할 때의 속력은 } A_{i} \text{와 } B_{j} \text{를 연결할 때의 속력보다 } 15\% \text{ 증가한다.} \\
\quad (i=1, 2, j=1, 2, 3, \dots, 7) \\
\text{Ⅱ. } A_{i} \text{와 } B_{j+1} \text{을 연결할 때의 속력은 } A_{i} \text{와 } B_{j} \text{를 연결할 때의 속력보다 } 20\% \text{ 증가한다.} \\
\quad (i=1, 2, 3, j=1, 2, 3, \dots, 6) \\
\hline
\end{array}
$$

이때, $B$ 축의 속력의 최댓값은 최솟값의 몇 배인가?
(단, $\log 1.15 = 0.06$, $\log 1.2 = 0.08$, $\log 2 = 0.30$)



> [!example] 📸 (그림/그래프)

## 보기
① $1.15$   ② $1.2$   ③ $2$   ④ $2.3$   ⑤ $4$

## 풀이
$A$ 축의 톱니바퀴 번호를 $i$, $B$ 축의 톱니바퀴 번호를 $j$라 하고, 이때의 $B$ 축의 속력을 $v(i, j)$라 하자.

조건 Ⅰ에 의하여 $A$ 축의 톱니바퀴 번호가 $1$ 증가할 때마다 속력이 $1.15$배가 되므로,
$v(i, j) = v(1, j) \times (1.15)^{i-1}$ 이 성립한다.

조건 Ⅱ에 의하여 $B$ 축의 톱니바퀴 번호가 $1$ 증가할 때마다 속력이 $1.2$배가 되므로,
$v(i, j) = v(i, 1) \times (1.2)^{j-1}$ 이 성립한다.

따라서 임의의 $i, j$에 대하여 $B$ 축의 속력 $v(i, j)$는 다음과 같다.
$v(i, j) = v(1, 1) \times (1.15)^{i-1} \times (1.2)^{j-1}$

$B$ 축 속력의 최솟값은 $i=1, j=1$일 때인 $v(1, 1)$이고,
최댓값은 $i=3, j=7$일 때인 $v(3, 7)$이다.
$v(3, 7) = v(1, 1) \times (1.15)^{3-1} \times (1.2)^{7-1} = v(1, 1) \times (1.15)^{2} \times (1.2)^{6}$

최댓값이 최솟값의 $k$배라고 하면,
$k = \frac{v(3, 7)}{v(1, 1)} = (1.15)^{2} \times (1.2)^{6}$

양변에 상용로그를 취하면
$\log k = \log ((1.15)^{2} \times (1.2)^{6})$
$\log k = 2 \log 1.15 + 6 \log 1.2$

주어진 조건 $\log 1.15 = 0.06$, $\log 1.2 = 0.08$을 대입하면
$\log k = 2(0.06) + 6(0.08) = 0.12 + 0.48 = 0.60$

이때 $\log 2 = 0.30$이므로
$\log k = 0.60 = 2 \times 0.30 = 2 \log 2 = \log 2^{2} = \log 4$

따라서 $k = 4$이다.
