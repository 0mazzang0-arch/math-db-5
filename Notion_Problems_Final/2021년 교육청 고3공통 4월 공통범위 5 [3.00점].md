## 문제
함수 $f(x)$에 대하여 $f'(x) = 2x + 4$이고 $f(-1) + f(1) = 0$일 때, $f(2)$의 값은?

## 보기
① $9$   ② $10$   ③ $11$   ④ $12$   ⑤ $13$

## 풀이
$f'(x) = 2x + 4$에서 
$f(x) = \int (2x + 4) dx = x^{2} + 4x + C$ ($C$는 적분상수)

$f(-1) + f(1) = 0$에서
$f(-1) = (-1)^{2} + 4(-1) + C = C - 3$
$f(1) = 1^{2} + 4(1) + C = C + 5$

$(C - 3) + (C + 5) = 2C + 2 = 0$
$C = -1$

이므로 $f(x) = x^{2} + 4x - 1$이다.
따라서 $f(2) = 2^{2} + 4(2) - 1 = 11$

정답: ③
